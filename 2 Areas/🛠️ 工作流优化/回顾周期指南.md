## 🔄 回顾体系概览

### 回顾频率表

| 周期 | 时长 | 关注重点 | 产出 |
|------|------|----------|------|
| **每日回顾** | 10-15 分钟 | 当日任务完成、明日计划 | 每日待办清单 |
| **每周回顾** | 30-60 分钟 | 项目进展、收件箱清理 | 周行动计划 |
| **每月回顾** | 1-2 小时 | Areas 维护、目标审视 | 月度总结 |
| **每季回顾** | 2-4 小时 | 方向调整、系统优化 | 季度计划 |
| **每年回顾** | 半天-一天 | 年度复盘、新年规划 | 年度总结 |

## 📅 每日回顾

### 时间安排

**推荐时段**: 晚间 21:00-21:30 或睡前

### 检查清单

- [ ] **今日完成**
  - [ ] 完成的主要任务
  - [ ] 重要决策
  - [ ] 学习收获

- [ ] **收件箱处理**
  - [ ] 所有新笔记已分类
  - [ ] 无用信息已删除

- [ ] **明日计划**
  - [ ] 最重要的 3 件事
  - [ ] 已排入日程的任务

- [ ] **问题记录**
  - [ ] 遇到的阻碍
  - [ ] 需要跟进的事项

### 每日回顾模板

```markdown
## 📅 每日回顾 - {{date:YYYY-MM-DD}}

### ✅ 今日完成
1. 完成事项 1
2. 完成事项 2

### 📥 收件箱状态
- 当前数量: X 条
- 已处理: Y 条

### 🎯 明日重点
1. 重点任务 1
2. 重点任务 2
3. 重点任务 3

### ⚠️ 注意事项
- 需要跟进的事项
```

## 📆 每周回顾

### 时间安排

**推荐时段**: 周五下午或周日晚上

### 检查清单

- [ ] **Projects 审查**
  - [ ] 每个进行中项目的状态
  - [ ] 下周需要推进的任务
  - [ ] 阻碍因素及解决方案

- [ ] **Areas 维护**
  - [ ] 各领域健康状况
  - [ ] 是否需要调整维护标准

- [ ] **收件箱清理**
  - [ ] 处理所有待分类笔记
  - [ ] 删除无用内容

- [ ] **Resources 更新**
  - [ ] 添加的新资源
  - [ ] 清理过时资料

- [ ] **下周规划**
  - [ ] 设定下周目标
  - [ ] 安排重要会议/任务

### 每周回顾模板

```markdown
## 📆 周回顾 - {{date:YYYY年第ww周}}

### 📊 本周概况
- 完成项目数: X
- 新增笔记数: Y
- 收件箱处理率: Z%

### 🎯 Projects 状态
- 进行中: X 个
- 本周完成: Y 个
- 新启动: Z 个

### 🔄 Areas 维护
| 领域 | 状态 | 备注 |
|------|------|------|
| 健康 | ✅ | 本周运动 3 次 |
| 财务 | ⚠️ | 需要处理账单 |

### 📋 下周计划
1. 重点任务 1
2. 重点任务 2
3. 重点任务 3

### 💡 本周洞察
- 学到的经验
- 需要改进的方面
```

## 📆 每月回顾

### 时间安排

**推荐时段**: 每月最后一个周五或次月第一天

### 检查清单

- [ ] **Areas 深度维护**
  - [ ] 每个领域的当前状态
  - [ ] 维护标准是否需要调整
  - [ ] 是否需要新项目支持

- [ ] **Projects 总结**
  - [ ] 本月完成的项目
  - [ ] 进行中项目的进展
  - [ ] 取消或搁置的项目

- [ ] **Goals 审视**
  - [ ] 年度目标进展
  - [ ] 是否需要调整方向

- [ ] **系统优化**
  - [ ] 工作流是否顺畅
  - [ ] 需要改进的流程

### 每月回顾模板

```markdown
## 📆 {{date:YYYY年MM月}} 月回顾

### 🎯 目标回顾
- 月初设定的目标
- 完成情况自评 (1-10)

### 📊 数据统计
- 完成项目: X 个
- 新增笔记: Y 篇
- 收件箱峰值: Z 条

### 🔄 Areas 状态
| 领域 | 健康度 | 维护行动 |
|------|--------|----------|
| 健康 | 8/10 | 继续保持运动频率 |
| 财务 | 6/10 | 需要关注预算 |

### 📈 本月成就
1. 成就 1
2. 成就 2

### ⚠️ 待改进
1. 问题 1
2. 问题 2

### 🎯 下月重点
1. 目标 1
2. 目标 2
3. 目标 3
```

## 💡 最佳实践

### ✅ 应该做的

1. **固定时间**: 设定固定的回顾时间，养成习惯
2. **创造环境**: 选择安静的环境专注回顾
3. **诚实面对**: 如实评估，避免自我欺骗
4. **产出行动**: 回顾要有具体产出和改进行动

### ❌ 不该做的

1. **流于形式**: 只填表格不思考
2. **过度自责**: 关注改进而非批评
3. **忽视问题**: 发现问题要及时解决
4. **拖延回顾**: 不要跳过或延迟回顾
