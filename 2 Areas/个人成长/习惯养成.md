## 💡 核心想法

习惯是一种自动化行为，可以：
- ⚡ 减少决策疲劳
- 🔄 提供稳定的行为基础
- 🎯 塑造长期身份
- 🌱 逐步实现目标

## 🧠 习惯循环

### 习惯的三个要素

```
┌─────────────────────────────────────────┐
│          习惯循环                   │
├─────────────────────────────────────────┤
│                                     │
│   ┌─────────┐    ┌─────────┐       │
│   │  提示   │ →  │  行为   │       │
│   └─────────┘    └─────────┘       │
│        ↑             ↓              │
│   ┌────────────────────┐            │
│   │     奖励          │            │
│   └────────────────────┘            │
│                                     │
└─────────────────────────────────────────┘
```

| 要素 | 说明 | 示例 |
|------|------|------|
| **提示** (Cue) | 触发习惯的信号 | 看到跑步鞋 |
| **行为** (Routine) | 执行的具体行动 | 穿鞋出门跑步 |
| **奖励** (Reward) | 行为带来的回报 | 内啡肽分泌、满足感 |

## 🛠️ 习惯养成方法

### 微习惯策略

> 从"不可能失败"的小习惯开始

| 常见目标 | 微习惯版本 |
|---------|-----------|
| 每天运动30分钟 | 每天做1个俯卧撑 |
| 每天读一本书 | 每天读1页 |
| 每天写日记 | 每天写1句话 |
| 每天冥想10分钟 | 每天冥想1分钟 |

**原理**：降低启动门槛，确保每天都有成功体验

### 21天习惯养成

| 阶段 | 天数 | 特点 | 策略 |
|------|------|------|------|
| **抵抗期** | 1-3天 | 新鲜但困难 | 严格执行 |
| **不稳定期** | 4-7天 | 容易放弃 | 适应调整 |
| **稳定期** | 8-21天 | 逐渐适应 | 持续强化 |
| **养成期** | 22-90天 | 自动化 | 巩固巩固 |

### 习惯叠加

> 在现有习惯上叠加新习惯

```
现有习惯 → 新习惯

示例：
- 刷牙后 → 立刻做5个深蹲
- 喝咖啡后 → 列出今日3个优先事项
- 晚餐后 → 写5分钟日记
```

## 📊 习惯追踪

### 追踪方法

| 方法 | 工具 | 优点 | 缺点 |
|------|------|------|------|
| **打勾表** | 纸笔/Excel | 简单直接 | 难以长期坚持 |
| **App 追踪** | Habitica, Streaks | 游戏化激励 | 可能分心 |
| **日记记录** | Obsidian 日记 | 深度反思 | 需要时间 |
| **可视化** | 墙贴日历 | 视觉激励 | 位置固定 |

## 🔄 习惯破坏与重建

### 识别坏习惯

使用 [[反思-闪念]] 中的 KPT 方法：

| Keep | Problem | Try |
|------|---------|------|
| 刷牙后 | 刷手机太长时间 | 替换为读1页书 |
| 午餐后 | 暴饮暴食 | 替换为散步10分钟 |

### 重建习惯

1. **识别提示** - 什么触发了坏习惯？
2. **替换行为** - 用更好的行为替代
3. **保持奖励** - 保留原有的奖励机制

## 📚 习惯的价值

### 短期价值
- ✅ 减少决策疲劳
- ✅ 提高执行力
- ✅ 建立日常节奏

### 长期价值
- 🎯 塑造身份认同
- 📈 累积复利效应
- 🌟 自动化成功

---

> [!tip] 实践建议
> 1. **一次一个** - 不要同时培养太多习惯
> 2. **从小开始** - 使用微习惯降低门槛
> 3. **持续追踪** - 可视化进度激励坚持
> 4. **允许失败** - 遗漏一天不要放弃，立即恢复
