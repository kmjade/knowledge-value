# Enterpy Trade 使用指南 v1.0

> 从安装到运行的完整操作手册。面向 EA 使用者，无需编程背景。

---

## 1. 快速开始（5 分钟）

### 1.1 安装

```
1. 关闭 MetaTrader 4 / 5
2. 将 Enterpy_v2 文件夹复制到 MT 数据目录：
   MT4: D:\Trade\MetaTrader 4\MQL4\
   MT5: D:\Trade\MetaTrader 5\MQL5\
3. 重启 MT4 / MT5
4. 按 F4 打开 MetaEditor
5. 找到 Experts → Enterpy_v2 → Enterpy.mq4，点 Compile (F7)
6. 同样编译 Indicators → Enterpy_v2 下的两个指标
```

**确认安装成功**：Navigator 面板 → Expert Advisors → 出现 `Enterpy_v2`；Indicators → 出现 `Enterpy_v2` 分组。

### 1.2 首次运行

```
1. 打开 EURUSD 日线图 (D1)
2. Navigator → Indicators → Enterpy_v2 → 拖入 Enterpy Donchian
   参数保持默认 (入场55 / 出场20)
3. Navigator → Indicators → Enterpy_v2 → 拖入 Enterpy ATR
4. Navigator → Expert Advisors → 拖入 Enterpy_v2 到图表
5. 参数保持默认，勾选 "Allow Auto Trading"，点 OK
6. 右上角笑脸图标确保是 ▶ 播放状态（自动交易已启用）
```

**检查运行状态**：图表右上角应出现 EA 名称 `Enterpy_v2` 和笑脸图标。MT4 终端 → Experts 标签应显示 `[INFO] Enterpy 交易系统 v0.1 正在初始化...` 和 `[INFO] 初始化完成，就绪。`

## 2. EA 参数配置

### 2.1 海龟策略参数

| 参数 | 默认 | 说明 | 调参建议 |
|:-----|:---:|:-----|:-----|
| S1 突破周期 | 20 | 快系统入场 Donchian 周期 | 短线交易可降至 10-15 |
| S1 出场周期 | 10 | 快系统出场反向突破周期 | 通常为入场周期的一半 |
| S2 突破周期 | 55 | 慢系统入场 Donchian 周期 | 长线交易可升至 80-100 |
| S2 出场周期 | 20 | 慢系统出场反向突破周期 | 通常为入场周期的 1/3 |
| ATR 计算周期 | 20 | ATR 平滑周期 | 海龟原版固定 20，不建议修改 |

### 2.2 仓位管理

| 参数 | 默认 | 说明 | 调参建议 |
|:-----|:---:|:-----|:-----|
| 每单位风险 | 1.0% | 单 Unit 占净值百分比 | 保守 0.5%，激进 2%。不超过 3% |
| 每系统最大单位 | 4 | 金字塔加仓上限 | 海龟原版 4，不建议超过 |
| 加仓步长 | 0.5 ATR | 两次加仓之间的 ATR 间距 | 海龟原版 0.5，短线可降至 0.25 |

### 2.3 止损设置

| 参数 | 默认 | 说明 | 调参建议 |
|:-----|:---:|:-----|:-----|
| 硬止损 | 2.0 ATR | 入场后最大容忍反向幅度 | 海龟原版 2.0，不低于 1.5 |
| 保本触发 | 1.0 ATR | 盈利多少后止损移至入场价 | 1.0-1.5，取决于波动率 |

### 2.4 风控参数

| 参数 | 默认 | 说明 | 调参建议 |
|:-----|:---:|:-----|:-----|
| 最大回撤暂停 | 20% | DD 超此值全平+停交易 | 实盘建议 15-20% |
| 恢复交易阈值 | 10% | DD 回到此值以下恢复交易 | 必须小于最大回撤 |

### 2.5 交易管理

| 参数 | 默认 | 说明 |
|:-----|:---:|:-----|
| 启用 S1 系统 | true | 20 日快系统开关 |
| 启用 S2 系统 | true | 55 日慢系统开关 |
| 固定手数 | 0 | 0=自动计算，>0=固定手数 |
| 魔术号 | 20240605 | 区分不同 EA 的订单 |
| 订单注释 | Enterpy | 显示在订单描述中 |

## 3. 指标使用

### 3.1 Donchian 通道

**拖入图表后看到的**：

```
┌──────────────────────────────────────┐
│  K 线图                               │
│                                      │
│  ━━━━━━━━ 绿线：入场 上轨 (55日) ━━━━ │
│  ━━━━━━━━ 红线：入场 下轨 (55日) ━━━━ │
│  --- 金线：出场 上轨 (20日) ---       │
│  --- 金线：出场 下轨 (20日) ---       │
│                                      │
│  绿色半透明带：入场通道填充区           │
│  ↑ 箭头：做多突破信号                 │
│  ↓ 箭头：做空突破信号                 │
└──────────────────────────────────────┘
```

**参数说明**：

| 参数 | 默认 | 用途 |
|:-----|:---:|:-----|
| EntryPeriod | 55 | 入场通道周期。S2 用 55，S1 用 20 |
| ExitPeriod | 20 | 出场通道周期 |
| ShowEntry | ON | 显示入场通道（绿/红线） |
| ShowExit | ON | 显示出场通道（金线） |
| ShowFill | ON | 显示通道填充色带 |
| ShowArrows | ON | 显示突破箭头标记 |
| 颜色参数 | 绿/红/金 | 可自定义各条线的颜色 |

**多周期部署**：

| 操作 | 用途 |
|:-----|:-----|
| 拖入一次，默认 55/20 | 观察 S2 慢系统 |
| 再拖入一次，EntryPeriod=20, ExitPeriod=10 | 观察 S1 快系统 |
| 用不同颜色区分 | S1 用蓝/红，S2 用绿/红 |

### 3.2 ATR 指标

拖入后在图表下方副图窗口显示一条蓝色曲线。

**解读**：
- 曲线上升 → 波动率增大（大行情、新闻事件）
- 曲线下降 → 波动率减小（盘整、低波动）
- 数值代表当前 ATR，单位与报价一致

**与 EA 的关系**：
- ATR 高 → 止损变宽、仓位变小
- ATR 低 → 止损变窄、仓位变大
- 这是自动的，无需手动干预

## 4. 交易信号解读

### 4.1 入场

| 信号 | 图表表现 | EA 动作 |
|:-----|:--------|:------|
| 做多 | 价格从下往上突破绿线（入场 上轨） | 开多仓 1 Unit，设 2 ATR 止损 |
| 做空 | 价格从上往下突破红线（入场 下轨） | 开空仓 1 Unit，设 2 ATR 止损 |
| 加仓 | 持仓中价格继续朝有利方向移动 ½ ATR | 加 1 Unit，最多 4 Unit |

**注意**：信号基于**收盘价**——不是盘中突破。等待 K 线收盘后下一根开盘时判断。

### 4.2 出场

| 信号 | 图表表现 | EA 动作 |
|:-----|:--------|:------|
| 多头出场 | 价格从上往下突破金线（出场 下轨） | 全平该系统多仓 |
| 空头出场 | 价格从下往上突破金线（出场 上轨） | 全平该系统空仓 |
| 止损出场 | 价格反向 2 ATR | 平掉该单位 |
| 回撤出场 | 净值回撤 > 20% | 全平所有仓位 + 暂停交易 |

## 5. 推荐品种与周期

### 5.1 推荐周期

| 周期 | 适用场景 | 信号频率 | 建议 |
|:-----|:--------|:------:|:-----|
| D1 日线 | **推荐**。海龟原版周期 | 每月 1-3 次 | 最适合海龟策略 |
| H4 四小时 | 偏短线，信号较多 | 每周 2-5 次 | 需要更大止损容忍 |
| H1 一小时 | 短线，噪音多 | 每天 1-3 次 | 假突破风险高，不推荐 |

### 5.2 推荐品种

| 品种 | 特性 | 适合度 |
|:-----|:-----|:-----:|
| EURUSD | 流动性最好，趋势性强 | ⭐⭐⭐⭐⭐ |
| GBPUSD | 波动大，趋势延续性好 | ⭐⭐⭐⭐ |
| USDJPY | 趋势性强，适合突破策略 | ⭐⭐⭐⭐ |
| XAUUSD | 波动极大，ATR 高 | ⭐⭐⭐（需调高止损倍数） |
| 交叉盘 | 流动性差，假突破多 | ⭐⭐ |

## 6. 日常操作

### 6.1 每日检查

```
1. 检查 MT4 终端 Experts 标签无 ERROR 日志
2. 检查图表右上角 EA 笑脸是 ▶ 状态
3. 打开 MT4 → 终端 → 交易，查看当前持仓
4. 观察 Donchian 通道：价格是否接近上/下轨
```

### 6.2 日志查看

日志文件位置（MT4 数据目录 `MQL4\Files\`）：

| 文件 | 内容 |
|:-----|:-----|
| `Enterpy_EURUSD_2026-06-05.log` | 每日运行日志 |
| `Enterpy_Trades_EURUSD_2026-06-05.csv` | CSV 交易记录 |

**CSV 导入 Excel**：数据 → 从文本/CSV → 逗号分隔 → 加载。可做数据透视分析。

### 6.3 暂停与恢复

| 操作 | 方法 |
|:-----|:-----|
| 暂停 EA | 点击图表右上角 EA 图标 → 右键 → Remove（或点 ▷ 变成 ∥） |
| 手动平仓 | MT4 终端 → 交易 → 右键持仓 → 平仓 |
| 恢复 EA | 重新拖入 EA 到图表，EA 会自动同步持仓状态 |
| 修改参数 | 图表右键 → Expert Advisors → Properties → Inputs 标签 |

## 7. 常见问题

### 7.1 EA 不开仓

| 原因 | 检查 |
|:-----|:-----|
| 自动交易未启用 | 右上角是否是 ▶ 播放状态 |
| 回撤保护触发 | Experts 标签是否显示 "最大回撤超限" |
| 无信号 | 价格是否在通道内（未突破） |
| 已达到最大单位 | 持仓是否已达到 8 Unit |
| 魔术号冲突 | 是否有其他 EA 使用相同 MagicNumber |

### 7.2 EA 重复开仓

| 原因 | 解决 |
|:-----|:-----|
| EA 拖了两次到图表 | 同品种只拖一次 |
| 多图表同时运行 | 每个品种只在一个图表上运行 EA |
| 旧版 Enterpy 同时在跑 | 检查是否有旧版 Enterpy EA 也在运行 |

### 7.3 止损被频繁触发

| 原因 | 调整 |
|:-----|:-----|
| 止损太紧 | 加大 `StopLoss_ATR`（如 2.0 → 2.5） |
| 周期太小 | 切换到更大周期（H1 → H4 → D1） |
| 品种波动大 | 换波动较小的品种 |

## 8. 参数模板

### 保守型（实盘推荐）

```
S1 突破周期     = 20
S2 突破周期     = 55
每单位风险      = 0.5%
硬止损          = 2.5 ATR
最大回撤暂停    = 15%
```

### 标准型（海龟原版）

```
S1 突破周期     = 20
S2 突破周期     = 55
每单位风险      = 1.0%
硬止损          = 2.0 ATR
最大回撤暂停    = 20%
```

### 激进型（仅模拟盘）

```
S1 突破周期     = 10
S2 突破周期     = 30
每单位风险      = 2.0%
硬止损          = 2.0 ATR
最大回撤暂停    = 25%
```
