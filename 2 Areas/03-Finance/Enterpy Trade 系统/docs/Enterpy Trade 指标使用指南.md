# Enterpy Trade 指标使用指南 v1.0

> Enterpy ATR 与 Enterpy Donchian 图表指标的安装、配置、解读与实战技巧。

---

## 1. 安装

### 1.1 确认文件存在

```
MT4: D:\Trade\MetaTrader 4\MQL4\Indicators\Enterpy_v2\
     ├── Enterpy_ATR.mq4
     └── Enterpy_Donchian.mq4

MT5: D:\Trade\MetaTrader 5\MQL5\Indicators\Enterpy_v2\
     ├── Enterpy_ATR.mq5
     └── Enterpy_Donchian.mq5
```

### 1.2 编译

```
1. 打开 MT4/MT5 → 按 F4 进入 MetaEditor
2. 左侧 Navigator → Indicators → Enterpy_v2
3. 右键 Enterpy_ATR → Compile
4. 右键 Enterpy_Donchian → Compile
5. 确认底部 Errors 标签为 0 errors, 0 warnings
```

### 1.3 加载到图表

```
方法一：Navigator 拖入
  Navigator → Indicators → Enterpy_v2 → 拖到图表上 → 点 OK

方法二：菜单插入
  插入 → 技术指标 → 自定义 → Enterpy Donchian / Enterpy ATR
```

## 2. Enterpy Donchian 指标

### 2.1 界面总览

```
┌──────────────────────────────────────────────────────┐
│  K 线图                                               │
│                                                      │
│  ═══════════ 绿实线：入场 上轨 (55日最高价) ══════════ │
│  ┌──────────────────────────────────────────────┐    │
│  │  绿色半透明填充区（通道内部）                    │    │
│  └──────────────────────────────────────────────┘    │
│  ═══════════ 红实线：入场 下轨 (55日最低价) ══════════ │
│                                                      │
│  ------ 金虚线：出场 上轨 (20日最高价) ------         │
│  ------ 金虚线：出场 下轨 (20日最低价) ------         │
│                                                      │
│  ↑ 绿箭头：价格突破上轨 → 做多信号                    │
│  ↓ 粉箭头：价格跌破下轨 → 做空信号                    │
└──────────────────────────────────────────────────────┘
```

**6 个视觉元素**：

| # | 元素 | 样式 | 含义 |
|:-:|:-----|:-----|:-----|
| 1 | 入场 上轨 | 绿色实线 | 55 日最高价（可配置） |
| 2 | 入场 下轨 | 红色实线 | 55 日最低价（可配置） |
| 3 | 出场 上轨 | 金色虚线 | 20 日最高价（可配置） |
| 4 | 出场 下轨 | 金色虚线 | 20 日最低价（可配置） |
| 5 | 通道填充 | 绿色半透明 | 入场通道区域 |
| 6 | 突破箭头 | ↑绿 ↓粉 | 突破信号标记 |

### 2.2 参数面板

拖入指标时弹出的参数窗口：

```
┌─────────────────────────────────────┐
│  Enterpy Donchian                   │
├─────────────────────────────────────┤
│  入场通道周期         [55]          │
│  出场通道周期         [20]          │
│  显示入场通道         [✔]           │
│  显示出场通道         [✔]           │
│  显示通道填充         [✔]           │
│  显示突破箭头         [✔]           │
│  入场上轨颜色         [绿色]         │
│  入场下轨颜色         [红色]         │
│  出场上轨颜色         [金色]         │
│  出场下轨颜色         [金色]         │
│  填充颜色             [绿色]         │
│  填充透明度 (0-255)   [30]           │
└─────────────────────────────────────┘
```

**参数说明**：

| 参数 | 默认 | 作用 |
|:-----|:---:|:-----|
| `EntryPeriod` | 55 | 入场通道的 N 日窗口。值越大通道越宽，信号越少 |
| `ExitPeriod` | 20 | 出场通道的 N 日窗口。通常 ≤ 入场周期 |
| `ShowEntry` | ON | 关闭后隐藏绿/红线，仅保留出场通道 |
| `ShowExit` | ON | 关闭后隐藏金线，仅保留入场通道 |
| `ShowFill` | ON | 关闭后移除绿色填充，通道只剩线条 |
| `ShowArrows` | ON | 关闭后不显示突破箭头 |
| `EntryUpperColor` | 绿色 | 入场上轨线条颜色 |
| `EntryLowerColor` | 红色 | 入场下轨线条颜色 |
| `ExitUpperColor` | 金色 | 出场上轨线条颜色 |
| `ExitLowerColor` | 金色 | 出场下轨线条颜色 |
| `FillColor` | 绿色 | 通道填充颜色 |
| `FillAlpha` | 30 | 填充透明度。0=完全不透明，255=完全透明 |

### 2.3 信号识别

#### 做多信号

```
条件：收盘价从通道下方突破上轨

前一根 K 线：收盘价 ≤ 上轨
当前 K 线：收盘价 > 上轨
箭头：↑ 绿色箭头出现在 K 线下方

示例：
        ┃                 ← 上轨
    ╱   ┃
   ╱ ↑  ┃                 ← 突破！箭头标记
  ╱     ┃
 ╱      ┃
```

#### 做空信号

```
条件：收盘价从通道上方跌破下轨

前一根 K 线：收盘价 ≥ 下轨
当前 K 线：收盘价 < 下轨
箭头：↓ 粉色箭头出现在 K 线上方

示例：
        ┃
        ┃  ╲
        ┃ ↓ ╲              ← 突破！箭头标记
        ┃    ╲
   ━━━━━┃━━━━━━            ← 下轨
```

#### 假突破识别

| 特征 | 真突破 | 假突破 |
|:-----|:-----:|:-----:|
| K 线实体 | 大阳/大阴线 | 小实体长影线 |
| 成交量 | 放量 | 缩量 |
| 突破后走势 | 继续朝突破方向 | 立即回撤至通道内 |
| 后续 K 线 | 连续同向 | 反转吞没 |

**提示**：箭头标记基于**收盘价**。盘中价格刺穿上轨但收盘回落不属于突破信号，不会标记箭头。

### 2.4 实战用法

#### 用法一：单指标观察 S2 慢系统

```
设置：EntryPeriod=55, ExitPeriod=20
用途：捕捉中长期趋势
品种：EURUSD D1
信号频率：每月 1-3 次
```

#### 用法二：双指标对比 S1+S2

```
指标 1（S1）：EntryPeriod=20, ExitPeriod=10, 颜色蓝/红
指标 2（S2）：EntryPeriod=55, ExitPeriod=20, 颜色绿/红

叠加效果：
  ━━━ 蓝线：S1 入场 上轨 (20日)
  ═══ 绿线：S2 入场 上轨 (55日)
  
当价格突破蓝线但未突破绿线：
  → S1 有信号，S2 无信号（仅快系统入场）
  
当价格同时突破蓝线和绿线：
  → S1 和 S2 都有信号（双系统共振，强信号）
```

#### 用法三：仅看入场通道

```
设置：ShowExit=OFF, ShowFill=ON
用途：简洁视图，只关注入场机会
适合：新手，减少视觉干扰
```

#### 用法四：通道收缩预警

```
观察通道宽度：
  通道持续收窄 → 波动率下降 → 突破临近
  通道持续扩张 → 波动率上升 → 趋势进行中
  
操作提示：
  收窄期间降低仓位预期，等待方向选择
  扩张期间顺势持有，注意出场信号
```

## 3. Enterpy ATR 指标

### 3.1 界面

```
┌──────────────────────────────────┐
│  Enterpy ATR(20)                  │  ← 副图窗口
│                                   │
│  0.00800 ┤                         │
│          │    ╱╲                    │
│  0.00600 ┤   ╱  ╲    ╱╲            │
│          │  ╱    ╲  ╱  ╲  ╱╲      │
│  0.00400 ┤ ╱      ╲╱    ╲╱  ╲     │
│          │╱                    ╲   │
│  0.00200 ┤                         │
│          │                         │
│  当前值: 0.00543 (54.3 点)          │
└──────────────────────────────────┘
```

### 3.2 参数

| 参数 | 默认 | 说明 |
|:-----|:---:|:-----|
| `ATR_Period` | 20 | ATR 平滑周期。海龟标准为 20 |

**只有一个参数**——设计原则是极简。

### 3.3 ATR 数值含义

ATR 是平均真实波幅，单位与报价一致。

| 品种 | 典型 ATR(20) D1 | 含义 |
|:-----|:-------------:|:-----|
| EURUSD | 0.0050-0.0080 | 日均波动 50-80 点 |
| GBPUSD | 0.0080-0.0120 | 日均波动 80-120 点 |
| USDJPY | 0.60-1.00 | 日均波动 60-100 点 |
| XAUUSD | 15-30 | 日均波动 15-30 美元 |

**ATR 与你的仓位**：1% 风险下，EURUSD 的 1 Unit ≈ ATR 越小仓位越大。

### 3.4 ATR 走势解读

```
ATR 上升（波动加剧）：
  ─── 通常伴随趋势行情、重大新闻、开盘跳空
  → 止损会放宽（2 ATR 的绝对值变大）
  → 仓位会减小（Unit 变小）
  → 加仓间距变大（0.5 ATR 的绝对值变大）

ATR 下降（波动收窄）：
  ─── 通常伴随盘整、假日、低流动性时段
  → 止损会收紧
  → 仓位会增大
  → 加仓间距变小

ATR 横盘（波动稳定）：
  → 策略参数稳定，适合趋势跟踪
```

### 3.5 实战技巧

#### 技巧一：用 ATR 预估止损距离

```
当前 EURUSD = 1.08500
当前 ATR(20) = 0.00600 (60 点)
止损 = 2 × 60 = 120 点

做多入场价 1.08500 → 止损价 1.08500 - 0.01200 = 1.07300
做空入场价 1.08500 → 止损价 1.08500 + 0.01200 = 1.09700
```

#### 技巧二：用 ATR 判断加仓时机

```
入场价 1.08500，ATR = 0.00600
第 1 次加仓触发价 = 1.08500 + 0.5 × 0.00600 = 1.08800
第 2 次加仓触发价 = 1.08500 + 1.0 × 0.00600 = 1.09100
第 3 次加仓触发价 = 1.08500 + 1.5 × 0.00600 = 1.09400
```

#### 技巧三：ATR 相对值判断

```
当前 ATR / 20 日平均 ATR > 1.5
  → 波动率异常偏高 → 谨慎开仓，可能处于极端行情

当前 ATR / 20 日平均 ATR < 0.5
  → 波动率异常偏低 → 突破可能临近，关注通道收窄
```

## 4. 指标组合使用

### 4.1 推荐图表布局

```
┌──────────────────────────────────────┐
│  主图：K 线 + Donchian 通道            │
│  ┌────────────────────────────────┐  │
│  │  ═══ 上轨                        │  │
│  │  K线  K线  K线                   │  │
│  │  ═══ 下轨                        │  │
│  └────────────────────────────────┘  │
├──────────────────────────────────────┤
│  副图 1：Enterpy ATR(20)              │
│  ┌────────────────────────────────┐  │
│  │  ATR 曲线                        │  │
│  └────────────────────────────────┘  │
└──────────────────────────────────────┘
```

**保存模板**：图表右键 → 模板 → 保存模板 → 命名为 `Enterpy`。以后右键 → 模板 → Enterpy 即可一键加载。

### 4.2 信号确认流程

```
Step 1: 看 Donchian — 价格接近上轨还是下轨？
         ↓
Step 2: 看 Donchian — 通道在扩张还是收窄？
         ↓
Step 3: 看 ATR — 波动率是升是降？
         ↓
Step 4: 看箭头 — 是否有突破信号标记？
         ↓
Step 5: 结合 EA 持仓 — 是否已入场？是否需要加仓？
```

## 5. 常见问题

### Donchian

| 问题 | 原因 | 解决 |
|:-----|:-----|:-----|
| 通道太宽，看不清 K 线 | 周期太大 | 减小 EntryPeriod |
| 箭头不出现 | ShowArrows=OFF 或无突破 | 检查参数；等待信号 |
| 填充遮挡 K 线 | 透明度太低 | 增大 FillAlpha (30→80) |
| 四个通道太乱 | 信息过载 | ShowExit=OFF，单看入场 |
| 线条颜色看不清 | 与背景色冲突 | 改为白色/黑色等高对比度色 |

### ATR

| 问题 | 原因 | 解决 |
|:-----|:-----|:-----|
| ATR 值太小看不清 | 品种报价位数多 | 无解，正常现象（如 EURUSD 0.005） |
| ATR 曲线太平 | 周期过长 | 减小 ATR_Period (20→10) |
| 副图空白 | 数据不足 | 等待 ATR_Period 根 K 线后自动出现 |
