# 財務規劃

## 📌 標準
- 每月記錄所有收支，了解金錢流向
- 儲蓄率維持在收入的 20% 以上
- 定期檢視投資組合表現
- 確保有足夠的緊急備用金
- 每年進行一次財務健康檢查

## 📅 財務目標

### 短期目標（1年內）
- [ ] 建立 3 個月緊急備用金
- [ ] 還清信用卡債務
- [ ] 設定自動儲蓄

### 中期目標（1-5年）
- [ ] 建立 6 個月緊急備用金
- [ ] 買房頭期款
- [ ] 投資帳戶達到 X 萬

### 長期目標（5年以上）
- [ ] 退休金帳戶達到 Y 萬
- [ ] 財務自由
- [ ] 被動收入覆蓋日常開支
