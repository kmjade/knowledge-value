# 运营流程

> [!info] 概述
> 健康养生馆日常运营标准化流程，从开店准备到关店收尾的完整工作流程。

## 开店前准备

### 每日开店流程（上午9:00前）

#### 环境准备
- [ ] 开启通风系统，更换新鲜空气
- [ ] 调节室内温度至舒适范围（22-24℃）
- [ ] 检查照明设备是否正常
- [ ] 开启背景音乐，调节音量适中
- [ ] 清洁整理各服务房间

#### 物品准备
- [ ] 检查消毒用品充足（酒精、消毒液等）
- [ ] 准备一次性用品（床单、纸巾、水杯等）
- [ ] 整理工作台和工具设备
- [ ] 补充消耗品（茶叶、水等）
- [ ] 检查设备运行状态

#### 人员准备
- [ ] 员工签到打卡
- [ ] 检查员工仪容仪表
- [ ] 召开晨会，确认当日安排
- [ ] 分配工作任务
- [ ] 确认预约客户和技师安排

### 预约确认
- 查看当日预约记录
- 发送当日预约提醒
- 确认技师到岗情况
- 准备应对临时加客

## 服务接待流程

### 客户到店接待

#### 标准接待流程
1. **问候**: 热情主动问候客户
2. **签到**: 引导客户签到登记
3. **需求了解**: 询问客户今日服务需求
4. **物品存放**: 协助客户存放个人物品
5. **基础服务**: 提供茶水、更换拖鞋

#### 预约客户接待
1. 确认预约信息和时间
2. 介绍当日服务技师
3. 引导客户进入服务房间
4. 协助客户准备服务

#### 散客接待
1. 询问客户需求和服务项目
2. 介绍可提供的服务和时间
3. 确认服务安排和技师
4. 引导客户进入服务流程

### 服务执行流程

#### 服务前
- 核对客户信息和需求
- 介绍服务流程和注意事项
- 准备所需工具和材料
- 调整房间环境

#### 服务中
- 按标准流程提供服务
- 定期询问客户感受
- 保持专业沟通
- 注意观察客户状态

#### 服务后
- 协助客户整理
- 询问服务感受
- 记录服务信息
- 预约下次服务

## 收款与送客流程

### 收款流程

#### 收款准备
- 确认服务项目和价格
- 检查优惠和会员权益
- 准备收银设备和票据

#### 收款执行
- 向客户说明服务项目和费用
- 确认付款方式（现金/微信/支付宝）
- 收取款项并核对
- 开具票据或小票

#### 收款后
- 感谢客户付款
- 介绍会员优惠（如适用）
- 邀请客户评价服务

### 送客流程
1. **整理**: 协助客户整理物品
2. **道别**: 礼貌道别
3. **送至门口**: 送到门口或电梯口
4. **送别**: 再次感谢光临

## 关店流程

### 每日关店流程（晚上21:00后）

#### 收尾工作
- [ ] 送走最后一批客户
- [ ] 清理公共区域
- [ ] 整理各服务房间
- [ ] 清洁工具和设备
- [ ] 记录当日服务数据

#### 设备关闭
- [ ] 关闭背景音乐
- [ ] 关闭非必要照明
- [ ] 关闭电器设备电源
- [ ] 检查电源安全
- [ ] 锁好门窗

#### 当日总结
- [ ] 统计当日营业数据
- [ ] 记录客户反馈
- [ ] 处理当日问题
- [ ] 规划明日工作
- [ ] 员工签退打卡

## 营业中异常处理

### 客户临时取消
- 礼貌表示理解
- 询问是否需要改期
- 记录取消原因
- 释放技师时间

### 客户迟到
- 电话联系确认情况
- 调整服务安排
- 等待时间过长时主动询问

### 设备故障
- 立即停止使用
- 向客户解释情况
- 提供替代方案
- 安排维修或更换

### 技师缺席
- 调整技师安排
- 与客户协商改期
- 提供适当补偿
- 事后跟进处理

## 周期性运营活动

### 周期活动

| 周期 | 活动内容 |
|------|----------|
| 每日 | 晨会、关店总结 |
| 每周 | 周例会、数据统计、卫生大扫除 |
| 每月 | 月度总结、员工培训、设备维护 |
| 每季 | 季度复盘、市场分析、促销活动 |
| 每年 | 年度总结、战略规划、年庆活动 |

### 周例会内容
- 上周运营数据回顾
- 客户反馈汇总分析
- 存在问题讨论
- 改进措施制定
- 本周工作安排

## 数据统计与分析

### 每日统计项
- 客流量
- 服务人次
- 营业收入
- 新增客户数
- 客户评价

### 周统计项
- 周客流量
- 周服务人次
- 周营业额
- 客户满意度
- 投诉统计

### 月统计项
- 月客流量
- 月服务人次
- 月营业额
- 服务项目分析
- 客户留存率
