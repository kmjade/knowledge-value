### 支出审批

#### 审批权限
- 小额支出（<500元）：前台负责人审批
- 中额支出（500-2000元）：店长审批
- 大额支出（>2000元）：店主审批

#### 审批流程
1. 填写支出申请单
2. 提供相关凭证
3. 逐级审批
4. 财务付款
5. 凭证归档

## 利润报表

### 利润计算

#### 毛利润
```
毛利润 = 收入总额 - 材料成本
```

#### 营业利润
```
营业利润 = 毛利润 - 人员支出 - 水电支出 - 其他运营支出
```

#### 净利润
```
净利润 = 营业利润 - 房租 - 折旧 - 财务费用 - 税费
```

## 财务分析

### 营收分析

#### 收入分析
- 收入来源分析
- 收入趋势分析

#### 客户分析
- 客户消费频次

### 成本分析

#### 成本率分析
```
成本率 = 成本总额 / 收入总额 × 100%
```

#### 费用率分析
- 人员费用率 = 人员支出 / 收入总额
- 材料费用率 = 材料支出 / 收入总额
- 营销费用率 = 营销支出 / 收入总额

### 利润分析

#### 利润率分析
```
利润率 = 净利润 / 收入总额 × 100%
```

## 财务档案管理

### 凭证管理

#### 收入凭证
- 收银小票
- 转账凭证
- 发票
- 收据

#### 支出凭证
- 发票
- 合同
- 收据
- 付款凭证

### 档案归档

#### 归档时间
- 每月凭证：次月15日前归档
- 合同文件：签署后立即归档
- 财务报表：编制后立即归档

#### 归档要求
- 分类整理
- 编号登记
- 妥善保管
- 便于查阅

### 保管期限

| 档案类型 | 保管期限 |
|----------|----------|
| 收支凭证 | 5年 |
| 合同文件 | 合同期满后2年 |
| 财务报表 | 5年 |
| 纳税凭证 | 永久 |
