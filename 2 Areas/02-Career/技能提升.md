## 学习方法

### 高效学习策略

#### 1. 70-20-10 法则

```
70% - 实践项目
├── 真实项目开发
├── 开源贡献
└── 技术博客写作

20% - 交流学习
├── 技术社区讨论
├── Code Review
└── 技术分享

10% - 正式学习
├── 在线课程
├── 书籍阅读
└── 官方文档
```

#### 2. 费曼学习法

1. **选择概念**: 选择要学习的主题
2. **教学讲解**: 用简单语言解释给他人
3. **识别空白**: 发现理解不足的地方
4. **重新学习**: 针对空白再次学习
5. **简化总结**: 用更简单的方式重新组织

#### 3. 项目驱动学习

以项目为中心的学习方式：

```
项目需求
    │
    ├─► 识别需要的技能
    │
    ├─► 边做边学
    │
    ├─► 遇到问题解决问题
    │
    └─► 完成项目，巩固技能
```
