# 配置

# 配置

#### 简体中文
1. **桥接模式（推荐）**
   - 关闭光猫的 DHCP 和路由功能
   - 光猫仅作为光电转换设备
   - PPPoE 拨号由主路由器完成

2. **路由模式（备选）**
   - 光猫进行 PPPoE 拨号
# 連接
# 配置

#### Traditional Chinese
1. **橋接模式（推薦）**
   - 關閉光貓的 DHCP 和路由功能
   - 光貓僅作為光電轉換設備
   - PPPoE 撥號由主路由器完成

2. **路由模式（備選）**
   - 光貓進行 PPPoE 撥號
   - 主路由器連接光貓 LAN 口
# 配置

#### English
1. **Bridge Mode (Recommended)**
   - Disable ONT's DHCP and routing
   - ONT acts as optical-to-electrical converter only
   - PPPoE dialing handled by main router

2. **Router Mode (Alternative)**
   - ONT handles PPPoE dialing
   - Main router connects to ONT LAN port
   - Requires DMZ or port forwarding

### 防火墙规则 / Firewall Rules

```yaml
# 简体中文 / Simplified Chinese
firewall:
  # 默认策略
  default_input: ACCEPT      # 默认接受入站
  default_output: ACCEPT     # 默认接受出站
# 轉發

  # 允许的规则
  rules:
    - name: "允许局域网访问"
      src: "192.168.1.0/24"
      dest: "192.168.1.1"
      action: ACCEPT

# 連接
      state: "ESTABLISHED,RELATED"
      action: ACCEPT

    - name: "允许 ICMP"
      proto: "icmp"
      action: ACCEPT

# Traditional Chinese / 繁體中文
firewall:
  # 預設策略
  default_input: ACCEPT
  default_output: ACCEPT
  default_forward: REJECT

  # 允許的規則
  rules:
    - name: "允許區域網訪問"
      src: "192.168.1.0/24"
      dest: "192.168.1.1"
      action: ACCEPT

    - name: "允許已建立的連接"
      state: "ESTABLISHED,RELATED"
      action: ACCEPT

    - name: "允許 ICMP"
      proto: "icmp"
      action: ACCEPT

# English
firewall:
  # Default policies
  default_input: ACCEPT
  default_output: ACCEPT
  default_forward: REJECT

  # Allowed rules
  rules:
    - name: "Allow LAN access"
      src: "192.168.1.0/24"
      dest: "192.168.1.1"
      action: ACCEPT

    - name: "Allow established connections"
      state: "ESTABLISHED,RELATED"
      action: ACCEPT

    - name: "Allow ICMP"
      proto: "icmp"
      action: ACCEPT
```
