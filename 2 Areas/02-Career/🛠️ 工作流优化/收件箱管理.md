## 📋 处理流程

### 每日处理（建议早晚各一次）

```
📥 打开收件箱
    ↓
# 筆記
    ↓
# 資訊
# 刪除
    └─ 是 → 下一步行动是什么？
            ↓
🎯 有下一步行动？
    ├─ 是 → 是否需要他人？
    │       ├─ 是 → 📋 等待清单
    │       └─ 否 → 是否在 2 分钟内完成？
    │               ├─ 是 → ⚡ 立即执行
# 專案
    └─ 否 → 有参考价值吗？
# 資源
# 刪除
```

### 2 分钟原则

如果一件事可以在 2 分钟内完成，**立即执行**，不要加入待办清单。

# 分類
|---------|------|--------|
| 需要采取行动吗？ | 否 | Resources 或 Archives |

| 只是供参考？ | 是 | Resources |
| 已完成不再需要？ | 是 | Archives |
