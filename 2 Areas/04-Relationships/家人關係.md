## 与父母关系

### 维护计划

| 频率 | 活动 | 说明 |
|------|------|------|
| 每周 1 次 | 电话问候 | 了解近况，简短交流 |
| 每月 1 次 | 视频通话 | 深入交流，解决关切 |
| 每月 1 次 | 实地探望 | 协助处理事务，陪伴聊天 |
| 节日 | 特殊安排 | 生日、节日礼物和聚餐 |

### 关心重点

**健康方面**：
- 定期询问身体状况
- 提醒体检和复查
- 协助购买药品和保健品

**生活方面**：
- 日常起居是否方便
- 家居环境是否安全
- 经济状况是否充足
- 社交活动是否丰富

**情感方面**：
- 倾听他们的烦恼和需求
- 表达感激和爱意
- 尊重他们的生活方式

### 待办

- [ ] 每周：电话问候，了解健康近况
- [ ] 每月：安排视频通话
- [ ] 每月：实地探望
- [ ] 年度：安排体检

## 与伴侣关系

### 维护节奏

| 频率 | 活动 |
|------|------|
| 每天 | 真诚交流，分享一天的经历和感受 |
| 每周 | 共同完成家务或活动 |
| 每月 | 安排一次约会（电影、餐厅、郊游） |
| 每年 | 庆祝纪念日和生日 |

### 沟通原则

1. **积极倾听** — 专注倾听不打断，理解对方需求，给予反馈
2. **表达欣赏** — 及时表达感谢和赞美，认可对方的付出
3. **尊重差异** — 接受不同观点和习惯，寻求妥协而非改变，保留个人空间
4. **共同成长** — 共同制定家庭目标，支持彼此的个人发展

### 共同目标

**短期（2026）**：
- [ ] 每月至少 1 次约会
- [ ] 每周共同完成 1 项家庭活动
- [ ] 建立「家庭会议」机制

**长期**：
- [ ] 家庭财务规划
- [ ] 子女教育规划
- [ ] 退休生活规划

## 与子女关系

### 教育理念

1. **尊重** — 尊重孩子的独立人格
2. **理解** — 理解孩子的成长阶段和需求
3. **引导** — 引导而非控制
4. **陪伴** — 高质量的陪伴时光

### 陪伴节奏

| 时段 | 活动 | 目的 |
|------|------|------|
| 早晨 | 共同早餐，准备上学 | 建立规律 |
| 晚间 | 阅读或游戏时间 | 情感交流 |
| 周末 | 户外活动或兴趣班 | 拓展视野 |

### 培养重点

**兴趣与能力**：
- 鼓励探索兴趣和天赋，不过度施加压力

**品格培养**：
- 以身作则，传递正面价值观
- 培养独立性和责任感
- 教导同理心和合作精神
- 鼓励表达和沟通

**身心健康**：
- 保证充足的睡眠和运动
- 创造安全的成长环境

### 待办

- [ ] 本月：安排一次家庭郊游
- [ ] 本月：参加一次学校活动
- [ ] 下月：规划暑假活动

## 家庭活动

### 每周常规

- **周五晚**：家庭电影夜
- **周六**：户外活动（公园、运动）
- **周日**：家庭聚餐或家务分工

### 每月常规

- 家庭会议（规划下月活动）
- 外出就餐（尝试新餐厅）
- 亲子活动（游乐园、博物馆）

### 季节性活动

| 季节 | 活动 |
|------|------|
| 春 | 踏青赏花、植树、户外野餐 |
| 夏 | 游泳、海滨度假、露营 |
| 秋 | 登山徒步、果园采摘、秋游 |
| 冬 | 滑雪/滑冰、温泉度假、室内活动 |

### 创意活动

- **家庭烹饪比赛**：每人负责一道菜
- **家庭才艺展示**：表演或作品展示
- **家庭读书会**：共同阅读一本书
- **家庭摄影日**：拍摄家庭合照
