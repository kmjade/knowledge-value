```
答案：O(log n)
解釋：Binary search divides the search interval in half each time.
```

# 方法

### 閃念複習規則

#### 間隔重複 Spaced Repetition

```
第1次：立即複習
第2次：1天後
第3次：3天後
第4次：1週後
第5次：2週後
第6次：1個月後
第7次：3個月後
```

#### 記憶技巧

**1. 聯想記憶法**
```
問題 + 圖像 = 記憶

示例：
問題：什麼是 binary search？
圖像：在字典裡找字，每次從中間翻
答案：O(log n) 的時間複雜度
```

**2. 記憶宮殿法**

```
將知識點放在熟悉的空間中

示例：
客廳 = Python 基礎
  - 沙發 = 數據類型
  - 電視 = 控制流語句
  - 茶几 = 函數
```

**3. 故事記憶法**

```
將知識編織成故事

示例：
變量的命名（變數是有名字的盒子）
  - `name` = 一個有標籤的盒子
  - `age` = 一個顯示年齡的數字顯示器
```

### 閃念創建原則

#### 1. 一張卡片一個概念

**✅ 好的做法：**
```
Q: What is the time complexity of binary search?
A: O(log n)
```

**❌ 不好的做法：**
```
Q: What are the time complexities of various sorting algorithms?
A: Bubble Sort: O(n²), Quick Sort: O(n log n), ...
```

#### 2. 使用簡潔清晰的語言

**✅ 好的做法：**
```
Q: What is the purpose of the `synchronized` keyword?
A: To control access to shared resources.
```

**❌ 不好的做法：**
```
Q: In multi-threaded programming, when we need to ensure that only one thread can access a shared resource at a time to avoid race conditions and data corruption, what Java keyword do we use?
A: synchronized
```

#### 3. 包含上下文但不過多

**✅ 好的做法：**
```
Q: What is overfitting?
A: Model performs well on training data but poorly on new data.
Context: Common in machine learning.
```

**❌ 不好的做法：**
```
Q: What is overfitting and how can we prevent it?
A: Overfitting is when a model learns the training data too well...
(too much information to remember)
```
