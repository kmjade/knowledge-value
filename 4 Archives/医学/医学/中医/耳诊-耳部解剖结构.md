# 耳诊 - 耳部解剖结构

耳部解剖結構是耳診的基礎，了解耳廓各部位對應的反射區對於準確診斷和治療至關重要。

## 耳廓基本结构

### 耳垂 (Lobule)

| 屬性 | 內容 |
|------|------|
| **形態特點** | 柔軟下垂，無軟骨 |
| **主要反射區** | 面部、牙齒、眼 |
| **臨床意義** | 面癱、牙痛、眼疾 |

**詳細對應:**
- 顔面區：面部疾病
- 牙區：牙齒問題
- 眼區：眼部疾病
- 內耳區：眩暈、平衡障礙
- 扁桃體區：咽喉疾病

### 耳輪 (Helix)

| 屬性 | 內容 |
|------|------|
| **形態特點** | 耳廓外側邊緣 |
| **主要反射區** | 肢體、脊柱 |
| **臨床意義** | 肢體疼痛、脊柱疾病 |

**詳細對應:**
- 頸椎：頸部疾病
- 胸椎：胸部疾病
- 腰椎：腰痛、腎虛
- 骶椎：下腰部問題
- 肩關節：肩周炎
- 肘關節：肘部疾病
- 腕關節：腕部問題
- 指關節：手指疾病

### 對耳輪 (Antihelix)

| 屬性 | 內容 |
|------|------|
| **形態特點** | 耳廓內側邊緣 |
| **主要反射區** | 軀幹、內臟 |
| **臨床意義** | 內臟疾病、軀幹疼痛 |

**詳細對應:**
- 腹部：腹痛、消化不良
- 胸部：胸悶、心悸
- 背部：背痛
- 頸部：頸部僵硬

### 耳舟 (Scapha)

| 屬性 | 內容 |
|------|------|
| **形態特點** | 耳輪與對耳輪之間 |
| **主要反射區** | 上肢 |
| **臨床意義** | 上肢疾病、肩頸痛 |

**詳細對應:**
- 指部：手指疾病
- 腕部：腕關節問題
- 肘部：肘部疾病
- 肩部：肩周炎、肩痛
- 鎖骨：鎖骨問題

### 三角窩 (Triangular Fossa)

| 屬性 | 內容 |
|------|------|
| **形態特點** | 對耳輪上下腳之間 |
| **主要反射區** | 盆腔、生殖器 |
| **臨床意義** | 婦科疾病、生殖系統 |

**詳細對應:**
- 子宮：月經不調、痛經
- 卵巢：卵巢疾病
- 內生殖器：生殖系統疾病
- 外生殖器：外生殖器問題
- 神門：鎮靜安神
- 盆腔：盆腔疾病

### 耳甲艇 (Cymba Conchae)

| 屬性 | 內容 |
|------|------|
| **形態特點** | 耳輪腳上方凹陷 |
| **主要反射區** | 腹腔臟器 |
| **臨床意義** | 消化系統疾病 |

**詳細對應:**
- 腎：腎臟疾病、腰痛
- 膀胱：泌尿系統疾病
- 輸尿管：輸尿管問題
- 胰膽：胰膽疾病
- 肝：肝病、眼疾
- 脾：消化不良、血液病
- 胃：胃痛、胃病
- 小腸：腸炎、消化不良

### 耳甲腔 (Cavum Conchae)

| 屬性 | 內容 |
|------|------|
| **形態特點** | 耳輪腳下方凹陷 |
| **主要反射區** | 胸腔臟器 |
| **臨床意義** | 心肺疾病 |

**詳細對應:**
- 心：心悸、胸悶
- 肺：咳嗽、哮喘
- 氣管：氣管炎
- 三焦：代謝疾病
- 內分泌：內分泌失調
- 皮质下：神經系統疾病

## 耳背结构

### 耳背溝 (Groove of Posterior Auricular Surface)

| 屬性 | 內容 |
|------|------|
| **主要反射區** | 降壓溝、脊椎 |
| **臨床意義** | 高血壓、脊椎病 |

**應用:**
- 降壓：高血壓患者主要穴位
- 脊椎：脊椎相關疾病

### 耳背心 (Posterior Auricular Heart Point)

| 屬性 | 內容 |
|------|------|
| **主要反射區** | 心臟、血管 |
| **臨床意義** | 心血管疾病 |

### 耳背脾 (Posterior Auricular Spleen Point)

| 屬性 | 內容 |
|------|------|
| **主要反射區** | 脾胃、消化 |
| **臨床意義** | 消化系統疾病 |

### 耳背腎 (Posterior Auricular Kidney Point)

| 屬性 | 內容 |
|------|------|
| **主要反射區** | 腎臟、泌尿 |
| **臨床意義** | 泌尿系統疾病 |

## 定位要點

### 常用定位標誌
- **耳輪腳**：耳甲的分界線
- **對耳輪上腳**：下肢區分界
- **對耳輪下腳**：下肢區分界
- **耳屏**：與耳垂的交界
- **對耳屏**：耳甲腔入口

### 定位技巧
1. **先找大結構**：確定耳輪、對耳輪等
2. **再找小區域**：細分至具體穴位
3. **結合解剖**：參考解剖位置
4. **比較定位**：左右耳對比

## 注意事項

### 觀察要點
- **形態**：觀察耳廓各部位形態
- **顏色**：注意顏色變化
- **光澤**：評估光澤度
- **彈性**：觸摸評估彈性

### 操作注意
- **定位準確**：準確定位反射區
- **力度適中**：避免過度刺激
- **衛生安全**：注意清潔消毒
- **個體差異**：考慮個體解剖差異
