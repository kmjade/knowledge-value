# 耳诊 - 重要耳穴详解

重要耳穴是臨床應用頻率最高、效果最顯著的穴位，掌握這些耳穴的定位、功能和主治對於耳診至關重要。

## 五臟六腑耳穴

### 心穴 (Heart Point)

| 屬性 | 詳情 |
|------|------|
| **定位** | 耳甲腔中央，三角窩後下方 |
| **參考定位** | 耳屏內側，耳甲腔最凹陷處 |
| **功能** | 寧心安神，調節心律 |
| **主治** | 心悸、胸悶、失眠、多夢、健忘 |

**詳細應用:**
- **心血管疾病**：心悸、胸悶、心律失常
- **神經系統**：失眠、多夢、健忘、神經衰弱
- **精神狀態**：焦慮、抑鬱、驚悸
- **相關穴位**：交感、腎上腺、神門

**操作要點:**
- **輕度刺激**：心穴敏感，不宜過度刺激
- **時間控制**：每次1-2分鐘
- **頻率**：每日1-2次

### 腎穴 (Kidney Point)

| 屬性 | 詳情 |
|------|------|
| **定位** | 對耳輪下腳下方，三角窩後壁 |
| **參考定位**：耳甲艇後下方，耳輪腳末端上方 |
| **功能** | 補腎益精，強壯腰膝 |
| **主治** | 腰痛、耳鳴、遺精、陽痿、腎炎 |

**詳細應用:**
- **泌尿系統**：腎炎、尿路感染、水腫
- **生殖系統**：遺精、陽痿、不孕不育
- **骨骼系統**：腰痛、骨質疏鬆
- **聽覺系統**：耳鳴、聽力下降
- **老年保健**：抗衰老、增強體質

**操作要點:**
- **補法**：宜用溫和刺激，補益為主
- **持續刺激**：可長期應用
- **配合穴位**：膀胱、尿道、腰椎

### 肝穴 (Liver Point)

| 屬性 | 詳情 |
|------|------|
| **定位** | 耳甲艇後部，胃穴後上方 |
| **參考定位**：耳甲艇外側緣，耳輪腳上方 |
| **功能** | 疏肝理氣，明目 |
| **主治** | 肝炎、眼疾、高血壓、月經不調 |

**詳細應用:**
- **肝膽疾病**：肝炎、膽囊炎、肝病
- **眼科疾病**：視物模糊、眼乾、眼疲勞
- **心血管**：高血壓、頭痛、眩暈
- **情志病**：情緒不穩、易怒、抑鬱
- **婦科**：月經不調、痛經

**操作要點:**
- **疏通為主**：重點疏通，不宜補益
- **配合穴位**：膽、脾、胃
- **注意禁忌**：肝火旺盛時慎用補法

### 脾穴 (Spleen Point)

| 屬性 | 詳情 |
|------|------|
| **定位** | 耳甲腔後下方，腰椎穴前下方 |
| **參考定位**：耳屏內側下方，靠近耳垂 |
| **功能** | 健脾益氣，統血 |
| **主治** | 消化不良、腹瀉、貧血、出血 |

**詳細應用:**
- **消化系統**：消化不良、腹瀉、腹脹、食欲不振
- **血液病**：貧血、出血、血小板減少
- **肌肉系統**：肌肉無力、疲勞
- **婦科**：月經過多、崩漏
- **全身狀態**：體虛、乏力

**操作要點:**
- **補益為主**：宜溫和刺激，補益脾胃
- **配合穴位**：胃、腸、三焦
- **飲食配合**：注意飲食調養

### 肺穴 (Lung Point)

| 屬性 | 詳情 |
|------|------|
| **定位** | 耳甲腔上部，心穴上方 |
| **參考定位**：耳甲腔外上方 |
| **功能** | 宣肺止咳，化痰平喘 |
| **主治** | 咳嗽、哮喘、支氣管炎、感冒 |

**詳細應用:**
- **呼吸系統**：咳嗽、哮喘、支氣管炎、感冒
- **咽喉疾病**：咽喉腫痛、音啞
- **皮膚病**：蕁麻疹、濕疹（肺主皮毛）
- **便秘**：肺與大腸相表裡，可治便秘

**操作要點:**
- **急性病**：急性發作時可加重刺激
- **慢性病**：慢性病宜溫和刺激
- **配合穴位**：氣管、內鼻、三焦

### 胃穴 (Stomach Point)

| 屬性 | 詳情 |
|------|------|
| **定位** | 耳甲艇前方，耳輪腳末端 |
| **參考定位**：耳甲艇前方凹陷處 |
| **功能** | 和胃止痛，降逆止嘔 |
| **主治** | 胃痛、胃脹、噁心、嘔吐、消化不良 |

**詳細應用:**
- **胃部疾病**：胃炎、胃潰瘍、胃痛
- **消化不良**：胃脹、消化不良、食欲不振
- **噁心嘔吐**：妊娠嘔吐、暈車嘔吐
- **全身狀態**：營養不良、消瘦

**操作要點:**
- **急性病**：急性胃痛可加重刺激
- **慢性病**：慢性胃病宜溫和刺激
- **配合穴位**：脾、腸、肝、膽

## 功能性耳穴

### 神門 (Shenmen Point)

| 屬性 | 詳情 |
|------|------|
| **定位** | 三角窩前1/3處，對耳輪上下腳分叉處 |
| **參考定位**：三角窩最凹陷處 |
| **功能** | 鎮靜安神，鎮痛消炎 |
| **主治** | 失眠、頭痛、神經衰弱、疼痛 |

**詳細應用:**
- **神經系統**：失眠、多夢、頭痛、神經衰弱
- **疼痛症**：各種疼痛（頭痛、牙痛、關節痛等）
- **精神疾病**：焦慮、抑鬱、癔症
- **過敏性疾病**：哮喘、蕁麻疹、過敏性鼻炎

**操作要點:**
- **鎮靜為主**：適用於興奮、失眠
- **鎮痛要穴**：各種疼痛均可配用
- **配合穴位**：皮质下、交感、相應病變穴位

### 皮质下 (Subcortex Point)

| 屬性 | 詳情 |
|------|------|
| **定位** | 對耳屏內側面前下方 |
| **參考定位**：對耳屏內側面最凹陷處 |
| **功能** | 調節大腦皮質，鎮靜安神 |
| **主治** | 神經衰弱、失眠、頭暈、記憶力減退 |

**詳細應用:**
- **神經系統**：神經衰弱、失眠、頭暈
- **大腦功能**：記憶力減退、注意力不集中
- **內分泌**：內分泌失調、甲狀腺疾病
- **消化系統**：胃腸功能紊亂

**操作要點:**
- **調節為主**：調節大腦皮質功能
- **配合穴位**：神門、心、腎
- **適用廣泛**：多種疾病均可配用

### 內分泌 (Endocrine Point)

| 屬性 | 詳情 |
|------|------|
| **定位** | 耳甲腔後壁，對耳屏下方 |
| **參考定位**：耳甲腔後下方，近耳輪腳 |
| **功能** | 調節內分泌，抗過敏 |
| **主治** | 內分泌失調、糖尿病、過敏性疾病 |

**詳細應用:**
- **內分泌疾病**：糖尿病、甲狀腺疾病、更年期綜合徵
- **過敏性疾病**：哮喘、蕁麻疹、過敏性鼻炎
- **婦科**：月經不調、痛經、不孕不育
- **皮膚病**：濕疹、痤瘡

**操作要點:**
- **調節為主**：調節內分泌功能
- **配合穴位**：卵巢、子宮、甲狀腺
- **適用廣泛**：內分泌相關疾病

### 交感 (Sympathetic Point)

| 屬性 | 詳情 |
|------|------|
| **定位** | 對耳輪下腳末端與耳輪交界處 |
| **參考定位**：對耳輪下腳末端 |
| **功能** | 調節自主神經，解痙止痛 |
| **主治** | 胃腸痙攣、心絞痛、頭痛 |

**詳細應用:**
- **心臟疾病**：心絞痛、心律失常
- **胃腸疾病**：胃腸痙攣、胃痛、腹痛
- **神經系統**：頭痛、眩暈
- **血管疾病**：血管痙攣、雷諾病

**操作要點:**
- **解痙為主**：解除平滑肌痙攣
- **急症要穴**：急性痙攣性疼痛
- **配合穴位**：相應臟腑穴位

## 常見症狀對應耳穴

| 症狀 | 主穴 | 配穴 |
|------|------|------|
| **失眠** | 神門、皮质下 | 心、腎、肝 |
| **頭痛** | 神門、額、枕 | 皮质下、交感 |
| **胃痛** | 胃、交感 | 脾、肝、膽 |
| **便秘** | 大腸、三焦 | 脾、肺 |
| **月經不調** | 子宮、內分泌 | 卵巢、腎、肝 |
| **高血壓** | 降壓溝、心 | 腎、肝、交感 |
| **咳嗽** | 肺、氣管 | 三焦、內鼻 |
| **腰痛** | 腰椎、腎 | 膀胱、神門 |

## 選穴原則

### 主穴配穴原則
1. **選主穴**：根據病變部位選取相應耳穴
2. **配穴位**：根據病證選配功能性耳穴
3. **虛實補瀉**：虛證用補法，實證用瀉法
4. **標本兼顧**：急則治標，緩則治本

### 選穴技巧
- **對症選穴**：根據症狀選取對應耳穴
- **辨證選穴**：根據辨證選取相應耳穴
- **經驗選穴**：參考經驗穴位
- **壓痛選穴**：通過壓痛確定反應點
