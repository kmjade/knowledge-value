# 指南

### 选择合适的模板

```
遇到问题？
├─ 是 → 使用问题卡
└─ 否
   有明确目标和截止日期？
   ├─ 是 → 使用 Project
   └─ 否
      是持续的责任？
      ├─ 是 → 使用 Area
      └─ 否
# 資源
         ├─ 是 → 使用 Resource Collection
         └─ 否
            突发想法/灵感？
            ├─ 是 → 使用 Idea Capture
            └─ 否 → 使用 Resource
```
