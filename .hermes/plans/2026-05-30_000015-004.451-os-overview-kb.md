# Plan: 004.451 操作系统总览 — KB 构建计划

## 定位

004.451 是 UDC 中「特定操作系统」的总类，作为 Windows Server / Linux / macOS / Android / HarmonyOS 五个子库的**父级索引与对比总览**。

## 章节内容

| # | 章节 | 核心内容 |
|---|------|----------|
| 01 | 操作系统谱系 | OS 家族树（Unix→Linux/macOS、Windows NT、Android/Linux、HarmonyOS）、历史时间线、市场份额变迁 |
| 02 | 架构对比 | 内核类型（宏内核/混合/微内核）对比表、进程调度、内存管理、文件系统、安全模型 — Windows vs Linux vs macOS vs Android vs HarmonyOS 横向对比 |
| 03 | 命令行对比 | Bash/zsh (Linux/macOS) vs PowerShell (Windows) vs hdc (HarmonyOS) vs adb (Android) — 等价命令速查表 |
| 04 | 开发环境对比 | IDE/工具链/包管理/调试器矩阵 — VS Code/Android Studio/DevEco/Xcode |
| 05 | 部署与运维对比 | 容器(Docker/Podman)、虚拟化、配置管理 (Ansible/Puppet)、CI/CD 集成 |
| 06 | 安全模型对比 | 权限模型、沙箱、SELinux/AppArmor/SIP、安全启动、应用签名 |
| 07 | 选型指南 | 按场景推荐 OS（服务器/桌面/移动/嵌入式/IoT/车载）、TCO 分析 |
| 08 | 互操作性 | 跨平台文件共享 (SMB/NFS)、远程桌面 (RDP/VNC)、WSL、虚拟化嵌套、AD/LDAP 集成 |
| 09 | 未来趋势 | 云原生 OS (Flatcar/Bottlerocket)、AI OS、RISC-V OS、鸿蒙跨设备融合 |
