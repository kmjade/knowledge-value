**infographic 布局 (21)**: bridge, circular-flow, comparison-table, do-dont, equation, feature-list, fishbone, funnel, grid-cards, iceberg, journey-path, layers-stack, mind-map, nested-circles, priority-quadrants, pyramid, sandwich, side-by-side, staircase, timeline, venn-diagram

**infographic 风格 (21)**: craft-handmade, technical-schematic, corporate-memphis, ...

每个内容技能列出完整的风格/布局选项表，标注默认值和推荐场景。

- **版本敏感**：baoyu-skills 活跃更新（19.8k⭐），KB 聚焦当前稳定功能，标注"以官方 README 为准"
- **与 vault 现有内容关系**：此 KB 独立于 DDC 分类体系，作为工具参考放在 `3 Resources/` 根级，与 Markdown/Obsidian/Notion KB 同级
- **不包含 screenshots**：仓库有 125 张截图，KB 中引用原始 GitHub URL，不本地存储
