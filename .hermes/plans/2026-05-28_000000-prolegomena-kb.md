# Plan: 构建 Prolegomena 知识库

**创建时间**: 2026-05-28
**状态**: 📋 规划中

---

## 1. 目标

在 `3 Resources/000 Knowledge/ram/` 下构建 **Prolegomena (知识学导论)** 知识库，覆盖知识与文化的基础理论——认识论 (Epistemology)、预备学科 (Propaedeutics) 和文化知识论。

---

## 2. 当前上下文

### 2.1 知识库定位

- DDC 对应: **001 Knowledge** — 知识本身的理论
- 作为 Knowledge-Systems 知识库的**哲学上层建筑**：KOS 回答「如何组织知识」，Prolegomena 回答「知识是什么」
- 与现有 `ram/Knowledge-Systems-知识系统/` (DDC 001 应用层) 形成互补：**基础理论 ↔ 应用系统**

### 2.2 现有相关结构

```
3 Resources/000 Knowledge/
├── ram/
│   ├── Knowledge-Systems/     ← KOS 九章（知识组织应用）
│   ├── DDC/                   ← 分类法实例
│   ├── UDC/                   ← 分类法实例
│   └── Prolegomena/           ← 🆕 本知识库
├── wiki/                      ← AI 编译产物
└── outputs/                   ← 最终制品
```

### 2.3 命名说明

- **Prolegomena** (προλεγόμενα): 希腊语「先说的话」，意为序论、导言。康德《未来形而上学导论》以此命名。
- **Propaedeutics**: 预备学科，入门教育——古希腊的「自由七艺」(Trivium + Quadrivium) 即 propaedeutic。
- 知识库英文: *Prolegomena: Fundamentals of Knowledge and Culture*
- 知识库中文: 知识学导论

### 章节内容大纲

| # | 章节 | 核心内容 |
|---|------|---------|
| 01 | **什么是知识** | 柏拉图《泰阿泰德》、JTB (Justified True Belief)、葛梯尔问题 (Gettier Problem)、当代定义 (reliabilism, virtue epistemology) |
| 02 | **知识的来源** | 理性主义 vs 经验主义、先验/后验、分析/综合、知觉、记忆、证言 (testimony)、直觉 |
| 03 | **知识的类型** | 命题知识 (knowing-that)、能力知识 (knowing-how)、亲知 (acquaintance)、默会知识 (tacit knowledge / Polanyi)、集体知识 |
| 04 | **真理理论** | 符合论 (correspondence)、融贯论 (coherence)、实用论 (pragmatic / James, Dewey)、冗余论 (deflationary)、蒯因与自然化认识论 |
| 05 | **知识与文化** | 文化作为集体知识储备、知识的社会建构、传统与传承、模因论 (memetics / Dawkins)、口头文化与文字文化、知识的社会分布 |
| 06 | **知识组织导论** | 从亚里士多德范畴到图书馆分类、知识分类的哲学基础、百科全书传统、分类法作为世界观（衔接 Knowledge-Systems KB） |
| 07 | **预备学科与博雅教育** | 希腊 Paideia、自由七艺 (Trivium + Quadrivium)、中世纪大学、洪堡大学理念、通识教育 (General Education)、现代 Liberal Arts |
| 08 | **知识论简史** | 古希腊 (柏拉图/亚里士多德) → 中世纪 (奥古斯丁/阿奎那) → 近代 (笛卡尔/洛克/休谟/康德) → 分析哲学 (罗素/维特根斯坦) → 当代 (Quine/Gettier/认知科学) |
| 09 | **应用认识论** | 科学方法论、信息素养 (information literacy)、批判性思维、数字时代的认识论问题（假新闻、回音室、AI生成知识）、知识管理实践 |
