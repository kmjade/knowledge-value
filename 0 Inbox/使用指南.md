## 快速开始 / Quick Start

### 新建待办

使用 `/para-创建项目` 或直接在 InBox 中创建待办事项。

### 快速分类

将选中的内容直接移动到对应 PARA 文件夹，支持批量操作。

### 自动归档

项目完成后，系统会自动归档到 `4 Archives/`。

## InBox 主索引详解

### 快速操作区域

主索引包含快速操作按钮，可以：
- 一键打开工作流配置
- 一键打开搜索
- 批量处理选中的内容

### 待办功能

- **`choose()` 选择器** - 交互式 PARA 分类选择
- 子任务支持 - 每个待办可以添加子任务
- 提醒设置 - 支持截止日期提醒

### 搜索功能

- **实时搜索** - 使用 `contains()` 匹配文件内容
- **搜索历史** - 保存搜索关键词
- **高级筛选** - 支持日期范围、作者筛选等

### 工作流自动化

支持以下自动触发器：
- 定时整理 InBox
- 项目完成自动归档
- 收件箱满时提醒

## 最佳实践

1. **使用 PARA 原则**
   - 项目：短期、有截止日期
   - 领域：长期、持续维护
   - 资源：持续感兴趣的话题
   - 归档：完成或不再活跃

2. **定期回顾**
   - 每周回顾项目
   - 每月回顾领域
   - 每季度清理归档

3. **快速整理**
   - 及时处理 InBox 内容
   - 避免 InBox 积压
   - 使用标签和分类
