## 📚 PARA 知识管理完整指南

PARA 是一个由 [Tiago Forte](https://fortelabs.co/) 开发的知识管理方法论，基于**行动性和当前相关性**而非主题分类来组织信息。

---

### 🎯 四大核心类别

|类别|缩写|定义|特征|示例|
|---|---|---|---|---|
|**Projects**|P|短期、有明确目标的任务|⏰ 有截止日期、🎯 有明确结果|"推出新网站"、"完成年度税务申报"|
|**Areas**|A|长期持续责任|♾️ 无截止日期、🔄 需要定期维护|"健康管理"、"财务管理"、"知识管理"|
|**Resources**|R|持续感兴趣的主题|🔍 主题导向、📚 参考性质|"AI工具研究"、"投资策略"、"生产力技巧"|
|**Archives**|A|已完成/不活跃项目|⚰️ 不活跃、📦 保存价值|"2024年度计划"、"旧学习笔记"|

### 🔄 分类决策流程

```
新信息进入收件箱 (InBox)
    ↓
是否当前活跃？
    ├── 否 → 归入 Archives
    └── 是 → 是否有截止日期？
              ├── 是 → Projects（短期项目）
              └── 否 → 是否持续性责任？
                        ├── 是 → Areas（长期领域）
                        └── 否 → Resources（兴趣资源）
```

---

### 🔗 与 Zettelkasten 的关系

|系统|用途|特点|
|---|---|---|
|**PARA**|组织行动和项目|按可执行性分类|
|**Zettelkasten**|建立知识网络|按概念链接|

**结合方式**：

- PARA 决定笔记的**存放位置**
- Zettelkasten 建立笔记之间的**链接关系**
- 原子笔记存储在 `5 Zettels/`，通过链接关联到 PARA 各分类

---

### 📊 回顾周期

|类别|频率|重点|
|---|---|---|
|**Projects**|每周|进展、下一步行动、是否完成|
|**Areas**|每月|维护状况、是否需要新项目|
|**Resources**|每季|相关性、是否需要清理|
|**Archives**|每年|清理、压缩、删除|

PARA 的核心理念是：**让信息跟随行动，而非被困在主题文件夹中**。这样可以快速找到当前需要的信息，保持系统的长期清晰度。
