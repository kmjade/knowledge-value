# Algorithmic Philosophy · 算法哲学

> 一种通过计算世界观来指导生成艺术创作的方法论——不是画一幅画，而是设计一个产生美的过程。

## 定义

算法哲学是一种**生成美学运动**的宣言。它不是描述最终图像，而是定义**计算过程**如何产生涌现之美。每一份算法哲学：
- 命名一个运动（1-2 词）
- 阐述美学立场（4-6 段）
- 指定算法表达方式（噪声场、粒子系统、力场、递归等）
- 为下一个 Claude 留下实现空间

## 核心原则

| 原则 | 说明 |
|------|------|
| **过程优于产品** | 美从算法执行中涌现，每次运行独一无二 |
| **参数表达** | 思想通过数学关系、力、行为交流——而非静态构图 |
| **涌现行为** | 复杂模式从简单规则中自发产生 |
| **种子随机性** | 使用 seed 确保可复现，但每次 seed 产生独特变体 |
| **受控混沌** | 随机在严格边界内变化——秩序从无序中涌现 |

## 经典运动示例

| 运动 | 核心算法 | 美学特征 |
|------|---------|---------|
| Organic Turbulence | 分层 Perlin 噪声流场 | 混沌受自然法则约束 |
| Quantum Harmonics | 粒子相位干涉 | 离散实体展现波状模式 |
| Recursive Whispers | L-系统递归细分 | 跨尺度自相似性 |
| Field Dynamics | 数学向量场 | 不可见力通过物质效应显现 |
| Stochastic Crystallization | Voronoi 松弛 | 随机过程结晶为有序结构 |
| **Cartographic Drift** | 传感器射线 + 密度积累 | 代理探索构建概率地图 |

## 工艺要求

算法哲学必须**反复强调**最终算法应呈现出：
- 经过无数次迭代精心打磨的感觉
- 由计算美学领域顶尖专家精炼而成
- 每个参数都在秩序与混沌之间找到精确平衡点
