## ⚙️ 快速安装指引

text

```
Step 1  安装 Templater 插件
        设置 → 插件市场 → 搜索 Templater → 安装并启用

Step 2  创建模板文件夹
        在Obsidian根目录新建文件夹：/templates

Step 3  将以上5个模板文件粘贴进去
        文件名严格按照上方命名（方便Templater识别）

Step 4  配置Templater
        设置 → Templater → Template folder location → 填写 "templates"
        开启 "Trigger Templater on new file creation"

Step 5  调用模板
        新建笔记时，按 Alt+E（或设置的快捷键）
        → 选择对应模板 → 自动填入日期/时间等动态字段
```
