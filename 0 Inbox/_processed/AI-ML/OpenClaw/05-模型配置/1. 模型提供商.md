# OpenClaw 模型提供商

> [!info] 概述
> OpenClaw 支持多种 AI 模型提供商，你可以根据需求选择最适合的模型。

## 支持的提供商

### 国际提供商

| 提供商 | 配置键 | 模型示例 | 特点 |
|--------|--------|----------|------|
| Anthropic | `anthropic` | claude-sonnet-4-6 | 推荐，能力强 |
| OpenAI | `openai` | gpt-4o | 生态丰富 |
| Google | `google` | gemini-pro | 免费额度 |
| Mistral | `mistral` | mistral-large | 开源模型 |

### 国内提供商

| 提供商 | 配置键 | 模型示例 | 特点 |
|--------|--------|----------|------|
| DeepSeek | `deepseek` | deepseek-chat | 性价比高 |
| 通义千问 | `qwen` | qwen-max | 阿里云 |
| 智谱 AI | `zhipu` | glm-4 | 国产模型 |
| Moonshot | `moonshot` | moonshot-v1 | 长上下文 |

## 配置 API Key

### 配置文件方式

```json
{
  "providers": {
    "anthropic": {
      "apiKey": "sk-ant-..."
    },
    "openai": {
      "apiKey": "sk-..."
    },
    "deepseek": {
      "apiKey": "sk-..."
    }
  }
}
```

### 环境变量方式

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
export OPENAI_API_KEY="sk-..."
export DEEPSEEK_API_KEY="sk-..."
```

### 命令行方式

```bash
openclaw provider set anthropic --api-key "sk-ant-..."
```

## Anthropic 配置

### 基础配置

```json
{
  "providers": {
    "anthropic": {
      "apiKey": "sk-ant-...",
      "models": {
        "claude-sonnet-4-6": {
          "maxTokens": 4096
        },
        "claude-opus-4-7": {
          "maxTokens": 4096
        }
      }
    }
  }
}
```

### 可用模型

| 模型 | 上下文 | 特点 |
|------|--------|------|
| claude-sonnet-4-6 | 200K | 平衡性能与成本 |
| claude-opus-4-7 | 200K | 最强能力 |
| claude-haiku-4-5-20251001 | 200K | 快速响应 |

### 获取 API Key

1. 访问 https://console.anthropic.com/
2. 创建账户
3. 在 API Keys 页面创建新 Key

## 代理配置

### 自定义 API 端点

```json
{
  "providers": {
    "anthropic": {
      "apiKey": "sk-ant-...",
      "baseURL": "https://your-proxy.com/v1"
    }
  }
}
```

### 使用代理

```json
{
  "providers": {
    "anthropic": {
      "apiKey": "sk-ant-...",
      "proxy": {
        "http": "http://proxy:8080",
        "https": "http://proxy:8080"
      }
    }
  }
}
```

## 模型选择

### 按场景选择

| 场景 | 推荐模型 | 原因 |
|------|----------|------|
| 日常对话 | claude-sonnet-4-6 | 平衡 |
| 复杂任务 | claude-opus-4-7 | 最强 |
| 代码生成 | claude-sonnet-4-6 | 专长 |
| 预算有限 | deepseek-chat | 性价比 |

### 配置默认模型

```json
{
  "agents": {
    "pi": {
      "provider": "anthropic",
      "model": "claude-sonnet-4-6"
    }
  }
}
```

## 多提供商负载均衡

### 配置多个提供商

```json
{
  "agents": {
    "pi": {
      "providers": [
        {
          "name": "anthropic",
          "model": "claude-sonnet-4-6",
          "weight": 70
        },
        {
          "name": "openai",
          "model": "gpt-4o",
          "weight": 30
        }
      ]
    }
  }
}
```

---

## 故障转移

### 配置备用提供商

```json
{
  "agents": {
    "pi": {
      "provider": "anthropic",
      "model": "claude-sonnet-4-6",
      "fallback": {
        "provider": "openai",
        "model": "gpt-4o"
      }
    }
  }
}
```

# 列出提供商
