# OpenCode 配置文件详解

> [!info] 概述
> OpenCode 使用 JSON 格式的配置文件进行设置。

## 配置文件位置

| 配置类型 | 路径 |
|----------|------|
| 项目配置 | `./opencode.json` |
| 全局配置 | `~/.config/opencode/opencode.json` |
| API 凭证 | `~/.local/share/opencode/auth.json` |

---

## 基础结构

```json
{
  "$schema": "https://opencode.ai/config.json",
  "model": "anthropic/claude-sonnet-4-20250514",
  "agent": {},
  "mcp": {},
  "permission": {}
}
```

---

## 核心配置项

### model - 默认模型

```json
{
  "model": "anthropic/claude-sonnet-4-20250514"
}
```

常用模型：
- `anthropic/claude-sonnet-4-20250514`
- `anthropic/claude-opus-4-20250514`
- `openai/gpt-4o`
- `google/gemini-pro`
- `deepseek/deepseek-chat`

### agent - 代理配置

```json
{
  "agent": {
    "build": {
      "mode": "primary",
      "model": "anthropic/claude-sonnet-4-20250514",
      "prompt": "{file:./prompts/build.txt}",
      "tools": {
        "write": true,
        "edit": true,
        "bash": true
      }
    },
    "code-reviewer": {
      "description": "Reviews code for best practices",
      "mode": "subagent",
      "model": "anthropic/claude-sonnet-4-20250514",
      "prompt": "You are a code reviewer...",
      "tools": {
        "write": false,
        "edit": false
      }
    }
  }
}
```

### mcp - MCP 服务器配置

```json
{
  "mcp": {
    "my-mcp-server": {
      "type": "local",
      "enabled": true
    },
    "remote-mcp": {
      "type": "remote",
      "url": "https://api.example.com/mcp"
    }
  }
}
```

### permission - 权限配置

```json
{
  "permission": {
    "edit": "deny",
    "bash": "ask",
    "webfetch": "allow"
  }
}
```

| 权限值 | 说明 |
|--------|------|
| `allow` | 允许直接执行 |
| `deny` | 完全禁止 |
| `ask` | 执行前需要人工确认 |

---

## 完整配置示例

```json
{
  "$schema": "https://opencode.ai/config.json",

  "model": "anthropic/claude-sonnet-4-20250514",

  "agent": {
    "build": {
      "mode": "primary",
      "model": "anthropic/claude-sonnet-4-20250514",
      "tools": {
        "write": true,
        "edit": true,
        "bash": true
      }
    },
    "plan": {
      "mode": "primary",
      "tools": {
        "edit": "ask",
        "bash": "ask"
      }
    }
  },

  "mcp": {
    "github": {
      "type": "local",
      "enabled": true
    }
  },

  "permission": {
    "bash": "ask",
    "edit": "allow",
    "webfetch": "allow"
  },

  "theme": "dark",
  "keybindings": {
    "custom": "ctrl+k"
  }
}
```

## 工具权限配置

### 内置工具权限

```json
{
  "permission": {
    "bash": "ask",      // Shell 命令执行
    "edit": "allow",    // 文件编辑
    "write": "allow",   // 文件创建
    "read": "allow",    // 文件读取
    "grep": "allow",    // 内容搜索
    "glob": "allow",    // 文件匹配
    "webfetch": "allow" // 网络请求
  }
}
```

### 通配符配置

```json
{
  "permission": {
    "mymcp_*": "ask"  // 所有来自该 MCP 的工具需审批
  }
}
```

## 环境变量

| 变量 | 说明 |
|------|------|
| `OPENCODE_MODEL` | 默认模型 |
| `OPENCODE_CONFIG` | 配置文件路径 |
| `OPENCODE_DISABLE_CLAUDE_CODE` | 禁用 Claude Code 兼容 |

## 配置优先级

1. 命令行参数
2. 环境变量
3. 项目配置 (./opencode.json)
4. 全局配置 (~/.config/opencode/opencode.json)
5. 默认值
