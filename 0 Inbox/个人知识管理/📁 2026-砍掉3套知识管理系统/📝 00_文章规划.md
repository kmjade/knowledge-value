# 📝 00_文章规划｜我亲手砍掉的 3 套知识管理系统
---

## 1. 目标读者是谁？

- 职业/阶段：普通上班族/技术人，自学党
- 特征：  
    - 信息焦虑，买了很多课程/书  
    - 折腾过很多笔记软件和系统  
    - 常常觉得「我不是不努力，而是不会管理知识」

## 2. 他们在什么场景会点开这篇文章？

- 正在纠结要不要换新工具/新模板
- 对「第二大脑」「知识管理」心动，但又怕被割
- 最近刚刚重构完自己的系统，半爽半心虚  
    

## 3. 我要给什么具体承诺？

- 告诉你我亲身踩过的 3 套系统是怎么死的
- 帮你看清常见的 3 个知识管理大坑
- 展示我现在在用的一套简单但能活下去的结构  
    

## 4. 想踩的几个心理点

- 「别重蹈覆辙」：浪费时间/钱的损失厌恶
- 「原来不是只有我这样」：同温层的安全感
- 「想偷看别人的后台长什么样」：好奇/窥探欲  
    

## 5. 初步结构想法

- 开头：自嘲 + 场景共鸣
- 三套系统的故事（Notion / 双链 / 过度 Obsidian）
- 每套故事后总结一个坑
- 最后说我现在的做法 + 给 3 条现实建议
