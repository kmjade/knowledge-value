# 💡 概念卡：卡片笔记法 (Zettelkasten)

> [!question] 什么是卡片笔记法？
>
> 卡片笔记法是一种由德国社会学家 Niklas Luhmann 发展的知识管理方法，通过创建相互连接的"原子化"笔记，构建一个能够自组织的知识网络。

## 核心概念

卡片笔记法的本质是：**不要把笔记当作容器，而要把笔记当作网络中的节点。**

### 关键特征

1. **原子化 (Atomicity)**
   - 每张卡片只包含一个想法
   - 可以独立理解和引用
   - 便于重新组合

2. **链接 (Linking)**
   - 使用 `[[双向链接]]` 连接相关笔记
   - 形成网状结构而非层级结构
   - 通过连接产生新见解

3. **自组织 (Self-Organization)**
   - 不预设分类结构
   - 让知识自然涌现
   - 结构从连接中产生

## 核心原则

> **"我们思考的不是单一的事实，而是事实之间的关系"**
> — Niklas Luhmann

## 卢曼的生产力

Niklas Luhmann 使用卡片笔记法在 30 年内出版了：
- 50+ 本书
- 500+ 篇论文
- 涵盖社会学、系统论等多个领域

他的秘诀：**让笔记系统替他思考**

## 与传统笔记的区别

| 传统笔记 | 卡片笔记法 |
|---------|-----------|
| 按主题分类 | 按连接组织 |
| 层级结构 | 网络结构 |
| 容易遗忘 | 容易发现 |
| 静态存储 | 动态生长 |
