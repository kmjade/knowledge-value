##### 2. 固件文件結構圖

```
固件文件結構
│
├── bootloader.bin
│   ├── 地址: 0x0000
│   ├── 大小: ~28KB
│   └── 功能: 啟動引導程序
│
├── partition-table.bin
│   ├── 地址: 0x8000
│   ├── 大小: ~4KB
│   └── 功能: 分區表信息
│
└── main.bin
    ├── 地址: 0x10000
    ├── 大小: 視應用而定
    └── 功能: 主程序固件
```

##### 3. 固件文件對應表

| 文件名稱 | 燒錄地址 | 大小 | 功能 | 備註 |
|---------|---------|------|------|------|
| **bootloader.bin** | 0x0000 | ~28KB | 啟動引導程序 | 第一個燒錄 |
| **partition-table.bin** | 0x8000 | ~4KB | 分區表 | 第二個燒錄 |
| **main.bin** | 0x10000 | 視應用 | 主程序固件 | 第三個燒錄 |
| **ota_data.bin** | 0xE000 | ~4KB | OTA 數據（可選） | OTA 模式使用 |

##### 6. esptool.py 基本燒錄命令

```bash
# 基本語法
esptool.py --chip esp32s3 \
  --port /dev/ttyUSB0 \
  --baud 921600 \
  write_flash \
  0x0000 bootloader.bin \
  0x8000 partition-table.bin \
  0x10000 main.bin
```

##### 7. 自動化燒錄腳本示例

```bash
#!/bin/bash
# flash_esp32s3.sh

# 設置變量
PORT="/dev/ttyUSB0"
BAUD="921600"
BOOTLOADER="build/bootloader/bootloader.bin"
PARTITION="build/partition_table/partition-table.bin"
MAIN="build/main.bin"

# 燒錄固件
esptool.py --chip esp32s3 \
  --port $PORT \
  --baud $BAUD \
  write_flash \
  0x0000 $BOOTLOADER \
  0x8000 $PARTITION \
  0x10000 $MAIN

echo "Flashing completed!"
```

##### 8. 常見問題解決方案 (7類問題)

**問題1: 找不到 COM 端口**
- 檢查 USB 連接
- 檢查驅動程序
- 重新安裝驅動
- 更換 USB 線
- 更換 USB 端口

**問題2: 連接超時**
- 進入 Boot 模式
- 降低波特率
- 檢查 USB 連接
- 重啟設備

**問題3: 寫入失敗**
- 降低波特率
- 檢查 Flash 大小配置
- 擦除 Flash
- 檢查固件文件

**問題4: 燒錄後無法啟動**
- 檢查串口輸出
- 檢查 Flash 大小
- 檢查 bootloader
- 檢查分區表
- 擦除 Flash 並重新燒錄

**問題5: 燒錄速度慢**
- 提高波特率
- 檢查 USB 2.0/3.0
- 縮小固件大小
- 檢查 USB 線質量

**問題6: 權限拒絕 (Linux)**
- 添加用戶到 dialout 組
- 重新登錄或立即生效
- 驗證權限

**問題7: macOS 驅動問題**
- 允許未簽名驅動
- 重啟系統
- 檢查設備
- 安裝驅動
