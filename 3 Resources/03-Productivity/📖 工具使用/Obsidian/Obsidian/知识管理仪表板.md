# 知识管理仪表板

> 📊 本仪表板使用 **Dataview** 插件实时展示当前的待办、项目进度、资源统计和归档情况。

## 📋 待办任务
```dataview
TABLE status, priority, due
FROM "#knowledge-management"
WHERE status = "on-hold" OR status = "active"
SORT due ASC
```

## 📈 项目进度
```dataview
TABLE status, updated
FROM "#knowledge-management"
WHERE contains(file.path, "1 Projects")
SORT updated DESC
```

## 📚 资源统计
```dataview
TABLE length(file.link) AS "链接数", tags
FROM "3 Resources"
GROUP BY tags
```

## 📦 已归档
```dataview
TABLE file.link, updated
FROM "#knowledge-management"
WHERE status = "completed"
SORT updated DESC
```

> 🛎️ 将此页固定在侧边栏，便于随时查看整体知识管理状态。
