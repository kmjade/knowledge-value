## 🚨 故障排除快速诊断 / Troubleshooting Quick Diagnosis

### 常见問題症状 / Common Issue Symptoms

| 症状 / Symptom | 可能原因 / Possible Cause | 快速解決方案 / Quick Fix |
|----------------|----------------------|-----------------------|
| **opencode命令未找到** | 未正确安裝或PATH問題 | 重新安裝或檢查PATH |
| **ollama服务未啟動** | 服务未運行或端口被占用 | `ollama serve` 或檢查端口 |
| **模型未找到** | 模型未下載或名称错误 | `ollama pull <model>` |
| **連接被拒絕** | 防火墙或網路問題 | 檢查防火墙設置 |
| **記憶體不足** | 模型太大或并发过多 | 使用更小模型或减少并发 |
