# Zettelkasten 方法指南

> Zettelkasten（卡片盒）是一种个人知识管理系统，通过原子化笔记、双向链接和有机成长来构建知识网络。

## 📚 指南结构

### 核心概念
- [[01-工作流指南]] - Zettelkasten 基础概念和工作流程
- [[02-原子化标准]] - 原子化原则和卡片类型
- [[03-进阶技巧]] - 高级技巧和自动化

### 快速开始

1. 阅读 [[01-工作流指南]] 理解基础概念
2. 学习 [[02-原子化标准]] 掌握原子化原则
3. 使用 [[03-进阶技巧]] 提升效率

## 🏗️ 核心原则

| 原则 | 说明 |
|------|------|
| **原子性** | 每张卡片只包含一个想法 |
| **自主性** | 每张卡片可独立理解 |
| **链接性** | 通过 wikilinks 建立知识网络 |
| **成长性** | 系统持续演化 |
