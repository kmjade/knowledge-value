# 示例笔记 - 健康管理与运动习惯

## 核心内容

健康管理是一个持续的过程，包括运动习惯、饮食规律和睡眠质量等多个方面。

### 运动习惯

- **每周运动目标**：3-4 次，每次 30-45 分钟
- **运动类型**：有氧运动 + 力量训练结合
- **最佳时间**：早晨或工作结束后

### 健康监测

- 定期体检（每年一次）
- 体重和血压监测（每月一次）
- 运动记录（每次运动后）
