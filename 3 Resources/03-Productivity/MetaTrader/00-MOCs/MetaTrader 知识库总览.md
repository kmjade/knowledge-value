## MT4 vs MT5

| 特性 | MT4 | MT5 |
|------|-----|-----|
| 发布时间 | 2005 | 2010 |
| 编程语言 | MQL4 | MQL5 |
| 市场深度 | ❌ | ✅ |
| 时间周期 | 9 种 | 21 种 |
| 内置指标 | 30 | 38 |
| 回测 | 单线程 | 多线程 |
| 资产类别 | 外汇为主 | 外汇 + 股票 + 期货 |
| 对冲系统 | ✅ | ✅ |
| 净额系统 | ❌ | ✅ |
| 社区 | 最大 | 快速增长 |

> 新手建议：如果做外汇 → MT4 足够。如果想交易多资产 → MT5。

## 什么是 CFD？

CFD（差价合约）= 不实际持有资产，只交易价格差。

```
买入 CFD → 价格上涨 → 赚差价
买入 CFD → 价格下跌 → 亏差价
（可以做多也可以做空）
```
