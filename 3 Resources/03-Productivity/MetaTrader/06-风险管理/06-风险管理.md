# 06 - MetaTrader 风险管理

> 交易的第一规则：**活下去**。先想亏损，再想盈利。

## 仓位计算

### 固定比例法

```
每笔交易风险 = 总资金的 1-2%

例如：账户 $10,000
  每笔最大亏损 = $10,000 × 2% = $200

止损 20 点 → 每点价值？
  如果做 EURUSD（1 手 = $10/点）
  200 / (20 × 10) = 1 手

止损 50 点：
  200 / (50 × 10) = 0.4 手
```

### 仓位计算公式

```
手数 = (本金 × 风险%) / (止损点数 × 每点价值)
```

## 止损策略

### 基于技术位

```
止损放在：
├── 支撑/阻力位外侧
├── 趋势线外侧
├── 前高/前低外侧
└── ATR（平均真实波幅）× 2
```

### 移动止损

```
价格向有利方向移动 X 点 → 止损跟随移动

MT4/MT5 设置：
  右键持仓 → 追踪止损 → 选择点数
```

## 止盈策略

### 盈亏比

```
最小盈亏比 = 1:2

止损 30 点 → 止盈至少 60 点
止损 50 点 → 止盈至少 100 点
```

### 分批止盈

```
仓位 1 手：
  到达 1:1 → 平仓 0.5 手（锁定利润）
  止损移到保本位
  到达 1:2 → 平仓剩余 0.5 手
```

## 资金管理法则

```
1. 单笔亏损 < 总资金 2%
2. 同时持仓 < 3 笔（分散风险）
3. 单日亏损 > 5% → 停止交易
4. 单周亏损 > 10% → 本周不交易
5. 单月亏损 > 20% → 反思策略
6. 永远不要补仓（加仓亏损单）
7. 不要在亏损后立即翻倍赌回来
```

## 心理管理

| 错误心态 | 正确心态 |
|----------|----------|
| 必须赚回来 | 亏损是交易的一部分 |
| 这次一定会涨 | 尊重止损 |
| 我比市场聪明 | 市场永远是对的 |
| 再等等就回来了 | 止损就是止损 |
