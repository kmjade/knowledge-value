# 08 - MetaTrader 经纪商选择

## 监管机构

| 监管 | 地区 | 可靠性 |
|------|------|--------|
| **FCA**（英国） | 英国 | ⭐⭐⭐⭐⭐ |
| **ASIC**（澳大利亚） | 澳大利亚 | ⭐⭐⭐⭐⭐ |
| **CySEC**（塞浦路斯） | 欧洲 | ⭐⭐⭐⭐ |
| **NFA/CFTC**（美国） | 美国 | ⭐⭐⭐⭐⭐ |
| **FSA**（日本） | 日本 | ⭐⭐⭐⭐⭐ |
| **FSCA**（南非） | 南非 | ⭐⭐⭐ |
| **FSC**（毛里求斯） | 毛里求斯 | ⭐⭐ |
| 无监管 | 离岸 | ⭐ |

> 优先选择 FCA / ASIC 监管的经纪商。

## 选择标准

```
□ 监管合规（最重要！）
□ 出入金是否顺畅？
□ 点差/佣金是否透明？
□ 是否支持 MT4/MT5？
□ 客服响应速度？
□ 成立时间 > 5 年？
□ 在 Myfxbook / ForexPeaceArmy 上的口碑？

⚠️ 避开：
  ❌ 无监管 / 离岸监管
  ❌ 出金需要额外"手续费"
  ❌ 承诺保本保收益
  ❌ 无法找到实体地址
```

## 常见经纪商类型

| 类型 | 模式 | 适合 |
|------|------|------|
| **ECN/STP** | 直通市场，赚佣金 | 透明交易 |
| **做市商** | 内部对冲 | 小额交易 |
| **混合** | ECN + 做市 | 大部分经纪商 |

## 出入金方式

| 方式 | 到账时间 | 手续费 |
|------|----------|--------|
| 银行电汇 | 1-5 天 | 高 |
| 信用卡 | 即时 | 低 |
| Skrill / Neteller | 即时 | 中 |
| USDT（加密货币） | 即时 | 低 |
| 银联 | 1-3 天 | 中 |

> 入金前先小额测试出入金流程是否顺畅。
