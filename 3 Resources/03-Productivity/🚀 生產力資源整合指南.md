#### 2. Getting Things Done (GTD)

**路徑**: `3 Resources/03-Productivity/Methods/Getting Things Done.md`

# 指南

**核心原則**:
1. **捕獲一切** - 不要依賴記憶，把所有事務記錄下來
2. **定期處理** - 系統性處理收集的項目
3. **明確下一步** - 每個項目都有明確的下一步行動
4. **歸檔清晰** - 所有項目都有明確歸屬
5. **定期回顧** - 保持系統新鮮和可靠
6. **根據情境行動** - 在適合的時間做適合的事

**五步法流程**:
```
捕獲 → 阐明 → 组织 → 反思 → 执行
```

**收集系統**:
- 物理位置：筆記本、便利貼、書桌
- 數字位置：筆記、郵件、瀏覽器標籤
- 心理位置：擔憂、想法、承諾、待辦

**情境分類**:
- @電腦：需要電腦的工作
- @電話：可以用電話完成
- @辦公室：需要在辦公室做
- @家：在家裡做的事
- @外出：出去時順便做
- @等待：等待他人回應的項目

**學習資源**:
- [[3 Resources/03-Productivity/Methods/Getting Things Done.md]]
- [Getting Things Done 官方網站](https://gettingthingsdone.com/)
- [Getting Things Done 書籍](https://www.amazon.com/Getting-Things-Done-Stress-Free-Productivity/dp/0142000280)

### 深度工作技巧

#### 深度工作環境

**物理環境**:
- [ ] 安靜的工作空間
- [ ] 舒適的椅子和桌子
- [ ] 充足的光線
- [ ] 適合的溫度（20-22℃）

**數字環境**:
- [ ] 關閉手機通知
- [ ] 使用網站攔截器
- [ ] 設定"勿擾"狀態
- [ ] 關閉不必要的標籤頁

**心理準備**:
- [ ] 設定明確的工作目標
- [ ] 準備好所有需要的資料
- [ ] 選擇高能量的時間段

**深度工作時段**:
- **番茄工作法**: 25分鐘專注 + 5分鐘休息
- **時間塊**: 為特定任務分配固定時間段（如：9:00-11:00 寫程式）
- **無干擾時間**: 設定每天的「深度工作時段」，此期間不接受打擾
