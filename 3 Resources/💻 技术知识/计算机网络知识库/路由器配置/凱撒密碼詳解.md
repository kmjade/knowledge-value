# 凱撒密碼詳解

> [!info] 概述
> 凱撒密碼（Caesar Cipher）是最古老的替換加密算法之一，相傳由尤利烏斯·凱撒發明。本文檔詳細介紹其原理、實現方法和應用場景。

---

## 一、凱撒密碼基礎

### 1.1 歷史背景

| 時間 | 人物 | 事件 |
|------|------|------|
| 公元前 100 年 | 尤利烏斯·凱撒 | 用於軍事通信 |
| 現代 | 密碼學研究 | 作為經典案例學習 |

### 1.2 基本原理

凱撒密碼通過將字母**位移固定位置**來進行加密：

```
原文:  ABCDEFGHIJKLMNOPQRSTUVWXYZ
密文:  DEFGHIJKLMNOPQRSTUVWXYZABC
       └─── 位移 3 位 ───┘
```

### 1.3 加密公式

```
密文位置 = (原文位置 + 位移量) mod 26
解密位置 = (密文位置 - 位移量 + 26) mod 26
```

## 三、程式實現

### 3.1 Python 實現

```python
def caesar_encrypt(text, shift):
    """
    凱撒密碼加密
    :param text: 明文
    :param shift: 位移量
    :return: 密文
    """
    result = ""
    for char in text:
        if char.isalpha():
            # 判斷大寫或小寫
            base = ord('A') if char.isupper() else ord('a')
            # 計算加密後的字符
            encrypted = (ord(char) - base + shift) % 26 + base
            result += chr(encrypted)
        else:
            # 非字母字符保持不變
            result += char
    return result

def caesar_decrypt(text, shift):
    """
    凱撒密碼解密
    :param text: 密文
    :param shift: 位移量
    :return: 明文
    """
    return caesar_encrypt(text, -shift)

# 使用示例
shift = 4

# 加密
plaintext = "telecomadmin"
encrypted = caesar_encrypt(plaintext, shift)
print(f"原文: {plaintext}")
print(f"密文: {encrypted}")  # 輸出: xipigsqehqmr

# 解密
decrypted = caesar_decrypt(encrypted, shift)
print(f"解密: {decrypted}")  # 輸出: telecomadmin
```

### 3.2 JavaScript 實現

```javascript
function caesarEncrypt(text, shift) {
    return text.split('').map(char => {
        if (char.match(/[a-z]/i)) {
            const base = char === char.toUpperCase() ? 65 : 97;
            return String.fromCharCode(
                (char.charCodeAt(0) - base + shift) % 26 + base
            );
        }
        return char;
    }).join('');
}

function caesarDecrypt(text, shift) {
    return caesarEncrypt(text, -shift);
}

// 使用示例
const shift = 4;
const plaintext = "telecomadmin";
const encrypted = caesarEncrypt(plaintext, shift);
console.log(`原文: ${plaintext}`);      // telecomadmin
console.log(`密文: ${encrypted}`);      // xipigsqehqmr
```

### 3.3 Bash 實現

```bash
#!/bin/bash

caesar_encrypt() {
    text=$1
    shift=$2
    result=""
    for ((i=0; i<${#text}; i++)); do
        char=${text:$i:1}
        if [[ $char =~ [a-zA-Z] ]]; then
            if [[ $char =~ [a-z] ]]; then
                base=97
            else
                base=65
            fi
            encrypted=$(($(printf '%d' "'$char") - $base + $shift))
            encrypted=$((($encrypted % 26 + 26) % 26 + $base))
            result+=$(printf "\\x%x" $encrypted | xxd -r -p)
        else
            result+=$char
        fi
    done
    echo $result
}

# 使用
caesar_encrypt "telecomadmin" 4  # 輸出: xipigsqehqmr
```

## 五、攻擊與防禦

### 5.1 暴力破解

```python
def brute_force_caesar(ciphertext):
    """
    暴力破解凱撒密碼
    嘗試所有 25 種可能的位移
    """
    results = []
    for shift in range(1, 26):
        decrypted = caesar_decrypt(ciphertext, shift)
        results.append((shift, decrypted))
    return results

# 使用
cipher = "xipigsqehqmr"
for shift, text in brute_force_caesar(cipher):
    print(f"位移 {shift:2d}: {text}")
```

```python
# 數字和符號保持不變
def caesar_encrypt_letters_only(text, shift):
    result = ""
    for char in text:
        if char.isalpha():
            base = ord('A') if char.isupper() else ord('a')
            encrypted = (ord(char) - base + shift) % 26 + base
            result += chr(encrypted)
        else:
            result += char
    return result
```

### 6.2ROT13（特殊位移）

ROT13 是一種特殊的凱撒密碼，位移量固定為 13：

```python
def rot13(text):
    return caesar_encrypt(text, 13)

# 特性：加密和解密是同一操作
# rot13(rot13(text)) == text
```

### 6.3 自定義字母表

```python
def custom_caesar(text, shift, alphabet="ABCDEFGHIJKLMNOPQRSTUVWXYZ"):
    result = ""
    for char in text.upper():
        if char in alphabet:
            index = alphabet.index(char)
            encrypted = alphabet[(index + shift) % len(alphabet)]
            result += encrypted
        else:
            result += char
    return result
```
