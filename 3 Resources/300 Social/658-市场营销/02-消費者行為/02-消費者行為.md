# 02 消費者行為

> People don't buy products; they buy better versions of themselves. — Samuel Hulick

---

## 消費者決策模型

### 五階段模型（EKB Model）

```
問題認知 → 資訊搜尋 → 方案評估 → 購買決策 → 購後行為
```

| 階段 | 行銷任務 | 觸發問題 |
|:-----|:---------|:---------|
| 1. 問題認知 | 創造需求意識 | "我需要什麼？" |
| 2. 資訊搜尋 | 確保品牌在考慮集合中 | "有什麼選擇？" |
| 3. 方案評估 | 突出差異化優勢 | "哪個最好？" |
| 4. 購買決策 | 降低購買障礙 | "現在買嗎？" |
| 5. 購後行為 | 管理期望·鼓勵推薦 | "買對了嗎？" |

### 涉入度（Involvement）

| 高涉入 | 低涉入 |
|:------|:------|
| 主動資訊搜尋 | 被動接收資訊 |
| 多方比較評估 | 習慣性購買 |
| 品牌忠誠度高 | 價格敏感 |
| 例：買車·買房·選大學 | 例：買鹽·買牙膏 |

## 心理因素

### 馬斯洛需求層次

```
自我實現 (Self-actualization)
    ↑
尊重需求 (Esteem)
    ↑
社交需求 (Love/Belonging)
    ↑
安全需求 (Safety)
    ↑
生理需求 (Physiological)
```

營銷啟示：產品應該對應哪一層需求？

### 知覺（Perception）

三個關鍵過程：

| 過程 | 說明 | 營銷應用 |
|:-----|:-----|:---------|
| **選擇性注意** | 消費者只注意部分刺激 | 打破 pattern、使用 contrast |
| **選擇性扭曲** | 按既有信念解讀資訊 | 先建立正面品牌形象 |
| **選擇性記憶** | 只記住支持信念的資訊 | 重複關鍵訊息·講故事 |

### 動機理論

| 理論 | 核心 | 應用 |
|:-----|:-----|:-----|
| Freud 潛意識動機 | 消費者受潛意識驅動 | 深度訪談·投射技術 |
| Maslow 需求層次 | 需求有優先順序 | 定位產品滿足哪層需求 |
| Herzberg 雙因素 | 不滿意因素 ≠ 激勵因素 | 消除 dissatisfier + 提供 satisfier |

## Cialdini 說服六原則

| 原則 | 說明 | 範例 |
|:-----|:-----|:-----|
| **互惠** Reciprocity | 給予 → 獲得回報 | 免費試用·樣品 |
| **承諾一致** Consistency | 小承諾 → 大行動 | 加入購物車·註冊 |
| **社會認同** Social Proof | 看別人怎麼做 | "最暢銷"·評價 |
| **喜好** Liking | 喜歡 → 更容易說 Yes | 相似性·讚美·合作 |
| **權威** Authority | 專家背書 | 認證·專家推薦 |
| **稀缺** Scarcity | 越少越想要 | "僅剩 3 件"·限時 |
