# 03 品牌與產品策略

> A brand is a promise. A product is the delivery of that promise.

---

## 品牌的本質

### 品牌 vs 產品

| | 產品 Product | 品牌 Brand |
|:--|:------------|:----------|
| 性質 | 有形/功能的 | 無形/情感的 |
| 在哪裡 | 貨架上 | 心智中 |
| 生命週期 | 有限 | 可無限延續 |
| 可複製性 | 容易被模仿 | 難以複製 |

### 品牌識別六面體（Kapferer's Brand Identity Prism）

```
      外部化 (Externalisation)
    ┌─────────────────────────┐
    │  Physique     Personality│
    │   (外形)        (個性)    │

    │  (反映)       (自我形象)  │
    └─────────────────────────┘
      內部化 (Internalisation)
```

### USP（Unique Selling Proposition 獨特賣點）

> 一句話說清楚：為什麼選你而不選競爭對手？

**好 USP 的標準**：
- 競爭對手做不到/沒說過
- 對目標消費者有意義
- 簡單到可以口耳相傳

**經典 USP 範例**：
| 品牌 | USP |
|:-----|:----|
| M&M's | "只溶在口，不溶在手" |
| Domino's | "30 分鐘送達，否則免費" |
| FedEx | "When it absolutely, positively has to be there overnight" |
| 小米 | "為發燒而生"（早期） |

## 產品生命週期 (PLC)

```
銷量
 │  Introduction  Growth   Maturity    Decline
 │    導入期       成長期    成熟期      衰退期
 │               ┌──────┐
 │              ╱        ╲──────
 │             ╱                 ╲
 │      ╱────╱                    ╲────
 │  ╱──╱
 └──────────────────────────────→ 時間
```

| 階段 | 特徵 | 策略重點 |
|:-----|:-----|:---------|
| 導入期 | 低銷量·高成本·無利潤 | 建立認知·早期採用者 |
| 成長期 | 快速增長·競爭者進入 | 擴大市佔·建立品牌 |
| 成熟期 | 增速放緩·利潤最高 | 差異化·降低成本 |
| 衰退期 | 銷量下降 | 收割/撤出/轉型 |

## BCG 矩陣

```
市場增長率
  ↑
  │  ★ Star        ❓ Question Mark
高│   (明星)          (問題兒童)
  │
  │  🐄 Cash Cow    🐕 Dog
低│   (金牛)          (瘦狗)
  │
  └─────────────────────→ 相對市場份額
     高                低
```

| 象限 | 策略 |
|:-----|:-----|
| **Star** 明星 | 投資成長 → 未來金牛 |
| **Cash Cow** 金牛 | 收割現金 → 餵養明星 |
| **Question Mark** 問題 | 選擇性投資或放棄 |
| **Dog** 瘦狗 | 剝離/收割 |
