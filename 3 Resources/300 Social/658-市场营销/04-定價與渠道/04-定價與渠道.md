# 04 定價與渠道

> Price is what you pay. Value is what you get. — Warren Buffett

---

## 第一部分：定價策略

### 定價的 3C 框架

```
     Cost（成本）
       ╱  ╲
      ╱    ╲
Customer ←→ Competition
（顧客）    （競爭）
```

好的定價在這三個 C 之間找到最優解。

### 主要定價方法

| 方法 | 邏輯 | 優勢 | 劣勢 |
|:-----|:-----|:-----|:-----|
| **成本加成** Cost-Plus | 成本 + 利潤率 | 簡單·保證利潤 | 忽略市場願付價格 |
| **競爭導向** Competition-Based | 跟隨市場價格 | 避免價格戰 | 同質化 |
| **價值導向** Value-Based | 顧客感知價值 | 利潤最大化 | 需深入理解顧客 |
| **動態定價** Dynamic Pricing | 供需實時調整 | 收益最優化 | 消費者抵觸 |

### 定價心理學

| 技巧 | 範例 | 原理 |
|:-----|:-----|:-----|
| **魅力定價** Charm Pricing | ¥99 vs ¥100 | 左位數效應 |
| **錨定效應** Anchoring | 先展示原價 ¥999，現價 ¥499 | 對比產生超值感 |
| **中間選項** Decoy Effect | 小杯 ¥3 / 中杯 ¥6.5 / 大杯 ¥7 | 中杯是 decoy，引導買大杯 |
| **捆綁定價** Bundling | 套餐比單點便宜 30% | 提高客單價 |
| **訂閱制** Subscription | 月付 ¥29 vs 一次性 ¥999 | 降低決策門檻 |

### 常見定價策略

| 策略 | 說明 | 場景 |
|:-----|:-----|:-----|
| **撇脂定價** Skimming | 高價進入，逐步降價 | 新技術·高端品牌 |
| **滲透定價** Penetration | 低價進入，搶市佔後提價 | 平台型產品 |
| **免費增值** Freemium | 基礎免費，高級收費 | SaaS·App |
| **差別定價** Price Discrimination | 不同顧客不同價格 | 早鳥票·學生價·地域價 |
| **損失領導** Loss Leader | 某商品虧本賣，引流 | 超市雞蛋·Costco 烤雞 |

## 第二部分：渠道策略

### 渠道層級

```
零級渠道： 生產者 ────────────→ 消費者 （D2C）
一級渠道： 生產者 → 零售商 → 消費者
二級渠道： 生產者 → 批發商 → 零售商 → 消費者
三級渠道： 生產者 → 代理商 → 批發商 → 零售商 → 消費者
```

### 渠道設計考量

| 因素 | 問題 |
|:-----|:-----|
| **市場覆蓋** | 密集分銷（FMCG）vs 獨家分銷（奢侈品）vs 選擇性分銷 |
| **渠道衝突** | 線上與線下如何不打架？ |
| **渠道利潤** | 每一層吃多少 margin？ |
| **控制力** | 品牌對渠道有多少話語權？ |

### 全渠道零售（Omnichannel）

> 消費者在任何渠道開始、任何渠道完成。

```
線上瀏覽 → 線下體驗 → 手機比價 → 店內購買 → 線上評價
   📱          🏪         📱          🏪         📱
```

關鍵：**所有渠道的庫存·價格·會員數據打通**。
