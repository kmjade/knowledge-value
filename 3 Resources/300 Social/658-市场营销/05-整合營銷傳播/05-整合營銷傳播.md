# 05 整合營銷傳播 (IMC)

> The aim of marketing is to make selling superfluous. — Peter Drucker

---

## IMC 是什麼？

IMC（Integrated Marketing Communications，整合營銷傳播）是**協調所有傳播工具和觸點，向消費者傳遞一致品牌訊息的策略框架**。

核心理念：
- **一個聲音**（One Voice）— 所有渠道傳遞同一品牌故事
- **360° 觸點**— 消費者在哪，品牌就在哪
- **雙向溝通**— 不是廣播，是對話

---

## 傳播工具組合 (Promotional Mix)

| 工具 | 特性 | 優勢 | 劣勢 |
|:-----|:-----|:-----|:-----|
| **廣告** Advertising | 付費·大眾·單向 | 覆蓋廣·可控 | 成本高·可信度下降 |
| **公關** PR | 贏得媒體·第三方背書 | 可信度高 | 不可控 |
| **銷售促進** Sales Promotion | 短期激勵 | 快速見效 | 可能損害品牌 |
| **直效營銷** Direct Marketing | 精準·個人化 | 可測量 | 侵入性 |
| **人員銷售** Personal Selling | 一對一·關係 | 說服力強 | 成本極高 |

### PESO 模型

```
Paid       Earned      Shared      Owned
付費媒體    贏得媒體     共享媒體     自有媒體
   │          │           │           │
 廣告·SEM   媒體報導    社交互動    官網·Blog
 贊助·KOL   口碑評價    UGC內容     App·Email
```

## AIDA 溝通模型

```
Attention  →  Interest  →  Desire  →  Action
  （注意）      （興趣）     （慾望）    （行動）
```

| 階段 | 目標 | 合適工具 |
|:-----|:-----|:---------|
| Attention 注意 | 打破 noise·被看見 | 戶外廣告·社群短影音·PR |
| Interest 興趣 | 引發好奇·深入了解 | 內容營銷·影片·Blog |
| Desire 慾望 | 建立偏好·想要擁有 | 試用·案例·評測 |
| Action 行動 | 促成購買/轉化 | 限時優惠·CTA·簡化結帳 |

### AIDA 的現代變體：See-Think-Do-Care（Avinash Kaushik）

| 階段 | 受眾 | 策略 |
|:-----|:-----|:-----|
| See | 最大範圍潛在受眾 | 品牌曝光·內容 |
| Think | 有初步意向 | 深度內容·比較 |
| Do | 準備購買 | 轉化優化·促銷 |
| Care | 已購買客戶 | 留存·忠誠·推薦 |

## 廣告策略基礎

### 創意簡報（Creative Brief）

一個好的創意簡報回答：

| 問題 | 說明 |
|:-----|:-----|
| **目標受眾是誰？** | 人口·心理·行為 |
| **我們想讓他們做什麼？** | 關鍵行為轉化 |
| **他們現在怎麼想？** | 現有認知/態度 |
| **我們想讓他們怎麼想？** | 期望認知/態度 |
| **單一最重要訊息是什麼？** | 如果只記住一句話 |
| **為什麼要相信我們？** | RTB (Reason to Believe) |

### 訊息策略

| 策略類型 | 說明 | 範例 |
|:---------|:-----|:-----|
| **理性訴求** Rational | 功能·數據·邏輯 | "省油 30%" |
| **感性訴求** Emotional | 情感·故事·價值觀 | Nike "Just Do It" |
| **恐懼訴求** Fear | 展示不使用的後果 | 保險·安全產品 |
| **幽默** Humor | 讓人發笑·記住 | 超級盃廣告 |
| **社會認同** Social Proof | "大家都在用" | 銷量第一·N 人已購買 |

## 促銷工具

| 工具 | 機制 | 風險 |
|:-----|:-----|:-----|
| 折扣/優惠券 | 價格降低 | 價格敏感化·品牌貶值 |
| 買贈/樣品 | 額外價值 | 成本 |
| 會員積分 | 累積回報 | 同質化 |
| 限時限量 | 稀缺性 | 消費者疲勞 |
| 競賽/抽獎 | 參與感 | 法規風險 |
| 聯合促銷 | 異業合作 | 品牌匹配度 |
