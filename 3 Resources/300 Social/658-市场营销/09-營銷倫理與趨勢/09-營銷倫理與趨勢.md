# 09 營銷倫理與未來趨勢

> The best marketing doesn't feel like marketing. — Tom Fishburne

---

## 第一部分：營銷倫理

### 核心倫理問題

| 議題 | 問題 | 應對 |
|:-----|:-----|:-----|
| **隱私** | 數據收集的邊界在哪？ | GDPR·CCPA 合規·透明告知 |
| **虛假廣告** | 誇大宣傳 vs 誤導 | 實質證據·第三方認證 |
| **兒童營銷** | 何時可以對未成年營銷？ | COPPA·自律規範 |
| **暗黑模式** Dark Patterns | 欺騙性 UI 設計 | 倫理審查·用戶測試 |
| **價格歧視** | 大數據殺熟 | 公平定價·透明度 |
| **綠色漂洗** Greenwashing | 假環保營銷 | 第三方認證·實質行動 |

### 道德營銷框架

```
                    合法性
                      ↑
        合法但不道德   │   合法且道德
         ──────────────┼──────────────
                      │
        違法且不道德   │   非法但合乎道德
                      │
                      └─────────────→ 道德性
```

**底線**: 合法且道德。Beyond compliance → 做對的事。

## 第二部分：ESG 與可持續營銷

### 三重底線（Triple Bottom Line）

```
     People（人）
       ╱  ╲
      ╱    ╲
Planet ←──→ Profit
（地球）      （利潤）
```

| 維度 | 營銷行動 |
|:-----|:---------|
| **環境** Planet | 減碳·可回收包裝·碳中和承諾 |
| **社會** People | 公平貿易·多元包容·社區貢獻 |
| **經濟** Profit | 長期價值·負責任供應鏈 |

## 第四部分：未來趨勢

| 趨勢 | 說明 | 影響 |
|:-----|:-----|:-----|
| **隱私優先** | Third-party cookie 消亡·First-party data 崛起 | 重新定義 targeting |
| **社群商務** Social Commerce | 抖音·小紅書·直播帶貨 | 內容即貨架 |
| **D2C 浪潮** | 品牌直銷跳過中間商 | 渠道權力轉移 |
| **元宇宙營銷** | 虛擬世界品牌體驗 | 新觸點實驗 |
| **營銷民主化** | No-code 工具·UGC·創作者經濟 | 人人都是營銷人 |
| **可持續性** | 消費者選「對的品牌」 | 價值觀驅動購買 |
