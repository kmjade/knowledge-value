# 06 市場研究

> Without data, you're just another person with an opinion. — W. Edwards Deming

---

## 市場研究流程

```
定義問題 → 研究設計 → 數據收集 → 數據分析 → 報告呈現 → 決策行動
```

| 階段 | 關鍵問題 |
|:-----|:---------|
| 定義問題 | 我們真正需要知道什麼？ |
| 研究設計 | 用什麼方法最有效？ |
| 數據收集 | 一手還是二手？定量還是定性？ |
| 數據分析 | 數據在說什麼故事？ |
| 呈現 | 如何讓決策者聽懂並行動？ |

---

## 研究方法對比

| | 定量研究 Quantitative | 定性研究 Qualitative |
|:--|:----------------------|:----------------------|
| 目標 | 測量·統計·驗證假設 | 探索·理解·發現洞察 |
| 樣本 | 大樣本（n ≥ 100） | 小樣本（n = 8–30） |
| 方法 | 問卷·實驗·數據分析 | 深度訪談·焦點小組·觀察 |
| 產出 | 數字·圖表·統計量 | 故事·主題·框架 |
| 問題類型 | "有多少人？" | "為什麼？" |

## 抽樣方法

```
抽樣
├── 概率抽樣（Probability）
│   ├── 簡單隨機 Simple Random
│   ├── 分層抽樣 Stratified
│   ├── 集群抽樣 Cluster
│   └── 系統抽樣 Systematic
└── 非概率抽樣（Non-probability）
    ├── 便利抽樣 Convenience
    ├── 判斷抽樣 Judgmental
    ├── 配額抽樣 Quota
    └── 滾雪球 Snowball
```

**關鍵考量**：樣本量、代表性、誤差範圍（±3–5% 常見）

## 問卷設計原則

### 好問題的特質

| 原則 | 差 | 好 |
|:-----|:---|:---|
| 簡潔 | "您過去 12 個月平均每月去幾次超市？" | "您每週去幾次超市？" |
| 中立 | "您同意我們的服務很棒嗎？" | "請評價我們的服務 1-5 分" |
| 單一 | "您覺得價格和品質如何？" | 價格和品質分開問 |
| 具體 | "您常買飲料嗎？" | "您過去一週買了幾瓶飲料？" |

---

## 關鍵指標

| 指標 | 公式 | 用途 |
|:-----|:-----|:-----|
| **NPS** (Net Promoter Score) | %推薦者 − %貶損者 | 忠誠度 |
| **CSAT** (Customer Satisfaction) | 滿意人數 ÷ 總回覆 | 滿意度 |
| **CES** (Customer Effort Score) | 平均努力評分 | 體驗順暢度 |
| **品牌知名度** | 知曉品牌人數 ÷ 總樣本 | 品牌健康 |
| **考慮度** | 會考慮購買 ÷ 知曉品牌 | 漏斗轉換 |
