# 人類學 Anthropology
研究人類文化、社會與生物多樣性的學科。

## 四大分支
| 分支 | 關注 |
|:----|:-----|
| **文化人類學** | 習俗·信仰·社會組織 |
| **體質人類學** | 人類演化·生物多樣性 |
| **考古學** | 過去社會的物質遺留 |
| **語言人類學** | 語言與文化的關係 |

## 核心概念
- **文化相對主義**：不以外在標準評判其他文化（博厄斯）
- **結構功能論**：文化元素服務社會整體功能（拉德克利夫-布朗）
- **深描**：厚實描述文化意義（格爾茨）

## Sources
- 博厄斯《原始人的心靈》(1911)
- 格爾茨《文化的解釋》(1973)
