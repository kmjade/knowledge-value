# 06 悖論 (Paradoxes)

## 1. 悖論的分類

| 類型 | 特徵 | 範例 |
|------|------|------|
| 語義悖論 (Semantic) | 涉及真理、指稱等語義概念 | Liar, Curry, Yablo |
| 集合論悖論 (Set-Theoretic) | 涉及集合、類等概念 | Russell, Cantor, Burali-Forti |
| 模糊性悖論 | 涉及模糊謂詞 | Sorites (見 [[../05-模糊邏輯/05-模糊邏輯\|Ch.05]]) |
| 理性悖論 | 涉及信念、決策 | Preface paradox, Lottery paradox |

## 2. 說謊者悖論 (The Liar Paradox)

### 經典形式

考慮語句 $L$：
$$L: \text{"This sentence is false" / 「本語句為假」}$$

若 $L$ 為真 → $L$ 所陳述的內容為真 → $L$ 為假 → 矛盾！
若 $L$ 為假 → $L$ 所陳述的內容為假 → $L$ 為真 → 矛盾！

∴ $L$ 既不能為真也不能為假 → 這挑戰了二值原則。

### 強化說謊者 (Strengthened Liar)

$$L': \text{"This sentence is not true" / 「本語句非真」}$$

若引入第三值試圖解決，$L'$ 會重新產生悖論：
- 若 $L'$ 為第三值 → $L'$ 非真 → $L'$ 為真 → 矛盾！

### Tarski 的層級論 (Tarski's Hierarchy)

Tarski (1933) 的解決方案：
- **對象語言 (Object language)** $\mathcal{L}_0$: 不含「真」謂詞
- **元語言 (Metalanguage)** $\mathcal{L}_1$: 包含 $\mathcal{L}_0$ 的真謂詞 $True_1$
- **元-元語言** $\mathcal{L}_2$: 包含 $\mathcal{L}_1$ 的真謂詞 $True_2$
- ...

> 一個語言的真理謂詞只能在更高層的語言中定義。

**優點**：避免自我指涉，保留古典邏輯
**缺點**：真謂詞的統一性喪失；無法表達「所有層級的真」；日常語言中「真」是統一的

## 3. Kripke 的固定點理論 (Kripke's Fixed-Point Theory)

Kripke (1975) 提出另一條進路，使用**三值邏輯**：

### 核心構造

- 語言 $\mathcal{L}$ 擴展含部分真謂詞 $T$
- 從空擴展開始，逐步將更多語句分類為真或假
- 達到**最小固定點 (least fixed point)**：在此點，$T$ 的擴展不再變化

### 結果

- 說謊者語句在固定點中**既非真也非假**（真值缺口）
- 強化說謊者不會出現：在固定點中，「本語句為真」也落在缺口
- 保留了直覺：日常語言中的「真」是統一的謂詞

### 比較 Tarski vs Kripke

| | Tarski 層級 | Kripke 固定點 |
|------|-------------|---------------|
| 邏輯 | 古典二值 | 三值 (Kleene strong) |
| 真謂詞 | 多層級 | 單一謂詞 |
| 說謊者 | 不合語法 (跨層) | 真值缺口 |
| 排中律 | 保留 | 失效 (缺口處) |

## 4. Dialetheism (雙真論) — Graham Priest

Priest 提出最激進的方案：**接受有些矛盾為真**。

### 核心主張

- 說謊者既是真也是假 → 這是**真矛盾 (true contradiction / dialetheia)**
- 使用**次協調邏輯 (paraconsistent logic)** LP：允許矛盾但不導致 trivialism
- 放棄爆炸原理 (Explosion)：$A \land \neg A \models B$

### 批評

- 直覺上難以接受 "真矛盾"
- 拒絕 (rational rejection) 的可理解性？
- Curry 悖論仍構成威脅（見下文）

## 5. Russell 悖論 (Russell's Paradox)

### 經典形式

令 $R = \{x \mid x \notin x\}$（所有不包含自身的集合的集合）。

$$R \in R \iff R \notin R$$

矛盾！

### 解決方案

- **Russell 的類型論 (Type Theory)**：$x \in y$ 僅當 $y$ 的類型比 $x$ 高一層
- **ZFC 集合論**：限制概括公理 (restricted comprehension)，$R$ 不是集合而是 proper class
- **NF (New Foundations)**：分層概括公理

## 6. Curry 悖論 (Curry's Paradox)

### 形式

令 $C$ 為語句：「若 $C$ 為真，則 $Q$」，其中 $Q$ 是任意命題。

證明 $Q$：
1. $C \equiv (True(C) \to Q)$ (by definition)
2. 假設 $True(C)$
3. 則 $True(C) \to Q$ (by 1)
4. 則 $Q$ (Modus Ponens, 2, 3)
5. ∴ $True(C) \to Q$ (discharging 2)
6. ∴ $C$ 為真 (by 1, 5 的等值)
7. ∴ $True(C)$
8. ∴ $Q$ (Modus Ponens, 5, 7)

### 重要性

Curry 悖論不依賴否定，因此**即使放棄排中律/矛盾律也無法避免**。對 Dialetheism 構成特殊挑戰。

## 7. Yablo 悖論 (Yablo's Paradox)

### 形式（非自我指涉）

考慮無限序列 $Y_1, Y_2, Y_3, \ldots$：

$$Y_k: \text{"For all } i > k, Y_i \text{ is false"}$$

- 若某 $Y_k$ 為真 → 所有 $Y_{k+1}, Y_{k+2}, \ldots$ 為假 → 矛盾
- 若某 $Y_k$ 為假 → 存在 $i > k$ 使得 $Y_i$ 為真 → 矛盾

### 哲學重要性

Yablo 聲稱此悖論**不涉及自我指涉 (self-reference)** — 每個 $Y_k$ 只指涉 $Y_{k+1}, Y_{k+2}$ 等，從不指涉自己。

**爭議**：Priest 認為 Yablo 悖論仍暗含循環性（依賴於非良基的指涉鏈）。
