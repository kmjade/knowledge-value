# 02 謂詞邏輯 (Predicate Logic)

## 1. 從命題邏輯到謂詞邏輯

命題邏輯將原子命題視為不可分析的最小單元。謂詞邏輯（一階邏輯, FOL）引入內部結構：

- **個體常元 (Individual constants)**: $a, b, c, \ldots$
- **個體變元 (Individual variables)**: $x, y, z, \ldots$
- **謂詞符號 (Predicate symbols)**: $P, Q, R, \ldots$ (帶有元數 arity)
- **量詞 (Quantifiers)**: $\forall$ (全稱, universal), $\exists$ (存在, existential)

### 範例對比

| 命題邏輯 | 謂詞邏輯 |
|----------|----------|
| $P$: Socrates is mortal | $M(s)$: $s$ = Socrates, $M$ = is mortal |
| $Q$: All men are mortal | $\forall x (Man(x) \to Mortal(x))$ |
| 無法表達關係 | $Loves(a, b)$: $a$ loves $b$ |

## 2. 語法 (Syntax)

### 項 (Terms)
1. 所有變元與常元都是 term
2. 若 $f$ 是 $n$-元函數符號，$t_1, \ldots, t_n$ 是 terms，則 $f(t_1, \ldots, t_n)$ 是 term

### 公式 (Formulas)
1. 若 $P$ 是 $n$-元謂詞，$t_1, \ldots, t_n$ 是 terms，則 $P(t_1, \ldots, t_n)$ 是原子公式
2. 命題連接詞規則同上
3. 若 $A$ 是公式，$x$ 是變元，則 $\forall x A$ 和 $\exists x A$ 是公式

### 自由變元與約束變元 (Free vs Bound Variables)

- $\forall x P(x)$：$x$ 是 bound
- $P(x)$：$x$ 是 free
- **語句 (Sentence)**: 無自由變元的公式

## 3. 語意學：模型論 (Semantics: Model Theory)

### 模型 (Model) $\mathcal{M} = \langle D, I \rangle$

- **Domain $D$**: 非空集合（個體域）
- **Interpretation function $I$**:
  - 常元 $c \mapsto I(c) \in D$
  - $n$-元謂詞 $P \mapsto I(P) \subseteq D^n$
  - $n$-元函數 $f \mapsto I(f): D^n \to D$

### 滿足關係 (Satisfaction) $\mathcal{M} \models_s A$

- $\mathcal{M} \models_s P(t_1, \ldots, t_n)$ iff $\langle val_s(t_1), \ldots, val_s(t_n) \rangle \in I(P)$
- $\mathcal{M} \models_s \forall x A$ iff for all $d \in D$, $\mathcal{M} \models_{s[x:=d]} A$
- $\mathcal{M} \models_s \exists x A$ iff for some $d \in D$, $\mathcal{M} \models_{s[x:=d]} A$

### 邏輯真理 (Logical Truth)

$A$ 在所有模型中為真 $\Rightarrow$ $\models A$

## 4. 等同 (Identity)

引入二元謂詞 $=$，滿足：

- **自反性**: $\forall x (x = x)$
- **Leibniz 律 (Indiscernibility of Identicals)**: $x = y \to (A(x) \leftrightarrow A(y))$

等同使我們能表達數量概念：
- "至少兩個": $\exists x \exists y (x \neq y \land P(x) \land P(y))$
- "恰好一個": $\exists x (P(x) \land \forall y (P(y) \to y = x))$

## 5. 限定描述詞 (Definite Descriptions)

Russell (1905) 的著名分析：

"The $F$ is $G$" → $\exists x (F(x) \land \forall y (F(y) \to y = x) \land G(x))$

即：恰好有一個 $F$，且那個 $F$ 是 $G$。

### Russell 分析的優勢

- 消解了 "The present King of France is bald" 的真值問題
- 將描述詞分析為量化結構，而非指稱表達式

### Strawson 的批評

Strawson 認為描述詞是**預設 (presupposition)** 而非斷言 (assertion)：當無 King of France 時，語句不是假的，而是**真值空缺 (truth-value gap)**。

## 6. 一階邏輯的元理論性質

| 性質 | 命題邏輯 | 一階邏輯 |
|------|----------|----------|
| Soundness 健全性 | ✓ | ✓ |
| Completeness 完備性 (Gödel 1930) | ✓ | ✓ |
| Compactness 緊緻性 | ✓ | ✓ |
| Decidability 可判定性 | ✓ (SAT) | ✗ (Church/Turing 1936) |
| Löwenheim-Skolem | N/A | 若有無限模型，則有任意無窮基數模型 |

### Gödel 不完備性定理的影響

一階邏輯本身是完備的，但包含算術的理論（如 PA）是不完備的。這不是邏輯的缺陷，而是算術理論的內在限制。
