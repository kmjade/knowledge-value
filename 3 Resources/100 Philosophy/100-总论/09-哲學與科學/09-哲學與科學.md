# 09 哲學與科學（Philosophy and Science）

科學哲學（Philosophy of Science）研究科學的基礎、方法與含義。核心問題包括：什麼區分科學與非科學？科學方法有何特殊？科學知識的性質是什麼？

## 劃界問題（The Demarcation Problem）

> 如何區分科學（science）與偽科學（pseudoscience）？

這是 Karl Popper（1902–1994）視為科學哲學核心的問題。

### 常見的劃界嘗試

| 標準 | 說明 | 問題 |
|------|------|------|
| **經驗驗證** | 科學命題可被經驗驗證 | 歸納問題；形而上學命題也可能被「驗證」 |
| **可否證性（Falsifiability）** | 科學命題必須能被經驗否證 | 歷史上有科學家不理會否證 |
| **典範共識** | 科學社群共享一套典範 | 怎麼判斷什麼是典範？ |
| **進步性** | 科學持續進步 | 什麼算「進步」？ |

## Popper 的證偽主義（Falsificationism）

### 核心主張

Karl Popper 在 *The Logic of Scientific Discovery*（1934/1959）中提出：

> 一個理論是科學的，若且唯若它能被經驗觀察所否證（falsifiable）。

| 概念 | 說明 |
|------|------|
| **證偽（Falsification）** | 與驗證（verification）對立：科學不是通過驗證前進，而是通過大膽猜想與嚴格否證 |
| **可否證性（Falsifiability）** | 科學理論必須做出可被經驗檢驗的預測，且這些預測存在被證明為假的可能性 |
| **逼真性（Verisimilitude）** | 被否證的理論仍可能比競爭理論更接近真理 |

### Popper 對馬克思主義與精神分析的批評

Popper 認為 Marxism 與 Freudian psychoanalysis 是偽科學，因為它們可以解釋任何觀察，無法被否證。相比之下，Einstein 的相對論做出了可被否證的大膽預測（如：1919 年日食觀測）。

### Popper 的科學方法圖景

```
問題（Problem） → 猜想（Conjecture） → 否證（Refutation） → 新問題（New Problem）
```

> 科學始於問題，而非觀察。觀察總是在理論的指導下進行（theory-laden）。

## Kuhn 的典範轉移（Paradigm Shifts）

Thomas Kuhn（1922–1996）在 *The Structure of Scientific Revolutions*（1962）中挑戰 Popper 與邏輯實證論。

### 核心概念

| 概念 | 說明 |
|------|------|
| **典範（Paradigm）** | 科學社群共享的理論框架、方法標準、模範解題 |
| **常態科學（Normal Science）** | 在典範框架內解決謎題（puzzle-solving） |
| **危機（Crisis）** | 異常（anomalies）積累，典範無法解釋 |
| **科學革命（Scientific Revolution）** | 舊典範被新典範取代（如：Newton → Einstein） |
| **不可共量性（Incommensurability）** | 新舊典範之間缺乏中立的比較標準 |

### Kuhn vs Popper 對比

| 維度 | Popper | Kuhn |
|------|--------|------|
| **科學圖景** | 持續革命（permanent revolution） | 常態科學 + 偶發革命 |
| **科學進步** | 通過否證逼近真理 | 無必然的真理趨近 |
| **方法論** | 規範性（科學應該怎麼做） | 描述性（科學實際上怎麼做） |
| **理性** | 科學是理性的 | 典範轉移有非理性因素 |
| **社會因素** | 不重要 | 核心（科學社群共識） |

## 其他重要科學哲學觀點

### Lakatos 的研究綱領（Research Programmes）

Imre Lakatos（1922–1974）試圖調和 Popper 與 Kuhn：

| 元素 | 說明 |
|------|------|
| **硬核（Hard Core）** | 綱領的基本假設，不容否證（如 Newton 力學三定律） |
| **保護帶（Protective Belt）** | 輔助假設，可調整以保護硬核 |
| **進步綱領（Progressive Programme）** | 能預測新事實 |
| **退化綱領（Degenerating Programme）** | 只能事後解釋，不能預測 |

### Feyerabend 的認識論無政府主義

Paul Feyerabend（1924–1994）：*Against Method*（1975）

> "Anything goes." — 沒有唯一正確的科學方法。科學的進步往往來自打破方法論規則。

## 科學實在論 vs 反實在論（Scientific Realism vs Anti-Realism）

| 立場 | 主張 | 代表人物 |
|------|------|----------|
| **科學實在論** | 成熟的科學理論（大致）為真；理論術語指稱真實存在的不可觀察實體 | Putnam, Boyd |
| **工具論（Instrumentalism）** | 科學理論只是預測工具，不需要為真 | Duhem, van Fraassen |
| **建構經驗論（Constructive Empiricism）** | 科學目標是經驗妥當性，而非真理 | van Fraassen |

### 無奇蹟論證（No-Miracles Argument）

Putnam 的論證支持實在論：如果科學理論不是（大致）真的，科學的成功就是一個奇蹟。科學成功的最佳解釋是理論為真。

### 悲觀歸納（Pessimistic Induction）

Laudan 的反駁：歷史上有許多成功的科學理論後來被證明是假的（如：燃素說、以太說）。我們現在的理論也可能如此。

## 科學解釋（Scientific Explanation）

### Hempel 的覆蓋律模型（Covering Law Model）

| 類型 | 說明 |
|------|------|
| **演繹-律則模型（D-N Model）** | 從普遍定律與初始條件演繹出待解釋現象 |
| **歸納-統計模型（I-S Model）** | 從統計定律導出待解釋現象為高概率 |

### 因果解釋與統一解釋

當代替代方案：
- **因果機械論（Causal-Mechanical）**：解釋是揭示因果機制（Salmon）
- **統一論（Unificationist）**：解釋是將現象統攝到更少的基本模式中（Kitcher）
