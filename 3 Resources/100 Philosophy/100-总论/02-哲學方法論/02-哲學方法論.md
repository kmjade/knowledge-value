# 02 哲學方法論（Philosophical Methodology）

哲學方法論探討哲學家如何進行研究與論證。與科學依賴實驗不同，哲學依賴概念分析、邏輯推理與思想實驗。

## 理性主義 vs 經驗主義（Rationalism vs Empiricism）

這是認識論與方法論上最根本的對立之一。

| 維度 | 理性主義（Rationalism） | 經驗主義（Empiricism） |
|------|------------------------|------------------------|
| **知識來源** | 理性（Reason）與先天觀念 | 感官經驗（Sense Experience） |
| **核心主張** | 某些知識獨立於經驗（a priori） | 所有知識來自經驗（a posteriori） |
| **典範** | 數學、邏輯 | 自然科學 |
| **代表** | Descartes, Spinoza, Leibniz | Locke, Berkeley, Hume |
| **口號** | "I think, therefore I am" | "Nothing in the intellect that was not first in the senses" |
| **優勢** | 必然性、普遍性 | 與科學相容、具體可驗證 |
| **弱點** | 如何確保理性對應實在？ | 如何解釋數學與邏輯知識？ |

### 康德綜合（Kant's Synthesis）

Kant（1724–1804）試圖調和兩者：知識始於經驗，但經驗的結構（時間、空間、範疇）來自心靈的先驗形式（a priori forms）。這是他的「哥白尼革命」（Copernican Revolution）。

## 主要哲學方法（Major Philosophical Methods）

### 1. 蘇格拉底方法（Socratic Method / Elenchus）
通過追問與反駁來檢驗信念。步驟：
1. 提出一個定義或主張
2. 追問其含義與後果
3. 揭示矛盾或不一致
4. 修正或放棄原主張

> 蘇格拉底自稱「助產士」（midwife），幫助他人產出真理。

### 2. 辯證法（Dialectic）
從 Hegel（1770–1831）到 Marx（1818–1883），辯證法描述思想的發展動力：

```
正題（Thesis） → 反題（Antithesis） → 合題（Synthesis）
```

Hegel 認為這是概念與歷史發展的內在邏輯。Marx 將其「顛倒」為唯物辯證法（dialectical materialism）。

### 3. 現象學（Phenomenology）
Husserl（1859–1938）創立的方法，口號：「回到事物本身！」（Zu den Sachen selbst!）

核心操作：
- **懸置（Epoché / Bracketing）**：擱置對外部世界存在的判斷
- **本質直觀（Eidetic Reduction）**：從具體經驗中直觀普遍本質
- **意向性（Intentionality）**：意識總是關於某物的意識

### 4. 分析哲學（Analytic Philosophy）
20 世紀英美哲學主流，強調：
- 語言分析與概念澄清
- 邏輯形式化
- 論證的精確重建
- 對形而上學思辨的警惕

代表人物：Russell, Wittgenstein（早期）, Quine, Kripke

### 5. 思想實驗（Thought Experiments）
在想像情境中測試概念與直覺：

| 思想實驗 | 提出者 | 用途 |
|----------|--------|------|
| 洞穴寓言（Allegory of the Cave） | Plato | 說明理型論 |
| 惡魔欺騙（Evil Demon） | Descartes | 極端懷疑論 |
| 中國房間（Chinese Room） | Searle | 反對強 AI |
| 電車難題（Trolley Problem） | Foot/Thomson | 測試倫理直覺 |
| 體驗機（Experience Machine） | Nozick | 挑戰享樂主義 |

### 6. 反思均衡（Reflective Equilibrium）
Rawls（1921–2002）發展的方法：在一般原則與具體判斷之間來回調整，直到達成一致。廣泛用於倫理學與政治哲學。

## 方法選擇指南（Method Selection Guide）

| 你要做什麼 | 適合方法 |
|-----------|---------|
| 檢驗定義 | Socratic Method |
| 分析概念 | Analytic Philosophy |
| 描述經驗結構 | Phenomenology |
| 測試直覺 | Thought Experiments |
| 建立體系 | Dialectic |
| 調和衝突 | Reflective Equilibrium |
