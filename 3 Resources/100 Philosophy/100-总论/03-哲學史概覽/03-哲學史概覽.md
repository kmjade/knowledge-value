# 03 哲學史概覽（History of Philosophy — Overview）

西方哲學史可大致分為四個時期：古代（Ancient）、中世紀（Medieval）、近代（Modern）與現當代（Contemporary）。每個時期都有其核心問題與典範轉移。

## 四大時期總覽（Four Major Periods）

| 時期 | 時間 | 核心關注 | 關鍵轉折 |
|------|------|----------|----------|
| **古代** | 6th C BC – 5th C AD | 宇宙論、理型、幸福 | 從神話到理性（Mythos → Logos） |
| **中世紀** | 5th – 15th C | 信仰與理性的關係 | 哲學服務於神學（ancilla theologiae） |
| **近代** | 16th – 19th C | 認識論轉向、主體性 | 從上帝到主體（From God to Subject） |
| **現當代** | 20th C – 至今 | 語言、意義、權力 | 語言學轉向、後現代批判 |

---

## 古代哲學詳細時間線（Ancient Philosophy Timeline）

| 時期 | 人物 | 年代 | 核心貢獻 |
|------|------|------|----------|
| **先蘇時期** | Thales | c.624–546 BC | 萬物源於水；理性解釋自然 |
| | Anaximander | c.610–546 BC | Apeiron（無定）為本源 |
| | Heraclitus | c.535–475 BC | 萬物皆流（Panta rhei）；Logos |
| | Parmenides | c.515–450 BC | 存在是一，變化是幻象 |
| | Democritus | c.460–370 BC | 原子論（Atomism） |
| **古典時期** | Socrates | 470–399 BC | 倫理學轉向；Socratic method |
| | Plato | 428–348 BC | 理型論（Theory of Forms）；《理想國》 |
| | Aristotle | 384–322 BC | 邏輯學奠基；形而上學；倫理學系統化 |
| **希臘化時期** | Epicurus | 341–270 BC | 快樂主義（Epicureanism） |
| | Zeno of Citium | c.334–262 BC | 斯多葛主義（Stoicism） |
| | Pyrrho | c.360–270 BC | 懷疑論（Skepticism） |

### 關鍵轉折：Mythos → Logos
從神話敘事（Mythos）轉向理性論證（Logos），這是哲學誕生的決定性時刻。Thales 首次以自然原因（水）解釋世界，而非訴諸神話。

---

## 中世紀哲學（Medieval Philosophy）

| 人物 | 年代 | 核心貢獻 |
|------|------|----------|
| Augustine | 354–430 | 信仰尋求理解；原罪；上帝之城 |
| Boethius | c.480–524 | 《哲學的慰藉》；共相問題 |
| Anselm | 1033–1109 | 本體論論證（Ontological Argument） |
| Aquinas | 1225–1274 | 綜合 Aristotle 與基督教；五路論證 |
| Ockham | 1287–1347 | Ockham's Razor；唯名論（Nominalism） |

### 核心張力：信仰 vs 理性（Faith vs Reason）
從 Augustine 的「信仰尋求理解」（fides quaerens intellectum）到 Aquinas 的系統綜合，再到 Ockham 將理性與信仰分離，預示近代的到來。

---

## 近代哲學（Modern Philosophy）

| 人物 | 年代 | 核心貢獻 |
|------|------|----------|
| Descartes | 1596–1650 | "I think, therefore I am"；心物二元論 |
| Spinoza | 1632–1677 | 泛神論（Pantheism）；倫理學幾何學證明 |
| Leibniz | 1646–1716 | 單子論（Monadology）；充足理由律 |
| Locke | 1632–1704 | 經驗主義；白板說（Tabula Rasa） |
| Berkeley | 1685–1753 | 唯心論（Idealism）：存在即被感知 |
| Hume | 1711–1776 | 極端經驗主義；因果懷疑論 |
| Kant | 1724–1804 | 先驗觀念論；定言令式 |
| Hegel | 1770–1831 | 絕對觀念論；歷史辯證法 |

### 認識論轉向（The Epistemological Turn）
Descartes 將哲學焦點從「什麼存在」轉向「我們如何知道」，開啟近代哲學的核心關切。

---

## 現當代哲學（Contemporary Philosophy）

| 人物 | 年代 | 核心貢獻 |
|------|------|----------|
| Nietzsche | 1844–1900 | 上帝已死；權力意志；永恆輪迴 |
| Husserl | 1859–1938 | 現象學奠基 |
| Heidegger | 1889–1976 | 存有問題；此在（Dasein） |
| Wittgenstein | 1889–1951 | 語言遊戲；意義即使用 |
| Sartre | 1905–1980 | 存在先於本質；存在主義 |
| Foucault | 1926–1984 | 權力/知識；規訓社會 |
| Derrida | 1930–2004 | 解構（Deconstruction） |
| Rawls | 1921–2002 | 正義論；反思均衡 |

### 語言學轉向（The Linguistic Turn）
20 世紀哲學的核心是語言：分析哲學與歐陸哲學雖然路徑不同，但都將語言視為哲學的關鍵場域。
