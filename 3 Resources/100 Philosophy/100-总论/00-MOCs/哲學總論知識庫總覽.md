# 哲學總論知識庫總覽（MOC: Philosophy General）

此地圖（Map of Content）提供 DDC 100 哲學總論知識庫的完整導航，涵蓋研究光譜、關鍵問題、與建議閱讀順序。

```
100-总论/
├── 01-哲學是什麼/          ← 哲學本質、分支、哲學態度
├── 02-哲學方法論/          ← 思維工具與方法
├── 03-哲學史概覽/          ← 思想史四大時期
├── 04-形而上學導論/        ← 存有、實體、因果

├── 06-倫理學導論/          ← 三大規範倫理理論
├── 07-邏輯學導論/          ← 形式邏輯與非形式謬誤
├── 08-美學導論/            ← 美與藝術的哲學
├── 09-哲學與科學/          ← 科學哲學與劃界
└── 99-資源收集/            ← 書籍、課程、期刊
```

## 研究光譜（Research Spectrum）

哲學研究從最抽象到最具體可排列為：

```
形而上學（最抽象）
    ↓
認識論
    ↓
邏輯
    ↓
倫理學
    ↓
美學
    ↓
科學哲學（最具體）
```

## 關鍵問題地圖（Key Questions Map）

每個分支的核心追問：

| 分支 | 核心問題 | 代表人物 |
|------|----------|----------|
| 形而上學 | 存在是什麼？有自由意志嗎？ | Aristotle, Kant, Heidegger |
| 認識論 | 知識的條件是什麼？我們能知道什麼？ | Plato, Descartes, Gettier |
| 倫理學 | 什麼是對的行為？善是什麼？ | Aristotle, Kant, Mill |
| 邏輯 | 什麼使一個論證有效？ | Aristotle, Frege, Gödel |
| 美學 | 美是主觀的還是客觀的？ | Kant, Hegel, Danto |
| 科學哲學 | 科學與偽科學的界限在哪？ | Popper, Kuhn, Feyerabend |

## 核心哲學家速查（Key Philosophers Quick Reference）

| 哲學家 | 年代 | 領域 | 核心貢獻 |
|--------|------|------|----------|
| Socrates | 470-399 BC | 倫理學、方法論 | Socratic method |
| Plato | 428-348 BC | 形而上學、認識論 | Theory of Forms |
| Aristotle | 384-322 BC | 全領域 | 邏輯、形而上學系統化 |
| Descartes | 1596-1650 | 認識論 | "I think, therefore I am", mind-body dualism |
| Kant | 1724-1804 | 認識論、倫理學 | 先驗觀念論、定言令式 |
| Nietzsche | 1844-1900 | 倫理學、形而上學 | 上帝已死、權力意志 |
| Wittgenstein | 1889-1951 | 邏輯、語言哲學 | 語言遊戲、意義即使用 |
