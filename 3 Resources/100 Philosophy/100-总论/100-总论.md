# 100 哲學總論（Philosophy — General）

哲學（Philosophy）源自希臘文 φιλοσοφία（philosophia），意為「愛智慧」（love of wisdom）。作為人類最古老的學科之一，哲學探討存在、知識、價值、理性、心靈與語言等根本問題。本知識庫為 DDC 100 的總論入口，提供系統性的哲學導覽。

## 核心問題（Core Questions）

哲學處理以下根本提問：

1. **形而上學（Metaphysics）**：什麼是真實的？（What is real?）存在是什麼？有無自由意志？
2. **認識論（Epistemology）**：我們如何知道？（How do we know?）知識的界限在哪裡？
3. **倫理學（Ethics）**：我們應當如何生活？（How should we live?）何謂善？
4. **邏輯（Logic）**：什麼構成好的推理？（What constitutes good reasoning?）
5. **美學（Aesthetics）**：什麼是美？（What is beauty?）藝術的本質為何？

## 關鍵術語速查（Key Terminology Quick Reference）

| 中文 | English | 簡要定義 |
|------|---------|----------|
| 形而上學 | Metaphysics | 研究存在與實在之本質 |
| 認識論 | Epistemology | 研究知識的性質、來源與範圍 |
| 倫理學 | Ethics | 研究道德價值與行為規範 |
| 邏輯 | Logic | 研究有效推理的原則 |
| 美學 | Aesthetics | 研究美、藝術與審美經驗 |
| 本體論 | Ontology | 形而上學分支，研究存有本身 |
| 先驗 | A priori | 獨立於經驗的知識 |
| 後驗 | A posteriori | 依賴於經驗的知識 |
| 辯證法 | Dialectic | 通過對立命題的對話尋求真理 |
