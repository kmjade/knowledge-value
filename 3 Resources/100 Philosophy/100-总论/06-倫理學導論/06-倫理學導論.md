# 06 倫理學導論（Introduction to Ethics）

倫理學（Ethics / Moral Philosophy）研究道德價值、行為規範與善的生活。核心問題：我們應當如何生活？（How ought we to live?）

## 倫理學的三個層次（Three Levels of Ethics）

| 層次 | 研究對象 | 核心提問 |
|------|----------|----------|
| **元倫理學（Meta-Ethics）** | 道德語言的意義與本質 | 「善」是什麼意思？道德是客觀的嗎？ |
| **規範倫理學（Normative Ethics）** | 行為的對錯標準 | 什麼使一個行為是對的？ |
| **應用倫理學（Applied Ethics）** | 具體道德問題 | 墮胎是否道德？安樂死是否正當？ |

## 三大規範倫理理論（Three Major Normative Theories）

### 1. 義務論（Deontology）

核心主張：行為的道德價值取決於它是否符合道德義務/規則，而非其後果。

**Kant 的定言令式（Categorical Imperative）**：

> 「只按照你能同時意願它成為普遍法則的格律去行動。」（*Groundwork of the Metaphysics of Morals*, 1785）

兩個主要表述：
1. **普遍法則公式（Formula of Universal Law）**：你的行動格律能否成為普遍法則？
2. **人性公式（Formula of Humanity）**：始終將人（包括自己）視為目的本身，而非僅僅手段

| 優點 | 缺點 |
|------|------|
| 尊重個人尊嚴與權利 | 義務衝突時難以抉擇 |
| 提供明確的行為指導 | 可能忽視後果（如：不能對兇手說謊） |
| 道德的普遍性與必然性 | 「哪些是真正的義務？」有爭議 |

### 2. 效益論（Utilitarianism / Consequentialism）

核心主張：行為的道德價值取決於其後果，正確的行為是能產生最大幸福的行為。

**Bentham（1748–1832）與 Mill（1806–1873）**：

> 「最大多數人的最大幸福。」（Greatest Happiness Principle）

| 類型 | 說明 |
|------|------|
| **行為效益論（Act Utilitarianism）** | 每個行為單獨計算後果 |
| **規則效益論（Rule Utilitarianism）** | 遵循通常能產生最佳後果的規則 |
| **偏好效益論（Preference Utilitarianism）** | 滿足偏好而非最大化快樂（Singer） |

| 優點 | 缺點 |
|------|------|
| 直覺上合理（後果確實重要） | 可能正義犧牲少數（如：懲罰無辜者） |
| 提供計算方法（功利計算） | 幸福/痛苦難以量化與比較 |
| 不依賴可疑的形而上學 | 過於苛求（demandingness objection） |

### 3. 德行倫理學（Virtue Ethics）

核心主張：倫理學的核心不是行為規則，而是培養良好的品格特質（virtues）。

**Aristotle 的 eudaimonia（幸福/興盛）**：

> 德性是使一個事物良好地執行其功能的特質。人的功能是理性的活動，因此人的善在於合乎德性的理性活動。

| 核心概念 | 說明 |
|----------|------|
| **Eudaimonia** | 人的終極目的：興盛/幸福（不僅是快樂） |
| **中庸之道（Golden Mean）** | 德性是兩個極端之間的中道（如：勇敢是怯懦與魯莽之間的中道） |
| **實踐智慧（Phronesis）** | 在具體情境中判斷如何行動的實踐理性 |

| 優點 | 缺點 |
|------|------|
| 關注整體人格而非單一行為 | 缺乏具體行為指導 |
| 符合日常道德經驗 | 不同文化的德性可能衝突 |
| 避免規則 vs 後果的對立 | 什麼算「德性」本身有爭議 |

## 三大理論比較（Comparison Table）

| 維度 | 義務論 | 效益論 | 德行倫理 |
|------|--------|--------|----------|
| **核心** | 義務/規則 | 後果/幸福 | 品格/德性 |
| **代表人物** | Kant | Bentham, Mill | Aristotle |
| **判斷對象** | 行為本身 | 行為後果 | 行為者 |
| **口號** | "Let justice be done though the heavens fall" | "The greatest good for the greatest number" | "Be virtuous" |
| **電車難題** | 不應推人（尊重權利） | 應推一人救五人（最大化幸福） | 有德者會如何做？ |

## 元倫理學（Meta-Ethics）速覽

| 立場 | 主張 |
|------|------|
| **道德實在論（Moral Realism）** | 道德事實客觀存在 |
| **道德反實在論（Moral Anti-Realism）** | 道德事實不存在（錯誤理論、情緒論、規範論） |
| **情緒論（Emotivism）** | 道德判斷只是情感表達（Ayer, Stevenson） |
| **錯誤理論（Error Theory）** | 所有道德命題皆假（Mackie） |
