# 170 倫理學 / Ethics

DDC 170 covers moral philosophy — the systematic study of right and wrong, good and bad,
duty and virtue. 倫理學是哲學的核心分支之一，探究人類行為的規範性問題。

## 核心問題 / Core Questions

1. What makes an action right or wrong? 什麼使行為對或錯？
2. Are moral truths objective? 道德真理是客觀的嗎？
3. Why be moral? 為什麼要有道德？

> *"Act only according to that maxim whereby you can at the same time will that it should become a universal law."* — Immanuel Kant
