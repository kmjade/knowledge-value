## 🧭 研究光譜

認識論的核心爭論可以沿以下軸線展開：

```
知識的本質
  ├── JTB 正統 ───────────── 葛梯爾挑戰 ────── 知識優先論
  │    (Plato)                   (Gettier)          (Williamson)
  │
知識的來源
  ├── 理性主義 ──────────── 康德綜合 ──────────── 經驗主義
  │    (Descartes)              (Kant)              (Hume)
  │
證成的結構
  ├── 基礎論 ──────────── 融貫論 ──────────── 無限論
  │    (Descartes)            (BonJour)            (Klein)
  │
懷疑論的挑戰
  ├── 皮浪主義 ────────── 笛卡爾式 ────────── 缸中之腦
  │    (Sextus)              (Descartes)          (Putnam)
  │
知識的範圍
  ├── 先驗知識 ────────── 經驗知識 ────────── 道德知識
  │    (Kant)                 (Locke)              (道德認識論)
  │
認識的社會維度
  ├── 個人主義 ────────── 社會認識論 ──────── 認識不正義
       (傳統)                 (Goldman)            (Fricker)
```
