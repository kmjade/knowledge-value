# 01 知識的定義 / The Definition of Knowledge

> 什麼是知識？柏拉圖在兩千四百年前提出、統治西方哲學至今的答案是：**知識是經過證成的真信念（Justified True Belief）**。

---

## 一、柏拉圖的《泰阿泰德篇》（Theaetetus）

柏拉圖的對話錄《泰阿泰德篇》是認識論的奠基文本。蘇格拉底與年輕數學家泰阿泰德探討「什麼是知識？」（*episteme*），依次檢驗了三種定義：

### 1. 知識即知覺（Knowledge is Perception）

泰阿泰德首先提出：知識就是知覺（perception）。蘇格拉底反駁：

- 知覺是相對的——同一陣風對你冷對我暖（普羅泰戈拉 relativism）
- 知覺無法解釋未來預測與抽象數學知識
- **結論**：知覺不是知識（但可能是知識的來源之一）

### 2. 知識即真信念（Knowledge is True Belief）

第二個提議：知識就是真信念（true belief / *alethes doxa*）。蘇格拉底反駁：

- 陪審團可以被說服而持有真信念，但他們並不「知道」——他們沒有親眼所見
- 真信念可以是運氣好猜對的，但猜對不是知道
- **結論**：真信念不足以構成知識——需要額外的東西

### 3. 知識即經過證成的真信念（Knowledge is Justified True Belief）

第三個提議：知識是真信念加上一個「說明」（*logos*）——即證成（justification）。蘇格拉底檢查了「說明」的三種含義：

1. **語言表達**：僅僅把信念說出來——太弱
2. **元素分析**：把事物分解為組成部分——但元素本身無法被說明
3. **區分標記**：指出使事物與眾不同的特徵——但這預設了我們已經知道這個事物

對話以**困惑（aporia）**結尾——柏拉圖最終沒有給出圓滿的答案，但奠定了 JTB 分析的基本框架。

## 二、JTB 的三個條件

### 條件一：信念（Belief）

$S$ knows that $p$ only if $S$ believes that $p$.

- 如果我不相信 $p$，我不可能「知道」$p$
- 信念是一種命題態度（propositional attitude）：對一個命題的肯定心理狀態
- 問題：信念是否是知識的必要條件？一些知識（如技能知識、know-how）不涉及命題信念

### 條件二：真（Truth）

$S$ knows that $p$ only if $p$ is true.

- 你不可能「知道」錯誤的事情——你可以**以為**你知道，但這不是知識
- 真理的對應論（correspondence theory）：$p$ 為真當且僅當 $p$ 對應於事實
- 替代理論：融貫論（coherence theory）、實用論（pragmatic theory）

### 條件三：證成（Justification）

$S$ knows that $p$ only if $S$ is justified in believing that $p$.

- 證成排除「運氣好猜對」——你必須有好的理由（good reasons）
- 證成有程度之分（degrees of justification）
- 內在論（internalism）：證成因素必須是主體可通達的（accessible）
- 外在論（externalism）：證成可以依賴主體無法意識到的因素（如可靠過程）

## 三、知識的類型學

| 類型 | 英文 | 說明 | 範例 |
|:-----|:-----|:-----|:-----|
| 命題知識 | Propositional knowledge / Know-that | 知道某事為真 | 我知道巴黎是法國首都 |
| 技能知識 | Procedural knowledge / Know-how | 知道如何做某事 | 我知道如何騎腳踏車 |
| 親知知識 | Knowledge by acquaintance | 直接熟悉某人或某物 | 我認識張三 |

認識論主要關注**命題知識**（know-that），但 know-how 與 know-by-acquaintance 是否可以還原為 know-that 是持續的爭論（Ryle vs 理智主義）。

## 四、知識的價值（The Value Problem）

柏拉圖在《美諾篇》（Meno）提出：為什麼知識比真信念更有價值？

- 真信念同樣可以指導正確行動（如果你真相信去 Larissa 的路是那條，你也能到達）
- 但知識像是被「拴住」的——你不會輕易改變它
- 蘇格拉底：知識是「經過證成的真信念」，證成（*logos*）將信念「拴住」，使其穩定

**當代討論**（Zagzebski, Kvanvig, Pritchard）：

- 葛梯爾問題表明：JTB+第四條件也可能只是「拴住」的巧合
- 是否有些價值是知識獨有的，真信念無法提供？——理解（understanding）或許是更好的候選

## 五、當代替代方案

| 理論 | 代表人物 | 核心主張 |
|:-----|:---------|:---------|
| 可靠論（Reliabilism） | Goldman (1967) | 知識 = 由可靠過程產生的真信念 |
| 德性認識論（Virtue Epistemology） | Sosa, Zagzebski | 知識 = 因認知德性而成功的真信念 |
| 知識優先論（Knowledge-First） | Williamson (2000) | 知識是基本的、不可分析的——用知識解釋信念和證成 |
| 語境論（Contextualism） | DeRose, Lewis | 「知道」的語義隨語境變化 |
