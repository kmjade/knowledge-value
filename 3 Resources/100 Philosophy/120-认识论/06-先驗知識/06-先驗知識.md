# 06 先驗知識 / A Priori Knowledge

> 先驗知識（a priori knowledge）是獨立於感官經驗即可被證成的知識。它與後驗知識（a posteriori knowledge）的區分是認識論最根本的分類之一——但這個區分本身比表面看起來複雜得多。

---

## 一、基本區分

### 先驗 vs 後驗（A Priori vs A Posteriori）

| 維度 | 先驗（A Priori） | 後驗（A Posteriori） |
|:-----|:-----------------|:---------------------|
| **證成來源** | 獨立於感官經驗 | 依賴感官經驗 |
| **典範** | 數學、邏輯 | 自然科學、日常觀察 |
| **確定性** | 必然真理（necessary） | 偶然真理（contingent） |
| **可否被經驗推翻** | 傳統上不可 | 可 |

### 分析 vs 綜合（Analytic vs Synthetic）

| 維度 | 分析命題（Analytic） | 綜合命題（Synthetic） |
|:-----|:---------------------|:----------------------|
| **真值來源** | 詞義與邏輯 | 詞義 + 世界的事實 |
| **謂詞** | 包含在主詞概念中 | 超出主詞概念 |
| **典範** | 「所有單身漢都是未婚的」 | 「這張桌子是棕色的」 |

---

## 二、康德的革命：綜合先驗判斷

### 《純粹理性批判》（*Critique of Pure Reason*, 1781/1787）

康德提出了一個革命性問題：**綜合先驗判斷如何可能？**（How are synthetic a priori judgments possible?）

在康德之前，主流觀點是：
- 理性主義者：先驗知識 = 分析命題
- 經驗主義者：綜合知識 = 後驗

康德論證存在**第三種**命題——綜合先驗命題：

|  | 分析 | 綜合 |
|:-|:----:|:----:|
| **先驗** | ✅ 所有單身漢都是未婚的 | ✅ 5+7=12、因果原則 |
| **後驗** | ❌ （概念上不可能） | ✅ 這張桌子是棕色的 |

### 為什麼 5+7=12 是綜合先驗？

康德論證：「12」這個概念並不包含在「5」、「7」、「+」這些概念的簡單組合中——你需要進行**直觀中的綜合**（synthesis in intuition）才能從 5+7 達到 12。因此它是綜合的（新增了信息），但又是先驗的（不依賴特定經驗）。

### 綜合先驗的典範

| 領域 | 綜合先驗命題 |
|:-----|:------------|
| 數學 | 7+5=12；兩點之間直線最短 |
| 自然科學 | 所有變化都有原因（因果律） |
| 形而上學 | 世界在時間上有開端 |

## 三、蒯因的批判：兩個教條

### W. V. O. Quine, "Two Dogmas of Empiricism" (1951)

蒯因攻擊了經驗主義的兩個基本教條：

### 教條一：分析/綜合的區分

蒯因論證：我們無法給出「分析性」的非循環定義：

- 「分析 = 根據意義為真」需要先定義「意義」
- 「分析 = 邏輯真理 + 同義詞替換」需要先定義「同義詞」
- 「同義詞」的定義本身依賴「分析」——循環！

**結論**：分析/綜合的界線是模糊的——只有程度差異，沒有種類差異。

### 教條二：還原論（Reductionism）

每個有意義的陳述都可以還原為關於直接經驗的陳述——蒯因拒絕這個教條。

### 知識的網絡隱喻（Web of Belief）

```
           ┌──────────────────────────────┐
           │        邏輯與數學              │  ← 網絡中心（最遠離經驗）
           │    ┌──────────────────┐       │
           │    │   自然定律         │       │
           │    │  ┌──────────┐    │       │
           │    │  │ 觀察陳述  │    │       │  ← 網絡邊緣（最近經驗）
           │    │  └──────────┘    │       │
           │    └──────────────────┘       │
           └──────────────────────────────┘
```

- 沒有命題是完全免疫於經驗修正的——**連邏輯和數學也可以被修正**
- 修正可以發生在網絡的任何一點——我們選擇修正何處取決於**實用考量（pragmatic considerations）**

## 四、先驗知識可修正嗎？

### 傳統觀點
先驗知識是必然的、不可被經驗推翻的。

### 蒯因的挑戰
所有知識都面對「經驗法庭」（tribunal of experience）——作為一個整體。即使是邏輯定律，在極端情況下也可以被修正以容納頑固的經驗。

### 當代辯論

| 立場 | 代表人物 | 觀點 |
|:-----|:---------|:-----|
| **強先驗論** | BonJour, Bealer | 先驗知識是獨立於經驗的理性洞察（rational insight） |
| **溫和先驗論** | Peacocke | 先驗證成可被經驗推翻，但不總是依賴經驗 |
| **徹底經驗論** | Quine, Devitt | 沒有實質性的先驗知識 |
| **語義先驗論** | Boghossian | 先驗性基於對概念的語義能力（semantic competence） |

---

## 五、Kripke 的挑戰：先驗偶然與後驗必然

### *Naming and Necessity* (1980)

Kripke 打破了「先驗 = 必然」的傳統等式：

|  | 必然 | 偶然 |
|:-|:----:|:----:|
| **先驗** | 數學真理 | 「標準米尺長一米」（先驗偶然！） |
| **後驗** | 「水是 H₂O」（後驗必然！） | 大多數日常事實 |

- **先驗偶然**：標準米尺的長度被定義為一米——先驗知道，但米尺可以有不同的長度（偶然）
- **後驗必然**：水 = H₂O 是形而上學必然，但我們通過經驗科學才知道
