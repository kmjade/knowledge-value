# 120 認識論 / Epistemology — 知識庫總覽

> 認識論（Epistemology）是哲學的核心分支，追問「什麼是知識？」、「我們如何知道？」、「知識的界限何在？」。本知識庫覆蓋從柏拉圖的 JTB 定義到當代社會認識論的完整光譜。
