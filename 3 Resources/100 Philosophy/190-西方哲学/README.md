# 190 西方哲學 · Western Philosophy

現代西方哲學從 17 世紀理性主義與經驗主義的對立開始，歷經康德的批判轉向、德國唯心論、尼采的生命哲學、現象學運動、存在主義、分析哲學，一直到當代政治哲學、後結構主義與思辨實在論。This knowledge base covers the entire sweep from Descartes to the present day.

1. **初學者**：從 01 理性主義 → 02 經驗主義 → 03 康德 依序閱讀，理解近代哲學的基本張力。
2. **中階讀者**：關注 04 德國唯心論 → 05 尼采 → 06 現象學 → 07 存在主義 這條歐陸脈絡。
3. **進階讀者**：對比 08 分析哲學與 09 當代哲學，理解英美與歐陸傳統的分合。
