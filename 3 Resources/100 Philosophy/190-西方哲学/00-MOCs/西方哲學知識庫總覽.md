## 核心辯論地圖 · Key Debates Map

### 軸線一：理性主義 vs. 經驗主義 (Rationalism vs. Empiricism)
```
Descartes ─┬─ Spinoza ── Leibniz    理性主義: 先天觀念，理性直觀
           │
Locke ─────┼─ Berkeley ── Hume       經驗主義: 一切來自經驗
           │
           └──→ Kant 綜合：先天綜合判斷
```

### 軸線二：實在論 vs. 觀念論 (Realism vs. Idealism)
```
樸素實在論 ──→ Locke 初性/次性 ──→ Berkeley 觀念論 ──→ Kant 先驗觀念論
                                                          │
                                Hegel 絕對觀念論 ←────────┘
```

### 軸線三：基礎主義 vs. 反基礎主義 (Foundationalism vs. Anti-foundationalism)
```
Descartes 不可懷疑的基礎 ──→ Husserl 先驗主體性 ──→ 早期分析哲學 邏輯原子主義
                                                          │
Nietzsche 視角主義 ────→ Heidegger 解釋學 ──→ Derrida 解構 ──→ Rorty 反基礎主義
```

### 軸線四：英美分析 vs. 歐陸傳統 (Analytic vs. Continental)
```
分析哲學：Frege → Russell → 早期 Wittgenstein → 維也納學圈 → Quine → Kripke
歐陸傳統：Hegel → Nietzsche → Husserl → Heidegger → Sartre → Foucault → Derrida
                      │                                    │
                      └──── 交集：後期 Wittgenstein ────────┘
                                   (語言遊戲、生活形式)
```
