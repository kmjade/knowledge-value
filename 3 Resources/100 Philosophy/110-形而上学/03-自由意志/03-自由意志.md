# 03 自由意志 (Free Will)

> *"Man can do what he wills but he cannot will what he wills."* — Schopenhauer

自由意志問題是形而上學中最具存在感的辯論。如果宇宙是決定的 (deterministic)，我們還能對自己的行為負責嗎？如果不自由，道德責任與刑罰的正當性何在？

---

## 一、問題框架 Framing the Problem

### 經典三難 (Classical Trilemma)
```
                Is determinism true?
                   /        \
                 YES         NO
                 /              \
        Do we have            Do we have
        free will?            free will?
        /      \              /      \
      YES      NO          YES       NO
       ↓        ↓           ↓         ↓
  Compatibilism  Hard    Libertarianism  Hard
  (soft         Determinism              Incompatibilism
   determinism)
```

### 核心術語
| 術語 | 定義 |
|------|------|
| **Determinism** | 過去狀態 + 自然律 → 唯一未來 |
| **Free Will** | 行動者有能力以其他方式行動 (ability to do otherwise) |
| **Compatibilism** | 自由意志與決定論相容 |
| **Incompatibilism** | 自由意志與決定論不相容 |
| **Libertarianism** | 決定論為假，人有自由意志 |
| **Hard Determinism** | 決定論為真，人無自由意志 |
| **Hard Incompatibilism** | 無論決定論是否為真，人都無自由意志 |

## 二、Compatibilism 相容論

### 2.1 Classical Compatibilism
- **Hobbes, Hume, Mill**：自由 = 不受外部強制地按自己的慾望行動
- "Freedom of spontaneity" vs "freedom of indifference"
- 自由不等於無原因；自由是從 coercion 中解放

### 2.2 對 Classical Compatibilism 的挑戰
**Consequence Argument (van Inwagen, 1983)**:
```
1. 沒有人能改變過去或自然律
2. 如果決定論為真，我們當前的行動是過去+自然律的必然結果
3. ∴ 沒有人能改變自己的行動
4. 自由意志要求能改變行動
5. ∴ 決定論與自由意志不相容
```

### 2.3 New Compatibilism (Mesh Theories)
- **Harry Frankfurt (1971)**：自由行動來自 higher-order volitions 與 first-order desires 的契合
- 吸毒者有 first-order desire 吸毒，但 higher-order volition 不想吸毒 → 不自由
- **Watson**：valuation vs motivation 的區分
- **Fischer & Ravizza**：reasons-responsiveness 理論

## 三、Frankfurt Cases 法蘭克福反例

### 挑戰 the Principle of Alternate Possibilities (PAP)

> **PAP**: 一個人對其行為負有道德責任，僅當他本可以採取其他行動

**Frankfurt 反例**：
```
Jones 決定殺 Smith。Black 是神經科學家，在 Jones 腦中植入晶片：
- 如果 Jones 決定不殺 Smith，晶片會強制他殺
- 但 Jones 自己決定殺 Smith，晶片未被觸發
→ Jones 負有責任，但他「不能以其他方式行動」（因為晶片會阻止）
→ ∴ PAP 為假
```

### 對此的爭議
- **Fischer**: Frankfurt cases 顯示責任不需要 alternate possibilities，但需要 guidance control
- **寬大論者 (Widerker, Ginet)**：反對 Frankfurt cases 的設計；存在「flicker of freedom」

---

## 四、Libertarianism 自由意志論

### 4.1 Agent Causation (Chisholm, O'Connor)
- 行動者 (agent) —— 而非事件 —— 是行動的 undetermined 原因
- 行動者作為「不移動的推動者」(unmoved mover)
- 挑戰：agent causation 是否連貫？是否僅推遲了問題？

### 4.2 Event-Causal Libertarianism (Kane)
- **Robert Kane**：自由行動發生在「自我形成行動」(self-forming actions, SFAs)
- SFAs 是 torn decision 狀態（同時有道德理由與自利理由）
- 量子不確定性 + 努力 = 最終選擇
- 但：如果最終結果是 indeterministic，行動者如何「控制」結果？(Luck objection)

---

## 五、Hard Incompatibilism 與道德責任

### 5.1 Pereboom's Hard Incompatibilism
- **Derk Pereboom**：無論決定論是否為真，人都沒有 free will in the sense required for moral responsibility
- 四案例論證 (Four-Case Argument)：manipulation cases 與 ordinary determinism 無本質區別
- 結果：我們應放棄 retributive punishment，轉向 quarantine model

### 5.2 Strawson's Reactive Attitudes
- **P.F. Strawson (1962)**："Freedom and Resentment"
- Reactive attitudes (resentment, gratitude, indignation) 是人際關係的 constitutive part
- 無論形而上學結論如何，我們無法真正放棄這些態度
- **Compatibilism from the participant stance**

### 5.3 Galen Strawson's Basic Argument
1. 你的一切行動來自你的性格與心理狀態
2. 但你的性格與心理狀態來自遺傳 + 環境（非你選擇）
3. 因此你對自己的行動不負終極責任 (ultimate responsibility)
4. 真正的道德責任要求終極責任
5. ∴ 沒有人真正負有道德責任
