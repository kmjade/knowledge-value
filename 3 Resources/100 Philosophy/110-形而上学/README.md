# 110 形而上學 (Metaphysics)

> 「形而上者謂之道。」——《易經·繫辭上》
> "That which is above form is called the Dao."

形而上學是哲學最古老的核心分支，探討 **存在本身 (being qua being)**、**實在的本質 (the nature of reality)**，以及超越經驗世界的基本結構。從 Aristotle 的「第一哲學」到當代 analytic metaphysics 的 grounding 與 fundamentality 辯論，形而上學持續追問：**What is there? What is it like?**

本知識庫涵蓋九大核心領域，每章提供關鍵論證、哲學家立場對照表，以及跨庫連接。
