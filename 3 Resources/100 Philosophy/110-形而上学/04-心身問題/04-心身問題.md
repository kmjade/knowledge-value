# 04 心身問題 (The Mind-Body Problem)

> *"What is it like to be a bat?"* — Thomas Nagel (1974)

心身問題追問：**意識與物理世界的關係是什麼？** 主觀經驗 (qualia) 如何從客觀的物理過程中浮現？這是形而上學中最持久的難題之一，橫跨哲學、神經科學、與物理學。

---

## 一、問題框架 The Central Problem

### 1.1 心靈的兩面
```
          心靈現象 Mental Phenomena
         /                          \
  意向性 (Intentionality)        意識 (Consciousness)
  "aboutness" — 思想關於某物      "what-it's-likeness" — 主觀經驗
  例：我相信巴黎在法國             例：紅色看起來的感覺
```

### 1.2 立場光譜 Spectrum of Positions

```
Physicalism ←───────────────────────────────→ Dualism
    │                                              │
Eliminativism  Behaviorism  Identity  Functionalism  Property Dualism  Substance Dualism
  (Churchlands)  (Ryle)    (Smart)    (Putnam)       (Chalmers)       (Descartes)
```

---

## 二、Substance Dualism 實體二元論

### 2.1 Descartes 的論證
1. I can clearly and distinctly conceive of myself existing without my body
2. Whatever is clearly and distinctly conceivable is possible (by God's power)
3. ∴ It is possible that I exist without my body
4. If A can exist without B, A ≠ B
5. ∴ I am not identical to my body

### 2.2 交互問題 The Interaction Problem
- 非物理的 mind 如何 causally interact with 物理的 body？
- **Princess Elisabeth of Bohemia (1643)** 的著名挑戰
- Descartes 的回應：pineal gland（松果體）但未解決問題
- **Occasionalism (Malebranche)**：God 在每個時刻中介
- **Parallelism (Leibniz)**：預定和諧

---

## 三、Physicalism 物理主義

### 3.1 行為主義 Behaviorism
- **Ryle (1949)**：*The Concept of Mind*
- Category mistake：將 mind 視為 "ghost in the machine"
- 心靈謂詞只是行為傾向的描述 (dispositions to behave)
- 問題：無法解釋 qualia 與內在經驗

### 3.2 Mind-Brain Identity Theory
- **Smart (1959)**, **Feigl**：mental states = brain states
- Type identity：pain = C-fiber firing
- **Multiple realizability 挑戰 (Putnam, 1967)**：
  - 章魚、外星人可能有 pain 但有不同的神經結構
  - ∴ pain ≠ C-fiber firing

### 3.3 Functionalism 功能主義
- **Putnam, Fodor**：mental states = functional states
- 心靈狀態由其因果角色定義 (causal role)，而非物理實現
- 心靈之於大腦如軟體之於硬體
- 優勢：允許多重實現性

## 四、The Hard Problem of Consciousness

### 4.1 Chalmers 的區分 (1995)
| 問題類型 | 描述 | 可解決性 |
|----------|------|----------|
| **Easy Problems** | 解釋認知功能：感知、記憶、學習、注意力 | 原則上可解決（神經科學） |
| **The Hard Problem** | 為什麼物理過程伴隨主觀經驗？為什麼不全是黑暗中進行？ | 極難，甚至原則上不可解 |

### 4.2 Jackson's Knowledge Argument: Mary 的房間
```
Mary 是位卓越的神經科學家，一生被關在黑白房間中，
從未見過顏色。她知道關於顏色感知的一切物理事實。
有一天，Mary 被釋放並第一次看到紅色。
Question: Mary 學到了新東西嗎？

If YES → 存在非物理的事實 → Physicalism is false.
```
- Jackson 後來轉向 physicalism 陣營，但 Knowledge Argument 持續引發辯論
- **Ability Hypothesis (Lewis/Nemirow)**：Mary 獲得的是能力 (knowing-how)，而非命題知識

### 4.3 Chalmers' Zombie Argument
- **Philosophical Zombie**：物理上與我們完全相同，但缺乏主觀經驗
- 殭屍是可設想的 (conceivable)
- 可設想的 → 可能的 (metaphysically possible)
- ∴ 意識不必然隨附於物理 → physicalism is false

### 4.4 Nagel's Bat (1974)
> *"What is it like to be a bat?"*
- 主觀經驗的觀點性 (point-of-view-ness) 是客觀科學無法捕捉的
- 即使知道蝙蝠的所有神經生理學，我們也不知道「身為蝙蝠是什麼感覺」
- 這指出了 physicalist reduction 的根本限制

---

## 五、當代理論發展

### 5.1 Panpsychism 泛心論
- **Goff, Strawson, Russell**：意識是物質的基本屬性
- Micropsychism：每個基本粒子都有 proto-consciousness
- 解決 hard problem 的激進方案：不「產生」意識，意識從一開始就在

### 5.2 Illusionism 幻覺論
- **Frankish, Dennett**：意識並不存在（作為非物理實體）
- Phenomenal consciousness is an illusion
- 我們有「似乎有 qualia」的信念，但信念本身是物理過程

### 5.3 Integrated Information Theory (IIT)
- **Tononi**：意識是 integrated information (Φ) 的量度
- 任何具有足夠 Φ 的系統都有意識
- 數學化、可量化的意識理論

| 理論 | 意識的地位 | 科學性 | 直覺符合度 |
|------|-----------|--------|-----------|
| Dualism | 基本非物理實體 | 低 | 高 |
| Physicalism | 物理的副產品 | 高 | 中 |
| Panpsychism | 基本物理屬性 | 中 | 低-中 |
| Illusionism | 不存在 | 高 | 低 |
| IIT | 可量化的物理屬性 | 高 | 中 |
