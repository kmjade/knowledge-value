### 核心立場矩陣 Position Matrix

| 問題域 | 實在論立場 | 反實在論/溫和立場 |
|--------|-----------|-------------------|
| Universals | Platonism, Aristotelian realism | Nominalism, Trope theory |
| Possible worlds | Modal realism (Lewis) | Actualism, Fictionalism |
| Time | Eternalism (B-theory) | Presentism (A-theory) |
| Mind | Dualism | Physicalism, Eliminativism |
| Free will | Libertarianism | Compatibilism, Hard determinism |
| Scientific entities | Scientific realism | Constructive empiricism |
