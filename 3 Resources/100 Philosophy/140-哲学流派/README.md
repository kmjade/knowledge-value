# 140 哲學流派 (Philosophical Schools)

Brief overview of the DDC 140 section: a systematic survey of major Western philosophical schools from Ancient Greece to Postmodernism, organized chronologically — covering their core doctrines, key figures, historical context, and enduring debates.

This knowledge base provides nine chapter-length deep dives, comparison matrices, philosopher timelines, and a guided four-week learning path.
