# 08 分析哲學 (Analytic Philosophy)

> Everything that can be said at all can be said clearly. — Ludwig Wittgenstein

---

## 什麼是分析哲學？

分析哲學不是一套學說，而是一種**風格**：通過語言分析和邏輯澄清來處理哲學問題。它起源於 20 世紀初對黑格爾式唯心論的反叛。

| 特徵 | 說明 |
|:-----|:-----|
| **語言轉向** Linguistic Turn | 哲學問題首先是語言問題 |
| **邏輯方法** | 用形式邏輯作為哲學工具 |
| **論證清晰** | 論證必須明確、步驟化、可檢驗 |
| **科學友好** | 與自然科學保持連續性 |

## 發展階段

### 第一階段：邏輯原子論 (1900–1930)

| 人物 | 貢獻 |
|:-----|:-----|
| **Gottlob Frege** (1848–1925) | 現代邏輯之父 — 量詞、命題函數、Sinn/Bedeutung 區分 |
| **Bertrand Russell** (1872–1970) | 確定描述詞理論 (On Denoting, 1905)、邏輯原子論、類型論 |
| **G.E. Moore** (1873–1958) | 常識哲學、自然主義謬誤、開放問題論證 |
| **早期 Wittgenstein** | 《邏輯哲學論》(Tractatus, 1921)：語言圖像論、可說/不可說 |

### 第二階段：邏輯實證主義 (1920s–1950s)

**維也納學圈** (Vienna Circle)：Schlick, Carnap, Neurath, Hahn

| 核心信條 | 說明 |
|:---------|:-----|
| **證實原則** Verification Principle | 一個命題的意義就是它的證實方法 |
| **拒斥形而上學** | 形而上學陳述沒有認知意義，只有情感表達 |
| **統一科學** | 所有科學可以化約為物理語言 |

### 第三階段：日常語言哲學 (1940s–1960s)

| 人物 | 貢獻 |
|:-----|:-----|
| **晚期 Wittgenstein** | 《哲學研究》(1953)：意義即使用·語言遊戲·家族相似性 |
| **J.L. Austin** (1911–1960) | 言語行為理論·以言行事 (How to Do Things with Words) |
| **Gilbert Ryle** (1900–1976) | 範疇錯誤·心靈概念 (The Concept of Mind) |

### 第四階段：後實證主義 (1960s–至今)

| 人物 | 貢獻 |
|:-----|:-----|
| **W.V.O. Quine** (1908–2000) | 〈經驗論的兩個教條〉— 拒斥分析/綜合區分、翻譯不確定性 |
| **Wilfrid Sellars** (1912–1989) | 「所與神話」·科學實在論 |
| **Donald Davidson** (1917–2003) | 徹底詮釋·異常一元論 |
| **Saul Kripke** (1940–2022) | 《命名與必然性》— 因果指稱理論·先驗偶然/後驗必然 |

### 第五階段：當代分析哲學

| 領域 | 代表人物 |
|:-----|:---------|
| 心靈哲學 | David Chalmers, Daniel Dennett, Thomas Nagel |
| 語言哲學 | Robert Brandom, John McDowell |
| 形而上學 | David Lewis, Kit Fine, Ted Sider |
| 認識論 | Timothy Williamson, Jennifer Lackey |
| 倫理學 | Derek Parfit, Peter Singer, Christine Korsgaard |

## 分析哲學 vs 歐陸哲學

| | 分析哲學 | 歐陸哲學 |
|:--|:--------|:--------|
| 方法 | 邏輯分析·論證 | 歷史詮釋·敘事 |
| 典範 | 自然科學 | 文學·歷史·社會理論 |
| 風格 | 論文 (paper) | 著作 (book) |
| 代表人物 | Russell, Quine, Kripke | Heidegger, Sartre, Derrida |

> 這個二分法本身有爭議 — 越來越多學者拒絕非此即彼的標籤。
