# 02 中世紀哲學 (Medieval Philosophy)

> *"Credo ut intelligam."* — Anselm of Canterbury
> 「我相信，以便理解。」

中世紀哲學 (約 5th–15th C) 的核心張力是 **信仰與理性 (fides et ratio)** 的關係。這一時期的哲學家——無論是基督教、伊斯蘭教還是猶太教——都試圖在希臘哲學遺產與啟示宗教之間建立橋樑。

---

## 一、教父哲學 Patristic Philosophy (約 2nd–7th C)

教父哲學是基督教神學的形成期，核心任務是闡明正統教義並回應異端挑戰。

### 1.1 Augustine of Hippo (354–430)

Augustine 是教父時期最偉大的哲學家，融合了新柏拉圖主義與基督教。

| 領域 | 核心貢獻 |
|------|----------|
| **認識論** | 信仰尋求理解 (fides quaerens intellectum)；反對懷疑論：Si fallor, sum (即使我犯錯，我存在) |
| **神學** | 原罪 (original sin)；恩典論；預定論 — 對抗 Pelagianism |
| **時間哲學** | 時間是心靈的 distentio animi；God 在時間之外 (永恆的現在) |
| **政治哲學** | 《上帝之城》(De Civitate Dei)：地上之城 vs 上帝之城 |
| **惡的問題** | 惡是善的缺乏 (privatio boni)；自由意志辯護 (free will defense) |
| **影響** | 塑造西方基督教思想直至 Aquinas，深刻影響 Descartes ("cogito" 的先驅) |

### 1.2 Boethius (c.480–524)

「最後的羅馬人，第一位經院哲學家」：

- 《哲學的慰藉》(*Consolation of Philosophy*)
- 共相問題 (Problem of Universals) 的經典表述
- 永恆與時間的區分：God's knowledge as timeless vs foreknowledge
- 在獄中寫成，被後世無數人閱讀

---

## 二、經院哲學 Scholasticism (約 11th–14th C)

經院哲學是中世紀哲學的黃金時期，以大學為中心，運用 dialectic 方法系統化基督教神學。

### 2.1 Anselm of Canterbury (1033–1109)

| 貢獻 | 內容 |
|------|------|
| **Ontological Argument** | *Proslogion*：God = "that than which nothing greater can be conceived" |
| **邏輯結構** | 從 God 的概念推導 God 的存在（a priori 論證） |
| **反駁** | Gaunilo 的「完美島嶼」反駁 → Anselm 的回應 |
| **影響** | Descartes、Leibniz、Gödel 均復興了本體論論證的變體 |

### 2.2 Thomas Aquinas (1225–1274)

Aquinas 是中世紀哲學的巔峰，將 Aristotle 系統性地整合進基督教神學框架。

| 領域 | 核心貢獻 | 關鍵著作 |
|------|----------|----------|
| **上帝證明** | Five Ways (五路論證)：motion, efficient cause, contingency, gradation, design | *Summa Theologica* I, Q2 |
| **存在論** | Essence (essentia) vs. Existence (esse)：在受造物中二者分離，唯獨在 God 中合一 | *De Ente et Essentia* |
| **自然法** | 四種法：Eternal Law → Natural Law → Human Law → Divine Law | *Summa Theologica* I-II |
| **倫理學** | 德性倫理學 + 自然法 + 終極目的 (beatitudo) | |
| **認識論** | 適度實在論 (moderate realism)：共相存在於事物中 (in re)、心靈中 (in mente)、God 中 (ante rem) | |

### Five Ways 詳解

| # | 論證 | 出發點 | 推理 | 結論 |
|---|------|--------|------|------|
| 1 | Motion | 事物在運動 | 運動需要推動者；無窮後退不可能 | First Mover = God |
| 2 | Efficient Cause | 因果鏈 | 無第一因則無後續因果 | First Cause = God |
| 3 | Contingency | 偶然存在物 | 並非一切都是偶然的 | Necessary Being = God |
| 4 | Gradation | 善/真/存在的等級 | 等級預設最高標準 | Maximum = God |
| 5 | Design | 自然的目的性 | 目的預設設計者 | Designer = God |

### 2.3 晚期經院哲學 Late Scholasticism

| 哲學家 | 貢獻與立場 |
|--------|-----------|
| **Duns Scotus** (c.1266–1308) | 意志優先於理智 (voluntarism)；haecceitas (此性)；univocity of being |
| **William of Ockham** (c.1287–1347) | Ockham's Razor (奧卡姆剃刀)；唯名論 (nominalism)：只有個別物存在；信仰與理性分離 |

Ockham 的唯名論切斷了信仰與理性的連接，為近代哲學的興起鋪平了道路。

## 三、伊斯蘭哲學 Islamic Philosophy

伊斯蘭哲學在中世紀扮演了關鍵角色：保存並發展了希臘哲學遺產，而後將其傳回西方。

### 3.1 Avicenna (Ibn Sina, 980–1037)

| 貢獻 | 內容 |
|------|------|
| **Flying Man 論證** | 懸浮在空中的人仍然意識到自己的存在 → 靈魂獨立於身體（自我意識的論證） |
| **Essence/Existence 區分** | 比 Aquinas 更早提出：essence 不蘊含 existence（偶然性論證） |
| **必要存在者** | Necessary Existent (واجب الوجود) 的證明 |
| **影響** | 深刻影響 Aquinas、Scotus 與後續西方哲學 |

### 3.2 Averroes (Ibn Rushd, 1126–1198)

| 貢獻 | 內容 |
|------|------|
| **The Commentator** | Aristotle 最偉大的注釋者；通過注釋將 Aristotle 傳回西方 |
| **雙重真理論** | 哲學真理與宗教真理可以共存但不必然重合 |
| **理智統一論** | 所有人共享一個 agent intellect (備受爭議) |
| **影響** | 拉丁 Averroism；巴黎大學的激烈辯論 |

### 3.3 Al-Ghazali (1058–1111)

- 《哲學家的不一致》(*Tahafut al-Falasifa*)：批判 Avicenna 等哲學家的 20 個命題
- 將 Sufi mysticism 與正統伊斯蘭結合
- Averroes 後來以《不一致的不一致》回應

## 四、猶太哲學 Jewish Philosophy

### Maimonides (Moses ben Maimon, 1138–1204)

| 貢獻 | 內容 |
|------|------|
| **著作** | 《迷途指津》(*Guide for the Perplexed*) |
| **核心問題** | 如何調和 Torah 與 Aristotelian 哲學？ |
| **否定神學** | God 只能通過否定來描述 (via negativa) — 我們只能說 God 不是什麼 |
| **影響** | 深刻影響 Aquinas、Spinoza 與後續猶太思想 |
