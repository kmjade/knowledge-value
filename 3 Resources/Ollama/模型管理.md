# 管理

# 查看
```bash
ollama list
```
会列出本地已下載的模型名称、大小以及最近使用時間。

## 下載新模型
```bash
ollama pull llama2
ollama pull mistral
```
- 根据需要選擇合适的模型，模型檔案会存放在 `~/.ollama/models/` 目錄下。

## 刪除模型
```bash
ollama rm llama2
```
- 刪除不再需要的模型以節省磁碟空間。

## 切換模型（使用示例）
在調用 API 时指定模型名称，例如：
```bash
curl -X POST http://localhost:11434/api/generate \
  -d '{"model":"mistral","prompt":"Explain quantum computing in 2 sentences."}'
```

## 常用模型推荐
| 模型 | 规模 | 适用場景 |
|------|------|----------|
| llama2 | 7B‑13B | 通用对话、代碼补全 |
| mistral | 7B | 高效推理、資源受限 |
| phi | 2.7B | 轻量嵌入式设备 |

> 📌 **提示**：在資源受限的机器上，优先使用 2‑7 B 规模的模型，以獲得更快響應。
