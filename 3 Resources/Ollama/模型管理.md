# 模型管理

## 查看已下载模型
```bash
ollama list
```
会列出本地已下载的模型名称、大小以及最近使用时间。

## 下载新模型
```bash
ollama pull llama2
ollama pull mistral
```
- 根据需要选择合适的模型，模型文件会存放在 `~/.ollama/models/` 目录下。

## 删除模型
```bash
ollama rm llama2
```
- 删除不再需要的模型以节省磁盘空间。

## 切换模型（使用示例）
在调用 API 时指定模型名称，例如：
```bash
curl -X POST http://localhost:11434/api/generate \
  -d '{"model":"mistral","prompt":"Explain quantum computing in 2 sentences."}'
```

## 常用模型推荐
| 模型 | 规模 | 适用场景 |
|------|------|----------|
| llama2 | 7B‑13B | 通用对话、代码补全 |
| mistral | 7B | 高效推理、资源受限 |
| phi | 2.7B | 轻量嵌入式设备 |

> 📌 **提示**：在资源受限的机器上，优先使用 2‑7 B 规模的模型，以获得更快响应。
