## 2. 模型下載很慢或中断？
- **網路代理**：檢查是否有全局代理，必要时設置 `HTTPS_PROXY` 環境变量。
- **镜像源**：官方提供 `OLLAMA_MIRROR` 環境变量，可指向国内镜像（如 `https://ollama-mirror.com`）。
- **磁碟空間**：确保有足夠的可用空間（模型大小通常在数 GB）。

## 3. 調用 API 返回错误 500？
- **模型名称错误**：確認请求体中的 `model` 与已下載模型列表一致（`ollama list`）。
- **JSON 格式**：使用有效的 JSON 且字段名正确（`model`, `prompt`, `stream` 等）。

## 4. 如何在 Obsidian 中切換本地模型？
1. 打開 **AI Assistant** 外掛設置。
2. 在 **模型列表** 中選擇已下載的模型（如 `llama2`）。
3. 儲存設置并重启外掛。

## 5. 本地模型卡顿，響應慢？
# 管理
- **GPU 加速**：若 GPU 支持，安裝对应的驅動程式并在 `config.yaml` 中启用 `gpu: true`。
- **模型大小**：使用更小的模型（2‑7 B）或开启 **量化**（`quantize: true`）以降低記憶體占用。
