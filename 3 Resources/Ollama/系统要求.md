# 系統要求

## 硬體
- CPU: 4 核心以上，推荐 8 核心
- 記憶體: 最低 8 GB，推荐 16 GB+
- 儲存: 至少 50 GB 可用空間（模型檔案）
- GPU: 可选，推荐 RTX 3060+ 用于加速推理

## 軟體
- 操作系統: Windows 10/11、macOS 12+、Linux 主流发行版
- Python: 3.8+（用于 Ollama Python 客户端）
# 部署
- Git: 用于获取模型仓库（可选）

## 網路
- 带宽: ≥10 Mbps（用于下載模型）
- 延遲: ≤100 ms（实时交互）
