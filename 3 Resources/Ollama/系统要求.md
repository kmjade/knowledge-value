# 系统要求

## 硬件
- CPU: 4 核心以上，推荐 8 核心
- 内存: 最低 8 GB，推荐 16 GB+
- 存储: 至少 50 GB 可用空间（模型文件）
- GPU: 可选，推荐 RTX 3060+ 用于加速推理

## 软件
- 操作系统: Windows 10/11、macOS 12+、Linux 主流发行版
- Python: 3.8+（用于 Ollama Python 客户端）
- Docker: 20.10+（可选，容器化部署）
- Git: 用于获取模型仓库（可选）

## 网络
- 带宽: ≥10 Mbps（用于下载模型）
- 延迟: ≤100 ms（实时交互）

> 📌 **提示**: 部署前确保系统满足以上最低要求，以免出现性能瓶颈。
