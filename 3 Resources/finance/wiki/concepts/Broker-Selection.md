# Broker Selection (经纪商选择)

## 定义

经纪商选择是交易中最重要的基础设施决策。好的经纪商提供安全资金保障、合理交易成本和可靠平台服务，差的经纪商可能导致资金损失。

## 核心原理

### 选择金字塔

```
         ┌──────────────┐
         │  交易体验     │ ← 平台、客服、出入金
         │              │
       ┌─┴──────────────┴─┐
       │   交易成本        │ ← 点差、佣金、隔夜利息
       │                  │
     ┌─┴──────────────────┴─┐
     │   监管合规（基石）    │ ← 资金安全、法律保障
     └──────────────────────┘
```

## 关键要点

### 全球主要监管机构

| 监管 | 地区 | 可靠性 | 杠杆限制 |
|------|------|--------|----------|
| **FCA** | 英国 | ⭐⭐⭐⭐⭐ | 30:1 |
| **ASIC** | 澳大利亚 | ⭐⭐⭐⭐⭐ | 30:1 |
| **NFA/CFTC** | 美国 | ⭐⭐⭐⭐⭐ | 50:1 |
| **FSA** | 日本 | ⭐⭐⭐⭐⭐ | 25:1 |
| **CySEC** | 塞浦路斯 | ⭐⭐⭐⭐ | 30:1 |
| **FSCA** | 南非 | ⭐⭐⭐ | 灵活 |
| 无监管 | 离岸 | ⭐ | 不推荐 |

### 选择清单

```
✅ 监管合规（最重要！）
✅ 出入金是否顺畅？
✅ 点差/佣金是否透明？
✅ 是否支持 MT4/MT5？
✅ 客服响应速度？
✅ 成立时间 > 5 年？
✅ Myfxbook / ForexPeaceArmy 口碑？

⚠️ 避开：
  ❌ 无监管 / 离岸监管
  ❌ 出金需要额外"手续费"
  ❌ 承诺保本保收益
  ❌ 无法找到实体地址
```

### 经纪商类型

| 类型 | 模式 | 适合 |
|------|------|------|
| **ECN/STP** | 直通市场，赚佣金 | 追求透明、低点差 |
| **做市商 (MM)** | 内部对冲，赚点差 | 小额交易者 |
| **混合型** | ECN + 做市 | 大多数经纪商 |
