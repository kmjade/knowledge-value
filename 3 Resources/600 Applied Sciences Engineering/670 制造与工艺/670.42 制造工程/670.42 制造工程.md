# 08-制造工程

> **DDC 670.42** · 制造工程 — **UDC 658.5** Production management. Process management
> **CLC TH16** 机械制造工艺 — **LCC TS176** Production planning. Production control
>
> 把设计变成产品——制造工程是工程学中最"接地气"的分支。没有制造，所有的设计都只存在于图纸上。

## 制造的演进

| 时代                 | 特征             | 代表            |
| ------------------ | -------------- | ------------- |
| **工业 1.0** (1780s) | 蒸汽机带动的机械化      | 纺织厂           |
| **工业 2.0** (1870s) | 电力驱动的流水线       | Ford T 型车生产线  |
| **工业 3.0** (1970s) | PLC 和计算机控制的自动化 | CNC 机床、工业机器人  |
| **工业 4.0** (2010s) | 物联网+AI+数字孪生    | 智能工厂、预测性维护    |
| **工业 5.0** (概念)    | 人机协作+可持续+韧性    | 协作机器人 (Cobot) |

## 核心制造工艺

### 成型工艺

| 工艺 | 原理 | 特点 |
|------|------|------|
| **铸造** | 熔融金属浇入模具 | 复杂形状→一次成型 |
| **锻造** | 压力使金属变形 | 强度高，晶粒流线优化 |
| **冲压** | 模具冲切/成型板材 | 极快，适合大批量 |
| **注塑** | 熔融塑料注入模具 | 塑料件大批量生产主力 |

### 减材制造

| 工艺 | 原理 | 精度 |
|------|------|------|
| 车削 | 工件旋转，刀具进给 | ~0.01mm |
| 铣削 | 刀具旋转，工件进给 | ~0.01mm |
| 磨削 | 砂轮高速旋转 | ~0.001mm |
| EDM (电火花) | 放电蚀除 | ~0.001mm |

### 增材制造 (3D 打印)

| 技术 | 材料 | 典型应用 |
|------|------|----------|
| FDM | 塑料丝 | 原型、业余 |
| SLA/DLP | 光敏树脂 | 牙科、珠宝 |
| SLS | 尼龙粉末 | 功能件 |
| SLM/DMLS | 金属粉末 | 航空、医疗 |

## 质量控制

### 六西格玛 (Six Sigma)

目标：每百万机会中缺陷数 (DPMO) < 3.4。

| σ 水平 | DPMO | 良率 |
|--------|------|------|
| 1σ | 690,000 | 31% |
| 2σ | 308,000 | 69% |
| 3σ | 66,800 | 93.3% |
| 4σ | 6,210 | 99.38% |
| 5σ | 233 | 99.977% |
| **6σ** | **3.4** | **99.99966%** |

### SPC (统计过程控制)

| 工具 | 用途 |
|------|------|
| 控制图 | 监控过程是否在统计控制范围内 |
| 过程能力指数 (Cpk) | 衡量过程满足规格限的能力 |
| 鱼骨图 | 分析质量问题的根本原因 |
| PDCA | Plan-Do-Check-Act 持续改进循环 |

## 精益制造

| 概念 | 含义 |
|------|------|
| **消除浪费 (Muda)** | 七种浪费：过量生产、等待、运输、过度加工、库存、动作、缺陷 |
| **JIT (准时制)** | 在需要的时候生产需要的量 |
| **看板 (Kanban)** | 可视化拉动系统 |
| **持续改善 (Kaizen)** | 小步快跑的日常改进 |

> Toyota 生产系统是精益制造的经典——不追求最快，追求消除一切不创造价值的活动。
