# 06-应用场景

> Cobot 已经渗透到制造业的几乎所有环节——从机床上下料到实验室移液。以下是六大经典场景。

## 1. 机床上下料

| 要素 | 说明 |
|------|------|
| 任务 | CNC/注塑机→Cobot 取成品+放毛坯 |
| 夹爪 | 双工位夹爪——一次取放 |
| 节拍 | Cobot 速度通常不是瓶颈——机床加工时间更长 |
| ROI | 一台 Cobot 替代一个操作员→回收期 12-18 月 |

## 2. 装配

| 要素 | 说明 |
|------|------|
| 任务 | 人做柔性操作（理线、卡扣），Cobot 做精确操作（拧螺丝、压入） |
| 技术 | 力控——感知接触力→柔顺插入 |
| 优势 | 人不会被重复劳动伤害，Cobot 不会疲劳 |

## 3. 质检

| 要素 | 说明 |
|------|------|
| 任务 | Cobot 带相机检测工件→OK/NG 判断 |
| 技术 | 机器视觉 + Cobot 精确定位 |
| 优势 | 不疲劳→一致性高→可 24h |

## 4. 包装码垛

| 要素 | 说明 |
|------|------|
| 任务 | Cobot 将产品放入包装箱→码垛到托盘 |
| 负载 | 需要大负载 Cobot (10-30kg) |
| ROI | 替代重体力劳动→减少工伤+提高效率 |

## 5. 焊接

| 要素 | 说明 |
|------|------|
| 任务 | Cobot 持焊枪焊接小批量多品种工件 |
| 优势 | 拖动示教快速编程→适合小批量 |
| 局限 | 速度不如传统焊接机器人 |

## 6. 实验室自动化

| 要素 | 说明 |
|------|------|
| 任务 | 移液、样本分装、试剂添加、试管搬运 |
| 优势 | 精度高、不疲劳、可追溯 |
