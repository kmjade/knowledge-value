# 01-物联网概述

> 物联网 (Internet of Things) 是通过嵌入式传感器、执行器和通信模块将物理对象连接到互联网，实现数据采集、远程监控和智能控制的网络。

## 核心定义

| 要素 | 含义 |
|------|------|
| **感知** | 传感器将物理量转为数字信号 |
| **连接** | 通过有线/无线网络传输数据 |
| **计算** | 在边缘或云端处理和分析数据 |
| **行动** | 基于分析结果的自动化响应 |

## IoT 四层架构

```
感知层: 传感器、执行器、RFID、摄像头
   ↓ (有线/无线)
网络层: WiFi/BLE/Zigbee/LoRa/5G/以太网
   ↓ (MQTT/HTTP/CoAP)
平台层: 设备管理、数据存储、规则引擎、分析
   ↓ (API/Dashboard)
应用层: 智慧工厂、智慧城市、智慧农业、车联网
```

## 市场规模

| 指标 | 2024 数据 |
|------|----------|
| 全球 IoT 连接数 | ~180 亿 |
| 预计 2030 | ~400 亿 |
| 最大垂直领域 | 制造业 (IIoT) |
| 增长最快 | 车联网、医疗 IoT |

## 消费 IoT vs 工业 IoT

| 维度 | 消费 IoT | 工业 IoT (IIoT) |
|------|----------|-----------------|
| 设备数 | 10-100/户 | 1000-100000/厂 |
| 可靠性 | 可接受偶尔断连 | 必须 99.9%+ |
| 安全 | 数据隐私 | 人身安全 + 生产安全 |
| 环境 | 舒适的室内 | -40~85°C、粉尘、震动 |
| 生命周期 | 2-5 年 | 10-20 年 |
| 协议 | WiFi、BLE | Modbus、Profinet、OPC UA |
| 例子 | 智能音箱、手环 | 产线监控、预测维护 |
