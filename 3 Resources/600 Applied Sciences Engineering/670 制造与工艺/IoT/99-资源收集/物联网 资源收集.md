# 物联网 资源收集

## 📡 芯片与模组

| 类别 | 代表 | 说明 |
|------|------|------|
| **MCU** | ESP32, STM32, nRF52 | IoT 终端核心 |
| **蜂窝模组** | Quectel, SIMCom | 4G/5G/NB-IoT |
| **LoRa 芯片** | Semtech SX1276 | LoRa 核心 |

## ☁️ IoT 平台

| 平台 | 特点 |
|------|------|
| **AWS IoT Core** | 全球最大——设备影子、规则引擎 |
| **Azure IoT Hub** | 企业级，DPS 自动注册 |
| **阿里云 IoT** | 中国最大 |
| **EMQX** | 开源 MQTT Broker——百万连接 |
| **ThingsBoard** | 开源 IoT 平台 |
