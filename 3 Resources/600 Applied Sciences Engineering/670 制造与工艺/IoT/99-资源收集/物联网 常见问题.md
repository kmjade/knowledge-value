# 物联网 常见问题

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| IoT = 智能家居 | 工业 IoT (IIoT) 的市场规模远大于消费 IoT——工厂、电网、油田才是主力 |
| 5G 是 IoT 的必须条件 | 大多数 IoT 设备用 WiFi/LoRa/NB-IoT 就够了。5G 只在高带宽低延迟场景（自动驾驶、工业控制）有优势 |
| 万物皆可 IoT | 加传感器的成本 + 通信 + 平台 > 产生的价值 → 不值得。ROI 要算清楚 |
| IoT 安全不重要 | 僵尸网络 Mirai (2016) 用百万 IoT 设备 DDoS 攻击 DNS——IoT 不安全会危害整个互联网 |
| 边缘计算 = 把云搬到本地 | 边缘不是"小云"，而是"在数据产生的地方做处理"——很多场景不需要云 |
| MQTT 已经过时 | MQTT 仍是 IIoT 最主流的协议——轻量、可靠、生态完整 |
| 买了平台就能做 IoT | 平台只是工具。真正的难点在硬件部署、网络覆盖、数据质量和运维 |
