# 03 — 住宅與居家管理

> 家是生活的中心——選擇、規劃和維護一個舒適、安全、高效的居住空間。

## 住宅選擇考量

| 因素 | 考量 |
|------|------|
| **區位** | 交通、學區、生活機能 |
| **類型** | 公寓、透天、電梯大樓 |
| **面積** | 家庭人數 × 空間需求 |
| **預算** | 房貸不超過月收入 30% |
| **安全** | 結構安全、消防、治安 |

## 空間規劃原則

| 區域 | 規劃重點 |
|------|---------|
| 公共區 | 客廳、餐廳：動線流暢、採光通風 |
| 私密區 | 臥室：安靜、隱私、收納 |
| 服務區 | 廚房、衛浴：防水、通風、安全 |
| 過渡區 | 玄關、走廊：引導與緩衝 |

## 居家維護管理

| 類別 | 項目 |
|------|------|
| **日常** | 清潔、通風、垃圾分類 |
| **週期** | 濾網更換、水垢清理、除蟲 |
| **季節** | 空調保養、排水檢查、防颱 |
| **年度** | 油漆、防水、管線檢查 |

## 智慧家庭

| 設備 | 功能 |
|------|------|
| 智慧燈光 | 遠端控制、情境設定 |
| 智慧家電 | 節能、排程運作 |
| 安全監控 | 攝影機、感測器、警報 |
| 語音助理 | 統一家電控制 |

> 💡 好的居家管理不是「大掃除」，而是「順手維護」——每天 5 分鐘勝過每月 5 小時。
