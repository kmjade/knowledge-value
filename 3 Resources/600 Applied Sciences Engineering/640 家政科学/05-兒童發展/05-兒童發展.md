# 05 — 兒童發展

> 兒童的成長是身、心、社、靈的全面發展——了解每個階段的特徵，給予適當的引導與支持。

## 發展領域

| 領域 | 內容 |
|------|------|
| **生理發展** | 身高體重、大小肌肉、感官 |
| **認知發展** | 語言、記憶、思考、問題解決 |
| **社會發展** | 人際互動、合作、同理心 |
| **情緒發展** | 自我認知、情緒調節、安全感 |

## Piaget 認知發展階段

| 階段 | 年齡 | 特徵 |
|------|------|------|
| **感覺動作期** | 0–2 歲 | 透過感官和動作認識世界、物體恆存 |
| **前運思期** | 2–7 歲 | 象徵思考、自我中心、萬物有靈 |
| **具體運思期** | 7–11 歲 | 邏輯思考、守恆概念、分類能力 |
| **形式運思期** | 11 歲以上 | 抽象推理、假設思考 |

## 教養風格

| 風格 | 關愛 | 要求 | 結果 |
|------|------|------|------|
| **權威型** | 高 | 高 | 自信、自律、社交能力強 |
| **獨裁型** | 低 | 高 | 順從但低自尊、焦慮 |
| **放任型** | 高 | 低 | 自我中心、缺乏自律 |
| **忽視型** | 低 | 低 | 各項發展均受負面影響 |

## 各階段發展重點

| 年齡 | 關鍵任務 |
|------|---------|
| 0–1 歲 | 建立安全依附 |
| 1–3 歲 | 自主性 vs 羞恥懷疑 |
| 3–6 歲 | 主動探索、語言爆發 |
| 6–12 歲 | 勤勉感、同儕關係 |

> 💡 最好的教養不是「教」，而是「陪伴」——孩子在安全感中自然成長。
