# 01 — 食品與營養

> 營養是健康的基礎——了解吃進身體的每一口食物，就是了解自己的健康。

## 六大營養素

| 營養素 | 功能 | 來源 |
|--------|------|------|
| **碳水化合物** | 主要能量來源 (4 kcal/g) | 米飯、麵包、馬鈴薯 |
| **蛋白質** | 組織修復、酶與激素 (4 kcal/g) | 肉、魚、豆、蛋、奶 |
| **脂肪** | 能量儲存、細胞膜 (9 kcal/g) | 油、堅果、酪梨 |
| **維生素** | 代謝調節 | 蔬果、全穀 |
| **礦物質** | 骨骼、神經、酵素 | 奶、鹽、紅肉 |
| **水** | 體溫調節、運輸、代謝 | 飲水、湯、蔬果 |

## 均衡飲食指南

| 食物類別 | 每日建議份量 |
|---------|------------|
| 全穀雜糧 | 1.5–4 碗 |
| 蔬菜 | 3–5 份 |
| 水果 | 2–4 份 |
| 豆魚蛋肉 | 3–8 份 |
| 乳製品 | 1.5–2 杯 |
| 油脂與堅果 | 3–7 茶匙 + 1 份堅果 |

## 食品安全

| 原則 | 說明 |
|------|------|
| **清潔** | 洗手、清潔器具與食材 |
| **分開** | 生熟食分開處理 |
| **煮熟** | 食物中心溫度達 70°C 以上 |
| **冷藏** | 5°C 以下保存，2 小時內冷藏 |
| **水源** | 使用安全飲用水 |

> 💡 均衡飲食的核心不是「少吃什麼」，而是「多吃什麼」——先確保營養充足，再考慮限制。
