# 06 — 消費者教育

> 每個人都消費，但不是每個人都懂消費——消費者教育教你聰明選擇、保護權益。

## 消費者八大權利

| 權利 | 說明 |
|------|------|
| **基本需求** | 獲得基本商品與服務 |
| **安全** | 免於有害產品的危害 |
| **資訊** | 獲得充分正確的消費資訊 |
| **選擇** | 多樣化產品的選擇自由 |
| **表達** | 消費者意見被聽見 |
| **救濟** | 損害賠償的權利 |
| **消費者教育** | 獲得消費知識的權利 |
| **健康環境** | 在健康環境中生活 |

## 理性消費決策

| 步驟 | 說明 |
|------|------|
| 確認需求 | 我真的需要這個嗎？ |
| 蒐集資訊 | 比較產品、價格、評價 |
| 評估選項 | CP 值、品質、售後服務 |
| 購買決策 | 選擇最佳方案 |
| 購後評估 | 滿意嗎？下次如何改進？ |

## 常見消費陷阱

| 陷阱 | 對策 |
|------|------|
| 限時優惠 | 先冷靜 24 小時 |
| 錨定效應 | 比較原價與實際價值 |
| 從眾心理 | 問自己「我真的想要嗎？」 |
| 沉沒成本 | 不因已花的錢繼續花錢 |

## 消費者保護

| 法規 | 保護內容 |
|------|---------|
| 消費者保護法 | 定型化契約、特種交易、損害賠償 |
| 公平交易法 | 不實廣告、獨占結合 |
| 商品標示法 | 成分、產地、有效日期 |

> 💡 聰明的消費者不是買最便宜的，而是買最有價值的——了解自己的需求是理性消費的第一步。
