# 08 — 家庭關係

> 家是人際關係最密集的場所——學習溝通與衝突解決，經營健康的家庭關係。

## 家庭生命週期

| 階段 | 任務 |
|------|------|
| 新婚期 | 建立伴侶關係、協調原生家庭 |
| 育兒期 | 適應父母角色、調整夫妻關係 |
| 青少年期 | 平衡自主與管教 |
| 空巢期 | 重新定義夫妻關係 |
| 老年期 | 退休適應、健康管理 |

## 家庭溝通

| 原則 | 說明 |
|------|------|
| **主動傾聽** | 不打斷、不評判、回應感受 |
| **我訊息 (I-Message)** | 「我感到...因為...我希望...」取代「你總是...」 |
| **非暴力溝通** | 觀察→感受→需要→請求 |
| **正向表達** | 多肯定、少指責 |

## 衝突解決

| 步驟 | 說明 |
|------|------|
| 1. 冷靜 | 情緒高漲時暫停對話 |
| 2. 定義問題 | 明確衝突的核心是什麼 |
| 3. 表達感受 | 使用我訊息 |
| 4. 共同尋找方案 | 腦力激盪可能的解決方法 |
| 5. 達成協議 | 可接受、可執行的方案 |

## 健康家庭的特徵

- 開放溝通：成員能自由表達感受
- 互相尊重：尊重個別差異
- 共同時間：有規律的家庭時間
- 明確界線：個人空間與責任清晰
- 支持系統：彼此是最大的支援

> 💡 家庭不是沒有衝突，而是知道如何處理衝突——健康的家庭不是完美的家庭，而是能修復的家庭。
