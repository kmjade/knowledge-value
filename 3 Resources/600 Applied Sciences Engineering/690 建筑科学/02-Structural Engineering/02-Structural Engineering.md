# 02 — Structural Engineering 結構工程

> How do buildings stand up? Structural engineering ensures buildings resist gravity, wind, earthquakes, and other loads safely and economically.
> 建築物如何站立？結構工程確保建築物安全且經濟地抵抗重力、風力、地震和其他荷載。

---

## 荷載類型 Load Types

| Load | Description | Example |
|------|-------------|---------|
| **Dead Load (DL)** | Permanent, self-weight | Structure, finishes, fixed equipment |
| **Live Load (LL)** | Temporary, movable | People, furniture, vehicles |
| **Wind Load (WL)** | Wind pressure/suction | Especially critical for tall buildings |
| **Seismic Load (EQ)** | Earthquake ground motion | Lateral force, base shear |
| **Snow Load** | Accumulated snow weight | Roof design in cold climates |
| **Thermal Load** | Temperature expansion/contraction | Long structures, bridges |

---

## 結構系統 Structural Systems

| System | Description | Typical Height |
|--------|-------------|---------------|
| **Load-Bearing Wall** | Walls carry load | 1–5 stories |
| **Frame** | Beam + column skeleton | 1–30 stories |
| **Shear Wall** | RC walls resist lateral loads | 10–40 stories |
| **Braced Frame** | Diagonal bracing for stiffness | 10–60 stories |
| **Tube** | Perimeter columns as hollow tube | 40–100+ stories |
| **Outrigger** | Perimeter + core connected by trusses | 50–100+ stories |
| **Diagrid** | Diagonal grid exterior | 30–80+ stories |

## 抗震設計 Seismic Design

| Principle | Description |
|-----------|-------------|
| **Ductility** | Deform without sudden failure |
| **Strong Column — Weak Beam** | Beams yield before columns |
| **Base Isolation** | Decouple building from ground motion |
| **Tuned Mass Damper** | Counterweight reduces sway (Taipei 101) |
| **Energy Dissipation** | Dampers absorb seismic energy |

---

> 💡 The art of structural engineering is finding the most efficient load path — from roof to foundation — using the least material while maintaining safety.
