# 04 — Architectural Design 建築設計

> Architecture is the art and science of designing buildings — balancing function, form, context, and human experience.
> 建築是設計建築物的藝術與科學——在功能、形式、環境與人類體驗之間取得平衡。

---

## 設計原則 Design Principles

| Principle | Description |
|-----------|-------------|
| **Function** | Does it serve its purpose? |
| **Form** | How does it look and feel? |
| **Space** | How are volumes organized? |
| **Light** | Natural + artificial illumination |
| **Context** | How does it relate to surroundings? |
| **Materiality** | What materials express the intent? |
| **Sustainability** | Environmental performance |

---

## 建築類型 Building Typologies

| Type | Key Considerations |
|------|-------------------|
| **Residential** | Privacy, comfort, daylight |
| **Commercial/Office** | Flexibility, efficiency, branding |
| **Institutional** | Durability, security, civic presence |
| **Industrial** | Clear spans, logistics, utilities |
| **Cultural** | Iconic form, visitor experience |
| **Healthcare** | Infection control, patient flow |
| **Educational** | Acoustics, natural light, safety |

---

## 建築史簡覽 Architectural History

| Period | Characteristics | Examples |
|--------|----------------|----------|
| **Classical** | Columns, symmetry, proportion | Parthenon, Pantheon |
| **Gothic** | Pointed arches, flying buttresses | Notre-Dame |
| **Renaissance** | Geometry, humanism | St. Peter's Basilica |
| **Modernism** | Form follows function, minimalism | Villa Savoye, Seagram Building |
| **Postmodernism** | Playful, historical references | AT&T Building |
| **Contemporary** | Parametric, sustainable, digital | Burj Khalifa, CCTV HQ |
