# 03 — Construction Methods 施工方法

> How are buildings actually built? From traditional craftsmanship to robotic prefabrication — the methods that transform designs into reality.
> 建築物如何實際建造？從傳統工藝到機器人預製——將設計轉化為現實的方法。

## 傳統施工 Traditional Construction

### Cast-in-Place (現場澆築)

| 步驟 | 說明 |
|------|------|
| Formwork | 模板搭建 |
| Reinforcement | 鋼筋綁紮 |
| Concrete Pouring | 混凝土澆築 |
| Curing | 養護（28 天達設計強度） |
| Formwork Removal | 拆模 |

### Masonry (砌築)

| Material | Use |
|----------|-----|
| Brick | Walls, arches |
| Concrete Block (CMU) | Load-bearing walls |
| Stone | Veneer, structural (historic) |

## 現代工法 Modern Methods

### Precast Concrete 預製混凝土

| Advantage | Description |
|-----------|-------------|
| Quality | Factory-controlled conditions |
| Speed | Parallel site prep + factory production |
| Weather | Independent of site weather |
| Disadvantage | Transport cost, crane needed |

### Modular Construction 模組化建築

```
Factory → Modules (rooms/units) → Transport → Stack on site → Connect utilities
```

- 80–90% completed in factory
- 30–50% faster than traditional
- Examples: CitizenM hotels, residential towers

### Steel Frame

```
Steel columns + beams → Bolted/welded connections → Composite deck floors
```

- Fast erection
- Long spans possible
- Common for commercial/high-rise

## 施工階段 Construction Phases

```
1. Site Prep      場地準備      Clearing, excavation, shoring
2. Foundation     基礎工程      Piles, footings, waterproofing
3. Superstructure 上部結構      Columns, beams, slabs, walls
4. Enclosure      外殼          Curtain wall, roofing, windows
5. MEP Rough-in   機電管線      HVAC ducts, plumbing, electrical
6. Interior       室內          Drywall, flooring, finishes
7. Commissioning  調試          Systems testing, punch list
8. Handover       移交          Owner occupancy
```

---

> 💡 The best construction method depends on project context — site, budget, schedule, labor, and local regulations all factor in.
