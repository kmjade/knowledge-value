# 08 — Urban Planning 城市規劃

> Urban planning shapes how cities grow, function, and serve their inhabitants — from zoning to transit, from density to public space.
> 城市規劃塑造城市的成長、功能和服務方式——從分區到交通，從密度到公共空間。

## Core Concepts

| Concept | Description |
|---------|-------------|
| **Zoning** | Land use regulation (residential, commercial, industrial) |
| **Density** | FAR (Floor Area Ratio), dwelling units per acre |
| **Mixed-Use** | Combining residential, commercial, office in one development |
| **TOD** (Transit-Oriented Development) | High density around transit stations |
| **Public Realm** | Streets, parks, plazas — spaces between buildings |
| **Urban Heat Island** | City warmer than surroundings — mitigation via green roofs, trees |

## Planning Scales

| Scale | Focus |
|-------|-------|
| **Regional** | Transportation networks, growth boundaries |
| **City** | Comprehensive plan, zoning map |
| **District/Neighborhood** | Character, density, amenities |
| **Site** | Building placement, landscaping, access |
