# 09 — Construction Management 施工管理

> Delivering projects on time, on budget, and to quality — the discipline of construction management.
> 按時、按預算、按品質交付專案——施工管理的學科。

## Project Delivery Methods

| Method | Description | Risk |
|--------|-------------|------|
| **Design-Bid-Build** | Traditional: design → tender → build | Owner bears risk |
| **Design-Build** | Single entity designs + builds | Contractor bears risk |
| **CM at Risk** | Construction manager guarantees max price | Shared risk |
| **IPD** (Integrated Project Delivery) | All parties share risk/reward | Collaborative |

## Project Phases

```
Pre-construction → Procurement → Construction → Closeout
    規劃採購             發包               施工               竣工
```

## BIM (Building Information Modeling)

> BIM is a digital representation of a building's physical and functional characteristics.

| Dimension | What it adds |
|-----------|-------------|
| **3D** | Geometry |
| **4D** | Schedule (time) |
| **5D** | Cost |
| **6D** | Sustainability |
| **7D** | Facility management |

| Software | Use |
|----------|-----|
| Revit | BIM authoring |
| Navisworks | Clash detection, 4D/5D |
| BIM 360 | Cloud collaboration |
| Tekla | Structural detailing |

## Cost Management

| Stage | Activity |
|-------|----------|
| Conceptual | Order-of-magnitude estimate (±30%) |
| Design Development | Budget estimate (±15%) |
| Construction Documents | Definitive estimate (±5%) |
| Bidding | Contractor bids |
| Construction | Change orders, progress payments |

## Scheduling

| Tool | Description |
|------|-------------|
| **Gantt Chart** | Bar chart — most common |
| **Critical Path Method (CPM)** | Longest path = project duration |
| **Last Planner System** | Lean construction, collaborative planning |
| **Pull Planning** | Work backwards from milestones |

> 💡 Construction is about managing uncertainty — weather, labor, materials, changes. The best managers anticipate problems before they become crises.
