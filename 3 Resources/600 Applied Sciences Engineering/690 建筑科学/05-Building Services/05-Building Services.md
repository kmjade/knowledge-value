# 05 — Building Services (MEP) 建築設備

> MEP = Mechanical, Electrical, Plumbing. The systems that make buildings functional and comfortable.
> 機電系統 = 機械、電氣、給排水——讓建築物運作和舒適的系統。

## HVAC (Heating, Ventilation, Air Conditioning)

| System | Function |
|--------|----------|
| **Heating** | Boilers, heat pumps, radiant floors |
| **Ventilation** | Fresh air, CO₂ control, exhaust |
| **Air Conditioning** | Chillers, cooling towers, VRF, split systems |

| Type | Description |
|------|-------------|
| Central AHU | Large buildings, ducted |
| VRF/VRV | Variable refrigerant flow — flexible zoning |
| Split System | Residential, small commercial |
| Radiant | Floor/ceiling panels, comfortable |

## Plumbing 給排水

| System | Components |
|--------|-----------|
| **Water Supply** | Mains → storage tank → booster pump → fixtures |
| **Drainage** | Gravity flow → stack → sewer |
| **Hot Water** | Boiler / heat pump / solar |
| **Fire Sprinkler** | Wet pipe / dry pipe / pre-action |

## Electrical 電氣

| System | Description |
|--------|-------------|
| Power Distribution | Transformer → switchgear → panel boards |
| Lighting | Ambient, task, accent; LED standard |
| Emergency Power | Generator, UPS |
| Low Voltage | Data, security, BMS, fire alarm |

## Fire Protection 消防

| System | Description |
|--------|-------------|
| Sprinkler System | Automatic fire suppression |
| Standpipe | Firefighter hose connection |
| Fire Alarm | Detection + notification |
| Smoke Control | Pressurization, exhaust |

## BMS (Building Management System)

```
Sensors → Controllers → BMS Server → Dashboard
   ↓
HVAC, Lighting, Security, Energy
```

> 💡 MEP is ~30-40% of construction cost — getting it right is critical to building performance and occupant comfort.
