# 07 — 永續農業

> 永續農業的目標：滿足當代糧食需求，同時不損害後代滿足其需求的能力。

## 永續農業原則

| 原則 | 說明 |
|------|------|
| 生態健全 | 維護生物多樣性、土壤健康 |
| 經濟可行 | 農民能獲利、農村能發展 |
| 社會公平 | 食物正義、勞動權益 |

## 有機農業

| 規範 | 說明 |
|------|------|
| 禁止合成農藥 | 使用生物防治、天然資材 |
| 禁止化學肥料 | 使用有機肥、綠肥 |
| 禁止 GMO | 不得使用基因改造生物 |
| 輪作與多樣化 | 維持生態平衡 |

## 再生農業 (Regenerative Agriculture)

| 實踐 | 效益 |
|------|------|
| 免耕/少耕 | 減少土壤擾動 |
| 覆蓋作物 | 保護土壤、固碳 |
| 輪牧 | 模擬自然草食動物遷徙 |
| 堆肥 | 增加土壤有機質 |
| 農林複合 | 樹木 + 作物/畜牧 |

## 碳匯農業

農業既是碳排放源（化肥、畜牧、農機），也是碳匯——
- 土壤有機碳：最大的陸地碳庫
- 每增加 1% 土壤有機質 = 固碳約 8–10 噸/公頃
- 碳權交易為農民創造新收入

> 💡 永續農業不是「回到過去」——它是用現代科學實現古老智慧：與自然合作而非對抗。
