# 06 — 農業工程

> 工程技術賦能農業——從灌溉到自動化，從溫室到無人機，科技重塑生產方式。

## 灌溉系統

| 方式 | 效率 | 適用 |
|------|------|------|
| 漫灌 | 30–50% | 傳統、低成本 |
| 噴灌 | 60–80% | 大面積 |
| 滴灌 | 85–95% | 節水、精準 |
| 地下灌溉 | 90%+ | 先進設施 |

## 農業機械化

| 機械 | 功能 |
|------|------|
| 耕耘機 | 翻土、整地 |
| 播種機 | 精量播種 |
| 插秧機 | 水稻插秧 |
| 收割機 | 穀物收穫 |
| 無人機 | 噴藥、施肥、巡田 |

## 設施農業

| 設施 | 功能 |
|------|------|
| 溫室 | 溫度、濕度、光照控制 |
| 網室 | 防蟲、遮陰 |
| 植物工廠 | 全人工環境、LED 光源 |
| 水耕/氣耕 | 無土栽培 |

## 智慧農業

| 技術 | 應用 |
|------|------|
| IoT 感測器 | 土壤濕度、氣象監測 |
| GPS 導航 | 自動駕駛農機 |
| AI 影像辨識 | 病蟲害診斷、果實計數 |
| 區塊鏈 | 產銷履歷不可篡改 |

> 💡 智慧農業不是為了「取代農民」，而是讓農民少做苦力、多做決策。
