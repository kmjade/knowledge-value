# 08 — 植物保護

> 全球每年因病蟲害損失約 20–40% 的作物產量——植物保護是農業生產的防線。

## 有害生物分類

| 類型 | 舉例 |
|------|------|
| 病害 | 稻熱病、小麥銹病、晚疫病 |
| 蟲害 | 蚜蟲、蝗蟲、果實蠅、斜紋夜蛾 |
| 草害 | 稗草、馬唐、牛筋草 |
| 鼠害 | 田鼠、家鼠 |

## 防治方法

| 方法 | 說明 |
|------|------|
| **農業防治** | 抗病品種、輪作、清園 |
| **物理防治** | 黏蟲板、防蟲網、燈光誘殺 |
| **生物防治** | 天敵釋放（瓢蟲→蚜蟲）、微生物製劑 |
| **化學防治** | 農藥——最後手段 |

## IPM 綜合害物管理

```
預防 → 監測 → 閾值判斷 → 多種防治組合 → 評估
         ↓
   優先使用非化學方法
         ↓
   化學防治作為最後選項
```

## 農藥安全

| 原則 | 說明 |
|------|------|
| 對症下藥 | 正確診斷後選擇適當藥劑 |
| 適時適量 | 遵守推薦濃度和安全採收期 |
| 輪替用藥 | 避免抗藥性產生 |
| 防護裝備 | 口罩、手套、防護衣 |
| 廢容器回收 | 三次清洗、不隨意丟棄 |

> 💡 最好的植物保護是「不讓病蟲害發生」——健康的植株是最好的防禦。
