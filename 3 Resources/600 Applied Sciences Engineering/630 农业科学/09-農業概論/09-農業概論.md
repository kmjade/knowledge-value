# 09 — 農業概論

> 農業是人類最古老的產業，也是唯一直接利用太陽能進行大規模生產的產業。

## 農業的定義

農業是利用土地和自然資源，透過種植作物和飼養動物，生產食物、纖維和其他產品的產業。

## 農業發展簡史

| 時期 | 特徵 |
|------|------|
| 原始農業 (~10,000 BC) | 刀耕火種、遷移耕種 |
| 傳統農業 (~5,000 BC) | 定居農耕、輪作、灌溉 |
| 工業化農業 (18–19世紀) | 機械化、化肥、農藥 |
| 綠色革命 (1960s) | 高產品種、灌溉擴張 |
| 精準農業 (2000s) | GPS、IoT、數據驅動 |
| 智慧農業 (2020s) | AI、無人機、區塊鏈 |

## 農業的分類

| 分類 | 說明 |
|------|------|
| 依生產方式 | 傳統農業 / 現代化農業 / 有機農業 |
| 依產品 | 種植業 / 畜牧業 / 水產業 / 林業 |
| 依規模 | 小農 / 家庭農場 / 企業化農場 |
| 依目的 | 自給 / 商業 / 休閒農業 |

## 現代農業的挑戰

- 糧食安全：2050 年需增加 60% 糧食產量
- 氣候變遷：極端天氣、作物減產
- 水資源短缺：農業用水佔全球 70%
- 土壤退化：侵蝕、鹽化、有機質流失
- 農村衰退：人口老化、勞動力不足

> 💡 農業不僅是糧食生產——它是生態系統管理、文化傳承與國家安全的基石。
