# 05 — 農業經濟

> 農業不只是生產——從農場到市場，從價格到政策，經濟學貫穿整個農業鏈條。

## 農產品供應鏈

```
生產 → 集貨 → 分級包裝 → 批發 → 零售 → 消費者
```

## 農業經濟特性

| 特性 | 說明 |
|------|------|
| **供需彈性低** | 必需品，價格波動對消費量影響小 |
| **生產週期長** | 供給調整滯後於價格變化 |
| **天候風險** | 產量受自然條件強烈影響 |
| **蛛網理論** | 價格→生產決策→供給→價格，循環波動 |

## 農業政策工具

| 工具 | 說明 |
|------|------|
| 價格支持 | 保證收購價格 |
| 休耕補貼 | 減少供給以穩定價格 |
| 直接給付 | 對農民直接補貼 |
| 關稅與配額 | 保護國內農業 |
| 農作物保險 | 降低天災風險 |

## 農業發展趨勢

- 產銷履歷：從產地到餐桌可追溯
- 地產地消：縮短食物里程
- 六級產業化：一級生產 × 二級加工 × 三級服務
- 農業電商：線上直銷、社群團購

> 💡 農業問題往往是「穀賤傷農、穀貴傷民」的兩難——政策在生產者與消費者之間找平衡。
