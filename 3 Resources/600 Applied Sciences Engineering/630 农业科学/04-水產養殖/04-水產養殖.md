# 04 — 水產養殖

> 水產養殖是全球成長最快的食物生產部門——從魚塭到海洋牧場。

## 養殖系統

| 系統 | 說明 |
|------|------|
| **池塘養殖** | 傳統土池、水泥池 |
| **箱網養殖** | 海水/淡水箱網 |
| **循環水系統 (RAS)** | 室內封閉循環、高密度 |
| **魚菜共生** | 養魚 + 種菜，水資源循環利用 |
| **海洋牧場** | 開放海域放養 |

## 主要養殖物種

| 類別 | 物種 |
|------|------|
| 魚類 | 吳郭魚、石斑、鮭魚、鯉魚 |
| 甲殼類 | 白蝦、草蝦、螃蟹 |
| 軟體動物 | 牡蠣、文蛤、鮑魚 |
| 藻類 | 海帶、紫菜、龍鬚菜 |

## 養殖管理

| 環節 | 關鍵 |
|------|------|
| 水質 | 溶氧、pH、氨氮、溫度 |
| 飼料 | 飼料係數 (FCR) = 飼料量/增重量 |
| 疫病 | 預防為主、隔離、水質管理 |
| 密度 | 適當密度，避免緊迫 |

## 永續挑戰

- 魚粉資源枯竭：開發替代蛋白質飼料
- 抗生素殘留：減抗、無抗養殖
- 環境汙染：廢水處理、底泥管理
- 逃逸風險：養殖魚影響野生族群

> 💡 海洋捕撈已達極限——水產養殖是未來蛋白質的主要來源。
