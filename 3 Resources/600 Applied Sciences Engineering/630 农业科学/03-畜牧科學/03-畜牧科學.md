# 03 — 畜牧科學

> 畜牧科學涵蓋家禽家畜的飼養、育種、營養、管理與福祉。

## 主要畜禽類別

| 類別 | 物種 | 主要產品 |
|------|------|---------|
| 家畜 | 牛、豬、羊 | 肉、乳、皮、毛 |
| 家禽 | 雞、鴨、鵝 | 肉、蛋 |
| 其他 | 蜜蜂、兔、鹿 | 蜜、肉、茸 |

## 動物營養

| 營養素 | 功能 |
|--------|------|
| 碳水化合物 | 能量來源 |
| 蛋白質 | 生長、產肉產蛋 |
| 脂肪 | 能量儲存 |
| 礦物質 | 骨骼（鈣磷）、代謝 |
| 維生素 | 各種生理功能 |
| 水 | 最重要的營養素 |

## 飼養管理

| 系統 | 說明 |
|------|------|
| **集約飼養** | 高密度、高效率、需嚴控環境與疫病 |
| **放牧飼養** | 低密度、動物福祉佳、土地需求大 |
| **混合系統** | 結合放牧與舍飼 |

## 動物育種

| 方法 | 說明 |
|------|------|
| 選拔 | 選擇優良性狀個體留種 |
| 雜交 | 利用雜交優勢（生長快、抗病強） |
| 人工授精 | 優秀種公畜的廣泛利用 |
| 胚胎移植 | 加速優良母畜繁殖 |

## 動物福祉五大自由

1. 免於飢渴
2. 免於不適
3. 免於痛苦、傷害與疾病
4. 表達正常行為
5. 免於恐懼與緊迫

> 💡 畜牧業的核心挑戰：在生產效率與動物福祉之間取得平衡。
