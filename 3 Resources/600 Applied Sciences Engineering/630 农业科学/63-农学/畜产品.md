# 畜产品

> **UDC 637**: Produce of domestic animals — 乳、肉、蛋、皮、毛的加工与品质

畜产品是畜牧业与**消费者餐桌**之间的桥梁——从活体动物到可食用/可使用产品的转化过程中，品质控制、安全管理和加工技术决定了最终价值。这个领域横跨农业、食品科技和公共卫生。

## 核心领域

| 分支 | 内容 |
|------|------|
| **乳品** | 鲜乳、酸奶、奶酪、黄油、奶粉——从挤奶到包装 |
| **肉品** | 屠宰、分割、成熟、加工（腌制、熏制、发酵） |
| **蛋品** | 鲜蛋分级、液蛋、蛋粉——壳蛋到加工蛋 |
| **毛皮** | 羊毛分级、生皮加工、羽绒处理 |
| **食品安全** | 微生物、兽药残留、可追溯体系 |

## 核心概念

### 乳品链：从挤奶到货架

```
挤奶 → 速冷(4°C) → 运输 → 检验 → 加工 → 包装 → 冷链 → 零售
 │                                 │
 └─ 体细胞数(SCC) ────────────────┘
    乳房健康指标                  巴氏杀菌/超高温
    <20万 cells/mL = 优质          72°C/15s 或 138°C/2s
```

**乳成分基准**:

| 成分 | 荷斯坦牛 | 娟姗牛 | 水牛 |
|------|----------|--------|------|
| 乳脂 (%) | 3.5-4.0 | 5.0-6.0 | 7.0-8.0 |
| 乳蛋白 (%) | 3.0-3.3 | 3.8-4.0 | 4.0-4.5 |
| 乳糖 (%) | 4.8-5.0 | 4.8-5.0 | 5.1 |

### 肉的转化：从肌肉到食用肉

屠宰后肌肉经历一系列生化变化——**处理得好是熟成牛肉，处理不好是僵直肉**：

| 阶段 | 时间 | 变化 | 品质影响 |
|------|------|------|----------|
| **预僵直** | 0-3h | ATP 充足，肌肉松弛 | 此时烹饪最嫩——"热鲜肉" |
| **僵直** | 3-24h | ATP 耗尽，肌动球蛋白不可逆结合 | 肉变硬——不应食用 |
| **解僵成熟** | 1-4 周 | 钙蛋白酶降解肌纤维 | 嫩度回升、风味形成——"熟成" |

**PSE 肉 vs DFD 肉**——两个经典的肉质缺陷：

| 缺陷 | 机制 | 外观 | 原因 |
|------|------|------|------|
| **PSE** (苍白、软、渗水) | 宰后 pH 过快下降 | 灰白、松软 | 宰前应激（猪应激综合征） |
| **DFD** (暗、硬、干) | 宰前糖原耗竭，pH 不降 | 深黑、坚硬 | 长时间运输应激 |

### 蛋品质量

| 指标 | 含义 | 优级标准 |
|------|------|----------|
| **哈氏单位** | 蛋白高度/蛋重——蛋白浓稠度 | >72（AA 级） |
| **蛋壳强度** | 抗破碎能力 | >3.5 kg/cm² |
| **蛋黄颜色** | 消费者直观感受——不影响营养 | Roche 色扇 8-12 |
| **血斑/肉斑** | 异物——消费者反感 | 0 |

### 可追溯体系

畜产品安全的最后防线——"从牧场到餐桌"的全程追溯：

```
养殖环节 → 运输环节 → 屠宰加工 → 流通环节 → 零售终端
   │           │           │           │           │
  耳标      运输记录    屠宰编号   批次号   销售记录
```

**核心要素**: 个体标识（耳标/RFID）→ 数据库记录 → 标签载体（二维码/条形码）→ 消费者可查询

**中国实践**: 非洲猪瘟后，生猪"点对点"调运 + 分区防控——畜牧业的可追溯正在从"自愿"转向"强制"。

## 当代挑战

- **抗生素残留**: 养殖用药 → 畜产品残留 → 消费者健康风险 + 出口贸易壁垒
- **动物福利与品质**: 宰前应激 → 肉质缺陷（PSE/DFD）→ 经济损失——动物福利与产品品质高度一致
- **人畜共患病**: 布鲁氏菌病（生乳）、沙门氏菌（蛋品）、旋毛虫（猪肉）——加工环节的卫生控制
- **替代蛋白**: 植物奶（燕麦、大豆、杏仁）→ 对传统乳业的冲击；细胞培养肉——尚在成本曲线的陡峭下降段
- **碳排放标签**: 欧盟正在推进——1kg 牛肉 ≈ 60kg CO₂-eq vs 1kg 鸡肉 ≈ 6kg CO₂-eq → 碳标签将改变消费选择

## 在三层模型中的位置

畜产品处于**农业终点**与**食品工业起点**的交界处：

- **科学层**: 动物生理学（泌乳、肌肉生物学）、微生物学（发酵乳品、肉品腐败）、毒理学（兽药残留）
- **知识层**: 加工工艺（奶酪的"手感"、火腿的"看霉"）——大量默会知识尚未工程化
- **学科层**: 畜产品加工属于"食品科技"学科——与畜牧学、公共卫生交叉
