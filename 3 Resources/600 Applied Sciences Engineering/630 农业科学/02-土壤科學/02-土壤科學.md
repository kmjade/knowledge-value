# 02 — 土壤科學

> 土壤是農業的根基——健康的土壤產出健康的作物。

## 土壤組成

| 成分 | 比例 |
|------|------|
| 礦物質 | 45% |
| 水 | 25% |
| 空氣 | 25% |
| 有機質 | 5% |

## 土壤質地

| 質地 | 粒徑 | 特性 |
|------|------|------|
| **砂土** | 0.05–2mm | 排水好、保肥差 |
| **壤土** | 混合 | 最佳耕作土壤 |
| **黏土** | <0.002mm | 保水保肥、排水差 |

## 土壤肥力

| 要素 | 功能 |
|------|------|
| **氮 (N)** | 葉片生長、蛋白質合成 |
| **磷 (P)** | 根系發展、能量轉移 |
| **鉀 (K)** | 抗病、果實品質 |
| **有機質** | 改善結構、養分緩釋 |
| **pH** | 多數作物適宜 pH 6.0–7.0 |

## 土壤退化問題

| 問題 | 成因 |
|------|------|
| 侵蝕 | 風蝕、水蝕、過度耕種 |
| 鹽化 | 灌溉不當、排水不良 |
| 酸化 | 過量氮肥、酸雨 |
| 壓實 | 機械碾壓、有機質不足 |

## 土壤保育

| 措施 | 說明 |
|------|------|
| 覆蓋作物 | 防止侵蝕、增加有機質 |
| 免耕/少耕 | 減少土壤擾動 |
| 輪作 | 平衡養分消耗 |
| 有機肥料 | 堆肥、綠肥 |

> 💡 土壤不是「泥土」——它是活的生態系統。一茶匙健康土壤中有數十億微生物。
