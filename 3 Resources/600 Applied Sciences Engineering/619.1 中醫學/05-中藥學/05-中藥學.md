# 05 — 中藥學

> 中藥是自然界中用於防治疾病的物質——每一味藥有其獨特的性味、歸經和功效。

## 四氣五味

| 四氣 | 說明 |
|------|------|
| 寒 | 清熱瀉火（黃連、石膏） |
| 熱 | 溫裡散寒（附子、乾薑） |
| 溫 | 溫和散寒（桂枝、生薑） |
| 涼 | 輕度清熱（薄荷、菊花） |

| 五味 | 作用 | 代表藥 |
|------|------|--------|
| 辛 | 發散、行氣 | 麻黃、薄荷 |
| 甘 | 補益、緩和 | 甘草、大棗 |
| 酸 | 收斂、固澀 | 五味子、烏梅 |
| 苦 | 燥濕、泄降 | 黃連、大黃 |
| 鹹 | 軟堅、瀉下 | 芒硝、牡蠣 |

## 中藥分類（按功效）

| 類別 | 功效 | 代表藥物 |
|------|------|---------|
| 解表藥 | 發散表邪 | 麻黃、桂枝、薄荷、菊花 |
| 清熱藥 | 清熱瀉火 | 黃連、黃芩、金銀花、連翹 |
| 祛濕藥 | 利水滲濕、化濕 | 茯苓、薏苡仁、藿香 |
| 理氣藥 | 行氣解鬱 | 陳皮、木香、柴胡 |
| 活血化瘀藥 | 促進血行 | 丹參、川芎、紅花 |
| 補虛藥 | 補氣養血滋陰助陽 | 人參、黃耆、當歸、熟地 |
| 安神藥 | 安神定志 | 酸棗仁、遠志 |
| 平肝息風藥 | 平抑肝陽 | 天麻、鉤藤 |

## 歸經

藥物對特定臟腑經絡的選擇性作用：
- 黃連 → 心、胃經（清心火、胃火）
- 麻黃 → 肺、膀胱經（宣肺、利水）
- 柴胡 → 肝、膽經（疏肝解鬱）

## 用藥禁忌

| 禁忌 | 說明 |
|------|------|
| 十八反 | 某些藥物配伍產生毒性 |
| 十九畏 | 某些藥物配伍降低療效 |
| 妊娠禁忌 | 孕婦禁用/慎用藥物 |
| 證候禁忌 | 根據證型選擇，如陰虛者忌辛溫 |

> 💡 中藥的核心是「藥性」而非「成分」——性味歸經是中藥獨特的藥理語言。
