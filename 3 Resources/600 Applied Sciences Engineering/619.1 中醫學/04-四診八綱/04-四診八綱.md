# 04 — 四診八綱

> 四診是收集病情的方法，八綱是分析病情的綱領——兩者構成中醫診斷體系的核心。

## 四診

### 望診

| 內容 | 觀察重點 |
|------|---------|
| **望神** | 精神狀態、目光、表情 |
| **望色** | 面色（青赤黃白黑對應五臟） |
| **望舌** | 舌質（色、形、態）+ 舌苔（色、厚薄、潤燥） |
| **望形態** | 體型、姿勢、動作 |

### 聞診

| 內容 | 說明 |
|------|------|
| **聽聲音** | 語聲高低、咳嗽、呼吸、腸鳴 |
| **嗅氣味** | 口氣、汗味、分泌物氣味 |

### 問診

十大問歌：一問寒熱二問汗，三問頭身四問便，五問飲食六問胸，七聾八渴俱當辨，九問舊病十問因。

### 切診

| 內容 | 說明 |
|------|------|
| **脈診** | 寸口脈法：寸關尺三部，浮中沉三候 |
| **觸診** | 按壓腹部、四肢溫度 |

## 常見脈象

| 脈象 | 特徵 | 主病 |
|------|------|------|
| 浮脈 | 輕取即得 | 表證 |
| 沉脈 | 重按始得 | 裡證 |
| 遲脈 | 一息不足四至 | 寒證 |
| 數脈 | 一息超過五至 | 熱證 |
| 虛脈 | 按之無力 | 虛證 |
| 實脈 | 按之有力 | 實證 |
| 弦脈 | 如按琴弦 | 肝病、痛證 |
| 滑脈 | 往來流利 | 痰飲、食積、妊娠 |

## 八綱辨證

| 綱領 | 對立 | 核心區別 |
|------|------|---------|
| **陰陽** | 陰證 / 陽證 | 總綱，統攝其餘六綱 |
| **表裡** | 表證 / 裡證 | 病位深淺 |
| **寒熱** | 寒證 / 熱證 | 疾病性質 |
| **虛實** | 虛證 / 實證 | 正氣與邪氣的力量對比 |

> 💡 八綱是辨證的總綱——任何疾病都可以先用八綱定性，再結合臟腑、六經等深入辨證。
