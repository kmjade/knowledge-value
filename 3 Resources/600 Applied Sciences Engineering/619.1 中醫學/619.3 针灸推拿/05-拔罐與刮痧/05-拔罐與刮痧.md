# 05 — 拔罐與刮痧

> 拔罐與刮痧是中醫最普遍的民間療法——簡便驗廉，效果顯著。

## 拔罐

### 罐具與吸拔方法

| 罐具 | 特點 |
|------|------|
| 玻璃罐 | 可觀察皮膚、適合走罐 |
| 竹罐 | 輕便、可藥煮 |
| 真空抽氣罐 | 安全、無火 |
| 陶罐 | 傳統、保溫 |

| 吸拔方法 | 操作 |
|---------|------|
| 火罐（閃火法） | 燃燒的棉球快速放入罐內 |
| 抽氣法 | 真空槍抽取空氣 |

### 拔罐方式

| 方式 | 操作 | 適應 |
|------|------|------|
| 留罐 | 吸附後留置 10–15min | 一般 |
| 走罐 | 塗抹潤滑油後滑動 | 背部膀胱經 |
| 閃罐 | 快速吸附後取下，反覆 | 面癱、局部麻木 |
| 刺絡拔罐 | 先刺絡再拔罐 | 瘀血、痛症 |

## 刮痧

| 工具 | 牛角板、玉石板、瓷勺 |
|------|------|
| 方向 | 由上到下、由內到外，單方向 |
| 力度 | 均勻適中，以出痧為度 |
| 部位 | 背部膀胱經、頸部、四肢 |

## 出痧的意義

- 痧色鮮紅：新病、病情較輕
- 痧色紫暗：久病、瘀血較重
- 痧點密集：病邪較盛

## 注意事項

- 拔罐後 4 小時內不洗澡、不吹風
- 同一部位痧退後方可再次刮痧
- 出血傾向者、孕婦腹部禁用

> 💡 拔罐和刮痧是「讓身體自己告訴你哪裡有問題」的診療一體方法——出痧的部位往往就是病灶所在。
