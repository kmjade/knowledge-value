# 02 — 腧穴各論

> 腧穴是經絡上氣血匯聚的特殊點位——361 個經穴 + 經外奇穴 + 阿是穴。

## 特定穴分類

| 類型 | 數量 | 說明 | 舉例 |
|------|------|------|------|
| **五輸穴** | 60 | 井滎輸經合，五行配屬 | 少商(井)、魚際(滎) |
| **原穴** | 12 | 臟腑原氣經過留止 | 合谷(大腸)、太淵(肺) |
| **絡穴** | 15 | 聯絡表裡經脈 | 列缺(肺絡大腸) |
| **郄穴** | 16 | 經氣深聚處，多治急症 | 孔最(肺)、梁丘(胃) |
| **俞穴** | 12 | 臟腑之氣輸注於背 | 肺俞、心俞、肝俞 |
| **募穴** | 12 | 臟腑之氣匯聚於腹 | 中府(肺)、巨闕(心) |
| **八會穴** | 8 | 臟腑氣血筋脈骨髓等 | 中脘(腑會)、膈俞(血會) |
| **八脈交會穴** | 8 | 通於奇經八脈 | 內關通陰維、外關通陽維 |

## 取穴方法

| 方法 | 說明 |
|------|------|
| 骨度分寸 | 以骨節為標誌等分測量 |
| 體表標誌 | 固定標誌（五官、毛髮）+ 活動標誌（關節） |
| 手指同身寸 | 中指、拇指、橫指 |
| 簡便取穴 | 經驗取穴（如列缺：兩虎口交叉食指尖處） |

## 十大要穴

| 穴位 | 經脈 | 定位 | 功效 |
|------|------|------|------|
| 合谷 LI4 | 大腸經 | 手背虎口 | 頭面諸疾、止痛 |
| 足三里 ST36 | 胃經 | 犢鼻下3寸 | 健脾胃、強身 |
| 內關 PC6 | 心包經 | 腕上2寸 | 寧心安神、止嘔 |
| 三陰交 SP6 | 脾經 | 內踝上3寸 | 婦科、泌尿 |
| 百會 GV20 | 督脈 | 頭頂正中 | 升陽、醒腦 |
| 關元 CV4 | 任脈 | 臍下3寸 | 補腎固本 |
| 湧泉 KI1 | 腎經 | 足心 | 滋陰降火 |
| 大椎 GV14 | 督脈 | 第七頸椎下 | 退熱、通陽 |
| 太衝 LR3 | 肝經 | 足背1-2蹠骨間 | 平肝、降壓 |
| 神門 HT7 | 心經 | 腕橫紋尺側 | 安神 |

> 💡 針灸的精髓在於「選穴」——辨證選穴、循經取穴，穴不在多，在於精準。
