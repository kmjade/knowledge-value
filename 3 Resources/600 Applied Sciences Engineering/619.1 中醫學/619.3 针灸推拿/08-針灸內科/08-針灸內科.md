# 08 — 針灸內科

> 針灸在內科疾病的應用——從中風到失眠，從頭痛到胃痛。

## 中風（腦卒中）

| 時期 | 治法 | 穴位 |
|------|------|------|
| 急性期 | 醒腦開竅 | 人中、內關、三陰交 |
| 恢復期 | 疏通經絡 | 頭皮針運動區 + 手足陽明經 |
| 後遺症期 | 調理肝腎 | 肝俞、腎俞、太衝、太谿 |

## 面癱（顏面神經麻痺）

| 分期 | 穴位 |
|------|------|
| 急性期（1–7天） | 風池、翳風、合谷（遠端取穴） |
| 恢復期 | 陽白、四白、地倉、頰車、下關 |
| 配合 | 閃罐、電針（疏波） |

## 頭痛

| 證型 | 治法 | 主穴 |
|------|------|------|
| 風寒 | 疏風散寒 | 風池、太陽、合谷 |
| 肝陽 | 平肝潛陽 | 太衝、太谿、百會 |
| 血虛 | 養血止痛 | 足三里、三陰交、百會 |
| 瘀血 | 活血化瘀 | 阿是穴、血海 |

## 失眠

| 證型 | 主穴 |
|------|------|
| 心脾兩虛 | 神門、三陰交、心俞、脾俞 |
| 肝火擾心 | 太衝、行間、神門 |
| 心腎不交 | 神門、太谿、心俞、腎俞 |

## 胃痛

| 證型 | 主穴 |
|------|------|
| 虛寒 | 足三里、中脘（灸） |
| 肝氣犯胃 | 太衝、期門、足三里 |

> 💡 WHO 推薦針灸治療的 100+ 種疾病中，神經系統疾病和消化道疾病佔了很大比例——這些正是針灸的強項。
