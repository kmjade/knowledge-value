# 11 — 現代研究與國際化

> 針灸是中醫國際化最成功的領域——從實驗室到臨床，從 NIH 到 WHO。

## 神經機制

| 機制 | 說明 |
|------|------|
| 閘門控制理論 | 針刺激活 Aβ 纖維關閉脊髓疼痛閘門 |
| 內源性鴉片肽 | β-內啡肽、腦啡肽、強啡肽 |
| 下行抑制通路 | PAG→NRM→脊髓背角 |
| 腺苷 A1 受體 | 針刺局部釋放腺苷，鎮痛 |

## fMRI 研究

- 針刺特定穴位激活對應腦區
- 合谷→體感皮層（手部代表區）
- 足三里→邊緣系統（內臟調節）
- 太衝→視覺相關皮層（肝開竅於目）
- 得氣狀態 vs 銳痛：不同腦網路激活模式

## 經絡實質研究

| 發現 | 說明 |
|------|------|
| 低電阻特性 | 經絡線電阻低於周圍組織 |
| 高導聲特性 | 經絡線對聲波傳導更佳 |
| 紅外輻射軌跡 | 與古典經絡路線一致 |
| 同位素追蹤 | 循經遷移現象 |
| 鈣離子富集 | 穴位處 Ca²⁺ 濃度較高 |

## 國際認可

| 里程碑 | 年份 |
|--------|------|
| WHO 發布針灸適應症清單 | 1979 |
| NIH 共識聲明：針灸對疼痛有效 | 1997 |
| 美國 FDA 將針灸針列為醫療器械 | 1996 |
| 德國法定保險覆蓋針灸（腰痛、膝痛） | 2007 |
| 全球 183 個國家使用針灸 | 2020s |

> 💡 針灸的科學驗證仍在進行中——已經驗證的部分足以支撐其臨床價值，未解之謎則激勵著新一代研究者。
