# 09 — 針灸外科與痛症

> 針灸治療痛症效果最為突出——針刺鎮痛的機制已被現代科學充分證實。

## 頸椎病

| 類型 | 主穴 |
|------|------|
| 頸型 | 風池、天柱、頸夾脊 |
| 神經根型 | 風池、頸夾脊 + 上肢循經取穴 |
| 椎動脈型 | 風池、百會、太衝 |

## 肩周炎

| 階段 | 主穴 |
|------|------|
| 疼痛期 | 肩髃、肩髎、肩貞、條口透承山 |
| 黏連期 | 加推拿鬆解 |

## 腰椎間盤突出

| 治法 | 穴位 |
|------|------|
| 腰部 | 腰夾脊、腎俞、大腸俞 |
| 放射痛（坐骨神經） | 環跳、委中、陽陵泉、崑崙 |

## 膝關節炎

| 治法 | 穴位 |
|------|------|
| 局部 | 鶴頂、內膝眼、外膝眼 |
| 遠端 | 足三里、陽陵泉 |
| 配合 | 艾灸、電針 |

## 針刺鎮痛機制

| 機制 | 說明 |
|------|------|
| 閘門控制 | 針刺訊號「關閉」脊髓疼痛閘門 |
| 內源性鴉片肽 | 釋放腦啡肽、β-內啡肽 |
| 下行抑制 | 激活中腦導水管周圍灰質 |

> 💡 針灸治療痛症是 EBM 證據最充分的領域——多篇 Cochrane 綜述確認針灸對慢性疼痛的有效性。
