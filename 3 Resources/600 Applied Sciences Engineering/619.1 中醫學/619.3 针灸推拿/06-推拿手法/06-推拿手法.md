# 06 — 推拿手法

> 推拿是中醫最古老的療法之一——以手法治療，疏通經絡，調理臟腑。

## 六大基本手法

| 手法 | 操作 | 作用 |
|------|------|------|
| **按法** | 手指或手掌垂直按壓 | 鎮靜止痛 |
| **摩法** | 環形撫摸、輕柔 | 和中理氣 |
| **推法** | 單方向推動 | 疏通經絡 |
| **拿法** | 捏而提起 | 鬆解肌肉 |
| **揉法** | 旋轉揉動 | 活血化瘀 |
| **點法** | 指尖點壓穴位 | 以指代針 |

## 其他常用手法

| 手法 | 操作 | 適應 |
|------|------|------|
| 㨰法 | 手背滾動 | 腰背、大腿 |
| 拍法 | 虛掌拍打 | 活血、鬆弛 |
| 抖法 | 牽拉後抖動 | 放鬆關節 |
| 搖法 | 關節環轉 | 增加活動度 |
| 扳法 | 快速小幅度的關節整復 | 頸椎腰椎錯位 |

## 適應證

| 類別 | 疾病 |
|------|------|
| 傷科 | 頸椎病、肩周炎、腰肌勞損、椎間盤突出 |
| 內科 | 頭痛、失眠、胃痛、便秘 |
| 兒科 | 小兒推拿：食積、腹瀉、夜啼、發熱 |
| 保健 | 全身放鬆、消除疲勞 |

## 小兒推拿特點

| 特點 | 說明 |
|------|------|
| 手法輕柔 | 力度遠小於成人 |
| 特定穴位 | 脾經、大腸經、板門、天河水等 |
| 介質 | 滑石粉、按摩油 |

> 💡 推拿是最安全的療法——不破皮、不用藥。小兒推拿尤其值得推廣，讓孩子在無痛中康復。
