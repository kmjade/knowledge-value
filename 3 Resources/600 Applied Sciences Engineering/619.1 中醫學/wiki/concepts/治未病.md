# 治未病 Preventive Treatment

> 聖人不治已病治未病，不治已亂治未亂。——《素問·四氣調神大論》

## 定義 Definition

「治未病」是中醫學最重要的預防醫學思想，強調在疾病發生之前進行預防，或在疾病發展的早期階段進行干預，防止病情加重和傳變。

## 三層含義 Three Levels

### 1. 未病先防 Prevention Before Illness

在健康狀態下採取措施防止疾病發生：

| 方法 | 說明 |
|------|------|
| **順應四時** | 春生夏長、秋收冬藏——起居與自然同步 |
| **調攝情志** | 喜怒憂思悲恐驚不可過度 |
| **飲食有節** | 五味調和、不可偏嗜 |
| **起居有常** | 規律作息、不妄作勞 |
| **適度運動** | 五禽戲、太極拳、八段錦 |

### 2. 既病防變 Preventing Progression

已病之後防止疾病向深層次發展：

> 「見肝之病，知肝傳脾，當先實脾。」——《金匱要略》

臟腑之間按五行生剋關係傳變——治療時預先強化將被傳變的臟腑。

### 3. 瘥後防復 Preventing Recurrence

病癒後防止復發——避免勞復、食復、情志復發。

## 臨床應用

| 層面 | 現代對應 |
|------|---------|
| 未病先防 | 疫苗接種、健康體檢、生活方式管理 |
| 既病防變 | 早期診斷、併發症預防 |
| 瘥後防復 | 康復管理、隨訪制度 |

## 經典闡發

> 「夫病已成而後藥之，亂已成而後治之，譬猶渴而穿井，鬥而鑄錐，不亦晚乎！」——《素問·四氣調神大論》

等到疾病已成才用藥，如同口渴了才去挖井——來不及了。這是對「治未病」最生動的比喻。
