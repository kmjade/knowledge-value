# 02 — 哲學基礎

> 黃帝內經的醫學理論建立在深厚的中國哲學基礎上——天人合一、氣一元論、形神合一。

## 天人合一

人與自然是一個有機整體：
- 「人以天地之氣生，四時之法成」（素問·寶命全形論）
- 人體生理節律與自然同步：晝夜節律、月經週期、四季脈象
- 養生須順應四時：春生、夏長、秋收、冬藏

## 氣一元論

萬物由氣構成——氣是宇宙的本源，也是生命的根基。
- 「天地合氣，命之曰人」（素問·寶命全形論）
- 氣的分類：天氣（自然之氣）、地氣（水穀之氣）、真氣（元氣）
- 疾病的根源 = 氣的異常（氣虛、氣滯、氣逆）

## 形神合一

| 概念 | 內容 |
|------|------|
| **形** | 身體、臟腑、經絡、氣血 |
| **神** | 精神、意識、思維、情緒 |
| **關係** | 形為神之宅，神為形之主 |

- 「心者，君主之官，神明出焉」（素問·靈蘭秘典論）
- 情志過極傷五臟：怒傷肝、喜傷心、思傷脾、悲傷肺、恐傷腎

## 整體觀念

| 層次 | 說明 |
|------|------|
| 人與自然 | 天人相應，四時調神 |
| 人體內部 | 五臟六腑經絡互相聯繫 |
| 身心統一 | 形神一體，情志致病 |

> 💡 內經的哲學基礎決定了中醫的方法論——不是頭痛醫頭，而是從整體尋找失衡的根源。
