# 09 — 針刺（靈樞篇）

> 靈樞是中醫針灸學的奠基之作——系統論述了針具、手法、補瀉、得氣與禁忌。

## 核心篇章

- **靈樞·九針十二原**：九針介紹、針刺原則
- **靈樞·小針解**：針刺手法詳解
- **靈樞·官針**：九種刺法
- **靈樞·終始**：針刺的開始與結束

## 九針

| 針具 | 形狀 | 用途 |
|------|------|------|
| 鑱針 | 頭大末銳 | 淺刺瀉熱 |
| 圓針 | 針身圓直 | 按摩穴位 |
| 鍉針 | 針尖如粟 | 按壓經脈 |
| 鋒針 | 三棱 | 放血瀉熱 |
| 鈹針 | 劍形 | 切開排膿 |
| 圓利針 | 針尖微大 | 治療暴痺 |
| 毫針 | 細長 | 最常用針具 |
| 長針 | 長而細 | 深刺 |
| 大針 | 粗大 | 瀉水 |

## 補瀉手法

| 手法 | 操作 | 適應證 |
|------|------|--------|
| **補法** | 進針慢、出針快、按壓針孔 | 虛證 |
| **瀉法** | 進針快、出針慢、搖大針孔 | 實證 |
| **徐疾補瀉** | 慢進快出為補，快進慢出為瀉 | |
| **呼吸補瀉** | 呼氣進針為補，吸氣進針為瀉 | |
| **迎隨補瀉** | 順經為補，逆經為瀉 | |

## 得氣

「刺之要，氣至而有效」（靈樞·九針十二原）
- 患者感：酸、麻、脹、重
- 醫者感：針下沉緊
- 得氣是針刺有效的關鍵

## 針刺禁忌

| 禁忌 | 說明 |
|------|------|
| 大醉、大怒、大勞 | 氣血逆亂不宜針 |
| 大飢、大渴 | 氣血不足不宜針 |
| 重要臟器部位 | 避免刺傷內臟 |
| 大血管處 | 避免出血 |

> 💡 靈樞的針刺體系至今仍是針灸臨床的基礎——兩千年來，毫針的形制和基本手法幾乎沒有改變。
