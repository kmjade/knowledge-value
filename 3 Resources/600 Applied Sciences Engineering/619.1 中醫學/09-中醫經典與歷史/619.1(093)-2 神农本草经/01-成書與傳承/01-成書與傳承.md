# 01 — 成書與傳承

> 神農本草經是中國最早的藥學專著，託名神農氏，實際上是漢代醫家集體創作。

## 成書背景

| 觀點 | 說明 |
|------|------|
| 主流共識 | 東漢時期（約 200 AD），非一人一時之作 |
| 託名神農 | 神農嘗百草的傳說，賦予藥學權威性 |
| 與內經關係 | 與《黃帝內經》同為中醫四大經典，理論互補 |

## 原始版本與輯佚

| 階段 | 說明 |
|------|------|
| 原始版本 | 東漢成書，已佚失 |
| 歷代引用 | 散見於《證類本草》《千金要方》等 |
| 輯佚復原 | 明清學者從古籍中輯出 |
| 主要輯本 | 孫星衍/孫馮翼輯本（1799）、顧觀光輯本（1844） |
| 日本輯本 | 森立之輯本（1854），考據精審 |

## 在醫學史上的地位

| 貢獻 | 影響 |
|------|------|
| 最早的藥學分類 | 三品分類法影響千年 |
| 藥性理論奠基 | 四氣五味學說沿用至今 |
| 七情配伍 | 藥物相互關係的系統總結 |
| 君臣佐使雛形 | 方劑配伍理論的源頭 |
| 後世本草之源 | 陶弘景《本草經集注》等皆以此為基礎 |

> 💡 我們今天看到的《神農本草經》是輯佚本——原書已佚，但它的核心內容透過歷代引用完整地流傳至今。
