# 01 — 張仲景與成書

> 張仲景（約 150–219 AD），名機，東漢南陽人，被譽為「醫聖」。

## 生平

| 事項 | 內容 |
|------|------|
| 時代 | 東漢末年，戰亂頻仍，瘟疫流行 |
| 家族悲劇 | 宗族二百餘人，十年內死亡三分之二，多死於傷寒 |
| 學醫背景 | 師從同郡張伯祖 |
| 著作動機 | 「感往昔之淪喪，傷橫夭之莫救」 |

## 版本流傳

| 階段 | 說明 |
|------|------|
| 原書 | 傷寒雜病論十六卷，東漢末成書 |
| 王叔和整理 | 晉代（~280 AD），編次傷寒部分 |
| 宋代分刊 | 1065 年校正醫書局刊行《傷寒論》和《金匱要略》 |
| 現代通行 | 宋本傷寒論、成無己《註解傷寒論》（1144） |

## 序言名句

> 「上以療君親之疾，下以救貧賤之厄，中以保身長全，以養其生。」
> 「觀今之醫，不念思求經旨，以演其所知；各承家技，始終順舊。」

> 💡 張仲景是中國醫學史上第一位將理論與臨床完美結合的醫家——「醫聖」之名，當之無愧。
