# 02 — 六經辨證總論

> 六經辨證是傷寒論的核心辨證體系，將外感熱病的發生發展分為六個階段（太陽→陽明→少陽→太陰→少陰→厥陰）。

## 六經總覽

| 經 | 性質 | 病位 | 病機 | 代表方 |
|----|------|------|------|--------|
| **太陽** | 表證 | 體表、膀胱 | 外邪初犯 | 桂枝湯、麻黃湯 |
| **陽明** | 裡熱實證 | 胃腸 | 熱盛津傷 | 白虎湯、承氣湯 |
| **少陽** | 半表半裡 | 膽、三焦 | 正邪分爭 | 小柴胡湯 |
| **太陰** | 裡虛寒證 | 脾胃 | 脾陽虛弱 | 理中湯 |
| **少陰** | 心腎虛衰 | 心腎 | 陰陽兩虛 | 四逆湯 |
| **厥陰** | 寒熱錯雜 | 肝 | 陰陽失調 | 烏梅丸 |

## 傳經規律

```
太陽 ──→ 陽明 ──→ 少陽 ──→ 太陰 ──→ 少陰 ──→ 厥陰
  │        │        │        │        │
  └────────┴────────┴── 越經傳 ─┘        │
  └────────────────── 直中 ──────────────┘
```

| 傳變 | 說明 |
|------|------|
| **循經傳** | 按六經順序傳變 |
| **越經傳** | 跳過某一經 |
| **直中** | 病邪直接侵犯三陰 |
| **合病** | 兩經或三經同時發病 |
| **併病** | 一經未罷，又傳另一經 |

> 💡 六經辨證不是解剖位置的劃分，而是疾病發展階段的動態模型——這是張仲景最偉大的臨床發明。
