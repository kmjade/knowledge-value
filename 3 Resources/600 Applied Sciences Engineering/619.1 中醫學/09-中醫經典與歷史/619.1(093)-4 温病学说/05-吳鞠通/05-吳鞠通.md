# 05 — 吳鞠通（1758–1836）

> 吳瑭，字鞠通，清代溫病學大家，《溫病條辨》作者，創立三焦辨證體系。

## 生平

| 事項 | 內容 |
|------|------|
| 籍貫 | 江蘇淮陰 |
| 學醫契機 | 父病不治，發奮學醫 |
| 遊歷 | 因京師溫疫流行，赴京行醫 |
| 著作 | 《溫病條辨》（1798） |

## 主要貢獻

| 貢獻 | 說明 |
|------|------|
| **三焦辨證** | 縱向劃分溫病病程 |
| **溫病條辨** | 系統整理溫病理法方藥 |
| **銀翹散** | 辛涼平劑，溫病第一方 |
| **安宮牛黃丸** | 溫病神昏急救要藥 |

## 溫病條辨名句

- 「治上焦如羽，非輕不舉」
- 「治中焦如衡，非平不安」
- 「治下焦如權，非重不沉」

## 葉天士與吳鞠通對比

| | 葉天士 | 吳鞠通 |
|------|--------|--------|
| 辨證 | 衛氣營血（橫向） | 三焦（縱向） |
| 著作 | 溫熱論 | 溫病條辨 |
| 方劑 | 多為經驗 | 系統整理方劑 |
| 影響 | 理論奠基 | 體系完成 |

> 💡 葉天士是開創者，吳鞠通是集大成者——兩人共同完成了溫病學從理論到臨床的完整體系。
