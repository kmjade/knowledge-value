# 03 — 三焦辨證

> 吳鞠通創立的縱向辨證體系——以上中下三焦劃分溫病的病程階段和病位。

## 三焦層次

| 焦 | 臟腑 | 病程階段 | 主症 | 代表方 |
|----|------|---------|------|--------|
| **上焦** | 肺、心包 | 初期 | 發熱、咳嗽、神昏 | 桑菊飲、銀翹散 |
| **中焦** | 脾胃、腸 | 中期 | 高熱、腹滿、便秘 | 白虎湯、承氣湯 |
| **下焦** | 肝腎 | 後期 | 低熱、手足心熱、抽搐 | 加減復脈湯 |

## 三焦傳變

```
上焦（肺）──→ 中焦（脾胃）──→ 下焦（肝腎）
  初期            極期            後期
```

## 衛氣營血 vs 三焦

| | 衛氣營血 | 三焦 |
|------|---------|------|
| 創立者 | 葉天士 | 吳鞠通 |
| 維度 | 橫向（淺深） | 縱向（上下 + 病程） |
| 核心 | 熱入營血的病理變化 | 臟腑定位與病程階段 |
| 關係 | 互相補充，臨床常結合使用 | |

## 結合應用

```
      衛      氣      營      血
上焦    ✓       ✓
中焦            ✓       ✓
下焦                            ✓
```

> 💡 吳鞠通說「傷寒論六經，由表入裡，由淺及深，須橫看；本論論三焦，由上及下，亦由淺入深，須豎看。」——橫看豎看，經緯交織，方為完整。
