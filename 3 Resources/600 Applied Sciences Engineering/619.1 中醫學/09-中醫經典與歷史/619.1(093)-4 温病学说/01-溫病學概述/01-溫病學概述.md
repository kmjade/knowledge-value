# 01 — 溫病學概述

> 溫病是由溫熱病邪引起的急性外感熱病，以發熱、化燥傷陰為主要特徵。

## 溫病學發展簡史

| 時期 | 醫家 | 貢獻 |
|------|------|------|
| 內經 | — | 「冬傷於寒，春必病溫」最早記載 |
| 傷寒論 | 張仲景 | 提及溫病，但詳寒略溫 |
| 宋代 | 龐安時 | 提出寒溫分治 |
| 金元 | 劉完素 | 「六氣皆從火化」，主寒涼 |
| 明代 | 吳又可 | 《溫疫論》，戾氣學說（病原微生物雛形） |
| 清代 | 葉天士 | 《溫熱論》，衛氣營血辨證 |
| 清代 | 吳鞠通 | 《溫病條辨》，三焦辨證 |
| 清代 | 王孟英 | 《溫熱經緯》，集大成 |

## 溫病的特點

| 特點 | 說明 |
|------|------|
| 發病急 | 起病急驟，傳變迅速 |
| 熱象重 | 以發熱為主，易化燥傷陰 |
| 易傳變 | 衛→氣→營→血層層深入 |
| 季節性 | 多發於春冬季節 |
| 傳染性 | 部分具有強烈傳染性（溫疫） |

> 💡 溫病學的誕生背景是明清時期瘟疫頻發——吳又可的「戾氣」說比西方細菌學說早了兩百年。
