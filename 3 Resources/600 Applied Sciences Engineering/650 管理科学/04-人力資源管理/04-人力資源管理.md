# 04 — 人力資源管理

> 人力資源管理涵蓋員工在組織中的整個生命週期——從招聘到離職，從培訓到績效，從薪酬到員工關係。
> HRM covers the full employee lifecycle — from hiring to departure, from training to performance, from compensation to relations.

---

## 員工生命週期

```
招聘 → 入職 → 培訓發展 → 績效管理 → 薪酬獎勵 → 留任/離職
```

---

## 招聘與甄選

| 步驟 | 說明 |
|------|------|
| 人力規劃 | 預測需求、工作分析、職位描述 |
| 招募 | 內部公告、招聘網站、獵頭、社交媒體 |
| 甄選 | 履歷篩選、測驗、面試、背景調查 |
| 錄用 | Offer、薪資談判、入職安排 |

| 甄選工具 | 效度 |
|---------|------|
| 結構化面試 | 高 |
| 工作抽樣 | 最高 |
| 認知能力測驗 | 中高 |
| 非結構化面試 | 低 |

---

## 培訓與發展

| 方法 | 說明 |
|------|------|
| **在職培訓 (OJT)** | 邊做邊學，最常見 |
| **課堂培訓** | 正式課程、研討會 |
| **導師制** | 資深員工指導 |
| **輪調** | 跨部門經驗 |
| **線上學習** | e-learning 平台 |
| **教練 (Coaching)** | 一對一發展對話 |

---

## 績效管理

| 方法 | 說明 |
|------|------|
| **MBO** (目標管理) | 設定具體可衡量目標 |
| **360 度回饋** | 上司+同事+下屬+自評 |
| **KPI** | 關鍵績效指標 |
| **OKR** | 目標與關鍵結果 |
| **BSC** (平衡計分卡) | 財務+客戶+流程+學習 |

---

## 薪酬與福利

| 類型 | 項目 |
|------|------|
| **直接財務** | 本薪、獎金、佣金、股票選擇權 |
| **間接財務** | 保險、退休金、帶薪休假 |
| **非財務** | 認可、發展機會、工作環境 |

### 薪酬設計原則

- **內部公平**：同工同酬
- **外部競爭**：市場薪資水準
- **績效連結**：貢獻越大回報越高
