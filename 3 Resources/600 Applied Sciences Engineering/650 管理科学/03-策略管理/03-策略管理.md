# 03 — 策略管理

> 策略管理是制定、執行和評估跨功能決策的藝術與科學，使組織能夠達成其長期目標。
> Strategic management is the art and science of formulating, implementing, and evaluating cross-functional decisions that enable an organization to achieve its long-term objectives.

---

## 策略規劃流程

```
外部環境分析 → 內部資源分析 → 策略制定 → 策略執行 → 策略評估與控制
     ↓               ↓
  機會/威脅        優勢/劣勢
     └──────┬──────┘
           SWOT
```

---

## 外部環境分析

### PESTEL 分析

| 構面 | 分析內容 |
|------|---------|
| **P**olitical | 政策、法規、稅收、貿易 |
| **E**conomic | GDP、利率、通膨、匯率 |
| **S**ocial | 人口、文化、消費趨勢 |
| **T**echnological | 新技術、數位化、自動化 |
| **E**nvironmental | 氣候、永續、環保法規 |
| **L**egal | 勞動法、智慧財產權 |

### Porter 五力分析

| 力量 | 問題 |
|------|------|
| **現有競爭者** | 產業內競爭強度如何？ |
| **潛在進入者** | 新對手進入的門檻多高？ |
| **替代品** | 客戶可轉用什麼替代方案？ |
| **供應商議價力** | 供應商有多大的定價權？ |
| **買方議價力** | 客戶有多大的議價空間？ |

---

## 內部資源分析

### VRIO 框架

| 標準 | 問題 |
|------|------|
| **V**aluable | 資源是否創造價值？ |
| **R**are | 資源是否稀有？ |
| **I**nimitability | 資源是否難以模仿？ |
| **O**rganized | 組織是否善用此資源？ |

### 核心能力 (Core Competence)

- 組織獨特的能力組合
- 競爭對手難以複製
- 為客戶創造獨特價值

---

## 策略層級

| 層級 | 問題 | 範例 |
|------|------|------|
| **公司層級** | 我們應該在哪些產業競爭？ | 多角化、購併、垂直整合 |
| **事業層級** | 如何在特定市場競爭？ | 成本領導、差異化、集中化 |
| **功能層級** | 各部門如何支援策略？ | 行銷、研發、人資策略 |

---

## Porter 一般性策略

| 策略 | 說明 | 範例 |
|------|------|------|
| **成本領導** | 以最低成本提供產品/服務 | Walmart, 比亞迪 |
| **差異化** | 提供獨特價值的產品 | Apple, Dyson |
| **集中化** | 專注特定市場區隔 | Rolex, Ferrari |

---

## 藍海策略

| 紅海 | 藍海 |
|------|------|
| 在既有市場競爭 | 創造新市場空間 |
| 擊敗競爭者 | 讓競爭變得無關 |
| 開發現有需求 | 創造新需求 |
| 價值-成本取捨 | 同時追求差異化與低成本 |

### 四項行動框架

```
消除 ───→ 提升
  │         │
  └── 降低 ←── 創造
```

---

> 💡 策略的本質是「選擇不做什麼」——取捨是策略的核心。Strategy is about choosing what NOT to do.
