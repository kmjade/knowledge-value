# 02 — 組織行為

> 組織行為研究個體、群體和組織結構對組織內部行為的影響，目的是提升組織效能。
> OB studies the impact of individuals, groups, and structure on behavior within organizations.

---

## 分析層次

```
個體層次 → 群體層次 → 組織層次
  人格         團隊動態       組織文化
  動機         溝通            組織變革
  態度         衝突            權力與政治
  知覺         領導            組織設計
```

---

## 個體層次：動機理論

| 理論 | 核心思想 |
|------|---------|
| **Maslow 需求層次** | 生理→安全→歸屬→尊重→自我實現 |
| **Herzberg 雙因子** | 保健因子（不滿）vs 激勵因子（滿意） |
| **McClelland 三需求** | 成就、權力、歸屬 |
| **期望理論 (Vroom)** | 動機 = 期望 × 工具性 × 價值 |
| **公平理論 (Adams)** | 人們比較投入/產出比 |
| **自我決定理論 (Deci & Ryan)** | 自主性、勝任感、關聯性 → 內在動機 |

---

## 個體層次：人格特質

| 模型 | 構面 |
|------|------|
| **Big Five (OCEAN)** | 開放性、盡責性、外向性、親和性、情緒穩定性 |
| **MBTI** | 外向/內向、實感/直覺、思考/情感、判斷/感知 |
| **黑暗三角** | 自戀、馬基維利主義、精神病態 |

---

## 群體層次

| 概念 | 說明 |
|------|------|
| **團隊發展階段 (Tuckman)** | Forming → Storming → Norming → Performing → Adjourning |
| **社會惰化** | 個人在群體中付出較少努力的傾向 |
| **群體思維** | 追求共識導致非理性決策 |
| **團隊凝聚力** | 成員間的吸引力與歸屬感 |

---

## 組織層次

| 概念 | 說明 |
|------|------|
| **組織文化** | 共享的價值觀、信念與行為規範 |
| **組織變革 (Kotter 八步)** | 緊迫感→聯盟→願景→溝通→授權→短期勝利→鞏固→制度化 |
| **組織政治** | 非正式的影響力運用 |
| **組織設計** | 功能式、事業部式、矩陣式、網絡式 |

---

> 💡 管理的核心挑戰不是「做什麼」，而是「人為什麼這樣做」。理解組織行為 = 理解人心。
