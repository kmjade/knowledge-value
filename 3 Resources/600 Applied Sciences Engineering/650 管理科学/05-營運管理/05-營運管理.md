# 05 — 營運管理

> 營運管理關注將輸入轉化為輸出的過程——以最有效率的方式生產產品和提供服務。

## 營運管理核心

| 要素 | 說明 |
|------|------|
| **品質** | 符合規格，滿足客戶期望 |
| **速度** | 從訂單到交貨的時間 |
| **彈性** | 適應需求變化的能力 |
| **成本** | 生產與交付的總成本 |
| **可靠性** | 準時交貨的承諾 |

## 供應鏈管理

供應商 → 製造 → 倉儲 → 配送 → 零售 → 客戶

| 策略 | 說明 |
|------|------|
| **垂直整合** | 自行掌控供應鏈上下游 |
| **外包** | 委託第三方處理非核心業務 |
| **JIT** | 零庫存，按需生產 |
| **安全庫存** | 保留緩衝庫存應對波動 |

## 精益生產 (Lean)

| 七大浪費 | 說明 |
|---------|------|
| 運輸 | 不必要的移動 |
| 庫存 | 過多存貨 |
| 動作 | 多餘的人員動作 |
| 等待 | 閒置時間 |
| 過度加工 | 超出需求的加工 |
| 過度生產 | 生產超過需求 |
| 缺陷 | 不良品與重工 |

## 流程改善工具

| 工具 | 用途 |
|------|------|
| 價值流圖 (VSM) | 視覺化流程 |
| 5S | 整理、整頓、清掃、清潔、素養 |
| 看板 (Kanban) | 拉動式生產控制 |
| PDCA | Plan-Do-Check-Act 循環 |

> 💡 營運管理的核心是「消除浪費，創造價值」——每一分不必要的成本都是競爭劣勢。
