# 09 — 品質管理

> 品質是「符合要求」而非「高級」——品質管理確保產品和服務持續滿足客戶期望。

## 品質管理演進

| 階段 | 焦點 |
|------|------|
| **品質檢驗** | 產品完成後檢查 |
| **品質控制** | 製程中統計監控 |
| **品質保證** | 系統性預防設計 |
| **全面品質管理 (TQM)** | 全組織全員參與 |

## 品質大師

| 大師 | 貢獻 |
|------|------|
| **Deming** | PDCA 循環、14 點管理原則 |
| **Juran** | 品質三部曲、80/20 法則 |
| **Crosby** | 零缺陷、品質是免費的 |
| **Ishikawa** | 魚骨圖、品質圈 |
| **Taguchi** | 品質損失函數、穩健設計 |

## TQM 核心原則

| 原則 | 說明 |
|------|------|
| 客戶導向 | 品質由客戶定義 |
| 全員參與 | 品質是每個人的責任 |
| 流程導向 | 改善流程而非責怪個人 |
| 持續改善 (Kaizen) | 小步快跑，持續進步 |
| 事實決策 | 基於資料而非直覺 |

## 六標準差 (Six Sigma)

| 等級 | DPMO | 良率 |
|------|------|------|
| 1σ | 690,000 | 31% |
| 3σ | 66,807 | 93.3% |
| **6σ** | **3.4** | **99.99966%** |

## ISO 9001 七大原則

1. 客戶焦點
2. 領導力
3. 全員參與
4. 流程方法
5. 改善
6. 證據決策
7. 關係管理

> 💡 品質不是檢查出來的——是設計和流程的結果。
