# 02 — 反應工程

> 反應工程研究化學反應器的設計、分析和最佳化。

## 反應器類型

| 類型 | 特點 | 應用 |
|------|------|------|
| 批次反應器 | 一次性投料 | 小批量、多品種 |
| CSTR | 連續攪拌、穩定出口 | 液相反應 |
| PFR | 管式、濃度沿管變化 | 氣相反應 |
| 固定床 | 催化劑填充 | 大規模氣固反應 |
| 流化床 | 固體顆粒懸浮 | 催化裂化 |

## 關鍵概念

| 概念 | 說明 |
|------|------|
| 停留時間 | 物料在反應器中的平均時間 |
| 轉化率 | 反應物轉化為產物的比例 |
| 選擇性 | 目標產物/所有產物 |
| 產率 | 轉化率 × 選擇性 |
| 催化劑 | 降低活化能，不改變平衡 |

> 💡 反應器設計的目標：最大化產率，最小化副產物和能耗。
