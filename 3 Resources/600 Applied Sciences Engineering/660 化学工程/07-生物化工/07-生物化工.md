# 07 — 生物化工

> 生物化工利用微生物、酶或細胞進行工業生產——從發酵到生物精煉。

## 主要領域

| 領域 | 產品 |
|------|------|
| 發酵工程 | 酒精、胺基酸、有機酸 |
| 抗生素生產 | 青黴素、紅黴素 |
| 酶工程 | 工業酶、診斷酶 |
| 生物燃料 | 乙醇、生物柴油 |
| 生物塑膠 | PLA、PHA |
| 合成生物學 | 基因工程菌生產化學品 |

## 發酵過程

```
菌種 → 種子培養 → 發酵罐 → 分離純化 → 產品
```

> 💡 生物化工是綠色化工的未來——用再生原料替代石化原料，用生物催化替代化學催化。
