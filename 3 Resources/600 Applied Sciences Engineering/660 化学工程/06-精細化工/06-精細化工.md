# 06 — 精細化工

> 精細化工生產高附加值、小批量的化學品——醫藥、農藥、染料、香料。

## 主要領域

| 領域 | 產品 |
|------|------|
| 醫藥中間體 | 原料藥、中間體 |
| 農藥 | 殺蟲劑、除草劑 |
| 染料與顏料 | 紡織染料、塗料顏料 |
| 香料與香精 | 日化香精、食品香料 |
| 介面活性劑 | 洗滌劑、乳化劑 |
| 電子化學品 | 光阻劑、蝕刻液 |

## 特點

| 特點 | 說明 |
|------|------|
| 小批量 | 年產量比大宗化工低幾個數量級 |
| 高附加值 | 單位重量價值高 |
| 技術密集 | 合成路徑複雜 |
| 品種多 | 市場需求多樣化 |

> 💡 精細化工的門檻不在於規模，而在於合成技術和純度控制。
