# 05 — 石油化工

> 石油化工將原油轉化為燃料和化學品——現代化學工業的基礎。

## 煉油流程

```
原油 → 常壓蒸餾 → 石腦油/柴油/重油
                      ↓
             催化裂化 → 乙烯、丙烯
             重整     → 芳烴（苯、甲苯、二甲苯）
```

## 主要產品鏈

| 原料 | 中間體 | 最終產品 |
|------|--------|---------|
| 乙烯 → | 聚乙烯、環氧乙烷 | 塑膠、防凍液 |
| 丙烯 → | 聚丙烯、丙烯腈 | 塑膠、腈綸 |
| 苯 → | 苯乙烯、環己烷 | PS、尼龍 |
| 二甲苯 → | 對苯二甲酸 | PET |

> 💡 石油化工是現代文明的基石——我們生活中 90% 以上的化學品都直接或間接來自石油。
