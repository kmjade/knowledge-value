# 04 — 高分子工程

> 高分子工程涵蓋塑膠、橡膠、纖維的生產——現代生活中無所不在的材料。

## 主要高分子

| 材料 | 單體 | 用途 |
|------|------|------|
| 聚乙烯 PE | 乙烯 | 包裝袋、管材 |
| 聚丙烯 PP | 丙烯 | 汽車零件、容器 |
| 聚氯乙烯 PVC | 氯乙烯 | 管材、窗框 |
| 聚苯乙烯 PS | 苯乙烯 | 包裝、絕熱 |
| 聚酯 PET | 對苯二甲酸+乙二醇 | 飲料瓶、纖維 |
| 聚氨酯 PU | 異氰酸酯+多元醇 | 泡棉、塗料 |
| 尼龍 PA | 己內醯胺 | 纖維、工程塑料 |

## 聚合方法

| 方法 | 說明 |
|------|------|
| 加成聚合 | 雙鍵打開連接（PE、PP、PVC） |
| 縮合聚合 | 脫去小分子連接（PET、尼龍） |
| 乳液聚合 | 單體分散在水中聚合 |

> 💡 高分子是 20 世紀最重要的材料革命——從包裝到航太，無處不在。
