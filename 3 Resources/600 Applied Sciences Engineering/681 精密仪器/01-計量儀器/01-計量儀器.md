# 01 — 計量儀器

> 計量儀器是測量的基礎——長度、品質、時間等基本物理量的標準化測量設備。

## 基本量與儀器

| 基本量 | 儀器 | 精度 |
|--------|------|------|
| 長度 | 雷射干涉儀 | nm 級 |
| 品質 | 電子天平 | μg 級 |
| 時間 | 原子鐘 | 10⁻¹⁵ |
| 溫度 | 鉑電阻溫度計 | mK 級 |
| 電流 | 量子電壓標準 | — |

## 三次元量測儀 (CMM)

| 類型 | 特點 |
|------|------|
| 橋式 | 最常見，高精度 |
| 龍門式 | 大型工件 |
| 水平臂 | 汽車車身 |
| 關節臂 | 可攜式、現場使用 |

> 💡 計量儀器是工業品質的基礎——沒有精確的測量，就沒有可靠的製造。
