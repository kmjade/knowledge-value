# 09 — 校準與標準

## 計量追溯鏈

```
國際標準 (SI) → 國家標準 → 實驗室標準 → 工作標準 → 現場儀器
```

## 校準概念

| 概念 | 說明 |
|------|------|
| 校準 | 與標準比對，確定誤差 |
| 檢定 | 法定判斷合格與否 |
| 調整 | 修正儀器偏差 |
| 不確定度 | 量測結果的可信範圍 |

## 常見標準

| 標準 | 範圍 |
|------|------|
| ISO 17025 | 實驗室能力 |
| ISO 9001 | 品質管理 |
| ISO 13485 | 醫療器械品質 |

> 💡 沒有追溯就沒有信任——校準確保全世界的量測結果可以互相比較。
