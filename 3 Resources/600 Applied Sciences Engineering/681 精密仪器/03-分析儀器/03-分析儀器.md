# 03 — 分析儀器

## 光譜分析

| 儀器 | 原理 | 用途 |
|------|------|------|
| UV-Vis | 紫外可見吸收 | 定量分析 |
| FTIR | 紅外吸收 | 官能團鑑定 |
| Raman | 散射光譜 | 分子結構 |
| XRF | X射線螢光 | 元素分析 |
| ICP-OES/MS | 電漿發射/質譜 | 微量元素 |

## 色譜分析

| 儀器 | 原理 | 用途 |
|------|------|------|
| GC | 氣相色譜 | 揮發性物質 |
| HPLC | 高效液相 | 非揮發性物質 |
| IC | 離子色譜 | 離子分析 |

## 質譜 (MS)

| 類型 | 特點 |
|------|------|
| Q-MS | 四極桿，最常用 |
| TOF-MS | 高解析度 |
| Orbitrap | 超高解析度 |
| MALDI-TOF | 大分子分析 |

> 💡 現代分析儀器的核心趨勢：更小、更快、更靈敏、更智慧。
