# 05 — 醫療儀器

## 診斷設備

| 設備 | 原理 | 用途 |
|------|------|------|
| X-Ray | 放射線穿透 | 骨骼、肺部 |
| CT | 旋轉掃描+重建 | 全身斷層 |
| MRI | 核磁共振 | 軟組織 |
| 超音波 | 聲波反射 | 腹部、心臟 |
| 內視鏡 | 光纖成像 | 腸胃道 |
| 心電圖 ECG | 心電訊號 | 心臟 |

## 監護設備

| 設備 | 監測參數 |
|------|---------|
| 病患監護儀 | ECG、SpO₂、血壓、呼吸 |
| 血糖儀 | 血糖 |
| 血氧儀 | SpO₂ |
| 呼吸機 | 輔助/控制呼吸 |

> 💡 醫療儀器的核心要求是安全性和可靠性——故障可能危及生命。
