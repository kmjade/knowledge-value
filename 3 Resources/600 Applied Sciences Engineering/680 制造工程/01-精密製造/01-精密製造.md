# 01 — 精密製造

> 精密製造追求微米級甚至奈米級的加工精度——是現代高科技產業的基礎。

## 核心技術

| 技術 | 精度 | 應用 |
|------|------|------|
| CNC 精密加工 | μm 級 | 航太、醫療器械 |
| 放電加工 (EDM) | μm 級 | 模具製造 |
| 雷射加工 | μm 級 | 切割、焊接、標記 |
| 超精密研磨 | nm 級 | 光學鏡片、晶圓 |
| 微機電系統 (MEMS) | μm–nm | 感測器、致動器 |
| 奈米加工 | nm 級 | 半導體、量子元件 |

## 關鍵要素

| 要素 | 說明 |
|------|------|
| 工具機精度 | 機台本身的幾何精度 |
| 環境控制 | 恆溫、防震、無塵 |
| 刀具技術 | 超硬材料、塗層技術 |
| 測量技術 | 雷射干涉儀、三次元量測 |

> 💡 精密製造是工業的「皇冠」——一個國家的精密製造水準決定了其高端製造業的競爭力。
