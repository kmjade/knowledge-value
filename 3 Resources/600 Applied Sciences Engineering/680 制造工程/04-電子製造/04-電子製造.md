# 04 — 電子製造

> 電子製造涵蓋從 PCB 到半導體封裝——現代電子設備的生產基礎。

## 核心工藝

| 工藝 | 說明 |
|------|------|
| PCB 製造 | 蝕刻、鍍銅、多層壓合 |
| SMT 組裝 | 表面貼裝技術、回流焊 |
| 半導體封裝 | 打線、覆晶、扇出型封裝 |
| 測試 | ICT、FCT、老化測試 |

## 半導體製造流程

```
晶圓 → 光刻 → 蝕刻 → 摻雜 → 金屬化 → 測試 → 切割 → 封裝 → 終測
```

| 環節 | 關鍵技術 |
|------|---------|
| 光刻 | EUV/DUV、光罩 |
| 蝕刻 | 乾式蝕刻、濕式蝕刻 |
| 封裝 | Flip-chip、TSV、Chiplet |

> 💡 電子製造是現代工業的基石——從手機到汽車，從家電到衛星，都離不開電子製造。
