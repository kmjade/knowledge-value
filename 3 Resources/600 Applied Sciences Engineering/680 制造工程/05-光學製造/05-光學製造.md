# 05 — 光學製造

> 光學製造涵蓋鏡頭、光纖、雷射等精密光學元件的生產。

## 主要產品

| 產品 | 關鍵工藝 |
|------|---------|
| 光學鏡片 | 研磨、拋光、鍍膜 |
| 光纖 | 預製棒→拉絲→塗覆 |
| 雷射器 | 晶體生長、光學鍍膜、封裝 |
| 光學薄膜 | 蒸鍍、濺鍍 |

## 鏡頭製造流程

```
玻璃熔煉 → 壓型 → 粗磨 → 精磨 → 拋光 → 定心 → 鍍膜 → 組裝
```

## 應用領域

| 領域 | 光學元件 |
|------|---------|
| 手機 | 微型鏡頭模組 |
| 相機 | 交換鏡頭 |
| 醫療 | 內視鏡、顯微鏡 |
| 半導體 | 光刻物鏡 |
| 航太 | 衛星光學 |

> 💡 手機鏡頭是光學製造的奇蹟——一個指甲蓋大小的模組中包含 5–7 片精密鏡片，每片精度達奈米級。
