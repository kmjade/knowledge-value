# 03 — 機械製造

> 機械製造涵蓋引擎、工業機械、動力設備的生產——工業的心臟。

## 主要領域

| 領域 | 內容 |
|------|------|
| 引擎製造 | 內燃機、渦輪機、電動馬達 |
| 工業機械 | 工具機、紡織機械、食品機械 |
| 動力傳動 | 齒輪、軸承、液壓系統 |
| 重型設備 | 工程機械、礦山設備 |

## 引擎製造

| 類型 | 關鍵工藝 |
|------|---------|
| 內燃機 | 鑄造缸體、精密搪缸、曲軸加工 |
| 燃氣渦輪 | 單晶葉片鑄造、熱障塗層 |
| 電動馬達 | 繞線、磁鋼裝配、動平衡 |

> 💡 引擎製造是機械製造的巔峰——一台現代汽車引擎包含數百個精密零件，每個都必須在極端溫度和壓力下完美配合。
