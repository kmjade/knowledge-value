# 09 — 工具與工坊

> 工欲善其事，必先利其器——好的工具和安全的環境是手工的基礎。

## 工坊設置三層次

### 初級：角落工坊（0 成本）
餐桌 + 收納箱 = 臨時工坊
— 鋪上切割墊保護桌面
— 做完收拾，不佔空間

### 中級：陽台/書房（¥500–2000）
一張工作桌 + 牆面收納 + 基礎照明

### 高級：獨立工坊
通風系統 + 集塵 + 專用工具櫃
（適合木工、金工等粉塵/氣味較多的手工）

## 安全守則

| 規則 | 說明 |
|------|------|
| 護目鏡 | 切割、打磨時必戴 |
| 防塵口罩 | 打磨、噴漆時 |
| 通風 | 膠水、溶劑、焊接 |
| 整潔 | 工具用完歸位 |
| 急救包 | 必備創可貼和消毒用品 |

> 💡 工坊不在大小，在於你願意花時間在那裡。一個角落、一張桌、一盞燈，就足夠開始。
