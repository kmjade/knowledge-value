# 04 — 業餘金工首飾

> 自己做首飾的滿足感是買不到的——親手打磨的戒指比珠寶店的更有意義。

## 必備工具（入門約 ¥500–800）

| 工具 | 用途 |
|------|------|
| 銼刀組 | 打磨金屬 |
| 焊槍（丁烷） | 焊接 |
| 焊藥+助焊劑 | 焊接材料 |
| 戒指棒+膠錘 | 戒指成型 |
| 拋光機/拋光布 | 表面處理 |
| 防護眼鏡 | 安全 |

## 常用金屬

| 金屬 | 硬度 | 價格 | 適合新手 |
|------|------|------|---------|
| 銅 | 軟 | 低 | ✅ |
| 黃銅 | 中 | 低 | ✅ |
| 純銀 | 軟 | 中 | ✅ |
| 不鏽鋼 | 硬 | 低 | ❌（太硬） |

## 新手專案（建議從銅開始）

1. **簡單戒指**（2hr）— 銅片彎曲成型 + 焊接 + 拋光
2. **刻字吊墜**（1hr）— 金屬片 + 鋼印敲字
3. **銀戒指**（3hr）— 進階

## 安全第一

- 焊接時戴防護眼鏡
- 在通風處操作
- 金屬加熱後極燙——等冷卻

> 💡 金屬焊接是金工的核心——看到焊藥融化流動的那一刻，你會感到真正的工匠成就感。
