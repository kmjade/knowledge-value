# 08 — 業餘電子DIY

> 電子DIY讓你有超能力——讓燈泡聽你的話、讓馬達轉起來、讓感測器感知世界。

## 入門平台

| 平台 | 難度 | 適合 |
|------|------|------|
| Arduino | ⭐ | 最推薦入門 |
| ESP32 | ⭐⭐ | WiFi/藍牙專案 |
| Raspberry Pi | ⭐⭐⭐ | 完整電腦 |

## 必備工具（入門約 ¥200–500）

| 工具 | 用途 |
|------|------|
| Arduino UNO 開發板 | 核心 |
| 麵包板+杜邦線 | 免焊接原型 |
| LED、電阻套件 | 基本元件 |
| 三用電錶 | 量測 |
| USB 線 | 供電+編程 |

## 新手專案

1. **閃爍LED**（10min）— 第一個程式
2. **交通燈**（30min）— 3個LED循環
3. **溫濕度顯示**（1hr）— DHT11感測器 + LCD

## 焊接基礎

| 步驟 | 說明 |
|------|------|
| 加熱焊點 | 烙鐵接觸焊盤和元件腳 |
| 送入焊錫 | 焊錫融化流動 |
| 移開 | 先移焊錫，再移烙鐵 |
| 檢查 | 光亮圓錐形 = 好焊點 |

> 💡 電子DIY最好的部分是「它真的動了！」——當你寫的程式讓 LED 開始閃爍，那種興奮是買不到的。Start with blinking an LED.
