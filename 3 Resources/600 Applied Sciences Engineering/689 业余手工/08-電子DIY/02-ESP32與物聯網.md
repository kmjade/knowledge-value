# 02 ESP32 與物聯網 ESP32 & IoT

> Arduino 能控制世界，ESP32 能讓世界互聯。

## 為什麼選 ESP32？

| 特性 | Arduino UNO | ESP32 |
|------|-----------|-------|
| 處理器 | ATmega328P 16MHz | Xtensa LX6 雙核 240MHz |
| RAM | 2KB | 520KB |
| Flash | 32KB | 4–16MB |
| WiFi | ❌ | ✅ 802.11 b/g/n |
| 藍牙 | ❌ | ✅ BLE 4.2 + Classic |
| GPIO | 14 | 34 |
| ADC | 10-bit | 12-bit |
| 價格 | ¥15–30 | ¥20–40 |

## 開發環境

### PlatformIO (VS Code) — 推薦

```bash
# 安裝 PlatformIO CLI
pip install platformio

# 建立 ESP32 專案
pio init --board esp32dev
```

### Arduino IDE

1. 檔案 → 偏好設定 → 附加開發板管理員網址
2. 貼上：`https://espressif.github.io/arduino-esp32/package_esp32_index.json`
3. 工具 → 開發板 → 開發板管理員 → 搜尋 `ESP32` → 安裝

### MicroPython

```bash
# 安裝 esptool
pip install esptool

# 燒錄 MicroPython 韌體
esptool.py --port /dev/ttyUSB0 erase_flash
esptool.py --port /dev/ttyUSB0 --baud 460800 write_flash -z 0x1000 firmware.bin
```

## WiFi 基礎

### 連接 WiFi (Arduino framework)

```cpp
#include <WiFi.h>

const char* ssid = "你的WiFi名稱";
const char* password = "你的密碼";

void setup() {
  Serial.begin(115200);
  WiFi.begin(ssid, password);
  
  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }
  
  Serial.println("\n已連接！IP: " + WiFi.localIP().toString());
}
```

### 建立 Web Server

```cpp
#include <WiFi.h>
#include <WebServer.h>

WebServer server(80);

void handleRoot() {
  server.send(200, "text/html", "<h1>Hello from ESP32!</h1>");
}

void setup() {
  // ... WiFi 連接 ...
  server.on("/", handleRoot);
  server.begin();
}

void loop() {
  server.handleClient();
}
```

## 藍牙 BLE

### BLE Server 範例

```cpp
#include <BLEDevice.h>
#include <BLEServer.h>
#include <BLEUtils.h>

#define SERVICE_UUID "4fafc201-1fb5-459e-8fcc-c5c9c331914b"
#define CHARACTERISTIC_UUID "beb5483e-36e1-4688-b7f5-ea07361b26a8"

void setup() {
  BLEDevice::init("ESP32_BLE");
  BLEServer *pServer = BLEDevice::createServer();
  BLEService *pService = pServer->createService(SERVICE_UUID);
  BLECharacteristic *pCharacteristic = pService->createCharacteristic(
    CHARACTERISTIC_UUID,
    BLECharacteristic::PROPERTY_READ | BLECharacteristic::PROPERTY_WRITE
  );
  pCharacteristic->setValue("Hello BLE!");
  pService->start();
  BLEAdvertising *pAdvertising = BLEDevice::getAdvertising();
  pAdvertising->start();
}
```

## IoT 協定

| 協定 | 特點 | 適用場景 |
|------|------|---------|
| **MQTT** | 輕量、發布/訂閱、QoS | 感測器數據上傳、遠端控制 |
| **HTTP** | 簡單、廣泛支援 | REST API、簡易Web控制 |
| **WebSocket** | 全雙工、低延遲 | 即時控制面板 |
| **CoAP** | 比MQTT更輕量、UDP | 極低功耗設備 |
| **LoRaWAN** | 長距離、低功耗 | 農業、偏遠地區 |

### MQTT 快速上手

```cpp
#include <PubSubClient.h>

WiFiClient espClient;
PubSubClient client(espClient);
const char* mqtt_server = "broker.emqx.io";  // 免費公共 Broker

void callback(char* topic, byte* payload, unsigned int length) {
  Serial.print("收到訊息: ");
  Serial.println(topic);
}

void setup() {
  client.setServer(mqtt_server, 1883);
  client.setCallback(callback);
}

void reconnect() {
  while (!client.connected()) {
    if (client.connect("ESP32_Client")) {
      client.subscribe("esp32/command");
    }
  }
}

void loop() {
  if (!client.connected()) reconnect();
  client.loop();
  client.publish("esp32/temperature", "25.5");
  delay(5000);
}
```

## 腳位注意事項

ES32 的 GPIO 不像 Arduino 那麼單純，有些腳位有特殊功能或限制：

| 腳位 | 限制 |
|------|------|
| GPIO 0 | 燒錄時需拉低（內建按鈕） |
| GPIO 6–11 | 連接內部 Flash，**不可用** |
| GPIO 34–39 | **僅輸入**，無內部上拉/下拉 |
| ADC2 | WiFi 開啟時不可用（用 ADC1：GPIO 32–39） |
