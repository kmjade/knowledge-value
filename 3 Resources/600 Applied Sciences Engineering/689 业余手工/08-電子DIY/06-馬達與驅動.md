# 06 馬達與驅動 Motors & Drivers

> 讓你的專案動起來——從讓一顆輪子轉動到控制機器人手臂。

## 馬達類型總覽

| 類型 | 控制難度 | 精度 | 扭力 | 適合 |
|------|---------|------|------|------|
| 直流馬達 DC Motor | ⭐ | 無 | 中高 | 風扇、車輪 |
| 減速馬達 Gear Motor | ⭐ | 無 | 高 | 機器人底盤 |
| 伺服馬達 Servo | ⭐⭐ | 角度 | 低中 | 機器人關節、轉向 |
| 步進馬達 Stepper | ⭐⭐⭐ | 步距 | 中 | 3D列印機、CNC |
| 無刷馬達 BLDC | ⭐⭐⭐⭐ | 需 ESC | 極高 | 無人機、高速 |

## 直流馬達 + L298N

### L298N 規格

| 參數 | 值 |
|------|-----|
| 驅動電壓 | 5–35V |
| 邏輯電壓 | 5V |
| 最大電流 | 2A/通道（峰值3A） |
| 通道數 | 2 (可驅動2個馬達或1個步進) |
| 價格 | ¥5–10 |

### 接線圖

```
Arduino         L298N           馬達/電源
─────────────────────────────────────────
D5 (PWM)  →  ENA (Enable A)
D6        →  IN1
D7        →  IN2
D9 (PWM)  →  ENB (Enable B)
D8        →  IN3
D10       →  IN4
5V        →  +5V (邏輯供電)
GND       →  GND
            VCC ← 外接電源 (6–12V)
            OUT1/OUT2 → 馬達A
            OUT3/OUT4 → 馬達B
```

### 控制程式

```cpp
// 馬達 A
#define ENA 5
#define IN1 6
#define IN2 7

// 馬達 B
#define ENB 9
#define IN3 8
#define IN4 10

void setup() {
  pinMode(ENA, OUTPUT);
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
  pinMode(ENB, OUTPUT);
  pinMode(IN3, OUTPUT);
  pinMode(IN4, OUTPUT);
}

void motorA_forward(int speed) {
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  analogWrite(ENA, speed);
}

void motorA_backward(int speed) {
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, HIGH);
  analogWrite(ENA, speed);
}

void motorA_stop() {
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, LOW);
}
```

### L298N vs L293D

| | L298N | L293D |
|---|-------|-------|
| 最大電流 | 2A | 0.6A |
| 散熱 | 需散熱片 | 內建 |
| 二極體 | 無（需外接） | 內建 |
| 尺寸 | 大 | 小(DIP) |
| 適合 | 中型馬達 | 微型馬達 |

## 伺服馬達 SG90 / MG996R

### SG90 (微型)

| 參數 | 值 |
|------|-----|
| 扭力 | 1.5 kg·cm |
| 速度 | 0.1s/60° |
| 角度 | 0–180° |
| 電壓 | 4.8–6V |
| 價格 | ¥5–8 |

### 控制原理

伺服馬達用 PWM 脈衝寬度控制角度：

```
脈寬 0.5ms → 0°
脈寬 1.0ms → 90°
脈寬 1.5ms → 180°
```

Arduino `Servo` 庫幫你處理這些細節：

```cpp
#include <Servo.h>

Servo myservo;

void setup() {
  myservo.attach(9);  // D9 接伺服信號線
}

void loop() {
  myservo.write(0);    // 0°
  delay(1000);
  myservo.write(90);   // 90°
  delay(1000);
  myservo.write(180);  // 180°
  delay(1000);
}
```

> ⚠️ 多個伺服馬達不要從 Arduino 5V 供電——用外接 5V/2A+ 電源！

## 步進馬達 28BYJ-48 + ULN2003

| 參數 | 值 |
|------|-----|
| 步距角 | 5.625° (64步/圈) |
| 減速比 | 1:64 |
| 實際步數 | 4096步/圈 |
| 電壓 | 5V |
| 價格 | ¥10–15 (含驅動板) |

```cpp
#include <Stepper.h>

const int stepsPerRevolution = 2048;  // 28BYJ-48 一圈
Stepper myStepper(stepsPerRevolution, 8, 10, 9, 11);

void setup() {
  myStepper.setSpeed(10);  // RPM
}

void loop() {
  myStepper.step(stepsPerRevolution);  // 正轉一圈
  delay(500);
  myStepper.step(-stepsPerRevolution); // 反轉一圈
  delay(500);
}
```

## 供電注意事項

| 問題 | 原因 | 解決 |
|------|------|------|
| 馬達不動/抖動 | Arduino 5V 供電不足 | 外接電源 |
| Arduino 重啟 | 馬達啟動時電壓驟降 | 加電容 (1000μF) 在電源端 |
| 干擾/雜訊 | 馬達電刷火花 | 馬達端加 0.1μF 陶瓷電容 |
| 控制異常 | 回流電流 | 加反向二極體（L298N已內建） |

## 安全提示

> ⚠️ **永遠先把馬達固定在桌面上再通電測試**——一顆通電的小車輪子能瞬間把你的麵包板掃到地上。
