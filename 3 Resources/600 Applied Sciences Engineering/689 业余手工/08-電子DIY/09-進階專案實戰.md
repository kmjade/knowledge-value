# 09 進階專案實戰 Advanced Projects

> 真正的學習來自實戰——用一個完整的專案把學過的技能串聯起來。

## 專案難度分級

| 等級 | 說明 | 時間 |
|------|------|------|
| 🟢 初階 | 單一功能、10行以內 | 30分鐘 |
| 🟡 中階 | 2–3個模組整合 | 2–4小時 |
| 🔴 進階 | 多模組 + 通訊 + 結構 | 1–2天 |
| ⚫ 高手 | RTOS、PID、電腦視覺 | 1–2週 |

## 🟢 初階專案

### 1. 自動感應燈

```
PIR人體感測 + 光敏電阻 + LED/繼電器
→ 有人 + 暗 = 開燈
```

材料：HC-SR501、LDR、LED、電阻 — ¥10

```
DHT11 + 1602 LCD
→ 即時顯示室內溫濕度
```

材料：DHT11、1602 LCD、I2C轉接板 — ¥20

## 🔴 進階專案

### 7. 自平衡小車

```
Arduino/ESP32 + MPU6050 + L298N + 2輪 + 編碼器馬達
→ PID平衡控制、手機遙控
```

核心演算法：

```cpp
// PID 控制器（簡化版）
float Kp = 30, Ki = 2, Kd = 5;
float error, lastError, integral, derivative;
float setpoint = 0;  // 目標角度 = 直立

float computePID(float currentAngle) {
  error = setpoint - currentAngle;
  integral += error;
  derivative = error - lastError;
  lastError = error;
  return Kp * error + Ki * integral + Kd * derivative;
}
```

> 詳細 PID 教學見 [[智能小车/智能小车|🚗 智能小車]] 中的 STM32/ESP32 實現。


## 專案開發流程

```
1. 需求定義   → 這個專案要做什麼？
2. 系統設計   → 方塊圖、選型
3. 原型搭建   → 麵包板測試各模組
4. 軟體開發   → 分模組寫、逐步整合
5. 整合測試   → 全部接在一起，找bug
6. 外殼/PCB   → 焊接、3D列印外殼
7. 文檔記錄   → README、接線圖、BOM表
```

## 專案管理建議

- **一個專案一個資料夾** → 程式碼、電路圖、照片放一起
- **用 Git 管理程式碼** → 改了什麼一目了然
- **拍照記錄每一步** → 麵包板佈局、焊接前後
- **寫 README** → 兩個月後你自己也看不懂當初怎麼接的
- **開源分享** → GitHub / Hackaday / Instructables
