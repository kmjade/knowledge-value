## 二、电机速度环 PID 原理

### 2.1 电机类型

电机的种类特别多，小型机器人（无人机、无人车）上普遍使用直流无刷电机：

| 应用 | 特点 | 示例 |
|------|------|------|
| **航模** | 高速低扭矩 | 无刷电机直驱 |
| **车模** | 低速高扭矩 | 无刷电机 + 减速器 |

### 2.2 大疆 M3508 电机

本文使用大疆的 M3508 电机，配套 C620 电调，使用 CAN 总线进行控制。

**电机特性：**

| 参数 | 值 |
|------|-----|
| **额定电压** | 24V |
| **空载转速** | 462 RPM |
| **额定扭矩** | 0.25 N·m |
| **编码器** | 8192 线增量式 |

### 2.3 CAN 总线简介

CAN 总线是汽车行业的应用标准，稍大型的电机很多是 CAN 总线控制的。

**CAN 总线优点：**

| 优点 | 说明 |
|------|------|
| **速度快** | 最高 1Mbps |
| **传输距离远** | 最长 40m（1Mbps） |
| **连接节点多** | 理论上无限制 |
| **抗干扰强** | 差分信号传输 |
| **错误检测** | 自动错误检测和恢复 |

### 2.4 电机三闭环控制

伺服电机有三种控制模式：

| 模式 | 控制目标 | 外环 | 中环 | 内环 |
|------|----------|------|------|------|
| **位置模式** | 位置 | 位置环 | 速度环 | 电流环 |
| **速度模式** | 速度 | - | 速度环 | 电流环 |
| **转矩模式** | 力矩 | - | - | 电流环 |

**三闭环控制结构：**

```
┌─────────────────────────────────────────────────────────────────────┐
│                       电机三闭环控制结构                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   目标位置 ──→ [位置环PID] ──→ 目标速度                              │
│                                      │                              │
│                                      ▼                              │
│                               [速度环PID] ──→ 目标电流              │
│                                      │                              │
│                                      ▼                              │
│                               [电流环PID] ──→ PWM输出              │
│                                      │                              │
│                                      ▼                              │
│                                   电机执行                          │
│                                      │                              │
│         ┌────────────────────────────┼────────────────────────────┐ │
│         │                            │                            │ │
│         ▼                            ▼                            ▼ │
│     位置反馈                     速度反馈                     电流反馈│
│   (编码器角度)                (编码器速度)                (电流采样) │
└─────────────────────────────────────────────────────────────────────┘
```

> [!important] 关键发现
> 外环的控制变量和内环的控制变量依次成导数关系：
> - 位置的导数 = 速度
> - 速度的导数 = 加速度（电流决定力矩，力矩产生加速度）

### 2.5 大疆 M3508 控制方式

大疆 M3508 电机驱动器只上报电机的速度、电流、转角、温度等信息，**需要控制器做闭环计算**，控制器将计算结果（电流值）传输给驱动器。

**简化后的速度闭环：**

```
目标速度 ──→ [速度环PID] ──→ 目标电流 ──→ CAN发送 ──→ C620电调 ──→ 电机
    ▲                                              │
    │                                              │
    └────────────── CAN反馈 ←─────────────────────┘
                   (速度/电流/角度/温度)
```

## 三、STM32 使用 CAN 总线实现电机速度闭环

### 3.1 PID 结构体定义

```c
// pid.h

typedef struct
{
    float target[3];    // 目标值，包含NOW, LAST, LLAST上上次
    float get[3];       // 测量值
    float err[3];       // 误差
    
    float pout;         // P输出
    float iout;         // I输出
    float dout;         // D输出
    
    float Kp;           // 比例系数
    float Ki;           // 积分系数
    float Kd;           // 微分系数
    
    float input_max_err;   // 输入误差最大值
    float integral_limit;  // 积分限幅
    float output_limit;    // 输出限幅
    
    void (*f_param_init)(struct pid_t *pid, 
                         float maxerr, 
                         float inte_limit, 
                         float output_limit);
    void (*f_pid_reset)(struct pid_t *pid, float kp, float ki, float kd);
    
} pid_t;
```

### 3.2 PID 初始化与计算

```c
// pid.c

// 参数初始化
static void pid_param_init(pid_t *pid, 
                           float maxerr, 
                           float inte_limit, 
                           float output_limit)
{
    pid->input_max_err = maxerr;
    pid->integral_limit = inte_limit;
    pid->output_limit = output_limit;
}

// 参数重置
static void pid_reset(pid_t *pid, float kp, float ki, float kd)
{
    pid->Kp = kp;
    pid->Ki = ki;
    pid->Kd = kd;
}

// 总体初始化
void PID_Init(pid_t *pid, float kp, float ki, float kd, 
              float maxerr, float inte_limit, float output_limit)
{
    pid->Kp = kp;
    pid->Ki = ki;
    pid->Kd = kd;
    pid_param_init(pid, maxerr, inte_limit, output_limit);
}

// PID 计算
float PID_Calc(pid_t *pid, float target, float feedback)
{
    // 更新目标值和测量值
    pid->target[2] = pid->target[1];
    pid->target[1] = pid->target[0];
    pid->target[0] = target;
    
    pid->get[2] = pid->get[1];
    pid->get[1] = pid->get[0];
    pid->get[0] = feedback;
    
    // 计算误差
    pid->err[2] = pid->err[1];
    pid->err[1] = pid->err[0];
    pid->err[0] = pid->target[0] - pid->get[0];
    
    // P 项
    pid->pout = pid->Kp * pid->err[0];
    
    // I 项（积分限幅）
    pid->iout += pid->Ki * pid->err[0];
    if (pid->iout > pid->integral_limit)
        pid->iout = pid->integral_limit;
    else if (pid->iout < -pid->integral_limit)
        pid->iout = -pid->integral_limit;
    
    // D 项
    pid->dout = pid->Kd * (pid->err[0] - pid->err[1]);
    
    // 总输出
    float output = pid->pout + pid->iout + pid->dout;
    
    // 输出限幅
    if (output > pid->output_limit)
        output = pid->output_limit;
    else if (output < -pid->output_limit)
        output = -pid->output_limit;
    
    return output;
}
```

### 3.3 电机数据结构

```c
// motor.h

#define MOTOR_NUM 4  // 电机数量

// 电机参数结构体
typedef struct
{
    uint8_t id;             // 电机ID
    
    int16_t angle;          // 机械角度 0~8191
    int16_t last_angle;     // 上次角度
    int16_t offset_angle;   // 偏移角度
    int16_t total_angle;    // 总角度
    
    int16_t speed_rpm;      // 转速 RPM
    int16_t real_current;   // 实际电流
    int16_t given_current;  // 给定电流
    
    uint8_t temperature;    // 温度
    
    pid_t speed_pid;        // 速度PID
    
} motor_t;

extern motor_t motor[MOTOR_NUM];
```

### 3.4 CAN 接收数据解析

```c
// motor.c

// CAN 接收数据解析
void CAN_GetMotorData(motor_t *ptr, uint8_t *rx_data)
{
    // 解析角度 (0~8191)
    ptr->last_angle = ptr->angle;
    ptr->angle = (rx_data[0] << 8) | rx_data[1];
    
    // 解析转速 RPM
    ptr->speed_rpm = (rx_data[2] << 8) | rx_data[3];
    
    // 解析实际电流
    ptr->real_current = (rx_data[4] << 8) | rx_data[5];
    
    // 解析温度
    ptr->temperature = rx_data[6];
    
    // 计算总角度
    get_total_angle(ptr);
}

// 计算总角度（处理角度跳变）
void get_total_angle(motor_t *p)
{
    int32_t res1, res2;
    
    // 处理 8191 -> 0 或 0 -> 8191 的跳变
    res1 = p->angle - p->last_angle;
    
    if (res1 < -4096)
        res1 += 8192;
    else if (res1 > 4096)
        res1 -= 8192;
    
    // 累加总角度
    p->total_angle += res1;
}
```

### 3.5 CAN 发送控制命令

```c
// CAN 发送电流控制命令
void CAN_SetMotorCurrent(int16_t current1, int16_t current2, 
                         int16_t current3, int16_t current4)
{
    CAN_TxHeaderTypeDef tx_header;
    uint8_t tx_data[8];
    uint32_t tx_mailbox;
    
    // 设置帧头
    tx_header.StdId = 0x200;        // 控制1-4号电机
    tx_header.IDE = CAN_ID_STD;
    tx_header.RTR = CAN_RTR_DATA;
    tx_header.DLC = 8;
    
    // 填充数据
    tx_data[0] = (current1 >> 8) & 0xFF;
    tx_data[1] = current1 & 0xFF;
    tx_data[2] = (current2 >> 8) & 0xFF;
    tx_data[3] = current2 & 0xFF;
    tx_data[4] = (current3 >> 8) & 0xFF;
    tx_data[5] = current3 & 0xFF;
    tx_data[6] = (current4 >> 8) & 0xFF;
    tx_data[7] = current4 & 0xFF;
    
    // 发送
    HAL_CAN_AddTxMessage(&hcan1, &tx_header, tx_data, &tx_mailbox);
}

// 设置单个电机速度
void Motor_SetSpeed(uint8_t motor_id, int16_t target_speed)
{
    motor_t *m = &motor[motor_id - 1];
    
    // PID 计算，输出电流值
    m->given_current = (int16_t)PID_Calc(&m->speed_pid, 
                                          target_speed, 
                                          m->speed_rpm);
}
```

### 3.6 CAN 接收中断回调

```c
// CAN 接收中断回调
void HAL_CAN_RxFifo0MsgPendingCallback(CAN_HandleTypeDef *hcan)
{
    CAN_RxHeaderTypeDef rx_header;
    uint8_t rx_data[8];
    
    if (HAL_CAN_GetRxMessage(hcan, CAN_RX_FIFO0, &rx_header, rx_data) == HAL_OK)
    {
        // 判断是否为电机反馈数据 (ID: 0x201-0x204)
        if (rx_header.StdId >= 0x201 && rx_header.StdId <= 0x204)
        {
            uint8_t motor_index = rx_header.StdId - 0x201;
            CAN_GetMotorData(&motor[motor_index], rx_data);
        }
    }
}
```

## 四、电机控制任务实现

### 4.1 电机初始化

```c
// 电机初始化
void Motor_Init(void)
{
    for (int i = 0; i < MOTOR_NUM; i++)
    {
        motor[i].id = i + 1;
        motor[i].angle = 0;
        motor[i].last_angle = 0;
        motor[i].total_angle = 0;
        motor[i].speed_rpm = 0;
        motor[i].given_current = 0;
        
        // 初始化速度 PID
        PID_Init(&motor[i].speed_pid, 
                 2.0f,    // Kp
                 0.1f,    // Ki
                 0.0f,    // Kd
                 500.0f,  // maxerr
                 5000.0f, // inte_limit
                 16000.0f // output_limit (±16384)
                 );
    }
}
```

### 4.2 电机控制任务

```c
// 电机控制任务
u8 motor_task(void *p_arg)
{
    OS_ERR err;
    CPU_SR_ALLOC();
    
    // 初始化电机
    Motor_Init();
    
    while (1)
    {
        // 设置电机目标速度（从姿态控制任务获取）
        // 左右电机差速
        int16_t left_speed = base_speed - yaw_output;
        int16_t right_speed = base_speed + yaw_output;
        
        // 设置电机速度
        Motor_SetSpeed(1, left_speed);   // 左前
        Motor_SetSpeed(2, left_speed);   // 左后
        Motor_SetSpeed(3, right_speed);  // 右前
        Motor_SetSpeed(4, right_speed);  // 右后
        
        // 发送电流控制命令
        CAN_SetMotorCurrent(motor[0].given_current,
                           motor[1].given_current,
                           motor[2].given_current,
                           motor[3].given_current);
        
        delay_ms(10);  // 100Hz 控制频率
    }
}
```

### 4.3 完整控制流程

```c
// 主控制循环中的数据流
void Control_Loop(void)
{
    // 1. 获取遥控器数据
    int16_t forward = sbus_to_pwm(SBUS_CH.CH3);   // 前进后退
    int16_t turn = sbus_to_pwm(SBUS_CH.CH1);      // 左右转向
    
    // 2. 获取 IMU 数据
    float yaw = imu_data.yaw;
    float yaw_rate = imu_data.yaw_rate;
    
    // 3. 姿态控制（串级 PID）
    float yaw_output = CascadePID_Calculate(&yaw_cascade_pid, 
                                            yaw_target, yaw, yaw_rate);
    
    // 4. 电机速度分配
    float base_speed = map(forward, 1000, 2000, -500, 500);
    float left_speed = base_speed - yaw_output;
    float right_speed = base_speed + yaw_output;
    
    // 5. 电机速度 PID 控制
    Motor_SetSpeed(1, (int16_t)left_speed);
    Motor_SetSpeed(2, (int16_t)left_speed);
    Motor_SetSpeed(3, (int16_t)right_speed);
    Motor_SetSpeed(4, (int16_t)right_speed);
    
    // 6. 发送电流命令
    CAN_SetMotorCurrent(motor[0].given_current,
                       motor[1].given_current,
                       motor[2].given_current,
                       motor[3].given_current);
}
```

## 五、CAN 通信协议

### 5.1 发送控制命令

| 参数 | 值 |
|------|-----|
| **标识符 (ID)** | 0x200（控制 1-4 号电机） |
| **帧类型** | 标准帧 |
| **DLC** | 8 字节 |

**数据格式：**

| 字节 | 内容 |
|------|------|
| Byte 0-1 | 电机1电流（高8位 + 低8位） |
| Byte 2-3 | 电机2电流 |
| Byte 4-5 | 电机3电流 |
| Byte 6-7 | 电机4电流 |

**电流范围：** -16384 ~ 16384，对应 -20A ~ 20A

### 5.2 接收反馈数据

| 参数 | 值 |
|------|-----|
| **标识符 (ID)** | 0x201 ~ 0x204（对应电机编号） |
| **帧类型** | 标准帧 |
| **DLC** | 8 字节 |

**数据格式：**

| 字节 | 内容 |
|------|------|
| Byte 0-1 | 机械角度（0~8191） |
| Byte 2-3 | 转速 RPM |
| Byte 4-5 | 实际电流 |
| Byte 6 | 温度 |
| Byte 7 | 保留 |

## 七、PID 参数整定

### 7.1 速度环 PID 整定步骤

```
步骤1: 设置 Kp
   ├── 从小值开始（如 0.5）
   ├── 逐渐增大直到电机开始振荡
   └── 回退到振荡前的值

步骤2: 设置 Ki
   ├── 从小值开始（如 0.05）
   ├── 观察静差是否消除
   └── 增大直到消除静差

步骤3: 设置 Kd（可选）
   ├── 一般不需要
   └── 噪声大时会放大干扰
```

### 7.2 参考参数

| 电机 | Kp | Ki | Kd |
|------|-----|-----|-----|
| **M3508** | 1.5 ~ 3.0 | 0.05 ~ 0.2 | 0 |
| **M2006** | 5.0 ~ 10.0 | 0.1 ~ 0.5 | 0 |

## 八、调试技巧

### 8.1 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 电机不转 | CAN 通信失败 | 检查 CAN 连接和波特率 |

| 有静差 | Ki 过小 | 增大 Ki |
