# MOC-學習路徑

> [!info] 概述
> RoboMaster EP 機器人系統化學習路徑，從環境搭建到多機編隊控制。

---

## 學習階段總覽

```
第一階段：環境搭建（1週）
    ↓
第二階段：SDK 基礎（2週）
    ↓
第三階段：EP 機器人控制（2-3週）
    ↓
第四階段：拓展模塊（2週）
    ↓
第五階段：多機控制（2-3週）
```

## 學習建議

1. **動手實踐**：每個接口都要親自測試
2. **查閱 API**：遇到問題查閱 API 文檔
3. **項目驅動**：設計小項目綜合練習
4. **記錄筆記**：記錄常用代碼和問題解決方案
