# 2. 编程环境安装

> [!info] 本章概述
> 介绍如何在 Windows 和 Ubuntu 系统上安装 Python 开发环境，为使用 RoboMaster SDK 做准备。

---

## 2.1 介绍

### 连接方式

用户在 PC 上通过以下方式与 EP 建立连接后，可以使用明文 SDK 跟 EP 进行通信：

| 连接方式 | 说明 |
|----------|------|
| **WIFI** | 无线连接，灵活方便 |
| **USB** | 有线连接，稳定可靠 |
| **UART** | 串口通信，适合嵌入式开发 |

### 开发语言选择

用户可在 PC 上使用多种语言进行编程：

| 语言 | 适用场景 |
|------|----------|
| **Python** | 快速开发、学习入门、SDK 支持 |
| **C++** | 高性能应用、嵌入式开发 |
| **C#** | Windows 应用开发 |
| **其他** | 根据开发能力选择 |

> [!tip] 建议
> 为方便学习本网站的 Python 示例代码，建议先安装 Python 环境。

---

## 2.2 在 Windows 上安装 Python

### 环境要求

| 项目 | 要求 |
|------|------|
| **操作系统** | Windows 10 64 位 |
| **Python 版本** | 3.6.6 ~ 3.8.9（推荐 3.7.8） |

### 安装步骤

#### 步骤 1：下载安装包

从 Python 官网下载安装包：[Python 官网下载页面](https://www.python.org/downloads/windows/)

> [!warning] 重要提示
> - 请确保下载的 `python.exe` 是 **64 位**版本
> - Python SDK 适配 **3.6.6 以上至 3.8.9** 版本
> - 否则会影响 Python SDK 的使用

![](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/win_python_setup1.png)

---

#### 步骤 2：安装配置

运行安装程序时，请注意以下三个步骤：

| 步骤 | 操作 | 说明 |
|------|------|------|
| **步骤 1** | 确认版本 | 确保安装包是 `64-bit` |
| **步骤 2** | 勾选选项 | 勾选 `Add Python 3.7 to Path` |
| **步骤 3** | 开始安装 | 点击 `Install Now` |

![](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/win_python_setup2.png)

> [!important] 关键步骤
> **必须勾选** `Add Python 3.7 to Path`，否则需要手动配置环境变量。

---

#### 步骤 3：验证安装

1. 按 `Win + R` 打开运行窗口
2. 输入 `cmd` 打开命令提示符
3. 输入 `python` 查看版本信息

![](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/python_version.png)

> [!note] 验证结果
> CMD 窗口会显示对应的 Python 版本信息。如果未显示，请从步骤 1 重新安装。

### 安装验证命令

```bash
# 查看 Python 版本
python --version

# 查看 pip 版本
pip --version

# 进入 Python 交互模式
python
```

## 2.3 在 Ubuntu 上安装 Python

### 环境要求

| 项目 | 要求 |
|------|------|
| **操作系统** | Ubuntu 16.04 64 位 |
| **Python 版本** | 3.7.8 |

### 安装步骤

#### 步骤 1：查看现有版本

Ubuntu 16.04 默认安装了 Python 2.7 和 3.5：

```bash
python
```

> [!warning] 注意
> 系统自带的 Python **千万不能卸载**，否则会导致系统问题。

---

#### 步骤 2：安装 Python 3.7

```bash
# 添加 PPA 源
sudo add-apt-repository ppa:jonathonf/python-3.7

# 更新软件包列表
sudo apt-get update

# 安装 Python 3.7
sudo apt-get install python3.7
```

---

#### 步骤 3：调整 Python 优先级

设置 Python 3.7 为默认版本：

```bash
# 配置 python3 优先级
sudo update-alternatives --install /usr/bin/python3 python3 /usr/bin/python3.6 1
sudo update-alternatives --install /usr/bin/python3 python3 /usr/bin/python3.7 2

# 配置 python 优先级
sudo update-alternatives --install /usr/bin/python python /usr/bin/python2 100
sudo update-alternatives --install /usr/bin/python python /usr/bin/python3 150
```

**优先级说明：**
- 数字越大，优先级越高
- Python 3 优先级设为 150（最高）
- Python 2 优先级设为 100

---

#### 步骤 4：验证安装

```bash
python
```

确认显示 Python 3.7 版本信息。

---

## 安装 RoboMaster SDK

完成 Python 环境安装后，可以安装 RoboMaster SDK：

```bash
# 使用 pip 安装
pip install robomaster

# 升级 SDK
pip install --upgrade robomaster
```

> [!info] 详细安装说明
> 详细的 SDK 安装步骤请参考：[[../02-SDK基础/SDK安装|SDK 安装指南]]

## 常见问题

### Windows 问题

| 问题 | 解决方案 |
|------|----------|
| `python` 命令无法识别 | 重新安装并勾选 `Add Python to Path` |
| 安装 SDK 失败 | 确认 Python 版本在 3.6.6~3.8.9 之间 |
| pip 版本过低 | 运行 `python -m pip install --upgrade pip` |

### Ubuntu 问题

| 问题 | 解决方案 |
|------|----------|
| 权限不足 | 使用 `sudo` 执行命令 |
| PPA 源添加失败 | 检查网络连接 |
| Python 版本切换 | 使用 `sudo update-alternatives --config python` |
