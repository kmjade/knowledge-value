## DBUS 协议

### 通信参数

| 参数 | 值 |
|------|-----|
| **波特率** | 100000 bps |
| **数据位** | 8 |
| **停止位** | 1 |
| **校验位** | Even (偶校验) |
| **数据长度** | 18 字节 |

### 数据帧结构

```
| 字节 | 内容 |
|------|------|
| 0 | 帧头 0x0F |
| 1-2 | 右摇杆 X (CH0) |
| 2-4 | 右摇杆 Y (CH1) |
| 4-6 | 左摇杆 X (CH2) |
| 6-8 | 左摇杆 Y (CH3) |
| 8-10 | 左拨杆 (CH4) |
| 10-11 | 右拨杆 (CH5) |
| 11-16 | 按键状态 |
| 16-17 | 鼠标 X/Y |
| 17-18 | 鼠标按键 |
```

### 通道定义

| 通道 | 摇杆 | 范围 | 说明 |
|------|------|------|------|
| CH0 | 右摇杆 X | -660 ~ 660 | 左右 |
| CH1 | 右摇杆 Y | -660 ~ 660 | 前后 |
| CH2 | 左摇杆 X | -660 ~ 660 | 左右 |
| CH3 | 左摇杆 Y | -660 ~ 660 | 前后 |
| CH4 | 左拨杆 | 1/2/3 | 三档开关 |
| CH5 | 右拨杆 | 1/2/3 | 三档开关 |

## 使用示例

### 基础使用

```c
#include "dbus.h"

int main(void)
{
    // HAL 初始化
    HAL_Init();
    SystemClock_Config();
    
    // 初始化 DBUS
    DBUS_Init();
    
    RC_Controller_t *rc;
    
    while (1)
    {
        // 获取遥控器数据
        rc = DBUS_GetData();
        
        if (DBUS_IsOnline()) {
            // 读取摇杆值
            int16_t right_x = rc->ch0;  // 右摇杆 X
            int16_t right_y = rc->ch1;  // 右摇杆 Y
            int16_t left_x = rc->ch2;   // 左摇杆 X
            int16_t left_y = rc->ch3;   // 左摇杆 Y
            
            // 读取拨杆值
            int16_t sw_l = rc->ch4;     // 左拨杆 (1/2/3)
            int16_t sw_r = rc->ch5;     // 右拨杆 (1/2/3)
        }
        
        HAL_Delay(10);
    }
}
```

### 底盘控制

```c
// 遥控器控制底盘运动
void RC_ControlChassis(void)
{
    RC_Controller_t *rc = DBUS_GetData();
    
    if (!DBUS_IsOnline()) {
        // 遥控器离线，停止运动
        Chassis_Stop();
        return;
    }
    
    // 摇杆值映射到速度 (-660~660 -> -MAX_SPEED~MAX_SPEED)
    float vx = (float)rc->ch3 / 660.0f * MAX_SPEED;
    float vy = (float)rc->ch2 / 660.0f * MAX_SPEED;
    float vw = (float)rc->ch0 / 660.0f * MAX_ROTATE_SPEED;
    
    // 设置底盘速度
    Chassis_SetSpeed(vx, vy, vw);
}
```

### 模式切换

```c
// 根据拨杆切换运行模式
void RC_UpdateMode(void)
{
    RC_Controller_t *rc = DBUS_GetData();
    
    switch (rc->ch4) {
        case 1:
            // 左拨杆在上，手动模式
            Robot_SetMode(MODE_MANUAL);
            break;
        case 2:
            // 左拨杆在中，自动模式
            Robot_SetMode(MODE_AUTO);
            break;
        case 3:
            // 左拨杆在下，调试模式
            Robot_SetMode(MODE_DEBUG);
            break;
    }
}
```

## 数据解析

### 协议解析代码

```c
// DBUS 数据解析
void DBUS_Decode(uint8_t *data)
{
    RC_Controller_t *rc = &rc_data;
    
    // 解析摇杆数据 (11bit each)
    rc->ch0 = (data[0] | data[1] << 8) & 0x07FF;
    rc->ch1 = (data[1] >> 3 | data[2] << 5) & 0x07FF;
    rc->ch2 = (data[2] >> 6 | data[3] << 2 | data[4] << 10) & 0x07FF;
    rc->ch3 = (data[4] >> 1 | data[5] << 7) & 0x07FF;
    rc->ch4 = (data[5] >> 4 | data[6] << 4) & 0x07FF;
    rc->ch5 = (data[6] >> 7 | data[7] << 1 | data[8] << 9) & 0x07FF;
    
    // 转换为有符号值 (-660 ~ 660)
    if (rc->ch0 > 1024) rc->ch0 -= 2048;
    if (rc->ch1 > 1024) rc->ch1 -= 2048;
    if (rc->ch2 > 1024) rc->ch2 -= 2048;
    if (rc->ch3 > 1024) rc->ch3 -= 2048;
    if (rc->ch4 > 1024) rc->ch4 -= 2048;
    if (rc->ch5 > 1024) rc->ch5 -= 2048;
}
```

## 安全机制

### 失控保护

```c
// 遥控器离线检测
#define RC_TIMEOUT_MS 100

uint32_t last_update_time = 0;

void DBUS_UpdateTime(void)
{
    last_update_time = HAL_GetTick();
}

uint8_t DBUS_IsOnline(void)
{
    return (HAL_GetTick() - last_update_time) < RC_TIMEOUT_MS;
}

// 安全停止
void Safety_Check(void)
{
    if (!DBUS_IsOnline()) {
        // 遥控器离线，紧急停止
        Emergency_Stop();
    }
}
```

## 调试方法

### 数据验证

1. **检查帧头**
   - 第一字节应为 0x0F

2. **检查数据范围**
   - 摇杆值应在 -660 ~ 660
   - 拨杆值应为 1、2 或 3

3. **检查更新频率**
   - 正常为 14ms 一帧

### 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 无数据 | 波特率错误 | 确认 100000 bps |
| 数据错误 | 校验位配置错误 | 设置为 Even |
| 频繁离线 | 信号干扰 | 检查天线连接 |
