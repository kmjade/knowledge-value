# 11. RoboMaster SDK 多机api汇总

> [!info] 本章概述
> RoboMaster SDK 多机控制 API 汇总，列出 EP 机器人和教育无人机支持的多机控制接口。

---

## 概述

目前多机支持的 API 使用大部分都与单机相同，本文档主要对多机目前支持的 API 做一个汇总。

> [!tip] 说明
> 本文单独介绍多机中与单机用法不同的接口，其余 API 的具体参数类型、取值范围以及返回值等详细介绍参考单机部分。

---

## 11.1 多机 API 汇总

### 11.1.1 EP 机器人部分

#### chassis 模块（底盘控制）

| API | 说明 |
|-----|------|
| `drive_wheels(w1, w2, w3, w4, timeout)` | 麦轮速度控制 |
| `drive_speed(x, y, z, timeout)` | 速度控制 |
| `move(x, y, z, xy_speed, z_speed)` | 相对移动 |

**示例：**

```python
# 组内所有机器人前进 1 米
robot_group.chassis.move(x=1, y=0, z=0, xy_speed=0.7).wait_for_completed()

# 组内所有机器人左转 90 度
robot_group.chassis.move(x=0, y=0, z=90, z_speed=45).wait_for_completed()

# 组内所有机器人以 0.5 m/s 前进
robot_group.chassis.drive_speed(x=0.5, y=0, z=0, timeout=5)
```

---

#### gimbal 模块（云台控制）

| API | 说明 |
|-----|------|
| `recenter(pitch_speed, yaw_speed)` | 回中 |
| `suspend()` | 暂停 |
| `resume()` | 恢复 |
| `move(pitch, yaw, pitch_speed, yaw_speed)` | 相对移动 |
| `moveto(pitch, yaw, pitch_speed, yaw_speed)` | 绝对移动 |

**示例：**

```python
# 组内所有机器人云台向左旋转 90 度
robot_group.gimbal.move(pitch=0, yaw=90).wait_for_completed()

# 组内所有机器人云台回中
robot_group.gimbal.recenter().wait_for_completed()
```

---

#### blaster 模块（发射器控制）

| API | 说明 |
|-----|------|
| `fire(fire_type, times)` | 发射 |
| `set_led(brightness, effect)` | 设置瞄准灯 |

**示例：**

```python
from robomaster import blaster

# 组内所有机器人发射 1 颗水弹
robot_group.blaster.fire(fire_type=blaster.WATER_FIRE, times=1)
```

---

#### led 模块（LED 控制）

| API | 说明 |
|-----|------|
| `set_led(comp, r, g, b, effect, freq)` | 设置灯效 |

**示例：**

```python
from robomaster import led

# 组内所有机器人设置为红色常亮
robot_group.led.set_led(comp=led.COMP_ALL, r=255, g=0, b=0, effect=led.EFFECT_ON)
```

---

#### robotic_arm 模块（机械臂控制）

| API | 说明 |
|-----|------|
| `recenter()` | 回中 |
| `move(x, y)` | 相对移动 |
| `moveto(x, y)` | 绝对移动 |

**示例：**

```python
# 组内所有机器人机械臂移动到指定位置
robot_group.robotic_arm.moveto(x=100, y=0).wait_for_completed()
```

---

#### gripper 模块（机械爪控制）

| API | 说明 |
|-----|------|
| `open(power)` | 打开 |
| `close(power)` | 关闭 |
| `pause()` | 暂停 |

**示例：**

```python
# 组内所有机器人打开机械爪
robot_group.gripper.open(power=50)
```

### 11.1.2 教育机器人部分

#### flight 模块（飞行控制）

| API | 说明 |
|-----|------|
| `takeoff()` | 起飞 |
| `land()` | 降落 |
| `up(distance)` | 向上飞 |
| `down(distance)` | 向下飞 |
| `forward(distance)` | 向前飞 |
| `backward(distance)` | 向后飞 |
| `left(distance)` | 向左飞 |
| `right(distance)` | 向右飞 |
| `rotate(angle)` | 旋转 |
| `flip_forward()` | 向前翻滚 |
| `flip_backward()` | 向后翻滚 |
| `flip_left()` | 向左翻滚 |
| `flip_right()` | 向右翻滚 |
| `go(distance)` | 飞到指定位置（多机特有） |
| `mission_pad_on()` | 开启挑战卡检测 |
| `mission_pad_off()` | 关闭挑战卡检测 |
| `motor_on()` | 开启电机 |
| `motor_off()` | 关闭电机 |

**示例：**

```python
# 组内所有无人机起飞
drone_group.flight.takeoff().wait_for_completed()

# 组内所有无人机向前飞 100cm
drone_group.flight.forward(distance=100).wait_for_completed()

# 组内所有无人机降落
drone_group.flight.land().wait_for_completed()
```

## 多机特有接口

以下接口在多机中的使用与单机不同：

### flight 模块的 go() 指令

**单机用法：** `go(x, y, z, speed, mid)`

**多机用法：**

```python
go(go_dict)
```

**参数：**

```python
go_dict = {
    robot_id1: [x1, y1, z1, speed1, mid1],
    robot_id2: [x2, y2, z2, speed2, mid2],
    ...
}
```

| 参数 | 说明 |
|------|------|
| `robot_id` | 飞机的编号 |
| `x, y, z` | 目标坐标 |
| `speed` | 移动速度 |
| `mid` | 挑战卡号码 |

**返回值：** `multi_action` 对象

**示例：**

```python
# 控制 0 号无人机飞到 (100, 50, 100)
# 控制 1 号无人机飞到 (200, 100, 120)
go_dict = {
    0: [100, 50, 100, 50, 1],
    1: [200, 100, 120, 50, 2]
}
drone_group.flight.go(go_dict).wait_for_completed()
```

---

### led 模块指令

**多机用法：**

新增 `command_dict` 参数，实现对单个飞机 LED 的控制。

```python
set_led(command_dict={...})
```

**参数：**

```python
command_dict = {
    robot_id1: [*args],
    robot_id2: [*args],
    ...
}
```

| 参数 | 说明 |
|------|------|
| `robot_id` | 飞机编号 |
| `*args` | 各 LED 模块的参数 |

**返回值：** `multi_action` 对象

**示例：**

```python
# 0 号无人机设置红色，1 号无人机设置蓝色
command_dict = {
    0: [255, 0, 0],    # 红色
    1: [0, 0, 255]     # 蓝色
}
drone_group.led.set_led(command_dict=command_dict)
```

---

### battery 模块的 get_battery() 指令

**多机用法：**

```python
get_battery()
```

**参数：** 无

**行为：** 会将飞机 ID 和对应的电量打印至控制台

**示例：**

```python
# 获取组内所有无人机电量
drone_group.battery.get_battery()

# 输出示例：
# Robot 0: 85%
# Robot 1: 92%
```

## 多机控制完整示例

### EP 多机编队示例

```python
from multi_robomaster import multi_robot

# 创建多机对象
multi_robots = multi_robot.MultiEP()
multi_robots.initialize()

# 编号
multi_robots.number_id_by_sn(
    [0, 'SN_XXXXX1'],
    [1, 'SN_XXXXX2']
)

# 分组
group_all = multi_robots.build_group([0, 1])

# 定义任务
def forward_task(group):
    group.chassis.move(x=1, y=0, z=0, xy_speed=0.7).wait_for_completed()

def led_red_task(group):
    from robomaster import led
    group.led.set_led(comp=led.COMP_ALL, r=255, g=0, b=0, effect=led.EFFECT_ON)

# 执行并行任务
multi_robots.run([group_all, forward_task])
multi_robots.run([group_all, led_red_task])

# 释放资源
multi_robots.close()
```

---

### 教育无人机多机编队示例

```python
from multi_robomaster import multi_robot

# 创建多机对象
multi_drones = multi_robot.MultiDrone()
multi_drones.initialize(2)  # 2 架无人机

# 编号
multi_drones.number_id_by_sn(
    [0, 'SN_XXXXX1'],
    [1, 'SN_XXXXX2']
)

# 分组
group_all = multi_drones.build_group([0, 1])

# 定义任务
def takeoff_task(group):
    for drone_id in group.robots_id_list:
        drone = group.get_robot(drone_id)
        drone.flight.takeoff().wait_for_completed()

def forward_task(group):
    group.flight.forward(distance=100).wait_for_completed()

def land_task(group):
    group.flight.land().wait_for_completed()

# 执行任务
multi_drones.run([group_all, takeoff_task])
multi_drones.run([group_all, forward_task])
multi_drones.run([group_all, land_task])

# 释放资源
multi_drones.close()
```

## 多机 API 对比表

### EP 机器人

| 模块 | 支持的 API 数量 | 主要功能 |
|------|----------------|----------|
| chassis | 3 | 移动控制 |
| gimbal | 5 | 云台控制 |
| blaster | 2 | 发射控制 |
| led | 1 | 灯效控制 |
| robotic_arm | 3 | 机械臂控制 |
| gripper | 3 | 机械爪控制 |

### 教育无人机

| 模块 | 支持的 API 数量 | 主要功能 |
|------|----------------|----------|
