# 05 感測器模組 Sensor Modules

> 感測器是電子設備的感官——讓你的電路能看見、聽見、感受到這個世界。

## 感測器分類

### 依物理量分類

| 類別 | 感測器 | 用途 |
|------|--------|------|
| 🌡️ 溫度/濕度 | DHT11, DHT22, DS18B20, LM35 | 氣象站、溫控 |
| 📏 距離 | HC-SR04 (超音波), VL53L0X (ToF), GP2Y0A21 (紅外線) | 避障、測距 |
| 💡 光線 | LDR (光敏電阻), BH1750, TSL2561 | 自動燈光、亮度檢測 |
| 🎵 聲音 | KY-038 聲音感測器, MAX9814 麥克風 | 聲控、噪音監測 |
| 🧭 運動/姿態 | MPU6050 (6軸), MPU9250 (9軸), ADXL345 | 平衡車、手勢控制 |
| 💨 氣體 | MQ-2 (煙霧), MQ-3 (酒精), MQ-135 (空氣品質) | 安全警報、空氣監測 |
| 🖐️ 觸摸 | TTP223 電容觸摸 | 觸摸開關 |
| 🏷️ RFID | RC522 (13.56MHz) | 門禁、物品識別 |
| 📡 人體 | HC-SR501 PIR (紅外線人體) | 自動燈、防盜 |
| 🧲 磁力 | KY-003 霍爾感測器, KY-025 磁簧開關 | 轉速測量、門窗感應 |
| 💧 液體 | 雨滴感測器, 水位感測器, YF-S201 流量計 | 澆水系統、液位監控 |
| ⚡ 電流/電壓 | INA219, ACS712 | 功耗監測 |

## 常用感測器詳解

### DHT11 / DHT22 — 溫濕度

```cpp
#include <DHT.h>

#define DHTPIN 2
#define DHTTYPE DHT11   // 或 DHT22

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float h = dht.readHumidity();
  float t = dht.readTemperature();
  
  Serial.print("濕度: "); Serial.print(h);
  Serial.print("%  溫度: "); Serial.print(t);
  Serial.println("°C");
  delay(2000);
}
```

| | DHT11 | DHT22 |
|---|-------|-------|
| 溫度範圍 | 0–50°C | -40–80°C |
| 精度 | ±2°C | ±0.5°C |
| 採樣率 | 1Hz | 0.5Hz |
| 價格 | ¥3–5 | ¥8–15 |

### HC-SR04 — 超音波測距

```cpp
#define TRIG 9
#define ECHO 10

void setup() {
  Serial.begin(9600);
  pinMode(TRIG, OUTPUT);
  pinMode(ECHO, INPUT);
}

void loop() {
  digitalWrite(TRIG, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG, LOW);
  
  long duration = pulseIn(ECHO, HIGH);
  float distance = duration * 0.034 / 2;  // cm
  
  Serial.print("距離: "); Serial.print(distance);
  Serial.println(" cm");
  delay(500);
}
```

> ⚠️ HC-SR04 盲區約 2cm，最大測距約 4m。不適合測量軟質/吸音表面。

### MPU6050 — 6軸陀螺儀+加速度計

```cpp
#include <Wire.h>
#include <MPU6050.h>

MPU6050 mpu;

void setup() {
  Serial.begin(115200);
  Wire.begin();
  mpu.initialize();
  
  if (mpu.testConnection()) {
    Serial.println("MPU6050 OK");
  }
}

void loop() {
  int16_t ax, ay, az, gx, gy, gz;
  mpu.getMotion6(&ax, &ay, &az, &gx, &gy, &gz);
  
  Serial.print("aX: "); Serial.print(ax);
  Serial.print(" gZ: "); Serial.println(gz);
  delay(100);
}
```

### VL53L0X — 雷射 ToF 測距

| 特性 | 值 |
|------|-----|
| 原理 | 飛時測距 (Time of Flight) |
| 範圍 | 30–2000mm |
| 精度 | ±3% |
| 介面 | I2C |
| 優勢 | 不受物體顏色/反射率影響 |

```cpp
#include <Wire.h>
#include <VL53L0X.h>

VL53L0X sensor;

void setup() {
  Serial.begin(9600);
  Wire.begin();
  sensor.init();
  sensor.setTimeout(500);
  sensor.startContinuous();
}

void loop() {
  Serial.print(sensor.readRangeContinuousMillimeters());
  Serial.println("mm");
}
```

## 感測器介面速查

| 介面 | 腳位數 | 速度 | 常用感測器 |
|------|-------|------|-----------|
| **數位 GPIO** | 1 | - | PIR、觸摸、霍爾（開關型） |
| **Analog** | 1 | - | LDR、聲音、氣體（MQ系列） |
| **I2C** | 2 (SDA+SCL) | 100/400kHz | BH1750、MPU6050、VL53L0X、BMP280 |
| **SPI** | 4 (MOSI/MISO/SCK/CS) | >1MHz | 高速ADC、LCD |
| **UART** | 2 (TX+RX) | 9600–115200 | GPS、指紋模組 |
| **OneWire** | 1 | - | DS18B20 |
| **PWM** | 1 | - | 伺服馬達 |

## 感測器選購建議

### 新手入門套件（¥30–50）

- DHT11 溫濕度 ×1
- HC-SR04 超音波 ×1
- LDR 光敏電阻 ×3
- KY-038 聲音感測器 ×1
- HC-SR501 PIR人體感應 ×1
- 紅外線避障模組 ×2

### 進階推薦

- DHT22（更高精度溫濕度）
- MPU6050（姿態感測，做平衡車必備）
- VL53L0X（精準測距，比超音波可靠）
- BH1750（數位光照度，比LDR準）
- BMP280（氣壓+溫度，測海拔）

## 除錯技巧

1. **I2C 設備找不到** → 用 I2C Scanner 確認位址
2. **數值亂跳** → 加中值濾波或移動平均
3. **讀數永遠0或最大值** → 檢查接線、供電電壓
4. **DHT11 讀取失敗** → 加 delay(2000)，不要頻繁讀取
5. **HC-SR04 不準** → 確認目標是硬質平面，避免軟/吸音材料
