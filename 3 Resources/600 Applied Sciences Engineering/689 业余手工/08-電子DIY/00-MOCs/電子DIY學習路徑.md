# 電子DIY — 學習路徑

## Phase 1：點亮第一顆LED
Arduino + LED + Blink → 按鈕 → 光敏電阻

## Phase 2：感知世界
DHT11 → HC-SR04 → 更多感測器

## Phase 3：讓東西動起來
伺服馬達 → 直流馬達 → 步進馬達

## Phase 4：連上網路
ESP32 WiFi → Web控制 → MQTT → IoT專案

> 💡 每完成一個Phase就做一個小專案。理論+實作 = 真正學會。
