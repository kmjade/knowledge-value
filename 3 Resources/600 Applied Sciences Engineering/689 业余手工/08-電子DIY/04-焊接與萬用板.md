# 04 焊接與萬用板 Soldering & Protoboard

> 麵包板是草稿，焊接是定稿。從原型到成品，只差一把烙鐵。

## 為什麼要學焊接？

- 麵包板接觸不良 → 疑難雜症來源
- 焊接後電路穩定、可長期使用
- 是從「做實驗」到「做產品」的分水嶺

## 焊接基礎

### 五步焊接法

```
1. 清潔 → 烙鐵頭沾錫、海綿擦拭
2. 加熱 → 烙鐵頭同時接觸焊盤 + 元件腳（約1–2秒）
3. 送錫 → 焊錫絲接觸焊盤（非烙鐵頭！）
4. 撤錫 → 錫量足夠後先移開焊錫絲
5. 撤烙鐵 → 保持0.5秒後移開烙鐵
```

### 好焊點 vs 壞焊點

| 好焊點 ✅ | 壞焊點 ❌ |
|-----------|----------|
| 光亮、圓潤、山形 | 霧面、毛刺 |
| 完全包覆焊盤與腳 | 錫量不足（冷焊） |
| 無橋接 | 連錫（短路） |
| 牢固不動 | 元件可晃動 |

### 常見問題與解決

| 問題 | 原因 | 解決 |
|------|------|------|
| 焊錫不沾 | 氧化 / 溫度不夠 | 加助焊劑、提高溫度 |
| 錫珠飛濺 | 溫度過高 / 助焊劑沸騰 | 降低溫度至 320–350°C |
| 焊點發灰 | 冷焊（移動了） | 重新加熱至焊錫流動 |
| 連錫/橋接 | 錫量過多 | 吸錫器清理、重新焊接 |
| 焊盤脫落 | 加熱太久 / 溫度太高 | 飛線修復 |

## 建議溫度

| 焊錫類型 | 溫度 |
|----------|------|
| 63/37 含鉛焊錫 | 320–350°C |
| 無鉛焊錫 | 350–380°C |
| SMD 小元件 | 300–320°C |
| 大面積接地焊盤 | 380–400°C |

## 萬用板（洞洞板）

### 類型

| 類型 | 說明 | 適合 |
|------|------|------|
| 單面焊盤 | 一面有銅箔焊盤 | 簡單電路 |
| 雙面焊盤 | 兩面都有焊盤 | 複雜佈線 |
| 連孔型 | 整排導通 | 電源/地線 |
| 麵包板佈局型 | 模仿麵包板連接 | 快速轉換麵包板原型 |

### 佈線技巧

```
1. 先規劃 → 紙上畫出元件位置與走線
2. 電源/地 → 用粗線或銅柱做電源總線
3. IC 先放 → 預留空間走線
4. 跳線 → 使用不同顏色（紅=VCC、黑=GND、其他=信號）
5. 焊接順序 → 矮元件先 → 高元件後
```

### 常用成品轉換

麵包板測試成功後，轉到萬用板上的步驟：

1. 拍照記錄麵包板佈局
2. 在萬用板上標記IC位置
3. 焊接IC座（不要直接焊IC）
4. 焊被動元件（電阻、電容）
5. 焊連接器、排針
6. 最後焊跳線
7. 目檢 + 萬用表測短路
8. 通電測試

## 安全須知

> ⚠️ **烙鐵頭溫度 300°C+，可瞬間造成三度灼傷。**

- 使用烙鐵架，烙鐵不用時放回架上
- 通風良好——焊錫煙霧含松香和鉛
- 焊接後洗手（含鉛焊錫）
- 電容先放電再焊接
- 戴護目鏡（剪腳時碎片可能飛濺）
