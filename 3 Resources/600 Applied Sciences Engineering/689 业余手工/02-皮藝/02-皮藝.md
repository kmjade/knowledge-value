# 02 — 業餘皮藝

> 皮革的魅力在於它會隨著使用而變化——你的包包、錢包會留下你的痕跡，越用越有味道。

## 必備工具（入門約 ¥300–500）

| 工具 | 用途 |
|------|------|
| 裁皮刀/美工刀 | 裁切 |
| 菱斬（4mm間距） | 打縫線孔 |
| 手縫針+蠟線 | 縫合 |
| 邊緣處理器 | 磨邊 |
| 床面處理劑 | 封邊 |
| 白膠 | 黏貼 |

## 皮革種類

| 類型 | 特點 | 適合 |
|------|------|------|
| 植鞣革 | 硬挺、可雕刻、會變色 | 錢包、皮帶 |
| 鉻鞣革 | 柔軟、顏色多 | 包包 |
| 瘋馬皮 | 油感、復古 | 粗獷風格 |

## 新手專案

1. **鑰匙扣**（30min）— 一片皮 + 打孔 + 五金
2. **卡套**（1hr）— 學習縫線
3. **簡約錢包**（3hr）— 多層結構

## 基礎縫法：馬鞍縫

雙針交叉縫合——最牢固的手縫方法。一根線兩頭穿針，從兩側穿過同一個孔。

> 💡 皮藝的秘訣是耐心——慢慢裁、慢慢縫，皮會回應你的用心。好皮可以用十年以上。
