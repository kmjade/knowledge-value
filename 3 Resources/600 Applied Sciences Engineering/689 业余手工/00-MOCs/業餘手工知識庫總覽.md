# 業餘手工 — 知識庫總覽

> 用雙手創造的快樂——各類手工的入門地圖。

## 手工分類

| 類別 | 材料 | 核心技能 |
|------|------|---------|
| 木工 | 木材 | 鋸、鑿、鉋、磨 |
| 皮藝 | 皮革 | 裁、縫、封邊 |
| 編織 | 毛線/繩 | 鉤、編、結 |
| 金工 | 金屬 | 鋸、焊、磨 |
| 模型 | 塑膠/紙 | 組裝、上色 |
| 陶藝 | 泥土 | 捏、拉、燒 |
| 縫紉 | 布料 | 縫、裁、熨 |
| 電子 | 電路 | 程式、焊接 |

> 💡 手工的共通法則：先完成再完美。第一個作品永遠是最醜的——但也是最珍貴的。
