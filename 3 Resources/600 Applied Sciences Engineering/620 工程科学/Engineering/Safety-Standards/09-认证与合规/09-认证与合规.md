# 09-认证与合规

> 不是设计出安全的产品就结束了——你需要**证明**它是安全的、**签署**法律文件、**粘贴**标识。

## CE 标识流程

### 1. 确定适用指令

| 指令 | 适用 |
|------|------|
| **2006/42/EC** (机械指令) | 几乎所有机器 |
| **2014/35/EU** (低电压指令) | 50-1000V AC / 75-1500V DC |
| **2014/30/EU** (EMC 指令) | 有电子元件的机器 |
| **2014/53/EU** (RED) | 无线电设备 |

### 2. 确定合格评定路径

| 路径 | 适用 |
|------|------|
| **自我声明 (附录 VIII)** | 大多数普通机器——制造商自己评估 |
| **EC 型式检验 (附录 IX)** | 高风险机器 (列入附录 IV)——公告机构介入 |
| **全面质量保证 (附录 X)** | 制造商有认证的质量体系 |

> 附录 IV 的机器包括：圆锯、木工机床、注塑机、升降平台、安全零部件等——需要公告机构 (Notified Body) 参与。

### 3. 技术文件 (TCF)

必须保存至少 10 年：

| 文件 | 内容 |
|------|------|
| 总图+控制图 | 机械结构+电气/液压/气动 |
| 风险评估 | 完整的风险评估报告 |
| 协调标准清单 | 你依据了哪些标准 |
| 测试报告 | 电气安全测试、功能安全计算等 |
| 说明书 | 使用国的语言 |
| **符合性声明 (DoC)** | 签名——"我声明符合所有适用指令" |

### 4. 符合性声明 (DoC)

最核心的法律文件——必须有：

```
产品名称/型号
制造商名称/地址
我们声明上述产品符合：
  - 机械指令 2006/42/EC
  - EMC 指令 2014/30/EU
依据标准：
  - EN ISO 12100:2010
  - EN 60204-1:2018
签名/日期/职位
```

### 5. 粘贴 CE 标识

| 要求 | 说明 |
|------|------|
| 位置 | 铭牌上——清晰、不可磨灭 |
| 大小 | 至少 5mm 高 |
| 比例 | 标准比例——不能变形 |

## 中国 CR/CCC

| 认证 | 产品 |
|------|------|
| **CCC** | 强制性——电气、汽车、玩具等 |
| **CR (中国机器人认证)** | 工业/协作机器人 |

## 违反的后果

| 后果 | 说明 |
|------|------|
| 产品召回 | 全部费用——可能数倍于合规成本 |
| 罚款 | 欧盟可达年营业额的 4% |
| 刑事责任 | 出了人命——相关责任人可能坐牢 |
| 市场禁入 | 产品退出市场 |
