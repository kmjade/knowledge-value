# 03-功能安全

> 功能安全是控制系统（而非物理防护）来保证安全——"控制系统本身不会导致危险，并且在需要时能可靠地执行安全功能"。

## 核心标准

| 标准 | 适用 | 安全等级 |
|------|------|----------|
| **ISO 13849-1** | 机械——控制系统安全 | **PL** (a→e) |
| **IEC 62061** | 机械——功能安全 | **SIL** (1→3) |
| **IEC 61508** | 所有行业——功能安全总则 | **SIL** (1→4) |
| **IEC 61511** | 过程工业 (化工/石化) | **SIL** (1→4) |

## PL vs SIL

| 维度 | PL (ISO 13849) | SIL (IEC 62061) |
|------|---------------|-----------------|
| 等级 | a b c d **e** | 1 2 3 |
| 每小时危险失效概率 | PLe: <10⁻⁷ | SIL3: <10⁻⁷ |
| 适用 | 机械（传统优势） | 电气/电子/可编程 |
| 关系 | PL e ≈ SIL 3 | SIL 3 ≈ PL e |

### 选 ISO 13849 还是 IEC 62061？

| 情况 | 推荐 |
|------|------|
| 简单机电系统 (继电器、接触器) | ISO 13849 |
| 复杂电子/可编程系统 (安全PLC) | IEC 62061 |
| 二者混合 | 都可以——选你熟悉的 |

## 安全功能示例

| 安全功能 | 触发 | 响应 |
|----------|------|------|
| 光幕被遮挡 | 人进入 | 电机立即停止 |
| 急停被按下 | 手动触发 | 所有运动停止 |
| 门联锁打开 | 防护门打开 | 主轴停止+制动 |
| 超速检测 | 速度 > 120% | 安全停止 |

## 安全回路的架构

### 单通道 (Cat B/1)

```
传感器 → 逻辑 → 执行器
```
任何单点故障→安全功能丧失→**不能用于高等级**

### 双通道 (Cat 3/4)

```
传感器A → 逻辑A → 执行器A
传感器B → 逻辑B → 执行器B
      ↘ 交叉诊断 ↙
```

单点故障→另一通道仍执行安全功能→检测到故障→安全停机

## PL 的确定

```
PL = f(类别, MTTFd, DC, CCF)

类别: B/1/2/3/4
MTTFd: 平均危险失效时间 (低/中/高)
DC: 诊断覆盖率 (无/低/中/高)
CCF: 共因失效防护措施
```

| PL | 每小时危险失效概率 | 典型应用 |
|----|------------------|----------|
| a | ≥10⁻⁵ | 低风险 |
| b | <10⁻⁵ | 轻微伤害 |
| c | <3×10⁻⁶ | 可逆伤害 |
| d | <10⁻⁶ | 严重不可逆伤害 |
| **e** | **<10⁻⁷** | **致命/多人伤害** |
