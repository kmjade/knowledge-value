# 04-机器人安全

> 机器人安全由 ISO 10218 (工业机器人) 和 ISO/TS 15066 (协作机器人) 两个标准主导。传统机器人用围栏隔离，协作机器人用力和速度限制。

## 核心标准

| 标准 | 内容 |
|------|------|
| **ISO 10218-1** | 机器人本体安全设计 |
| **ISO 10218-2** | 机器人系统集成安全 |
| **ISO/TS 15066** | 协作机器人——四种模式 |

## ISO 10218-1 (本体)

| 要求 | 说明 |
|------|------|
| 安全停止 | 至少一个独立的安全停止输入 |
| 使能装置 | 示教时的三段式使能开关 |
| 速度限制 | 示教模式限速 250mm/s |
| 关节限制 | 软件/机械限制运动范围 |
| 标识 | 警告标签+技术数据 |

## ISO 10218-2 (系统集成)

| 要求 | 说明 |
|------|------|
| 围栏设计 | 高度≥1400mm，底部间隙≤200mm |
| 安全门联锁 | 门开→安全停机 |
| 光幕/扫描仪 | 检测人进入→减速或停止 |
| 末端执行器 | 工具的夹点也需要防护 |
| 风险评估 | 必须针对整个应用——不只是机器人 |

## ISO/TS 15066 四种协作模式

| 模式 | 原理 | 传感器 |
|------|------|--------|
| **安全级监控停止** | 人靠近→停止 | 激光扫描仪 |
| **手动引导** | 人手引导运动 | 力/扭矩传感器 |
| **速度分离监控** | 距离越近越慢→停止 | 视觉/激光 |
| **功率和力限制** | 碰撞力<人体承受极限 | 关节力矩传感器 |

### PFL 力限值

| 身体部位 | 准静态 | 瞬态 |
|----------|--------|------|
| 头/脸 | 65N | — |
| 手/手指 | 140N | 280N |
| 手臂 | 150N | 300N |
| 腹部 | 110N | 220N |
| 大腿 | 180N | 360N |

> 这些值来自医学文献——超过这个力可能导致挫伤、骨折。Cobot 必须证明碰撞力低于限值。

## 风险评估特殊考量

传统机器人：人不进入工作区（围栏）
Cobot：人就在旁边 → 要额外考虑：

| 考量 | 说明 |
|------|------|
| 末端执行器 | 夹爪/刀具比 Cobot 手臂更危险 |
| 工件 | 锐利边角、高温、脱落风险 |
| 夹点 | Cobot 和固定物体之间的挤压点 |
| 操作员行为 | 可能绕过安全措施——设计防呆 |
