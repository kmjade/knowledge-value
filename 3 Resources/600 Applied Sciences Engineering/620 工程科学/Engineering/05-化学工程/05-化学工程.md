# 05-化学工程

> 化学工程是将实验室化学转化为工业规模生产的学科——从原油裂解到药物合成，从塑料到化肥。核心是"单元操作"和"反应工程"。

## 化工的核心：单元操作

无论产品是什么，化工过程由有限的"单元操作"组合而成：

| 单元操作 | 原理 | 应用 |
|----------|------|------|
| **蒸馏** | 利用沸点差异分离液体混合物 | 原油分馏→汽油/柴油/沥青 |
| **吸收** | 气体被液体吸收 | 烟气脱硫 |
| **萃取** | 用溶剂从混合物中提取目标物 | 咖啡因提取、植物油萃取 |
| **过滤** | 用多孔介质分离固体和液体 | 水处理、制药 |
| **结晶** | 从溶液中析出纯固体 | 制糖、制药 |
| **干燥** | 去除水分 | 食品、化工产品 |
| **反应器** | 发生化学反应 | 所有化工过程的心脏 |

> 化工的哲学：一个年产百万吨的化工厂，其实就是把实验室烧瓶里的反应放大——但"放大"是化工最难的挑战。

## 反应工程

### 反应器类型

| 类型 | 特点 | 应用 |
|------|------|------|
| **间歇反应器** | 一批一批地生产 | 制药（小批量、高价值） |
| **连续搅拌釜 (CSTR)** | 连续进料连续出料 | 大宗化学品 |
| **管式反应器 (PFR)** | 物料在管道中推进 | 裂解、聚合 |
| **固定床反应器** | 流体通过固体催化剂床 | 催化裂化、合成氨 |

### 催化剂

| 类型 | 相态 | 例子 | 应用 |
|------|------|------|------|
| 均相 | 催化剂与反应物同相 | 酸/碱催化 | 酯化、水解 |
| 非均相 | 催化剂与反应物异相 | 固体催化剂+气体/液体 | 哈伯法合成氨、催化裂化 |
| 酶催化 | 生物催化剂 | 脂肪酶、蛋白酶 | 生物柴油、食品加工 |

> 哈伯法 (Haber-Bosch)：N₂+3H₂→2NH₃。用空气制造化肥——养活了地球一半的人口。这是 20 世纪最重要的化学工程突破。

## 过程控制

| 概念 | 含义 |
|------|------|
| PID 控制 | 比例-积分-微分——工业控制的标准算法 |
| 反馈控制 | 测量输出 → 与设定值比较 → 调整输入 |
| 串级控制 | 两个 PID 控制器串联——更精确 |
| 安全仪表系统 (SIS) | 独立于过程控制的紧急停车系统 |

## 安全：化工的第一要务

| 概念 | 说明 |
|------|------|
| **HAZOP** (危险与可操作性分析) | 系统性地分析每个节点可能的偏差和后果 |
| **LOPA** (保护层分析) | 一层一层的安全保护是否足够？ |
| **本质安全设计** | 从根本上减少危险物质和能量——比"加保护"更有效 |

### 博帕尔灾难 (1984)

印度博帕尔农药厂异氰酸甲酯泄漏——历史上最严重的工业灾难。直接死 3,800+ 人，影响 50 万+ 人。核心教训：安全系统不能只是"加上去的"——要在设计中就消除危险。
