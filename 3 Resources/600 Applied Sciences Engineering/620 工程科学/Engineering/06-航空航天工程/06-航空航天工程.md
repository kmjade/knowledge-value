# 06-航空航天工程

> 航空工程研究大气层内的飞行器，航天工程研究大气层外的航天器。两者共享流体力学、推进和结构的基础，但面临截然不同的挑战。

## 航空 vs 航天

| 维度 | 航空 | 航天 |
|------|------|------|
| 环境 | 大气层内 | 真空中 |
| 速度 | 亚音速/超音速 | 轨道速度 (~7.8 km/s, LEO) |
| 推进 | 吸气式（喷气发动机） | 自带氧化剂（火箭） |
| 返回 | 着陆 | 再入大气层 |
| 典型例子 | 波音 737, F-22 | 长征火箭, 国际空间站 |

## 空气动力学

### 四个力

```
             升力 (Lift)
                ↑
                |
   阻力 ←───── 飞机 ─────→ 推力 (Thrust)
  (Drag)        |
                ↓
             重力 (Weight)
```

| 力 | 来源 | 飞机如何管理 |
|----|------|-------------|
| 升力 | 机翼上下表面的压力差（伯努利原理+攻角） | 机翼形状、襟翼 |
| 重力 | 地球引力 | 轻量化结构、材料 |
| 推力 | 发动机 | 喷气发动机、螺旋桨 |
| 阻力 | 空气阻碍运动 | 流线型、翼型优化 |

### 音速与马赫数

| 马赫数 | 速度范围 | 现象 |
|--------|----------|------|
| M < 0.8 | 亚音速 | 空气可视为不可压缩 |
| M 0.8-1.2 | 跨音速 | 局部激波→阻力急增（音障） |
| M 1.2-5.0 | 超音速 | 激波锥，Concorde 在此飞行 |
| M > 5.0 | 高超音速 | 空气化学解离→极端加热 |

## 推进系统

| 类型 | 原理 | 应用 |
|------|------|------|
| 涡扇发动机 | 风扇+核心机，大部分推力来自风扇 | 客机 (CFM LEAP, GE9X) |
| 涡喷发动机 | 纯喷气 | 战斗机 |
| 火箭发动机 | 自带氧化剂+燃料 | 航天发射 |
| 电推进 | 电力加速离子 | 深空探测、卫星维持 |

### 齐奥尔科夫斯基火箭方程

```
Δv = v_exhaust × ln(m_initial / m_final)
```

这是航天工程最基本的方程——速度增量取决于排气速度和初始/最终质量比。

> 这就是为什么火箭 90% 以上的质量是燃料——要把剩下的 10% 加速到轨道速度，需要 90% 的燃料来"抬"燃料本身。

## 航天器系统

| 子系统 | 功能 |
|--------|------|
| 结构与机构 | 承受发射载荷，展开太阳能板 |
| 热控 | 真空中只能靠辐射散热 |
| 电源 | 太阳能板+电池，深空用核电池 (RTG) |
| 姿轨控 (GNC) | 确定方向和位置，推力器调整 |
| 通信 | 与地面的数据链路 |
| 生命保障 | 载人航天必需——空气、水、温度 |
