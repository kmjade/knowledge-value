## 工程的本质

| 核心 | 含义 |
|------|------|
| **约束下求解** | 在成本、时间、安全、法规等约束中找到可行最优解 |
| **安全边际** | 设计能力远超预期负载——工程不是赌博 |
| **故障预防** | 预想所有可能的失败模式，在设计阶段就消除 |
| **权衡取舍** | 快 vs 好 vs 便宜——不可能三角 |
| **系统思维** | 整体 > 部分之和，关注交互而非孤立的组件 |
