# 07 — 工業工程

> 工業工程是「效率的科學」——最佳化人員、物料、設備、能源和資訊的整合系統。

## 核心領域

| 領域 | 內容 |
|------|------|
| **作業研究 (OR)** | 線性規劃、排隊理論、賽局理論 |
| **生產管理** | 排程、產能規劃、庫存控制 |
| **品質管理** | SPC、六標準差、抽樣檢驗 |
| **人因工程** | 人機介面、工作場所設計 |
| **物流與供應鏈** | 運輸、倉儲、配送網路 |
| **模擬** | 離散事件模擬、蒙地卡羅 |

## 精實生產 (Lean)

| 浪費 | 說明 |
|------|------|
| 等待 | 閒置時間 |
| 搬運 | 不必要的移動 |
| 不良品 | 重工與廢品 |
| 過度加工 | 超出需求 |
| 庫存 | 過多物料 |
| 動作 | 多餘人員動作 |
| 過度生產 | 最大浪費 |

## 工業 4.0

| 技術 | 應用 |
|------|------|
| IoT | 設備即時監控 |
| 數位分身 | 虛擬模擬與最佳化 |
| AI | 預測維護、品質檢測 |
| AGV/AMR | 自主搬運 |
| 區塊鏈 | 供應鏈追溯 |

> 💡 工業工程不生產任何實體產品——它讓其他工程師的產品生產得更快、更好、更便宜。
