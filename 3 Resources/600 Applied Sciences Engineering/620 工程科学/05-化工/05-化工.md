# 05 — 化學工程

> 化學工程將實驗室的化學反應放大為工業量產——從石油煉製到藥品製造，從塑膠到清淨能源。

## 核心概念

| 概念 | 說明 |
|------|------|
| **質量平衡** | 輸入 = 輸出 + 積累 + 消耗 |
| **能量平衡** | 能量守恆在製程中的應用 |
| **動量傳遞** | 流體力學，泵與管道設計 |
| **熱量傳遞** | 熱交換器、加熱/冷卻 |
| **質量傳遞** | 蒸餾、吸收、萃取、乾燥 |

## 單元操作 Unit Operations

| 操作 | 目的 | 設備 |
|------|------|------|
| 蒸餾 | 依沸點分離液體混合物 | 蒸餾塔 |
| 過濾 | 固液分離 | 過濾機 |
| 乾燥 | 去除水分 | 乾燥機 |
| 結晶 | 純化固體 | 結晶器 |
| 混合 | 均勻化 | 攪拌槽 |
| 熱交換 | 加熱/冷卻 | 熱交換器 |

## 反應器設計

| 類型 | 說明 |
|------|------|
| CSTR (連續攪拌槽) | 穩定狀態、均勻混合 |
| PFR (柱塞流) | 沿管長度濃度變化 |
| 批次反應器 | 一次性投料反應 |

## 化工的現代應用

- 綠氫：電解水產氫（潔淨能源載體）
- 碳捕集與利用 (CCUS)：捕獲 CO₂ 轉化為產品
- 生物精煉：生質原料取代石化原料
- 製藥工程：連續製造取代批次生產

> 💡 化工將化學從「燒杯」放大到「工廠」——規模化是化工的核心命題。
