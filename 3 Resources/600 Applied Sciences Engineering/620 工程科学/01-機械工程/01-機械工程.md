# 01 — 機械工程

> 機械工程是工程學的基石——研究力、運動、能量和材料，設計和製造機械系統。

## 核心領域

| 領域 | 內容 |
|------|------|
| **靜力學** | 靜止物體的力平衡 |
| **動力學** | 運動物體的力與加速度 |
| **材料力學** | 應力、應變、強度 |
| **熱力學** | 能量轉換、熱機、冷凍循環 |
| **流體力學** | 液體與氣體的行為 |
| **機構設計** | 齒輪、連桿、凸輪 |
| **製造工程** | CNC、3D列印、鑄造、焊接 |

## 熱力學四大定律

| 定律 | 內容 |
|------|------|
| 第零 | 熱平衡的遞移性 |
| 第一 | 能量守恆 (ΔU = Q − W) |
| 第二 | 熵增原理，熱不能自發從冷到熱 |
| 第三 | 絕對零度不可達 |

## 製造方法

| 方法 | 說明 |
|------|------|
| 鑄造 | 液態金屬倒入模具 |
| 鍛造 | 高壓成形 |
| 切削 (CNC) | 減材製造 |
| 3D列印 | 增材製造 |
| 焊接 | 熔接連接 |

## 新興領域

- **機電整合** (Mechatronics)：機械 + 電子 + 控制
- **微機電系統** (MEMS)：微型感測器與致動器
- **軟性機器人**：仿生、柔性致動

> 💡 機械工程是「萬物皆可動」的學問——從鐘錶齒輪到火箭引擎。
