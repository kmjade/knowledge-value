# 09 — 工程概論

> 工程是將科學發現轉化為實用解決方案的藝術與科學。

## 什麼是工程？

工程是應用數學和自然科學原理，透過設計和建造來解決人類問題的專業。
- **科學**：發現自然規律 (Why?)
- **工程**：應用規律解決問題 (How?)

## 工程設計流程

```
需求定義 → 概念設計 → 細部設計 → 原型測試 → 生產部署 → 維護更新
```

## 工程倫理

| 原則 | 說明 |
|------|------|
| 公眾安全優先 | 保護公眾健康、安全與福祉 |
| 誠實正直 | 不誤導、不隱瞞風險 |
| 能力範圍 | 只在自身專業領域內執業 |
| 永續發展 | 考慮環境與後代影響 |

## 工程的歷史里程碑

| 時期 | 里程碑 |
|------|--------|
| 古代 | 金字塔、羅馬水道、長城 |
| 工業革命 | 蒸汽機、鐵路、鋼鐵結構 |
| 20世紀 | 汽車、飛機、電腦、核能、太空 |
| 21世紀 | AI、基因編輯、量子、新能源 |

> 💡 工程師不是解決教科書上已有答案的問題——而是解決從未被解決過的問題。
