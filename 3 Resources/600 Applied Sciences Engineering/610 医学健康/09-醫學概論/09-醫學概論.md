# 09 — 醫學概論

> 醫學是維護、促進和恢復人類健康的科學與實踐。

## 醫學簡史

| 時期 | 里程碑 |
|------|--------|
| 古代 | 希波克拉底誓言、黃帝內經、蓋倫解剖 |
| 中世紀 | 醫院興起、阿拉伯醫學黃金時代 |
| 文藝復興 | Vesalius 人體解剖、Harvey 血液循環 |
| 19世紀 | 麻醉、消毒、細菌理論 (Pasteur, Koch) |
| 20世紀 | 抗生素、疫苗、醫學影像、器官移植 |
| 21世紀 | 基因組學、免疫療法、AI 診斷 |

## 醫學模式

| 模式 | 核心理念 |
|------|---------|
| **生物醫學模式** | 疾病 = 生理異常 |
| **生物-心理-社會模式** | 健康 = 生理 + 心理 + 社會 |
| **實證醫學 (EBM)** | 決策基於最佳證據 |
| **精準醫學** | 基於基因和環境的個人化治療 |
| **中醫模式** | 陰陽平衡、整體調理 |

## 醫學倫理四大原則

| 原則 | 說明 |
|------|------|
| **自主** | 尊重患者決定權 |
| **行善** | 為患者最大利益行動 |
| **不傷害** | Primum non nocere (首先不傷害) |
| **公正** | 公平分配醫療資源 |

> 💡 醫學的本質是「有時治癒，經常緩解，總是安慰」— Cure sometimes, treat often, comfort always.
