# 04 — 臨床醫學

> 臨床醫學是醫學的核心——在病床邊診斷、治療和照護患者。

## 診斷流程

```
主訴 → 病史 → 理學檢查 → 鑑別診斷 → 檢驗/影像 → 確診
```

## 主要臨床科別

| 科別 | 範圍 |
|------|------|
| 內科 | 心臟、胸腔、腸胃、腎臟、內分泌... |
| 外科 | 手術治療 |
| 婦產科 | 女性生殖、產科 |
| 小兒科 | 嬰幼兒、兒童 |
| 急診醫學 | 緊急狀況初步處理 |
| 影像醫學 | X光、CT、MRI、超音波 |
| 病理科 | 組織/細胞診斷 |

## 常見診斷工具

| 工具 | 用途 |
|------|------|
| 血液檢查 | CBC、生化、電解質、發炎指標 |
| X光 | 骨骼、肺部 |
| 超音波 | 腹部、心臟、血管 |
| CT | 全身斷層掃描 |
| MRI | 軟組織、神經系統 |
| 內視鏡 | 腸胃道、呼吸道 |
| 心電圖 (ECG) | 心臟電活動 |

## 治療方式

| 方式 | 說明 |
|------|------|
| 藥物治療 | 最常見的第一線治療 |
| 手術 | 切除、修復、移植 |
| 放射治療 | 針對腫瘤的放射線 |
| 物理治療 | 復健、功能恢復 |
| 心理治療 | 認知行為、諮商 |

> 💡 臨床決策 = 證據 + 經驗 + 病人偏好。三者缺一不可。Evidence-based medicine is the integration of best research evidence with clinical expertise and patient values.
