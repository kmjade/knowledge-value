# 03 — 藥理學

> 藥理學研究藥物如何在體內作用——藥效學（藥物對身體做什麼）和藥動學（身體對藥物做什麼）。

## 藥物分類

| 類別 | 作用 | 範例 |
|------|------|------|
| **抗生素** | 殺菌/抑菌 | Penicillin, Amoxicillin |
| **止痛藥** | 緩解疼痛 | Ibuprofen, Morphine |
| **降血壓藥** | 擴張血管/減少心輸出 | ACEI, CCB |
| **降血糖藥** | 增加胰島素/降低血糖 | Metformin, Insulin |
| **抗癌藥** | 抑制細胞增生 | Chemotherapy, Targeted therapy |
| **抗憂鬱藥** | 調節神經傳遞物質 | SSRI (Prozac) |

## 藥動學 (ADME)

| 階段 | 說明 |
|------|------|
| **A**bsorption 吸收 | 藥物如何進入血液（口服、注射、吸入） |
| **D**istribution 分布 | 藥物在體內如何擴散 |
| **M**etabolism 代謝 | 肝臟如何分解藥物 |
| **E**xcretion 排泄 | 腎臟如何排出藥物 |

## 藥物安全

| 概念 | 說明 |
|------|------|
| 治療指數 (TI) | LD50/ED50；TI 越大越安全 |
| 副作用 | 非預期的藥理作用 |
| 藥物交互作用 | 兩種藥物互相影響 |
| 抗藥性 | 病原體/癌細胞不再對藥物反應 |
| 過敏反應 | 免疫系統對藥物的異常反應 |

> 💡 所有的藥物都是毒物——差別在於劑量。All substances are poisons; the dose makes the poison. — Paracelsus
