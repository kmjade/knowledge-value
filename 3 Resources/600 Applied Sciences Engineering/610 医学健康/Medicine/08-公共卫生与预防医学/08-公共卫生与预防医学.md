# 08-公共卫生与预防医学

> 公共卫生关注的是**人群**而非个体——"临床医学拯救一个患者，公共卫生拯救一百万人口。" 预防医学则聚焦于疾病发生之前的干预。

## 预防的三级模型

| 级别 | 时机 | 做什么 | 例子 |
|------|------|--------|------|
| **一级预防** | 疾病发生前 | 防止疾病发生 | 疫苗、健康教育、控烟 |
| **二级预防** | 疾病早期 | 早发现早治疗 | 癌症筛查（乳腺、宫颈、结直肠） |
| **三级预防** | 疾病发生后 | 减少并发症和残疾 | 糖尿病足护理、卒中康复 |

> 一级预防的性价比远高于治疗。疫苗的成本效益比是所有医疗干预中最高的之一。

## 流行病学

流行病学是公共卫生的核心科学——研究疾病在人群中的分布和决定因素。

### 关键指标

| 指标 | 含义 | 公式 |
|------|------|------|
| **发病率** | 新病例的比例 | 新病例数 / 风险人口 × 时间 |
| **患病率** | 现有病例的比例 | 现有病例数 / 总人口 |
| **死亡率** | 死亡的比例 | 死亡数 / 总人口 × 时间 |
| **病死率** | 得病后死亡的比例 | 死亡数 / 病例数 |

### 研究设计

| 类型 | 说明 | 证据强度 |
|------|------|----------|
| **队列研究** | 按暴露分组，前瞻追踪 | 高 |
| **病例对照** | 按结局分组，回顾暴露 | 中 |
| **横断面** | 同一时点的快照 | 较低 |
| **生态学研究** | 群体水平分析 | 最低（生态学谬误） |

### 基本繁殖数 R₀

一个感染者平均传染多少人。

| 疾病 | R₀ |
|------|-----|
| 季节性流感 | ~1.3 |
| COVID-19 (原始株) | ~2-3 |
| COVID-19 (Delta) | ~5-7 |
| 麻疹 | ~12-18 |

> 如果 R₀ > 1，疫情扩散。疫苗接种覆盖率需要 > 1 - 1/R₀ 才能实现群体免疫。

## 疫苗

### 疫苗类型

| 类型 | 原理 | 例子 |
|------|------|------|
| **减毒活疫苗** | 活但弱化的病原体 | 麻疹、风疹、水痘 |
| **灭活疫苗** | 杀死的病原体 | 脊髓灰质炎 (IPV)、甲肝 |
| **亚单位/重组** | 病原体的一部分 | HPV、乙肝 |
| **mRNA** | 教身体自己制造抗原 | COVID-19 (Pfizer/Moderna) |
| **病毒载体** | 用无害病毒递送抗原基因 | COVID-19 (阿斯利康) |

### 中国免疫规划

| 疫苗 | 预防疾病 |
|------|----------|
| 卡介苗 (BCG) | 结核病 |
| 乙肝疫苗 | 乙型肝炎 |
| 脊髓灰质炎疫苗 | 小儿麻痹 |
| 百白破 (DPT) | 百日咳、白喉、破伤风 |
| 麻腮风 (MMR) | 麻疹、腮腺炎、风疹 |
| 乙脑疫苗 | 流行性乙型脑炎 |

## 筛查的原则 (Wilson & Jungner, 1968)

不是所有疾病都适合筛查：

| 原则 | 说明 |
|------|------|
| 疾病是重要的健康问题 | 常见且严重 |
| 有可治疗的早期阶段 | 发现早期能改变结局 |
| 筛查方法可接受 | 安全、不太痛苦、不太贵 |
| 有确诊和治疗的设施 | 查出问题后能处理 |

> 符合这些原则的筛查：宫颈癌（巴氏涂片）、乳腺癌（钼靶）、结直肠癌（结肠镜/便潜血）。

## 全球健康

### 主要挑战

| 挑战 | 数据 |
|------|------|
| 孕产妇死亡 | 每天 ~800 人（绝大多数可预防） |
| 5 岁以下儿童死亡 | 每年 ~500 万（营养不良、肺炎、腹泻） |
| 结核病 | 每年 ~1000 万新发病例 |
| 疟疾 | 每年 ~2.5 亿病例 |
| HIV/AIDS | ~3900 万人带毒生存 |
