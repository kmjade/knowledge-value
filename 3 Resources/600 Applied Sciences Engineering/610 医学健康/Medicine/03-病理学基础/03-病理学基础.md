# 03-病理学基础

> 病理学是研究疾病的本质——**为什么会生病？细胞和组织发生了什么变化？** 这是连接基础医学和临床医学的桥梁。

## 疾病的四大病因

| 病因类型 | 说明 | 例子 |
|----------|------|------|
| **感染性** | 病原体入侵 | 细菌（肺炎链球菌→肺炎）、病毒（HBV→肝炎）、真菌、寄生虫 |
| **遗传性** | 基因突变 | 镰刀细胞贫血（单基因）、乳腺癌 BRCA1/2 突变 |
| **环境性** | 外部因素 | 吸烟→肺癌、石棉→间皮瘤、UV→皮肤癌 |
| **多因素** | 基因+环境 | 大多数慢性病（高血压、2 型糖尿病、大部分癌症） |

## 细胞损伤与死亡

### 细胞损伤的可逆与不可逆

```
正常细胞 → 应激/损伤 → 适应
                ↓（损伤超出自适应能力）
            可逆损伤（细胞肿胀、脂肪变）
                ↓（损伤持续/加重）
            不可逆损伤 → 坏死 或 凋亡
```

### 坏死 vs 凋亡

| 特征 | 坏死 (Necrosis) | 凋亡 (Apoptosis) |
|------|----------------|-----------------|
| 触发 | 严重损伤（缺血、毒素） | 编程性（发育、免疫、DNA 损伤） |
| 过程 | 细胞肿胀、膜破裂 | 细胞收缩、形成凋亡小体 |
| 炎症 | 引发炎症 | 不引发炎症（安静死亡） |
| 生理性 | 病理性 | 生理性+病理性 |

> 每天约有 500-700 亿个细胞通过凋亡被清除——这就是你每天都在"换新"而不引起炎症的原因。

## 炎症

炎症不是疾病——是保护性反应。但过度的、持续的炎症本身就是疾病。

### 急性炎症的五大经典体征

| 体征 | 拉丁文 | 机制 |
|------|--------|------|
| 红 | Rubor | 血管扩张 |
| 热 | Calor | 血流增加 |
| 肿 | Tumor | 血管通透性增加→液体渗出 |
| 痛 | Dolor | 炎症介质刺激神经末梢 |
| 功能障碍 | Functio laesa | 肿胀+疼痛+组织损伤 |

### 慢性炎症

持续数月到数年的炎症。关键特征：**炎症 + 组织破坏 + 修复同时进行**。

| 疾病 | 慢性炎症的角色 |
|------|---------------|
| 动脉粥样硬化 | 血管壁慢性炎症→斑块形成 |
| 类风湿关节炎 | 免疫系统攻击关节滑膜 |
| 慢性肝炎 | HBV/HCV 持续感染→肝硬化→肝癌 |
| 炎症性肠病 (IBD) | 肠道慢性炎症 |

## 肿瘤 (Neoplasia)

### 良性 vs 恶性

| 特征 | 良性 | 恶性（癌症） |
|------|------|-------------|
| 生长速度 | 慢 | 快 |
| 边界 | 有包膜、清晰 | 浸润性、不清 |
| 转移 | 无 | 有（通过血液/淋巴） |
| 细胞分化 | 接近正常 | 异常（去分化） |
| 复发 | 少 | 常见 |
| 致命性 | 通常不（除非压迫重要器官） | 是主要死因 |

### 癌的六大标志 (Hallmarks of Cancer)

Hanahan & Weinberg (2000, 2011):

1. 持续的增殖信号
2. 逃避生长抑制
3. 抵抗细胞死亡（凋亡）
4. 无限复制能力（端粒酶）
5. 诱导血管生成（为自己"修路"供血）
6. 激活侵袭和转移
7. （2011 新增）代谢重编程、逃避免疫

## 遗传病

| 类型 | 模式 | 例子 |
|------|------|------|
| 常染色体显性 | 一个突变等位基因就发病 | 亨廷顿病、马凡综合征 |
| 常染色体隐性 | 两个等位基因都突变才发病 | 镰刀细胞贫血、囊性纤维化 |
| X 连锁隐性 | 男性发病（只有一条 X） | 血友病 A、Duchenne 肌营养不良 |
| 线粒体遗传 | 母系遗传 | Leber 遗传性视神经病 |
