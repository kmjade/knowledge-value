# 05 — 公共衛生

> 公共衛生關注群體健康——從流行病學到衛生政策，從防疫到健康促進。

## 公共衛生核心功能

| 功能 | 說明 |
|------|------|
| **流行病學** | 研究疾病在人群中的分布與決定因素 |
| **衛生政策** | 制定與推動健康相關法規 |
| **健康促進** | 教育與環境改善 |
| **疾病監測** | 追蹤傳染病、慢性病趨勢 |
| **環境衛生** | 水、空氣、食品安全 |

## 流行病學指標

| 指標 | 定義 |
|------|------|
| 發生率 (Incidence) | 新病例數/人口/時間 |
| 盛行率 (Prevalence) | 現有病例數/人口 |
| 死亡率 (Mortality) | 死亡人數/人口 |
| 致死率 (CFR) | 死亡數/確診數 |
| R₀ (基本傳染數) | 一個感染者平均傳染多少人 |

## 流行病學研究設計

| 設計 | 說明 | 證據強度 |
|------|------|---------|
| RCT | 隨機對照試驗 | 最高 |
| Cohort | 追蹤暴露組與非暴露組 | 高 |
| Case-Control | 回溯比較病例與對照 | 中 |
| Cross-Sectional | 單一時點調查 | 低 |
| Case Series | 病例系列報告 | 最低 |

## 三級預防

| 層級 | 目標 | 範例 |
|------|------|------|
| **初級** | 防止疾病發生 | 疫苗、衛生教育 |
| **次級** | 早期發現 | 癌症篩檢 |
| **三級** | 減少併發症 | 復健、慢性病管理 |

> 💡 公共衛生的目標不是治療一個人，而是讓所有人更健康。Public health saves lives — millions at a time.
