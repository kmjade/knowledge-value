# 07 — 精神健康

> 精神健康是整體健康不可分割的一部分——沒有心理健康，就沒有真正的健康。
> There is no health without mental health.

## 常見精神疾病

| 疾病 | 核心特徵 | 盛行率 |
|------|---------|--------|
| **憂鬱症** | 持續低落、失去興趣、能量下降 | ~5% |
| **焦慮症** | 過度擔憂、恐慌、逃避行為 | ~7% |
| **躁鬱症** | 情緒極端波動（躁期+鬱期） | ~1% |
| **思覺失調症** | 幻覺、妄想、思維紊亂 | ~0.5% |
| **強迫症** | 反覆想法與行為 | ~2% |
| **PTSD** | 創傷後壓力反應 | ~4% |
| **失智症** | 認知功能退化 | >65歲 ~5–8% |

## 憂鬱症

| 面向 | 說明 |
|------|------|
| 情緒 | 悲傷、空虛、無望 |
| 生理 | 疲勞、食慾/睡眠改變 |
| 認知 | 注意力下降、自責、自殺意念 |
| 持續時間 | ≥ 2 週 |

## 治療方法

| 方法 | 說明 |
|------|------|
| **藥物** | SSRI、SNRI、抗精神病藥 |
| **心理治療** | CBT（認知行為治療）、IPT（人際治療） |
| **生活調整** | 運動、睡眠衛生、社交連結 |
| **rTMS/ECT** | 難治型憂鬱症的非藥物治療 |

## 心理健康維護

| 策略 | 說明 |
|------|------|
| 社交連結 | 維持有意義的人際關係 |
| 規律運動 | 每週 150 分鐘中強度運動 |
| 充足睡眠 | 7–9 小時 |
| 正念練習 | 冥想、呼吸練習 |
| 限制螢幕 | 減少社群媒體比較 |

> 💡 精神疾病不是軟弱的表現——它是需要治療的疾病，就像糖尿病和高血壓一樣。It's okay to not be okay.
