# 02 — 病理學

> 病理學研究疾病的本質——病因（為什麼生病）和發病機制（如何生病）。

## 疾病四大範疇

| 範疇 | 機制 | 範例 |
|------|------|------|
| **發炎** | 組織對損傷的反應 | 肺炎、關節炎 |
| **腫瘤** | 細胞不受控制增生 | 癌症 |
| **退化** | 組織功能逐漸喪失 | 阿茲海默症、骨關節炎 |
| **代謝異常** | 生化反應失調 | 糖尿病、甲狀腺疾病 |

## 細胞傷害機制

| 機制 | 說明 |
|------|------|
| 缺氧 | 氧氣供應不足（最常見原因） |
| 自由基 | 氧化壓力損傷細胞膜 |
| 發炎介質 | 過度發炎反應損害正常組織 |
| 基因突變 | DNA 損傷導致癌症或遺傳病 |
| 蛋白質錯誤折疊 | 導致細胞功能障礙 |

## 腫瘤學基礎

| 概念 | 說明 |
|------|------|
| 良性 vs 惡性 | 惡性 = 侵襲 + 轉移 |
| 致癌基因 | 促進細胞增生 |
| 抑癌基因 | 抑制細胞增生 (p53) |
| 轉移 | 癌細胞擴散至遠處器官 |

## 常見疾病機制

| 疾病 | 病理機制 |
|------|---------|
| 動脈粥狀硬化 | 脂質沉積→發炎→斑塊→狹窄/破裂 |
| 糖尿病 (Type 2) | 胰島素阻抗→高血糖→血管損傷 |
| 高血壓 | 血管收縮/硬化→心臟負荷增加 |
| 肝硬化 | 慢性損傷→纖維化→肝功能喪失 |

> 💡 病理學是醫學的「偵探工作」——從症狀和檢驗中推理疾病的根源。
