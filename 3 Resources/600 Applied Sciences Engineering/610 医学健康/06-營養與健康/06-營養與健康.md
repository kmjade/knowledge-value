# 06 — 營養與健康

> 營養是健康的基石——飲食影響從細胞代謝到慢性病風險的各個層面。

## 六大營養素

| 營養素 | 能量 | 功能 |
|--------|------|------|
| 碳水化合物 | 4 kcal/g | 主要能量來源 |
| 蛋白質 | 4 kcal/g | 組織修復、酶 |
| 脂肪 | 9 kcal/g | 細胞膜、激素 |
| 維生素 | 0 | 代謝輔因子 |
| 礦物質 | 0 | 骨骼、神經、酵素 |
| 水 | 0 | 溶解、運輸、調溫 |

## 飲食與慢性病

| 疾病 | 飲食因素 |
|------|---------|
| 肥胖 | 熱量過剩、精緻糖 |
| 糖尿病 Type 2 | 高 GI 飲食、含糖飲料 |
| 心血管疾病 | 飽和脂肪、反式脂肪、高鈉 |
| 高血壓 | 高鈉、低鉀 |
| 骨質疏鬆 | 鈣與維生素 D 不足 |
| 大腸癌 | 紅肉、加工肉品、低纖維 |

## 健康飲食模式

| 模式 | 特色 |
|------|------|
| 地中海飲食 | 橄欖油、魚、蔬果、全穀、堅果 |
| DASH 飲食 | 低鈉、高鉀鈣鎂，降血壓 |
| 植物性飲食 | 以植物為主，減少動物性食物 |

## 體重管理

| 概念 | 說明 |
|------|------|
| BMI | 體重(kg)/身高(m)²；18.5–24 正常 |
| 能量平衡 | 攝入 = 消耗 → 體重穩定 |
| TDEE | 每日總能量消耗 |

> 💡 最好的飲食是可以長期堅持的飲食——不存在魔法減肥法，只有持續的能量平衡。
