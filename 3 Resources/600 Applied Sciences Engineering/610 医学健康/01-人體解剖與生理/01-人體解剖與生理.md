# 01 — 人體解剖與生理

> 解剖學研究人體結構，生理學研究人體功能——兩者互為表裡。

## 人體系統

| 系統 | 主要器官 | 核心功能 |
|------|---------|---------|
| **循環** | 心臟、血管 | 運輸氧氣與養分 |
| **呼吸** | 肺、氣管 | 氣體交換 |
| **消化** | 胃、腸、肝 | 消化吸收 |
| **神經** | 腦、脊髓、神經 | 控制與協調 |
| **內分泌** | 甲狀腺、胰島 | 激素調節 |
| **骨骼** | 206 塊骨頭 | 支撐、保護、造血 |
| **肌肉** | 600+ 塊肌肉 | 運動、產熱 |
| **免疫** | 淋巴結、脾臟 | 防禦病原 |
| **泌尿** | 腎、膀胱 | 過濾廢物 |
| **生殖** | 卵巢/睪丸 | 繁衍後代 |

## 恆定 (Homeostasis)

人體維持內部環境穩定的能力：
- 體溫：36.5–37.5°C
- 血壓：120/80 mmHg
- 血糖：70–110 mg/dL (空腹)
- pH：7.35–7.45

## 細胞 → 組織 → 器官 → 系統

```
細胞 (40兆個) → 組織 (4大類) → 器官 (78個) → 系統 (11個)
```

> 💡 人體是已知最複雜的機器——了解它是醫學的第一步。
