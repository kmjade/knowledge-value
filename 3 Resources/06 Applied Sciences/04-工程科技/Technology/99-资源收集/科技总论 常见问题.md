# 科技总论 常见问题

## 十大常见误区

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| 科技 = 电子设备 | 科技包括任何系统性应用知识的方法——轮子、农业、印刷术都是科技 |
| 新技术总是更好 | 新技术更贵、更不稳定、生态不成熟。成熟技术往往更可靠 |
| 科技是中性的 | 科技的设计本身就嵌入了价值观——算法可以放大偏见，武器的存在改变了政治博弈 |
| 摩尔定律适用于一切 | 摩尔定律只描述了集成电路的晶体管密度——电池、材料、航空都没有指数增长 |
| AI 将取代所有工作 | 历史表明技术替代的是任务而非职业——但过渡期的痛苦是真实的 |
| 5G = 更快的 4G | 5G 的低延迟和网络切片能力开启了工业自动化、远程手术等新应用——不只是速度 |
| 开源 = 不安全 | 开源意味着更多人审查代码——安全隐患更快被发现。闭源 ≠ 安全 |
| 自动驾驶已经成熟 | L5（完全无人驾驶）仍未实现——技术瓶颈在边缘案例而非高速公路 |
| 量子计算即将取代经典计算机 | 量子计算只在特定问题上（因数分解、模拟）有优势——不会替代经典计算机 |
| 技术可以解决一切问题 | 技术是工具——气候变化、不平等、隐私等问题的核心是政治和社会选择 |
