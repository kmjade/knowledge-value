# 02-土木工程

> 土木工程是最古老的工程学科——从金字塔到港珠澳大桥，人类文明建立在土木工程之上。涉及建筑、桥梁、道路、隧道、水利和岩土。

## 分支领域

| 分支 | 做什么 | 代表项目 |
|------|--------|----------|
| **结构工程** | 设计承受荷载的骨架 | 高层建筑、桥梁、体育场 |
| **岩土工程** | 研究土壤和岩石的力学行为 | 地基、边坡、隧道 |
| **交通工程** | 规划道路、铁路和机场 | 高速公路、高铁线路 |
| **水利工程** | 水资源管理和防洪 | 水库、大坝、排水系统 |
| **环境工程** | 水处理、固废、污染 | 污水处理厂、垃圾填埋场 |

## 结构的荷载

| 荷载类型 | 说明 | 例子 |
|----------|------|------|
| **恒载** | 结构自身的重量 | 楼板、墙体 |
| **活载** | 人和可移动物品 | 人、家具、车辆 |
| **风荷载** | 风产生的力 | 高层建筑必须抗风 |
| **地震荷载** | 地震产生的惯性力 | 抗震设计 |
| **雪荷载** | 积雪重量 | 北方地区屋顶设计 |
| **温度荷载** | 热胀冷缩 | 桥梁伸缩缝 |

## 结构形式

| 形式 | 原理 | 例子 |
|------|------|------|
| **梁** | 承受弯矩 | 楼板梁、桥的主梁 |
| **柱** | 承受轴向压力 | 建筑立柱 |
| **桁架** | 三角形单元组成的框架 | 屋顶、桥梁 |
| **拱** | 将竖向荷载转为推力 | 拱桥、拱门 |
| **悬索** | 钢缆承受拉力 | 悬索桥（金门大桥） |
| **壳体** | 薄而弯曲的表面 | 穹顶、薄壳屋顶 |

> 为什么拱桥能用石头建造？石头抗压很强但抗拉很弱。拱把所有的力转化为压力——这恰好是石头擅长的。

## 建筑材料

| 材料 | 优点 | 缺点 | 典型应用 |
|------|------|------|----------|
| **混凝土** | 抗压、耐火、便宜 | 抗拉弱、重 | 基础、柱、大坝 |
| **钢筋混凝土** | 混凝土抗压+钢筋抗拉 | 会开裂、需要维护 | 绝大多数现代结构 |
| **钢** | 强度高、延性好 | 耐火差（需防火涂层）、易腐蚀 | 高层框架、桥梁 |
| **木材** | 可再生、轻 | 易燃、强度偏低 | 住宅、低层建筑 |
| **预应力混凝土** | 提前压紧混凝土→抗裂 | 施工复杂 | 大跨度桥梁、核电站 |
