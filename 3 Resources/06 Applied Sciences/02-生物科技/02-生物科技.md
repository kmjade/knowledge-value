# 02-生物科技

> 生物技术是利用生物体或其组成部分来制造产品的技术——从千年前的面包发酵到今天的 CRISPR 基因编辑，人类一直在"劫持"生物系统为我们服务。

## 生物技术的演进

| 时代 | 技术 | 例子 |
|------|------|------|
| **传统生物技术** (>1000 年前) | 发酵、育种 | 面包、啤酒、杂交水稻 |
| **经典生物技术** (1970s-2000s) | 重组 DNA、PCR | 胰岛素（大肠杆菌生产）、DNA 鉴定 |
| **现代生物技术** (2010s-) | 基因编辑、合成生物学、mRNA | CRISPR、实验室培养肉、mRNA 疫苗 |

## 核心技术

### CRISPR-Cas9

"基因剪刀"——2020 年诺贝尔化学奖。

```
工作原理:
1. guide RNA 引导 Cas9 蛋白到目标 DNA 序列
2. Cas9 在指定位置切割 DNA 双链
3. 细胞自身的修复机制在修复时产生突变（敲除基因）
   或者：同时提供模板 DNA → 精确编辑（插入/替换基因）
```

| 应用 | 现状 |
|------|------|
| 治疗遗传病（镰刀细胞贫血） | 已获批 (Casgevy, 2023) |
| 癌症免疫疗法（编辑 T 细胞） | 临床试验中 |
| 农业（抗病作物） | 已商业化 |
| 基因驱动（消灭蚊媒疾病） | 实验室/伦理争议中 |

### PCR (聚合酶链式反应)

在试管中扩增特定 DNA 片段——分子生物学的"复印机"。从一滴血中扩增出足够的 DNA 用于检测。

### DNA 测序

| 技术 | 年代 | 特点 |
|------|------|------|
| Sanger 测序 | 1977 | 金标准，慢但精确 |
| 下一代测序 (NGS) | 2005 | 高通量、低成本 |
| 三代测序 (PacBio, Nanopore) | 2010s | 长读长，实时 |

> 人类基因组计划 (1990-2003)：$2.7B。今天测一个人类基因组：~$500。

## 合成生物学

不满足于编辑现有基因——设计和建造全新的生物系统。

| 概念 | 说明 |
|------|------|
| 标准化生物部件 (BioBricks) | 像电子元件一样标准化的 DNA 片段 |
| 基因回路 | 编程细胞行为 |
| 最小基因组 | "一个生命最少需要多少基因？" (JCVI-syn3.0: 473 个基因) |

### 应用前景

| 领域 | 例子 |
|------|------|
| 医药 | 用酵母生产青蒿素（抗疟疾药） |
| 材料 | 蜘蛛丝蛋白（比钢铁强 5 倍） |
| 食品 | 精密发酵（人造奶蛋白、血红素） |
| 能源 | 工程藻类生产生物燃料 |

## 生物制药

| 类型 | 例子 | 说明 |
|------|------|------|
| **单克隆抗体** | Humira, Keytruda | 精确靶向特定蛋白——生物药的主力 |
| **mRNA 疫苗** | Pfizer/BioNTech, Moderna | 让身体自己生产抗原 |
| **细胞疗法** | CAR-T | 取患者 T 细胞，编辑后回输抗癌 |
| **基因疗法** | Luxturna, Zolgensma | 用病毒载体送入正确基因 |

## 伦理问题

| 议题 | 核心问题 |

| 基因歧视 | GATTACA 预言——保险/就业会以基因为标准吗？ |
