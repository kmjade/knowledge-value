# 农事（畜牧与养殖）

> **UDC 636**: Animal husbandry — 家畜、家禽的饲养、育种与管理

"农事"在中文传统中指**农家之事**——包括耕作与饲养。此处聚焦畜牧维度。

## 核心领域

| 分支 | 内容 |
|------|------|
| **家畜饲养** | 牛、猪、羊、马等大型家畜的营养、饲料、管理 |
| **家禽养殖** | 鸡、鸭、鹅——全球最大的动物蛋白来源 |
| **动物育种** | 遗传改良——选择、杂交、分子育种 |
| **动物营养** | 饲料科学——配方、添加剂、消化生理 |
| **动物健康** | 兽医预防、生物安全、人畜共患病 |
| **畜牧设施** | 圈舍设计、通风、粪污处理 |

## 核心概念

### 畜牧系统谱系

```
粗放 ←──────────────────────────→ 集约
  │                                    │
游牧、放牧      半舍饲      工业化集中养殖
低投入低产出                  高投入高产出
生态友好？                    效率但易有动物福利问题
```

### 育种三阶段

| 阶段 | 方法 | 速度 |
|------|------|------|
| **传统选育** | 根据表型选择配种 | 慢（数十年） |
| **数量遗传** | BLUP 育种值估计、人工授精 | 中（十年内） |
| **分子育种** | 基因组选择、标记辅助选择 | 快（数年） |

### 饲料转化率（FCR）

衡量畜牧效率的核心指标——生产 1kg 活重需要的饲料：

| 动物 | FCR |
|------|-----|
| 肉鸡 | ~1.6 |
| 猪 | ~2.5-3.0 |
| 牛 | ~6-10 |
| 鱼（养殖） | ~1.0-1.5 |

## 当代挑战

- **抗生素耐药性**: 养殖业是全球抗生素消费最大的部门之一
- **温室气体**: 反刍动物甲烷排放——全球 GHG 的 ~14%
- **动物福利**: 工业化养殖的密集约束——社会接受度下降
- **替代蛋白**: 植物肉、细胞培养肉对传统畜牧的挑战
