# 08-化工与材料

> 材料科学是研究材料的组成、结构、加工、性能之间关系的学科。人类文明的阶段以材料命名——石器时代、青铜时代、铁器时代——下一个也许是硅时代或碳时代。

## 材料的四面体

```
        组成 (Composition)
            /\
           /  \
          /    \
         /      \
结构 (Structure)──性能 (Properties)
         \      /
          \    /
           \  /
            \/
        加工 (Processing)
```

四个顶点互相影响——改变任何一个都会影响其他。

## 四大材料家族

| 家族 | 典型 | 关键特性 | 应用 |
|------|------|----------|------|
| **金属** | 钢、铝、钛合金 | 强度、延展性、导电 | 建筑、汽车、飞机 |
| **陶瓷** | 氧化铝、碳化硅、玻璃 | 硬、耐热、脆 | 刀具、涡轮叶片、手机屏幕 |
| **聚合物** | 塑料、橡胶、纤维 | 轻、易成型、便宜 | 包装、服装、管道 |
| **复合材料** | 碳纤维、玻璃钢 | 结合多种材料的优势 | 飞机机身、F1 赛车、钓竿 |

## 半导体材料

信息时代的基石：

| 材料 | 应用 | 特点 |
|------|------|------|
| **硅 (Si)** | CPU、内存、几乎所有芯片 | 便宜、工艺成熟 |
| **砷化镓 (GaAs)** | 射频芯片、LED | 电子迁移率更高 |
| **氮化镓 (GaN)** | 快充、5G 基站、雷达 | 高频、高压、耐热 |
| **碳化硅 (SiC)** | 电动车逆变器、电网 | 更高效率、更高温度 |
| **磷化铟 (InP)** | 光通信激光器 | 光子学应用 |

> 摩尔定律的放缓不是因为"造不出更小的晶体管"，而是因为量子隧穿效应——在 ~1nm 尺度，电子可以"穿墙"。新材料（二维材料、拓扑绝缘体）是延续摩尔定律的希望。

## 纳米材料：在原子尺度设计

> 纳米技术已归入本章——它本质是材料科学在 1-100nm 尺度的延伸。详见下文。

### 为什么纳米尺度特别？

| 效应 | 说明 |
|------|------|
| **表面效应** | 纳米颗粒 90% 的原子在表面——比表面积巨大 |
| **量子效应** | 电子被限制→离散能级→新光学/电学性质 |
| **分子识别** | 纳米尺度与 DNA、蛋白质天然匹配 |

> 金：宏观是黄色的。纳米金颗粒：可以是红色的（13nm）或紫色的（50nm）——颜色取决于尺寸，而非化学组成。

### 纳米材料的维度分类

| 维度 | 类型 | 例子 | 应用 |
|------|------|------|------|
| **0D** (三维都纳米) | 纳米颗粒、量子点 | 金纳米颗粒、CdSe 量子点 | 医学成像、显示器 |
| **1D** (两维纳米) | 纳米管、纳米线 | 碳纳米管、硅纳米线 | 晶体管、传感器 |
| **2D** (一维纳米) | 纳米薄膜 | 石墨烯、MoS₂ | 柔性电子、过滤膜 |
| **3D** (纳米结构) | 纳米多孔材料 | 沸石、金属有机框架 (MOF) | 催化、储气 |

### 纳米电子学：晶体管的极限

| 节点 | 年份 | 特点 |
|------|------|------|
| 7nm | 2018 | FinFET |
| 5nm | 2020 | FinFET |
| 3nm | 2022 | GAA (Gate-All-Around) |
| 2nm | 2025 | GAA, 背面供电 |
| <1nm | 未来 | 量子隧穿主导——需要全新范式 |

> 在 <3nm 节点，晶体管栅极只有几十个原子宽。从"制造晶体管"过渡到"排列原子"。

### 纳米技术的应用

| 领域 | 应用 | 现状 |
|------|------|------|
| **靶向给药** | 纳米颗粒+抗体识别癌细胞 | 已上市 (Doxil) |
| **mRNA 疫苗** | 脂质纳米颗粒 (LNP) 递送 | 已大规模使用 |
| **量子点显示** | QLED 电视——色彩更纯 | 已商业化 |
| **催化** | MOF/纳米颗粒→更高效率 | 工业应用 |

### 纳米技术的风险

| 风险 | 说明 |
|------|------|
| 纳米毒性 | 高反应性+穿透生物屏障→潜在健康风险 |
| 环境累积 | 纳米颗粒在环境中的行为不明确 |
| 监管空白 | 许多纳米产品缺乏专门安全评估 |

| 材料 | 特点 | 潜在应用 |
|------|------|----------|
| **石墨烯** | 单原子层碳，最强、最导电 | 柔性电子、超级电容 |
| **碳纳米管** | 管状石墨烯，强度是钢的 100 倍 | 太空电梯（理论）、晶体管 |
| **拓扑绝缘体** | 表面导电、内部绝缘 | 量子计算 |
| **钙钛矿** | 光伏效率飙升（25%+） | 下一代太阳能电池 |
| **形状记忆合金** | 变形后加热恢复原状 | 医疗支架、航天器展开机构 |
| **自修复材料** | 受损后自动修复 | 涂层、混凝土、电子产品 |
| **超材料** | 天然材料不具备的性质（负折射率） | 隐形斗篷、完美透镜 |

## 材料的失效

| 模式 | 原因 | 例子 |
|------|------|------|
| **疲劳** | 反复应力导致裂纹扩展 | 飞机金属疲劳（彗星号客机事故） |
| **蠕变** | 高温下持续负载导致变形 | 涡轮叶片的蠕变 |
| **腐蚀** | 化学/电化学反应 | 生锈 |
| **脆性断裂** | 低温或无预警的快速断裂 | 泰坦尼克号（铆钉低温变脆） |
