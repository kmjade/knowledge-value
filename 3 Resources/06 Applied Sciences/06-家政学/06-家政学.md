# 06-家政学

> 家政学（Home Economics / Family and Consumer Sciences）是将科学、经济学和管理学应用于家庭生活的学科——从营养膳食到家庭预算，从纺织品护理到儿童发展。在中国学科目录中属于"应用科学"门类。

## 学科全景

| 分支 | 研究什么 |
|------|----------|
| **食品与营养** | 膳食设计、食品安全、营养搭配 |
| **家庭经济** | 预算管理、消费决策、家庭财务 |
| **居住与环境** | 住宅设计、室内环境、家居安全 |
| **纺织与服装** | 织物性能、服装护理、功能性服装 |
| **儿童发展与家庭关系** | 育儿科学、家庭沟通、老龄化照护 |
| **家庭管理** | 时间管理、家务效率、家庭标准化 |

## 食品与营养

### 三大营养素

| 营养素 | 功能 | 来源 | 每克能量 |
|--------|------|------|----------|
| **碳水化合物** | 主要能量来源 | 米面、薯类、糖 | 4 kcal |
| **蛋白质** | 构建和修复组织 | 肉、蛋、豆、奶 | 4 kcal |
| **脂肪** | 能量储存、激素合成 | 油、坚果、肥肉 | 9 kcal |

### 均衡膳食

| 原则 | 说明 |
|------|------|
| 食物多样 | 每天 12 种以上食物 |
| 谷类为主 | 占总能量 50-65% |
| 蔬菜水果 | 每天 300-500g 蔬菜 + 200-350g 水果 |
| 适量蛋白 | 鱼/禽/蛋/瘦肉 + 大豆 |
| 少盐少油 | 盐 <5g/天，油 25-30g/天 |
| 足量饮水 | 每天 1.5-2L |

> 数据来源：《中国居民膳食指南 (2022)》

### 食品安全

| 危险温度区 | 说明 |
|-----------|------|
| 5-60°C | 细菌快速繁殖——食物不宜在此区间超过 2 小时 |
| 冷藏 | ≤4°C——抑制细菌 |
| 冷冻 | ≤-18°C——停止细菌生长 |

## 家庭经济管理

### 家庭预算模型

```
月收入
├── 储蓄/投资 (20%)
├── 住房 (25-30%)
├── 食品 (15-20%)
├── 交通 (10-15%)
├── 医疗/保险 (5-10%)
├── 教育/娱乐 (5-10%)
└── 其他 (5-10%)
```

### 消费决策框架

| 原则 | 说明 |
|------|------|
| 区分需要 vs 想要 | 需要 = 维持生活必需的；想要 = 提升品质的 |
| 比较单价 | 不看包装价格，看每克/每毫升价格 |
| 考虑全生命周期成本 | 便宜的电器耗电多——总成本可能更高 |
| 延迟满足 | 大额消费等 48 小时再做决定 |

## 居住与环境

| 方面 | 要点 |
|------|------|
| 室内空气质量 | 通风 (每天 2-3 次)、控制湿度 40-60%、少用 VOC 材料 |
| 照明 | 自然光优先、色温 2700K (暖) 用于休息区、4000K (冷) 用于工作区 |
| 收纳 | 分类→定位→标签——所有物品有固定位置 |
| 安全 | 烟雾报警器、灭火器、急救箱 |

## 纺织与服装

| 纤维类型 | 特点 | 洗涤 |
|----------|------|------|
| **棉** | 吸汗、透气、易皱 | 可机洗、可高温 |
| **羊毛** | 保暖、弹性好 | 冷水/干洗、易缩水 |
| **涤纶** | 耐磨、快干、不吸汗 | 可机洗、低温 |
| **丝绸** | 光滑、吸湿、娇贵 | 手洗/干洗 |

### 衣物护理标识

| 符号 | 含义 |
|------|------|
| 🪣 盆 | 可水洗——数字=最高水温 |
| △ | 可漂白 |
| □ | 可烘干——点数=温度 |
| 🔲 熨斗 | 可熨烫——点数=温度 |
| ○ | 可干洗——字母=溶剂类型 |

## 时间与家务管理

| 原则 | 方法 |
|------|------|
| 批量处理 | 同类家务集中做——一次性备餐、集中洗衣 |
| 15 分钟法则 | 不想做时告诉自己"只做 15 分钟"——通常做起来就停不下了 |
| 分工与授权 | 家庭成员分工——不是一个人的事 |
| 标准化 | 固定流程——购物清单模板、每周菜单循环 |

## 中国家庭变迁

| 指标 | 1980s | 2020s |
|------|-------|-------|
| 平均家庭规模 | 4.4 人 | 2.6 人 |
| 女性劳动参与 | 高 | 仍然高 (全球前列) |
| 双职工家庭 | 少数 | 多数 |
| 家务时间 | 女性>>男性 | 差距缩小但仍存在 |
| 外卖渗透率 | 0% | 50%+ (城市) |
