# MicroROS机器人控制板

> **产品厂商**: 深圳市亚博智能科技有限公司（Yahboom）
>
> **产品类型**: 机器人控制板 / MicroROS开发平台

---

## 📋 产品概览

### 核心特性

- ✅ **支持虚拟机主控远程MicroROS应用**
- ✅ **支持多ROS主控**: 树莓派5/Jetson 等
- ✅ **板载IMU六轴姿态传感器/雷达接口**
- ✅ **支持四路编码器电机，2路PWM舵机**

### 技术参数

| 特性 | 参数 |
|------|------|
| 主控芯片 | ESP32 |
| IMU传感器 | 六轴姿态传感器 |
| 雷达接口 | 支持 |
| 电机驱动 | 四路编码器电机 |
| 舵机控制 | 2路PWM舵机 |
| ROS支持 | MicroROS / ROS2 |
