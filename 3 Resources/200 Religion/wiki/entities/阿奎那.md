# 托馬斯·阿奎那 Thomas Aquinas
中世紀最重要的神學家與哲學家。

## 主要貢獻
- 《神學大全》— 天主教神學的權威著作
- 五路論證：五種證明神存在的方式
- 調和亞里士多德哲學與基督教神學

## Sources
- 阿奎那《神學大全》(1265-1274)
