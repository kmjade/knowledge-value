## 学科简史

### 哲学 → 心理学的分化

| 时期 | 事件 |
|------|------|
| 古希腊 | 柏拉图、亚里士多德：灵魂、知识、德性的哲学思辨 |
| 17 世纪 | 笛卡尔「我思故我在」— 心物二元论的确立 |
| 18 世纪 | 康德：现象界 vs 物自身，认知的先验条件 |
| **1879** | **冯特在莱比锡建立第一个心理学实验室 — 心理学独立** |
| 1890 | James《心理学原理》— 美国功能主义心理学 |
| 1900 | 弗洛伊德《释梦》— 精神分析的诞生 |
| 1913 | Watson 宣言行为主义 — 只研究可观察行为 |
| 1950s | 认知革命 — 心智重新成为科学对象 |
| 1990s | 认知神经科学 — 大脑、心智与行为的整合 |

### DDC 中的分类沿革

| 版本 | 变化 |
|------|------|
| DDC 1st (1876) | Class 1 = Philosophy，心理学为 150 |

## 关键跨域概念

| 概念 | 哲学视角 | 心理学视角 |
|------|----------|------------|
| **意识** | 现象学：意识的结构和意向性 | 认知心理学：意识的信息加工模型 |
| **自由意志** | 相容论 vs 不相容论 vs 决定论 | 意志的神经机制、延迟满足实验 |
| **自我** | 人格同一性：什么使我是我 | 自我概念、自尊、自我效能 |
| **道德** | 功利主义、义务论、美德伦理 | 道德发展 (Kohlberg)、道德直觉 (Haidt) |
| **知识** | JTB 理论、怀疑论、自然化认识论 | 认知偏差、元认知、学习理论 |
| **情感** | 斯多葛主义、情绪与理性 | 情绪的基本维度、情绪调节 |
