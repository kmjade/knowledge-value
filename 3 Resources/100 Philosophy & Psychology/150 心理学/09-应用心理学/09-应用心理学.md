# 09-应用心理学

> 如果把基础心理学比作物理学，应用心理学就是工程学——用科学原理解决现实世界的问题。

## 主要应用领域一览

| 领域 | 核心问题 | 典型工作 |
|------|----------|----------|
| **教育心理学** | 人如何学习？如何教得更好？ | 课程设计、学习评估、特殊教育 |
| **工业与组织心理学（I/O）** | 如何让人在工作上更幸福、更高效？ | 招聘、培训、领导力发展 |
| **健康心理学** | 心理因素如何影响身体健康？ | 压力管理、慢性病行为干预 |
| **消费心理学** | 消费者为什么买？如何决策？ | 广告设计、用户体验研究 |
| **司法心理学** | 心理学如何在法律中应用？ | 目击者证词评估、犯罪心理画像 |
| **积极心理学** | 人如何变得更幸福、更有意义？ | 幸福感研究、优势培养 |
| **运动心理学** | 心理如何影响运动表现？ | 竞赛焦虑管理、团队凝聚力 |

## 教育心理学

### 学习的关键原理

| 原理 | 说明 | 实用建议 |
|------|------|----------|
| **间隔效应** | 分散学习比集中学习效果好 | 不要考前通宵复习，每天学一点 |
| **测试效应** | 提取练习比重复阅读效果好 | 用自测代替反复看笔记 |
| **交错练习** | 混合不同题型比集中练习一种效果好 | 数学题混合加减乘除而非分类练习 |
| **生成效应** | 自己产出答案比看答案记得牢 | 先尝试回答再看答案 |
| **双重编码** | 图文结合比纯文字效果好 | 用思维导图、信息图表学习 |

### 成长型思维（Growth Mindset）

**Carol Dweck** 的研究：

| 信念 | 面对困难时的反应 |
|------|------------------|
| **固定型思维**："智力是固定的" | 回避挑战、轻易放弃 |
| **成长型思维**："智力可以发展" | 迎接挑战、从失败中学习 |

> 夸孩子"你真聪明"可能适得其反——让他们害怕挑战以免失去聪明标签。夸"你真努力"会培养成长型思维。

## 工业与组织心理学（I/O）

### 动机理论在工作中的应用

| 理论 | 核心 | 对管理的启示 |
|------|------|-------------|
| **自我决定理论** (SDT) | 内在动机需要三要素：自主、胜任、关系 | 给员工自主权 > 只给钱 |
| **目标设置理论** | 具体 + 有挑战但可达的目标效果最好 | 模糊的"尽力而为"不如"提高 15%" |
| **期望理论** | 动机 = 期望 × 价值 | 员工需要看到努力 → 绩效 → 奖励的链条 |

### 人格与工作绩效

- **尽责性** — 所有工作绩效的最强预测因素（r ≈ 0.2-0.3）
- **外向性** — 销售和管理岗位的成功预测因素
- **认知能力** — 复杂工作绩效的最佳单一预测因素

## 健康心理学

### 压力与应对

| 压力类型 | 说明 |
|----------|------|
| **急性压力** | 短期、有明确的开始和结束 |
| **慢性压力** | 持续存在的压力源（工作不满意、经济困难） → 危害最大 |

### 压力管理策略

| 策略 | 说明 | 实证支持 |
|------|------|----------|
| 运动 | 降低皮质醇、释放内啡肽 | 强 |
| 正念冥想 | 减少反刍思维，增加当下觉察 | 强（MBSR 有 RCT 支持） |
| 社会支持 | 有人可以倾诉 | 强——预测寿命和健康的独立因素 |
| 认知重评 | 改变对压力事件的解读 | 强（CBT 核心技术） |

### 安慰剂效应

> 信念可以让大脑释放真正的止痛物质（内啡肽）——安慰剂不只是心理的，它也有生物学基础。

## 积极心理学

Seligman 的 **PERMA 模型**（幸福感五要素）：

| 要素 | 含义 |
|------|------|
| **P**ositive Emotion（积极情绪） | 体验快乐、感恩、宁静 |
| **E**ngagement（投入） | 沉浸在心流中（见 Csikszentmihalyi） |
| **R**elationships（关系） | 高质量的人际关系 |
| **M**eaning（意义） | 服务于大于自己的目标 |
| **A**ccomplishment（成就） | 追求和达成目标带来的满足 |

## 司法心理学

| 应用 | 心理学发现 |
|------|-----------|
| 目击者证词 | 信心 ≠ 准确性。事后信息可以改写记忆 |
| 虚假供述 | 某些审讯技术会导致无辜者供认 |
| 陪审团决策 | 事前态度和故事叙事方式影响判决 |

## 消费心理学

| 效应 | 应用 |
|------|------|
| 锚定效应 | 先展示高价商品，再推荐"实惠"款 |
| 稀缺效应 | "限量""仅剩 2 件" 增加购买欲 |
| 峰终定律 | 体验的评价取决于峰值和结尾，而非平均值 |
| 选择过载 | 选项太多反而降低购买率 |
