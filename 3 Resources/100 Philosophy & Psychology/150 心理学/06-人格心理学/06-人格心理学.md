# 06-人格心理学

> 人格是使一个人区别于他人的、持久而独特的思维、情感和行为模式。人格心理学试图回答：**是什么让你成为独一无二的你？**

## 四大取向速览

| 取向 | 核心问题 | 代表学者 | 方法 |
|------|----------|----------|------|
| **特质** | 人格由哪些基本维度组成？ | Allport, Cattell, Eysenck | 因素分析 |
| **精神分析** | 潜意识如何驱动行为？ | Freud, Jung, Adler | 临床观察 |
| **人本** | 人如何实现潜能和成长？ | Rogers, Maslow | 治疗对话 |
| **社会认知** | 认知和环境如何交互塑造人格？ | Bandura, Mischel | 实验 |

## 特质理论

### Big Five（大五人格，OCEAN）— 当前学界主流

| 特质 | 低分 | 高分 | 生活相关 |
|------|------|------|----------|
| **开放性** (Openness) | 喜欢常规、传统 | 好奇、富有想象力、审美 | 创意工作绩效 |
| **尽责性** (Conscientiousness) | 无计划、散漫 | 自律、有条理、可靠 | 工作绩效最强预测因素 |
| **外向性** (Extraversion) | 安静、独处偏好 | 健谈、喜欢社交、活力充沛 | 销售岗位绩效 |
| **宜人性** (Agreeableness) | 竞争、质疑 | 合作、信任、共情 | 团队协作 |
| **神经质** (Neuroticism) | 情绪稳定、平静 | 焦虑、情绪波动 | 生活满意度、婚姻质量 |

> Big Five 有跨文化的一致性，且在整个成年期相对稳定（虽然可以变化）。

### 其他特质理论

| 理论 | 学者 | 核心内容 |
|------|------|----------|
| 16PF（16 种人格因素） | Cattell | 用因素分析提取 16 个基本特质 |
| PEN 三因素 | Eysenck | 精神质、外向性、神经质 |
| 黑暗三人格 | Paulhus & Williams | 自恋、马基雅维利主义、精神病态——与社交攻击相关 |

## 精神分析取向

### Freud 的结构模型

```
┌──────────────────────────────┐
│          意识                 │
│     ┌──────────┐             │
│     │   自我    │ ← 现实原则  │
│     │  (Ego)   │   协调者    │
│     └──────────┘             │
│  ┌────┐              ┌────┐  │
│  │本我│ ← 快乐原则    │超我│  │
│  │(Id)│   我想要！    │(Superego)│
│  └────┘              │应该怎样│
│                       └────┘  │
│     潜意识                    │
└──────────────────────────────┘
```

### 防御机制（简化）

| 防御机制 | 含义 | 例子 |
|----------|------|------|
| 压抑 | 把不愉快的推到潜意识 | 忘记童年创伤 |
| 投射 | 把自己不能接受的冲动归给别人 | "不是我讨厌他，是他讨厌我" |
| 合理化 | 给不可接受的行为找合理借口 | 没考上 → "那个学校其实也不怎么样" |
| 转移 | 把情绪转向安全的目标 | 被老板骂了回家对伴侣发火 |
| 升华（最成熟） | 把冲动转化为社会接受的活动 | 攻击冲动 → 成为外科医生或运动员 |

### 新弗洛伊德学派

| 学者 | 与 Freud 的区别 |
|------|----------------|
| **Jung** | 集体潜意识、原型、内外向 |
| **Adler** | 强调社会因素、自卑感与补偿、出生顺序 |
| **Horney** | 反对"阴茎羡妒"，强调文化和社会关系 |

## 人本取向

| 理论 | 核心 |
|------|------|
| **Rogers 自我理论** | 人需要无条件积极关注（Unconditional Positive Regard）。条件性积极关注导致"价值条件化"——做着别人期待的自己 |
| **Maslow 需求层次** | 生理→安全→归属→尊重→自我实现。只有基本需求满足，高层的自我实现才成为驱动力 |

> 人本主义的核心信念：**人性本善，每个人内心都有朝向成长的积极力量。**

## 社会认知取向

### Bandura 的交互决定论

```
        个人因素（认知、信念）
            ↙        ↘
    行为 ────────── 环境
```

三者互相影响，没有单向的因果。

### 关键概念

| 概念 | 含义 |
|------|------|
| **自我效能感** | 对自己"能完成某事"的信念——Bandura 认为这是行为最强的预测因素 |
| **观察学习** | 通过观察他人行为及其后果来学习（Bobo doll 实验） |
| **控制点** | 内控（自己的行为决定结果）vs 外控（命运/运气决定结果） |

## 人格测评

| 工具 | 类型 | 用途 |
|------|------|------|
| NEO-PI-R | 自陈量表（Big Five） | 科研、职业评估 |
| MMPI-2 | 自陈量表（临床） | 精神病理评估 |
| 罗夏墨迹测验 | 投射测验 | （争议大，效度不足） |
| TAT（主题统觉测验） | 投射测验 | 测量成就动机等 |

## 情境 vs 人格之争

> "人的行为到底是人格决定的，还是情境决定的？"

- **Mischel 的情境主义**：跨情境的行为一致性只有 r ≈ 0.3。情境比我们以为的更重要。
- **交互作用论**（现代共识）：行为 = 人格 × 情境。两者不可分。
