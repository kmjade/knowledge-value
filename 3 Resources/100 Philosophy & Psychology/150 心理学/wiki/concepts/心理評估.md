# 心理評估 Psychological Assessment

> 你不能管理你無法測量的東西。——心理測量學的核心前提。

## 定義

心理評估是使用標準化工具和方法來**測量心理特質、能力和狀態**的系統化過程。

## 測量品質三大指標

### 信度 Reliability — 一致性

| 類型 | 說明 |
|------|------|
| **再測信度** | 不同時間測量結果一致 |
| **內部一致性** | 同一測驗內各題目的一致性 (Cronbach's α > 0.7) |
| **評分者信度** | 不同評分者的結果一致 |

### 效度 Validity — 準確性

| 類型 | 說明 |
|------|------|
| **內容效度** | 測驗是否覆蓋了全部相關內容 |
| **效標效度** | 與外部標準的一致性（預測效度） |
| **構念效度** | 是否真正測量了目標構念 |

### 標準化 Standardization

測驗需在**統一條件**下施測，並有**常模 Norms** 進行比較。

## 常用評估工具

| 工具 | 測量內容 | 類型 |
|------|---------|------|
| **WAIS-IV** | 智力 (IQ) | 個別施測、16–90 歲 |
| **MMPI-2** | 人格與精神病理 | 自評、567 題 |
| **NEO-PI-R** | Big Five 人格 | 自評、240 題 |
| **Rorschach** | 人格動力（投射） | 墨跡測驗 |
| **Beck Depression Inventory** | 憂鬱程度 | 自評、21 題 |
| **Raven's Matrices** | 流體智力 | 非語言、文化公平 |

## 智力理論

| 理論 | 主張 |
|------|------|
| **Spearman** (1904) | g 因子（一般智力）+ s 因子（特殊能力） |
| **Gardner** (1983) | 多元智能（語言、邏輯、空間、音樂…） |
| **Sternberg** (1985) | 三元論（分析、創造、實踐） |

## 倫理考量

- **知情同意** — 被評估者知道評估目的
- **保密** — 結果僅用於授權目的
- **能力** — 評估者需有相應資格
- **文化公平** — 避免文化偏差
