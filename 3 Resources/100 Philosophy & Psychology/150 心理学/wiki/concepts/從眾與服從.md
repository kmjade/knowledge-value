# 從眾與服從 Conformity & Obedience

> 人有兩種屈服於社會壓力的方式：從眾（你自願跟隨）和服從（你被命令跟隨）。

## 從眾 Conformity — Asch 實驗 (1951)

### 實驗設計

- 參與者與 7 個假被試（演員）一起判斷線段長度
- 演員一致給出明顯錯誤的答案
- 測試真實參與者是否屈從於群體壓力

### 結果

| 指標 | 數據 |
|------|:--:|
| 至少一次從眾 | **75%** |
| 平均從眾率 | 37% |
| 完全獨立（從不從眾） | 25% |

### 影響從眾的因素

| 因素 | 效果 |
|------|------|
| **群體規模** | 3–5 人最有效，超過後效果趨平 |
| **一致性** | 只要有一個異議者，從眾率大幅下降 |
| **任務難度** | 越難越從眾 |
| **文化** | 集體主義文化從眾率高於個人主義文化 |

## 服從 Obedience — Milgram 實驗 (1963)

### 實驗設計

- 參與者扮演"教師"，對"學習者"（演員）施加電擊
- 電擊從 15V 增加到 450V（標記為 XXX 危險）
- 學習者表現出痛苦、哀求停止、最終沉默
- 實驗者要求繼續：「實驗需要你繼續」

### 結果（驚人）

| 指標 | 數據 |
|------|:--:|
| 施加到最大 450V | **65%** |
| 無人在 300V 之前停止 | 0% |
| 專家預測的服從率 | 僅 1–3%（嚴重低估） |

### 影響服從的因素

| 因素 | 服從率 |
|------|:--:|
| 實驗者在場 | 65% |
| 實驗者通過電話指令 | 21% |
| 學習者在同一房間 | 40% |
| 教師需強按學習者的手 | 30% |
| 其他教師拒絕繼續 | 10% |

## 倫理爭議

兩項實驗引發了心理學研究的倫理改革：
- **知情同意**：參與者必須了解研究風險
- **事後解釋 Debriefing**：必須告知真實研究目的
- **退出權**：隨時可以退出（Milgram 實驗中被弱化）
- **IRB 審查**：所有研究須經倫理委員會批准

## 從眾 vs 服從

| | 從眾 | 服從 |
|---|------|------|
| 壓力來源 | 同儕群體（橫向） | 權威（縱向） |
| 行為性質 | 模仿、跟隨 | 執行命令 |
| 意識程度 | 常常無意識 | 有意識 |
| 經典實驗 | Asch (1951) | Milgram (1963) |
