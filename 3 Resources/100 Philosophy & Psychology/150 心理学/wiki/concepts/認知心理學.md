# 認知心理學 Cognitive Psychology

> 心智是一個資訊處理系統——輸入→處理→輸出。

## 定義

認知心理學研究人類如何**感知、注意、記憶、思考、解決問題和使用語言**。它將心智類比為電腦的資訊處理系統。

## 核心領域

### 記憶模型 Memory Models

| 模型 | 容量 | 持續時間 | 編碼 |
|------|:--:|:--:|------|
| **感覺記憶** | 極大 | <1 秒 | 感官原始形式 |
| **短期記憶 STM** | 7±2 個組塊 | ~20 秒 | 聲學為主 |
| **工作記憶 WM** | 4±1 個組塊 | 主動維持 | 中央執行 + 語音/視空 |
| **長期記憶 LTM** | 無限 | 永久 | 語義/情節/程序 |

### 注意力 Attention

| 類型 | 說明 |
|------|------|
| **選擇性注意** | Cherry 的雞尾酒會效應 (1953) |
| **分配性注意** | 同時處理多個任務的能力 |
| **持續性注意** | 長時間維持注意力（如雷達監控） |

> **Stroop 效應** (1935)：說出字的顏色比說出字本身更慢——自動化處理干擾控制處理。

## 語言 Language

| 理論 | 主張 |
|------|------|
| **Chomsky** (1957) | 語言習得裝置 LAD、普遍語法 |
| **Skinner** (1957) | 語言是操作制約的產物 |
| **Sapir-Whorf** | 語言決定思維（語言相對論） |

## 認知革命 Cognitive Revolution

1950–1970 年代，心理學從行為主義轉向認知研究——**Ulric Neisser** 的《認知心理學》(1967) 標誌著學科的正式誕生。
