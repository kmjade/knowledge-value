# 臨床心理學 Clinical Psychology

> 整合科學、理論與實踐——理解、預防和緩解心理困擾。

## 定義

臨床心理學是應用心理學的分支，專注於**心理障礙的評估、診斷、治療和預防**。

## DSM-5 分類系統

《精神疾病診斷與統計手冊》第五版 (DSM-5, 2013) 是當前標準：

| 類別 | 代表障礙 |
|------|---------|
| **焦慮障礙** | 廣泛性焦慮、恐慌症、社交焦慮、特定恐懼症 |
| **情感障礙** | 重度憂鬱症、雙相障礙 (Bipolar) |
| **精神分裂譜系** | 精神分裂症、分裂情感障礙 |
| **創傷相關** | PTSD、急性壓力障礙 |
| **強迫相關** | OCD、囤積症、身體畸形障礙 |
| **人格障礙** | 邊緣型、自戀型、反社會型 |
| **神經發育** | ADHD、自閉症譜系 (ASD) |

## 主要治療取向

| 取向 | 核心理念 | 適用 |
|------|---------|------|
| **CBT 認知行為治療** | 改變失功能思維→改變行為 | 憂鬱、焦慮、OCD (實證最強) |
| **精神動力** | 探索無意識衝突 | 人格障礙、長期問題 |
| **人本主義** | 無條件接納、自我實現 | 成長性困擾 |
| **藥物治療** | 神經化學平衡 | 精神分裂、雙相、重度憂鬱 |
| **DBT 辯證行為治療** | 情緒調節 + 正念 | 邊緣型人格 (BPD) |

## 療效研究 Evidence-Based Practice

| 療法 | 障礙 | 效果量 |
|------|------|:--:|
| CBT | 憂鬱症 | 大 (d=0.8) |
| CBT | 焦慮症 | 大 |
| 暴露療法 | 恐懼症 | **非常大** (d=1.0+) |
| DBT | BPD | 中到大 |
| 正念 (MBCT) | 憂鬱復發預防 | 中 |

> **Dodo Bird 效應**：多數療法效果相當——治療關係的品質 > 特定技術。
