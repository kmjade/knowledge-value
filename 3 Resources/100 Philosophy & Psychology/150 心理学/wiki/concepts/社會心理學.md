# 社會心理學 Social Psychology

> 一個人的行為不是由其性格決定的，而是由其所處的情境。——Kurt Lewin

## 定義

社會心理學研究**個體的思想、情感和行為如何被他人（真實的、想像的或隱含的）所影響**。

## 核心主題

### 社會認知 Social Cognition

| 概念 | 說明 |
|------|------|
| **歸因理論** | 如何解釋他人行為的原因（內部 vs 外部） |
| **基本歸因錯誤** | 高估人格因素、低估情境因素 |
| **自我服務偏誤** | 成功歸於自己、失敗歸於環境 |

### 態度與態度改變

| 理論 | 說明 |
|------|------|
| **認知失調** (Festinger, 1957) | 態度與行為不一致時產生的不適感驅使改變 |
| **精細可能性模型 ELM** | 說服的兩條路徑：中心路徑（理性）和外圍路徑（情感） |

### 群體行為

| 現象 | 說明 |
|------|------|
| **社會助長** | 他人在場提升簡單任務表現、降低複雜任務表現 |
| **社會懈怠** | 群體中個人努力減少（搭便車） |
| **群體極化** | 群體討論使觀點更極端 |
| **去個體化** | 匿名狀態下自我約束降低 |

## 經典實驗

| 實驗 | 研究者 | 年份 | 發現 |
|------|--------|:--:|------|
| **從眾實驗** | Asch | 1951 | 75% 參與者至少一次屈從於錯誤群體判斷 |
| **服從實驗** | Milgram | 1963 | **65%** 參與者服從指令施以"致命電擊" |
| **監獄實驗** | Zimbardo | 1971 | 情境力量可將普通人變成虐待者（6天終止） |
| **旁觀者效應** | Latané & Darley | 1968 | 目擊者越多，每個人介入的可能性越低 |

## 旁觀者效應 Bystander Effect

```
決策樹：
1. 注意到事件？
2. 解釋為緊急情況？
3. 承擔個人責任？（人數越多，責任分散越嚴重）
4. 知道如何幫助？
5. 實施幫助？
```
