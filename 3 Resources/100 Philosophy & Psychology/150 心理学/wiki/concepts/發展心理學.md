# 發展心理學 Developmental Psychology

> 從搖籃到墳墓——研究人類一生中的心理變化。

## 定義

發展心理學研究人類從出生到死亡全程的心理發展規律，包括認知、情感、社會和道德發展。

## Piaget 認知發展階段

| 階段 | 年齡 | 特徵 |
|------|:--:|------|
| **感覺運動期** | 0–2 | 通過感官和動作認識世界、物體恆存 |
| **前運思期** | 2–7 | 符號使用、自我中心、無守恆概念 |
| **具體運思期** | 7–11 | 邏輯思維、守恆概念、分類能力 |
| **形式運思期** | 11+ | 抽象思維、假設推理、系統性問題解決 |

### 核心概念

- **基模 Schema**：認知結構，隨經驗而調整
- **同化 Assimilation**：新經驗融入已有基模
- **調適 Accommodation**：修改基模以適應新經驗

## Kohlberg 道德發展階段

| 層次 | 階段 | 道德推理 |
|------|:--:|------|
| **前習俗** | 1 | 服從以避免懲罰 |
| | 2 | 互惠——你幫我、我幫你 |
| **習俗** | 3 | 好孩子取向——符合他人期望 |
| | 4 | 法律與秩序——遵守社會規則 |
| **後習俗** | 5 | 社會契約——法律可因更大利益而改變 |
| | 6 | 普遍倫理原則——良心與正義 |

## Vygotsky 社會文化理論

- **近側發展區 ZPD**：學習者獨立能做到的 vs 在幫助下能做到的差距
- **鷹架 Scaffolding**：成人/同儕提供的暫時性支持
- 語言是思維的工具——先外化（對他人說），後內化（對自己說）
