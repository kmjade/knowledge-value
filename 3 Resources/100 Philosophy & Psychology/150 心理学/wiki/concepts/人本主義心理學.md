# 人本主義心理學 Humanistic Psychology

> 成為你自己。——存在的目的不是成為別人期望的你，而是成為真正的你。

## 定義

人本主義心理學被稱為「第三勢力」（相對於行為主義和精神分析），強調人的**自由意志、自我實現和內在成長潛能**。它關注健康的人格發展而非心理疾病。

## 需求層次理論 Maslow's Hierarchy

```
        ┌──────────────┐
        │  自我實現     │ ← 創造力、道德、潛能發揮
       ┌┴──────────────┴┐
       │   尊重需求      │ ← 自尊、成就、被認可
      ┌┴────────────────┴┐
      │   歸屬與愛       │ ← 友誼、家庭、親密關係
     ┌┴──────────────────┴┐
     │    安全需求         │ ← 人身安全、健康、財產
    ┌┴────────────────────┴┐
    │    生理需求           │ ← 食物、水、睡眠、呼吸
    └──────────────────────┘
```

> 下層需求必須基本滿足後，上層需求才會成為行為的驅動力。

## Rogers 的自我理論

### 實現傾向 Actualizing Tendency
每個人都有朝向成長、健康、自我實現的內在驅力。

### 價值條件 Conditions of Worth
他人（尤其是父母）對我們的評價變成我們評判自己的標準——當這些條件與真實自我衝突時，產生心理困擾。

### 治療三要素

| 要素 | 說明 |
|------|------|
| **真誠一致 Congruence** | 治療師真實表達自己 |
| **無條件積極關注** | 接納來訪者的全部 |
| **同理心 Empathy** | 理解來訪者的主觀世界 |

## 高峰體驗 Peak Experience

Maslow 描述的一種短暫的、強烈的自我實現時刻——感覺與宇宙合一、時間停止、完全的滿足感。
