# Psychology 常见问题

> 心理学的十大常见误解。如果你只记住一件事，记住这条：**心理学不是弗洛伊德，也不是心灵鸡汤。**

## 十大常见误解

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| 心理学 = 读心术 | 心理学不能"读心"，它通过可观察的行为和可测量的数据来推断心理过程 |
| 弗洛伊德 = 心理学的全部 | 弗洛伊德只是精神分析流派的创始人。现代心理学以实证研究为主，精神分析只占很小一部分 |
| 心理学 = 心理有问题的人才需要 | 心理学是研究**所有**人类行为和心智的科学，就像生物学不只研究疾病 |
| 心理学 = 星座/MBTI/性格色彩 | 这些都是未经科学验证的通俗分类。科学心理学使用经过信效度检验的心理测量工具 |
| 常识 = 心理学 | 很多"常识"是事后聪明偏差——你知道结果之后觉得"我早就知道"。心理学用实验验证而非凭直觉 |
| 心理学实验 = 折磨人 | 现代心理学有严格的伦理规范：知情同意、最小伤害、保密、可随时退出 |
| 左手右手性格测试有科学依据 | 无。大脑偏侧化不等于性格，这些测试没有预测效度 |
| 微表情能测谎 | 微表情的"测谎"能力被严重夸大，科学证据不足，法庭上不承认 |
| 我们只用了 10% 的大脑 | 神经影像学证明，几乎所有脑区在一天中都有活动。这是典型的伪科学 |
| 积极思考 = 心理学 | 积极思考本身不是科学心理学。积极心理学（Positive Psychology）才是有实证基础的子领域 |

## 如何分辨科学与伪心理学？

1. **可证伪性**：这个说法能被证明是错的吗？如果不能，就不是科学
2. **同行评审**：它发表在正规学术期刊上吗？
3. **可重复性**：其他研究者能复现这个结果吗？
4. **操作性定义**：概念有没有清晰、可测量的定义？
5. **控制变量**：结论是否排除了其他可能的解释？

> 💡 一个简单的测试：如果某个心理学说法让你觉得"哇这不就是我吗"，先别信。这可能是巴纳姆效应（对泛泛而谈的人格描述产生高度认同）。
