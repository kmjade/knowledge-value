# 03-认知心理学

> 认知心理学研究的是 "大脑的操作系统"——我们如何感知、注意、记忆、思考和说话。

## 信息加工模型

认知心理学的核心隐喻：大脑 ≈ 计算机。

```
输入（刺激）→ 感觉记忆 → 注意 → 工作记忆 → 编码/提取 ⇄ 长时记忆
                  ↓                ↓
               遗忘           通过复述保持
```

## 感知（Perception）

我们不是被动接收信息，而是主动构建感知。

| 概念 | 说明 | 示例 |
|------|------|------|
| **自上而下加工** | 用已有的知识和期望来解读感知 | 看到"HELLO"而不是"H E L L O" |
| **自下而上加工** | 从感觉信息出发构建感知 | 第一次看到不认识的文字 |
| **知觉恒常性** | 物体即使感官输入变化，我们仍感知为同一物体 | 门从不同角度看形状不同，但我们知道它是长方形 |

## 注意（Attention）

| 类型 | 说明 | 示例 |
|------|------|------|
| **选择性注意** | 聚焦于一个刺激，过滤其他 | 鸡尾酒会效应：嘈杂派对中听到自己的名字 |
| **分配性注意** | 同时处理多个任务 | 边开车边说话（但效率都降低） |
| **持续性注意** | 长时间保持警觉 | 雷达监控员 |

> ⚠️ **多任务处理的迷思**：人脑不能真正"并行"处理复杂认知任务。所谓的 multitasking 其实是快速任务切换，每次切换都有"切换成本"。

## 记忆（Memory）

### Atkinson-Shiffrin 三阶段模型

| 阶段 | 容量 | 持续时间 | 编码方式 |
|------|------|----------|----------|
| **感觉记忆** | 极大 | 0.5-3 秒 | 感觉信号的精确副本 |
| **工作记忆（短时记忆）** | 7±2 个组块 | 约 20 秒 | 语音/视觉编码 |
| **长时记忆** | 几乎无限 | 永久（理论上） | 语义编码为主 |

### 记忆的类型

```
        记忆
       /    \
  外显记忆  内隐记忆
 （有意识） （无意识）
   /    \        \
语义记忆 情景记忆  程序性记忆
（知识） （经历） （技能，如骑车）
```

### 遗忘的原因

| 理论 | 说明 |
|------|------|
| 编码失败 | 根本没有真正注意和学习 |
| 存储衰退 | 不使用的记忆痕迹随时间减弱 |
| 提取失败 | 信息还在，但找不到了（舌尖现象） |
| 干扰 | 新旧信息互相干扰 |

## 思维与决策

### 问题解决策略

| 策略 | 说明 | 代价 |
|------|------|------|
| 算法（Algorithm） | 系统地尝试所有可能 | 耗时长，但保证找到答案 |
| 启发式（Heuristic） | 心理捷径，凭经验快速判断 | 快速，但可能出错 |

### 常见认知偏差

| 偏差 | 含义 | 示例 |
|------|------|------|
| **确认偏差** | 倾向寻找支持已有信念的证据 | 只看支持自己观点的文章 |
| **可得性启发** | 根据想到例子的容易程度做判断 | 看新闻飞机失事后高估飞行风险 |
| **锚定效应** | 过度依赖最先获得的信息 | 原价 999 → 现价 399，觉得便宜 |
| **框架效应** | 同一信息的不同表达方式影响决策 | "存活率 90%" vs "死亡率 10%" |
| **过度自信** | 对自己判断准确性的高估 | 认为自己驾驶水平高于平均水平 |

## 语言

### Chomsky vs Skinner 的语言之争

| 立场 | 主张 |
|------|------|
| **Skinner（行为主义）** | 语言是学来的——通过强化和模仿 |
| **Chomsky（先天论）** | 语言是天赋的——大脑有通用语法（Universal Grammar） |

现在的主流观点：语言是先天能力与环境输入共同作用的产物。
