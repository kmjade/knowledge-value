# 02-超感官知觉 (ESP)

> 不通过已知的五感——人能否获取远距离的、被屏蔽的、甚至未来的信息？ESP 研究是超心理学的核心战场。

## ESP 的类型

| 类型 | 英文 | 含义 | 经典实验 |
|------|------|------|---------|
| **心电感应** | Telepathy | 人与人之间的直接信息传递 | Ganzfeld 发送者-接收者 |
| **遥视** | Clairvoyance | 不通过感官感知远处物体/事件 | Remote Viewing / Stargate |
| **预知** | Precognition | 对未来事件的感知 | Bem 的「感觉未来」实验 |
| ** retrocognition** | Retrocognition | 对过去不可知事件的感知 | 较少研究——更难验证 |

## Ganzfeld 实验：最著名的 ESP 范式

### 实验流程

```
发送者 (Sender)              接收者 (Receiver)
    │                              │
    │ 观看随机选中的图片/视频        │ 眼睛盖上半透明半球
    │                              │ 耳机播放白噪声
    │                              │ 红光照射面部
    │                              │ （感觉剥夺状态）
    │              ↓                │
    │     「发送」心理印象 → → → → →│
    │                              │
    │                              │ 口头描述接收到的印象
    │                              ↓
    │              从 4 张图片中选出目标
    │              （随机概率 = 25%）
```

### 元分析结果

| 研究 | 年份 | 实验数 | 击中率 | 统计显著性 |
|------|------|--------|--------|-----------|
| Honorton | 1985 | 28 | 35% (vs 25%) | p < 0.001 |
| Bem & Honorton | 1994 | 11 (autoganzfeld) | 32% | p = 0.002 |
| Storm et al. | 2010 | 108 | 30% | 显著 |
| 怀疑论者批评 | — | — | 选择性报告 | 文件抽屉问题 |

> Ganzfeld 效应量约 0.15~0.20 —— 这意味着「平均来说，接收者比随机猜测好约 5-10%」。很小，但如果在大量实验中一致，就可能需要解释。

## Stargate 计划：美国政府的遥视项目

| 事实 | 内容 |
|------|------|
| 时间 | 1978-1995 |
| 经费 | ~2000 万美元 |
| 资助方 | CIA / DIA (国防情报局) |
| 目的 | 用遥视获取情报——定位人质、描述军事设施 |
| 结果 | 部分案例「惊人准确」，但整体「不能可靠用于情报」 |
| 终结 | 1995 年 CIA 委托 AIR (美国研究所) 评估 → 结论：没有情报价值 |

### Stargate 的矛盾

| 支持方说 | 怀疑方说 |
|---------|---------|
| 有多个「统计学显著」的遥视者 | 显著 ≠ 有用——情报需要高精确度 |
| 某些案例超出巧合 | 谁能排除信息泄露、前期研究、冷读？ |
| 20 年 2000 万——为什么继续？ | 冷战偏执——苏联也在做 |

## Bem (2011)：「感觉未来」实验

Daryl Bem 在《人格与社会心理学杂志》(JPSP) 发表 9 个实验，声称证明了**预知**：

| 实验 | 范式 | 结果 |
|------|------|------|
| 1-2 | 词/图记忆——先测试、后学习 | 已「预知」的词回忆更好 |
| 3-4 | 情色图片位置预测 | 预测准确率 > 50% |
| 5-8 | 隐性联想、 priming | 时间反转 priming 效应 |
| 9 | 习惯化 | 对未来的刺激习惯化 |

> **争议**：Bem 使用标准心理学方法——但结果「不可能」。这引发了一场方法论战争：是 Bem 发现的效应是真实的，还是心理学的研究方法本身有问题？

## Sheldrake 的延伸意识假说

Rupert Sheldrake 提出：意识不是封闭在颅骨内的——它是**延伸的场**：

| 现象 | 实验 | 结果 |
|------|------|------|
| **被注视感** | 背后有人盯着你——你能感觉到吗？ | 部分实验显著 |
| **电话预知** | 电话响前就知道是谁 | 部分显著 |
| **宠物预知** | 狗在主人在回家路上就激动 | 部分显著 |

> Sheldrake 被主流科学界严厉批评——但他的实验范式很简单，任何人都可以在家里试。

## ESP 的替代解释

| 解释 | 说明 |
|------|------|
| **感官泄露** | 实验中存在微妙的感官线索——半透明眼罩、发送者的声音 |
| **选择性报告** | 只发表阳性结果——阴性结果留在抽屉里 |
| **p-hacking** | 测试多个因变量，只报告显著的 |
| **统计失误** | 随机性被误解——小样本的波动被当作效应 |
| **欺诈** | 少数实验者/被试造假——影响整个领域可信度 |
