# 轉世記憶 Reincarnation Memories

## 定義
轉世記憶研究調查兒童聲稱記得「前世」的案例，由 Ian Stevenson 在維吉尼亞大學系統化開展。

## Stevenson 研究方法

1. 收集兒童自發報告的「前世記憶」（通常 2–5 歲開始）
2. 尋找聲稱的前世人物及其家庭
3. 驗證兒童無法通過正常途徑獲得的資訊
4. 記錄身體印記（胎記、先天缺陷對應前世創傷）

## 統計數據 (Stevenson, 1997)

| 指標 | 數據 |
|------|:--:|
| 調查案例總數 | 2,500+ |
| 「已解決」案例（找到前世人物） | ~1,200 |
| 兒童開始講述的平均年齡 | 35 個月 |
| 停止講述的平均年齡 | 7 歲 |
| 有身體印記的案例 | 35% |
| 暴力死亡的前世 | 61% |

## 替代解釋

| 解釋 | 說明 |
|------|------|
| **幻想/暗示** | 父母無意引導、社會文化影響 |
| **隱性記憶** | 兒童通過正常途徑獲得的資訊被遺忘來源 |
| **詐欺** | 父母或研究者有意造假 |

## 批評
- 多數案例來自相信轉世的文化（印度、斯里蘭卡）
- 驗證過程可能受確認偏誤影響
- 未能提供可重複的實驗證據
