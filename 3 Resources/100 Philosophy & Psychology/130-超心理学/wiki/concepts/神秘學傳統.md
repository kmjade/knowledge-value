# 神秘學傳統 Esoteric Traditions

## 定義
神秘學是指東西方文化中關於隱藏知識、靈性修行和超自然能力的傳統體系。

## 西方神秘傳統

| 傳統 | 核心 |
|------|------|
| **赫密斯主義 Hermeticism** | 「如其在上，如其在下」 |
| **卡巴拉 Kabbalah** | 生命之樹、十個 Sephirot |
| **煉金術 Alchemy** | 物質轉化→心靈轉化 |
| **神智學 Theosophy** | Blavatsky—東西方神秘學融合 |

## 東方傳統

| 傳統 | 核心 |
|------|------|
| **瑜伽 Yoga** | 意識控制→超常能力 (Siddhis) |
| **道教內丹** | 精氣神轉化→長生久視 |
| **藏密** | 中陰 (Bardo)、轉世活佛 |

## 超心理學的關係
- 神秘傳統是超心理學研究的**現象來源**
- 但學術超心理學強調**科學方法驗證**，而非信仰接受
