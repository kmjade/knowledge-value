# 170 伦理学 / Ethics

> 「我应该怎样活？」(How ought I to live?) — 苏格拉底之问，伦理学的原点。
> 伦理学不是教条清单，而是关于对错的理性探究。

## 一、元伦理学 (Meta-ethics) — 道德的本质是什么？

### 核心问题

| 问题 | 说明 |
|------|------|
| 道德事实是否存在？ | 道德实在论 vs 反实在论 |
| 道德判断是真假命题吗？ | 认知主义 vs 非认知主义 |
| 道德有客观基础吗？ | 客观主义 vs 相对主义 |
| 道德动机从何而来？ | 内在论 vs 外在论 |

### 主要立场

| 立场 | 核心主张 | 代表人物 |
|------|----------|----------|
| **道德实在论** | 道德事实独立于我们而存在 | Moore, Brink |
| **错误理论** | 道德判断都是假的（因为道德事实不存在） | Mackie |
| **表达主义** | 道德判断不是真假命题，是情感/态度的表达 | Ayer, Blackburn |
| **规约主义** | 道德判断是普遍的规约 (prescription) | Hare |
| **建构主义** | 道德事实由理性程序建构 | Korsgaard, Rawls |

> **简化版**：当你说「杀人是错的」——
> - 实在论者：这是一个客观事实
> - 表达主义者：你在表达「杀人——呸！」
> - 错误理论家：你错了，不存在「错」这种东西

### 三大经典理论

### ②A 功利主义 (Utilitarianism) — 看结果

> **最大多数人的最大幸福。** 行为的对错取决于它带来的后果。

| 维度 | 说明 |
|------|------|
| 核心原则 | 效用最大化 (maximize utility)；以结果评判行为 |
| 决策方法 | 计算每个选项的预期幸福总量，选最大的 |
| 代表人物 | Bentham (古典功利主义)、Mill (规则功利主义)、Singer (偏好功利主义) |
| 经典测试 | 电车难题：牺牲 1 人救 5 人 → 功利主义说「应该」 |

| 变体 | 核心主张 |
|------|----------|
| **行动功利主义** | 每个具体行动都要计算效用 |
| **规则功利主义** | 遵循「通常最大化效用的规则」 |
| **偏好功利主义** | 最大化偏好的满足（而非快乐） |

| 批评 | 回应 |
|------|------|
| 可能为多数牺牲少数无辜者 | 规则功利主义：长期来看，牺牲无辜的规则降低总效用 |
| 幸福难以量化 | 不需要精确数字，只需可比较 |
| 要求太高 (too demanding) | 只需做「够好」而非「最优」 |

---

### ②B 义务论 (Deontology) — 看规则

> **某些行为本身就错，不论后果如何。** 道德是义务，不是计算。

| 维度 | 说明 |
|------|------|
| 核心原则 | 绝对命令 (Categorical Imperative)：只按你能同时意愿它成为普遍法则的准则行动 |
| 决策方法 | 检查你的行动准则能否被普遍化而不自相矛盾 |
| 代表人物 | 康德 (经典义务论)、Ross (初始义务) |
| 经典测试 | 电车难题：故意推人下桥救 5 人 → 义务论说「不能」，人不是工具 |

| 康德的三个公式 |
|----------------|
| ① **普遍法则公式**：只按你能同时意愿它成为普遍法则的准则行动 |
| ② **人性公式**：永远把人当作目的，而不仅仅是手段 |
| ③ **目的王国公式**：好像你是目的王国中自主的立法者那样行动 |

| 批评 | 回应 |
|------|------|
| 义务冲突时怎么办？ | Ross：初始义务 (prima facie duties) 可以有优先级 |
| 过于僵化 | 规则总有例外；义务论不等于绝对主义 |

---

### ②C 德性伦理 (Virtue Ethics) — 看品格

> **正确的行动来自有德性的人。** 问的不是「我该做什么」，而是「我该成为什么样的人」。

| 维度 | 说明 |
|------|------|
| 核心原则 | 德性 (virtue) = 使事物很好发挥其功能的品质；幸福 (eudaimonia) = 灵魂合乎德性的活动 |
| 决策方法 | 问「一个有德性的人会怎么做？」而非「什么规则适用？」 |
| 代表人物 | 亚里士多德 (古典)、Anscombe、MacIntyre (当代复兴) |

| 核心德性 (亚里士多德) |
|-----------------------|
| **理智德性**：智慧 (sophia)、明智 (phronesis)、理解 (nous) |
| **道德德性**：勇敢、节制、慷慨、正义、友善、诚实 |

| **中庸之道 (Golden Mean)** |
| 勇敢 = 怯懦 (不足) ← → 鲁莽 (过度) 之间的中道 |
| 每种德性都是两种恶之间的平衡点 |

| 批评 | 回应 |
|------|------|
| 如何确定哪些是德性？ | 社群实践 + 人的功能论证 |
| 无法指导具体行动 | 当代德性伦理已发展出具体指导 |

---

### 三大理论对比

| 维度 | 功利主义 | 义务论 | 德性伦理 |
|------|---------|--------|---------|
| 核心问题 | 什么带来最好结果？ | 什么是我的义务？ | 什么是有德之人？ |
| 评判对象 | 行为后果 | 行为准则 | 行为者品格 |
| 时间焦点 | 未来 (预测后果) | 当下 (检验准则) | 一生 (品格养成) |
| 强项 | 直觉匹配、可量化 | 尊重个人权利 | 完整的「好生活」愿景 |
| 弱项 | 量化困难、牺牲少数 | 义务冲突、僵化 | 行动指导不精确 |

---

### ②D 关怀伦理 (Care Ethics) — 20C 新范式

> **道德的核心不是抽象规则，而是关系中的关怀与回应。** 女性主义伦理学的核心贡献。

| 维度 | 说明 |
|------|------|
| 核心主张 | 道德应基于关怀关系，而非抽象规则或效用计算 |
| 代表人物 | Gilligan, Noddings |
| 对比传统 | 传统伦理强调自主、理性、普遍规则；关怀伦理强调关系、情感、具体情境 |

## 伦理的心理学基础

| 心理学发现 | 伦理意义 |
|------------|----------|
| **电车难题 fMRI 实验** | 推人 (义务论禁止) 激活情绪脑区，拉杆 (功利主义) 激活认知脑区 |
| **Haidt 道德基础理论** | 6 大道德基础：关怀/伤害、公平/欺骗、忠诚/背叛、权威/颠覆、圣洁/堕落、自由/压迫 |
| **Kohlberg 道德发展阶段** | 前习俗 → 习俗 → 后习俗；最高阶段是社会契约与普遍原则 |
| **婴儿道德实验** | 6 月龄婴儿偏好「帮助者」而非「阻碍者」— 道德感可能有先天基础 |

## 历史脉络

| 时期 | 人物/事件 | 贡献 |
|------|----------|------|
| 古希腊 | 苏格拉底、柏拉图、亚里士多德 | 「应该怎样活」；德性伦理 |
| 希腊化 | 斯多葛派、伊壁鸠鲁派 | 幸福 = 宁静 (ataraxia)；美德即自足 |
| 中世纪 | 阿奎那 | 自然法理论：道德法则源于上帝理性 |
| 启蒙 (18C) | 休谟 | 是/应该鸿沟 (is-ought gap)；道德感源于情感 |
| 启蒙 (18C) | **康德 (1785)** | **义务论**：《道德形而上学奠基》 |
| 19C | Bentham, Mill | **功利主义**：最大幸福原则 |
| 20C 初 | Moore | 自然主义谬误；元伦理学开端 |
| 20C 中 | Anscombe, MacIntyre | 德性伦理复兴 |
| 1971 | **Rawls** | **《正义论》**：无知之幕、两个原则 |
| 1980s | Gilligan | 关怀伦理：不同的声音 |
| 2000s | Singer, Parfit | 有效利他主义、道德数学 |

## 经典思想实验

| 实验 | 场景 | 测试的伦理问题 |
|------|------|---------------|
| **电车难题** | 是否牺牲 1 人救 5 人？ | 功利主义 vs 义务论 |
| **器官移植** | 可否杀死 1 个人用其器官救 5 人？ | 行动的边界 |
| **体验机 (Nozick)** | 如果有一台机器给你一切快乐体验，你进去吗？ | 快乐是唯一价值吗？ |
| **无知之幕 (Rawls)** | 在不知道自己的社会地位时设计社会，你会怎么设计？ | 正义的起点 |
| **池塘中的孩子 (Singer)** | 路过池塘见孩子溺水，救他需要弄脏衣服，你救吗？ | 对远方苦难的义务 |
