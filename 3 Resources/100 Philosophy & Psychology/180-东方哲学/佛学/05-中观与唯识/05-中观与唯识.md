# 05-中观与唯识

> 大乘佛学有两大哲学巅峰：中观——龙树说「一切法空」；唯识——无著和世亲说「万法唯识」。一个是空的哲学，一个是心的哲学。合起来再加上如来藏——「一切众生皆有佛性」——构成大乘思想的三根支柱。

## 中观 (Madhyamaka)

### 龙树 (Nagarjuna, ~2世纪)

| 概念 | 内容 |
|------|------|
| **核心** | 《中论》——「众因缘生法，我说即是空」 |
| **空** (Shunyata) | 不是虚无——是**无自性**——任何事物都没有独立的、不变的本质 |
| **中道** | 不落「有」(常见) 也不落「无」(断见) ——空超越有无 |

### 归谬法 (Prasanga)

> 龙树不建立自己的立场——他用对方的逻辑推出矛盾。

| 例子 | 内容 |
|------|------|
| 如果你说「事物有自性」——自性意味着不依赖条件 | 但一切事物都依赖条件——所以自性概念自相矛盾 |
| 结论 | 空不是「另一种理论」——是**一切理论的解构** |

### 二谛

| 谛 | 含义 |
|----|------|
| **世俗谛** (Samvrti) | 日常语言中的「有」——桌子、人——在约定俗成的层面是有的 |
| **胜义谛** (Paramartha) | 空——没有自性——在终极层面没有一个「桌子本身」 |
| **关系** | 不了解二谛——就不了解佛法 |

> 中观的空不是「什么都没有」——是「没有自性的有」——缘起 = 空。

---

## 唯识 (Yogacara)

### 无著与世亲 (~4-5世纪)

| 概念 | 内容 |
|------|------|
| **核心** | 「三界唯心，万法唯识」——我们感知的世界是识的投影 |
| **针对什么** | 中观说一切空——但「空的认识」本身从哪里来？唯识回答：从心来 |

### 八识

| 识 | 功能 |
|----|------|
| 前五识 | 眼耳鼻舌身——感官 |
| 第六识 (意识) | 思维、判断、分别 |
| 第七识 (末那识) | 自我意识——持续地把阿赖耶识执为「我」 |
| **第八识 (阿赖耶识)** | 种子识——储存一切过去经验的「种子」——轮回的承载者 |

> 阿赖耶识回答了上座部被大乘追问的问题：「如果没有我——什么在轮回？」——阿赖耶识。

### 三性

| 性 | 含义 |
|----|------|
| **遍计所执性** | 我们把概念投射到世界上——以为「桌子」「我」是实在的 |
| **依他起性** | 一切事物依赖条件——缘起 |
| **圆成实性** | 看到依他起性后——不再遍计所执——这就是真如 |

## 如来藏 (Tathagatagarbha)

| 概念 | 内容 |
|------|------|
| **核心** | 「一切众生皆有佛性」——成佛的能力不是你修来的——是你本来就有的 |
| **经典** | 《涅槃经》《胜鬘经》《楞伽经》 |
| **比喻** | 佛性像金矿里的黄金——被烦恼矿石包裹——但黄金本身是纯净的 |

### 三派关系

```
中观 (空的哲学) ——破斥一切实体
    │
    ├→ 唯识 (心的哲学) ——解释「空的认识」从哪里来
    │
    └→ 如来藏 (佛性论) ——为什么众生「能」成佛
```
