# 06-禅宗

> 「不立文字，教外别传，直指人心，见性成佛。」——禅宗自称是佛陀的「心传」：佛陀拈花，迦叶微笑——不需要语言，直接传递。

## 禅宗的谱系

| 人物 | 时代 | 贡献 |
|------|------|------|
| **摩诃迦叶** | 佛陀时代 | 拈花一笑——「禅」的开始（传说） |
| **菩提达摩** | 5-6世纪 | 从印度来华——禅宗初祖——少林寺面壁九年 |
| **慧可** | 二祖 | 「觅心了不可得」——断臂求法 |
| **僧璨** | 三祖 | 《信心铭》 |
| **道信** | 四祖 | 开始建立禅宗僧团——定居而非游方 |
| **弘忍** | 五祖 | 黄梅东山——禅宗制度化 |
| **惠能** | 六祖 (638-713) | 《坛经》——「本来无一物」——南宗禅奠基 |

---

## 神秀 vs 惠能

### 神秀的偈

```
身是菩提树，心如明镜台。
时时勤拂拭，勿使惹尘埃。
```

→ 渐修——需要不断擦——慢慢来。

### 惠能的偈

```
菩提本无树，明镜亦非台。
本来无一物，何处惹尘埃？
```

→ 顿悟——镜子本来干净——不需要擦——一念觉悟即是佛。

> 弘忍把衣钵传给惠能——南宗禅从此成为中国禅宗的主流。

---

## 惠能的核心思想

### 无念·无相·无住

| 概念 | 内容 |
|------|------|
| **无念** | 不是没有念头——是于念而离念——念头生起但不执著 |
| **无相** | 不执著于事物的相状——「外离相为禅」 |
| **无住** | 不停留——不执著——像水流一样 |

### 定慧一体

> 「即定之时慧在定，即慧之时定在慧。」——定和慧不是先后关系——是同一体的两面。

---

## 禅宗的主要形式

### 公案 (Koan)

| 概念 | 内容 |
|------|------|
| **定义** | 一个无法用逻辑回答的问题/故事——逼你跳出理性思维 |
| **著名公案** | 「父母未生你之前，你的本来面目是什么？」 |
| | 「单手拍掌是什么声音？」 |
| | 赵州和尚：「狗有佛性吗？」——「无！」 |

> 公案不是谜语——它是一种**禅修技术**——通过逻辑的死胡同逼出直觉的开悟。

### 默照禅

| 概念 | 内容 |
|------|------|
| **代表** | 曹洞宗——宏智正觉 |
| **方法** | 「只管打坐」——不参公案——只是坐着——默然——照见 |
| 对比临济 | 临济宗用公案——曹洞宗用默照——「看话」vs「默照」 |
