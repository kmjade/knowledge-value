# 04-大乘佛教

> 「大乘」(Mahayana) ——大的运载工具。运的不是一个人，是一切众生。大乘批评上座部「只顾自己解脱」——他们说真正的佛教是菩萨道：推迟自己的涅槃，先度化一切众生。

## 大乘 vs 上座部

| 维度 | 上座部 | 大乘 |
|------|--------|------|
| 理想 | 阿罗汉——个人解脱 | 菩萨——度一切众生 |
| 佛陀观 | 释迦牟尼是历史人物 | 佛陀有无数——法身遍一切处 |
| 空性 | 「人无我」——没有永恒的自我 | 「法无我」——一切法也空 |
| 经典 | 巴利三藏 | 大乘经——般若、法华、华严 |
| 语言 | 巴利语 | 梵语 → 汉语、藏语 |
| 地区 | 斯里兰卡、东南亚 | 中国、日本、韩国、越南、西藏 |

## 菩萨道

### 菩萨的定义

> 「菩提萨埵」= 觉悟 + 有情——追求觉悟的有情——但为了众生而推迟进入涅槃。

### 六度 (六波罗蜜)

| 波罗蜜 | 含义 | 实践 |
|--------|------|------|
| **布施** (Dana) | 给予——财物、无畏、法 | 慈善 |
| **持戒** (Sila) | 道德自律 | 菩萨戒 |
| **忍辱** (Ksanti) | 忍受逆境和伤害 | 不报复 |
| **精进** (Virya) | 不懈努力 | 不退转 |
| **禅定** (Dhyana) | 心的专注 | 止观 |
| **智慧** (Prajna) | 空性的洞察 | 般若 |

### 四弘誓愿

```
众生无边誓愿度  烦恼无尽誓愿断
法门无量誓愿学  佛道无上誓愿成
```

## 核心经典

### 般若经系

> 《心经》——「色即是空，空即是色」——二百六十字的大乘核心。

| 经典 | 核心命题 |
|------|---------|
| 《心经》 | 五蕴皆空——度一切苦厄 |
| 《金刚经》 | 「凡所有相，皆是虚妄」 |
| 《大般若经》 | 空性的最详尽展开——600 卷 |

### 法华经

> 「一切众生皆可成佛。」——《法华经》把之前的大小乘对立统一了：小乘的「阿罗汉」只是方便说——最终都要走向成佛。

### 净土经典

| 概念 | 内容 |
|------|------|
| 阿弥陀佛 | 在因地发了四十八大愿——建立了西方极乐世界 |
| 净土法门 | 念佛——「南无阿弥陀佛」——靠他力往生 |
