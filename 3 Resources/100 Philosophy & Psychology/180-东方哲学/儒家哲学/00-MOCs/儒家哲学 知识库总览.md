# 儒家哲学 知识库总览 (DDC 180)

> 儒家是中国文化的「主流操作系统」——它奠定了东亚社会的伦理框架、家庭结构、政治理念和教育传统。从孔子的「仁」到王阳明的「良知」，儒家两千年不断自我更新。

## 核心张力：孟学 vs 荀学

```
        人性本善？(孟子)                 人性本恶？(荀子)
              │                              │
        扩充四端                          化性起伪
        内省·养心                         礼法·教化
        浩然之气                         劝学·积善
              │                              │
              └──────────┬──────────────────┘
                         ↓
                   宋明理学综合
               天地之性(善) vs 气质之性(恶)
                        │
               ┌────────┴────────┐
               ↓                  ↓
          程朱理学            陆王心学
         性即理 · 格物       心即理 · 致良知
```
