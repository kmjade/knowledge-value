# 魔兽世界 - 防御圣骑士职业定位

> [!info] 核心定位
> 防御圣骑士（Protection Paladin）是魔兽世界的主要坦克职业之一，以高生存能力、团队保护和强力爆发著称。

---

## 职业角色定位

### 主角色：坦克（Tank）

| 维度 | 评分 | 说明 |
|------|------|------|
| 生存能力 | ★★★★★ | 神圣之盾、圣佑术、圣盾术 |
| 仇恨建立 | ★★★★★ | 审判、十字军打击、复仇 |
| 团队保护 | ★★★★★ | 自由、保护、牺牲祝福 |
| 爆发能力 | ★★★☆ | 复仇之怒、终极技能 |
| 操作难度 | ★★★☆ | 需要掌握技能时机和团保使用 |

---

## 核心优势

### 1. 高生存能力

#### 神圣之盾
- **效果**：每 3 秒抵挡一次物理伤害，持续 15 秒
- **用途**：拉怪、应对爆发伤害、生存关键时刻
- **冷却**：30 秒

#### 圣佑术
- **效果**：20% 物理减伤，持续 8 秒
- **用途**：应对高伤害BOSS技能
- **冷却**：3 分钟

#### 圣盾术（无敌）
- **效果**：免疫所有伤害，持续 8 秒
- **用途**：应对必杀技能、紧急保命
- **冷却**：5 分钟

#### 守护者形态（终极技能）
- **效果**：50% 护甲提升、神圣力量获取翻倍、持续 20 秒
- **用途**：应对高强度伤害、团队副本关键时刻
- **冷却**：3 分钟

---

### 2. 团队保护能力

#### 自由祝福
- **效果**：解除所有控制（昏迷、定身、减速、沉默）
- **持续时间**：6 秒
- **冷却**：6 分钟
- **用途**：解控、保护输出和治疗

#### 保护祝福
- **效果**：免疫所有物理伤害
- **持续时间**：8 秒
- **冷却**：6 分钟
- **用途**：保护队友物理伤害、应对关键技能
- **注意**：保护期间目标无法攻击

#### 牺牲祝福
- **效果**：分担目标 50% 伤害
- **持续时间**：12 秒
- **冷却**：6 分钟
- **用途**：保护治疗、分担团队伤害

---

### 3. 强力仇恨建立

#### 仇恨倍率
- **基础仇恨**：所有攻击技能仇恨值提升 200%
- **复仇技能**：仇恨传递率 200%
- **审判**：高额仇恨 + 2 点神圣力量

#### 爆发仇恨
- **复仇之怒**：30% 伤害提升 → 仇恨提升 30%
- **守护者形态**：神圣力量获取翻倍 → 仇恨获取翻倍

---

## 核心劣势

### 1. 输出较低
- **单攻输出**：相比输出职业，防骑输出较低
- **AoE能力**：正义复仇和神圣风暴 AoE 范围有限
- **依赖团队**：需要团队输出职业配合

### 2. 爆发间隔长
- **终极技能 CD**：守护者形态、天使形态等 3 分钟 CD
- **复仇之怒 CD**：30 秒（天赋减少到 15 秒）
- **爆发期后**：技能 CD 期伤害大幅下降

### 3. 移动能力弱
- **无位移技能**：只能靠坐骑和自由祝福移动
- **机动性差**：相比盗贼、武僧、德鲁伊机动性较弱
- **依赖站位**：需要预判走位

### 4. 团保限制
- **保护限制**：保护期间目标无法攻击
- **牺牲风险**：只分担 50% 伤害，目标仍可能死亡
- **祝福冲突**：同类型祝福不能叠加

---

## 适合场景

### 最适合
- **团队副本坦克**：高生存和团保能力，适合主/副坦克
- **5人副本**：拉怪能力强，仇恨稳定
- **小规模PVP**：团保能力在战场优势明显
- **新手法师**：容错率高，适合新手法师

### 较适合
- **大型副本**：需要团队配合，适合主/副坦克
- **竞技场**：团保能力重要，但输出较低
- **野外BOSS**：生存能力强，但输出有限

### 不太适合
- **纯输出需求**：输出较低，不适合输出位置
- **高机动性需求**：位移需求高的场景不占优

## 配置建议

### 新手配置
```
主天赋：
- 圣洁光环（团队 5% 暗影减伤）
- 勇敢之心（圣佑术 +3 秒）
- 强化十字军打击（+15% 伤害，+1 神圣力量）

副天赋：
- 神圣之盾强化（移动速度 +20%）
- 守护之盾（吸收量 +20%）

核心技能优先级：
1. 审判
2. 十字军打击
3. 复仇（复仇之怒）
4. 神圣之盾
5. 神圣震击
```

### 进阶配置
```
主天赋：
- 圣洁光环
- 强化十字军打击
- 强化审判（+20% 伤害，神圣力量获取翻倍）
- 神圣审判（目标神圣伤害 +10%）

副天赋：
- 神圣之盾强化
- 守护之盾
- 神圣复仇（复仇之怒 CD -15 秒）

终极技能：守护者形态
```

## 关键要点总结

### 必须掌握的核心
1. **神圣之盾时机**：拉怪前、应对爆发、关键时刻
2. **团保使用**：自由（解控）、保护（物理伤害）、牺牲（分摊）
3. **仇恨管理**：审判 → 十字军打击 → 复仇循环
4. **减伤规划**：圣佑术、圣盾术、守护者形态的时机

### 新手常见错误
1. **忘记开盾**：拉怪前忘记开神圣之盾
2. **团保用错**：保护输出职业、不用保护坦克
3. **仇恨丢失**：忽视审判和复仇，仇恨不稳定
4. **过度依赖圣盾术**：频繁使用无敌，浪费 CD
