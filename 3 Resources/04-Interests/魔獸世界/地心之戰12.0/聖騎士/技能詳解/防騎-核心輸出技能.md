# 魔兽世界 - 防御圣骑士核心输出技能

> [!info] 输出技能
> 核心输出技能是防骑建立仇恨的主要手段。掌握核心输出技能的使用时机和输出循环，是成为优秀防骑的基础。

---

## 🎯 核心输出技能概述

### 输出技能分类
| 分类 | 技能 | 作用 | 冷却 |
|------|------|------|------|
| 主要输出 | 审判 | 产生神圣力量 | 6 秒 |
| 主要输出 | 十字军打击 | 产生神圣力量 | 4 秒 |
| 爆发输出 | 神圣震击 | 消耗神圣力量 | 6 秒 |
| 爆发输出 | 愤怒之锤 | 消耗神圣力量 | 8 秒 |
| AOE 输出 | 神圣风暴 | 消耗神圣力量 | 10 秒 |
| AOE 输出 | 正义复仇 | 产生仇恨 | 10 秒 |

### 输出循环原则
1. **优先产生神圣力量**：审判和十字军打击优先
2. **合理消耗神圣力量**：根据神圣力量数量选择消耗技能
3. **保持仇恨稳定**：持续输出，确保仇恨不丢失
4. **爆发规划**：合理规划神圣力量使用时机

## ⚖️ 审判（Judgment）

### 技能详情
| 属性 | 数值 |
|------|------|
| 伤害类型 | 神圣 |
| 伤害范围 | 20m |
| 冷却时间 | 6 秒 |
| 消耗 | 无 |
| 神圣力量 | 产生 2 点 |
| GCD | 是 |

### 效果说明
- **神圣伤害**：对目标造成神圣伤害
- **产生神圣力量**：产生 2 点神圣力量
- **远程技能**：20m 距离，适合远程拉怪
- **仇恨输出**：仇恨输出稳定

### 使用场景

#### 拉怪阶段
- **起手技能**：作为起手技能，产生神圣力量
- **远程拉怪**：20m 距离，可远程拉怪
- **快速建立仇恨**：快速建立初始仇恨

#### 稳定仇恨阶段
- **循环使用**：每 6 秒使用一次，持续产生神圣力量
- **补充神圣力量**：神圣力量不足时，优先使用审判
- **远程输出**：距离较远时，使用审判输出

#### 爆发阶段
- **爆发期间**：爆发期间持续使用审判
- **神圣力量支持**：为爆发技能提供神圣力量支持

### 使用技巧

#### 技巧 1：起手审判
```lua
1. 起手审判 → 产生 2 点神圣力量
2. 十字军打击 → 再产生 1 点神圣力量
3. 神圣震击 → 消耗 3 点神圣力量爆发
```

#### 技巧 2：循环使用
```
审判 → 十字军打击 → 审判 → 神圣震击
```

#### 技巧 3：远程输出
```
保持 20m 距离，循环使用审判
```

### 优先级
- **优先级**：⭐⭐⭐⭐⭐（最高）
- **理由**：产生神圣力量，远程拉怪，仇恨输出稳定

## ⚠️ 常见错误

### 错误 1：不优先产生神圣力量
**问题**：神圣力量不足时，不优先使用审判和十字军打击
**解决**：神圣力量不足时，优先使用审判和十字军打击

### 错误 2：神圣力量浪费
**问题**：神圣力量满后，不及时使用消耗技能
**解决**：神圣力量满后，立即使用神圣震击或神圣风暴

### 错误 3：不规划爆发
**问题**：不规划神圣力量使用时机，爆发效果不佳
**解决**：规划神圣力量使用时机，在关键时刻爆发

### 错误 4：不与团队配合
**问题**：不与团队配合，仇恨输出不足或过高
**解决**：与团队配合，确保仇恨稳定，输出效率
