# 魔獸世界-地心之戰12.0知識庫建設方案

> **概述**：地心之戰(The War Within)是魔獸世界二十週年紀念資料片，標誌著職業設計的重大轉型。本方案規劃如何系統化構建地心之戰版本的知識庫。

---

## 📋 建設目標

### 核心目標
- 系統化記錄地心之戰版本的所有機制變化
- 建立職業分析框架追蹤12.0職業重構
- 整理新內容（地區、任務、裝備）的知識體系

### 建設原則
- **版本獨立性**：與經典版本知識庫分離
- **深度優先**：聚焦遊戲機制分析
- **持續更新**：跟隨版本更新迭代

## 🔑 核心內容分類

### 1. 英雄天賦系統（Hero Talents）

#### 設計理念
- **職業通用**：每個職業3個英雄天賦
- **天賦點獨立**：不影響現有天賦選擇
- **玩法定義**：提供獨特的操作風格

#### 職業天賦分布

| 職業 | 英雄天賦 #1 | 英雄天賦 #2 | 英雄天賦 #3 |

### 2. 職業機制重構

#### 重點職業

| 職業 | 重構內容 | 變化程度 |
|------|---------|----------|
| 術士 | 毀滅/惡魔學家/痛苦全面重做 | ★★★★★ |
| 德魯伊-守護者 | 移除怒氣，改用原始怒火 | ★★★★☆ |
| 聖騎士-神聖 | 聖能機制重新設計 | ★★★★☆ |
| 狂戰士 | 怒氣生成從普攻改為技能命中 | ★★★☆☆ |

### 3. 新增種族：土靈（Earthen）

#### 基本資料
- **陣營**：聯盟
- **類型**：石製構裝體
- **可選職業**：戰士、聖騎士、德魯伊、獵人、法師、術士、牧師、薩滿

#### 種族天賦
| 天賦 | 效果 |
|------|------|
| 石膚 | 受到治療時獲得臨時生命值 |
| 大地堅韌 | 昏迷時間減少 |
| 地底感知 | 礦物採集專業加成 |
