# 魔兽世界 - 快速技能释放宏

> [!tip] 使用提示
> - 魔兽世界单个宏最多 255 字符
> - 可使用多个宏组合使用
> - 需要根据职业和天赋调整

---

## 基础宏类型

### 1. 连招宏（一键释放多个技能）

#### 法师 - 冰霜连招
```lua
#showtooltip 冰霜箭
/castsequence reset=3 冰枪术,冰霜箭,寒冰箭
```

#### 战士 - 致死打击 + 旋风斩
```lua
#showtooltip 致死打击
/castsequence reset=target 致死打击,旋风斩,致死打击
```

---

### 2. 智能选择宏（条件判断）

#### 盗贼 - 自动选择潜行状态技能
```lua
#showtooltip 背刺
/cast [stealth] 伏击; 背刺
```

#### 德鲁伊 - 形态切换
```lua
#showtooltip 旅行形态
/cast [swimming] 水栖形态; [flyable, outdoors] 飞行形态; 旅行形态
```

---

### 3. 目标选择宏（智能锁定目标）

#### 自动选择目标 + 攻击
```lua
#showtooltip 影袭
/cleartarget
/targetenemy [noharm][dead]
/cast 影袭
/startattack
```

#### 辅助目标（治疗职业）
```lua
#showtooltip 快速治疗
/cast [target=mouseover,exists,help] 快速治疗; [target=target,exists,help] 快速治疗; 快速治疗
```

---

## 高级宏技巧

### 1. 宏组合（超长宏）

由于单个宏限制 255 字符，可以使用多个宏组合：

```lua
#showtooltip 宏1
/run if not IsShiftKeyDown() then CastSpellByName("技能1") else CastSpellByName("技能2") end
```

### 2. 使用 /run 命令

```lua
/run if UnitHealth("target")/UnitHealthMax("target")<0.2 then CastSpellByName("爆发技能") end
```

### 3. 条件组合

```lua
/cast [mod:alt, target=player] 自我技能; [mod:ctrl] 辅助技能; 主技能
```

```lua

/targetenemy [noharm][dead]

```

```lua
/script TargetUnit("target")
/sendmessage 当前目标: %t, 1.0, 1.0, 0
```

## 使用建议

### 1. 宏放置位置
- 放置在易于按到的键位
- 常用技能放在 1-5 键位
- 辅助技能放在 Q、E、F 键位

### 2. 避免冲突
- 不要将宏放在与默认技能冲突的键位
- 测试宏功能是否正常

### 3. 个性化调整
- 根据自己的职业天赋调整
- 根据个人按键习惯调整
- 根据团队配合需求调整

### 4. 宏管理
- 命名规范：`职业-功能-技能名`
- 定期清理不用的宏
- 备份重要宏

## 故障排除

### 常见问题

#### 问题 1：宏无法工作
**解决**：
- 检查语法是否正确
- 检查技能名称是否正确
- 检查宏是否超过 255 字符

#### 问题 2：技能无法连招
**解决**：
- 检查公共冷却时间 (GCD)
- 使用 reset 参数调整序列重置
- 检查技能前置条件

#### 问题 3：目标选择错误
**解决**：
- 使用 /cleartarget 清除目标
- 添加 [noharm][dead] 条件判断
- 检查目标类型是否正确

## 注意事项

> [!warning] 游戏规则
> - 宏必须符合游戏服务条款
> - 禁止使用自动化脚本（按键精灵等）
> - 禁止使用宏插件实现全自动挂机
