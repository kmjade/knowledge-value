# 魔獸世界 - 知識庫整理方案 (World of Warcraft - Knowledge Base Cleanup Plan)

> **本文件位置**: [[魔獸世界-MOC|⬅️ 返回主索引]]

---

## 📋 整理目標 (Cleanup Goals)

### 🎯 主要目標
- 🔍 **識別重複內容**: 找出重複或相似的文件
- 🗂️ **刪除過時文件**: 移除不再需要的舊文件
- 📁 **整合分散內容**: 將分散的內容合併到統一文件
- 🔗 **優化鏈接結構**: 確保所有鏈接正確且高效
- 🏷️ **統一標籤系統**: 規範化所有文件的標籤使用
- 📊 **建立清晰架構**: 創建易於理解和維護的目錄結構

---

## 🔍 當前問題分析 (Current Issues Analysis)

### 已識別的問題

#### 1. 文件分散 (Scattered Files)
**問題描述**:
- 相關內容分散在多個文件夾中
- 缺乏統一的組織結構
- 難以找到特定內容

**具體例子**:
- 職業分析文件分散在多個文件夾
- 機制分析重複內容
- PVP環境分析分散
- 地心之戰12.0有多個版本

#### 2. 重複內容 (Duplicate Content)
**問題描述**:
- 多個文件包含相同或相似的內容
- 版本控制不清晰
- 難以確定哪個是最新版本

**具體例子**:
- 多個「地心之戰12.0」變體
- 重複的職業機制分析
- 相似的工作清單文件
- 重複的英雄天賦系統記錄

#### 3. 命名不一致 (Inconsistent Naming)
**問題描述**:
- 文件命名風格不統一
- 中文和英文混用不一致
- 難以搜索和定位

**具體例子**:
- 有的使用「地心之戰12.0」，有的使用「地心之戰」
- 有的使用全稱，有的使用縮寫
- 標點符號使用不統一

#### 4. 鏈接混亂 (Chaotic Links)
**問題描述**:
- 鏈接指向錯誤文件
- 重定向鏈接缺失
- 外部鏈接失效

**具體例子**:
- 某些MOC中的鏈接指向不存在
- 版本比較鏈接缺失
- 舊資料片鏈接未更新

## 📁 整理方案 (Cleanup Plan)

### 階段 1: 文件審查 (Phase 1: File Review)

#### 審查清單
- [ ] **列出所有文件**: 使用Glob列出所有.md文件
- [ ] **分類整理**: 按類型和主題分類
- [ ] **識別重複**: 找出重複內容
- [ ] **標記過時**: 標記需要刪除或合併的文件

#### 審查標準
| 標準 | 說明 | 操作 |
|------|------|------|
| **重複內容** | 文件包含與其他文件相同的內容 | 合併或刪除舊版 |
| **過時信息** | 信息已不再準確或相關 | 更新或刪除 |
| **孤立文件** | 沒有被引用的孤立文件 | 合併到相關文件 |
| **模板文件** | 可能有重複的模板 | 保留最新版本 |
