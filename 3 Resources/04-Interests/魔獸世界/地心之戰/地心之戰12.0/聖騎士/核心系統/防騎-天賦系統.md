# 魔兽世界 - 防御圣骑士天赋系统

> [!info] 天赋系统重做
> 地心之战 12.0 版本圣骑士天赋系统全面重做，采用新的天赋树结构，共 10 层天赋，每层 3 个选项。

---

## 天赋树结构

### 天赋树概览
- **层数**：10 层
- **每层选项**：3 个天赋节点
- **前置条件**：必须按顺序解锁下一层
- **重置消耗**：免费重置（12.0 改动）

### 专精分支
- **神圣（Holy）**：治疗专精
- **防护（Protection）**：坦克专精 ← 防骑
- **惩戒（Retribution）**：输出专精

## 通用天赋推荐（防骑优先）

### 第 1 层（等级 11）
```
推荐：[中] 圣洁光环
效果：自身及周围 40 米友方受到的暗影伤害降低 5%
理由：团队必备，提升团队整体生存
优先级：★★★★★
```

### 第 2 层（等级 14）
```
推荐：[中] 圣光灌注
效果：神圣震击施放时间减少 0.5 秒
理由：提升神圣震击施放速度，增加输出灵活性
优先级：★★★☆☆
```

### 第 3 层（等级 18）
```
推荐：[左] 强化十字军打击
效果：十字军打击伤害提升 15%，并产生 1 点额外神圣力量
理由：提升十字军打击价值，神圣力量获取更高效
优先级：★★★★★
```

### 第 4 层（等级 21）
```
推荐：[左] 神圣愤怒
效果：愤怒之锤冷却时间减少 3 秒
理由：缩短打断技能 CD，提升打断效率
优先级：★★★★☆
```

### 第 5 层（等级 25）
```
推荐：[左] 强化审判
效果：审判伤害提升 20%，神圣力量获取翻倍
理由：防骑核心技能，大幅提升神圣力量获取效率
优先级：★★★★★（必须天赋）
```

### 第 6 层（等级 29）
```
推荐：[右] 神圣复仇
效果：复仇（防骑复仇之怒）冷却时间减少 15 秒
理由：缩短主要爆发技能 CD，提升爆发频率
优先级：★★★★★
```

### 第 7 层（等级 34）
```
推荐：[左] 神圣之盾强化
效果：神圣之盾效果期间移动速度提升 20%
理由：提升神圣之盾期间机动性，更方便拉怪
优先级：★★★★☆
```

### 第 8 层（等级 39）
```
推荐：[右] 天使降临
效果：圣疗施放时，获得 10 秒 50% 伤害减免
理由：圣疗后提升自身生存，应对可能报复
优先级：★★★★★
```

### 第 9 层（等级 44）
```
推荐：[中] 神圣审判
效果：审判造成伤害时，目标被"审判"效果影响，受到的所有神圣伤害提升 10%
理由：提升神圣伤害整体输出，团队配合更强大
优先级：★★★★★
```

### 第 10 层（等级 50 - 终极天赋）
```
推荐：[中] 守护者形态
效果：变为守护者形态 20 秒，护甲值提升 50%，神圣力量获取翻倍，神圣之盾冷却减少 50%
理由：防骑终极爆发技能，护甲+神圣力量获取+神圣之盾CD三重提升
优先级：★★★★★★（终极必点）
```

## 必点天赋（防骑）

### 核心天赋（必须点）
1. **圣洁光环**（第 1 层中）
   - 团队必备，5% 暗影减伤

2. **强化十字军打击**（第 3 层左）
   - 十字军打击价值提升，神圣力量获取+1

### 推荐天赋（强烈推荐）
7. **圣光灌注**（第 2 层中）
   - 神圣震击施法时间-0.5 秒

8. **神圣愤怒**（第 4 层左）
   - 愤怒之锤 CD-3 秒

## 天赋配置方案

### 方案 1：生存型配置
```
特点：最大化生存能力，适合新手

核心天赋：
- 圣洁光环
- 勇敢之心
- 守护之盾
- 神圣之盾强化
- 守护者形态

优点：
- 生存能力强
- 容错率高
- 适合坦克新手

缺点：
- 输出较低
- 仇恨建立较慢
```

### 方案 2：仇恨型配置
```
特点：最大化仇恨建立，适合团队副本

核心天赋：
- 强化十字军打击
- 强化审判
- 神圣复仇
- 神圣审判
- 守护者形态

优点：
- 仇恨建立极快
- 适合拉怪
- 团队副本优势明显

缺点：
- 生存相对较弱
- 输出依赖神圣力量
```

### 方案 3：平衡型配置（推荐）
```
特点：平衡生存和输出，适合大多数场景

核心天赋：
- 圣洁光环
- 强化十字军打击
- 强化审判
- 神圣复仇
- 神圣审判
- 守护者形态

优点：
- 生存和输出平衡
- 神圣力量获取高效
- 适用场景广泛

缺点：
- 需要一定操作技巧
- 需要把握时机
```

### 方案 4：爆发型配置
```
特点：最大化爆发伤害，适合需要高仇恨场景

核心天赋：
- 强化审判
- 神圣复仇
- 神圣审判
- 神圣之怒
- 守护者形态

优点：
- 爆发能力强
- 仇恨建立极快
- 守护者形态效果最大化

缺点：
- 爆发间隔长
- 爆发期后较弱
- 生存相对降低
```

## 天赋与技能配合

### 神圣力量获取最大化
```
强化审判天赋（第 5 层）：
- 审判产生 4 点神圣力量（翻倍）
- 每 8 秒一次 → 0.5 点/秒

标准审判：
- 审判产生 2 点神圣力量
- 每 8 秒一次 → 0.25 点/秒

提升幅度：100% 提升
```

### 爆发最大化
```
守护者形态（第 10 层）：
- 护甲 +50%
- 神圣力量获取翻倍（审判 4 点）
- 神圣之盾 CD-50%

复仇之怒开启：
- 30% 伤害提升
- 神圣之怒天赋额外 +30% 伤害
- 总伤害提升：60%
```

### 生存最大化
```
神圣之盾强化 + 勇敢之心：
- 神圣之盾期间移动速度+20%
- 圣佑术持续时间+3 秒（共 11 秒）
- 天使降临：圣疗后 10 秒 50% 减伤

组合效果：
- 连续减伤时间：11 秒（圣佑术）+ 10 秒（天使降临）= 21 秒
- 神圣之盾机动性提升
- 圣疗后生存保障
```

## 天赋切换场景

### 副本天赋配置
```
副本场景特点：
- 需要高生存
- 需要稳定仇恨
- 需要团队贡献

推荐配置：平衡型配置
理由：生存和输出平衡，适用副本各种场景
```

### PVP 天赋配置
```
战场场景特点：
- 需要解控能力
- 需要保护队友
- 需要生存能力

推荐配置：生存型配置
理由：圣洁光环、天使降临、奉献之愿等团保天赋优势明显

竞技场场景特点：
- 需要爆发能力
- 需要控制应对
- 需要保护配合

推荐配置：平衡型配置 + 奉献之愿
理由：爆发和生存兼顾，牺牲祝福团队价值高
```

## 进阶技巧

### 天赋模拟器使用
1. **测试配置**：在天赋模拟器中测试不同配置
2. **数据对比**：对比输出、生存、仇恨等数据
3. **实战验证**：在副本和 PVP 中验证配置效果
4. **持续优化**：根据实战反馈调整天赋

### 天赋与装备配合
1. **命中配天赋**：装备命中不足时选择提高命中的天赋
2. **暴击配天赋**：装备暴击高时选择暴击相关天赋
3. **加速配天赋**：装备加速高时选择减少 CD 的天赋

### 团队天赋配合
1. **沟通配置**：与其他职业沟通天赋配置
2. **互补选择**：选择与团队职业互补的天赋
3. **团保优先**：优先选择团队保护天赋
