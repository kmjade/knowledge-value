## ⚒️ 製造專業

### 三大製造專業

#### 鍛造 (Blacksmithing)

| 項目 | 內容 |
|------|------|
| 可製作 | 護甲、武器、珠寶 |
| 核心材料 | 錘煉恐渣、裂顱者礦石 |
| 推薦度 | ⭐⭐⭐⭐ |

**推薦產品**：
- 護甲片（升級必備）
- 武器（初期高價值）
- 珠寶（後期高收益）

#### 珠寶設計 (Jewelcrafting)

| 項目 | 內容 |
|------|------|
| 可製作 | 珠寶、寶石、戒指 |
| 核心材料 | 璀璨寶石、龍眼石 |
| 推薦度 | ⭐⭐⭐⭐⭐ |

**推薦產品**：
- 插槽寶石（必備）
- 戒指（高需求）
- 項鏈（初期高價值）

#### 製皮 (Leatherworking)

| 項目 | 內容 |
|------|------|
| 可製作 | 護甲、護腕、皮革 |
| 核心材料 | 原始皮革、堅硬肌腱 |
| 推薦度 | ⭐⭐⭐⭐ |

**推薦產品**：
- 護甲片（升級必備）
- 皮甲（高需求）
- 護腕（前期高價值）

### 輔助製造專業

#### 裁縫 (Tailoring)

| 項目 | 內容 |
|------|------|
| 可製作 | 布甲、包包、附魔材料 |
| 核心材料 | 夢境絲線、布匹 |
| 推薦度 | ⭐⭐⭐⭐ |

**推薦產品**：
- 包包（必備）
- 布甲（法系必備）
- 附魔卷軸（高需求）

#### 銘文 (Inscription)

| 項目 | 內容 |
|------|------|
| 可製作 | 藥水、卷軸、飾品 |
| 核心材料 | 墨水、草藥 |
| 推薦度 | ⭐⭐⭐⭐ |

**推薦產品**：
- 藥水（消耗品）
- 捲軸（必備）
- 飾品（後期高價值）

#### 工程施工 (Engineering)

| 項目 | 內容 |
|------|------|
| 可製作 | 槍械、護甲、玩具 |
| 核心材料 | 發條零件、蒸汽 |
| 推薦度 | ⭐⭐⭐ |

**推薦產品**：
- 護甲（特色）
- 玩具/寵物（趣味）
- 槍械（收藏）

## ⛏️ 收集專業

### 雙採專業

#### 採礦 (Mining)

| 礦石 | 等級 | 地點 |
|------|------|------|
| 裂顱者礦石 | 1-50 | 卡茲阿爾加 |
| 恐渣 | 50-100 | 阿茲-卡赫特 |

**採集地點推薦**：
- 卡茲阿爾加礦脈
- 阿茲-卡赫特礦坑
- 地底王國礦洞

#### 草藥學 (Herbalism)

| 草藥 | 等級 | 地點 |
|------|------|------|
| 夢境葉 | 1-50 | 卡茲阿爾加 |
| 深淵菇 | 50-100 | 阿茲-卡赫特 |

**採集地點推薦**：
- 卡茲阿爾加森林
- 阿茲-卡赫特沼澤
- 地底王國洞穴

#### 剝皮 (Skinning)

| 材料 | 等級 | 地點 |
|------|------|------|
| 原始皮革 | 1-50 | 卡茲阿爾加 |
| 堅硬肌腱 | 50-100 | 阿茲-卡赫特 |

**剝皮地點推薦**：
- 卡茲阿爾加野外
- 阿茲-卡赫特生物
- 地底王國

#### 分解 (Disenchanting)

**推薦分解裝備**：
- 綠裝（初期）
- 藍裝（中後期）
- 史詩裝（後期）
