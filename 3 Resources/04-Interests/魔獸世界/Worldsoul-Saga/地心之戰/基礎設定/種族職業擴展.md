## 📋 擴展概述

### 🎯 變化統計

#### 主要變化數據
| 變化類型 | 數量 | 說明 |
|----------|------|------|
| **新增職業組合** | 44個 | 11種族×4個職業 |
| **新種族引入** | 1個 | 土靈種族加入聯盟 |
| **職業組合限制** | 大幅開放 | 大部分組合限制被移除 |
| **角色扮演可能性** | 大幅提升 | 新的角色扮演可能性 |

#### 影響範圍
| 影響領域 | 變化程度 | 主要影響 |
|----------|----------|----------|
| **角色創造** | 🔴 大幅變化 | 更多角色創造選擇 |
| **隊伍配置** | 🟡 中等變化 | 更靈活的隊伍配置 |
| **陣營平衡** | 🟡 中等變化 | 陣營間職業分布更均衡 |
| **社區互動** | 🔴 大幅變化 | 新的社區互動模式 |

### 🔧 設計理念

#### 核心原則
1. **包容性**：讓更多玩家可以選擇喜歡的種族職業組合
2. **文化連貫**：考慮種族文化背景和職業連貫性
3. **平衡性**：確保陣營間的職業分布平衡
4. **創造性**：激發玩家的創造力和角色扮演热情

#### 技術考慮
1. **性能優化**：確保新的組合不影�遊戲性能
2. **平衡性維護**：持續監控和調整平衡性
3. **內容更新**：更新相關的任務和故事內容
4. **社區管理**：管理社區對新組合的反饋和期望

## 🆕 土靈種族

### 🏔️ 基本特徵

#### 種族背景
- **陣營**：聯盟
- **起源**：矮人的親族，居住在深大地底
- **文化**：石製構裝體，高度發展的地底文明
- **歷史**：被遺忘者詛咒後變形的矮人後裔

#### 外觀特徵
- **體型**：比普通矮人更高大
- **皮膚**：岩石般的質感
- **眼睛**：發光的礦物色
- **髮型**：岩石般的質感，多為黑色或深灰色

#### 文化特色
- **社會結構**：以礦物和岩石為基礎的社會
- **宗教信仰**：崇拜大地和礦物
- **技術水平**：高度發達的石製技術
- **戰鬥方式**：利用岩石和礦物進行戰鬥

### 🎯 角色扮演建議

#### 背景故事建議
- **起源**：可以描寫為被詛咒後變形的矮人
- **使命**：尋找恢復原形的辦法或接受新的身份
- **目標**：探索深大地底的秘密或恢復矮人的榮光

#### 社會關係
- **與矮人**：複雜的親族關係，有時有矛盾
- **與聯盟**：忠誠的盟友，貢獻自己的力量

#### 性格特徵
- **堅韌**：像岩石一樣堅韌不拔
- **智慧**：地底文明帶來的智慧
- **孤獨**：與普通種族的隔閡感

## 📊 影響分析

### 🎯 對遊戲體驗的影響

#### 正面影響
1. **角色創造自由度**：可以自由選擇喜歡的種族職業組合
2. **文化豐富性**：新的組合帶來了豐富的文化背景
3. **社區互動**：新的組合帶來了新的社區互動模式
4. **遊戲多樣性**：增加了遊戲的玩法和內容多樣性

#### 潛在挑戰
1. **平衡性維護**：新的組合可能帶來平衡性挑戰
2. **文化連貫性**：新的組合可能存在文化連貫性問題
3. **玩家習慣**：需要時間適應新的組合選擇
4. **社區反饋**：需要處理社區對新組合的反饋
