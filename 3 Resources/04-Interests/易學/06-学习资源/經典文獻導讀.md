# 經典文獻導讀

> 🎯 **學習目標**：瞭解易學經典文獻，掌握《周易本義》《易經來注》《周易折中》等重要經典著作的內容和特點，能夠選擇適合的經典文獻進行深入學習。

## 📚 經典文獻概述

### 易學經典的重要性

易學經典文獻是易學研究的基石，是學習易學的重要資源。歷代易學家通過註釋、解說、研究經典，形成了豐富的易學文獻體系。研讀經典文獻，可以幫助我們深入理解易理，掌握易學的真諦。

### 經典文獻的分類

**原著經典**：《周易》經傳原文

**註疏經典**：歷代易學家的註疏和解說

**研究經典**：歷代易學家的研究著作

## 🔄 原著經典

### 周易

#### 《周易》的結構

**《周易》的組成**：
- 《經》：六十四卦、三百八十四爻、卦辭、爻辭
- 《傳》：《十翼》：彖傳、象傳、繫辭傳、文言傳、說卦傳、序卦傳、雜卦傳

**《周易》的特點**：
- 經傳結合：經文和傳文結合，相互補充
- 義理象數：義理和象數並重
- 哲學占卜：哲學和占卜並重

#### 《周易》的研讀

**研讀方法**：
1. **先讀原文**：先讀《周易》原文，建立初步認識
2. **參考註疏**：參考歷代注家，理解不同解讀
3. **融會貫通**：將八卦、五行、六十四卦聯繫起來
4. **生活實踐**：將易學智慧應用於日常生活

**研讀建議**：
- 循序漸進：從簡單到複雜，循序漸進
- 多讀多思：多讀原文，多思考易理
- 結合實踐：將易理應用於生活實踐
