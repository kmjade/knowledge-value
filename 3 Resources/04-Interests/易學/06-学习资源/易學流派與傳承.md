# 易學流派與傳承

> 🎯 **學習目標**：瞭解易學的主要流派和傳承歷史，掌握象數易、義理易等流派的特點，理解易學的發展脈絡，能夠根據個人興趣選擇適合的易學流派深入學習。

## 📚 易學流派概述

### 易學流派的形成

易學在長期的發展過程中，形成了不同的流派。各流派在研究方法、學術觀點、傳承方式等方面各有特點，共同構成了豐富多彩的易學體系。

### 易學流派的分類

**象數易**：重視象數，研究卦象和數理

**義理易**：重視義理，研究哲學內涵

**醫易同源**：研究易學與醫學的關係

**兵法易學**：研究易學與兵法的關係
