# 140 哲学流派 / Philosophical Schools

> 不是「哪个哲学家说了什么」，而是「思考世界有几套底层语法」。
> 哲学流派是思想的范式 (paradigm) —— 每一种流派提供了一套不兼容的预设。

## 九大流派详解

### ① 理性主义 (Rationalism) · 17-18C

> **知识的主要来源是理性 (reason)，而非感官经验。**

| 维度 | 立场 |
|------|------|
| 认识论 | 先天知识 (a priori) 存在且优先；数学是最佳模型 |
| 形而上学 | 实体二元论 (笛卡尔)、一元论 (斯宾诺莎)、单子论 (莱布尼茨) |
| 方法 | 演绎推理；从自明公理出发 |
| 代表人物 | 笛卡尔、斯宾诺莎、莱布尼茨 |
| 口号 | 「我思故我在」(Cogito ergo sum) |

### ② 经验主义 (Empiricism) · 17-18C

> **知识的唯一来源是感官经验 (experience)。**

| 维度 | 立场 |
|------|------|
| 认识论 | 心灵是「白板」(tabula rasa)；无先天观念 |
| 形而上学 | 对实体、因果持怀疑态度 |
| 方法 | 归纳；从观察到概括 |
| 代表人物 | 洛克、贝克莱、休谟 |
| 口号 | 「知觉之外无物存在」(贝克莱) |

### ③ 先验观念论 (Kantianism / Transcendental Idealism) · 1781+

> **康德：「理性无经验则空，经验无理性则盲」—— 理性主义与经验主义的综合。**

| 维度 | 立场 |
|------|------|
| 认识论 | 现象界 (可认识) vs 物自身 (不可认识)；心灵有先天结构 (范畴) |
| 形而上学 | 不直接讨论物自身；限定理性的合法范围 |
| 方法 | 先验论证：使经验可能的必要条件 |
| 代表人物 | 康德 |
| 口号 | 「哥白尼革命」：不是知识符合对象，而是对象符合知识 |

### ④ 唯心主义 (Idealism) · 18-19C

> **实在的本质是精神/观念 (mind/idea)，而非物质。**

| 维度 | 立场 |
|------|------|
| 认识论 | 一切知识都是心灵对自身内容的意识 |
| 形而上学 | 绝对精神 (黑格尔)、意志 (叔本华)；世界是表象 |
| 方法 | 辩证法 (正→反→合) |
| 代表人物 | 贝克莱、黑格尔、叔本华、布拉德雷 |
| 口号 | 「存在即被感知」(贝克莱) / 「凡是现实的都是合理的」(黑格尔) |

### ⑤ 实在论 (Realism) · 贯穿

> **外部世界独立于心灵而真实存在。**

| 维度 | 立场 |
|------|------|
| 认识论 | 我们能认识独立于心灵的世界（不是一切都被心灵构造） |
| 形而上学 | 共相实在论 (柏拉图) 或科学实在论 (实体不依赖观察) |
| 方法 | 诉诸常识与科学 |
| 代表人物 | 亚里士多德、摩尔 (常识实在论)、普特南 (内在实在论) |
| 口号 | 「这里有一只手」(摩尔举起手) |

### ⑥ 实用主义 (Pragmatism) · 19C末-20C

> **一个观念的真理性在于它的实际效果 (practical consequences)。**

| 维度 | 立场 |
|------|------|
| 认识论 | 真理 ≠ 符合实在，真理 = 有用的信念 |
| 形而上学 | 反形而上学空谈；形而上学问题应被消解 |
| 方法 | 皮尔士：「考虑你的概念会有什么实际效果」 |
| 代表人物 | 皮尔士、詹姆斯、杜威、罗蒂 |
| 口号 | 「真的不过是在信念方式上权宜的」(詹姆斯) |

### ⑦ 现象学 (Phenomenology) · 20C初

> **回到事物本身 (Zu den Sachen selbst) — 描述意识的结构。**

| 维度 | 立场 |
|------|------|
| 认识论 | 悬搁 (epoché)：搁置关于外部世界存在的问题；描述意识体验 |
| 形而上学 | 不预设，只描述「给予意识的东西」 |
| 方法 | 现象学还原；意向性分析 |
| 代表人物 | 胡塞尔、海德格尔、梅洛-庞蒂 |
| 口号 | 「意识总是关于某物的意识」(意向性) |

### ⑧ 分析哲学 (Analytic Philosophy) · 20C初-

> **哲学问题可以通过语言分析来澄清或消解。**

| 维度 | 立场 |
|------|------|
| 认识论 | 强调清晰性、逻辑分析；哲学是「概念工程」 |
| 形而上学 | 早期反对（逻辑实证主义）；后期复兴（Kripke 可能世界） |
| 方法 | 逻辑分析、概念分析、思想实验 |
| 代表人物 | 弗雷格、罗素、维特根斯坦、奎因、Kripke |
| 口号 | 「凡可说的都能说清楚，不可说的应当沉默」(维特根斯坦) |

### ⑨ 存在主义 (Existentialism) · 19-20C

> **存在先于本质 (Existence precedes essence) — 人先存在，再通过选择定义自己。**

| 维度 | 立场 |
|------|------|
| 认识论 | 主观真理 > 客观真理；真正重要的知识是「我如何活」 |
| 形而上学 | 人的存在不可还原为概念或科学描述 |
| 方法 | 文学、日记、现象学描述 |
| 代表人物 | 克尔凯郭尔、尼采、萨特、加缪、波伏娃 |
| 口号 | 「人是被判定为自由的」(萨特) |

## 流派对照矩阵

| 流派 | 知识的来源 | 实在的本质 | 人的位置 | 关键词 |
|------|-----------|-----------|---------|--------|
| 理性主义 | 理性 | 理性可把握 | 思维实体 | 先天、演绎 |
| 经验主义 | 感官 | 物质/感觉 | 白板 | 归纳、习惯 |
| 康德主义 | 先天结构+经验 | 现象/物自身 | 先验主体 | 范畴、界限 |
| 唯心主义 | 心灵 | 精神/观念 | 精神的一部分 | 绝对、表象 |
| 实在论 | 理性+经验 | 独立于心灵 | 认识者 | 常识、独立 |
| 实用主义 | 行动效果 | 不追问 | 行动者 | 效用、后果 |
| 现象学 | 意识描述 | 悬搁 | 意向性存在 | 体验、生活世界 |
| 分析哲学 | 语言/逻辑 | 经过语言分析 | 语言使用者 | 清晰、论证 |
| 存在主义 | 主观体验 | 荒谬/无意义 | 自由的行动者 | 选择、责任 |

## 历史时间线

```
1600 ─┬─ 理性主义兴起 (笛卡尔 1641《第一哲学沉思集》)
      │
1700 ─┼─ 经验主义回应 (洛克 1689《人类理解论》)
      │
      ├─ 休谟 (1748) 将经验主义推向怀疑论
      │
      ├─ 康德 (1781) 综合理性与经验 —「哥白尼革命」
      │
1800 ─┼─ 德国观念论 (黑格尔 1807《精神现象学》)
      │
      ├─ 叔本华 (1819) 意志形而上学
      │
      ├─ 克尔凯郭尔 (1843) 存在主义先驱
      │
      ├─ 实用主义诞生 (皮尔士 1870s)
      │
      ├─ 尼采 (1883-1888)「上帝已死」
      │
1900 ─┼─ 胡塞尔现象学 (1900-01《逻辑研究》)
      │
      ├─ 分析哲学兴起 (罗素/摩尔 1903 反唯心主义)
      │
      ├─ 海德格尔 (1927《存在与时间》)
      │
      ├─ 逻辑实证主义 (维也纳学派 1920s-30s)
      │
      ├─ 萨特 (1943《存在与虚无》)
      │
      ├─ 维特根斯坦后期 (1953《哲学研究》— 语言游戏)
      │
      ├─ 奎因 (1951) 批判分析/综合区分
      │
      ├─ 罗蒂 (1979) 新实用主义
      │
2000 ─┴─ 当代：分析-大陆对话、实验哲学、AI 哲学
```

## 三大对立轴

### 轴 1: 理性的角色
```
理性主义 ←──────────────────────────→ 经验主义
(笛卡尔)                              (休谟)
```

### 轴 2: 实在的性质
```
唯心主义 ←──────────────────────────→ 实在论
(黑格尔)        康德在中间              (摩尔)
```

### 轴 3: 哲学的方法
```
分析哲学 ←──────────────────────────→ 大陆哲学
(清晰·论证·逻辑)        (现象学·存在主义·解释学)
```

> 当代趋势：这三条轴不再是二选一，而是被重新审视和融合。
