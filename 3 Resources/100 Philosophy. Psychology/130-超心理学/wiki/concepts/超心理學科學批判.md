# 超心理學科學批判 Scientific Critique

## 核心批評

| 批評 | 說明 |
|------|------|
| **無物理機制** | 無已知物理理論解釋 ESP/PK |
| **複製失敗** | 正面結果難以被獨立實驗室複製 |
| **檔案抽屜問題** | 陰性結果不被發表，導致發表偏誤 |
| **實驗者效應** | 相信者獲正面結果、懷疑者獲陰性結果 |

## 偽科學問題

超心理學是否符合科學標準？

| 科學標準 | 超心理學 |
|---------|:--:|
| 可證偽性 | ✅（可通過實驗證偽） |
| 可複製性 | ⚠️（部分複製、爭議） |
| 可預測性 | ❌（無預測模型） |
| 簡約性 | ❌（更簡單的解釋已足夠） |

## 超心理學家的回應
- 效應量雖小但統計顯著（元分析支持）
- 物理學本身也在發展（量子力學的非定域性）
- 要求「超常宣稱需要超常證據」——超心理學家同意此標準
