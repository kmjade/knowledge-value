# 09-科学与批判

> 「超常的主张需要超常的证据。」——卡尔·萨根。超心理学的核心张力在于：它声称的现象如果真实，需要改写物理学和神经科学——而它提供的证据尚未达到这个门槛。

## 为什么超心理学不被主流科学接受？

| 原因 | 说明 |
|------|------|
| **效应量太小** | ESP 效应量 ~0.02-0.05——远远低于其他心理学领域 |
| **不可复现** | 没有一个 psi 实验范式能被任何独立实验室稳定复现 |
| **无理论机制** | 没有已知的物理机制可以解释 ESP/PK |
| **欺诈史** | 多个高调案例被揭露为故意欺诈 |
| **非累积性** | 60 年研究——没有「越来越强」的证据链 |
| **选择性报告** | 文件抽屉问题严重——阴性结果不发表 |

---

## 著名的欺诈案例

| 案例 | 时间 | 事件 | 后果 |
|------|------|------|------|
| **福克斯姐妹** | 1848/1888 | 用脚趾关节发出叩击声——假装是灵体 | 击垮了第一波灵性主义运动 |
| **Soal 实验** | 1940s-1950s | 英国数学家 Soal 声称 ESP 显著——后来发现篡改数据 | 超心理学遭受重创 |
| **Project Alpha** | 1979-1983 | James Randi 派遣两名年轻魔术师潜入 McDonnell 实验室——伪造 PK 能力——未被识破 | 暴露了超心理学家的「确认偏差」 |
| **Uri Geller** | 1970s | 声称 PK——被 Randi 逐一复制 | 推动了公众怀疑论 |

## James Randi 与怀疑论运动

| 要素 | 内容 |
|------|------|
| **人物** | James Randi (1928-2020) ——魔术师、怀疑论者 |
| **百万美元挑战** | 任何人能在控制条件下展示超常能力——奖励 100 万美元 |
| **持续时间** | 1964-2015 |
| **结果** | 无人通过初步测试 |
| **批评** | 测试条件过于苛刻——否认了「微弱的 psi 效应」存在的可能性 |

## 元分析之争

### 支持方的元分析

| 研究 | 现象 | 结论 |
|------|------|------|
| Radin (1997) | RNG 微 PK | 效应显著但极小——文件抽屉不可能解释 |
| Storm et al. (2010) | Ganzfeld ESP | 108 项研究——效应显著 |
| Bem et al. (2015) | 预知 | 90 项研究——效应显著 |

### 怀疑方的反驳

| 批评 | 说明 |
|------|------|
| **文件抽屉校正不够** | 传统校正低估了未发表的阴性结果数量 |
| **研究质量与效应量负相关** | 方法越严格——效应量越小——暗示效应可能是方法学噪声 |
| **p-curve 分析** | 显著结果的 p 值分布模式暗示 p-hacking 而非真实效应 |
| **预注册后消失** | 当要求预注册——效应消失 |

## 复制危机与超心理学

心理学整体在经历「复制危机」(Replication Crisis)——许多经典心理学效应无法被复现。这对超心理学意味着什么？

| 观点 | 说明 |
|------|------|
| **「我们都一样」** | 超心理学声称的效应与其他心理学领域的效应同样脆弱——问题出在心理学方法论本身 |
| **「更糟」** | 超心理学的效应比「正常」心理学更小、更不稳定、更依赖可疑统计操作 |
| **「先见之明」** | 超心理学 60 年来一直在讨论方法学问题——在其他领域最近才发现时，超心理学早就在自省 |

---

## 科学的自我纠正

| 改进 | 与超心理学的关系 |
|------|-----------------|
| **预注册** | 事先声明假设、方法、分析计划——消除 p-hacking |
| **大样本** | 更大的样本——检测真实微小效应 |
| **开放数据** | 公开原始数据和代码——防止选择性报告 |
| **Registered Reports** | 基于方法而非结果的同行评议——消除发表偏差 |
| **多实验室复现** | Many Labs 项目——多个实验室独立复现同一实验 |

> 如果 psi 效应是真实的——这些改进应该让它更清晰，而不是消失。

## 结论：证据的平衡

| 立场 | 主张 | 代表人物 |
|------|------|---------|
| **坚信者** | Psi 效应已被元分析确证——怀疑来自「科学主义的偏见」 | Radin, Sheldrake |
| **温和开放** | 有些现象（Ganzfeld, 微 PK）值得继续研究——但证据尚不充分 | Cardeña, 部分认知科学家 |
| **温和怀疑** | 当前证据不能排除方法学噪声——需要更严格的研究 | Wiseman, French |
| **坚定怀疑** | 超心理学 130 年无进展——资源应该投向更有效的方向 | Hyman, Blackmore |
