## 关键方法论提示

在学习超心理学时，持续追问四个问题：

| 问题 | 说明 |
|------|------|
| **效应量多大？** | 统计显著 ≠ 实际显著——ESP 的效应量通常很小 |
| **可重复吗？** | 单一实验不能说明问题——元分析怎么说？ |
| **有替代解释吗？** | 选择性报告？文件抽屉问题？感官泄露？冷读？ |
| **与既有科学兼容吗？** | 如果 ESP 存在，物理学需要修改什么？ |
