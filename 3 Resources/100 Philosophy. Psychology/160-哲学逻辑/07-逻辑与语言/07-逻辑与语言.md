# 07-逻辑与语言

> 逻辑的工具不是为数学发明的——弗雷格发明谓词逻辑的初衷，是理解语言如何表达思想。逻辑与语言的关系是分析哲学的核心。

## 弗雷格：意义与指称

### 晨星 = 昏星？

| 表达式 | 指称 (Bedeutung) | 意义 (Sinn) |
|--------|-----------------|------------|
| 「晨星」 | 金星 (Venus) | 清晨出现在东方的那颗明亮天体 |
| 「昏星」 | 金星 (Venus) | 黄昏出现在西方的那颗明亮天体 |

> 「晨星就是昏星」——这是一个**信息增量**的判断——而「金星就是金星」不是。它们的意义不同，但指称相同。

### 弗雷格的贡献

| 概念 | 内容 |
|------|------|
| **命题的态度语境** | 「John 相信晨星是行星」——不能把「晨星」替换为「昏星」因为 John 可能有不同的信念 |
| **函项-论证分析** | 「苏格拉底是人」→ 函项 Human(苏格拉底) ——谓词是函项，主词是论证 |
| **量词** | 「所有人都会死」= ∀x (Human(x) → Mortal(x)) ——「所有」是二阶函项 |

## 罗素：摹状词理论 (1905)

> 「当今法国国王是秃头。」——这个命题是假的吗？

| 问题 | 说明 |
|------|------|
| 当今法国没有国王——如果命题是假的，它的否定「当今法国国王不是秃头」应该为真 | 但那也为假 |
| 排中律失效 | 经典逻辑无法处理指称失败 |

### 摹状词分析

罗素的方案——把「当今法国国王是秃头」（表面上是「F(a)」的形式）重新分析为：

```
∃x (KingOfFrance(x) ∧ ∀y (KingOfFrance(y) → y=x) ∧ Bald(x))
存在 x：x 是法国国王，且只有 x 是法国国王，且 x 是秃头。
```

因为不存在满足前两个条件的 x ——整个命题为**假**（不是无意义！）。它的否定也就可以是真了。

> 罗素的摹状词理论被誉为「分析哲学的典范」——用逻辑分析消解哲学困惑。

## 早期维特根斯坦：《逻辑哲学论》

| 命题 | 内容 |
|------|------|
| 1 | 世界是一切实际如此的情况 |
| 7 | 对于那些不可说的东西，必须保持沉默 |

### 语言图像论

| 概念 | 内容 |
|------|------|
| **核心** | 命题是现实的图像——命题的结构映射事态的结构 |
| **真值表** | 维特根斯坦发明了真值表作为逻辑工具 |
| **重言式** | 在所有可能情况下为真——不说关于世界的任何东西——是逻辑本身的 |
| **可说 vs 不可说** | 自然科学的命题可说——伦理、美学、宗教、逻辑形式本身——不可说 |

## 逻辑实证主义

| 要素 | 内容 |
|------|------|
| **口号** | 「一个命题的意义就是它的证实方法」 |
| **核心** | 有意义的命题要么是分析的（逻辑/数学），要么是综合的（可经验证实） |
| **被拒绝** | 形而上学——无意义——不是假，是无法判断真假——伪命题 |
| **问题** | 「可证实性原则」本身是分析的还是综合的？——它既不能分析也不能经验证实 |

## 日常语言转向

| 人物 | 主张 |
|------|------|
| **后期维特根斯坦** | 语言的意义在使用中——不是图像。「语言游戏」——不同语境 = 不同规则 |
| **J.L. Austin** | 「如何以言行事」——说话本身就是行动（承诺、命令、宣告） |
| **Grice** | 会话含义——说出来的 vs 暗示的——逻辑只捕捉了前者 |
