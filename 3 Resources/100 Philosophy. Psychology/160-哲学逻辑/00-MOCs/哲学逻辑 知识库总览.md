# 哲学逻辑 知识库总览 (DDC 160)

> 哲学逻辑（Philosophical Logic）研究推理的有效性、逻辑真理的本质和形式系统的哲学基础。从亚里士多德的三段论到哥德尔不完备定理，从经典二值逻辑到模糊逻辑——这是思维的骨架。对应 DDC 160「哲学逻辑」。

## 逻辑的演进

```
公元前 4 世纪        19 世纪末             20 世纪初            1931
┌──────────┐     ┌──────────────┐     ┌──────────────┐     ┌──────────┐
│ 亚里士多德  │ → │ 弗雷格         │ → │ 罗素 & 怀特海  │ → │ 哥德尔     │
│ 三段论      │   │ 概念文字       │   │ 数学原理       │   │ 不完备定理  │
└──────────┘     └──────────────┘     └──────────────┘     └──────────┘
 传统逻辑           现代逻辑诞生         逻辑主义纲领           逻辑的边界
```
