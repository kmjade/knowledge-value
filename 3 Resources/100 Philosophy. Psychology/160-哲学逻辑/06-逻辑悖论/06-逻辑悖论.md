# 06-逻辑悖论

> 一个命题从看似合理的前提，通过看似正确的推理，竟然推出了矛盾——**悖论**。悖论不是逻辑的失败——它们是逻辑的边界指示牌。

## 什么是悖论？

| 要素 | 内容 |
|------|------|
| **定义** | 从看似可接受的前提出发，通过看似有效的推理，得出不可接受的结论 |
| **结构** | 前提 → 推理 → 矛盾/荒谬结论 |
| **三种反应** | ① 拒绝前提 ② 拒绝推理步骤 ③ 接受结论（悖论不是悖论） |

---

## 说谎者悖论

> 「这句话是假的。」

| 分析 | 内容 |
|------|------|
| 假设它为真 → 它说「这句话是假的」→ 所以它为假 | 矛盾 |
| 假设它为假 → 它说「这句话是假的」→ 所以它为真 | 矛盾 |

### 为什么它重要？

说谎者悖论直接威胁了逻辑的基本假设——**二值原则**（非真即假）和**不矛盾律**（不能既真又假）。

### 塔斯基的解决

| 步骤 | 内容 |
|------|------|
| 分层 | 对象语言 (Object Language) + 元语言 (Metalanguage) |
| 规则 | 「真」谓词不能出现在对象语言中——只能出现在元语言中 |
| 后果 | 「这句话是假的」——这句话不能合法地构造——它跨越了语言分层 |

> 塔斯基：自然语言是「语义封闭」的（包含自己的真谓词）——这就是悖论的根源。

## 罗素悖论

> 定义 R = { x | x ∉ x } ——所有不属于自身的集合的集合。
> 问：R 属于 R 吗？

| 假设 | 结果 |
|------|------|
| R ∈ R → 根据定义，R 中的元素都不属于自身 → R ∉ R | 矛盾 |
| R ∉ R → 根据定义，R 收集所有不属于自身的集合 → R ∈ R | 矛盾 |

### 后果

罗素悖论击溃了弗雷格的逻辑主义大厦——弗雷格收到罗素的信后写道：「一个科学家最不愿看到的事情，就是在工作完成时发现基础崩塌。」

| 解决方案 | 内容 |
|---------|------|
| **类型论** (Russell) | 给集合分层——x ∈ y 仅在 x 的类型比 y 低一级时合法 |
| **ZFC 集合论** | 限制概括公理——不能随意定义「所有满足 φ(x) 的 x 的集合」 |

## 谷堆悖论 (Sorites Paradox)

> 1 粒沙子不是谷堆。
> 如果 n 粒沙子不是谷堆，那么 n+1 粒沙子也不是谷堆。
> ∴ 10000 粒沙子也不是谷堆。

| 问题 | 说明 |
|------|------|
| **根源** | 模糊谓词——「谷堆」没有精确边界 |
| **经典逻辑的困境** | 经典逻辑要求精确真值——模糊概念无法处理 |
| **出路** | [[../05-非经典逻辑/05-非经典逻辑\|模糊逻辑]] ——允许多值真值 |

## 芝诺悖论

### 阿喀琉斯与乌龟

> 阿喀琉斯领先乌龟 100 米。当他跑到乌龟的位置时，乌龟又前进了一点。当他跑完这「一点」时，乌龟又前进了……∴ 阿喀琉斯永远追不上乌龟。

| 分析 | 内容 |
|------|------|
| **数学解决** | 无穷级数 1/2 + 1/4 + 1/8 + ... = 1 ——有限时间内跨过无限分划 |
| **哲学问题** | 任何有限区间都包含无限多个点——如何「跨过」无限？ |

## 悖论的类型

| 类型 | 例子 | 根源 |
|------|------|------|
| **语义悖论** | 说谎者、Grelling | 自指 + 真/满足概念 |
| **集合论悖论** | 罗素、Cantor | 无限制概括 + 自指 |
| **模糊悖论** | 谷堆 | 模糊谓词 + 经典逻辑 |
| **知识悖论** | 意外绞刑、知道者 | 自指 + 认知概念 |
| **决策悖论** | 纽康难题、囚徒困境 | 因果 + 合理选择 |
