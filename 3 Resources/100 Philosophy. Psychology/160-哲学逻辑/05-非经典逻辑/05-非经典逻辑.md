# 05-非经典逻辑

> 经典逻辑假设每个命题非真即假——排中律 (P∨¬P) 永远成立。但如果现实不是非黑即白的呢？如果矛盾可以「局部」容忍呢？非经典逻辑质疑经典逻辑的一个或多个假设。

## 为什么需要非经典逻辑？

| 经典假设 | 挑战 | 非经典逻辑 |
|---------|------|-----------|
| **二值原则**：每个命题非真即假 | 模糊概念、未定义项、未来偶然命题 | 多值逻辑、模糊逻辑 |
| **排中律**：P∨¬P | 数学构造主义——不构造出来不算存在 | 直觉主义逻辑 |
| **爆炸律**：矛盾推出一切 | 处理不一致信息——数据库、法律 | 弗协调逻辑 |
| **单调性**：新前提不撤销旧结论 | 日常推理——新信息推翻旧结论 | 非单调逻辑 |
| **联结词的函项性** | 「因为 A 所以 B」要求 A 和 B 有内容关联 | 相干逻辑 |

---

## 直觉主义逻辑 (Intuitionistic Logic)

| 要素 | 内容 |
|------|------|
| **创始人** | L.E.J. Brouwer (1881-1966) |
| **核心思想** | 数学真理 = 被构造出来的证明——不能说「存在一个 x 使得 P(x)」除非你能**构造**出这个 x |
| **被拒绝** | 排中律 (P∨¬P) ——未证明不等于假 |
| **双重否定** | ¬¬P 不等于 P ——「不是假的」不等于「是真的」 |

### 经典 vs 直觉主义

| 经典有效 | 直觉主义？ |
|---------|:--------:|
| P ∨ ¬P | ❌ |
| ¬¬P → P | ❌ |
| (¬P → ⊥) → P | ❌ (同上) |
| ¬(P∧Q) → (¬P∨¬Q) | ❌ |
| (P→Q) ∨ (Q→P) | ❌ |

> 直觉主义逻辑在计算机科学中找到了意想不到的家——Curry-Howard 同构：直觉主义证明 = 程序，公式 = 类型。

## 弗协调逻辑 (Paraconsistent Logic)

| 要素 | 内容 |
|------|------|
| **动机** | 经典逻辑中，从一个矛盾可以推出任何命题 (爆炸律)——但这不反映人类推理：我们从数据库中发现了不一致，但我们不会因此相信一切 |
| **核心** | 拒绝爆炸律：A, ¬A ⊬ B |
| **应用** | 不一致的数据库、法律系统（冲突的证据）、科学理论（新旧理论的过渡期） |

### 爆炸律为什么不好？

```
经典逻辑中：
  矛盾产生 ⇨ 一切皆可证 ⇨ 逻辑无用
弗协调逻辑中：
  矛盾产生 ⇨ 逻辑仍然能区分合理的和不合理的推论
```

## 多值逻辑与模糊逻辑

### 三值逻辑 (Łukasiewicz, 1920)

| 真值 | 含义 |
|------|------|
| 1 | 真 |
| ½ | 不确定 (可能的——未来偶然命题) |
| 0 | 假 |

> 「明天将有一场海战」——亚里士多德认为这个命题现在既不真也不假。

### 模糊逻辑 (Fuzzy Logic — Zadeh, 1965)

| 要素 | 内容 |
|------|------|
| **核心** | 真值是 [0,1] 中的**任何实数**——不是 {0, ½, 1} |
| **例子** | 「张三很高」——身高 175cm → 真值 0.7；180cm → 0.9 |
| **应用** | 控制系统 (洗衣机、空调)、AI 推理、决策支持 |
| **特点** | 模糊逻辑不是概率——它不涉及随机性，而是概念的边界不清晰 |

---

## 相干逻辑 (Relevance Logic)

| 要素 | 内容 |
|------|------|
| **动机** | 经典实质条件 P→Q 不要求 P 和 Q 有任何内容关联——导致「悖论」 |
| **核心** | 有效的条件句必须保证前件和结论之间有**内容上的关联** |
| **被拒绝** | P → (Q → P) 和 ¬P → (P → Q) ——经典逻辑的「悖论式定理」 |

---

## 非经典逻辑谱系

```
经典逻辑 (Classical)
  ├─ 放弃排中律 ──→ 直觉主义逻辑 (Intuitionistic)
  ├─ 放弃二值 ────→ 多值逻辑 ──→ 模糊逻辑
  ├─ 放弃爆炸律 ──→ 弗协调逻辑 (Paraconsistent)
  ├─ 放弃单调性 ──→ 非单调逻辑 (Non-monotonic)
  └─ 加强条件句 ──→ 相干逻辑 (Relevance)
```
