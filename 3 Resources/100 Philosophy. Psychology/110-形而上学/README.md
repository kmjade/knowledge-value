# 110 形而上学 / Metaphysics

> 追问「什么是真实」——对存在的最底层探索。
> 亚里士多德称之为「第一哲学」(First Philosophy)，因为它研究的是「作为存在的存在」(being qua being)。

## 六大核心问题

### ① 本体论 (Ontology) — 存在什么？

| 问题 | 立场 A | 立场 B |
|------|--------|--------|
| 共相是否存在？ | **实在论**: 共相真实存在 (柏拉图) | **唯名论**: 只有殊相存在 (奥卡姆) |
| 什么是实体？ | **实体二元论**: 物质 + 心灵 | **物理主义**: 一切是物质 |
| 抽象对象？ | **柏拉图主义**: 数、命题独立存在 | **虚构主义**: 抽象对象只是有用的虚构 |

### ② 因果关系 (Causation) — 为什么事情发生？

| 理论 | 核心主张 | 代表人物 |
|------|----------|----------|
| **休谟规则性** | 因果只是恒常连结 (constant conjunction) | Hume |
| **反事实理论** | A 导致 B = 若无 A 则无 B | Lewis |
| **操纵理论** | 因果 = 可干预性 (intervention) | Woodward |
| **亚里士多德四因** | 质料因·形式因·动力因·目的因 | Aristotle |

### ③ 自由意志 (Free Will) — 我们能选择吗？

| 立场 | 核心主张 | 代表人物 |
|------|----------|----------|
| **自由意志论** | 人有真正的选择自由 | 笛卡尔、萨特 |
| **硬决定论** | 一切被因果决定，无自由 | 斯宾诺莎 |
| **相容论** | 自由与决定论可以共存 | 休谟、丹尼特 |
| **自由意志怀疑论** | 即使非决定，也不是"自由" | 斯特劳森 |

### ④ 时间 (Time) — 时间是真实的吗？

| 立场 | 核心主张 | 代表人物 |
|------|----------|----------|
| **A 理论** | 过去/现在/未来有真实差异；现在是特殊的 | 麦克塔格特 |
| **B 理论** | 时间只是事件的排序关系，"现在"只是索引词 | 罗素、爱因斯坦 |
| **永恒论** | 过去、现在、未来同样真实 (块宇宙) | — |
| **现在论** | 只有现在存在；过去已逝，未来未至 | — |

### ⑤ 可能世界 (Possible Worlds) — 事情可以不同吗？

| 立场 | 核心主张 | 代表人物 |
|------|----------|----------|
| **模态实在论** | 可能世界和我们世界一样真实 | David Lewis |
| **抽象实在论** | 可能世界是抽象的命题集合 | Plantinga |
| **模态虚构论** | 可能世界是方便的说法，不真实存在 | Rosen |

### ⑥ 个人同一性 (Personal Identity) — 什么使你成为你？

| 立场 | 核心主张 | 代表人物 |
|------|----------|----------|
| **心理连续性** | 记忆、性格、信念的连续性 = 同一性 | Locke, Parfit |
| **身体连续性** | 同一身体 = 同一人 | — |
| **灵魂/单纯自我** | 不可分割的灵魂实体 = 同一性 | 笛卡尔 |
| **虚无主义** | 「自我」是幻觉，没有持存的同一性 | 休谟、佛教 |

## 主要立场速查

```
                    真有共相？
                   /        \
                 是          否
                /              \
          柏拉图实在论        只存在殊相？
                            /         \
                          是           否
                         /               \
                   殊相 + 相似关系       殊相 + 抽象归因
                   (唯名论)              (概念论)

                    自由意志？
                   /    |    \
                  /     |     \
            自由论  相容论  决定论
           (意志    (自由   (一切
            为因)   在规律   被决定)
                    之内)
```

## 历史脉络

| 时期 | 核心人物 | 核心问题 |
|------|----------|----------|
| 古希腊 | 柏拉图、亚里士多德 | 理型 vs 实体、「作为存在的存在」 |
| 中世纪 | 阿奎那、奥卡姆 | 上帝存在论证、共相之争 |
| 理性主义 (17C) | 笛卡尔、斯宾诺莎、莱布尼茨 | 实体、心身、上帝 |
| 经验主义 (18C) | 洛克、贝克莱、休谟 | 实在的性质、因果与归纳 |
| **康德 (1781)** | 康德 | **哥白尼革命**：不是知识符合对象，而是对象符合知识 |
| 德国观念论 | 黑格尔 | 绝对精神、辩证法 |
| 分析哲学 (20C) | 罗素、维特根斯坦、奎因 | 语言→实在、本体论承诺 |
| **当代 (20C 末-)** | Kripke, Lewis, Dennett | 可能世界语义学、意识研究 |
