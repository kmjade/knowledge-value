# 07-现象学

> 「回到事物本身！」(Zu den Sachen selbst!) ——胡塞尔的口号。不要理论预设，不要形而上学体系——直接去**描述**意识中呈现的东西本身。现象学是 20 世纪欧陆哲学最强大的方法论。

## 胡塞尔 (Husserl, 1859-1938)

### 现象学还原

| 概念 | 内容 |
|------|------|
| **自然态度** | 日常中我们「自然地」相信世界存在——桌子在那、别人有心智 |
| **悬搁** (Epoché) | 把这个自然信念**放进括号**——不否定它——暂停它 |
| **还原到纯粹意识** | 剩下的不是空——是意识的流动本身——**纯粹现象** |

### 意向性

> 「一切意识都是关于某物的意识。」

| 要素 | 内容 |
|------|------|
| **意向行为** (Noesis) | 意识的动作：看、想、回忆、判断 |
| **意向对象** (Noema) | 意识指向的东西——不是物理事物——是在意识中呈现的那个「对象」 |

> 意向性打破了笛卡尔的「内在心灵 vs 外部世界」的二分。意识从来不是封闭的——它总是在「指向」某物。

### 生活世界 (Lebenswelt)

| 概念 | 内容 |
|------|------|
| 科学描述的世界不是我们**活着的世界** | 我们活在感知的、意义的、文化的生活世界中——科学是对它的抽象 |

## 海德格尔 (Heidegger, 1889-1976)

### 存在的遗忘

> 哲学两千年来在问「存在者是什么」——忘了问「存在本身是什么意思」。这是「存在的遗忘」。

| 概念 | 内容 |
|------|------|
| **存在者** (Seiendes) | 桌子、人、原子、上帝——具体的东西 |
| **存在** (Sein) | 存在者存在这个「事实」本身——为什么要是有而不是无？ |

### 此在 (Dasein)

> 「此在」——人的存在方式——它的存在就是去**理解**自己的存在。

| 特征 | 内容 |
|------|------|
| **在世存在** (In-der-Welt-sein) | 人不是「在一个世界里」——人是「在世界中」——世界是我们关切和操劳的场域 |
| **被抛** (Geworfenheit) | 我们被「扔」进了一个我们没有选择的存在——已经在这里了 |
| **向死而生** (Sein-zum-Tode) | 死亡是我的「最本己的可能性」——意识到死亡让生命从日常消散中收回自己 |
| **本真/非本真** | 非本真 = 随大流 (das Man) ——做「大家」做的事；本真 = 面对死亡，决心成为自己 |

### 后期海德格尔

| 转向 (Kehre) | 从「此在」转向「存在本身」——语言是存在之家——技术是「座架」(Gestell) ——把万物变成可计算的资源 |

---

## 梅洛-庞蒂 (Merleau-Ponty, 1908-1961)

### 身体主体

| 概念 | 内容 |
|------|------|
| 不是「我有一个身体」——是「我就是我的身体」 | 意识不是在大脑中——意识是**身体化的** (embodied) |
| **肉身** (La chair) | 世界和身体是同一种「肉身」——打破了主体/客体的二分 |

---

## 现象学的影响

| 领域 | 影响 |
|------|------|
| 存在主义 | 萨特——存在先于本质——来自海德格尔 |
| 心理学 | 格式塔、身体化认知 |
| 社会学 | 舒茨——生活世界 |
| AI/认知科学 | 身体化认知——Dreyfus 对 AI 的现象学批判 |
