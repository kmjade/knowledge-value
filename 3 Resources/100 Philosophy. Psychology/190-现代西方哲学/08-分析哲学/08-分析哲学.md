# 08-分析哲学

> 如果说欧陆哲学关心「存在」「焦虑」「身体」，分析哲学只关心一件事：**论证的清晰性**。它的核心信念是：许多哲学问题其实是语言的误用——澄清语言，问题就消失了。

## 语言转向

| 概念 | 内容 |
|------|------|
| **语言转向** | 哲学从追问「世界是什么」「心灵是什么」——转向追问「我们在说什么？」 |
| **逻辑作为工具** | 弗雷格发明现代逻辑——不是为了数学——是为了哲学——精确陈述思想 |

---

## 弗雷格 (Frege, 1848-1925)

| 贡献 | 内容 |
|------|------|
| 概念文字 | 第一个形式逻辑系统——谓词逻辑 |
| 意义与指称 | 「晨星」和「昏星」——同一指称，不同意义——开创语言哲学 |
| 语境原则 | 不要孤立问词的意思——词在命题中才有意义 |

---

## 罗素 (Russell, 1872-1970)

### 摹状词理论

> 「当今法国国王是秃头。」——没有国王——这个命题是假还是无意义？

罗素的分析：把它翻译成谓词逻辑——
∃x (KingOfFrance(x) ∧ ∀y(KoF(y)→y=x) ∧ Bald(x)) ——假。不是无意义。

| 意义 | 这个分析消除了「不存在的东西怎么能是命题的主语」这个困惑 |

---

## 早期维特根斯坦：《逻辑哲学论》

> 「世界是一切实际如此的情况。」「对于那些不可说的，必须保持沉默。」

| 命题 | 内容 |
|------|------|
| 语言是现实的图像 | 命题的结构映射事实的结构 |
| 真值表 | 发明真值表——逻辑 = 重言式 |
| 可说的 vs 不可说的 | 科学命题可说——伦理、美学、人生意义——不可说——只能显示 |
| 「我的语言的界限意味着我的世界的界限」| 5.6 |

> 维特根斯坦认为《逻辑哲学论》解决了所有哲学问题——然后去做了园丁。

---

## 逻辑实证主义 (维也纳学派)

| 要素 | 内容 |
|------|------|
| **口号** | 「一个命题的意义就是它的证实方法」 |
| **有意义的命题** | 要么是分析的（逻辑/数学），要么是可经验证实的（科学） |
| **形而上学** | 无意义——不是假——是无法判断——伪命题 |
| **问题** | 可证实性原则本身既非分析也非经验可验证——**自毁** |

---

## 后期维特根斯坦：《哲学研究》

> 语言的意义不在「图画事实」——在**使用**中。

| 概念 | 内容 |
|------|------|
| **语言游戏** | 语言像下棋——每种用法有不同的规则。承诺、命令、演戏——不是同一个「语言游戏」 |
| **家族相似** | 「游戏」没有共同本质——只有一张重叠相似的网络——「家族相似」 |
| **哲学作为治疗** | 哲学问题 = 语言的痉挛——哲学的任务是让语言「回到粗糙的地面」 |

---

## 蒯因 (Quine, 1908-2000)

| 概念 | 内容 |
|------|------|
| **经验论的两个教条** | 1) 分析/综合的区分是假的 2) 意义可以还原为经验——也是假的 |
| **整体论** | 不是单个命题面对经验——是整个信念之网——科学 + 逻辑 + 日常信念——一起面对经验 |
| **自然化认识论** | 认识论不是哲学的前置——是心理学的一章 |

---

## 分析哲学 vs 欧陆哲学

| 维度 | 分析哲学 | 欧陆哲学 |
|------|---------|---------|
| 方法 | 论证、逻辑 | 历史、诠释、体验 |
| 核心 | 语言、逻辑 | 存在、意识、权力 |
| 风格 | 清晰、技术性 | 晦涩、文学性 |
| 代表 | Frege, Russell, Quine | Hegel, Husserl, Derrida |
