# 现代西方哲学 知识库总览 (DDC 190)

> 「现代」哲学从笛卡尔开始——不再问「世界是什么」(那是古代的问题)，而是问「我怎么能确定我知道世界是什么」。这是认识论的转向——主体成为哲学的起点。

## 现代哲学的两条道路

```
笛卡尔 (1641) —「我思故我在」
        │
        ├──→ 理性主义 (Spinoza, Leibniz)     ┐
        │    天赋观念 · 演绎 · 几何学方法     │
        │                                    ├──→ 康德综合 (1781)
        └──→ 经验主义 (Locke, Berkeley, Hume) │    先验综合判断
             白板 · 归纳 · 怀疑论极限         │
                                             │
                    ┌────────────────────────┘
                    │
         ┌──────────┼──────────┐
         ↓          ↓          ↓
     黑格尔      叛逆者     分析vs欧陆
     绝对精神    尼采等     20世纪分化
```

## 核心张力

| 张力 | 正题 | 反题 |
|------|------|------|
| 知识来源 | 理性主义——天赋观念 | 经验主义——白板 |
| 物自体 | 可知 (Hegel) | 不可知 (Kant) |
| 哲学方法 | 分析——逻辑·语言 | 欧陆——历史·体验 |
| 启蒙理性 | 肯定 (Kant, Hegel) | 质疑 (Nietzsche, Adorno) |
