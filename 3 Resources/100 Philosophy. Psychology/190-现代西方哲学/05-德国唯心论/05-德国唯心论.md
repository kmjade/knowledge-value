# 05-德国唯心论

> 康德说物自体不可知。费希特说：「那就取消它！」——德国唯心论的主题：取消物自体，一切都是精神（Geist）的自我展开。在黑格尔那里，整个宇宙是绝对精神的自我认识过程。

## 从康德到黑格尔

| 人物 | 对康德的修正 |
|------|------------|
| **康德** | 物自体不可知——认知有边界 |
| **费希特** (Fichte, 1762-1814) | 物自体不存在——「自我」设定「非我」——一切从绝对自我出发 |
| **谢林** (Schelling, 1775-1854) | 自然和精神是同一绝对者的两面——自然哲学补充了费希特的主观唯心论 |
| **黑格尔** (Hegel, 1770-1831) | 绝对者不是静态的——它是历史过程——通过辩证法自我展开 |

---

## 黑格尔：绝对精神的自我实现

### 核心命题

> 「凡是现实的都是合理的，凡是合理的都是现实的。」

| 概念 | 内容 |
|------|------|
| **绝对精神** (Geist) | 宇宙的终极实在——不是静止的实体——是自我展开的**过程** |
| **目标** | 精神的自我认识——通过历史——最终在哲学中达到绝对知识 |

### 辩证法

```
正题 (Thesis) → 反题 (Antithesis) → 合题 (Synthesis)
     ↓                                    ↓
  存在本身                            (又被否定)
     │                                    │
     └──────── 新一轮正反合 ←───────────────┘
```

| 阶段 | 内容 |
|------|------|
| **正题** | 一个立场 / 概念 |
| **反题** | 它的否定——不是简单的反对——是内在矛盾的展开 |
| **合题** | 扬弃 (Aufhebung) ——既否定又保留——两个环节的更高统一 |

> 「Aufhebung」——黑格尔最关键的词：否定、保留、提升——三合一。

### 主奴辩证法

> 两个自我意识相遇——为了承认而斗争。一方愿意冒死——成为主人。另一方怕死——成为奴隶。但奴隶通过劳动**改造了世界**——从而获得独立意识——主人却依赖奴隶的承认。

| 阶段 | 内容 |
|------|------|
| 斗争 | 两个自我意识相遇——都要被对方承认 |
| 主奴产生 | 不怕死的 → 主人；怕死的 → 奴隶 |
| 反转 | 奴隶劳动→改造世界→获得自我；主人依赖奴隶→失去独立性 |

> 这是黑格尔最被反复诠释的段落——马克思主义、存在主义、后殖民理论都从这里汲取资源。

## 《精神现象学》：意识的旅程

| 阶段 | 意识形式 | 例子 |
|------|---------|------|
| 意识 | 感性确定性→知觉→知性 | 「这个」东西是什么 |
| 自我意识 | 欲望→主奴→斯多葛→苦恼意识 | 人与人的承认斗争 |
| 理性 | 观察的理性→实践的理性 | 科学、伦理 |
| **精神** | 伦理→教化→道德→宗教→绝对知 | 文化、历史、艺术、宗教、哲学 |

> 终点：「绝对知」——精神完全认识了自己。一切外化都回归自身。

## 黑格尔的遗产

| 继承者 | 继承了什么 | 反对了什么 |
|--------|-----------|-----------|
| **马克思** | 辩证法、历史哲学 | 唯心论——把「精神」换成「物质」 |
| **克尔凯郭尔** | — | 体系——「存在不能被体系化」 |
| **尼采** | — | 理性——「理性是权力意志的工具」 |
| **20世纪法国哲学** | 主奴辩证法、欲望 | 黑格尔的封闭体系 |
