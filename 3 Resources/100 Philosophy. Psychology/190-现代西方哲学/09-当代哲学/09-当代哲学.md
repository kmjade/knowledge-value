# 09-当代哲学

> 20 世纪后半叶：存在主义说「存在先于本质」，后现代主义说「没有本质——只有文本」，政治哲学复兴——罗尔斯追问「什么是正义」。哲学不再是哲学家的自留地——它进入了公共辩论。

## 存在主义：萨特 (Sartre, 1905-1980)

### 「存在先于本质」

| 概念 | 内容 |
|------|------|
| **核心** | 人没有先天的「本质」 (人性) ——先存在——然后通过选择和行动定义自己 |
| **自由** | 人是「被判为自由的」——不能逃避选择——「不选择」也是一种选择 |
| **自欺** (Bad Faith) | 假装自己没有自由——「我是服务员所以我必须这样」——你在自欺 |
| **他人即地狱** | 在他人的注视下——你被「客体化」了——失去了主体性 |

## 后现代主义

### 福柯 (Foucault, 1926-1984)

| 概念 | 内容 |
|------|------|
| **知识 = 权力** | 知识不是中立的——它产生权力——同时又是由权力结构塑造的 |
| **真理体制** | 每个时代有特定的「真理生产机制」——什么是可以说的——什么是被排斥的 |
| **规训** | 现代社会通过**让人自我监控**来运作——全景敞视——你永远不知道是否被看——于是你总是「守规矩」 |
| **主体之死** | 「人」是现代知识型的发明——像沙滩上的一张脸——会被海浪抹去 |

### 德里达 (Derrida, 1930-2004)

| 概念 | 内容 |
|------|------|
| **解构** | 不是「拆毁」——是揭示文本如何自己**瓦解自己的前提** |
| **逻各斯中心主义** | 西方哲学一直相信有一个「终极所指」(上帝、理性、存在)——好能稳定整个意义系统 |
| **延异** (Différance) | 意义永远在**区别和延迟**中——永远不会「完全在场」 |
| **文本之外别无他物** | 不是「世界不存在」——是世界总是已经被「文本化」——被语言和概念中介 |

### 利奥塔 (Lyotard, 1924-1998)

> 「我把『后现代』定义为对元叙事的怀疑。」——「理性在进步」「人类在解放」——这些宏大叙事不再可信。

## 政治哲学：罗尔斯 (Rawls, 1921-2002)

### 《正义论》(1971)

| 概念 | 内容 |
|------|------|
| **无知之幕** | 如果你不知道你在社会中是什么身份——你会设计什么样的正义原则？ |
| **两条原则** | 1) 平等的自由；2) 不平等只有在对最不利者有利时才被允许 (差别原则) |
| **功绩** | 复兴了政治哲学——1945-1971 政治哲学基本停摆 |
