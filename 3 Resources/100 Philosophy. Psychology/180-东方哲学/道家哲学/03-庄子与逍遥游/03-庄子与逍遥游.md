# 03-庄子与逍遥游

> 「昔者庄周梦为蝴蝶，栩栩然蝴蝶也……不知周之梦为蝴蝶与？蝴蝶之梦为周与？」——《庄子·齐物论》。如果说老子是道家的智慧，庄子就是道家的诗意。

## 庄子其人

| 传说 | 内容 |
|------|------|
| **姓名** | 庄周 |
| **时代** | 约公元前 369-286——战国中期 |
| **生平** | 蒙（今河南商丘）漆园吏——一个小官。楚王请他做宰相——庄子说「我宁愿在泥里打滚的活乌龟」 |
| **著作** | 《庄子》33 篇——内篇 7 (庄子自著)、外篇 15、杂篇 11 |
| **风格** | 寓言、重言、卮言——用故事说哲学，而不是用论证 |

---

## 《庄子》的结构

| 部分 | 篇章 | 主题 |
|------|------|------|
| **内篇** (7) | 逍遥游、齐物论、养生主、人间世、德充符、大宗师、应帝王 | 核心——庄子本人的思想 |
| **外篇** (15) | 骈拇、马蹄、天地、秋水…… | 内篇的发挥——可能出自后学 |
| **杂篇** (11) | 庚桑楚、徐无鬼、则阳、天下…… | 杂录——包含对诸子的评论 |

## 核心篇章

### 逍遥游

> 大鹏展翅九万里——蝉和斑鸠嘲笑它飞得太高。

| 主题 | 内容 |
|------|------|
| **大小之辨** | 大鹏和蜩鸠各有各的「逍遥」——真正的逍遥超越了大小的比较 |
| **无待** | 「若夫乘天地之正，而御六气之辩，以游无穷者，彼且恶乎待哉」——不依赖任何东西 |
| **无用之大用** | 樗树（臭椿）因为无用而长得最大——「无用」反而保全了生命 |

### 齐物论

> 「天地与我并生，而万物与我为一。」

| 主题 | 内容 |
|------|------|
| **齐物** | 万物在「道」的层面是平等的——高低、是非、美丑都是人的分别心 |
| **天籁** | 地籁（风吹万物的声音）、人籁（人吹乐器）、天籁（万物自己发出的声音）——天籁是最高境界 |
| **蝴蝶梦** | 庄子梦见自己是蝴蝶——醒来不知道自己是庄子梦见蝴蝶，还是蝴蝶梦见庄子——「物化」 |
| **语言的局限** | 「大辩不言」——真正的道理不需要争辩。语言只能指向道，不能捕捉道 |

### 养生主

> 「吾生也有涯，而知也无涯。以有涯随无涯，殆已。」

| 主题 | 内容 |
|------|------|
| **庖丁解牛** | 庖丁解牛「以神遇而不以目视」——刀刃十九年如新——顺其自然则无往不利 |
| **缘督以为经** | 顺着中道——不偏不倚 |

### 人间世

| 主题 | 内容 |
|------|------|
| **心斋** | 不是不吃肉——是「虚其心」——清空心智才能听到道 |
| **无用之用** | 在乱世中——「无用」是保身之道 |

## 庄子与老子的对比

| 维度 | 老子 | 庄子 |
|------|------|------|
| **文体** | 箴言——短小、凝练 | 寓言——故事、幽默、想象力 |
| **态度** | 治世——「无为而治」 | 出世——「逍遥游」——不关心政治 |
| **「道」的取向** | 道生万物——宇宙论 | 道在蝼蚁、在屎溺——万物皆道 |
| **语言观** | 道不可言说 | 语言是「荃」——得到鱼后忘掉工具 |
| **生死观** | 长生久视 | 生死一如——安时而处顺 |
| **情感基调** | 冷静、深邃 | 幽默、荒诞、诗意 |

---

## 为什么庄子重要？

| 维度 | 说明 |
|------|------|
| **哲学** | 「齐物」打破二元对立——比德里达早了两千年 |
| **文学** | 中国最美的哲学散文——李白、苏轼的精神祖先 |
| **美学** | 「天地有大美而不言」——中国审美精神的源头之一 |
| **现代性** | 庄子对语言、真实、自我边界的追问——与现代后结构主义深度共振 |
