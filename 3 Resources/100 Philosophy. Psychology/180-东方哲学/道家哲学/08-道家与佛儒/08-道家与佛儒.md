# 08-道家与佛儒

> 「三教合一」是中国思想史上的核心母题——儒家提供伦理秩序，道家提供自然超越，佛家提供心性解脱。三者两千年来互相碰撞、吸收、融合。

## 三教对照

| 维度 | 儒家 | 道家 | 佛家 |
|------|------|------|------|
| **终极关怀** | 建立人间秩序 | 回归自然/道 | 解脱生死/涅槃 |
| **路径** | 修身齐家治国平天下 | 无为、自然、虚静 | 戒定慧、八正道 |
| **理想人格** | 圣人 | 真人、至人 | 佛、菩萨 |
| **对语言的态度** | 正名——语言必须准确 | 道不可言——语言指向道但非道 | 不立文字——但留下了最多经典 |
| **社会态度** | 入世——改造世界 | 出世/避世——顺其自然 | 出世——但大乘「度众生」是入世 |

## 魏晋玄学：道与儒的第一次融合

| 要素 | 内容 |
|------|------|
| **时代** | 魏晋 (3-6世纪) |
| **核心问题** | 「名教」与「自然」的关系——儒家礼教 vs 道家自然而然 |
| **王弼** | 「以无为本」——用老子注释《易经》和《论语》——道家的「无」作为儒家的基础 |
| **郭象** | 「独化论」——每个事物「自己如此」——不需要「造物主」——庄子注 |

### 竹林七贤

| 人物 | 态度 |
|------|------|
| 嵇康 | 「越名教而任自然」——激进的个体自由——被司马氏处死 |
| 阮籍 | 醉酒避世——「礼岂为我辈设也？」 |
| 向秀 | 调和——「名教即自然」——儒道不是对立的 |

## 禅宗与道家的共振

佛教传入中国后，禅宗是「最道家化」的佛教宗派：

| 禅宗特点 | 道家根源 |
|---------|---------|
| **不立文字** | 老子「道可道非常道」——真理不在文字中 |
| **顿悟** | 庄子「朝彻」(一朝豁然贯通) |
| **平常心是道** | 庄子「道在屎溺」——道不在特殊之处，在平常日用中 |
| **无念、无住** | 庄子「心斋」「坐忘」 |
| **自然** | 马祖道一「饥来吃饭，困来即眠」——与道家「自然」高度一致 |

### 格义

> 佛教初传中国时，用道家的词汇翻译佛教概念——「道」译 Dharma，「无为」译 Nirvana——这是「格义」。它帮助佛教进入中国——但也导致了概念的混淆。

## 宋明理学：儒家「吸道」

| 概念 | 道家源头 | 理学改造 |
|------|---------|---------|
| **无极而太极** | 老子「无极」——最原初的未分化状态 | 周敦颐《太极图说》——儒家宇宙论 |
| **理** | 道家「道」的理性化 | 程朱理学——「理」= 宇宙的秩序原理 |
| **静坐** | 道家「虚静」 | 朱熹教人「半日静坐，半日读书」——但目的不同（不是修道，是修身） |
| **万物一体** | 庄子「天地与我并生」 | 王阳明「万物一体之仁」——儒家的道德版本 |

## 全真道的三教合一

> 王重阳：「儒门释户道相通，三教从来一祖风。」

全真道以道家内丹为基础，吸收：

| 来源 | 吸收了什么 |
|------|-----------|
| 儒家 | 忠孝——出家也要忠君孝亲 |
| 佛家 | 出家制度、素食、戒律、心性修炼 |
| 道家 | 内丹——炼精化气、炼气化神 |
