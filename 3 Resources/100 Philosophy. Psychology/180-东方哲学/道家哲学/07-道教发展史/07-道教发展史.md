# 07-道教发展史

> 从老庄哲学到天师道，从外丹到内丹，从山林隐修到制度化宗教——道家的宗教化是中国思想史上最复杂的演变之一。

## 从哲学到宗教：关键转折

| 阶段 | 时代 | 事件 |
|------|------|------|
| **哲学源头** | 春秋战国 | 老子《道德经》、庄子——原始道家哲学 |
| **黄老之学** | 战国末—汉初 | 道家 + 法家 = 为政之术 |
| **老子神化** | 汉代 | 老子从「智者」变为「神」——太上老君 |
| **制度创立** | 东汉 (2世纪) | 张道陵创天师道——道教作为宗教正式诞生 |

---

## 早期道教：天师道

### 张道陵与五斗米道

| 要素 | 内容 |
|------|------|
| **创立** | 张道陵 (张陵) ——东汉顺帝年间 (142 年)，四川鹤鸣山 |
| **核心** | 太上老君 (神化老子) 授予张道陵「天师」之位——「正一盟威之道」 |
| **经典** | 《老子想尔注》——将《道德经》宗教化 |
| **组织** | 二十四治 (教区) ——信徒交五斗米——又称「五斗米道」 |
| **特点** | 治病 (符水)、忏悔、道德教化——中国第一个有组织的道教团体 |

### 张鲁的汉中政权

| 要素 | 内容 |
|------|------|
| 张鲁 (张道陵之孙) 在汉中建立了 30 年的「道治」政权 |
| 免费提供食物、住处——「义舍」——中国历史上唯一的宗教乌托邦实践 |
| 215 年投降曹操——天师道北传 |

---

## 中古道教：上清派

| 要素 | 内容 |
|------|------|
| **时代** | 东晋 (4世纪) |
| **创教者** | 杨羲——声称受魏华存 (南岳夫人) 降授《上清经》 |
| **核心修炼** | 存思——冥想体内神祇和宇宙星宿——不是炼丹服药，是精神修炼 |
| **与天师道区别** | 天师道：集体、仪式、治病；上清派：个人、冥想、精英 |

---

## 灵宝派与科仪

| 要素 | 内容 |
|------|------|
| **时代** | 东晋末—南北朝 |
| **经典** | 《灵宝经》——吸收了佛教的大量元素（轮回、地狱、功德） |
| **贡献** | 系统化道教仪式 (科仪) ——斋醮——影响了后世所有道教仪式 |
| **陆修静** (406-477) | 整理《三洞经书》——道藏的基础 |

---

## 唐：道教的黄金时代

| 要素 | 内容 |
|------|------|
| **国教地位** | 唐朝皇室姓李——认老子 (李耳) 为祖先——道教成为国教 |
| **经典编纂** | 唐玄宗编撰《开元道藏》——最早的道藏 |
| **外丹的辉煌与衰落** | 唐代多位皇帝死于服丹药——外丹术开始衰落 |
| **内丹兴起** | 外丹失败→「身体内部炼丹」——内丹术成为主流 |

---

## 宋金元：全真道的兴起

### 王重阳与全真道

| 要素 | 内容 |
|------|------|
| **创立** | 王重阳 (1112-1170) ——金代陕西 |
| **核心** | **三教合一**——「儒门释户道相通，三教从来一祖风」 |
| **修炼** | 内丹为主——「先性后命」——先修心性，后修命功 |
| **戒律** | 出家、素食、独身——深受佛教影响 |

### 全真七子

| 弟子 | 影响 |
|------|------|
| **马钰** | 全真第二代掌教——「清净派」 |
| **丘处机** | 远赴西域见成吉思汗——「一言止杀」——全真道最辉煌的时刻 |
| **孙不二** | 女性修炼——全真道的女性传统 |

---

## 正一道与南宗

| 派别 | 特点 |
|------|------|
| **正一道** | 天师道的延续——江西龙虎山——张天师世袭——符箓、斋醮 |
| **南宗 (金丹派)** | 张伯端 (紫阳真人) ——「先命后性」——与全真「先性后命」相反 |

> 元代以后，道教分为两大主流：**全真道** (内丹、出家) 和**正一道** (符箓、在家)。

## 道藏的编纂

| 时期 | 内容 |
|------|------|
| 唐 | 《开元道藏》——最早的道藏 |
| 宋 | 《大宋天宫宝藏》——张君房编 |
| 明 | 《正统道藏》(1445) + 《万历续道藏》(1607) ——现存版本——5485 卷 |

## 现代道教

| 地区 | 现状 |
|------|------|
| **中国大陆** | 文革重创——80 年代恢复——白云观、武当山、青城山——信众回升 |
| **台湾** | 道教 + 民间信仰——非常活跃。行天宫、指南宫 |
| **香港** | 黄大仙祠——香火鼎盛 |
| **海外** | 太极拳、风水、中医带动了道家的全球传播 |
