# 05-道家与养生

> 「长生久视」——道家不只是一种哲学，它是一套完整的身体观和实践体系。从老子的「专气致柔」到内丹的「炼精化气」，道家追求的是身心灵一体的长生。

## 道家身体观

| 概念 | 内容 |
|------|------|
| **形-气-神** | 身体 = 形 (肉体) + 气 (生命能量) + 神 (意识/精神) ——三者一体 |
| **小宇宙** | 人体是一个小宇宙——与天地大同构 |
| **贵生** | 生命是珍贵的——「名与身孰亲？身与货孰多？」(老子 Ch.44) |

---

## 气的概念

| 维度 | 说明 |
|------|------|
| **字源** | 气 = 云气——流动的能量 |
| **道家义** | 气是万物的生命力——道通过气化生万物 |
| **人体中的气** | 气在经络中运行——「气贯丹田」 |
| **修炼目标** | 使气在体内通畅——「专气致柔，能婴儿乎？」(老子 Ch.10) |

---

## 内丹术 (Neidan)

> 「炼丹」——外丹 (实验室炼丹) 后来被内丹 (身体内部炼丹) 取代。人体是丹炉——精、气、神是原料。

### 内丹三阶段

| 阶段 | 内容 | 目标 |
|------|------|------|
| **炼精化气** | 将身体精华 (精) 转化为气 | 百日筑基——经络通畅 |
| **炼气化神** | 将气转化为神 | 十月养胎——丹田成「丹」 |
| **炼神还虚** | 将神回归虚无——与道合一 | 九年面壁——成仙 |

### 丹田

| 丹田 | 位置 | 功能 |
|------|------|------|
| **下丹田** | 脐下三寸 | 储能——炼精化气 |
| **中丹田** | 膻中 (胸口) | 炼气化神 |
| **上丹田** | 印堂 (眉心) | 炼神还虚 |

## 气功 (Qigong)

> 「气功」是现代名称——古代叫「导引」「行气」「吐纳」。

| 类型 | 说明 | 代表 |
|------|------|------|
| **医疗气功** | 治病健身 | 八段锦、五禽戏 |
| **武术气功** | 增强武术功力 | 硬气功、铁布衫 |
| **道家气功** | 内丹基础——修炼成仙 | 小周天、大周天 |

### 小周天

> 气沿任督二脉循环——「筑基」的核心功法。

## 太极拳 (Taijiquan)

| 要素 | 内容 |
|------|------|
| **起源** | 传为张三丰 (元明之际) 创立——与武当道教关系密切 |
| **核心原则** | 以柔克刚、以静制动、四两拨千斤 |
| **道家哲学体现** | 阴阳相济、顺其自然、意到气到 |
| **现代发展与争议** | 实战性争议——但作为养生运动被世界接受 |

---

## 辟谷 (Bigu)

| 要素 | 内容 |
|------|------|
| **含义** | 「辟」= 避开，「谷」= 五谷——断食或限食修炼 |
| **目的** | 清净身体、减少浊气——让「气」取代食物作为能量 |
| **现代视角** | 间歇性断食 (Intermittent Fasting) 在一定程度上与之呼应 |

---

## 房中术与养生

| 要素 | 内容 |
|------|------|
| **核心** | 节欲保精——「精」是炼内丹的原料，不可耗散 |
| **原则** | 「还精补脑」——将性能量升华为精神能量 |
| **争议** | 被一些人发展为纵欲——背离道家「节欲」本义 |

---

## 道家养生的现代意义

| 贡献 | 说明 |
|------|------|
| **整体观** | 身心灵一体——比现代医学的分离视角超前 |
| **预防医学** | 「上医治未病」——道家的预防优先思维 |
| **慢性病管理** | 太极拳、气功对高血压、关节炎、焦虑的实证效果 |
| **正念 (Mindfulness)** | 「虚静」与当代正念冥想的呼应 |
