# 03-上座部佛教

> 「小乘」是大乘给的贬称——他们自称为「上座部」(Theravada)，意思是「长老们的教导」。这是现存最接近原始佛教的传承。

## 为什么叫「上座部」？

| 事实 | 内容 |
|------|------|
| **起源** | 佛陀灭度后——僧团分裂为大众部 (Mahasanghika) 和上座部 (Sthavira) |
| **上座部** | 保守派——坚持长老们的传统——严格持戒、重视禅修 |
| **南传路线** | 斯里兰卡 → 缅甸 → 泰国 → 老挝 → 柬埔寨 |
| **经典** | **巴利三藏**——唯一保存完整的早期佛典——用巴利语写成 |

---

## 巴利三藏

| 藏 | 内容 | 著名经典 |
|----|------|---------|
| **经藏** (Sutta Pitaka) | 佛陀的教诫——五部尼柯耶 | 《长部》《中部》《杂部》《增支部》《小部》 |
| **律藏** (Vinaya Pitaka) | 僧团戒律 | 227 条比丘戒 |
| **论藏** (Abhidhamma Pitaka) | 对教义的系统哲学分析 | 心所 (cetasika) 分类——最精细的意识分析 |

> 《法句经》(Dhammapada) ——最受欢迎的佛教经典之一——在《小部》中。

---

## 修行目标：阿罗汉

| 概念 | 内容 |
|------|------|
| **阿罗汉** (Arahant) | 「应供」——断尽烦恼的人——此生是最后一生——死后不再轮回 |
| **四果** | 须陀洹 (入流) → 斯陀含 (一来) → 阿那含 (不还) → 阿罗汉 |
| **与菩萨的区别** | 阿罗汉追求个人解脱；菩萨推迟自己解脱以度化众生——这是大小乘的根本分歧 |

---

## 内观禅修 (Vipassana)

| 概念 | 内容 |
|------|------|
| **核心** | 如实观察身心现象——看到它们的无常、苦、无我 |
| **方法** | 观呼吸 → 观身体感受 → 观心念生灭 |
| **与止禅的区别** | 止 (Samatha) = 专注——让心安静；观 (Vipassana) = 洞察——看到实相 |
| **当代传播** | S.N. Goenka、Mahasi Sayadaw——内观走向全球 |

### 四念处

| 念处 | 观察对象 |
|------|---------|
| 身念处 | 呼吸、姿势、身体部分 |
| 受念处 | 苦受、乐受、不苦不乐受 |
| 心念处 | 贪心、嗔心、散乱心…… |
| 法念处 | 五盖、五蕴、七觉支 |

## 上座部的现代特征

| 特征 | 说明 |
|------|------|
| **保守** | 2000 年变化最小——最接近原始形态 |
| **学术化** | 巴利圣典学会 (PTS, 1881) ——西方学者翻译巴利藏 |
| **正念运动** | 内观从寺院走向世俗——MBSR (正念减压) 的源头 |
