# 08-佛教中国化

> 佛教从印度来到中国——遇到了儒家和道家。一千年后，它已经不是「印度的佛教」——是天台、华严、禅宗——是「中国的佛教」。

## 中国化的三个阶段

### 1. 格义时期 (汉—魏晋)

| 方法 | 用道家词汇翻译佛教概念——「道」译 Dharma，「无为」译 Nirvana |
|------|------|
| **问题** | 概念混淆——但帮助了佛教进入中国 |

### 2. 独立时期 (南北朝—隋唐)

| 事件 | 内容 |
|------|------|
| **鸠摩罗什** (344-413) | 最伟大的佛经翻译家——《金刚经》《法华经》《中论》 |
| **玄奘** (602-664) | 西行取经 17 年——《西游记》原型——翻译 1335 卷 |
| **中国宗派兴起** | 天台、华严、禅宗、净土——不再是印度宗派的分支 |

### 3. 融合时期 (宋—明清)

> 三教合一——儒释道不再是三个——是一个中国人的三种精神面向。

## 天台宗

| 要素 | 内容 |
|------|------|
| **创立** | 智顗 (538-597) ——天台山 |
| **核心** | 「一念三千」——一念心中具足三千世界 |
| **判教** | 「五时八教」——把佛陀一生的教法按时间层次排列——《法华经》最高 |

---

## 华严宗

| 要素 | 内容 |
|------|------|
| **核心** | 「一即一切，一切即一」——理事无碍、事事无碍 |
| **帝网珠** | 因陀罗网上的宝珠——每一颗都映现所有其他珠子的影像——互入互即 |
| **经典** | 《华严经》——最庞大的大乘经典 |

---

## 佛教与中国社会的冲突与适应

| 冲突 | 适应 |
|------|------|
| **不拜君王**——僧人不拜皇帝 | 后来——「沙门敬王者」——部分妥协 |
| **出家 vs 孝道**——不娶不嗣——不孝之大 | 中国佛教强调「出家是更大的孝」——度七世父母 |
| **乞食 vs 劳动**——印度僧侣乞食 | 百丈怀海——「一日不作，一日不食」——农禅 |

---

## 居士佛教

> 中国佛教不只是僧侣的——大量在家修行者形成了「居士佛教」。

| 人物 | 内容 |
|------|------|
| **维摩诘** | 《维摩诘经》——居士的身份——智慧超过所有菩萨 |
| **苏东坡** | 大文豪——禅宗居士——「溪声尽是广长舌」 |
| **杨仁山** (1837-1911) | 近代居士佛教先驱——金陵刻经处 |
