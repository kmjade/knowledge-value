# 07-藏传佛教

> 藏传佛教 = 印度晚期大乘 + 密宗 + 西藏本土苯教元素。它不是「佛教的西藏变体」——它是大乘佛教发展最晚的一个阶段——金刚乘 (Vajrayana)。

## 藏传佛教的特征

| 特征 | 内容 |
|------|------|
| **显密双修** | 显教 (大乘经论——中观、唯识) + 密教 (灌顶、本尊、曼荼罗) |
| **上师** | Guru——比佛还重要——「没有上师，连佛的名字都听不到」——对上师的绝对依止 |
| **即身成佛** | 密宗认为不需要无量劫——这一生就可以成佛 |
| **丰富的象征体系** | 曼荼罗、咒语、手印、法器——多用象征而非文字 |

## 四大宗派

| 宗派 | 创立 | 特征 | 最高领袖 |
|------|------|------|---------|
| **宁玛** (红教) | 8世纪·莲花生 | 最古老——大圆满法 (Dzogchen) | — |
| **噶举** (白教) | 11世纪·马尔巴 | 那洛六法·大手印——米拉日巴 | 噶玛巴 (大宝法王) |
| **萨迦** (花教) | 11世纪 | 道果法——显密并重 | 萨迦法王 |
| **格鲁** (黄教) | 14世纪·宗喀巴 | 戒律最严——显教先、密教后 | 达赖喇嘛·班禅 |

## 宗喀巴与格鲁派

### 宗喀巴 (1357-1419)

| 贡献 | 内容 |
|------|------|
| **整顿戒律** | 当时西藏僧侣戒律松弛——宗喀巴恢复严格戒律——黄帽是标志 |
| **显密次第** | 《菩提道次第广论》——先学显教 (出离心、菩提心、空正见) ——再入密教 |
| **三大寺** | 甘丹、哲蚌、色拉——格鲁派学术中心 |
| **学位制度** | 格西 (Geshe) ——相当于佛学博士——通过**辩经**考试 |

### 活佛转世

| 要素 | 内容 |
|------|------|
| **起源** | 噶玛噶举最早 (12世纪) ——格鲁派后来采纳 |
| **原理** | 高僧圆寂后——悲愿力投生再来——寻访转世灵童 |
| **达赖喇嘛** | 格鲁派最高活佛——观世音菩萨化身——现任第十四世 (丹增嘉措) |
| **班禅额尔德尼** | 第二高——阿弥陀佛化身 |

---

## 辩经

> 藏传佛教学僧每天在辩经场——站立者拍手发问——坐着的人必须回答。

| 要素 | 内容 |
|------|------|
| **目的** | 不是「赢」——是破除执著——用逻辑检验对空性的理解 |
| **形式** | 「如果是 A，应该是 B——但经典说不是 B——为什么？」——层层追问 |
| **哲学基础** | 以月称的《入中论》为最高中观教材——中观应成派 |

---

## 藏传佛学对死亡的关注

| 概念 | 内容 |
|------|------|
| **中阴** (Bardo) | 死亡和再生之间的中间状态——《中阴闻教得度》(西藏度亡经) |
| **颇瓦法** | 往生法——将神识迁往净土 |
| **虹光身** | 大圆满的最高成就——圆寂时肉体化为虹光——只留下指甲和头发 |
