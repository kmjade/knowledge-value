# 01-佛学概述

> 「诸行无常，诸法无我，涅槃寂静。」——三法印是佛教的「DNA检测」：一部经典是否是佛法，用这三个标准来检验。

## 三法印

| 法印 | 含义 | 破除的执著 |
|------|------|-----------|
| **诸行无常** | 一切有为法都在变化——没有永恒不变的东西 | 对「常」的执著 |
| **诸法无我** | 一切法没有独立、自存的「我」或本质 | 对「我」的执著 |
| **涅槃寂静** | 熄灭贪嗔痴——解脱即是寂静 | 对「有」的执著 |

---

## 四圣谛

> 佛陀初转法轮的核心教义——医生的诊断和治疗方案。

| 谛 | 含义 | 类比 |
|----|------|------|
| **苦** (Dukkha) | 生命中有不满足——生老病死、爱别离、怨憎会、求不得 | 有病 |
| **集** (Samudaya) | 苦的原因——贪爱 (tanha)、执取 | 病因 |
| **灭** (Nirodha) | 苦是可以熄灭的——涅槃 | 可治愈 |
| **道** (Magga) | 熄灭苦的方法——八正道 | 处方 |

---

## 八正道

| 类别 | 道支 | 含义 |
|------|------|------|
| **慧** | 正见 | 理解四谛 |
| | 正思维 | 出离、无嗔、无害的念头 |
| **戒** | 正语 | 不妄语、不两舌、不恶口、不绮语 |
| | 正业 | 不杀生、不偷盗、不邪淫 |
| | 正命 | 正当的谋生方式 |
| **定** | 正精进 | 防止恶念、培育善念 |
| | 正念 | 对身体、感受、心念的觉知 |
| | 正定 | 禅定——心的专注与清净 |

> 戒定慧三学——不是三个阶段——是三个相互支撑的支柱。

## 十二因缘

> 苦从哪里来？佛陀追溯了十二个环节的因果链。

```
无明→行→识→名色→六入→触→受→爱→取→有→生→老死
│                                              │
└──────────── 轮回 (samsara) ──────────────────┘
```

| 环节 | 含义 |
|------|------|
| 无明 | 不知四谛——根本的迷惑 |
| 爱 | 贪爱——苦的直接原因 |
| **关键** | 在「受」和「爱」之间——如果能不因感受而生贪嗔——链条就断了 |
