# 02-佛陀与原始佛教

> 一个王子离开宫殿，在菩提树下坐了四十九天——然后他站起来说：「我觉悟了。」你问他悟到了什么？他说：缘起。

## 佛陀生平

| 阶段 | 内容 |
|------|------|
| **出生** | 悉达多·乔达摩——迦毗罗卫国 (今尼泊尔) 王子——约公元前 5 世纪 |
| **四门出游** | 见到老人、病人、死人、沙门——第一次意识到生命中有老病死 |
| **出家** | 29 岁——夜别妻儿——寻求解脱之道 |
| **六年苦行** | 向多位老师学习——极端断食——几乎饿死——发现苦行无用 |
| **中道** | 放弃苦行——接受牧羊女的乳糜供养——不苦不乐的中道 |
| **觉悟** | 35 岁——菩提树下——降魔——证悟缘起——成为「佛陀」(觉悟者) |
| **初转法轮** | 鹿野苑——对五比丘说四圣谛——佛教诞生 |
| **涅槃** | 80 岁——「一切有为法皆归散灭——当精勤修行」——逝世 |

---

## 根本教义

### 缘起

> 「此有故彼有，此生故彼生；此无故彼无，此灭故彼灭。」

| 含义 | 内容 |
|------|------|
| **核心** | 一切事物依赖条件而生起——没有独立的、自存的「东西」 |
| **破斥** | 破「常见」(永恒的灵魂) 和「断见」(死后什么都没有) |
| **与无我的关系** | 缘起必然推出「无我」——如果有永恒的「我」，它就不依赖条件 |

### 无我 (Anatta)

| 问题 | 回答 |
|------|------|
| 「我」是什么？ | 五蕴 (色受想行识) 的和合——像车是零件的组合——拆开没有「车本身」 |
| 没有我——谁在轮回？ | 像一根蜡烛点燃另一根——火焰传递但不是「同一个」火焰 |
| 无我不是虚无主义 | 是说没有永恒的我——不是说没有因果相续 |

## 原始僧团

| 要素 | 内容 |
|------|------|
| **僧伽** (Sangha) | 佛教的出家团体——比丘 (男) 和比丘尼 (女) |
| **戒律** | 227 条比丘戒——不杀不盗不淫不妄语等 |
| **生活方式** | 托钵乞食、雨季安居、集体诵戒 |
| **民主** | 僧团决策通过「羯磨」(投票表决)——最早的民主实践之一 |

---

## 原始佛教的特征

| 特征 | 内容 |
|------|------|
| **理性** | 「不要因为传统就相信——自己验证」(卡拉玛经) ——前现代罕见的经验主义态度 |
| **实用** | 佛陀拒绝回答形而上学问题——「箭喻」——你中了毒箭，先拔箭，不问谁射的 |
| **自力** | 佛陀不是救世主——他只是指路人——「你们要自己努力」 |

### 十四无记 (佛陀不回答的问题)

| 问题 | 例子 |
|------|------|
| 世界是永恒的吗？ | 不回答 |
| 世界是有限的吗？ | 不回答 |
| 灵魂与身体是一是异？ | 不回答 |
| 如来死后存在吗？ | 不回答 |

> 为什么不回答？——这些问题的争论无助于解脱。
