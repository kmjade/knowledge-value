# 09-现代佛学

> 20 世纪以来，佛学经历了三次「现代化」：亚洲内部的人间佛教改革、西方学术界的佛学研究、以及正念 (Mindfulness) 的全球流行。佛教从亚洲寺院走向了世界。

## 人间佛教

| 人物 | 主张 |
|------|------|
| **太虚大师** (1890-1947) | 「人生佛教」——佛教不是为死人超度——是为活人服务 |
| **印顺导师** (1906-2005) | 回到原始佛教——「人间佛教」的学术基础 |
| **星云大师** (1927-2023) | 佛光山——「人间佛教」的全球推广 |
| **证严法师** (1937-) | 慈济——佛教慈善——全球最大的佛教 NGO |

### 人间佛教的核心

| 原则 | 内容 |
|------|------|
| **此岸，非彼岸** | 不是追求来世——是在此世「建设人间净土」 |
| **入世，非出世** | 不逃避社会——积极参与慈善、教育、文化 |
| **自利利他** | 大乘菩萨道的现代版本——修自己，帮别人 |

---

## 一行禅师与「入世佛教」

| 概念 | 内容 |
|------|------|
| **Engaged Buddhism** | 佛教不应回避社会问题——战争、贫困、环境 |
| **正念** | 把禅修带入日常生活——走路、吃饭、洗碗——都可以是禅 |
| **全球影响** | 一行禅师 (1926-2022) 把正念带给西方——卖了几百万本书 |

---

## 正念 (Mindfulness) 的全球化

| 阶段 | 内容 |
|------|------|
| **源** | 上座部内观——四念处 |
| **去宗教化** | Jon Kabat-Zinn——MBSR (正念减压, 1979) ——把佛教冥想去掉宗教外衣——变成医疗技术 |
| **爆发** | Google、Goldman Sachs、美国军队——正念进入「主流」 |
| **争议** | 「McMindfulness」批判——去宗教化的正念是不是「把佛教的灵魂掏空了」？ |

---

## 西方佛学研究

| 阶段 | 特征 |
|------|------|
| **19世纪** | 殖民学者的「文本佛学」——把佛教当作文献来研究——忽视活着的实践 |
| **20世纪** | 藏传佛学进入西方学术——藏传僧侣流亡后在大学设教席 |
| **当代** | 「佛教现代主义」——佛教与科学对话（神经科学+冥想）、佛教与心理学对话 |

### 神经科学与冥想

> 长期禅修者的大脑发生了什么？——前额叶增厚、杏仁核缩小、默认模式网络活动减弱——「科学验证」了冥想的生理效果。

| 争议 | 内容 |
|------|------|
| fMRI 能测量「觉悟」吗？ | 不能——神经科学只测量了禅修的副产品 |

## 达赖喇嘛与全球佛教

| 贡献 | 内容 |
|------|------|
| **科学对话** | 与神经科学家、物理学家的定期对话——Mind & Life Institute |
