## 🧠 佛学传承谱系

```
佛陀 (~500 BCE)
    │
    ├──→ 上座部 (南传)
    │    巴利藏 · 内观 · 阿罗汉
    │    斯里兰卡 → 东南亚
    │
    └──→ 大乘 (北传)
          │
          ├──→ 中观 (龙树) ——空性
          ├──→ 唯识 (无著·世亲) ——万法唯识
          ├──→ 如来藏 ——一切众生皆有佛性
          │
          ├──→ 汉传 (中国)
          │    ├── 天台 · 华严
          │    └── 禅宗 → 日本 Zen
          │
          └──→ 藏传 (西藏)
               ├── 宁玛 · 噶举 · 萨迦
               └── 格鲁 (宗喀巴→达赖/班禅)
```
