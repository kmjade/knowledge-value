# 03-孟子与性善

> 「恻隐之心，仁之端也；羞恶之心，义之端也。」孟子把孔子「仁」的模糊暗示变成了一个完整的人性论体系——性善论。他被后世尊为「亚圣」。

## 孟子其人

| 事实 | 内容 |
|------|------|
| 姓名 | 孟轲 |
| 时代 | 公元前 372-289——战国中期 |
| 师承 | 子思（孔子之孙）的门人——属于孔门正统 |
| 生涯 | 像孔子一样周游列国——游说梁惠王、齐宣王——推行仁政——失败。晚年与弟子著书 |
| 著作 | 《孟子》七篇——比《论语》更长、更系统 |

## 性善论

### 核心论证

> 「人皆有不忍人之心……今人乍见孺子将入于井，皆有怵惕恻隐之心。」

| 步骤 | 内容 |
|------|------|
| 1 | 看见小孩快掉进井里——你的第一反应是「怵惕恻隐」——想去救 |
| 2 | 这不是因为你想讨好小孩的父母、或想获得名声——而是**人性本然** |
| 3 | 如果你没有这种反应——「非人也」——你就不是人 |

### 四端

| 端 (萌芽) | 对应情感 | 发展为 |
|-----------|---------|--------|
| **恻隐之心** | 不忍他人受苦 | → 仁 |
| **羞恶之心** | 对自己的恶感到羞耻 | → 义 |
| **辞让之心** | 礼让、恭敬 | → 礼 |
| **是非之心** | 判断对错 | → 智 |

> 四端是**萌芽**——不是成品。孟子说人**天生有向善的倾向**——但如果不培育，就像不浇水的幼苗会枯死。

### 性善论不是什么

| 误解 | 孟子的真实意思 |
|------|-------------|
| 人性已经是善的——不需要努力 | 性善是「端」——萌芽——需要扩充和培育 |
| 所有人行为都是善的 | 人有善的倾向——但环境、欲望可以使人为恶 |
| 恶不存在 | 「富岁子弟多赖，凶岁子弟多暴」——环境塑造人 |

## 浩然之气

> 「我善养吾浩然之气……其为气也，至大至刚，以直养而无害，则塞于天地之间。」

| 维度 | 内容 |
|------|------|
| **浩然之气** | 道德勇气 + 生命力 + 精神充盈——不是一种技巧——是道德修养的副产品 |
| **养的方法** | 「配义与道」——做正确的事——长期积累——「集义」 |
| **不能做的** | 「助长」——不要「拔苗助长」——不要强求——自然养成 |

---

## 民本思想

> 「民为贵，社稷次之，君为轻。」

| 概念 | 内容 |
|------|------|
| **革命权** | 暴君不是君主——是「一夫」——可以诛杀 |
| **仁政** | 给百姓恒产——「有恒产者有恒心」——基本经济保障是道德的前提 |
| **与民同乐** | 君主不应独享——应当与百姓共享快乐 |

## 孟子与孔子的对比

| 维度 | 孔子 | 孟子 |
|------|------|------|
| 人性论 | 「性相近也，习相远也」——模糊 | 明确的性善论 |
| 风格 | 温和、简练 | 雄辩、激情 |
| 政治 | 「君君臣臣」——秩序 | 「诛一夫」——革命权 |
| 心性 | 不深入讨论 | 尽心→知性→知天——心性论奠基 |
