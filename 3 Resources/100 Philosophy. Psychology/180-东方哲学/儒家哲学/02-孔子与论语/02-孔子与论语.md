# 02-孔子与论语

> 「己所不欲，勿施于人。」——这八个字被刻在联合国总部。孔子不写系统著作，他和弟子聊天，弟子记下来，就是《论语》。

## 孔子其人

| 事实 | 内容 |
|------|------|
| 姓名 | 孔丘，字仲尼 |
| 时代 | 公元前 551-479——春秋末期 |
| 出身 | 鲁国（今山东曲阜）——没落贵族 |
| 生涯 | 年轻时做过委吏（仓库管理）、乘田（畜牧）。中年开始教学。51 岁做中都宰→司空→大司寇。55 岁开始 14 年周游列国——推行政治理想——失败。68 岁回鲁——整理六经、教学 |
| 学生 | 弟子三千，贤者七十二 |

## 《论语》的结构

| 特征 | 内容 |
|------|------|
| 形式 | 20 篇，499 章——语录体——没有系统章节 |
| 作者 | 孔子弟子及再传弟子编纂——非孔子手著 |
| 风格 | 简练——往往一句话就说完了——给后人留下巨大的诠释空间 |

## 核心概念

### 仁 (Ren)

> 「仁者爱人。」(颜渊)

| 维度 | 内容 |
|------|------|
| **字源** | 仁 = 人 + 二 ——人与人之间 |
| **定义** | 爱人、对他人的关怀——儒家的最高道德 |
| **实践** | 「克己复礼为仁」——约束自己，回到礼的规范 |
| **方法** | 「能近取譬」——从身边的关系推扩出去：孝→悌→忠→信→泛爱众 |
| **否定式** | 「己所不欲，勿施于人」(恕) |

> 「仁」不是一个可以精确定义的概念——孔子对不同弟子给出不同的回答。**因材施教的「仁」是对不同人的不同「药方」。**

### 礼 (Li)

| 维度 | 内容 |
|------|------|
| **含义** | 礼仪、规范、制度——从祭祀到日常行为准则 |
| **功能** | 「不学礼，无以立」——礼是人在社会中立足的框架 |
| **与仁的关系** | 仁是内在情感（爱），礼是外在形式（规范）。「人而不仁，如礼何？」——没有仁的礼是空壳 |

### 正名

> 「名不正则言不顺，言不顺则事不成。」

| 维度 | 内容 |
|------|------|
| **核心** | 名称必须与实际一致——「君君、臣臣、父父、子子」 |
| **含义** | 不是保守的等级制——是每个人的角色要与其名称匹配——做君主的要像君主 |

### 中庸

> 「中庸之为德也，其至矣乎！」

| 维度 | 内容 |
|------|------|
| **含义** | 不偏不倚——无过无不及——恰到好处 |
| **不是** | 平庸、妥协——是在两端之间找到最恰当的那个点 |

---

## 孔子的教育

| 原则 | 内容 |
|------|------|
| **有教无类** | 不论出身——只要愿意学 |
| **因材施教** | 同一个问题——对不同弟子不同回答 |
| **启发式** | 「不愤不启，不悱不发」——不到他想不通的地步，不去点拨他 |

---

## 孔子的影响

| 层面 | 内容 |
|------|------|
| **伦理** | 以「仁」为核心的道德体系——定义了东亚的「好人」概念 |
| **政治** | 「为政以德」——德治而非法治——影响了整个东亚的政治理念 |
| **教育** | 私人讲学传统——打破了贵族对知识的垄断 |
| **文化** | 六经的整理——保存了上古文化 |
