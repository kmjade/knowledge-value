# 01-儒家哲学概述

> 「儒」——最初是殷周时代主持礼仪的术士。孔子把它变成了一个道德体系，后来成为中国两千年的官方意识形态。儒家不是宗教，但它承担了宗教的功能；不是哲学，但它有最完整的伦理体系。

## 什么是「儒」？

| 维度 | 内容 |
|------|------|
| **字源** | 儒 = 人 + 需 ——人之所需？或与「濡」(浸润)相关——以礼乐浸润人心 |
| **起源** | 殷周之际的礼仪专家——主持祭祀、丧葬、冠礼 |
| **孔子的转化** | 从「术士」变成「君子」——从仪式执行者变成道德实践者 |
| **核心关怀** | 人如何在这个世界上**正确地生活**——不是来世、不是超越——是此世的人伦秩序 |

## 六经：儒家的文本基础

| 经典 | 内容 | 孔子关系 |
|------|------|---------|
| **《诗》** | 诗歌 305 篇——民间的歌、贵族的颂 | 孔子删定——「诗三百，一言以蔽之，曰思无邪」 |
| **《书》** | 上古政治文献——周公的诰命 | 孔子编纂 |
| **《礼》** | 礼仪规范——周礼、仪礼、礼记 | 孔子以礼教弟子 |
| **《乐》** | 音乐——与礼配合 | 已失传 |
| **《易》** | 占卜 + 哲学——阴阳变化 | 孔子晚年喜易——「韦编三绝」 |
| **《春秋》** | 鲁国编年史——微言大义 | 孔子修订——「一字之褒，荣于华衮」 |

## 礼乐文明：儒家的社会理想

| 概念 | 功能 |
|------|------|
| **礼** | 社会行为的规范——区别上下、亲疏、男女——「礼之用，和为贵」 |
| **乐** | 情感的调和——音乐使人心和谐——「乐者，天地之和也」 |
| **礼乐互补** | 礼区分——乐融合。一张一弛——社会才能有序而不僵硬 |

## 儒家历史分期

| 时期 | 时代 | 代表 | 核心 |
|------|------|------|------|
| **原始儒家** | 春秋战国 | 孔子、孟子、荀子 | 奠定基本概念——仁、义、礼 |
| **经学时代** | 汉—唐 | 董仲舒、郑玄、孔颖达 | 经典注疏——五经确立为官学 |
| **理学时代** | 宋—明 | 朱熹、王阳明 | 哲学化——理、气、心、性 |
| **考据时代** | 清 | 戴震、钱大昕 | 「回到汉学」——反对空谈义理 |
| **现代转型** | 20世纪— | 牟宗三、杜维明 | 与民主、科学对话 |

## 儒家精神的核心

| 精神 | 说明 |
|------|------|
| **此世性** | 「未知生，焉知死」——不在彼岸寻找意义，在此岸建立秩序 |
| **关系性** | 人不是孤立的自我——是在五伦（君臣、父子、夫妇、兄弟、朋友）中定义的 |
| **修身本位** | 一切从修身开始——「自天子以至于庶人，壹是皆以修身为本」 |
| **道德自主** | 「为仁由己，而由人乎哉？」——道德不是外部命令——是内在的自觉 |
| **入世取向** | 改造世界，不是逃避世界——与道家的出世形成对照 |
