# 09-当代新儒家

> 五四运动喊出「打倒孔家店」——儒家似乎死了。但一批思想家没有放弃——他们试图让儒家与民主、科学对话。这就是「当代新儒家」。

## 五四对儒家的冲击

| 层面 | 批判 |
|------|------|
| 政治 | 儒家是专制帮凶——「礼教吃人」(鲁迅) |
| 伦理 | 三纲五常——压抑个体 |
| 知识 | 科举八股——扼杀创新 |
| 经济 | 重农抑商——阻碍资本主义 |
| **结论** | 中国要现代化——必须先「打倒孔家店」 |

---

## 三代新儒家

### 第一代 (1920s-1940s)

| 人物 | 贡献 |
|------|------|
| **梁漱溟** (1893-1988) | 《东西文化及其哲学》——中国文化有自己的路——不是落后的西方式 |
| **熊十力** (1885-1968) | 《新唯识论》——融合佛学与儒学——为心学建立现代哲学基础 |
| **冯友兰** (1895-1990) | 《中国哲学史》——用西方哲学框架整理中国思想 |

### 第二代 (1950s-1970s) — 港台新儒家

| 人物 | 贡献 |
|------|------|
| **牟宗三** (1909-1995) | 最系统的新儒家哲学家——「良知自我坎陷」开出民主与科学 |
| **徐复观** (1903-1982) | 儒家与民主——儒家本身有「民本」传统——可以接民主 |
| **唐君毅** (1909-1978) | 《中国文化的精神价值》——中西文化各有其价值——平等对话 |

### 牟宗三的核心命题

> 「良知自我坎陷」——良知主动「退让一步」——从道德领域转为认知和政治领域——开出科学与民主。

| 概念 | 内容 |
|------|------|
| **老内圣开新外王** | 传统的内圣 (修身) 不变——但外王不是齐家治国平天下——是民主与科学 |
| **坎陷** | 良知退让——从道德主体变为认知主体和政治主体——这是儒家自我转化的关键 |

### 第三代 (1980s-至今)

| 人物 | 贡献 |
|------|------|
| **杜维明** | 儒家与全球化的对话——「文化中国」——儒家可以超越国界 |
| **刘述先** | 儒家与基督教对话 |
| **余英时** | 历史学视角——《朱熹的历史世界》 |

## 东亚现代性与儒家

| 现象 | 与儒家的可能关联 |
|------|----------------|
| **东亚经济奇迹** (日韩台港新) | 「儒家资本主义」——重视教育、勤劳、储蓄、家庭 |
| **权威主义** | 儒家「尊尊」——接受等级——东亚民主的「非自由」特征 |
| **高储蓄率** | 儒家「勤俭」 |
| **教育狂热** | 儒家「万般皆下品，惟有读书高」——科举传统的现代变形 |

> 争议：东亚经济奇迹到底是因为儒家，还是因为（美国市场、冷战地缘、产业政策）？——没有定论。

## 当代儒学实践

| 实践 | 内容 |
|------|------|
| **读经运动** | 儿童诵读儒家经典——「国学热」——有争议 |
| **儒家宪政** | 蒋庆——恢复儒家的政治地位——非常小众 |
| **企业儒学** | 儒家伦理应用于企业管理——「儒商」 |
| **生活儒学** | 儒家不是学术——是日常伦理——「温良恭俭让」 |
