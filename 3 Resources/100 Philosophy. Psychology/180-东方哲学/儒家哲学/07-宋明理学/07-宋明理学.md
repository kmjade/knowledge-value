# 07-宋明理学

> 「存天理，灭人欲。」——朱熹这句被骂了八百年的口号，其实是说：保存天赋予你的道德本性，消除过度的私欲。宋明理学是儒家对佛道挑战的回应——一场持续六百年的哲学革命。

## 为什么会有理学？

| 背景 | 说明 |
|------|------|
| **佛道挑战** | 魏晋到唐——佛道在形而上学上碾压儒家。儒家只有伦理，没有宇宙论和心性论 |
| **经学僵化** | 汉唐经学只是注疏——没有哲学创造力 |
| **回应** | 理学吸收佛道的形而上学资源——构建了儒家的宇宙论和心性论——而又坚持入世伦理 |

---

## 核心概念

### 理 (Li)

| 维度 | 内容 |
|------|------|
| **含义** | 宇宙的秩序原理——万物的「所以然」和「所当然」 |
| **超越性** | 理先于气——理是形而上的、永恒的、不变的 |
| **内在性** | 同时理就在万物之中——「理一分殊」——一个理分殊到万物中 |

### 气 (Qi)

| 维度 | 内容 |
|------|------|
| **含义** | 构成万物的质料——有清浊厚薄 |
| **与理的关系** | 理是形式，气是质料——理气相即不离 |

### 性即理

| 维度 | 内容 |
|------|------|
| **天地之性** | 纯善无恶——理在人性中的体现 |
| **气质之性** | 理+气——气有清浊——所以人有善恶差异 |
| **修养目标** | 「变化气质」——让气变得清纯——充分呈现天地之性 |

> 这个区分融合了孟子的性善和荀子的性恶——天地之性善，气质之性可善可恶。

## 程朱理学

### 程颐

| 概念 | 内容 |
|------|------|
| 「涵养须用敬，进学则在致知」 | 修养的两条路径——敬 (态度) + 知 (学习) |

### 朱熹：理学集大成者

| 概念 | 内容 |
|------|------|
| **格物致知** | 探究事物中的理——积累到一定程度——「豁然贯通」 |
| **修养方法** | 居敬 (内心恭敬专注) + 穷理 (探究事物之理) |
| **四书系统** | 朱熹将《大学》《中庸》《论语》《孟子》编为「四书」——并作注——成为科举标准 |

## 陆王心学

### 陆九渊

> 「宇宙便是吾心，吾心即是宇宙。」

| 概念 | 内容 |
|------|------|
| **心即理** | 理不在外面——就在你的心中。不需要去外面「格物」——回到内心 |
| **发明本心** | 去掉私欲遮蔽——让本心呈现——「剥落」功夫 |

### 王阳明：心学的巅峰

| 概念 | 内容 |
|------|------|
| **心即理** | 「心外无物，心外无理」——理不在外面等着你去格——理就在你心中 |
| **致良知** | 人人都有良知——「见父自然知孝，见兄自然知悌」——不需要学 |
| **知行合一** | 「知而不行，只是未知」——真正的知必然导致行动——嘴上的「知」是假的 |

### 王阳明的四句教

```
无善无恶心之体  ——心的本体超越善恶
有善有恶意之动  ——意念一动就有了善恶
知善知恶是良知  ——良知自然会判断
为善去恶是格物  ——实践就是「格物」
```

## 程朱 vs 陆王

| 维度 | 程朱理学 | 陆王心学 |
|------|---------|---------|
| 理的所在 | 在物中——需要「格」 | 在心中——需要「致」 |
| 修养方法 | 读书 + 格物 | 静坐 + 反省 |
| 知与行 | 知先行后 | 知行合一 |
| 路径 | 向外——积累——贯通 | 向内——顿悟——呈现 |
| 比喻 | 渐进——磨镜使之光 | 直指——镜本自光——去尘即可 |
