# 05-社会心理学

> 社会心理学研究一个核心问题：**情境如何塑造人的思维、情感和行为？** ——即使我们认为自己是独立自主的个体。

## 核心主题：情境的力量

社会心理学最震撼的发现：我们**系统性低估**了情境对行为的影响，而**系统性高估**了个性的作用。

这就是**基本归因错误**（Fundamental Attribution Error）：
- 别人迟到了 → "他真不靠谱"（归因于他的个性）
- 自己迟到了 → "路上太堵了"（归因于外在情境）

## 社会认知

### 归因理论

| 概念 | 含义 |
|------|------|
| **内部归因** | 行为原因来自个人（能力、性格、态度） |
| **外部归因** | 行为原因来自情境（运气、任务难度、他人） |
| **基本归因错误** | 对他人做内部归因，对自己做外部归因（西方文化中更明显） |
| **自利偏差** | 成功时归因于自己，失败时归因于外部 |

### 态度与行为

| 概念 | 含义 |
|------|------|
| **认知失调**（Festinger） | 当态度和行为不一致时产生不适，人会改变态度来合理化行为。例如"这家餐厅又贵又难吃但排队很久 → 我告诉自己这一定是高级品味" |
| **说服的中心路径** | 通过逻辑和证据说服——适用于受众有能力且动机强 |
| **说服的外周路径** | 通过情感、颜值、权威等边缘线索——适用于受众不思考 |

## 社会影响

### 从众（Conformity）

**Asch 实验**（1951）：让被试判断线段长度，周围所有人（演员）故意选明显错误的答案。结果是 76% 的被试至少从众一次，37% 的错误回答是出自从众。

从众的两种类型：

| 类型 | 原因 | 表现 |
|------|------|------|
| **信息性从众** | 认为群体知道得更多 | 真的改变了看法 |
| **规范性从众** | 不想被排斥或尴尬 | 表面服从，内心不信 |

### 服从（Obedience）

**Milgram 实验**（1963）：最著名（也是最具争议）的心理学实验。被试在"权威人物"（穿白大褂的实验员）的指令下，愿意对"学习者"施加致命的电击（当然电击是假的）。65% 的被试服从到了最高电压。

> "普通人，只是执行任务，也可以成为可怕毁灭过程的一部分。" —— Milgram

### 群体行为

| 现象 | 含义 |
|------|------|
| **社会助长** | 有他人在场时，擅长任务的绩效提高 |
| **社会懈怠** | 群体中个人付出减少（"别人会做的"） |
| **去个体化** | 在群体中匿名时自我约束降低 |
| **群体极化** | 群体讨论使已有倾向强化（谨慎的更谨慎，冒险的更冒险） |
| **群体思维** | 为维护群体和谐压制异议（挑战者号灾难的心理学分析） |

## 人际关系

### 吸引力法则

| 因素 | 作用 |
|------|------|
| **接近性** | 物理距离越近，越容易喜欢（单纯曝光效应） |
| **相似性** | 态度和价值观相似 → 最强的吸引力预测因素（不是"互补"！） |
| **外表吸引力** | 有光环效应——好看的人被认为更聪明、更善良 |
| **互惠** | 你喜欢我 → 我更可能喜欢你 |

### 亲社会与反社会行为

| 概念 | 含义 |
|------|------|
| **旁观者效应** | 旁观者越多，每个人帮助的可能性越低（责任分散） |
| **攻击的挫折-攻击假说** | 挫折增加攻击的可能性 |
| **利他的共情-利他假说** | 真正的利他来自共情，不总是出于自私 |

## 经典实验总结

| 实验 | 研究者 | 发现 |
|------|--------|------|
| Asch 从众实验 | Asch, 1951 | 群体压力让多数人说显而易见的错话 |
| Milgram 服从实验 | Milgram, 1963 | 普通人会服从权威伤害他人 |
| 斯坦福监狱实验 | Zimbardo, 1971 | 角色和情境可以迅速塑造极端行为（方法论和伦理有严重争议） |
| 旁观者干预 | Latané & Darley | 人多不一定力量大 |
