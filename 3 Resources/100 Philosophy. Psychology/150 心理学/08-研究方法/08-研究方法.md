# 08-心理学研究方法

> 心理学之所以是科学而不是哲学或常识，核心在于**方法**——系统性地收集证据来检验假设。

## 科学方法的步骤

```
观察/提问 → 提出假设 → 操作化定义 → 收集数据 → 分析 → 得出结论 → 发表/复现
    ↑                                                              ↓
    └────────────────── 修正理论 ←──────────────────────────────────┘
```

### 操作化定义

把抽象概念变成可测量的具体变量。

| 抽象概念 | 操作化定义 |
|----------|-----------|
| 焦虑 | 状态焦虑量表（STAI）得分 |
| 攻击性 | 给对手的辣酱量（是的，这真的有人做过） |
| 利他行为 | 是否停下帮助掉落文件的路人 |
| 记忆力 | 20 分钟后能回忆的单词数 |

> 好的操作化定义是心理学研究的命脉。没有它，研究就无法重复。

## 主要研究方法

### 1. 实验法（最严格的因果推断）

| 要素 | 说明 |
|------|------|
| **自变量 (IV)** | 研究者操纵的变量（如：是否服用药物） |
| **因变量 (DV)** | 研究者测量的结果（如：焦虑量表得分） |
| **实验组** | 接受处理的组 |
| **控制组** | 不接受处理（或接受安慰剂）的组 |
| **随机分配** | 让每个被试有同等机会进入各组 → 这是实验法的灵魂 |

> 只有实验法能得出**因果关系**的结论。相关法不能。

### 2. 相关法

| 特征 | 说明 |
|------|------|
| 测量 | 测量两个或多个变量，看它们如何自然共变 |
| 相关系数 r | -1.0（完美负相关）～ 0 ～ +1.0（完美正相关）|
| 优势 | 可以研究不能操纵的变量（如人格、创伤经历） |
| 局限 | **相关 ≠ 因果**：冰淇淋销量和溺水死亡相关，因为两者都受"夏天"影响（第三变量问题） |

### 3. 观察法

| 类型 | 说明 |
|------|------|
| **自然观察** | 在自然环境中观察行为，不干预（如 Jane Goodall 观察黑猩猩） |
| **参与观察** | 研究者成为群体的一员（如研究某个亚文化） |
| **结构化观察** | 在控制的设置中观察（如家长-孩子互动实验室） |

### 4. 个案研究

深入研究单个个体（如 Phineas Gage——铁棍穿过前额叶后人格改变）。优势是深度，局限是缺乏泛化性。

### 5. 调查法

用问卷或访谈收集大量数据。注意：社会期望偏差（被试说"对的"答案而非真实的答案）。

## 统计基础速览

| 概念 | 含义 | 例子 |
|------|------|------|
| **平均数 (Mean)** | 所有值的算术平均 | 全班平均分 |
| **中位数 (Median)** | 排中间的值 | 受极端值影响小 |
| **标准差 (SD)** | 离散程度 | SD 大 = 个体差异大 |
| **正态分布** | 钟形曲线 | 很多心理特质近似正态分布 |
| **p 值** | 如果零假设为真，观察到这个结果（或更极端）的概率 | p < 0.05 通常被视为"统计显著" |

> ⚠️ p 值危机：p < 0.05 不等于"假设一定是真的"。复制危机是心理学近年最大的挑战——许多经典结果无法被独立实验室复现。

## 研究伦理

| 原则 | 含义 |
|------|------|
| **知情同意** | 被试必须知道研究目的、风险和权利，自愿参加 |
| **最小伤害** | 风险和不适降到最低，收益必须大于风险 |
| **保密** | 个人信息严格保密 |
| **可随时退出** | 任何时间可退出，不受惩罚 |
| **事后说明** | 研究结束后解释真正目的（尤其是使用了欺骗时） |

### 有争议的经典研究

| 研究 | 伦理问题 |
|------|----------|
| Milgram 服从实验 | 被试经历极端心理压力 |
| 斯坦福监狱实验 | 失去控制，参与者受到心理创伤 |
| Little Albert 实验 | 对婴儿制造恐惧，未消除条件化 |

> 这些研究贡献了重要发现，但今天不可能通过伦理审查委员会（IRB）的批准。

## 如何批判性地读心理学文章

看到心理学文章时，问自己：

1. 是相关还是因果？（"导致"这个词需要实验证据）
2. 样本多大？代表性如何？（WEIRD 问题：大多数研究基于西方、受过教育、工业化、富裕、民主的人口）
3. 可重复吗？（有没有独立团队复现？）
4. 效应量多大？（统计显著 ≠ 实际重要）
5. 谁资助的研究？（利益冲突）
