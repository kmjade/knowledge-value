# 07-变态与临床心理学

> 变态心理学研究心理障碍的性质、原因和治疗。临床心理学则应用这些知识来评估和治疗心理问题。

## 什么是"异常"？4D 标准

| 标准 | 含义 |
|------|------|
| **偏离** (Deviance) | 行为、思维或情绪与文化和统计常态显著不同 |
| **痛苦** (Distress) | 给本人带来主观痛苦 |
| **功能失调** (Dysfunction) | 影响日常生活、工作或社交功能 |
| **危险** (Danger) | 对自己或他人构成安全风险 |

> 单一标准不足以判断"异常"。一个人只是"与众不同"不一定有障碍。必须综合考虑 4D。

## DSM-5 主要分类

| 类别 | 代表性障碍 | 核心特征 |
|------|-----------|----------|
| **焦虑障碍** | 广泛性焦虑、特定恐怖症、社交焦虑、惊恐障碍 | 过度恐惧/担忧 + 回避行为 |
| **心境障碍** | 抑郁症 (MDD)、双相障碍 I/II | 极端情绪（低落或高涨）持续数周 |
| **强迫及相关障碍** | OCD、囤积障碍 | 强迫思维 + 强迫行为循环 |
| **创伤及应激相关障碍** | PTSD、适应障碍 | 暴露于创伤事件后的持续反应 |
| **精神分裂症谱系** | 精神分裂症 | 幻觉、妄想、思维混乱、阴性症状 |
| **人格障碍** | 边缘型、反社会型、自恋型 | 持久、僵化、适应不良的人格模式 |
| **进食障碍** | 神经性厌食症、贪食症 | 严重紊乱的进食行为 |
| **物质相关及成瘾障碍** | 酒精、阿片类、赌博 | 失控使用 + 严重后果 |
| **神经发育障碍** | ADHD、自闭症谱系 | 发育期出现的脑功能差异 |

## 焦虑障碍详解

| 障碍 | 核心特征 |
|------|----------|
| **广泛性焦虑 (GAD)** | 对多种事物过度担忧 ≥ 6 个月，伴肌肉紧张、失眠 |
| **惊恐障碍** | 反复、意外的惊恐发作 + 持续担心下一次发作 |
| **特定恐怖症** | 对特定对象或情境的不合理恐惧（动物、高度、飞行等） |
| **社交焦虑障碍** | 对被他人评判或尴尬的强烈恐惧 |

## 抑郁症 (Major Depressive Disorder)

### 诊断标准（DSM-5 概要）

在 2 周内出现 ≥ 5 个以下症状，且至少包含①或②：

1. 抑郁心境（整天，几乎每天）
2. 兴趣或快感明显减退（快感缺失）
3. 显著体重变化或食欲改变
4. 失眠或嗜睡
5. 精神运动性激动或迟缓
6. 疲劳或精力丧失
7. 无价值感或过度内疚
8. 注意力和决策能力下降
9. 反复想到死亡或自杀

> 抑郁症是全球致残的主要原因之一。但它是可治疗的——CBT 和抗抑郁药物都有实证支持。

## 主要治疗取向

| 取向 | 核心理念 | 适用障碍 | 典型技术 |
|------|----------|----------|----------|
| **认知行为治疗 (CBT)** | 思维、情绪和行为相互影响；改变不合理的思维模式 | 抑郁、焦虑、OCD、PTSD | 认知重构、暴露、行为激活 |
| **精神分析/心理动力学** | 意识不到的冲突驱动症状 | 人格障碍、长期存在的人际问题 | 自由联想、移情分析 |
| **人本主义治疗** | 提供无条件的积极关注，来访者自带答案 | 自我探索、轻度心境问题 | 积极倾听、共情、真实（Genuineness） |
| **生物/药物治疗** | 心理障碍有神经化学基础 | 精分、双相、严重抑郁 | SSRI（抗抑郁）、锂盐、抗精神病药 |
| **正念疗法** | 当下觉察、不评判的接纳 | 抑郁复发预防、焦虑 | MBCT（正念认知）、DBT（辩证行为）中的正念 |
