# 行為主義 Behaviorism

> 給我十幾個健康的嬰兒，我可以在任意指定的環境中把他們培養成任意類型的專家——醫生、律師、藝術家，甚至乞丐和小偷。——John B. Watson

## 定義

行為主義強調心理學應研究**可觀察的行為**，而非不可觀察的意識或心理過程。行為是通過與環境的互動而習得的。

## 核心概念

### 古典制約 Classical Conditioning
```
食物 (US) → 唾液 (UR)
鈴聲 (CS) + 食物 (US) → 唾液 (UR)
鈴聲 (CS) → 唾液 (CR)
```

### 操作制約 Operant Conditioning

| | 增加行為 | 減少行為 |
|---|:---:|:---:|
| **給予刺激** | 正強化 (+) | 正懲罰 |
| **移除刺激** | 負強化 (-) | 負懲罰 |

### 強化時程 Reinforcement Schedules

| 時程 | 特點 | 反應模式 |
|------|------|---------|
| 固定比率 (FR) | 每 N 次反應後強化 | 高反應率、短暫停頓 |
| 可變比率 (VR) | 平均每 N 次強化 | **最高且最穩定的反應率** |
| 固定時距 (FI) | 每 N 分鐘後強化 | 扇形反應模式 |
| 可變時距 (VI) | 平均每 N 分鐘強化 | 穩定中等反應率 |

## 批評 Critiques

- 忽略認知過程（思維、記憶、問題解決）
- 動物實驗結果未必適用於人類
- 語言習得無法僅用制約解釋（Chomsky, 1959）
