# 120 认识论 / Epistemology

> 「我们如何知道？」— 知识是什么？我们如何获得它？我们能知道什么？
> 认识论是哲学的基石：在追问「什么是真实」之前，先问「我们凭什么声称知道」。

## ① 知识是什么？— JTB 与葛梯尔

### 经典分析：JTB (Justified True Belief)

> 柏拉图《泰阿泰德》以来的标准答案：知识 = 得到辩护的真信念

| 条件 | 含义 | 反例 |
|------|------|------|
| **信念 (Belief)** | S 相信 P | 你不相信的东西不能算你知道 |
| **真 (True)** | P 是真的 | 假的信念不可能是知识 |
| **辩护 (Justified)** | S 有理由/证据支持 P | 靠运气猜对不算知识 |

```
知识 (Knowledge) = 辩护 (Justification) + 真 (Truth) + 信念 (Belief)

     ┌─────────┐
     │  知识    │  ← JTB 三者交集
     │  ┌─────┐ │
     │  │JTB  │ │
     │  └─────┘ │
     └─────────┘

  ┌─────┐    ┌─────┐    ┌─────┐
  │辩护  │    │ 真  │    │信念  │
  │     │    │     │    │     │
  └─────┘    └─────┘    └─────┘
```

### 葛梯尔问题 (Gettier Problem, 1963)

> Edmund Gettier 用两页纸摧毁了两千年的共识。

| 经典葛梯尔案例 | 说明 |
|---------------|------|
| **停了的钟** | 你看钟显示 12:00，钟一向准。恰好此刻真的是 12:00 —— 但钟其实 24 小时前就停了。你有一个**得到辩护的真信念**，但直觉上这不是知识。 |
| **伪造的羊** | 你看到田野里有只羊 (其实是块像羊的石头)，你相信「田野里有羊」。但田野里确实有只羊在石头后面你看不到的地方。JTB 满足，但不是知识。 |

> **葛梯尔的洞见**：JTB 是知识**必要**但不**充分**的条件。辩护与真理之间的联系可能只是巧合。

### 后葛梯尔方案

| 方案 | 核心主张 | 代表人物 |
|------|----------|----------|
| **无假引理 (No False Lemma)** | 推理链条中不能有假前提 | Clark |
| **因果理论** | 信念与事实之间必须有因果联系 | Goldman |
| **可靠性理论** | 信念必须由可靠的过程产生 | Goldman |
| **德性认识论** | 知识 = 由认知德性产生的真信念 | Sosa, Zagzebski |
| **安全条件** | 在邻近可能世界中，信念不会轻易变假 | Pritchard |

## ② 知识的来源

### 理性 vs 经验

| 维度 | 理性主义 (Rationalism) | 经验主义 (Empiricism) |
|------|------------------------|----------------------|
| 知识来源 | 理性 (reason) | 感官经验 (experience) |
| 先天知识 | 存在且优先 | 不存在（或只是分析命题） |
| 范式知识 | 数学、逻辑 | 自然科学 |
| 代表人物 | 笛卡尔、斯宾诺莎、莱布尼茨 | 洛克、贝克莱、休谟 |
| 康德 | **综合**：先验范畴 + 后天经验 = 知识 |

### 知识的类型

| 类型 | 定义 | 示例 |
|------|------|------|
| **命题知识 (know-that)** | 知道某个事实 | 「我知道巴黎是法国首都」 |
| **能力知识 (know-how)** | 知道如何做 | 「我知道怎么骑自行车」 |
| **亲知知识 (acquaintance)** | 直接熟悉 | 「我认识她」 |
| **先天知识 (a priori)** | 不依赖经验 | 2+2=4、「所有单身汉都未婚」 |
| **后天知识 (a posteriori)** | 依赖经验 | 「水在 100°C 沸腾」 |

## ③ 真理理论

| 理论 | 核心主张 | 真理 = ? | 代表人物 |
|------|----------|----------|----------|
| **符合论 (Correspondence)** | 命题符合事实 | 信念 ↔ 世界 | 亚里士多德、罗素 |
| **融贯论 (Coherence)** | 命题与信念系统一致 | 信念 ↔ 信念系统 | Spinoza, Bradley |
| **实用论 (Pragmatic)** | 真 = 有用/有效 | 信念 → 实际后果 | 詹姆斯、杜威 |
| **紧缩论 (Deflationary)** | 「真」没有本质，只是语言工具 | 「P 是真的」=「P」 | Ramsey, 奎因 |
| **语义论 (Semantic)** | 由语言结构定义真 | Tarski's T-schema | Tarski |

> Tarski: 「『雪是白的』是真的，当且仅当雪是白的。」

## ④ 怀疑论 (Skepticism)

### 经典怀疑论挑战

| 类型 | 场景 | 颠覆了... |
|------|------|-----------|
| **笛卡尔的恶魔** | 一个全能的恶魔系统性地欺骗你 | 对外部世界的知识 |
| **缸中之脑 (Putnam)** | 你的大脑在缸中，一切体验是模拟 | 对外部世界的知识 |
| **庄周梦蝶** | 你梦为蝶，还是蝶梦为你？ | 自我同一性的知识 |
| **休谟的归纳问题** | 过去 ≠ 保证未来 | 对自然规律的知识 |

### 回应策略

| 策略 | 核心主张 | 代表人物 |
|------|----------|----------|
| **摩尔式回应** | 「这里有一只手」比任何怀疑论证更确定 | G.E. Moore |
| **语境主义** | 「知道」在日常语境中是真的，在怀疑语境中不同 | Lewis, DeRose |
| **可谬论** | 知识不需要绝对确定性；可错的知识仍然是知识 | — |
| **自然化认识论** | 别问「我们能知道吗」，去研究我们实际如何认知 | 奎因 |

## ⑤ 内在论 vs 外在论

| 维度 | 内在论 (Internalism) | 外在论 (Externalism) |
|------|---------------------|---------------------|
| 核心主张 | 辩护因素必须在认知者「头脑内部」可通达 | 辩护可以依赖外部因素（可靠性、因果链） |
| 论证方式 | 我必须**能意识到**我为什么知道 | 只要**事实上**可靠即可 |
| 强项 | 匹配我们的主观感受 | 解决葛梯尔问题，适合动物/儿童知识 |
| 代表人物 | 笛卡尔、Chisholm | Goldman, Dretske |

## ⑥ 社会认识论 (Social Epistemology)

> 知识不是孤立的个人成就——它是社会互动的产物。

| 问题 | 说明 |
|------|------|
| **证词 (Testimony)** | 我们能从他人的话中获得知识吗？信任的边界在哪？ |
| **分歧 (Disagreement)** | 当认知对等者 (epistemic peers) 意见不一致，谁应该修正？ |
| **集体知识** | 群体/组织/文化能够「知道」吗？还是只有个体能知道？ |
| **回音室与过滤泡** | 数字时代：社会结构如何扭曲知识获取？ |
