# 魔兽世界 - 防御圣骑士神圣力量系统

> [!info] 核心资源
> 神圣力量（Holy Power）是圣骑士的核心资源系统，所有消耗神圣力量的技能都有高额效果，正确管理神圣力量是防骑的关键。

---

## 神圣力量基础

### 资源定义
- **资源名称**：Holy Power（神圣力量）
- **最大上限**：5 点（专精加成可提升）
- **基础上限**：3 点（通过专精提升到 5 点）
- **获取方式**：特定技能施放
- **消耗方式**：神圣震击、愤怒之锤、神圣风暴等

### 资源特点
- **不可自然恢复**：无法自然回复，只能通过技能获取
- **高优先级**：满神圣力量时应立即使用消耗技能
- **爆发核心**：复仇之怒爆发期间快速获取并消耗

### 获取技能一览

| 技能 | 获取量 | 冷却时间 | 优先级 | 说明 |
|------|---------|------------|---------|------|
| **审判** | 2 点 | 8 秒 | ★★★★★ | 主要获取方式，高额仇恨 |

| **愤怒之锤** | 1 点 | 12 秒 | ★★★☆☆ | 打断用，仇恨较低 |
| **神圣风暴** | 1 点 | 9 秒 | ★★★☆☆ | AoE 用，多目标获取 |

### 获取优先级分析

#### 高优先级（必须优先）
1. **审判**
   - 原因：2 点神圣力量 + 高额仇恨
   - 使用时机：CD 一好就使用
   - 仇恨贡献：★★★★★

2. **正义复仇**
   - 原因：2 点神圣力量 + AoE 伤害
   - 使用时机：聚怪时优先使用
   - 仇恨贡献：★★★★☆

#### 中优先级（填补用）
3. **十字军打击**
   - 原因：1 点神圣力量 + 快速 CD
   - 使用时机：审判 CD 期使用
   - 仇恨贡献：★★★☆☆

4. **神圣风暴**
   - 原因：1 点神圣力量 + AoE 伤害
   - 使用时机：3 个以上目标时使用
   - 仇恨贡献：★★★☆☆

#### 低优先级（辅助用）
5. **愤怒之锤**
   - 原因：1 点神圣力量 + 打断功能
   - 使用时机：打断施法时使用
   - 仇恨贡献：★★☆☆☆

## 神圣力量管理策略

### 1. 标准输出循环

#### 神圣力量不满时
```
优先级顺序：
1. 审判（获取 2 点）→ 神圣力量：2-4 点
2. 神圣震击（消耗 3 点）→ 神圣力量：0-2 点
3. 十字军打击（获取 1 点）→ 神圣力量：1-3 点
4. 神圣震击（消耗 3 点）→ 神圣力量：0 点
循环 2-4
```

#### 神圣力量满时（5 点）
```
优先级顺序：
1. 神圣震击（消耗 3 点）→ 神圣力量：2 点
2. 审判（获取 2 点）→ 神圣力量：4 点
3. 神圣震击（消耗 3 点）→ 神圣力量：1 点
4. 十字军打击（获取 1 点）→ 神圣力量：2 点
```

---

### 2. 爆发期输出循环（复仇之怒）

#### 开启复仇之怒后
```
复仇之怒效果：30% 伤害提升（天赋可额外 +30%）
优先级顺序：
1. 审判（获取 2 点，仇恨 200%）
2. 神圣震击（消耗 3 点，高额伤害）
3. 十字军打击（获取 1 点）
4. 愤怒之锤（消耗 3 点，打断）
5. 神圣风暴（消耗 3 点，AoE）
6. 循环 2-5
```

#### 天赋加持
- **神圣之怒天赋**：复仇之怒期间额外 +30% 伤害
- **强化审判天赋**：审判神圣力量获取翻倍（4 点）
- **神圣复仇天赋**：复仇之怒 CD 减少 15 秒

---

### 3. AoE 场景管理

#### 多目标场景（3+ 敌人）
```
优先级顺序：
1. 正义复仇（获取 2 点，AoE 聚怪）
2. 神圣风暴（消耗 3 点，AoE 伤害 + 治疗）
3. 审判（获取 2 点）
4. 神圣风暴（消耗 3 点）
5. 十字军打击（获取 1 点）
6. 重复 2-5
```

#### 单目标 Boss 战
```
优先级顺序：
1. 审判（获取 2 点，高额仇恨）
2. 神圣震击（消耗 3 点，最高伤害）
3. 十字军打击（获取 1 点）
4. 重复 1-3
```

### 4. 团队副本坦怪场景

#### 拉怪阶段
```
神圣力量管理：
1. 开启神圣之盾 → 减免拉怪伤害
2. 正义复仇 → 聚怪，获取 2 点
3. 审判 → 建立仇恨，获取 2 点
```

#### 仇恨维持
```
神圣力量目标：保持 3-4 点，确保随时可以使用神圣震击
循环：
审判（+2）→ 神圣震击（-3）→ 十字军打击（+1）→ 神圣震击（-3）
结果：仇恨稳定，伤害输出
```

## 常见问题与解决方案

### 问题 1：神圣力量总是满
**原因**：忘记使用消耗技能

**解决方案**：
1. 配置宏：审判 + 神圣震击一键连发
2. 设置提醒：神圣力量满时提醒
3. 训练反应：养成满神圣力量立即使用的习惯

---

### 问题 2：神圣力量总是空
**原因**：获取技能使用太少

**解决方案**：
1. 优先审判：确保每 8 秒使用一次审判
2. 提升命中：确保技能命中，避免浪费 CD
3. 调整循环：增加审判和十字军打击使用频率

---

### 问题 3：仇恨不稳定
**原因**：神圣力量消耗节奏不当

**解决方案**：
1. 保持神圣力量 3-4 点：确保随时可用神圣震击
2. 优先审判：审判提供最高仇恨
3. 使用复仇技能：传递仇恨，稳定仇恨值

---

## 进阶技巧

### 1. 审判仇恨最大化
- **天赋加持**：强化审判（+20% 伤害，神圣力量获取翻倍）
- **时机选择**：仇恨建立期优先审判
- **目标优先**：优先审判当前威胁最高的目标

### 2. 神圣力量预留
- **预留策略**：保持神圣力量 3 点，应对突发情况
- **打断优先**：愤怒之锤不占用预留神圣力量
- **爆发规划**：复仇之怒爆发前确保神圣力量 5 点

### 3. CD 期管理
- **审判 CD 期**：使用十字军打击填补获取空窗
- **神圣震击 CD 期**：使用审判获取神圣力量
- **神圣风暴 CD 期**：单体输出优先使用神圣震击

## 数据分析

### 仇恨贡献计算

#### 基础仇恨公式
```
基础仇恨 = 技能伤害 × 仇恨倍率 × 神圣力量消耗系数

仇恨倍率（防骑）：200%
神圣力量消耗系数：
- 神圣震击（3 点）：1.5
- 愤怒之锤（3 点）：1.3
- 神圣风暴（3 点）：1.0
- 审判（不消耗）：1.0
- 十字军打击（1 点）：0.8
```

#### 实际仇恨计算
```
示例：神圣震击造成 5000 伤害
仇恨 = 5000 × 200% × 1.5 = 15,000

示例：审判造成 3000 伤害
仇恨 = 3000 × 200% × 1.0 = 6,000
```
