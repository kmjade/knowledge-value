# 魔兽世界 - 防御圣骑士宏

> [!info] 防骑定位
> 防御圣骑士是魔兽世界的主要坦克职业之一，以高生存能力、团队保护和强力爆发著称。
> - 核心属性：力量、护甲、精通
> - 核心技能：圣佑、圣盾术、神圣之盾
> - 团保技能：自由祝福、保护祝福、牺牲祝福

---

## 核心输出宏

### 1. 复仇之怒爆发宏

```lua
#showtooltip 复仇之怒
/cast [mod:alt] 复仇之怒; 复仇之怒
/cast [mod:alt] 复仇
/startattack
```

**说明**：
- 普通按键：复仇之怒 + 自动攻击
- Alt + 按键：复仇 + 自动攻击

---

### 2. 复仇之怒 + 裁决爆发

```lua
#showtooltip 复仇之怒
/castsequence reset=10 复仇之怒,公正之剑,复仇之怒,十字军打击,复仇之怒
```

**说明**：
- 自动循环复仇之怒和主要输出技能
- reset=10 表示 10 秒后重置序列

---

### 3. 自动攻击 + 裁决

```lua
#showtooltip 裁决
/startattack
/cast 裁决
```

**说明**：
- 确保进入战斗就自动攻击
- 适合放在主要输出键位

---

### 4. 愤怒之锤（圣闪）宏

```lua
#showtooltip 愤怒之锤
/cast [mod:alt] 圣盾术; 愤怒之锤
```

**说明**：
- 普通按键：愤怒之锤（打断技能）
- Alt + 按键：圣盾术（保命）

## 实战使用技巧

### 1. 拉怪阶段
```
1. 开启神圣之盾
2. 使用正义复仇聚怪
3. 施放公正之剑建立仇恨
4. 切换圣印（正义圣印）

```

2. 保持正义圣印

4. 适时使用正义复仇
```

### 3. 爆发阶段
```
1. 开启复仇之怒
2. 使用神圣之盾减伤
3. 高频率输出技能
4. 保持圣印循环
```

### 4. 应急阶段
```
1. 血量危险 → 圣佑术
2. 即将死亡 → 圣盾术
3. 治疗自己 → 神恩术
4. 保护队友 → 保护祝福

## 故障排除

### 常见问题

#### 问题 1：技能无法施放
**解决**：
- 检查技能名称是否正确
- 确认是否在学习该技能
- 检查是否有前置条件

#### 问题 2：圣印无法切换
**解决**：
- 使用 /castsequence 而非 /cast
- 添加 reset 参数
- 检查圣印学习状态

#### 问题 3：祝福无效
**解决**：
- 确认目标为友方单位
- 检查目标是否已有祝福状态
- 确认祝福类型选择正确

#### 问题 4：宏超过 255 字符
**解决**：
- 拆分为多个宏
- 使用插件支持超长宏
- 简化条件判断

## 插件推荐

### 宏辅助插件

#### GS-E (Gnome Sequencer - Enhanced)
- 功能：支持超长序列宏
- 优势：突破 255 字符限制
- 适合：复杂循环输出

#### Macro Toolkit
- 功能：宏编辑器增强
- 优势：更好的编辑界面
- 适合：大量宏管理

#### Super Duper Macro (SDM)
- 功能：超长宏支持
- 优势：宏字符数不受限制
- 适合：需要复杂宏

#### Clique
- 功能：鼠标施法宏
- 优势：快速治疗和施法
- 适合：坦克快速治疗队友

#### TellMeWhen
- 功能：技能冷却监控
- 优势：显示所有技能状态
- 适合：优化输出循环
