## 📚 工作类型

### 工作分類

| 类型 | 特點 | 時間投入 | 例子 |
|------|------|---------|------|
| **深度工作** | 需要高度專注和认知投入 | 长期块 | 編程、写作、研究 |
| **協作工作** | 需要与團隊成员沟通合作 | 中期 | 會議、專案协调 |
| **浅度工作** | 常规性、低认知要求 | 短期 | 電子郵件、行政任務 |
| **创造性工作** | 需要灵感和創新思维 | 不定 | 設計、创意、策划 |

### 工作模式

```
全职工作模式：
  ┌──────────────────────────┐
  │  09:00-12:00  深度工作  │

  │  13:00-15:00  協作工作   │
  │  15:00-18:00  深度工作  │
  └──────────────────────────┘

  ┌──────────────────────────┐
```

# 工作流

**1. 每日准备（15分钟）**
- 回顧昨天的完成情况
- 確定今天的优先事項（MIT）
- 准备所需材料和工具
- 清理工作空間

**2. 深度工作（90分钟块）**
- 单任務执行
- 拒絕所有干扰
- 保持專注
- 記錄進展

**3. 短暂休息（20分钟）**
- 完全放松
- 远离屏幕
- 活动身体
- 不看手机

**4. 重复循环**
- 上午：2个深度工作块
- 下午：1-2个深度工作块

## 🔄 工作循环

### 每周工作循环

```
周一：計劃周 → 啟動專案
周二-周四：执行任務 → 深度工作
# 整理
周六-周日：休息充电 → 个人成长
```

### 專案工作循环

```
啟動 → 执行 → 監控 → 收尾

1. 啟動：明确目標、分配資源
2. 执行：按計劃推进、解決問題
3. 監控：追踪進展、调整策略
4. 收尾：總結成果、經驗教训
```

## 💡 工作原則

### 帕累托原則（80/20）
> 80% 的成果来自 20% 的關鍵活动

- 识别高價值任務
- 将大部分時間投入20%
- 减少或委派80%的活动

### 帕金森定律
> "工作会自動膨胀，直到占满所有可用時間"

- 设定明确的截止時間
- 提高時間感知的紧迫感
- 避免"無限時間"心态

### 单任務原則
# 效率

- 一次只做一件事
- 完成後再開始下一个
- 减少任務切換成本

## 📅 工作与生活平衡

# 管理

# 方法
|------|------|------|
| **设定边界** | 明确工作時間和个人時間 | 避免过度工作 |
| **优先休息** | 设定强制休息時間 | 保持长期可持續 |
# 效率
| **定期断连** | 每周设定"無電子郵件"時間 | 减少数字压力 |

### 生活品質

| 方面 | 策略 | 频率 |
|------|------|------|
| **健康** | 运动、健康饮食 | 每天 |
| **學習** | 阅读书籍、學習新技能 | 每周 |
| **社交** | 与朋友家人相处 | 每周 |
| **兴趣** | 培养爱好、创造時間 | 每周 |

> [!tip] 實踐建議
