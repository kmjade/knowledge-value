# 管理

### GTD (Getting Things Done)

```
收集 → 處理 → 组织 → 回顧 → 执行
```

| 步骤 | 目標 | 工具 |
|------|------|------|
| **收集** | 捕获所有想法 | InBox、備忘录 |
| **處理** | 清空收集箱 | 遵循 2分钟原則 |
# 排序
# 更新
| **执行** | 根据情境行动 | 下一步清單 |

### PARA + GTD 结合

```
GTD 流程：
  InBox 收集
    ↓
  處理和分類
    ↓
  PARA 歸檔：
    - 有明确目標 → Projects
    - 长期责任 → Areas
    - 參考资料 → Resources
    - 完成專案 → Archives
```

# 管理

### SMART 目標原則

| 原則 | 說明 | 示例 |
|------|------|------|
| **S** - Specific | 具体明确 | "每天运动30分钟" |
| **M** - Measurable | 可衡量 | "完成10个專案" |
| **A** - Achievable | 可達成 | 符合能力范围 |
| **R** - Relevant | 相關性 | 与长期目標一致 |
| **T** - Time-bound | 有时限 | "在3个月内完成" |
