### 🔄 分類決策流程

```
新資訊进入收件箱 (InBox)
    ↓
是否当前活跃？
    ├── 否 → 归入 Archives
    └── 是 → 是否有截止日期？
              ├── 是 → Projects（短期專案）
              └── 否 → 是否持續性责任？
                        ├── 是 → Areas（长期领域）
                        └── 否 → Resources（兴趣資源）
```

### 🔗 与 Zettelkasten 的关系

|系統|用途|特點|

# 知識

**结合方式**：

- PARA 决定筆記的**存放位置**
- Zettelkasten 建立筆記之间的**連結关系**
- 原子筆記儲存在 `5 Zettels/`，通過連結关联到 PARA 各分類

### 📊 回顧周期

|类别|频率|重点|
|---|---|---|
|**Projects**|每周|進展、下一步行动、是否完成|
|**Areas**|每月|維護状况、是否需要新專案|
|**Resources**|每季|相關性、是否需要清理|
|**Archives**|每年|清理、压缩、刪除|

PARA 的核心理念是：**让資訊跟随行动，而非被困在主題資料夾中**。这样可以快速找到当前需要的資訊，保持系統的长期清晰度。
