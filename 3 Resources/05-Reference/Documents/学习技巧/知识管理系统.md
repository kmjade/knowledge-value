# 管理

### PARA 系統

```
Projects（專案）：
  - 有明确目標和截止日期
  - 短期可执行
  - 完成後歸檔

Areas（领域）：
  - 长期责任
  - 持續維護
  - 没有截止日期

Resources（資源）：
  - 持續感兴趣的话题
  - 參考资料
  - 按主題分類

Archives（歸檔）：
  - 完成或不活跃的專案
  - 歷史記錄
  - 不再需要频繁访问
```

# 方法

```
核心原則：

1. 原子化：每张筆記一个想法
2. 自主性：筆記可以独立存在
3. 双向連結：通過 [[]] 連接相關概念
# 修改
# 知識

連結方式：
- 直接連結：[[筆記名]]
- 嵌入連結：![[筆記名]]
- 块引用：[[筆記名#^block-id]]
# 查看
```

# 工作流

```
收集 → 组织 → 提炼 → 表达

1. 收集 (Capture):
   - 快速記錄所有想法
   - 使用 InBox、闪念筆記
   - 不編輯，只記錄

2. 组织 (Organize):
# 整理
   - 分類到 PARA
   - 新增標籤和連結

3. 提炼 (Distill):
   - 提取核心观点
   - 创造原子筆記
# 知識

4. 表达 (Express):
   - 写文章、做專案
   - 教授他人
   - 创造新價值
```
