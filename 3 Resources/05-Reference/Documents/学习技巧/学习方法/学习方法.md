## 📚 學習类型

### 學習分類

| 类型 | 特點 | 時間投入 | 示例 |
|------|------|---------|------|
| **深度學習** | 系統化、结构化 | 长期 | 學習編程语言 |
| **碎片學習** | 短时、随机性 | 短期 | 阅读文章 |
| **專案學習** | 目標导向、實踐性 | 中期 | 完成網站專案 |
| **社交學習** | 互动、讨论 | 不定 | 参加课程/社區 |

### 學習模式

```
輸入 → 處理 → 輸出 → 反思 → 優化

1. 輸入: 阅读书、看視訊、听课
2. 處理: 理解、總結、关联
3. 輸出: 写筆記、做專案、教别人
4. 反思: 評估效果、识别不足
# 方法
```

## 🎯 學習策略

### 费曼技巧
> "如果你不能简单解釋它，你就没有真正理解它。"

**步骤：**
1. 選擇要學習的概念
2. 用简单语言解釋（假设给5岁孩子讲）
3. 找出解釋不通的地方
4. 回顧资料并补充
5. 簡化语言重新解釋

### 番茄工作法學習
```
25分钟專注學習 + 5分钟休息

- 專注期间：只做學習任務
- 休息期间：完全放松
- 4个番茄钟后：长休息15-30分钟
```

### 间隔重复 (Spaced Repetition)
- **第1次**：立即复习
- **第2次**：1天后
- **第3次**：3天后
- **第4次**：7天后
- **第5次**：15天后

使用工具：Anki、Flashcards

### 主动回忆
> 不要重读筆記，而是尝试从记忆中提取

- 闭眼回忆刚才学到的內容
- 写下记住的關鍵点
- 對比筆記，补充遗漏部分

# 方法

### 康奈尔筆記法

```
┌─────────────────────────────────────┐
│  記錄区 (Record)              │
│  - 記錄课堂/阅读內容           │
│  - 使用简写和符号             │
│                               │
│  线条...                      │
├─────────────────────────────────────┤
│  提示区 (Cue)   │
│  - 写下關鍵词/問題             │
│  - 用于复习时自我測試           │
├─────────────────────────────────────┤
│  總結区 (Summary) │
│  - 總結主要內容                │
│  - 用自己的话写               │
└─────────────────────────────────────┘
```

### 思维导图筆記
- 中心主題 → 主要分支 → 次要分支 → 细节
- 使用颜色、圖示、連接线
- 工具：XMind、MindNode、Obsidian Canvas

### Zettelkasten 筆記
- 原子化：每张筆記一个想法
- 双向連結：通過 `[[]]` 連接相關概念

## 🔄 學習循环

### PDCA 學習法

```
Plan (計劃) → Do (执行) → Check (檢查) → Act (行动)

# 方法
2. Do: 按計劃执行學習
3. Check: 評估學習效果
# 方法
```

### 深度工作學習
```
# 分享

准备:
  - 明确學習目標
  - 准备學習材料
  - 關閉干扰源

深度學習:
  - 專注2-3小时
  - 主动思考和應用程式
  - 記錄筆記

# 整理
  - 總結關鍵点
# 知識
  - 创造輸出成果

# 分享
  - 教授他人
  - 写文章/博客
  - 参与讨论
```

## 💡 學習原則

### 80/20 原則
# 知識
- 先學習核心概念，再深入细节

### 刻意练习 (Deliberate Practice)
- 專注弱点，而非舒适区
- 即时回饋
- 重复练习
- 保持挑戰性

### 费曼學習 + 教学相长
- 教是最好的學習方式
- 通過輸出驗證理解

> [!tip] 實踐建議
> 1. **设定明确目標** - 學習前知道要达到什么程度
> 2. **主动學習** - 不只是阅读，要思考和應用程式
> 3. **定期輸出** - 写筆記、做專案、教别人
> 4. **保持耐心** - 學習是长期過程，不要急于求成
