### 步驟一：登錄 useradmin

1. 打開瀏覽器，訪問 `http://192.168.1.1`
2. 使用 useradmin 帳戶登錄
3. **重要：不要關閉登錄視窗**

| 輸入項 | 內容 |

| 用戶名 | `useradmin` |

## 三、密碼解密

### 3.1 Python 解密腳本

```python
# 光貓密碼解密腳本
# 將 ASCII 編碼轉換為實際密碼

# 方法
ascii_codes = [120, 105, 112, 105, 103, 115, 113, 101, 104, 113, 109, 114, 57, 56, 49, 54, 56, 56, 53, 54]

password = ""
for code in ascii_codes:
    password += chr(code)

print(f"解密後的密碼: {password}")
# 輸出：xipigsqehqmr98168856
```

```python
# 方法
ascii_codes = [120,105,112,105,103,115,113,101,104,113,109,114,57,56,49,54,56,56,53,54]
result = ''.join(chr(code) for code in ascii_codes)
print(result)  # xipigsqehqmr98168856
```

### 3.2 在線工具解密

訪問以下網站進行 ASCII 解碼：

| 網站 | 用途 |
|------|------|
| ASCIITable.com | ASCII 轉字符 |
| CyberChef | 綜合解碼工具 |

### 3.3 密碼結構

| 部分 | 說明 | 位數 |
|------|------|------|
| **前綴** | 固定字符串 | 12 位 |
| **後綴** | 數字隨機碼 | 8 位 |
| **總長度** | 完整密碼 | 20 位 |

**示例**：`xipigsqehqmr98168856`
- 前綴：`xipigsqehqmr`（12 位）
- 後綴：`98168856`（8 位）

## 五、自動化腳本

### 5.1 Bash 腳本（Linux/macOS）

```bash
#!/bin/bash
# 光貓密碼解密腳本

# 輸入 ASCII 編碼
ascii_codes="120&105&112&105&103&115&113&101&104&113&109&114&57&56&49&54&56&56&53&54&"

# 提取數字並轉換
IFS='&' read -ra codes <<< "$ascii_codes"
password=""
for code in "${codes[@]}"; do
    if [ -n "$code" ]; then
        password+=$(printf "\\x%x" "$code")
    fi
done

echo "解密後的密碼: $password"
```

### 5.2 PowerShell 腳本（Windows）

```powershell
# 光貓密碼解密腳本

$asciiCodes = @(120, 105, 112, 105, 103, 115, 113, 101, 104, 113, 109, 114, 57, 56, 49, 54, 56, 56, 53, 54)

$password = ""
foreach ($code in $asciiCodes) {
    $password += [char]$code
}

Write-Host "解密後的密碼: $password"
```

# 分析

| 原字符 | 對應關係 | 說明 |
|--------|----------|------|
| `t` | → `x` | 向前位移 4 位 |
| `e` | → `i` | 向前位移 4 位 |
| `l` | → `p` | 向前位移 4 位 |
| ... | ... | ... |

### 8.2 為什麼這樣設計

| 原因 | 說明 |
|------|------|
| **簡單加密** | 防止普通用戶直接看到密碼 |
