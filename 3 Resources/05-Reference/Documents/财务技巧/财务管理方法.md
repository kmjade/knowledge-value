### 零基预算 (Zero-Based Budgeting)

**步骤：**
1. 从0開始：将所有收入分配到具体支出类别
2. 每元都有用途：确保收入=支出
3. 调整平衡：如果有结余，调整储蓄或投资

### 定期定额投资 (DCA)

**优点：**
- 平均成本：在市场高低点都买入
- 避免择时：不需要预测市场走势
- 情绪稳定：减少投资焦虑

### 债务偿还策略

**债务雪球法 (Debt Snowball):**
1. 列出所有债务
# 排序
3. 最小化最低余额债务的还款
4. 还清一个后，将金额转移到下一个债务

**债务雪崩法 (Debt Avalanche):**
1. 列出所有债务
# 排序
3. 集中所有资金还最高利率债务
4. 还清一个后，转移到下一个债务

## 💡 财务最佳實踐

### 巴菲特原則

1. **支出第一**：收入 - 储蓄 = 支出
2. **避免高息债务**：信用卡利息是财富杀手
3. **长期投资**：让复利为你工作
4. **持續學習**：投资自己是最重要的投资

### 防刷爆原則

> "不要借你無法在30天内还清的钱"

### 财务自由計算

```
财务自由 = (被动收入 × 12) / 年支出

被动收入来源：
- 股息、分红
- 房租收入
- 版权收入
- 企业收入
```

> [!tip] 实施建議
> 1. **从简单開始** - 先記錄一个月支出，了解消费模式
> 2. **建立紧急基金** - 3-6个月生活費用作为安全网
> 3. **还清高息债务** - 优先偿还利率最高的债务
> 4. **開始定期投资** - 使用定投策略，分散风险
> 5. **定期回顧** - 每月檢查财务健康状况
