# 管理

### 四象限分類

```
紧急度
  ↑
  │  🔥 第一象限            📞 第二象限
  │  重要且紧急            重要不紧急
  │  - 危机處理            - 战略規劃
  │  - 最后期限            - 能力提升
  │  - 重要任務            - 关系維護
  │
  │  📮 第三象限            🎮 第四象限
  │  不重要但紧急          不重要不紧急
  │  - 某些會議            - 社交媒体
  │  - 部分電子郵件            - 消费性娱乐
  │  - 打扰                - 浪费時間
  │
  └──────────────────────────→ 重要性
        不重要 ←─────────→ 重要
```

### 四象限策略

| 象限 | 策略 | 行动 |
|------|------|------|
| **第一象限**（重要且紧急） | 立即處理 | 现在就做 |
| **第二象限**（重要不紧急） | 計劃和优先 | 安排時間 |
| **第三象限**（不重要但紧急） | 委托或拒絕 | 尽量避免 |
| **第四象限**（不重要不紧急） | 刪除或限制 | 几乎不做 |

# 方法

### GTD (Getting Things Done)

```
收集 → 處理 → 组织 → 回顧 → 执行

1. 收集 (Capture):
   - 記錄所有想法和任務
   - 使用 InBox、筆記本、手机

2. 處理 (Process):
   - 清空收集箱
   - 决定每个專案的下一步
   - 2分钟原則：能立即做的马上做

3. 组织 (Organize):
   - 專案：需要多步完成的任務
   - 下一步：每个專案的具体行动
   - 等待：委派给他人
   - 将来/也许：未来可能的專案
   - 日历：有截止日期的任務

4. 回顧 (Review):
   - 每周回顧所有專案和任務
# 更新
   - 清理已完成的專案

5. 执行 (Do):
   - 根据情境選擇任務
   - 按優先級执行
   - 保持專注
```

# 方法

```
示例每日時間块：

┌──────────────────────────────────┐
│ 09:00-10:30  深度工作块    │
│ 10:30-10:50  休息            │
│ 11:00-12:30  深度工作块    │
│ 12:30-13:30  午餐            │
│ 14:00-15:00  會議/協作時間    │
│ 15:00-15:20  休息            │
│ 15:30-17:00  深度工作块    │
# 整理
└──────────────────────────────────┘

原則：
- 時間块是固定的
- 相似任務集中處理
- 休息時間也是時間块
- 保護深度工作時間
```

### 番茄工作法

```
25分钟專注 + 5分钟休息

流程：
1. 選擇任務
2. 设定计时器（25分钟）
3. 專注工作，不中断
4. 休息5分钟
5. 重复4个番茄钟
6. 长休息15-30分钟

好处：
- 明确的專注時間
- 定期休息保持精力
- 可量化的进度
```

## 📅 時間規劃工具

### 每日規劃

```
早晨15分钟規劃：

1. 回顧昨天的完成情况
2. 列出今天的所有任務
3. 识别最重要的3个任務（MIT）
4. 将任務分配到時間块
5. 预留缓冲時間
```

### 每周規劃

```
周日晚30分钟規劃：

1. 回顧上周成就和不足
2. 列出本周所有專案和任務
3. 识别本周最重要的目標
4. 将大任務分解为小步骤
5. 为每个任務分配時間和優先級
6. 预留弹性時間
```

### 每月規劃

```
月初1小时規劃：

1. 回顧上月完成情况
2. 识别本月的重点领域
3. 设定3-5个主要目標
4. 为每个目標制定計劃
5. 预留學習和成长時間
6. 设定檢查点
```

# 管理

### 帕金森定律
> "工作会自動膨胀，直到占满所有可用時間"
- 设定明确的截止時間
- 提高時間感知的紧迫感
- 避免"無限時間"心态

### 80/20 原則
> 80% 的成果来自 20% 的行动
- 识别高價值任務
- 将時間集中在最重要的任務
- 刪除或委派低價值活动

# 管理
> 時間是有限的，精力是可再生的
- 在精力最好的時間做重要任務
- 休息恢復精力
- 不要过度消耗精力
