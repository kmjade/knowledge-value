## 提醒类型 / Reminder Types
---

### 截止日期提醒

当檔案包含截止日期时自動創建提醒：

```dataview
TABLE WITHOUT ID
  file.link as "檔案",
  due_date as "截止日期",
  (date(today) - date(due_date)).days as "剩余天数",
  priority as "優先級"
FROM "0 Personals/📥 00_InBox"
WHERE due_date AND date(due_date) >= date(today)
SORT due_date ASC
```

### 定期提醒

按時間间隔触发的提醒：

```dataview
TABLE WITHOUT ID
  file.link as "任務",
  reminder_frequency as "频率",
  next_reminder as "下次提醒",
  last_completed as "上次完成"
FROM "0 Personals/📥 00_InBox"
WHERE contains(tags, "#recurring")
SORT next_reminder ASC
```

### 过期提醒

標記为过期未處理的內容：

```dataview
TABLE WITHOUT ID
  file.link as "檔案",
  due_date as "原截止日期",
  (date(today) - date(due_date)).days as "过期天数",
  action as "建議操作"
FROM "0 Personals/📥 00_InBox"
WHERE due_date AND date(due_date) < date(today) AND !contains(tags, "#completed")
SORT (date(today) - date(due_date)).days DESC
```

### 新內容提醒

InBox 中最近新增的內容（24 小时内）：

```dataview
TABLE WITHOUT ID
  file.link as "新檔案",
  file.ctime as "創建時間",
  urgency as "處理建議"
FROM "0 Personals/📥 00_InBox"
WHERE date(file.ctime) >= date(today) - dur(1 days) AND file.name != this.file.name
SORT file.ctime DESC
```

## 設置提醒 / Set Reminders
---

### 新增截止日期

在檔案 frontmatter 中新增：

```yaml
---
due_date: 2024-01-25
priority: high
---
```

### 新增定期提醒

```yaml
---
reminder_frequency: weekly
reminder_day: Monday
next_reminder: 2024-01-22
---
```

频率选项：
- `daily` - 每天
- `weekly` - 每周（可指定星期）
- `monthly` - 每月（可指定日期）
- `yearly` - 每年

### 快速新增提醒

```
/提醒: [[檔案名]] - due: 2024-01-25 - priority: high
```

### 批量新增提醒

```
/批量提醒: due: 2024-01-31 - from: "0 Personals/📥 00_InBox/0-待辦"
```

## 提醒儀表板 / Reminder Dashboard
---

### 今日提醒

```dataview
TABLE WITHOUT ID
  file.link as "任務",
  time as "時間",
  type as "类型",
  priority as "優先級"
FROM "0 Personals/📥 00_InBox"
WHERE
  (due_date AND date(due_date) = date(today)) OR
  (next_reminder AND date(next_reminder) = date(today)) OR
  (contains(tags, "#today"))
SORT priority ASC, time ASC
```

### 本周提醒

```dataview
TABLE WITHOUT ID
  dateformat(date(due_date), "E") as "星期",
  file.link as "任務",
  due_date as "日期",
  priority as "優先級"
FROM "0 Personals/📥 00_InBox"
WHERE due_date AND
      date(due_date) >= date(today) AND
      date(due_date) < date(today) + dur(7 days)
SORT due_date ASC
```

### 即将到期（3 天内）

```dataview
TABLE WITHOUT ID
  file.link as "任務",
  due_date as "截止日期",
  (date(due_date) - date(today)).days as "剩余天数",
  priority as "優先級"
FROM "0 Personals/📥 00_InBox"
WHERE due_date AND
      date(due_date) >= date(today) AND
      date(due_date) < date(today) + dur(3 days) AND
      !contains(tags, "#completed")

## 快速操作 / Quick Actions
---

### 標記完成

```
/完成: [[檔案名]]
```

自動：
- 新增 `#completed` 標籤
- 記錄完成時間
- 如果是定期提醒，設置下次提醒時間

```

```

```
# 整理
```

創建临时提醒（不关联檔案）。

### 智能提醒

```
/智能提醒

## 完成追踪 / Completion Tracking
---

### 今日完成

```dataview
TABLE WITHOUT ID
  file.link as "任務",
  completed_date as "完成時間",
  effort as "用时"
FROM "0 Personals/📥 00_InBox"
WHERE contains(tags, "#completed") AND completed_date AND date(completed_date) = date(today)
SORT completed_date DESC
```

### 本周完成

```dataview
TABLE WITHOUT ID
  dateformat(completed_date, "E") as "日期",
  length(rows) as "完成数"
FROM "0 Personals/📥 00_InBox"
WHERE contains(tags, "#completed") AND completed_date AND
      date(completed_date) >= date(today) - dur(7 days)
GROUP BY dateformat(date(completed_date), "E")
SORT completed_date DESC
```

### 完成率

```dataview
TABLE WITHOUT ID
  "本周完成率" as "指标",
  choice(length(rows) > 0,
         round((length(filter(rows, (r) => contains(r.file.tags, "#completed"))) / length(rows)) * 100, 1),
         0) as "百分比"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name
GROUP BY true
```

> - 清理过期內容
> - 新增缺失標籤
