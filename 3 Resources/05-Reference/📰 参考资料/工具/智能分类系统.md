## 分類规则 / Categorization Rules
---

### 專案特征 (Projects)

自動识别特征：
- 包含明确時間期限（`截止日期`、`deadline`、`due`）
- 包含行动动词（`完成`、`實現`、`開發`、`構建`）
- 包含階段性术语（`v1.0`、`beta`、`階段`、`里程碑`）
- 標籤包含 `#project`、`#active`、`#in-progress`

```dataview
TABLE WITHOUT ID
  file.link as "建議專案",
  confidence as "置信度",
  reason as "原因"
FROM "0 Personals/📥 00_InBox"
WHERE
  contains(tags, "#project") OR
  contains(file.name, "完成") OR
  contains(file.name, "實現") OR
  regexmatch("v\\d+\\.\\d+", file.name) OR
  contains(file.name, "deadline") OR contains(file.name, "截止") OR contains(file.name, "due")
SORT file.mtime DESC
```

### 领域特征 (Areas)

自動识别特征：
# 管理
- 包含责任领域（`健康`、`财务`、`學習`、`工作`、`家庭`）
- 標籤包含 `#area`、`#responsibility`、`#ongoing`

```dataview
TABLE WITHOUT ID
  file.link as "建議领域",
  confidence as "置信度",
  reason as "原因"
FROM "0 Personals/📥 00_InBox"
WHERE
  contains(tags, "#area") OR
# 管理
  contains(file.name, "維護") OR
  contains(file.name, "运营") OR
  contains(file.name, "健康") OR contains(file.name, "财务") OR contains(file.name, "學習") OR contains(file.name, "工作") OR contains(file.name, "家庭")
SORT file.mtime DESC
```

### 資源特征 (Resources)

自動识别特征：
# 教程
- 包含內容类型（`book`、`article`、`video`、`course`）
- 標籤包含 `#resource`、`#learn`、`#reference`、`#bookmark`

```dataview
TABLE WITHOUT ID
  file.link as "建議資源",
  confidence as "置信度",
  reason as "原因"
FROM "0 Personals/📥 00_InBox"
WHERE
  contains(tags, "#resource") OR
  contains(file.name, "學習") OR
  contains(file.name, "筆記") OR
# 教程
  contains(tags, "#book") OR
  contains(tags, "#article") OR
  contains(tags, "#video")
SORT file.mtime DESC
```

### 歸檔特征 (Archives)

自動识别特征：
- 標記为完成（`#done`、`#completed`、`#finished`）
# 修改
- 標記为废弃（`#archived`、`#obsolete`）

```dataview
TABLE WITHOUT ID
  file.link as "建議歸檔",
  confidence as "置信度",
  reason as "原因"
FROM "0 Personals/📥 00_InBox"
WHERE
  contains(tags, "#archived") OR
  contains(tags, "#completed") OR
  (file.mtime < date(today) - dur(30 days) AND !contains(tags, "#todo"))
SORT file.mtime DESC
```

### 待分類內容

```dataview
TABLE WITHOUT ID

# 修改
```

## 快速操作 / Quick Actions
---

### 自動分類

# 分析

```
/智能分類
```

系統会：
# 分析
2. 生成分類建議列表
3. 标注置信度（高/中/低）
4. 提供一键分類选项

### 批量分類

```
/批量分類: 1 Projects
```

将所有高置信度的專案建議移动到 Projects 資料夾。

### 確認分類

```
/確認分類: [[檔案名]]
```

对单个檔案执行分類操作。

## 分類置信度 / Confidence Levels

| 级别 | 阈值 | 建議 |
|------|------|------|
| 高 | > 80% | 可以直接分類 |
| 中 | 50-80% | 需要人工確認 |
| 低 | < 50% | 建議手動分類 |

---

> [!tip] 提升分類准确率
> - 新增明确的標籤
> - 使用規範化的檔案命名
> - 包含關鍵词（如 `專案`、`學習`、`參考`）
