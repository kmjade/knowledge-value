## 概览统计 / Overview Statistics
---

### InBox 状态概览

```dataview
TABLE WITHOUT ID
  length(rows) as "总文件数",
  sum(map(rows, (r) => length(r.file.tags))) as "总标签数",
  round(7, 1) as "平均周处理数"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name
GROUP BY true
```

### 文件类型分布

```dataview
TABLE WITHOUT ID
  file_type as "文件类型",
  length(rows) as "数量"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name
FLATTEN choice(contains(file.folder, "0-待办"), "待办", choice(contains(file.folder, "1-链接"), "链接", choice(contains(file.folder, "2-素材"), "素材", "根目录"))) as file_type

```

## 时间分析 / Time Analysis
---

### 每周新增文件

```dataview
TABLE WITHOUT ID
  dateformat(file.ctime, "yyyy-Www") as "周",
  length(rows) as "新增文件数"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name
GROUP BY dateformat(file.ctime, "yyyy-Www")
SORT dateformat(file.ctime, "yyyy-Www") DESC
LIMIT 12
```

### 最近活跃度

```dataview
TABLE WITHOUT ID
  dateformat(file.mtime, "yyyy-MM-dd") as "日期",
  length(rows) as "修改文件数"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name
GROUP BY dateformat(file.mtime, "yyyy-MM-dd")
SORT dateformat(file.mtime, "yyyy-MM-dd") DESC
```

### 处理速度分析

```dataview
TABLE WITHOUT ID
  "平均处理时间" as "指标",
  choice(length(rows) > 0,
         round(sum(map(rows, (r) => (r.file.mtime - r.file.ctime).days)) / length(rows), 1),
         0) as "天数"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name AND file.mtime != file.ctime
GROUP BY true
```

## 标签分析 / Tag Analysis

### 使用最多的标签

```dataview
TABLE WITHOUT ID
  tag as "标签",
  length(rows) as "使用次数"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name
FLATTEN file.tags as tag
WHERE !contains(tag, "#inbox")
GROUP BY tag
SORT length(rows) DESC
LIMIT 15
```

### PARA 分类倾向

```dataview
TABLE WITHOUT ID
  para_type as "分类倾向",
  length(rows) as "文件数"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name
FLATTEN choice(any(file.tags, (t) => contains(t, "project")), "Projects", choice(any(file.tags, (t) => contains(t, "area")), "Areas", choice(any(file.tags, (t) => contains(t, "resource")), "Resources", choice(any(file.tags, (t) => contains(t, "archive")), "Archives", "未分类")))) as para_type
GROUP BY para_type
```

## 健康指标 / Health Metrics

### 需要关注的文件

```dataview
TABLE WITHOUT ID
  file.link as "文件",
  file.mtime as "最后修改",
  round((date("now") - file.mtime).days) as "待处理天数",
  health_status as "状态"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name AND
      (date("now") - file.mtime).days > 3
FLATTEN choice((date("now") - file.mtime).days > 14, "⚠️ 需要处理", choice((date("now") - file.mtime).days > 7, "⚡ 应关注", "✅ 正常")) as health_status
SORT (date("now") - file.mtime).days DESC
LIMIT 20
```

### 清理建议

```dataview
TABLE WITHOUT ID
  recommendation as "建议",
  length(rows) as "文件数"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name
FLATTEN choice((date("now") - file.mtime).days > 14, "处理过期文件", choice(length(file.tags) < 2, "整理零散标签", choice((date("now") - file.mtime).days > 7, "应关注文件", "正常文件"))) as recommendation
GROUP BY recommendation
SORT length(rows) DESC
```

| 建议类型 | 优先级 | 说明 |
|---------|--------|------|
| 处理过期文件 | 🔴 高 | 超过 14 天未处理的文件 |
| 整理零散标签 | 🟡 中 | 使用次数 < 2 的标签 |
| 清理重复内容 | 🟡 中 | 标题相似的文件 |
| 归档完成内容 | 🟢 低 | 标记为 #completed 的文件 |

## 内容质量 / Content Quality

### 文件完整性检查

```dataview
TABLE WITHOUT ID
  file.link as "文件",
  quality_score as "质量评分",
  issues as "问题"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name
FLATTEN (10 + choice(length(file.tags) > 0, 20, 0) + choice(file.size > 100, 30, 0) + choice(file.size > 500, 40, 0)) as quality_score
FLATTEN choice(length(file.tags) = 0, "缺少标签", choice(file.size < 100, "内容过短", "无问题")) as issues
SORT quality_score DESC
LIMIT 20
```

| 评分 | 说明 | 改进建议 |
|------|------|---------|
| 优秀 (90-100) | 有标题、标签、内容 | 保持 |
| 良好 (70-89) | 有标题、标签，内容简短 | 补充内容 |
| 一般 (50-69) | 有标题，缺少标签 | 添加标签 |
| 需改进 (<50) | 缺少标题或标签 | 完善元数据 |

### 内容深度分析

```dataview
TABLE WITHOUT ID
  depth_level as "深度",
  length(rows) as "文件数"
FROM "0 Personals/📥 00_InBox"
WHERE file.name != this.file.name
FLATTEN choice(file.size < 100, "简单记录", choice(file.size < 500, "标准笔记", choice(file.size < 2000, "详细文档", "综合资料"))) as depth_level
GROUP BY depth_level
```

| 深度级别 | 特征 | 占比 |
|---------|------|------|
| 简单记录 | < 100 字，无结构 | - |
| 标准笔记 | 100-500 字，有段落 | - |
| 详细文档 | > 500 字，有标题 | - |
| 综合资料 | 有引用、链接、分类 | - |
