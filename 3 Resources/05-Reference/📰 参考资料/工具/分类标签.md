> [!note] 分类标签系统 / Tag Classification System
> 用于管理 InBox 内容的分类标签，支持快速筛选和自动分类

## 标签体系
---

### PARA 分类标签

```dataview
LIST WITHOUT id
WHERE contains(file.name, "#para-")
```

### 状态标签

```dataview
LIST WITHOUT id
WHERE contains(file.name, "#status-")
```

### 优先级标签

```dataview
LIST WITHOUT id
WHERE contains(file.name, "#priority-")
```

### 自定义标签

用户可以创建自定义标签用于更精细的分类：

| 标签类别 | 示例 | 用途 |
|-----------|------|------|
| 内容类型 | #article, #book, #video | 区分资源类型 |
| 项目关联 | #project-公众号, #project-网站 | 按项目分组 |
| 领域关联 | #area-健康, #area-财务 | 按领域分组 |
| 情绪标签 | #灵感, #问题, #idea | 标记思维状态 |
| 行动标签 | #today, #this-week | 标记需要行动的事项 |

## 快速操作
---

### 添加标签

在 InBox 文件中快速添加标签：

```markdown
- [ ] 待办内容 #todo #status-待处理
- [ ] 紧急内容 #urgent #priority-high
- [ ] 待定内容 #later #status-待确定
```

### 标签查询

查询特定标签下的所有内容：

```dataview
TABLE WITHOUT id file.link, tags as "标签"
FROM ""
WHERE contains(tags, "#tag")
```

> [!tip] 标签使用建议
> - 使用层次化的标签系统
> - 每个文件添加 2-3 个标签
> - 定期清理不常用的标签
> - 标签使用英语单词，便于搜索
