> [!note] 歸檔触发系統 / Archive Trigger System
> 自動或手動触发 PARA 內容的歸檔流程

## 触发条件

### 自動触发条件

| 条件 | 說明 | 處理方式 |
|------|------|---------|
| 專案完成 | Projects 中所有任務完成时 | 移动到 Archives |
| 專案取消 | status 设为 cancelled 时 | 移动到 Archives |
| 过期未處理 | InBox 中超过 7 天未處理 | 標記需要處理 |
| 定期歸檔 | 每周自動歸檔完成的 InBox 內容 | 按周創建歸檔記錄 |

### 手動触发

```
┌────────────────────────────────────────┐
│  📥 触发歸檔                      │
│  ↓ 選擇歸檔类型                     │
│  ┌────────────────────────────────┐   │
│  │  [ ] 專案完成歸檔                 │
│  │  [ ] 手動歸檔                   │
│  │  [ ] 批量歸檔                   │
│  │  [ ] 清空 InBox                  │
│  └────────────────────────────────┘   │
│  ↓ 創建歸檔記錄                     │
│  ┌────────────────────────────────┐   │
│  │  ✅ 歸檔記錄已創建            │
│  └────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

## 歸檔記錄格式

### 歸檔專案

```
---
title: "2024-W1-歸檔記錄"
archived: 2024-01-18
original_category: projects
items:
  - [[1 Projects/測試專案]]
reason: 專案完成
notes: 專案于 2024-01-15 完成
---
```

### 歸檔資源

```
---
title: "2024-W1-歸檔記錄"
archived: 2024-01-18
original_category: resources
items:
  - [[3 Resources/測試資源]]
reason: 已處理，不再需要保留
notes: 資訊已提取到筆記中
---
```

## 歸檔回顧

創建歸檔回顧命令 `/para-歸檔回顧`：
# 查看
- 評估歸檔策略
- 恢復已歸檔的內容

---

# 管理
> - 歸檔前提取有價值內容到 PARA
> - 記錄歸檔原因便于未来尋找
> - 定期回顧歸檔內容，清理無用檔案
> - 歸檔完成後清理 InBox 原內容
