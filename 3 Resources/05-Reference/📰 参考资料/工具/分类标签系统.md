# 管理

## 標籤体系
---

### PARA 分類標籤

```dataview
LIST WITHOUT id
WHERE contains(file.name, "#para-")
```

### 狀態標籤

```dataview
LIST WITHOUT id
WHERE contains(file.name, "#status-")
```

### 優先級標籤

```dataview
LIST WITHOUT id
WHERE contains(file.name, "#priority-")
```

### 自定义標籤

使用者可以創建自定义標籤用于更精细的分類：

| 標籤类别 | 示例 | 用途 |
|-----------|------|------|
| 內容类型 | #article, #book, #video | 区分資源类型 |
| 專案关联 | #project-公众号, #project-網站 | 按專案分组 |
| 领域关联 | #area-健康, #area-财务 | 按领域分组 |
| 情绪標籤 | #灵感, #問題, #idea | 標記思维狀態 |
| 行动標籤 | #today, #this-week | 標記需要行动的事項 |

## 快速操作
---

### 新增標籤

在 InBox 檔案中快速新增標籤：

```markdown
- [ ] 待辦內容 #todo #status-待處理
- [ ] 紧急內容 #urgent #priority-high
- [ ] 待定內容 #later #status-待確定
```

### 標籤查詢

查詢特定標籤下的所有內容：

```dataview
TABLE WITHOUT id file.link, tags as "標籤"
FROM ""
WHERE contains(tags, "#tag")
```
