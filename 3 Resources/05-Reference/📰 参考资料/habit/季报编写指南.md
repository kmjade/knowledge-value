# 季报编写指南

> 复盘每一季，连接月与年。

---

## 为什么写季报？

季报是连接月报和年报的重要桥梁：

| 作用 | 说明 |
|------|------|
| **中期检视** | 在年度中途进行深度复盘 |
| **目标校准** | 检查季度目标完成情况 |
| **趋势分析** | 发现跨月的模式 |
| **计划调整** | 根据数据调整下季度计划 |
| **承上启下** | 总结过去，展望未来 |
| **年中/年终复盘** | Q2和Q4是重要的里程碑 |

---

## 何时写季报？

### 推荐时间

| 时间 | 场景 | 优点 |
|------|------|------|
| **季度末最后一天** (19:00-21:00) | 复盘整季，规划下季 | 数据完整，衔接流畅 |
| **季度初第一天** (9:00-11:00) | 开启新季度 | 充满能量，方向清晰 |
| **Q2结束** (6月30日) | 年中复盘 | 重要里程碑 |
| **Q4结束** (12月31日) | 年终复盘 | 年度收官 |

### 季度划分

| 季度 | 月份 | 天数 | 特点 |
|------|------|------|------|
| **Q1** | 1-3月 | 90/91天 | 开局之季，定调全年 |
| **Q2** | 4-6月 | 91天 | 年中检视，调整方向 |
| **Q3** | 7-9月 | 92天 | 加速推进，保持动力 |
| **Q4** | 10-12月 | 92天 | 年终冲刺，年度收官 |

### 核心原则

1. **固定时间** - 每季度固定一天进行
2. **充足时间** - 1-2小时完成
3. **数据驱动** - 基于3个月报数据
4. **连接年度** - 与年度目标对齐

## 季报结构

```
1. 本季概览       → 一页纸了解整季
2. 年度目标进度   → 对照年度计划
3. Projects      → 项目进展与评估
4. Areas         → 领域状态与长期目标
5. Resources     → 学习与资源积累
6. 反思与收获    → 深度思考
7. 数据分析      → 量化自我
8. 人际关系      → 连接与关系
9. 财务总结      → 财务状况分析
10. 下季度计划   → 明确行动
11. 年度复盘     → 年中/年末深度复盘
```

### 各模块说明

#### 1. 本季概览
- **关键指标** - 与上季度对比
- **本季主题** - 用一个词概括本季
- **关键成果** - 最重要的4个成就

#### 2. 年度目标进度
- **年度目标对照** - 4个季度的累计进度
- **本季度目标** - 计划 vs 实际
- **达成分析** - 完成/超额/未达

#### 3. Projects
- **本季度完成** - 项目清单和贡献
- **进行中** - 当前进度和下季度计划
- **本季度启动** - 新项目状态
- **阻塞/风险** - 问题与应对

#### 4. Areas
- **本季度关注** - 各领域的进展
- **年度目标进度** - 各领域年度目标
- **待改进** - 发现的问题

#### 5. Resources
- **新增资源** - 收藏和积累
- **学习完成** - 本季度学习成果
- **技能提升** - 能力增长

#### 6. 反思与收获
- **本季亮点** - 做对了什么
- **改进空间** - 可以如何优化
- **学习心得** - 认知升级
- **困惑与突破** - 成长卡点
- **认知升级** - 思维转变

#### 7. 数据分析
- **习惯养成** - 季度习惯追踪
- **季度对比** - 4个季度趋势
- **时间分配** - 时间和精力分配

#### 8. 人际关系
- **本季度连接** - 关系质量变化
- **季度变化** - 4个季度的关系趋势
- **新关系** - 新建立的关系

#### 9. 财务总结
- **季度收支** - 财务概况
- **投资回报** - 学习/成长投资
- **资产变化** - 净资产变化

#### 10. 下季度计划
- **季度主题** - 新季度关键词
- **季度目标** - 具体可衡量
- **重点项目** - 关键里程碑
- **月度分解** - 每月重点任务

#### 11. 年度复盘（Q2/Q4特有）
- **上半年回顾**（Q2） - 对照年初期望
- **下半年展望**（Q2） - 下半年目标
- **年终总结**（Q4） - 年度复盘

## 季报与其他复盘的关系

```
日报 → 周报 → 月报 → 季报 → 年报
          ↓              ↓
       每周          每季度
       汇总          中期检视
```

### 数据流向

| 月报内容 | 季报提取 |
|----------|----------|
| 每月任务完成 | 季度任务汇总 |
| 每月反思 | 本季亮点/改进 |
| 每月数据 | 季度趋势分析 |
| 每月目标 | 季度目标达成 |

### 最佳实践

**月报记录时**：
- 保持3篇月报完整
- 标注关键成果和问题
- 保持数据一致性

**季报生成时**：
- 回顾3篇月报
- 提取关键数据
- 分析季度趋势
- 提炼季度洞察

---

## 最佳实践

### ✅ 推荐做法

- **季度末/初** - 数据最完整/能量最足
- **基于月报数据** - 准确可靠
- **充足时间** - 1-2小时深度复盘
- **诚实复盘** - 不回避问题
- **设定下季主题** - 聚焦重点
- **连接年度目标** - 保持对齐
- **Q2/Q4重点** - 年中/年终深度复盘

### ❌ 避免做法

- **拖延到下季初** - 数据遗忘
- **只看最后1月** - 以偏概全
- **流于形式** - 没有深度
- **只报喜不报忧** - 失去意义
- **写完不规划** - 流于表面

## 常见问题

### Q1: 季报和月报内容重复怎么办？

**A**: 季报是月报的提炼和升华
- 月报：记录细节
- 季报：萃取精华 + 趋势分析 + 目标对照
- 相同内容用"详见X月月报"概括

### Q2: 季报太长了怎么办？

**A**: 精简原则
- 成果不超过5项
- 问题不超过3个
- 计划不超过5项
- 总字数控制在 2000-3500 字

### Q3: 如何从季报看趋势？

**A**: 季度对比
- **横向对比** - 与上季度对比
- **季度趋势** - 4个季度趋势线
- **归因分析** - 为什么会有这个趋势
- **预测未来** - 趋势会持续吗？

### Q4: Q2和Q4的季报有什么不同？

**A**: Q2和Q4有特殊的年度复盘模块

| 模块 | Q1/Q3 | Q2 | Q4 |
|------|-------|----|----|
| 年度复盘 | 无 | 上半年回顾 + 下半年展望 | 年终总结 + 下年规划 |
| 篇幅 | 标准 | 加长 | 加长 |
| 重要性 | 标准 | **年中里程碑** | **年终收官** |

### Q5: 季报要给别人看吗？

**A**: 视情况而定
- **个人成长** - 重点写给自己看
- **团队同步** - 突出项目成果
- **上级汇报** - 调整结构和侧重点

## 4季复盘挑战

尝试连续4个季度写季报：

| 季度 | 时间 | 目标 | 重点 |
|------|------|------|------|
| **Q1** | 1-3月 | 建立习惯 | 季度复盘模式 |
| **Q2** | 4-6月 | 数据积累 | **年中深度复盘** |
| **Q3** | 7-9月 | 趋势分析 | 分析原因模式 |
| **Q4** | 10-12月 | 优化迭代 | **年终全面复盘** |

完成一年4篇季报后，撰写年度总结。

## 复盘周期对比

| 维度 | 季报 | 月报 | 周报 | 年报 |
|------|------|------|------|------|
| **频率** | 每季 | 每月 | 每周 | 每年 |
| **时间** | 1-2小时 | 30-60分钟 | 15-30分钟 | 2-3小时 |
| **深度** | 中深 | 中 | 浅 | 深 |
| **范围** | 本季 | 本月 | 本周 | 全年 |
| **作用** | 中期检视 | 趋势洞察 | 进度追踪 | 全局规划 |
| **里程碑** | Q2/Q4重要 | 标准 | 标准 | 年底 |

---

> [!tip] 小提示
> 季报是年度的1/4检视。
> 不要只做汇总，要做趋势分析；
> 不要只写事实，要写季度洞察；
> 连接月度与年度，形成完整复盘链。
