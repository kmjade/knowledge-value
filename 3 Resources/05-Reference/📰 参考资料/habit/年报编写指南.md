# 年报编写指南

> 回顾每一年，展望下一程。

---

## 为什么写年报？

年报是深度复盘和长期规划的最佳时机：

| 作用 | 说明 |
|------|------|
| **全局视角** | 从年度高度审视人生 |
| **趋势洞察** | 发现多年跨度的模式 |
| **目标校准** | 检查年度和长期目标 |
| **成就盘点** | 积累成功的资产 |
| **经验萃取** | 从经验中学习 |
| **方向调整** | 根据数据调整战略 |
| **人生里程碑** | 记录重要时间节点 |

---

## 何时写年报？

### 推荐时间

| 时间 | 场景 | 优点 |
|------|------|------|
| **12月31日** | 年末复盘 | 数据完整，记忆清晰 |
| **1月1日** | 新年规划 | 充满能量，展望未来 |
| **1月第1周** | 深度思考 | 时间充裕，思考深入 |

### 核心原则

1. **固定时间** - 每年固定一天进行
2. **充足时间** - 至少2-3小时
3. **数据驱动** - 基于12个月报数据
4. **深度复盘** - 不只是汇总，要分析
5. **连接长期** - 与3-5年愿景对齐

## 年报结构

```
1. 年度概览       → 一页纸了解全年
2. 目标达成       → 年度目标完成情况
3. Projects      → 项目回顾与经验
4. Areas         → 各领域状态评估
5. Resources     → 学习与资源积累
6. 深度反思      → 亮点与改进
7. 数据分析      → 量化自我
8. 人际关系      → 连接与关系
9. 财务总结      → 财务状况分析
10. 年度之最     → 有趣的记录
11. 下年计划     → 新年规划
12. 十年愿景     → 长期视角
```

### 各模块说明

#### 1. 年度概览
- **关键指标** - 与去年对比
- **年度主题** - 用一个词/短语概括全年
- **年度关键词** - 3个核心词
- **关键成果** - 最重要的5个成就

#### 2. 年度目标达成
- **年初目标回顾** - 对照检查
- **季度分解** - 各季度完成情况
- **达成分析** - 超额/按计划/未达预期

#### 3. Projects
- **完成项目** - 清单和贡献度
- **进行中项目** - 进度和下年计划
- **新启动项目** - 预期影响
- **项目经验** - 做对了什么/需要改进

#### 4. Areas
- **年度进展** - 各领域变化
- **长期目标** - 3年/5年目标进度
- **深度评估** - 健康/职业/关系/财务/成长

#### 5. Resources
- **新增资源** - 收藏和积累
- **学习完成** - 年度学习成果
- **技能提升** - 能力增长
- **知识积累** - 笔记和理解深化

#### 6. 深度反思
- **年度亮点** - 最成功/骄傲/突破
- **年度改进** - 最遗憾/教训/改进
- **年度学习** - 认知升级/思维转变
- **年度困惑** - 已解决/待解决

#### 7. 数据分析
- **习惯统计** - 全年习惯追踪
- **时间分配** - 时间和精力分配
- **生产力评估** - 效率变化

#### 8. 人际关系
- **年度连接** - 关系质量变化
- **重要关系** - 投入与收获
- **新关系** - 新建立的关系

#### 9. 财务总结
- **年度收支** - 财务概况
- **投资回报** - 学习/成长投资
- **资产变化** - 净资产变化

#### 10. 年度之最
- **有趣的记录** - 最成功/突破/开心等

#### 11. 下年计划
- **年度主题** - 新年关键词
- **年度目标** - 具体可衡量
- **Q1计划** - 近期计划
- **习惯目标** - 新年习惯

#### 12. 十年愿景
- **长期目标对照** - 愿景进度
- **人生阶段评估** - 各维度状态

## 使用模板

1. 按 `Ctrl/Cmd + P` 打开命令面板
2. 搜索 "Templater: Insert Template"

2. 复制全部内容
3. 创建新笔记，粘贴内容
4. 修改占位符

## 年报与月报的关系

```
日报 → 周报 → 月报 → 季报 → 年报 → 3年总结 → 10年总结
```

### 数据流向

| 月报内容 | 年报提取 |
|----------|----------|
| 每月任务完成 | 年度任务汇总 |
| 每月反思 | 年度亮点/改进 |
| 每月数据 | 年度趋势分析 |
| 每月目标 | 年度目标达成 |

### 最佳实践

**月报记录时**：
- 保持12篇月报完整
- 标注关键成果和问题
- 保持数据一致性

**年报生成时**：
- 回顾12篇月报
- 提取关键数据
- 分析年度趋势
- 提炼年度洞察

---

## 最佳实践

### ✅ 推荐做法

- **年末/年初** - 数据最完整/能量最足
- **基于月报数据** - 准确可靠
- **充足时间** - 2-3小时深度复盘
- **诚实复盘** - 不回避问题
- **设定新年主题** - 聚焦重点
- **连接长期愿景** - 保持对齐
- **保存数据** - 积累长期资产

### ❌ 避免做法

- **拖延到2月** - 数据遗忘
- **只看最后几月** - 以偏概全
- **流于形式** - 没有深度
- **过度乐观** - 高估成就
- **过度悲观** - 低估进步
- **写完不规划** - 流于表面

## 常见问题

### Q1: 第一年没有月报怎么写年报？

**A**:
1. 基于现有数据（日报、周报）
2. 回顾日历和重要事件
3. 诚实在数据缺失处标注
4. 下年确保月报完整

### Q2: 年报太长了怎么办？

**A**: 精简原则
- 成果不超过10项
- 反思不超过5点
- 计划不超过5项
- 总字数控制在 3000-5000 字

### Q3: 如何从年报看长期趋势？

**A**: 多年对比
- **横向对比** - 与去年对比
- **多年趋势** - 3-5年趋势线
- **归因分析** - 为什么会有这个趋势
- **预测未来** - 趋势会持续吗？

### Q4: 年报要给别人看吗？

**A**: 视情况而定
- **个人成长** - 重点写给自己看
- **团队分享** - 突出团队成果
- **家庭汇报** - 调整侧重点

## 10年复盘挑战

尝试连续10年写年报：

| 阶段 | 年份 | 目标 | 重点 |
|------|------|------|------|
| **第一阶段** | 第1-2年 | 建立习惯 | 每年坚持完成 |
| **第二阶段** | 第3-4年 | 数据积累 | 开始趋势分析 |
| **第三阶段** | 第5-6年 | 深度复盘 | 分析原因模式 |
| **第四阶段** | 第7-8年 | 优化迭代 | 调整和提升 |
| **第五阶段** | 第9-10年 | 十年总结 | 里程碑回顾 |

完成10年年报后，撰写十年总结报告。

## 年报 vs 其他复盘

| 维度 | 日报 | 周报 | 月报 | 年报 |
|------|------|------|------|------|
| **频率** | 每天 | 每周 | 每月 | 每年 |
| **时间** | 5-10分钟 | 15-30分钟 | 30-60分钟 | 2-3小时 |
| **深度** | 记录 | 汇总 | 分析 | 深度复盘 |
| **范围** | 当天 | 本周 | 本月 | 全年 |
| **作用** | 习惯养成 | 进度追踪 | 趋势洞察 | 全局规划 |

---

> [!tip] 小提示
> 年报是人生的年度检视。
> 不要只做汇总，要做深度分析；
> 不要只写事实，要写人生洞察；
> 不要只顾眼前，要连接长期愿景。
