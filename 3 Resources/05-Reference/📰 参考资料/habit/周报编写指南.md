## 为什么写周报？

周报是連接日报和月度复盘的重要桥梁：

| 作用 | 說明 |
|------|------|
| **宏观视角** | 从每日细节跳脱出来，看清整體進展 |
# 分析
| **重点聚焦** | 识别本周最重要的3件事 |
| **計劃優化** | 根据周數據调整下周計劃 |
| **沟通工具** | 向上汇报、團隊同步的好帮手 |

---

## 何时写周报？

### 推荐時間

| 時間 | 場景 | 优点 |
|------|------|------|
| **周日晚上** (20:00-21:00) | 回顧本周，規劃下周 | 有完整數據，衔接新一周 |
| **周一早晨** (8:00-9:00) | 开启新一周 | 充满能量，方向清晰 |

### 核心原則

1. **固定時間** - 养成稳定的周回顧习惯
2. **數據驅動程式** - 基于日报數據，而非凭印象
3. **简洁高效** - 15-30分钟完成
4. **重点突出** - 聚焦關鍵成果和關鍵問題

---

## 周报结构

```
1. 本周概览       → 一页纸了解整周
2. Projects      → 專案進展与問題
3. Areas         → 领域關注与改进
4. Resources     → 資源与學習動態
5. 反思与收获    → 深度思考
6. 下周計劃      → 明确行动
7. 數據追踪      → 量化自我
```

### 各模块說明

#### 1. 本周概览
- **關鍵指标** - 完成任務数、專注时长等
- **本周主題** - 用一句话概括本周焦点
- **關鍵成果** - 最重要的3个成就

#### 2. Projects
- **本周完成** - 列清單，勾选狀態
- **進行中** - 当前进度百分比
# 分析

#### 3. Areas
- **本周關注** - 各领域的進展
- **待改进** - 發現的問題

#### 4. Resources
# 收藏
# 知識
- **待學習** - 學習計劃清單

#### 5. 反思与收获
- **本周亮点** - 做对了什么
- **改进空間** - 可以如何優化
- **學習心得** - 认知升級
- **待解決問題** - 困惑与卡点

#### 6. 下周計劃
- **高優先級** - 必须完成
- **中優先級** - 尽量完成
- **低優先級** - 有空就做

#### 7. 數據追踪
- **习惯养成** - 周习惯完成率
- **目標进度** - 长期目標進展

## 周报与日报的关系

```
日报（每日） → 周报（每周） → 月报（每月） → 年报（每年）
```

### 數據流向

| 日报內容 | 周报提取 |
|----------|----------|
| 每日完成任務 | 本周完成清單 |
| 每日反思 | 本周亮点/改进 |
| 明日計劃 | 下周計劃 |
| 情绪狀態 | 本周狀態趨勢 |

### 最佳實踐

**日报記錄时**：
- 使用標籤分類（#project, #area, #resource）
- 填寫 `status` 字段（in-progress, completed）
- 記錄 `due_date` 便于追踪

**周报生成时**：
- 从日报提取结构化數據
- 關注趨勢和模式
- 提炼關鍵洞察

---

## 最佳實踐

### ✅ 推荐做法

- **周日生成** - 數據完整，時間充裕
- **基于日报** - 减少记忆偏差
- **量化追踪** - 用數據说话
- **連接PARA** - 同步專案狀態
- **诚实复盘** - 不回避問題
- **设定下周** - 明确優先級

### ❌ 避免做法

- **拖延到周一** - 數據可能遗忘
- **凭印象写** - 不准确
- **只报喜不报忧** - 失去复盘意义
- **流水账** - 没有重点
- **写完就忘** - 不对照执行

## 常见問題

### Q1: 周报和日报內容重复怎么办？

**A**: 周报是日报的提炼和升华
- 日报：記錄细节
- 周报：萃取精华
- 相同內容用"详见X月X日日报"概括

### Q2: 一周没写几篇日报怎么写周报？

**A**:
1. 诚实标注"數據不完整"
2. 重点写有記錄的那几天
3. 反思为什么没写日报
4. 設置下周改进目標

### Q3: 周报太长了怎么办？

**A**: 精简原則
- 成果不超过3项
- 問題不超过3个
- 計劃不超过5项
- 总字数控制在 800-1500 字

### Q4: 可以用周报向上级汇报吗？

**A**: 可以，但需要调整
- 增加业务指标
- 减少个人反思
- 突出團隊贡献
- 明确下周目標
