# 指南

> 复盘每一月，成长每一步。

---

## 为什么写月报？

月报是深度复盘的最佳周期：

| 作用 | 說明 |
|------|------|
| **趨勢洞察** | 發現跨周的模式和长期趨勢 |
| **目標校准** | 檢查月度目標完成情况 |
| **习惯检验** | 評估习惯养成进度 |
# 分析
| **战略调整** | 根据數據调整方向 |
| **成长記錄** | 积累长期數據资产 |

---

## 何时写月报？

### 推荐時間

| 時間 | 場景 | 优点 |
|------|------|------|
| **月末最后一天** (20:00-22:00) | 复盘整月，規劃下月 | 數據完整，衔接流畅 |
| **月初第一天** (9:00-11:00) | 开启新月 | 充满能量，方向清晰 |

### 核心原則

1. **固定時間** - 每月固定一天進行
2. **數據驅動程式** - 基于周报數據
# 分析
4. **連接长期** - 与年度目標对齐

---

## 月报结构

```
1. 本月概览       → 一页纸了解整月
2. Projects      → 專案進展与評估
3. Areas         → 领域狀態与长期目標
4. Resources     → 學習与資源积累
5. 反思与收获    → 深度思考
# 分析
7. 下月計劃      → 明确行动
8. 主題复盘      → 月初目標回顧
```

### 各模块說明

#### 1. 本月概览
- **關鍵指标** - 与上月對比
- **本月主題** - 用一个词概括本月
- **關鍵成果** - 最重要的3个成就

#### 2. Projects
- **本月完成** - 專案清單和贡献
- **進行中** - 当前进度
- **新啟動** - 新專案計劃
# 分析

#### 3. Areas
- **本月關注** - 各领域的進展
- **长期目標** - 年度目標进度
- **待改进** - 發現的問題

#### 4. Resources
# 收藏
- **學習完成** - 本月學習成果
- **技能提升** - 能力增长

#### 5. 反思与收获
- **本月亮点** - 做对了什么
- **改进空間** - 可以如何優化
- **學習心得** - 认知升級
- **困惑与突破** - 成长卡点
- **认知升級** - 思维转变

# 分析
- **习惯养成** - 月度习惯追踪
- **目標達成** - 目標完成率
- **時間分配** - 時間和精力分配

#### 7. 下月計劃
- **優先級高** - 必须完成的任務
- **優先級中** - 計劃完成的任務
- **新專案** - 計劃啟動的專案
- **习惯目標** - 下月习惯目標

#### 8. 主題复盘
- **月初目標回顧** - 对照檢查
- **下月主題设定** - 聚焦重点

## 月报与周报的关系

```
日报 → 周报 → 月报 → 季报 → 年报
```

### 數據流向

| 周报內容 | 月报提取 |
|----------|----------|
| 每周任務完成 | 月度任務彙總 |
| 每周反思 | 本月亮点/改进 |
| 每周計劃 | 下月計劃 |
# 分析

### 最佳實踐

**周报記錄时**：
- 完成周报，勿拖延
- 标注關鍵成果和問題
- 保持數據一致性

**月报生成时**：
- 回顧4篇周报
- 提取關鍵數據
# 分析
- 提炼洞察

---

## 最佳實踐

### ✅ 推荐做法

- **月末最后一天** - 數據最完整
- **基于周报數據** - 准确可靠
# 分析
- **诚实复盘** - 不回避問題
- **设定下月主題** - 聚焦重点
- **連接年度目標** - 保持对齐

### ❌ 避免做法

- **拖延到下月初** - 數據遗忘
- **只看最后一周** - 以偏概全
- **流于形式** - 没有深度
- **只报喜不报忧** - 失去意义
- **写完不执行** - 流于表面

## 进阶技巧

### 月报彙總

从月报生成季度/年度報告：

```
本季度主題：
├── 本月完成：
│   ├── 第1月：任務X
│   ├── 第2月：任務X
│   └── 第3月：任務X
├── 專案進展：
│   ├── 專案A：X% → X% → X%
│   └── 專案B：X% → X% → X%
├── 习惯养成：
│   ├── 写日报：X/90天
│   ├── 写周报：X/12周
# 整理
├── 關鍵學習：
│   ├── 學習主題1
│   ├── 學習主題2
│   └── 學習主題3
└── 下季度重点：
    ├── 重点1
    └── 重点2
```

### 年度回顧

从12篇月报生成年度報告：

```
年度總結 {{year}}：
├── 年度目標達成：
│   ├── 目標1：X%完成
│   ├── 目標2：X%完成
│   └── 目標3：X%完成
├── 年度成果：
│   ├── 專案完成：X个
│   ├── 技能提升：X项
│   └── 习惯养成：X个
├── 年度之最：
│   ├── 最成功的月份：X月
│   ├── 最大突破：XXX
│   └── 最大學習：XXX
├── 經驗萃取：
│   ├── 做对了什么
│   └── 需要改进什么
└── 下年度規劃：
    ├── 年度主題：XXX
    └── 關鍵目標：X个
```

## 常见問題

### Q1: 周报没写齐怎么写月报？

**A**:
1. 诚实标注"數據不完整"
# 分析
3. 反思为什么周报缺失
4. 下月改进周报习惯

### Q2: 月报太长了怎么办？

**A**: 精简原則
- 成果不超过5项
- 問題不超过3个
- 計劃不超过5项
- 总字数控制在 1500-2500 字

### Q3: 如何从月报看趨勢？

# 分析
- **横向對比** - 与上月對比
- **纵向對比** - 与歷史同期對比
- **趨勢判斷** - 是在进步还是退步？
# 分析

### Q4: 月报要给别人看吗？

**A**: 视情况而定
- **个人复盘** - 重点写给自己看
- **團隊同步** - 突出成果和計劃
- **向上汇报** - 调整结构和侧重点
