# 指南

> 記錄每一天，成长每一周。

---

## 为什么写日报？

日报是个人成长的重要工具，帮助您：

- **保持專注** - 每天明确最重要的任務
- **追踪進展** - 可视化您的进步轨迹
- **深度反思** - 从經驗中學習，持續改进
- **建立习惯** - 通過规律記錄培养自律

---

## 何时写日报？

### 推荐時間

| 時間 | 場景 |
|------|------|
| **晚间** (21:00-22:00) | 回顧一天完成的事項，規劃明日 |
| **次日早晨** (7:00-8:00) | 回顧昨日，總結今日計劃 |

### 核心原則

1. **固定時間** - 選擇适合自己的时段，坚持执行
2. **简短精炼** - 5-10分钟完成，避免负担过重
3. **真实記錄** - 如实反映情况，不夸大不隐瞒
4. **持續行动** - 每天記錄，哪怕只有一句话

---

## 日报结构

### 核心模块

```
1. 今日概览        → 快速了解今日整體狀態
2. 今日完成        → 記錄完成的任務和成果
3. 待辦事項        → 追踪未完成事項和明日計劃
4. 反思与收获      → 深度思考，萃取經驗
# 知識庫
```

### 各模块說明

#### 1. 今日概览
- 基礎統計數據（筆記編輯数、完成任務数等）
- 今日情绪/狀態評估
- 快速自检今日精力状况

#### 2. 今日完成
- **主要成果** - 最重要的3件事
- **已完成任務** - 勾选清單中的完成项
- 成就感来源，增强动力

#### 3. 待辦事項
- **今日未完成** - 延续到明天
# 排序
- 避免任務遗漏，保持连续性

#### 4. 反思与收获
- **今日亮点** - 做得好的是什么
- **改进空間** - 可以如何優化
# 知識
- **困惑与問題** - 待解決的疑问

#### 5. 关联筆記
- 今日編輯/創建的筆記連結
- 新增標籤彙總

## 最佳實踐

### ✅ 推荐做法

- **每天坚持** - 即使只写一句话也比不写好
- **诚实記錄** - 真实反映情况，便于复盘
- **關注感受** - 記錄情绪狀態，了解自己
# 知識
- **定期回顧** - 每周/每月回顧旧日报

### ❌ 避免做法

- **拖延** - 不要等到周末才补
- **流水账** - 避免無意义的细节
- **过度完美** - 不需要每次都写得很详尽
- **只记待辦** - 反思和收获同样重要

## 进阶技巧

### 周报彙總

从日报中提取關鍵資訊，生成周报：

```
本周主題：
├── 完成的任務数：X
├── 遇到的挑戰：
│   ├── 挑戰1
│   └── 挑戰2
├── 学到的新知：
│   ├── 新知1
│   └── 新知2
└── 下周計劃：
    ├── 計劃1
    └── 計劃2
```

### 標籤系統

使用標籤追踪特定主題：

| 標籤 | 用途 |
|------|------|
| `#daily-report` | 所有日报 |
| `#今日完成` | 已完成任務 |
| `#待辦` | 待辦事項 |
| `#反思` | 反思內容 |
| `#學習` | 學習心得 |

# 分析

定期統計日报數據：

- 完成率 = 完成任務 / 計劃任務
- 專注领域分布
- 情绪波动趨勢

## 常见問題

### Q1: 每天写日报太费時間怎么办？

**A**: 从简单開始。第一周只写3行：
1. 今天做了什么
2. 学到了什么
3. 明天要做什么

逐步增加內容深度。

### Q2: 有时候一天什么都没完成，怎么写？

**A**: 这种情况更有記錄價值！
- 诚实記錄狀態
# 分析
- 调整第二天的計劃

### Q3: 日报應該放在哪里？

# 管理

### Q4: 如何避免日报变成流水账？

**A**: 聚焦于：
- **成果** 而非活动
- **反思** 而非描述
- **問題** 而非现象

每次问自己："这对未来的我有什么價值？"

## 30天日报挑戰

试试连续30天写日报，见证改变：

| 周次 | 目標 | 重点 |
|------|------|------|
| 第1周 | 每天記錄 | 建立习惯，坚持最重要 |
| 第2周 | 增加反思 | 開始深度思考 |
# 知識
| 第4周 | 優化流程 | 根据經驗调整模板 |

完成挑戰后，回顧这30天的日报，撰写一篇總結。
