# 04-秦汉帝国

> 中国「大一统」的模板——2000年的政治基因来自这400年。

## 秦 (221-206 BCE)

| 秦始皇: 统一文字·度量衡·驰道——「车同轨——书同文」 |
|------|
| 长城: 不是新建——是连接——把各国的墙连起来 |
| 速亡: 暴政——「戍卒叫——函谷举」——15年——但制度留下来 |

## 汉 (206 BCE-220 CE)

| 刘邦: 布衣天子——「约法三章」——「安得猛士兮守四方」 |
|------|
| 汉武帝: 北击匈奴——卫青·霍去病——「匈奴未灭——何以家为」 |
| 丝绸之路: 张骞出使——打开了中西交流的通道 |
| 司马迁《史记》: 十二本纪·三十世家·七十列传——中国正史的模版 |
