# 01-先秦至隋唐概述

> 从神话到帝国——中国文明的形成与第一个鼎盛。

## 朝代速览

| 朝代 | 时间 | 关键词 |
|------|:---:|------|
| 夏 | ~2070-1600 BCE | 传说——大禹治水 |
| 商 | ~1600-1046 BCE | 甲骨文——青铜器 |
| 周 | 1046-256 BCE | 分封·百家争鸣 |
| 秦 | 221-206 BCE | 统一——15年——「昙花一现——影响千年」 |
| 汉 | 206 BCE-220 CE | 盛世——丝绸之路 |
| 三国 | 220-280 | 战争——英雄——演义 |
| 魏晋南北朝 | 220-589 | 分裂——佛教传入——民族融合 |
| 隋 | 581-618 | 统一——大运河——37年 |
| 唐 | 618-907 | 黄金时代——世界帝国 |

## 核心线索

| 统一与分裂的交替——王朝从兴起到腐败到崩溃——「天下大势——分久必合——合久必分」 |
