# 929(510)Mao — 毛泽东传记分类码详解

> 以 UDC `929(510)Mao` 为锚点，展开四体系毛泽东传记分类码的完整对照。

## 一、UDC 929(510)Mao 完整拆解

```
929     (510)     Mao
 ↑        ↑        ↑
传记    中国     人名扩展
主类    地点     字母指定
```

| 元素 | 来源 | 含义 |
|------|------|------|
| `929` | UDC 主表 | 传记与相关研究 (Biographical studies) |
| `(510)` | Table 1e 地点辅助表 | 中国 (China) |
| `Mao` | 字母扩展惯例 | 指定毛泽东个体 |

### UDC 合成规则

```
UDC 传记 = 929 + (地点复分) + 人名 + [:主题复分] + ["时间复分"]

929(510)Mao            → 中国 · 毛泽东 · 综合传记
929(510)Mao:32         → 中国 · 毛泽东 · 政治学
929(510)Mao:355        → 中国 · 毛泽东 · 军事学
929(510)Mao"1893/1976" → 中国 · 毛泽东 · 生平时段
```

## 八、分类哲学分析

### 为什么 CLC 赋予毛泽东独立类号？

| 体系 | 做法 | 哲学 |
|------|------|------|
| **DDC** | 归入中国史，不特殊处理 | 以历史时期为分类标准 |
| **UDC** | 归入传记，人名扩展 | 以传记类型 + 地点 + 个体为分类标准 |
| **CLC** | **A 类独立设 A7/A75** | 以思想体系地位为分类标准 — 意识形态嵌入 |
| **LCC** | 归入中国史，Cutter 区分 | 以历史地理 + 字母序为分类标准 |

> CLC 的 A75 是**分类非中立性**的经典案例：它将毛泽东从「历史人物」提升到「思想体系创始人」，与马克思、恩格斯、列宁、斯大林平级。
