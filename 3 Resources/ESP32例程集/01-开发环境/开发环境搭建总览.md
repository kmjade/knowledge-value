## 🚀 快速开始

### 前置要求
- Ubuntu 20.04/22.04 虚拟机或物理机
- 至少 10GB 可用磁盘空间
- 稳定的网络连接

### 开发环境组成

```
开发环境
├── ESP32-IDF v5.1.2      (ESP32官方SDK)
├── micro-ROS组件           (ROS2 for ESP32)
├── micro-ROS代理           (通信桥梁)
├── ROS2 Humble             (机器人操作系统)
└── Flash烧录工具           (固件烧录)
```
