## 📖 学习路径建议

### 入门路径

1. **环境搭建**：从开发环境开始，安装ESP32-IDF和microROS组件
2. **基础外设**：学习GPIO、UART、ADC等基础外设控制
3. **电机控制**：学习电机驱动、编码器读取和PID控制
4. **传感器**：学习IMU和激光雷达数据读取

### MicroROS路径

1. 先完成基础外设和传感器学习
2. 学习发布话题和订阅话题
3. 进阶到多话题处理
4. 学习速度、IMU、雷达等传感器话题发布
5. 学习自定义传输方式

### 机器人控制路径

1. 完成所有MicroROS基础学习
2. 烧录出厂固件并连接代理
3. 学习机器人信息发布
4. 学习键盘和手柄控制
5. 学习标定和URDF模型
