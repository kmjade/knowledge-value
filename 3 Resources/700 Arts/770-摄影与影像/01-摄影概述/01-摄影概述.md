# 01-摄影概述

> 「摄影是光线的绘画」——用机械捕捉瞬间——改变了人类看世界的方式。

## 摄影的本质

| 光→镜头→感光材料/传感器→图像——「照片是时间的切片」 |
|------|

## 摄影的类型

| 类型 | 目的 |
|------|------|
| 纪实 | 记录真实——不干预——新闻·社会 |
| 艺术 | 表达——个人视角——画廊 |
| 商业 | 广告·时尚·产品——「卖东西」的摄影 |
| 人像 | 人——身份·情感·记忆 |
| 风光 | 自然——美的追求——Ansel Adams |
