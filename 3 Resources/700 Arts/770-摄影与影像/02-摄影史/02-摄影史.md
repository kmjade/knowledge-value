# 02-摄影史

> 从8小时曝光到1/8000秒——从化学到数字。

## 技术里程碑

| 年份 | 事件 |
|:---:|------|
| 1826 | Niépce「窗外的风景」——第一张照片——曝光8小时 |
| 1839 | 达盖尔银版法公布——「摄影的生日」 |
| 1888 | Kodak——「你按快门——剩下的我们来做」——摄影大众化 |
| 1935 | Kodachrome——彩色反转片——鲜活的色彩 |
| 1975 | 柯达工程师发明数码相机——但柯达不敢推(怕冲击胶卷)——数码相机最终毁灭了柯达 |
| 2007 | iPhone——相机无处不在——「最好的相机是你手里的」 |

## 摄影与绘画

| 摄影发明→「画得像」不再是绘画的目标→推动了印象派和现代艺术 |
