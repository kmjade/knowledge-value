# 01-音乐概述

> 最抽象的艺术——只有时间——没有空间——但能让你哭。

## 音乐的元素

| 元素 | 说明 |
|------|------|
| 旋律 | 音高的序列——「你能哼出来的部分」 |
| 节奏 | 时间中的组织——快慢·节拍——身体最先感受到 |
| 和声 | 多个音同时——协和/不协和——色彩 |
| 音色 | 同一音高——钢琴和小提琴听起来不同——「声音的质感」 |
| 结构 | 整曲的布局——奏鸣曲式·ABA·verse-chorus |

## 音乐的功能

| 仪式·娱乐·表达·治愈·政治——没有一种文化没有音乐 |
