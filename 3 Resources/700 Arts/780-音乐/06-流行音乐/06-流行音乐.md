# 06-流行音乐

> 大众的音乐——从爵士到K-pop——「流行的」=「人民的」。

## 流派演变

| 年代 | 流派 | 特征 |
|:---:|------|------|
| 1920s | 爵士 | 即兴·摇摆——新奥尔良→芝加哥→纽约 |
| 1950s | 摇滚 | 电吉他——反叛——Elvis·Beatles——「年轻人的音乐」 |
| 1970s | 迪斯科·朋克 | 跳舞→「去你的」 |
| 1980s | 嘻哈·电子 | Bronx→全球——采样·节奏·说唱 |
| 1990s | 另类·电子舞曲 | Nirvana·Techno |
| 2000s- | 流媒体时代 | 歌单取代专辑——TikTok制造金曲 |

## K-pop

| BTS·Blackpink——韩国流行征服世界——训练体系·编舞·社交媒体——「偶像工业」 |
