# 07-民族音乐 (World Music)

> 每一种文化有自己的音乐「语言」——不是「落后」——是不同的逻辑。

## 世界民族音乐

| 地区 | 特征 |
|------|------|
| 印度 | 拉格(旋律框架)+塔拉(节奏循环) ——西塔琴——即兴 |
| 非洲 | 鼓——节奏复杂(交叉节奏) ——「参与」——没有「观众」——大家都是参与者 |
| 印尼 | 加美兰(gamelan) ——金属打击乐——循环·层次——「声音的曼陀罗」 |
| 中东 | 微分音(比半音还小) ——装饰音——「旋律的阿拉伯花纹」 |
