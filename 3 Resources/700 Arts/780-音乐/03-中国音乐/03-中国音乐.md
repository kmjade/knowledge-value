# 03-中国音乐

> 五千年——从编钟到「中国风」流行音乐。

## 传统音乐

| 古琴: 文人乐器——三千首曲谱——「琴棋书画」之首——伯牙子期「高山流水」 |
|------|
| 琵琶: 「大珠小珠落玉盘」——《十面埋伏》——叙事性 |
| 二胡: 两根弦——最像人声——《二泉映月》(阿炳·盲人流浪艺人) |
| 古筝·笛子·唢呐——合奏(江南丝竹·广东音乐) |

## 中国音乐的特征

| 五声音阶为主·旋律性强·标题音乐(有故事)——不是「抽象」的音乐 |

## 近现代

| 学堂乐歌→抗战歌曲→红色经典→改革开放→「中国风」(周杰伦·方文山) ——传统+流行 |
