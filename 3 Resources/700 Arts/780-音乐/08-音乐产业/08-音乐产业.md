# 08-音乐产业

> 音乐是艺术——也是生意——$260亿的全球产业。

## 产业的变革

| 时代 | 模式 |
|------|------|
| 唱片时代 (1950s-90s) | 卖实体——黑胶→磁带→CD——每张$15 |
| 下载时代 (2000s) | iTunes——$0.99/首——盗版肆虐(Napster) |
| 流媒体时代 (2010s-) | Spotify——月费——单次播放收入$0.003-0.005——「100万播放=$3000-5000」 |

## 谁赚钱？

| 巡演: 现在是音乐人最主要的收入——唱片不赚钱——「卖T恤比卖唱片赚得多」 |
|------|
