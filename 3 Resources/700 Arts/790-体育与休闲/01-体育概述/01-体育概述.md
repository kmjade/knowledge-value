# 01-体育概述

> 身体的竞技——从古希腊到电子竞技——体育是人类最古老的文化活动之一。

## 体育的功能

| 功能 | 说明 |
|------|------|
| 健康 | 锻炼——但职业体育往往不健康(伤病) |
| 娱乐 | 观看的快感——英雄·戏剧·归属 |
| 教育 | 规则·团队·公平竞争——「体育是最好的品格教育」 |
| 政治 | 奥运会是政治的延伸——1936柏林·1980莫斯科抵制 |
| 商业 | $5000亿全球产业——转播权·赞助·博彩 |
