# 04-篮球与其他球类运动

## 篮球

| Naismith 1891发明——把桃子筐钉在墙上——「用足球投进桃筐」 |
|------|
| NBA: 全球最赚钱的篮球联赛——乔丹·科比·勒布朗——「全球化」的典范 |
| 中国: 姚明——最高——但也最有技巧的大个子——连接了中国和NBA |

## 其他球类

| 网球: 四大满贯——费德勒·纳达尔·德约——「黄金一代」 |
|------|
| 乒乓球: 中国「国球」——基本上统治了这项运动——外交(乒乓外交1971) |
| 羽毛球·排球·棒球·板球(印度·巴基斯坦·英国——20亿观众!) |
