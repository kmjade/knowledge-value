# 08-体育产业

> $5000亿——而且还在增长——体育是最大的娱乐产业。

## 产业链

| 赛事(门票·转播权)——这是核心——英超转播权每年£50亿 |
|------|
| 赞助(Nike·Adidas)——运动员=活的广告牌 |
| 博彩——合法化浪潮——但也带来成瘾和腐败风险 |
| 周边(球衣·纪念品)——球迷「投票」用钱包 |

## 运动员经济

| 顶流: 梅西·C罗——年薪+赞助>$1亿——但99%的职业运动员收入普通——「运动生涯短」 |
