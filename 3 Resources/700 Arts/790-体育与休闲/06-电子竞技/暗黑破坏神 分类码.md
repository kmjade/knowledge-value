# 暗黑破坏神 分类码对照

> Diablo 系列在四大图书馆分类体系中的精确分类号。

---

## 一、四体系总览

|   体系    |       类号        | 名称                             |
| :-----: | :-------------: | ------------------------------ |
| **DDC** |    `793.932`    | Computer adventure / RPG games |
| **UDC** |    `794:004`    | Computer games                 |
| **CLC** |    `G898.3`     | 电子游戏                           |
| **LCC** | `GV1469.25.D53` | Diablo (Cutter)                |

---

## 二、DDC 层级

```
700 — 艺术与休闲 (Arts & Recreation)
  └── 790 — 休闲娱乐
       └── 793 — 室内游戏
            └── 793.9 — 其他室内游戏
                 └── 793.93 — 冒险游戏
                      └── 793.932 — 计算机冒险/RPG
```

> 与魔兽世界共用 `793.932`——ARPG 和 MMORPG 在 DDC 中归属同一子类。

---

## 三、CLC 层级（中图法）

```
G — 文化、科学、教育、体育
  └── G8 — 体育
       └── G89 — 文体活动
            └── G898 — 游戏
                 └── G898.3 — 电子游戏
```

### Diablo 系列

| 作品 | 年份 | CLC |
|------|:---:|:---:|
| Diablo I | 1996 | `G898.3` |
| Diablo II | 2000 | `G898.3` |
| Diablo III | 2012 | `G898.3` |
| Diablo IV | 2023 | `G898.3` |
| Diablo Immortal | 2022 | `G898.3` + `TN929.53` (手游) |

---

## 四、LCC 层级

```
G — 地理·人类学·休闲
  └── GV — 休闲 (Recreation. Leisure)
       └── GV1469.2-1469.35 — 电子游戏
            └── GV1469.25 — 特定游戏
                 └── GV1469.25.D53 — Diablo
```

---

## 五、与同类游戏的对比

| 游戏 | DDC | CLC | LCC | 类型 |
|------|:---:|:---:|:---:|------|
| Diablo | `793.932` | `G898.3` | `GV1469.25.D53` | ARPG |
| 魔兽世界 | `793.932` | `G898.3` | `GV1469.25.W67` | MMORPG |
| 星际争霸 | `793.932` | `G898.3` | `GV1469.25.S73` | RTS |
| 守望先锋 | `793.932` | `G898.3` | `GV1469.25.O94` | FPS |

> **暴雪全家桶**在 CLC 中全部归入 `G898.3`——中国分类法对电子游戏不做子类细分。LCC 通过 Cutter 号区分每款游戏。
