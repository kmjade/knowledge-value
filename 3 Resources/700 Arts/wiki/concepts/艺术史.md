# 藝術史 Art History
研究視覺藝術在歷史中的發展與演變。

## 主要時期
| 時期 | 年代 | 特徵 |
|:----|:----|:-----|
| **古典** | ~500 BCE–500 CE | 希臘羅馬·理想比例 |
| **中世紀** | 500–1400 | 宗教藝術·拜占庭·哥德式 |
| **文藝復興** | 1400–1600 | 透視法·人本主義·達文西·米開朗基羅 |
| **巴洛克** | 1600–1750 | 戲劇性·光影·卡拉瓦喬·林布蘭 |
| **新古典/浪漫** | 1750–1850 | 理性 vs 情感 |
| **現代** | 1850–1970 | 印象派→立體派→抽象表現主義 |
| **當代** | 1970– | 觀念藝術·裝置·數位藝術 |
