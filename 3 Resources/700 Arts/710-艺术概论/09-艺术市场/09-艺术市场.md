# 09-艺术市场

> 艺术是商品——而且是世界上最贵的商品之一。

## 谁在买卖艺术？

| 玩家 | 角色 |
|------|------|
| 艺术家 | 生产者——但大多数赚不到钱——「梵高一生卖了一幅画」 |
| 画廊 | 中间商——展览·推广·销售——抽成~50% |
| 拍卖行 | 二级市场——苏富比·佳士得——天价纪录的发生地 |
| 收藏家 | 终极买家——投资·炫耀·热爱——三者难以区分 |
| 美术馆 | 公共收藏——不能卖——「艺术的教堂」 |

## 天价作品

| 达芬奇《救世主》$4.5亿(2017) ——为什么这么贵？——稀缺+品牌+洗钱? |
|------|
| Beeple NFT $6900万(2021) ——数字文件——不可复制？——其实可以——但「原件」只有一份 |
