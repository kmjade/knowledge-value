# 07-艺术创作

> 艺术是怎么做出来的？——灵感·技巧·劳动。

## 创作过程

| 阶段 | 说明 |
|------|------|
| 灵感 | 触发——一幅风景·一个梦·一个社会事件——但灵感不是「等来的」——是在工作中遇到的 |
| 构思 | 从模糊到清晰的意象——草图·笔记·实验 |
| 执行 | 最长的阶段——技巧·劳动·「一万小时」 |
| 完成? | 什么时候画「完了」？——有些画家永远觉得没完——「作品不是完成的——是被放弃的」 |

## 灵感的神话

| 浪漫主义把灵感神圣化——「缪斯降临」——但大多数艺术家说: 灵感 = 1% + 99% 汗水 |
|------|
| 毕加索: 「灵感存在——但它必须在你工作时找到你」 |

## 天才 vs 训练

| 基因还是环境？——莫扎特4岁作曲——但莫扎特的父亲是音乐教师——从出生起就被训练 |
|------|
