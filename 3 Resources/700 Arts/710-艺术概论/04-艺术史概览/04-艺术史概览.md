# 04-艺术史概览

> 艺术风格的演变——从洞穴壁画到NFT。

## 西方艺术史时间线

| 时期 | 时间 | 特征 |
|------|:---:|------|
| 史前 | -30000 | 洞穴壁画——拉斯科·阿尔塔米拉 |
| 古典 | -800-500 | 希腊的理想人体·罗马的写实 |
| 中世纪 | 500-1400 | 宗教·象征——不是「不会画」——是「不想画得像」 |
| 文艺复兴 | 1400-1600 | 透视·解剖·人文——达芬奇·米开朗基罗 |
| 巴洛克 | 1600-1750 | 戏剧性·动感·光影——卡拉瓦乔·伦勃朗 |
| 印象派 | 1860-1900 | 光与色——莫奈·雷诺阿——「未完成」的感觉 |
| 现代主义 | 1900-1970 | 毕加索·杜尚·波洛克——「什么是艺术？」 |
| 当代 | 1970- | 观念·装置·数字——边界消失 |

## 中国艺术史概要
