# 08-艺术批评

> 批评不是「挑毛病」——是「解释、评价、放置于语境中」。

## 批评的功能

| 功能 | 说明 |
|------|------|
| 描述 | 这作品是什么样的？——客观——材料·尺寸·形式 |
| 分析 | 它是怎么构成的？——构图·色彩·技法 |
| 解释 | 它意味着什么？——符号·隐喻·语境 |
| 评价 | 它好不好？——最主观——最需要论证 |

## 批评的视角

| 视角 | 问题 |
|------|------|
| 形式主义 | 线条·色彩·构图——不看内容——「画本身说了什么」 |
| 马克思主义 | 谁生产？谁消费？——阶级和权力 |
| 女性主义 | 谁被表现了？谁在看？——「男性凝视」(male gaze) |
| 后殖民 | 谁的「艺术史」？——西方标准是普遍的吗？ |
