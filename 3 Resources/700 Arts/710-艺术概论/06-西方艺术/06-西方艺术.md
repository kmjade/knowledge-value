# 06-西方艺术传统

> 从模仿到抽象——2000年最终走向对「艺术」本身的质疑。

## 西方艺术的核心线索

| 线索 | 说明 |
|------|------|
| 模仿→表现 | 文艺复兴前: 画得像=好 → 摄影发明后: 画得像不如拍得快→转向「怎么画」 |
| 宗教→世俗 | 中世纪: 圣经插图 → 文艺复兴: 神话+肖像 → 17世纪后: 风景·静物·日常 |
| 贵族→公众 | 赞助人从教会·国王→资产阶级→艺术市场——谁付钱决定了画什么 |
| 确定性→多元 | 现代之前: 什么是好艺术有共识 → 当代: 没有标准——有时候连「是不是艺术」都不知道 |

## 西方艺术的转折点

| 摄影术(1839) | 「画得像」不再是艺术的目标——解放了绘画 |
|------|------|
| 杜尚(1917) | 「现成品」——观念比手艺重要——一切都可以是艺术 |
| 抽象表现主义(1940s-) | 艺术中心从巴黎→纽约——波洛克滴画——过程就是作品 |
