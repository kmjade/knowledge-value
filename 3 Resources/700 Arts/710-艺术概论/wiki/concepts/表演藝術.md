# 表演藝術 Performing Arts

## 定義
表演藝術是藝術家在**觀眾面前**進行的現場表演。

## 主要門類
| 門類 | 媒介 |
|------|------|
| **戲劇 Theatre** | 演員/舞台/文本 |
| **舞蹈 Dance** | 身體動作 |
| **音樂 Music** | 聲音/節奏 |
| **歌劇 Opera** | 音樂+戲劇 |

## 特點
- 即時性 — 每場演出獨一無二
- 互動性 — 表演者與觀眾的即時互動
- 不可複製 — 錄像 ≠ 現場體驗
