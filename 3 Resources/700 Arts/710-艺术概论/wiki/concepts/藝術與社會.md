# 藝術與社會 Art & Society

## 核心關係
- 藝術**反映**社會價值與衝突
- 藝術**塑造**公眾意識
- 藝術作為**政治工具**（宣傳 vs 抗議）

## 關鍵議題
| 議題 | 例子 |
|------|------|
| **審查 Censorship** | 被禁的藝術作品 |
| **文化挪用** | 誰有權使用某種文化符號？ |
| **藝術市場** | 藝術品的商品化 |
| **公共藝術** | 城市雕塑、壁畫 |
