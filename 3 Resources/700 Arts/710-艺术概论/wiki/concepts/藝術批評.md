# 藝術批評 Art Criticism

## 定義
藝術批評是對藝術作品進行**描述、分析、解釋和評價**的活動。

## 四步驟 (Feldman)
1. **描述** — 客觀描述你看見的
2. **分析** — 構圖、色彩、線條的組織
3. **解釋** — 作品的意義和意圖
4. **評價** — 判斷作品的藝術價值

## 批評流派
| 流派 | 關注焦點 |
|------|---------|
| 形式主義 | 視覺元素本身 |
| 脈絡主義 | 歷史、社會、文化背景 |
| 心理分析 | 藝術家潛意識 |
| 女性主義 | 性別視角 |
