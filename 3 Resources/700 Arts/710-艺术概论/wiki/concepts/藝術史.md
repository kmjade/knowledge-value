# 藝術史 Art History

## 定義
藝術史研究人類**視覺藝術的歷史發展**——從史前洞穴壁畫到當代裝置藝術。

## 主要時期
| 時期 | 年代 | 特點 |
|------|:--:|------|
| 古典 | -500–500 | 希臘-羅馬理想美 |
| 中世紀 | 500–1400 | 宗教藝術 |
| 文藝復興 | 1400–1600 | 透視、解剖、人文主義 |
| 巴洛克 | 1600–1750 | 動感、戲劇性光影 |
| 印象派 | 1860–1900 | 光與色彩 |
| **現代主義** | 1900–1970 | 抽象、立體、超現實 |
| 當代 | 1970– | 觀念、裝置、數位 |
