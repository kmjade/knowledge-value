# 05-中国艺术传统

> 书画同源——以笔墨为语言——追求「似与不似之间」。

## 中国艺术的独特特征

| 特征 | 说明 |
|------|------|
| 书画同源 | 书法和绘画用同样的工具——笔·墨·纸·砚——审美相通 |
| 散点透视 | 不是「站在一个点看」——是「边走边看」——《清明上河图》 |
| 诗书画印 | 一幅画上有诗·书法·印章——完整的文化产品——外国艺术没有 |
| 写意 | 不是不写实——是「意」重于「形」——「妙在似与不似之间」(齐白石) |

## 文人画

| 不是「职业画家」画的——是文人(官员·学者)的业余创作——强调「画中有诗」 |
|------|
| 不追求「像」——追求「意境」——苏轼: 「论画以形似——见与儿童邻」 |
