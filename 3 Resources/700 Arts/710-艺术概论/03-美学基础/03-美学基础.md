# 03-美学基础

> 美学 = 感性认识的学问——不仅关于「美」——关于「感性经验」。

## 西方美学

| 哲学家 | 核心观点 |
|------|------|
| 柏拉图 | 美是理念——艺术模仿自然——自然模仿理念——所以艺术是模仿的模仿 |
| 康德 | 审美判断——无关利害——主观但要求普遍同意——「无目的的合目的性」 |
| 黑格尔 | 美是理念的感性显现——艺术是绝对精神的一个阶段 |
| 本雅明 | 机械复制时代——艺术品失去了「灵韵」(aura) |

## 中国美学

| 概念 | 含义 |
|------|------|
| 气韵 | 作品中流动的生命力——「气韵生动」是六法之首 |
| 意境 | 情与景的交融——「意在画外」 |
| 留白 | 空不是「没有」——是「余地」——让观者参与 |
| 虚实 | 实景和虚境的辩证——山是实的——云雾是虚的 |

## 崇高 vs 优美

| 优美: 和谐·小·精致——一朵花 / 崇高: 压倒性·无限·恐怖——暴风雨中的大海 |
