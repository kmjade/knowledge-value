# 01-艺术概论概述

> 艺术是什么？——这个问题被追问了2500年——答案在不断变化。

## 艺术的功能

| 功能 | 说明 |
|------|------|
| 表达 | 艺术家传达情感·思想·体验 |
| 记录 | 历史事件·社会面貌——《清明上河图》 |
| 仪式 | 宗教·葬礼·庆典——艺术曾经是巫术 |
| 审美 | 美的体验——但「美」不是艺术的必要条件(杜尚的小便池) |
| 政治 | 宣传·抗议·颠覆——《格尔尼卡》 |

## 艺术的门类

| 门类 | 特征 |
|------|------|
| 视觉艺术 | 绘画·雕塑·建筑·摄影 |
| 表演艺术 | 音乐·舞蹈·戏剧 |
| 文学 | 诗歌·小说——文字的「艺术」 |
| 综合艺术 | 电影·歌剧·数字艺术 |

## 艺术的边界

| 杜尚的《泉》(1917): 一个小便池签名「R.Mutt」——是不是艺术？ |
|------|
| 如果小便池放在美术馆是艺术——那什么是艺术？——「艺术是被艺术世界承认的东西」 |
