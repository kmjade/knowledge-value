# Diablo III — 奪魂之鐮 (Reaper of Souls)

> 2012 年 5 月 15 日發售，歷經拍賣行爭議、戰利品 2.0 重製，最終成為最具深度的終局刷寶 ARPG。

## 一、核心資訊

|    項目    | 內容 |
| :------: | ---- |
|  開發商 | Blizzard Entertainment |
| 資料片 | Reaper of Souls (2014) · Rise of the Necromancer (2017) |
|  章節數 | 5 幕 |
|  職業數 | 7 (原版 5 + RoS 1 + DLC 1) |
| 難度 | Normal → Torment I–XVI |
|  賽季 | 第 1 賽季 (2014) — 至今持續更新 |
| 平台 | PC · PS4 · Xbox One · Switch |

## 二、七大職業

| 職業 | 英文 | 來源 | 核心資源 |
|:---:|------|:---:|:------:|
| 野蠻人 | Barbarian | 原版 | 怒氣 (Fury) |
| 狩魔獵人 | Demon Hunter | 原版 | 憎恨/戒律 (Hatred/Discipline) |
| 武僧 | Monk | 原版 | 精氣 (Spirit) |
| 巫醫 | Witch Doctor | 原版 | 魔力 (Mana) |
| 秘術師 | Wizard | 原版 | 奧能 (Arcane Power) |
| 聖教軍 | Crusader | RoS | 聖怒 (Wrath) |
| 死靈法師 | Necromancer | DLC | 魂能 (Essence) |

## 三、技能符文系統 (Skill Rune System)

取代 D2 的技能樹：每技能 5 種符文，改變效果/屬性。

| 符文 | 英文 | 效果示例 |
|:---:|------|---------|
| 無符文 | No Rune | 基礎效果 |
| 雪暴 | Snowbound | 降低消耗 |
| 冰霜射線 | Cold Snap | 冰凍傷害 |
| 連鎖閃電 | Chain Lightning | 閃電連鎖 |
| 火焰 | Conflagration | 火焰燃燒 |

## 四、終局系統

|   系統   |        描述        |
| :---: | :--------------: |
| 涅法雷姆秘境 (Nephalem Rift) | 隨機地圖，擊殺怪物填滿進度條 |
| 大秘境 (Greater Rift) | 限時挑戰，無限層級 (GR 150 上限) |
| 懸賞任務 (Bounties) | 章節任務，獎勵赫拉迪姆寶箱 |
| 卡奈魔方 (Kanai's Cube) | 萃取傳奇能力、重鑄裝備 |

## 五、套裝地城與賽季

| 賽季特色 | 描述 |
|---------|------|
| 賽季旅程 | 章節式任務，獎勵套裝、寵物、頭像框 |
| 海德格禮物 | 完成賽季旅程贈送完整職業套裝 |
| 賽季主題 | 每季獨特機制 (如第 28 賽季「聖化」) |
| 排行榜 | 大秘境職業/團隊排名 |
