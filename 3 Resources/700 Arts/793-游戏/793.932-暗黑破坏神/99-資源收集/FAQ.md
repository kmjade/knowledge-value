# 暗黑破壞神 — 常見問題 (FAQ)

> 暗黑系列知識庫的快速問答索引。

## 一、入門

| 問題 | 答案 |
|------|------|
| 新手推薦哪一代？ | D2R (經典系統) 或 D4 (現代體驗)。D3 最易上手 |
| 暗黑系列要買全部嗎？ | 不需要，每代獨立故事線，但共享世界觀 |
| D2R 和 D2 有什麼區別？ | D2R 是 2021 重製版，保留原版引擎但畫面現代化 |
| 有沒有中文版？ | 全部繁中/簡中在地化，含語音 |

## 二、遊戲機制

| 問題 | 答案 |
|------|------|
| 什麼是 MF？ | Magic Find，提高魔法/稀有/傳奇裝備掉落率 |
| 賽季是什麼？ | 週期性重置 (3-4 月)，所有玩家從零開始 |
| 專家模式 (HC) 是什麼？ | Hardcore：角色死亡永久消失 |
| 什麼是 Uber Boss？ | D2/D3 超級首領，掉落特定稀有裝備 |
| 交易系統如何？ | D2 自由交易，D3 限時限人，D4 傳奇以上綁定 |

## 四、技術

| 問題 | 答案 |
|------|------|
| D2 斷線複製 (Duping)？ | 已修復，不建議嘗試 |
| D4 跨平台？ | 支持 PC/主機跨平台連線與進度 |
| 離線模式？ | D2R 支援；D3 主機版支援；D4 需全程連線 |
| 硬體需求？ | D2R/D3 低；D4 需 SS + 中高階顯卡 |
