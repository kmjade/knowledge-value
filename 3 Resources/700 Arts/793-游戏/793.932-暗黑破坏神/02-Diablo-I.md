# Diablo I — 崔斯特瑞姆的黑暗 (The Darkening of Tristram)

> 1996 年 12 月 31 日發售，開創了現代 ARPG 類型的奠基之作。崔斯特瑞姆小鎮被魔王迪亞布羅的恐懼吞噬，玩家深入大教堂底下的十六層迷宮。

## 一、核心資訊

|    項目    | 內容 |
| :------: | ---- |
|  開發商 | Blizzard North |
|   平台   | PC (Windows 95) |
|  地城層數 | 16 層 (教堂 1-4 → 地下墓穴 5-8 → 洞穴 9-12 → 地獄 13-16) |
|  可選職業 | 戰士 (Warrior)、遊俠 (Rogue)、魔法師 (Sorcerer) |
|  擴展包 | Hellfire (1997, Synergistic Software) |
|  最大玩家數 | 4 (Battle.net) |

## 二、三大原初職業

| 職業 | 英文 | 主屬性 | 特點 |
|:---:|------|:-----:|------|
| 戰士 | Warrior | 力量 | 近戰耐久、修復裝備 |
| 遊俠 | Rogue | 敏捷 | 遠程弓箭、發現陷阱 |
| 魔法師 | Sorcerer | 魔力 | 法術 (火球/閃電/石化) |

## 三、地城層級

```
教 堂 (1-4)  →  地下墓穴 (5-8)  →  洞 穴 (9-12)  →  地 獄 (13-16)
  ↓               ↓                ↓                  ↓
 骷髏·墮落者    山羊人·蝙蝠      石像鬼·法師         地獄騎士·迪亞布羅
```

## 四、標誌性首領

|    首領    | 英文 | 層數 |        特點        |
| :-----: | :---: | :-: | :--------------: |
|  屠夫 | The Butcher |  2  | 「Fresh meat!」 |
|  骷髏王 | King Leoric |  3  | 崔斯特瑞姆的前國王 |
| 大主教拉薩雷斯 | Archbishop Lazarus | 15  | 叛徒，召喚迪亞布羅 |
| 迪亞布羅 | Diablo | 16  | 恐懼之王，終極首領 |

## 五、核心系統

|   系統   |        描述        |
| :---: | :--------------: |
| 隨機地城 | 每次新遊戲重新生成全部樓層 |
| 神殿 | 臨時增益/詛咒（隱藏效果） |
| 法術書 | 任何職業可學，魔法書消耗品 |
| 物品耐久 | 裝備有耐久度，需修理 |
| 死亡機制 | 掉裝撿屍體 |

> Diablo I 確立了「點擊移動 + 刷怪掉寶」的核心循環，為後續所有 Diablo-like 遊戲的模板。
