# Diablo Immortal — 暗黑破壞神：不朽

> 2022 年 6 月 2 日發售，與網易 (NetEase) 聯合開發的大型多人手遊。繼承經典暗黑元素，引入陣營 PvP 與大規模多人內容。

## 一、核心資訊

|    項目    | 內容 |
| :------: | ---- |
|  開發商 | Blizzard × NetEase |
|  平台 | iOS · Android · PC (Beta) |
|  時序 | D2 與 D3 之間 |
|  職業數 | 8 |
| 陣營 | 不朽者 (Immortals) vs 暗影會 (Shadows) |
|  PvE  | 地城副本、上古秘境、地獄骨匣 |
|  PvP  | 寶庫守衛、鬥爭循環、戰場 |

## 三、陣營 PvP 循環

```
  暗影會 (Shadows)                   不朽者 (Immortals)
       │                                  │
  完成契約任務  ────→  暗影等級提升  ────→  挑戰不朽者
       │                                  │
  陣營戰 (鬥爭循環) ←──→ 寶庫防禦           寶庫守衛
```

## 四、貨幣化爭議

| 機制 | 描述 | 爭議點 |
|------|------|--------|
| 傳奇寶石 | 核心裝備強化，需重複抽取升級 | P2W (付費取勝) |
| 戰鬥通行證 | 賽季付費軌道 | 可接受 |
| 拍賣行 | 玩家間交易系統 | 價格混亂 |
| 上古秘境 | 寶石獲取地城 | 付費加速 |

> 傳奇寶石的滿級成本推估超 10 萬美元，引發「Do YoU gUyS nOt HaVe PhOnEs?」事件。
