# 魔獸世界 — 社群與文化 (Community and Culture)

> WoW 的玩家社群是遊戲長壽的核心支柱。從公會文化、RP 角色扮演、經典版懷舊運動、到插件生態系統，WoW 社群創造了 MMORPG 歷史上最豐富的玩家驅動文化。

## 一、公會系統 (Guilds)

### 公會功能

| 功能 | 說明 |
| :---: | :--- |
| 團隊組織 | 固定/半固定團隊副本活動 |
| 公會銀行 | 共享存放空間，權限分級管理 |
| 公會成就 | 團隊副本、PvP、專業等集體成就 |
| 公會每日 | 完成公會挑戰獲得公會經驗與獎勵 |
| 跨陣營 | Dragonflight 10.1 起支援跨陣營公會 |
| 戰隊 | TWW 起帳號共享聲望/銀行，公會角色隨意切換 |

### 公會類型

| 類型 | 說明 |
| :---: | :--- |
| 進度團隊 (CE/Mythic) | 追求史詩團隊全通，固定出團時間 |
| 休閒公會 (Casual) | 輕鬆社交，彈性活動 |
| PvP 公會 | 競技場/積分戰場為主 |
| 升級公會 | 新手/小號友善，提供經驗加成 |
| RP 公會 | 角色扮演為核心 |
| 經濟公會 | AH 操作、壟斷市場 |
| 社群公會 (Communities) | BfA 引入跨服社群系統 (非傳統公會) |

## 二、角色扮演伺服器 (RP Servers)

### RP 伺服器特性

| 要素 | 說明 |
| :---: | :--- |
| 伺服器 | 月光林地 (Moonglade) · 血色十字軍 等 |
| 命名規則 | 嚴格命名政策，不符合 RP 命名可被舉報 |
| 活動 | RP 步行、酒館聚會、故事活動、陣營劇情 |
| 插件 | Total RP 3 (TRP3) · MyRolePlay (MRP) 角色描述插件 |
| 文化 | 角色背景故事設定、職業/種族 RP 搭配 |

### 著名 RP 事件

| 事件 | 描述 |
| :---: | :--- |
| 腐化之血事件 (Corrupted Blood) | 2005 年祖爾格拉布首領哈卡瘟疫機制洩漏到戶外，引發全服"疫情"，成為流行病學研究案例 |
| 婚禮葬禮 | 玩家自發性 NPC 葬禮、遊戲內婚禮 |
| 說書人巡遊 | RP 玩家在各主城進行故事表演 |

## 三、經典版與探索賽季 (Classic and SoD)

### WoW Classic 版本線

| 版本 | 發佈 | 階段 | 特色 |
| :---: | :--: | :--: | :--- |
| Classic (2019) | 2019.08 | 6 階段 | 忠實復刻 1.12 版本 |
| TBC Classic | 2021.06 | 5 階段 | 外域 + 血精靈/德萊尼 |
| WotLK Classic | 2022.09 | 5 階段 | 北裂境 + 死亡騎士 |
| Cataclysm Classic | 2024.05 | 持續更新 | 大災變經典版 |
| 探索賽季 (SoD) | 2023.11 | 7 階段 | 經典+實驗：符文系統、新團隊機制 |
| 20 週年紀念版 | 2024.11 | 持續 | 永久 60 級，無後續資料片 |

### 探索賽季 (Season of Discovery) 核心特性

| 要素 | 說明 |
| :---: | :--- |
| 符文系統 | 裝備符文獲得新技能 (如法師治療、盜賊坦克) |
| 等級階段 | 25→40→50→60 每階段提升上限 |
| 職業重塑 | 經典職業打破角色定位 (補法、坦賊、坦術、坦薩) |
| 新團隊 | 黑暗深淵 (25) · 諾姆瑞根 (40) · 沉沒的神廟 (50) · 熔火之心 (60) |
| 全新掉落 | 經典團隊副本獲得全新設計裝備 |

| 要素 | 說明 |
| :---: | :--- |

## 六、重要文化事件

| 事件 | 年份 | 描述 |
| :---: | :--: | :--- |
| 莉亞·詹金斯 (Leeroy Jenkins) | 2005 | 著名黑石塔上層「衝衝衝」影片，成為網路迷因 |
| 亡靈入侵事件 (Zombie Infestation) | 2008 | WotLK 前夕活動，瘟疫感染全城玩家，史詩級社群混亂 |
| 墮落之血 (Corrupted Blood) | 2005 | 祖爾格拉布瘟疫外洩，流行病學研究引用 |
| 哀悼日 (Funeral Raid Crash) | 2006 | 遊戲內葬禮被敵對陣營 PvP 洗劫 |
| 塞拉摩毀滅 | 2012 | MoP 前夕，塞拉摩被炸為廢墟 |
| 經典版回歸運動 | 2016-2019 | Nostalrius 私服關閉後社群請願，導致暴雪推出 WoW Classic |
| 泰達希爾之焚 | 2018 | BfA 前夕，希瓦娜斯燒毀夜精靈家園，引爆社群討論 |
| 中國伺服器停運 | 2023 | 網易暴雪合約終止，中國玩家遷徙至台服/國際服 |
