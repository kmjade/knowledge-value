# 魔獸世界 — 主知識庫索引 (World of Warcraft — Master Knowledge Base MOC)

> **Overview**: 暴雪娛樂 (Blizzard Entertainment) 旗下旗艦 MMORPG，自 2004 年上線以來持續營運逾 20 年，定義了現代大型多人線上角色扮演遊戲。本知識庫涵蓋從經典舊世 (Vanilla) 到地心之戰 (The War Within) 的全部核心內容。

## 🌍 艾澤拉斯世界觀 (Azeroth Cosmology)

```
        光明 (Light) ───────── 虛無 (Void)
              ╲                  ╱
               秩序 (Order) ─ 混亂 (Fel / Disorder)
              ╱                  ╲
        生命 (Life) ───────── 死亡 (Death)
              │                  │
         泰坦 (Titans)      燃燒軍團 (Burning Legion)
         上古之神 (Old Gods)  巫妖王 (Lich King)
```

**六大宇宙原力 (Six Cosmic Forces)**: Light · Void · Order · Disorder · Life · Death

**核心陣營 (Core Factions)**:
