# 魔獸世界 — PvP 與戰場 (PvP and Battlegrounds)

> WoW PvP 涵蓋競技場、積分戰場、野外 PvP、戰爭模式及單排競技場。從經典軍階系統到現代 Solo Shuffle，PvP 擁有獨立的裝備、獎勵與評分體系。

## 一、PvP 模式總覽

| 模式 | 英文 | 規模 | 排隊方式 | 獎勵系統 | 引入版本 |
| :--: | :---: | :--: | :-----: | :-----: | :-----: |
| 競技場 2v2 | Arena 2v2 | 2v2 | 預組/單排 | 征服/榮譽、競技場頭銜 | TBC |
| 競技場 3v3 | Arena 3v3 | 3v3 | 預組 | 角鬥士頭銜、季節坐騎 | TBC |
| 單排競技場 | Solo Shuffle | 6 人輪換 3v3 | 單人排隊 | 征服/榮譽、專屬頭銜 | Dragonflight |
| 積分戰場 | Rated Battleground (RBG) | 10v10 | 預組 | 大元帥/高階督軍頭銜 | Cataclysm |
| 積分閃擊戰 | Rated Battleground Blitz | 8v8 | 單人排隊 | 與 RBG 共享獎勵線 | TWW |

| 野外 PvP | World PvP | 不限 | 即時 | 榮譽 | Classic→現在 |

### Solo Shuffle 機制

| 要素 | 說明 |
| :--- | :--- |
| 配對規模 | 6 名玩家：2 治療 + 4 DPS (或特殊隊形) |
| 輪換 | 每輪與不同搭檔配對進行 3v3，共 6 輪 |
| 評分 | 依據勝場數計分，無平局 |
| 獎勵 | 專屬頭銜、季節坐騎變色版、征服點數 |
| 排隊時間 | DPS 較長 (10-30 分鐘)，治療幾乎秒排 |

### 積分戰場 vs 積分閃擊戰

| 要素 | 積分戰場 (RBG) | 積分閃擊戰 (Blitz) |
| :--- | :-----------: | :--------------: |
| 規模 | 10v10 | 8v8 |
| 排隊 | 需預組團隊 | 單人排隊 |
| 地圖池 | 選定戰場 | 選定戰場 (小規模) |
| 節奏 | 團隊配合、語音指揮 | 快速節奏、個人表現 |
| 獎勵 | 征服、頭銜、季節坐騎 | 同左 |

## 四、戰爭模式 (War Mode)

| 要素 | 說明 |
| :--- | :--- |
| 啟用方式 | 天賦頁面切換 (限暴風城/奧格瑪開啟) |
| 加成 | 啟用後獲得 10-15% 經驗/金幣加成 (依陣營人數動態) |
| PvP 天賦 | 解鎖額外 PvP 天賦欄位 (4 個) |
| 戰爭補給箱 | 空投降落補給箱，爭奪獲取征服點數與裝備 |
| 賞金獵人 | 擊殺敵對玩家後累積賞金，被擊殺時敵方獲得 |

## 五、榮譽與征服系統

### 榮譽系統

| 等級 | 榮譽等級 | 獎勵解鎖 |
| :--: | :-----: | :-----: |
| 1-500 | 榮譽等級 | 頭銜、寵物、坐騎、外觀 |
| 里程碑 | 每 10 級 | PvP 坐騎 (聯盟/部落各一) |
| 500 | 最高 | 聲望級坐騎 |

### 征服點數賽季

| 要素 | 說明 |
| :--- | :--- |
| 每週上限 | 初始上限 + 每週累積 (錯過週數可追趕) |
| 裝備購買 | 使用征服點數購買當季 PvP 裝備 |
| PvP 裝等 | PvP 環境內裝等自動提升 (基礎+14 裝等) |
| 階級 | 無評分 (Unrated) → 戰鬥者 (Combatant) → 挑戰者 (Challenger) → 競爭者 (Rival) → 決鬥者 (Duelist) → 精英 (Elite) |
