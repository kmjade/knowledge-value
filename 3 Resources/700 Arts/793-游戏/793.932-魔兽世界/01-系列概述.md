# 魔獸世界 — 系列概述 (World of Warcraft Series Overview)

> 暴雪娛樂旗艦 MMORPG，2004 年 11 月 23 日北美首發，全球累計註冊帳號逾 1 億，定義了現代 MMORPG 標準。歷經 10 部資料片，仍在持續更新。

## 三、MMORPG 類型定義

| 要素 | WoW 貢獻 |
|------|---------|
| 三位一體 | 確立 Tank/Healer/DPS 鐵三角 PvE 框架 |
| 團隊副本 | 從 40 人 MC 到 20 人史詩，分層難度體系 |
| 任務系統 | 感嘆號/問號 UI 範式，成為行業標準 |
| 陣營對立 | 聯盟 vs 部落的 PvP/PvE 雙軌設計 |
| 經濟生態 | 拍賣場 (AH)、專業系統、金幣經濟 |
| 插件生態 | LUA API 開放，WeakAuras/DBM 等第三方生態 |
| 賽季模式 | M+ 賽季、PvP 賽季、世界首殺競賽 |
