# 魔獸世界 — 經濟與專業 (Economy and Professions)

> WoW 的經濟體系包含專業技能、拍賣場交易、金幣經濟，以及龐大的收藏系統（坐騎、寵物、幻化）。Dragonflight 對專業系統進行了全面重製，引入品質層級與訂單系統。

## 二、Dragonflight 專業重製

### 品質層級系統

| 品質 | 星級 | 說明 |
| :--: | :--: | :--- |
| 1 星 | ★ | 基礎品質，基本屬性 |
| 2 星 | ★★ | 中級品質，屬性提升 |
| 3 星 | ★★★ | 最高品質，最大屬性 |

> 成品品質取決於：材料品質、專業專精、裝備工具、技能等級、靈感觸發。

### 專業專精 (Specializations)

| 專業 | 專精樹 | 說明 |
| :---: | :---: | :--- |
| 鍛造 | 武器鍛造 / 鎧甲鍛造 | 副專精影響品質與效率 |
| 煉金術 | 藥水 / 藥劑 / 轉化 | — |
| 工程學 | 炸彈 / 小工具 / 裝置 | — |

### 訂單系統 (Crafting Orders)

| 類型 | 說明 |
| :---: | :--- |
| 公開訂單 (Public) | 所有人可見，先到先得 |
| 公會訂單 (Guild) | 公會內可見 |
| 私人訂單 (Personal) | 指定工匠，一對一 |
| 客戶提供材料 | 可指定材料由客戶提供或賣家提供 |
| 佣金 | 客戶設定佣金金額 |

## 三、拍賣場經濟 (Auction House Economy)

### 拍賣場機制

| 要素 | 說明 |
| :---: | :--- |
| 跨陣營 | 拍賣場自 WoD 版本起即為中立跨陣營 |
| 同服共用 | 同一伺服器的聯盟與部落共用拍賣場 |
| 跨服商品 | 材料類 (礦石/草藥/布料等) 已跨服統一 |
| 裝備 | 部分拾綁 (BoP) 無法交易，裝綁 (BoE) 可拍賣 |
| 代幣 (WoW Token) | 金幣↔遊戲時間/戰網幣的合法交易 |

### 賺金方式

| 方式 | 說明 | 投入 |
| :---: | :--- | :--: |
| 專業製造 | 製作並出售高品質裝備/藥水/附魔 | 高 |
| 採集 | 雙採集 (礦/草) 出售原材料 | 中 |
| 世界任務 | 每日金幣獎勵累積 | 低 |
| 傳承副本 | 單刷舊團隊副本，賣掉掉落物 | 低 |
| 躺屍老闆 | 帶打 M+/團隊換取金幣 | 極低 (需技術) |
| 塑形/寵物翻賣 | 跨服購買稀有外觀/寵物轉售 | 中 |
| 代幣翻轉 | 購買 WoW 代幣在戰網幣/金幣間轉換 | 需現實金錢 |

## 五、寵物對戰 (Pet Battles)

### 系統概覽

| 要素 | 說明 |
| :---: | :--- |
| 引入 | MoP (5.0) |
| 類型 | 回合制 3v3 寵物對戰 |
| 收集 | 1,700+ 寵物可收集 |
| 品質 | 灰色 (Poor) → 白色 (Common) → 綠色 (Uncommon) → 藍色 (Rare) |
| 等級 | 1-25 級 |
| 對戰 | 野生寵物、訓練師、PvP 寵物對戰 |
| 成就 | 寵物收集成就解鎖寵物、坐騎 |

### 寵物來源

| 來源 | 範例 |
| :---: | :--- |
| 野外捕獲 | 各地圖野生寵物 |
| 團隊/地城 | 首領掉落 |
| 節日 | 節日專屬寵物 |
| 專業 | 工程學機械寵物、附魔寵物 |
| 商城/活動 | 暴雪嘉年華限定 |
| 成就 | TCG 碼寵物復刻 |

## 六、幻化 (Transmogrification)

### 幻化系統

| 要素 | 說明 |
| :---: | :--- |
| 引入 | Cataclysm (4.3) |
| 機制 | 將現有裝備外觀變更為已收集外觀 |
| 收集 | 裝備自動加入外觀收集 (拾取綁定後) |
| 規則 | 同護甲類型、同武器類型、可裝備 |
| 套裝 | T0→T21 各職業套裝，以及資料片 PvP 套裝 |

### 外觀收集擴展

| 更新 | 說明 |
| :---: | :--- |
| Legion (7.0) | 幻化衣櫃系統：自動收集 + 套裝管理 |
| Shadowlands | 非護甲部位擴展 (披風/武器附魔等) |
| Dragonflight | 白/灰品質裝備加入收集 |
| TWW | 戰隊系統：帳號共享幻化解鎖進度 |
