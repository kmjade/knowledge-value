# 魔獸世界 — 暗影國度至地心之戰 (Shadowlands to TWW: Post-Squish Era)

> 2020 年等級壓縮後，WoW 進入現代化時代。從暗影國度的盟約系統到巨龍時代的飛龍騎術與跨陣營，再到地心之戰的探究與戰隊系統 — 世界之魂傳奇 (World Soul Saga) 三部曲正式啟航。

## 一、版本速覽

| 版本 | 等級 | 新大陸 | 核心系統 | 新職業/種族 | 標誌性變化 |
| :--: | :-: | :---: | :-----: | :--------: | :-------: |
| Shadowlands (9.x) | 60 | 暗影之境 | 盟約、爬塔、靈魄 | — | 等級壓縮、新玩家島 |
| Dragonflight (10.x) | 70 | 巨龍群島 | 飛龍騎術、跨陣營、全新天賦樹 | 龍人喚能師 | UI 重製、專業重做 |
| TWW (11.x) | 80 | 卡茲阿爾加 | 探究、戰隊、英雄天賦 | 土靈 | 世界之魂傳奇首部曲 |

## 四、Dragonflight 核心系統

### 飛龍騎術 (Dragonriding)

| 要素 | 說明 |
| :--- | :--- |
| 機制 | 動量飛行系統：加速、爬升、俯衝、精力管理 |
| 龍族 | 四種可自訂飛龍坐騎，收集外觀雕紋 |
| 天賦 | 飛龍騎術天賦樹，解鎖技能與被動 |
| 速度 | 最高可超過 900% 移動速度 (遠超傳統 310% 飛行) |

### 全新天賦樹 (Talent Revamp)

| 要素 | 說明 |
| :--- | :--- |
| 結構 | 職業天賦樹 + 專精天賦樹雙軌 |
| 點數 | 1-70 級獲取，滿等 61 點 (31 職業 + 30 專精) |
| 設計 | 回歸樹狀結構，但無需投入前置點數過多 |
| 匯入/匯出 | 支援字串分享，一鍵載入配置 |

### 跨陣營遊戲 (Cross-Faction)

| 要素 | 說明 |
| :--- | :--- |
| 跨陣營副本 | M+、團隊副本支援聯盟/部落混合組隊 |
| 跨陣營公會 | 10.1 版本引入，公會成員可為雙陣營 |
| PvP | 競技場/戰場仍維繫陣營分流；戰爭模式保持陣營限定 |
| 拍賣場 | 拍賣場已跨陣營 (自 WoD) |
