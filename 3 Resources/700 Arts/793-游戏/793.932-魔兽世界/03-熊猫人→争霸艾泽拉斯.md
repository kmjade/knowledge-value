# 魔獸世界 — 熊貓人之謎至爭霸艾澤拉斯 (MoP to BfA: 90-120 Era)

> 2012-2020 年間，WoW 經歷了重大系統變革：彈性團隊、M+ 地城、神器武器、同盟種族。此階段奠定了現代 WoW 的 PvE 框架。

## 一、版本速覽

| 版本 | 等級 | 團隊規模 | 新大陸 | 核心系統創新 | 新職業/種族 |
| :--: | :-: | :-----: | :---: | :---------: | :--------: |
| MoP (5.x) | 90 | 10/25 + 彈性 | 潘達利亞 | 寵物對戰、彈性團隊、場景戰役 | 武僧、熊貓人 |
| WoD (6.x) | 100 | 20 史詩 / 彈性 | 德拉諾 | 要塞、史詩難度團隊 | — |
| Legion (7.x) | 110 | 20 史詩 / 彈性 | 破碎群島 | 神器武器、M+、世界任務、職業大廳 | 惡魔獵手 |
| BfA (8.x) | 120 | 20 史詩 / 彈性 | 庫爾提拉斯/贊達拉 | 同盟種族、戰爭模式、海島探險、艾澤萊晶岩 | 10 同盟種族 |

## 二、核心系統演變

### 團隊副本規模演化

| 版本 | 模式 | 說明 |
| :--: | :--- | :--- |
| Classic→Cata | 40 → 25/10 固定 | 固定規模，無彈性 |
| MoP | 25H/10H + 彈性 | 首次引入彈性 (Flex) 模式 (圍攻奧格瑪) |
| WoD | 史詩 (20 固定) / 英雄 (10-30) / 普通 / LFR | 確立四層難度體系 |
| Legion→TWW | 史詩 (20) / 英雄 (10-30) / 普通 / LFR | 標準化四層難度 |

### 難度分層體系

| 難度 | 英文 | 規模 | 鎖定機制 | 裝等差距 |
| :--: | :---: | :--: | :-----: | :-----: |
| 隨機團隊 | LFR (Looking for Raid) | 10-30 | 無鎖定 | 最低 |
| 普通 | Normal | 10-30 | 彈性 | +13 |
| 英雄 | Heroic | 10-30 | 伺服器鎖定 | +13 |
| 史詩 | Mythic | 20 固定 | 伺服器鎖定 (跨服需名人堂) | +13 |

### M+ 地城系統起源 (Legion 7.0)

| 要素 | 說明 |
| :--- | :--- |
| 起源 | 挑戰模式 (MoP/WoD) → 史詩鑰石地城 (Legion 7.0) |
| 機制 | 鑰石層級遞增 (M2→M+∞)，每層增加傷害/血量 |
| 詞綴 | 每週輪換，M2+/M4+/M7+/M10+ 各啟動不同詞綴 |
| 賽季 | 每賽季更換地城池 (4 舊 + 4 新) |
| 評分 | Raider.IO 評分 (BfA) → 遊戲內建評分 (Shadowlands+) |

## 三、神器與艾澤萊晶岩系統

### 神器武器 (Legion 7.0)

| 職業 | 專精 | 神器名稱 |
| :--: | :--: | :-----: |
| 戰士 | 武器/狂怒/防護 | 斯多姆卡·滅戰者 / 華爾拉亞之戰劍 / 鱗片壁壘 |
| 死亡騎士 | 血魄/冰霜/穢邪 | 詛咒之喉 / 墮落王子雙劍 / 天啟 |
| 惡魔獵手 | 災虐/復仇 | 欺詐者雙刃 / 奧達奇戰刃 |
| 聖騎士 | 神聖/防護/懲戒 | 白銀之手 / 真理守護者 / 灰燼使者 |

> 每把神器有獨特外觀解鎖、天賦樹 (神器之力)、聖物插槽系統。7.3 版本加入熔爐系統，擴展神器能力上限。

### 艾澤拉斯之心 (BfA 8.0)

| 系統 | 說明 |
| :--- | :--- |
| 艾澤拉斯之心 (Heart of Azeroth) | 取代神器武器的項鍊裝備 |
| 艾澤萊晶岩護甲 (Azerite Armor) | 頭/肩/胸三部位，解鎖特質環 |
| 精華系統 (Essences) | 8.2 引入，主動/被動鑲嵌效果 |
| 腐化系統 (Corruption) | 8.3 引入，強力負面效果換取屬性增益 |

### 系統對比

| 系統 | 版本 | 核心裝備 | 進階系統 | 評價 |
| :--- | :-: | :-----: | :-----: | :--: |
| 神器武器 | 7.0 | 武器+聖物 | 熔爐 (7.3) | 多數正面，外觀收集經典 |
| 艾澤萊晶岩 | 8.0-8.2 | 頭/肩/胸 | 精華 (8.2) | 初期負評，精華改善 |
| 腐化 | 8.3 | 全裝備 | 傳奇披風 | 強力但 RNG 重 |
