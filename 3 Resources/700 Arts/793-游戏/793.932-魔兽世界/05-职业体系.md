# 魔獸世界 — 職業體系 (Class Systems)

> WoW 共有 13 個職業，每個職業 2-4 個專精，分佈於坦/補/輸出三位一體。從經典天賦樹到 Dragonflight 雙軌天賦樹，再到 TWW 英雄天賦，職業系統持續演化。

## 六、資源系統

| 職業 | 資源類型 | 機制 |
| :--: | :-----: | :---: |
| 戰士 | 怒氣 (Rage) | 承受傷害/自動攻擊產生，技能消耗 |
| 聖騎士 | 聖能 (Holy Power) | 建構→消耗循環 (1-5) |
| 死亡騎士 | 符文 (Runes) + 符能 (Runic Power) | 雙資源系統 |
| 惡魔獵手 | 魔怒 (Fury) | 技能產生，核心技能消耗 |
| 武僧 | 能量 (Energy) + 真氣 (Chi) | 能量回復 + 真氣建構 |
| 德魯伊 | 怒氣/能量/法力 (依形態) | 形態決策資源 |
| 盜賊 | 能量 (Energy) + 連擊點數 (Combo Points) | 建構→終結技 |
| 薩滿 | 法力/漩渦 (Maelstrom) | 依專精 |
| 獵人 | 集中值 (Focus) | 穩定回復，技能消耗 |
| 喚能師 | 法力 + 精華 (Essence) | 蓄力法術不消耗 |
| 牧師 | 法力 + 瘋狂 (Insanity) | 暗影專精特殊 |
| 法師 | 法力 | 傳統法力池 |
| 術士 | 法力 + 靈魂裂片 (Soul Shards) | 建構→爆發 |
