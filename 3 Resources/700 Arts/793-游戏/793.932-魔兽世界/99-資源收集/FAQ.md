# 魔獸世界 — 常見問題 (FAQ)

> WoW 知識庫的快速問答索引。

## 一、入門

| 問題 | 答案 |
|------|------|
| 2024 年新手適合入坑嗎？ | 非常適合。TWW 推出戰隊系統與新玩家島，升級體驗現代化，克羅米時間讓新手自由選擇資料片 |
| WoW 需要買哪些版本？ | 僅需訂閱月費即可玩到 Dragonflight (等級 70) 之前全部內容。最新資料片 (TWW) 需單獨購買 |
| 經典版 (Classic) 和正式版 (Retail) 如何選擇？ | Classic：緩慢節奏、社交為重、懷舊體驗；Retail：快速升級、豐富終局內容、現代化系統 |
| 繁體中文支援如何？ | 台服 (亞服) 完全繁體中文在地化，含語音與文字 |
| 推薦哪個伺服器？ | 台服：暗影之月 (聯盟多) / 阿薩斯 (部落多)；美服：Tichondrius / Illidan (華人多) |

## 二、遊戲機制

| 問題 | 答案 |
|------|------|
| 裝等 (ilvl) 是什麼？ | 物品等級，決定裝備強度。為 WoW 玩家能力的最核心指標 |
| 什麼是寶庫 (Great Vault)？ | 每週重置後根據上週 PvE/PvP 表現獲得裝備三選一的系統 |
| M+ 是什麼？ | 史詩鑰石地城，遞增難度的 5 人地城挑戰系統，有時間限制與每週詞綴 |
| 賽季是多久？ | 通常 5-6 個月一個 PvE/PvP 賽季，每次伴隨新團隊副本與 M+ 地城池輪換 |
| 帳號綁定 (Warbound) 裝備是什麼？ | TWW 戰隊系統引入：可轉移給同帳號其他角色的裝備 |
| 飛龍騎術 (Dragonriding) 能在舊世界使用嗎？ | TWW 起全艾澤拉斯可使用飛龍騎術 (動量飛行) |
| 什麼是時光漫遊 (Timewalking)？ | 每週限時活動，可排隊體驗舊資料片地城/團隊，裝備等級同步 |

## 三、職業

| 問題 | 答案 |
|------|------|
| 新手推薦什麼職業？ | 獵人 (獸王)：遠程、寵物坦、容錯高；聖騎士：三系可坦可補可輸出；德魯伊：四系全體驗 |
| 最難的職業？ | 戒律牧師 (預判治療)、釀酒武僧 (醉拳管理)、敏銳盜賊 (隱身窗口期) |
| 什麼是英雄天賦 (Hero Talents)？ | TWW 新系統：71-80 級每專精可選兩條英雄天賦路線之一，增強核心技能與玩法 |
| 換專精需要重練嗎？ | 不需要。隨時切換專精，裝備需隨之更換 (部分共用) |
| 喚能師 (Evoker) 值得玩嗎？ | 獨特蓄力法術機制 + 增輝專精 (輔助型 DPS)，玩法獨一無二 |

## 四、PvE

| 問題 | 答案 |
|------|------|
| LFR 和普通團隊的區別？ | LFR 自動配對、難度最低、獎勵最低；普通需手動組隊，難度適中 |
| M+ 需要什麼裝備才能開始？ | M0 (593+) 即可開始 M2-4 層級，逐步提升 |
| 探究 (Delves) 能取代 M+ 嗎？ | 部分可以。探究獎勵可達英雄團隊級別，但僅限寶庫機制，無法完全取代 M+ 的無限層級挑戰 |
| 世界首殺 (RWF) 在哪看？ | Twitch 直播，Liquid (NA) / Echo (EU) 為主要競速公會 |
| 團隊副本要多少人？ | LFR/普通/英雄 10-30 人彈性；史詩固定 20 人 |

## 五、PvP

| 問題 | 答案 |
|------|------|
| PvP 裝備需要專門收集嗎？ | PvP 裝備在 PvP 環境中裝等自動提升 (+14)，但仍建議收集全套征服裝備 |
| Solo Shuffle 和競技場 3v3 的區別？ | Solo Shuffle 單排輪換 6 名玩家；3v3 需預組固定隊伍，獎勵更豐厚 (角鬥士) |
| 戰爭模式值得開嗎？ | 升級期間：+10-15% 經驗加成；滿等後：PvP 天賦解鎖 + 征服獲取 |

| 問題 | 答案 |
