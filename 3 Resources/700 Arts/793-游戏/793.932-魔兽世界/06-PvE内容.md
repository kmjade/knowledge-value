# 魔獸世界 — PvE 內容 (PvE Content)

> WoW PvE 涵蓋團隊副本、M+ 地城、世界任務、探究 (Delves) 等多層級內容。從 LFR 到世界首殺競賽，PvE 是 WoW 最核心的遊戲支柱。

## 一、團隊副本難度體系

### 四層難度

| 難度 | 英文 | 規模 | 裝等 (TWW S1) | 鎖定 | 目標玩家 |
| :--: | :---: | :--: | :----------: | :--: | :-----: |
| 隨機團隊 | LFR | 10-30 | 584-593 | 無鎖定 | 休閒/觀光 |
| 普通 | Normal | 10-30 | 597-606 | 彈性 (可跨服) | 一般玩家 |
| 英雄 | Heroic | 10-30 | 610-619 | 彈性 (可跨服) | 進階團隊 |
| 史詩 | Mythic | 20 固定 | 623-632 | 伺服器鎖定 (前期) | 核心團隊 |

### 世界首殺競賽 (Race to World First — RWF)

| 要素 | 說明 |
| :--- | :--- |
| 定義 | 全球團隊競速，爭奪最高難度團隊副本最終首領的首次擊殺 |
| 時間 | 通常在新團隊副本史詩難度開放後 1-3 週內完成 |
| 知名公會 | Liquid (NA) · Echo (EU) · Method (EU) |
| 直播 | Twitch 直播，數十萬人同時觀看 |
| 獎勵 | 無官方獎勵，純競技榮譽 |

### 傳奇難度團隊副本鎖定

| 時期 | 鎖定規則 |
| :--- | :-----: |
| 賽季初 (名人堂未滿) | 同伺服器鎖定：只能與同服玩家組隊 |
| 名人堂滿 200 名後 | 跨服開放：聯盟/部落各 100 名 |
| 賽季末 | 完全解鎖 |

## 二、M+ 史詩鑰石地城

### 鑰石層級系統

| 層級 | 血量/傷害加成 | 額外詞綴 | 獎勵 (終點) | 寶庫裝等 (TWW S1) |
| :--: | :----------: | :-----: | :--------: | :--------------: |
| M0 | 基礎 | 無 | 593 (探險者) | — |
| M2 | +7% | M2 詞綴 | 597 (冒險者) | 597 |
| M3 | +14% | M2 | 597 | 597 |
| M4 | +22% | M2 + M4 詞綴 | 600 (冒險者) | 600 |
| M5 | +30% | M2 + M4 | 603 | 603 |
| M6 | +39% | M2 + M4 | 606 | 606 |
| M7 | +48% | M2 + M4 + M7 詞綴 | 610 (老兵) | 610 |
| M8 | +58% | M2 + M4 + M7 | 610 | 610 |
| M9 | +68% | M2 + M4 + M7 | 613 | 613 |
| M10 | +79% | M2 + M4 + M7 + M10 詞綴 | 613 | 616 (英雄) |

> 每層級疊加 ~7-10% 怪物血量與傷害；M10+ 後無新詞綴，純數值增長。

### TWW S1 詞綴體系

| 啟動層級 | 詞綴類別 | 詞綴列表 |
| :-----: | :-----: | :-----: |
| M2 | 舊詞綴 (輪換) | 暴君 (Tyrannical) / 強悍 (Fortified) |
| M4 | 新·舊詞綴 (輪換) | 激勵 (Inspiring) · 屍爆 (Bursting) · 膿瘡 (Sanguine) · 惡意 (Spiteful) 等 |
| M7 | 專屬詞綴 (輪換) | 火山 (Volcanic) · 風暴 (Storming) · 糾纏 (Entangling) 等 |
| M10 | 賽季詞綴 | 薩拉塔斯的交易：每賽季變化 (S1：吸收球) |

### M+ 評分系統

| 評分來源 | 說明 |
| :-----: | :--- |
| M+ 評分 (Mythic+ Rating) | 遊戲內建，基於全地城最高層級與完成時間 |
| Raider.IO | 第三方插件/網站，更精細評分 + 歷史記錄 |
| 獎勵門檻 | 2,000 分 → 傳送門獎勵；2,500 分 → 成就 |
| 鑰石大師 (KSM) | 賽季內達指定分數門檻，獲得坐騎獎勵 |
| 鑰石英雄 (KSH) | 更高門檻，外觀解鎖 |

## 三、世界任務與戶外 PvE

| 系統 | 說明 | 版本 |
| :--- | :--- | :--: |
| 世界任務 (World Quests) | 每日輪換的戶外任務，獎勵金幣/裝備/聲望 | Legion→現在 |
| 特使任務 (Emissary) | 完成 3 個指定區域世界任務，獲額外獎勵 | Legion→BfA |
| 呼喚 (Callings) | 暗影國度版的每天精選任務 | Shadowlands |
| 世界首領 (World Boss) | 每週刷新，掉落高等級裝備 | 全版本 |
| 稀有精英 (Rares) | 地圖標記稀有怪，掉落坐騎/玩具/裝備 | 全版本 |
| 區域事件 (Zone Events) | 軍團入侵、阿拉希高地前線、巨龍群島風暴等 | Legion→現在 |

## 四、探究 (Delves) — TWW 全新 PvE

### 探究機制

| 要素 | 說明 |
| :--- | :--- |
| 規模 | 1-5 人 (含布萊恩·銅鬚 NPC 同伴) |
| 層級 | Tier 1-11，越高越難 |
| 賽季輪換 | 每賽季更新探究場景與主題 |
| 獎勵 | 寶庫解鎖 + 終點寶箱，可達英雄團隊級別 |
| 目標 | 填補單人/小團隊玩家與團隊/M+ 之間的獎勵鴻溝 |

### 探究類型

| 類型 | 描述 |
| :--- | :--- |
| 礦坑 (Mines) | 清除礦井敵怪，收集資源 |
| 挖掘 (Excavation) | 考古主題，尋找遺物 |
| 洞穴 (Caverns) | 探索地下洞穴，擊敗最終首領 |
| 真菌 (Fungal) | 孢子環境，特殊機制 |

## 五、每週活動輪換

| 活動 | 類型 | 獎勵 |
| :--- | :--- | :--- |
| 時光漫遊 (Timewalking) | 舊資料片地城/團隊 | 時光扭曲徽章、坐騎 |

| 寵物對戰獎勵週 | 寵物對戰加成 | 寵物經驗 |
| M+ 獎勵週 | 地城尾王多一件掉落 | 額外裝備 |

## 六、寶庫系統 (Great Vault)

| 內容類型 | 解鎖條件 | 裝備等級 |
| :-----: | :-----: | :-----: |
| 團隊副本 | 擊敗 2/4/6 個團隊首領 | 依難度 (LFR→史詩) |
| M+ 地城 | 完成 1/4/8 次 M+ | 依最高層級 |
| 探究 | 完成 2/4/8 次探究 | 依最高層級 |
| PvP | 獲得榮譽點數 | 依 PvP 階級 |

> 每週最多獲得一件裝備 (三選一)，為 WoW 主要裝備獲取途徑。
