# 04-UI与UX设计

> UI(用户界面) = 看起来如何 / UX(用户体验) = 用起来如何。

## UX的核心

| 可用性 = 用户能否完成任务？ / 愉悦性 = 用户完成任务时开心吗？ |
|------|
| 「别让我思考」(Don't Make Me Think) ——Steve Krug——UX的金科玉律 |

## UI设计趋势

| 扁平化(2013-): iOS7起——去阴影·去质感——极简 |
|------|
| 新拟态(Neumorphism): 软阴影——像浮雕——好看但可访问性差 |
| 暗黑模式: 省电·护眼——几乎每个App都有了 |

## 移动优先

| 先设计手机版——再扩展到桌面——不是反过来的 |
|------|
