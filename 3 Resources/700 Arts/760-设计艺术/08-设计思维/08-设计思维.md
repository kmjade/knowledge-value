# 08-设计思维 (Design Thinking)

> 设计师解决问题的方法——不只是给设计师用的——任何人都可以用。

## 五个阶段

| 阶段 | 内容 |
|------|------|
| 共情 (Empathize) | 理解用户——不是「我觉得」——是「他们需要」——观察·访谈 |
| 定义 (Define) | 把问题说清楚——「我们如何帮助X完成Y？」 |
| 构思 (Ideate) | 大量想法——不评判——先把量做上去——然后收敛 |
| 原型 (Prototype) | 快速·便宜——纸模型也可以——「如果图片值千言——原型值千图」 |
| 测试 (Test) | 让用户用——失败越早越便宜——迭代 |

## 双钻模型

| 发散→收敛——发散→收敛——打开问题空间——定义——打开解决方案——交付 |
