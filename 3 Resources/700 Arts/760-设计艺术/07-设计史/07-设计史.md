# 07-设计史

> 设计的历史是「大众」取代「精英」的历史——工业化让好东西不再是富人的专利。

## 关键节点

| 时期 | 事件 |
|:---:|------|
| 1851 | 水晶宫博览会——展示工业品——但丑陋——「设计的诞生」是对工业丑陋的回应 |
| 1919 | 包豪斯——艺术+工艺+技术——现代设计教育的鼻祖——师生名单是「谁是谁」 |
| 1950s | 德国乌尔姆——从包豪斯的手工艺→科学化设计——系统设计 |
| 1980s | 后现代——孟菲斯——「少即是无聊」——大胆色彩·几何——好玩 |
| 2000s | 数字设计——UI/UX——设计思维——设计从「物」走向「体验」 |

## 中国设计

| 民国: 图案教育 / 1949后: 实用美术 / 改革开放: 从代工到原创——「中国设计」仍在形成中 |
