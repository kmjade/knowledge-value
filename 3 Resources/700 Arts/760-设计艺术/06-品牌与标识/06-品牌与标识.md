# 06-品牌与标识设计

> Logo ≠ 品牌。Logo是品牌的「脸」——品牌是消费者心中的「人」。

## Logo 设计的类型

| 类型 | 例子 |
|------|------|
| 文字标 | Google·Coca-Cola——文字本身就是Logo |
| 图形标 | Apple(苹果)·Twitter(鸟)——单独图形 |
| 组合标 | 图形+文字——最常见 |
| 抽象标 | Nike(勾)——不代表任何具体东西——但全世界都知道 |

## 品牌不只是Logo

| 品牌 = 消费者的所有联想——颜色·声音·气味·体验——每一次接触都在塑造品牌 |
|------|
| 「品牌是一个承诺——一个公司做出的——消费者记住的」 |
