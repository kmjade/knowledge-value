# 02-平面设计

> 文字+图像在二维平面上的编排——从海报到App界面。

## 核心原则 (CRAP)

| 原则 | 说明 |
|------|------|
| 对比 (Contrast) | 差异越大越好——不要「差不多」——大胆 |
| 重复 (Repetition) | 一致的视觉元素——建立节奏 |
| 对齐 (Alignment) | 元素之间要有视觉联系——不是「随便放」 |
| 亲密性 (Proximity) | 相关的近——不相关的远——组织信息 |

## 网格系统

| 网格 = 隐形的骨架——提供秩序——打破网格之前——先学会用网格 |
|------|
| 瑞士设计(国际主义风格): 网格·无衬线字体·不对称——Helvetica——现代平面设计的「语法」 |
