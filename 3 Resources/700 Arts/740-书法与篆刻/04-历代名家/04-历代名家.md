# 04-历代书法名家

> 「书圣」只有一个——王羲之——但每个时代都有自己的巨人。

| 书家 | 时代 | 成就 |
|------|:---:|------|
| 王羲之 | 东晋 | 书圣——《兰亭序》天下第一行书——真迹已失传(唐太宗带入陵墓) |
| 颜真卿 | 唐 | 《祭侄文稿》天下第二行书——雄浑——「字如其人」(忠烈) |
| 柳公权 | 唐 | 「颜筋柳骨」——骨力——「用笔在心——心正则笔正」 |
| 欧阳询 | 唐 | 楷书极致——「欧体」——初学者入门范本 |
| 苏轼 | 宋 | 《黄州寒食帖》天下第三行书——文人书法的代表 |
| 米芾 | 宋 | 疯狂——「刷字」——八面出锋——个人风格极强 |
| 赵孟頫 | 元 | 复古——回归晋唐——优雅——但被批评「缺乏骨气」(降元) |
| 董其昌 | 明 | 影响最大——「平淡天真」——清代皇帝学他的字 |
