# 05-篆刻基础

> 方寸之间——气象万千——刻印是书法+雕刻+设计的结合。

## 印章的类型

| 类型 | 特征 |
|------|------|
| 白文(阴刻) | 文字凹下——钤出白字——沉着 |
| 朱文(阳刻) | 文字凸起——钤出红字——清秀 |
| 朱白相间 | 混合——变化 |

## 篆刻三要素

| 要素 | 说明 |
|------|------|
| 篆法 | 用什么字体——主要用篆书——要懂六书(造字原理) |
| 章法 | 方寸之内的布局——疏密·虚实·呼应 |
| 刀法 | 冲刀(推)·切刀(切)——石头的阻力反哺了风格 |

## 著名流派

| 浙派(西泠八家): 切刀——刚健 / 皖派(邓石如): 「书从印入——印从书出」 |
|------|
