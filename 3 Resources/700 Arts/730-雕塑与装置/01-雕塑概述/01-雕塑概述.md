# 01-雕塑概述

> 三维的艺术——从石头里「释放」出形象——米开朗基罗如是说。

## 雕塑的类型

| 类型 | 说明 |
|------|------|
| 圆雕 | 360°可观看——完全独立 |
| 浮雕 | 附着在背景上——浅浮雕(硬币)·高浮雕 |
| 环境 | 整个空间——观者走入其中 |
| 动态 | 会动的——Calder的活动雕塑 |
| 装置 | 物体+空间的关系——不一定是「雕刻」出来的 |

## 雕塑的四要素

| 要素 | 说明 |
|------|------|
| 体量 | 占据空间——雕塑是空间的「入侵者」 |
| 质感 | 光滑·粗糙——大理石 vs 青铜 vs 木材 |
| 光线 | 雕塑是和光线的合作——光改变雕塑的观感 |
| 空间 | 雕塑周围的「空」也是雕塑的一部分 |
