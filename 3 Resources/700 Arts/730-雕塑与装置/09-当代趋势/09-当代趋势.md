# 09-当代趋势

> 雕塑已经溢出到空间·时间·体验——边界不断消融。

## 趋势

| 趋势 | 说明 |
|------|------|
| 数字雕塑 | 3D建模→打印——不需要「手」作——算法生成形式 |
| 生物艺术 | 用活体组织——「生长」的雕塑——伦理争议 |
| 参与式 | 观者变成「完成者」——作品没有固定形态 |
| 临时性 | 存在几小时或几天——然后消失——只有照片留下 |

## 数字时代

| VR雕塑: 在虚拟空间中「雕刻」——没有重力·可以无限大——但「看」需要设备 |
|------|
| NFT: 数字雕塑的「原件」概念——稀缺性人为创造 |
