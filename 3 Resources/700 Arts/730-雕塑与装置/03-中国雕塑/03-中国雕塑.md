# 03-中国雕塑

> 中国雕塑长期被忽视——但兵马俑之后——世界重新审视。

## 历代特征

| 时期 | 代表作 | 特征 |
|------|------|------|
| 先秦 | 青铜器(司母戊鼎) | 礼器——装饰性 |
| 秦 | 兵马俑 | 真人大小——写实——8000+——个个不同 |
| 汉 | 霍去病墓石雕 | 写意——利用石头天然形状——「马踏匈奴」 |
| 南北朝-唐 | 敦煌·云冈·龙门 | 佛教雕塑——印度影响+中国化 |
| 宋 | 晋祠侍女像 | 世俗化——有血有肉的人 |
| 明清 | 装饰雕塑 | 工艺精湛——但缺乏创新 |

## 「雕塑」在中国

| 中国古代没有「雕塑家」这个概念——制作者是「工匠」——不被看作艺术家 |
|------|
| 直到近代——受西方影响——才有了「雕塑」这个艺术门类 |
