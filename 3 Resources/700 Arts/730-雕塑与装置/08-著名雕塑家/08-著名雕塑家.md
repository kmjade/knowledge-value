# 08-著名雕塑家

| 雕塑家 | 时代 | 成就 |
|------|:---:|------|
| 菲迪亚斯 | 古希腊 | 帕特农神庙雕塑——虽已不存——但罗马仿制品确立了古典标准 |
| 米开朗基罗 | 文艺复兴 | 「大卫」「哀悼基督」——从石头里「释放」生命 |
| 罗丹 | 19-20世纪 | 「思想者」——现代雕塑之父——打破光滑传统 |
| 布朗库西 | 20世纪 | 极简——「鸟在空间」——被美国海关当作「金属零件」征税 |
| 亨利·摩尔 | 20世纪 | 孔洞·母性——最成功的现代雕塑家——遍布全球 |
| 野口勇 | 20世纪 | 日裔美国人——雕塑+园林+家具——跨界 |
| 展望 | 当代中国 | 「假山石」——不锈钢复制太湖石——传统vs现代 |
