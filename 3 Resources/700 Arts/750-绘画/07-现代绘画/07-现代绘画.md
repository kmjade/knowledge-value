# 07-现代绘画

> 1900-1970——绘画对自己发起了革命——每一代都问「绘画还能是什么？」

## 关键运动

| 运动 | 核心 |
|------|------|
| 野兽派(1905) | 马蒂斯——色彩不是描绘——是表达——女人脸是绿的——不重要 |
| 立体主义(1907) | 毕加索——打碎再拼——多个视角——「像相机拍了几百张照片叠在一起」 |
| 未来主义(1909) | 速度——机械——动感——「飞驰的汽车比胜利女神更美」 |
| 抽象(1910s-) | 康定斯基——音乐没有「像什么」——画也可以——纯粹的形式 |
| 达达(1916) | 反艺术——反理性——杜尚——小便池——「一切都是艺术——艺术什么都不是」 |
| 超现实主义(1920s) | 达利——梦境·潜意识——钟表融化——不合逻辑但「感觉真实」 |
| 抽象表现主义(1940s) | 波洛克——滴画——画布铺地上——「在画中行走」 |
