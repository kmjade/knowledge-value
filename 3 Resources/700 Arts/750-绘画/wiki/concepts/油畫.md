# 油畫 Oil Painting

## 定義
油畫使用**乾性油**（亞麻仁油、核桃油）調和顏料，在畫布或木板上作畫。

## 特點
- 乾燥慢 → 可反覆修改、暈染
- 色彩飽和、層次豐富
- 可薄塗 (glaze) 或厚塗 (impasto)

## 代表人物
| 畫家 | 風格 |
|------|------|
| Jan van Eyck | 油畫技法先驅 |
| Rembrandt | 光影大師 (chiaroscuro) |
| **梵谷** | 厚塗、強烈筆觸 |
