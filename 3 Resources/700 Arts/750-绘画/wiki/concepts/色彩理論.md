# 色彩理論 Color Theory

## 色彩輪
| 關係 | 說明 |
|------|------|
| **互補色** | 色輪對面的顏色（紅←→綠） |
| **類似色** | 相鄰的顏色（和諧） |
| **三色組** | 均勻間隔的三色（活力） |

## 色彩屬性
- **色相 Hue** — 顏色的名稱
- **飽和度 Saturation** — 顏色的純度
- **明度 Value** — 顏色的明暗
