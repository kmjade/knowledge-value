# 透視法 Perspective

## 定義
透視法是在二維平面上創造**三維空間深度**的技術。

## 類型
| 類型 | 說明 |
|------|------|
| **線性透視** | 平行線匯聚於消失點 (文藝復興發明) |
| **大氣透視** | 遠處物體顏色變淡、變藍 |
| **色彩透視** | 暖色前進、冷色後退 |
