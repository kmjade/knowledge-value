# 繪畫流派 Painting Schools

## 主要流派
| 流派 | 年代 | 特點 |
|------|:--:|------|
| **文藝復興** | 1400–1600 | 透視、解剖、人文 |
| **巴洛克** | 1600–1750 | 動感、光影對比 |
| **洛可可** | 1700s | 華麗、輕盈 |
| **新古典** | 1780–1850 | 回歸希臘羅馬 |
| **浪漫主義** | 1800–1850 | 情感、自然 |
| **印象派** | 1860–1900 | 光與色彩 |
| **表現主義** | 1900– | 主觀情感 (梵谷, Munch) |
| **抽象** | 1910– | 純粹形式 (Kandinsky) |
