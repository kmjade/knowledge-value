# 構圖 Composition

## 定義
構圖是畫面上視覺元素的**組織和安排**，引導觀者的視線。

## 經典構圖
| 構圖 | 說明 |
|------|------|
| **三分法 Rule of Thirds** | 主體放在 1/3 處 |
| **黃金比例** | 1:1.618 |
| **三角形構圖** | 穩定感（文藝復興常用） |
| **S 形構圖** | 引導視線流動 |
