# 03-西方油画

> 从杨·凡·艾克发明油画(15世纪)到当代——600年的传统。

## 油画的特征

| 油作为介质——干燥慢——可以反复修改·叠加——光泽·深度 |
|------|
| 和蛋彩(tempera)比——油画更灵活——可以薄涂(glaze)也可以厚涂(impasto) |

## 流派速览

| 流派 | 时间 | 特征 |
|------|:---:|------|
| 文艺复兴 | 15-16c | 透视·解剖——达芬奇·拉斐尔 |
| 巴洛克 | 17c | 戏剧性·光影——卡拉瓦乔·伦勃朗 |
| 洛可可 | 18c | 轻佻·粉彩——布歇·弗拉戈纳尔 |
| 新古典 | 18-19c | 回归希腊罗马——大卫·安格尔 |
| 浪漫主义 | 19c | 情感·自然——德拉克罗瓦 |
| 印象派 | 19c末 | 光与色——莫奈·雷诺阿——「未完成」 |
| 后印象 | 19-20c | 梵高·塞尚·高更——各自走向现代 |
| 立体主义 | 20c | 毕加索——多个视角同时在平面上 |
| 抽象 | 20c | 蒙德里安·康定斯基——纯粹的形式 |
