# 01-绘画概述

> 平面的艺术——用颜料在表面上创造图像——从岩画到iPad。

## 绘画的要素

| 要素 | 说明 |
|------|------|
| 线条 | 轮廓——也可以是情绪——急促/流畅 |
| 色彩 | 冷暖·对比·和谐——色彩本身可以表达情感 |
| 构图 | 元素如何安排——三分法·对角线·对称 |
| 光影 | 明暗——体积感——卡拉瓦乔的「明暗对照」 |

## 绘画的类型

| 西方 | 油画·水彩·丙烯·蛋彩·壁画 |
|------|------|
| 中国 | 水墨(写意·工笔)·重彩·界画 |
| 其他 | 日本浮世绘·波斯细密画·印度壁画 |
