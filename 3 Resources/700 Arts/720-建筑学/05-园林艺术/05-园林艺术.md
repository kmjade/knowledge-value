# 05-园林艺术

> 园林 = 理想世界的微缩——中国人造山水——西方人驯服自然。

## 中国园林

| 特征 | 说明 |
|------|------|
| 师法自然 | 「虽由人作——宛自天开」——模仿自然——但不复制 |
| 借景 | 把园外的山·塔「借」进园内——扩大空间感 |
| 曲径通幽 | 不是一眼看尽——曲折·遮挡·「别有洞天」 |
| 四季 | 选择不同季节开花的植物——确保任何时候都有景 |

## 苏州园林

| 拙政园·留园·网师园——世界文化遗产——「咫尺之内再造乾坤」 |
|------|

## 日本园林

| 枯山水: 砂=水·石=山——不是「看」——是「冥想」——龙安寺石庭 |
|------|

## 西方园林

| 法式(凡尔赛): 几何·对称·征服自然——人是中心 |
|------|
| 英式: 自然风景——「如画」(picturesque) ——浪漫主义的影响 |
