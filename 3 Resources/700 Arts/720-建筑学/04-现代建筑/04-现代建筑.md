# 04-现代建筑

> 「形式追随功能」(Louis Sullivan) ——「少即是多」(Mies van der Rohe)。

## 现代主义的原则

| 原则 | 说明 |
|------|------|
| 功能决定形式 | 不需要装饰——结构本身就是美 |
| 新材料 | 钢+玻璃+混凝土——工业革命带来的可能性 |
| 开放平面 | 墙不再承重——空间可以自由分割 |
| 诚实表达 | 不伪装——混凝土就是混凝土——不涂成石头色 |

## 包豪斯 (1919-1933)

| 一所学校改变了世界——艺术+工艺+技术——「让建筑成为一切艺术的终极目标」 |

## 后现代与当代

| 后现代: 「少即是无聊」——把历史元素当「引用」——像穿戏服 |
|------|
| 参数化: 计算机算法生成形式——扎哈·哈迪德的流体曲线——「不可能」的形状 |
