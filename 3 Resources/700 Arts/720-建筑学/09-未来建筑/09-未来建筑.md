# 09-未来建筑

> 3D打印房子？火星基地？——建筑正在被重新定义。

## 趋势

| 趋势 | 说明 |
|------|------|
| 3D打印 | 24小时打印一座房子——减少人工·降低成本——月球基地？ |
| 模块化 | 工厂造→现场拼——速度快·质量可控 |
| 智能建筑 | 传感器·AI——建筑「知道」你冷了你热了 |
| 木结构回归 | 工程木材(CLT) ——可以盖摩天楼——低碳 |
| 火星建筑 | 3D打印火星土壤——穹顶——NASA竞赛 |

## 垂直城市

| 城市向上——沙特「The Line」: 170公里长·500米高——镜像墙——疯狂还是未来？ |

## 建筑与气候

| 建筑消耗40%能源——未来建筑必须产出的能量 ≥ 消耗的能量 |
