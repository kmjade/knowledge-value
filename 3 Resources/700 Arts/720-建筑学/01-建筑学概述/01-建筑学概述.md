# 01-建筑学概述

> 建筑 = 实用+坚固+美观 (Vitruvius, ~25 BCE) ——2000年后仍然是最简洁的定义。

## 建筑三原则

| 原则 | 含义 |
|------|------|
| 实用 (utilitas) | 能用——功能——不能满足功能的建筑是雕塑(或失败) |
| 坚固 (firmitas) | 不塌——结构——物理的约束 |
| 美观 (venustas) | 好看——美学——这是把建筑变成艺术的部分 |

## 建筑的类型

| 类型 | 例子 |
|------|------|
| 居住 | 住宅·公寓——最日常——也最直接影响生活 |
| 宗教 | 教堂·寺庙·清真寺——往往是一个文明最高建筑成就 |
| 公共 | 政府·博物馆·图书馆——公民空间 |
| 商业 | 办公楼·商场——现代城市的主体 |
| 基础设施 | 桥梁·车站·机场——工程的建筑表达 |
