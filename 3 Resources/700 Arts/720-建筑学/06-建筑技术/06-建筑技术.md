# 06-建筑技术

> 建筑是工程+艺术——技术决定了「能盖成什么样」。

## 结构系统

| 系统 | 原理 | 代表 |
|------|------|------|
| 梁柱 | 最古老——水平梁+垂直柱 | 古希腊神庙 |
| 拱 | 压力转化为推力 | 罗马水道·哥特教堂 |
| 桁架 | 三角形——轻+强 | 桥梁·大跨度屋顶 |
| 壳体 | 曲面——薄壳——如蛋壳 | 悉尼歌剧院 |
| 悬索 | 张力——反重力 | 金门大桥 |

## 材料

| 材料 | 特征 |
|------|------|
| 木材 | 可再生·温暖·怕火——中国传统主角 |
| 石材 | 耐久·受压强受拉弱——西方传统主角 |
| 钢 | 拉伸强——让摩天楼成为可能 |
| 混凝土 | 可塑·防火——现代建筑的主角 |
| 玻璃 | 透明——幕墙——但能耗高 |

## 绿色建筑

| LEED·被动房·零能耗——建筑占全球碳排放~40%——必须改变 |
