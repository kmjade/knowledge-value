# 03-中国古建筑

> 木结构+斗拱+院落——不同于西方的石结构传统。

## 木结构的逻辑

| 为什么是木？——抗震(柔性连接)·可再生·快速建造——但怕火——所以古建筑留存少 |
|------|
| 斗拱: 梁柱之间的榫卯连接——把屋顶重量传递到柱子——不用一颗钉子 |

## 中国建筑的哲学

| 原则 | 说明 |
|------|------|
| 对称 | 中轴线——主次分明——故宫是最极致的表达 |
| 院落 | 不是单体——是群体——四合院——「建筑是围合空间的艺术」 |
| 天人合一 | 建筑融入自然——不是征服——看苏州园林 |
| 等级 | 屋顶形式·颜色·装饰——从庑殿顶(最高)·歇山·悬山到硬山 |

## 代表作

| 故宫(600年·世界最大宫殿)·天坛(宇宙观的建筑表达)·应县木塔(1000年·世界最高木塔) |
