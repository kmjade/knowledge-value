# 競品分析工作流

使用 Claude Code 進行自動化競品分析的完整工作流程。

---

## 工作流程

```
1. 定義競品列表
      ↓
2. Agent Browser 抓取信息
      ↓
3. Skills 分析處理
      ↓
4. 生成結構化報告
```

---

## 效率對比

| 指標 | 傳統方式 | 自動化方式 |
|------|----------|------------|
| 5 個競品 | 2 天 | 2 小時 |
| 信息收集 | 手動搜索 | 自動抓取 |
| 分析報告 | 手動撰寫 | 自動生成 |
