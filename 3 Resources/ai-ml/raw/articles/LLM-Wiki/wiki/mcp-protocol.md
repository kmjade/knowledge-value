# MCP Protocol

Model Context Protocol（模型上下文協議），Anthropic 推出的開放協議，讓 AI 模型能夠安全地連接外部工具和數據源。

---

## 核心概念

- **標準化接口**：統一的工具調用協議
- **安全連接**：AI 通過 MCP 安全訪問外部資源
- **生態整合**：支持多種工具和服務

---

## 應用場景

| 場景 | 說明 |
|------|------|
| n8n MCP Server | 讓 [[3 Resources/ai-ml/raw/articles/LLM-Wiki/wiki/claude-code]] 能直接操作 [[n8n]] |
| 數據庫訪問 | AI 安全查詢數據庫 |
| API 整合 | 標準化調用外部 API |
