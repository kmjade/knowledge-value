# 应用科学 常见问题

## 十大常见误区

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| 应用科学比基础科学"低级" | 两者是共生关系——没有 Maxwell 方程就没有无线电（基础→应用），没有显微镜就没有细胞学说（应用→基础） |
| 医学 = 吃药+手术 | 现代医学是循证医学——流行病学、临床试验、卫生经济学、预防医学、公共卫生，药物和手术只是其中一部分 |
| 天然 = 安全，化学 = 有害 | "天然"物质也能剧毒（蓖麻毒素、河豚毒素）。安全性取决于剂量和化学结构，而非来源 |
| 基因编辑 = 设计婴儿 | CRISPR 的临床应用主要在治疗遗传病和癌症（体细胞编辑），生殖细胞编辑（可遗传）有严格的伦理禁令 |
| 工程 = 应用物理 | 工程有自己的方法论——系统工程、可靠性工程、安全工程，物理只是工具箱之一 |
| 可再生能源已经取代化石燃料 | 2026 年全球化石燃料仍占一次能源的约 80%。转型需要数十年 |
| 核能 = 切尔诺贝利/福岛 | 核能是单位发电量致死人数最少的能源之一（低于煤炭数百倍）。第四代反应堆更安全 |
| 纳米技术只是"做小东西" | 在纳米尺度，物质的物理化学性质发生变化——量子效应主导，这是全新的设计空间 |
| 疫苗里的"有害成分" | 铝佐剂用量极低（< 0.85mg/剂），婴儿每天从母乳中摄取的铝更多。硫柳汞（含汞防腐剂）在大多数疫苗中早已移除 |
| 气候变化只是"变暖" | 气候变化是系统性的：极端天气频率增加、海洋酸化、生物多样性丧失、粮食安全风险——不只是温度升高几度 |

## 如何分辨真应用科学 vs 伪科学炒作？

| 迹象 | 含义 |
|------|------|
| "革命性突破" + 没有发表在同行评审期刊 | 可能是夸大 |
| 声称治疗多种不相关疾病 | 万灵药不存在 |
| 只有公司新闻稿，没有独立验证 | 利益冲突 |
| 基于"古老智慧"而非现代证据 | 传统知识有价值，但需要科学验证 |
| 技术成熟度被跳过——从 TRL 2 直接跳到"产品即将上市" | 忽略工程化的漫长过程 |
