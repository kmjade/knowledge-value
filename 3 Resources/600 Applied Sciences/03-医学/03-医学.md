# 03-医学

> 医学是应用科学中最古老、与每个人最相关的分支——从希波克拉底誓言到 CRISPR 基因编辑，从放血疗法到 mRNA 疫苗。

## 现代医学的四根支柱

| 支柱 | 核心问题 | 工具 |
|------|----------|------|
| **循证医学 (EBM)** | 这真的有效吗？ | 随机对照试验 (RCT)、系统综述、Meta 分析 |
| **精准医学** | 为什么对张三有效对李四无效？ | 基因组学、蛋白质组学、生物标志物 |
| **预防医学** | 如何在生病前阻止？ | 疫苗、筛查、生活方式干预 |
| **公共卫生** | 如何保护整个人口？ | 流行病学、卫生政策、健康传播 |

## 循证医学的证据金字塔

```
         ┌──────────────┐
         │ 系统综述/Meta  │  ← 最高质量
         ├──────────────┤
         │  随机对照试验   │  ← 金标准
         ├──────────────┤
         │  队列研究      │
         ├──────────────┤
         │  病例对照研究   │
         ├──────────────┤
         │  病例系列      │
         ├──────────────┤
         │  专家意见      │  ← 最低质量
         └──────────────┘
```

> RCT（随机对照试验）为什么是"金标准"？因为它通过随机分组消除了混杂因素。但你不可能 RCT 一切——你不能随机分配人去吸烟 20 年。

## 药物研发流程

```
基础研究 → 靶点发现 → 先导化合物 → 临床前试验 → 
  (2-5年)    (1-2年)    (1-2年)      (1-2年)

Phase I（安全性） → Phase II（有效性） → Phase III（大规模验证） → 
  (1-2年, 20-80人)    (1-2年, 100-300人)    (2-3年, 1000-3000人)

FDA/NMPA 审批 → Phase IV（上市后监测）
  (1-2年)           (持续)
```

| 数据 | 数值 |
|------|------|
| 总时间 | 10-15 年 |
| 总成本 | 平均 $1-2.6B（含失败药物分摊） |
| 成功率 | 进入 Phase I 的药物只有 ~10% 最终获批 |

## 重大医学突破

| 突破 | 年代 | 影响 |
|------|------|------|
| **疫苗** (Jenner, 1796) | 牛痘预防天花 | 最终消灭了天花 (1980) |
| **抗生素** (Fleming, 1928) | 青霉素 | 细菌感染从致命变为可治疗 |
| **DNA 双螺旋** (1953) | 分子生物学起点 | 精准医学的基础 |
| **CT/MRI** (1970s-80s) | 无创看体内 | 诊断革命 |
| **HIV 抗逆转录病毒** (1996) | 从绝症到慢性病 | 寿命接近正常 |
| **人类基因组计划** (2003) | 30 亿碱基对测序 | 精准医学的基础 |
| **免疫检查点抑制剂** (2011) | 癌症免疫疗法 | 诺贝尔奖 2018 |
| **CRISPR-Cas9** (2012) | 基因编辑 | 诺贝尔奖 2020 |
| **mRNA 疫苗** (2020) | COVID-19 疫苗 | 疫苗技术范式转变 |
