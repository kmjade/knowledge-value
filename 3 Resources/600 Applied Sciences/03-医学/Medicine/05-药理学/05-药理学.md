# 05-药理学

> 药理学研究药物如何影响身体（药效学）和身体如何处理药物（药代动力学）。简单说：**药物对你做了什么，你对药物做了什么。**

## 药代动力学 (ADME)

药物在体内的四个过程：

| 过程 | 含义 | 影响因素 |
|------|------|----------|
| **A**bsorption (吸收) | 药物如何进入血液 | 给药途径（口服/注射/吸入）|
| **D**istribution (分布) | 药物在体内的扩散 | 血流、蛋白结合、血脑屏障 |
| **M**etabolism (代谢) | 身体如何改造/破坏药物 | 肝脏（CYP450 酶系） |
| **E**xcretion (排泄) | 药物如何离开身体 | 肾脏（尿）、肝脏（胆汁→粪便） |

> 肝脏代谢 + 肾脏排泄是两大清除途径。肝或肾功能不全的患者，药物剂量需要调整——否则可能中毒。

### 半衰期

药物浓度降到一半所需的时间。**约 5 个半衰期后药物基本从体内清除。**

| 药物 | 半衰期 | 意义 |
|------|--------|------|
| 肾上腺素 | ~2 分钟 | 急救用，作用极快 |
| 布洛芬 | ~2 小时 | 需要每 6-8 小时服药 |
| 地西泮（安定） | 20-100 小时 | 长期服用会蓄积 |

## 药效学

### 药物如何起作用

| 机制 | 说明 | 例子 |
|------|------|------|
| **受体激动剂** | 激活受体 → 产生效果 | 沙丁胺醇激活 β₂ 受体→扩张支气管 |
| **受体拮抗剂** | 阻断受体 → 阻止效果 | 普萘洛尔阻断 β 受体→降心率降压 |
| **酶抑制剂** | 抑制酶的活性 | 阿司匹林抑制 COX→抗炎止痛 |
| **离子通道调节** | 阻断/开放离子通道 | 硝苯地平阻断 Ca²⁺ 通道→降血压 |
| **转运体抑制** | 阻止物质转运 | SSRIs 抑制 5-HT 再摄取→抗抑郁 |

### 治疗指数

```
治疗指数 (TI) = TD50 / ED50

TD50 = 50% 个体中毒的剂量
ED50 = 50% 个体有效的剂量
```

| TI | 安全性 | 例子 |
|----|--------|------|
| 大 (>10) | 安全 | 青霉素 |
| 小 (<2) | 危险，需监测血药浓度 | 华法林、地高辛、锂盐 |

## 抗生素

### 作用机制分类

| 机制 | 代表 | 靶点 |
|------|------|------|
| 抑制细胞壁合成 | 青霉素、头孢 | 细菌有细胞壁，人没有 → 选择性毒性 |
| 抑制蛋白质合成 | 四环素、红霉素 | 细菌核糖体 |
| 抑制 DNA 复制 | 环丙沙星（喹诺酮类） | 细菌 DNA 旋转酶 |
| 抑制叶酸合成 | 磺胺类 | 人从食物获取叶酸，细菌需自合成 |

### 抗生素耐药

WHO 列出的全球十大健康威胁之一。机制：

| 机制 | 说明 |
|------|------|
| 产生灭活酶 | β-内酰胺酶破坏青霉素 |
| 改变靶点 | MRSA 改变了青霉素结合蛋白 |
| 外排泵 | 把药物泵出细胞 |
| 减少摄入 | 改变细胞膜通透性 |

> 你能做的：不要感冒就吃抗生素（感冒是病毒）。遵医嘱完成全程，不要"感觉好了就停药"——这是在筛选耐药菌。

## 药物相互作用

| 机制 | 例子 |
|------|------|
| 竞争 CYP450 酶 | 葡萄柚汁抑制 CYP3A4→多种药物浓度升高 |
| 增加毒性 | 华法林 + 阿司匹林 → 出血风险 |
| 拮抗 | 布洛芬 + 阿司匹林 → 减弱阿司匹林的心血管保护 |

> 告诉医生你吃的**所有**东西——处方药、OTC、中药、保健品。药物相互作用可能是致命的。
