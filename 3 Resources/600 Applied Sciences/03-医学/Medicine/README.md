## 特点

- **系统深度** — 从解剖→病理→诊断→药理→治疗的完整链条
- **临床思维** — 每章关联临床实践，不止于理论
- **数据说话** — 血压分级、R₀ 值、药物半衰期——精确量化
- **患者视角** — "如何成为聪明的患者"——不只是给医生看的知识
- **子知识库** — 位于 Applied Sciences 之下，与上层通识互补
