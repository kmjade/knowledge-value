# 04-诊断学

> 诊断是医学的核心技能——从患者的症状和体征出发，通过问诊、体检和辅助检查，确定疾病的性质和原因。诊断错误是所有医疗错误中最常见也最危险的一类。

## 诊断过程

```
主诉 (患者说) → 病史采集 → 体格检查 → 初步诊断
                                    ↓
                          辅助检查 ← 鉴别诊断
                              ↓
                          确诊 → 治疗方案
```

### 鉴别诊断 (Differential Diagnosis)

列出所有可能的疾病，用证据逐一排除。胸痛可能是：心肌梗死、肺栓塞、气胸、肋骨骨折、胃食管反流、焦虑发作……医生的任务是找出正确的那个。

## 病史采集

| 要素 | 内容 |
|------|------|
| **主诉** | 用患者自己的话——"胸口疼了 3 小时" |
| **现病史** | 何时开始？什么感觉？加重/缓解因素？伴随症状？ |
| **既往史** | 得过什么病？做过什么手术？ |
| **用药史** | 正在吃什么药？（包括中药、保健品！） |
| **家族史** | 直系亲属有什么病？ |
| **社会史** | 吸烟？饮酒？职业暴露？ |

> 有人说 80% 的诊断信息来自病史。一个好医生首先是好听众。

## 体格检查

| 方法 | 检查什么 |
|------|----------|
| **视诊** | 看——面色、体态、皮肤、呼吸模式 |
| **触诊** | 摸——腹部压痛、甲状腺结节、淋巴结 |
| **叩诊** | 敲——肺和腹部的声音（清音/浊音） |
| **听诊** | 听——心音、呼吸音、肠鸣音 |

### 生命体征

| 体征 | 正常范围 | 危急值示例 |
|------|----------|-----------|
| 体温 | 36.1-37.2°C | >41°C 或 <35°C |
| 心率 | 60-100 bpm | <40 或 >150 |
| 呼吸 | 12-20 次/分 | <8 或 >30 |
| 血压 | <120/<80 | >180/120 |
| 血氧 (SpO₂) | 95-100% | <90% |

## 辅助检查

### 实验室检查

| 类别 | 常见项目 | 筛查什么 |
|------|----------|----------|
| **血常规** | WBC, RBC, Hb, PLT | 感染、贫血、出血风险 |
| **生化** | 肝功、肾功、血糖、电解质 | 器官功能 |
| **血脂** | TC, LDL, HDL, TG | 心血管风险 |
| **肿瘤标志物** | AFP, CEA, CA-125, PSA | 癌症筛查（不能单独诊断） |
| **感染** | CRP, PCT, 血培养 | 炎症和感染 |

### 影像学

| 方法 | 原理 | 擅长 | 辐射 |
|------|------|------|------|
| **X 光** | X 射线穿透 | 骨折、肺炎、肺水肿 | 低 |
| **CT** | X 射线旋转扫描+计算机重建 | 脑出血、肿瘤、复杂骨折 | 中-高 |
| **超声** | 声波反射 | 腹部、心脏、血管、产科 | 无 |
| **MRI** | 强磁场+射频 | 脑、脊柱、关节软组织 | 无 |
| **PET-CT** | 放射性示踪剂+CT | 肿瘤代谢、转移灶 | 高 |
| **核医学** | 放射性同位素 | 骨扫描、甲状腺 | 中 |

### 病理检查

**活检 (Biopsy)**：取组织在显微镜下看——诊断的金标准。对于肿瘤，只有病理报告能确定是良性还是恶性。
