# 医学 知识库总览

> 医学是研究人体健康与疾病、诊断、治疗和预防的科学与艺术。从希波克拉底到 CRISPR，从听诊器到 PET-CT——医学是人类对抗疾病最长久的战争。

## 现代医学的原则

| 原则 | 含义 |
|------|------|
| **循证医学 (EBM)** | 临床决策基于最佳可用证据，而非经验或权威 |
| **患者中心** | 治疗的是人，不只是病 |
| **不伤害 (Primum non nocere)** | 希波克拉底誓言的核心——首先，不伤害 |
| **知情同意** | 患者有权了解并选择治疗方案 |
| **多学科协作 (MDT)** | 复杂病例需要多科室联合诊治 |

## 疾病谱的变迁

| 时代 | 主要死因 | 医学应对 |
|------|----------|----------|
| 1900 年前 | 传染病（天花、鼠疫、结核） | 疫苗、抗生素、公共卫生 |
| 1900-1950 | 传染病 + 营养不良 | 抗生素黄金时代 |
| 1950-2000 | 慢性病（心脑血管、癌症） | 降压药、化疗、手术进步 |
| 2000-至今 | 慢性病 + 精神健康 + 老龄化 | 精准医学、免疫治疗、AI |

> 医学最伟大的成就不是某种特效药，而是公共卫生——清洁饮水、疫苗、抗生素让人类平均寿命从 ~30 岁翻倍到 ~70 岁。
