# 医学 常见问题

## 十大常见误区

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| 抗生素治感冒 | 感冒是病毒引起的，抗生素对病毒无效。滥用抗生素导致细菌耐药——这是 WHO 列出的全球十大健康威胁之一 |
| 输液比吃药好 | 能口服不注射，能注射不输液。输液直接入血，风险远高于口服（感染、过敏、静脉炎） |
| 中药没有副作用 | "天然"不等于"无害"。马兜铃酸（存在于某些中药）有明确肾毒性和致癌性。任何有效成分都有副作用 |
| 癌症 = 绝症 | 许多癌症现在可以被治愈或作为慢性病管理（如早期乳腺癌 5 年生存率 >90%）。免疫疗法改变了晚期癌症的治疗格局 |
| MRI 有辐射 | MRI 使用强磁场和射频脉冲，完全没有电离辐射。CT 和 X 光才有辐射 |
| 打了疫苗就不会得病 | 疫苗不提供 100% 保护。它的主要作用是：降低感染风险 + 即使感染，症状也轻得多 |
| 头痛就是脑子有问题 | 脑实质本身没有痛觉感受器。大多数头痛来自颅外的血管、肌肉或神经 |
| 高血压没感觉就不用治 | 高血压是"沉默杀手"——在你没有感觉的时候持续损害心、脑、肾。发现高血压就要管理 |
| 手术后不能吃"发物" | "发物"没有科学定义。术后营养支持应根据具体手术和个体情况，而非笼统忌口 |
| 医学是精确科学 | 医学是概率科学。同一个病因、同一种治疗对不同人的效果不同。医生的决策基于概率和证据，而非确定性 |

## 如何成为聪明的患者？

| 做法 | 原因 |
|------|------|
| 带问题清单去看病 | 平均门诊时间只有几分钟，有准备的问诊效率高得多 |
| 告诉医生你在吃什么药 | 包括中药、保健品——药物相互作用可能致命 |
| 问"证据是什么" | 医生建议的检查/治疗，背后的证据强度如何？ |
| 寻求第二意见 | 重大决策（手术、癌症治疗）至少要两个专家的意见 |
| 不要用搜索引擎替代医生 | Google 不能区分你的症状是感冒还是白血病——因为它们的早期症状可能相似 |
