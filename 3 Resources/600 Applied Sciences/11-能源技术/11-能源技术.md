# 09-能源技术

> 能源是人类文明的血液——从烧木头到可控核聚变，能源技术的每一次革命都改变了社会形态。今天面临的挑战：在满足不断增长的能源需求的同时，实现碳中和。

## 全球能源格局 (2024 数据)

| 能源 | 全球一次能源占比 | 趋势 |
|------|-----------------|------|
| 石油 | ~31% | 平稳 |
| 煤炭 | ~27% | 发达国家下降，发展中国家上升 |
| 天然气 | ~24% | 上升（过渡燃料） |
| 核能 | ~4% | 平稳 |
| 水力 | ~7% | 平稳（增长空间有限） |
| 其他可再生 | ~7% | 快速增长（风、光） |

> 化石燃料仍占 ~82%。能源转型不是"替换"，而是马拉松。

## 能源密度对比

| 能源 | 能量密度 (MJ/kg) | 说明 |
|------|-----------------|------|
| 木材 | 15 | 人类最早的燃料 |
| 煤炭 | 24 | 工业革命的基础 |
| 汽油 | 46 | 交通的主力 |
| 天然气 | 55 | 最清洁的化石燃料 |
| 锂电池 | ~0.9 | 低，但可充电 |
| 铀-235 (核裂变) | 80,000,000 | 比化石燃料高 6 个数量级 |
| 氘-氚 (核聚变) | 340,000,000 | 终极能源（如果实现） |

> 为什么电动车比燃油车重？因为电池的能量密度比汽油低 50 倍。这是物理定律的硬约束。

## 可再生能源

### 太阳能

| 技术 | 效率 | 特点 |
|------|------|------|
| 单晶硅 | 22-26% | 主流技术 |
| 钙钛矿 | 25%+ (实验室) | 快速增长，有望超越硅 |
| 叠层 (硅+钙钛矿) | 33%+ | 下一代方向 |
| 薄膜 (CdTe, CIGS) | 18-22% | 轻、柔性 |

### 风能

| 类型 | 特点 |
|------|------|
| 陆上风电 | 成熟，成本低 |
| 海上风电 | 风更大更稳定，但建设贵 |

> 中国是全球最大的风电和光伏装机国——占全球 1/3 以上。

### 储能

可再生能源的致命弱点——不持续。储能是关键。

| 技术 | 适用 | 局限 |
|------|------|------|
| **锂电池** | 小时级 | 贵、原材料供应链 |
| **抽水蓄能** | 天级 | 需要特定地形 |
| **钠离子电池** | 电网级 | 能量密度低于锂，但便宜 |
| **液流电池** | 电网级 | 能量密度低，但可无限循环 |
| **氢能** | 季节性 | 效率低（电→氢→电 ~30%） |
| **压缩空气** | 电网级 | 需要地下洞穴 |

## 核能

### 裂变（现有技术）

| 代 | 特点 | 例子 |
|----|------|------|
| Gen I | 早期原型 | 已退役 |
| Gen II | 现在运行的大多数 | 压水堆 (PWR)、沸水堆 (BWR) |
| Gen III/III+ | 改进安全性 | AP1000, EPR, 华龙一号 |
| Gen IV | 更安全/更高效/减废 | 研发中（钠冷快堆、熔盐堆） |

### 聚变（终极目标）

ITER（国际热核聚变实验堆）正在法国建造。目标：Q > 10（输出能量是输入的 10 倍）。

- 2022：NIF（美国国家点火装置）实现 Q > 1（净能量增益）——首次在实验室中。
- 乐观估计：商用聚变在 2040-2060 年代。

## 碳中和路径

| 策略 | 说明 |
|------|------|
| 电气化 | 交通、供暖转向电力（前提：电力本身清洁） |
| 能效提升 | 用更少的能源做同样的事 |
| 核能/可再生 | 清洁电力来源 |
| 碳捕获 (CCS) | 从排放源捕获 CO₂ |
| 碳移除 (CDR) | 从大气中移除 CO₂（植树、直接空气捕获） |
| 碳定价 | 碳交易/碳税——让碳排放有经济成本 |
