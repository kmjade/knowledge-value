# 03 — 施工方法

> 建筑物如何实际建造？从传统工艺到机器人预制——将设计转化为现实的方法。

---

## 一、传统施工

### 现浇混凝土

| 步骤 | 说明 |
|------|------|
| 模板 | 搭建 |
| 钢筋 | 绑扎 |
| 浇筑 | 混凝土灌注 |
| 养护 | 28天达设计强度 |
| 拆模 | 模板拆除 |

| 材料 | 用途 |
|------|------|

| 石材 | 饰面·结构(历史建筑) |

## 二、现代工法

### 预制混凝土

| 优势 | 说明 |
|------|------|
| 质量 | 工厂控制条件 |
| 速度 | 现场与工厂并行 |
| 受天气影响小 | 不受现场天气影响 |
| 缺点 | 运输成本·需吊装 |

### 模块化建筑

```
工厂 → 模块(房间/单元) → 运输 → 现场堆叠 → 连接管线
```

- 工厂完成 80-90%
- 比传统快 30-50%
- 案例：CitizenM酒店·住宅塔楼

### 钢结构

```
钢柱+钢梁 → 螺栓/焊接连接 → 组合楼板
```

- 快速安装
- 大跨度
- 商业/高层常用

## 四、施工阶段

```
1. 场地准备  清表·开挖·支护
2. 基础工程  桩·基础·防水
3. 上部结构  柱·梁·板·墙
4. 外围护    幕墙·屋面·门窗
5. 机电管线  暖通·给排水·电气
6. 室内装修  隔墙·地面·饰面
7. 调试验收  系统测试·整改
8. 移交      业主入驻
```

> 最佳施工方法取决于项目条件——场地·预算·工期·人工·当地规范都需要综合考虑。
