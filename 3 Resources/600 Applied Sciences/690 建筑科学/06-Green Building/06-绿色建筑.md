# 06 — 绿色建筑

> 绿色建筑在最大化使用者的健康、舒适和生产力的同时，最小化环境影响。

---

## 一、评估体系

| 体系 | 来源 | 特点 |
|------|------|------|
| LEED | 美国 (USGBC) | 全球应用最广 |
| BREEAM | 英国 | 最早体系 (1990) |
| Green Mark | 新加坡 | 热带气候导向 |
| WELL | 国际 | 健康与福祉 |
| 被动房 | 德国 | 超低能耗 |
| 绿色建筑评价标准 (GB/T 50378) | 中国 | 三星认证 |
| 綠建築標章 (EEWH) | 台湾 | 生态·节能·减废·健康 |

---

## 二、设计策略

| 策略 | 说明 |
|------|------|
| 被动式设计 | 朝向·遮阳·自然通风 |
| 高性能围护 | 保温·气密·高性能玻璃 |
| 高效暖通 | 热泵·能量回收新风 |
| 可再生能源 | 光伏·地热·风能 |
| 节水 | 节水器具·雨水收集·中水回用 |
| 可持续材料 | 再生材料·本地采购·低VOC |
| 自然采光 | 最大化自然光·减少人工照明 |

---

## 三、净零能耗

```
建筑能耗 − 现场可再生能源 = 0 (或正收益)
```

三步实现：
1. 降低需求（被动设计）
2. 提升效率（设备·照明）
3. 现场发电（光伏·地热）

---

## 四、隐含碳 vs 运营碳

| 类型 | 定义 | 生命周期占比 |
|------|------|:---:|
| 隐含碳 | 材料+施工产生的碳排放 | ~30-50% |
| 运营碳 | 建筑使用产生的碳排放 | ~50-70% |

> 最绿色的建筑是已经建成的建筑——改造优于重建。运营效率最重要。
