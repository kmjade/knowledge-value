# 07 — Building Codes & Safety 建築法規與安全

> Building codes ensure minimum standards for safety, health, and welfare in the built environment.
> 建築法規確保建成環境中的安全、健康和福祉最低標準。

## Major Code Systems

| Code | Region | Scope |
|------|--------|-------|
| **IBC** (International Building Code) | US/International | Comprehensive building code |
| **Eurocodes** | EU | Structural design standards |
| **GB 50000 series** | China | Chinese building standards |
| **Building Act + Regulations** | Taiwan (建築法/技術規則) | Local code framework |

## Key Safety Areas

| Area | Requirements |
|------|-------------|
| **Structural** | Design loads, material strength, seismic |
| **Fire** | Sprinklers, alarms, egress, compartmentation |
| **Accessibility** | Ramps, elevators, accessible restrooms |
| **Energy** | Insulation, HVAC efficiency, lighting |
| **Ventilation** | Fresh air rates, exhaust |
| **Plumbing** | Water supply, drainage, fixture count |

## Fire Safety 消防安全

| Element | Purpose |
|---------|---------|
| Fire Resistance Rating | How long a component resists fire (1hr, 2hr, 3hr) |
| Means of Egress | Exit path: exit access → exit → exit discharge |
| Compartmentation | Fire walls/floors to contain fire spread |
| Smoke Control | Pressurized stairwells, atrium exhaust |
| Active Systems | Sprinklers, alarms, standpipes |

## Accessibility 無障礙

| Requirement | Standard |
|-------------|----------|
| Door width | ≥ 810mm clear |
| Ramp slope | ≤ 1:12 |
| Elevator | Required for buildings > 3 stories |
| Accessible toilet | Grab bars, turning space |

## Permitting Process

```
Design → Submit to authority → Plan review → Corrections → Permit issued → Inspections → Certificate of Occupancy
```

> 💡 Codes are minimums, not targets. Good design exceeds code requirements — especially for safety and accessibility.
