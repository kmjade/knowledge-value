# 07 — 建筑规范与安全

> 建筑规范确保建成环境中的安全、健康和福祉最低标准。

---

## 一、主要规范体系

| 规范 | 地区 | 范围 |
|------|------|------|
| IBC 国际建筑规范 | 美国/国际 | 综合建筑规范 |
| Eurocodes 欧洲规范 | 欧盟 | 结构设计标准 |
| GB 50000 系列 | 中国 | 中国建筑标准 |
| 建築法/技術規則 | 台湾 | 地方规范框架 |

---

## 二、安全要点

| 领域 | 要求 |
|------|------|
| 结构 | 设计荷载·材料强度·抗震 |
| 消防 | 喷淋·报警·疏散·防火分区 |
| 无障碍 | 坡道·电梯·无障碍卫生间 |
| 节能 | 保温·暖通效率·照明 |
| 通风 | 新风量·排风 |
| 给排水 | 供水·排水·卫生器具 |

---

## 三、消防安全

| 要素 | 作用 |
|------|------|
| 耐火等级 | 构件耐火时限 (1h/2h/3h) |
| 疏散通道 | 疏散门→走道→楼梯→出口 |
| 防火分区 | 防火墙/楼板阻止火势蔓延 |
| 防烟控制 | 加压楼梯间·中庭排烟 |
| 主动系统 | 喷淋·报警·消火栓 |

## 四、无障碍设计

| 要求 | 标准 |
|------|:---:|
| 门净宽 | ≥ 810mm |
| 坡道坡度 | ≤ 1:12 |
| 电梯 | 3层以上须设 |
| 无障碍卫生间 | 扶手·回转空间 |

## 五、审批流程

```
设计 → 报建 → 图纸审查 → 修改 → 施工许可 → 过程检查 → 竣工验收
```

> 规范是最低标准，不是设计目标。好设计超越规范要求——尤其在安全和无障碍方面。
