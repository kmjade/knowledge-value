# 01 — 建筑材料

> 材料决定强度、耐久性、成本、美学与环境影响。

---

## 一、混凝土

| 特性 | 说明 |
|------|------|
| 组成 | 水泥 + 骨料 + 水 + 外加剂 |
| 抗压强度 | 20–100+ MPa |
| 抗拉强度 | 低（需钢筋补强） |
| 优点 | 耐火 · 可塑 · 耐久 · 经济 |
| 缺点 | 高碳排放 · 脆性 · 自重大 |

| 类型 | 用途 |
|------|------|
| 钢筋混凝土 (RC) | 梁·柱·板 |
| 预应力混凝土 | 大跨桥梁·停车场 |
| 高性能混凝土 (HPC) | 超高层·海洋结构 |
| 自密实混凝土 (SCC) | 复杂模板 |

## 二、钢材

| 特性 | 说明 |
|------|------|
| 强度 | 抗拉+抗压均高 |
| 延性 | 优异——弯而不断 |
| 可回收 | 100% 可回收 |
| 缺点 | 锈蚀·需防火保护 |

| 类型 | 用途 |
|------|------|
| 型钢 (工字钢·H型钢) | 框架结构·超高层 |
| 钢筋 | 混凝土配筋 |
| 冷弯薄壁型钢 | 轻钢龙骨·住宅 |

## 三、木材

| 特性 | 说明 |
|------|------|
| 可再生 | 固碳材料 |
| 强重比 | 优异 |
| 缺点 | 防火·防潮·防虫 |

| 工程木 | 说明 |
|------|------|
| 胶合木 (Glulam) | 梁·拱 |
| 正交胶合木 (CLT) | 墙体·楼板 |
| 单板层积材 (LVL) | 结构构件 |
| 胶合板 | 板材 |

## 四、现代材料

| 材料 | 应用 |
|------|------|
| 玻璃 | 幕墙·结构玻璃 |
| FRP 纤维增强聚合物 | 加固·防腐结构 |
| UHPC 超高性能混凝土 | 轻薄·高强·耐久 |
| 气凝胶 | 超级保温 |
| 自修复混凝土 | 细菌基裂缝修复 |
| 石墨烯增强材料 | 更强·更轻的复合材料 |
