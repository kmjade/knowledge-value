# 01 — Building Materials 建築材料

> The substances from which buildings are made. Materials determine strength, durability, cost, aesthetics, and environmental impact.
> 建築物的構成物質——材料決定了強度、耐久性、成本、美學與環境影響。

## 主要建築材料 Major Materials

### Concrete 混凝土

| 特性 | 說明 |
|------|------|
| **Composition** | Cement + aggregates + water + admixtures |
| **Compressive Strength** | 20–100+ MPa |
| **Tensile Strength** | Low (reinforced with steel rebar) |
| **Advantages** | Fire-resistant, moldable, durable, cost-effective |
| **Disadvantages** | High CO₂ footprint, brittle, heavy |

| Type | Use |
|------|-----|
| Reinforced Concrete (RC) | Beams, columns, slabs |
| Prestressed Concrete | Long-span bridges, parking structures |
| High-Performance Concrete (HPC) | Skyscrapers, marine structures |
| Self-Compacting Concrete (SCC) | Complex formwork |

### Steel 鋼材

| 特性 | 說明 |
|------|------|
| **Strength** | High tensile + compressive |
| **Ductility** | Excellent — bends before breaking |
| **Recyclability** | 100% recyclable |
| **Disadvantages** | Corrosion, fire protection needed |

| Type | Use |
|------|-----|
| Structural Steel (I-beam, H-beam) | Frame structures, skyscrapers |
| Rebar | Reinforcing concrete |
| Cold-Formed Steel | Light-gauge framing, residential |

### Timber / Wood 木材

| 特性 | 說明 |
|------|------|
| **Renewable** | Carbon-sequestering |
| **Strength-to-weight** | Excellent |
| **Disadvantages** | Fire, moisture, pests |

| Engineered Wood | Description |
|-----------------|-------------|
| Glulam | Glued laminated timber — beams, arches |
| CLT (Cross-Laminated Timber) | Mass timber panels — walls, floors |
| LVL | Laminated veneer lumber |
| Plywood | Sheet material |

## 現代材料 Modern Materials

| Material | Application |
|----------|-------------|
| **Glass** | Curtain walls, structural glazing |
| **FRP** (Fiber-Reinforced Polymer) | Strengthening, corrosion-resistant structures |
| **UHPC** (Ultra-High Performance Concrete) | Thin, strong, durable |
| **Aerogel** | Super-insulation |
| **Self-Healing Concrete** | Bacteria-based crack repair |
| **Graphene-Enhanced Materials** | Stronger, lighter composites |

> 💡 The three most used materials on Earth: concrete (by mass), steel (by value), and timber (historically). Each has its place in the builder's palette.
