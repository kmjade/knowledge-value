# 05 — 建筑设备 (MEP)

> MEP = 机械(Mechanical) + 电气(Electrical) + 给排水(Plumbing)。让建筑运作和舒适的系统。

---

## 一、暖通空调 (HVAC)

| 系统 | 功能 |
|------|------|
| 供暖 | 锅炉·热泵·地暖 |
| 通风 | 新风·CO₂控制·排风 |
| 空调 | 冷水机组·冷却塔·VRF·分体 |

| 类型 | 说明 |
|------|------|
| 集中式 AHU | 大型建筑·风管系统 |
| VRF/VRV | 变制冷剂流量——灵活分区 |
| 分体系统 | 住宅·小型商业 |
| 辐射式 | 地板/天花板辐射——舒适 |

---

## 二、给排水

| 系统 | 组成 |
|------|------|
| 供水 | 市政→水箱→增压泵→用水点 |
| 排水 | 重力流→立管→市政管网 |
| 热水 | 锅炉/热泵/太阳能 |
| 消防喷淋 | 湿式/干式/预作用 |

---

## 三、电气

| 系统 | 说明 |
|------|------|
| 变配电 | 变压器→开关柜→配电箱 |
| 照明 | 环境光·任务光·重点照明；LED 为标准 |
| 应急电源 | 发电机·UPS |
| 弱电 | 数据·安防·楼控·消防报警 |

---

## 四、消防

| 系统 | 说明 |
|------|------|
| 喷淋 | 自动灭火 |
| 消火栓 | 消防员供水接口 |
| 火灾报警 | 探测+通知 |
| 防排烟 | 加压送风·排烟 |

---

## 五、楼宇自控 (BMS)

```
传感器 → 控制器 → BMS 服务器 → 监控大屏
   ↓
暖通·照明·安防·能源
```

> MEP 占建造成本约 30-40%——做好机电是建筑性能和舒适度的关键。
