# 09 — 施工管理

> 按时、按预算、按品质交付项目——施工管理的核心学科。

---

## 一、项目交付模式

| 模式 | 说明 | 风险承担 |
|------|------|:---:|
| **设计-招标-施工** (DBB) | 传统：设计→招标→施工 | 业主 |
| **设计-施工** (DB) | 单一主体设计+施工 | 承包商 |
| **CM at Risk** | 施工经理担保最高价 | 共担 |
| **一体化交付** (IPD) | 所有方共担风险/共享收益 | 协作 |

---

## 二、项目阶段

```
规划采购 → 发包 → 施工 → 竣工
Pre-construction → Procurement → Construction → Closeout
```

| 阶段 | 核心工作 |
|------|------|
| 规划采购 | 可行性研究 · 预算编制 · 招标文件 |
| 发包 | 评标 · 合同签订 |
| 施工 | 进度控制 · 质量管理 · 安全管理 |
| 竣工 | 验收 · 移交 · 结算 |

---

## 三、BIM 建筑信息模型

BIM 是建筑物物理和功能特性的数字化表达。

| 维度 | 增加的内容 | 应用 |
|:---:|------|------|
| 3D | 几何模型 | 碰撞检查 |
| 4D | 进度（时间） | 施工模拟 |
| 5D | 成本 | 工程量统计 |
| 6D | 可持续性 | 能耗分析 |
| 7D | 设施管理 | 运维管理 |

### 常用软件

| 软件 | 用途 |
|------|------|
| Revit | BIM 建模 |
| Navisworks | 碰撞检测 · 4D/5D |
| BIM 360 | 云端协作 |
| Tekla | 钢结构深化 |

---

## 四、成本管理

| 阶段 | 工作 | 精度 |
|------|------|:---:|
| 概念阶段 | 量级估算 | ±30% |
| 设计深化 | 预算估算 | ±15% |
| 施工图 | 确定性估算 | ±5% |
| 招标 | 承包商报价 | — |
| 施工 | 变更管理 · 进度付款 | — |

---

## 五、进度管理

| 工具 | 说明 |
|------|------|
| **甘特图** | 横道图——最常用 |
| **关键路径法 (CPM)** | 最长路径 = 项目总工期 |
| **末位计划系统** | 精益施工 · 协作计划 |
| **拉动式计划** | 从里程碑反向推导 |

---

## 六、质量管理

| 环节 | 内容 |
|------|------|
| 材料检验 | 进场验收 · 见证取样 |
| 过程控制 | 隐蔽工程验收 · 旁站 |
| 竣工验收 | 分部分项验收 · 资料归档 |

---

## 七、安全管理

| 要素 | 内容 |
|------|------|
| 三级教育 | 公司·项目·班组 |
| 危险源识别 | 高空·临电·机械 |
| 应急预案 | 演练·物资·响应 |

> 施工管理的本质是管理不确定性——天气、人工、材料、变更。最好的管理者在问题变成危机之前就已经预见。
