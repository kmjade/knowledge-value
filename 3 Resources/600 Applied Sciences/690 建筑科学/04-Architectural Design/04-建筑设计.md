# 04 — 建筑设计

> 建筑是设计建筑物的艺术与科学——在功能、形式、环境与人类体验之间取得平衡。

---

## 一、设计原则

| 原则 | 核心问题 |
|------|------|
| 功能 | 是否满足使用需求？ |
| 形式 | 外观与感受如何？ |
| 空间 | 体量如何组织？ |
| 光线 | 自然+人工照明 |
| 环境 | 与周边关系如何？ |
| 材料 | 用什么材料表达意图？ |
| 可持续 | 环境性能 |

---

## 二、建筑类型

| 类型 | 关键考量 |
|------|------|
| 住宅 | 隐私·舒适·采光 |
| 商业/办公 | 灵活性·效率·品牌 |
| 公共建筑 | 耐久·安全·城市形象 |
| 工业 | 大跨度·物流·设备 |
| 文化 | 标志性造型·参观体验 |
| 医疗 | 感染控制·患者流线 |
| 教育 | 声学·自然光·安全 |

---

## 三、建筑史简览

| 时期 | 特点 | 代表作 |
|------|------|------|
| 古典 | 柱式·对称·比例 | 帕特农·万神殿 |
| 哥特 | 尖拱·飞扶壁 | 巴黎圣母院 |
| 文艺复兴 | 几何·人文主义 | 圣彼得大教堂 |
| 现代主义 | 形式服从功能·极简 | 萨伏伊别墅·西格拉姆大厦 |
| 后现代 | 趣味·历史引用 | AT&T大楼 |
| 当代 | 参数化·可持续·数字 | 哈利法塔·央视大楼 |

---

## 四、设计流程

```
规划 → 概念设计 → 深化设计 → 施工图 → 招标 → 施工管理
```

| 阶段 | 产出 |
|------|------|
| 规划 | 设计任务书 |
| 概念设计 | 方案草图·体量模型 |
| 深化设计 | 平立剖·材料选择 |
| 施工图 | 施工图·节点详图 |
| 招标 | 招标文件 |
| 施工管理 | 现场配合·设计变更 |

> "建筑是对空间的深思熟虑。"——路易斯·康。好的设计解决问题，伟大的设计提升人类精神。
