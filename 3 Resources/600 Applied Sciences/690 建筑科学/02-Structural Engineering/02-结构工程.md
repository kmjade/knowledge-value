# 02 — 结构工程

> 建筑物如何站立？结构工程确保建筑安全经济地抵抗重力、风力、地震等荷载。

---

## 一、荷载类型

| 荷载 | 说明 | 示例 |
|------|------|------|
| 恒荷载 (DL) | 永久·自重 | 结构·装修·固定设备 |
| 活荷载 (LL) | 临时·可变 | 人员·家具·车辆 |
| 风荷载 (WL) | 风压/风吸 | 高层建筑尤为关键 |
| 地震荷载 (EQ) | 地面运动 | 水平力·基底剪力 |
| 雪荷载 | 积雪重量 | 寒冷地区屋顶设计 |
| 温度荷载 | 热胀冷缩 | 长结构·桥梁 |

---

## 二、结构体系

| 体系 | 说明 | 典型高度 |
|------|------|:---:|
| 承重墙 | 墙体承重 | 1-5层 |
| 框架 | 梁柱骨架 | 1-30层 |
| 剪力墙 | RC墙抵抗水平力 | 10-40层 |
| 支撑框架 | 斜撑增加刚度 | 10-60层 |
| 筒体 | 外围柱形成空心筒 | 40-100+层 |
| 伸臂桁架 | 核心筒+外围桁架连接 | 50-100+层 |
| 斜交网格 | 外立面斜交网格 | 30-80+层 |

---

## 三、基础工程

| 类型 | 深度 | 适用 |
|------|:---:|------|
| **浅基础** | < 3m | 良好地基·轻荷载 |
| 独立基础 | | 单柱 |
| 条形基础 | | 承重墙 |
| 筏板基础 | | 软弱土·重荷载 |
| **深基础** | > 3m | 表层土软弱 |
| 桩基础 | | 端承或摩擦 |
| 沉井 | | 大直径·钻孔 |
| 地下连续墙 | | 地下工程 |

---

## 四、抗震设计

| 原则 | 说明 |
|------|------|
| 延性 | 变形而不突然破坏 |
| 强柱弱梁 | 梁先于柱屈服 |
| 基础隔震 | 建筑与地面运动解耦 |
| 调谐质量阻尼器 | 配重减少摇摆（台北101） |
| 耗能装置 | 阻尼器吸收地震能量 |

> 结构工程的艺术是找到最高效的荷载传递路径——从屋顶到基础——用最少的材料实现最大的安全。
