# 08 — 城市规划

> 城市规划塑造城市的成长、功能与服务方式——从分区到交通，从密度到公共空间。

---

## 一、核心概念

| 概念 | 说明 |
|------|------|
| **分区 (Zoning)** | 土地使用管制（居住·商业·工业） |
| **容积率 (FAR)** | 建筑面积与用地面积之比 |
| **混合用途** | 居住·商业·办公一体化开发 |
| **TOD 公交导向开发** | 围绕公交站点高密度开发 |
| **公共领域** | 街道·公园·广场——建筑之间的空间 |
| **城市热岛效应** | 城市温度高于周边——绿色屋顶·行道树缓解 |

---

## 二、规划尺度

| 尺度 | 关注点 |
|------|------|
| **区域级** | 交通网络 · 增长边界 |
| **城市级** | 总体规划 · 分区地图 |
| **片区/社区** | 风貌 · 密度 · 配套设施 |
| **地块级** | 建筑布局 · 景观 · 出入口 |

---

## 三、15 分钟生活圈

```
居住 → 工作 → 购物 → 学习 → 休闲
  └──── 全部在 15 分钟内可达 ────┘
```

核心要素：步行友好 · 自行车基础设施 · 混合用途 · 本地服务

---

## 四、智慧城市

| 技术 | 应用 |
|------|------|
| IoT 传感器 | 交通 · 空气质量 · 垃圾管理 |
| 数字孪生 | 城市模拟与规划 |
| 智能电网 | 能源优化 |
| 自动驾驶公交 | 无人巴士 · 配送机器人 |

## 五、中国规划体系

| 层级 | 规划类型 |
|------|------|
| 全国 | 全国国土空间规划纲要 |
| 省级 | 省级国土空间规划 |
| 市级 | 城市总体规划 · 详细规划 |
| 乡镇 | 乡镇国土空间规划 |

### 控制性指标

| 指标 | 控制方式 |
|------|:---:|
| 容积率 | 上限 |
| 建筑密度 | 上限 |
| 建筑高度 | 上限 |
| 绿地率 | 下限 |
| 停车位 | 下限 |

> 伟大的城市不是从零规划的——它们在精心引导下有机演化。最好的城市主义把人放在车之前。
