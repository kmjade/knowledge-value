# 02 — 氣血津液

> 氣、血、津液是構成人體和維持生命活動的基本物質。

## 氣

| 功能 | 說明 |
|------|------|
| **推動** | 推動血液循環、津液輸布、臟腑活動 |
| **溫煦** | 維持體溫、促進代謝 |
| **防禦** | 抵抗外邪（免疫功能） |
| **固攝** | 統攝血液、控制津液不妄洩 |
| **氣化** | 物質與能量的轉化 |

### 氣的分類

| 類型 | 來源 | 功能 |
|------|------|------|
| **元氣** | 先天（腎精） | 生命活動的根本動力 |
| **宗氣** | 肺吸入清氣 + 脾胃水穀精微 | 推動呼吸和血液循環 |
| **營氣** | 水穀精微 | 營養全身、化生血液 |
| **衛氣** | 水穀精微 | 防禦外邪、調節體溫 |

---

## 血

| 功能 | 說明 |
|------|------|
| 滋養全身 | 為臟腑組織提供營養 |
| 載氣 | 氣依附於血運行 |
| 神志基礎 | 血足則神清 |

---

## 津液

| 類型 | 特性 | 分布 |
|------|------|------|
| **津** | 清稀、流動性大 | 體表、肌肉、孔竅 |
| **液** | 稠厚、流動性小 | 關節、臟腑、腦髓 |

---

## 氣血關係

| 關係 | 說明 |
|------|------|
| **氣為血之帥** | 氣能生血、行血、攝血 |
| **血為氣之母** | 血能載氣、養氣 |

> 💡 氣血是生命的根本——氣血充盈則健康，氣血失調則百病生。
