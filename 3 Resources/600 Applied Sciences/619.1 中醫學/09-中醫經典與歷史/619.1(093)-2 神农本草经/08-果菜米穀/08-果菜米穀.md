# 08 — 果菜米穀

> 果菜米穀是本草經中的「藥食同源」藥物——既是日常食物，又有治療功效。

## 代表藥物

| 藥物 | 性味 | 功效 | 日常用途 |
|------|------|------|---------|
| **大棗** | 溫、甘 | 補中益氣、養血安神 | 食材、藥膳 |
| **生薑** | 溫、辛 | 發散風寒、溫中止嘔 | 調料 |
| **薏苡仁** | 微寒、甘淡 | 利水滲濕、健脾除痺 | 雜糧 |
| **黑芝麻** | 平、甘 | 補肝腎、益精血、潤腸 | 食材 |
| **山藥** | 平、甘 | 補脾養胃、生津益肺 | 蔬菜 |
| **蓮子** | 平、甘澀 | 補脾止瀉、益腎固精 | 食材 |
| **赤小豆** | 平、甘酸 | 利水消腫、解毒排膿 | 雜糧 |
| **杏仁** | 微溫、苦 | 止咳平喘、潤腸通便 | 堅果 |

## 藥食同源理念

| 層次 | 說明 |
|------|------|
| **食療** | 以日常食物調理身體 |
| **藥膳** | 藥物與食物結合 |
| **治療** | 食物作為藥物使用 |

## 食療優勢

| 優勢 | 說明 |
|------|------|
| 安全性高 | 多為上品，無毒副作用 |
| 長期可用 | 適合慢性調理 |
| 易於接受 | 口感好，依從性高 |

> 💡 神農本草經的果菜米穀類是中醫「藥食同源」思想的源頭——最好的藥物就在日常飲食中。Food is medicine.
