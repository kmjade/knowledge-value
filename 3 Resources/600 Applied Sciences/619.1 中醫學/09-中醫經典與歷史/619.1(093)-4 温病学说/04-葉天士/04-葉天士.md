# 04 — 葉天士（1667–1746）

> 葉桂，字天士，清代著名醫家，溫病學派的奠基人，創立衛氣營血辨證。

## 生平

| 事項 | 內容 |
|------|------|
| 籍貫 | 江蘇吳縣（今蘇州） |
| 學醫 | 先後拜十七位老師，博採眾長 |
| 專長 | 溫熱病、雜病、兒科 |
| 著作 | 《溫熱論》《臨證指南醫案》 |

## 主要貢獻

| 貢獻 | 說明 |
|------|------|
| **衛氣營血辨證** | 溫病辨證的核心理論 |
| **辨舌驗齒** | 創立舌診在溫病中的系統應用 |
| **絡病學說** | 「久病入絡」理論 |
| **胃陰學說** | 重視養胃陰 |

## 溫熱論名句

- 「溫邪上受，首先犯肺，逆傳心包」
- 「在衛汗之，到氣清氣，入營透熱轉氣，入血涼血散血」
- 「救陰不在血，而在津與汗」

> 💡 葉天士一生忙於診務，著作多為弟子整理——《溫熱論》雖篇幅短小，但字字珠璣，奠定了溫病學的理論基石。
