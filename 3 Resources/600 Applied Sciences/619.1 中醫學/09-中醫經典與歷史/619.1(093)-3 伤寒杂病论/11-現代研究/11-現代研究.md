# 11 — 現代研究與應用

> 經方在現代臨床和科研中持續發光——從實驗室到臨床，兩千年前的智慧正被一一驗證。

## 經方現代應用

| 經方 | 現代應用 |
|------|---------|
| 桂枝湯 | 感冒、過敏性鼻炎、蕁麻疹 |
| 小柴胡湯 | 肝炎、膽囊炎、抑鬱症、月經不調 |
| 白虎湯 | 高熱、中暑、糖尿病初起 |
| 四逆湯 | 休克、心力衰竭、低血壓 |
| 腎氣丸 | 糖尿病腎病、前列腺增生、衰老相關疾病 |
| 真武湯 | 慢性心力衰竭、腎病症候群 |
| 半夏瀉心湯 | 慢性胃炎、消化性潰瘍 |

## 藥理研究

| 方劑 | 藥理作用 |
|------|---------|
| 桂枝湯 | 調節體溫、發汗、抗炎、免疫調節 |
| 小柴胡湯 | 保肝、抗炎、免疫調節、抗抑鬱 |
| 白虎湯 | 解熱、抗炎、降血糖 |
| 大承氣湯 | 促進腸蠕動、抗感染 |
| 四逆湯 | 強心、升壓、改善微循環 |

## 循證醫學證據

| 研究類型 | 發現 |
|---------|------|
| RCT | 小柴胡湯治療慢性肝炎有效 |
| Meta分析 | 半夏瀉心湯治療功能性消化不良 |
| 系統回顧 | 真武湯治療慢性心衰竭 |

> 💡 經方的現代研究證明：中醫不是「經驗醫學」——它的療效可以被科學方法驗證。經方的價值在於它提供了經過兩千年篩選的有效治療方案。
