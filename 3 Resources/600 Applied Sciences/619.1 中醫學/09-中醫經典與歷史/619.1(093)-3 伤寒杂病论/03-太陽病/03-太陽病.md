# 03 — 太陽病

> 太陽病是外感病的初始階段——病邪在表，正氣抗邪於體表。

## 提綱

> 「太陽之為病，脈浮，頭項強痛而惡寒。」（第 1 條）

## 分類

### 太陽中風（表虛證）

| 特徵 | 說明 |
|------|------|
| 症狀 | 發熱、惡風、汗出、脈浮緩 |
| 病機 | 衛強營弱，營衛不和 |
| 主方 | **桂枝湯** |
| 組成 | 桂枝、芍藥、生薑、大棗、甘草 |

### 太陽傷寒（表實證）

| 特徵 | 說明 |
|------|------|
| 症狀 | 惡寒、發熱、無汗、脈浮緊 |
| 病機 | 風寒束表，衛陽被遏 |
| 主方 | **麻黃湯** |
| 組成 | 麻黃、桂枝、杏仁、甘草 |

## 太陽腑證

| 證型 | 病機 | 主方 |
|------|------|------|
| 蓄水證 | 膀胱氣化不利 | 五苓散 |
| 蓄血證 | 瘀熱結於下焦 | 桃核承氣湯 |

## 變證

太陽病失治或誤治後出現的變證：
- 協熱下利：葛根芩連湯
- 奔豚：桂枝加桂湯
- 結胸：大陷胸湯

> 💡 太陽病是傷寒論篇幅最大的篇章——178 條。「善治太陽者，治傷寒之半矣。」
