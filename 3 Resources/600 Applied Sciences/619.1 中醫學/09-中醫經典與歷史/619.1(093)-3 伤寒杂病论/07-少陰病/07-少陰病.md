# 07 — 少陰病

> 少陰病是三陰病中最危重的階段——心腎陽虛，陰寒內盛，生死關頭。

## 提綱

> 「少陰之為病，脈微細，但欲寐也。」（第 281 條）

## 分類

### 少陰寒化證

| 特徵 | 說明 |
|------|------|
| 病機 | 心腎陽虛，陰寒內盛 |
| 症狀 | 惡寒蜷臥、四肢厥冷、下利清穀、脈微欲絕 |
| 主方 | **四逆湯** |

### 四逆湯

| 組成 | 功效 |
|------|------|
| 附子（生用） | 回陽救逆（君） |
| 乾薑 | 溫中散寒（臣） |
| 甘草 | 調和緩毒 |

### 少陰熱化證

| 病機 | 主方 |
|------|------|
| 陰虛火旺，心腎不交 | 黃連阿膠湯 |
| 陰虛水熱互結 | 豬苓湯 |

## 四逆湯類方

| 方 | 組成 | 主治 |
|----|------|------|
| 四逆湯 | 附子、乾薑、甘草 | 少陰寒化 |
| 通脈四逆湯 | 四逆湯加大劑量 | 陰盛格陽 |
| 白通湯 | 附子、乾薑、蔥白 | 陰盛戴陽 |
| 四逆加人參湯 | 四逆湯加人參 | 亡血 |

> 💡 少陰病是傷寒論的危重症篇章——四逆湯回陽救逆，至今仍是急救要方。少陰病多死證，治當急溫。
