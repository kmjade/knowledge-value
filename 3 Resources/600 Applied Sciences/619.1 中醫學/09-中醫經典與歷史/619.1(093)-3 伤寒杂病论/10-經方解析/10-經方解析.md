# 10 — 經方解析

> 經方是傷寒雜病論所載方劑的統稱——組方嚴謹、藥簡效宏，被譽為「眾方之宗」。

## 經方特點

| 特點 | 說明 |
|------|------|
| 藥味少 | 平均 4–6 味，少則 1 味（甘草湯） |
| 劑量重 | 單味藥劑量大，力專效宏 |
| 配伍精 | 一味之差，主治迥異 |
| 療效確 | 歷經兩千年臨床驗證 |

## 十大經典經方

| 方名 | 組成 | 功效 | 主治 |
|------|------|------|------|
| 桂枝湯 | 桂枝、芍藥、薑、棗、草 | 解肌發表、調和營衛 | 太陽中風 |
| 麻黃湯 | 麻黃、桂枝、杏仁、草 | 發汗解表、宣肺平喘 | 太陽傷寒 |
| 小柴胡湯 | 柴胡、黃芩、參、夏、薑、棗、草 | 和解少陽 | 少陽病 |
| 白虎湯 | 石膏、知母、粳米、草 | 清熱生津 | 陽明經證 |
| 大承氣湯 | 大黃、芒硝、枳實、厚朴 | 峻下熱結 | 陽明腑實 |
| 四逆湯 | 附子、乾薑、草 | 回陽救逆 | 少陰寒化 |
| 理中湯 | 參、朮、薑、草 | 溫中健脾 | 太陰病 |
| 烏梅丸 | 烏梅、連、柏、附、桂、薑等 | 寒熱並調 | 厥陰病 |
| 腎氣丸 | 地黃、山藥、茱、苓、澤、丹、桂、附 | 補腎助陽 | 虛勞 |
| 小建中湯 | 桂枝湯倍芍藥加飴糖 | 溫中補虛 | 虛勞腹痛 |

## 經方配伍規律

| 藥對 | 功效 | 代表方 |
|------|------|--------|
| 桂枝+芍藥 | 調和營衛 | 桂枝湯 |
| 麻黃+桂枝 | 發汗解表 | 麻黃湯 |
| 石膏+知母 | 清熱生津 | 白虎湯 |
| 大黃+芒硝 | 瀉下攻積 | 大承氣湯 |
| 附子+乾薑 | 回陽救逆 | 四逆湯 |
| 柴胡+黃芩 | 和解少陽 | 小柴胡湯 |

> 💡 經方的核心是「方證對應」——有是證，用是方。藥味少而精，充分體現了中醫「簡便驗廉」的優勢。
