# 05 — 少陽病

> 少陽病介於太陽之表和陽明之裡之間——「半表半裡」，以樞機不利為核心病機。

## 提綱

> 「少陽之為病，口苦，咽乾，目眩也。」（第 263 條）

## 主證

| 特徵 | 說明 |
|------|------|
| 症狀 | 往來寒熱、胸脅苦滿、嘿嘿不欲飲食、心煩喜嘔 |
| 脈象 | 弦 |
| 病機 | 邪犯少陽，膽火上炎，樞機不利 |
| 主方 | **小柴胡湯** |

## 小柴胡湯

| 組成 | 功效 |
|------|------|
| 柴胡 | 疏散少陽之邪（君） |
| 黃芩 | 清少陽膽熱（臣） |
| 人參、甘草 | 益氣和中 |
| 半夏、生薑 | 和胃降逆 |
| 大棗 | 調和營衛 |

## 小柴胡湯四大主證

> 「傷寒中風，有柴胡證，但見一證便是，不必悉具。」（第 101 條）

1. 往來寒熱
2. 胸脅苦滿
3. 嘿嘿不欲飲食
4. 心煩喜嘔

## 少陽兼證

| 兼證 | 主方 |
|------|------|
| 兼太陽 | 柴胡桂枝湯 |
| 兼陽明 | 大柴胡湯 |
| 兼太陰 | 柴胡桂枝乾薑湯 |

> 💡 小柴胡湯是傷寒論中應用最廣的方劑之一——現代臨床用於感冒、肝炎、胃炎、抑鬱症等多種疾病。
