# 06 — 太陰病

> 太陰病是三陰病的初始階段——脾胃虛寒，運化失職。

## 提綱

> 「太陰之為病，腹滿而吐，食不下，自利益甚，時腹自痛。」（第 273 條）

## 主證

| 特徵 | 說明 |
|------|------|
| 病性 | 裡虛寒證 |
| 病位 | 脾胃（足太陰脾經） |
| 病機 | 脾陽虛弱，寒濕內盛 |
| 主方 | **理中湯（丸）** |

## 理中湯

| 組成 | 功效 |
|------|------|
| 人參 | 補氣健脾 |
| 乾薑 | 溫中散寒 |
| 白朮 | 健脾燥濕 |
| 甘草 | 補中調和 |

## 太陰病治療原則

| 原則 | 說明 |
|------|------|
| 溫中散寒 | 恢復脾陽 |
| 健脾燥濕 | 運化水濕 |
| 忌攻下 | 太陰病本虛，下之更傷陽氣 |

## 太陰病變證

| 變證 | 病機 | 主方 |
|------|------|------|
| 太陰腹痛 | 脾絡不和 | 桂枝加芍藥湯 |
| 太陰發黃 | 寒濕內阻 | 茵陳朮附湯 |

> 💡 太陰病的治療核心是「當溫之」——以理中湯為代表的溫中健脾法，是後世脾胃學說的源頭。
