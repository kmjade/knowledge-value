# 09 — 雜病篇（金匱要略）

> 《金匱要略》是《傷寒雜病論》的雜病部分，以臟腑經絡辨證論治內傷雜病。

## 金匱要略結構

| 篇章 | 內容 | 代表方 |
|------|------|--------|
| 痙濕暍病 | 痙病、濕病、中暑 | 栝蔞桂枝湯 |
| 百合狐惑陰陽毒 | 精神疾病、黏膜病變 | 百合地黃湯 |
| 瘧病 | 瘧疾 | 鱉甲煎丸 |
| 中風歷節 | 中風、關節痛 | 桂枝芍藥知母湯 |
| 血痺虛勞 | 神經痛、虛損 | 黃耆桂枝五物湯、腎氣丸 |
| 肺痿肺癰咳嗽上氣 | 呼吸系統 | 葶藶大棗瀉肺湯 |
| 胸痺心痛短氣 | 心血管 | 栝蔞薤白白酒湯 |
| 腹滿寒疝宿食 | 消化系統 | 大建中湯 |
| 痰飲咳嗽 | 水液代謝病 | 苓桂朮甘湯 |
| 水氣病 | 水腫 | 防己黃耆湯 |
| 婦人病 | 婦科（三篇） | 溫經湯、當歸芍藥散 |

## 傷寒與金匱對比

| | 傷寒論 | 金匱要略 |
|------|--------|---------|
| 辨證 | 六經辨證 | 臟腑經絡辨證 |
| 病因 | 外感 | 內傷 |
| 傳變 | 循六經傳變 | 依臟腑生剋 |
| 治法 | 祛邪為主 | 扶正祛邪 |

> 💡 金匱要略是中醫內科學的源頭——胸痺（冠心病）、水腫（腎病）、婦人病的經典方劑至今仍在臨床一線使用。
