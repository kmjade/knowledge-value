# 06 — 病因病機

> 內經系統論述了疾病的原因（病因）和發生發展的機制（病機），建立了中醫病理學的基礎框架。

## 核心篇章

- **素問·調經論**：陰陽失調致病
- **素問·至真要大論**：病機十九條
- **素問·舉痛論**：九氣為病
- **靈樞·百病始生**：外邪侵襲路徑

## 病因分類

### 外因 — 六淫

| 邪氣 | 性質 | 致病特點 |
|------|------|---------|
| **風** | 陽邪，善行數變 | 遊走性疼痛、眩暈、抽搐 |
| **寒** | 陰邪，收引凝滯 | 疼痛、畏寒、氣血凝滯 |
| **暑** | 陽邪，炎熱升散 | 高熱、多汗、耗氣傷津 |
| **濕** | 陰邪，重濁黏滯 | 沉重、腫脹、纏綿難癒 |
| **燥** | 陽邪，乾澀 | 口鼻乾燥、皮膚乾裂 |
| **火** | 陽邪，炎上 | 高熱、面紅、出血、神昏 |

### 內因 — 七情

| 情志 | 傷及 | 機制 |
|------|------|------|
| 怒 | 肝 | 氣逆（怒則氣上） |
| 喜 | 心 | 氣緩（喜則氣緩） |
| 思 | 脾 | 氣結（思則氣結） |
| 悲 | 肺 | 氣消（悲則氣消） |
| 恐 | 腎 | 氣下（恐則氣下） |
| 驚 | 心 | 氣亂 |
| 憂 | 肺 | 氣鬱 |

### 不內外因
飲食不節、勞逸失度、外傷、蟲獸傷。

## 發病機制

| 概念 | 內經原文 | 說明 |
|------|---------|------|
| 正邪關係 | 「正氣存內，邪不可干」 | 正氣強則不病 |
| | 「邪之所湊，其氣必虛」 | 正氣虛則邪氣入侵 |
| 陰陽失調 | 「陰勝則陽病，陽勝則陰病」 | 陰陽平衡被打破 |

## 病機十九條（素問·至真要大論）

| 歸屬 | 條文舉例 |
|------|---------|
| 肝 | 「諸風掉眩，皆屬於肝」 |
| 心 | 「諸痛癢瘡，皆屬於心」 |
| 脾 | 「諸濕腫滿，皆屬於脾」 |
| 肺 | 「諸氣膹鬱，皆屬於肺」 |
| 腎 | 「諸寒收引，皆屬於腎」 |

> 💡 內經病機的核心思想：疾病的發生不是單一原因，而是正邪力量對比的結果——扶正氣、祛邪氣是治療的總原則。
