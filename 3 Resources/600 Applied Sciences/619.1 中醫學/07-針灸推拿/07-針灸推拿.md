# 07 — 針灸推拿

> 針灸推拿是中醫的非藥物療法——透過刺激經絡穴位，調節氣血，恢復健康。

## 針刺療法

| 方法 | 說明 |
|------|------|
| **毫針** | 最常用的針具，直徑 0.25–0.40mm |
| **電針** | 針刺後接通微弱電流 |
| **溫針** | 針柄燃燒艾絨，針刺+溫熱 |
| **皮膚針** | 多針淺刺皮膚 |

### 針刺手法

| 手法 | 說明 |
|------|------|
| 提插 | 針體上下運動 |
| 捻轉 | 針體旋轉運動 |
| 補法 | 輕柔緩和，順經捻轉 |
| 瀉法 | 強烈快速，逆經捻轉 |

## 得氣

針刺後患者感到酸、麻、脹、重的感覺，醫者感到針下沉緊——表示經氣已被激發。

## 艾灸

| 方法 | 說明 |
|------|------|
| 直接灸 | 艾炷直接放在穴位上 |
| 間接灸 | 隔薑、隔蒜、隔鹽灸 |
| 溫和灸 | 艾條懸於穴位上方 |
| 溫灸器灸 | 使用灸盒或灸筒 |

## 拔罐

| 方法 | 說明 |
|------|------|
| 留罐 | 吸附後留置 10–15 分鐘 |
| 走罐 | 罐體在皮膚上滑動 |
| 閃罐 | 快速吸附後立即取下，反覆多次 |

## 推拿

| 手法 | 說明 |
|------|------|
| 按法 | 用手指或手掌按壓 |
| 摩法 | 環形撫摸，輕柔 |
| 推法 | 單方向推動 |
| 拿法 | 捏而提起 |
| 揉法 | 旋轉揉動 |
| 點法 | 指尖點壓穴位 |

> 💡 針灸的療效已被現代研究證實——WHO 推薦針灸治療 100+ 種疾病。針刺鎮痛的機理與內源性鴉片肽釋放有關。
