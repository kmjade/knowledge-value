# 張仲景 Zhang Zhongjing

> 醫聖——《傷寒雜病論》作者，奠定了中醫臨床辨證論治體系。

## 基本信息

- **類型**: 醫學人物 (Person)
- **年代**: 東漢 (約公元 150–219 年)
- **稱號**: 醫聖
- **著作**: 《傷寒雜病論》（後分為《傷寒論》與《金匱要略》）
- **籍貫**: 南陽郡涅陽（今河南南陽）

## 核心貢獻

### 六經辨證體系

張仲景最大的貢獻是將《黃帝內經》的理論與臨床實踐結合，創立了 **六經辨證** 體系：

```
太陽病 → 陽明病 → 少陽病 → 太陰病 → 少陰病 → 厥陰病
 表證      裡實熱    半表半裡   裡虛寒    全身虛寒   寒熱錯雜
```

### 方書之祖

《傷寒雜病論》載方 269 首，被譽為「方書之祖」：

| 代表方劑 | 功效 | 現代應用 |
|---------|------|---------|
| 桂枝湯 | 解肌發表、調和營衛 | 感冒、過敏 |
| 麻黃湯 | 發汗解表、宣肺平喘 | 風寒感冒 |
| 小柴胡湯 | 和解少陽 | 感冒、肝炎 |
| 四逆湯 | 回陽救逆 | 休克、心衰 |
| 白虎湯 | 清熱生津 | 高熱、感染 |

## 學術思想

> 「觀其脈證，知犯何逆，隨證治之。」——《傷寒論》

核心原則是 **辨證論治**——根據患者的具體症狀和體徵來確定治療方案，而非按病名套方。
