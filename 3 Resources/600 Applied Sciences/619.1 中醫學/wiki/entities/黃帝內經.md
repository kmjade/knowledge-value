# 黃帝內經 Huangdi Neijing

> 中醫學理論體系的奠基之作——兩千年來中醫臨床與理論的最高經典。

## 基本信息

- **類型**: 經典著作 (Classic Text)
- **成書年代**: 戰國至漢代（約公元前 475 年–公元 220 年）
- **作者**: 託名黃帝，實為多代人集體創作
- **結構**: 素問 81 篇 + 靈樞 81 篇 = 162 篇

## 內容結構

### 素問 (Suwen) — Basic Questions
共 81 篇，側重於：
- 陰陽五行理論基礎
- 臟腑經絡學說
- 病因病機理論
- 診斷治療原則
- 養生預防思想

### 靈樞 (Lingshu) — Spiritual Pivot
共 81 篇（現存），側重於：
- 經絡腧穴體系（核心）
- 針刺手法與原則
- 營衛氣血運行
- 體質分類

## 核心貢獻

| 領域 | 貢獻 |
|------|------|
| **陰陽五行** | 奠定中醫哲學基礎 |
| **臟腑學說** | 系統描述五臟六腑功能 |
| **經絡學說** | 十二經脈 + 奇經八脈體系 |
| **病因病機** | 六淫、七情、飲食勞倦等病因論 |
| **治未病** | 「聖人不治已病治未病」的預防思想 |
| **針灸理論** | 腧穴定位、針刺深淺、補瀉手法 |
