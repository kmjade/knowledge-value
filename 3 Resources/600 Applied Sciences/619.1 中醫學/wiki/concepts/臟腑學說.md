# 臟腑學說 Zang-Fu Organ Theory

> 五臟者，所以藏精神血氣魂魄者也；六腑者，所以化水穀而行津液者也。——《靈樞·本臟》

## 定義 Definition

臟腑學說是研究人體 **五臟、六腑、奇恆之腑** 的生理功能、病理變化及其相互關係的理論，是中醫學生理學的核心。

## 五臟 Five Zang (Yin Organs)

五臟主「藏精氣而不瀉」，屬陰。

| 臟 | 五行 | 主要功能 | 外合 | 開竅 |
|----|:--:|------|------|:--:|
| **心 Heart** | 火 | 主血脈、藏神 | 脈 | 舌 |
| **肝 Liver** | 木 | 主疏泄、藏血 | 筋 | 目 |
| **脾 Spleen** | 土 | 主運化、統血 | 肉 | 口 |
| **肺 Lung** | 金 | 主氣、司呼吸 | 皮 | 鼻 |
| **腎 Kidney** | 水 | 藏精、主水液 | 骨 | 耳 |

## 六腑 Six Fu (Yang Organs)

六腑主「傳化物而不藏」，屬陽。

| 腑 | 功能 |
|----|------|
| 膽 | 貯存膽汁、主決斷 |
| 胃 | 受納腐熟水穀 |
| 小腸 | 受盛化物、分清泌濁 |
| 大腸 | 傳導糟粕 |
| 膀胱 | 貯存與排泄尿液 |
| 三焦 | 元氣通道、水液運行 |

## 臟腑關係 Key Relationships

### 表裡關係 (Interior-Exterior)

```
肝 ↔ 膽、心 ↔ 小腸、脾 ↔ 胃
肺 ↔ 大腸、腎 ↔ 膀胱、心包 ↔ 三焦
```

### 生剋關係 (Generating & Restraining)

```
腎水生肝木 → 肝木生心火 → 心火生脾土 → 脾土生肺金 → 肺金生腎水
```

## 臨床應用

| 應用 | 示例 |
|------|------|
| **辨證定位** | 咳嗽 → 病位在肺；水腫 → 病位在肺脾腎 |
| **五行治則** | 培土生金（補脾益肺）、滋水涵木（補腎養肝） |
| **臟腑同治** | 肝氣犯胃 → 疏肝和胃 |
