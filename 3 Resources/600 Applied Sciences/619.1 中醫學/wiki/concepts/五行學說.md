# 五行學說 Five Elements Theory

> 五行者，金木水火土也。更貴更賤，以知死生，以決成敗。——《素問·臟氣法時論》

## 定義 Definition

五行學說是將自然界和人體的一切現象歸納為 **木、火、土、金、水** 五種基本屬性及其運動變化規律的哲學體系，是中醫學解釋臟腑關係、疾病傳變的核心理論工具。

## 五行歸類 Five Element Classification

| 類別 | 木 Wood | 火 Fire | 土 Earth | 金 Metal | 水 Water |
|------|---------|---------|----------|----------|----------|
| **方向** | 東 | 南 | 中 | 西 | 北 |
| **季節** | 春 | 夏 | 長夏 | 秋 | 冬 |
| **五臟** | 肝 Liver | 心 Heart | 脾 Spleen | 肺 Lung | 腎 Kidney |
| **五腑** | 膽 | 小腸 | 胃 | 大腸 | 膀胱 |
| **五官** | 目 | 舌 | 口 | 鼻 | 耳 |
| **五體** | 筋 | 脈 | 肉 | 皮 | 骨 |
| **五志** | 怒 | 喜 | 思 | 悲 | 恐 |
| **五味** | 酸 | 苦 | 甘 | 辛 | 鹹 |
| **五色** | 青 | 赤 | 黃 | 白 | 黑 |

## 生剋制化 Relationships

### 相生 (Generating Cycle)
```
木 → 火 → 土 → 金 → 水 → 木
木生火、火生土、土生金、金生水、水生木
```

### 相剋 (Restraining Cycle)
```
木 → 土 → 水 → 火 → 金 → 木
木剋土、土剋水、水剋火、火剋金、金剋木
```

### 相乘與相侮 (Over-acting & Insulting)
- **相乘**：過度的剋制（病理狀態）
- **相侮**：反向剋制（被剋者反剋）

## 臨床應用 Clinical Application

| 應用 | 示例 |
|------|------|
| **臟腑關聯** | 肝木剋脾土 → 肝氣鬱結可致脾胃虛弱 |
| **疾病傳變** | 見肝之病，知肝傳脾，當先實脾 |
| **藥物歸經** | 青色入肝、赤色入心、黃色入脾 |
| **飲食養生** | 春食酸（應肝）、夏食苦（應心） |
| **情志調節** | 怒傷肝、喜傷心、思傷脾、悲傷肺、恐傷腎 |

## 經典闡發

> 「亢則害，承乃制，制則生化。」——《素問·六微旨大論》

五行之間通過生剋制化維持動態平衡——任何一行過亢都會產生危害，需要有制約才能生化不息。
