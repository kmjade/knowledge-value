# 氣血津液 Qi, Blood & Body Fluids

> 人之血氣精神者，所以奉生而周於性命者也。——《靈樞·本臟》

## 定義 Definition

氣、血、津液是構成人體和維持生命活動的基本物質。**氣** 屬陽，主推動、溫煦；**血與津液** 屬陰，主滋養、濡潤。

## 氣 Qi

### 氣的分類

| 分類 | 來源/位置 | 功能 |
|------|---------|------|
| **元氣** (Yuan Qi) | 先天之精，藏於腎 | 生命原動力 |
| **宗氣** (Zong Qi) | 肺吸入清氣 + 脾運化水穀 | 司呼吸、行血脈 |
| **營氣** (Ying Qi) | 水穀精微 | 營養全身、化生血液 |
| **衛氣** (Wei Qi) | 水穀之悍氣 | 溫養肌膚、抵禦外邪 |

### 氣的功能

- **推動** — 促進生長發育、推動血行
- **溫煦** — 維持體溫
- **防禦** — 抵抗外邪
- **固攝** — 統攝血液、控制汗液
- **氣化** — 物質與能量的轉化

## 血 Blood

### 血的生成

```
水穀精微 + 營氣 + 津液 ──（心化赤）──→ 血
```

### 血的功能

- **營養滋潤**全身組織器官
- **載氣**——血為氣之母
- 精神活動的物質基礎

## 津液 Body Fluids

| | 津 Jin | 液 Ye |
|---|--------|-------|
| 質地 | 清稀、流動性大 | 稠厚、流動性小 |
| 分佈 | 皮膚、肌肉、孔竅 | 關節、臟腑、腦髓 |
| 功能 | 滋潤 | 濡養 |

## 氣血津液關係

```
氣為血之帥 → 氣能生血、行血、攝血
血為氣之母 → 血能載氣、養氣
津血同源  → 津液與血液相互轉化
```
