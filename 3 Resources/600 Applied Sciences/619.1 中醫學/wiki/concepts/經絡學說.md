# 經絡學說 Meridian & Collateral Theory

> 經脈者，所以能決死生、處百病、調虛實，不可不通。——《靈樞·經脈》

## 定義 Definition

經絡是人體運行氣血、聯絡臟腑肢節、溝通上下內外的通道系統。**經** 是主幹（縱行），**絡** 是分支（網絡），二者構成遍布全身的網絡系統。

## 經絡功能 Functions

| 功能 | 說明 |
|------|------|
| **運行氣血** | 將氣血輸送到全身各處 |
| **聯絡臟腑** | 溝通五臟六腑與體表 |
| **感應傳導** | 傳遞針刺、推拿的刺激 |
| **調節平衡** | 通過經絡調節臟腑功能 |

## 臨床應用 Clinical Application

### 針灸 Acupuncture
通過刺激特定經絡上的腧穴來調節臟腑功能。

### 診斷應用
- 經絡循行部位的疼痛、麻木可定位病變臟腑
- 特定腧穴的壓痛可輔助診斷
