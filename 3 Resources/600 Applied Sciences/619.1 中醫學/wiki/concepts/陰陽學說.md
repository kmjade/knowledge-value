# 陰陽學說 Yin-Yang Theory

> 陰陽者，天地之道也，萬物之綱紀，變化之父母，生杀之本始，神明之府也。——《素問·陰陽應象大論》

## 定義 Definition

陰陽學說是中醫學的核心哲學基礎，認為宇宙萬物皆由 **陰** 與 **陽** 兩種對立統一的屬性構成，二者的消長平衡、相互轉化是生命活動的根本規律。

## 核心屬性 Core Properties

| 屬性 | 陰 Yin | 陽 Yang |
|------|--------|---------|
| 方向 | 下、內、右 | 上、外、左 |
| 溫度 | 寒、涼 | 溫、熱 |
| 亮度 | 暗 | 明 |
| 動態 | 靜、降、斂 | 動、升、散 |
| 物質/功能 | 物質 (形) | 功能 (氣) |
| 臟腑 | 五臟 (心肝脾肺腎) | 六腑 (膽胃大腸小腸膀胱三焦) |

## 陰陽關係 Relationships

### 1. 對立制約 Opposition
陰陽相互對立、相互制約。如：寒制熱、靜制動。

### 2. 互根互用 Interdependence
陰陽互為存在的條件。「孤陰不生，獨陽不長」——沒有陰就沒有陽，反之亦然。

### 3. 消長平衡 Dynamic Balance
陰陽處於不斷的消長變化中，維持動態平衡。例如：一日之中，子時陰極陽生，午時陽極陰生。

### 4. 相互轉化 Mutual Transformation
在一定條件下，陰陽可以相互轉化。「重陰必陽，重陽必陰」——極端狀態會向對立面轉化。

## 臨床應用 Clinical Application

| 應用 | 說明 |
|------|------|
| **病理診斷** | 陽盛則熱、陰盛則寒；陽虛則寒、陰虛則熱 |
| **治療原則** | 「謹察陰陽所在而調之，以平為期」——調整陰陽至平衡 |
| **藥物分類** | 寒涼藥屬陰、溫熱藥屬陽 |
| **養生指導** | 「春夏養陽，秋冬養陰」——順應四時陰陽變化 |

## 經典闡發

> 「陰在內，陽之守也；陽在外，陰之使也。」——《素問·陰陽應象大論》

陰主內守、陽主外護——二者分工協作，共同維持人體正常功能。
