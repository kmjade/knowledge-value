# 制造与工艺 (UDC 67-68)

> UDC 67-68 涵盖从冶金制品到家具、从服装到玩具的各种工业、贸易与手工艺——将原材料转化为最终产品的制造过程。

## 关键制造工艺

### 成型工艺

| 工艺 | 材料 | 说明 |
|------|------|------|
| **铸造** | 金属 | 熔融→浇入模具→凝固 |
| **注塑** | 塑料 | 熔融塑料→高压注入模具 |
| **冲压** | 金属板材 | 模具冲切/弯折→一次成型 |
| **锻造** | 金属 | 压力→改变形状+改善晶粒 |
| **挤出** | 塑料/铝 | 材料→通过模具口→连续截面 |

### 连接工艺

| 工艺 | 适用 | 特点 |
|------|------|------|
| **焊接** | 金属 | 永久连接 |
| **榫接** | 木材 | 传统木工精髓 |
| **缝制** | 纺织/皮革 | 柔性连接 |
| **胶合** | 通用 | 简单但有老化问题 |
| **铆接/螺栓** | 金属 | 可拆卸 |

### 表面处理

| 工艺 | 作用 | 应用 |
|------|------|------|
| **电镀** | 防腐蚀+装饰 | 镀铬、镀锌 |
| **涂装** | 保护+颜色 | 喷漆、粉末涂装 |
| **染色** | 着色 | 纺织品、皮革 |
| **抛光** | 光滑+光泽 | 金属、石材、木材 |

## 工业 4.0 在手工艺中的渗透

|------|----------|
| CAD/CAM | 手工绘图→数字设计→CNC 加工 |

| 激光切割 | 手工锯切→激光精密切割 |
| IoT | 手工记录→传感器实时监测 |
