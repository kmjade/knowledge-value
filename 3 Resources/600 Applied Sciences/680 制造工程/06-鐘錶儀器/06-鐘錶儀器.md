# 06 — 鐘錶與儀器製造

> 鐘錶與儀器製造是精密機械的極致——從機械錶到科學儀器。

## 鐘錶製造

| 類型 | 技術 |
|------|------|
| 機械錶 | 擒縱機構、陀飛輪、發條 |
| 石英錶 | 晶體振盪、步進馬達 |
| 原子鐘 | 銫原子共振（精度 10⁻¹⁵） |

## 測量儀器

| 儀器 | 用途 |
|------|------|
| 三次元量測儀 (CMM) | 精密尺寸量測 |
| 雷射干涉儀 | 奈米級位移量測 |
| 光譜儀 | 物質成分分析 |
| 電子顯微鏡 | 微觀結構觀察 |

> 💡 瑞士機械錶是精密製造的典範——數百個微小零件在方寸之間精準運作，誤差以秒/天計。
