# 09 — 品質控制

> 品質控制確保製造產品符合規格——從進料到出貨。

## 檢測方法

| 方法 | 說明 |
|------|------|
| 外觀檢查 | 目視/AOI |
| 尺寸量測 | 卡尺、CMM、雷射 |
| 材料分析 | 光譜、硬度、拉伸 |
| 功能測試 | 性能、耐久、環境 |
| 非破壞檢測 | X-Ray、超音波、渦電流 |

## 品質管理體系

| 體系 | 說明 |
|------|------|
| ISO 9001 | 品質管理系統 |
| IATF 16949 | 汽車業品質標準 |
| Six Sigma | 3.4 DPMO |
| SPC | 統計製程管制 |

## 計量與標準

| 概念 | 說明 |
|------|------|
| 校準 | 儀器與標準比對 |
| 追溯 | 量測結果可追溯至國家標準 |
| 不確定度 | 量測結果的可信範圍 |

> 💡 沒有測量就沒有品質——品質控制的核心是精準的計量。
