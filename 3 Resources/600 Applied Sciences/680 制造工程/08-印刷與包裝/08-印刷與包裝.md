# 08 — 印刷與包裝

> 印刷與包裝是產品走向市場的最後一哩——從標籤到外包裝。

## 印刷技術

| 技術 | 特點 | 應用 |
|------|------|------|
| 平版印刷 | 高品質、大量 | 書刊、海報 |
| 柔版印刷 | 適合包裝材料 | 紙箱、標籤 |
| 凹版印刷 | 高耐久 | 鈔票、郵票 |
| 網版印刷 | 多材質 | 紡織、電子 |
| 數位印刷 | 可變資料、短版 | 個人化印刷 |
| 3D 列印 | 增材製造 | 原型、小批量 |

## 包裝工程

| 功能 | 說明 |
|------|------|
| 保護 | 防震、防潮、防氧化 |
| 標識 | 條碼、QR Code、RFID |
| 永續 | 可回收、可降解材料 |

> 💡 包裝是產品的第一印象——好的包裝設計同時滿足保護、行銷和永續三大需求。
