# 02 — 服裝與紡織

> 衣著不僅是自我表達，更是材料科學——了解纖維、設計與保養，延長衣物的生命。

## 紡織纖維分類

| 類別 | 纖維 | 特性 |
|------|------|------|
| **天然植物** | 棉、麻 | 吸濕透氣、舒適 |
| **天然動物** | 羊毛、蠶絲 | 保暖、光澤、彈性 |
| **再生纖維** | 嫘縈、莫代爾 | 柔軟、吸濕 |
| **合成纖維** | 聚酯、尼龍、彈性纖維 | 耐用、快乾、抗皺 |

## 服裝設計要素

| 要素 | 說明 |
|------|------|
| **線條** | 垂直線顯瘦、水平線顯寬 |
| **色彩** | 暖色前進、冷色後退 |
| **質地** | 光滑反光顯胖、粗糙吸光顯瘦 |
| **圖案** | 大圖案膨脹、小圖案收縮 |

## 服裝保養

| 標示 | 說明 |
|------|------|
| 水洗 | 數字為最高水溫 (°C) |
| 漂白 | △ = 可漂白；△╳ = 不可 |
| 熨燙 | 點數 = 溫度（1低 2中 3高） |
| 乾洗 | ○ = 可乾洗 |
| 烘乾 | □ = 可烘乾 |

## 膠囊衣櫥 (Capsule Wardrobe)

以最少件數創造最多搭配——約 30–40 件核心單品，每季可配出數十種組合。

> 💡 最好的衣物不是最貴的，而是最常穿的——購買前問自己：我會穿 30 次以上嗎？
