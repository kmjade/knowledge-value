# 07 — 室內設計

> 室內設計不只是美化空間——它是創造功能、舒適與美感兼備的生活環境。

## 設計要素

| 要素 | 說明 |
|------|------|
| **空間** | 虛與實的關係、動線規劃 |
| **色彩** | 影響情緒與空間感 |
| **光線** | 自然光 + 人工照明 |
| **材質** | 質感、觸感、耐用性 |
| **家具** | 功能、比例、風格統一 |
| **裝飾** | 藝術品、織品、植栽 |

## 色彩心理學

| 色系 | 感受 | 適合空間 |
|------|------|---------|
| 暖色 (紅、橙、黃) | 溫暖、活力、食慾 | 餐廳、客廳 |
| 冷色 (藍、綠、紫) | 平靜、放鬆、專注 | 臥室、書房 |
| 中性 (白、灰、米) | 乾淨、寬敞、百搭 | 任何空間 |

## 照明設計

| 類型 | 功能 |
|------|------|
| **環境照明** | 基礎照明，均勻分布 |
| **任務照明** | 特定工作區域（閱讀燈、廚房檯面） |
| **重點照明** | 強調藝術品或建築特色 |
| **裝飾照明** | 純美學效果 |

## 空間風格

| 風格 | 特色 |
|------|------|
| 北歐風 | 簡約、自然材質、淺色調 |
| 日式風 | 極簡、木質、留白 |
| 工業風 | 裸露管線、金屬、水泥 |
| 鄉村風 | 溫暖、復古、手工感 |
| 現代風 | 乾淨線條、中性色、玻璃與金屬 |

> 💡 好的室內設計不是花更多錢，而是做更好的決策——比例、光線和動線比昂貴的材料更重要。
