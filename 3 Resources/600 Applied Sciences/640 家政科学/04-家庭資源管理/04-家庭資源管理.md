# 04 — 家庭資源管理

> 家庭資源是有限的——金錢、時間、精力都需要被有效管理，才能實現家庭目標。

## 家庭資源類型

| 資源 | 特性 | 管理挑戰 |
|------|------|---------|
| **金錢** | 可量化、可儲存 | 收入有限、慾望無限 |
| **時間** | 每人每天 24 小時 | 無法儲存、一去不回 |
| **精力** | 體力與心力 | 需休息恢復 |
| **人際** | 家庭關係、社會網絡 | 需經營維護 |
| **知識** | 技能與資訊 | 需持續學習 |

## 家庭預算管理

| 類別 | 佔收入比例 |
|------|-----------|
| 住房 | ≤ 30% |
| 飲食 | 15–20% |
| 交通 | 10–15% |
| 儲蓄/投資 | ≥ 20% |
| 保險 | 5–10% |
| 娛樂/其他 | 10–15% |

## 時間管理矩陣 (Eisenhower)

|  | 緊急 | 不緊急 |
|------|------|--------|
| **重要** | 立即處理 (危機、截止日) | 規劃執行 (學習、健康、關係) |
| **不重要** | 委派處理 (干擾、部分會議) | 減少/消除 (滑手機、追劇) |

## 家庭決策模型

| 步驟 | 說明 |
|------|------|
| 1. 定義問題 | 明確需要決策的事項 |
| 2. 蒐集資訊 | 收集相關資料與選項 |
| 3. 評估選項 | 比較各方案的優缺點 |
| 4. 做出決策 | 選擇最佳方案 |
| 5. 執行與評估 | 行動並檢討結果 |

> 💡 家庭理財的第一原則：先存後花，而非先花後存。Pay yourself first.
