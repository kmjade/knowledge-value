# 傷寒雜病論常見問題

## 1. 「傷寒就是感冒？」

**迷思**：傷寒論就是治療感冒的書。
**事實**：傷寒論中的「傷寒」是廣義的外感熱病，涵蓋從感冒到重症感染的多種疾病。六經辨證體系可用於各類內外疾病，遠超「感冒」範疇。

## 2. 「經方不能加減？」

**迷思**：經方必須原方原量，不能改動。
**事實**：張仲景本人就大量加減——桂枝湯的加減方達數十首。經方的核心是方證對應的思路，而非僵化的固定組成。

## 3. 「六經等於十二經脈？」

**迷思**：六經辨證 = 經絡辨證。
**事實**：六經辨證是疾病發展階段的綜合模型，包含臟腑、經絡、氣化多層次內涵，不只是經絡循行的概念。

## 4. 「傷寒論只有 113 方，能治多少病？」

**事實**：經方貴在配伍精妙——桂枝湯加減可衍生數十方。方不在多，在於辨證準確。核心經方不過數十首，但臨床應用涵蓋內外婦兒各科。

## 5. 「兩千年前的方子現在還有效嗎？」

**事實**：人體的生理病理規律沒有改變。桂枝湯治感冒、白虎湯退高熱、四逆湯抗休克——經方的療效已被兩千年臨床實踐和現代研究雙重驗證。
