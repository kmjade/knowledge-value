# 07 — 領導力

> 領導不是職位，而是影響力——讓他人願意追隨你，共同達成願景。

## 管理 vs 領導

| 管理 | 領導 |
|------|------|
| 處理複雜性 | 處理變革 |
| 規劃與預算 | 設定方向 |
| 組織與配員 | 協調與溝通 |
| 控制與解決問題 | 激勵與啟發 |
| 把事情做對 | 做對的事情 |

## 主要領導理論

| 理論 | 時期 | 核心思想 |
|------|------|---------|
| **特質理論** | 1940s | 領導者天生具有某些特質 |
| **行為理論** | 1950s | 領導是可以學習的行為 |
| **情境理論** | 1970s | 最佳風格取決於下屬成熟度 |
| **轉換型領導** | 1980s | 理想化影響、激勵、智力啟發、個別關懷 |
| **僕人式領導** | 1970s | 領導者首先是僕人 |

## 領導風格

| 風格 | 說明 | 適合情境 |
|------|------|---------|
| **權威型** | "跟我來" | 危機、轉型 |
| **民主型** | "你怎麼看？" | 需要團隊共識 |
| **親和型** | "人最重要" | 修復團隊關係 |
| **教練型** | "試試看" | 發展人才 |
| **領跑型** | "照我做的做" | 高績效團隊 |
| **命令型** | "照我說的做" | 緊急情況 |

## GROW 教練模型

| 步驟 | 問題 |
|------|------|
| **G**oal | 你想要達成什麼？ |
| **R**eality | 現狀是什麼？ |
| **O**ptions | 有哪些選擇？ |
| **W**ill | 你將採取什麼行動？ |

> 💡 最好的領導者不是最有權力的人，而是最能激發他人潛力的人。
