# 08 — 商業溝通

> 溝通是管理的基本工具——管理者 80% 的時間花在溝通上。有效的溝通不是「說了什麼」，而是「對方收到什麼」。

## 溝通模型

發送者 → 編碼 → 訊息 → 管道 → 解碼 → 接收者

- 噪音干擾影響每個環節
- 回饋是溝通的閉環

## 溝通障礙

| 障礙 | 說明 |
|------|------|
| **選擇性知覺** | 人們聽到他們想聽的 |
| **資訊過載** | 太多資訊導致忽略 |
| **情緒** | 情緒影響訊息解讀 |
| **語言** | 專業術語、文化差異 |
| **過濾** | 訊息在傳遞中被扭曲 |

## 談判策略

| 策略 | 說明 |
|------|------|
| **分配式談判** | 零和博弈，分割固定資源 |
| **整合式談判** | 創造價值，雙贏 |
| **BATNA** | 談判協議的最佳替代方案 |

## 衝突管理風格

| 風格 | 說明 |
|------|------|
| **競爭** | 我贏你輸（緊急情況） |
| **合作** | 雙贏（最佳但耗時） |
| **妥協** | 各退一步 |
| **迴避** | 暫時不理 |
| **遷就** | 你贏我輸（關係重要時） |

## Hofstede 文化維度

| 維度 | 高 | 低 |
|------|-----|-----|
| 權力距離 | 接受階層 | 追求平等 |
| 個人主義 | 個人優先 | 群體優先 |
| 不確定性規避 | 需要規則 | 容忍模糊 |
| 長期導向 | 著眼未來 | 重視傳統 |

> 💡 溝通最大的問題是「以為已經溝通好了」。
