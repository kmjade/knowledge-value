# 03-通信技术

> 通信技术解决"如何让信息跨越空间"——从烽火台到光纤，核心目标始终是更快、更远、更可靠。

## 通信的演进

| 时代 | 技术 | 关键突破 |
|------|------|----------|
| 1840s | 电报 | 首次即时远距离通信 |
| 1876 | 电话 | 传输声音 |
| 1895 | 无线电 | 无线传输——马可尼 |
| 1948 | 信息论 | 香农——通信的数学基础 |
| 1960s | 卫星通信 | 全球覆盖 |
| 1970s | 光纤 | 海量带宽 |
| 1980s | 蜂窝网络 (1G) | 移动通信开始 |
| 2020s | 5G/6G | 不只是手机——万物互联 |

## 移动通信世代

| 世代 | 技术 | 速率 | 标志性应用 |
|------|------|------|-----------|
| **1G** | 模拟 | ~2.4 kbps | 语音 |
| **2G** | GSM/CDMA 数字 | ~64 kbps | 短信 |
| **3G** | WCDMA | ~2 Mbps | 移动网页 |
| **4G** | LTE | ~100 Mbps | 视频、App |
| **5G** | NR | ~1-10 Gbps | IoT、自动驾驶、远程手术 |
| **6G** (2030s) | 太赫兹+AI | ~1 Tbps | 全息通信、数字孪生 |

### 5G 不只是"更快的 4G"

| 特性 | 说明 |
|------|------|
| **eMBB** (增强移动宽带) | 超高速——8K 视频 |
| **uRLLC** (超低延迟) | <1ms——远程手术、工业控制 |
| **mMTC** (海量机器通信) | 每平方公里百万设备——智能城市 |

## 香农定理

通信的物理极限：

```
C = B × log₂(1 + S/N)

C = 信道容量 (bps)
B = 带宽 (Hz)
S/N = 信噪比
```

> 这是"为什么不能无限提速"的物理答案——给定带宽和信噪比，最大速率是有限的。

## 光纤

| 类型 | 特点 |
|------|------|
| 单模光纤 | 细芯 (~9μm)，长距离，昂贵 |
| 多模光纤 | 粗芯 (~50μm)，短距离，便宜 |

> 一根光纤的带宽：理论 ~100 Tbps，目前商用 ~100 Gbps-1 Tbps 每波长。跨太平洋海底光缆承载 99% 的国际数据。

## 卫星通信

| 轨道 | 高度 | 延迟 | 用途 |
|------|------|------|------|
| **GEO** (地球同步) | 35,786 km | ~600ms | 电视广播 |
| **MEO** | 2,000-35,000 km | ~100ms | GPS |
| **LEO** (低轨) | 200-2,000 km | ~25ms | Starlink, 千帆星座 |

> Starlink 已发射 5000+ 颗 LEO 卫星——目标是低延迟全球宽带。中国"千帆星座"跟进。
