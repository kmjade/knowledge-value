# 03-机械工程

> 机械工程涉及运动和能量的传递与转换——从汽车发动机到风力涡轮，从钟表齿轮到机器人关节。

## 三大基础

| 基础 | 内容 | 核心定律 |
|------|------|----------|
| **热力学** | 能量转换和热传递 | 能量守恒、熵增原理 |
| **力学** | 力和运动 | F=ma、应力-应变 |
| **材料** | 金属、合金、复合材料的力学行为 | 疲劳、蠕变、断裂韧性 |

### 热力学四定律

| 定律 | 含义 |
|------|------|
| **第零定律** | 如果 A=B 和 B=C 温度相同，则 A=C——温度的定义 |
| **第一定律** | 能量守恒——能量不能凭空产生或消失 |
| **第二定律** | 热量不能自发地从低温到高温——熵总是增加 |
| **第三定律** | 绝对零度不可达到 |

### 热机

将热能转化为机械功的装置。

| 循环 | 应用 | 效率范围 |
|------|------|----------|
| 奥托循环 | 汽油发动机 | ~25-30% |
| 狄塞尔循环 | 柴油发动机 | ~35-45% |
| 布雷顿循环 | 燃气轮机（喷气发动机、发电） | ~35-60% |
| 朗肯循环 | 蒸汽轮机（火电、核电） | ~35-45% |

> 汽车发动机只有约 25% 的燃油能量变成推动力——其余以热量浪费。这就是为什么电气化是效率革命。

## 机械设计要素

| 要素 | 功能 | 例子 |
|------|------|------|
| **轴承** | 支撑旋转轴，减少摩擦 | 滚珠轴承、滑动轴承 |
| **齿轮** | 传递扭矩和改变转速 | 变速器、差速器 |
| **弹簧** | 储存和释放弹性能 | 悬挂系统 |
| **密封** | 防止泄漏 | O 形圈、垫片 |
| **紧固件** | 连接零件 | 螺栓、铆钉、焊接 |

### 失效模式

| 模式 | 原因 | 预防 |
|------|------|------|
| 疲劳 | 反复应力 | 疲劳寿命设计、避免应力集中 |
| 磨损 | 摩擦 | 润滑、硬化表面 |
| 腐蚀 | 化学/电化学反应 | 涂层、材料选择 |
| 蠕变 | 高温+持续应力 | 选择耐蠕变合金 |
| 屈服 | 应力超过屈服强度 | 安全系数 |

## 制造工艺

| 工艺 | 原理 | 适用 |
|------|------|------|
| **铸造** | 熔融金属浇入模具 | 复杂形状、大批量 |
| **锻造** | 压力使金属成型 | 高强度零件 |
| **机加工** | 切削去除材料 | 精密零件 |
| **焊接** | 熔化连接 | 结构件 |
| **3D 打印** | 逐层堆积 | 复杂几何、小批量 |
