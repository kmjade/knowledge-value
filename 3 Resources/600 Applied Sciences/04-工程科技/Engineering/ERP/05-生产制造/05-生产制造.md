# 05-生产制造

> ERP 的生产模块回答三个问题：做什么？需要什么？能做吗？——对应 BOM、MRP 和产能计划。

## BOM (物料清单)

产品的"配方"：

```
成品 (1个)
├── 半成品A (1个)
│   ├── 原材料1 (2个)
│   └── 原材料2 (3个)
└── 半成品B (2个)
    └── 原材料3 (1个)
```

## MRP (物料需求计划)

```
销售预测/订单 → 需要多少成品？
       ↓
BOM 展开 → 需要多少原材料/半成品？
       ↓
库存减去 → 还需要采购多少？生产多少？
       ↓
生成采购申请 + 生产工单
```

## 生产工单

| 要素 | 说明 |
|------|------|
| 工单号 | 唯一标识 |
| BOM | 需要什么物料 |
| 工艺路线 | 经过哪些工序 (车→铣→磨→检) |
| 数量 | 生产多少 |
| 时间 | 何时开始/结束 |
| 状态 | 创建→下达→发料→报工→完工 |

## MES vs ERP

| 维度 | ERP (PP 模块) | MES |
|------|-------------|-----|
| 层级 | 计划层 | 执行层 |
| 时间 | 天/周 | 秒/分 |
| 粒度 | 工单 | 每件/每工序 |
| 数据 | 计划数 | 实际数 |

> ERP 说"今天做 100 件"，MES 说"第 37 件在第 3 道工序，温度 85°C"。
