# 02-核心模块

> ERP 由多个功能模块组成——每个模块管理一个业务领域，底层共享同一数据库。

## 模块全景

```
        ┌──────────────────────────┐
        │      财务管理 (FI/CO)      │ ← 心脏
        └──────────┬───────────────┘
                   │
  ┌────┬────┬──────┼──────┬────┬────┐
  ↓    ↓    ↓      ↓      ↓    ↓    ↓
 采购  库存  生产  销售   HR  项目  质量
```

## 核心模块

| 模块 | 英文 | 做什么 |
|------|------|--------|
| **财务会计** | FI | 总账、应收应付、资产 |
| **管理会计** | CO | 成本中心、利润中心、产品成本 |
| **采购** | MM/PUR | 供应商、采购订单、入库 |
| **库存** | IM/WM | 库存数量/价值、仓库管理 |
| **生产** | PP | BOM、工单、MRP、排产 |
| **销售** | SD | 客户、报价、订单、发货、开票 |
| **人力资源** | HCM | 组织、薪资、考勤、招聘 |
| **项目** | PS/PM | 项目成本、进度、设备维护 |
| **质量** | QM | 检验、质量通知 |

## 模块间的信息流

```
销售订单 → MRP 计算 → 采购申请 → 采购订单 → 入库 → 
生产工单 → 领料 → 完工入库 → 发货 → 开票 → 收款 → 总账
```

> 这就是 ERP 的魔力——一笔销售自动触发采购、生产和财务的所有后续动作。
