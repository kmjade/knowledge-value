# 09-ERP 与工业4.0

> ERP 在工业 4.0 自动化金字塔的顶端——是业务决策和车间执行之间的桥梁。

## 自动化金字塔中的 ERP

```
ERP ───── 天/周 ──── 订单、财务、计划
  ↕
MES ───── 分/时 ──── 排产、追踪、质量
  ↕
SCADA ─── 秒/分 ──── 监控、报警
  ↕
PLC ───── ms ─────── 逻辑控制
  ↕
现场 ──── μs ─────── 传感器/执行器
```

## ERP + IoT

| 场景 | 说明 |
|------|------|
| 自动收货 | 地磅联网→自动创建收货单 |
| 自动报工 | 设备自动报产量→ERP 实时更新 |
| 预测维护 | 设备 IoT 数据→触发维修工单 |
| 质量闭环 | 在线检测→不良自动 ERP 冻结批次 |
