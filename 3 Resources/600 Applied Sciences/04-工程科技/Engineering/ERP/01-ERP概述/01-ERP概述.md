# 01-ERP 概述

> ERP 是整合企业所有核心业务流程的软件系统——财务、采购、生产、销售、HR 共用一个数据库，实时共享信息。

## 为什么需要 ERP？

| 没有 ERP | 有 ERP |
|----------|--------|
| 财务一套 Excel、仓库一套 Excel——对不上 | 一个数据库——所有人看同一组数 |
| 采购不知道库存还有多少 | 库存实时可见→按需采购 |
| 销售接了单不知道能不能做出来 | MRP 自动算物料和产能 |
| 月度报表要手工汇总一周 | 实时报表 |

## 核心特征

| 特征 | 说明 |
|------|------|
| **集成** | 所有模块共享一个数据库——没有信息孤岛 |
| **实时** | 业务发生→实时反映到财务 |
| **标准化** | 流程按最佳实践标准化 |
| **可追溯** | 任何一笔业务都可追溯到源头 |
