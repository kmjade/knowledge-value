# 03-财务与会计

> 财务是 ERP 的心脏——所有业务模块的最终结果都汇集到财务。财务会计 (FI) 对外报告，管理会计 (CO) 对内决策。

## FI vs CO

| 维度 | FI (财务会计) | CO (管理会计) |
|------|-------------|-------------|
| 对象 | 外部（税务局、审计） | 内部（管理层） |
| 标准 | 会计准则 | 内部管理需求 |
| 粒度 | 公司代码级 | 成本中心/利润中心级 |
| 例子 | 资产负债表、损益表 | 哪种产品最赚钱？ |

## FI 核心

| 子模块 | 做什么 |
|--------|--------|
| **总账 (GL)** | 所有会计科目的中心记录 |
| **应收账款 (AR)** | 客户欠我们多少钱 |
| **应付账款 (AP)** | 我们欠供应商多少钱 |
| **固定资产 (AA)** | 设备、建筑的折旧和价值 |

### 业务→分录：自动化

```
采购收货 → 自动生成: 借 库存 贷 GR/IR
发票校验 → 自动生成: 借 GR/IR 贷 应付账款
付款     → 自动生成: 借 应付账款 贷 银行存款
```

> ERP 的最大价值之一：业务和财务自动集成。不再需要"把入库单送到财务部→财务手工做账"。

## CO 核心

| 对象 | 用途 |
|------|------|
| **成本中心** | 部门花了多少钱？ |
| **利润中心** | 哪个事业部赚钱？ |
| **内部订单** | 某次市场活动的花费？ |
| **产品成本** | 生产一个产品花了多少材料+人工+制造费？ |
