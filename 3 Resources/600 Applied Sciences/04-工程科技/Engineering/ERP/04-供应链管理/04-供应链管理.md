# 04-供应链管理

> ERP 中的供应链管理覆盖了从采购到付款 (P2P) 和从订单到收款 (O2C) 的全过程。

## 采购到付款 (P2P)

```
采购申请 → 询价 → 采购订单 → 收货 → 发票校验 → 付款
```

| 步骤 | ERP 做什么 |
|------|-----------|
| 采购申请 | 自动生成——MRP 发现缺料 |
| 采购订单 | 记录数量/价格/交期→发给供应商 |
| 收货 | 入库→库存增加→自动生成财务凭证 |
| 发票校验 | 三单匹配(PO+收货+发票)→一致才付款 |

## 库存管理

| 概念 | 说明 |
|------|------|
| **安全库存** | 低于这个数→自动触发采购 |
| **批次管理** | 每批物料有唯一批号→可追溯 |
| **先进先出 (FIFO)** | 先入库的先用 |
| **ABC 分类** | A 类 (高价值 10%)、B 类 (中 20%)、C 类 (低 70%) |

## 从订单到收款 (O2C)

```
客户询价 → 报价 → 销售订单 → 发货 → 开票 → 收款
```
