# 08-云ERP与SaaS

> 2026 年，云 ERP 已是主流趋势——不再需要买服务器、装数据库、雇运维团队。

## 部署方式对比

| 维度 | 本地部署 | 云 ERP (SaaS) |
|------|----------|-------------|
| 初始成本 | 高 (服务器+许可证) | 低 (月付) |
| 运维 | 需要 IT 团队 | 厂商负责 |
| 升级 | 手动+痛苦 | 自动+平滑 |
| 定制 | 高度可定制 | 配置为主，定制有限 |
| 数据控制 | 数据在自己手里 | 数据在云端 |
| 适合 | 大型企业/特殊需求 | 中小企业/标准流程 |

## 多租户架构

```
┌─────────────────────────────┐
│        同一套代码             │
├─────────┬─────────┬─────────┤
│ Tenant A │ Tenant B │ Tenant C │ ← 数据隔离
│ (公司A)  │ (公司B)  │ (公司C)  │
└─────────┴─────────┴─────────┘
```

> 多租户的优势：厂商只需维护一套代码，所有客户同时受益于升级。
