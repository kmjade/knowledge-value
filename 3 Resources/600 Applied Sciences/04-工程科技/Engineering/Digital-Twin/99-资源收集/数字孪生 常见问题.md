# 数字孪生 常见问题

## 十大常见误区

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| 数字孪生 = 3D 模型 | 3D 可视化只是"皮"。真正的价值在于数据同步、仿真和预测——没有数据流的 3D 模型是数字模型，不是数字孪生 |
| 数字孪生 = 仿真 | 仿真可以是离线的、基于假设的。数字孪生是在线、实时同步的。两者互补，但不同 |
| 数字孪生只用于制造业 | 建筑(BIM→CIM)、医疗(人体数字孪生)、能源(电网孪生)、城市(新加坡 Virtual Singapore)——应用广泛 |
| 必须做完整的数字孪生 | 可以从一个关键设备开始，逐步扩展。最小的数字孪生只包含几个关键参数 |
| 数字孪生是 IT 项目 | 本质是 OT+IT 融合。需要领域专家（懂设备）、数据工程师和 IT 架构师协作 |
| 买一个平台就能建数字孪生 | 平台是工具，不是答案。真正的挑战是数据质量、模型准确性和组织变革 |
| 数字孪生是一次性项目 | 是持续演进的产品——物理资产会老化、改造，数字孪生必须同步更新 |
| 数字孪生适用于所有设备 | 投资回报取决于资产价值和复杂性。一台普通水泵的回本周期可能不值得 |
| 只有大公司能用 | 云平台和工业 IoT 降低了门槛。中小企业在关键设备上也能获益 |
| 数字孪生 = 工业 4.0 的全部 | 数字孪生是工业 4.0 的核心概念之一，但不是全部——还有增材制造、自主机器人等 |
