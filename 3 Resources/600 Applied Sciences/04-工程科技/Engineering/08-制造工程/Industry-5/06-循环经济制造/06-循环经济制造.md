# 06-循环经济制造

> 线性经济 = 取→造→扔。循环经济 = 设计→制造→使用→回收→再生。目标是零废弃——每一件产品在生命结束时都成为下一件产品的原料。

## 线性 vs 循环

```
线性: 原料 → 制造 → 使用 → 废弃 → 填埋/焚烧
循环: 原料 → 制造 → 使用 → 回收 → 再制造 → 原料
              ↑___________________________↓
```

## 循环策略 (R-阶梯)

优先级从高到低：

| 策略 | 说明 | 例子 |
|------|------|------|
| **R0 Refuse** | 根本不需要——拒绝 | 共享而非拥有 |
| **R1 Rethink** | 重新思考使用方式 | 产品即服务 |
| **R2 Reduce** | 减少材料和能耗 | 轻量化设计 |
| **R3 Reuse** | 直接再利用 | 可重复灌装的瓶子 |
| **R4 Repair** | 维修 | 换屏而非换手机 |
| **R5 Refurbish** | 翻新 | 二手设备翻新到出厂标准 |
| **R6 Remanufacture** | 再制造 | 发动机全部拆解→检测→更换磨损件→装配→测试 |
| **R7 Repurpose** | 改作他用 | 退役风电叶片→桥梁 |
| **R8 Recycle** | 回收材料 | 熔铝回收（节能 95%） |
| **R9 Recover** | 能量回收 | 焚烧发电（最末选择） |

> R0-R3 最优先——最高价值保留。R8-R9 是不得已的办法。循环经济不是"更好的回收"——是"不产生废弃物"。

## 面向循环的设计 (DfX)

| 原则 | 实施 |
|------|------|
| **模块化** | 容易拆解和更换——不用坏一个零件扔整机 |
| **标准化紧固件** | 减少工具种类——容易维修 |
| **单材料** | 减少材料种类——容易回收 |
| **标记材料** | 零件上标注材料类型 |
| **避免永久连接** | 卡扣替代胶水 |
| **长寿命设计** | 关键部件冗余设计——延长使用期 |

## 再制造

与维修的区别——再制造恢复到出厂标准：

| 步骤 | 说明 |
|------|------|
| 1. 全部拆解 | 100% 分解 |
| 2. 清洗 | 去除油污和残留 |
| 3. 检测 | 每个零件检测→分拣可用/可修/报废 |
| 4. 修复/更换 | 磨损件换新 |
| 5. 装配 | 和新品一样的装配流程 |
| 6. 测试 | 和新品一样的出厂测试 |

### 再制造的经济效益

| 产品 | 再制造成本 | 节能 | 节材 |
|------|-----------|------|------|
| 发动机 | 新品的 50-60% | 68-83% | 60-70% |
| 打印机墨盒 | 新品的 30-50% | — | 减少填埋 90%+ |

## 产品即服务 (PaaS)

不卖产品，卖使用效果：

| 例子 | 传统 | PaaS |
|------|------|------|
| 照明 | 买灯泡 | 按 lux 付费——飞利浦维护 |
| 发动机 | 卖发动机 | 按飞行小时付费——劳斯莱斯 |
| 压缩机 | 卖设备 | 按压缩空气 m³ 付费 |

> 好处：制造商有动力设计长寿命+易维修的产品——因为坏了要自己修。
