# 07-供应链韧性

> 韧性是工业 5.0 的第三支柱——供应链不仅要高效，还要能在冲击下存活并快速恢复。COVID-19、苏伊士运河堵塞、俄乌战争——每个事件都暴露了全球精益供应链的脆弱性。

## 效率 vs 韧性

| 维度 | 精益供应链 (4.0) | 韧性供应链 (5.0) |
|------|-----------------|-----------------|
| 目标 | 最低成本 | 适度成本+抗冲击 |
| 库存 | 最小化 (JIT) | 战略缓冲 |
| 供应商 | 单一最优 | 多元 (至少 2-3) |
| 制造地点 | 最低成本国 | 多区域布局 |
| 信息 | 按需 | 全链可视 |
| 恢复 | 不是优先项 | 预设的恢复方案 |

## 韧性策略

### 供应商多元化

| 策略 | 说明 |
|------|------|
| **双源/多源** | 关键物料至少 2 个合格供应商 |
| **地理分散** | 供应商分布在不同的国家和地区 |
| **供应商深度** | 了解你的供应商的供应商——Tier 2/3 风险 |

### 库存策略

| 策略 | 说明 |
|------|------|
| **战略缓冲** | 关键物料保持安全库存——不追求零库存 |
| **共享库存** | 同行业企业共享安全库存 |
| **近线存储** | 库存靠近产线——快速调用 |

### 制造布局

| 策略 | 说明 |
|------|------|
| **近岸制造 (Nearshoring)** | 生产靠近消费市场——缩短供应链 |
| **友岸制造 (Friendshoring)** | 在政治友好的国家布局 |
| **多工厂产能** | 同一产品可在多个工厂生产 |
| **微型工厂** | 靠近客户的分布式小工厂 |

### 数字供应链

| 技术 | 作用 |
|------|------|
| **供应链控制塔** | 实时全链可视化——库存/在途/需求 |
| **AI 风险预警** | 天气/政治/金融事件自动预警供应链风险 |
| **数字孪生供应链** | 模拟"如果 X 工厂停产，影响多大？" |
| **区块链** | 供应链溯源——从矿山到成品的透明追溯 |

## 韧性指标

| 指标 | 含义 |
|------|------|
| **恢复时间 (RTO)** | 中断后恢复到正常运营需要多长时间？ |
| **存活时间** | 在供应中断的情况下，现有库存能撑多久？ |
| **冲击吸收能力** | 多大程度的冲击（如供应商停产 50%）仍能维持？ |
| **多样性指数** | 供应商和地理分布的集中度 |

## 实际案例

| 事件 | 教训 |
|------|------|
| 2011 日本地震+海啸 | 瑞萨电子停产→全球汽车减产——单一来源的代价 |
| 2020 COVID-19 | PPE（口罩/防护服）过度依赖单一国家生产 |
| 2021 苏伊士运河 | 一艘船堵塞→全球贸易损失 $9.6B/天 |
| 2022 俄乌战争 | 氖气（芯片用）供应中断、能源价格飙升 |
