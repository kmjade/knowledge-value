# 08-弹性制造系统

> 弹性制造是产线层面的韧性——快速切换产品、自适应异常、吸收波动。如果说供应链韧性是"外部"的，弹性制造是"内部"的。

## 弹性制造 vs 传统产线

| 维度 | 传统产线 | 弹性产线 |
|------|----------|----------|
| 产品切换 | 数天-数周 | 数分钟-数小时 |
| 批量 | 大批量单一品种 | 多品种小批量 |
| 布局 | 固定排列 | 可重构模块 |
| 控制 | 集中式 PLC | 分布式+自主决策 |
| 异常响应 | 人工处理 | 自动检测+自动切换 |

## 弹性制造技术

### 快速换型 (SMED)

| 步骤 | 说明 |
|------|------|
| 分离内外作业 | 区分必须停机做和可以不停机做的 |
| 内转外 | 把停机作业转为外作业 |
| 简化 | 标准化夹具、快换接头、免工具 |

> SMED 的经典案例：丰田将冲压模具换型时间从 4 小时→3 分钟。

### 可重构制造系统 (RMS)

| 特征 | 说明 |
|------|------|
| **模块化** | 标准化的加工单元——可快速重组 |
| **可集成** | 新模块"即插即用" |
| **可转换** | 同一系统可生产不同产品族 |
| **可扩缩** | 按需求增加或减少产能模块 |

### 自主移动机器人 (AMR)

不同于 AGV（固定路线），AMR 自主导航——适合弹性产线：

| AMR | 应用 |
|-----|------|
| 物料搬运 | 动态规划路径→绕过障碍 |
| 产线重组 | 移动工作站临时组成新产线 |

### AI 驱动的调度

| 能力 | 说明 |
|------|------|
| 动态排产 | 订单变化→实时重排 |
| 异常处理 | 某工位故障→自动绕过或替代 |
| 多目标优化 | 交期+能耗+成本同时优化 |

## 分布式制造

| 模式 | 说明 |
|------|------|
| 云制造 | 像云一样调用分布式制造资源——谁的产线空闲就给谁 |
| 3D 打印分布式网络 | 需要零件→最近的有 3D 打印能力的工厂打印→配送 |
| 微型工厂 | 5000m² 以下的全自动柔性工厂→靠近客户 |

## 韧性设计模式

| 模式 | 说明 |
|------|------|
| 松耦合 | 各工位之间通过缓冲解耦→一个工位故障不影响全局 |
| 降级运行 | 部分设备故障→降速但不停产 |
| 热备 | 关键设备有备份→秒级切换 |
| 跨线互助 | 两条线可以互相"借"产能 |
