# 02-安全技术

> 安全是 Cobot 与传统机器人的分水岭——Cobot 可以没有围栏，因为它"知道"有人在旁边，并限制自己的力量和速度。

## 安全标准

| 标准 | 内容 |
|------|------|
| **ISO 10218-1** | 机器人本体安全要求 |
| **ISO 10218-2** | 机器人系统集成安全 |
| **ISO/TS 15066** | **协作机器人专项——定义了四种协作模式** |
| **ISO 13849** | 安全相关控制系统——PL (性能等级) |

## 四种协作模式

### 1. 安全级监控停止

| 特性 | 说明 |
|------|------|
| 原理 | 人进入协作区→机器人停止 |
| 恢复 | 人离开→自动恢复或手动恢复 |
| 典型 | 人偶尔进入——取放工件 |

### 2. 手动引导

| 特性 | 说明 |
|------|------|
| 原理 | 人手握住机器人末端→跟随人手运动 |
| 力反馈 | 机器人感知人的意图→平滑地辅助 |
| 典型 | 拖动示教、辅助搬运重物 |

### 3. 速度和分离监控

| 特性 | 说明 |
|------|------|
| 原理 | 距离越近速度越慢→接触前停止 |
| 传感器 | 激光扫描仪、视觉系统监控人的位置 |
| 公式 | 最小距离 = f(机器人速度, 人的速度, 停止距离) |

### 4. 功率和力限制 (PFL)

**最常用的模式**——机器人本身"无力"到足以安全接触：

| 身体部位 | 准静态接触力上限 | 瞬态接触力上限 |
|----------|----------------|---------------|
| 头部 | 65N | — |
| 脸部 | 65N | — |
| 手/手指 | 140N | 280N |
| 手臂 | 150N | 300N |

> 这些值来自医学研究——超过这个力可能造成损伤。Cobot 通过关节力矩传感器确保碰撞力低于这些值。

## 实现安全的技术

| 技术 | 说明 |
|------|------|
| **关节力矩传感器** | 每个关节实时测量力矩→检测碰撞 (<10ms) |
| **电流监测** | 无传感器方案——电机电流异常→碰撞 |
| **双通道安全** | 两条独立的检测通道→任一触发就停止 |
| **功率限制** | 机械设计限制最大输出力 |
| **圆角设计** | 无尖锐边角→减少挤压伤害 |
| **轻量设计** | 自重越轻→碰撞能量越低 |

## 风险评估

不是 Cobot 绝对安全——要根据具体应用做评估：

| 步骤 | 内容 |
|------|------|
| 1. 确定限制 | 空间/速度/力/负载 |
| 2. 识别危险 | 夹点/碰撞/工件脱落/末端执行器 |
| 3. 风险估计 | 严重度 × 发生概率 |
| 4. 风险降低 | 降低速度/加安全传感器/防护罩 |
| 5. 验证 | 实际测试确认安全 |

> Cobot 末端装一把刀→它就不再是"安全的"Cobot。风险评估要评估整个应用——不只看机器人本体。
