# Cobot 常见问题

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| Cobot 替代传统机器人 | 互补——Cobot 擅长柔性小批量，传统机器人擅长高速大批量 |
| Cobot 绝对安全 | 安全取决于风险评估——末端装刀具的 Cobot 仍然危险 |
| Cobot 不需要编程 | 拖动示教只是最简单的方式——复杂任务仍需编写逻辑 |
| Cobot 速度慢所以效率低 | 节拍取决于应用——上下料等任务 Cobot 速度足够，瓶颈在人不在机 |
| 买了 Cobot 就能直接部署 | 需要设计夹具、安全评估、布局调整、操作员培训 |
| Cobot = UR | UR 是市场先驱但不是唯一——FANUC/ABB/KUKA 各有优势 |
