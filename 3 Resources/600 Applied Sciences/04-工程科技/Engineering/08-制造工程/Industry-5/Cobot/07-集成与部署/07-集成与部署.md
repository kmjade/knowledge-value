# 07-集成与部署

> 买 Cobot 只是第一步——如何把它集成到产线中、设计安全方案、计算节拍、培训操作员，决定了项目的成败。

## 部署流程

```
需求分析 → 方案设计 → 风险评估 → 安装 → 编程 → 
试运行 → 安全验收 → 培训 → 正式投产
```

## 需求分析

| 问题 | 说明 |
|------|------|
| 要做什么？ | 上下料/装配/质检？ |
| 工件是什么？ | 重量/形状/材料→决定夹爪 |
| 节拍要求？ | 每小时多少件？ |
| 空间多大？ | Cobot 安装位置和可达范围 |
| 现有设备？ | CNC/传送带/视觉→接口方式 |
| 谁操作？ | 操作员的技能水平 |

## 空间布局

| 配置 | 说明 |
|------|------|
| **固定台面** | Cobot 固定在台面上——最常用 |
| **移动推车** | Cobot 在移动底座上→多工位共享 |
| **倒装** | Cobot 倒挂安装→节省台面空间 |
| **壁挂** | 侧面安装→特殊空间 |

## 风险评估实操

| 步骤 | 工具/方法 |
|------|----------|
| 识别危险 | 观察操作过程——哪些地方可能夹手/碰撞 |
| 风险矩阵 | 严重度 (1-3) × 概率 (1-3) |
| 降低措施 | 降速/安全传感器/光幕/防护罩 |
| 验证 | 用测试载荷实际碰撞→测量接触力 |
| 文档 | 保存风险评估报告——法规要求 |

## 节拍计算

```
单件周期 = 取料时间 + 移动时间 + 放料时间 + 信号等待
```

| 优化策略 | 效果 |
|----------|------|
| 双工位夹爪 | 一次取放→节拍减半 |
| 过渡半径 | 航点间平滑过渡→减少加减速 |
| 并行操作 | 移动同时夹爪开合 |
| 减少等待 | 提前发信号→不等信号 |

## 培训

| 阶段 | 内容 | 时长 |
|------|------|------|
| 基础操作 | 开/关/模式切换/紧急停止 | 2h |
| 编程 | 拖动示教+图形化编程 | 1天 |
| 维护 | 清洁/润滑/备份 | 2h |
| 排障 | 常见错误代码和复位 | 2h |

## 常见错误

| 错误 | 后果 | 避免 |
|------|------|------|
| 未做风险评估 | 安全事故 | 强制做 |
| 夹持力不够 | 工件脱落 | 安全系数够 |
| 电缆干扰 | 运动不流畅 | 线缆管理 |
| 忽略环境 | 振动/温度/粉尘影响 | 提前评估 |
