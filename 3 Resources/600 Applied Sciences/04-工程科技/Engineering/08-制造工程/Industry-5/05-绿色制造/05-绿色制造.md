# 05-绿色制造

> 绿色制造是工业 5.0 的环境支柱——在产品的全生命周期最小化环境影响。不是"边污染边治理"，而是从设计开始就消除污染和浪费。

## 碳核算

### 范围 1 / 2 / 3

| 范围 | 含义 | 制造例子 |
|------|------|----------|
| **Scope 1** | 直接排放——自己的烟囱和车辆 | 锅炉燃气、自有卡车 |
| **Scope 2** | 间接排放——买的电和热 | 外购电力（火电） |
| **Scope 3** | 供应链上下游 | 原材料碳足迹、产品使用中的排放 |

> Scope 3 通常占总碳排放的 70-90%——也是最难计算和减排的部分。

## 绿色制造策略

### 能源

| 策略 | 实施 |
|------|------|
| 可再生能源 | 屋顶光伏+PPA 绿电采购 |
| 能效提升 | 变频器、余热回收、空压机优化 |
| 储能 | 电池储能削峰填谷 |

### 材料

| 策略 | 实施 |
|------|------|
| 轻量化 | 零件减重→减少材料和运输能耗 |
| 材料替代 | 生物基塑料替代石化塑料 |
| 减少浪费 | 近净成形 (Near Net Shape) 减少加工去除量 |

### 水

| 策略 | 实施 |
|------|------|
| 闭路循环 | 冷却水循环使用 |
| 中水回用 | 处理后的废水用于非关键清洗 |
| 零排放 (ZLD) | 所有废水内部处理→不排放 |

### 排放

| 策略 | 实施 |
|------|------|
| 碳捕获 (CCS) | 从烟囱中捕获 CO₂ |
| 燃料替代 | 煤→天然气→氢 |
| 电气化 | 电弧炉替代高炉 (钢铁) |

## 净零工厂路径

```
1. 能源审计 → 2. 节能改造 → 3. 屋顶光伏 → 4. 绿电采购 → 
5. 余热回收 → 6. 碳补偿 (最后手段) → 净零
```

## 绿色供应链

| 策略 | 说明 |
|------|------|
| 供应商碳评估 | 要求供应商披露碳足迹 |
| 低碳物流 | 铁路/电动卡车替代柴油 |
| 近岸采购 | 减少运输距离 |
| 包装减量 | 可回收/可复用包装 |

## 产品碳足迹 (PCF)

```
原材料开采 → 运输 → 制造 → 分销 → 使用 → 回收
   ↓碳        ↓碳    ↓碳    ↓碳    ↓碳    ↓碳
   └───────────────── 总和 = PCF ──────────────────┘
```
