# 09-社会可持续性

> 工业 5.0 的可持续不只是环境的——还包括社会的。绿色转型不能以牺牲工人和社区为代价。公平转型 (Just Transition) 是核心原则。

## 可持续的三重底线

```
      环境可持续
          /\
         /  \
        /    \
       / 5.0  \
      /        \
     /__________\
经济可持续      社会可持续
```

## 公平转型

向绿色和数字化经济转型的过程中，如何保护工人和社区？

| 原则 | 实施 |
|------|------|
| **技能再培训** | 化石燃料工人→可再生能源技术员 |
| **社会保障** | 转型期的收入支持和再就业服务 |
| **社区投资** | 依赖煤炭/钢铁的城镇→新兴产业 |
| **社会对话** | 政府+企业+工会三方协商 |

### 德国的煤炭退出

2038 年前逐步淘汰煤电——配套：
- 400 亿欧元的结构转型基金
- 受影响工人提前退休或再培训
- 煤矿区投资新产业和研究机构

## 体面工作

| 维度 | 工业 5.0 目标 |
|------|-------------|
| 安全 | 零死亡、零重伤——Cobot+外骨骼减少工伤 |
| 健康 | 减少重复劳损、减少化学暴露 |
| 工作时间 | 柔性排班——工作生活平衡 |
| 收入 | 技能升级→更高的薪酬 |
| 话语权 | 工人参与技术和流程决策 |

## 供应链的人权

| 议题 | 应对 |
|------|------|
| 强迫劳动 | 供应链审计+区块链溯源 |
| 冲突矿产 (3TG) | 负责任矿产倡议 (RMI) |
| 童工 | 供应商行为准则+第三方审核 |
| 生活工资 | 不只是法定最低工资——而是能体面生活的工资 |

### 德国供应链法 (2023)

德国《供应链尽职调查法》要求大企业：
- 识别供应链中的人权和环境风险
- 采取措施预防和补救
- 每年公开报告

> 欧盟正在制定的 CSDDD（企业可持续发展尽职调查指令）范围更广、更严格。

## 包容性制造

| 维度 | 说明 |
|------|------|
| 性别 | 制造业女性占比仅 10-30%——如何吸引和保留？ |
| 残障 | 自适应工作站 + Cobot 辅助 |
| 年龄 | 老龄化社会——如何让 50-65 岁工人继续贡献？ |
| 地理 | 不把工厂搬走让社区空心化 |

## 社会影响评估

| 指标 | 衡量 |
|------|------|
| 本地就业率 | 工厂雇佣了多少本地工人？ |
| 培训投入 | 每位员工每年的培训时数和费用 |
| 工伤率 | 可记录工伤率 (TRIR) |
| 供应商合规率 | 通过人权审核的供应商比例 |
| 社区投资 | 对当地教育和基础设施的投入 |
