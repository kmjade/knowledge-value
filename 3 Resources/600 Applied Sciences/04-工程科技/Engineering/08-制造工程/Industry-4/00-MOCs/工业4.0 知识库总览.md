# 工业4.0 知识库总览

> 工业 4.0 是 2011 年德国提出的第四次工业革命——通过 CPS（信息物理系统）、IoT、AI 和数字孪生实现智能制造。它是工业 5.0 的数字底座。

## 工业革命代际

| 代 | 驱动 | 标志 |
|----|------|------|
| 1.0 (1780s) | 蒸汽 | 机械化 |
| 2.0 (1870s) | 电力 | 流水线 |
| 3.0 (1970s) | 电子+计算机 | 自动化 |
| **4.0 (2010s)** | **IoT+AI+数字孪生** | **智能制造** |
| 5.0 (2020s) | 人+地球+韧性 | 可持续制造 |
