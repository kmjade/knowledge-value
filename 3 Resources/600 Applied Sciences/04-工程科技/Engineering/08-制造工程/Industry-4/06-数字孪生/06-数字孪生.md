# 06-数字孪生

> 数字孪生是工业 4.0 的核心应用——在产品设计、制造和运行阶段都有对应的数字镜像。

## 三阶段孪生

| 阶段 | 孪生类型 | 用途 |
|------|----------|------|
| **设计** | 产品孪生 (CAE) | 虚拟测试、优化 |
| **制造** | 产线孪生 | 虚拟调试、瓶颈分析 |
| **运行** | 运行孪生 | 实时监控、预测维护 |

## 虚拟调试

在新产线硬件到位前，用数字孪生调试 PLC 程序：

```
虚拟 PLC → 虚拟 IO → 虚拟产线 (3D + 物理引擎) → 虚拟传感器反馈
```

成果：现场调试时间缩短 50-80%。

## 工业案例

| 案例 | 成果 |
|------|------|
| GE 航空发动机孪生 | 每台引擎云端孪生→优化维护计划 |
| 西门子安贝格 | 全厂孪生→质量率 99.9988% |
| 特斯拉 | 每辆车有数字孪生→OTA 诊断 |
