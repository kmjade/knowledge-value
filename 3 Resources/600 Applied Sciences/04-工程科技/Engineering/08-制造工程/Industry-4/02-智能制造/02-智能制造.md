# 02-智能制造

> 智能制造是工业 4.0 的核心场景——通过 IoT 数据 + AI 分析 + 自动化执行，实现自我感知、自我决策、自我执行的工厂。

## 智能工厂 vs 传统工厂

| 维度 | 传统 | 智能 |
|------|------|------|
| 排产 | 人工排程 | AI 动态优化 |
| 质量 | 抽检+事后 | 100%在线预测 |
| 维护 | 定期/事后 | 预测性维护 |
| 物流 | 固定路线 AGV | AMR 自主导航 |
| 信息流 | 纸+Excel | 数字主线 |
| 异常响应 | 人工发现+处理 | 自动检测+自动处理 |

## 关键技术

| 技术 | 功能 |
|------|------|
| **MES/MOM** | 生产执行——排产、追踪、质量 |
| **APS** | 高级排产——AI 优化调度 |
| **数字孪生** | 产线数字镜像——虚拟调试+实时仿真 |
| **AOI/机器视觉** | 自动光学检测——在线质检 |
| **AGV/AMR** | 自主搬运——柔性物流 |
