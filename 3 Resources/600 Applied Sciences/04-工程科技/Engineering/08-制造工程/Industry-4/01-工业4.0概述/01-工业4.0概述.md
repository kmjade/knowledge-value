# 01-工业4.0概述

> 工业 4.0 (Industrie 4.0) 是德国 2011 年汉诺威工博会上首次提出的概念——通过信息物理系统 (CPS)、物联网和人工智能实现制造业的智能化转型。

## 核心定义

| 特征 | 说明 |
|------|------|
| **互联** | 机器、产品、人——实时互联 |
| **信息透明** | 传感器数据→数字孪生→全局可视 |
| **技术辅助** | AI 辅助人做复杂决策 |
| **分散决策** | CPS 自主决策——不依赖中央控制 |

## 各国路径对比

| 国家/地区 | 名称 | 侧重 | 代表 |
|-----------|------|------|------|
| **德国** | Industrie 4.0 | 制造装备+自动化 | Siemens, Bosch |
| **美国** | Industrial Internet | IT+数据 | GE, PTC |
| **中国** | 中国制造 2025/智能制造 | 制造业全面升级 | 政策驱动 |
| **日本** | Society 5.0 / 互联工业 | 社会整体数字化 | 政策+企业 |

## RAMI 4.0 三维架构

```
         层级 (Hierarchy)
         ↑
   ┌─────┼─────┐  企业/工厂/产线/设备/产品
   │     │     │
   │    /│     │
   │   / │     │
   │  /  │     │
   └─┼───┼─────┼──→ 生命周期 (设计→制造→服务)
     │   │    /
     │   │   /
     └───┼──┘
         ↓
    IT 层级 (业务→功能→信息→通信→集成→资产)
```

> RAMI 4.0 是工业 4.0 的"地图"——任何一个工业 4.0 用例都可以在这个三维空间中找到位置。

## 工业 4.0 vs 5.0

| | 4.0 | 5.0 |
|---|-----|------|
| 核心驱动 | 技术 (IoT, AI, 数字孪生) | 价值 (人, 地球, 韧性) |
| 人与机器 | 机器增强人 | 人机深度协作 |
| 环境 | 效率附带减碳 | 可持续性为核心 |
