# 05-大数据与AI

> 工业 4.0 时代，数据是新的石油——但原油需要提炼。大数据和 AI 是将原始的机器数据转化为可执行的洞察。

## 工业数据的特征

| 特征 | 说明 |
|------|------|
| 时序为主 | 传感器每秒/每毫秒产生带时间戳的数据 |
| 体量大 | 一个工厂每天 TB 级 |
| 多源异构 | PLC、振动、视觉、MES、ERP |
| 价值密度低 | 99% 的正常数据——异常才是金矿 |

## AI 在制造中的应用

| 应用 | 方法 | 输入 → 输出 |
|------|------|------------|
| **异常检测** | Isolation Forest, Autoencoder | 振动/温度 → 异常分数 |
| **预测性维护** | LSTM, XGBoost | 历史退化数据 → 剩余寿命(RUL) |
| **质量预测** | 随机森林, 神经网络 | 过程参数 → 质量OK/NG |
| **视觉质检** | CNN | 图像 → 缺陷分类 |
| **排产优化** | 遗传算法, 强化学习 | 订单+约束 → 最优排程 |

## 从数据到行动的闭环

```
传感器 → 流处理 → 异常检测 → 告警 → 维修 → 反馈
  ↑__________________________________________↓
              持续学习 (数据回流→重训练)
```
