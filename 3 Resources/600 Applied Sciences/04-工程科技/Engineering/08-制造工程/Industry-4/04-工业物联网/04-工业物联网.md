# 04-工业物联网

> IIoT 是工业 4.0 的连接基础——让机器、传感器和系统互联互通。不同于消费 IoT，IIoT 要求 99.9%+ 可靠性。

## IIoT 架构

```
现场层 (PLC/传感器/执行器)
   ↓ Modbus/Profinet/OPC UA
边缘层 (网关/边缘服务器)
   ↓ MQTT/OPC UA
平台层 (IIoT 平台/MES)
   ↓ API
企业层 (ERP)
```

## 关键技术

| 技术 | 作用 |
|------|------|
| **OPC UA** | 工业互操作标准——传数据也传语义 |
| **MQTT** | 轻量消息协议——设备→云端 |
| **TSN** (时间敏感网络) | 确定性的工业以太网——μs 级同步 |
| **边缘计算** | 本地实时处理——不依赖云 |

## 核心应用

| 应用 | 说明 |
|------|------|
| 设备联网 | 老旧设备加装传感器→数字化 |
| OEE 监控 | 实时计算设备综合效率 |
| 能源管理 | 分项计量→优化能耗 |
| 预测维护 | 振动/温度→寿命预测 |
