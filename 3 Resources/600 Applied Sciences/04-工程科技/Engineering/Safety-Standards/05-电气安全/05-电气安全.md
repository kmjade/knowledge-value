# 05-电气安全

> IEC 60204-1 是机械电气设备安全的"宪法"——每台机器的电控柜都必须符合它。覆盖电击防护、过流保护、等电位联结和急停。

## IEC 60204-1 核心要求

### 电击防护

| 措施 | 说明 |
|------|------|
| **直接接触防护** | 带电部分不外露——IP2X 防护（手指进不去） |
| **间接接触防护** | 绝缘故障时自动断电——漏电保护/等电位联结 |
| **SELV/PELV** | 安全特低电压——≤50V AC / 120V DC |

### 过流保护

| 保护 | 保护什么 | 器件 |
|------|----------|------|
| 过载 | 电机/电缆长时间过流→发热 | 热继电器、电机保护断路器 |
| 短路 | 瞬间大电流→烧毁 | 断路器、熔断器 |

### 等电位联结

```
所有金属外壳、门板、安装板 → 连接到主接地端子
     ↓
  接地电阻 < 0.1Ω (任何两点之间)
```

> 目的：确保所有外露导电部分在同一电位——即使漏电，人也不会触电。

### 急停 (再次强调)

| 要求 | IEC 60204-1 细化 |
|------|-----------------|
| 停止类别 | Cat 0: 直接断电 / Cat 1: 可控停止后断电 |
| 颜色 | 红色蘑菇头+黄色背景——没有例外 |
| 复位 | 必须手动——自动复位违规 |
| 布线 | 硬线连接——不能经过 PLC/软件 |

## 电控柜设计

| 要求 | 说明 |
|------|------|
| IP 防护 | 至少 IP54（防尘防溅水），户外 IP65 |
| 散热 | 柜内温度 ≤ 器件额定温度 |
| 开门断电 | 高电压柜需联锁 |
| 线缆颜色 | PE=黄绿双色, 交流=棕/蓝, 直流=红/蓝(±) |
| 标识 | 每个端子、每根线都有标签 |

## 绝缘测试

| 测试 | 电压 | 标准 |
|------|------|------|
| 绝缘电阻 | 500V DC | >1MΩ |
| 耐压测试 | AC 2×额定+1000V, 1min | 无击穿/闪络 |
| 接地连续性 | 大电流 (≥10A) | <0.1Ω |
