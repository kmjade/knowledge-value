# 02-机械安全

> ISO 12100 是所有机械安全标准的基础——定义了风险评估和风险降低的通用方法。任何一个机器安全设计都从它开始。

## ISO 12100 三步法

```
Step 1: 本质安全设计 (消除危险)
    ↓ (仍有残余风险)
Step 2: 安全防护和补充措施 (防护罩、光幕、联锁)
    ↓ (仍有残余风险)
Step 3: 使用信息 (警示标识、手册、培训)
    ↓
残余风险可接受？→ 是 → 完成
```

### Step 1: 本质安全设计

| 策略 | 例子 |
|------|------|
| 消除夹点 | 增大间隙→手无法进入 |
| 降低能量 | 降低速度/力/电压到安全水平 |
| 自动化 | 机器人替代人手——人不进入危险区 |
| 人因工程 | 减少误操作可能 |

### Step 2: 安全防护

| 防护类型 | 说明 | 标准 |
|----------|------|------|
| **固定防护罩** | 只有用工具才能打开 | ISO 14120 |
| **活动防护罩+联锁** | 打开→机器停机 | ISO 14119 |
| **光幕** | 光束被遮挡→停机 | IEC 61496 |
| **双手控制** | 必须双手同时按→手在安全位置 | ISO 13851 |
| **安全垫** | 踩上→停机 | ISO 13856 |

### Step 3: 使用信息

| 形式 | 要求 |
|------|------|
| 警示标识 | 黄色背景+黑色符号——国际通用 |
| 信号灯 | 红=危险/停机, 黄=警告, 绿=安全 |
| 手册 | 语言是使用国的语言 |
| 培训 | 操作员必须培训并记录 |

## 安全距离 (ISO 13857)

防护装置离危险区需要多远？

| 危险区高度 | 防护高度 | 最小安全距离 |
|-----------|---------|-------------|
| 通过上方到达 | — | ≥ 防护高度 |
| 通过开口到达 | 开口 ≤ 40mm | 距离 ≥ 200mm |
| 通过开口到达 | 开口 40-120mm | 距离 ≥ 850mm |

```
手 → 开口 40mm → 伸进去 200mm → 够不着危险区？
                                    ↓
                              需要更大的安全距离
```

## 急停 (ISO 13850)

| 要求 | 说明 |
|------|------|
| 红色蘑菇头+黄色背景 | 必须一眼可见 |
| 按下自锁 | 必须手动旋转/拉出才能复位 |
| 每个操作位 | 操作员站的地方都要有 |
| 复位≠重启 | 急停复位后机器不能自动重启 |
