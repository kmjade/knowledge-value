# 安全标准 知识库总览

> 工业安全标准是保障工人生命安全的技术法规——从机械防护到功能安全，从风险评估到安全完整性等级。每个标准背后都有血的教训。

## 标准的三级结构

```
A 类 (基础安全标准)
  ISO 12100 — 风险评估和风险降低的基本原则
    ↓
B 类 (通用安全标准)
  B1: 安全距离 (ISO 13857)、表面温度、噪声...
  B2: 安全装置 (ISO 13850 急停)、联锁、光幕...
    ↓
C 类 (机器专用安全标准)
  ISO 10218 工业机器人、ISO 16090 加工中心...
```
