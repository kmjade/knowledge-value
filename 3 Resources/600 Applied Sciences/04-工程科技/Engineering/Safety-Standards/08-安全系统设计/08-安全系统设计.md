# 08-安全系统设计

> 从纸面标准到实际电路——如何设计一个符合 PL e / SIL 3 的安全回路。

## 安全回路的组成

```
传感器子系统 → 逻辑子系统 → 执行器子系统
 (急停/光幕/门锁)  (安全继电器/PLC)  (接触器/变频器STO)
```

三个子系统必须都达到所需的 PL/SIL——短板决定整体。

## 安全继电器

| 特性 | 说明 |
|------|------|
| 冗余触点 | 至少两个强制导向触点 |
| 自诊断 | 检测触点焊接/线圈故障 |
| 手动/自动复位 | 可选——高风险建议手动复位 |
| PL | 典型 PL e (配合双通道) |

```
急停 (双通道) → 安全继电器 → 接触器 (双通道)
    NC+NC        强制导向     串联断开
```

## 安全 PLC

| 特性 | 说明 |
|------|------|
| 冗余处理器 | 两个 CPU 交叉比较 |
| 安全 I/O | 双通道输入、安全输出 |
| 安全通信 | PROFIsafe, CIP Safety |
| 编程 | 安全逻辑和标准逻辑分开 |
| PL | 可达 PL e / SIL 3 |

## PL 计算 (SISTEMA)

用 SISTEMA (免费工具) 计算：

| 参数 | 值 | 来源 |
|------|-----|------|
| MTTFd (每个元件) | 查数据手册 | 制造商提供 |
| DC (诊断覆盖率) | 低/中/高 | 设计决定 |
| CCF (共因失效) | 得分表 | ISO 13849 附录 F |
| 架构 | Cat 2/3/4 | 电路设计 |

```
输入每个元件 → 选择架构 → SISTEMA 计算 → 得出 PL
```

## 设计原则

| 原则 | 说明 |
|------|------|
| **故障安全** | 任何故障 → 安全状态 (断电=停止) |
| **冗余** | 关键功能不能单点故障 |
| **多样性** | 冗余通道用不同类型器件——防共因失效 |
| **测试** | 定期测试安全功能——发现隐藏的故障 |
| **隔离** | 安全回路和标准回路物理分开 |

## 常见错误

| 错误 | 正确做法 |
|------|----------|
| 安全回路经过 PLC 标准 I/O | 安全回路必须硬线或安全 PLC |
| 单接触器断开动力 | 必须双接触器串联 |
| 复位按钮和启动按钮共用 | 必须分开——防止误复位 |
| 不计算 PL | 计算——凭感觉是违规 |
