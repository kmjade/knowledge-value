# 03-PLC 编程

> IEC 61131-3 是 PLC 编程的国际标准，定义了 5 种编程语言。梯形图 (LD) 最经典，结构化文本 (ST) 越来越流行。

## IEC 61131-3 五种语言

| 语言 | 类型 | 看起来像 | 适合 |
|------|------|----------|------|
| **LD** (梯形图) | 图形 | 电路图 | 逻辑控制、电工过渡 |
| **FBD** (功能块图) | 图形 | 逻辑门图 | 过程控制、信号处理 |
| **ST** (结构化文本) | 文本 | Pascal/C | 复杂算法、数据处理 |
| **SFC** (顺序功能图) | 图形 | 流程图 | 顺序控制、步骤流程 |
| **IL** (指令表) | 文本 | 汇编 | 已淘汰 (IEC 61131-3 第三版移除) |

## 梯形图 (LD) 基础

### 基本触点

```
    X0          Y0
  ──┤ ├────────(  )──    常开触点：X0=ON → Y0=ON
    X0          Y0
  ──┤/├────────(  )──    常闭触点：X0=OFF → Y0=ON
```

### 经典电路

**启保停电路** (Start/Stop)——PLC 入门的 "Hello World"：

```
   X0(启动)    X1(停止)     Y0
  ──┤ ├──┬────┤/├────────(  )──
         │
         │    Y0
         └───┤ ├──            自锁触点
```

> 这是用梯形图实现锁存——按一下启动，Y0 保持 ON，直到按停止。对应继电器世界的"自锁"。

### 常用指令

| 指令 | 符号 | 功能 |
|------|------|------|
| 输出线圈 | `( )` | 赋值输出 |
| 置位 | `(S)` | 设为 ON 并保持 |
| 复位 | `(R)` | 设为 OFF |
| 上升沿检测 | `(P)` | 信号从 0→1 时导通一个周期 |
| 下降沿检测 | `(N)` | 信号从 1→0 时导通一个周期 |
| 定时器 (TON) | `TON` | 延时接通 |
| 计数器 (CTU) | `CTU` | 加计数 |

## 结构化文本 (ST)

```pascal
// 启保停电路用 ST 写
IF StartButton THEN
    Motor := TRUE;
END_IF;

IF StopButton THEN
    Motor := FALSE;
END_IF;

// PID 控制
PID_Controller(
    SetPoint := 100.0,
    Input := Temperature,
    ManualMode := FALSE
);
Temperature := PID_Controller.Output;
```

> ST 是趋势——年轻工程师更喜欢写代码而非画梯形图。复杂算法（PID、数据处理、通信）用 ST 比梯形图高效得多。

## 程序组织单元 (POU)

| 类型 | 说明 | 特点 |
|------|------|------|
| **函数 (FUN)** | 纯计算，无内部状态 | 相同的输入→相同的输出 |
| **功能块 (FB)** | 有内部状态和记忆 | 如定时器——多次调用有不同状态 |
| **程序 (PROG)** | 顶层入口 | 主循环 |

## 常见编程模式

| 模式 | 用途 |
|------|------|
| 顺序控制 (步进) | 将复杂过程分解为步骤 |
| 报警处理 | 故障检测和连锁保护 |
| 模拟量处理 | 量程转换、滤波、标定 |
| 配方管理 | 多产品参数切换 |
| 通信处理 | 与第三方设备数据交换 |
