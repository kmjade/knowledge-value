# 自动化控制 常见问题

## 十大常见误区

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| PLC 就是高级继电器 | PLC 是现代工业计算机——有 CPU、内存、通信、PID、运动控制。继电器只是它最原始的应用 |
| 梯形图 = PLC 编程的全部 | IEC 61131-3 定义了 5 种语言：LD(梯形图)、FBD(功能块)、ST(结构化文本)、SFC(顺序功能图)、IL(指令表)。ST 越来越主流 |
| 一台 PLC 控制一切 | 复杂系统使用分布式 I/O + 多 PLC + DCS。单 PLC 有 IO 点数和扫描周期限制 |
| 学会了某个品牌就会了全部 | 编程理念相通，但软件和通信配置完全不一样。跨品牌迁移成本高 |
| 继电器输出和晶体管输出一样 | 继电器输出可以带交流和直流负载、电流大、但寿命有限（机械触点）。晶体管输出只带直流、寿命长、可高频 |
| PLC 不需要网络安全 | 现代 PLC 都联网（Profinet、Ethernet/IP）→ 是网络攻击目标。Stuxnet 就是攻击 PLC 的 |
| 扫描周期不重要 | 高速应用（包装、印刷）需要扫描周期 <1ms。普通应用 10-50ms 够用 |
| PID 参数随便调 | PID 整定是门技术——错误参数导致震荡或不响应。有自整定但不可完全依赖 |
| 伺服就是高级步进 | 伺服有反馈（编码器）、闭环控制、过载能力强。步进是开环、高频失步。应用场景不同 |
| 自动化 = 让人失业 | 自动化替代的是重复性、危险性劳动——释放人去从事更高价值的创造性工作 |
