# 01-自动化控制概述

> 自动化控制是在没有（或极少）人工干预的情况下，使设备或过程按预定目标运行的技术。从恒温器到化工厂，核心原理相同。
> 相关入口：[[Automation|自动化控制总览]]

## 开环 vs 闭环

| 类型 | 原理 | 例子 | 特点 |
|------|------|------|------|
| **开环** | 给定输入→执行，无反馈 | 定时喷灌 | 简单、便宜，抗干扰弱 |
| **闭环** | 测量输出→与目标比较→调整输入 | 空调恒温 | 抗干扰，可能有震荡 |

```
开环: 设定 → 控制器 → 执行器 → 过程 → 输出

闭环: 设定 → ⓔ→ 控制器 → 执行器 → 过程 → 输出
             ↑                              ↓
             └──── 传感器 ←─────────────────┘
```

## PID 控制

工业控制中使用最广泛的算法——90%+ 的控制回路是 PID。

| 项 | 作用 | 效果 |
|----|------|------|
| **P** (比例) | 误差越大，输出越大 | 减小稳态误差，但可能震荡 |
| **I** (积分) | 累积误差→消除稳态误差 | 消除残余误差，但可能超调 |
| **D** (微分) | 预测误差变化趋势 | 减小超调，但对噪声敏感 |

```
u(t) = Kp·e(t) + Ki·∫e(t)dt + Kd·de(t)/dt
        └─比例─┘   └──积分──┘   └──微分──┘
```

### PID 整定口诀

| 问题 | 调整 |
|------|------|
| 响应太慢 | 增大 P |
| 震荡 | 减小 P 或增大 D |
| 有余差 | 增大 I |
| 超调太大 | 减小 P 或增大 D |

## 自动化金字塔

```
层级          系统            周期        关注
─────────────────────────────────────────────
ERP           SAP/Oracle      天-月       订单、财务
MES           生产执行系统      小时-天     排产、质量
SCADA         WinCC/IFix      秒-分       监控、报警
PLC/DCS       S7-1500/PCS7    ms-秒      逻辑、PID
现场设备       传感器/电机       μs-ms      物理信号
```

## 控制系统的类型

| 类型 | 适用规模 | 特点 |
|------|----------|------|
| **PLC** | 单机-产线 | 逻辑控制为主，离散制造 |
| **DCS** | 全厂 | 过程控制为主，连续流程（石化、电力） |
| **PAC** | 中型 | PLC + PC 能力混合 |
| **RTU** | 远程分散 | 油田、管线监控 |
| **CNC** | 单机 | 精密运动控制 |
