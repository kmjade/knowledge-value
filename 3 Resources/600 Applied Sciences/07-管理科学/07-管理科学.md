# 07-管理科学 (UDC 65) 

> 管理科学是将科学方法、数学模型和行为理论应用于组织管理——从办公室运作到企业战略，从会计到市场营销。UDC 65 涵盖管理、组织、通信和出版。

## UDC 65 分支

| UDC | 分支 | 核心内容 |
|-----|------|----------|
| **651** | 办公管理 | 文书、档案、办公自动化 |
| **654** | 电信 | 通信系统管理（技术见 UDC 621.39） |
| **655** | 印刷与出版 | 图书、期刊、数字出版 |
| **656** | 运输与邮政 | 物流管理、邮政服务 |
| **657** | 会计学 | 财务会计、成本会计、审计 |
| **658** | 企业管理 | 战略、组织、生产、营销、HR |
| **659** | 广告与公关 | 品牌传播、公共关系 |

## 企业管理 (UDC 658)

### 管理职能

| 职能 | 核心问题 |
|------|----------|
| **战略** | 我们要去哪里？为什么？ |
| **组织** | 谁做什么？如何协调？ |
| **领导** | 如何激励和引导人？ |
| **控制** | 我们在正确的轨道上吗？ |

### 组织结构

| 类型 | 特点 | 适合 |
|------|------|------|
| **职能型** | 按专业分组 (市场/研发/生产) | 单一产品、稳定环境 |
| **事业部** | 按产品/地区分组 | 多产品线、大型企业 |
| **矩阵型** | 职能+项目双线汇报 | 项目型业务 |
| **扁平/网络** | 少层级、自组织 | 科技创业公司 |

### 生产与运营

| 概念 | 说明 |
|------|------|
| **精益生产** | 消除浪费——Toyota 生产系统 |
| **六西格玛** | 减少质量波动——DPMO < 3.4 |
| **约束理论 (TOC)** | 瓶颈决定整体产出 |
| **敏捷/Scrum** | 迭代交付、快速反馈 |

## 会计学 (UDC 657)

| 分支 | 服务对象 | 核心产出 |
|------|----------|----------|
| **财务会计** | 外部（税务、投资者） | 资产负债表、损益表 |
| **管理会计** | 内部（管理层） | 成本分析、预算 |
| **审计** | 外部/内部 | 合规性、真实性验证 |

## 营销 (UDC 658.8 + 659)

| 概念 | 说明 |
|------|------|
| **4P** | Product (产品) / Price (价格) / Place (渠道) / Promotion (促销) |
| **STP** | Segmentation (细分) → Targeting (目标) → Positioning (定位) |
| **品牌** | 不只是 logo——是顾客心中的全部联想 |
| **数字营销** | SEO/SEM、社交媒体、内容营销、数据分析 |

## 人力资源管理

| 职能 | 说明 |
|------|------|
| 招聘与选拔 | 找到对的人 |
| 培训与发展 | 让人变得更好 |
| 绩效管理 | 评估和反馈 |
| 薪酬福利 | 激励和保留 |
| 员工关系 | 文化、沟通、合规 |
