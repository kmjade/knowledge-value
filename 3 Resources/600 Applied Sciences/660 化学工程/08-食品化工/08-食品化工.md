# 08 — 食品化工

> 食品化工將化學工程原理應用於食品加工——從保存到風味。

## 食品加工技術

| 技術 | 目的 |
|------|------|
| 熱加工 | 殺菌、熟化（巴氏殺菌、UHT） |
| 冷凍 | 長期保存 |
| 乾燥 | 脫水保存（噴霧乾燥、冷凍乾燥） |
| 發酵 | 風味、保存（酸奶、醬油） |
| 擠壓 | 膨化食品 |
| 包裝 | 氣調包裝、真空包裝 |

## 食品添加劑

| 類別 | 功能 |
|------|------|
| 防腐劑 | 延長保存期 |
| 抗氧化劑 | 防止氧化變質 |
| 乳化劑 | 油水混合 |
| 增稠劑 | 改善口感 |
| 甜味劑 | 提供甜味 |

> 💡 現代食品工業是化學工程的典範應用——從農場到餐桌的每一個環節都離不開化工技術。
