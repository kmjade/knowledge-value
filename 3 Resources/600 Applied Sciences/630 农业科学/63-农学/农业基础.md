# 农业基础

> **UDC 631**: Agriculture in general — 土壤、肥料、农机、灌溉、耕作制度

农业基础是农学的**底层基础设施**——不直接生产作物，但决定了所有农业生产的上限。它回答了"在什么条件下种、用什么工具种、如何维持地力"。

## 核心领域

| 分支 | 内容 |
|------|------|
| **土壤科学** | 土壤物理（质地、结构、水分）、土壤化学（pH、CEC、养分）、土壤生物（微生物组、蚯蚓） |
| **肥料与施肥** | 化肥（N-P-K）、有机肥、绿肥、精准施肥 |
| **农业机械** | 耕整地、播种、植保、收获、运输机械 |
| **灌溉与排水** | 地表灌溉、喷灌、滴灌、暗管排水 |
| **耕作制度** | 轮作、间作、免耕、保护性耕作 |
| **农业气象** | 积温、霜冻预警、气候区划 |

## 核心概念

### 土壤肥力三角

```
         物理性质
         (结构、质地)
           /\
          /  \
         /    \
        /  肥  \
       /   力   \
      /__________\
  化学性质        生物性质
 (养分、pH)      (微生物、有机质)
```

三者的交互决定了土壤的**真实生产力**——单一维度的改良往往失效。例如：大量施化肥（化学↑）但土壤板结（物理↓），最终产量仍下降。

### 肥料三要素

| 元素 | 符号 | 作用 | 缺乏症状 |
|------|------|------|----------|
| **氮** | N | 叶片生长、蛋白质合成 | 老叶黄化 |
| **磷** | P | 根系发育、能量代谢（ATP） | 叶片紫红、矮小 |
| **钾** | K | 抗逆性、品质、水分调节 | 叶缘焦枯 |

**施肥的 4R 原则**: Right source（正确肥源）、Right rate（正确用量）、Right time（正确时间）、Right place（正确位置）。

### 灌溉效率

| 方式 | 水利用效率 | 适用场景 |
|------|-----------|----------|
| 漫灌 | 30-50% | 传统、低成本、浪费大 |
| 喷灌 | 60-80% | 大田、均匀性好 |
| 滴灌 | 85-95% | 设施农业、缺水区 |
| 地下滴灌 | 90-95%+ | 高价值作物、技术门槛高 |

**全球现状**: 农业消耗全球淡水取用量的 ~70%，但其中超过一半因低效灌溉而浪费。

### 保护性耕作三原则

1. **免耕/少耕**: 不翻动土壤，减少侵蚀和有机质流失
2. **覆盖**: 秸秆还田或覆盖作物，保护地表
3. **轮作**: 不同作物轮替，打断病虫循环、均衡养分消耗

**效果**: 美国中西部大规模采用免耕后，土壤侵蚀减少 60-90%，但需要配套除草剂管理。

## 当代挑战

- **土壤退化**: 全球 33% 的土壤已中度到高度退化——侵蚀、盐碱化、有机质下降
- **水资源竞争**: 城市、工业、生态用水与农业争水——地下水超采严重
- **磷峰值**: 磷矿是不可再生资源——摩洛哥控制全球 70% 储量，地缘政治风险
- **微塑料**: 地膜残留和污泥农用导致农田微塑料积累——对土壤生物的影响未知
- **农机智能化**: GPS 导航、自动驾驶——效率提升但也加剧了小农户的竞争劣势

## 在三层模型中的位置

农业基础体现了应用科学中**科学与工程的交织**：

- **科学层**: 土壤学、水文学、气象学——观察和实验揭示机理
- **知识层**: 累积了大量程序性知识（农民"摸土知墒"的默会知识）和命题知识（土壤测试数据）
- **学科层**: 农业院校的土壤系/农机系——建制化程度高
