# 智慧农业

> 将信息技术、自动化与数据科学系统性地应用于农业生产与管理的全链条。

## 定义与范围

智慧农业（Smart Agriculture / Precision Agriculture）是通过传感器、物联网、人工智能、机器人和大数据分析等技术，实现农业生产过程的**精准感知、智能决策、自动执行**。它不是农业的一个独立分支，而是对传统农业各分支的**技术重构**。

| 层次 | 描述 | 关键词 |
|------|------|--------|
| **精准农业** (Precision Ag) | 基于空间变异性的变量投入 | GPS、VRT、产量图 |
| **数字农业** (Digital Ag) | 全链条数字化建模与决策 | 数字孪生、大数据、DSS |
| **智慧农业** (Smart Ag) | AI 驱动的自主决策与执行 | 机器人、边缘计算、自动驾驶 |

三者是**递进关系**：精准→数字→智慧，后者包含前者。

## 核心技术栈

### 感知层：数据获取

| 技术 | 应用 | 精度/特点 |
|------|------|-----------|
| **卫星遥感** | 大范围作物长势、土壤墒情 | 10-30m（Sentinel）、<1m（商业） |
| **无人机** | 高分辨率田间巡查、变量喷洒 | cm 级，灵活性高 |
| **IoT 传感器** | 土壤温湿度、气象、虫情实时监测 | 连续时间序列 |
| **机器视觉** | 杂草识别、果实成熟度、体况评分 | RGB/多光谱/高光谱 |
| **基因组测序** | 标记辅助育种、品种鉴定 | 成本已降至 $100/样本以下 |

### 决策层：数据→知识

| 技术 | 应用 | 特点 |
|------|------|------|
| **作物模型** | APSIM、DSSAT——模拟作物生长与环境互作 | 机理驱动，需参数校准 |
| **机器学习** | 产量预测、病虫害预警、推荐系统 | 数据驱动，黑箱可解释性差 |
| **数字孪生** | 农田/牧场全要素虚拟映射 | 实时同步、情景模拟 |
| **知识图谱** | 农技知识的结构化表示与推理 | 将分散的专家经验系统化 |

### 执行层：决策→行动

| 技术 | 应用 | 成熟度 |
|------|------|--------|
| **自动驾驶农机** | 耕作、播种、收割的无人化 | 商用化（John Deere、CNH） |
| **变量施药/施肥** | 按需精准投放，减少化学物质用量 30-50% | 广泛推广 |
| **智能灌溉** | 基于土壤墒情和蒸散模型的精准灌溉 | 节水 20-40% |
| **挤奶机器人** | 奶牛自动识别、自动挤奶 | 成熟商用（Lely、DeLaval） |
| **采摘机器人** | 果蔬选择性采摘 | 研发中，柔软果实是难点 |
| **放牧无人机** | 替代牧羊犬的畜群管理 | 新西兰、澳大利亚已商用 |

## 核心概念

### VRT：变量作业技术

智慧农业的标志性技术——不再对整块田地"一视同仁"，而是根据**空间变异性**（soil variability）进行差异化操作：

```
传统：整块地均匀施肥 200kg/ha
    ↓
VRT：A 区 150kg, B 区 250kg, C 区 100kg
    ↓
结果：总用量减少 15-25%，产量不变或略增
```

**变量依据**：土壤采样图、历年产量图、遥感植被指数（NDVI）、电导率（ECa）

### 农业数据链

```
传感器 ──→ 数据 ──→ 模型 ──→ 决策 ──→ 执行 ──→ 反馈
  ↑                                              │
  └──────────────────────────────────────────────┘
```

数据流是智慧农业的血液循环系统。中断任何一环，系统退化为传统农业。

### 数字孪生农田

将物理农田的**全要素**（土壤、作物、气象、设施）映射到数字空间：

- 实时同步传感器数据更新虚拟农田状态
- 模拟不同管理方案（施肥量、灌溉时间、收获日期）的结果
- 以最低成本探索最优方案——"在数字空间犯错，在物理空间正确"

## 在三层模型中的位置

智慧农业完美体现了**应用科学的跨层整合**特征：

| 层 | 关联 |
|----|------|
| **科学层** | 作物生理学、土壤学、气象学提供机理模型；AI/ML 提供数据驱动方法 |
| **知识层** | 命题知识（作物模型参数）+ 程序知识（农事操作技能）+ 机器知识（ML 模型权重） |
| **学科层** | 农业工程、计算机科学、数据科学的交叉——新学科"数字农业"正在建制化 |

**认识论启示**：智慧农业暴露了农业知识的**两种逻辑**之间的张力——

| 逻辑 | 来源 | 特点 |
|------|------|------|
| **农民的逻辑** | 经验、直觉、默会知识 | 整体性、情境敏感、难以形式化 |
| **算法的逻辑** | 数据、模型、优化目标 | 精确、可复制、忽略不可量化的维度 |

真正有效的智慧农业必须是二者的**融合**，而非替代。

## 场景应用

### 种植业

- **精准播种**：根据土壤条件变量调整播种密度
- **养分管理**：氮肥的实时诊断与变量追肥
- **病虫害预警**：基于气象+孢子捕获+图像识别的早期预警
- **产量预测**：结合遥感、气象、作物模型的区域级产量预估

### 畜牧业

- **精准饲喂**：根据个体产奶量/增重调整日粮配方
- **健康监测**：反刍传感器、活动量监测——提前 2-3 天发现疾病
- **繁殖管理**：发情检测、最佳配种时间预测
- **个体追溯**：区块链记录从出生到餐桌的完整生命周期

### 水产养殖

- **水质实时监控**：溶解氧、pH、氨氮——异常自动预警
- **智能投喂**：基于鱼类摄食行为识别饱食度的精准投喂
- **水下机器人**：网箱巡检、死鱼清理、网衣清洗

## 当代挑战

### 技术层面

- **可解释性**：AI 模型推荐"今天浇水"，但不告诉你为什么——农民不信任
- **数据孤岛**：不同设备/平台的数据格式不统一——互操作性差
- **模型泛化**：在 A 县训练好的模型到了 B 县可能完全失效
- **基础设施**：农村网络覆盖不足——没有 5G/宽带，智慧农业是空中楼阁

### 社会经济层面

- **数字鸿沟**：大农场用得起卫星数据，小农户连智能手机都不会用——智慧农业加深了不平等
- **数据主权**：农民的数据归谁？设备商？平台？还是农民自己？
- **技能断层**：传统农民转型为"农业数据操作员"——谁来培训？
- **技术锁定**：购买某品牌的智能农机，维修和数据都在它手里——"你拥有拖拉机，但它不听你的话"
- **资本替代劳动**：智慧农业减少体力劳动需求，但对农村就业的影响是双刃剑

### 认识论层面

- **默会知识的命运**：当算法替代了"看天吃饭"的经验判断，几个世纪积累的农事默会知识会不会消失？
- **简化的代价**：传感器只测量它能测量的东西（温度、湿度、叶绿素）——那些不可量化的维度（土壤的"活力"、作物的"长势感觉"）被系统性地忽略
