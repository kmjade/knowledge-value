# 植物保护

> **UDC 632**: Plant damage, plant diseases, pests, weeds, plant protection

植物保护是农业的**防御系统**——作物在田间面临病原菌、昆虫、杂草和恶劣环境的四面围攻。全球因病虫草害损失的作物产量达 **20-40%**（FAO 估算），植保的核心任务是将这个损失降到最低。

## 核心领域

| 分支 | 内容 |
|------|------|
| **植物病理学** | 真菌、细菌、病毒、线虫病害——鉴定、流行规律、防控 |
| **农业昆虫学** | 害虫生物学、种群动态、天敌利用 |
| **杂草科学** | 杂草生态、除草剂、生物除草 |
| **农药科学** | 杀虫剂、杀菌剂、除草剂——化学、毒理、剂型 |
| **生物防治** | 天敌昆虫、微生物制剂（Bt、白僵菌）、信息素 |
| **综合防治（IPM）** | 多策略协调——"不是消灭，是管理" |

## 核心概念

### 病害三角

```
           病原 (Pathogen)
              /\
             /  \
            /    \
           /  病  \
          /   害   \
         /__________\
    寄主 (Host) ──── 环境 (Environment)
```

病害发生需要三者**同时满足**——任一角的削弱都可阻止病害。这是制定防控策略的理论基础。

### 综合防治（IPM）

Integrated Pest Management——从"有虫就打药"到"系统治理"的范式转变：

| 层级 | 策略 | 示例 |
|------|------|------|
| **1. 预防** | 不让病虫发生 | 抗性品种、轮作、健康种苗 |
| **2. 监测** | 知道什么时候需要行动 | 孢子捕捉器、性诱剂监测、田间调查 |
| **3. 经济阈值** | 不到阈值不防治 | "有几只虫不代表需要喷药" |
| **4. 干预** | 过了阈值才出手 | 生物→物理→化学（化学是最后手段） |

**核心理念**: 不是"消灭所有害虫"——那既不经济也不生态——而是将害虫种群控制在**经济损害水平以下**。

### 农药的演化

| 世代 | 代表 | 特点 |
|------|------|------|
| **第一代** | 无机物：砷、硫磺、波尔多液 | 广谱、高毒 |
| **第二代** | 有机合成：DDT、有机磷、氨基甲酸酯 | 高效、残留问题 |
| **第三代** | 拟生农药：保幼激素、信息素 | 专一性强、环境友好 |
| **第四代** | 生物制剂 + RNAi | 精准靶向、基因层面干扰 |

### 杀菌剂作用机制

杀菌剂按作用方式分为两类：

| 类型 | 机制 | 代表 |
|------|------|------|
| **保护性** | 在病原侵入前覆盖植物表面 | 百菌清、代森锰锌 |
| **内吸性** | 被植物吸收后在体内运转 | 三唑类、甲氧基丙烯酸酯类 |

**抗药性**是核心问题——病原菌对单一作用位点的内吸剂产生抗性的速度远超保护性杀菌剂。FRAC（杀菌剂抗性行动委员会）制定分类标签指导轮换使用。

## 当代挑战

- **抗药性**: 全球已记录 600+ 种害虫/病原/杂草对农药产生抗性——新农药研发速度跟不上
- **农药残留**: 消费者对"零农残"的要求与田间实际需要之间的矛盾
- **新入侵物种**: 全球贸易加速病虫传播——草地贪夜蛾 3 年从美洲入侵到亚非
- **气候变化**: 暖冬使害虫越冬范围扩大、世代数增加；病害流行区向北移动
- **生物多样性**: 广谱杀虫剂杀灭天敌——"越打药虫越多"的恶性循环
- **蜜蜂危机**: 新烟碱类杀虫剂的亚致死效应——全球传粉昆虫下降

## 在三层模型中的位置

植物保护是典型的**应用科学**，但植保实践体现了科学知识与田间经验的持续对话：

- **科学层**: 病原生物学、毒理学——揭示了病虫害的分子和生态机制
- **知识层**: 丰富的过程性知识——"望闻问切"式的田间诊断依赖默会知识
- **学科层**: 植物保护已是成熟学科——植保学院 + 农药登记制度 + 专业化服务
