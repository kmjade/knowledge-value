# 01 — 作物科學

> 作物科學研究作物的生長、發育、產量與品質——從基因到田間管理。

## 主要作物分類

| 類別 | 作物 | 用途 |
|------|------|------|
| **糧食作物** | 水稻、小麥、玉米 | 主食 |
| **豆類作物** | 大豆、花生、綠豆 | 蛋白質、油脂 |
| **油料作物** | 油菜、向日葵、芝麻 | 食用油 |
| **經濟作物** | 棉花、甘蔗、茶、咖啡 | 工業原料、飲料 |
| **蔬果作物** | 番茄、白菜、柑橘 | 蔬菜水果 |
| **飼料作物** | 苜蓿、玉米青貯 | 畜牧飼料 |

## 作物生長因子

| 因子 | 影響 |
|------|------|
| **光** | 光合作用、光週期、開花誘導 |
| **溫度** | 發芽、生長速率、耐寒性 |
| **水** | 養分運輸、細胞膨壓 |
| **養分** | N-P-K（氮磷鉀）+ 微量元素 |
| **土壤** | 根系發展、養分供應 |

## 作物育種

| 方法 | 說明 |
|------|------|
| **選拔育種** | 從現有群體中選擇優良個體 |
| **雜交育種** | 不同親本雜交產生後代 |
| **突變育種** | 誘導突變篩選有利性狀 |
| **分子育種** | MAS 標誌輔助選拔 |
| **基因編輯** | CRISPR 精準修改基因 |

## 栽培管理

| 環節 | 措施 |
|------|------|
| 播種 | 適期播種、合理密度 |
| 施肥 | 基肥 + 追肥、測土配方 |
| 灌溉 | 滴灌、噴灌、浸灌 |
| 輪作 | 豆科與禾本科輪作、防止連作障礙 |

> 💡 作物科學的核心是「在對的時間做對的事」——農業的節奏由自然決定。
