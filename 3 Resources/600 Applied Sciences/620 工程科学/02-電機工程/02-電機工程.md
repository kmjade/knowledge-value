# 02 — 電機工程

> 電機工程研究電、電磁和電子——從發電到晶片，從通訊到控制。

## 核心領域

| 領域 | 內容 |
|------|------|
| **電路理論** | 電壓、電流、電阻、Kirchhoff 定律 |
| **電子學** | 二極體、電晶體、運算放大器 |
| **電磁學** | Maxwell 方程、天線、傳輸線 |
| **訊號與系統** | 傅立葉轉換、濾波器、取樣 |
| **控制系統** | PID、回授、穩定性分析 |
| **電力系統** | 發電、輸電、配電、馬達 |
| **通訊系統** | 調變、編碼、通道容量 |
| **數位邏輯** | 布林代數、組合/序向電路 |

## 電子元件

| 元件 | 符號 | 功能 |
|------|------|------|
| 電阻 (R) | Ω | 限制電流 |
| 電容 (C) | F | 儲存電荷 |
| 電感 (L) | H | 儲存磁能 |
| 二極體 | →| | 單向導通 |
| 電晶體 | BJT/MOSFET | 放大/開關 |

## 現代電機工程

- **VLSI**：數十億電晶體在單一晶片上
- **嵌入式系統**：微控制器、物聯網
- **電力電子**：電能轉換與控制
- **訊號處理**：數位訊號處理 (DSP)、機器學習

> 💡 電機工程是現代文明的「神經系統」——一切電子設備的基礎。
