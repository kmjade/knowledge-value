# 01 — 機械工程

> **DDC 621** · 機械工程 — **UDC 621** Mechanical engineering
> **CLC TH** 機械、儀表工業 — **LCC TJ** Mechanical engineering and machinery
>
> 機械工程是工程學的基石——研究力、運動、能量和材料，設計和製造機械系統。從汽車發動機到風力渦輪，從鐘錶齒輪到機器人關節。

## 核心領域

| 領域 | 內容 |
|------|------|
| **靜力學** | 靜止物體的力平衡 |
| **動力學** | 運動物體的力與加速度 |
| **材料力學** | 應力、應變、強度 |
| **熱力學** | 能量轉換、熱機、冷凍循環 |
| **流體力學** | 液體與氣體的行為 |
| **機構設計** | 齒輪、連桿、凸輪 |
| **製造工程** | CNC、3D列印、鑄造、焊接 |

## 熱力學四大定律

| 定律 | 含義 |
|------|------|
| **第零定律** | 熱平衡的遞移性——若 A=B 且 B=C 溫度相同，則 A=C |
| **第一定律** | 能量守恆 (ΔU = Q − W) |
| **第二定律** | 熵增原理，熱不能自發從冷到熱 |
| **第三定律** | 絕對零度不可達 |

### 熱機循環

將熱能轉化為機械功的裝置。

| 循環 | 應用 | 效率範圍 |
|------|------|----------|
| 奧托循環 | 汽油發動機 | ~25–30% |
| 狄塞爾循環 | 柴油發動機 | ~35–45% |
| 布雷頓循環 | 燃氣輪機（噴氣發動機、發電） | ~35–60% |
| 朗肯循環 | 蒸汽輪機（火電、核電） | ~35–45% |

> 汽車發動機僅約 25% 的燃油能量變成推動力——其餘以熱量浪費。這就是電氣化是效率革命的原因。

## 機械設計要素

| 要素 | 功能 | 例子 |
|------|------|------|
| **軸承** | 支撐旋轉軸，減少摩擦 | 滾珠軸承、滑動軸承 |
| **齒輪** | 傳遞扭矩和改變轉速 | 變速器、差速器 |
| **彈簧** | 儲存和釋放彈性能 | 懸掛系統 |
| **密封** | 防止洩漏 | O 形圈、墊片 |
| **緊固件** | 連接零件 | 螺栓、鉚釘、焊接 |

### 失效模式

| 模式 | 原因 | 預防 |
|------|------|------|
| 疲勞 | 反覆應力 | 疲勞壽命設計、避免應力集中 |
| 磨損 | 摩擦 | 潤滑、硬化表面 |
| 腐蝕 | 化學/電化學反應 | 塗層、材料選擇 |
| 潛變 | 高溫 + 持續應力 | 選擇耐潛變合金 |
| 降伏 | 應力超過降伏強度 | 安全係數 |

## 製造方法

| 方法 | 原理 | 適用 |
|------|------|------|
| 鑄造 | 液態金屬倒入模具 | 複雜形狀、大批量 |
| 鍛造 | 高壓成形 | 高強度零件 |
| 機加工 (CNC) | 切削去除材料 | 精密零件 |
| 3D 列印 | 逐層堆積 | 複雜幾何、小批量 |
| 焊接 | 熔化連接 | 結構件 |

## 新興領域

- **機電整合** (Mechatronics)：機械 + 電子 + 控制
- **微機電系統** (MEMS)：微型感測器與致動器
- **軟性機器人**：仿生、柔性致動
