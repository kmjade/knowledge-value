# 01-Cobot 概述

> 协作机器人 (Collaborative Robot, Cobot) 是专为人机安全协作而设计的机器人。与传统工业机器人的本质区别：**不需要围栏，共享工作空间**。

## 演进史

| 时期 | 事件 |
|------|------|
| 1996 | J. Edward Colgate 和 Michael Peshkin (西北大学) 首次提出 Cobot 概念 |
| 2008 | **Universal Robots 推出 UR5**——首个商用协作机器人 |
| 2014 | ABB 推出 YuMi——双臂协作机器人 |
| 2015 | FANUC 推出 CR 系列 |
| 2016 | ISO/TS 15066 协作机器人安全标准发布 |
| 2020s | 中国品牌崛起 (越疆/节卡/珞石)，AI+视觉融入 Cobot |

## 为什么需要 Cobot？

| 驱动力 | 说明 |
|--------|------|
| **多品种小批量** | 传统产线换型慢→需要灵活的自动化 |
| **劳动力短缺** | 招工难、老龄化→机器人辅助而非替代 |
| **空间约束** | 没有地方加围栏→Cobot 直接嵌入现有产线 |
| **人机协作** | 人做判断和柔性，Cobot 做精确和重复 |

## 核心特征

| 特征 | 说明 |
|------|------|
| **力控** | 感知接触力→安全停止或柔顺运动 |
| **拖动示教** | 手把手教机器人——不需要编程 |
| **快速部署** | 数小时到数天——传统机器人需数周 |
| **轻量** | 10-35kg 自重→可移动安装 |

## 市场格局

| 排名 | 品牌 | 市场份额 |
|------|------|----------|
| 1 | Universal Robots | ~40-50% |
| 2 | FANUC | ~10-15% |
| 3 | ABB/KUKA | 各 ~5-10% |
| 4 | 越疆/节卡 (中国) | 快速增长 |

> 2024 年全球 Cobot 市场 ~$20 亿，年增长 20%+。中国是增长最快的市场。
