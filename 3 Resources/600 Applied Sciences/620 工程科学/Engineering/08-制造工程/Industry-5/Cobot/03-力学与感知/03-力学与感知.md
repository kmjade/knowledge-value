# 03-力学与感知

> Cobot 的"感觉"来自关节力矩传感器和末端力传感器——这是实现安全、拖动示教和柔顺控制的基础。

## 力矩感知

### 关节力矩传感器

每个关节内置——测量输出扭矩：

| 位置 | 实现 |
|------|------|
| 电机输出端 | 应变片测量轴上的扭矩 |
| 谐波减速器后 | 更精确——排除减速器摩擦影响 |

### 如何检测碰撞？

```
期望力矩 (动力学模型) - 实际力矩 (传感器) = 外力矩
                                           ↓
                              > 阈值 → 碰撞 → 停止 (几ms内)
```

| 步骤 | 耗时 |
|------|------|
| 传感器读取 | ~0.1ms |
| 碰撞判断 | ~1ms |
| 停止指令 | ~2ms |
| 机器人停止 | ~10-50ms (取决于速度) |

## 力/扭矩传感器

安装在 Cobot 末端和工具之间的六维传感器：

| 维度 | 测量 |
|------|------|
| Fx, Fy, Fz | 三个方向的力 |
| Tx, Ty, Tz | 三个方向的扭矩 |

### 应用

| 功能 | 说明 |
|------|------|
| **力控装配** | 将轴插入孔——感知接触力→调整位置 |
| **恒力打磨** | 保持接触力恒定→不因工件形状变化 |
| **拖动示教** | 感知手的推力→跟随运动 |

## 拖动示教的力学原理

```
手推 Cobot → 末端力传感器感知力的方向和大小
                    ↓
        阻抗控制: F = M·a + B·v + K·x
                    ↓
        机器人向力的方向移动→感觉像"推一扇很轻的门"
```

| 参数 | 感觉 |
|------|------|
| M 大 (惯性大) | 推起来"重" |
| B 大 (阻尼大) | 推起来"粘" |
| K 大 (刚度大) | 松开后快速回弹 |

> 好的拖动示教体验：M 和 B 都小——推起来轻、松手即停、运动平滑。

## 柔顺控制

| 模式 | 说明 |
|------|------|
| **位置控制** | 精确到目标位置——不在乎力 |
| **力控制** | 精确到目标力——不在乎位置 |
| **阻抗控制** | 动态调节力-位置关系——"像弹簧一样" |
| **导纳控制** | 测量力→调整位置——"像阻尼器一样" |
