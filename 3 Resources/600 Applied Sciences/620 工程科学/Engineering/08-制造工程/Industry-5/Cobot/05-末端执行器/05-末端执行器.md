# 05-末端执行器

> 末端执行器 (End-of-Arm Tooling, EOAT) 是 Cobot 的"手"——没有它 Cobot 只是一个会动的金属臂。选对夹爪比选对 Cobot 更重要。

## 末端执行器类型

| 类型 | 原理 | 负载 | 适合 |
|------|------|------|------|
| **气动平行夹爪** | 压缩空气驱动两指开合 | 1-20kg | 通用——最常用 |
| **电动夹爪** | 电机驱动——力控精准 | 1-15kg | 柔性工件、需要力控 |
| **真空吸盘** | 负压吸起 | <5kg | 平整表面——包装、上料 |
| **磁力吸盘** | 电磁铁 | 取决于磁场 | 铁磁性工件 |
| **三指定心夹爪** | 三指同时收拢 | 1-10kg | 圆柱形工件 |
| **柔性夹爪** | 硅胶手指+气压 | <1kg | 异形、易碎物品 |
| **专用工具** | 焊枪、打磨头、螺丝刀 | — | 特定工艺 |

## 夹爪选型

| 参数 | 说明 |
|------|------|
| 工件重量 | 夹持力 ≥ 重量 × (安全系数 3-5) × 摩擦系数 |
| 工件形状 | 平行面→平行夹爪；圆柱→三指；不平整→柔性 |
| 工件材料 | 硬→金属夹爪；脆/软→硅胶垫 |
| 节拍要求 | 气动快 (0.1s)、电动慢 (0.5s) |
| 环境 | 洁净间→电动；油污→气动 |

## 快换装置

让 Cobot 自动切换不同工具：

```
Cobot 末端 → 快换母头 → 快换公头 → 夹爪A
                   公头 → 焊接头
                   公头 → 吸盘
```

| 品牌 | 特点 |
|------|------|
| **Schunk** | 精密、气+电+数据一体 |
| **ATI** | 机器人快换先驱 |
| **OnRobot** | 即插即用——Cobot 专用 |

## 视觉引导的末端执行

| 应用 | 说明 |
|------|------|
| 无序抓取 (Bin Picking) | 视觉找到工件位置→Cobot 去抓 |
| 定位补偿 | 工件有偏差→视觉测量偏差→夹爪位置实时调整 |
| 质检抓取 | 视觉检测→OK 放到 A 箱、NG 放到 B 箱 |
