# 01-工业5.0概述

> 工业 5.0 是欧盟 2021 年正式提出的概念——补足工业 4.0 在人文、环境和社会层面的缺失。它不替代 4.0，而是将 4.0 的技术能力导向更宏大的目标。

## 工业革命的代际

| 代 | 时期 | 驱动 | 标志 |
|----|------|------|------|
| 1.0 | 1780s | 蒸汽 | 机械化 |
| 2.0 | 1870s | 电力 | 流水线、大规模生产 |
| 3.0 | 1970s | 电子+计算机 | 自动化 |
| 4.0 | 2010s | IoT+AI+数字孪生 | 智能制造 |
| **5.0** | **2020s** | **人+地球+韧性** | **以人为本的可持续制造** |

## 三大支柱

### 以人为本

| 原则 | 含义 |
|------|------|
| 技术服务于人 | 不是人适应机器——机器适应人 |
| 增强而非替代 | Cobot 辅助操作员——做重活、精活、危险活 |
| 技能升级 | 操作员→数字工匠——培训+终身学习 |

### 可持续性

| 原则 | 含义 |
|------|------|
| 碳中和 | 制造过程净零排放 |
| 循环经济 | 设计→制造→使用→回收→再生——闭环 |
| 尊重地球边界 | 不超越地球系统的承载极限 |

### 韧性

| 原则 | 含义 |
|------|------|
| 抗冲击 | 疫情、战争、自然灾害——供应链不能断 |
| 适应性 | 快速换型、多产品混线 |
| 分布式 | 不依赖单一来源/单一工厂 |

## 为什么需要 5.0？

| 4.0 的局限 | 5.0 的回应 |
|-----------|-----------|
| 技术中心——人只是"用户" | 人机协作——人是决策者 |
| 效率最大化忽视了环境成本 | 可持续性内置于生产系统 |
| 全球最优供应链→脆弱 | 韧性供应链——成本不是唯一考量 |
| 以发达国家为中心 | 包容全球南方和中小企业 |
