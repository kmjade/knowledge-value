# 工业5.0 知识库总览

> 工业 5.0 是欧盟 2021 年提出的下一代工业范式——在工业 4.0 的数字底座上，补充三个核心价值：**人机协作、可持续性、韧性**。它不替代 4.0，而是超越。

> **两个入口**: 本页是完整目录索引；[[Industry-5|工业5.0]] 是概念导读——含五代工业革命全景、三大支柱展开、三层模型定位、核心概念速览。

## 工业 4.0 → 5.0

| 维度 | 工业 4.0 | 工业 5.0 |
|------|----------|----------|
| 核心驱动 | 技术 (IoT, AI, 数字孪生) | 价值 (人、地球、韧性) |
| 人与机器 | 机器替代人 | 人机协作 (Cobot) |
| 环境 | 效率优先 | 可持续性为核心 |
| 供应链 | 全球最优成本 | 韧性+可持续+成本 |
| 目标 | 智能制造 | 以人为本的可持续制造 |

```
        以人为本 ←──→ 可持续性
           ↖    ／
            韧性
```

> 三者的交集处是工业 5.0 的核心——技术服务于人，生产不破坏地球，系统经得起冲击。
