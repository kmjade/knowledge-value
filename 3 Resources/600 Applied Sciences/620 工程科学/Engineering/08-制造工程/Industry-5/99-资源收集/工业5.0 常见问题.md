# 工业5.0 常见问题

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| 工业5.0 替代 4.0 | 5.0 补充 4.0——4.0 提供数字底座，5.0 加入人文和可持续维度 |
| 协作机器人取代工业机器人 | Cobot 和传统工业机器人互补——Cobot 擅长柔性、人机协作；传统机器人擅长高速、大批量 |
| 可持续 = 环保主义 | 可持续 = 经济可行+环境可承受+社会公平——三重底线 |
| 韧性 = 冗余库存 | 韧性是多维的——供应商多元化、数字可视性、快速换型都是韧性 |
| 只是欧洲的概念 | 人机协作、碳中和、供应链韧性是所有制造业大国的共同趋势 |
| 中小企业做不了 | 韧性（多供应商）和可持续（节能）恰恰是中小企业最容易起步的方向 |
