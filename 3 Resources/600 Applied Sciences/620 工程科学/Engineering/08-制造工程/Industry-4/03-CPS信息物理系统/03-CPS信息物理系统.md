# 03-CPS 信息物理系统

> CPS (Cyber-Physical System) 是工业 4.0 的理论核心——计算、通信和控制深度嵌入物理过程，形成闭环。

## 核心概念

```
物理世界 ──传感器──→ 数字世界 (Cyber)
   ↑                       ↓
   └── 执行器 ←── 控制/优化 ←─┘
```

| 特征 | 说明 |
|------|------|
| **实时感知** | 传感器持续采集物理状态 |
| **数字映射** | 在数字空间中精确反映物理实体 |
| **计算决策** | AI/规则引擎分析→决定做什么 |
| **闭环控制** | 决策通过执行器作用于物理世界 |

## CPS vs 传统自动化

| 维度 | 传统自动化 | CPS |
|------|----------|-----|
| 感知范围 | 单设备 | 全系统互联 |
| 决策 | 预设规则 | AI 动态优化 |
| 适应性 | 固定 | 自适应 |
| 人机关系 | 人监控机器 | 人机协同 |
| 信息模型 | 点对点信号 | 语义模型+数字孪生 |

## CPS 5C 架构

| 层级 | 功能 |
|------|------|
| **Connection** | 连接——传感器+通信 |
| **Conversion** | 转换——数据→信息 |
| **Cyber** | 数字空间——孪生+仿真 |
| **Cognition** | 认知——AI 诊断和预测 |
| **Configuration** | 配置——反馈控制 |
