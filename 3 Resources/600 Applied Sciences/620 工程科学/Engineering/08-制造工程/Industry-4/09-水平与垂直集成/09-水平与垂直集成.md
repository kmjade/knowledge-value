# 09-水平与垂直集成

> 工业 4.0 不只是车间里的事——它需要从供应商到客户的水平集成，以及从传感器到 ERP 的垂直集成。

## 三大集成维度

```
           垂直集成 ↑↓
    ┌──────────────────────┐
    │   ERP (企业资源计划)    │
    ├──────────────────────┤
    │   MES (制造执行系统)    │
    ├──────────────────────┤
    │   SCADA/PLC           │
    ├──────────────────────┤
    │   传感器/执行器         │
    └──────────────────────┘
              ↕
    水平集成 ←──→ 水平集成
 (与供应商)        (与客户)
```

## 垂直集成

从现场层到企业层的信息贯通：

| 实现 | 技术 |
|------|------|
| 数据上流 | OPC UA→MES→ERP——生产数据自动同步 |
| 指令下流 | ERP→MES→PLC——订单自动下发 |
| 效果 | 无纸化、实时化、可追溯 |

## 水平集成 (端到端工程)

从供应商→制造→客户的全价值链：

| 阶段 | 集成内容 |
|------|----------|
| 供应商 | 订单实时共享、质量数据回流 |
| 制造 | 从设计到制造的数字主线 |
| 客户 | 定制化配置→直接下发产线 |

## 数字主线 (Digital Thread)

```
设计 → 工艺 → 制造 → 检验 → 交付 → 服务
  ↑___________数据持续贯通___________↓
```

> 没有数字主线，设计是一个孤岛，制造是另一个。出了问题不知道是设计锅还是工艺锅。
