# 07-建筑与基础设施

> 建筑数字孪生从 BIM 出发——将设计模型和运行数据融合，贯穿建筑的全生命周期（设计→施工→运营）。

## BIM → 数字孪生

| 维度 | BIM | 数字孪生 |
|------|-----|---------|
| 阶段 | 设计与施工 | 全生命周期（侧重运营） |
| 数据 | 静态（几何、材料） | 动态（IoT 实时数据） |
| 更新 | 设计变更时 | 持续同步 |
| 用户 | 建筑师、工程师 | 设施经理、运维团队 |

```
BIM 模型 (Revit) ──→ 数字孪生平台 ──→ 运营优化
                         ↑
                  IoT 实时数据 (传感器、电表、HVAC)
```

## 建筑运营的核心应用

### 能耗管理

| 应用 | 说明 |
|------|------|
| 能耗监测 | 分项计量——照明、空调、动力各自用了多少 |
| 异常检测 | "今天空调能耗比昨天高 30%，但温度差不多→可能设备故障" |
| 预测优化 | 基于天气预报和人流预测，提前调整 HVAC 策略 |
| 碳追踪 | 实时显示碳足迹——满足 ESG 报告要求 |

### HVAC 优化

```
建筑孪生 + 天气预报 + 室内人流数据
        ↓
    AI 预测模型
        ↓
  最优冷量供给方案 → 下发到 BACnet/Modbus 控制器
```

> 这是建筑数字孪生 ROI 最高的场景——HVAC 占建筑能耗 40-60%，优化 15-25% 是可行的。

### 空间管理

| 应用 | 说明 |
|------|------|
| 工位利用率 | 传感器监测实际使用率→优化空间分配 |
| 会议室管理 | 预订 vs 实际使用→释放闲置会议室 |
| 寻路导航 | 室内导航+AR 指引 |

## 基础设施孪生

| 类型 | 监测什么 | 价值 |
|------|----------|------|
| **桥梁** | 结构应变、振动、位移 | 安全预警、预测维护 |
| **隧道** | 变形、渗漏、空气质量 | 运营安全 |
| **水处理厂** | 流量、水质、设备状态 | 合规+节能 |
| **电网** | 负荷、电压、设备温度 | 动态调度、故障预警 |

## 城市数字孪生 (CIM)

从建筑到城市——CIM (City Information Model)：

| 城市 | 项目 | 内容 |
|------|------|------|
| 新加坡 | Virtual Singapore | 全城 3D 孪生——洪涝模拟、风道分析、太阳能潜力 |
| 赫尔辛基 | Helsinki 3D+ | 城市能耗模拟和碳中和规划 |
| 雄安新区 | 数字雄安 | 从设计阶段就建立数字孪生——地上地下全要素 |
