# 08-运动控制

> 运动控制是自动化的精密分支——不只要"动"，还要精确控制位置、速度和扭矩。从 CNC 机床到贴片机，从机器人到印刷机。

## 驱动类型对比

| 类型 | 反馈 | 精度 | 成本 | 典型应用 |
|------|------|------|------|----------|
| **变频器 + 异步电机** | 无/编码器 | 低-中 | 低 | 风机、水泵、传送带 |
| **步进电机** | 开环 | 中 | 低 | 简单定位、3D 打印 |
| **伺服电机** | 编码器闭环 | 高 | 中-高 | 机器人、CNC、贴片 |
| **直线电机** | 光栅尺闭环 | 极高 | 高 | 半导体、精密平台 |
| **力矩电机** | 编码器 | 极高扭矩 | 高 | 转台、直驱 |

## 伺服系统

### 三环控制结构

```
位置设定 ──→ⓔ→ 位置环(P) ──→ⓔ→ 速度环(PI) ──→ⓔ→ 电流环(PI) → 电机
              ↑                       ↑                    ↑
              └─── 编码器位置 ─────────└── 编码器速度 ──────┘
```

| 环 | 周期 | 作用 |
|----|------|------|
| 电流环 | ~50μs | 控制扭矩——最快 |
| 速度环 | ~200μs | 控制转速 |
| 位置环 | ~1ms | 控制最终位置 |

### 伺服选型要点

| 参数 | 含义 |
|------|------|
| 额定功率 (kW) | 持续输出的机械功率 |
| 额定扭矩 (Nm) | 持续输出的扭矩 |
| 峰值扭矩 | 短时过载（通常 3 倍额定） |
| 额定转速 (rpm) | 额定扭矩下的最高转速 |
| 编码器分辨率 | 如 20bit = 1,048,576 脉冲/圈 |
| 惯量比 | 负载惯量 / 电机惯量——<10:1 理想 |

## 步进电机

| 优点 | 缺点 |
|------|------|
| 便宜 | 高速扭矩急剧下降 |
| 开环控制——简单 | 过载失步（没有反馈不知道） |
| 静止时有保持扭矩 | 效率低、发热 |
| 适合低转速定位 | 噪音和振动 |

### 步进 vs 伺服——怎么选？

| 选步进 | 选伺服 |
|--------|--------|
| 低成本 | 高性能 |
| 开环够用 | 需要反馈保障 |
| 低速 (<1000rpm) | 高速或变速 |
| 恒定负载 | 变负载/冲击负载 |
| 精度要求不高 | 高精度定位 |

## 运动控制功能

| 功能 | 说明 |
|------|------|
| **点位 (PTP)** | 从 A 到 B，路径不重要 |
| **电子齿轮** | 从轴跟随主轴——固定速比 |
| **电子凸轮** | 从轴根据曲线函数跟随主轴 |
| **插补** | 多轴协调走指定轨迹——直线、圆弧 |
| **回零** | 找机械原点 |
| **飞剪** | 运动中同步切割——包装机典型应用 |

## CNC

计算机数字控制——运动控制的典型应用：

```
G-code (NC 程序)
    ↓
CNC 控制器 (插补器)
    ↓
伺服驱动器 × 3 (X/Y/Z)
    ↓
加工
```

| G 代码 | 功能 |
|--------|------|
| G00 | 快速定位 |
| G01 | 直线插补 |
| G02/G03 | 顺/逆时针圆弧插补 |
| G90/G91 | 绝对/增量坐标 |
