# 02-传感器技术

> 传感器是 IoT 的"感官"——将温度、压力、振动、光线等物理量转换为电信号。MEMS 技术让传感器从工业设备缩小到指尖大小。

## 传感器分类

| 类别 | 测量 | 典型传感器 | 输出 |
|------|------|-----------|------|
| **温度** | -200~2000°C | PT100, 热电偶, NTC | 4-20mA/数字 |
| **压力** | 0~1000bar | 应变片, MEMS | 4-20mA/I²C |
| **振动** | 加速度 | MEMS 加速度计, 压电 | 模拟/SPI |
| **位移/位置** | 距离/角度 | 编码器, LVDT, 激光 | 脉冲/4-20mA |
| **流量** | 液体/气体 | 电磁, 超声波, 科氏力 | 4-20mA/脉冲 |
| **气体** | ppm 浓度 | 电化学, NDIR, MOS | 模拟/I²C |
| **图像** | 视觉 | CMOS 摄像头 | MIPI/USB |
| **声音** | 噪声 | MEMS 麦克风 | I²S/PDM |
| **环境** | 湿度/光照/PM2.5 | 温湿度一体, 颗粒物 | I²C/UART |

## MEMS——传感器的革命

| 特点 | 说明 |
|------|------|
| 微型化 | 毫米级芯片包含机械结构+电路 |
| 低成本 | 硅工艺批量制造——一个加速度计 <$0.5 |
| 低功耗 | μA 级——电池供电可行 |
| 集成化 | 6 轴 IMU = 3 轴加速度+3 轴陀螺仪 |

> 你手机里有 10-20 个 MEMS 传感器——加速度计、陀螺仪、磁力计、气压计、麦克风、接近传感器...每个都曾是数万美元的工业设备。

## 传感器接口

| 接口 | 类型 | 距离 | 速率 |
|------|------|------|------|
| **模拟** (4-20mA/0-10V) | 模拟 | 数百米 | — |
| **I²C** | 数字 | <1m | 100k-3.4Mbps |
| **SPI** | 数字 | <1m | 10-50Mbps |
| **UART** | 数字 | <10m | 9600-4Mbps |
| **IO-Link** | 工业数字 | 20m | 230kbps |
| **RS-485** | 工业数字 | 1200m | 100k-10Mbps |

## 传感器融合

单传感器有局限——多传感器融合提高精度和可靠性：

| 融合 | 传感器 | 输出 |
|------|--------|------|
| IMU+GPS | 加速度+陀螺+磁力+GPS | 精确的位置和姿态 |
| 振动+温度+电流 | 加速度+PT100+CT | 设备健康状态 |
| 雷达+摄像头 | mmWave+视觉 | 自动驾驶环境感知 |

## 选型考量

| 因素 | 问题 |
|------|------|
| 量程 | 测 -40~+125°C 还是 0~50°C？ |
| 精度 | 需要 ±0.1% 还是 ±5%？ |
| 响应时间 | 温度变化需要多快感知？ |
| 环境 | 防爆？防水(IP67)？耐腐蚀？ |
| 功耗 | 电池供电→必须低功耗 |
| 成本 | 10 个 vs 10,000 个——价差巨大 |
