# 04-电气与电子工程

> 电气工程研究电力的产生、传输和使用。电子工程研究电子器件和电路——从晶体管到微处理器。两者是现代文明的"神经系统"。

## 电力系统

```
发电厂 → 升压变压器 → 输电网 (高压) → 降压变压器 → 配电网 → 用户
```

### 发电方式对比

| 方式 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **火电** (煤/气) | 烧水→蒸汽→涡轮→发电机 | 稳定、可调度 | CO₂ 排放 |
| **核电** | 核裂变→烧水→涡轮→发电机 | 零碳、基载 | 废料、安全、成本 |
| **水电** | 水流→涡轮→发电机 | 可再生、可调度 | 生态影响 |
| **风电** | 风→叶片→发电机 | 可再生 | 间歇性 |
| **光伏** | 光子→电子（半导体） | 可再生、无运动部件 | 间歇性 |

### 交流 vs 直流

| 特征 | 交流 (AC) | 直流 (DC) |
|------|-----------|-----------|
| 变压 | 容易（变压器） | 困难（需电力电子） |
| 长距离传输 | 高效（高压交流） | 更高效（高压直流 HVDC） |
| 应用 | 电网、家用电器 | 电池、电子设备、数据中心 |

> 交流胜出的历史原因：19 世纪末特斯拉的交流电可以用变压器轻松升降压，爱迪生的直流不能。但今天的高压直流 (HVDC) 在超长距离传输上更优秀。

## 电子学

### 从真空管到晶体管到集成电路

| 时代 | 器件 | 集成度 |
|------|------|--------|
| 1900s-1950s | 真空管 | 1 个/玻璃管 |
| 1947 | 晶体管 (Bardeen, Brattain, Shockley) | 1 个/器件 |
| 1958 | 集成电路 (Kilby, Noyce) | 多个/芯片 |
| 1971 | 微处理器 (Intel 4004) | 2300 晶体管 |
| 2024 | 最新 GPU | 2080 亿晶体管 |

### 基本元件

| 元件 | 功能 | 符号 |
|------|------|------|
| **电阻** | 限制电流 | R |
| **电容** | 储存电荷 | C |
| **电感** | 储存磁能 | L |
| **二极管** | 单向导电 | D |
| **晶体管** | 放大/开关 | Q |

### 摩尔定律

> 集成电路上的晶体管数量约每两年翻一番。(Gordon Moore, 1965)

| 年份 | 制程 | 代表芯片 |
|------|------|----------|
| 1971 | 10μm | Intel 4004 |
| 2000 | 180nm | Pentium 4 |
| 2010 | 32nm | Core i7 |
| 2020 | 5nm | Apple M1 |
| 2024 | 3nm | Apple M3, Snapdragon 8 Gen3 |

> 摩尔定律正在放缓——不是"死了"，而是每次翻倍越来越贵。物理极限（量子隧穿）在逼近。

## 信号处理

| 概念 | 含义 |
|------|------|
| 模拟信号 | 连续变化的信号（声音、光线） |
| 数字信号 | 离散的 0/1 |
| ADC/DAC | 模拟↔数字转换 |
| 采样定理 (Nyquist) | 采样频率 ≥ 信号最高频率 × 2 才能完整还原 |
| 傅里叶变换 | 将信号从时域转为频域——任何信号都可以分解为正弦波的和 |
