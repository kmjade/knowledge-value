# ERP 常见问题

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| ERP = 财务软件 | 财务只是核心模块之一——ERP 覆盖财务/供应链/生产/HR/CRM |
| 上了 ERP 就能解决问题 | ERP 是工具——混乱的流程 × ERP = 更快的混乱。先优化流程 |
| ERP 实施 = IT 项目 | ERP 是管理变革项目——业务部门的投入比 IT 更重要 |
| 大厂 ERP 太贵 | 云 ERP (SaaS) 降低了门槛——中小企业月付 ¥几千起 |
| ERP 上线=项目结束 | 上线是开始——持续优化、升级、培训 |
