# 工程学 常见问题

## 十大常见误区

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| 工程 = 应用科学 | 工程有自己独立的方法论——安全系数、FMEA、冗余设计。科学追求"真"，工程追求"够好且安全" |
| 工程师 = 穿工装的技工 | 工程师设计系统，技工执行操作。两者是不同但互补的职业 |
| 软件工程不是"真正的工程" | 软件工程同样面临约束优化、安全边际和故障模式——只是故障模式不同（没有物理磨损，但有技术债务） |
| 建筑倒了 = 设计问题 | 可能是设计、材料、施工、维护或使用不当——工程失败通常是多因素 |
| 中国造不出 X = 技术不行 | 很多时候是经济不可行而非技术不可能。工程的第一约束是成本 |
| 越多冗余越安全 | 冗余增加复杂度和故障模式——飞机三个引擎 ≠ 两个引擎的三分之二安全性 |
| CAD 取代了工程师的判断 | CAD 是工具，不能替代工程直觉。哈勃望远镜的错误就来自过于相信测量仪器 |
| 工程师不需要写作能力 | 工程文档、规范、事故报告——沟通能力是工程师的核心竞争力 |
| 最新的技术就是最好的 | 成熟技术比新技术可靠得多。工程偏爱"经过验证的"而非"最先进的" |
| 工程问题有"正确"答案 | 工程没有唯一正确答案——只有在不同约束下的最优权衡 |
