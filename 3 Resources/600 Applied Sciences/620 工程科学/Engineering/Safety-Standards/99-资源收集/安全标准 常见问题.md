# 安全标准 常见问题

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| 安全标准 = 建议 | 违反安全标准 → 出了事故 → 刑事责任。标准是强制性的（通过法规引用） |
| SIL 越高越好 | SIL 3/PL e 系统昂贵、复杂——能用 SIL 2 解决就不上 SIL 3 |
| 加个安全继电器就安全了 | 安全是系统性的——传感器+逻辑+执行器+诊断+定期测试 |
| 风险评估做一次就够了 | 每次改造、换产品、换工艺都要重新评估 |
| CE 标识 = 欧盟认证 | CE 是制造商自我声明符合性——不是第三方认证。公告机构介入仅在高风险产品 |
