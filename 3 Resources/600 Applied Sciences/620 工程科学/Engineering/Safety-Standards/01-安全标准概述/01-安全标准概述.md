# 01-安全标准概述

> 工业安全标准不是建议——是法律强制执行的技术规范。每个标准都遵循"三层结构"：A 类基础 → B 类通用 → C 类专用。

## 为什么需要安全标准？

| 驱动力 | 说明 |
|--------|------|
| **血的教训** | 几乎每个标准修订都源于真实事故 |
| **法律责任** | 不合规 → 出了事故 → 刑事责任 |
| **市场准入** | 没有 CE/CR → 不能在欧洲/中国销售 |
| **保险要求** | 保险公司要求合规 |

## 标准三兄弟：ISO / IEC / EN

| 组织 | 范围 | 例子 |
|------|------|------|
| **ISO** | 国际——机械、机器人 | ISO 12100, ISO 10218 |
| **IEC** | 国际——电气、电子 | IEC 60204-1, IEC 61508 |
| **EN** | 欧洲——采用 ISO/IEC | EN ISO 12100 |

> EN ISO xxxxx = 欧洲采纳了 ISO 标准。技术上一样，但 EN 版本是欧盟法规下的"协调标准"——满足它就推定满足法规。

## 法规 vs 标准

| 维度 | 法规 (Regulation/Directive) | 标准 (Standard) |
|------|---------------------------|-----------------|
| 制定者 | 政府/议会 | 标准组织 (ISO/IEC/CEN...) |
| 强制性 | 法律强制 | 自愿——但法规引用后变强制 |
| 内容 | "必须安全" | "怎样才算安全" |
| 例子 | 欧盟机械指令 2006/42/EC | ISO 12100 |

> 法规说"机器必须安全"。标准说"按照这个 checklist 做，你的机器就是安全的"。

## 主要认证体系

| 认证 | 地区 | 法规 |
|------|------|------|
| **CE** | 欧盟/欧洲经济区 | 机械指令、低电压指令、EMC 指令 |
| **UKCA** | 英国 (脱欧后) | 英国版 CE |
| **CR** | 中国 | CCC 认证 |
| **NRTL** (UL/CSA) | 北美 | OSHA 认可的测试实验室 |
| **NR-12** | 巴西 | 巴西机械安全法规 |

## CE 标识流程

```
产品 → 确定适用指令 → 选择合格评定路径 → 满足协调标准 
→ 编制技术文件 → 签署符合性声明 (DoC) → 粘贴 CE 标识
```

## A/B/C 标准层级

| 层级 | 说明 | 标准 |
|------|------|------|
| **A** | 所有机械的基础 | ISO 12100 |
| **B1** | 特定安全方面 | ISO 13857 (安全距离) |
| **B2** | 安全装置 | ISO 13850 (急停) |
| **C** | 特定机器 | ISO 10218 (机器人) |

> 优先级：C 类 > B 类 > A 类。如果 C 类标准有规定，不一定需要回到 A 类。
