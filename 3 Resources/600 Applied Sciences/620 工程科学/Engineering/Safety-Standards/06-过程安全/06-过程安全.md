# 06-过程安全

> 过程安全与机械安全的区别：机械安全防止夹手，过程安全防止爆炸。适用化工、石化、电力——连续流程、危险物质。

## 核心标准

| 标准 | 内容 |
|------|------|
| **IEC 61508** | 功能安全总则——所有行业 |
| **IEC 61511** | 过程工业功能安全 |

## HAZOP (危险与可操作性分析)

系统性地分析每一个节点、每一个参数偏离的后果：

| 引导词 | 参数 | 偏差 | 原因 | 后果 | 措施 |
|--------|------|------|------|------|------|
| 无/No | 流量 | 无流量 | 泵故障 | 反应器过热 | 备用泵+流量低联锁 |
| 多/More | 压力 | 压力高 | 出口阀误关 | 管道破裂 | 安全阀+压力高联锁 |
| 少/Less | 温度 | 温度低 | 加热蒸汽不足 | 反应停止 | 温度低报警 |

## LOPA (保护层分析)

一层一层分析——每道保护层独立，任一层的失效都应由其他层兜底：

```
┌─────────────────────────────┐
│         社区应急响应           │ ← 最外层
├─────────────────────────────┤
│       工厂应急响应             │
├─────────────────────────────┤
│   物理防护 (防爆墙、围堰)       │
├─────────────────────────────┤
│   安全仪表系统 (SIS)           │ ← 自动联锁停车
├─────────────────────────────┤
│   报警+操作员干预              │
├─────────────────────────────┤
│   基本过程控制 (BPCS)          │
├─────────────────────────────┤
│       工艺设计                 │ ← 最内层 (本质安全)
└─────────────────────────────┘
```

### SIL 定级

| SIL | PFD (低要求模式) | 风险降低 |
|-----|-----------------|----------|
| 1 | 10⁻²~10⁻¹ | 10-100× |
| 2 | 10⁻³~10⁻² | 100-1000× |
| 3 | 10⁻⁴~10⁻³ | 1000-10000× |
| 4 | 10⁻⁵~10⁻⁴ | 10000-100000× |

> 通常工厂需要 SIL 1-3。SIL 4 极少——如果有 SIL 4 的风险，通常应该从根本上重新设计工艺。

## SIS 架构

```
传感器 (变送器/开关) → 安全PLC → 执行器 (阀门/断路器)
  ↑ 2oo3 投票              ↑ 冗余          ↑ 部分行程测试
```

| 投票逻辑 | 含义 |
|----------|------|
| **1oo1** | 1 个传感器→单点故障→失效 |
| **1oo2** | 任一触发→执行→安全性高 |
| **2oo2** | 两个都触发→执行→可用性高 |
| **2oo3** | 3 取 2→兼顾安全+可用 |
