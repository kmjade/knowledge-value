# 04 — 材料工程

> 材料是工程的基礎——每一件產品都是由材料決定的。材料工程研究材料的結構、性質、製程與性能。

## 材料四大類

| 類別 | 特性 | 應用 |
|------|------|------|
| **金屬** | 高強度、導電導熱、延展 | 結構、導線、引擎 |
| **陶瓷** | 高硬度、耐高溫、脆性 | 切削工具、引擎部件、生醫 |
| **高分子** | 輕量、可塑性、絕緣 | 包裝、管材、紡織 |
| **複合材料** | 結合多種材料優點 | 航太、運動器材、風電葉片 |

## 材料性質

| 性質 | 指標 | 說明 |
|------|------|------|
| **強度** | 降伏強度、抗拉強度 | 承受荷載能力 |
| **剛性** | 彈性模數 (Young's Modulus) | 抵抗變形 |
| **韌性** | 應力-應變曲線下面積 | 吸收能量 |
| **硬度** | Rockwell, Brinell, Vickers | 抵抗表面壓痕 |
| **疲勞** | S-N 曲線 | 反覆荷載下的壽命 |

## 材料的微觀結構

| 尺度 | 結構 |
|------|------|
| 原子 | 鍵結、晶體結構 (BCC, FCC, HCP) |
| 微米 | 晶粒、晶界、析出物 |
| 毫米 | 相分佈、孔隙、缺陷 |

## 材料選擇方法 (Ashby 圖)

```
強度 vs 密度圖 → 找出最輕但最強的材料
成本 vs 性能 → 找出最經濟的選擇
```

## 新興材料

- 石墨烯 (Graphene)：單原子層碳，超高強度與導電性
- 形狀記憶合金 (SMA)：加熱恢復原形
- 自癒合材料：自動修補裂紋
- 超材料 (Metamaterial)：負折射率、隱形斗篷

> 💡 材料的突破往往帶來文明的躍進——石器、青銅、鐵器、鋼鐵、矽、碳纖維...
