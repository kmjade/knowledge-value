# 06 — 航太工程

> 航太工程涵蓋航空（大氣層內）與太空（大氣層外）——從飛機構型到火箭推進，從衛星到太空站。

## 航空 vs 太空

| 領域 | 環境 | 關鍵挑戰 |
|------|------|---------|
| **航空** | 大氣層內 | 空氣動力、升力、阻力、燃油效率 |
| **太空** | 真空/微重力 | 推進、熱控、輻射、生命維持 |

## 空氣動力學基礎

| 力 | 說明 |
|----|------|
| **升力** | 機翼上下壓力差產生 |
| **阻力** | 空氣摩擦 + 壓力差 |
| **推力** | 引擎產生 |
| **重力** | 飛機重量 |

## 推進系統

| 類型 | 應用 | 原理 |
|------|------|------|
| 活塞引擎 | 小型飛機 | 內燃機驅動螺旋槳 |
| 渦輪噴射 | 早期噴射機 | 高溫高速排氣 |
| 渦輪扇 | 現代客機 | 旁通比大、效率高 |
| 火箭引擎 | 太空發射 | 自帶氧化劑、真空中可用 |
| 電推進 | 深空探測 | 離子推進、高比衝 |

## 太空任務階段

```
發射 → 軌道插入 → 在軌運行 → 脫軌 → 再入 → 著陸/濺落
```

## 新興趨勢

- 可重複使用火箭 (SpaceX Falcon 9, Starship)
- 小型衛星星系 (Starlink, OneWeb)
- 太空旅遊與商業太空站
- 高超音速飛行 (Mach 5+)

> 💡 航太工程是在最極端的環境中追求最高的可靠性——「失敗不是選項」不是口號，是設計要求。
