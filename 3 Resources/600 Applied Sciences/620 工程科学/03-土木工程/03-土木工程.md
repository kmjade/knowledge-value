# 03 — 土木工程

> 土木工程設計和建造基礎設施——橋樑、道路、水壩、建築、給排水系統。

## 核心領域

| 領域 | 內容 |
|------|------|
| **結構工程** | 建築、橋樑的設計分析 |
| **大地工程** | 土壤、岩石、基礎 |
| **水利工程** | 水壩、防洪、灌溉 |
| **交通工程** | 道路、鐵路、機場 |
| **環境工程** | 給水、汙水、廢棄物處理 |
| **測量工程** | 地形測繪、GIS、GPS |

## 結構分析

| 荷載 | 說明 |
|------|------|
| 靜載重 | 結構自身重量 |
| 活載重 | 人員、車輛、家具 |
| 風載重 | 風壓、風力 |
| 地震力 | 地表加速度 |
| 溫度 | 熱脹冷縮 |

## 主要材料

| 材料 | 特性 |
|------|------|
| 混凝土 | 抗壓強、抗拉弱、耐久 |
| 鋼筋 | 抗拉強、韌性佳 |
| 鋼結構 | 輕、強、可回收 |
| 木材 | 可再生、碳儲存 |
| FRP | 輕、耐腐蝕 |

## 基礎設施生命週期

```
規劃 → 設計 → 施工 → 營運維護 → 除役/更新
```

> 💡 土木工程是「留下來的工程」——橋樑和建築物可能比設計者存在更久。責任與影響超越當下。
