# 04-自动化与机器人

> 自动化和机器人技术将人类从重复性、危险性劳动中解放——从工厂机械臂到 Roomba 扫地机器人，从自动驾驶到无人机。

## 自动化的层次

| 层次 | 说明 | 例子 |
|------|------|------|
| **固定自动化** | 做一件事，高速大批量 | 汽车焊装线 |
| **可编程自动化** | 可重编程做不同产品 | CNC 机床 |
| **柔性自动化** | 自动切换产品，无需停机 | FMS (柔性制造系统) |
| **智能自动化** | AI 驱动，适应未知情况 | 自适应机器人 |

## 工业机器人

| 类型 | 结构 | 典型应用 |
|------|------|----------|
| **关节型 (6 轴)** | 多关节臂 | 焊接、喷涂、搬运 |
| **SCARA** | 水平关节 | 装配、PCB 贴装 |
| **直角坐标** | XYZ 三直线轴 | 点胶、3D 打印 |
| **并联 (Delta)** | 多臂并联 | 高速分拣、包装 |
| **协作机器人 (Cobot)** | 力控+安全设计 | 与人并肩工作 — 详见 [[02-协作机器人\|工程学·Cobot 深度]] |

## 控制系统

| 概念 | 说明 |
|------|------|
| **开环控制** | 给定输入→执行——不做反馈修正 |
| **闭环控制** | 测量输出→与目标比较→调整输入 |
| **PID 控制** | 比例(P)+积分(I)+微分(D)——工业标准 |
| **模型预测控制 (MPC)** | 用系统模型预测未来状态→优化控制 |

### 自动驾驶的 SAE 等级

| 等级 | 名称 | 谁驾驶 |
|------|------|--------|
| **L0** | 无自动化 | 人类 |
| **L1** | 驾驶辅助 | 人类（辅助巡航/车道保持） |
| **L2** | 部分自动化 | 人类（但车可同时控制油门+转向） |
| **L3** | 有条件自动化 | 车（人类需准备接管） |
| **L4** | 高度自动化 | 车（限定区域） |
| **L5** | 完全自动化 | 车（任何地方、任何条件） |

> 2026 年：L2 普及，L3 刚起步 (Mercedes Drive Pilot)，L4 在限定区域运营 (Waymo, 百度 Apollo)，L5 仍未实现。
