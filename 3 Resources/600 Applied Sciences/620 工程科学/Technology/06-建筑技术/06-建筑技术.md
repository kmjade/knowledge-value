# 06-建筑技术

> 建筑技术是人类最古老的技术之一——从洞穴到摩天大楼，从土坯到碳纤维。建筑消耗全球 40% 的能源和原材料。

## 高层建筑的技术挑战

| 挑战 | 解决方案 |
|------|----------|
| 重力 | 高强度混凝土+钢框架 |
| 风 | 调谐质量阻尼器 (台北101的660吨金色球) |
| 地震 | 基础隔震、耗能支撑、延性设计 |
| 电梯 | 双层轿厢、智能调度、Sky Lobby |
| 消防 | 加压楼梯间、避难层、主动防火 |

### 超高层建筑的"钱德勒曲线"

```
每平方米造价
    ↑
    │                ┌────── 超高层附加成本
    │          ┌────┘
    │    ┌────┘
    │   /  正常建筑+ 高层附加成本  + 超高层额外成本
    └──────────────────────────→ 层数
```

## 建筑材料革命

| 材料 | 创新 |
|------|------|
| 超高强度混凝土 (UHPC) | 抗压 >150 MPa（普通混凝土 ~30 MPa） |
| 工程竹/CLT | 高层木结构——18 层木结构建筑 (挪威 Mjøstårnet) |
| 自修复混凝土 | 含细菌孢子的混凝土——裂缝自动修复 |
| 透明铝 (ALON) | 防弹、透明——下一代幕墙 |
| 气凝胶隔热 | 世界最轻固体——R 值远超常规保温材料 |

## 智能建筑

| 系统 | 功能 |
|------|------|
| **BMS (楼宇管理系统)** | 监控和控制 HVAC、照明、安防 |
| **智能照明** | 占空感应+日光采集 |
| **预测性维护** | 传感器监测设备状态→故障前预警 |
| **数字孪生** | 建筑的实时虚拟镜像 |

## 绿色建筑

| 认证 | 说明 |
|------|------|
| **LEED** | 美国——全球最广泛 |
| **BREEAM** | 英国——最古老 |
| **中国绿色建筑评价标准** | 三星制 |
| **WELL** | 聚焦健康和福祉（而非环境） |
| **Passivhaus** | 超低能耗——几乎不需主动供暖 |

### 净零建筑

| 策略 | 说明 |
|------|------|
| 被动设计 | 朝向、自然通风、遮阳 |
| 超保温围护 | 高性能窗户+墙体保温 |
| 高效设备 | 热泵、LED、能量回收通风 |
| 可再生能源 | 光伏、地源热泵 |
| 智能控制 | 只在需要时、需要处使用能源 |

## 模块化建筑

工厂预制 → 现场拼装。优点：更快、更可控质量、减少浪费。缺点：运输限制、设计灵活性降低。
