# 02-信息技术

> 信息技术 (IT) 是处理、存储和传输信息的科技——从 1940 年代的电子管计算机到今天的云计算和智能手机。

## 计算的演进

| 时代 | 代表 | 关键变化 |
|------|------|----------|
| 大型机 (1950s-70s) | IBM System/360 | 计算→机构 |
| 个人电脑 (1980s-90s) | Apple II, IBM PC | 计算→桌面 |
| 互联网 (1990s-2000s) | Web, 宽带 | 计算→连接 |
| 移动 (2007-) | iPhone, Android | 计算→口袋 |
| 云计算 (2010s-) | AWS, 阿里云 | 计算→水电般的基础设施 |
| AI 时代 (2020s-) | LLM, AI Agent | 计算→认知 |

## 软件

| 类型 | 说明 | 例子 |
|------|------|------|
| **系统软件** | 管理硬件和基础功能 | 操作系统、驱动程序 |
| **应用软件** | 完成特定任务 | Word、微信、Chrome |
| **中间件** | 连接不同的软件 | 数据库、消息队列 |
| **嵌入式软件** | 运行在设备内部 | 汽车 ECU、洗衣机控制 |

### 开源 vs 闭源

| 维度 | 开源 | 闭源 |
|------|------|------|
| 代码 | 公开 | 保密 |
| 商业模式 | 服务/支持/云 | 许可证 |
| 安全 | 更多人审查 | 安全靠隐蔽 |
| 例子 | Linux, PostgreSQL, VS Code | Windows, Oracle, Photoshop |

## 云计算

| 模式 | 含义 | 例子 |
|------|------|------|
| **IaaS** | 租用虚拟机和存储 | AWS EC2 |
| **PaaS** | 租用开发和部署平台 | Vercel, Heroku |
| **SaaS** | 租用软件 | Gmail, Notion |
| 私有云 | 自己的云基础设施 | 企业内部云 |
| 混合云 | 私有+公有混合 | 常见于大型企业 |

## 数据

| 概念 | 说明 |
|------|------|
| 结构化数据 | 行列格式——数据库、Excel |
| 非结构化数据 | 文本、图片、视频——占 80%+ |
| 大数据 (3V) | Volume (数量) + Velocity (速度) + Variety (多样性) |
| 数据湖 | 以原始格式存储所有数据 |
