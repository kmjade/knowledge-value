# 07 — 溫病條辨

> 吳鞠通《溫病條辨》（1798）是溫病學的集大成之作，系統整理了三焦辨證下的理法方藥。

## 結構

| 卷 | 內容 |
|----|------|
| 卷一·上焦篇 | 上焦溫病（肺、心包） |
| 卷二·中焦篇 | 中焦溫病（脾胃、腸） |
| 卷三·下焦篇 | 下焦溫病（肝腎） |
| 卷四·雜說 | 理論闡發 |
| 卷五·解產難 | 產後溫病 |
| 卷六·解兒難 | 小兒溫病 |

## 三焦治則

| 焦 | 治則 | 說明 |
|----|------|------|
| **上焦** | 如羽，非輕不舉 | 用輕清宣透之品 |
| **中焦** | 如衡，非平不安 | 調整氣機升降 |
| **下焦** | 如權，非重不沉 | 用滋填重鎮之品 |

## 經典方劑來源

| 方劑 | 出處 | 功效 |
|------|------|------|
| 銀翹散 | 上焦篇 | 辛涼平劑 |
| 桑菊飲 | 上焦篇 | 辛涼輕劑 |
| 清營湯 | 上焦篇（營分） | 清營透熱 |
| 三仁湯 | 上焦篇（濕溫） | 宣暢氣機、清熱利濕 |
| 安宮牛黃丸 | 上焦篇 | 清熱開竅 |

> 💡 溫病條辨的價值在於「系統化」——它把散在的溫病理論整理成完整的辨證論治體系，使溫病學真正成為一門獨立的學科。
