# 07-主要厂商

## 全球格局

| 厂商 | 产品 | 部署 | 目标客户 | 年收入 |
|------|------|------|----------|--------|
| **SAP** | S/4HANA | 云/本地 | 大型+中型 | ~$30B |
| **Oracle** | Fusion Cloud | 云 | 大型+中型 | ERP ~$15B |
| **Microsoft** | Dynamics 365 | 云 | 中型 | ~$20B (Dynamics) |
| **Infor** | CloudSuite | 云 | 行业垂直 | ~$3B |
| **Workday** | Workday | 云 | HR+财务 | ~$7B |
| **用友** | U8+/YonSuite | 云/本地 | 大中小 | ~$1.5B |
| **金蝶** | 云星空/苍穹 | 云/本地 | 大中小 | ~$1B |

## 选型参考

| 企业规模 | 推荐 |
|----------|------|
| 超大型 (集团/跨国) | SAP S/4HANA, Oracle Fusion |
| 大型 (全国) | SAP, Oracle, 用友 NC |
| 中型 | Dynamics 365, 用友 U8+, 金蝶云星空 |
| 小型 | 用友畅捷通, 金蝶精斗云 |
