# Cobot 知识库总览

> 协作机器人 (Cobot) 是专为人机安全协作而设计的机器人——不需要围栏，通过力控、碰撞检测和速度限制，与人共享工作空间。

## Cobot vs 传统工业机器人

| | 传统 | Cobot |
|---|------|-------|
| 安全 | 围栏隔离 | 力/速限制→直接接触安全 |
| 编程 | 专业程序员 | 拖动示教/图形化 |
| 部署 | 数周 | 数小时 |
| 负载 | 10-1000+ kg | 3-25 kg |
| 成本 | ¥200K-1M+ | ¥50K-300K |
