# 野生动物研究与狩猎渔业

> **UDC 639**: Hunting. Fishing. Wildlife management

## 核心领域

| 分支 | 内容 |
|------|------|
| **野生动物管理** | 种群监测、栖息地管理、承载力估算 |
| **狩猎管理** | 可持续猎获量、狩猎许可、传统狩猎权利 |
| **渔业管理** | 最大可持续产量（MSY）、配额、禁渔期 |
| **保护生物学** | 濒危物种恢复、迁地保护、重引入 |
| **人兽冲突** | 捕食家畜、毁坏作物——缓解策略 |

## 核心概念

### 最大可持续产量（MSY）

渔业和狩猎管理的核心模型——在种群能自我补充的前提下可以收获的最大量。

$$MSY = \frac{rK}{4}$$

其中 r = 内禀增长率，K = 环境承载力。

**局限**: MSY 假设环境恒定——现实中环境波动和气候变化使这个模型不可靠。

### 野生动物管理的三种范式

| 范式 | 核心理念 | 时期 |
|------|----------|------|
| **控制自然** | 消灭害兽、最大化可猎物种 | 19–20 世纪初 |
| **科学管理** | 基于种群生态学的定量管理 | 1930s–1980s |
| **生态系统管理** | 整体生态系统健康，生物多样性优先 | 1990s–现在 |

### 保护 vs 利用的张力

野生动物的双重属性——既是**保护对象**又是**利用资源**：

| 立场 | 主张 |
|------|------|
| 纯粹保护 | 野生动物应有不受人类干预的存在权 |
| 可持续利用 | 合理利用为保护提供资金和社会支持（"use it or lose it"） |
| 社区共管 | 将管理权交还当地社区——他们承担成本也应获得利益 |

## 当代挑战

- **第六次大灭绝**: 当前物种灭绝速率是背景值的 100–1000 倍
- **栖息地破碎化**: 农业扩张、城市化——野生动物岛状化
- **气候变化**: 物候不匹配——迁徙时间与食物高峰不再同步
- **非法贸易**: 象牙、犀角、穿山甲——跨国犯罪网络
- **人兽共患病**: COVID-19 凸显了野生动物-人类界面的疾病风险

## 在三层模型中的位置

野生动物研究体现了三层模型的**边界张力**：

- **科学层**: 种群生态学、行为学——但野外研究难以控制变量
- **知识层**: 命题知识（种群模型）+ 传统生态知识（猎人的默会知识）
- **学科层**: "野生动物管理"是生态学、兽医学和公共政策的交叉——尚未完全建制化
