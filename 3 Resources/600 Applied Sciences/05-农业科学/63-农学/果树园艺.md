# 果树园艺

> **UDC 634**: Fruit growing — 果树栽培、育种、果园管理

果树园艺是农业中**时间跨度最长**的分支——种一季水稻只需 4 个月，但种一棵苹果树要等 3-5 年才能结果，盛果期可持续 20-50 年。这种长周期决定了果树园艺在经济、生态和知识上的独特逻辑。

## 核心领域

| 分支 | 内容 |
|------|------|
| **果树育种** | 芽变选种、杂交育种、砧木选育 |
| **果园建立** | 园址选择、定植密度、树形管理（纺锤形、开心形） |
| **花果管理** | 授粉、疏花疏果、果实套袋 |
| **土肥水管理** | 果园生草、水肥一体化 |
| **采收与采后** | 成熟度判断、预冷、冷链、气调贮藏 |

## 主要果树分类

### 按果实类型

| 类别 | 代表树种 | 特点 |
|------|----------|------|
| **仁果类** | 苹果、梨 | 假果，由花托发育——耐贮藏 |
| **核果类** | 桃、李、杏、樱桃 | 真果，内果皮木质化——不耐贮 |
| **浆果类** | 葡萄、猕猴桃、蓝莓 | 多汁柔软——极不耐贮 |
| **坚果类** | 核桃、板栗、榛子 | 可食部分为种子——耐贮 |
| **柑橘类** | 橙、橘、柚、柠檬 | 柑果，特殊结构——较耐贮 |
| **热带果树** | 香蕉、芒果、荔枝、龙眼 | 对低温敏感 |

## 核心概念

### 砧木——果树的"地下半身"

果树几乎全部采用**嫁接繁殖**——优良品种（接穗）嫁接到砧木上。砧木决定了：

| 砧木功能 | 说明 |
|----------|------|
| **矮化** | 控制树体大小——M9 砧木使苹果树从 6m 降到 2.5m，密植高产 |
| **抗逆** | 抗根腐病、抗线虫、耐盐碱 |
| **早果** | 缩短童期——从定植到结果的时间 |

**苹果矮化密植革命**: M9 矮化砧 + 纺锤形树形 → 密度从 100 株/亩提升到 200-400 株/亩 → 早期产量提升 3-5 倍。这是现代果树产业的基础。

### 童期——果树特有的生命阶段

从种子萌发到首次开花需要经历**童期**（juvenility）——无法通过任何栽培措施诱导开花：

| 树种 | 童期年限 |
|------|----------|
| 桃 | 2-4 年 |
| 苹果 | 4-8 年 |
| 柑橘 | 5-10 年 |

**育种含义**: 果树育种极其耗时——一个杂交组合从播种到看到第一个果实可能等 5-10 年。这是果树育种落后于大田作物的根本原因。

### 大小年现象

果树产量在年份之间剧烈波动的现象——今年结太多（大年），明年就很少（小年）：

- **原因**: 大年消耗过多养分，花芽分化不足——生理性自我调节
- **调控**: 疏花疏果——在大年主动减少果实数量，换取来年的稳定产量
- **经济学**: 大小年造成价格剧烈波动——大年果贱伤农，小年价高无货

## 当代挑战

- **劳动力**: 修剪、疏果、套袋、采摘——果树管理是劳动密集型的巅峰，"谁来务果"比"谁来种粮"更严峻
- **气候异常**: 倒春寒→花期冻害（2023 年中国北方苹果花期降雪，减产 30-50%）；暖冬→需冷量不足→休眠不充分
- **连作障碍**: 老果园更新时——重茬导致根系生长受阻（自毒物质 + 微生物群落失衡）
- **品种同质化**: 富士苹果占中国 70% 面积——一旦爆发针对富士的新病害，后果灾难性
- **果品安全**: 套袋与"不套袋裸果"之争——前者防病虫但增加成本和白色污染，后者更自然但农残风险高

## 在三层模型中的位置

果树园艺的长周期特性使其成为检验知识积累性质的独特案例：

- **科学层**: 果树生理学揭示了童期调控、花芽分化的分子机制——但实验室发现转化为田间实践需数十年
- **知识层**: 修剪是最高度的程序性知识——如何"看树修剪"无法从书本学会，必须师徒传承
- **学科层**: 果树学是园艺学的核心分支——但各地品种、气候、栽培传统的巨大差异使得"通用知识"的边界很窄
