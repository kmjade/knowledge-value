# 大田作物

> **UDC 633**: Field crops — 谷物、纤维、糖料、油料、刺激类作物

大田作物是**人类卡路里的主引擎**。全球每年收获约 30 亿吨谷物——水稻、小麦、玉米三种作物提供了人类热量摄入的 ~50%。大田作物的丰歉直接关系粮食安全与社会稳定。

## 核心领域

| 分支 | 内容 |
|------|------|
| **谷物** | 水稻、小麦、玉米、大麦、高粱、小米 |
| **纤维作物** | 棉花、麻类（亚麻、大麻、黄麻） |
| **糖料作物** | 甘蔗、甜菜 |
| **油料作物** | 大豆、油菜、花生、向日葵、油棕 |
| **豆类作物** | 食用豆——大豆、绿豆、鹰嘴豆、扁豆 |
| **刺激类作物** | 咖啡、茶、可可、烟草 |

## 核心作物画像

### 三大主粮

| 作物 | 全球年产量 | 核心产区 | 关键特征 |
|------|-----------|----------|----------|
| **玉米** | ~12 亿吨 | 美国中西部、中国东北 | C4 光合、高产、饲用为主 |
| **小麦** | ~8 亿吨 | 黑海、北美大平原、中国华北 | C3 光合、广泛适应性、主粮 |
| **水稻** | ~8 亿吨 | 南亚、东南亚、东亚 | 水生环境、全球半数人口主粮 |

### 玉米：现代农业的缩影

- **杂交种革命**: 1908 年 Shull 发现杂种优势——现代玉米育种的开端。单产从 1930s 的 1.5 t/ha 提升到现在的 10+ t/ha
- **利用方式**: 全球 60% 用作饲料，20% 用作生物燃料（美国），仅 ~10% 直接食用
- **C4 优势**: 比 C3 作物（小麦、水稻）光合同化效率高 ~50%

### 大豆：从东方到全球

- 原产中国——5000 年栽培史。现在全球最大生产国是美国和巴西
- 蛋白质含量 ~40%（远高于谷物），油分 ~20%——双重价值
- 中国年进口 ~1 亿吨——占全球大豆贸易的 60%，主要用作饲料蛋白（豆粕）和食用油

## 核心概念

### 绿色革命

1960s-70s 以小麦和水稻为核心的农业技术变革：

| 要素 | 贡献 |
|------|------|
| **矮秆品种** | 诺曼·博洛格培育的半矮秆小麦——高产不倒伏 |
| **化肥** | 氮磷钾的大量投入——产量翻倍 |
| **灌溉** | 保障水供应——解除旱灾约束 |
| **农药** | 防止病虫夺取增产成果 |

**成就**: 印度小麦产量 1965-1972 增长 3 倍，从饥荒边缘到自给自足。博洛格获 1970 年诺贝尔和平奖。

**代价**: 地下水耗竭（印度旁遮普）、化肥面源污染、品种单一化——生物多样性丧失。

### 作物驯化综合征

野生植物→作物的转变涉及一系列共同的性状变化：

| 驯化性状 | 野生祖先 | 栽培种 |
|----------|----------|--------|
| 落粒性 | 成熟即落（散播种子的策略） | 不落粒（便于收获） |
| 种子休眠 | 深度休眠（分批萌发避险） | 休眠弱（出苗整齐） |
| 籽粒大小 | 小 | 大 |
| 分蘖 | 多而乱 | 少而齐 |

### 种植制度

| 制度 | 描述 | 实例 |
|------|------|------|
| **连作** | 同一块地连续种同一作物 | 美国玉米带——但病害积累严重 |
| **轮作** | 按顺序轮替种植不同作物 | 玉米→大豆→小麦（北美经典轮作） |
| **间作** | 同时种两种以上作物 | 玉米 + 大豆（中国东北传统） |
| **复种** | 一年种多季 | 稻—稻—麦（华南） |

## 当代挑战

- **产量天花板**: 主要谷物遗传产量潜力的增速在放缓——过去 30 年年均增 1%，未来可能降到 0.5%
- **氮肥困境**: 世界一半人口的蛋白质来自哈伯-博斯法合成的氨——但其能耗占全球能源的 1-2%，排放大量 CO₂
- **气候冲击**: 高温导致水稻结实率下降；干旱使小麦灌浆期缩短——每升温 1°C，小麦减产 ~6%
- **生物燃料竞争**: 美国 40% 玉米用于乙醇——"用粮食烧汽车"的伦理争议
- **营养转型**: 全球饮食结构从谷物向肉蛋奶转移——但生产 1kg 牛肉需要 ~7kg 谷物

## 在三层模型中的位置

大田作物体现了农业中**科学与实践的深层纠缠**：

- **科学层**: 植物生理学、遗传学、育种学——揭示了产量形成的机制
- **知识层**: 育种家的"眼力"（对优良单株的直觉选择）——大量默会知识尚未被基因组选择替代
- **学科层**: 作物科学是最成熟的农学分支——CIMMYT（玉米小麦改良中心）、IRRI（国际水稻研究所）等国际机构引领全球育种
