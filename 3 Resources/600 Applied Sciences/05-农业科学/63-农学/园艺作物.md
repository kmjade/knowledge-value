# 园艺作物

> **UDC 635**: Garden plants, horticulture — 蔬菜、花卉、观赏植物

园艺作物是大田作物的**精细化版本**——单位面积产值是大田作物的 5-20 倍，但需要的管理精细度也同等放大。蔬菜提供维生素和微量营养，花卉满足人类的美学需求——二者共同构成了"小面积、高价值"的农业模式。

## 核心领域

| 分支 | 内容 |
|------|------|
| **蔬菜栽培** | 设施蔬菜（温室/大棚）、露地蔬菜、芽苗菜 |
| **花卉栽培** | 切花、盆栽、花坛花卉、草坪 |
| **设施园艺** | 温室设计、环境调控（温光水气）、无土栽培 |
| **种子种苗** | 蔬菜种子生产、穴盘育苗、嫁接育苗 |
| **采后处理** | 预冷、分级包装、冷链物流 |

## 蔬菜分类

### 按食用部位

| 类型 | 代表 | 特点 |
|------|------|------|
| **叶菜类** | 白菜、菠菜、生菜、芹菜 | 生长周期短（20-60 天）、不耐贮 |
| **果菜类** | 番茄、辣椒、茄子、黄瓜 | 需授粉/单性结实、价格波动大 |
| **根茎类** | 萝卜、胡萝卜、马铃薯、甘薯 | 耐贮性好、机械化程度高 |
| **花菜类** | 花椰菜、西兰花、黄花菜 | 采收窗口窄——错过一天就可能开花散球 |
| **葱蒜类** | 大葱、洋葱、大蒜、韭菜 | 含硫化合物——特殊风味和杀菌作用 |

### 设施栽培方式

| 类型 | 环境控制 | 投入成本 | 适用 |
|------|----------|----------|------|
| **露地** | 无 | 低 | 大宗蔬菜 |
| **地膜覆盖** | 增温、保墒 | 极低 | 早春提早上市 |
| **塑料大棚** | 半封闭、保温 | 中 | 春秋延后/提早 |
| **日光温室** | 被动蓄热 | 高 | 北方冬季生产 |
| **连栋温室** | 主动调控（加热、降温、补光） | 极高 | 高端蔬果、花卉 |
| **植物工厂** | 全人工环境（LED + 营养液） | 极高 | 叶菜、育苗 |

## 核心概念

### 设施园艺的"环境三角"

设施园艺的核心是在封闭/半封闭空间中维持三个变量的适度平衡：

```
      温度
       /\
      /  \
     /    \
    / 作   \
   /  物   \
  /__________\
光照 ──────── 湿度
      + CO₂
```

- **温度**: 绝大多数蔬菜适宜 15-30°C——低于 10°C 停止生长，高于 35°C 热害
- **光照**: 冬季弱光是北方设施蔬菜的首要限制因子——补光灯成本高
- **湿度**: 过高引发灰霉病、霜霉病——过低抑制光合——通风与保温的矛盾

### 连作障碍

同一块地（特别是设施内）连续种植同种或同科蔬菜后，出现的生长抑制问题：

| 原因 | 机制 | 对策 |
|------|------|------|
| **自毒物质** | 根系分泌物在土壤中积累 | 轮作、嫁接、活性炭吸附 |
| **土传病害** | 病原菌（枯萎病、根结线虫）累积 | 土壤消毒（蒸汽/化学）、抗性砧木 |
| **养分失衡** | 特定元素被持续消耗 | 测土配方、有机肥 |
| **微生物群落恶化** | 有益菌减少、病原菌增多 | 生物有机肥、微生物菌剂 |

**设施内连作障碍比露地严重 3-5 倍**——因为温室内的淋洗和生物降解更弱。

### 蔬菜嫁接

将优质接穗嫁接到抗病砧木上——日本和韩国的设施蔬菜几乎 100% 嫁接：

| 作物 | 常用砧木 | 目的 |
|------|----------|------|
| 黄瓜 | 黑籽南瓜 | 抗枯萎病、耐低温 |
| 西瓜 | 葫芦、南瓜 | 抗枯萎病、提高品质 |
| 番茄 | 抗性番茄品种 | 抗青枯病、根结线虫 |
| 茄子 | 托鲁巴姆（野生茄） | 抗黄萎病、增强长势 |

## 花卉：美学经济

花卉是农业中唯一以**观赏价值**为核心的产品：

| 类别 | 代表 | 全球主要产区 |
|------|------|-------------|
| **切花** | 玫瑰、百合、菊花、康乃馨 | 荷兰、哥伦比亚、肯尼亚、中国云南 |
| **盆栽** | 蝴蝶兰、红掌、仙客来 | 荷兰、中国、日本 |
| **球根** | 郁金香、风信子、百合 | 荷兰（占全球 ~80%） |

**花卉物流**: 切花从肯尼亚早上采摘到阿姆斯特丹拍卖再到伦敦花店——全程 <48 小时、4°C 冷链、全程无断链。这是全球最精密的农产品物流。

## 当代挑战

- **能源成本**: 北方温室冬季加温——燃料成本占运营成本的 30-50%，"种一季菜不够烧煤钱"
- **食品安全**: 叶菜类蔬菜的农药残留——短周期 + 直接食用 = 对农残极度敏感
- **劳动力**: 蔬菜定植、整枝、采收——高度依赖手工，尤其是果菜类
- **种业安全**: 高端蔬菜种子（西兰花、胡萝卜、菠菜）高度依赖进口品种——"洋种子"危机
- **植物工厂泡沫**: 资本涌入植物工厂——建得起，但 LED 电费 + 折旧——绝大多数不盈利

## 在三层模型中的位置

园艺作物的知识特征——高度精细化和情境依赖：

- **科学层**: 植物生理学揭示了温光水气的响应曲线——但"一个温室一个气候"，理论只能指示方向
- **知识层**: 种菜能手的"手感"——浇水时机、通风量的判断——是典型的默会知识，AI 短期难以替代
- **学科层**: 园艺学科建制完善——但设施园艺是园艺+建筑+自动化+计算机的交叉，学科边界模糊
