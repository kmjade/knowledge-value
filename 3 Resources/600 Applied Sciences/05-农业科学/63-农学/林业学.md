# 林业学

> **UDC 630**: Forestry — 森林培育、经营、保护与可持续利用

## 核心领域

| 分支 | 内容 |
|------|------|
| **森林培育**（Silviculture） | 人工更新、抚育、间伐——控制森林的建立、组成和生长 |
| **森林经营** | 森林资源规划、采伐量计算、永续利用 |
| **森林保护** | 防火、病虫害防治、气候变化适应 |
| **森林生态** | 森林生态系统的结构、功能和演替 |
| **林产品** | 木材、松脂、林下经济 |

## 核心概念

### 永续利用原则

林业的核心理念——森林应该在采伐后可持续地再生。这是可持续性概念的最早系统表达之一（18 世纪德国林业科学）。

### 森林的多功能

现代林业不只看木材产量：

| 功能 | 说明 |
|------|------|
| **生产** | 木材、薪材、非木林产品 |
| **生态** | 碳汇、水土保持、生物多样性 |
| **社会** | 游憩、文化、景观 |

### 人工林 vs 天然林经营

| 人工林 | 天然林经营 |
|--------|----------|
| 高产、短轮伐期 | 生物多样性高 |
| 单一树种风险大 | 生态系统服务完整 |
| 适合工业原料需求 | 适合保护性目标 |

## 当代挑战

- **气候变化**: 树种分布北移、病虫害范围扩大、火灾频率增加
- **生物多样性**: 人工林替代天然林导致的物种损失
- **碳市场**: 森林碳汇作为气候解决方案——但 MRV（监测、报告、核查）困难
