# 昆虫养殖

> **UDC 638**: Keeping, breeding and management of insects — 蜜蜂、蚕及其他经济昆虫

昆虫养殖是农业中**最小众但最精妙**的分支——研究的不是大型家畜，而是体型微小、生命周期复杂、行为高度特化的昆虫。养蜂和养蚕是人类历史上最早建立**跨物种共生关系**的实践——数千年驯化，却从未完全"驯服"。

## 核心领域

| 分支 | 内容 |
|------|------|
| **养蜂** | 蜜蜂生物学、蜂群管理、蜂蜜生产、授粉服务 |
| **养蚕** | 桑树栽培、蚕种繁育、蚕茧生产、缫丝 |
| **其他经济昆虫** | 紫胶虫、五倍子蚜、胭脂虫（色素） |
| **新兴昆虫养殖** | 黑水虻（饲料蛋白）、黄粉虫（食品）、蟋蟀 |

## 养蜂

### 蜜蜂的社会结构

蜜蜂是**真社会性昆虫**——一个蜂群是一个超级有机体：

| 成员 | 数量 | 寿命 | 职能 |
|------|------|------|------|
| **蜂王** | 1 只 | 2-5 年 | 唯一产卵者——日产卵 1500-2000 粒 |
| **工蜂** | 2-6 万只 | 夏季 4-6 周，冬季 4-5 月 | 内勤（哺育、酿蜜）→外勤（采集、守卫） |
| **雄蜂** | 数百只 | 交配后即死或秋季被驱逐 | 唯一功能——与处女王交配 |

### 蜂产品的多层次价值

| 产品 | 来源 | 价值 |
|------|------|------|
| **蜂蜜** | 花蜜经蜜蜂酿造 | 直接食品——全球年产量 ~180 万吨 |
| **蜂花粉** | 采集的花粉团 | 高蛋白、"全营养食品" |
| **蜂王浆** | 工蜂咽下腺分泌物 | 高端保健品——蜂王终身食用 |
| **蜂胶** | 植物树脂+蜂蜡混合物 | 天然抗生素、保健品 |
| **蜂蜡** | 工蜂蜡腺分泌 | 化妆品、蜡烛、药品辅料 |
| **蜂毒** | 毒囊分泌物 | 蜂针疗法——关节炎 |

### 授粉服务：比蜂蜜更值钱的产品

蜂蜜只是养蜂的**副产品**——蜜蜂真正的经济价值在于**授粉**：

| 作物 | 依赖蜜蜂授粉程度 | 无蜂减产 |
|------|-----------------|----------|
| 苹果 | 90% | -80% |
| 杏仁 | 100% | -100% |
| 蓝莓 | 100% | -90% |
| 油菜 | 30% | -20%（但含油率下降） |

**美国加州杏仁产业**: 每年 2 月需要 180 万群蜜蜂授粉——全美 60% 的商业蜂群被卡车运到加州。授粉合同费用 ~$200/群——养蜂人的主要收入来源已从卖蜜转向"出租蜜蜂"。

### 蜂群崩溃症（CCD）

2006 年首次报道——工蜂突然消失，留下蜂王和少量幼蜂。至今没有单一原因确定：

| 假说 | 证据 |
|------|------|
| 新烟碱类杀虫剂 | 亚致死剂量影响蜜蜂导航——"回不了家" |
| 瓦螨（Varroa destructor） | 吸食蜜蜂体液——传播病毒 |
| 营养不足 | 单一作物大面积开花——"只有面包没有菜" |
| 多重胁迫交互 | 农药 + 寄生虫 + 营养不良 → 超过蜂群承受阈值 |

## 养蚕

### 蚕的生命周期

```
卵 → 幼虫(5龄) → 蛹 → 蛾
       ↑
    约 25-28 天
    蚕食桑叶 → 吐丝结茧
```

一只蚕在 5 龄后期用 **2-3 天**吐出一根连续不断的丝——长度 **1000-1500 米**。这根丝是蚕的一生——蚕蛹在茧内如果未被杀死就会羽化成蛾，咬破蚕丝钻出——破坏丝的连续性。

**"蚕的一生是一种化学"**：蚕食入桑叶的蛋白质 → 在丝腺中重组为丝素蛋白 → 以液体形式吐出 → 接触空气立刻固化为丝。

### 丝绸之路的生物学基础

| 要素 | 说明 |
|------|------|
| **桑树** | 蚕的唯一饲料——"桑基鱼塘"是珠三角的经典生态农业模式 |
| **蚕种** | 中国的多化性蚕种（一年多代）vs 欧洲的一化性（一年一代） |
| **缫丝** | 煮茧→找丝头→6-8 根单丝并合→生丝——纯手工技艺，机械替代困难 |

**当代地位**: 全球生丝年产量 ~10 万吨——中国占 ~70%，印度 ~15%。但丝绸在全球纤维市场占比 <0.2%——这是极少数中国依然主导的奢侈品农业。

## 新兴昆虫养殖

### 昆虫作为饲料和食品

| 物种 | 用途 | 优势 |
|------|------|------|
| **黑水虻** | 饲料蛋白 | 幼虫以有机废弃物为食——"垃圾变蛋白"，蛋白质含量 ~42% |
| **黄粉虫** | 食品/饲料 | 易于规模化、蛋氨酸含量高 |
| **蟋蟀** | 食品 | 传统食用昆虫——泰国、墨西哥有成熟市场 |
| **蝇蛆** | 饲料 | 繁殖速度极快——"48 小时从卵到蛆" |

**为什么昆虫**: FCR（饲料转化率）远低于传统家畜——蟋蟀 ~1.7（接近肉鸡），产生的温室气体是牛的 1/100。欧盟 2021 年批准黄粉虫作为"新型食品"——开启昆虫食品商业化的监管通道。

## 当代挑战

- **传粉昆虫危机**: 全球传粉昆虫多样性在下降——不仅是蜜蜂，野生蜂、蝴蝶、食蚜蝇都在减少
- **蜂螨抗药性**: 瓦螨对主流杀螨剂的抗性发展迅速——养蜂业的化学武器正在失效
- **养蚕业萎缩**: 中国蚕农老龄化——"年轻人不会养蚕，也不愿养蚕"——桑园面积持续下降
- **昆虫食品接受度**: 西方消费者对昆虫食品的"恶心感"——文化壁垒远大于技术壁垒
- **新兴昆虫养殖的监管空白**: 黑水虻的规模化养殖尚无完善的兽医卫生规范

## 在三层模型中的位置

昆虫养殖是**未完全建制化**的知识领域：

- **科学层**: 蜜蜂生物学（Von Frisch 的蜜蜂舞蹈语言——1973 年诺贝尔奖）、昆虫生理学——科学积累精良但规模小
- **知识层**: 养蜂人和蚕农的默会知识极为丰富——"看蜂识流蜜"、"摸茧知丝质"——正在快速流失
- **学科层**: 养蜂学/蚕学是农业院校中的"小众专业"——经常面临招生困难、被合并的风险
