# 05 — 臟象學說

> 臟象學說是研究臟腑生理功能、病理變化及其相互關係的學說。「臟」指藏於體內的内臟，「象」指表現於外的徵象。

## 核心篇章

- **素問·靈蘭秘典論**：十二臟腑功能比喻
- **素問·六節藏象論**：臟腑與自然的對應

## 五臟核心功能

| 臟 | 內經描述 | 核心功能 |
|----|---------|---------|
| **心** | 「君主之官，神明出焉」 | 主血脈、主神志 |
| **肺** | 「相傅之官，治節出焉」 | 主氣司呼吸、主宣發肅降 |
| **肝** | 「將軍之官，謀慮出焉」 | 主疏泄、主藏血 |
| **脾** | 「倉廩之官，五味出焉」 | 主運化、主統血 |
| **腎** | 「作強之官，伎巧出焉」 | 主藏精、主水、主納氣 |

## 六腑功能

| 腑 | 功能 | 特點 |
|----|------|------|
| 膽 | 貯藏排泄膽汁、主決斷 | 奇恆之腑 |
| 胃 | 受納腐熟水穀 | 後天之本 |
| 小腸 | 受盛化物、泌別清濁 | |
| 大腸 | 傳導糟粕 | |
| 膀胱 | 貯存排泄尿液 | |
| 三焦 | 通行元氣、運行水液 | 上中下三焦 |

## 臟腑關係

| 關係 | 說明 |
|------|------|
| 臟與臟 | 五行生剋（心腎相交、肝腎同源、脾胃相合） |
| 臟與腑 | 表裡關係（心合小腸、肺合大腸...） |
| 臟與竅 | 開竅（肝開竅於目、腎開竅於耳...） |

> 💡 「象」是臟象學說的精髓——觀察外在表現（面色、脈象、舌象），推斷內在臟腑的狀態。這是中醫「以外揣內」的診斷方法論基礎。
