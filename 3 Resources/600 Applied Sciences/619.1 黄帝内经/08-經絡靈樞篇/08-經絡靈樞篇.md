# 08 — 經絡（靈樞篇）

> 靈樞建立了完整的經絡系統理論——十二經脈、奇經八脈、十五絡脈，構成氣血運行的網絡。

## 核心篇章

- **靈樞·經脈**：十二經脈的循行、是動病、所生病
- **靈樞·經別**：十二經別
- **靈樞·經筋**：十二經筋
- **靈樞·脈度**：經脈長度

## 經絡的功能

| 功能 | 說明 |
|------|------|
| 運行氣血 | 「經脈者，所以行血氣而營陰陽」 |
| 聯絡臟腑 | 溝通內外、聯繫上下 |
| 感應傳導 | 傳遞針刺感應 |
| 調節平衡 | 維持人體陰陽平衡 |

## 十二經脈循行規律

| 部位 | 陰經（內側） | 陽經（外側） |
|------|------------|------------|
| 手 | 太陰（前）、厥陰（中）、少陰（後） | 陽明（前）、少陽（中）、太陽（後） |
| 足 | 太陰（前）、厥陰（中）、少陰（後） | 陽明（前）、少陽（中）、太陽（後） |

## 靈樞·經脈篇的結構

每條經脈的論述模式：
1. 循行路線（起於...終於...）
2. 是動病（經氣變動引起的病證）
3. 所生病（本經主治的病證）
4. 治療原則（實則瀉之，虛則補之）

> 💡 靈樞·經脈篇是中醫經絡學的源頭——後世所有的經絡理論和針灸實踐都建立在這一篇的基礎上。
