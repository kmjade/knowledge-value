# 06 — 業餘陶藝

> 陶藝是最原始的創作——雙手與泥土的對話，火與土的藝術。

## 成型方法

| 方法 | 難度 | 特點 |
|------|------|------|
| 手捏 | ⭐ | 不需要任何工具 |
| 泥條盤築 | ⭐ | 大型器皿 |
| 陶板成型 | ⭐⭐ | 方形器皿 |
| 拉坯 | ⭐⭐⭐ | 需要拉坯機和大量練習 |

## 從泥到器

```
成型 → 乾燥 → 素燒（~800°C）→ 上釉 → 釉燒（~1200°C）→ 成品
```

## 釉藥基礎

| 釉藥 | 效果 |
|------|------|
| 透明釉 | 顯現土色 |
| 青瓷釉 | 青綠色 |
| 天目釉 | 黑底結晶 |
| 志野釉 | 白色溫潤 |

## 新手建議

- 從手捏開始——不需要拉坯機
- 找當地陶藝教室，用他們的窯
- 第一批作品可能全軍覆沒——正常！

> 💡 陶藝教會我們接受不完美——釉藥在高溫下的流動無法完全控制，每一個作品都是獨一無二的。
