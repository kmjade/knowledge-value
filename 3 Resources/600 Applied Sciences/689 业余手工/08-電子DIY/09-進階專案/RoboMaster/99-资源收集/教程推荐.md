## 學習建議

### 學習順序

```
1. Python 基礎 → 語法、面向對象
2. SDK 入門 → 連接、基礎控制
3. 拓展模塊 → 機械臂、傳感器
4. 多機控制 → 編隊開發
5. 項目實踐 → 綜合應用
```

### 實踐方法

1. **跟著官方文檔做**：每個示例都要親自運行
2. **修改代碼嘗試**：在示例基礎上修改參數
3. **設計小項目**：如自動巡邏、目標追蹤
4. **參加比賽**：以賽促學
