### 开发板 C 型示例仓库

**Development-Board-C-Examples** 是 RoboMaster 开发板 C 型嵌入式软件教程，包含 20+ 示例：

| 示例类型 | 包含示例 |

| **传感器** | IST8310 磁力计、BMI088 IMU、OLED 显示 |
| **实时系统** | FreeRTOS 任务管理 |
| **机器人系统** | 底盘任务、云台任务、惯导系统、完整机器人 |

详细信息：[[开发板C型示例概览]]

### RoboRTS 机器人系统仓库

**RoboRTS** 是 RoboMaster AI 挑战赛机器人系统，基于 ROS 的自主移动机器人框架：

| 模块 | 功能 |
|------|------|
| roborts_base | 底层通信接口 |
| roborts_camera | 相机驱动 |
| roborts_detection | 装甲板检测与 PnP 解算 |
| roborts_localization | AMCL 定位 |
| roborts_decision | 行为树决策框架 |
| roborts_planning | A*/DWA 路径规划 |
| roborts_bringup | 系统启动包 |

**技术栈：** C++、ROS、Eigen3、OpenCV、FreeRTOS

详细信息：[[RoboRTS教程概览]]

## 快速下载

### SDK 下载

```bash
# pip 安装
pip install robomaster

# 从 GitHub 克隆
git clone https://github.com/dji-sdk/RoboMaster-SDK.git

# 从 Gitee 克隆（国内加速）
git clone https://gitee.com/xitinglin/RoboMaster-SDK.git
```

### 示例代码下载

```bash
# 下载完整示例
git clone https://github.com/dji-sdk/RoboMaster-SDK.git
cd RoboMaster-SDK/examples

# 运行示例
python 01_robot/01_get_version.py
```
