# 4. RoboMaster SDK 新手入门 - 基础篇

> [!info] 本章概述
> 介绍 RoboMaster SDK 的基本功能和第一个 SDK 程序的编写方法。

---

## 4.1 SDK 能做什么?

### 支持的产品

RoboMaster SDK（以下简称 SDK）是一套面向大疆 RoboMaster 系列产品的开发工具包。

| 产品类型 | 产品名称 |
|----------|----------|
| **EP 系列** | RoboMaster EP, RoboMaster EP Core |
| **无人机系列** | Tello EDU, Tello Talent |

### 核心功能

通过 SDK，用户可以实现在 PC 上：

| 功能类别 | 具体功能 |
|----------|----------|
| **运动控制** | 控制机器人运动（底盘、云台） |
| **传感器获取** | 获取机器人传感器的相关信息 |
| **模块控制** | 控制发射器、LED、机械臂等模块 |
| **视频流** | 获取相机视频流 |
| **音频流** | 获取麦克风音频流 |
| **智能识别** | 视觉识别、标签识别 |

## 4.2 第一个 SDK 程序

接下来从如何获取 RoboMaster SDK 的版本号来编写第一个 SDK 程序。

### 步骤 1：导入模块

首先从安装的 `robomaster` 包中导入需要的模块，这里导入包含获取 SDK 版本信息的 `version` 模块：

```python
from robomaster import version
```

### 步骤 2：获取版本号

通过 `version` 模块中的 `__version__` 属性可以获得 SDK 的版本号，并将其打印：

```python
sdk_version = version.__version__
print("sdk version:", sdk_version)
```

### 步骤 3：运行程序

运行程序，可以看到打印的结果：

```
sdk version: 0.1.1.29
```

---

### 完整示例代码

参考例程：`/examples/00_general/01_sdk_version.py` ([GitHub](https://github.com/dji-sdk/robomaster-sdk))

```python
from robomaster import version

if __name__ == "__main__":
    sdk_version = version.__version__
    print("sdk version:", sdk_version)
```

## SDK 基础概念

### 模块结构

RoboMaster SDK 采用模块化设计，主要模块如下：

| 模块 | 说明 |
|------|------|
| `robomaster.robot` | 机器人主体 |
| `robomaster.version` | SDK 版本信息 |
| `robomaster.chassis` | 底盘控制 |
| `robomaster.gimbal` | 云台控制 |
| `robomaster.blaster` | 发射器控制 |
| `robomaster.armor` | 装甲板控制 |
| `robomaster.led` | LED 灯效 |
| `robomaster.camera` | 相机控制 |
| `robomaster.vision` | 视觉识别 |

### 导入方式

```python
# 导入整个模块
import robomaster

# 导入特定模块
from robomaster import robot
from robomaster import chassis
from robomaster import gimbal

# 导入多个模块
from robomaster import robot, chassis, gimbal
```

## 快速验证安装

### 方法一：命令行验证

```bash
# 查看 SDK 版本
pip show robomaster

# 进入 Python 交互模式验证
python -c "from robomaster import version; print(version.__version__)"
```

### 方法二：Python 脚本验证

```python
# verify_sdk.py
from robomaster import version

print("=" * 40)
print("RoboMaster SDK 验证")
print("=" * 40)
print(f"SDK 版本: {version.__version__}")
print("SDK 安装成功!")
print("=" * 40)
```

## 常见问题

### 1. 模块导入失败

**错误信息：** `ModuleNotFoundError: No module named 'robomaster'`

**解决方案：**
```bash
# 安装 SDK
pip install robomaster

# 或使用国内镜像
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple robomaster
```

### 2. 版本过低

**错误信息：** 某些功能不可用

**解决方案：**
```bash
# 升级到最新版本
pip install --upgrade robomaster
```

### 3. Python 版本不兼容

**错误信息：** 运行时报错

**解决方案：**
- 确保使用 Python 3.6.6 - 3.8.9 版本
- 确保使用 64 位 Python
