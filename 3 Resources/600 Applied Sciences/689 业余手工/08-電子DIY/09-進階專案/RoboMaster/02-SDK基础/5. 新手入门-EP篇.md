# 5. RoboMaster SDK 新手入门 - EP 篇

> [!info] 本章概述
> 详细介绍如何使用 RoboMaster SDK 控制 EP 机器人，包括初始化、模块操作、查询/设置/动作类接口以及多媒体功能。

---

## 5.1 初始化机器人

在进行与机器人相关的操作之前，需要根据指定的配置初始化机器人对象。

### 步骤 1：导入模块

首先从安装的 `robomaster` 包中导入 `robot` 模块：

```python
from robomaster import robot
```

### 步骤 2：指定本地 IP（可选）

如需手动指定本地 IP 地址：

```python
robomaster.config.LOCAL_IP_STR = "192.168.2.20"
```

> [!tip] 自动获取
> 大部分情况下 SDK 能够自动获取正确的本地 IP，无需手动指定。但当 SDK 运行在多网卡同时使用的设备时，自动获取的 IP 可能不是与机器人进行连接的 IP，此时需要手动指定。

**Windows 下查看 IP：**
1. 按 `Win + R`，输入 `cmd`
2. 在 CMD 窗口中输入 `ipconfig`

### 步骤 3：创建机器人对象

创建 `Robot` 类的实例对象：

```python
ep_robot = robot.Robot()
```

### 步骤 4：初始化机器人

```python
# 使用默认配置（WiFi 直连模式 + UDP 通讯）
ep_robot.initialize()

# 手动指定连接方式
ep_robot.initialize(conn_type="sta")  # 组网模式

# 同时指定连接方式和通讯方式
ep_robot.initialize(conn_type="sta", proto_type="tcp")
```

**连接方式参数：**

| 参数值 | 说明 |
|--------|------|
| `ap` | WiFi 直连模式（默认） |
| `sta` | WiFi 组网模式 |
| `rndis` | USB 连接模式 |

**设置默认配置：**

```python
from robomaster import config

config.DEFAULT_CONN_TYPE = "sta"    # 默认连接方式
config.DEFAULT_PROTO_TYPE = "tcp"   # 默认通讯方式
```

## 5.2 获取模块对象

部分 SDK 接口属于 Robot 对象本身，可以直接调用。但一些接口属于其他模块，需要先获取相应的对象。

### 方法一：直接使用 `.` 运算符

```python
ep_led = ep_robot.led
ep_chassis = ep_robot.chassis
ep_gimbal = ep_robot.gimbal
```

### 方法二：使用 `get_module()` 方法

```python
ep_led = ep_robot.get_module("led")
ep_chassis = ep_robot.get_module("chassis")
```

### 模块对照表

| 模块名 | 获取方法 | 功能 |
|--------|----------|------|
| `led` | `ep_robot.led` | LED 灯效控制 |
| `chassis` | `ep_robot.chassis` | 底盘控制 |
| `gimbal` | `ep_robot.gimbal` | 云台控制 |
| `blaster` | `ep_robot.blaster` | 发射器控制 |
| `armor` | `ep_robot.armor` | 装甲板控制 |
| `camera` | `ep_robot.camera` | 相机控制 |
| `vision` | `ep_robot.vision` | 视觉识别 |

## 5.3 释放机器人资源

在程序的最后，应该手动释放机器人对象相关的资源：

```python
ep_robot.close()
```

> [!warning] 重要
> 为了避免一些意外错误，记得在程序的最后调用 `close()` 方法！

## 5.4 查询类接口的使用

查询类接口即数据获取类接口，用于获取机器人自身的状态信息以及传感器状态等信息。

### 5.4.1 示例一：查询机器人版本

```python
from robomaster import robot

if __name__ == '__main__':
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="ap")
    
    ep_version = ep_robot.get_version()
    print("Robot Version: {0}".format(ep_version))
    
    ep_robot.close()
```

**运行结果：**
```
Robot Version: xx.xx.xx.xx
```

> [!info] 示例文件
> `/examples/01_robot/01_get_version.py`

---

### 5.4.2 示例二：获取机器人 SN 号

```python
from robomaster import robot

if __name__ == '__main__':
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="sta")
    
    SN = ep_robot.get_sn()
    print("Robot SN:", SN)
    
    ep_robot.close()
```

> [!info] 示例文件
> `/examples/01_robot/02_get_sn.py`

## 5.5 设置类接口的使用

设置类接口可以完成对机器人相关模块的设置。

### 5.5.1 示例一：设置机器人整机运动模式

**机器人运动模式：**

| 模式 | 说明 |
|------|------|
| `FREE` | 自由模式 |
| `GIMBAL_LEAD` | 底盘跟随云台 |
| `CHASSIS_LEAD` | 云台跟随底盘 |

```python
from robomaster import robot

if __name__ == '__main__':
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="ap")
    
    # 设置为底盘跟随云台模式
    ep_robot.set_robot_mode(mode=robot.GIMBAL_LEAD)
    
    ep_robot.close()
```

> [!info] 示例文件
> `/examples/01_robot/09_set_mode.py`

---

### 5.5.2 示例二：设置机器人装甲灯

```python
import time
from robomaster import robot
from robomaster import led

if __name__ == '__main__':
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="sta")
    
    ep_led = ep_robot.led
    
    # 设置所有装甲灯为红色常亮
    ep_led.set_led(comp=led.COMP_ALL, r=255, g=0, b=0, effect=led.EFFECT_ON)
    
    time.sleep(2)
    ep_robot.close()
```

**LED 参数说明：**

| 参数 | 说明 |
|------|------|
| `comp` | 装甲灯选择：`led.COMP_ALL`（全部）或其他 |
| `r, g, b` | RGB 颜色值（0-255） |
| `effect` | 灯效：`led.EFFECT_ON`（常亮）、`led.EFFECT_OFF`（熄灭）、`led.EFFECT_FLASH`（闪烁）、`led.EFFECT_BREATH`（呼吸） |

**亮度递增示例：**

```python
import time
from robomaster import robot
from robomaster import led

if __name__ == '__main__':
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="sta")
    ep_led = ep_robot.led
    
    bright = 1
    for i in range(0, 8):
        ep_led.set_led(comp=led.COMP_ALL, r=bright << i, g=bright << i, b=bright << i, effect=led.EFFECT_ON)
        time.sleep(1)
        print("brightness: {0}".format(bright << i))
    
    ep_robot.close()
```

> [!info] 示例文件
> `/examples/01_robot/07_led.py`

## 5.6 动作类接口的使用

动作类接口用于控制机器人执行指定动作，分为 **即时动作控制** 和 **任务动作控制** 两类。

### 5.6.1 即时动作控制

即时控制类动作是指设置后马上生效的动作。

#### 5.6.1.1 示例一：控制发射器射击

```python
import time
from robomaster import robot
from robomaster import blaster

if __name__ == '__main__':
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="sta")
    
    ep_blaster = ep_robot.blaster
    
    # 发射1颗水弹
    ep_blaster.fire(times=1)
    time.sleep(2)
    
    # 发射3颗水弹
    ep_blaster.fire(fire_type=blaster.WATER_FIRE, times=3)
    time.sleep(2)
    
    # 发射红外子弹
    ep_blaster.fire(fire_type=blaster.INFRARED_FIRE, times=3)
    time.sleep(2)
    
    ep_robot.close()
```

**发射类型：**

| 参数 | 说明 |
|------|------|
| `blaster.WATER_FIRE` | 水弹 |
| `blaster.INFRARED_FIRE` | 红外弹 |

> [!info] 示例文件
> `/examples/06_blaster/01_fire.py`

---

#### 5.6.1.2 示例二：控制底盘速度

```python
import time
from robomaster import robot

if __name__ == '__main__':
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="sta")
    
    ep_chassis = ep_robot.chassis
    
    # 前进 3秒
    ep_chassis.drive_speed(x=0.5, y=0, z=0, timeout=5)
    time.sleep(3)
    
    # 后退 3秒
    ep_chassis.drive_speed(x=-0.5, y=0, z=0, timeout=5)
    time.sleep(3)
    
    # 左移 3秒
    ep_chassis.drive_speed(x=0, y=-0.3, z=0, timeout=5)
    time.sleep(3)
    
    # 右转 3秒
    ep_chassis.drive_speed(x=0, y=0, z=30, timeout=5)
    time.sleep(3)
    
    # 停止
    ep_chassis.drive_speed(x=0, y=0, z=0, timeout=5)
    
    ep_robot.close()
```

**速度参数说明：**

| 参数 | 说明 | 单位 |
|------|------|------|
| `x` | 前进/后退速度 | m/s |
| `y` | 横移速度 | m/s |
| `z` | 旋转速度 | °/s |
| `timeout` | 超时时间（超时未收到指令则停止） | 秒 |

> [!info] 示例文件
> `/examples/02_chassis/03_speed.py`

### 5.6.2 任务动作控制

任务动作是指需要持续一段时间才能完成的动作。

> [!important] 注意事项
> 1. 任务动作接口返回 `action` 对象，提供 `wait_for_completed(timeout)` 方法
> 2. 同一模块同一时间只能执行一个动作（互斥）
> 3. 不同模块之间的动作可以同时执行

#### 5.6.2.1 示例一：控制底盘移动指定距离

```python
from robomaster import robot

if __name__ == '__main__':
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="sta")
    
    ep_chassis = ep_robot.chassis
    
    # 前进 0.5米
    ep_chassis.move(x=0.5, y=0, z=0, xy_speed=0.7).wait_for_completed()
    
    # 后退 0.5米
    ep_chassis.move(x=-0.5, y=0, z=0, xy_speed=0.7).wait_for_completed()
    
    # 左移 0.6米
    ep_chassis.move(x=0, y=-0.6, z=0, xy_speed=0.7).wait_for_completed()
    
    # 左转 90度
    ep_chassis.move(x=0, y=0, z=90, z_speed=45).wait_for_completed()
    
    ep_robot.close()
```

**move 参数说明：**

| 参数 | 说明 | 单位 |
|------|------|------|
| `x` | X 轴移动距离 | m |
| `y` | Y 轴移动距离 | m |
| `z` | Z 轴旋转角度 | ° |
| `xy_speed` | XY 轴运动速度 | m/s |
| `z_speed` | Z 轴旋转速度 | °/s |

**三种调用方式：**

```python
# 方法一：立即等待完成（推荐）
ep_chassis.move(x=0.5, y=0, z=0, xy_speed=0.7).wait_for_completed()

# 方法二：稍后等待
chassis_action = ep_chassis.move(x=0.5, y=0, z=0, xy_speed=0.7)
# 可以执行其他操作
chassis_action.wait_for_completed()

# 方法三：使用延时（不推荐）
ep_chassis.move(x=0.5, y=0, z=0, xy_speed=0.7)
time.sleep(10)  # 需要估算足够的时间
```

> [!info] 示例文件
> `/examples/02_chassis/01_move.py`

## 5.7 多媒体接口的使用

多媒体接口主要包括视频流与音频流两部分。

### 5.7.1 示例一：获取视频流

#### 方法一：直接显示视频流

```python
import time
from robomaster import robot
from robomaster import camera

if __name__ == '__main__':
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="sta")
    
    ep_camera = ep_robot.camera
    
    # 显示十秒图传
    ep_camera.start_video_stream(display=True, resolution=camera.STREAM_360P)
    time.sleep(10)
    ep_camera.stop_video_stream()
    
    ep_robot.close()
```

#### 方法二：使用 OpenCV 处理

```python
import cv2
from robomaster import robot

if __name__ == '__main__':
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="sta")
    
    ep_camera = ep_robot.camera
    
    # 显示200帧图传
    ep_camera.start_video_stream(display=False)
    for i in range(0, 200):
        img = ep_camera.read_cv2_image()
        cv2.imshow("Robot", img)
        cv2.waitKey(1)
    
    cv2.destroyAllWindows()
    ep_camera.stop_video_stream()
    ep_robot.close()
```

**分辨率参数：**

| 参数 | 说明 |
|------|------|
| `camera.STREAM_360P` | 360P 分辨率 |
| `camera.STREAM_720P` | 720P 分辨率 |

> [!info] 示例文件
> - `/examples/04_camera/01_video_with_display.py`
> - `/examples/04_camera/03_video_without_display.py`

### 5.7.2 示例二：获取音频流

```python
from robomaster import robot

if __name__ == '__main__':
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="sta")
    
    ep_camera = ep_robot.camera
    
    # 录制5秒音频，保存为 WAV 文件
    ep_camera.record_audio(save_file="output.wav", seconds=5, sample_rate=16000)
    
    ep_robot.close()
```

**音频参数说明：**

| 参数 | 说明 |
|------|------|
| `save_file` | 保存文件名 |
| `seconds` | 录制时长（秒） |
| `sample_rate` | 采样率 |

## 接口类型总结

| 接口类型 | 说明 | 示例 |
|----------|------|------|
| **查询类** | 获取机器人状态信息 | `get_version()`, `get_sn()` |
| **设置类** | 设置机器人参数 | `set_robot_mode()`, `set_led()` |
| **即时动作类** | 立即生效的动作 | `drive_speed()`, `fire()` |
| **任务动作类** | 需要持续一段时间的动作 | `move()` |
| **多媒体类** | 视频/音频流获取 | `start_video_stream()`, `record_audio()` |
