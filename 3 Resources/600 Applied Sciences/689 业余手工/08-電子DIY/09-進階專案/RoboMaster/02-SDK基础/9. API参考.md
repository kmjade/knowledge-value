# 9. RoboMaster SDK APIs

> [!info] 本章概述
> RoboMaster SDK API 接口列表，提供所有模块的快速索引和参考。

---

## SDK 模块概览

RoboMaster SDK 采用模块化设计，主要包含以下模块：

### 核心模块

| 模块 | 说明 | 适用产品 |
|------|------|----------|
| `robomaster.robot` | 机器人主体控制 | EP / 无人机 |
| `robomaster.version` | SDK 版本信息 | 全部 |

### 运动控制模块

| 模块 | 说明 | 适用产品 |
|------|------|----------|
| `robomaster.chassis` | 底盘控制 | EP |
| `robomaster.gimbal` | 云台控制 | EP |
| `robomaster.flight` | 飞行控制 | 无人机 |

### 功能模块

| 模块 | 说明 | 适用产品 |
|------|------|----------|
| `robomaster.blaster` | 发射器控制 | EP |
| `robomaster.armor` | 装甲板控制 | EP |
| `robomaster.led` | LED 灯效 | EP / TT |
| `robomaster.camera` | 相机控制 | EP / 无人机 |
| `robomaster.vision` | 视觉识别 | EP |
| `robomaster.battery` | 电池信息 | EP / 无人机 |

### 拓展模块

| 模块 | 说明 | 适用产品 |
|------|------|----------|
| `robomaster.robotic_arm` | 机械臂控制 | EP |
| `robomaster.gripper` | 机械爪控制 | EP |
| `robomaster.servo` | 舵机控制 | EP |
| `robomaster.sensor` | 传感器接口 | EP |
| `robomaster.uart` | 串口通信 | EP |
| `robomaster.ai_module` | AI 模块 | EP |

### 其他模块

| 模块 | 说明 |
|------|------|
| `robomaster.action` | 动作控制基类 |
| `robomaster.exceptions` | 异常定义 |

## 模块详细索引

### robomaster.robot

机器人主体模块，提供初始化、连接、资源管理等功能。

**主要类：**
- `Robot` - EP 机器人对象
- `Drone` - 无人机对象

**主要方法：**

| 方法 | 说明 | 返回值 |
|------|------|--------|
| `initialize()` | 初始化机器人 | None |
| `close()` | 释放资源 | None |
| `get_version()` | 获取机器人版本 | str |
| `get_sn()` | 获取 SN 号 | str |
| `set_robot_mode()` | 设置运动模式 | bool |

**获取子模块：**
```python
ep_robot.chassis    # 底盘
ep_robot.gimbal     # 云台
ep_robot.blaster    # 发射器
ep_robot.armor      # 装甲板
ep_robot.led        # LED
ep_robot.camera     # 相机
ep_robot.vision     # 视觉
```

---

### robomaster.chassis

底盘控制模块。

**主要方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `move(x, y, z, xy_speed, z_speed)` | 相对移动 | 位置、速度 |
| `drive_speed(x, y, z, timeout)` | 速度控制 | 速度、超时 |
| `drive_wheels(w1, w2, w3, w4)` | 麦轮速度 | 各轮速度 |
| `get_attitude()` | 获取姿态 | 无 |
| `get_position()` | 获取位置 | 无 |

**示例：**
```python
# 前进 1 米
ep_chassis.move(x=1, y=0, z=0, xy_speed=0.7).wait_for_completed()

# 以 0.5 m/s 前进
ep_chassis.drive_speed(x=0.5, y=0, z=0, timeout=5)
```

---

### robomaster.gimbal

云台控制模块。

**主要方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `move(pitch, yaw, pitch_speed, yaw_speed)` | 相对移动 | 角度、速度 |
| `moveto(pitch, yaw, pitch_speed, yaw_speed)` | 绝对移动 | 角度、速度 |
| `suspend()` | 暂停 | 无 |
| `resume()` | 恢复 | 无 |
| `get_attitude()` | 获取姿态 | 无 |

**示例：**
```python
# 云台向左旋转 90 度
ep_gimbal.move(pitch=0, yaw=90).wait_for_completed()

# 云台指向中心位置
ep_gimbal.moveto(pitch=0, yaw=0).wait_for_completed()
```

---

### robomaster.blaster

发射器控制模块。

**主要方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `fire(times, fire_type)` | 发射 | 次数、类型 |
| `set_led(brightness)` | 设置 LED 亮度 | 亮度值 |

**发射类型：**
- `blaster.WATER_FIRE` - 水弹
- `blaster.INFRARED_FIRE` - 红外弹

**示例：**
```python
# 发射 1 颗水弹
ep_blaster.fire(times=1)

# 发射 3 颗红外弹
ep_blaster.fire(fire_type=blaster.INFRARED_FIRE, times=3)
```

### robomaster.led

LED 灯效控制模块。

**主要方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `set_led(comp, r, g, b, effect)` | 设置灯效 | 组件、颜色、效果 |

**组件常量：**
- `led.COMP_ALL` - 全部装甲灯
- `led.COMP_TOP` - 顶部灯
- `led.COMP_BOTTOM` - 底部灯

**效果常量：**
- `led.EFFECT_ON` - 常亮
- `led.EFFECT_OFF` - 熄灭
- `led.EFFECT_FLASH` - 闪烁
- `led.EFFECT_BREATH` - 呼吸

**示例：**
```python
# 设置红色常亮
ep_led.set_led(comp=led.COMP_ALL, r=255, g=0, b=0, effect=led.EFFECT_ON)
```

### robomaster.flight

无人机飞行控制模块。

**主要方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `takeoff()` | 起飞 | 无 |
| `land()` | 降落 | 无 |
| `forward(distance)` | 向前飞 | 距离(cm) |
| `backward(distance)` | 向后飞 | 距离(cm) |
| `left(distance)` | 向左飞 | 距离(cm) |
| `right(distance)` | 向右飞 | 距离(cm) |
| `up(distance)` | 向上飞 | 距离(cm) |
| `down(distance)` | 向下飞 | 距离(cm) |
| `rotate(angle)` | 旋转 | 角度(°) |
| `rc(a, b, c, d)` | 遥控杆量 | 各轴速度 |

**示例：**
```python
# 起飞
tl_flight.takeoff().wait_for_completed()

# 向前飞 50cm
tl_flight.forward(distance=50).wait_for_completed()

# 降落
tl_flight.land().wait_for_completed()
```

## API 使用模式

### 查询类接口

```python
# 获取版本
version = ep_robot.get_version()

# 获取电量
battery = ep_robot.battery.get_battery_percentage()

# 获取底盘位置
position = ep_chassis.get_position()
```

### 设置类接口

```python
# 设置运动模式
ep_robot.set_robot_mode(mode=robot.GIMBAL_LEAD)

# 设置 LED
ep_led.set_led(comp=led.COMP_ALL, r=255, g=0, b=0, effect=led.EFFECT_ON)
```

### 动作类接口

```python
# 任务动作（需要等待完成）
ep_chassis.move(x=1, y=0, z=0, xy_speed=0.7).wait_for_completed()

# 即时动作（立即生效）
ep_chassis.drive_speed(x=0.5, y=0, z=0, timeout=5)
```
