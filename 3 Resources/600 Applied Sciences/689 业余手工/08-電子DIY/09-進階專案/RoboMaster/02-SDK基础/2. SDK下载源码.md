# 2. RoboMaster SDK 下载源码

> [!info] 本章概述
> 介绍如何从 GitHub 和 Gitee 下载 RoboMaster SDK 源码和示例代码。

---

## 源码仓库

RoboMaster SDK 是开源项目，可以从以下平台获取源码：

### GitHub（官方仓库）

| 项目 | 链接 |
|------|------|
| **RoboMaster SDK** | [github.com/dji-sdk/robomaster-sdk](https://github.com/dji-sdk/robomaster-sdk) |
| **RoboMaster-SDK（文档）** | [github.com/dji-sdk/RoboMaster-SDK](https://github.com/dji-sdk/RoboMaster-SDK) |

**功能：**
- Clone - 克隆仓库到本地
- Fork - 复制到自己的 GitHub 账户
- Issues - 报告问题或提出建议

---

### Gitee（国内镜像）

| 项目 | 链接 |
|------|------|
| **RoboMaster SDK** | [gitee.com/xitinglin/RoboMaster-SDK](https://gitee.com/xitinglin/RoboMaster-SDK) |

> [!tip] 推荐
> 国内用户可使用 Gitee 镜像，**下载速度更快**。

---

## 克隆仓库

### 从 GitHub 克隆

```bash
# HTTPS 方式
git clone https://github.com/dji-sdk/robomaster-sdk.git

# SSH 方式（需配置 SSH 密钥）
git clone git@github.com:dji-sdk/robomaster-sdk.git
```

### 从 Gitee 克隆

```bash
# HTTPS 方式
git clone https://gitee.com/xitinglin/RoboMaster-SDK.git
```

---

## 仓库结构

```
robomaster-sdk/
├── robomaster/           # SDK 源码
│   ├── __init__.py
│   ├── robot.py          # 机器人模块
│   ├── chassis.py        # 底盘模块
│   ├── gimbal.py         # 云台模块
│   ├── blaster.py        # 发射器模块
│   ├── armor.py          # 装甲板模块
│   ├── led.py            # LED 模块
│   ├── camera.py         # 相机模块
│   ├── vision.py         # 视觉模块
│   ├── robotic_arm.py    # 机械臂模块
│   ├── gripper.py        # 机械爪模块
│   ├── servo.py          # 舵机模块
│   └── sensor.py         # 传感器模块
├── sample/               # 示例代码
│   ├── ep/               # EP 示例
│   └── drone/            # 无人机示例
├── docs/                 # 文档源码
├── tests/                # 测试代码
└── setup.py              # 安装配置
```

## 相关命令

```bash
# 查看 SDK 版本
pip show robomaster

# 查看 SDK 安装位置
pip show robomaster | grep Location

# 查看本地仓库状态
cd robomaster-sdk
git status

# 更新本地仓库
git pull origin master
```
