# 系统架构

> [!summary] 概述
> RoboRTS 双层系统架构设计，上位机 ROS 与底层 STM32 的协同工作方式。

## 通信机制

### ROS 话题通信

| 话题 | 类型 | 方向 | 说明 |
|------|------|------|------|
| `/camera/image` | sensor_msgs/Image | 发布 | 相机图像 |
| `/scan` | sensor_msgs/LaserScan | 发布 | 激光数据 |
| `/odom` | nav_msgs/Odometry | 发布 | 里程计 |
| `/cmd_vel` | geometry_msgs/Twist | 订阅 | 速度指令 |
| `/armor_info` | roborts_msgs/ArmorInfo | 发布 | 装甲板信息 |

### ROS 服务通信

| 服务 | 类型 | 说明 |
|------|------|------|
| `/gimbal_ctrl` | roborts_msgs/GimbalControl | 云台控制 |
| `/shoot_ctrl` | roborts_msgs/ShootControl | 发射控制 |

### Actionlib 通信

| Action | 说明 |
|--------|------|
| `global_planner` | 全局路径规划 |
| `local_planner` | 局部路径规划 |
| `detection` | 装甲板检测任务 |

## 模块职责

### 上位机模块

| 模块 | 职责 | 频率 |
|------|------|------|
| `roborts_camera` | 图像采集、发布 | 30-60 Hz |
| `roborts_detection` | 装甲板识别、PnP 解算 | 30 Hz |
| `roborts_localization` | AMCL 定位 | 10 Hz |
| `roborts_decision` | 行为树决策 | 20 Hz |
| `roborts_planning` | 路径规划、避障 | 20 Hz |

### 底层模块

| 模块 | 职责 | 频率 |
|------|------|------|
| `chassis` | 底盘运动控制 | 500 Hz |
| `gimbal` | 云台姿态控制 | 500 Hz |
| `shooter` | 发射机构控制 | 500 Hz |
| `sensor` | IMU、编码器读取 | 1000 Hz |

## 线程模型

### 上位机线程

```
Main Thread
├── ROS Spinner (回调处理)
├── Camera Thread (图像采集)
├── Detection Thread (目标检测)
├── Localization Thread (定位更新)
├── Decision Thread (行为决策)
└── Planning Thread (路径规划)
```

### 底层线程 (FreeRTOS)

```
IMU Task (1000 Hz)
├── 读取 IMU 数据
└── 更新姿态估计

Motor Task (500 Hz)
├── 读取电机反馈
├── PID 控制计算
└── 发送控制指令

Communication Task (100 Hz)
├── 接收上位机指令
└── 发送传感器数据
```
