# 1. RoboMaster Python 编程介绍

> [!info] 本章概述
> 介绍 RoboMaster EP App 内置 Python 编程环境的功能特性、界面说明和开发资源。

---

## 1.1 介绍

本章介绍的 Python 编程指的是使用设备连接 EP 机器人后，进入 **App 实验室**进行 Python 编程。

### 功能特性

在 Python 编程界面，允许玩家基于 **Python 3.6.6** 版本的基础语法完成 Python 编程，并可以：

- 调用 RoboMaster EP 提供的编程接口编写自己的 Python 程序
- 生成的 Python 程序可被装配成自主程序或自定义技能
- 实现多机通信，让多台机器人相互通信
- 编程自定义 UI 系统，设计交互界面

### 核心能力

| 功能 | 说明 |
|------|------|
| **Python 3.6.6 环境** | 完整的 Python 基础语法支持 |
| **机器人控制 API** | 底盘、云台、发射器等控制接口 |
| **多机通信** | 多台机器人局域网通信 |
| **自定义 UI** | 虚拟控件，交互界面设计 |
| **程序装配** | 生成自主程序或自定义技能 |

## 1.5 与 Python SDK 的区别

> [!important] 重要区分
> App 内置 Python 编程与 RoboMaster Python SDK 是两种不同的开发方式。

| 对比项 | App 内置 Python | Python SDK |
|--------|-----------------|------------|
| **环境** | App 实验室 | 独立 Python 环境 |
| **Python 版本** | 3.6.6 | 3.6.6 - 3.8.9 |
| **API 前缀** | `robot_ctrl`、`chassis_ctrl` 等 | `ep_robot.chassis` 等 |
| **运行位置** | 机器人端/移动设备 | PC 端 |
| **适用场景** | 快速原型、教学 | 专业开发、复杂应用 |

### API 命名差异

**App 内置 Python：**
```python
# 获取电量
battery = robot_ctrl.get_battery_percentage()

# 底盘移动
chassis_ctrl.move(1, 0, 0, 0.7)
```

**Python SDK：**
```python
from robomaster import robot

ep_robot = robot.Robot()
ep_robot.initialize(conn_type="ap")

# 获取电量
battery = ep_robot.battery.get_battery_percentage()

# 底盘移动
ep_robot.chassis.move(x=1, y=0, z=0, xy_speed=0.7).wait_for_completed()
```

## 1.6 快速开始

### 第一个程序

```python
# 导入模块
import robot_ctrl
import chassis_ctrl

# 获取电量
battery = robot_ctrl.get_battery_percentage()
print(f"电量: {battery}%")

# 前进 1 米
chassis_ctrl.move(1, 0, 0, 0.7)

# 发射
import gun_ctrl
gun_ctrl.fire_once()
```

## 常见问题

### 1. 无法运行 Python 程序

**解决方案：**
- 确认已连接 EP 机器人
- 检查 Python 语法是否正确
- 确认已进入编程模式

### 2. API 找不到

**解决方案：**
- 确认使用正确的 API 前缀（`robot_ctrl`、`chassis_ctrl` 等）
- 参考编程模块手册
- 查看本章 [[3. Python API]]

### 3. 与 SDK 代码不兼容

**解决方案：**
- App 内置 Python 与 Python SDK 使用不同的 API
- 参考 [[../02-SDK基础]] 了解 SDK 编程方式
