# 大疆电机M3508 PWM控制

> [!summary] 概述
> 本文介绍 M3508 电机通过 PWM 信号进行控制的方法，包括 C620 电调 PWM 模式配置、校准流程、STM32 定时器配置及代码实现。

---

## 一、PWM 控制模式概述

![[Pasted image 20260524122701.png]]

### 1.1 PWM 控制与 CAN 控制

M3508 电机支持两种控制方式：

| 控制方式 | 接口 | 特点 |
|----------|------|------|
| **CAN 控制** | CAN 总线 | 功能完整，支持反馈数据 |
| **PWM 控制** | PWM 信号 | 简单直接，无需 CAN 通信 |

### 1.2 PWM 控制模式

C620 电调支持两种 PWM 控制模式：

| 模式 | 脉宽范围 | 说明 |
|------|----------|------|
| **单向模式** | 1000μs ~ 2000μs | 仅单向转动 |
| **正反转模式** | 1000μs ~ 2000μs | 支持正反转控制 |

> [!tip] 模式选择
> 通过上位机软件可以切换单向模式和正反转模式，同时可以选择旋转方向（顺时针/逆时针）。

### 1.3 C620 电调 PWM 规格

| 参数 | 值 |
|------|-----|
| **PWM 频率** | 500Hz（推荐） |
| **有效脉宽范围** | 1000μs ~ 2000μs |
| **中位脉宽** | 1500μs |
| **校准脉宽** | 1480μs ~ 1520μs |

### 2.2 C620 电调配置

> [!warning] 重要设置
> 使用 PWM 控制模式前，必须调整 C620 电调上的 **120Ω 电阻**到 **OFF** 挡位。

**配置步骤：**

1. **断电状态下**调整电阻位置
2. 将 120Ω 电阻拨到 **OFF** 位置
3. 使用上位机配置控制模式
4. 设置旋转方向

### 2.3 PWM 信号引脚

C620 电调 PWM 信号输入：

| 引脚 | 说明 |
|------|------|
| **SIG** | PWM 信号输入 |
| **GND** | 信号地 |
| **VCC** | 信号电源（可选） |

## 三、电机校准

### 3.1 校准必要性

在控制前，需要 PWM 输出**最低行程脉宽（1480-1520 微秒）**校准电机，否则 C620 电调会一直闪黄灯。

### 3.2 校准流程

```
1. 上电前：确保 PWM 输出 1500μs 中位信号
2. 上电：给 C620 电调供电
3. 观察：
   - 黄灯闪烁：未校准成功
   - 绿灯闪烁：校准成功
4. 等待：听到电机发出"滴"声表示校准完成
```

### 3.3 校准代码

```c
/**
 * @brief  电机校准 - 输出中位 PWM
 */
void Motor_Calibrate(void)
{
    // 输出 1500μs 脉宽
    __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, 1500);

    // 等待校准完成（约 2-3 秒）
    HAL_Delay(3000);
}
```

### 3.4 故障排除

| 现象 | 原因 | 解决方法 |
|------|------|----------|
| 黄灯一直闪烁 | 校准失败 | 检查 PWM 输出是否为 1480-1520μs |
| 绿灯闪烁 | 正常工作 | 可以开始控制 |
| 无反应 | 电阻未调整 | 将 120Ω 电阻拨到 OFF |

> [!tip] 调试建议
> 如果 1500μs 无法校准，可以尝试 **500μs** 脉宽，观察电调状态。

## 四、STM32 定时器配置

### 4.1 PWM 参数计算

以 STM32F4 为例，TIM2 挂载在 APB1 总线上：

| 参数 | 值 | 说明 |
|------|-----|------|
| APB1 时钟 | 84 MHz | 总线时钟 |
| 预分频 (PSC) | 83 | 预分频系数 |
| 自动重装载 (ARR) | 1999 | 周期计数值 |

**频率计算：**

```
定时器时钟 = APB1 / (PSC + 1)
           = 84MHz / 84 = 1MHz

PWM 周期 = (ARR + 1) / 定时器时钟
         = 2000 / 1MHz = 2ms = 2000μs

PWM 频率 = 1 / PWM 周期
         = 1 / 2ms = 500Hz
```

### 4.2 占空比计算

```
占空比 = CCR / (ARR + 1)
       = CCR / 2000

脉宽 = CCR × 1μs

例如：
- CCR = 1500 → 脉宽 = 1500μs
- CCR = 1000 → 脉宽 = 1000μs
- CCR = 2000 → 脉宽 = 2000μs
```

### 4.3 CubeMX 配置

**步骤 1：选择定时器**

在 Pinout & Configuration → Timers → TIM2：
- **Channel1**：PWM Generation CH1

**步骤 2：参数配置**

| 参数 | 值 |
|------|-----|
| Prescaler (PSC) | 83 |
| Counter Period (ARR) | 1999 |
| Pulse (CCR) | 1500 |
| Counter Mode | Up |
| auto-reload preload | Enable |

**步骤 3：时钟验证**

```
PWM 频率 = 84MHz / (84 × 2000) = 500Hz ✓
PWM 周期 = 2ms = 2000μs ✓
```

## 五、代码实现

### 5.1 PWM 初始化

```c
/**
 * @brief  PWM 初始化
 */
void PWM_Init(void)
{
    // 启动 PWM 输出
    HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_1);

    // 设置初始值（中位）
    __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, 1500);
}
```

### 5.2 电机控制函数

#### 单向模式

```c
/**
 * @brief  设置电机速度（单向模式）
 * @param  speed: 速度百分比 (0-100)
 */
void Motor_SetSpeed_Unidirectional(uint8_t speed)
{
    if (speed > 100) speed = 100;

    // 脉宽范围：1000-2000μs
    // speed=0 → 1000μs, speed=100 → 2000μs
    uint16_t pulse = 1000 + speed * 10;

    __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, pulse);
}
```

#### 正反转模式

```c
/**
 * @brief  设置电机速度（正反转模式）
 * @param  speed: 速度百分比 (-100 ~ 100)
 *                 正值正转，负值反转，0 停止
 */
void Motor_SetSpeed_Bidirectional(int8_t speed)
{
    if (speed > 100) speed = 100;
    if (speed < -100) speed = -100;

    // 脉宽范围：1000-2000μs
    // speed=-100 → 1000μs, speed=0 → 1500μs, speed=100 → 2000μs
    uint16_t pulse = 1500 + speed * 5;

    __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, pulse);
}
```

### 5.3 完整示例代码

```c
/* main.c */

#include "main.h"
#include "tim.h"

#define PWM_MIN    1000
#define PWM_CENTER 1500
#define PWM_MAX    2000

void PWM_Init(void);
void Motor_Calibrate(void);
void Motor_SetSpeed(uint16_t pulse);

int main(void)
{
    // HAL 库初始化
    HAL_Init();
    SystemClock_Config();

    // 外设初始化
    MX_GPIO_Init();
    MX_TIM2_Init();

    // PWM 初始化
    PWM_Init();

    // 电机校准
    Motor_Calibrate();

    while (1)
    {
        // 单向模式示例：逐渐加速
        for (uint16_t pulse = PWM_MIN; pulse <= PWM_MAX; pulse += 10)
        {
            Motor_SetSpeed(pulse);
            HAL_Delay(20);
        }

        // 逐渐减速
        for (uint16_t pulse = PWM_MAX; pulse >= PWM_MIN; pulse -= 10)
        {
            Motor_SetSpeed(pulse);
            HAL_Delay(20);
        }

        HAL_Delay(1000);
    }
}

/**
 * @brief  PWM 初始化
 */
void PWM_Init(void)
{
    HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_1);
    __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, PWM_CENTER);
}

/**
 * @brief  电机校准
 */
void Motor_Calibrate(void)
{
    // 输出中位 PWM 等待校准
    __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, PWM_CENTER);
    HAL_Delay(3000);
}

/**
 * @brief  设置电机速度
 * @param  pulse: PWM 脉宽 (1000-2000)
 */
void Motor_SetSpeed(uint16_t pulse)
{
    if (pulse < PWM_MIN) pulse = PWM_MIN;
    if (pulse > PWM_MAX) pulse = PWM_MAX;

    __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, pulse);
}
```

## 六、多电机控制

### 6.1 多路 PWM 配置

使用同一定时器的多个通道控制多个电机：

| 定时器 | 通道 | 引脚 | 电机 |
|--------|------|------|------|
| TIM2 | CH1 | PA0 | 电机 1 |
| TIM2 | CH2 | PA1 | 电机 2 |
| TIM2 | CH3 | PA2 | 电机 3 |
| TIM2 | CH4 | PA3 | 电机 4 |

### 6.2 多电机控制代码

```c
/**
 * @brief  设置多路电机速度
 * @param  channel: 通道号 (1-4)
 * @param  pulse: PWM 脉宽 (1000-2000)
 */
void Motor_SetSpeed_Multi(uint8_t channel, uint16_t pulse)
{
    if (pulse < 1000) pulse = 1000;
    if (pulse > 2000) pulse = 2000;

    switch (channel)
    {
        case 1:
            __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, pulse);
            break;
        case 2:
            __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_2, pulse);
            break;
        case 3:
            __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_3, pulse);
            break;
        case 4:
            __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_4, pulse);
            break;
    }
}

/**
 * @brief  启动所有 PWM 通道
 */
void PWM_StartAll(void)
{
    HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_1);
    HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_2);
    HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_3);
    HAL_TIM_PWM_Start(&htim2, TIM_CHANNEL_4);
}
```

## 七、PWM 与 CAN 控制对比

### 7.1 功能对比

| 功能 | PWM 控制 | CAN 控制 |
|------|----------|----------|
| **速度控制** | ✓ | ✓ |
| **方向控制** | ✓（正反转模式） | ✓ |
| **位置反馈** | ✗ | ✓ |
| **速度反馈** | ✗ | ✓ |
| **电流反馈** | ✗ | ✓ |
| **温度反馈** | ✗ | ✓ |
| **精确控制** | 一般 | 高 |
| **接线复杂度** | 简单 | 复杂 |

### 7.2 应用场景

| 场景 | 推荐方式 |
|------|----------|
| **简单速度控制** | PWM |
| **开环控制** | PWM |
| **精确速度控制** | CAN |
| **位置控制** | CAN |
| **需要反馈数据** | CAN |
| **快速原型验证** | PWM |

## 八、调试与故障排除

### 8.1 调试步骤

1. **检查电阻设置**：确认 120Ω 电阻在 OFF 位置
2. **检查 PWM 输出**：用示波器测量 PWM 信号
3. **校准电机**：输出中位 PWM 等待校准
4. **观察指示灯**：黄灯闪烁表示异常，绿灯表示正常

### 8.2 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 电调黄灯闪烁 | 校准失败 | 重新输出中位 PWM |
| 电机不转 | PWM 范围错误 | 检查脉宽是否在 1000-2000 |
| 电机抖动 | PWM 频率不对 | 确认 500Hz 频率 |
| 无法反转 | 模式设置错误 | 使用上位机设置正反转模式 |
| 转速不对 | 脉宽计算错误 | 检查 ARR 和 PSC 设置 |

### 8.3 示波器验证

```c
// 验证 PWM 参数
// 期望：频率 500Hz，周期 2ms，脉宽 1000-2000μs

// 测试代码
__HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, 1500);  // 1.5ms 脉宽
// 用示波器测量：
// - 周期应为 2ms
// - 高电平时间应为 1.5ms
// - 频率应为 500Hz
```
