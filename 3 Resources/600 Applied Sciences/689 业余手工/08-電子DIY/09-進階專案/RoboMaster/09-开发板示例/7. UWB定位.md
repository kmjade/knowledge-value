## UWB 技术

### 技术概述

UWB (Ultra-Wideband，超宽带) 是一种无线通信技术：

| 特性 | 说明 |
|------|------|
| **频段** | 3.1 ~ 10.6 GHz |
| **带宽** | > 500 MHz |
| **定位精度** | 厘米级 (10-30cm) |
| **测距范围** | 0-100m |
| **功耗** | 低功耗 |

### 定位原理

UWB 定位主要使用以下方法：

| 方法 | 原理 | 精度 |
|------|------|------|
| **TOF** | Time of Flight，飞行时间测距 | 高 |
| **TDOA** | Time Difference of Arrival，到达时间差 | 中 |
| **AOA** | Angle of Arrival，到达角度 | 中 |
| **RSSI** | 信号强度测距 | 低 |

## 使用示例

### 基础测距

```c
#include "uwb.h"

int main(void)
{
    HAL_Init();
    SystemClock_Config();
    
    // 初始化 UWB
    if (UWB_Init() != 0) {
        printf("UWB init failed!\n");
        Error_Handler();
    }
    
    // 设置本机地址
    UWB_SetAddress(0x0001);
    
    float distance;
    
    while (1)
    {
        // 测量与目标 0x0002 的距离
        if (UWB_GetDistance(0x0002, &distance) == 0) {
            printf("Distance: %.2f m\n", distance);
        }
        
        HAL_Delay(100);
    }
}
```

### 定位系统

```c
// 锚点配置
typedef struct {
    uint16_t id;
    float x, y;
} UWB_Anchor_t;

UWB_Anchor_t anchors[4] = {
    {0x0001, 0.0f, 0.0f},    // 锚点 1：原点
    {0x0002, 10.0f, 0.0f},   // 锚点 2：X 轴
    {0x0003, 10.0f, 10.0f},  // 锚点 3：对角
    {0x0004, 0.0f, 10.0f},   // 锚点 4：Y 轴
};

// 初始化定位系统
void UWB_Positioning_Init(void)
{
    UWB_Init();
    
    // 设置锚点位置
    for (int i = 0; i < 4; i++) {
        UWB_SetAnchorPosition(anchors[i].id, anchors[i].x, anchors[i].y);
    }
}

// 获取位置
void UWB_UpdatePosition(void)
{
    float x, y;
    
    if (UWB_GetPosition(&x, &y) == 0) {
        printf("Position: (%.2f, %.2f)\n", x, y);
        
        // 更新机器人位置
        Robot_SetPosition(x, y);
    }
}
```

### 多机器人编队

```c
// 编队定位
#define ROBOT_COUNT 3
uint16_t robot_addrs[ROBOT_COUNT] = {0x0001, 0x0002, 0x0003};
float robot_positions[ROBOT_COUNT][2];

void Formation_UpdatePositions(void)
{
    float distance;
    
    for (int i = 0; i < ROBOT_COUNT; i++) {
        if (UWB_GetDistance(robot_addrs[i], &distance) == 0) {
            // 根据距离更新相对位置
            UpdateRobotPosition(i, distance);
        }
    }
}

// 编队控制
void Formation_Control(void)
{
    // 保持编队间距
    float target_distance = 1.0f;  // 1米间距
    
    for (int i = 0; i < ROBOT_COUNT - 1; i++) {
        float actual_distance = CalculateDistance(
            robot_positions[i], robot_positions[i + 1]);
        
        if (actual_distance > target_distance + 0.1f) {
            // 距离过大，缩小
            Robot_MoveCloser(i, i + 1);
        } else if (actual_distance < target_distance - 0.1f) {
            // 距离过小，扩大
            Robot_MoveApart(i, i + 1);
        }
    }
}
```

```c
// 位置卡尔曼滤波
typedef struct {
    float x, y;
    float vx, vy;
    float P[4][4];  // 误差协方差矩阵
} KalmanFilter_t;

KalmanFilter_t kf;

void Kalman_Init(void)
{
    kf.x = 0; kf.y = 0;
    kf.vx = 0; kf.vy = 0;
    // 初始化协方差矩阵
    memset(kf.P, 0, sizeof(kf.P));
    kf.P[0][0] = kf.P[1][1] = 1.0f;
}

void Kalman_Update(float measured_x, float measured_y)
{
    // 预测步骤
    float dt = 0.1f;  // 时间步长
    float pred_x = kf.x + kf.vx * dt;
    float pred_y = kf.y + kf.vy * dt;
    
    // 更新步骤
    float K[2];  // 卡尔曼增益
    // ... 计算卡尔曼增益
    
    kf.x = pred_x + K[0] * (measured_x - pred_x);
    kf.y = pred_y + K[1] * (measured_y - pred_y);
}
```

### 与机器人控制集成

```c
// 使用 UWB 定位的导航控制
void Navigation_Control(void)
{
    float current_x, current_y;
    float target_x = 5.0f, target_y = 5.0f;
    
    // 获取当前位置
    UWB_GetPosition(&current_x, &current_y);
    
    // 计算目标方向
    float dx = target_x - current_x;
    float dy = target_y - current_y;
    float distance = sqrtf(dx * dx + dy * dy);
    float angle = atan2f(dy, dx);
    
    // 控制机器人移动
    if (distance > 0.2f) {
        Chassis_MoveTo(dx, dy, angle);
    } else {
        Chassis_Stop();
    }
}
```

### 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 无法定位 | 锚点未配置 | 检查锚点设置 |
| 定位跳变 | 多径效应 | 增加滤波 |
| 距离不准 | 时钟漂移 | 校准时钟 |
| 信号弱 | 遮挡/距离远 | 调整位置/增加锚点 |
