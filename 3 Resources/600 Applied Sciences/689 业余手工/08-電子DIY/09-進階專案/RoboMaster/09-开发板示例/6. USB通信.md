# USB通信

> [!summary] 概述
> STM32 开发板 USB 通信示例，实现虚拟串口 (CDC) 功能，用于与上位机通信。

---

## 功能说明

### 核心功能

- USB 虚拟串口 (CDC)
- 与 PC 通信
- 数据收发

### 应用场景

| 场景 | 说明 |
|------|------|
| **调试接口** | 打印调试信息 |
| **数据传输** | 上位机数据通信 |
| **参数配置** | 通过串口配置参数 |
| **固件升级** | USB DFU 模式 |

---

## 技术架构

### USB 协议

| 项目 | 说明 |
|------|------|
| **类别** | CDC (Communication Device Class) |
| **功能** | 虚拟串口 |
| **端点** | Control IN/OUT, Interrupt IN, Bulk IN/OUT |
| **速度** | Full Speed (12Mbps) |

### USB 设备库

STM32 USB 设备库提供完整的 USB 协议栈：

```
Middlewares/ST/STM32_USB_Device_Library/
├── Core/                   # USB 核心库
│   ├── Src/
│   │   ├── usbd_core.c
│   │   ├── usbd_ctlreq.c
│   │   └── usbd_ioreq.c
│   └── Inc/
└── Class/                  # USB 类
    └── CDC/               # CDC 虚拟串口
        ├── Src/
        │   └── usbd_cdc.c
        └── Inc/
            └── usbd_cdc.h
```

## 使用示例

### 初始化

```c
#include "usb_device.h"
#include "usbd_cdc_if.h"

int main(void)
{
    HAL_Init();
    SystemClock_Config();
    
    // 初始化 USB CDC
    MX_USB_DEVICE_Init();
    
    while (1)
    {
        // 主循环
        HAL_Delay(100);
    }
}
```

### 数据发送

```c
// 发送字符串
void USB_SendString(const char *str)
{
    CDC_Transmit_FS((uint8_t *)str, strlen(str));
}

// 发送格式化数据
void USB_Printf(const char *fmt, ...)
{
    char buffer[256];
    va_list args;
    
    va_start(args, fmt);
    vsnprintf(buffer, sizeof(buffer), fmt, args);
    va_end(args);
    
    CDC_Transmit_FS((uint8_t *)buffer, strlen(buffer));
}

// 使用示例
USB_Printf("Hello, USB CDC!\r\n");
USB_Printf("Value: %d, Float: %.2f\r\n", 123, 45.67);
```

### 数据接收

```c
// 重写接收回调函数
void CDC_Receive_FS(uint8_t *Buf, uint32_t *Len)
{
    // 处理接收到的数据
    for (uint32_t i = 0; i < *Len; i++) {
        ProcessReceivedByte(Buf[i]);
    }
    
    // 可选：回显
    CDC_Transmit_FS(Buf, *Len);
}
```

### 命令解析

```c
#define CMD_BUFFER_SIZE 128
static char cmd_buffer[CMD_BUFFER_SIZE];
static uint16_t cmd_index = 0;

void ProcessReceivedByte(uint8_t data)
{
    if (data == '\r' || data == '\n') {
        if (cmd_index > 0) {
            cmd_buffer[cmd_index] = '\0';
            ProcessCommand(cmd_buffer);
            cmd_index = 0;
        }
    } else if (cmd_index < CMD_BUFFER_SIZE - 1) {
        cmd_buffer[cmd_index++] = data;
    }
}

void ProcessCommand(char *cmd)
{
    USB_Printf("Received: %s\r\n", cmd);
    
    if (strcmp(cmd, "help") == 0) {
        USB_Printf("Commands: help, info, led, motor\r\n");
    }
    else if (strcmp(cmd, "info") == 0) {
        USB_Printf("Device: RoboMaster Dev Board\r\n");
        USB_Printf("Version: 1.0.0\r\n");
    }
    else if (strncmp(cmd, "led ", 4) == 0) {
        int state;
        sscanf(cmd + 4, "%d", &state);
        SetLED(state);
        USB_Printf("LED set to %d\r\n", state);
    }
    else {
        USB_Printf("Unknown command: %s\r\n", cmd);
    }
}
```

## 上位机通信

### Python 示例

```python
import serial
import time

# 打开虚拟串口
ser = serial.Serial('COM3', 115200, timeout=1)

# 发送数据
ser.write(b'Hello STM32!\r\n')

# 接收数据
while True:
    if ser.in_waiting:
        data = ser.readline()
        print(f"Received: {data.decode()}")
    
    # 发送命令
    cmd = input("Enter command: ")
    ser.write((cmd + '\r\n').encode())
    
ser.close()
```

### C# 示例

```csharp
using System.IO.Ports;

SerialPort port = new SerialPort("COM3", 115200);
port.Open();

// 发送数据
port.WriteLine("info");

// 接收数据
while (true)
{
    if (port.BytesToRead > 0)
    {
        string line = port.ReadLine();
        Console.WriteLine($"Received: {line}");
    }
}

port.Close();
```

```
  idVendor:           0x0483 (STMicroelectronics)
  idProduct:          0x5740 (Virtual COM Port)

```

### 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 设备未识别 | 驱动问题 | 安装 ST 驱动 |
| 无法打开串口 | 被占用 | 检查其他软件 |
| 数据丢失 | 发送太快 | 添加延时 |
| 枚举失败 | 描述符错误 | 检查配置 |

### 数据监控

## 进阶应用

### 多端点通信

```c
// 自定义端点通信
#define EP_DATA_IN  0x81
#define EP_DATA_OUT 0x01

void Custom_EP_Transmit(uint8_t *data, uint16_t len)
{
    USBD_LL_Transmit(&hUsbDeviceFS, EP_DATA_IN, data, len);
}
```

### USB HID 模式

可修改为 HID 设备（键盘、鼠标等）：

```c
// 使用 HID 类代替 CDC
#include "usbd_hid.h"

// HID 报告描述符
uint8_t HID_ReportDesc[] = {
    // 鼠标报告描述符
    0x05, 0x01,  // Usage Page (Generic Desktop)
    0x09, 0x02,  // Usage (Mouse)
    // ...
};
```
