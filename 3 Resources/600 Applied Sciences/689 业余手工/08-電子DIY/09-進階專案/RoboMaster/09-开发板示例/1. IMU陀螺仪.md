# IMU陀螺仪

> [!summary] 概述
> STM32 开发板 IMU 陀螺仪数据读取示例，用于获取机器人姿态和运动信息。

---

## 功能说明

### 核心功能

- 读取 IMU 传感器原始数据
- 计算姿态角度（俯仰、横滚、偏航）
- 提供加速度和角速度数据

### 应用场景

| 场景 | 说明 |
|------|------|
| **姿态控制** | 获取机器人当前姿态 |
| **平衡控制** | 用于自平衡机器人 |
| **运动检测** | 检测机器人运动状态 |
| **导航定位** | 惯性导航辅助 |

---

## 硬件接口

### IMU 传感器

| 参数 | 说明 |
|------|------|
| **传感器型号** | BMI055 / ICM20602 等 |
| **通信接口** | SPI / I2C |
| **测量范围** | ±250°/s ~ ±2000°/s (陀螺仪) |
| **加速度范围** | ±2g ~ ±16g |

### 引脚连接

| 信号 | GPIO | 说明 |
|------|------|------|
| SCK | SPI_SCK | SPI 时钟 |
| MISO | SPI_MISO | SPI 主入从出 |
| MOSI | SPI_MOSI | SPI 主出从入 |
| CS_GYRO | GPIO | 陀螺仪片选 |
| CS_ACC | GPIO | 加速度计片选 |

### API 接口

#### 初始化

```c
/**
 * @brief 初始化 IMU
 * @return 0: 成功, -1: 失败
 */
int IMU_Init(void);

/**
 * @brief 检测 IMU 是否在线
 * @return 1: 在线, 0: 离线
 */
uint8_t IMU_IsOnline(void);
```

#### 数据读取

```c
/**
 * @brief 获取加速度数据
 * @param acc_x X轴加速度 (mg)
 * @param acc_y Y轴加速度 (mg)
 * @param acc_z Z轴加速度 (mg)
 */
void IMU_GetAcceleration(int16_t *acc_x, int16_t *acc_y, int16_t *acc_z);

/**
 * @brief 获取角速度数据
 * @param gyro_x X轴角速度 (dps)
 * @param gyro_y Y轴角速度 (dps)
 * @param gyro_z Z轴角速度 (dps)
 */
void IMU_GetGyroscope(int16_t *gyro_x, int16_t *gyro_y, int16_t *gyro_z);

/**
 * @brief 获取姿态角度
 * @param pitch 俯仰角 (度)
 * @param roll  横滚角 (度)
 * @param yaw   偏航角 (度)
 */
void IMU_GetEulerAngles(float *pitch, float *roll, float *yaw);
```

## 使用示例

### 基础使用

```c
#include "bmi055.h"

int main(void)
{
    // HAL 初始化
    HAL_Init();
    SystemClock_Config();
    
    // 初始化 IMU
    if (IMU_Init() != 0) {
        Error_Handler();
    }
    
    int16_t acc_x, acc_y, acc_z;
    int16_t gyro_x, gyro_y, gyro_z;
    float pitch, roll, yaw;
    
    while (1)
    {
        // 读取加速度
        IMU_GetAcceleration(&acc_x, &acc_y, &acc_z);
        
        // 读取角速度
        IMU_GetGyroscope(&gyro_x, &gyro_y, &gyro_z);
        
        // 读取姿态角
        IMU_GetEulerAngles(&pitch, &roll, &yaw);
        
        // 数据处理...
        
        HAL_Delay(10);
    }
}
```

### 姿态解算

```c
// 使用互补滤波计算姿态角
void IMU_Update(void)
{
    static float pitch = 0, roll = 0;
    float dt = 0.01f; // 10ms
    
    int16_t acc_x, acc_y, acc_z;
    int16_t gyro_x, gyro_y, gyro_z;
    
    IMU_GetAcceleration(&acc_x, &acc_y, &acc_z);
    IMU_GetGyroscope(&gyro_x, &gyro_y, &gyro_z);
    
    // 加速度计计算角度
    float acc_pitch = atan2(acc_y, acc_z) * 180.0f / M_PI;
    float acc_roll = atan2(-acc_x, acc_z) * 180.0f / M_PI;
    
    // 互补滤波
    pitch = 0.98f * (pitch + gyro_x * dt) + 0.02f * acc_pitch;
    roll = 0.98f * (roll + gyro_y * dt) + 0.02f * acc_roll;
}
```

## 调试方法

### 数据校验

1. **检查通信**
   - 使用逻辑分析仪查看 SPI 波形
   - 确认片选信号正确

2. **验证数据**
   - 静止时加速度 Z 轴约 1g (1000 mg)
   - 静止时角速度接近 0

3. **常见问题**

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 数据全为 0 | SPI 通信失败 | 检查接线 |
| 角度漂移 | 零点偏移 | 添加零点校准 |
| 数据跳变 | 供电不稳定 | 检查电源滤波 |

```c
#define CALIBRATION_SAMPLES 1000

void IMU_Calibrate(void)
{
    int32_t gyro_x_sum = 0, gyro_y_sum = 0, gyro_z_sum = 0;
    int16_t gx, gy, gz;
    
    for (int i = 0; i < CALIBRATION_SAMPLES; i++) {
        IMU_GetGyroscope(&gx, &gy, &gz);
        gyro_x_sum += gx;
        gyro_y_sum += gy;
        gyro_z_sum += gz;
        HAL_Delay(1);
    }
    
    // 计算零点偏移
    int16_t offset_x = gyro_x_sum / CALIBRATION_SAMPLES;
    int16_t offset_y = gyro_y_sum / CALIBRATION_SAMPLES;
    int16_t offset_z = gyro_z_sum / CALIBRATION_SAMPLES;

}
```
