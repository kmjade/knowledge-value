# OLED显示

> [!summary] 概述
> RoboMaster 开发板 OLED 屏幕演示示例，包含 BSP 硬件抽象层，支持图形和文字显示。

---

## 功能说明

### 核心功能

- OLED 屏幕驱动
- 图形绘制（点、线、矩形、圆）
- 文字显示（ASCII、汉字）
- 图片显示

### 应用场景

| 场景 | 说明 |
|------|------|
| **状态显示** | 显示机器人运行状态 |
| **调试输出** | 显示传感器数据 |
| **用户界面** | 参数设置界面 |
| **动画效果** | 启动动画、LOGO |

---

## 硬件规格

### OLED 参数

| 参数 | 值 |
|------|-----|
| **分辨率** | 128 x 64 像素 |
| **驱动芯片** | SSD1306 |
| **通信接口** | I2C / SPI |
| **显示颜色** | 单色（白/蓝） |
| **工作电压** | 3.3V / 5V |

### 引脚连接

#### I2C 模式

| 信号 | GPIO | 说明 |
|------|------|------|
| SCL | I2C_SCL | I2C 时钟 |
| SDA | I2C_SDA | I2C 数据 |

#### SPI 模式

| 信号 | GPIO | 说明 |
|------|------|------|
| SCK | SPI_SCK | SPI 时钟 |
| MOSI | SPI_MOSI | SPI 数据 |
| DC | GPIO | 数据/命令选择 |
| CS | GPIO | 片选 |
| RES | GPIO | 复位 |

## 使用示例

### 基础显示

```c
#include "oled.h"

int main(void)
{
    HAL_Init();
    SystemClock_Config();
    
    // 初始化 OLED
    OLED_Init();
    OLED_Clear();
    
    // 显示文字
    OLED_ShowString(0, 0, "RoboMaster", 16);
    OLED_ShowString(0, 16, "Hello World!", 16);
    
    // 显示数字
    int count = 0;
    
    while (1)
    {
        OLED_ShowNum(0, 32, count++, 5, 16);
        HAL_Delay(100);
    }
}
```

### 状态显示界面

```c
// 显示机器人状态
void Display_RobotStatus(void)
{
    char buf[20];
    
    OLED_Clear();
    
    // 标题
    OLED_ShowString(0, 0, "Robot Status", 12);
    
    // 电池电量
    sprintf(buf, "BAT:%d%%", battery_percent);
    OLED_ShowString(0, 12, buf, 12);
    
    // 温度
    sprintf(buf, "TMP:%dC", temperature);
    OLED_ShowString(64, 12, buf, 12);
    
    // 速度
    sprintf(buf, "VX:%.1f", velocity_x);
    OLED_ShowString(0, 24, buf, 12);
    
    sprintf(buf, "VY:%.1f", velocity_y);
    OLED_ShowString(64, 24, buf, 12);
    
    // 模式
    OLED_ShowString(0, 48, "Mode:", 12);
    switch (robot_mode) {
        case 0: OLED_ShowString(48, 48, "MANUAL", 12); break;
        case 1: OLED_ShowString(48, 48, "AUTO  ", 12); break;
        case 2: OLED_ShowString(48, 48, "DEBUG ", 12); break;
    }
}
```

### 绘图示例

```c
void Display_DrawDemo(void)
{
    OLED_Clear();
    
    // 画边框
    OLED_DrawRect(0, 0, 128, 64);
    
    // 画圆
    OLED_DrawCircle(64, 32, 20);
    
    // 画线
    OLED_DrawLine(10, 10, 118, 54);
    
    // 画点
    for (int i = 0; i < 128; i += 4) {
        OLED_DrawPoint(i, 32 + sin(i * 0.1) * 10, 1);
    }
}
```

## 调试方法

### 显示测试

```c
// 全屏填充测试
void OLED_TestFill(void)
{
    // 全亮
    for (int i = 0; i < 1024; i++) {
        OLED_WriteData(0xFF);
    }
    HAL_Delay(1000);
    
    // 全灭
    for (int i = 0; i < 1024; i++) {
        OLED_WriteData(0x00);
    }
}
```

### 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 无显示 | 初始化失败 | 检查 I2C 地址 |
| 显示花屏 | 数据错误 | 检查时序 |
| 部分区域异常 | 显存映射错误 | 检查坐标计算 |
