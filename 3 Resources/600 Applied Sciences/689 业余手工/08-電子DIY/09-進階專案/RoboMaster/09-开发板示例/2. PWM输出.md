# PWM输出

> [!summary] 概述
> STM32 开发板 PWM 信号输出示例，用于电机控制、舵机控制和 LED 调光等应用。

---

## 功能说明

### 核心功能

- 多路 PWM 信号输出
- 可调频率和占空比
- 支持多种 PWM 模式

### 应用场景

| 场景 | 说明 |
|------|------|
| **电机控制** | 调节电机速度 |
| **舵机控制** | 控制舵机角度 |
| **LED 调光** | 调节 LED 亮度 |
| **音频输出** | 蜂鸣器驱动 |

---

## PWM 基础

### 参数说明

| 参数 | 说明 | 单位 |
|------|------|------|
| **频率** | PWM 信号每秒周期数 | Hz |
| **周期** | 一个完整波形的时间 | ms |
| **占空比** | 高电平时间/周期 | % |
| **分辨率** | 占空比调节精度 | bit |

### 常用频率

| 应用 | 推荐频率 |
|------|----------|
| 直流电机 | 10kHz ~ 20kHz |
| 舵机 | 50Hz (20ms 周期) |
| LED 调光 | 100Hz ~ 1kHz |
| 蜂鸣器 | 2kHz ~ 5kHz |

## 硬件接口

### PWM 输出引脚

| 通道 | GPIO | 定时器 | 说明 |
|------|------|--------|------|
| PWM1 | PA0 | TIM2_CH1 | 通用 PWM 输出 |
| PWM2 | PA1 | TIM2_CH2 | 通用 PWM 输出 |
| PWM3 | PA2 | TIM2_CH3 | 通用 PWM 输出 |
| PWM4 | PA3 | TIM2_CH4 | 通用 PWM 输出 |

### API 接口

#### 初始化

```c
/**
 * @brief 初始化 PWM
 * @param hz PWM 频率 (Hz)
 */
void PWM_Init(uint32_t hz);

/**
 * @brief 设置 PWM 频率
 * @param hz 频率 (Hz)
 */
void PWM_SetFrequency(uint32_t hz);
```

#### 占空比控制

```c
/**
 * @brief 设置 PWM 占空比
 * @param channel PWM 通道 (1-4)
 * @param duty 占空比 (0-1000, 对应 0%-100%)
 */
void PWM_SetDuty(uint8_t channel, uint16_t duty);

/**
 * @brief 设置 PWM 脉宽 (用于舵机)
 * @param channel PWM 通道
 * @param pulse_us 脉宽 (微秒)
 */
void PWM_SetPulse(uint8_t channel, uint16_t pulse_us);
```

## 使用示例

### 基础使用

```c
#include "pwm.h"

int main(void)
{
    // HAL 初始化
    HAL_Init();
    SystemClock_Config();
    
    // 初始化 PWM，频率 1kHz
    PWM_Init(1000);
    
    while (1)
    {
        // 设置 50% 占空比
        PWM_SetDuty(1, 500);
        
        // 呼吸灯效果
        for (int i = 0; i <= 1000; i += 10) {
            PWM_SetDuty(1, i);
            HAL_Delay(10);
        }
        for (int i = 1000; i >= 0; i -= 10) {
            PWM_SetDuty(1, i);
            HAL_Delay(10);
        }
    }
}
```

### 电机控制

```c
// 电机速度控制 (0-100%)
void Motor_SetSpeed(uint8_t motor_id, int8_t speed)
{
    if (speed >= 0) {
        // 正转
        PWM_SetDuty(motor_id * 2 - 1, speed * 10);
        PWM_SetDuty(motor_id * 2, 0);
    } else {
        // 反转
        PWM_SetDuty(motor_id * 2 - 1, 0);
        PWM_SetDuty(motor_id * 2, -speed * 10);
    }
}
```

### 舵机控制

```c
// 舵机角度控制 (0-180度)
void Servo_SetAngle(uint8_t channel, uint8_t angle)
{
    // 舵机脉宽范围: 500us(0°) - 2500us(180°)
    uint16_t pulse = 500 + (angle * 2000 / 180);
    PWM_SetPulse(channel, pulse);
}

// 使用示例
void Servo_Sweep(void)
{
    // 50Hz PWM，周期 20ms
    PWM_Init(50);
    
    while (1) {
        // 0° -> 180° -> 0°
        for (int i = 0; i <= 180; i += 5) {
            Servo_SetAngle(1, i);
            HAL_Delay(20);
        }
        for (int i = 180; i >= 0; i -= 5) {
            Servo_SetAngle(1, i);
            HAL_Delay(20);
        }
    }
}
```

## 配置说明

### STM32CubeMX 配置

1. 选择定时器通道
2. 设置模式为 PWM Generation
3. 配置参数：
   - Prescaler (PSC)
   - Counter Period (ARR)
   - Pulse (CCR)

### 计算公式

```
PWM频率 = 时钟频率 / ((PSC + 1) * (ARR + 1))
占空比 = CCR / (ARR + 1) * 100%
```

**示例配置** (72MHz 时钟, 1kHz PWM):

| 参数 | 值 |
|------|-----|
| PSC | 71 (72MHz / 72 = 1MHz) |
| ARR | 999 (1MHz / 1000 = 1kHz) |
| CCR | 0-999 (0%-100%) |

## 调试方法

### 示波器测量

1. **检查频率**
   - 测量一个完整周期时间
   - 验证频率是否符合设定值

2. **检查占空比**
   - 测量高电平时间
   - 计算占空比是否正确

3. **检查波形**
   - 上升沿/下降沿是否陡峭
   - 是否有抖动或噪声

### 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 无输出 | GPIO 未配置 | 检查 CubeMX 配置 |
| 频率不对 | 时钟配置错误 | 检查 SystemClock_Config |
| 占空比异常 | ARR 值设置错误 | 重新计算 ARR |
