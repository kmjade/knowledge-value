## 二、CAN 通讯协议

### 2.1 CAN 简介

**控制器局域网（Controller Area Network，CAN）** 是一种多主方式的串行通讯总线，具有高可靠性、实时性强的特点。

### 2.2 CAN 基本特性

| 特性 | 说明 |
|------|------|
| **传输速率** | 最高 1Mbps（40m 内） |
| **传输距离** | 最远 10km（5kbps） |
| **节点数量** | 理论上无限制 |
| **错误检测** | CRC、ACK、格式检查 |
| **仲裁机制** | 基于优先级的非破坏性仲裁 |

### 2.3 CAN 数据帧结构

```
标准 CAN 数据帧结构：
┌─────────┬──────┬─────────┬──────────┬────────┬──────┬─────────┐
│  帧起始  │ 仲裁域 │ 控制域   │  数据域   │ CRC域  │ ACK域 │  帧结束  │
│  SOF    │ ID    │ DLC     │ 0-8 Byte │ 15 bit │ 2 bit│  EOF    │
└─────────┴──────┴─────────┴──────────┴────────┴──────┴─────────┘
```

| 字段 | 长度 | 说明 |
|------|------|------|
| **SOF** | 1 bit | 帧起始位 |
| **ID** | 11 bit | 标准标识符 |
| **DLC** | 4 bit | 数据长度码（0-8） |
| **Data** | 0-64 bit | 数据域 |
| **CRC** | 15 bit | 循环冗余校验 |
| **ACK** | 2 bit | 应答位 |
| **EOF** | 7 bit | 帧结束 |

## 三、M3508 电调通信协议

### 3.1 发送控制报文（STM32 → 电调）

![电调接收报文格式](https://i-blog.csdnimg.cn/blog_migrate/23e2878df16208702063b41281a553e9.png)

控制 1-4 号电调，发送 ID 为 **0x200**：

| 字节 | 数据 | 说明 |
|------|------|------|
| Byte 0 | CH0_H | 电机1电流高8位 |
| Byte 1 | CH0_L | 电机1电流低8位 |
| Byte 2 | CH1_H | 电机2电流高8位 |
| Byte 3 | CH1_L | 电机2电流低8位 |
| Byte 4 | CH2_H | 电机3电流高8位 |
| Byte 5 | CH2_L | 电机3电流低8位 |
| Byte 6 | CH3_H | 电机4电流高8位 |
| Byte 7 | CH3_L | 电机4电流低8位 |

> [!tip] 控制多组电机
> - 控制 1-4 号电机：CAN ID = **0x200**
> - 控制 5-8 号电机：CAN ID = **0x1FF**

### 3.2 接收反馈报文（电调 → STM32）

每个电调独立发送反馈数据，ID 对应电机编号：

| 电调编号 | CAN ID | 说明 |
|----------|--------|------|
| 1号电机 | 0x201 | 反馈数据 |
| 2号电机 | 0x202 | 反馈数据 |
| 3号电机 | 0x203 | 反馈数据 |
| 4号电机 | 0x204 | 反馈数据 |
| ... | ... | ... |
| 8号电机 | 0x208 | 反馈数据 |

### 3.3 反馈数据格式

| 字节 | 数据 | 范围 | 说明 |
|------|------|------|------|
| Byte 0-1 | 机械角度 | 0-8191 | 转子机械角度（0~360°） |
| Byte 2-3 | 转速 | -32768~32767 | 转子转速（RPM） |
| Byte 4-5 | 实际转矩电流 | -16384~16384 | 实际电流值 |
| Byte 6 | 温度 | 0-255 | 电机温度（℃） |
| Byte 7 | 保留 | - | 预留字节 |

### 3.4 数据解析公式

```c
// 机械角度（0~8191 对应 0~360°）
uint16_t angle = (data[0] << 8) | data[1];

// 转速（RPM，有符号）
int16_t speed = (data[2] << 8) | data[3];

// 实际转矩电流（有符号）
int16_t current = (data[4] << 8) | data[5];

// 温度
uint8_t temp = data[6];
```

## 四、PID 控制基础

### 4.1 PID 简介

**PID 控制器** 是一种线性控制器，根据给定值与实际输出值构成控制偏差：

```
e(t) = r(t) - y(t)
```

PID 控制规律：

```
u(t) = Kp * e(t) + Ki * ∫e(t)dt + Kd * de(t)/dt
```

### 4.2 PID 各项作用

| 项 | 公式 | 作用 | 影响 |
|---|------|------|------|
| **P（比例）** | Kp × e | 即时响应误差 | 响应速度 |
| **I（积分）** | Ki × ∫e dt | 消除稳态误差 | 精度 |
| **D（微分）** | Kd × de/dt | 抑制误差变化 | 稳定性 |

### 4.3 PID 参数调节

> [!tip] 参数整定口诀
> 参数整定找最佳，从小到大顺序查
> 先是比例后积分，最后再把微分加
> 曲线振荡很频繁，比例度盘要放大
> 曲线漂浮绕大弯，比例度盘往小扳
> 曲线偏离回复慢，积分时间往下降
> 曲线波动周期长，积分时间再加长
> 曲线振荡频率快，先把微分降下来
> 动差大来波动慢，微分时间应加长

### 4.4 PID 控制框图

```
        ┌─────────┐      ┌───────┐      ┌─────────┐
 r(t) → │   PID   │ ───→ │ 电机  │ ───→ │  y(t)   │
        └────┬────┘      └───────┘      └─────────┘
             │                                │
             └────────────────────────────────┘
                        反馈回路
```

## 五、STM32CubeMX 配置

### 5.1 CAN 配置

#### 步骤 1：开启 CAN

在 Pinout & Configuration → Connectivity → CAN1：
- **Master Mode**：勾选

#### 步骤 2：参数配置

| 参数 | 值 | 说明 |
|------|-----|------|
| **Prescaler** | 3 | 预分频系数 |
| **Time Quanta** | 配置 | 时间份额 |
| **Time Quanta in Bit Segment 1** | 10 Times |
| **Time Quanta in Bit Segment 2** | 3 Times |
| **ReSynchronization Jump Width** | 1 Time |

#### 步骤 3：波特率计算

```
CAN 波特率 = APB1_Clock / (Prescaler × (SyncSeg + BS1 + BS2))

以 APB1 = 42MHz 为例：
波特率 = 42MHz / (3 × (1 + 10 + 3))
       = 42MHz / 42
       = 1MHz = 1000kbps
```

### 5.2 CAN 过滤器配置

## 六、代码实现

### 6.1 数据结构定义

```c
/* motor.h */

// 电机数据结构
typedef struct
{
    uint16_t angle;       // 机械角度 (0-8191)
    int16_t speed;        // 转速 (RPM)
    int16_t current;      // 实际电流
    uint8_t temp;         // 温度
    int16_t set_current;  // 设定电流
} Motor_t;

// PID 结构体
typedef struct
{
    float Kp;             // 比例系数
    float Ki;             // 积分系数
    float Kd;             // 微分系数
    float target;         // 目标值
    float error;          // 当前误差
    float last_error;     // 上次误差
    float integral;       // 积分项
    float max_integral;   // 积分限幅
    float output;         // 输出值
    float max_output;     // 输出限幅
} PID_t;

// 电机实例
extern Motor_t motor[4];
```

### 6.2 CAN 初始化

```c
/**
 * @brief  CAN 初始化
 */
void CAN_Init(void)
{
    // 配置过滤器
    CAN_FilterTypeDef sFilterConfig;
    sFilterConfig.FilterBank = 0;
    sFilterConfig.FilterMode = CAN_FILTERMODE_IDMASK;
    sFilterConfig.FilterScale = CAN_FILTERSCALE_32BIT;
    sFilterConfig.FilterIdHigh = 0x0000;
    sFilterConfig.FilterIdLow = 0x0000;
    sFilterConfig.FilterMaskIdHigh = 0x0000;
    sFilterConfig.FilterMaskIdLow = 0x0000;
    sFilterConfig.FilterFIFOAssignment = CAN_RX_FIFO0;
    sFilterConfig.FilterActivation = ENABLE;
    sFilterConfig.SlaveStartFilterBank = 14;

    HAL_CAN_ConfigFilter(&hcan1, &sFilterConfig);

    // 启动 CAN
    HAL_CAN_Start(&hcan1);

    // 使能接收中断
    HAL_CAN_ActivateNotification(&hcan1, CAN_IT_RX_FIFO0_MSG_PENDING);

```

### 6.3 电机数据解码

```c
/**
 * @brief  解析电机反馈数据
 * @param  id: CAN ID (0x201-0x208)
 * @param  data: 数据缓冲区
 */
void Motor_Decode(uint32_t id, uint8_t *data)
{
    uint8_t index = id - 0x201;  // 计算电机索引

    if (index > 7) return;  // 超出范围

    // 解析数据
    motor[index].angle = (data[0] << 8) | data[1];
    motor[index].speed = (int16_t)((data[2] << 8) | data[3]);
    motor[index].current = (int16_t)((data[4] << 8) | data[5]);
    motor[index].temp = data[6];
}
```

### 6.4 CAN 发送控制命令

```c
/**
 * @brief  发送电机电流控制命令
 * @param  id: CAN ID (0x200 或 0x1FF)
 */
void Motor_SendCurrent(uint32_t id)
{
    CAN_TxHeaderTypeDef txHeader;
    uint8_t txData[8];
    uint32_t txMailbox;

    // 设置帧头
    txHeader.StdId = id;
    txHeader.IDE = CAN_ID_STD;
    txHeader.RTR = CAN_RTR_DATA;
    txHeader.DLC = 8;

    // 填充数据（4个电机的电流值）
    txData[0] = (motor[0].set_current >> 8) & 0xFF;
    txData[1] = motor[0].set_current & 0xFF;
    txData[2] = (motor[1].set_current >> 8) & 0xFF;
    txData[3] = motor[1].set_current & 0xFF;
    txData[4] = (motor[2].set_current >> 8) & 0xFF;
    txData[5] = motor[2].set_current & 0xFF;
    txData[6] = (motor[3].set_current >> 8) & 0xFF;
    txData[7] = motor[3].set_current & 0xFF;

    // 发送数据
    HAL_CAN_AddTxMessage(&hcan1, &txHeader, txData, &txMailbox);
}
```

### 6.5 PID 控制

```c
/**
 * @brief  PID 初始化
 * @param  pid: PID 结构体指针
 * @param  Kp, Ki, Kd: PID 参数
 * @param  max_integral: 积分限幅
 * @param  max_output: 输出限幅
 */
void PID_Init(PID_t *pid, float Kp, float Ki, float Kd,
              float max_integral, float max_output)
{
    pid->Kp = Kp;
    pid->Ki = Ki;
    pid->Kd = Kd;
    pid->target = 0;
    pid->error = 0;
    pid->last_error = 0;
    pid->integral = 0;
    pid->max_integral = max_integral;
    pid->output = 0;
    pid->max_output = max_output;
}

/**
 * @brief  PID 计算
 * @param  pid: PID 结构体指针
 * @param  feedback: 反馈值
 * @retval 控制输出
 */
float PID_Calc(PID_t *pid, float feedback)
{
    // 计算误差
    pid->error = pid->target - feedback;

    // 积分项（带限幅）
    pid->integral += pid->error;
    if (pid->integral > pid->max_integral)
        pid->integral = pid->max_integral;
    else if (pid->integral < -pid->max_integral)
        pid->integral = -pid->max_integral;

    // PID 计算
    pid->output = pid->Kp * pid->error
                + pid->Ki * pid->integral
                + pid->Kd * (pid->error - pid->last_error);

    // 输出限幅
    if (pid->output > pid->max_output)
        pid->output = pid->max_output;
    else if (pid->output < -pid->max_output)
        pid->output = -pid->max_output;

    // 保存上次误差
    pid->last_error = pid->error;

    return pid->output;
}
```

### 6.6 电机速度控制

```c
/* motor_control.c */

// PID 控制器实例
PID_t speed_pid[4];

/**
 * @brief  电机速度控制初始化
 */
void Motor_Control_Init(void)
{
    // 初始化 PID 参数
    for (int i = 0; i < 4; i++)
    {
        // 典型 PID 参数（需根据实际情况调整）
        PID_Init(&speed_pid[i], 5.0f, 0.5f, 0.1f, 1000.0f, 16000.0f);
    }
}

/**
 * @brief  设置电机目标速度
 * @param  index: 电机索引 (0-3)
 * @param  speed: 目标速度 (RPM)
 */
void Motor_SetSpeed(uint8_t index, int16_t speed)
{
    if (index > 3) return;
    speed_pid[index].target = speed;
}

/**
 * @brief  电机控制循环（在定时器中断中调用）
 */
void Motor_Control_Loop(void)
{
    for (int i = 0; i < 4; i++)
    {
        // PID 计算输出电流
        float output = PID_Calc(&speed_pid[i], motor[i].speed);

        // 设置电机电流
        motor[i].set_current = (int16_t)output;
    }

    // 发送控制命令
    Motor_SendCurrent(0x200);
}
```

### 6.7 CAN 接收中断回调

```c
/**
 * @brief  CAN 接收 FIFO0 回调函数
 * @param  hcan: CAN 句柄
 */
void HAL_CAN_RxFifo0MsgPendingCallback(CAN_HandleTypeDef *hcan)
{
    CAN_RxHeaderTypeDef rxHeader;
    uint8_t rxData[8];

    // 获取接收数据
    if (HAL_CAN_GetRxMessage(hcan, CAN_RX_FIFO0, &rxHeader, rxData) == HAL_OK)
    {
        // 判断是否为电机反馈数据 (ID: 0x201-0x208)
        if (rxHeader.StdId >= 0x201 && rxHeader.StdId <= 0x208)
        {
            // 解析电机数据
            Motor_Decode(rxHeader.StdId, rxData);
        }
    }
}
```

## 八、调试与优化

### 8.1 PID 调试方法

> [!tip] 调试步骤
> 1. 先将 Ki、Kd 设为 0，调节 Kp
> 2. 增加 Kp 直到系统开始振荡，取 0.6 倍
> 3. 加入 Ki 消除稳态误差
> 4. 最后加入 Kd 提高动态响应

### 8.2 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 电机不转 | CAN 通信异常 | 检查接线和波特率 |
| 转速不稳 | PID 参数不当 | 重新整定 PID |
| 响应慢 | Kp 过小 | 增大 Kp |
| 震荡 | Kd 过大或过小 | 调整 Kd |
| 稳态误差 | Ki 过小 | 增大 Ki |

### 8.3 优化建议

1. **积分分离**：误差大时取消积分作用
2. **梯形积分**：提高积分精度
3. **抗积分饱和**：限制积分项输出
4. **前馈控制**：提高响应速度

```c
/**
 * @brief  带积分分离的 PID 计算
 */
float PID_Calc_Enhanced(PID_t *pid, float feedback)
{
    pid->error = pid->target - feedback;

    // 积分分离：误差较大时不积分
    if (fabs(pid->error) < 100)  // 误差阈值
    {
        pid->integral += pid->error;
        // 积分限幅
        pid->integral = fmax(fmin(pid->integral, pid->max_integral), -pid->max_integral);
    }

    // PID 计算
    pid->output = pid->Kp * pid->error
                + pid->Ki * pid->integral
                + pid->Kd * (pid->error - pid->last_error);

    // 输出限幅
    pid->output = fmax(fmin(pid->output, pid->max_output), -pid->max_output);

    pid->last_error = pid->error;

    return pid->output;
}
```
