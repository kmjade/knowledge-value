# 3. 传感任务——姿态角解算

> [!summary] 概述
> 本文介绍如何使用 STM32 实现四驱小车的姿态角测量，涵盖惯性传感器原理、姿态估计方法（互补滤波）、传感器数据读取与滤波处理等内容。

---

## 一、绪论

本文接上一篇 [[2. 通信任务——遥控器SBUS通信]]，在今天的文章中解决小车的姿态角测量问题，实现操作系统中的传感任务。

对小车来说只需要测量航向角（YAW），方法就是使用姿态角传感器，通过串口直接读三轴角度。但是为了知识体系的完整性，本文会从传感器测量原理到姿态估计讲起，讲解标准对标多旋翼的姿态解算与状态估计。

> [!tip] 阅读建议
> 本文的结构是"高高拿起，轻轻放下"。如果只是想实现功能，看最后二节代码部分就行了。

## 二、惯性传感器测量原理

### 2.1 三轴加速度计

三轴加速度计是一种惯性传感器，能够测量物体的**比力**，即去掉重力后的整体加速度。

#### 核心特性

| 特性 | 说明 |
|------|------|
| **测量原理** | 弹簧压缩量由加速度计与地面的角度决定 |
| **核心功能** | **测量角度** |
| **优势** | 无累计误差 |
| **劣势** | 动态响应差、噪声大 |
| **频率特性** | **低频特性好，高频特性差** |

> [!important] 关键结论
> **在没有外力作用的情况下，加速度计能够准确的测量俯仰角和滚转角，且没有累计误差。** 这是实现互补滤波、卡尔曼滤波的基础。

![三轴加速计](https://i-blog.csdnimg.cn/blog_migrate/486ca0535dd402215706ca4713deb222.png)

#### 角度计算公式

加速度计近似测量俯仰角和滚转角的公式：

```
θ_pitch = arctan(ax / √(ay² + az²))
φ_roll  = arctan(ay / √(ax² + az²))
```

### 2.2 三轴陀螺仪

MEMS 陀螺仪是基于科里奥利力工作的传感器。

#### 核心特性

| 特性 | 说明 |
|------|------|
| **测量原理** | 科里奥利力正比于角速度 |
| **测量对象** | 三轴角速度 |
| **角度获取** | 角速度积分 |
| **优势** | 动态响应快、测量精度高 |
| **劣势** | 积分漂移（累计误差） |
| **频率特性** | **高频特性好，低频特性差** |

#### 工作原理

```
科里奥利力 → 电容变化 → 角速度计算 → 积分得角度
```

### 2.3 三轴磁力计

磁力计是提供导航及基于位置服务的重要组成部分。

#### 核心特性

| 特性 | 说明 |
|------|------|
| **测量原理** | 各向异性磁致电阻 / 霍尔效应 / 洛伦兹力 |
| **测量对象** | 三轴磁场强度 |
| **核心功能** | **测量偏航角** |
| **应用** | 导航、位置服务 |

> [!note] 滤波作用
> 磁力计测量的偏航角在滤波中将会作为观测值。

## 三、传感器特性对比

### 3.1 特性总结

| 传感器 | 测量量 | 优势 | 劣势 | 频率特性 |
|--------|--------|------|------|----------|
| **加速度计** | 比力 → 角度 | 无漂移 | 噪声大 | 低频好 |
| **陀螺仪** | 角速度 | 响应快、精度高 | 积分漂移 | 高频好 |
| **磁力计** | 磁场 → 航向角 | 无漂移 | 易受干扰 | 低频好 |

### 3.2 融合原理

```
┌─────────────┐                    ┌─────────────┐
│  加速度计    │                    │   陀螺仪    │
│ (低频好)    │                    │ (高频好)    │
│ (无漂移)    │                    │ (有漂移)    │
└──────┬──────┘                    └──────┬──────┘
       │                                  │
       │         ┌──────────────┐         │
       └────────→│  互补滤波     │←────────┘
                 │  卡尔曼滤波   │
                 └──────┬───────┘
                        │
                        ▼
                 ┌──────────────┐
                 │  姿态角估计   │
                 └──────────────┘
```

## 四、状态估计

状态估计是控制与决策的基础，具有十分重要的作用。

### 4.1 状态估计类型

| 类型 | 说明 | 传感器 |
|------|------|--------|
| **姿态估计** | 估计姿态角 | IMU |
| **速度估计** | 估计速度 | IMU + GPS/光流 |
| **位置估计** | 估计位置 | GPS/视觉/UWB |

### 4.2 姿态角的表示方法

| 表示方法 | 说明 | 特点 |
|----------|------|------|
| **欧拉角** | Roll/Pitch/Yaw | 直观，有万向锁 |
| **旋转矩阵** | 3×3 矩阵 | 无奇异性，冗余 |
| **四元数** | 4 元素 | 无奇异性，计算高效 |

## 五、姿态估计方法

### 5.1 传感器特性分析

姿态信息主要利用三轴加速度计、三轴陀螺仪、三轴磁力计的数据，通过互补滤波或卡尔曼滤波获得。

**传感器特性：**

| 传感器 | 特性 |
|--------|------|
| **陀螺仪** | 动态响应快，测量精度高，但求解姿态角时需要对角速度积分，会产生累计误差（积分漂移） |
| **加速度计/磁力计** | 可以得到稳定的姿态角，无累计误差，但动态响应特性差、测量噪声大 |

### 5.2 线性互补滤波器

#### 基本定义

三轴姿态角向量：**s = [φ, θ, ψ]ᵀ**（滚转角、俯仰角、偏航角）

机体旋转角速度：**ω = [p, q, r]ᵀ**

#### 俯仰角互补滤波

俯仰角的拉氏变换表示为：

```
θ̂(s) = (1 / (τs + 1)) · θ_acc(s) + (τs / (τs + 1)) · θ_gyro(s)
```

其中：
- `1 / (τs + 1)` 表示低通滤波器的传递函数
- `τ` 表示时间常数
- `τs / (τs + 1)` 表示高通滤波器的传递函数

**加速度计测量模型：**

```
θ_acc = θ_true + n_high
```

- `n_high` 表示高频噪声
- 加速度计测量得到的俯仰角无漂移，但噪声大

**陀螺仪积分模型：**

```
θ_gyro = θ_true + c/s
```

- `c/s` 表示常值漂移
- 角速度积分得到的俯仰角噪声小，但漂移大

#### 线性互补滤波器标准形式

```
θ̂(s) = [τs / (τs + 1)] · θ_gyro(s) + [1 / (τs + 1)] · θ_acc(s)
```

### 5.3 非线性互补滤波器

非线性互补滤波器直接在时域进行融合，计算更简单：

```c
// 俯仰角更新
theta = theta + (q - theta_acc * K) * dt;

// 滚转角更新
phi = phi + (p - phi_acc * K) * dt;

// 其中 K 为滤波增益系数
```

### 5.4 卡尔曼滤波器

卡尔曼滤波器是一种最优估计器，能够融合多个传感器数据。

#### 基本结构

```
预测步骤：
  x̂⁻ = A·x̂ + B·u
  P⁻ = A·P·Aᵀ + Q

更新步骤：
  K = P⁻·Hᵀ·(H·P⁻·Hᵀ + R)⁻¹
  x̂ = x̂⁻ + K·(z - H·x̂⁻)
  P = (I - K·H)·P⁻
```

#### 状态向量

```
x = [θ, θ̇]ᵀ  // 角度 + 角速度
```

## 六、姿态角测量实现

### 6.1 方案一：JY901B 模块

JY901B 是一款集成姿态解算的 IMU 模块，直接输出姿态角。

#### 通信参数

| 参数 | 值 |
|------|-----|
| **接口** | UART / I2C |
| **波特率** | 9600 / 115200 |
| **输出** | Roll / Pitch / Yaw |
| **更新频率** | 最高 200Hz |

#### 数据格式

```c
// JY901B 数据帧格式
typedef struct {
    float roll;     // 滚转角
    float pitch;    // 俯仰角
    float yaw;      // 偏航角
    float gx;       // 陀螺仪 X
    float gy;       // 陀螺仪 Y
    float gz;       // 陀螺仪 Z
    float ax;       // 加速度 X
    float ay;       // 加速度 Y
    float az;       // 加速度 Z
} JY901_Data_t;
```

### 6.2 方案二：十轴惯导模块

使用原始传感器数据，自行实现姿态解算。

#### 常见芯片

| 芯片 | 类型 | 接口 |
|------|------|------|
| **MPU9250** | 九轴 (IMU + 磁力计) | I2C / SPI |
| **BMI088** | 六轴 IMU | SPI |
| **IST8310** | 磁力计 | I2C |

## 七、滑动平均滤波

### 7.1 滤波原理

滑动平均滤波是对一段时间内的数据进行平均，消除随机噪声。

```c
// 滑动平均滤波
#define FILTER_SIZE 10

typedef struct {
    float buffer[FILTER_SIZE];
    int index;
    float sum;
} MovingAverage_t;

float MovingAverage_Update(MovingAverage_t *filter, float new_value)
{
    filter->sum -= filter->buffer[filter->index];
    filter->buffer[filter->index] = new_value;
    filter->sum += new_value;
    filter->index = (filter->index + 1) % FILTER_SIZE;
    return filter->sum / FILTER_SIZE;
}
```

### 7.2 角度和角速度滤波

```c
// IMU 数据结构
typedef struct {
    float yaw;              // 偏航角
    float yaw_rate;         // 偏航角速度
    float pitch;            // 俯仰角
    float pitch_rate;       // 俯仰角速度
    float roll;             // 滚转角
    float roll_rate;        // 滚转角速度
} IMU_Data_t;

IMU_Data_t imu_data;

// 滤波器
MovingAverage_t yaw_filter;
MovingAverage_t yaw_rate_filter;

// 更新滤波后的角度
void IMU_Update(void)
{
    // 读取原始数据
    float raw_yaw = JY901_ReadYaw();
    float raw_yaw_rate = JY901_ReadYawRate();
    
    // 滤波
    imu_data.yaw = MovingAverage_Update(&yaw_filter, raw_yaw);
    imu_data.yaw_rate = MovingAverage_Update(&yaw_rate_filter, raw_yaw_rate);
}
```

## 八、传感任务实现

### 8.1 任务框架

```c
// 传感任务
u8 sensor_task(void *p_arg)
{
    OS_ERR err;
    CPU_SR_ALLOC();
    
    // 初始化滤波器
    MovingAverage_Init(&yaw_filter);
    MovingAverage_Init(&yaw_rate_filter);
    
    while (1)
    {
        // 读取 IMU 数据
        IMU_Update();
        
        // 数据可用于其他任务
        // (通过全局变量或消息队列传递)
        
        delay_ms(10);  // 100Hz 更新率
    }
}
```

### 8.2 串口读取 JY901B

```c
// UART 接收中断
void USART2_IRQHandler(void)
{
    static u8 rx_buffer[11];
    static u8 rx_index = 0;
    
    if (USART_GetITStatus(USART2, USART_IT_RXNE) != RESET)
    {
        u8 data = USART_ReceiveData(USART2);
        
        // 解析数据帧
        if (rx_index == 0 && data == 0x55)
        {
            rx_buffer[rx_index++] = data;
        }
        else if (rx_index > 0)
        {
            rx_buffer[rx_index++] = data;
            
            if (rx_index >= 11)
            {
                // 校验并解析
                if (CheckSum(rx_buffer) == rx_buffer[10])
                {
                    ParseJY901Data(rx_buffer);
                }
                rx_index = 0;
            }
        }
    }
}
```

### 8.3 I2C 读取 MPU9250

```c
// 读取加速度计
void MPU9250_ReadAccel(float *ax, float *ay, float *az)
{
    u8 buffer[6];
    
    I2C_ReadBytes(MPU9250_ADDR, ACCEL_XOUT_H, buffer, 6);
    
    int16_t accel_x = (buffer[0] << 8) | buffer[1];
    int16_t accel_y = (buffer[2] << 8) | buffer[3];
    int16_t accel_z = (buffer[4] << 8) | buffer[5];
    
    // 转换为 g
    *ax = accel_x / 16384.0f;
    *ay = accel_y / 16384.0f;
    *az = accel_z / 16384.0f;
}

// 读取陀螺仪
void MPU9250_ReadGyro(float *gx, float *gy, float *gz)
{
    u8 buffer[6];
    
    I2C_ReadBytes(MPU9250_ADDR, GYRO_XOUT_H, buffer, 6);
    
    int16_t gyro_x = (buffer[0] << 8) | buffer[1];
    int16_t gyro_y = (buffer[2] << 8) | buffer[3];
    int16_t gyro_z = (buffer[4] << 8) | buffer[5];
    
    // 转换为 deg/s
    *gx = gyro_x / 131.0f;
    *gy = gyro_y / 131.0f;
    *gz = gyro_z / 131.0f;
}
```

## 十一、调试技巧

### 11.1 验证 IMU 数据

1. 静止状态：角度应稳定，角速度接近 0
2. 旋转测试：角度变化与实际旋转一致
3. 长时间测试：观察是否有漂移

### 11.2 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 角度跳变 | 数据帧校验失败 | 加强校验逻辑 |
| 静止漂移 | 陀螺仪零偏 | 加上电校准 |
| 响应延迟 | 滤波窗口过大 | 减小 FILTER_SIZE |
| 通信失败 | 波特率/地址错误 | 检查配置参数 |
