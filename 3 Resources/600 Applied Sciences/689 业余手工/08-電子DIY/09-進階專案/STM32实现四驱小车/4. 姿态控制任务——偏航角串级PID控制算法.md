# 4. 姿态控制任务——偏航角串级PID控制算法

> [!summary] 概述
> 本文深入探讨串级 PID 控制原理及其在 STM32 上的实现方法，重点介绍角度环和角速度环的串级 PID 控制策略，并通过 UCOS-III 实现小车的姿态控制任务。

---

## 一、绪论

这一部分是核心内容，讲解姿态角的串级 PID 控制。在智能小车、四旋翼、四足机器人等一系列机器人的控制系统中，姿态控制（俯仰角、滚转角、偏航角）都是核心内容，它决定了小车开得直不直，飞机飞得稳不稳。

虽然现在先进的、智能的控制算法有很多，如自适应控制、神经网络控制、模糊控制等在机器人控制系统的设计上有了很多应用，但是**最常用的最好用的依然是 PID 控制器**，搞通了 PID 控制器就能够应付绝大多数场合了。

本文续接上一篇 [[3. 传感任务——姿态角解算]]。

## 二、PID 控制器原理

### 2.1 基本算法

PID 控制器是一种线性控制器，根据给定值和实际输出值的偏差构成控制偏差。

![PID 控制器](https://i-blog.csdnimg.cn/blog_migrate/f795c05bbcd55f00ef7293f6d8761e88.png)

**控制偏差：**

```
e(t) = r(t) - y(t)
```

**PID 控制律：**

```
u(t) = Kp·e(t) + Ki·∫e(t)dt + Kd·de(t)/dt
```

或表示为：

```
u(t) = Kp·[e(t) + (1/Ti)·∫e(t)dt + Td·de(t)/dt]
```

其中：
- **Kp** 为比例系数
- **Ti** 为积分时间常数
- **Td** 为微分时间常数

### 2.2 各环节作用

| 环节 | 作用 | 特点 |
|------|------|------|
| **比例（P）** | 成比例反应偏差信号 | 偏差一旦产生立即控制，**不能消除稳态误差** |
| **积分（I）** | 消除静差，提高无差度 | 积分时间常数越大，作用越弱 |
| **微分（D）** | 反映偏差变化趋势 | 超前校正，加快响应，**噪声大时慎用** |

### 2.3 PID 参数整定经验

> [!tip] 参数整定步骤
> 1. **比例环节**：从小到大逐渐调整，直到系统有发散趋势，然后往回取适中值
> 2. **积分环节**：确定比例系数后，从小到大增大积分系数，直到有发散趋势
> 3. **微分环节**：延迟要求不高时可不加，噪声大时会放大噪声

### 2.4 数字 PID 控制

#### 位置式 PID

```c
u(k) = Kp·e(k) + Ki·Σe(j) + Kd·[e(k) - e(k-1)]
```

**特点：** 输出是执行机构的实际位置

```c
// 位置式 PID 实现
typedef struct {
    float Kp;           // 比例系数
    float Ki;           // 积分系数
    float Kd;           // 微分系数
    float target;       // 目标值
    float error;        // 当前误差
    float last_error;   // 上次误差
    float integral;     // 积分累计
    float output;       // 输出值
    float integral_limit;  // 积分限幅
    float output_limit;    // 输出限幅
} PID_t;

float PID_Calculate(PID_t *pid, float feedback)
{
    pid->error = pid->target - feedback;
    
    // 积分
    pid->integral += pid->error;
    
    // 积分限幅
    if (pid->integral_limit != 0)
    {
        if (pid->integral > pid->integral_limit)
            pid->integral = pid->integral_limit;
        else if (pid->integral < -pid->integral_limit)
            pid->integral = -pid->integral_limit;
    }
    
    // PID 计算
    pid->output = pid->Kp * pid->error 
                + pid->Ki * pid->integral 
                + pid->Kd * (pid->error - pid->last_error);
    
    // 输出限幅
    if (pid->output_limit != 0)
    {
        if (pid->output > pid->output_limit)
            pid->output = pid->output_limit;
        else if (pid->output < -pid->output_limit)
            pid->output = -pid->output_limit;
    }
    
    pid->last_error = pid->error;
    
    return pid->output;
}
```

#### 增量式 PID

```c
Δu(k) = Kp·[e(k) - e(k-1)] + Ki·e(k) + Kd·[e(k) - 2e(k-1) + e(k-2)]
```

**特点：** 输出是控制量的增量

```c
// 增量式 PID 实现
typedef struct {
    float Kp;
    float Ki;
    float Kd;
    float target;
    float error;
    float last_error;
    float prev_error;    // 前次误差
    float output;
    float output_limit;
} PID_Inc_t;

float PID_Inc_Calculate(PID_Inc_t *pid, float feedback)
{
    pid->error = pid->target - feedback;
    
    // 增量计算
    float delta = pid->Kp * (pid->error - pid->last_error)
                + pid->Ki * pid->error
                + pid->Kd * (pid->error - 2 * pid->last_error + pid->prev_error);
    
    pid->output += delta;
    
    // 输出限幅
    if (pid->output_limit != 0)
    {
        if (pid->output > pid->output_limit)
            pid->output = pid->output_limit;
        else if (pid->output < -pid->output_limit)
            pid->output = -pid->output_limit;
    }
    
    pid->prev_error = pid->last_error;
    pid->last_error = pid->error;
    
    return pid->output;
}
```

## 三、串级 PID 控制原理

### 3.1 单环 vs 串级

| 类型 | 结构 | 特点 |
|------|------|------|
| **单环 PID** | 目标 → PID → 执行器 | 简单，响应慢 |
| **串级 PID** | 外环 → 内环 → 执行器 | 响应快，抗干扰强 |

### 3.2 姿态角串级 PID 结构

```
┌─────────────────────────────────────────────────────────────────────┐
│                        串级 PID 控制结构                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  目标角度 ──→ [+ ] ──→ 角度环PID ──→ 目标角速度 ──→ [+ ]            │
│              (-)        (外环)                      (-)             │
│               │                                        │            │
│               │                                        ▼            │
│            实际角度 ◄─────────────────────────────── 实际角速度     │
│                        (IMU测量)                        (IMU测量)   │
│                                                                     │
│                                       目标角速度 ──→ 角速度环PID    │
│                                                     (内环)          │
│                                                          │          │
│                                                          ▼          │
│                                                     电机控制量       │
│                                                          │          │
│                                                          ▼          │
│                                                       电机执行       │
└─────────────────────────────────────────────────────────────────────┘
```

### 3.3 串级 PID 优势

| 优势 | 说明 |
|------|------|
| **响应更快** | 内环角速度环响应快，快速抑制扰动 |
| **控制精度高** | 外环角度环保证稳态精度 |
| **抗干扰强** | 内环可快速消除角速度扰动 |
| **解耦效果好** | 角度和角速度分别控制 |

### 3.4 偏航角串级 PID 原理

**外环（角度环）：**
- 输入：目标偏航角 - 实际偏航角
- 输出：目标偏航角速度
- 作用：保证角度跟踪精度

**内环（角速度环）：**
- 输入：目标偏航角速度 - 实际偏航角速度
- 输出：电机差速控制量
- 作用：快速响应，抑制扰动

## 四、串级 PID 代码实现

### 4.1 串级 PID 结构体

```c
// 串级 PID 结构体
typedef struct
{
    // 外环（角度环）
    PID_t angle_pid;
    
    // 内环（角速度环）
    PID_t rate_pid;
    
    // 输出
    float output;
    
} CascadePID_t;
```

### 4.2 串级 PID 计算

```c
/**
 * @brief  串级 PID 计算
 * @param  pid: 串级 PID 结构体
 * @param  angle_target: 目标角度
 * @param  angle_feedback: 实际角度
 * @param  rate_feedback: 实际角速度
 * @return 控制输出
 */
float CascadePID_Calculate(CascadePID_t *pid, 
                           float angle_target, 
                           float angle_feedback, 
                           float rate_feedback)
{
    // 外环：角度环，输出目标角速度
    pid->angle_pid.target = angle_target;
    float rate_target = PID_Calculate(&pid->angle_pid, angle_feedback);
    
    // 内环：角速度环，输出控制量
    pid->rate_pid.target = rate_target;
    pid->output = PID_Calculate(&pid->rate_pid, rate_feedback);
    
    return pid->output;
}
```

### 4.3 PID 初始化

```c
// 偏航角串级 PID
CascadePID_t yaw_cascade_pid;

void YawPID_Init(void)
{
    // 角度环参数
    yaw_cascade_pid.angle_pid.Kp = 2.0f;
    yaw_cascade_pid.angle_pid.Ki = 0.0f;    // 角度环一般不加积分
    yaw_cascade_pid.angle_pid.Kd = 0.0f;
    yaw_cascade_pid.angle_pid.integral_limit = 0;
    yaw_cascade_pid.angle_pid.output_limit = 200.0f;  // 最大角速度
    
    // 角速度环参数
    yaw_cascade_pid.rate_pid.Kp = 5.0f;
    yaw_cascade_pid.rate_pid.Ki = 0.1f;
    yaw_cascade_pid.rate_pid.Kd = 0.0f;
    yaw_cascade_pid.rate_pid.integral_limit = 500.0f;
    yaw_cascade_pid.rate_pid.output_limit = 1000.0f;  // 最大输出
}
```

## 五、姿态控制任务实现

### 5.1 控制流程

```
遥控器输入 ──→ 目标偏航角
                    │
                    ▼
              ┌──────────┐
              │ 角度环PID │ ──→ 目标角速度
              └──────────┘        │
                    ▲             ▼
                    │       ┌────────────┐
               实际偏航角   │ 角速度环PID │ ──→ 差速控制量
                    │       └────────────┘
                    │             ▲
                    │             │
               IMU测量 ◄───── 实际角速度
                              (IMU测量)
```

### 5.2 任务代码实现

```c
// 姿态控制任务
void stabilization_task(void *p_arg)
{
    OS_ERR err;
    CPU_SR_ALLOC();
    
    float yaw_target = 0.0f;      // 目标偏航角
    float yaw_output = 0.0f;      // 控制输出
    
    // 初始化 PID
    YawPID_Init();
    
    while (1)
    {
        // 获取目标偏航角（从遥控器）
        // 这里假设遥控器摇杆控制偏航角速度，积分得到目标角度
        float yaw_rate_cmd = sbus_to_Range(command[YAW], -100, 100);  // deg/s
        yaw_target += yaw_rate_cmd * 0.01f;  // 积分，10ms周期
        
        // 获取 IMU 数据
        float yaw_actual = imu_data.yaw;
        float yaw_rate_actual = imu_data.yaw_rate;
        
        // 串级 PID 计算
        yaw_output = CascadePID_Calculate(&yaw_cascade_pid, 
                                          yaw_target, 
                                          yaw_actual, 
                                          yaw_rate_actual);
        
        // 输出到电机差速控制
        // yaw_output > 0: 左转
        // yaw_output < 0: 右转
        Motor_SetYawRate(yaw_output);
        
        delay_ms(10);  // 100Hz 控制频率
    }
}
```

### 5.3 电机差速控制

```c
// 基础速度（前进）
float base_speed = 500.0f;

// 根据偏航控制量设置电机差速
void Motor_SetYawRate(float yaw_rate)
{
    // 左右电机差速
    float left_speed = base_speed - yaw_rate;
    float right_speed = base_speed + yaw_rate;
    
    // 设置电机速度
    Motor_SetSpeed(MOTOR_LEFT_FRONT, left_speed);
    Motor_SetSpeed(MOTOR_LEFT_REAR, left_speed);
    Motor_SetSpeed(MOTOR_RIGHT_FRONT, right_speed);
    Motor_SetSpeed(MOTOR_RIGHT_REAR, right_speed);
}
```

### 5.4 完整控制任务

```c
// 姿态控制任务（完整版）
void stabilization_task(void *p_arg)
{
    OS_ERR err;
    CPU_SR_ALLOC();
    
    float yaw_target = 0.0f;
    float pitch_target = 0.0f;
    float roll_target = 0.0f;
    
    // 初始化所有 PID
    YawPID_Init();
    PitchPID_Init();
    RollPID_Init();
    
    while (1)
    {
        // ========== 获取遥控器指令 ==========
        // 偏航角速度指令
        float yaw_rate_cmd = sbus_to_Range(command[YAW], -100, 100);
        
        // 俯仰/滚转角指令（可选）
        // ...
        
        // 更新目标角度
        yaw_target += yaw_rate_cmd * 0.01f;
        
        // ========== 读取 IMU 数据 ==========
        float yaw = imu_data.yaw;
        float yaw_rate = imu_data.yaw_rate;
        
        // ========== 串级 PID 控制 ==========
        float yaw_output = CascadePID_Calculate(&yaw_cascade_pid,
                                                 yaw_target, yaw, yaw_rate);
        
        // ========== 电机输出 ==========
        Motor_SetYawRate(yaw_output);
        
        // 10ms 周期，100Hz 控制频率
        delay_ms(10);
    }
}
```

## 六、PID 参数整定指南

### 6.1 整定步骤

```
步骤1: 内环（角速度环）整定
   ├── 设置角速度目标，观察响应
   ├── 先调 Kp，从小到大
   ├── 再调 Ki，消除静差
   └── Kd 一般不加

步骤2: 外环（角度环）整定
   ├── 设置角度目标，观察跟踪
   ├── 只调 Kp
   └── 角度环一般不加 I 和 D

步骤3: 联调优化
   ├── 联合测试
   ├── 微调参数
   └── 检查稳定性
```

### 6.2 参数参考值

| 控制对象 | 外环 Kp | 内环 Kp | 内环 Ki |
|----------|---------|---------|---------|
| **小型四驱车** | 1.0~3.0 | 3.0~8.0 | 0.05~0.2 |
| **RoboMaster** | 2.0~5.0 | 5.0~15.0 | 0.1~0.5 |
| **四旋翼** | 3.0~8.0 | 8.0~20.0 | 0.1~0.3 |

### 6.3 常见问题与解决

| 现象 | 原因 | 解决方案 |
|------|------|----------|
| 持续振荡 | Kp 过大 | 减小 Kp |
| 响应太慢 | Kp 太小 | 增大 Kp |
| 有静差 | 无积分作用 | 增大 Ki |
| 输出饱和 | 积分饱和 | 添加积分限幅 |
| 高频震荡 | Kd 过大 | 减小或去掉 Kd |
