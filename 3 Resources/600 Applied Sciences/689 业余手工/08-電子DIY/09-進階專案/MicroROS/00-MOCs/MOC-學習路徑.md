# MOC-學習路徑

> [!info] 概述
> MicroROS 機器人控制板系統化學習路徑，從零基礎到機器人控制實踐。

---

## 學習階段總覽

```
第一階段：開發環境（1-2週）
    ↓
第二階段：ESP32基礎（2-3週）
    ↓
第三階段：MicroROS基礎（2-3週）
    ↓
第四階段：ROS2課程（3-4週）
    ↓
第五階段：機器人控制（持續）
```

## 學習建議

1. **動手實踐**：每學完一個例程都要親自燒錄測試
2. **記錄筆記**：記錄遇到的問題和解決方案
3. **循序漸進**：按順序學習，不要跳躍
4. **理論結合實踐**：理解原理後再動手編碼
