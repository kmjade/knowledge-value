# UGV01 vs UGV02 對比與選型指南 Comparison & Selection Guide

> 兩款微雪 (Waveshare) 開源機器人底盤——履帶旗艦 vs 六輪猛獸。

## 一句話總結 One-Liner

| 型號 | 定位 |
|------|------|
| **UGV01** | 履帶旗艦——多獨立懸掛，極限越野 |
| **UGV02** | 六輪猛獸——橡膠輪胎，四驅重載 |

## 核心差異 Key Differences

| 維度 | UGV01 | UGV02 |
|------|-------|-------|
| **底盤結構** | 履帶式 | 六輪四驅 |
| **懸掛系統** | 每側 4 個獨立懸掛 | 橡膠輪胎自然減震 |
| **驅動輪數** | 2 驅（後輪驅動） | 4 驅（後四輪驅動） |
| **輪胎** | 橡膠履帶 | 柔軟橡膠輪胎 |
| **載重能力** | 中等 | 更強（六輪分攤） |
| **爬坡角度** | 極佳（履帶抓地） | 優秀（四驅推力） |
| **轉彎半徑** | 較大（履帶摩擦） | 較小（差速轉向） |
| **速度** | 中等 | 較快 |
| **噪音** | 較大（履帶噪音） | 較小 |
| **適合地形** | 泥地、沙地、碎石 | 平地、草地、輕越野 |
| **底盤尺寸** | 緊湊型 | 更大（六輪） |
| **擴展導軌** | 1020 歐標鋁型材 ×2 | 1020 鋁型材 ×2（203mm 間距） |

## 共同特性 Shared Features

兩款產品共享以下核心架構：

### 硬體平台

| 組件 | 規格 |
|------|------|
| **下位機** | ESP32（非 ESP32-S3） |
| **驅動板** | General Driver for Robots |
| **顯示** | 0.96" OLED |
| **IMU** | 九軸（加速度 + 陀螺儀 + 磁力計） |
| **儲存** | TF 卡槽 |
| **電池** | 3S 18650 鋰電池（3 串聯），7800mAh |
| **充電** | 12.6V 2A，支援邊充邊放 |
| **電源監控** | INA219 晶片（電壓 / 電流即時監控） |
| **保護** | 防過充、防過放、防過流、短路保護 |
| **擴展輸出** | 5V / 3.3V 供外部設備 |

### 軟體平台

| 特性 | 說明 |
|------|------|
| **開發環境** | Arduino IDE（無需手動配置） |
| **WiFi 模式** | AP（預設熱點 UGV / 密碼 12345678）+ STA |
| **Web 控制** | `192.168.4.1`，基於 Chromium 瀏覽器 |
| **通訊協議** | JSON 指令集（統一格式） |
| **通訊方式** | 串口 (115200) / USB / HTTP / ESP-NOW |
| **心跳機制** | 3 秒無移動指令自動停止 |
| **代碼開源** | 全部開源，含 WebPage.h 可自定義 UI |

### 上位機支援

兩款均支援以下上位機（通過串口連接）：
- **樹莓派** (Raspberry Pi 4B/5)
- **NVIDIA Jetson Nano / Orin Nano**
- **地平線旭日 X3 派** (RDK X3)
- 任何支援 UART 串口的 Linux/Windows/Mac 設備

## OLED 顯示對比

| 行 | UGV01 | UGV02 |
|----|-------|-------|
| 1 | AP 模式 + 熱點名 (UGV) | AP 模式 + 熱點名 (UGV) |
| 2 | STA 模式狀態 | STA 模式狀態 |
| 3 | MAC 地址 | 型號 (UGV Rover) |
| 4 | 電源電壓 | 電壓 + 設備類型編碼 |

UGV02 第四行 "s 20" 編碼含義：
- s = 設備類型設置
- 第一個數字：2=UGV Rover, 1=RaspRover, 3=UGV Beast
- 第二個數字：0=無擴展, 1=RoArm-M2, 2=Camera PT

## 選型建議 Selection Guide

| 場景 | 推薦 | 原因 |
|------|------|------|
| 🏔️ 極限越野（泥地、沙地） | **UGV01** | 履帶 + 獨立懸掛 = 無敵通過性 |
| 🏢 室內/平地巡檢 | **UGV02** | 橡膠輪安靜、不傷地板 |
| 📦 重載運輸 | **UGV02** | 六輪分散負重，四驅推力強 |
| 🎓 教學/實驗 | **UGV01** | 結構更簡單直觀，兩輪差速易理解 |
| 🤖 ROS 機器人開發 | 兩者皆可 | 均支援 ROS 控制指令 (T=2) |
| 🎯 精準定位 | **UGV02** | 橡膠輪 + 四驅 = 更精確里程計 |
| ⚡ 高速運動 | **UGV02** | 橡膠輪滾動阻力小 |
| 🔇 低噪音要求 | **UGV02** | 橡膠輪遠比履帶安靜 |

## JSON 指令相容性

大部分 JSON 指令兩者通用，少數指令的 T 值在固件版本間有差異。詳見 [[UGV-JSON指令集完整手冊|JSON 指令集完整手冊]]。

## 上位機安裝（通用流程）

```bash
# 適用於樹莓派 (UGV01/UGV02)
cd ugv_rpi/
sudo chmod +x setup.sh autorun.sh
sudo ./setup.sh          # 國際用戶
sudo ./setup.sh -i       # 國內用戶（使用國內鏡像源）
./autorun.sh             # 配置開機自啟動
```

WiFi 配置工具：
```bash
cd ugv_rpi/AccessPopup/
sudo chmod +x installconfig.sh
sudo ./installconfig.sh
```
