# 2. RoboMaster 明文 SDK 接入方式

> [!info] 本章概述
> 介绍 RoboMaster 明文 SDK 的多种连接方式，包括 WiFi 直连、WiFi 路由器、USB 和 UART 连接。

---

## 2.1 连接方式概览

机器人支持多种连接方式，可通过任意一种连接方式接入使用 SDK 功能。

### 直接连接方式

| 连接方式 | 说明 | 默认 IP/配置 |
|----------|------|--------------|
| **WiFi 直连** | 连接机器人的 WiFi 热点 | 192.168.2.1 |
| **USB 连接** | 通过智能中控 USB 端口 | 192.168.42.2 |
| **UART 连接** | 通过运动控制器 UART 接口 | 115200 波特率 |

### 组网连接方式

| 连接方式 | 说明 |
|----------|------|
| **WiFi 路由器** | 机器人加入局域网，IP 由路由器分配 |

---

## 2.2 连接参数

### WiFi 直连 / 组网 / USB 连接参数

#### IP 地址说明

| 连接模式 | IP 地址 |
|----------|---------|
| WiFi 直连模式 | 机器人默认 IP：`192.168.2.1` |
| WiFi 组网模式 | IP 由路由器动态分配，通过监听 IP 广播获取 |
| USB 连接模式 | 机器人默认 IP：`192.168.42.2` |

#### 端口说明

| 端口 | 用途 |
|------|------|
| **40923** | 明文 SDK 控制端口（TCP） |
| **40924** | 视频流端口（TCP） |
| **40925** | 音频流端口（TCP） |
| **40926** | IP 广播端口（UDP） |

---

## 2.3 连接示例

以下以 Python 编程语言为基础，介绍多种连接方式的使用范例。

### 2.3.1 WiFi 直连模式

#### 环境准备

1. 准备一台 PC 电脑，需具备 WiFi 功能

#### 建立连接

**步骤 1：开启电源**

开启机器人电源，切换智能中控的连接模式开关至 **直连模式**。

**步骤 2：连接 WiFi**

在 PC 上搜索并连接机器人的 WiFi 热点（默认名称格式：`RoboMaster-XXXXXX`）。

**步骤 3：建立 TCP 连接**

```python
import socket

# 连接机器人
sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
sock.connect(('192.168.2.1', 40923))

# 进入 SDK 模式
sock.send(b'command;')
response = sock.recv(1024)
print(f"响应: {response.decode()}")

# 发送控制命令
sock.send(b'robot battery?;')
response = sock.recv(1024)
print(f"电量: {response.decode()}")

# 关闭连接
sock.close()
```

---

### 2.3.2 WiFi 路由器模式

#### 环境准备

1. 准备一台 PC 电脑，具备网络功能
2. 准备一台家用路由器

#### 建立连接

**步骤 1：开启电源**

开启机器人电源，切换智能中控的连接模式开关至 **组网模式**。

**步骤 2：机器人入网**

使用 RoboMaster App 扫描二维码，让机器人加入路由器的 WiFi 网络。

**步骤 3：获取 IP 地址**

通过监听 IP 广播获取机器人的 IP 地址：

```python
import socket

# 监听 IP 广播
sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
sock.bind(('0.0.0.0', 40926))

print("等待机器人 IP 广播...")

while True:
    data, addr = sock.recvfrom(1024)
    message = data.decode()
    if message.startswith('robot ip'):
        robot_ip = message.split()[-1].rstrip(';')
        print(f"发现机器人 IP: {robot_ip}")
        break

sock.close()
```

**步骤 4：建立 TCP 连接**

```python
import socket

# 使用获取到的 IP 连接
robot_ip = '192.168.1.100'  # 替换为实际 IP

sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
sock.connect((robot_ip, 40923))

sock.send(b'command;')
response = sock.recv(1024)
print(f"响应: {response.decode()}")

sock.close()
```

### 2.3.3 USB 连接

USB 连接模式使用 RNDIS 协议，将机器人 USB 设备虚拟为网卡设备。

> [!note] RNDIS
> Remote Network Driver Interface Specification，微软定义的网络协议。
> 更多信息请参考 [RNDIS Wikipedia](https://www.wikipedia.org/wiki/RNDIS)

#### 环境准备

1. 准备一台具备 RNDIS 功能的 PC 电脑
2. 准备一根 Micro-USB 数据线

#### 建立连接

**步骤 1：开启电源**

开启机器人电源，无需关心连接模式开关位置。

**步骤 2：连接 USB**

使用 Micro-USB 数据线连接机器人和 PC。

**步骤 3：安装驱动**

- **Windows**：通常会自动安装 RNDIS 驱动
- **Linux/macOS**：可能需要手动配置网络

**步骤 4：建立连接**

```python
import socket

# USB 连接默认 IP
sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
sock.connect(('192.168.42.2', 40923))

sock.send(b'command;')
response = sock.recv(1024)
print(f"响应: {response.decode()}")

sock.close()
```

### 2.3.4 UART 连接

#### 环境准备

1. 一台 PC 电脑，已安装 USB 转串口模块驱动
2. USB 转串口模块
3. 三根杜邦线

#### 引脚连接

| EP UART 引脚 | USB 转串口模块 |
|--------------|----------------|
| TX | RX |
| RX | TX |
| GND | GND |

#### 串口配置

| 参数 | 值 |
|------|-----|
| 波特率 | 115200 |
| 数据位 | 8 |
| 停止位 | 1 |
| 奇偶校验 | None |

#### 建立连接

```python
import serial
import time

# 打开串口
ser = serial.Serial('COM3', 115200, timeout=1)
time.sleep(1)  # 等待串口稳定

# 进入 SDK 模式
ser.write(b'command;')
time.sleep(0.1)
response = ser.readline()
print(f"响应: {response.decode()}")

# 查询电量
ser.write(b'robot battery?;')
time.sleep(0.1)
response = ser.readline()
print(f"电量: {response.decode()}")

ser.close()
```

## 2.4 连接方式对比

| 连接方式 | 优点 | 缺点 | 适用场景 |
|----------|------|------|----------|
| **WiFi 直连** | 无需额外设备 | 距离有限 | 快速开发测试 |
| **WiFi 路由器** | 多机控制、距离远 | 需路由器 | 多机编队 |
| **USB 连接** | 稳定、速度快 | 有线连接 | 桌面开发 |
| **UART 连接** | 通用性强、嵌入式友好 | 需要串口模块 | 嵌入式开发 |

## 2.5 完整连接示例

### TCP 连接封装类

```python
import socket

class RoboMasterConnection:
    """RoboMaster 明文 SDK 连接类"""
    
    def __init__(self, ip='192.168.2.1', port=40923):
        self.ip = ip
        self.port = port
        self.socket = None
    
    def connect(self):
        """建立连接"""
        self.socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        self.socket.connect((self.ip, self.port))
        
        # 进入 SDK 模式
        self.send_command('command')
        print(f"已连接到 {self.ip}:{self.port}")
    
    def disconnect(self):
        """断开连接"""
        if self.socket:
            self.send_command('quit')
            self.socket.close()
            self.socket = None
            print("已断开连接")
    
    def send_command(self, command):
        """发送命令"""
        if not command.endswith(';'):
            command += ';'
        self.socket.send(command.encode())
        response = self.socket.recv(1024).decode().strip()
        return response
    
    def __enter__(self):
        self.connect()
        return self
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        self.disconnect()

# 使用示例
if __name__ == "__main__":
    with RoboMasterConnection('192.168.2.1') as robot:
        # 查询电量
        battery = robot.send_command('robot battery?')
        print(f"电量: {battery}")
        
        # 控制底盘
        robot.send_command('chassis speed 0.3 0 0')
```

## 常见问题

### 1. WiFi 连接失败

**解决方案：**
- 确认机器人已开机
- 检查 WiFi 密码是否正确
- 尝试重启机器人 WiFi

### 2. USB 连接无响应

**解决方案：**
- 确认 RNDIS 驱动已安装
- 检查 USB 线缆是否正常
- 尝试更换 USB 端口

### 3. UART 通信乱码

**解决方案：**
- 确认波特率为 115200
- 检查 TX/RX 是否接反
- 验证 GND 是否共地
