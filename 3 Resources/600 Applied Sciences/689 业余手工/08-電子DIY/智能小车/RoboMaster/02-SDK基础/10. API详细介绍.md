# 10. RoboMaster SDK API 详细介绍

> [!info] 本章概述
> RoboMaster SDK 各模块 API 详细介绍，包含方法签名、参数说明和使用示例。

## 10.1 robomaster package

RoboMaster SDK 主包，包含所有功能模块。

```python
import robomaster
from robomaster import robot
```

## 10.1.1 robomaster.action

动作基类模块，提供任务动作的控制接口。

### Action 类

所有任务动作的基类，提供统一的动作控制接口。

**主要方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `wait_for_completed(timeout)` | 等待动作完成 | timeout: 超时时间(秒) |
| `wait_for_started(timeout)` | 等待动作开始 | timeout: 超时时间(秒) |

**示例：**

```python
# 等待动作完成
action = ep_chassis.move(x=1, y=0, z=0, xy_speed=0.7)
action.wait_for_completed(timeout=10)

# 带超时检测
try:
    action.wait_for_completed(timeout=5)
except Exception as e:
    print(f"动作执行超时: {e}")
```

## 10.1.2 robomaster.armor

装甲板控制模块。

### Armor 类

**主要方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `set_sensitivity(comp, sensitivity)` | 设置灵敏度 | comp: 装甲板, sensitivity: 1-10 |
| `get_sensitivity(comp)` | 获取灵敏度 | comp: 装甲板 |
| `get_hit_event(comp)` | 获取打击事件 | comp: 装甲板 |

**装甲板常量：**

| 常量 | 说明 |
|------|------|
| `armor.COMP_ALL` | 全部装甲板 |
| `armor.COMP_FRONT` | 前装甲板 |
| `armor.COMP_BACK` | 后装甲板 |
| `armor.COMP_LEFT` | 左装甲板 |
| `armor.COMP_RIGHT` | 右装甲板 |

**示例：**

```python
from robomaster import armor

# 设置前装甲板灵敏度
ep_armor.set_sensitivity(comp=armor.COMP_FRONT, sensitivity=5)

# 获取打击事件
hit_event = ep_armor.get_hit_event(comp=armor.COMP_ALL)
print(f"打击位置: {hit_event}")
```

## 10.1.3 robomaster.battery

电池信息模块。

### Battery 类

**主要方法：**

| 方法 | 说明 | 返回值 |
|------|------|--------|
| `get_battery_percentage()` | 获取电量百分比 | int (0-100) |
| `get_battery_temp()` | 获取电池温度 | float (摄氏度) |

发射器控制模块。

### Blaster 类

**主要方法：**

| 方法 | 说明 | 参数 |

| `fire(times, fire_type)` | 发射 | times: 次数, fire_type: 类型 |

```python
# 设置瞄准灯亮度
ep_blaster.set_led(brightness=255)
```

## 10.1.5 robomaster.camera

相机控制模块。

### Camera 类

**主要方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `start_video_stream(display, resolution)` | 开始视频流 | display: 是否显示, resolution: 分辨率 |
| `stop_video_stream()` | 停止视频流 | 无 |
| `read_cv2_image()` | 读取 OpenCV 图像 | 无 |
| `record_audio(save_file, seconds, sample_rate)` | 录制音频 | 文件名、时长、采样率 |

**分辨率常量：**

| 常量 | 说明 |
|------|------|
| `camera.STREAM_360P` | 360P 分辨率 |
| `camera.STREAM_720P` | 720P 分辨率 |

**示例：**

```python
from robomaster import camera
import cv2

# 方法一：直接显示
ep_camera.start_video_stream(display=True, resolution=camera.STREAM_360P)
time.sleep(10)
ep_camera.stop_video_stream()

# 方法二：OpenCV 处理
ep_camera.start_video_stream(display=False)
for i in range(200):
    img = ep_camera.read_cv2_image()
    cv2.imshow("Robot", img)
    cv2.waitKey(1)
ep_camera.stop_video_stream()

# 录制音频
ep_camera.record_audio(save_file="output.wav", seconds=5, sample_rate=16000)
```

## 10.1.6 robomaster.chassis

底盘控制模块。

### Chassis 类

**移动控制方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `move(x, y, z, xy_speed, z_speed)` | 相对移动 | x/y: 距离(m), z: 角度(°), 速度 |
| `moveto(x, y, z, xy_speed, z_speed)` | 绝对移动 | x/y: 位置(m), z: 角度(°) |
| `drive_speed(x, y, z, timeout)` | 速度控制 | x/y: 速度(m/s), z: 角速度(°/s) |
| `drive_wheels(w1, w2, w3, w4)` | 麦轮速度 | 各轮速度 |

**状态获取方法：**

| 方法 | 说明 | 返回值 |
|------|------|--------|
| `get_attitude()` | 获取姿态 | (pitch, roll, yaw) |
| `get_position()` | 获取位置 | (x, y, z) |
| `get_speed()` | 获取速度 | (vx, vy, vz) |

**示例：**

```python
# 相对移动 - 前进 1 米
ep_chassis.move(x=1, y=0, z=0, xy_speed=0.7).wait_for_completed()

# 相对移动 - 左转 90 度
ep_chassis.move(x=0, y=0, z=90, z_speed=45).wait_for_completed()

# 速度控制 - 以 0.5 m/s 前进
ep_chassis.drive_speed(x=0.5, y=0, z=0, timeout=5)
time.sleep(3)
ep_chassis.drive_speed(x=0, y=0, z=0, timeout=5)  # 停止

# 麦轮速度控制
ep_chassis.drive_wheels(w1=100, w2=100, w3=100, w4=100)

# 获取位置
pos = ep_chassis.get_position()
print(f"位置: x={pos[0]}, y={pos[1]}, z={pos[2]}")
```

## 10.1.7 robomaster.exceptions

异常定义模块。

### 异常类

| 异常 | 说明 |
|------|------|
| `RobomasterException` | SDK 基础异常 |
| `ConnectionError` | 连接错误 |
| `TimeoutError` | 超时错误 |
| `SDKError` | SDK 错误 |

**示例：**

```python
from robomaster.exceptions import *

try:
    ep_robot.initialize(conn_type='ap')
except ConnectionError as e:
    print(f"连接失败: {e}")
except TimeoutError as e:
    print(f"连接超时: {e}")
```

## 10.1.8 robomaster.flight

无人机飞行控制模块。

### Flight 类

**基础飞行方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `takeoff()` | 起飞 | 无 |
| `land()` | 降落 | 无 |
| `hover()` | 悬停 | 无 |
| `emergency_stop()` | 紧急停止 | 无 |

**方向飞行方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `forward(distance)` | 向前飞 | distance: 距离(cm) |
| `backward(distance)` | 向后飞 | distance: 距离(cm) |
| `left(distance)` | 向左飞 | distance: 距离(cm) |
| `right(distance)` | 向右飞 | distance: 距离(cm) |
| `up(distance)` | 向上飞 | distance: 距离(cm) |
| `down(distance)` | 向下飞 | distance: 距离(cm) |

**旋转方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `rotate(angle)` | 旋转 | angle: 角度(°) |
| `rotate_cw(angle)` | 顺时针旋转 | angle: 角度(°) |
| `rotate_ccw(angle)` | 逆时针旋转 | angle: 角度(°) |

**遥控杆量控制：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `rc(a, b, c, d)` | 遥控杆量 | a:横滚, b:俯仰, c:油门, d:偏航 |

**示例：**

```python
# 起飞并飞行
tl_flight.takeoff().wait_for_completed()
tl_flight.forward(distance=100).wait_for_completed()  # 前进 100cm
tl_flight.rotate(angle=90).wait_for_completed()       # 旋转 90°
tl_flight.land().wait_for_completed()

# 遥控杆量控制
tl_flight.rc(a=20, b=0, c=0, d=0)  # 左飞
time.sleep(2)
tl_flight.rc(a=0, b=0, c=0, d=0)   # 停止
```

|------|------|------|
| `move(pitch, yaw, pitch_speed, yaw_speed)` | 相对移动 | pitch/yaw: 角度(°), 速度 |
| `moveto(pitch, yaw, pitch_speed, yaw_speed)` | 绝对移动 | pitch/yaw: 角度(°), 速度 |

```python
# 相对移动 - 向左旋转 30 度
ep_gimbal.move(pitch=0, yaw=30).wait_for_completed()
```

## 10.1.11 robomaster.led

LED 灯效控制模块。

### LED 类

**主要方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `set_led(comp, r, g, b, effect)` | 设置灯效 | 组件、RGB、效果 |

**组件常量：**

| 常量 | 说明 |
|------|------|
| `led.COMP_ALL` | 全部装甲灯 |
| `led.COMP_TOP_ALL` | 全部顶部灯 |
| `led.COMP_BOTTOM_ALL` | 全部底部灯 |
| `led.COMP_TOP_FRONT` | 顶部前灯 |
| `led.COMP_TOP_BACK` | 顶部后灯 |
| `led.COMP_BOTTOM_FRONT` | 底部前灯 |
| `led.COMP_BOTTOM_BACK` | 底部后灯 |
| `led.COMP_BOTTOM_LEFT` | 底部左灯 |
| `led.COMP_BOTTOM_RIGHT` | 底部右灯 |

**效果常量：**

| 常量 | 说明 |
|------|------|
| `led.EFFECT_ON` | 常亮 |
| `led.EFFECT_OFF` | 熄灭 |
| `led.EFFECT_FLASH` | 闪烁 |
| `led.EFFECT_BREATH` | 呼吸 |
| `led.EFFECT_SCROLLING` | 流水 |

**示例：**

```python
from robomaster import led

# 设置所有灯为红色常亮
ep_led.set_led(comp=led.COMP_ALL, r=255, g=0, b=0, effect=led.EFFECT_ON)

# 设置顶部灯为绿色呼吸
ep_led.set_led(comp=led.COMP_TOP_ALL, r=0, g=255, b=0, effect=led.EFFECT_BREATH)

# 关闭所有灯
ep_led.set_led(comp=led.COMP_ALL, r=0, g=0, b=0, effect=led.EFFECT_OFF)
```
