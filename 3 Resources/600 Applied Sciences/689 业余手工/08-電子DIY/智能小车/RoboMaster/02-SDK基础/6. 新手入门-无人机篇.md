# 6. RoboMaster SDK 新手入门 - 教育系列无人机篇

> [!info] 本章概述
> 详细介绍如何使用 RoboMaster SDK 控制教育系列无人机（Tello EDU、Tello Talent），包括初始化、飞行控制、LED 设置和视频流获取。

---

## 支持的产品

| 产品 | 说明 |
|------|------|
| **Tello EDU** | 教育编程无人机 |
| **Tello Talent** | 教育编程无人机（带扩展模块） |

---

## 6.1 初始化机器人

在进行与机器人相关的操作之前，需要根据指定的配置初始化机器人对象。

### 步骤 1：导入模块

```python
from robomaster import robot
```

### 步骤 2：指定本地 IP（可选）

```python
import robomaster
robomaster.config.LOCAL_IP_STR = "192.168.10.20"
```

> [!tip] 自动获取
> 大部分情况下 SDK 能够自动获取正确的本地 IP，无需手动指定。但当 SDK 运行在多网卡同时使用的设备时，需要手动指定。

### 步骤 3：创建无人机对象

```python
tl_drone = robot.Drone()
```

### 步骤 4：初始化

```python
# 教育系列无人机的初始化不需要传入任何参数
tl_drone.initialize()
```

---

## 6.2 获取模块对象

部分 SDK 接口属于 Drone 对象本身，可以直接调用。但一些接口属于其他模块，需要先获取相应的对象。

### 方法一：直接使用 `.` 运算符

```python
tl_flight = tl_drone.flight
tl_led = tl_drone.led
tl_camera = tl_drone.camera
```

### 方法二：使用 `get_module()` 方法

```python
tl_flight = tl_drone.get_module("flight")
```

### 模块对照表

| 模块名 | 获取方法 | 功能 |
|--------|----------|------|
| `flight` | `tl_drone.flight` | 飞行控制 |
| `led` | `tl_drone.led` | 扩展 LED 控制 |
| `camera` | `tl_drone.camera` | 相机控制 |

---

## 6.3 释放机器人资源

在程序的最后，应该手动释放机器人对象相关的资源：

```python
tl_drone.close()
```

> [!warning] 重要
> 为了避免一些意外错误，记得在程序的最后调用 `close()` 方法！

---

## 6.4 查询类接口的使用

查询类接口用于获取机器人自身的状态信息。

### 6.4.1 示例一：查询机器人固件 SDK 版本

```python
import robomaster
from robomaster import robot

if __name__ == '__main__':
    tl_drone = robot.Drone()
    tl_drone.initialize()
    
    # 获取飞机版本信息
    drone_version = tl_drone.get_sdk_version()
    print("Drone sdk version: {0}".format(drone_version))
    
    tl_drone.close()
```

**运行结果：**
```
Drone sdk version: xx.xx.xx.xx
```

> [!info] 示例文件
> `/examples/12_drone/02_get_version.py`

---

### 6.4.2 示例二：获取机器人 SN 号

```python
import robomaster
from robomaster import robot

if __name__ == '__main__':
    tl_drone = robot.Drone()
    tl_drone.initialize()
    
    # 获取飞机SN信息
    SN = tl_drone.get_sn()
    print("drone sn: {0}".format(SN))
```

## 6.5 设置类接口的使用

> [!warning] 注意
> 设置扩展 LED 灯目前只有 **Tello Talent** 机器支持！

### 6.5.1 示例一：设置机器人扩展 LED 模块

```python
import time
import robomaster
from robomaster import robot

if __name__ == '__main__':
    tl_drone = robot.Drone()
    tl_drone.initialize()
    
    tl_led = tl_drone.led
    
    # 设置为红色
    tl_led.set_led(r=255, g=0, b=0)
    time.sleep(1)
    
    tl_drone.close()
```

**颜色循环示例：**

```python
import time
import robomaster
from robomaster import robot

if __name__ == '__main__':
    tl_drone = robot.Drone()
    tl_drone.initialize()
    
    tl_led = tl_drone.led
    tl_led.set_led(r=0, g=0, b=0)
    
    rgb_list = [
        (100, 100, 100), (255, 255, 255), 
        (255, 0, 0), (0, 0, 255),
        (0, 255, 0), (255, 255, 0), 
        (255, 0, 255), (0, 255, 255)
    ]
    
    for rgb_info in rgb_list:
        tl_led.set_led(r=rgb_info[0], g=rgb_info[1], b=rgb_info[2])
        time.sleep(0.5)
    
    tl_drone.close()
```

> [!info] 示例文件
> `/examples/12_drone/20_led.py`

## 6.6 动作类接口的使用

动作类接口分为 **任务动作控制** 和 **即时动作控制** 两类。

> [!important] 飞行前须知
> 由于飞机类机器人本身的特性决定必须要起飞后才能进行控制，因此首先介绍任务动作控制。

### 6.6.1 任务动作控制

任务动作是指需要持续一段时间才能完成的动作。

> [!important] 注意事项
> 1. 任务动作接口返回 `action` 对象，提供 `wait_for_completed(timeout)` 方法
> 2. 同一模块同一时间只能执行一个动作（互斥）
> 3. 不同模块之间的动作可以同时执行

#### 6.6.1.1 示例一：控制飞机起飞并前后飞行

```python
import robomaster
from robomaster import robot

if __name__ == '__main__':
    tl_drone = robot.Drone()
    tl_drone.initialize()
    
    tl_flight = tl_drone.flight
    
    # 起飞
    tl_flight.takeoff().wait_for_completed()
    
    # 向前飞50厘米
    tl_flight.forward(distance=50).wait_for_completed()
    
    # 向后飞50厘米
    tl_flight.backward(distance=50).wait_for_completed()
    
    # 降落
    tl_flight.land().wait_for_completed()
    
    tl_drone.close()
```

**飞行控制方法：**

| 方法 | 说明 | 参数 |
|------|------|------|
| `takeoff()` | 起飞 | 无 |
| `land()` | 降落 | 无 |
| `forward(distance)` | 向前飞行 | distance: 距离(cm) |
| `backward(distance)` | 向后飞行 | distance: 距离(cm) |
| `left(distance)` | 向左飞行 | distance: 距离(cm) |
| `right(distance)` | 向右飞行 | distance: 距离(cm) |
| `up(distance)` | 向上飞行 | distance: 距离(cm) |
| `down(distance)` | 向下飞行 | distance: 距离(cm) |
| `rotate(angle)` | 旋转 | angle: 角度(°) |

**三种调用方式：**

```python
# 方法一：立即等待完成（推荐）
tl_flight.forward(distance=50).wait_for_completed()

# 方法二：稍后等待
flight_action = tl_flight.forward(distance=50)
# 可以执行其他操作
flight_action.wait_for_completed()

# 方法三：使用延时（不推荐）
tl_flight.forward(distance=50)
time.sleep(8)  # 需要估算足够的时间
```

> [!info] 示例文件
> `/examples/12_drone/07_forward_backward.py`

### 6.6.2 即时动作控制

即时控制类动作是指设置后马上生效的动作。

#### 6.6.2.1 示例一：控制遥控器杆量

```python
import time
import robomaster
from robomaster import robot

if __name__ == '__main__':
    tl_drone = robot.Drone()
    tl_drone.initialize()
    
    tl_flight = tl_drone.flight
    
    # 起飞
    tl_flight.takeoff().wait_for_completed()
    
    # 左移（a=20）
    tl_flight.rc(a=20, b=0, c=0, d=0)
    time.sleep(4)
    
    # 右移（a=-20）
    tl_flight.rc(a=-20, b=0, c=0, d=0)
    time.sleep(3)
    
    # 停止
    tl_flight.rc(a=0, b=0, c=0, d=0)
    
    # 降落
    tl_flight.land().wait_for_completed()
    
    tl_drone.close()
```

**遥控器杆量参数：**

| 参数 | 说明 | 范围 |
|------|------|------|
| `a` | 横滚（左右） | -100 ~ 100 |
| `b` | 俯仰（前后） | -100 ~ 100 |
| `c` | 油门（上下） | -100 ~ 100 |
| `d` | 偏航（旋转） | -100 ~ 100 |

**参数方向说明：**
- `a > 0`：向左飞行
- `a < 0`：向右飞行
- `b > 0`：向前飞行
- `b < 0`：向后飞行
- `c > 0`：向上飞行
- `c < 0`：向下飞行
- `d > 0`：顺时针旋转
- `d < 0`：逆时针旋转

> [!info] 示例文件
> `/examples/12_drone/13_rc.py`

## 6.7 多媒体接口的使用

教育系列无人机的多媒体部分主要指获取视频流。

### 6.7.1 示例一：获取视频流

#### 方法一：直接显示视频流

```python
import time
import robomaster
from robomaster import robot

if __name__ == '__main__':
    tl_drone = robot.Drone()
    tl_drone.initialize()
    
    tl_camera = tl_drone.camera
    
    # 显示十秒图传
    tl_camera.start_video_stream(display=True)
    time.sleep(10)
    tl_camera.stop_video_stream()
    
    tl_drone.close()
```

#### 方法二：使用 OpenCV 处理

```python
import cv2
import robomaster
from robomaster import robot

if __name__ == '__main__':
    tl_drone = robot.Drone()
    tl_drone.initialize()
    
    tl_camera = tl_drone.camera
    
    # 设置视频参数
    tl_camera.start_video_stream(display=False)
    tl_camera.set_fps("high")
    tl_camera.set_resolution("high")
    tl_camera.set_bitrate(6)
    
    # 显示视频帧
    for i in range(0, 302):
        img = tl_camera.read_cv2_image()
        cv2.imshow("Drone", img)
        cv2.waitKey(1)
    
    cv2.destroyAllWindows()
    tl_camera.stop_video_stream()
    tl_drone.close()
```

**视频参数设置：**

| 方法 | 说明 | 参数值 |
|------|------|--------|
| `set_fps()` | 设置帧率 | `"low"`, `"middle"`, `"high"` |
| `set_resolution()` | 设置分辨率 | `"low"`, `"middle"`, `"high"` |
| `set_bitrate()` | 设置码率 | 1-5（自动）或具体数值 |

> [!info] 示例文件
> `/examples/12_drone/01_video_with_display.py`

## 接口类型总结

| 接口类型 | 说明 | 示例 |
|----------|------|------|
| **查询类** | 获取无人机状态信息 | `get_sdk_version()`, `get_sn()` |
| **设置类** | 设置无人机参数 | `set_led()` |
| **任务动作类** | 需要持续一段时间的动作 | `takeoff()`, `land()`, `forward()` |
| **即时动作类** | 立即生效的动作 | `rc()` |
| **多媒体类** | 视频流获取 | `start_video_stream()` |

## EP 与无人机 SDK 对比

| 功能 | EP 机器人 | 教育无人机 |
|------|----------|------------|
| **对象类型** | `robot.Robot()` | `robot.Drone()` |
| **初始化参数** | 需要指定连接方式 | 无需参数 |
| **通讯协议** | TCP/UDP 可选 | 仅 UDP |
| **底盘控制** | `chassis` 模块 | `flight` 模块 |
| **LED 控制** | 装甲灯 | 扩展 LED（仅 TT） |
| **发射器** | 支持 | 不支持 |
| **云台控制** | 支持 | 不支持 |

## 飞行安全提示

> [!warning] 安全须知
> 1. **起飞前检查**：确保无人机周围无障碍物
> 2. **电池电量**：确保电池电量充足
> 3. **飞行环境**：在室内或无风环境下飞行
> 4. **紧急停止**：了解紧急停止方法
> 5. **降落确认**：确保降落区域平整无障碍
