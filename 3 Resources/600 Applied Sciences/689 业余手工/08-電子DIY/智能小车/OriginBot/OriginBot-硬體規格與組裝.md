# OriginBot 硬體規格與組裝 Hardware & Assembly

> OriginBot 是地平線 (D-Robotics) 官方推出的 RDK X3 開源機器人套件——從零組裝一台具備 AI 視覺 + ROS2 自主導航能力的智能小車。

## 硬體規格 Hardware Specs

### 核心組件

| 組件 | 規格 | 說明 |
|------|------|------|
| **主控** | RDK X3 (旭日X3派) | 5 TOPS BPU、4 核 ARM A53 |
| **驅動方式** | 差速驅動 | 兩主動輪 + 一從動萬向輪 |
| **馬達** | 直流減速馬達 ×2 (帶編碼器) | 閉環速度控制 |
| **底盤** | 鋁合金框架 + 亞克力擴展板 | 多層結構，方便擴展 |
| **電池** | 12V 鋰電池組 | 續航 2–4 小時 |
| **電源管理** | 降壓模組 (12V→5V/3.3V) | 為 RDK X3 和外設供電 |

### 感測器套件

| 感測器 | 型號 | 接口 | 用途 |
|--------|------|------|------|
| **雷射雷達** | RPLIDAR A1 / YDLIDAR X4 | USB/UART | 360° 距離掃描、SLAM |
| **深度相機** | Astra Pro / Intel RealSense | USB 3.0 | 3D 視覺、物體檢測 |
| **IMU** | MPU6050 / BMI088 | I2C | 姿態感知、里程計融合 |
| **超音波** | HC-SR04 ×3 | GPIO | 近距離避障 |

### 擴展接口

| 接口 | 用途 |
|------|------|
| 40PIN GPIO | 感測器、LED、舵機 |
| USB 3.0 ×4 | 相機、雷達 |
| MIPI CSI ×2 | MIPI 攝像頭模組 |
| UART | MCU 通訊（備用） |
| I2C / SPI | 外設感測器 |

## 零件清單 BOM

| # | 零件 | 數量 | 備註 |
|:--:|------|:--:|------|
| 1 | RDK X3 開發板 | 1 | 主控制器 |

| 10 | 杜邦線、螺絲、銅柱 | 若干 | 連接與固定 |

## 組裝步驟 Assembly

### Step 1: 底盤組裝

```
1. 將兩個馬達安裝到底盤框架的馬達座上
   → 用 M3 螺絲固定，確保馬達軸水平
2. 安裝橡膠輪胎到馬達軸上
   → 用聯軸器連接，鎖緊頂絲
3. 安裝萬向輪到底盤後方
   → 確保底盤水平不傾斜
```

### Step 2: 電控安裝

```
1. 將馬達驅動板 (TB6612/L298N) 固定在底盤上
2. 連接馬達線到驅動板輸出端
   TB6612:
   - AO1/AO2 → 左馬達
   - BO1/BO2 → 右馬達
3. 連接編碼器線（如有）
4. 安裝降壓模組，12V 輸入 → 5V 輸出供 RDK X3
```

### Step 3: RDK X3 安裝

```
1. 用銅柱將 RDK X3 固定到亞克力擴展板上
2. 將擴展板安裝到底盤上方
3. 連接 GPIO 到驅動板：
   RDK X3 40PIN        TB6612
   ─────────────────────────────
   GPIO 13 (PWM)  →   PWMA
   GPIO 16        →   AIN1
   GPIO 17        →   AIN2
   GPIO 18 (PWM)  →   PWMB
   GPIO 19        →   BIN1
   GPIO 20        →   BIN2
   5V             →   VCC (邏輯)
   GND            →   GND
```

### Step 4: 感測器安裝

```
1. 雷射雷達：安裝到頂層亞克力板，USB 連接 RDK X3
2. 深度相機：安裝到前方，USB 連接
3. IMU：I2C 連接到 RDK X3 40PIN
   - VCC → 3.3V (Pin 1)
   - GND → GND (Pin 6)
   - SDA → GPIO 2 (Pin 3)
   - SCL → GPIO 3 (Pin 5)
```

### Step 5: 供電接線

```
電池 12V ──┬── 馬達驅動板 VMOT (馬達電源)
           ├── 降壓模組 → 5V → RDK X3 USB-C
           └── 雷射雷達 (如有獨立供電)
```

## 開機檢查 Power-On Checklist

```
□ 馬達線連接牢固，無短路風險
□ 電池充滿電 (12.6V)
□ RDK X3 正常開機（綠燈亮）
□ 雷射雷達旋轉正常
□ 深度相機可以被 lsusb 識別
□ SSH 可以連接 RDK X3
□ 馬達在手動推動時無卡頓
```

> ⚠️ **安全第一**：首次通電前將小車架空（輪子離地），避免意外跑走。
