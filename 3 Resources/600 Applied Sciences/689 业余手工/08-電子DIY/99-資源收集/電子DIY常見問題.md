# 電子DIY常見問題

## 1. 「我不會程式設計，能學嗎？」
Arduino 非常簡單——Blink 只有 8 行程式碼。第一小時就讓 LED 閃起來。

## 2. 「Arduino UNO 還是 ESP32？」
零基礎選 Arduino UNO（簡單穩定）；想做 WiFi 選 ESP32。

## 3. 「為什麼我的 LED 燒了？」
忘了串聯限流電阻！LED 直接接 5V = 秒燒。永遠記得：LED + 220Ω 電阻。

## 4. 「板子壞了怎麼辦？」
先檢查 USB 線（很多問題是線壞了）、換一個 USB 口、檢查電腦是否認到板子。
