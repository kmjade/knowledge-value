# 07 — 業餘縫紉

> 縫紉是最實用的手工——修改不合身的衣服、做一個獨一無二的包，省錢又有成就感。

## 工具

| 工具 | 必備 |
|------|------|
| 縫紉機（入門級 ¥500–1000） | 推薦 |
| 手縫針+線 | ✅ |
| 布剪（專用，不可剪紙！） | ✅ |
| 拆線器 | ✅ |
| 軟尺 | ✅ |
| 熨斗 | ✅ |

## 基礎技法

| 技法 | 用途 |
|------|------|
| 直線縫 | 基本拼接 |
| 回針縫（手縫） | 牢固縫合 |
| 捲邊 | 邊緣處理 |
| 上拉鍊 | 包包、褲子 |

## 新手專案

1. **手縫杯墊**（30min）— 兩塊布 + 手縫
2. **環保購物袋**（1hr）— 學習直線縫
3. **修改褲長**（30min）— 最實用技能

## 布料新手推薦

| 布料 | 難度 |
|------|------|
| 棉布 | ⭐ 最佳入門 |
| 亞麻 | ⭐ |
| 牛仔布 | ⭐⭐ |
| 絲綢 | ⭐⭐⭐（太滑） |

> 💡 縫紉機是值得的投資——一條修改褲子的錢可能就夠買一台入門機了。學會縫紉 = 終身省錢技能。
