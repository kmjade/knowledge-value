# 05 — 業餘模型製作

> 模型是微型的世界——在方寸之間重現歷史、科幻和想像。

## 模型類型

| 類型 | 難度 | 適合 |
|------|------|------|
| 塑膠模型（Gunpla） | ⭐ | 機器人、戰車 |
| 紙模 | ⭐⭐ | 建築、角色 |
| 木質模型 | ⭐⭐ | 船、建築 |
| 金屬蝕刻片 | ⭐⭐⭐ | 細節升級 |
| 微縮場景 | ⭐⭐⭐ | 鐵道、戰爭場景 |

## 必備工具

| 工具 | 用途 |
|------|------|
| 模型剪鉗 | 從框架取下零件 |
| 筆刀 | 修整水口 |
| 鑷子 | 精細操作 |
| 砂紙（#400–#2000） | 打磨 |
| 模型膠水 | 黏合 |
| 補土 | 填補縫隙 |

## 新手推薦：Gunpla（鋼彈模型）

不需要膠水，不需要上色，直接組裝就有很好的效果。從 HG（1/144）開始。

## 新手專案

1. **HG Gunpla**（2–3hr）— 素組（不需上色）
2. **簡易紙模**（1–2hr）— 建築紙模免費下載
3. **上色練習**（半天）— 噴罐或筆塗

> 💡 模型製作是對耐心的極致考驗——一個零件可能要打磨半小時。但完成那一刻的成就感，值得。
