# 02-人体系统

> 人体由约 37 万亿个细胞构成，组织为 11 大系统。每个系统各司其职，协同维持生命——这是医学的"底层架构"。

## 人体 11 大系统速览

| 系统 | 核心功能 | 关键器官/结构 |
|------|----------|--------------|
| **骨骼系统** | 支撑、保护、造血、钙储存 | 206 块骨、关节、韧带 |
| **肌肉系统** | 运动、产热、姿势维持 | 600+ 块骨骼肌、平滑肌、心肌 |
| **神经系统** | 感知、整合、指挥 | 脑、脊髓、神经 |
| **循环系统** | 运输氧气/营养/废物 | 心脏、血管、血液 |
| **呼吸系统** | 气体交换 (O₂↔CO₂) | 鼻、气管、肺、膈肌 |
| **消化系统** | 分解食物、吸收营养 | 口→食管→胃→小肠→大肠→肝→胰 |
| **泌尿系统** | 过滤血液、排出废物 | 肾、输尿管、膀胱 |
| **内分泌系统** | 激素调控全身 | 垂体、甲状腺、肾上腺、胰腺 |
| **免疫系统** | 防御病原体 | 白细胞、淋巴结、脾、骨髓 |
| **生殖系统** | 繁衍 | 卵巢/睾丸、子宫、相关管道 |
| **皮肤系统** | 屏障、体温调节、感觉 | 皮肤、毛发、指甲、汗腺 |

> 没有一个系统是独立工作的——循环系统运输呼吸系统交换的氧气、内分泌系统调节循环系统的血压、神经系统控制一切。

## 神经系统

### 中枢 vs 周围

```
神经系统
├── 中枢神经系统 (CNS)
│   ├── 脑
│   │   ├── 大脑（思维、意识、运动、感觉）
│   │   ├── 小脑（平衡、协调）
│   │   └── 脑干（呼吸、心跳、觉醒）
│   └── 脊髓（信息高速公路）
└── 周围神经系统 (PNS)
    ├── 感觉神经（传入）
    └── 运动神经（传出）
```

### 神经元与突触

- **神经元**：神经系统的基本单位，~860 亿个
- **突触**：神经元之间的连接点，通过神经递质传递信号
- **动作电位**："全或无"的电信号，沿轴突传播

## 循环系统

### 心脏——四腔泵

```
上腔静脉 → 右心房 → 右心室 → 肺动脉 → 肺
                                       ↓
         ← 左心房 ← 左心室 ← 肺静脉 ←───┘
         ↓
       主动脉 → 全身
```

| 参数 | 正常值 |
|------|--------|
| 心率 | 60-100 次/分 |
| 血压 | 120/80 mmHg |
| 心输出量 | ~5 L/分 |
| 总血量 | ~5 L |

## 免疫系统

### 三道防线

| 防线 | 组成 | 作用时间 |
|------|------|----------|
| **第一道** | 皮肤、黏膜、分泌物 | 即时 |
| **第二道（先天免疫）** | 巨噬细胞、中性粒细胞、炎症反应 | 数小时 |
| **第三道（适应性免疫）** | B 细胞（抗体）、T 细胞（杀伤） | 数天-数周 |

> 疫苗的原理：提前"展示"病原体的特征，让适应性免疫系统做好准备。当真正感染时，免疫系统能快速响应。

## 内分泌系统

| 腺体 | 激素 | 作用 |
|------|------|------|
| 垂体 | 生长激素、促激素 | "主腺"——控制其他腺体 |
| 甲状腺 | T3, T4 | 调节代谢率 |
| 肾上腺 | 肾上腺素、皮质醇 | 应激反应、代谢 |
| 胰腺 | 胰岛素、胰高血糖素 | 血糖调节 |
| 卵巢/睾丸 | 雌激素/睾酮 | 性发育和生殖 |
