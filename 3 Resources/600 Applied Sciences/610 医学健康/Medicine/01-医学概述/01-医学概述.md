# 01-医学概述

> 医学是一门同时依赖科学证据和临床判断的学科——它基于生物学和化学，但最终关乎人的健康和尊严。

## 医学简史

| 时期 | 里程碑 | 范式 |
|------|--------|------|
| 古代 | 希波克拉底 (前 460-370)："医学之父"，体液学说，希波克拉底誓言 | 经验+哲学 |
| 2 世纪 | 盖伦：解剖学和生理学的早期系统化（基于动物解剖，很多是错的） | 权威主义 |
| 1543 | Vesalius《人体的构造》——首次基于人体解剖 | 观察替代权威 |
| 1628 | Harvey 发现血液循环 | 实验医学开端 |
| 1840s | 麻醉术 (乙醚、氯仿) | 手术不再等于酷刑 |
| 1860s | Lister 无菌术、Pasteur 微生物理论 | 感染可预防 |
| 1895 | Röntgen 发现 X 光 | 影像诊断开端 |
| 1928 | Fleming 发现青霉素 | 抗生素时代 |
| 1953 | Watson & Crick DNA 双螺旋 | 分子医学 |
| 1970s | CT 扫描 | 计算机辅助诊断 |
| 1990 | 人类基因组计划启动 | 精准医学前夜 |
| 2012 | CRISPR-Cas9 | 基因编辑 |
| 2020 | mRNA 疫苗 | 疫苗技术范式转变 |

## 循证医学 (Evidence-Based Medicine)

循证医学不等于"只看论文"。它是三者的交集：

```
     证据（研究数据）
          /\
         /  \
        /    \
       / EBM  \
      /        \
     /          \
临床经验 ──────── 患者价值观
```

| 要素 | 含义 |
|------|------|
| 最佳研究证据 | 来自系统综述、RCT 的可靠证据 |
| 临床专业经验 | 医生的诊断直觉和技能 |
| 患者价值观 | 每位患者的独特偏好、关切和期望 |

### 证据等级

| 等级 | 证据类型 |
|------|----------|
| **I** | 多个高质量 RCT 的系统综述/Meta 分析 |
| **II** | 单个高质量 RCT |
| **III** | 队列研究、病例对照研究 |
| **IV** | 病例系列、专家意见 |
| **V** | 个人经验、 anecdotes |

## 医学伦理四大原则

| 原则 | 含义 | 应用 |
|------|------|------|
| **自主 (Autonomy)** | 尊重患者自己的决定 | 知情同意 |
| **有利 (Beneficence)** | 做对患者有益的事 | 选择最有效的治疗 |
| **不伤害 (Non-maleficence)** | 首先，不伤害 | 权衡治疗的风险和收益 |
| **公正 (Justice)** | 公平分配医疗资源 | 医保政策、器官分配 |
