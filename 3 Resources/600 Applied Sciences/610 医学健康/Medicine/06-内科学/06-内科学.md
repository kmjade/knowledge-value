# 06-内科学

> 内科学是用非手术方法诊断和治疗成人疾病的学科——药物、生活方式干预、介入操作是其工具。内科是临床医学中最大的分支。

## 心血管系统

### 高血压

| 分级 | 收缩压 | 舒张压 |
|------|--------|--------|
| 正常 | <120 | <80 |
| 高血压前期 | 120-139 | 80-89 |
| 1 级高血压 | 140-159 | 90-99 |
| 2 级高血压 | ≥160 | ≥100 |

> 高血压是全球死亡的首要危险因素——它"沉默"地损害心脏、脑、肾脏和血管。控制血压是降低心血管风险最有效的事。

### 冠心病

冠状动脉粥样硬化 → 心肌缺血。最严重的后果：**心肌梗死**（心脏病发作）。

| 类型 | 说明 |
|------|------|
| 稳定型心绞痛 | 运动时胸痛，休息缓解 |
| 急性冠脉综合征 (ACS) | 不稳定心绞痛 + 心肌梗死——急症！ |

## 呼吸系统

### 肺炎

| 类型 | 人群 | 常见病原体 |
|------|------|-----------|
| 社区获得性 (CAP) | 普通人群 | 肺炎链球菌、病毒 |
| 医院获得性 (HAP) | 住院 48h 后 | 耐药菌更多 |
| 吸入性 | 吞咽困难者 | 口腔厌氧菌 |

### COPD (慢性阻塞性肺疾病)

吸烟是最主要的病因——不可逆的气道阻塞。全球第三大死因。

## 消化系统

### 肝病谱

```
健康肝 → 脂肪肝 → 肝炎 → 肝纤维化 → 肝硬化 → 肝癌
  ↑                              ↓
 可逆                          不可逆
```

| 病因 | 说明 |
|------|------|
| 病毒性 (HBV/HCV) | 中国肝硬化和肝癌的主要病因 |
| 酒精性 | 长期大量饮酒 |
| 非酒精性脂肪肝 (NAFLD) | 与肥胖/糖尿病相关，发病率飙升 |

### 消化性溃疡

| 病因 | 比例 |
|------|------|
| 幽门螺杆菌 (H. pylori) | ~70% |
| NSAIDs (阿司匹林、布洛芬等) | ~25% |

> 发现 H. pylori 是溃疡的病因是医学史上的经典——Barry Marshall 甚至自己喝下细菌证明了这一点（2005 年诺贝尔奖）。

## 内分泌系统

### 糖尿病

| 类型 | 机制 | 占糖尿病比例 |
|------|------|-------------|
| **1 型** | 自身免疫破坏胰岛 β 细胞→绝对胰岛素缺乏 | ~5-10% |
| **2 型** | 胰岛素抵抗 + 相对胰岛素缺乏 | ~90% |

| 指标 | 正常 | 糖尿病前期 | 糖尿病 |
|------|------|-----------|--------|
| 空腹血糖 | <5.6 | 5.6-6.9 | ≥7.0 mmol/L |
| HbA1c | <5.7% | 5.7-6.4% | ≥6.5% |

### 甲状腺

| 状态 | TSH | T3/T4 | 症状 |
|------|-----|-------|------|
| 甲亢 | ↓ | ↑ | 心悸、手抖、怕热、消瘦 |
| 甲减 | ↑ | ↓ | 乏力、怕冷、体重增加、抑郁 |
