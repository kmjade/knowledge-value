# 07-外科学

> 外科学是用手术方法治疗疾病的学科。在麻醉和无菌术发明之前，手术等同于酷刑。今天的外科是精确、微创、快速恢复的艺术。

## 外科的四大基石

| 基石 | 发明者/年代 | 解决了什么 |
|------|-----------|-----------|
| **麻醉** | Morton, 1846 (乙醚) | 疼痛——手术从折磨变为可耐受 |
| **无菌术** | Lister, 1860s | 感染——术后死亡率大幅下降 |
| **输血** | Landsteiner, 1901 (血型) | 失血——大手术成为可能 |
| **抗生素** | Fleming, 1928 | 术后感染——最后一道防线 |

> 在这四项发明之前，外科手术的死亡率极高。今天，一位健康的年轻人做阑尾切除术的死亡率 <0.1%。

## 手术的演进

| 时代 | 方式 | 特点 |
|------|------|------|
| 传统开放手术 | 大切口 | 视野好，创伤大，恢复慢 |
| 腹腔镜 (1990s-) | 几个小孔 + 摄像头 | 创伤小，恢复快，已成标准 |
| 机器人手术 | da Vinci 等机器人系统 | 3D 视野、震颤过滤、更灵活 |
| 自然腔道内镜 (NOTES) | 经口/肛门等自然开口 | 无体表切口——探索中 |

## 创伤与急救

### "黄金一小时"

严重创伤后第一个小时的处理决定生死。

### ABCDE 评估

| 步骤 | 含义 | 检查 |
|------|------|------|
| **A**irway | 气道 | 通畅吗？ |
| **B**reathing | 呼吸 | 在呼吸吗？有效吗？ |
| **C**irculation | 循环 | 脉搏？血压？出血？ |
| **D**isability | 神经 | 意识？瞳孔？ |
| **E**xposure | 暴露 | 全身检查+保温 |

### 止血

| 方法 | 适用 |
|------|------|
| 直接压迫 | 大多数外出血 |
| 止血带 | 四肢严重出血（记录时间！） |
| 止血剂 | 军用/急救（QuikClot 等） |

## 常见外科急症

| 急症 | 典型表现 | 处理 |
|------|----------|------|
| 急性阑尾炎 | 转移性右下腹痛 | 阑尾切除术 |
| 肠梗阻 | 腹痛+呕吐+停止排气排便 | 保守或手术 |
| 胆囊炎 | 右上腹痛+Murphy 征 | 胆囊切除术 |
| 骨折 | 畸形+肿胀+功能障碍 | 复位+固定 |
| 气胸 | 突发胸痛+呼吸困难 | 胸腔引流 |

### 移植的三大障碍

| 障碍 | 解决 |
|------|------|
| 手术技术 | 已基本解决 |
| 排斥反应 | 免疫抑制剂（环孢素, 1980s → 革命性突破） |
| 器官短缺 | 待解决——异种移植（猪→人）是希望 |
