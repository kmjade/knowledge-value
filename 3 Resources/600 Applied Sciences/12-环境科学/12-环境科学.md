# 10-环境科学

> 环境科学是研究人类活动与自然系统相互作用的跨学科学科。它不是"环保主义"，而是基于数据和模型的系统性科学。

## 地球系统的关键指标

| 指标 | 工业化前 | 现在 | 趋势 |
|------|---------|------|------|
| 大气 CO₂ | ~280 ppm | ~425 ppm (2025) | ↑ 持续上升 |
| 全球平均温度 | 基准 | +1.2°C | ↑ 每十年 ~0.2°C |
| 北极海冰范围 | 基准 | 减少 ~40% | ↓ 加速减少 |
| 海平面 | 基准 | +20cm | ↑ 加速上升 |
| 物种灭绝率 | 背景速率 | 100-1000× 背景速率 | ↑ 第六次大灭绝 |

## 气候变化的科学

### 温室效应——基础物理

```
太阳短波辐射 → 穿过大气 → 被地表吸收 → 地表释放长波辐射
                                           ↓
                              CO₂/CH₄/H₂O 吸收长波 → 大气变暖
```

这不是"理论"或有争议——这是 19 世纪就已确立的物理学（傅里叶、廷德尔、阿伦尼乌斯）。

### 碳循环

```
大气 CO₂ ←→ 海洋溶解 ←→ 光合作用/呼吸 ←→ 化石燃料燃烧
   ↓
 每年人类排放 ~37Gt CO₂
   ↓
 约 50% 被海洋和陆地吸收，50% 留在大气中
```

## 九大地球边界

Rockström 等人提出的框架——地球系统能安全运行的边界：

| 边界 | 状态 |
|------|------|
| 气候变化 | ❌ 已越过 |
| 生物圈完整性（生物多样性） | ❌ 已越过 |
| 氮循环 | ❌ 已严重越过 |
| 磷循环 | ❌ 已越过 |
| 土地利用变化 | ⚠️ 接近边界 |
| 淡水使用 | ⚠️ 接近边界 |
| 海洋酸化 | ⚠️ 接近边界 |
| 大气气溶胶 | ⚠️ 未量化 |
| 新化学物质 | ⚠️ 未量化 |

> 9 个边界中 4 个已被越过——我们正在安全运行空间之外。

## 污染治理

| 类型 | 主要来源 | 治理技术 |
|------|----------|----------|
| **空气污染** (PM2.5, SO₂, NOₓ) | 燃煤、汽车、工业 | 脱硫脱硝、电动化 |
| **水污染** (重金属、营养物) | 工业废水、农业径流 | 膜过滤、生物处理 |
| **土壤污染** | 工业场地、农药 | 生物修复、土壤淋洗 |
| **塑料污染** | 一次性塑料 | 减少使用、可降解材料、回收 |
| **微塑料** | 塑料降解、合成纤维 | 源头控制 |

## 循环经济

线性经济（取→造→扔）→ 循环经济：

```
        原料
         ↓
    产品设计（可维修、可回收）
         ↓
      使用 ←── 维修/翻新
         ↓        ↑
    回收 ← 再利用
```

| 策略 | 说明 |
|------|------|
| Reduce | 少用原料 |
| Reuse | 重复使用产品 |
| Repair | 坏了修而非扔 |
| Remanufacture | 翻新到新标准 |
| Recycle | 回收材料（最末选择） |

## 中国环境治理

| 政策/行动 | 内容 |
|-----------|------|
| **双碳目标** | 2030 碳达峰、2060 碳中和 |
| **蓝天保卫战** | PM2.5 大幅下降（北京 2013-2023 下降 ~60%） |
| **长江十年禁渔** | 2021-2030，恢复长江生态 |
| **全国碳市场** | 2021 年启动，发电行业先行 |
