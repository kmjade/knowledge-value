# 07 — 獸禽蟲魚

> 動物藥是中藥的重要組成部分——多為血肉有情之品，善於補益精血。

## 代表藥物

| 藥物 | 性味 | 功效 | 來源 |
|------|------|------|------|
| **鹿茸** | 溫、甘鹹 | 補腎陽、益精血、強筋骨 | 鹿角 |
| **阿膠** | 平、甘 | 補血滋陰、潤燥止血 | 驢皮 |
| **蜂蜜** | 平、甘 | 補中緩急、潤燥解毒 | 蜂 |
| **龜甲** | 寒、甘鹹 | 滋陰潛陽、補腎健骨 | 龜 |
| **鱉甲** | 寒、鹹 | 滋陰潛陽、軟堅散結 | 鱉 |
| **牡蠣** | 微寒、鹹 | 重鎮安神、軟堅散結 | 貝殼 |
| **牛黃** | 涼、苦甘 | 清熱解毒、息風止痙 | 牛膽結石 |
| **麝香** | 溫、辛 | 開竅醒神、活血通經 | 麝 |

## 動物藥特點

| 特點 | 說明 |
|------|------|
| 血肉有情 | 補益力強，善於填精補髓 |
| 藥力峻猛 | 麝香、牛黃效力強勁 |
| 來源珍貴 | 部分藥物稀有（麝香、牛黃） |
| 現代替代 | 人工麝香、人工牛黃等替代品 |

> 💡 動物藥在中藥中有不可替代的地位——阿膠補血、鹿茸補陽，至今臨床廣泛使用。
