# 06 — 木部

> 木部收載木本植物藥物——多為樹皮、樹枝、果實、樹脂等。

## 代表藥物

| 藥物 | 性味 | 功效 | 藥用部位 |
|------|------|------|---------|
| **桂枝** | 溫、辛甘 | 發汗解肌、溫通經脈 | 嫩枝 |
| **杜仲** | 溫、甘 | 補肝腎、強筋骨 | 樹皮 |
| **枸杞** | 平、甘 | 滋補肝腎、明目 | 果實 |
| **酸棗仁** | 平、甘酸 | 養心安神、斂汗 | 種子 |
| **山茱萸** | 微溫、酸 | 補益肝腎、收澀固脫 | 果肉 |
| **吳茱萸** | 熱、辛苦 | 散寒止痛、降逆止嘔 | 果實 |
| **厚朴** | 溫、苦辛 | 行氣消積、燥濕除滿 | 樹皮 |
| **大棗** | 溫、甘 | 補中益氣、養血安神 | 果實 |

## 木部藥特點

| 特點 | 說明 |
|------|------|
| 藥性溫和 | 多為上品，可久服 |
| 補益肝腎 | 杜仲、枸杞、山茱萸為補腎要藥 |
| 安神 | 酸棗仁為安神第一要藥 |
| 樹皮多用 | 厚朴、杜仲、肉桂均為樹皮 |

> 💡 木部藥多為補益和安神之品——枸杞、酸棗仁至今仍是日常養生的常用藥材。
