# 09-建筑科学 (UDC 69) (UDC 69)

> UDC 69 涵盖建筑实践的全过程——从地基到屋顶，从砖石到幕墙，从管道到暖通。它是将设计图纸转化为实体空间的应用科学。

## UDC 69 分支

| UDC | 分支 | 核心内容 |
|-----|------|----------|
| **691** | 建筑材料 | 混凝土、钢材、木材、玻璃、砖石 |
| **692** | 结构构件 | 基础、墙体、柱、梁、楼板、屋顶 |
| **693** | 砌体/混凝土/金属施工 | 砖石砌筑、混凝土浇筑、钢结构 |
| **694** | 木结构 | 木框架、CLT 正交胶合木、榫卯 |
| **696** | 建筑设备 | 给排水、电气、燃气 |
| **697** | 暖通空调 (HVAC) | 供暖、通风、空调、制冷 |
| **698** | 装修与装饰 | 抹灰、涂装、铺地、吊顶 |
| **699.8** | 建筑防护 | 防水、防火、隔热、隔声 |

## 建筑材料 (UDC 691)

| 材料 | 抗压 | 抗拉 | 典型应用 | 碳排放 |
|------|------|------|----------|--------|
| **混凝土** | ★★★★★ | ★ | 基础、柱、楼板 | 高 (~300kgCO₂/m³) |
| **钢筋** | ★★★★ | ★★★★★ | 与混凝土复合 | 中-高 |
| **木材** | ★★ | ★★★ | 低层结构、CLT | 负 (储碳) |
| **砖** | ★★★★ | ★ | 墙体 | 中 |
| **玻璃** | ★★★ | ★ | 幕墙、窗户 | 中-高 |
| **石材** | ★★★★★ | ★ | 基础、装饰 | 低 |

## 结构体系

| 体系 | 原理 | 高度 | 代表 |
|------|------|------|------|
| **框架结构** | 梁柱承重，墙不承重 | 低-超高层 | 大多数现代建筑 |
| **剪力墙** | 混凝土墙抵抗水平力 | 中-高层 | 住宅 |
| **筒体结构** | 外围密柱+核心筒 | 超高层 | 哈利法塔 |
| **木结构** | 木材框架/CLT | 低-中层 (≤18层) | Mjøstårnet |

## 建筑施工

### 基础工程

| 类型 | 适用 | 深度 |
|------|------|------|
| **独立基础** | 承载力好的土 | 浅 |
| **条形基础** | 墙下 | 浅 |
| **筏板基础** | 软弱土 | 浅-中 |
| **桩基础** | 深层承载力 | 深 (数十米) |

### 主体施工

| 工艺 | 说明 |
|------|------|
| 钢筋混凝土 | 支模→绑钢筋→浇筑→养护→拆模 |
| 钢结构 | 工厂预制→现场螺栓/焊接连接 |
| 预制装配式 | 工厂预制构件→现场拼装——速度快 30-50% |
| 逆作法 | 地上地下同时施工——缩短工期 |

## 建筑设备 (UDC 696-697)

| 系统 | 功能 |
|------|------|
| **给排水** | 供水、排水、消防栓、喷淋 |
| **电气** | 配电、照明、弱电、智能化 |
| **暖通** | 供暖 (锅炉/热泵)、通风 (新风)、空调 (VRV/中央) |
| **电梯** | 乘客、货梯、扶梯 |

## 绿色建筑

| 认证 | 地区 | 重点 |
|------|------|------|
| **LEED** | 美国→全球 | 能源、水、材料、室内环境 |
| **中国绿色建筑** | 中国 | 三星制：节地/节能/节水/节材 |
| **Passivhaus** | 德国 | 超低能耗——几乎不需主动供暖 |
| **WELL** | 全球 | 人居健康——空气/水/光/舒适 |
