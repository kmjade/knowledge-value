# 08-现代主义文学 (1900-1950)

> 世界不再是确定的——文学也不再是「讲故事」——形式本身的革命。

## 特征

| 意识流: 不是「他想了什么」——是思想的河流——跳跃·混乱——像真正的思维 |
|------|
| 碎片化: 叙事不连贯——时间跳跃——需要读者「拼」 |
| 神话方法: 用古代神话作为现代故事的骨架——《尤利西斯》=《奥德赛》 |

## 代表

| 乔伊斯《尤利西斯》(1922): 一天——都柏林——荷马史诗的现代版本——「最难读的小说」 |
|------|
| 伍尔夫: 「1920年12月左右——人的性格变了」 |
| T.S.艾略特《荒原》: 「四月是最残酷的月份」——现代世界的「荒原」 |
