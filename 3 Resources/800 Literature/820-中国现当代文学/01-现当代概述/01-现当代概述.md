# 01-中国现当代文学概述

> 1917年文学革命至今——一百年的文学——在革命、战争和改革开放中蜕变。

## 分期

| 时期 | 年代 | 关键词 |
|------|:---:|------|
| 五四时期 | 1917-1927 | 白话文·启蒙·个性解放 |
| 左翼时期 | 1927-1937 | 革命文学——阶级斗争 |
| 战争时期 | 1937-1949 | 抗日·内战——文学为战争服务 |
| 十七年 | 1949-1966 | 社会主义现实主义——「三红一创」 |
| 文革 | 1966-1976 | 样板戏——文学凋零 |
| 新时期 | 1978-1989 | 伤痕·反思·先锋——文学爆炸 |
| 后新时期 | 1990-2010 | 市场化——多元化 |
| 新世纪 | 2010- | 网络文学·科幻——新媒介 |
