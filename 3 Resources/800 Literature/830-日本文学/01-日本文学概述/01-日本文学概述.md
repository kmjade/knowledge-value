# 01-日本文学概述

> 从《源氏物语》(世界第一部小说)到村上春树——日本文学在世界文学中地位独特。

## 分期

| 时期 | 时代 | 代表 |
|------|------|------|
| 上古 | 奈良(710-794) | 《古事记》《万叶集》 |
| 中古 | 平安(794-1185) | 《源氏物语》《枕草子》 |
| 中世 | 镰仓·室町(1185-1603) | 军记物语·能乐·俳谐 |
| 近世 | 江户(1603-1868) | 井原西鹤·近松门左卫门 |
| 近代 | 明治·大正(1868-1926) | 夏目漱石·森鸥外 |
| 现代 | 昭和(1926-1989) | 川端康成·三岛由纪夫 |
| 当代 | 平成·令和(1989-) | 村上春树·吉本芭娜娜 |
