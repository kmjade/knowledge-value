# 05-元曲与杂剧

> 元代市民文化崛起——韵文走向民间——散曲和杂剧成为「一代之文学」。

## 元曲

| 散曲: 更自由——可以加衬字——更口语——马致远「枯藤老树昏鸦」 |
|------|

## 元杂剧

| 中国戏剧的成熟——四折一楔子——唱·白·科 |
|------|
| 关汉卿《窦娥冤》: 「地也——你不分好歹何为地！天也——你错勘贤愚枉做天！」 |
| 王实甫《西厢记》: 「愿天下有情人终成眷属」 |
