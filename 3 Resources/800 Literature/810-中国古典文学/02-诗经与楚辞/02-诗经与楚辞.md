# 02-诗经与楚辞

> 中国文学的双源——北方《诗经》的现实——南方《楚辞》的浪漫。

## 《诗经》

| 中国第一部诗歌总集——305首——西周初年至春秋中叶(~公元前1000-600年) |
|------|
| 风(民歌160篇)·雅(宫廷105篇)·颂(祭祀40篇) |
| 「关关雎鸠——在河之洲——窈窕淑女——君子好逑」——中国人背了2500年 |

## 《楚辞》

| 南方——楚国——屈原——《离骚》——「长太息以掩涕兮——哀民生之多艰」 |
|------|
| 和《诗经》的四言不同——《楚辞》长短句——更自由——更个人化 |
| 开创了中国文学的浪漫主义传统——「香草美人」的象征体系 |
