# 文學理論 Literary Theory
研究文學的本質、形式、功能與批評方法的理論體系。

## 主要流派
| 流派 | 關注點 | 代表人物 |
|:----|:------|:--------|
| **形式主義** | 文學語言的獨特性 | 什克洛夫斯基·雅科布森 |
| **結構主義** | 文學的深層結構 | 巴特·熱奈特 |
| **後結構主義** | 意義的不確定性 | 德希達·傅柯 |
| **馬克思主義** | 文學與階級意識 | 伊格頓·詹明信 |
| **女性主義** | 性別與文學 | 西蘇·巴巴拉·史密斯 |
| **後殖民** | 殖民歷史與文化身份 | 薩依德·史碧娃克 |

## 核心概念
- **陌生化** (Defamiliarization)：文學語言使日常事物變得陌生
- **互文性** (Intertextuality)：每一文本都是其他文本的引用
