# 01-文学理论概述

> 文学理论不是「怎么欣赏文学」——是「文学是什么」。

## 基本问题

| 问题 | 分支 |
|------|------|
| 文学是什么？ | 本质论 |
| 意义从哪里来？作者？文本？读者？ | 诠释学 |
| 文学和社会什么关系？ | 意识形态批评 |
| 「好」文学的标准是什么？ | 价值论 |

## 作者→文本→读者

| 19世纪: 意义=作者的意图 (「他想说什么」) |
|------|
| 20世纪中叶: 意义=文本本身 (「文本说了什么」——不管作者) |
| 20世纪后期: 意义=读者的建构 (「你读到了什么」) |
