# 結構主義 Structuralism
## 定義
結構主義將文化現象（語言、文學、神話）視為**符號系統**，意義來自系統內部的關係而非外部現實。

## 核心思想 (Saussure → Lévi-Strauss → Barthes)
- 語言是差異的系統（無正面項）
- 二元對立是意義的基本結構
- 「作者已死」(Barthes, 1967)
