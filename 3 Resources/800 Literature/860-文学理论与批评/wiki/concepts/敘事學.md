# 敘事學 Narratology
## 定義
敘事學研究**故事如何被講述**——敘事的結構、視角、時間和敘述者。

## 核心概念
| 概念 | 說明 |
|------|------|
| **敘述者 Narrator** | 誰在講故事？（第一人稱/第三人稱/全知/不可靠） |
| **視角 Focalization** | 通過誰的眼睛看？（Genette） |
| **故事 vs 話語** | Story (內容) vs Discourse (表達方式) |
| **敘事時間** | 順序、時長、頻率 |

## 代表人物: Gerard Genette, Vladimir Propp
