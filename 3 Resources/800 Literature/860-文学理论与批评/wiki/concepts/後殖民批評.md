# 後殖民批評 Post-Colonial Criticism
## 定義
研究**殖民主義及其後果**如何在文學中被呈現、抵抗和重構。

## 核心概念 (Said, Spivak, Bhabha)
| 概念 | 說明 |
|------|------|
| **東方主義** (Said) | 西方如何建構「東方」為他者 |
| **庶民能否發聲？** (Spivak) | 被邊緣化者的聲音 |
| **混雜性 Hybridity** (Bhabha) | 文化接觸中的混合與創造 |
