# 04-西方诗歌传统

| 荷马: 史诗——英雄 / 萨福: 抒情——爱情——「你来了——我为你痴狂」 |
|------|
| 但丁《神曲》: 地狱·炼狱·天堂——「在人生中途——我迷失了正路」 |
| 莎士比亚十四行诗: 「我能否将你比作夏日？」 |

## 浪漫主义

| 华兹华斯: 「诗是强烈情感的自然流露」 / 济慈: 「美即是真——真即是美」 |
