# 創作者偏誤 (Writer-Centric Bias)

## 定義

**創作者偏誤**指主流筆記方法論（Zettelkasten、PARA、CODE、費曼學習法等）的發明者大多是專業寫作者和學者，他們的方法預設了「知識輸出 = 寫作/出版」，不適合以行動和決策為主要輸出的普通知識工作者。

## 核心原理

### 方法論發明者的背景分析

| 方法論 | 發明者 | 職業 | 輸出形式 |
|--------|--------|------|----------|
| Zettelkasten | Niklas Luhmann | 社會學教授 | 70 本書、400+ 論文 |
| PARA / CODE | Tiago Forte | 作家、講師 | 書籍、課程 |
| 費曼學習法 | Richard Feynman | 物理學教授 | 教材、演講 |
| 漸進式摘要 | Tiago Forte | 作家 | 書籍 |
| 康奈爾筆記 | Walter Pauk | 教育學教授 | 教材 |

**共同特徵**: 他們的「產品」是文字，筆記是寫作的原材料。

### 普通知識工作者的真實需求

| 知識工作者的輸出 | 需要的知識形式 |
|-----------------|---------------|
| 做一個商業決策 | 關鍵事實 + 選項 + 風險 |
| 解決一個技術問題 | 診斷邏輯 + 解決方案 |
| 說服一個同事 | 論據 + 數據 + 故事 |
| 學習一項新技能 | 操作步驟 + 常見錯誤 |

**他們需要的不是「卡片」，而是可被激活的知識。**

## 實踐啟示

1. **不要盲目跟隨方法論** — 先問：這個方法的發明者是為了解決什麼問題？我和他的使用場景一樣嗎？
2. **以輸出倒推輸入** — 你的最終輸出是什麼？反向設計你的知識管理流程
3. **放棄完美主義** — 你不是在寫百科全書，你是在建立一個支持決策的系統
