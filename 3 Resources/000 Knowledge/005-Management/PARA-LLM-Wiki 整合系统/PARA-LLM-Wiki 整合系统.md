## 🔄 核心工作流

```
每日開機                          工作中                           每日關機
   │                               │                               │
   ├─ /daily-open                  ├─ 信息捕獲 → Inbox              ├─ /lint --quick
   ├─ /context --quick             ├─ Inbox > 閾值 → /triage        ├─ git status
   ├─ 確認今日意圖                  ├─ raw/ 新材料 → /wiki-compile   ├─ git commit + push
   └─ 檢視優先級                    └─ 隨時 /context 查看狀態         └─ 更新日終回顧
```

> 📖 完整規則見 [[05-日常維護流程]]
