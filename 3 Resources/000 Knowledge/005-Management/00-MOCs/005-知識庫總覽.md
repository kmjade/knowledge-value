# 知識管理知識庫總覽 (DDC 005)

> 知識管理（Knowledge Management）是組織和個人系統性地創造、獲取、共享和應用知識的學科。
> 從 Nonaka 的 SECI 螺旋到 AI 驅動的知識激活——這是 KM 的完整圖譜。

## 知識類型光譜

| | 顯性知識 Explicit | 隱性知識 Tacit |
|:--|:-----------------|:--------------|
| 性質 | 可編碼·可文檔化 | 個人化·難以言說 |
| 傳遞 | 文檔·數據庫·手冊 | 師徒制·觀察·實踐 |
| 例子 | SOP·專利·公式 | 騎車·烹飪火候·談判直覺 |
| KM 挑戰 | 組織·檢索·更新 | **外化** (Externalization) |
