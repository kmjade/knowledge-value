# 06-文献学

> 文献学是中国独有的学术传统，研究文献的形态、版本、目录和校勘——"辨章学术，考镜源流"。在数字时代，它的方法同样适用于电子文献的版本管理和来源追溯。

## 文献学的三大分支

| 分支 | 核心问题 | 方法 |
|------|----------|------|
| **版本学** | 这本书有多少个版本？哪个最可靠？ | 版本比对、版式分析、避讳字考 |
| **目录学** | 有什么书？这些书讲什么？ | 书目编纂、分类体系、提要撰写 |
| **校勘学** | 文本中的错漏如何纠正？ | 对校、本校、他校、理校 |

## 版本学

### 版本类型

| 类型 | 说明 |
|------|------|
| **稿本** | 作者手稿——最原始版本 |
| **抄本** | 手工抄写——印刷术之前的主要传播方式 |
| **刻本** | 雕版印刷——宋刻本最珍贵 |
| **活字本** | 活字印刷——毕昇发明，但中国以刻本为主流 |
| **石印本** | 清末民初的石版印刷技术 |
| **排印本** | 铅字排版印刷（近现代） |

> 善本：具有历史文物性、学术资料性和艺术代表性的古籍版本。宋元刻本、名家稿本、精校精注本都属于善本。

### 现代版本意识

版本意识不只是古籍的事：

- 一本书的第 1 版和第 3 版可能完全不同
- 翻译版本 vs 原版
- 论文的 preprint → accepted → published 之间可能有关键差异
- 软件和代码也有版本（Git 就是版本管理工具）

## 目录学

### 从《七略》到《四库全书》

| 时代 | 代表著作 | 分类体系 |
|------|----------|----------|
| 西汉 | 刘向、刘歆《七略》 | 六艺、诸子、诗赋、兵书、术数、方技（六分法） |
| 唐 | 《隋书·经籍志》 | **经史子集 四分法**正式确立 |
| 清 | 《四库全书总目》 | 四分法的巅峰，200 卷 |
| 近代 | 中图法 | 现代分类法 |

### 经史子集

| 部类 | 包含 | 相当于 |
|------|------|--------|
| **经** | 儒家经典及注释 | 意识形态/核心价值观 |
| **史** | 历史、地理、政书 | 历史学+政治学 |
| **子** | 诸子百家、宗教、科技 | 哲学+宗教+自然科学 |
| **集** | 诗文词曲 | 文学 |

## 校勘学

### 陈垣"校法四例"

| 校法 | 方法 | 适用 |
|------|------|------|
| **对校** | 用同一书的不同版本互相对比 | 最基本的校勘方法 |
| **本校** | 用本书前后文互证 | 作者自相矛盾之处 |
| **他校** | 用其他书中引用的本书文字来校 | 本书失传但被别书引用 |
| **理校** | 无版本依据，靠逻辑推理 | 万不得已时用，风险最高 |

### 校勘记的格式

```
原文: 学而时习之，不亦说乎
校记: "说"——皇侃本作"悦"，《说文》："说，释也"，"悦"为后起字。
```

## 数字文献学

| 挑战 | 说明 |
|------|------|
| 电子书版本 | Kindle 上悄悄改了错字——你的版本和别人的不一样 |
| 网页存档 | 网页随时可能变化或消失——Internet Archive 是数字时代的"藏本" |
| 数据溯源 | 数据集从哪来的？经过了什么处理？——数字文献学的新问题 |
