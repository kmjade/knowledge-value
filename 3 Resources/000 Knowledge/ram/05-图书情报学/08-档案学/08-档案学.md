# 08-档案学

> 档案学是研究档案的形成、管理和长期保存的学科。档案与图书的本质区别：档案是原始记录（唯一性），图书是出版品（多副本）。数字保存是档案学增长最快的领域。

## 档案 vs 图书 vs 出版物

| 维度 | 档案 | 图书 | 期刊论文 |
|------|------|------|----------|
| 性质 | 原始记录（唯一） | 出版品（多副本） | 出版品（多副本） |
| 产生 | 自然积累（工作过程） | 有意创作 | 有意创作 |
| 组织 | 全宗原则（按来源） | 分类法（按主题） | ISSN/卷期 |
| 价值 | **证据价值** > 信息价值 | 信息价值 | 信息价值 |
| 访问 | 受控（隐私、保密） | 开放 | 开放（可能付费） |

## 档案管理三大原则

### 来源原则

**档案按产生机构/个人组织**，不能打散按主题重组——这是档案学的基石。

```
全宗 (Fonds) — 一个机构的全部档案
  └── 系列 (Series) — 按职能分组
        └── 案卷 (File) — 同一事务的文件
              └── 件 (Item) — 单份文件
```

### 原始顺序

保持形成时的原始顺序——这个顺序本身包含信息。

### 鉴定

决定什么值得永久保存。档案鉴定是最考验判断力的档案工作。

| 价值 | 说明 |
|------|------|
| 证据价值 | 证明机构活动和权利 |
| 信息价值 | 对历史研究的意义 |
| 内在价值 | 物理特征不可替代（如签名原件） |

## OAIS 数字保存模型

OAIS (ISO 14721) 是数字保存的国际标准：

```
生产者 → SIP → [摄取 → 归档存储 → 访问] → DIP → 消费者
                ↑           ↓
              数据管理    行政管理
```

| 包类型 | 含义 |
|--------|------|
| **SIP** | 提交信息包——原始数据 |
| **AIP** | 存档信息包——完整元数据+数据 |
| **DIP** | 分发信息包——用户获取的版本 |

## 数字保存的挑战

| 挑战 | 例子 |
|------|------|
| **格式过时** | 1990 年代的 WordStar 文件今天怎么打开？ |
| **介质退化** | 光盘寿命 10-50 年，硬盘 3-5 年 |
| **软件依赖** | Flash 动画依赖 Flash 播放器（已终止支持） |
| **规模爆炸** | 数字信息指数增长，不可能全保存 |
| **真实性证明** | 怎么证明数字文件没被篡改？ |

### 保存策略

| 策略 | 方法 |
|------|------|
| **迁移** | 转成新格式（最常用） |
| **仿真** | 模拟原始软件环境运行（游戏、交互性作品） |
| **封装** | 数据+描述打包，附带"说明书" |
