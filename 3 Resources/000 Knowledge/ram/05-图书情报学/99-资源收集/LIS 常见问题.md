# LIS 常见问题

## 十大常见误解

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| 图书情报学 = 学整理书 | 这是研究知识的生产、组织、传播和利用的跨学科学科——整理书只是图书馆学 1% 的内容 |
| 图书馆员只是借还书的 | 学科馆员做研究支持、数据管理、文献计量分析、系统开发——很多馆员有双硕士甚至博士 |
| 学情报学就是学间谍 | 中国"情报学"源自日本汉字的 information，对应西方的 Information Science——不是 intelligence |
| 这个学科找不到工作 | 数据管理、信息架构、知识管理、学术出版、竞争情报——都是 LIS 毕业生在做的事 |
| Google 淘汰了信息检索 | Google 是 Web 搜索——学术数据库、机构知识库、档案系统需要完全不同的检索方法 |
| 档案学就是存档文件 | 数字保存是硬核技术问题——"让 WordPerfect 5.1 文件在今天还能打开"需要深度的格式知识 |
| 文献学只研究古籍 | 文献学的方法（版本、目录、校勘）同样适用于数字文献——电子书的版本管理就是现代文献学 |
| 影响因子 = 论文质量 | 期刊影响因子不能评价单篇论文——这是 DORA 宣言的核心论点 |
| 开放获取 = 免费看论文 | OA 是学术出版商业模式的根本变革，涉及版权改革和知识民主化 |
| 数字人文 = 用电脑研究文科 | DH 是方法论革命——远读、网络分析、GIS 正在重新定义人文学科的研究范式 |

## "图书情报学" vs "信息科学" vs "图书馆学"

| 概念 | 关系 |
|------|------|
| 图书馆学 | 研究图书馆（理论与实践）——最古老的分支 |
| 信息科学 / 情报学 | 研究信息的性质和检索——1960s 从图书馆学中发展而来 |
| 图书情报学 (LIS) | **图书馆学 + 信息科学的整合**——现代学科名称 |
| 图书情报与档案管理 | 中国学科目录中的完整名称——再加档案学 |

```
        图书情报与档案管理 (中国)
        ┌─────────┬─────────┬──────────┐
        │ 图书馆学  │ 情报学    │  档案学    │
        └─────────┴─────────┴──────────┘
        └──────────┬──────────┘
              图书情报学 (LIS)
```

> 本知识库采用"图书情报学"（LIS）作为总称，覆盖图书馆学+情报学+档案学+文献学+出版学五大分支。
