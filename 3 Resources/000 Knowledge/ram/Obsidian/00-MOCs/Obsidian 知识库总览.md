## 为什么用 Obsidian？

| 特性 | 说明 |
|------|------|
| **本地优先** | 数据完全在你手上，Markdown 纯文本 |
| **双向链接** | `[[wikilink]]` 构建知识网络 |
| **图谱视图** | 可视化笔记之间的关联 |
| **插件生态** | 2000+ 社区插件，高度可定制 |
| **完全免费** | 个人使用全功能免费 |
| **离线可用** | 无需网络，秒开秒写 |
| **跨平台** | Windows / macOS / Linux / iOS / Android |
