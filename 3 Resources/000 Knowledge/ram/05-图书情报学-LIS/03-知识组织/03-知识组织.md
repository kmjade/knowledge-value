# Knowledge Organization

> 知识组织 (Knowledge Organization, KO) 是 LIS 的核心技术——研究如何系统地描述、分类和关联知识，使之可发现、可理解、可复用。分类法、叙词表、元数据和本体是其四大支柱。

## 知识组织的层次

```
语义丰富度 ↑
┌──────────────────┐
│   本体 (Ontology)       │  概念的关系网络 + 逻辑推理
├──────────────────┤
│   叙词表 (Thesaurus)     │  层级 + 相关关系 + 同义词控制
├──────────────────┤
│   分类法 (Classification)│  层级结构，学科分类
├──────────────────┤
│   元数据 (Metadata)      │  描述单个资源的属性
├──────────────────┤
│   标签 (Tags / Folksonomy)│  非受控的用户标注
└──────────────────┘
```

## 分类法

### 三大分类法对比

| 分类法           | 创建者          | 年代   | 结构            | 使用范围     |
| ------------- | ------------ | ---- | ------------- | -------- |
| **DDC**       | Melvil Dewey | 1876 | 000-999 十进    | 全球最广     |
| **UDC**       | Paul Otlet   | 1905 | DDC 扩展，支持组合符号 | 欧洲       |
| **LCC**       | 美国国会图书馆      | 1897 | 字母+数字         | 英美研究型图书馆 |
| **中图法 (CLC)** | 中国           | 1975 | 22 大类，字母      | 中国所有图书馆  |

### 中图法 22 大类

| 字母 | 大类 | 字母 | 大类 |
|------|------|------|------|
| A | 马、列、毛 | N | 自然科学总论 |
| B | 哲学、宗教 | O | 数理科学和化学 |
| C | 社会科学总论 | P | 天文学、地球科学 |
| D | 政治、法律 | Q | 生物科学 |
| E | 军事 | R | 医药、卫生 |
| F | 经济 | S | 农业科学 |
| **G** | **文化、科学、教育、体育** | T | 工业技术 |
| H | 语言、文字 | U | 交通运输 |
| I | 文学 | V | 航空、航天 |
| J | 艺术 | X | 环境科学 |
| K | 历史、地理 | Z | 综合性图书 |

> G25 = 图书馆学。G35 = 情报学。T 大类是工业技术，TP 是自动化/计算机。

## 元数据

元数据 = 关于数据的数据。好元数据让资源从"找不到"变成"瞬间找到"。

### Dublin Core (都柏林核心)

15 个元素——最通用的元数据标准：

| 元素 | 说明 |
|------|------|
| Title | 资源名称 |
| Creator | 创建者 |
| Subject | 主题 |
| Description | 描述 |
| Publisher | 发布者 |
| Date | 日期 |
| Type | 类型 |
| Format | 格式 |
| Identifier | 唯一标识（DOI, ISBN） |
| Source | 来源 |
| Language | 语言 |
| Relation | 相关资源 |
| Coverage | 时空范围 |
| Rights | 权限 |
| Contributor | 贡献者 |

## 叙词表 (Thesaurus)

受控词表，标注术语间的三种关系：

| 符号 | 关系 | 例子 |
|------|------|------|
| BT | 上位词 (Broader) | 猫 BT 哺乳动物 |
| NT | 下位词 (Narrower) | 猫 NT 暹罗猫 |
| RT | 相关词 (Related) | 猫 RT 宠物 |
| USE/UF | 规范词 / 非规范词 | "猫咪" USE "猫" |

## 本体 (Ontology)

最形式化的知识组织——明确定义类、属性和关系，支持逻辑推理。

知识图谱就是大规模本体的应用实例。Google 知识图谱、Wikidata 都是以本体为核心构建的。
