# 05-图书馆学

> 图书馆学是 LIS 最古老的分支，研究图书馆的运作原理、社会功能和信息服务。现代图书馆是社区信息中心、数字资源门户和信息素养教育者——远超"藏书楼"。

## 图书馆学五定律 (Ranganathan, 1931)

五条简洁而深刻的定律——

| 定律 | 原句 | 现代解读 |
|------|------|----------|
| 1 | Books are for use | 资源是为了用的，不是供着的 |
| 2 | Every reader his/her book | 每位用户都能找到需要的信息 |
| 3 | Every book its reader | 每份资源都有需要它的用户 |
| 4 | Save the time of the reader | 用户体验第一——系统要高效 |
| 5 | Library is a growing organism | 图书馆随社会需求不断演进 |

> 用"信息"替换"book"，这五条完全适用于今天的任何信息系统。

## 图书馆类型

| 类型 | 服务对象 | 核心使命 |
|------|----------|----------|
| **国家图书馆** | 全国 | 保存国家出版物（呈缴本制度）、国家书目 |
| **公共图书馆** | 社区居民 | 免费信息获取、阅读推广、社区空间 |
| **高校图书馆** | 师生 | 教学科研支持、学科服务、信息素养教育 |
| **专业图书馆** | 机构/行业 | 深度主题服务（法律、医学、企业） |
| **中小学图书馆** | 中小学生 | 阅读培养、基础信息技能 |

## 核心业务：资源建设

### 采访 → 编目 → 典藏

| 环节 | 做什么 |
|------|--------|
| **采访** | 决定买什么（选书、数据库订阅谈判） |
| **编目** | 用 MARC/RDA 标准描述资源，创建书目记录，赋予索书号 |
| **典藏** | 上架、保存、剔除（剔旧） |

### FRBR 模型

书目世界的实体-关系模型：

```
作品 (Work) ── 《红楼梦》
  ├── 内容表达 (Expression) ── 曹雪芹原文 / 英文译本 / 评注本
  │     └── 载体表现 (Manifestation) ── 人民文学 2020 / 中华书局 2019
  │           └── 单件 (Item) ── 北大馆藏的那一本 / 清华馆藏的那一本
```

> FRBR 让用户看到"同一作品的不同版本"——这对学术研究至关重要。

## 读者服务

| 服务 | 说明 |
|------|------|
| **参考咨询** | 从"那本书在哪"到"帮我做文献综述"——图书馆员的核心价值 |
| **信息素养教育** | 教用户找、评、用信息——21 世纪基础技能 |
| **学科服务** | 学科馆员嵌入院系，做深度支持 |
| **馆际互借** | 本馆没有？从其他馆调 |

## 数字图书馆

| 要素 | 说明 |
|------|------|
| 数字资源管理 | 电子书、数据库、数字化特藏 |
| 发现系统 | 跨数据库的统一搜索（如 Primo, Summon） |
| 机构知识库 | 大学学术成果的数字仓储（DSpace 等平台） |
| 长期保存 | 数字资源不因技术过时而丢失 |
