# 07-出版学

> 出版学研究知识和信息如何被创造、选择、编辑、生产和传播——从论文投稿到书籍发行，从同行评审到开放获取运动。

## 学术出版的生命周期

```
研究 → 写作 → 投稿 → 编辑初审 → 同行评审 → 修改 → 录用 
  → 编辑加工 → 排版 → 出版 → 传播 → 索引收录 → 引用
```

## 文献类型谱系

| 类型 | 速度 | 评审 | 权威性 | 典型 |
|------|------|------|--------|------|
| 预印本 (Preprint) | 即时 | 无 | 需谨慎 | arXiv |
| 会议论文 | 较快 | 同行评审 | 高 (取决于会议) | CS 领域主要形式 |
| 期刊论文 | 1-2 年 | 同行评审 (2-4 审稿人) | 最高 | Nature, 情报学报 |
| 专著 | 2-5 年 | 编辑+同行评审 | 高 | 学术书籍 |
| 学位论文 | — | 导师+评审委员会 | 高 | 硕士/博士论文 |

## 同行评审

| 模式 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| **单盲** | 审稿人知道作者 | 有上下文 | 可能偏见 |
| **双盲** | 互不知道 | 减少偏见 | 大牛可能被认出 |
| **开放** | 双方知晓，报告可能公开 | 透明 | 年轻学者不敢批评大佬 |
| **发表后评审** | 先发再评 | 速度快 | 质量控制弱 |

> 同行评审不完美——慢、保守、偶尔不公平。但至今没有更好的替代方案。

## 开放获取 (Open Access)

| 模式 | 读者 | 作者成本 | 代表 |
|------|------|----------|------|
| **金色 OA** | 免费 | 付 APC (~$2000-5000) | PLOS, eLife |
| **绿色 OA** | 免费 | 免费（自存档） | arXiv, 机构知识库 |
| **钻石 OA** | 免费 | 免费 | 资助型期刊 |
| **订阅制** | 付费 | 免费 | 传统期刊 |

### Plan S

2021 年起，欧洲 cOAlition S 资助的研究必须**立即** OA，不允许 embargo。激进但推动了变革。

### 学术出版的悖论

```
研究者免费提供内容 → 免费审稿 → 机构花巨资买回
```

这就是 OA 运动批判的核心。

## 学术标识符

| 标识符 | 标识什么 | 例子 |
|--------|----------|------|
| **DOI** | 论文/数据集的永久链接 | 10.1000/xyz123 |
| **ISBN** | 图书唯一标识 | 978-7-xxx-xxxxx-x |
| **ISSN** | 期刊唯一标识 | 1002-1027 |
| **ORCID** | 研究者唯一标识 | 解决"李伟"是谁的问题 |
| **ROR** | 研究机构唯一标识 | 解决"清华"是哪个清华 |

## 版权与许可

| 许可 | 含义 |
|------|------|
| **CC BY** | 署名即可——最开放 |
| **CC BY-SA** | 署名+相同方式共享（维基百科使用） |
| **CC BY-NC** | 署名+非商业使用 |
| **CC BY-ND** | 署名+禁止演绎 |
| **CC0** | 完全放弃版权（公共领域） |
