# Science of Science & Epistemology

> 科学学 (Science of Science) 用科学方法研究科学本身——知识是如何被创造的？什么是"科学知识"？如何衡量科学的影响力？这章提供图书情报学的知识论基础。

## 知识论基础

### 什么是知识？

柏拉图以来，知识被定义为 **"被证成的真信念" (Justified True Belief)**：

| 要素 | 含义 |
|------|------|
| 信念 (Belief) | 你必须相信它 |
| 真 (Truth) | 它必须是真的 |
| 证成 (Justification) | 你必须有理由相信它 |

> Gettier 问题 (1963) 挑战了这个经典定义——存在"被证成的真信念"但似乎不是知识的反例。至今争论未息。

### 显性知识 vs 隐性知识

| 类型 | 特征 | 传递方式 | 例子 |
|------|------|----------|------|
| **显性知识** | 可以编码、写下来 | 文档、公式、数据库 | 操作手册、数学定理 |
| **隐性知识** | 难以言传、嵌入经验 | 师徒制、实践、观察 | 骑自行车、诊断直觉 |

> 图书情报学主要处理显性知识，但隐性知识的捕获和转化（SECI 模型）是知识管理的关键。

## 科学学

### 科学如何生产知识？

| 阶段 | 说明 |
|------|------|
| 观察/提问 | 发现现象，提出问题 |
| 假设 | 提出可检验的假说 |
| 实验/数据 | 收集证据 |
| 同行评审 | 经过学术共同体检验 |
| 发表 | 知识进入公共领域 |
| 复现 | 其他研究者独立验证 |
| 共识形成 | 经过反复验证后成为科学共识 |

> **可证伪性** (Popper)：科学命题必须能被证明是错误的。不能被证明错误的不是科学（如弗洛伊德理论）。

### 科学的规范结构 (Merton, 1942)

| 规范 | 含义 |
|------|------|
| **公有性 (Communism)** | 科学发现属于科学共同体，不是个人私有 |
| **普遍性 (Universalism)** | 科学评价不取决于发现者的个人属性（种族、国籍、身份） |
| **无私利性 (Disinterestedness)** | 科学家的动机应该是求知，而非个人利益 |
| **有组织的怀疑 (Organized Skepticism)** | 一切都要经过批判性检验 |

> 这四条是理想化的——现实中科学家也有竞争、偏见和利益冲突。但它们定义了科学的规范目标。

### 库恩的科学革命

**Thomas Kuhn** 提出科学不是线性累积的：

```
前范式 → 常规科学 → 反常 → 危机 → 科学革命 → 新范式
  ↑________________________________________________↓
```

| 概念 | 含义 |
|------|------|
| **范式 (Paradigm)** | 一个科学共同体共享的理论框架、方法和假设 |
| **常规科学** | 在范式内"解谜" |
| **反常** | 当前范式解释不了的现象 |
| **科学革命** | 旧范式被新范式取代（如牛顿→爱因斯坦） |

> 图书情报学关注：不同范式下的知识组织方式也不同。DDC 反映的是 19 世纪的学科分类范式。

## 知识社会学

知识不是凭空产生的——社会因素影响知识的创造和接受：

| 概念 | 含义 | 例子 |
|------|------|------|
| 社会建构 | 知识受到社会环境的影响 | 不同文化对"疾病"的分类不同 |
| 马太效应 | 已有的获得更多，没有的连已有的也被夺走 | 高被引论文被引更多 |
| 无形学院 | 通过非正式交流形成的精英研究者网络 | 某领域的核心邮件列表/闭门会议 |

## 科学计量学

用定量方法研究科学的生产和传播——详见 [[02-Science-Epistemology|本章]] 核心：

| 指标 | 含义 | 局限 |
|------|------|------|
| 影响因子 (IF) | 期刊"平均"影响力 | 不适合评价单篇论文 |
| h 指数 | 研究者影响力 | 不能跨学科比较 |
| 引文分析 | 知识的流动路径 | 引用动机复杂 |
| Altmetrics | 在线关注度 | 不等于质量 |

> DORA 宣言 (2012)：**不要用影响因子评价单个研究者或论文。**
