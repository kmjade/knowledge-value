## 🔬 核心数据库

| 数据库 | 类型 | 特点 |
|--------|------|------|
| **Web of Science** (SCI/SSCI/AHCI) | 引文数据库 | 引文分析权威 |
| **Scopus** | 引文数据库 | 覆盖更广 |
| **CNKI (中国知网)** | 全文+引文 | 中文学术最全 |
| **万方数据** | 全文 | 学位论文强 |

| **Google Scholar** | 引文搜索 | 免费，覆盖广 |

## 📄 核心标准

| 标准 | 用途 |
|------|------|
| **MARC** | 图书馆书目记录格式 |
| **Dublin Core** | 通用元数据标准 |
| **FRBR** | 书目记录的实体-关系模型 |
| **RDA** | 新一代编目规则 |
| **OAIS** (ISO 14721) | 数字保存参考模型 |
| **SKOS** | 叙词表/分类法的语义网表达 |
| **FAIR** | 科研数据管理原则 |
