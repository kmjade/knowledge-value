# 01-学科概述

> 图书情报学 (LIS) 是研究知识和信息的产生、组织、存储、检索、传播和利用的学科。它跨越人文、社会科学和技术，是信息社会的基础设施学科。

## 学科的三条腿

```
         ┌──────────────┐
         │   图书馆学     │  ← 人文传统：服务、文化记忆、信息素养
         │  (最古老)      │
         └──────┬───────┘
                │
    ┌───────────┼───────────┐
    │           │           │
    ↓           ↓           ↓
┌────────┐ ┌────────┐ ┌────────┐
│ 情报学   │ │ 档案学   │ │ 出版学   │
│(信息行为)│ │(记录保存)│ │(知识传播)│
└────────┘ └────────┘ └────────┘
    ↑           ↑           ↑
    └───────────┼───────────┘
                │
         ┌──────────────┐
         │   文献学       │  ← 中国独有的学术传统
         │ (版本/目录/校勘) │
         └──────────────┘
```

## 学科简史

| 时期 | 里程碑 |
|------|--------|
| 古代 | 中国：刘向《别录》(前 26 年) 编纂最早的国家书目。西方：亚历山大图书馆 |
| 19 世纪末 | Melvil Dewey 创建杜威十进分类法 (DDC, 1876)，图书馆学专业化开始 |
| 1895 | Paul Otlet 和 Henri La Fontaine 创建国际书目学会——信息科学先驱 |
| 1931 | S. R. Ranganathan 提出"图书馆学五定律" |
| 1945 | Vannevar Bush 发表 "As We May Think"——提出 Memex 概念 |
| 1960s | "信息科学" (Information Science) 成为独立学科 |
| 1970s-80s | 在线检索系统、图书馆自动化 |
| 1990s | 互联网和数字图书馆革命 |
| 2000s | iSchools 运动——信息学院整合 CS + LIS |
| 2010s-至今 | 数据科学、数字人文、AI 对 LIS 的影响 |

## 核心概念

### DIKW 金字塔

知识的层次模型：

```
         /\
        /智慧\        应用知识做出正确的判断
       /Wisdom\
      /────────\
     /   知识    \     被内化和理解的信息
    / Knowledge  \
   /──────────────\
  /      信息       \   有上下文和意义的数据
 /   Information    \
/────────────────────\
        数据           原始的事实和符号
        Data
```

### 信息生命周期

```
创造 → 采集 → 组织 → 存储 → 检索 → 传播 → 使用 → 保存/销毁
```
