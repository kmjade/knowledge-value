# 知识学导论 知识库总览 (DDC 001)

> 知识学导论（Prolegomena）追问知识本身——它是什么、从哪里来、有哪些形态、如何与文化和真理关联。作为 DDC 001「知识」的哲学地基，本库为 Knowledge-Systems (KOS) 知识库提供理论基础。
