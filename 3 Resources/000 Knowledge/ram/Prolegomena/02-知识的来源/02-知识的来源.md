# Sources of Knowledge

> 知识从哪里来？理性主义说「从思维中来」，经验主义说「从感官中来」。这场争论持续了四百年，最终的答案是：两者都需要。

## 理性主义 vs 经验主义

| 维度 | 理性主义 (Rationalism) | 经验主义 (Empiricism) |
|------|----------------------|---------------------|
| 知识来源 | 理性、思维、演绎 | 感官、经验、归纳 |
| 代表 | 笛卡尔、斯宾诺莎、莱布尼茨 | 洛克、贝克莱、休谟 |
| 典范 | 数学、逻辑 | 自然科学 |
| 先天观念 | 存在（天赋观念） | 不存在（心灵是白板） |
| 方法 | 演绎推理 | 归纳推理 |

### 笛卡尔的激进怀疑

> 「我思故我在」(Cogito, ergo sum) — 笛卡尔

| 步骤 | 内容 |
|------|------|
| 1 | 怀疑一切——感官会骗人，梦境与醒难以区分 |
| 2 | 唯一不能怀疑的：我在怀疑这一事实本身 |
| 3 | 从「我思」出发，用理性演绎一切知识 |

### 洛克的白板说

> 「心灵是一张白纸，没有任何先天字符」— 洛克《人类理解论》

| 观念类型 | 来源 | 例子 |
|---------|------|------|
| 简单观念 | 直接来自感觉 | 颜色、声音、味道 |
| 复杂观念 | 心灵对简单观念的组合 | 美、正义、独角兽 |

## 先验与后验 (A Priori / A Posteriori)

| 类型 | 含义 | 例子 | 确证方式 |
|------|------|------|---------|
| **先验** (A Priori) | 独立于经验的知识 | 2+2=4、所有单身汉都未婚 | 纯理性 |
| **后验** (A Posteriori) | 依赖经验的知识 | 水在 100°C 沸腾、地球绕太阳转 | 观察/实验 |

### 分析 vs 综合 (康德的关键区分)

| 类型 | 谓词是否蕴含于主词？ | 例子 |
|------|---------------------|------|
| **分析命题** | 是——仅靠定义即可判断 | 「所有单身汉都未婚」 |
| **综合命题** | 否——需要经验判断 | 「这朵玫瑰是红色的」 |

> 康德追问：有没有**先天综合知识**？——先于经验但增进了我们对世界的理解？他的答案：数学和形而上学就是。

## 知识的其他来源

| 来源 | 说明 | 认识论地位 |
|------|------|-----------|
| **知觉** (Perception) | 五官对外部世界的直接感知 | 经验主义的基础——但幻觉和错觉是问题 |
| **记忆** (Memory) | 对过去经验的存储和调用 | 次生来源——但记忆会扭曲和遗忘 |
| **证言** (Testimony) | 他人告诉我们的知识 | 大多数知识来自证言——但信任需要理由 |
| **内省** (Introspection) | 对自己心理状态的觉察 | 笛卡尔式的特权通道——但弗洛伊德质疑其可靠性 |
| **直觉** (Intuition) | 不经过推理的直接把握 | 数学直觉、道德直觉——但可能出错 |
| **推理** (Inference) | 从已知推导未知 | 演绎（必然）和归纳（概然） |

### 证言：被低估的知识来源

> 我们相信地球是圆的、DNA 是双螺旋、拿破仑在滑铁卢战败——不是因为我们自己验证了这些，而是因为有人告诉我们。

| 问题 | 讨论 |
|------|------|
| 证言是知识的基础来源还是派生来源？ | 还原论（可以还原为知觉+推理） vs 非还原论（证言是独立来源） |
| 什么时候该相信证言？ | Hume: 见证者的可靠性 × 事件的先天概率 |

## 知识的限度

| 限度类型 | 内容 | 代表人物 |
|---------|------|---------|
| **休谟的归纳问题** | 归纳没有理性保证——太阳明天会升起只是习惯 | 休谟 |
| **康德的物自体** | 我们只能认识现象，不能认识物自体 | 康德 |
| **哥德尔不完备** | 任何足够强的形式系统都有不可证的真理 | 哥德尔 |
| **认知科学的边界** | 进化给我们的是「够用」的认知，不是完美的 | 进化认识论 |
