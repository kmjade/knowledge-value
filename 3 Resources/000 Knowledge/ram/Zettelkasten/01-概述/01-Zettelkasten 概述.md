# 01 - Zettelkasten 概述

## 起源

Niklas Luhmann（1927-1998），德国社会学家。

```
Luhmann 的工作方式：

每天读 → 写卡片（手写小纸条）→ 编号 → 放入盒子 → 互相关联

30 年 → 9 万张卡片 → 70 本专著 + 400+ 篇论文
```

他称自己的卡片盒为"第二个我"（ein zweites Ich）。

## 核心哲学

### 1. 不是收集，是对话

```
❌ 传统笔记：记录 → 归档 → 遗忘
✅ Zettelkasten：记录 → 思考 → 链接 → 生长
```

### 2. 惊喜的价值

> "Zettelkasten 最好的时刻，是当你看到两条看似无关的卡片，突然产生了连接。"

### 3. 用自己的话写

```
原文：The fundamental problem of communication is reproducing at one point a message selected at another point. (Shannon)

❌ 复制粘贴
✅ 香农认为，通信的本质是在另一个地方复现信息。这让我想到...（你的思考）
```

## 三类笔记

### 闪念笔记（Fleeting Notes）

```
作用：快速捕获想法
存放：Inbox，1-2 天内处理
例子：
  "GPT-4o 的视觉能力可以用于代码审查？"
  "刚刚会议上张总说的那个问题，值得深入"
```

### 文献笔记（Literature Notes）

```
作用：记录你读了什么，用自己的话
存放：参考资料区，关联源材料
要求：
  - 简短（一页以内）
  - 用你自己的话复述
  - 标注出处
  - 选择性记录
```

### 永久笔记（Permanent Notes）

```
作用：思考的结晶，Zettelkasten 的核心
存放：永久卡片盒，与现有卡片链接
要求：
  - 一条笔记 = 一个想法
  - 完整的句子（脱离上下文也能理解）
  - 链接到相关的卡片
  - 问自己"这和已有的卡片有什么关系？"
```

## 处理流程

```
阅读/思考/会议
      │
      ↓
闪念笔记（快速记下）
      │
      ↓（1-2 天内）
      │
文献笔记（选择性记录）
      │
      ↓
思考：这让我想到什么？和已有卡片有什么关系？
      │
      ↓
永久笔记（一条一个想法 + 链接）
      │
      ↓
放入卡片盒，参与对话
```

## 与"普通记笔记"的区别

| 普通笔记 | Zettelkasten |
|----------|-------------|
| 按主题/课程分类 | 按链接自然生长 |
| 记录 = 终点 | 记录 = 起点 |
| 写了就不再看 | 持续回看和链接 |
| 孤岛 | 网络 |
| 为了考试/备忘 | 为了思考和创造 |
