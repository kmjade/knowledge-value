# 03 - PARA Areas（领域）

## 定义

> 领域 = 需要长期维护、没有截止日期的责任范围

**特点**：
- 永远不"完成"，持续进行
- 有某种标准需要维护
- 是你生活中的角色和责任

## 常见领域

```
工作：
├── 产品管理
├── 团队领导
├── 客户关系
└── 职业发展

个人：
├── 健康
├── 财务
├── 人际关系
├── 学习成长
├── 家庭
└── 精神/心灵

混合：
├── 汽车保养
├── 家庭维修
└── 宠物照顾
```

## 领域 vs 项目

| | 领域 | 项目 |
|---|------|------|
| 截止日 | 无 | 有 |
| 完成状态 | 永不"完成" | 可以完成 |
| 内容 | 标准、原则、参考 | 任务、进度、产出 |
| 示例 | 健康 | 减重 5kg（截止 8/1） |

```
领域：健康
  ├── 项目：年度体检（截止 6/30）
  ├── 项目：减重 5kg（截止 8/1）
  ├── 参考资料：体检报告、饮食计划
  └── 标准：BMI < 24、每周运动 3 次
```

## 发现你的领域

问自己这些问题：

```
1. 我负责什么？（工作角色）
2. 我关心什么？（个人关注）
3. 什么事出问题我会被问责？
4. 什么持续需要我的注意力？
```

## 领域维护

```
每周回顾时检查：
□ 这个领域有没有需要发起的新项目？
□ 有没有积累的问题需要处理？
□ 参考信息是否过时？
```

## 领域数量

```
工作领域：3-5 个
个人领域：5-7 个
总计：< 12 个

太多 → 合并同类
太少 → 拆解太粗略
```
