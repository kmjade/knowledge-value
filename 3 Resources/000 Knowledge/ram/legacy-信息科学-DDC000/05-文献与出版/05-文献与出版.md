# 05-文献与出版

> 文献与出版研究学术信息如何被创造、传播、获取和保存——从论文写作到期刊出版，从同行评审到开放获取运动。

## 学术出版生命周期

```
研究 → 写作 → 投稿 → 同行评审 → 修改 → 录用 → 出版 → 传播 → 引用
  ↑______________________________________________________________↓
                          新一轮研究
```

## 文献类型

| 类型 | 特点 | 时效性 | 权威性 |
|------|------|--------|--------|
| **期刊论文** | 经过同行评审的原创研究 | 1-2 年发表周期 | 最高 |
| **会议论文** | 学术会议发表，CS 领域尤其重要 | 较快 | 高（取决于会议质量） |
| **学位论文** | 硕士/博士论文，系统深入 | — | 高 |
| **预印本 (Preprint)** | 未经同行评审的论文初稿 | 即时 | 未经评审，需谨慎引用 |
| **专著 (Monograph)** | 系统性的学术著作 | 2-5 年 | 高 |
| **技术报告** | 项目或机构的非正式报告 | 快 | 中等 |
| **灰色文献** | 不通过商业出版的文献 | 不定 | 依赖来源 |

## 同行评审 (Peer Review)

| 类型 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| **单盲** | 审稿人知道作者，作者不知道审稿人 | 审稿人有上下文 | 可能偏袒/歧视 |
| **双盲** | 双方都不知道对方 | 减少偏见 | 大牛的作品仍可被认出 |
| **开放评审** | 双方都知道，报告可能公开 | 透明、负责 | 审稿人可能不敢批评 |
| **发表后评审** | 先发表，再公开评审 | 快速、民主 | 缺乏质量控制 |

> 同行评审不是完美的——它慢、保守、有时不公平。但至今没有更好的替代方案。"同行评审只是科学质量的最低保证，不是最高标准。"

## 开放获取 (Open Access)

| 模式 | 含义 | 谁出钱 | 例子 |
|------|------|--------|------|
| **金色 OA** | 发表后立即可免费获取 | 作者付 APC (文章处理费) | PLOS, BioMed Central |
| **绿色 OA** | 在机构知识库或预印本服务器自存档 | 不收费 | arXiv |
| **青铜 OA** | 出版社免费提供但没有开放许可 | 出版社 | 某些期刊促销期 |
| **钻石/白金 OA** | 免费发表 + 免费阅读 | 机构/基金资助 | 部分学会期刊 |

### Plan S

由欧洲研究资助机构联盟 cOAlition S 推动的激进 OA 政策：从 2021 年起，公共资助的研究必须立即开放获取——不允许 embargo（延迟期）。这是学术出版史上最大的变革之一。

## 学术出版的经济学

```
研究者(免费提供内容+免费审稿)
        ↓
出版商(组织审稿、编辑、排版、发行)
        ↓
图书馆(以高昂价格买回订阅)
```

> 这个循环被称为"学术出版的 triple-pay 系统"——研究者免费提供内容、免费审稿、机构再花巨资买回来。这是开放获取运动的核心批判点。

## 文献计量标识符

| 标识符 | 用途 |
|--------|------|
| **DOI** (Digital Object Identifier) | 论文/数据集的永久链接 |
| **ISBN** | 图书的唯一标识 |
| **ISSN** | 期刊的唯一标识 |
| **ORCID** | 研究者的唯一标识（解决重名问题） |
| **ROR** | 研究机构的唯一标识 |

## 文献学 (Documentation)

文献学在中文语境中有独特含义——与西方 Documentation 不同：

| 维度 | 中国文献学 | 西方 Documentation |
|------|-----------|-------------------|
| 核心关注 | 古籍版本、校勘、目录 | 科技文献的组织和检索 |
| 代表学者 | 郑樵、章学诚 | Paul Otlet, Suzanne Briet |
| 方法 | 版本学、校勘学、目录学 | 分类法、索引、摘要 |
| 现代发展 | 中国古典文献学 | 数字人文、信息科学 |
