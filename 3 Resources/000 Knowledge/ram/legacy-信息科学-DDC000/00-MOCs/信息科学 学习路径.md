# 信息科学 学习路径

> 面向对信息、知识、图书馆和出版感兴趣的读者。2 阶段，每阶段约 2 周。

## 阶段一：核心理念（第 1-2 周）

1. [[01-Overview]] — 建立全景认知，理解 DIKW 和信息生命周期
2. [[02]] — 分类法、元数据、本体——信息的"骨架"
3. [[03]] — 搜索引擎如何工作？为什么你的搜索有时找不到东西？
4. **实践**：用分类法整理自己的 Obsidian vault 标签体系

## 阶段二：机构与应用（第 3-4 周）

5. [[04]] — 图书馆不只是放书的地方
6. [[05]] — 学术出版的全流程
7. [[06]] — 如何让信息保存 100 年？
8. [[07]] — 如何衡量知识的影响力？
9. [[08]] — 信息组织的运作
10. [[09]] — 数字化时代的人文研究
11. **实践**：用 [[信息计量学|文献计量]] 方法分析一个你感兴趣的研究领域

## 完整阅读顺序

```
信息科学 知识库总览
  └── 信息科学 学习路径 ◀ 你在这里
        ├── 01-Overview
        ├── 02-知识组织
        ├── 03-信息检索
        ├── 04-图书馆学
        ├── 05-文献与出版
        ├── 06-档案学与数字保存
        ├── 07-信息计量学
        ├── 08-信息机构与管理
        └── 09-数字人文与数据管理
```

## 选学路径

- **知识管理爱好者** → 01 → 02 → 04 → 08
  > 结合 [[PARA 知识库总览|PARA]] 和 [[GTD 知识库总览|GTD]] 效果更佳
- **学术研究者** → 01 → 03 → 05 → 07
- **信息架构/UX** → 01 → 02 → 03
- **数据管理/数字保存** → 01 → 06 → 09

> 这个学科在中国常被称为"图书情报与档案管理"（一级学科），在西方则统一在 "Library and Information Science" (LIS) 或 "iSchools" 联盟下。这是一个被严重低估但极其有用的学科。
