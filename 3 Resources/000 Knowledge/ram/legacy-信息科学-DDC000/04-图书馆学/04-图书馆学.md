# 04-图书馆学

> 图书馆学是研究图书馆的运作原理、社会功能和服务的学科。现代图书馆早已不是"藏书楼"——它是社区信息中心、数字资源门户和信息素养教育者。

## 图书馆类型

| 类型 | 服务对象 | 核心使命 |
|------|----------|----------|
| **国家图书馆** | 全国 | 保存国家出版物（呈缴本制度）、编制国家书目 |
| **公共图书馆** | 社区居民 | 免费的信息获取、阅读推广、社区活动 |
| **高校图书馆** | 师生 | 支持教学和科研、学科服务、信息素养教育 |
| **专业图书馆** | 特定机构/行业 | 深度主题服务（医学、法律、企业） |
| **中小学图书馆** | 中小学生 | 阅读兴趣培养、基础信息素养 |

## 图书馆的核心业务

### 资源建设

| 环节 | 内容 |
|------|------|
| **采访 (Acquisition)** | 选购、捐赠、交换——决定买什么 |
| **编目 (Cataloging)** | 按标准（MARC, RDA）描述资源，创建书目记录 |
| **分类 (Classification)** | 按分类法赋予索书号，确定上架位置 |

### 读者服务

| 服务 | 说明 |
|------|------|
| **参考咨询** | 帮助用户找到所需信息——从"那本书在哪"到"帮我做文献综述" |
| **流通服务** | 借还、预约、馆际互借 |
| **信息素养教育** | 教用户如何找、评、用信息——比任何具体技能都重要 |
| **学科服务** | 嵌入院系的专业图书馆员服务（高校图书馆特有） |

## 编目：让信息可发现

### FRBR 实体-关系模型

将书目世界分为三组实体：

| 组 | 实体 | 含义 |
|----|------|------|
| 第 1 组 | 作品 (Work) → 内容表达 (Expression) → 载体表现 (Manifestation) → 单件 (Item) | 知识创作的抽象层次 |
| 第 2 组 | 个人 (Person) → 团体 (Corporate Body) | 创作者 |
| 第 3 组 | 概念、物体、事件、地点 | 主题 |

```
《红楼梦》 (Work)
  ├── 曹雪芹原文 (Expression)
  ├── 英文译本 (Expression)
  └── 评注本 (Expression)
        ├── 人民文学出版社 2020 版 (Manifestation)
        │     ├── 北大图书馆藏本 (Item)
        │     └── 清华图书馆藏本 (Item)
        └── 中华书局 2019 版 (Manifestation)
```

> FRBR 是理解 RDA 编目规则和现代图书馆目录设计的核心概念。

## 数字图书馆

| 要素 | 说明 |
|------|------|
| 数字资源 | 电子书、数据库、扫描件、原生数字资料 |
| 数字资源管理 | 数字对象的存储、描述、长期保存 |
| 发现系统 | 统一搜索——跨数据库、跨格式的联邦检索 |
| 机构知识库 | 大学/机构的学术成果数字仓储（如 DSpace） |

## 信息素养 (Information Literacy)

ACRL 框架（2016）的六大核心概念：

| 概念 | 核心问题 |
|------|----------|
| **权威是建构的和情境的** | 谁说了算？在不同情境下权威来源不同 |
| **信息创建是一种过程** | 这个信息是怎么产生的？ |
| **信息具有价值** | 谁从信息中获利？知识产权、付费墙 |
| **研究即探究** | 研究是迭代的提问过程 |
| **学术即对话** | 每个研究都在回应前人的研究 |
| **搜索即策略探索** | 搜索不是一次性的，需要不断调整策略 |
