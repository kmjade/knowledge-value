## 📄 核心标准

| 标准 | 全称 | 用途 |
|------|------|------|
| **MARC** | Machine-Readable Cataloging | 图书馆书目记录格式 |
| **Dublin Core** | DC 元数据 | 最简洁的通用元数据标准 |
| **FRBR** | Functional Requirements for Bibliographic Records | 书目记录的实体-关系模型 |
| **OAIS** | Open Archival Information System | 数字保存参考模型 (ISO 14721) |
| **RDA** | Resource Description and Access | 新一代编目规则（替代 AACR2） |
| **SKOS** | Simple Knowledge Organization System | 叙词表/分类法的语义网表达 |
| **FAIR** | Findable, Accessible, Interoperable, Reusable | 科研数据管理原则 |

## 🗄️ 重要数据库

| 数据库 | 类型 | 说明 |
|--------|------|------|
| **Web of Science** | 引文数据库 | SCI/SSCI/AHCI，引文分析权威 |
| **Scopus** | 引文数据库 | Elsevier，与 WoS 竞争 |
| **CNKI (中国知网)** | 全文+引文 | 中文学术资源最全 |
| **万方/维普** | 全文+引文 | 中文学术（与知网互补） |
| **Google Scholar** | 引文搜索 | 免费，覆盖广但质量控制弱 |
| **PubMed** | 生物医学 | 免费，医学必用 |
| **arXiv** | 预印本 | 物理/CS/数学的开放获取预印本 |
