# DDC 23 与 UDC 全面对比 / DDC 23 vs UDC Comprehensive Comparison

## 基本信息對比 / Basic Information

| 维度 Dimension | DDC 23 (杜威十进分类法) | UDC (國際十進分類法 / 国际十进分类法) |
|:---|:---|:---|
| 全称 Full Name | Dewey Decimal Classification | Universal Decimal Classification |
| 創建 Creation | 1876, Melvil Dewey (美国) | 1905, Paul Otlet & Henri La Fontaine (比利时) |
| 最新版 Latest | 23rd ed., 2011 (印刷) + WebDewey (在线) | UDC MRF (Master Reference File, 年度更新) |
| 维护者 Maintainer | OCLC (美国) | UDC Consortium (荷兰海牙) |
| 卷数 Volumes | 4 卷 / 4 volumes | 线上为主 / Primarily online |
| 页数 Pages | 4,276 页 | 60,000+ 类号 / classes |
| 语言 Languages | 30+ 种 | 57 种 / 57 languages |
| 使用国 Countries | 135+ 国家 | 30+ 国家 (欧洲、拉美、非洲为主) |
| 网址 Website | oclc.org/dewey | udcc.org |

## 結構對比 / Structure Comparison

| 特征 Feature | DDC 23 | UDC |
|:---|:---|:---|
| 大类数 Main Classes | 10 | 10 (無第4類 / 无第4类) |
| 标记法 Notation | 純數字 / Pure numbers | 數字+符號 / Numbers+symbols |
| 最小位数 Minimum digits | 3 位 | 無限制 / Unlimited |
| 小数点 Decimal point | 第3位後 / After 3rd digit | 可任意位置 / Any position |
| 組合能力 Combination | 弱——單一類號 / Weak—single number | 強——符號組合 / Strong—symbol combination |
| 复分表 Auxiliary Tables | 6 张 (Table 1-6) | 辅助符号体系 + 专用复分 |
| 索引 Index | Relative Index (Vol 4) | 无独立索引 / No separate index |

## 設計哲學對比 / Design Philosophy

| 维度 Dimension | DDC 23 | UDC |
|:---|:---|:---|
| 核心目标 Core goal | 简单——任何图书馆员都能用 | 精确——能描述文献的所有维度 |
| 设计为 Designed for | 排架 / Shelf arrangement | 文献检索 / Document retrieval |
| 思维方式 Thinking | **一本书 → 一个号码** / One book → one number | **一本书 → 一个表达式** / One book → one expression |
| 适合 Suitable for | 公共图书馆 / Public libraries | 专业图书馆 / Special libraries |
| 学习曲线 Learning curve | 低 / Low | 高 / High |
| 数字时代适配 Digital age | WebDewey 月度更新 | UDC MRF 年度更新 + Linked Data |

## 優缺點對比 / Pros & Cons

| | DDC 23 | UDC |
|:---|:---|:---|
| ✅ 优势 Pros | 全球最广泛、简单易学、OCLC 强力维护 | 表达力最强、多语言、Linked Data 原生、符号组合 |
| ❌ 劣势 Cons | 西方中心论、不能组合主题、单一类号限制 | 过于复杂、学习曲线陡峭、维护频率较低、第4类空缺 |
| 🔧 数字应对 | WebDewey (按月更新) | UDC Summary (免费), UDC MRF (年度) |
