# 03-标记法与复分表

> DDC 的魔力在小数点之后——通过标准化的复分表，任何主题都可以用统一的规则深入细分。

## 标记法规则

| 规则 | 说明 | 例子 |
|------|------|------|
| 最小三位 | 类号至少 3 位——不足补零 | 5 → 500（科学） |
| 小数点 | 第三位后加小数点 | 636.7（狗） |
| 分层 | 每多一位=细分 10 倍 | 6→63→636→636.7→636.75 |

## 标准复分表 (Table 1)

九个通用复分——可附加到任何类号后：

| 复分 | 含义 | 例子 |
|------|------|------|
| **-01** | 哲学与理论 | 500.1 = 科学的哲学 |
| **-03** | 词典、百科全书 | 503 = 科学词典 |
| **-05** | 期刊 | 505 = 科学期刊 |
| **-07** | 教育与研究 | 507 = 科学教育 |
| **-09** | 历史 | 509 = 科学史 |

## 文学复分 (Table 3)

用于 800 文学类的细分：

```
823.912 = 英国小说, 20世纪早期
 800 文学
  820 英国文学
   823 英国小说
    .9 20世纪
      .91 1900-1945 → .912
```

## 助记符 (Mnemonics)

| 数字 | 含义 | 贯穿各处的使用 |
|------|------|---------------|
| 1 | 哲学/理论 | 101 哲学理论、301 社会学理论 |
| 2 | 杂项/纲要 | 102 哲学杂集、502 科学杂集 |
| 3 | 词典 | 103 哲学词典、503 科学词典 |
| 5 | 期刊 | 105 哲学期刊、605 技术期刊 |
| 7 | 教育 | 107 哲学教育、507 科学教育 |
| 9 | 历史 | 109 哲学史、509 科学史 |

> 这就是 DDC 的优雅之处——一旦学会了这套语言，看到任何类号都能大致猜出它的含义。
