# 06-分类实践

> 给一本书分配 DDC 类号的实战方法。

## 分类步骤

```
1. 确定主题 —— 这本书到底讲什么？
2. 找到大类 —— 000-900 哪个？
3. 细分 —— 找到最精确的类号
4. 加复分 —— 需要加形式/地区复分吗？
5. 检查 —— 类号是否唯一且有意义？
```

## 实例

### 《中国的人工智能发展史》

```
1. 主题: 人工智能 (计算机科学)
2. 大类: 000
3. 细分: 006.3 (人工智能)
4. 复分: + 09 (历史) + 51 (中国)
5. 类号: 006.30951
```

### 《德国哲学词典》

```
1. 主题: 哲学
2. 大类: 100
3. 细分: 103 (词典)
4. 复分: + 43 (德国)
5. 类号: 103.43
```

## 常见陷阱

| 陷阱 | 正确做法 |
|------|----------|
| 按书名分类 | 按内容——"苹果"可能是水果(634)或计算机(004) |
| 分得太粗 | 尽可能具体——636 不如 636.7 |
| 忽略复分 | 历史/词典/地区复分让类号更精确 |
| 跨学科犹豫 | 放在主要学科——交叉学科放在最强调的方面 |
