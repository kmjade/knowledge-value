# 01-DDC 概述

> DDC (Dewey Decimal Classification) 由 Melvil Dewey 于 1876 年创建——将人类全部知识分为 10 个大类，每类再细分为 10 个，以此类推，无限细分。

## Melvil Dewey (1851-1931)

| 事实 | 说明 |
|------|------|
| 21 岁 | 在 Amherst 学院图书馆工作时创建 DDC |
| 1876 | 首次出版——仅 42 页 |
| 贡献 | 分类法、图书馆事业专业化、美国图书馆协会 (ALA) 创始人 |
| 争议 | 性别歧视、种族歧视——ALA 最高奖项已移除其名字 |

## DDC 的核心原理

### 十进分层

```
600 技术（应用科学）
 630 农业
   636 动物饲养
     636.7 狗
       636.75 猎犬
         636.753 腊肠犬
```

> 每一层都是上一层的 1/10。理论上可以无限细分——小数点后可以任意延长。

### 三要素

| 要素 | 说明 | 例子 |
|------|------|------|
| **类号** | 数字表示的主题 | 636.7 = 狗 |
| **索引** | 主题→类号的查找表 | "狗"→636.7 |
| **复分表** | 标准附加规则 | -03 = 词典 |

## DDC 十大类概览

| 类 | 内容 | 关键细分 |
|----|------|----------|
| **000** | 计算机、信息、总论 | 004 计算机、020 图书馆学、030 百科全书 |
| **100** | 哲学、心理学 | 150 心理学、170 伦理学 |
| **200** | 宗教 | 220 圣经、280 基督教 |
| **300** | 社会科学 | 330 经济、340 法律、370 教育 |
| **400** | 语言 | 420 英语、490 其他语言 |
| **500** | 自然科学 | 510 数学、530 物理、570 生物 |
| **600** | 技术（应用科学） | 610 医学、620 工程、640 家政 |
| **700** | 艺术与娱乐 | 720 建筑、750 绘画、780 音乐 |
| **800** | 文学 | 810 美国文学、820 英国文学 |
| **900** | 历史与地理 | 910 地理、930 古代史、940 欧洲史 |
