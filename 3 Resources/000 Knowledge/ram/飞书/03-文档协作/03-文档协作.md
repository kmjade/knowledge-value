# 03 - 飞书文档协作

## 文档类型

| 类型 | 说明 | 对标 |
|------|------|------|
| **飞书文档** | 富文本文档 | Notion / Google Docs |
| **飞书表格** | 电子表格 | Google Sheets / Excel |
| **思维笔记** | 思维导图 | XMind |
| **多维表格** | 在线数据库 | Airtable |
| **知识库** | Wiki 系统 | Confluence / Notion Wiki |
| **画板** | 白板 | Miro |
| **幻灯片** | PPT | PowerPoint |

## 飞书文档

### 块类型

```
/ 呼出命令菜单
/h1 → 大标题
/h2 → 中标题  
/h3 → 小标题
/高亮块 → 彩色卡片
/分栏 → 多栏布局
/图片 → 插入图片
/文件 → 插入附件
/引用 → 引用块
/多维表格 → 内嵌数据库
```

### 协作功能

```
选中文字 → 评论（Ctrl+Alt+M）
右上角 → 分享 → 设置权限
右上角 → 历史版本 → 回溯
@某人 → 提及通知
```

### 权限控制

| 权限 | 说明 |
|------|------|
| 可管理 | 完全控制 |
| 可编辑 | 编辑 + 评论 |
| 可评论 | 仅评论 |
| 可阅读 | 只读 |

## 飞书表格

```
新建 → 飞书表格
```

**核心功能**：
- 公式（类似 Excel）
- 数据透视表
- 条件格式
- 数据验证
- 保护工作表
- 评论与协作

**常用公式**：

```excel
=IF(A1>100, "超预算", "正常")
=VLOOKUP(A1, B:C, 2, FALSE)
=SUMIF(B:B, "已完成", C:C)
=CONCATENATE("项目-", A1)
```

## 知识库

```
云文档 → 知识库 → 新建知识库
```

**适合场景**：团队 Wiki、产品文档、新人手册

**结构**：
- 知识库名称
- 目录导航
- 文档页面（树状结构）

## 思维笔记

```
新建 → 思维笔记
```

- 大纲式 / 思维导图双模式
- 支持一键切换视图
- 可嵌入文档中
