# 01 - 飞书平台概览

## 注册与创建团队

```
飞书官网 https://www.feishu.cn/ → 注册 → 创建团队 → 邀请成员
```

**Lark（国际版）**：https://www.larksuite.com/

## 下载客户端

| 平台 | 下载 |
|------|------|
| Windows | 飞书官网 → 下载 |
| macOS | 飞书官网 或 Mac App Store |
| Linux | 飞书官网 → 下载 deb/rpm |
| iOS | App Store → 飞书 |
| Android | 应用商店 → 飞书 |

> Web 版也可用：https://www.feishu.cn/ 直接登录

## 核心功能一览

| 模块 | 功能 | 对标 |
|------|------|------|
| **消息** | 即时通讯 | Slack / 微信 |
| **日历** | 日程管理 | Google Calendar |
| **云文档** | 在线文档/表格 | Notion / Google Docs |
| **多维表格** | 在线数据库 | Airtable |
| **会议** | 视频会议 | Zoom / Teams |
| **审批** | 审批流程 | 钉钉审批 |
| **工作台** | 应用入口 | — |
| **开放平台** | 机器人/API | Slack Bot |

## 核心快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` | 快速搜索（聊天/文档/联系人） |
| `Ctrl+Shift+K` | 全局搜索 |
| `Ctrl+Shift+N` | 新建文档 |
| `Ctrl+Shift+2` | 快速截图 |
| `Alt+Shift+D` | 先 Do Not Disturb |
| `Ctrl+/` | 快捷键帮助 |
