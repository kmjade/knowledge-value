# 04 - 飞书多维表格（Bitable）

> 飞书多维表格是类 Airtable 的在线数据库，融合电子表格的灵活与数据库的强大。

## 基础概念

```
多维表格 = 表（Table）+ 视图（View）

表 → 包含字段（列）和记录（行）
视图 → 同一张表的不同视角（表格/看板/日历/画廊/甘特）
```

## 字段类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 文本 | 单行/多行文本 | 姓名 |
| 数字 | 数值（支持货币/百分比） | 金额 |
| 选择 | 单选下拉 | 状态 |
| 多选 | 多个标签 | 标签 |
| 日期 | 日期时间 | 截止日 |
| 复选框 | 是/否 | 是否完成 |
| 人员 | 选择成员 | 负责人 |
| 附件 | 文件/图片上传 | 相关文件 |
| 链接 | URL + 显示文字 | 参考链接 |
| 公式 | 计算字段 | 天数差 |
| 关联 | 关联到其他表 | 项目↔任务 |
| 汇总 | 从关联表汇总 | 任务数 |
| 自动编号 | 自增 ID | ID-001 |
| 创建时间 | 自动时间戳 | — |
| 修改时间 | 自动更新时间 | — |
| 创建人/修改人 | 自动记录 | — |

## 六种视图

| 视图 | 适用场景 |
|------|----------|
| **表格** | 数据一览，批量编辑 |
| **看板** | 按状态分组（项目管理） |
| **日历** | 按日期展示（日程管理） |
| **画廊** | 卡片展示（图片/员工名录） |
| **甘特** | 时间线（项目排期） |
| **表单** | 收集数据 |

## 公式

```
[字段名]      → 引用字段
IF()          → 条件判断
AND()/OR()    → 逻辑运算
SUM()/AVG()   → 聚合函数
DATEIF()      → 日期差
CONCAT()      → 字符串拼接
```

**常用示例**：

```
# 状态图标
IF({状态}="完成", "✅", IF({状态}="进行中", "🔄", "⏳"))

# 逾期天数
DATEIF({截止日}, TODAY(), "D")

# 完成率
ROUND(COUNTIF({关联任务}.{状态}, "完成") / COUNTA({关联任务}) * 100, 0) & "%"
```

## 关联与汇总

```
创建「关联」字段 → 指向另一个表
创建「汇总」字段 → 从关联表统计
```

**实战：项目管理**

```
项目表 ──关联──→ 任务表
任务表 ←──汇总── 完成率、总数

项目表（看板视图）
├── 待开始
├── 进行中
│   ├── 任务A ○
│   └── 任务B ●
└── 已完成
```

## 数据同步

```
多维表格 ↔ 飞书表格：数据可双向同步
多维表格 ↔ API：通过开放平台读写
```

## 免费版限制

- 每表最多 1000 条记录
- 商业/企业版不限
