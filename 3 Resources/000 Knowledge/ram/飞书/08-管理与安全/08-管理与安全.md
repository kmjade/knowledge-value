# 08 - 飞书管理与安全

## 管理后台

```
飞书桌面端 → 头像 → 管理后台
或 https://www.feishu.cn/admin
```

## 组织架构

```
管理后台 → 组织架构 → 通讯录
├── 部门管理
├── 成员管理
├── 职位/职级
└── 批量导入
```

## 权限管理

### 管理员角色

| 角色 | 权限 |
|------|------|
| **超级管理员** | 全部权限（最多 3 人） |
| **分级管理员** | 指定模块权限 |
| **应用管理员** | 管理特定应用 |

### 成员权限

```
安全设置 → 权限管理
├── 谁能创建群组
├── 谁能添加外部联系人
├── 谁能使用 API
└── 谁能导出数据
```

## 安全设置

| 设置 | 说明 | 建议 |
|------|------|------|
| 登录 IP 限制 | 限制登录 IP | 开启（企业版） |
| 水印 | 文档/聊天水印 | 开启 |
| 截屏限制 | 禁止移动端截屏 | 按需开启 |
| 外部联系人 | 控制外部沟通 | 按需限制 |
| 离职交接 | 自动交接文档/审批 | 开启 |
| 数据导出 | 导出审计 | 按需关闭 |

## 数据合规

```
飞书（国内版）→ 数据存储在中国
Lark（国际版）→ 数据存储在海外（新加坡/美国）

SOC2 / ISO 27001 / GDPR 合规
```

## 企业设置

| 设置项 | 路径 |
|--------|------|
| 企业信息 | 管理后台 → 企业设置 |
| 域名验证 | 管理后台 → 企业设置 → 域名 |
| SSO 单点登录 | 管理后台 → 安全 → SSO |
| 审计日志 | 管理后台 → 安全 → 审计日志 |
| 数据归档 | 管理后台 → 安全 → 数据归档 |

## 常见管理操作

```text
✅ 批量导入成员：通讯录 → 导入 → 上传 Excel
✅ 离职处理：成员管理 → 选择 → 操作 → 停用账号
✅ 重置密码：成员 → 操作 → 重置密码
✅ 群组管理：群组 → 查看所有群 → 管理
✅ 应用管理：应用 → 应用管理 → 启用/停用
```
