# 05 - 飞书审批流程

## 审批入口

```
工作台 → 审批 → 发起审批 / 管理后台
```

> 仅商业版/企业版支持自定义审批，免费版无此功能。

## 内置审批模板

| 模板 | 适用 |
|------|------|
| 请假 | 年假、病假、事假 |
| 报销 | 费用报销 |
| 加班 | 加班申请 |
| 出差 | 出差申请 |
| 采购 | 采购审批 |
| 用章 | 公章使用 |
| 合同 | 合同审批 |

## 自定义审批

### 创建步骤

```
管理后台 → 审批 → 创建审批 → 
  1. 基础信息：名称、图标、说明
  2. 表单设计：拖拽控件
  3. 流程设计：审批节点
  4. 高级设置：权限、通知
```

### 表单控件

| 控件 | 说明 |
|------|------|
| 单行文本 | 短文字输入 |
| 多行文本 | 长文字输入 |
| 数字 | 金额/数量 |
| 日期 | 日期选择 |
| 选择 | 单选/多选 |
| 人员 | 选择成员 |
| 部门 | 选择部门 |
| 附件 | 文件上传 |
| 明细 | 子表格（报销清单等） |
| 计算公式 | 自动计算 |

### 审批流程

```
发起人
  → 直属上级（自动）
    → 部门负责人
      → 人事审批
        → 结束（自动通知）
```

**节点类型**：
- 审批节点：指定审批人
- 抄送节点：通知但不审批
- 条件分支：按条件走不同流程
- 并行审批：多人同时审批

## 审批管理

```
工作台 → 审批 → 我的审批
├── 待审批
├── 已审批
├── 我发起的
└── 抄送我的
```

## 高级功能

| 功能 | 说明 |
|------|------|
| **条件分支** | 金额>5000 走经理，<5000 走组长 |
| **多人会签** | 需多人全部同意 |
| **或签** | 任一人同意即可 |
| **加签** | 审批中可加人 |
| **转交** | 转给他人审批 |
| **自动驳回** | 超时自动驳回 |
