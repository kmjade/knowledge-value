# 02 - 飞书即时通讯

## 消息类型

| 类型 | 说明 |
|------|------|
| 单聊 | 一对一私聊 |
| 群聊 | 最多 5000 人 |
| 话题群 | 消息按话题分组（类似 Slack Thread） |
| 频道 | 基于主题的讨论空间 |

## 群组管理

### 创建群组

```
联系人 → 创建群组 → 搜索成员 → 确定
```

### 群组设置

```
群设置 → 群管理
├── 群名称/头像
├── 入群方式（邀请/链接/审批）
├── 群公告（置顶重要信息）
├── 群机器人（添加 Bot）
└── 群文件（共享文件）
```

### 话题群 vs 普通群

| | 话题群 | 普通群 |
|---|--------|--------|
| 消息组织 | 按话题分组 | 按时间线 |
| 避免刷屏 | ✅ | ❌ |
| 查找历史 | 清晰 | 混乱 |
| 适合场景 | 团队讨论、项目沟通 | 通知群、闲聊 |

## 频道（Channels）

```
消息 → 频道 → 创建频道
```

**频道类型**：
- 公开频道：团队所有人可加入
- 私有频道：仅受邀成员可见

**适合场景**：跨部门话题、兴趣小组、公告频道

## 高效沟通技巧

```
@某人 → 定向提醒
@所有人 → 全员通知（谨慎使用）
@日期时间 → 快速创建日程

# 话题群回复
鼠标悬停消息 → 回复 → 创建新话题
```

## 消息操作

| 操作 | 方式 |
|------|------|
| 引用回复 | 右键消息 → 引用回复 |
| 表情回复 | 鼠标悬停 → 表情 |
| 翻译 | 右键 → 翻译 |
| 收藏 | 右键 → 收藏 |
| 待办 | 右键 → 标记为待办 |
| 转发 | 右键 → 转发 |

## 快捷入口

```
侧边栏顶部搜索框 → 搜索联系人/群组/消息
Ctrl+K → 快速跳转
```
