# 06 - 飞书自动化

## 自动化入口

```
多维表格 → 右上角「自动化」按钮
飞书后台 → 工作台 → 自动化
```

## 触发条件（Trigger）

| 触发类型 | 示例 |
|----------|------|
| **记录变更** | 新增记录、字段被修改 |
| **定时触发** | 每天 9:00、每周一 |
| **表单提交** | 有人提交表单后 |
| **审批通过** | 审批流程结束后 |
| **Webhook** | 外部系统推送 |

## 执行动作（Action）

| 动作类型 | 示例 |
|----------|------|
| **修改记录** | 自动更新字段值 |
| **发送消息** | 飞书消息、群通知 |
| **创建记录** | 新建一条记录 |
| **发送 HTTP 请求** | 调用外部 API |
| **发送邮件** | 邮件通知 |
| **执行代码** | 自定义脚本 |

## 常用自动化场景

### 场景 1：任务自动通知

```
触发：任务表新增记录
动作：发送飞书消息 → 通知负责人 "你有新任务：[任务名]"
```

### 场景 2：状态变更提醒

```
触发：项目表「状态」字段变更为「完成」
动作：发送群消息 → 通知全员 "恭喜！[项目名] 已完成 🎉"
```

### 场景 3：定时日报

```
触发：每天 9:00
动作：查询当天截止的任务 → 发送消息 "今日待办：3 项任务"
```

### 场景 4：审批自动归档

```
触发：审批通过
动作：在多维表格中创建一条审批记录
```

## 自动化正则与条件

```
仅当满足条件时触发：
  状态 = "待处理"
  AND 优先级 = "高"
  AND 截止日 <= TODAY()
```

## 执行日志

```
自动化设置 → 运行日志 → 查看每次执行结果
```

可追踪：触发时间、是否成功、失败原因
