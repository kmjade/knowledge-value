# 07 - 飞书机器人开发

## 飞书开放平台

```
https://open.feishu.cn/
```

### 创建应用

```
开发者后台 → 创建企业自建应用 → 
  1. 填写应用名称/描述
  2. 配置应用权限
  3. 发布版本 → 管理员审核
```

## 机器人类型

| 类型 | 说明 | 创建方式 |
|------|------|----------|
| **自定义 Bot** | 群 Webhook 机器人 | 群设置 → 添加机器人 |
| **企业自建应用** | 内部应用 | 开放平台创建 |
| **应用商店应用** | 对外发布 | 开放平台 → 发布 |

## Webhook 机器人

### 添加

```
群设置 → 群机器人 → 添加机器人 → 自定义机器人 → 复制 Webhook URL
```

### 发送消息

```python
import requests
import json

webhook = "https://open.feishu.cn/open-apis/bot/v2/hook/xxx"

# 文本消息
data = {
    "msg_type": "text",
    "content": {
        "text": "新任务提醒：请查看项目管理表"
    }
}

# 富文本消息（卡片）
data = {
    "msg_type": "interactive",
    "card": {
        "header": {
            "title": {"tag": "plain_text", "content": "新任务分配"}
        },
        "elements": [
            {"tag": "div", "text": {"tag": "plain_text", "content": "你有新任务"}},
            {
                "tag": "action",
                "actions": [{
                    "tag": "button",
                    "text": {"tag": "plain_text", "content": "查看详情"},
                    "url": "https://example.com/task/123"
                }]
            }
        ]
    }
}

requests.post(webhook, json=data)
```

## 消息卡片设计

### 卡片组件

| 组件 | 说明 |
|------|------|
| `header` | 标题栏（颜色自定义） |
| `div` | 文本块 |
| `hr` | 分割线 |
| `image` | 图片 |
| `note` | 脚注 |
| `action` | 按钮组 |
| `column_set` | 多列布局 |

### 卡片模板

```json
{
  "msg_type": "interactive",
  "card": {
    "header": {
      "title": {"content": "📋 任务提醒", "tag": "plain_text"},
      "template": "blue"
    },
    "elements": [
      {"tag": "div", "text": {"content": "**任务：** 完成 Q2 报告", "tag": "lark_md"}},
      {"tag": "div", "text": {"content": "**截止：** 2026-06-30", "tag": "lark_md"}},
      {"tag": "hr"},
      {
        "tag": "action",
        "actions": [
          {"tag": "button", "text": {"content": "确认", "tag": "plain_text"}, "type": "primary"},
          {"tag": "button", "text": {"content": "推迟", "tag": "plain_text"}, "type": "danger"}
        ]
      }
    ]
  }
}
```

## API 认证

```
获取 tenant_access_token：
POST https://open.feishu.cn/open-apis/auth/v3/tenant_access_token/internal
Body: {"app_id": "xxx", "app_secret": "xxx"}
→ 得到 token，有效期 2 小时
```

## 常用 API

```python
# 发送消息
POST /open-apis/im/v1/messages?receive_id_type=open_id

# 读取多维表格
GET /open-apis/bitable/v1/apps/{app_token}/tables/{table_id}/records

# 获取用户信息
GET /open-apis/contact/v3/users/{user_id}
```
