# 08-分类法批判

> 任何分类法都反映了创建时代的世界观——DDC 是 19 世纪白人男性的知识地图。批判它不是否定它，而是理解它的局限。

## 主要批判

### 西方中心论

| 问题 | 例子 |
|------|------|
| 宗教失衡 | 200-280 全是基督教 (~80%)，其他宗教挤在 290 |
| 语言倾斜 | 420-490 中 420-480 是欧洲语言，中文在 495.1 的小数点后 |
| 文学倾斜 | 810-880 是欧美文学——整个亚洲文学在 890 |

### 偏见嵌入

| 领域 | 批判 |
|------|------|
| 性别 | LGBT 长期被归类为"异常"(616.85)而非正常人性 |
| 种族 | 原住民文化被归入"原始"(572) |
| 殖民 | "殖民地"持续到 20 世纪后期 |

### 更新滞后

| 问题 | 例子 |
|------|------|
| 新学科 | 计算机科学花了数十年才找到合适位置 |
| 社会变革 | 同性婚姻合法化后 DDC 仍多年不变 |

## 修正努力

| 行动 | 说明 |
|------|------|
| 22nd→23rd | 大量重新归类——LGBT 从 616→306 |
| 多样性编委会 | 增加非西方、非白人成员 |
| 社区反馈 | 开放用户建议渠道 |
| 可选类号 | 允许图书馆使用替代类号 |

## 根本性的问题

> "完美的分类法存在吗？"——不存在。所有分类法都是特定时代的产物，携带了那个时代的偏见。关键不是消除偏见（不可能），而是持续审视和修正。
