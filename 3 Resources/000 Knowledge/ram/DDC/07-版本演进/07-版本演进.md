# 07-DDC 版本演进与 DDC 23

> 从 1876 年的 42 页小册子到 DDC 23 的四卷巨著——DDC 的版本史是知识增长史的镜子。DDC 23 (2011) 是截至 2026 年最新的完整印刷版。

## 关键版本

| 版本 | 年份 | 页数 | 里程碑 |
|------|------|------|--------|
| 1st | 1876 | 42 | 首创——Amherst 学院图书馆 |
| 2nd | 1885 | 314 | 引入小数点 |
| 12th | 1927 | 1,243 | 首次突破千页 |
| 15th | 1951 | 1,961 | 战后大修订 |
| 19th | 1979 | 3,738 | 三卷 |
| 20th | 1989 | 3,388 | 计算机辅助编辑 |
| 22nd | 2003 | 4,000+ | 四卷，大幅修订 |
| **23rd** | **2011** | **4,276** | **最新印刷版** |
| WebDewey | 持续 | 在线 | 每月更新 |

# DDC 23 详解

## 四卷结构

| 卷 | 内容 | 页数 |
|----|------|------|
| **Vol 1** | 导言、手册、Table 1-6（复分表） | ~900 |
| **Vol 2** | 类表：000-599 | ~1,200 |
| **Vol 3** | 类表：600-999 | ~1,200 |
| **Vol 4** | 索引 (Relative Index) | ~900 |

## DDC 23 相比 DDC 22 的主要变化

### 重大修订

| 领域 | 变化 |
|------|------|
| **004-006 计算机** | 彻底重组——反映云计算、移动计算、社交媒体的兴起 |
| **200 宗教** | 减少基督教偏见——290 其他宗教大幅扩展 |
| **340 法律** | 国际法和比较法大幅更新 |
| **370 教育** | 特殊教育和多元文化教育扩展 |
| **610 医学** | 新增补充医学、中医学分类 |

### 社会敏感修订

| 议题 | DDC 22 | DDC 23 |
|------|--------|--------|
| 同性恋 | 归类于"性障碍" | → 移至 306.76（性取向与社会） |
| 跨性别 | 未明确 | → 新增 306.768 |
| 气候变化 | 551.6 (气象) | → 新增环境视角 363.738 |
| 伊斯兰教 | 297 简短 | → 297 大幅扩展 |

## OCLC 的持续维护

| 机制 | 说明 |
|------|------|
| Editorial Policy Committee | 国际编委会——10 人，来自 5 大洲 |
| 定期更新 | WebDewey 每月更新（比等 7 年出新版快） |
| 用户建议 | 任何图书馆员可提交新类号建议 |
| 翻译 | 30+ 种语言——中文版由国家图书馆编译 |
| DDC 24? | OCLC 未宣布——WebDewey 持续更新可能替代印刷新版 |
