# 05-DDC 在数字时代

> 一个 1876 年的分类系统如何在 Internet 时代生存？答案是持续进化和拥抱关联数据。

## WebDewey

OCLC 的在线平台——替代印刷版：

| 功能 | 说明 |
|------|------|
| 实时更新 | 不再等 7 年出印刷新版 |
| 搜索 | 关键词搜索类号和标题 |
| 映射 | DDC ↔ LCSH (国会图书馆主题词) |
| 用户注释 | 图书馆员添加本地实践 |

## dewey.info——关联数据

DDC 作为 Linked Data 发布：

```
http://dewey.info/class/636.7/
  → 返回 RDF 数据: "这是关于狗的类号"
  → 多语言标签: en=Dog, zh=狗, fr=Chien
  → 关联到: LCSH, VIAF, Wikidata
```

## DDC 在分类之外

| 应用 | 说明 |
|------|------|
| 书架浏览 | 按 DDC 排架→相似主题的书籍在一起 |
| 发现系统 | 用 DDC 做分面导航 |
| 数字图书馆 | Europeana、WorldCat 使用 DDC |
| 数据分析 | 分析图书馆藏书的学科分布 |

## 挑战

| 挑战 | 现状 |
|------|------|
| Amazon/Google 替代 | 用户不再从分类号找书 |
| 跨学科主题 | 气候变化在 363 还是 551？ |
| 数字资源 | 电子书不需要"排架" |
| AI 分类 | 机器学习能否自动分类？ |
