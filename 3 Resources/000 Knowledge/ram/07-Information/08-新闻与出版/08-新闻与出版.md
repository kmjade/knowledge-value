## 新闻业态变化

| 传统 | 2020s |
|------|-------|
| 报纸印刷 | 数字优先 |
| 广告收入 | 订阅+广告 |
| 专业记者 | 公民记者+AI |
| 编辑把关 | 算法推荐 |

## 出版业价值链

```
作者 → 代理商 → 出版社 → 印刷/数字 → 发行 → 书店/平台 → 读者
        ↑ 编辑、设计、营销、版权管理 ↑
```
