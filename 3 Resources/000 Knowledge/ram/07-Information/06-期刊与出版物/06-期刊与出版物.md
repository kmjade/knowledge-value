# 06-期刊与出版物 (DDC 050)

> DDC 050 是期刊、杂志和连续出版物——定期出版的、无限延续的出版物。

## 期刊特征

| 特征 | 说明 |
|------|------|
| 连续性 | 无限期出版——不像书有"完成" |
| 周期性 | 周刊/月刊/季刊/年刊 |
| 编号 | 卷(Volume) + 期(Issue) |
| ISSN | 期刊唯一标识——8 位数字 |

## ISSN

```
ISSN 1234-5678
     1234 = 期刊代码
     5678 = 校验位
```

与 ISBN 的区别：ISSN 标识的是"期刊"而非"期刊中的某一篇"。

## 数字时代的期刊

| 变化 | 说明 |
|------|------|
| 电子期刊 | PDF/HTML 替代印刷 |
| 开放获取 (OA) | 免费阅读——PLOS, eLife |
| 预印本 | arXiv——绕过期刊直接发布 |
| 连续出版 | 不再"期"——接收一篇发一篇 |
