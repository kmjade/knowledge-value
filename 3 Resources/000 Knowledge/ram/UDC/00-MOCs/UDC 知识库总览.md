# UDC 知识库总览

> UDC (Universal Decimal Classification) 是 Paul Otlet 和 Henri La Fontaine 于 1905 年基于 DDC 创建的分类系统。它的独特之处在于**辅助符号**——可以用 "+/:=()" 组合任意主题，表达力远超 DDC。

## UDC vs DDC

| 维度 | DDC | UDC |
|------|-----|-----|
| 创建 | 1876, Dewey | 1905, Otlet / La Fontaine |
| 基础 | — | 基于 DDC 扩展 |
| 标记 | 纯数字 | 数字 + 辅助符号 |
| 组合 | 单一类号 | 类号可组合——"+"连接两个主题 |
| 关系 | 隐含 | 显式——":"表示关系 |
| 适用 | 公共/学校图书馆 | 专业/研究图书馆、文献数据库 |
| 使用 | 全球 135+ 国家 | 欧洲、拉美、非洲 30+ 国家 |
