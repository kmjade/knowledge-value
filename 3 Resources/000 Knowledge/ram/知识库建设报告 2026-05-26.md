## 设计原则

1. **统一结构** — 每个知识库：MOC + 分章节 + 资源收集 + README
2. **学习路径** — 每个知识库配有 4 周学习计划
3. **双向链接** — `[[wikilink]]` 跨知识库互联
4. **工具实践** — 方法论知识库提供 Obsidian/Notion/飞书 三种方案
5. **代码示例** — 含可运行的代码（MQL、Python、Templater）
