# Classification Systems体系

> 分类法是最古老也是最基础的知识系统——将知识划分为层级结构，让相似主题的书籍和信息聚集在一起。

## 四大分类法

| 分类法 | 创建 | 使用 | 特征 |
|--------|------|------|------|
| **DDC** | 1876 | 135+ 国 | 纯数字、10 大类 |
| **UDC** | 1905 | 30+ 国 | 数字+符号、可组合 |
| **CLC (中图法)** | 1975 | 中国 | 字母+数字、22 大类 |
| **LCC** | 1897 | 美国研究型 | 字母+数字、灵活 |

## 分类法的三个功能

| 功能 | 说明 |
|------|------|
| 排架 | 让同一主题的书在一起——浏览发现 |
| 导航 | 图书馆目录的骨架 |
| 统计 | 馆藏学科分布分析 |

## 分类的认知基础

| 概念 | 分类法体现 |
|------|-----------|
| 层级 | 从一般到特殊——600→610→616 |
| 互斥 | 每本书只能在一个位置 |
| 穷举 | 所有知识都应有位置 |
| 共识 | 学科划分反映时代共识 |
