# Knowledge Management

> 知识管理是组织层面的"知识系统"——如何在企业/机构中捕获、组织、共享和应用知识。

## SECI 模型

| 模式 | 转化 | 例子 |
|------|------|------|
| **S**ocialization | 隐性→隐性 | 师徒制、观摩 |
| **E**xternalization | 隐性→显性 | 把经验写成手册 |
| **C**ombination | 显性→显性 | 整合多个报告 |
| **I**nternalization | 显性→隐性 | 读完手册后会做 |

## KM 工具

| 工具 | 功能 |
|------|------|
| Wiki/Confluence | 组织的集体显性知识 |
| 实践社区 CoP | 同行的非正式交流 |
| 事后回顾 AAR | 项目结束的经验沉淀 |
| 专家黄页 | "谁知道什么" |
