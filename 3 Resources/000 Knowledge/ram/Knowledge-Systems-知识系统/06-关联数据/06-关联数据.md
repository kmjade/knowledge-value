# Linked Data与语义网

> Tim Berners-Lee 的愿景：Web 不仅是文档的互联，更是数据的互联。关联数据是语义网的实现路径。

## 关联数据四原则

1. **用 URI 命名事物**——每本书、每个人、每个地点有唯一 URL
2. **使用 HTTP URI**——通过 Web 协议访问
3. **提供有用信息**——用标准格式 (RDF) 返回数据
4. **包含链接**——指向其他 URI，形成数据网络

## 五星开放数据

| 星 | 条件 |
|----|------|
| ★ | 数据在网上 (任何格式) |
| ★★ | 结构化数据 (Excel 而非图片) |
| ★★★ | 开放格式 (CSV 而非 Excel) |
| ★★★★ | **使用 URI 标识事物** (RDF) |
| ★★★★★ | **链接到其他数据** (Linked Data) |

## SPARQL——语义网的 SQL

```sparql
# Wikidata: 曹雪芹写了什么？
SELECT ?work WHERE {
  wd:Q10581 wdt:P800 ?work.
}
```
