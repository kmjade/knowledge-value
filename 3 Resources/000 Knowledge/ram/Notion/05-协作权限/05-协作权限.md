# 05 - Notion 协作与权限

## 分享页面

```
页面右上角 Share → 选择分享方式
```

### 三种分享方式

| 方式 | 权限 | 适用 |
|------|------|------|
| **邀请成员** | 完全访问/可编辑/可评论/可查看 | 团队协作者 |
| **分享链接** | 可编辑/可评论/可查看 | 外部人员 |
| **发布到 Web** | 只读（公开） | 文档站 |

### 权限等级

| 等级 | 能做什么 |
|------|----------|
| **完全访问** | 编辑 + 管理权限 + 删除 |
| **可编辑** | 编辑 + 评论 |
| **可评论** | 仅评论 + 查看 |
| **可查看** | 仅查看 |

## 团队空间（Teamspace）

```
Settings → Teamspaces → New teamspace
```

**适合**：部门/项目组共享空间

**特点**：
- 独立导航栏
- 独立权限设置
- 成员管理

## 访客（Guests）

免费版不需要付费席位：

```
Share → Invite → 输入邮箱 → 设为 Guest
```

**访客限制**：最多 10 个（免费版）、100 个（Plus）

## 安全与隐私

| 设置 | 路径 |
|------|------|
| 禁止复制 | Share → 关闭 Allow duplicate as template |
| 禁止搜索索引 | Share → 关闭 Search engine indexing |
| 页面锁 | 页面右上角 ⋮ → Lock page |
| 评论管理 | 页面右上角 ⋮ → Resolve comments |

## 实时协作技巧

```
@某人     → 提及/通知
@日期     → 日期提醒
Ctrl+Shift+M → 添加评论
页面顶部 → 查看在线用户
```
