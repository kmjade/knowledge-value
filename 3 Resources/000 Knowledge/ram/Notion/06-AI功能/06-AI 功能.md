# 06 - Notion AI 功能

## 启动方式

```
空格键 → 弹出 AI 菜单
选中文字 → Ask AI
Ctrl+J → 继续编写
```

## AI 命令大全

### 写作

| 命令 | 说明 |
|------|------|
| Continue writing | AI 续写 |
| Summarize | 总结段落 |
| Improve writing | 润色优化 |
| Fix spelling & grammar | 修正拼写语法 |
| Make longer / shorter | 扩写/缩写 |
| Change tone | 改变语气（专业/随意/直白/友好） |
| Simplify language | 简化语言 |

### 翻译

| 命令 | 说明 |
|------|------|
| Translate | 翻译为多种语言 |

### 创意

| 命令 | 说明 |
|------|------|
| Brainstorm ideas | 头脑风暴 |
| Draft a plan | 草拟计划 |
| Write an outline | 写大纲 |
| Create a pros and cons list | 利弊分析 |

### 编辑

| 命令 | 说明 |
|------|------|
| Explain this | 解释内容 |
| Find action items | 提取行动项 |
| Translate | 翻译 |

## Notion AI 适用场景

```
✅ 会议纪要 → AI 提取行动项
✅ 长文档 → AI 生成摘要
✅ 双语协作 → AI 实时翻译
✅ 头脑风暴 → AI 展开创意
✅ 邮件草稿 → AI 润色语气
```

## 定价

Notion AI 是付费附加功能：
- 包含在 Plus / Business / Enterprise 方案中
- 免费版有少量试用额度

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` | 呼出 AI 菜单 |
| `Ctrl+J` | AI 续写 |
| `Tab` | 接受 AI 建议 |
| `Esc` | 关闭 AI 面板 |
