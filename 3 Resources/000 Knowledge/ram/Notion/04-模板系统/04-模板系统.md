# 04 - Notion 模板系统

## 内置模板

```
新建页面 → Templates → 选择模板
```

**分类**：
- Personal：个人日记、习惯追踪
- Education：课程笔记、论文规划
- Engineering：Sprint 计划、Bug 追踪
- Design：设计规范、情绪板
- Marketing：内容日历、社交媒体
- Sales：CRM、销售漏斗

## 自定义模板

### 页面模板

```
1. 创建页面，编辑好内容和结构
2. 页面右上角 ⋮ → Save as template
3. 命名模板 → 保存
```

### 数据库模板

```
数据库视图 → ▼ → New template → 编辑默认字段值 → 命名
```

**数据库模板可以预设**：
- 属性默认值
- 页面内容结构
- 子 Block（折叠、表格等）

### 模板按钮

创建一个「模板按钮」Block：

```
/button → 选择模板 → 一键生成
```

> 完美适合：每日日记、周报、会议纪要等重复性内容。

## 按钮自动化

Notion Button（新功能）：

```
/button → Insert blocks → 配置要插入的 Block
/button → Add page to → 配置页面位置
/button → Edit pages in → 批量编辑
/button → Send notification → 发送通知
```

## 模板设计原则

1. **少即是多** — 属性别超过 10 个，避免压迫感
2. **预设默认** — 填好常用字段默认值
3. **图标区分** — 用 emoji 快速识别
4. **注释引导** — 关键位置加注释
5. **视图搭配** — 至少配 2 种视图
