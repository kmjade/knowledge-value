# 01 - Notion 基础入门

## 注册与登录

1. 访问 https://www.notion.so/
2. 使用 Google 账号或邮箱注册
3. 选择用途 → 跳过模板 → 进入空白工作区

> 💡 学生/教师可用学校邮箱换免费 Plus 方案

## 第一个页面

```
打开 Notion → 新建页面（Ctrl+N / Cmd+N）→ 输入标题 → 开始写
```

**试试这些**：
```text
/ 呼出命令菜单（最核心的快捷键）
/h1 一级标题
/h2 二级标题  
/bullet 无序列表
/todo 待办清单
/page 新建子页面
```

## 核心快捷键

| 快捷键 | 功能 |
|--------|------|
| `/` | 命令菜单（最重要！） |
| `Ctrl+N` | 新建页面 |
| `Ctrl+Shift+N` | 新建窗口 |
| `Ctrl+P` | 快速搜索/跳转 |
| `Ctrl+[` | 返回上一页 |
| `Ctrl+]` | 前进 |
| `Ctrl+Shift+L` | 切换暗色模式 |
| `Esc` | 选中当前 Block |
| `Ctrl+D` | 复制 Block |
| `:/` | 快速 Emoji |

## Block 拖拽

```
将鼠标悬停在 Block 左侧 → 看到 ⋮⋮ 图标 → 拖拽移动
选中多个 Block → 批量拖拽
```

> Block 可以拖到任何位置，包括并排排列。

## 设置

左侧边栏顶部 → Settings & members：
- My account：个人信息
- My notifications：通知设置
- Language & region：语言（支持中文）
- Appearance：暗色/亮色/跟随系统
