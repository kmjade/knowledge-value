# 07 - Notion 集成与自动化

## 官方集成（Built-in）

```
Settings → My connections → 连接第三方应用
```

**支持的服务**：

| 服务 | 用途 |
|------|------|
| **Slack** | 消息通知 |
| **GitHub** | 关联 PR/Issue |
| **Google Calendar** | 日历同步 |
| **Jira** | 任务同步 |
| **Figma** | 设计稿嵌入 |
| **Zoom** | 会议记录 |
| **Zapier** | 万能自动化 |

## Notion API

```
https://api.notion.com/v1/
```

### 创建集成

```
https://www.notion.so/my-integrations → New integration → 获取 API Key
```

### Python 示例

```python
import requests

headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json",
    "Notion-Version": "2022-06-28"
}

# 查询数据库
response = requests.post(
    f"https://api.notion.com/v1/databases/{DATABASE_ID}/query",
    headers=headers
)

# 创建页面
data = {
    "parent": {"database_id": DATABASE_ID},
    "properties": {
        "Name": {"title": [{"text": {"content": "新任务"}}]},
        "Status": {"select": {"name": "待办"}}
    }
}
response = requests.post(
    "https://api.notion.com/v1/pages",
    headers=headers, json=data
)
```

## Zapier 自动化

```
Notion 触发器 → Zapier → 目标动作
```

**常用 Zap**：

| 触发 | 动作 |
|------|------|
| Notion 新增数据库项 | → Slack 通知 |
| Google 表单提交 | → 写入 Notion 数据库 |
| Gmail 收到邮件 | → 创建 Notion 任务 |
| Notion 任务到期 | → 发送邮件提醒 |
| GitHub Issue 更新 | → 同步 Notion 数据库 |

## Webhooks（第三方）

无内置 Webhook，通过以下方式实现：

1. Zapier Webhook → Notion
2. Make (Integromat) → Notion
3. n8n (自托管) → Notion
4. Pipedream → Notion API
