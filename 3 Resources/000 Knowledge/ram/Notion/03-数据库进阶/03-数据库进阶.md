# 03 - Notion 数据库进阶

## 公式（Formula）

### 基本语法

```
prop("属性名")          → 引用属性
if(条件, true, false)   → 条件判断
format()                → 格式化输出
round()                 → 四舍五入
dateBetween(d1, d2, "days") → 日期差
```

### 常用公式示例

```
# 进度百分比
round(prop("已完成") / prop("总数") * 100) + "%"

# 任务状态图标
if(prop("状态") == "完成", "✅", "🔄")

# 截止日期倒计时
dateBetween(prop("截止日期"), now(), "days") + " 天"

# 优先级标签
if(prop("优先级") == "高", "🔴", 
  if(prop("优先级") == "中", "🟡", "🟢"))

# 自动编号
"TASK-" + format(prop("创建时间").id())
```

## 关联（Relation）

```
在一个数据库中创建「关联」属性 → 关联另一个数据库
```

**典型场景**：
- 项目 ↔ 任务（一个项目对应多个任务）
- 客户 ↔ 订单
- 作者 ↔ 书籍

## 汇总（Rollup）

```
基于「关联」属性，从关联的数据库中汇总数据
```

**统计方式**：
- 显示原始 → 列出所有值
- 计数 → 关联数量
- 求和/平均/最大/最小/中位数 → 数字汇总
- 最早/最晚日期

**实例**：
```
任务数据库中 → 汇总项目的「总预算」
项目数据库中 → 汇总任务的「完成率」
```

## 视图（View）

每个数据库可以有多个视图，**同一数据不同视角**。

### 过滤

```
Add filter → 选择属性 → 设置条件
```

**示例**：
- 只看「未完成」的任务
- 只看「本月」的笔记
- 只看「高优先级」的项目

### 排序

```
Add sort → 按优先级降序 → 按截止日期升序
```

### 分组

```
按「状态」分组 → 自动归类为「待办/进行中/完成」
按「负责人」分组 → 每人一组
```

### 数据库模板

```
表格右上角 ▼ → New template → 设置默认字段值
```

> 新建行时自动填入模板内容，超好用！

## 实战：项目管理数据库

```
Projects 数据库
├── 属性：项目名、状态、优先级、负责人、开始日、截止日、预算
├── 关联 → Tasks 数据库
├── 视图1：看板（按状态分组）
├── 视图2：时间线（甘特图）
└── 视图3：日历（截止日视图）

Tasks 数据库
├── 属性：任务名、状态、优先级、截止日、预估工时
├── 关联 → Projects 数据库
├── 汇总 → 项目名（从 Projects）
├── 视图1：表格（全部任务）
└── 视图2：看板（按状态）
```
