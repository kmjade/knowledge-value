# Knowledge & Culture

> 知识不只是头脑中的东西——它嵌在语言、传统、制度和集体实践中。文化是一个社会的知识储备库。

## 文化作为集体知识储备

| 维度 | 个人知识 | 文化知识 |
|------|---------|---------|
| 持有者 | 个体 | 群体、社会、文明 |
| 传递方式 | 学习、思考 | 语言、传统、教育、仪式 |
| 持久性 | 一生 | 数代、数千年 |
| 修改方式 | 个人信念改变 | 文化演化、革命 |
| 丢失方式 | 遗忘、死亡 | 文化断层、文明失落 |

> 每一次你开口说话，你就在使用几万年来积累的文化知识——语法、词汇、概念都是祖先的遗产。

## 知识的社会建构

| 理论 | 核心主张 | 代表 |
|------|---------|------|
| **知识社会学** | 知识的形态受社会结构影响 | 曼海姆 (Karl Mannheim) |
| **社会建构论** | 科学事实也是社会协商的产物 | 布鲁尔 (Bloor), 拉图尔 (Latour) |
| **社会认识论** | 知识是一种社会现象——需要研究其社会维度 | 戈德曼 (Alvin Goldman) |

### 强纲领 (Strong Programme) 的四原则

| 原则 | 内容 |
|------|------|
| **因果性** | 信念的形成有社会原因 |
| **无偏性** | 真信念和假信念应被同等方式解释 |
| **对称性** | 同类型的原因解释真信念和假信念 |
| **反身性** | 这些原则也适用于社会学本身 |

## 知识的文化传递

| 机制 | 说明 | 例子 |
|------|------|------|
| **直接教学** | 有意识的知识传递 | 学校教育、师徒制 |
| **模仿** | 观察他人行为并复制 | 儿童学习母语 |
| **叙事** | 故事承载知识 | 神话、史诗、案例研究 |
| **仪式** | 重复性行为固化知识 | 宗教仪式、学术典礼 |
| **制度** | 组织结构保存并传递知识 | 大学、图书馆、科学院 |

## 模因论 (Memetics)

> 理查德·道金斯 (Richard Dawkins, 1976) 提出「模因」(meme) 作为文化的基因——一个自我复制、变异、选择的文化传递单位。

| 概念 | 类比 |
|------|------|
| **基因** (Gene) | 生物遗传的基本单位 |
| **模因** (Meme) | 文化传递的基本单位——曲调、观念、标语、时尚 |
| **模因池** (Meme Pool) | 一个文化中所有模因的总和 |
| **模因选择** | 有的模因更容易传播（简单、易记、情感冲击） |

### 模因论的批评

| 批评 | 内容 |
|------|------|
| 单位不明确 | 「模因」的边界是什么？一首歌？一段旋律？一个乐句？ |
| 过度简化 | 文化不是简单的「复制+选择」 |
| 缺乏因果力 | 模因只是一种隐喻，不是真实的因果实体 |

## 口头文化与文字文化

| 维度 | 口头文化 | 文字文化 |
|------|---------|---------|
| 知识存储 | 人脑中 | 外部媒介（泥板、纸、硬盘） |
| 知识形式 | 格律、重复、故事（便于记忆） | 抽象、分析、系统化 |
| 权威来源 | 长者、吟游诗人 | 文本、作者、图书馆 |
| 改变速度 | 慢——口述传统保守 | 快——书写允许批判和颠覆 |
| 知识民主化 | 精英垄断 | 可广泛传播 |

> 沃尔特·翁 (Walter Ong): 文字不仅记录了知识——它改变了人类思维的结构。抽象推理、逻辑体系、科学方法——这些都是文字文化的产物。

## 知识的社会分布 (Distributed Cognition)

> 没有人知道一切。现代社会运行在**认知分工**之上。

| 概念 | 内容 |
|------|------|
| 认知分工 | 知识分布在不同的个体和机构中 |
| 知识权威 | 我们依赖专家系统——医生、工程师、科学家 |
| 信任结构 | 社会运转的前提是我们信任他人的知识 |
| 知识不公 (Epistemic Injustice) | 某些群体的知识被系统性地忽视或贬低 (Fricker, 2007) |
