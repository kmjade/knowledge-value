# Brief History of Epistemology

> 从柏拉图到葛梯尔，从《泰阿泰德》到认知科学——两千四百年的人类追问：我们如何知道？我们知道了什么？

## 1. 古希腊：认识论的诞生

### 柏拉图 (Plato, 427-347 BCE)

| 概念 | 内容 |
|------|------|
| **知识 vs 意见** | 知识是确证的、不变的、关于理念 (Forms) 的——意见则是关于可感世界的 |
| **回忆说** | 学习就是回忆——灵魂在出生前已见过理念 |
| **线喻** | 可知世界（理念） > 可感世界（事物）——知识在线的上端 |
| **《泰阿泰德》** | 知识 = 被确证的真信念 (JTB) ——认识论的奠基文本 |

### 亚里士多德 (Aristotle, 384-322 BCE)

| 概念 | 内容 |
|------|------|
| **经验主义开端** | 知识从感官经验开始——「灵魂中没有不先经过感官的东西」 |
| **范畴论** | 十大范畴——知识分类的第一个系统 |
| **证明科学** | 从第一原理 (First Principles) 出发的演绎体系 |

### 怀疑论 (Skepticism)

| 学派 | 主张 |
|------|------|
| **皮浪主义** (Pyrrhonism) | 悬置判断 (epochē) ——对任何命题都可以找到对等的反面论证 |
| **学园派怀疑论** | 没有知识是可能的——最多只有「概然的」信念 |

> 怀疑论是认识论的影子：每一次认识论的前进，都是对怀疑论的一次回应。

## 2. 中世纪：信仰寻求理解

| 人物 | 主张 |
|------|------|
| **奥古斯丁** (354-430) | 「我相信以便理解」(credo ut intelligam)——信仰是知识的起点 |
| **安瑟伦** (1033-1109) | 本体论证明：上帝存在的先天论证 |
| **阿奎那** (1225-1274) | 综合亚里士多德与基督教——理性和信仰各有疆域 |
| **司各脱 & 奥卡姆** | 意志优先于理智——「如无必要，勿增实体」(奥卡姆剃刀) |

> 中世纪的核心张力：**启示 vs 理性**——被揭示的真理和人类推理获得的真理，关系是什么？

## 3. 近代：认识论的黄金时代

### 理性主义路线

| 人物 | 核心 | 名言 |
|------|------|------|
| **笛卡尔** (1596-1650) | 方法怀疑 → 「我思故我在」 → 天赋观念 | Cogito, ergo sum |
| **斯宾诺莎** (1632-1677) | 几何学式的哲学——从公理出发演绎一切 | Deus sive Natura |
| **莱布尼茨** (1646-1716) | 单子论、充足理由律、可能世界中最好的世界 | —— |

### 经验主义路线

| 人物 | 核心 | 名言 |
|------|------|------|
| **洛克** (1632-1704) | 心灵是白板 (tabula rasa)——一切观念来自经验 | —— |
| **贝克莱** (1685-1753) | 存在即被感知 (esse est percipi)——物质不存在 | —— |
| **休谟** (1711-1776) | 因果只是习惯联想的归纳问题——极端怀疑论 | —— |

### 康德的哥白尼革命

> 「不是知识必须符合对象，而是对象必须符合我们的认知形式。」

| 概念 | 内容 |
|------|------|
| **先验综合判断** | 先于经验但对世界有所断言——数学、自然科学的基本原理 |
| **现象 vs 物自体** | 我们只能认识现象（被认知形式过滤后的世界）——物自体不可知 |
| **范畴** | 12 个先天知性范畴——人类认知的通用模板 |

## 4. 现代：分析哲学与实用主义

| 人物 | 贡献 |
|------|------|
| **罗素** (1872-1970) | 亲知 (acquaintance) vs 描述 (description) 的区分 |
| **早期维特根斯坦** (1889-1951) | 《逻辑哲学论》——语言图像论：「语言的界限就是世界的界限」 |
| **逻辑实证主义** | 可证实性原则——一个有意义的命题要么是分析的，要么是可经验证实的 |
| **詹姆斯 & 杜威** | 实用主义——真理是「有用」的信念 |

## 5. 当代：葛梯尔之后

### 1963：葛梯尔的炸弹

> Edmund Gettier 用三页纸推翻了 JTB——两千四百年后，我们仍然不知道「知识」的精确定义。

| 方向 | 代表人物 | 核心 |
|------|---------|------|
| **可靠主义** | Alvin Goldman | 知识的来源必须是可靠的认知过程 |
| **德性认识论** | Ernest Sosa, Linda Zagzebski | 知识来自认知德性——细心、开放、诚实 |
| **语境主义** | David Lewis, Keith DeRose | 「知道」的标准随语境变化 |
| **自然化认识论** | W.V.O. Quine | 认识论是心理学的一章——用科学方法研究科学 |

### 认知科学与认识论

| 领域 | 对认识论的贡献 |
|------|--------------|
| **认知心理学** | 人实际上如何形成信念？——不完全符合任何哲学模型 |
| **神经科学** | 知识有神经基础——但神经元和「知识」之间的桥梁仍然模糊 |
| **进化认识论** | 我们的认知能力是自然选择的产物——适合生存，不一定适合真理 |
| **AI 与机器学习** | 机器能「知道」吗？——大语言模型是否有信念？ |

## 关键转折点

| 时间 | 事件 | 影响 |
|------|------|------|
| ~380 BCE | 柏拉图《泰阿泰德》 | 定义了「知识是什么」的问题 |
| 1641 | 笛卡尔《第一哲学沉思集》 | 开启了近代理性主义认识论 |
| 1781 | 康德《纯粹理性批判》 | 「哥白尼革命」——认识论的结构性转向 |
| 1963 | 葛梯尔三页论文 | 推翻 JTB——当代认识论的起点 |
| 1969 | 蒯因「自然化认识论」 | 认识论的去哲学化——转向科学 |
