# Theories of Truth

> 「真」是什么意思？「雪是白的」为真——但「为真」意味着什么？哲学家争论了两千年，给出了至少四种回答。

## 真理理论的四大流派

| 理论 | 核心主张 | 「雪是白的」为什么为真？ | 代表 |
|------|---------|------------------------|------|
| **符合论** | 命题符合事实 | 因为雪确实是白的 | 亚里士多德、罗素、早期维特根斯坦 |
| **融贯论** | 命题与信念系统一致 | 因为它与我们的其他信念融洽 | 黑格尔、布拉德利 |
| **实用论** | 真理 = 有用的信念 | 因为相信「雪是白的」在实践中有效 | 詹姆斯、杜威 |
| **冗余论** | 「为真」不增加信息 | 「雪是白的」就够了，加「为真」是废话 | 拉姆齐、蒯因 |

## 符合论 (Correspondence Theory)

> 「说是者为是，非者为非，这就是真。」— 亚里士多德《形而上学》

| 要素 | 说明 |
|------|------|
| **真理承载者** | 命题、信念、语句 |
| **真理制造者** | 事实、事态、世界 |
| **符合关系** | 命题与世界之间的某种对应 |

### 符合论的困难

| 问题 | 内容 |
|------|------|
| **什么是「事实」？** | 事实本身是不是命题的投影？——循环？ |
| **如何符合？** | 命题「猫在垫子上」与事实「猫在垫子上」如何「对应」？ |
| **负面真理** | 「独角兽不存在」——它与什么事实符合？ |
| **数学真理** | 「2+2=4」——它与哪个事实符合？ |

## 融贯论 (Coherence Theory)

> 一个信念为真，当且仅当它与信念系统中的其他信念融贯一致。

| 特点 | 说明 |
|------|------|
| **整体论** | 单个命题不能独立为真——真理是信念之网的整体属性 |
| **理想主义背景** | 黑格尔：实在是精神的——没有独立于心灵的「外部事实」 |
| **科学中的应用** | 一个科学理论被接受，更多是因为它与其他理论融贯，而非与「事实」一一对应 |

### 融贯论的困难

| 问题 | 内容 |
|------|------|
| **可能有多套融贯系统** | 两套互相矛盾但各自内部融贯的信念系统——哪套是真？ |
| **脱离世界** | 一个完全融贯的童话世界——但它不是真的 |
| **融贯的标准** | 什么算「融贯」？——没有一致定义 |

## 实用论 (Pragmatic Theory)

> 「真理是信念的一种属性，当信念在实践中被证明有用时，它就是真的。」— 威廉·詹姆斯

| 代表 | 主张 |
|------|------|
| **皮尔士** (C.S. Peirce) | 真理是探究的终点——理想科学共同体最终会同意的东西 |
| **威廉·詹姆斯** (William James) | 真理是「现金价值」——一个信念为真当它「好用」 |
| **约翰·杜威** (John Dewey) | 真理 = 「有根据的可断言性」——被经验证据支持的主张 |

### 实用论的吸引力与问题

| 优势 | 问题 |
|------|------|
| 解释了真理与行动的联系 | 「有用」不等于「真」——假信念也可能有用 |
| 适合科学进步的解释 | 真理变成了时间依赖的——哥白尼之前「太阳绕地球」是真的吗？ |
| 避免了形而上学难题 | 如何定义「有用」本身就有争议 |

## 冗余论与紧缩论 (Deflationism)

> 「『雪是白的』为真」——这句话没有比「雪是白的」多说什么。

| 版本 | 主张 |
|------|------|
| **冗余论** (Ramsey) | 「为真」是一个多余的谓词 |
| **去引号论** (Quine) | 「『雪是白的』为真」↔「雪是白的」（只需去引号） |
| **极小论** (Horwich) | 真理没有本质——只要接受所有形如「『P』为真当且仅当 P」的命题就足够了 |

> **紧缩论的核心洞见**：我们不需要一个厚重的「真理本质」理论——只需要知道「为真」这个词怎么用。

## 真理理论的当代格局

| 取向 | 特征 | 流行领域 |
|------|------|---------|
| **符合论** | 直觉最强，仍在科学实在论中占主流 | 分析哲学、科学哲学 |
| **融贯论** | 在反实在论和整体论中仍有影响 | 欧陆哲学、系统论 |
| **实用论** | 强调真理与探究的关联 | 新实用主义 (Rorty, Putnam) |
| **紧缩论** | 技术性强，避免形而上学 | 当代分析哲学主流之一 |

> **蒯因与自然化认识论**：蒯因主张「认识论是心理学的一章」——关于知识的问题最终是经验科学问题。真理亦然：我们如何形成「真」的判断是一个认知科学问题。
