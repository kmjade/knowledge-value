# 09-Information-Institutions与管理

> 本章覆盖信息机构的运营管理、组织中的知识管理，以及数字人文——LIS 与人文研究交叉的前沿领域。

## 信息机构类型

| 类型 | 代表 | 核心使命 |
|------|------|----------|
| 图书馆 | 国家图书馆、公共图书馆、高校图书馆 | 信息获取和素养 |
| 档案馆 | 国家/地方/企业档案馆 | 证据保存和长期访问 |
| 数据中心 | 科研数据中心、政府数据平台 | 数据组织和共享 |
| 博物馆 | 历史/艺术/科学博物馆 | 实物+信息的保存展示 |
| 情报机构 | 竞争情报、科技情报 | 信息分析和决策支持 |

## 知识管理 (KM)

### SECI 模型（野中郁次郎）

知识转化的四种模式：

| 模式 | 转化 | 例子 |
|------|------|------|
| **S**ocialization | 隐性 → 隐性（观察模仿） | 跟资深馆员学参考咨询 |
| **E**xternalization | 隐性 → 显性（写下来） | 把经验编成培训手册 |
| **C**ombination | 显性 → 显性（整合） | 合并多个数据库使用说明 |
| **I**nternalization | 显性 → 隐性（内化） | 读了手册后在实践中形成直觉 |

### 知识管理工具

| 工具 | 说明 |
|------|------|
| 知识库/Wiki | 组织的集体显性知识 |
| 实践社区 (CoP) | 同行的非正式交流 |
| 事后回顾 (AAR) | 项目结束后的经验总结 |
| 专家黄页 | 组织中"谁知道什么"的目录 |

## 数字人文 (Digital Humanities)

DH 不是"用电脑研究文科"——是方法论的根本转变：

| 传统人文 | 数字人文 |
|----------|----------|
| 精读 (Close Reading) | 远读 (Distant Reading) |
| 单个文本 | 大规模语料库 |
| 定性 | 定量+定性混合 |
| 个人 | 团队（学者+程序员+设计师） |

### DH 方法

| 方法 | 做什么 | 例子 |
|------|--------|------|
| 文本挖掘 | 从海量文本中提取模式 | 分析 100 年报纸中的性别语言变迁 |
| 网络分析 | 实体间的关系网络 | 宋代文人社交网络 |
| GIS | 人文数据映射地理空间 | 文学家的行迹地图 |
| 3D 重建 | 虚拟复原历史遗迹 | 圆明园数字复原 |

### 中国 DH 重点项目

| 项目 | 内容 |
|------|------|
| **CBDB** | 中国历代人物传记数据库（哈佛） |
| **CTEXT** | 中国哲学书电子化计划 |
| **唐宋文学编年地图** | 作家行迹的时空可视化 |

## 信息政策与伦理

| 议题 | 核心问题 |
|------|----------|
| 信息自由 | 公众获取政府信息的权利 |
| 隐私保护 | 个人信息在信息机构中的保护 |
| 知识产权 | 版权、合理使用、开放许可 |
| 数字包容 | 缩小数字鸿沟 |
| FAIR 原则 | Findable, Accessible, Interoperable, Reusable |
