# CLC 知识库总览

> Chinese Library Classification — 中国图书馆分类法

## 快速概览

| 维度 | 说明 |
|------|------|
| 全称 | 中国图书馆分类法 (Chinese Library Classification) |
| 简称 | CLC / 中图法 |
| 创建年份 | 1975 年（第一版） |
| 管理机构 | 中国图书馆分类法编辑委员会 |
| 标记体系 | 字母 + 数字（A-Z + 0-9） |
| 大类数量 | 22 个基本大类 |
| 使用范围 | 中国大陆绝大多数图书馆 |
| 当前版本 | 第 5 版（2010 年） |
