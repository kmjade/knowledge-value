# 07-信息计量学

> 信息计量学用定量方法研究信息和知识的产生、传播和使用——回答"谁的研究最有影响力？""哪些领域在兴起？""知识如何流动？"

## 三大计量学

| 学科 | 研究对象 | 典型问题 |
|------|----------|----------|
| **文献计量学 (Bibliometrics)** | 文献和出版物 | 某篇论文被引了多少次？ |
| **科学计量学 (Scientometrics)** | 科学活动和系统 | 一个国家的科研产出趋势？ |
| **信息计量学 (Informetrics)** | 广义信息 | 网站之间的链接模式？ |

> 三者边界模糊，实践中常互用。学术评价主要用文献计量学。

## 核心指标

### 期刊指标

| 指标 | 含义 | 计算方法 | 局限 |
|------|------|----------|------|
| **影响因子 (IF)** | 期刊"平均"影响力 | 前两年发表论文在当年的篇均被引 | 不适合评价单篇论文、被综述拉高、学科不可比 |
| **CiteScore** | Scopus 的影响因子替代 | 四年窗口 | 窗口更长，免费 |
| **H 指数** | 研究者影响力 | 有 h 篇论文被引至少 h 次 | 不能跨学科比较、只升不降 |
| **Eigenfactor** | 期刊"总"影响力（类似 PageRank） | 加权被引 + 排除自引 | 大期刊天然占优 |
| **SJR** (SCImago Journal Rank) | 期刊声望 | 类似 Eigenfactor，基于 Scopus | |

### 研究者指标

| 指标 | 含义 |
|------|------|
| **h 指数** | 平衡了产量和影响力——一个 h=20 的研究者有 20 篇论文至少被引 20 次 |
| **g 指数** | h 指数的改进——给高被引论文更多权重 |
| **i10 指数** | 被引 ≥ 10 次的论文数（Google Scholar 简单指标） |

### 论文指标

| 指标 | 含义 |
|------|------|
| **被引次数** | 最直接但最容易被滥用的指标 |
| **FWCI** (Field-Weighted Citation Impact) | 经学科、年份、文献类型标准化后的被引——比原始被引更公平 |
| **Altmetrics** | 在线关注度：新闻提及、推文、Mendeley 保存数 |

## 引文分析

### 引用行为的动机（为什么引用？）

引用不只是"受影响"——有复杂的社会和策略因素：

| 引用动机 | 说明 |
|----------|------|
| 承认知识来源 | 规范的、应然的引用 |
| 说服 | 引用权威研究增加自己论证的力度 |
| 社交 | 引用潜在审稿人、圈内同行 |
| 自引 | 引用自己的之前工作 |
| 负面引用 | 引用某研究来反驳它 |

> 这就是为什么"高被引 ≠ 高质量"——引用是复杂社会行为，不是纯科学度量。

### 引文网络

```
论文 A ──引用──→ 论文 B ──引用──→ 论文 D
  │                │
  └────引用────────┘
        论文 C ──引用──→ 论文 E
```

通过引文网络可以分析：
- **共引 (Co-citation)**：两篇论文同时被第三篇引用 → 它们可能相关
- **文献耦合 (Bibliographic Coupling)**：两篇论文引用了同一篇文献 → 它们可能相关
- **研究前沿**：共引论文在近期密集增长 → 新兴热点

## 科学计量定律

| 定律 | 内容 | 含义 |
|------|------|------|
| **洛特卡定律** | 发表 n 篇论文的作者数量 ≈ 1/n² | 少数高产作者贡献多数论文 |
| **布拉德福定律** | 某一学科的核心期刊数量有限 | 大部分重要论文集中在少数期刊 |
| **齐普夫定律** | 词频 × 排名 ≈ 常数 | 少量高频词覆盖大量出现 |
| **普赖斯定律** | 科学文献呈指数增长 | 每 10-15 年翻一番 |

## 替代计量学 (Altmetrics)

传统引文太慢——从发表到被引可能需要数年。Altmetrics 追踪即时在线关注：

| 来源 | 追踪什么 |
|------|----------|
| 新闻媒体 | 论文被多少新闻提及 |
| 社交媒体 | Twitter/X 提及、Facebook 分享 |
| 参考文献管理 | Mendeley 保存数、Zotero 收藏 |
| 政策文件 | 被政府/政策文件引用 |
| 维基百科 | 被维基百科条目引用 |

> Altmetrics 测量的是"关注度"而非"质量"。一篇论文因为错误而被大量讨论，Altmetrics 也会很高。

## 负责任地使用指标

### 莱顿宣言 (Leiden Manifesto, 2015)

十条原则——最重要的几条：

1. 定量评估应支持而非取代定性专家判断
2. 根据机构、学科和国家的不同使命来衡量绩效
3. 保护卓越的本地化研究
4. 保持数据收集和分析过程的开放和透明
5. 允许被评估者验证数据和分析结果

### DORA 宣言 (2012)

**不要用期刊影响因子来评价单个研究者或论文。** 论文的 merit 在于其内容，不是发表在哪。
