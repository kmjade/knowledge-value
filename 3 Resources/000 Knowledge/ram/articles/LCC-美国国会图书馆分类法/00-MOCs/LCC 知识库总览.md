# LCC 知识库总览

> Library of Congress Classification — 美国国会图书馆分类法

## 快速概览

| 维度 | 说明 |
|------|------|
| 全称 | Library of Congress Classification |
| 简称 | LCC |
| 创建年份 | 1897 年（开始），1902 年（首个大纲） |
| 管理机构 | 美国国会图书馆 (Library of Congress) |
| 标记体系 | 字母 + 数字（A-Z + 1-9999） |
| 大类数量 | 21 个基本大类（A-Z，跳过 I、O、W、X、Y） |
| 使用范围 | 北美研究型/大学图书馆 |
| 特点 | 无严格十进制、灵活扩展、按学科逐类发展 |
