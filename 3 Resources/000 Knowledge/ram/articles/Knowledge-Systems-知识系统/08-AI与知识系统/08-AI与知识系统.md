# 08-AI 与知识系统

> AI 正在重塑知识系统——LLM 可以"理解"自然语言，知识图谱提供事实基础。两者结合是 2024-2026 的核心趋势。

## LLM vs 知识图谱

| 维度 | LLM | 知识图谱 |
|------|-----|----------|
| 知识存储 | 模型参数 (隐式) | 三元组 (显式) |
| 更新 | 重训练 | 实时 |
| 推理 | 统计 | 逻辑 |
| 可解释 | 弱 | 强 |
| 幻觉 | 有 | 无 |

> 趋势：**LLM + KG = RAG**——检索知识图谱中的事实，LLM 生成自然语言回答。

## RAG (检索增强生成)

```
用户提问 → 检索相关文档/图谱 → 拼入Prompt → LLM生成 → 带引用回答
```

## 向量数据库——新知识系统

| 数据库 | 特点 |
|--------|------|
| Pinecone | 商业 |
| Chroma | 开源 |
| Weaviate | 开源、GraphQL |
| Milvus | 中国开源 |

> 向量数据库是"非结构化的索引"——不靠分类号，靠语义相似度。
