# 知识系统 知识库总览 (DDC 001)

> 知识系统（Knowledge Systems / Knowledge Organization Systems）是对知识进行结构化描述、组织和检索的体系——从分类法到知识图谱，从叙词表到本体论。对应 DDC 001「知识」与 025.4「分类与索引」。

## 知识系统的层次

```
语义丰富度 ↑
┌──────────────────────┐
│   知识图谱 (Knowledge Graph)│ 实体+关系+推理
├──────────────────────┤
│   本体 (Ontology)          │ 概念层级+逻辑约束
├──────────────────────┤
│   叙词表 (Thesaurus)       │ 层级+同义词+关联
├──────────────────────┤
│   分类法 (Classification)  │ 学科层级 (DDC/UDC/CLC)
├──────────────────────┤
│   标签 (Folksonomy)        │ 用户自由标注
└──────────────────────┘
       ↑ 结构化程度递增
```
