## 🧠 知识系统层次

```
语义丰富度 ↑
┌──────────────────────────────────────┐
│   知识图谱 (Knowledge Graph)         │ 实体 + 关系 + 推理    │
├──────────────────────────────────────┤
│   本体 (Ontology)                    │ 概念层级 + 逻辑约束    │
├──────────────────────────────────────┤
│   叙词表 (Thesaurus)                 │ 层级 + 同义词 + 关联   │
├──────────────────────────────────────┤
│   分类法 (Classification)            │ 学科层级 (DDC/UDC)    │
├──────────────────────────────────────┤
│   标签 (Folksonomy)                  │ 用户自由标注          │
└──────────────────────────────────────┘
       ↑ 结构化程度递增
```

## 📌 注意事項

> [!tip] 这个知识库的价值
> 它回答了一个根本问题：**知识如何被结构化地组织**。从古老的 DDC 分类法到现代的 LLM + Knowledge Graph，这个知识库帮你理解知识组织的过去、现在和未来。

> [!info] 与 Obsidian 的关系
> 你的 Obsidian 笔记系统本质上是一个"个人知识组织系统" (Personal KOS)，理解 KOS 理论能帮你更好地设计和管理你的第二大脑。
