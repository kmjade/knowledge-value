# 知识系统 资源收集

## 📚 核心标准

| 标准 | 领域 |
|------|------|
| **SKOS** (W3C) | 叙词表/分类法的语义网表达 |
| **OWL** (W3C) | 本体语言 |
| **RDF** (W3C) | 资源描述框架 |
| **SPARQL** (W3C) | RDF 查询语言 |
| **ISO 25964** | 叙词表国际标准 |

## 🧠 知识图谱

| 图谱 | 规模 |
|------|------|
| **Wikidata** | 1 亿+ 实体——开源 |
| **Google Knowledge Graph** | 数十亿实体——闭源 |
| **DBpedia** | 从 Wikipedia 提取 |

## 🛠️ 工具

| 工具 | 用途 |
|------|------|
| **Protégé** | 本体编辑器 (Stanford) |
| **Neo4j** | 图数据库 |
| **GraphDB** | RDF 三元组存储 |
| **Zotero** | 个人知识管理——参考文献 |
