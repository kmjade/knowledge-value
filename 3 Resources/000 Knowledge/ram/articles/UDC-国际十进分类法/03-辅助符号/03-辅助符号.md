# 03-辅助符号

> 辅助符号是 UDC 的精髓——它们让 UDC 从"给一个主题一个编号"升级到"用符号语言精确描述文献内容"。

## 并列与延伸

| 符号 | 含义 | 例子 | 读法 |
|------|------|------|------|
| **+** | 并列（两个独立主题） | 622+669 | 采矿**和**冶金 |
| **/** | 范围（连续类号） | 621.3/.4 | 从 621.3 **到** 621.4 |

## 关系

| 符号 | 含义 | 例子 | 读法 |
|------|------|------|------|
| **:** | 关系（两个主题的关联） | 37:004 | 教育**与**计算机 |
| **::** | 固定关系（不可逆） | 61::004 | 医学**的**计算机方面 |

```
37:004 —— 教育中的计算机（编程教育）
004:37 —— 计算机在教育中的应用（教育软件）
→ 冒号前后顺序表达不同的侧重点！
```

## 通用辅助符号

### 语言 (=...)

| 符号 | 语言 |
|------|------|
| =111 | 英语 |
| =133.1 | 法语 |
| =161.1 | 俄语 |
| =581 | 中文 |

```
61=111 —— 医学（英文文献）
```

### 形式 (0...)

| 符号 | 形式 |
|------|------|
| (035) | 手册、指南 |
| (038) | 词典 |
| (043) | 学位论文 |
| (051) | 期刊 |
| (075) | 教科书 |
| (091) | 历史 |

```
61(035) —— 医学手册
```

### 地点 (...)

| 符号 | 地点 |
|------|------|
| (510) | 中国 |
| (520) | 日本 |
| (73) | 美国 |
| (4) | 欧洲 |
| (5) | 亚洲 |

```
61(510)(091) —— 中国医学史
```

### 时间 "..."

| 符号 | 时间 |
|------|------|
| "19" | 20 世纪 |
| "20" | 21 世纪 |
| "2019" | 公元 2019 年 |

```

## 组合示例

| 文献 | UDC 类号 | 拆解 |
|------|----------|------|
| 中国医学史（英文） | 61(510)(091)=111 | 医学 + 中国 + 历史 + 英语 |
| 计算机伦理学手册 | 004:17(035) | 计算机与伦理 + 手册 |
| 21世纪德国高等教育 | 378(430)"20" | 高等教育 + 德国 + 21世纪 |
| 日本采矿与冶金 | 622+669(520) | 采矿与冶金 + 日本 |

## 引用顺序

UDC 的辅助符号有严格的引用顺序：

```
主类号 :: 固定关系 : 关系 + 并列
  = 语言
  (0...) 形式
  (...) 地点
  "..." 时间
  -0 通用特征
  .0 专用复分
```

> 这是 UDC 最复杂也最强大的部分——掌握了符号，就掌握了精确描述任何文献的能力。
