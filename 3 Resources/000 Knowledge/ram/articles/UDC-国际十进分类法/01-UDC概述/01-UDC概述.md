# 01-UDC 概述

> UDC (Universal Decimal Classification) 是比利时学者 Paul Otlet 和 Henri La Fontaine 于 1905 年创建的分类系统——它从 DDC 出发，但走得比 DDC 远得多。

## 创建者：Paul Otlet (1868-1944)

| 事实 | 说明 |
|------|------|
| 信息科学之父 | 比 Vannevar Bush 早 50 年提出"世界脑" |
| 1895 | 创建国际书目学会 (IIB) |
| 1905 | 基于 DDC 发布 UDC 第一版 |
| 愿景 | 用分类法+索引卡片组织人类全部知识——Google 的维多利亚时代版本 |
| 命运 | 二战中纳粹摧毁了他的 Mundaneum（世界知识中心） |

## UDC 的核心理念

| 理念 | 含义 |
|------|------|
| **通用性** | 覆盖所有知识领域，不限语言 |
| **分面性** | 可以从多个角度描述同一文献 |
| **组合性** | 用符号连接多个主题——"农业+经济"、"医学史在中国" |
| **可扩展性** | 新主题可以插入而不破坏整体结构 |

## UDC 的符号哲学

UDC 的最大创新：**辅助符号**。DDC 只能给一本书一个类号。UDC 可以组合：

```
DDC: 636.7 (狗)
UDC: 636.7+636.8 (狗+猫——两者都讨论)
UDC: 636.7:616 (狗与医学——兽医学)
UDC: 636.7(510)(091) (中国的狗的历史)
```

| 符号 | 含义 | 例子 |
|------|------|------|
| **+** | 并列 | 622+669 = 采矿与冶金 |
| **/** | 范围 | 621.3/.4 = 电气工程到热机 |
| **:** | 关系 | 37:004 = 教育与计算机 |
| **=** | 语言 | 61=111 = 医学(英文) |
| **(0...)** | 形式 | (035) = 手册、(043) = 论文 |
| **(...)** | 地点 | (510) = 中国、(73) = 美国 |
| **"..."** | 时间 | "20" = 21 世纪 |

## 十大类概览

| 类 | 内容 |
|----|------|
| **0** | 总论、计算机、信息 |
| **1** | 哲学、心理学 |
| **2** | 宗教、神学 |
| **3** | 社会科学、统计、政治、经济 |
| **4** | (空——原语言类已并入 8) |
| **5** | 数学、自然科学 |
| **6** | 应用科学、医学、科技 |
| **7** | 艺术、娱乐、体育 |
| **8** | 语言、语言学、文学 |
| **9** | 地理、传记、历史 |

> 注意：UDC 没有 4 类——1980 年代语言类从 4 并入了 8。这是 UDC 不同于 DDC 的一个明显特征。
