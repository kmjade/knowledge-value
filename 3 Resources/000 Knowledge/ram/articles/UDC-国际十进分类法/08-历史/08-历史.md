# 08-UDC 历史

## 时间线

| 年份 | 事件 |
|------|------|
| 1895 | Otlet 和 La Fontaine 创建国际书目学会 (IIB) |
| 1905 | UDC 第一版——33,000 类号 (法语) |
| 1920s | 扩展到 70,000 类号 |
| 1934 | Otlet 出版《文献学专著》——"世界脑"愿景 |
| 1940 | 纳粹摧毁 Mundaneum（15M 索引卡片被毁） |
| 1992 | UDC Consortium 成立——接管维护 |
| 2011 | UDC MRF 完全数字化 |

## Otlet 的"世界脑"

> 在 Google 出现之前 50 年，Otlet 已经在用手工索引卡片建造一个"世界知识网络"——用户寄信提问，工作人员用 UDC 检索卡片，回信给出答案。每年处理 1,500+ 查询。

类同于今天的 Wikipedia + Google，但用的是纸和分类法。
