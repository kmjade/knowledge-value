# 超心理学 知识库总览 (DDC 130)

> 超心理学（Parapsychology）研究超出常规心理学解释范围的现象：超感官知觉 (ESP)、意念致动 (PK)、濒死体验 (NDE)、转世记忆、灵性现象——以及它们背后可能的意识-物理交互机制。对应 DDC 130「超心理学与神秘学」。

## 研究谱系

```
证据强度 ↑
┌──────────────────────────────────────┐
│  实验超心理学 (Experimental)          │ Ganzfeld · 微PK · RNG · 预知实验  │
├──────────────────────────────────────┤
│  自发案例研究 (Spontaneous Cases)     │ NDE · 前世记忆 · 灵媒 · 骚灵      │
├──────────────────────────────────────┤
│  理论假说 (Theoretical)              │ 量子意识 · 场论 · 观察者效应       │
├──────────────────────────────────────┤
│  神秘学传统 (Occult Traditions)       │ 占卜 · 魔法 · 星象 · 炼金术       │
└──────────────────────────────────────┘
     ↑ 可重复性 / 可证伪性递减
```
