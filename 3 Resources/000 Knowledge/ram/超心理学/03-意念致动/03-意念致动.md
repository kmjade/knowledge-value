# 03-意念致动 (PK)

> 「用意念弯曲勺子」——这是 PK 的大众形象。但学术 PK 研究更关心的是：意识能否对随机物理系统产生微小的、统计上可检测的偏差？

## PK 的分类

| 类型 | 英文 | 说明 | 实验范式 |
|------|------|------|---------|
| **微 PK** | Micro-PK | 意识对随机系统施加微小统计偏斜 | RNG 实验 |
| **宏观 PK** | Macro-PK | 肉眼可见的物体移动/变形 | 勺子弯曲、物体悬浮 |
| **生物 PK** | Bio-PK | 意识影响生物系统 | DMILS (Direct Mental Interaction with Living Systems) |
| **骚灵现象** | Poltergeist / RSPK | 自发性反复出现的宏观 PK——常与特定人物关联 | 案例研究 |

---

## 微 PK：RNG 实验

### PEAR 实验室 (Princeton, 1979-2007)

Robert Jahn (工程院长) 和 Brenda Dunne 用随机事件发生器 (REG/RNG) 进行了 28 年的微 PK 实验：

| 参数 | 内容 |
|------|------|
| 设备 | 基于量子隧穿效应的真随机数发生器 |
| 任务 | 被试尝试「意愿」RNG 偏离 50/50 |
| 实验次数 | 数千次 |
| 总试次 | 数百万 |
| 效应量 | ~0.0001——极其微小 |
| 显著性 | 元分析：p < 0.001 |

> 批评：效应量太小，需要海量试次——任何微小的设备偏差、环境变动都可能积累成「显著性」。

### 微 PK 的特征

| 发现 | 内容 |
|------|------|
| **目标导向** | 效应取决于「目的」而不是「过程」——与物理学预期相反 |
| **距离无关** | 被试在另一大洲与在现场效果无差异 |
| **时间无关** | 被试在实验前几天或几天后「意图」也有效——时间位移 |
| **个体差异** | 部分被试表现持续优于随机——但无法可靠预测谁 |

## 全球意识计划 (Global Consciousness Project)

Roger Nelson (原 PEAR 成员) 在全球部署了 70+ 个 RNG 节点，连续运行 24/7：

| 假说 | 全球性事件（如 911、大型灾难）会引发「集体意识波动」——反映在 RNG 网络的偏差上 |
|------|------|

| 事件 | RNG 偏差 | 显著性 |
|------|---------|--------|
| 911 恐怖袭击 (2001) | 显著偏斜 | p < 0.01 |
| 戴安娜王妃葬礼 (1997) | 显著偏斜 | p < 0.01 |
| 2004 印度洋海啸 | 偏斜 | 显著 |
| 奥巴马就职 (2009) | 偏斜 | 显著 |
| 对照 (随机日期) | 无偏斜 | — |

> 批评：「显著事件」的选择是事后回溯的——如果选不同的时间窗口和事件定义，结果可能不同。

## 宏观 PK：勺子弯曲

| 人物 | 事件 | 证据状态 |
|------|------|---------|
| **Uri Geller** | 1970s 电视上弯曲勺子——声称是 PK | 魔术师 James Randi 复制了所有效果——判定为魔术 |
| **Nina Kulagina** | 苏联——影片显示她用意念移动物体 | 从未在控制条件下复现 |
| **Matthew Manning** | 英国——声称有 PK 能力 | 实验室测试结果模棱两可 |

> 宏观 PK 的核心问题：**缺乏控制条件下的可复现证据**。所有著名案例要么被揭露为魔术/欺诈，要么无法在实验室中复现。

## 骚灵现象 (Poltergeist / RSPK)

Recurrent Spontaneous Psychokinesis (RSPK)：

| 特征 | 内容 |
|------|------|
| **现象** | 物体自行移动、飞起、破碎、水龙头自开、电灯闪烁 |
| **关联** | 几乎总是围绕一个「焦点人物」——通常是青春期青少年 |
| **持续时间** | 数周到数月，然后自行消退 |
| **假说** | 无意识的 PK——焦点人物在情绪压力下无意识地释放 psi 能量 |

### 著名案例

| 案例 | 时间 | 地点 | 特征 |
|------|------|------|------|
| **Enfield 骚灵** | 1977-79 | 英国伦敦 | 围绕 11 岁女孩——著名超心理学案例 |
| **Rosenheim 骚灵** | 1967-68 | 德国 | 围绕 19 岁秘书——电灯爆炸、画框旋转 |
| **Miami 骚灵** | 1984 | 美国 | 仓库中物品自行移动——被录像 |

## PK 研究的替代解释

| 解释 | 说明 |
|------|------|
| **设备故障** | RNG 的「量子随机性」可能不是真正的随机——电子噪声、温度漂移 |
| **统计误用** | 停止规则 (optional stopping) ——收集数据直到显著 |
| **欺诈** | 宏观 PK 案例大量涉及魔术技巧 |
| **心理暗示** | 骚灵现象的目击者受到期望和暗示影响 |
| **未知物理** | 也许是未知的物理机制——但这不是「意识」 |
