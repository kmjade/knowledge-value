# 05 - GTD 回顾（Reflect）

## 每周回顾——GTD 的灵魂

> "如果你只做 GTD 中的一件事，就做每周回顾。" — David Allen

### 为什么至关重要？

```
系统 → 5-7 天不维护 → 不再可信 → 大脑重新囤积 → GTD 失效
                                      ↑
每周回顾 ──── 重新校准 ──── 恢复可信 ────┘
```

## 每周回顾清单

### 准备

```
□ 清空桌面
□ 准备好清单系统
□ 30-60 分钟不被打扰
□ 一杯咖啡/茶
```

### GET CLEAR（清空）

```
□ 收集散落的便签/纸张
□ 处理未读邮件 → 存档或转入收集篮
□ 清空微信/钉钉/飞书待处理消息
□ 整理桌面文件
□ 检查手机备忘录
```

### GET CURRENT（更新）

```
□ 回顾下一步行动清单 → 划掉已完成
□ 回顾项目清单 → 确保每个项目有下一步
□ 回顾等待清单 → 催促进展
□ 回顾日历 → 过去一周 + 未来两周
□ 清空收集篮 → 逐条理清
□ 回顾将来/也许清单 → 也许现在可以启动了？
```

### GET CREATIVE（创造）

```
□ 有什么新想法？
□ 有什么想改变的方向？
□ 当前项目是否需要调整？
□ 有没有新灵感要加入将来/也许？
```

## 每周回顾时间

```
固定时间：每周五下午 4:00 或 周日晚上
时长：30-60 分钟
地点：安静的书桌
```

## 每日检视

| 时间 | 做什么 |
|------|--------|
| 早上 | 查看日历 + 下一步行动清单，确定今天做什么 |
| 上午 | 检查邮件，处理 2 分钟能做的 |
| 下午 | 清理收集篮 |
| 傍晚 | 记录今天完成的事，把未完的移回清单 |

## 回顾频率总结

| 频率 | 做什么 | 时长 |
|------|--------|------|
| 每日 | 检视日历 + 下一步 | 5 分钟 |
| 每周 | 完整回顾 | 30-60 分钟 |
| 每月 | 检查项目方向 + 将来也许 | 30 分钟 |
| 每季/年 | 更高层面的目标评估 | 1-2 小时 |
