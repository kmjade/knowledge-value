# 02 - GTD 收集

## 核心原则

> **100% 收集**：任何引起你注意的事项，必须立刻捕获到收集篮。

## 收集什么？

```
工作相关：
├── 邮件需要回复
├── 会议待办
├── 项目想法
├── 老板交代的任务
└── 同事的请求

个人相关：
├── 家里灯泡坏了
├── 想学的新技能
├── 需要预约的体检
├── 想买的书
└── 突然的灵感
```

## 收集工具

| 工具 | 适合场景 | 推荐 |
|------|----------|------|
| 随身小本 | 随时随地 | ⭐⭐⭐⭐ |
| 手机备忘录 | 日常记录 | ⭐⭐⭐⭐⭐ |
| 语音助手 | 开车/走路 | ⭐⭐⭐ |
| 邮件草稿箱 | 工作想法 | ⭐⭐⭐ |
| A4 纸 | 大扫除式收集 | ⭐⭐⭐⭐ |
| Obsidian 速记 | 数字笔记 | ⭐⭐⭐⭐⭐ |

## 收集的纪律

```text
✅ 立刻记下，不超过 5 秒
✅ 只收集，不处理（不要边收集边整理）
✅ 一个收集篮（或定期合并）
✅ 大脑不再记忆，只信任系统
```

## 收集清单（触发列表）

用这个清单触发遗漏事项：

### 工作
```
□ 待回复的邮件
□ 未完的项目
□ 等待别人回复的事
□ 需要准备的会议
□ 需要阅读/审批的文件
□ 需要沟通的人
□ 即将截止的任务
□ 职业发展相关
```

### 个人
```
□ 家庭维修
□ 健康和医疗
□ 财务事项（账单、保险）
□ 想买的物品
□ 想学的东西
□ 朋友聚会
□ 兴趣爱好
□ 旅行计划
□ 阅读清单
```

## 大扫除（Mind Sweep）

每周/每月做一次彻底的大脑清空：

```
1. 坐下来，准备纸笔（或 Obsidian）
2. 逐项扫描触发列表
3. 写下每件想到的事（不处理！）
4. 直到「心里空了」为止
5. 通常 50-200 条
```

## 常见误区

❌ 觉得「这个太小不用记」 → 大脑还是会惦记
❌ 边收集边整理 → 打乱收集节奏
❌ 用多个收集篮 → 信息分散
