# 01 - GTD 概述

## 核心原则

> "大脑是用来产生想法的，不是用来存储想法的。" — David Allen

### 三个关键认知

1. **清空大脑**：所有未完成事项必须移出大脑，进入外部系统
2. **下一步行动**：每件事都要明确「具体的下一步物理动作」
3. **定期回顾**：系统需要定期维护，否则会失效

## 五步流程

```
┌─────────────────────────────────────────────┐
│                   收集                       │
│          把所有事项倒入收集篮                │
└──────────────┬──────────────────────────────┘
               ↓
┌─────────────────────────────────────────────┐
│                   理清                       │
│       逐条判断：可执行？下一步是什么？       │
└──────────────┬──────────────────────────────┘
               ↓
┌─────────────────────────────────────────────┐
│                   整理                       │
│      归类到合适的清单：下一步/项目/等待/...  │
└──────────────┬──────────────────────────────┘
               ↓
┌─────────────────────────────────────────────┐
│                   回顾                       │
│     每日检视 + 每周回顾，保持系统更新        │
└──────────────┬──────────────────────────────┘
               ↓
┌─────────────────────────────────────────────┐
│                   执行                       │
│     根据情境、时间、精力选择下一步行动       │
└─────────────────────────────────────────────┘
```

## 心智如水（Mind Like Water）

GTD 的目标状态：大脑像平静的水面，投入石子（新事项）时产生涟漪（响应），然后恢复平静。

```
混乱大脑：💭💭💭💭💭💭💭💭💭💭
→ GTD 系统替代记忆
清晰大脑：😌 + 📋 外部系统
```

## 横向 vs 纵向管理

| | 横向管理 | 纵向管理 |
|---|----------|----------|
| 范围 | 所有事务 | 单个项目 |
| 方法 | 五步流程 | 自然式规划法 |
| 频率 | 持续 | 按需 |

### 自然式规划法（纵向）

```
1. 定义目标和原则
2. 展望结果
3. 头脑风暴
4. 组织整理
5. 明确下一步行动
```

## 适合谁？

✅ 事务繁多、容易焦虑的人
✅ 觉得「有太多事要做」的人
✅ 想要提升个人效率的知识工作者
✅ 需要兼顾工作和生活的多角色人士
