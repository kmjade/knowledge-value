# 04 - GTD 整理（Organize）

## 核心清单系统

```
GTD 需要维护的清单：

1. 下一步行动清单      ← 按情境分类
2. 项目清单            ← 所有活跃项目
3. 等待清单            ← 等待他人完成
4. 将来/也许清单       ← 未来想做
5. 日历                ← 硬性时间
6. 参考资料            ← 存档
```

## 1. 下一步行动清单

> 所有等你去执行的下一步动作。

### 按情境分类

| 情境 | 示例 |
|------|------|
| @电脑 | 写报告、发邮件、整理文件 |
| @手机 | 打电话、回复微信、查资料 |
| @办公室 | 找同事、打印、开会 |
| @家里 | 修灯泡、整理书架 |
| @外出 | 购物、取快递 |
| @任何地方 | 思考、阅读、听播客 |

> 情境把你从「应该做」的焦虑中解放 → 看你现在在哪儿，就做什么。

## 2. 项目清单

```
活跃项目（所有需要 2 个以上步骤完成的事）

□ Q2 销售报告
□ 网站改版
□ 驾照更换
□ 年度体检计划
□ 孩子生日聚会
...
```

**项目必须关联下一步行动**！

```
项目：Q2 销售报告
  ├── 下一步：从系统中导出 Q2 数据
  ├── 下一步：制作图表
  └── 下一步：写分析结论
```

## 3. 等待清单

| 等待谁 | 等待什么 | 日期 |
|--------|----------|------|
| 张总 | 报销审批 | 5/20 |
| IT | 账号开通 | 5/18 |
| 供应商 | 报价单 | 5/22 |

> 每周回顾时检查：该催的催。

## 4. 将来/也许清单

```
想学 → 吉他、日语、陶艺
想去 → 西藏、冰岛
想做 → 写本书、开副业
想买 → 无人机、电子书阅读器
...
```

## 5. 日历

> ⚠️ 日历只放「硬性时间」的事项。

| ✅ 放日历 | ❌ 不放日历 |
|-----------|-------------|
| 会议（固定时间） | 今天要写的报告 |
| 预约 | 想读的书 |
| 截止日 | 要打的电话 |

## 清单管理工具

| 工具 | 优势 |
|------|------|
| Obsidian + Tasks 插件 | 本地、灵活、可关联笔记 |
| Notion 数据库 | 多视图、协作、模板 |
| Todoist | 轻量、跨平台、自然语言 |
| Things (Mac) | 设计精美、GTD 原生 |
| 纸笔 | 零学习、可触摸 |
| Microsoft To Do | 免费、Office 集成 |
