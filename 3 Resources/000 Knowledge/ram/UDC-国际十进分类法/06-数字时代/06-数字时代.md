# 06-UDC 在数字时代

## UDC MRF (Master Reference File)

| 事实 | 说明 |
|------|------|
| 语言 | 57 种——中文版也在其中 |
| 容量 | 60,000+ 类号 |
| 维护 | UDC Consortium (海牙) |
| 更新 | 年度发布——印刷版已成历史 |

## UDC Summary

| URL | udcsummary.info |
|-----|-----------------|
| 用途 | 免费浏览 UDC 结构 |
| 语言 | 57 种——可切换 |
| 层次 | 从大类到细分 |
| API | 可通过 URL 直接查询 (`?id=xxxxx`) |

→ [[../../../06-Applied-Sciences/README|Applied Sciences]] 就基于 `udcsummary.info?id=37275` (UDC 6) 构建。

## UDC Linked Data

| 特性 | 说明 |
|------|------|
| SKOS | 用 W3C 标准表达 UDC |
| 多语言 | 每个类号有 57 种语言标签 |
| 可查询 | SPARQL 端点——"给我所有关于中国医学的类号" |

## 与 DDC 数字化的对比

| 维度 | UDC | DDC |
|------|-----|-----|
| 在线平台 | UDC MRF | WebDewey |
| 免费版本 | UDC Summary | Classify, dewey.info |
| 更新频率 | 年度 | 月度 |
| 多语言 | 57 种 | 30+ 种 |
