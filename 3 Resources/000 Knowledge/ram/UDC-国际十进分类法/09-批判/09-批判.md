# 09-UDC 批判

| 问题 | 说明 |
|------|------|
| **欧洲中心论** | UDC 源于欧洲学术传统——基督教 2 类占 80% |
| **过于复杂** | 辅助符号让 UDC 表达力极强但也极难学会 |
| **维护压力** | 60,000+ 类号，更新慢——DDC 月更，UDC 年更 |
| **数字替代** | 关键词搜索比分类号更直观——很多图书馆弃用 |
| **第 4 类空缺** | 语言类从 4 移到 8 后留下空缺——一直未补 |

## UDC 的辩护

| 优点 | 回应 |
|------|------|
| 表达力 | 复杂 = 精确——UDC 是给专业图书馆的分类语言 |
| 过时 | 持续更新——UDC MRF 包含 AI、区块链等新主题 |
| 小众 | 欧洲+拉美+非洲 30+ 国使用——DDC 并非唯一选择 |
