# 07-分类实践

## 分类步骤

```
1. 确定主题——这本书讲什么？
2. 找主类号——0-9 哪个大类？
3. 细化——找到最具体的类号
4. 加辅助符号——需要语言/地点/时间/形式吗？
5. 排序符号——按引用顺序排列
```

## 实例

### 《中国 21 世纪人工智能教育手册》(英文)

```
1. 主题: 人工智能 (004.8) + 教育 (37)
2. 关系: 004.8:37
3. 地点: (510)
4. 时间: "20"
5. 形式: (035)
6. 语言: =111
→ UDC: 004.8:37(510)"20"(035)=111
```

### 《日本采矿与冶金史》(日文)

```
1. 主题: 采矿(622) + 冶金(669)
2. 并列: 622+669
3. 地点: (520)
4. 形式: (091)
5. 语言: =521
→ UDC: 622+669(520)(091)=521
```

## 常见错误

| 错误 | 正确 |
|------|------|
| 符号顺序随意 | 严格按引用顺序——语言→形式→地点→时间 |
| 缺少空格 | UDC 不使用空格——但要可读 |
| 辅助符号过度使用 | 不是每个文献都需要所有符号——够用即可 |
