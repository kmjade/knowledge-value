# SECI Model (Nonaka, 1995)

| 社会化(S): 隐性→隐性——师徒——观察·模仿——不通过语言 |
|------|
| 外化(E): 隐性→显性——把经验写成手册——最难——「我们知道的多于我们能说的」 |
| 组合(C): 显性→显性——把分散的文档整合——数据库·报告 |
| 内化(I): 显性→隐性——看手册→实践——变成自己的 |
