# 敘詞表 Thesaurus

## 定義
敘詞表是控制詞彙的結構化集合，通過詞間關係（同義·上下位·相關）指引檢索。

## 關係類型
| 關係 | 標記 | 例子 |
|:----|:----|:-----|
| **同義** | USE / UF | 「電腦」USE「計算機」 |
| **上下位** | BT / NT | 哺乳動物 BT 動物 / NT 鯨魚 |
| **相關** | RT | 教育 RT 學習 |

## 主要敘詞表
- MeSH (醫學) — 美國國家醫學圖書館
- LCSH (通用) — 美國國會圖書館
- AGROVOC (農業) — FAO
