# 08-知识论简史

| 柏拉图: 知识=真信念+logos(理性说明) ——洞穴喻 |
|------|
| 笛卡尔: 「我思故我在」——怀疑一切——只有怀疑本身不可怀疑——理性主义的起点 |
| 洛克: 白板——一切来自经验——和笛卡尔对着干 |
| 休谟: 因果只是「恒常连接」——我们看不到「因果力」本身——「理性是激情的奴隶」 |
| 康德: 「哥白尼革命」——不是知识符合对象——是对象符合我们的认知结构 |
| 维特根斯坦: 「不可说的——必须保持沉默」——语言游戏——知识的形式由语言的边界决定 |
| Gettier(1963): 三页纸——推翻了2500年的定义——认识论的新纪元 |
