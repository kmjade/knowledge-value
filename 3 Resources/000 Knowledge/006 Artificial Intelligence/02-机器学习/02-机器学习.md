# 02 — Machine Learning 機器學習

> Machine Learning is the subfield of AI that studies how computers can **learn from data** — automatically improving performance through experience, without explicit programming.
> 機器學習是 AI 的子領域，研究如何讓電腦**從資料中學習**——通過經驗自動改進效能，而不需要顯式程式設計。

---

## 機器學習的三種範式 Three Paradigms

| 範式 Paradigm | 資料要求 Data | 學習方式 Approach | 典型任務 Tasks |
|-------------|-------------|-----------------|--------------|
| **監督學習 Supervised** | 帶標籤的資料 (x → y) | 學習輸入到輸出的映射 | 分類、迴歸 |
| **非監督學習 Unsupervised** | 無標籤的資料 | 發現資料中的隱藏結構 | 聚類、降維、異常檢測 |
| **強化學習 Reinforcement** | 環境反饋（獎勵/懲罰） | 試錯學習最佳策略 | 遊戲、機器人控制 |

---

## 核心概念 Core Concepts

### 訓練流程 Training Pipeline

```
Raw Data → Feature Engineering → Model Training → Evaluation → Deployment
  原始資料      特徵工程              模型訓練          評估            部署
```

| 步驟 | 說明 |
|------|------|
| **特徵工程** | 將原始資料轉化為模型可用的數值特徵 |
| **Train/Val/Test Split** | 訓練集 (70%) / 驗證集 (15%) / 測試集 (15%) |
| **Cross-Validation** | K-fold 交叉驗證，充分利用有限資料 |
| **Hyperparameter Tuning** | 網格搜索、隨機搜索、Bayesian Optimization |

### 偏差-方差權衡 Bias-Variance Tradeoff

| 問題 | 表現 | 解法 |
|------|------|------|
| **High Bias (Underfitting)** | 訓練誤差高，模型過於簡單 | 增加模型複雜度、更多特徵 |
| **High Variance (Overfitting)** | 訓練誤差低但測試誤差高 | 正則化、更多資料、簡化模型 |

## 經典演算法 Classic Algorithms

### 監督學習 Supervised Learning

| 演算法 | 類型 | 核心思想 | 使用場景 |
|--------|------|---------|---------|
| **Linear Regression** | Regression | 擬合最佳線性關係 | 房價預測、趨勢分析 |
| **Logistic Regression** | Classification | 線性分類 + sigmoid 機率輸出 | 二元分類 baseline |
| **Decision Tree** | Both | 樹狀規則分割 | 可解釋性要求高的場景 |
| **Random Forest** | Both | 多棵決策樹的集成 (Bagging) | 結構化資料的首選 |
| **SVM** | Classification | 尋找最大邊界超平面 | 高維資料、小樣本 |
| **XGBoost / LightGBM** | Both | Gradient Boosting，逐步修正殘差 | Kaggle 競賽標配 |
| **k-NN** | Both | 最近鄰居投票 | 小規模基準測試 |

### 非監督學習 Unsupervised Learning

| 演算法 | 核心思想 | 使用場景 |
|--------|---------|---------|
| **K-Means** | 將資料分割為 K 個聚類 | 客戶分群、影像壓縮 |
| **DBSCAN** | 基於密度的聚類，自動發現群數 | 任意形狀的聚類 |
| **PCA** | 線性降維，保留最大方差 | 視覺化、除噪、加速訓練 |
| **t-SNE / UMAP** | 非線性降維 | 高維資料視覺化 |

## 評估指標 Evaluation Metrics

### 分類 Classification

| 指標 | 公式/說明 | 何時使用 |
|------|----------|---------|
| **Accuracy** | (TP+TN) / Total | 類別平衡時 |
| **Precision** | TP / (TP+FP) | 降低誤報重要時（垃圾郵件過濾） |
| **Recall** | TP / (TP+FN) | 降低漏報重要時（疾病檢測） |
| **F1 Score** | 2×P×R/(P+R) | Precision 和 Recall 的調和平均 |
| **ROC-AUC** | ROC 曲線下面積 | 二元分類的整體能力評估 |
| **Confusion Matrix** | 矩陣視覺化 | 了解各類別的錯誤模式 |

### 迴歸 Regression

| 指標 | 說明 |
|------|------|
| **MAE** (Mean Absolute Error) | 平均絕對誤差，對異常值不敏感 |
| **MSE** (Mean Squared Error) | 平均平方誤差，懲罰大誤差 |
| **RMSE** | MSE 的平方根，與目標同單位 |
| **R²** | 解釋方差比例，0–1 之間 |

## 特徵工程 Feature Engineering

| 技術 | 說明 |
|------|------|
| **Normalization** | Min-Max 縮放至 [0,1] |
| **Standardization** | Z-score: (x−μ)/σ |
| **One-Hot Encoding** | 類別變數轉二元向量 |
| **Target Encoding** | 用目標變數的統計量替換類別 |
| **Feature Crossing** | 組合多個特徵創造新特徵 |
