# oh-my-opencode 快速开始

> 安装和配置 oh-my-opencode 的完整指南。

---

## 📥 安装

### 基本安装

```bash
# 使用 npm 安装
npm oh-my-opencode install

# 或使用 bun
bun oh-my-opencode install
```

### 非交互式安装

```bash
npx oh-my-opencode install --no-tui \
  --claude=max20 \
  --gemini=no \
  --copilot=no \
  --skip-auth
```

### 安装选项

| 选项 | 说明 | 可选值 |
|------|------|--------|
| `--claude` | Claude 订阅 | `no`, `yes`, `max20` |
| `--openai` | OpenAI 订阅 | `no`, `yes` |
| `--gemini` | Gemini 集成 | `no`, `yes` |
| `--copilot` | GitHub Copilot | `no`, `yes` |
| `--opencode-zen` | OpenCode Zen | `no`, `yes` |
| `--zai-coding-plan` | Z.ai Coding Plan | `no`, `yes` |
| `--kimi-for-coding` | Kimi For Coding | `no`, `yes` |
| `--skip-auth` | 跳过认证设置 | - |

## ⚙️ 配置

### 配置文件位置

- Windows: `C:\Users\<用户名>\.config\opencode\oh-my-opencode.json`
- macOS/Linux: `~/.config/opencode/oh-my-opencode.json`

### 基础配置示例

```json
{
  "$schema": "https://raw.githubusercontent.com/code-yeongyu/oh-my-opencode/master/assets/oh-my-opencode.schema.json",

      "model": "anthropic/claude-opus-4-6",
```

## 🚀 基本使用

### 启动 OpenCode

```bash
opencode
```

### 使用特定 Agent

```bash
# 使用 Sisyphus 代理（默认）
opencode --agent sisyphus

# 使用 Librarian 代理
opencode --agent librarian

# 使用 Explore 代理
opencode --agent explore

# 使用 Oracle 代理
opencode --agent oracle
```

### 使用技巧

在提示中包含以下关键字：

| 关键字 | 功能 |
|--------|------|
| `ultrawork` / `ulw` | 启用并行代理、后台任务、深度探索 |
| `quick` | 使用快速模型处理简单任务 |
| `ultrabrain` | 使用最强大的模型处理复杂任务 |

---

## 🔧 常用命令

```bash
opencode init                    # 初始化配置
opencode --agent <name>          # 使用指定代理
opencode --model <model>         # 指定模型
opencode --help                  # 查看帮助
```

## ✅ 验证安装

安装完成后，运行以下命令验证：

```bash
opencode --version
```

应该显示类似 `oh-my-opencode v3.x.x` 的版本信息。
