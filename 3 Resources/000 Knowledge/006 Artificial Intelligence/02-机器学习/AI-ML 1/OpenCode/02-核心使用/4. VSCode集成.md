# OpenCode VSCode 集成

> [!info] 概述
> OpenCode 可以与 VSCode 集成，在编辑器中直接使用 AI 辅助编程。

## 安装方式

### 方式一：扩展市场

1. 打开 VSCode
2. 进入扩展市场（Ctrl+Shift+X）
3. 搜索 "OpenCode"
4. 点击安装

### 方式二：命令行安装

```bash
code --install-extension opencode.opencode-vscode
```

---

## 功能特性

### 核心功能

| 功能 | 说明 |
|------|------|
| 内嵌终端 | 在 VSCode 中启动 OpenCode TUI |
| 代码补全 | AI 辅助代码补全 |
| 快捷命令 | 快速执行 OpenCode 命令 |
| 文件上下文 | 自动注入当前文件上下文 |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+P` → "OpenCode" | 打开命令面板 |
| 自定义 | 启动 OpenCode 终端 |

---

## 配置

### settings.json

```json
{
  "opencode.path": "/usr/local/bin/opencode",
  "opencode.model": "claude-sonnet",
  "opencode.terminal.integrated": true
}
```

### 配置选项

| 选项 | 说明 |
|------|------|
| `opencode.path` | OpenCode 可执行文件路径 |
| `opencode.model` | 默认使用的模型 |
| `opencode.terminal.integrated` | 使用集成终端 |

---

## 使用方式

### 启动 OpenCode

1. 打开命令面板（Ctrl+Shift+P）
2. 输入 "OpenCode: Start"
3. OpenCode 终端会在下方打开

### 快速操作

| 操作 | 方式 |
|------|------|
| 分析当前文件 | 右键 → "OpenCode: Analyze" |
| 优化选中代码 | 选中代码 → 右键 → "OpenCode: Optimize" |
| 添加注释 | 右键 → "OpenCode: Add Comments" |

## 工作流示例

### 1. 代码审查

1. 打开文件
2. 选中代码
3. 右键 → "OpenCode: Review"
4. 查看审查结果

### 2. 重构代码

1. 打开文件
2. 选中需要重构的代码
3. 右键 → "OpenCode: Refactor"
4. 选择重构方式

### 3. 生成测试

1. 打开源文件
2. 右键 → "OpenCode: Generate Tests"
3. 选择测试框架
4. 生成测试文件

## 常见问题

### 找不到 OpenCode 命令

1. 确认 OpenCode 已安装
2. 检查 `opencode.path` 配置
3. 重启 VSCode

### 终端无法启动

1. 检查终端配置
2. 确认 OpenCode 在 PATH 中
3. 尝试使用外部终端
