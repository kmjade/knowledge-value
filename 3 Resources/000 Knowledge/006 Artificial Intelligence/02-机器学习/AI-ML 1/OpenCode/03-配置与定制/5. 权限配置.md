# OpenCode 权限配置

> [!info] 概述
> **Tools = OpenCode 的执行能力，而权限 = 安全边界**
> 默认情况下，所有工具都是启用的，生产环境必须配置权限。

## 权限值

| 值 | 说明 |
|----|------|
| `allow` | 允许直接执行 |
| `deny` | 完全禁止 |
| `ask` | 执行前需要人工确认 |

---

## 基础配置

```json
{
  "$schema": "https://opencode.ai/config.json",
  "permission": {
    "edit": "deny",
    "bash": "ask",
    "webfetch": "allow"
  }
}
```

---

## 内置工具权限

### bash - 命令执行

```json
{
  "permission": {
    "bash": "ask"
  }
}
```

**作用：** 执行 shell 命令（如 `npm install`、`git status`）

**风险：** 高（建议设为 `ask`）

### edit - 文件编辑

```json
{
  "permission": {
    "edit": "allow"
  }
}
```

**作用：** 修改现有文件

**说明：** 这是 AI 修改代码的核心能力

### write - 文件创建

**作用：** 创建新文件、覆盖已有文件

**注意：** write 受 edit 权限控制

### read - 文件读取

```json
{
  "permission": {
    "read": "allow"
  }
}
```

**作用：** 读取代码文件

**风险：** 低（建议 always allow）

### grep - 内容搜索

```json
{
  "permission": {
    "grep": "allow"
  }
}
```

**作用：** 正则搜索代码

### glob - 文件匹配

**作用：** 查找文件路径（如 `src/**/*.ts`）

### webfetch - 网络请求

```json
{
  "permission": {
    "webfetch": "allow"
  }
}
```

**作用：** 访问网络资源

---

## 通配符配置

批量控制来自特定来源的工具：

```json
{
  "permission": {
    "mymcp_*": "ask"
  }
}
```

所有来自 `mymcp` 的 MCP 工具都需要审批。

---

## 推荐配置

### 开发环境

```json
{
  "permission": {
    "bash": "ask",
    "edit": "allow",
    "write": "allow",
    "read": "allow",
    "grep": "allow",
    "glob": "allow",
    "webfetch": "allow"
  }
}
```

### 生产环境

```json
{
  "permission": {
    "bash": "deny",
    "edit": "ask",
    "write": "ask",
    "read": "allow",
    "grep": "allow",
    "glob": "allow",
    "webfetch": "ask"
  }
}
```

### 只读模式

```json
{
  "permission": {
    "bash": "deny",
    "edit": "deny",
    "write": "deny",
    "read": "allow",
    "grep": "allow",
    "glob": "allow"
  }
}
```

## 工具权限速查表

| 工具 | 功能 | 风险 | 推荐设置 |
|------|------|------|----------|
| `bash` | 执行命令 | 高 | `ask` |
| `edit` | 编辑文件 | 中 | `allow`/`ask` |
| `write` | 创建文件 | 中 | `allow`/`ask` |
| `read` | 读取文件 | 低 | `allow` |
| `grep` | 搜索内容 | 低 | `allow` |
| `glob` | 查找文件 | 低 | `allow` |
| `webfetch` | 网络请求 | 中 | `ask` |

---

```json
{
  "permission": { ... }
}
```
