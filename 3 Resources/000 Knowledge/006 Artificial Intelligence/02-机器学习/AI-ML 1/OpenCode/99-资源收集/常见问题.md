# OpenCode 常见问题

## 安装问题

### 命令找不到

**问题**：运行 `opencode` 提示命令不存在。

**解决方案**：
1. 确认安装成功
2. 检查 PATH 环境变量
3. 重新打开终端

### 权限被拒绝

**问题**：`EACCES: permission denied`

**解决方案**：
```bash
sudo chown -R $(whoami) ~/.local
chmod -R 755 ~/.local
```

或使用 sudo：
```bash
sudo opencode
```

### 网络超时

**问题**：安装时网络超时。

**解决方案**：
- 使用代理
- 切换网络
- 手动下载二进制文件

## 配置问题

### API Key 无效

**问题**：API Key 配置后报错。

**解决方案**：
1. 检查 Key 是否正确
2. 确认 Key 未过期
3. 检查账户余额

### 模型不可用

**问题**：选择的模型不可用。

**解决方案**：
1. 检查 API 权限
2. 确认模型名称正确
3. 使用 `/models` 查看可用模型

---

## 使用问题

### 响应慢

**问题**：AI 响应很慢。

**解决方案**：
1. 使用更快的模型（如 Haiku）
2. 减少上下文大小
3. 使用 `/compact` 压缩上下文

### 上下文溢出

**问题**：上下文超过模型限制。

**解决方案**：
```bash
/compact  # 压缩上下文
/new      # 新建会话
```

### 文件引用失败

**问题**：`@file` 找不到文件。

**解决方案**：
1. 检查文件路径
2. 使用模糊搜索
3. 确认文件存在

## 工具问题

### bash 命令失败

**问题**：执行 bash 命令报错。

**解决方案**：
1. 检查命令语法
2. 确认权限设置
3. 查看错误日志

### 文件编辑失败

**问题**：edit 工具无法修改文件。

**解决方案**：
1. 检查权限配置
2. 确认文件可写
3. 检查文件编码

---

## MCP 问题

### MCP 连接失败

**问题**：MCP 服务器无法连接。

**解决方案**：
1. 检查 MCP 命令是否正确
2. 确认 MCP 服务器已安装
3. 查看调试日志：
```bash
opencode mcp debug
```

### MCP 工具不可用

**问题**：MCP 工具不显示。

**解决方案**：
1. 检查 `enabled: true`
2. 重启 OpenCode
3. 检查权限配置

---

## 性能问题

### 内存占用高

**问题**：OpenCode 占用大量内存。

**解决方案**：
1. 减少上下文大小
2. 关闭不需要的 MCP
3. 重启 OpenCode

### CPU 占用高

**问题**：CPU 使用率持续很高。

**解决方案**：
1. 检查后台任务
2. 减少并发请求
3. 更新到最新版本

## 其他问题

### 如何更新

```bash
# npm 安装的
npm update -g opencode-ai

# Homebrew 安装的
brew upgrade opencode

# 一键安装的
curl -fsSL https://opencode.ai/install | bash
```

### 如何卸载

```bash
# npm 安装的
npm uninstall -g opencode-ai

# Homebrew 安装的
brew uninstall opencode
```

### 如何重置配置

```bash
# 删除配置目录
rm -rf ~/.config/opencode
rm -rf ~/.local/share/opencode
```
