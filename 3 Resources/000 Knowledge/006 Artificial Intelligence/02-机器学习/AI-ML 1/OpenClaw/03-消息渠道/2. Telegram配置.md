# OpenClaw Telegram 配置

> [!info] 概述
> Telegram 是最容易配置的渠道，推荐作为第一个集成的平台。

## 创建 Telegram Bot

### 步骤 1：找到 BotFather

1. 打开 Telegram
2. 搜索 `@BotFather`
3. 发送 `/newbot` 命令

### 步骤 2：设置 Bot 信息

```
/newbot

BotFather: Alright, a new bot. How are we going to call it?
你: MyOpenClawBot

BotFather: Good. Now let's choose a username for your bot.
你: my_openclaw_bot

BotFather: Done! Here's your token:
123456789:ABCdefGHIjklMNOpqrsTUVwxyz
```

### 步骤 3：保存 Token

记录下 Bot Token，格式为：`数字:字母数字混合`

---

## 配置 OpenClaw

### 命令行配置

```bash
openclaw channel add telegram --token 123456789:ABCdefGHIjklMNOpqrsTUVwxyz
```

### 配置文件

```json
{
  "channels": {
    "telegram": {
      "token": "123456789:ABCdefGHIjklMNOpqrsTUVwxyz",
      "enabled": true
    }
  }
}
```

---

## 高级配置

### 权限控制

```json
{
  "channels": {
    "telegram": {
      "token": "your-token",
      "allowFrom": [123456789, 987654321],
      "groups": {
        "*": { "requireMention": true },
        "-1001234567890": { "requireMention": false }
      }
    }
  }
}
```

| 配置项 | 类型 | 说明 |
|--------|------|------|
| `allowFrom` | 数字数组 | 允许的用户 ID |
| `groups` | 对象 | 群组权限配置 |
| `requireMention` | 布尔 | 是否需要 @ 机器人 |

### 获取用户 ID

1. 在 Telegram 中搜索 `@userinfobot`
2. 发送任意消息
3. 它会返回你的 User ID

## Bot 命令

### 设置 Bot 命令

发送给 @BotFather：

```
/setcommands

start - 开始对话
help - 获取帮助
reset - 重置会话
```

### OpenClaw 内置命令

| 命令 | 说明 |
|------|------|
| `/start` | 开始与 AI 对话 |
| `/help` | 显示帮助信息 |
| `/reset` | 重置当前会话 |
| `/model` | 切换模型 |
| `/agent` | 切换智能体 |

## 群组使用

### 添加 Bot 到群组

1. 在群组设置中添加 Bot
2. 给 Bot 管理员权限（可选）

### 群组配置

```json
{
  "channels": {
    "telegram": {
      "groups": {
        "*": {
          "requireMention": true,
          "allowCommands": ["/help", "/reset"]
        }
      }
    }
  }
}
```

### 消息提及

在群组中，需要 @ 你的 Bot 才会触发回复：

```
@my_openclaw_bot 你好，请帮我写一段代码
```

## Webhook 模式（可选）

### 配置 Webhook

```bash
openclaw channel set telegram --webhook https://your-domain.com/webhook/telegram
```

### 自签名证书

```bash
openclaw channel set telegram \
  --webhook https://your-domain.com/webhook/telegram \
  --cert /path/to/cert.pem
```

## 常见问题

### Bot 无响应

1. 检查 Token 是否正确
2. 确认 Bot 未被封禁
3. 查看 Gateway 日志
4. 检查网络连接

### 群组不回复

1. 确认已添加到群组
2. 检查 `requireMention` 配置
3. 尝试 @ Bot 触发

### 权限问题

1. 确认用户 ID 在 `allowFrom` 列表中
2. 检查 Bot 是否有群组消息权限

openclaw gateway logs --channel telegram

  -d "text=测试消息"
