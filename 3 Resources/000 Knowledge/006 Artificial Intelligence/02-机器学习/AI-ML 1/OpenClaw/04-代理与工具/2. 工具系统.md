# OpenClaw 工具系统

> [!info] 概述
> 工具系统让 AI 能够执行实际操作，如搜索网页、运行代码、管理文件等。

## 内置工具

### 核心工具

| 工具 | 说明 | 状态 |
|------|------|------|
| `web_search` | 网页搜索 | ✅ |
| `code_interpreter` | 代码执行 | ✅ |
| `file_manager` | 文件管理 | ✅ |
| `http_client` | HTTP 请求 | ✅ |
| `calculator` | 计算器 | ✅ |

### 扩展工具

| 工具 | 说明 | 状态 |
|------|------|------|
| `image_generator` | 图片生成 | ⚠️ 需配置 |
| `database` | 数据库操作 | ⚠️ 需配置 |
| `shell` | Shell 命令 | ⚠️ 受限 |

---

## 启用工具

### 配置文件

```json
{
  "agents": {
    "pi": {
      "tools": ["web_search", "code_interpreter"]
    }
  }
}
```

### 命令行

```bash
# 启用工具
openclaw tool enable web_search --agent pi

# 禁用工具
openclaw tool disable web_search --agent pi
```

## 工具配置

### Web Search

```json
{
  "tools": {
    "web_search": {
      "provider": "google",
      "apiKey": "your-api-key",
      "searchEngineId": "your-engine-id"
    }
  }
}
```

### Code Interpreter

```json
{
  "tools": {
    "code_interpreter": {
      "enabled": true,
      "timeout": 30000,
      "languages": ["python", "javascript", "bash"]
    }
  }
}
```

### File Manager

```json
{
  "tools": {
    "file_manager": {
      "enabled": true,
      "allowedPaths": [
        "/home/user/documents",
        "/home/user/projects"
      ],
      "maxFileSize": 10485760
    }
  }
}
```

## 自定义工具

### 工具定义

```json
{
  "tools": {
    "my_custom_tool": {
      "name": "my_custom_tool",
      "description": "自定义工具描述",
      "parameters": {
        "type": "object",
        "properties": {
          "input": {
            "type": "string",
            "description": "输入参数"
          }
        },
        "required": ["input"]
      },
      "handler": "/path/to/handler.js"
    }
  }
}
```

### Handler 示例

```javascript
// handler.js
module.exports = async function(params, context) {
  const { input } = params;

  // 处理逻辑
  const result = `处理结果: ${input}`;

  return {
    success: true,
    result: result
  };
};
```

## 工具权限

### 权限级别

| 级别 | 说明 |
|------|------|
| `full` | 完全访问 |
| `limited` | 限制访问 |
| `readonly` | 只读访问 |
| `disabled` | 禁用 |

### 权限配置

```json
{
  "tools": {
    "file_manager": {
      "permission": "limited",
      "allowedPaths": ["/home/user/documents"]
    },
    "shell": {
      "permission": "disabled"
    }
  }
}
```

## 工具调用流程

### 调用流程

```
用户消息 → AI 分析 → 决定使用工具 → 执行工具 → 返回结果 → AI 生成回复
```

### 示例对话

```
用户: 帮我搜索一下最新的 AI 新闻
AI: [调用 web_search 工具]
工具: 返回搜索结果
AI: 根据搜索结果，最新的 AI 新闻如下...
```

## 安全考虑

### 工具安全

> [!warning] 重要
> 启用工具前，请考虑安全影响：

1. **Shell 工具** - 可执行任意命令，谨慎启用
2. **File Manager** - 限制可访问路径
3. **HTTP Client** - 可能泄露敏感信息

### 最佳实践

- 只启用必要的工具
- 配置严格的权限
- 定期审计工具调用日志
- 使用沙箱环境测试

## MCP 工具

OpenClaw 支持 MCP (Model Context Protocol) 工具：

### 添加 MCP 服务器

```bash
openclaw mcp add filesystem --config '{"path": "/home/user/documents"}'
```

### 查看 MCP 工具

```bash
openclaw mcp list
```
