# 03-深度学习

> 深度学习是机器学习的一个子集，使用多层（深度）人工神经网络自动学习数据的层次化表示——从低级特征构建到高级抽象。

## 为什么深度学习？

传统 ML 需要**人工设计特征**。深度学习**自动学习特征**。

| 方法 | 特征工程 | 适用场景 |
|------|----------|----------|
| 传统 ML | 人工设计和提取 | 结构化表格数据 |
| 深度学习 | 自动学习层次化特征 | 图像、文本、音频等非结构化数据 |

## 神经网络基础

### 感知机 → 多层感知机 (MLP)

```
输入层     隐藏层     输出层
  x₁ ─→ ○ ─→ ○ ─→ y₁
  x₂ ─→ ○ ─→ ○ ─→ y₂
  x₃ ─→ ○ ─→ ○
```

每个神经元：
1. 计算加权和：z = Σ(wᵢxᵢ) + b
2. 通过激活函数：a = f(z)

### 激活函数

| 函数 | 公式 | 特点 | 使用场景 |
|------|------|------|----------|
| **Sigmoid** | 1/(1+e⁻ˣ) | 输出 (0,1)，有梯度消失问题 | 二分类输出层 |
| **Tanh** | (eˣ-e⁻ˣ)/(eˣ+e⁻ˣ) | 输出 (-1,1)，零中心 | 曾经流行，现已少用 |
| **ReLU** | max(0, x) | 计算简单、缓解梯度消失 | 隐藏层默认选择 |
| **GELU** | x·Φ(x) | 平滑版 ReLU | Transformer 标准 |
| **Softmax** | eˣᵢ/Σeˣⱼ | 输出概率分布 | 多分类输出层 |

### 反向传播 (Backpropagation)

训练神经网络的核心算法：

```
前向传播：输入 → 网络 → 预测 → 计算损失
                                      ↓
反向传播：梯度 ← 损失对参数的偏导 ← 链式法则
                                      ↓
参数更新：w = w - η × 梯度  (η = 学习率)
```

## 三大经典架构

### CNN（卷积神经网络）— 处理网格数据

| 组件 | 作用 |
|------|------|
| **卷积层** | 用可学习的滤波器扫描输入，检测局部模式（边缘、纹理等） |
| **池化层** | 下采样，减少计算量、增加平移不变性 |
| **全连接层** | 最后几层做分类/回归 |

经典架构演进：
```
LeNet (1998) → AlexNet (2012) → VGG (2014) → ResNet (2015) → EfficientNet (2019)
```

**ResNet 的关键创新**：跳跃连接（Skip Connection），解决了深层网络梯度消失问题，使 100+ 层的网络可训练。

### RNN / LSTM / GRU — 处理序列数据

| 架构 | 特点 | 问题 |
|------|------|------|
| **RNN** | 循环处理序列，有"记忆" | 梯度消失/爆炸，难以处理长序列 |
| **LSTM** | 门控机制（遗忘门/输入门/输出门） | 解决了长程依赖，但计算重 |
| **GRU** | 简化版 LSTM（两个门） | 参数更少，通常与 LSTM 效果相当 |

### Transformer — 现代标准

2017 年论文 "Attention Is All You Need" 彻底改变了深度学习。

| 组件 | 作用 |
|------|------|
| **自注意力 (Self-Attention)** | 每个位置关注序列中所有其他位置，捕获全局依赖 |
| **多头注意力 (Multi-Head Attention)** | 并行做多次注意力，从不同"角度"看数据 |
| **位置编码** | 注入序列顺序信息（Transformer 本身是位置无关的） |
| **前馈网络 (FFN)** | 每个位置的独立非线性变换 |
| **层归一化 + 残差连接** | 稳定训练 |

> Transformer 的优势：并行计算（不像 RNN 顺序处理）、捕获任意距离的依赖关系、可扩展到超大规模。

## 训练技巧

### 优化器

| 优化器 | 特点 |
|--------|------|
| SGD + Momentum | 经典，需要仔细调参 |
| Adam | 自适应学习率，默认首选 |
| AdamW | Adam + 解耦权重衰减，Transformer 训练常用 |

### 正则化

| 方法 | 作用 |
|------|------|
| **Dropout** | 随机丢弃神经元，防止共适应 |
| **Weight Decay** | L2 正则化，限制权重不要太大 |
| **Batch Normalization** | 标准化每层输入，稳定和加速训练 |
| **Layer Normalization** | 类似 BN，但不依赖 batch size（Transformer 用这个） |
| **数据增强** | 扩充训练数据（图像旋转/翻转、文本回译等） |

### 学习率调度

| 策略 | 说明 |
|------|------|
| **阶梯衰减** | 每隔 N 步学习率乘以系数 |
| **余弦退火** | 余弦曲线平滑降低 |
| **Warmup** | 先从小学习率线性增加到目标值（Transformer 训练必备） |

## 迁移学习与微调

现代 DL 的常见工作流：不从头训练，而是：

1. 取预训练模型（如 ImageNet 上的 ResNet / 大规模语料上的 BERT）
2. **微调 (Fine-tune)**：在自己的任务数据上继续训练

| 方法 | 说明 |
|------|------|
| **全量微调** | 更新所有参数（需要较多数据和计算） |
| **冻结特征提取** | 冻结预训练层，只训练最后的新分类层 |
| **LoRA** | 只训练低秩矩阵增量，大幅减少可训练参数（LLM 微调常用） |
