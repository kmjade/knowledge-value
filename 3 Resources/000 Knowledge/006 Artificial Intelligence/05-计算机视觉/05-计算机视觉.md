# 05 — Computer Vision 電腦視覺

> Computer Vision enables machines to see, understand, and reason about visual information. From edge detection to diffusion models — the evolution of machine perception.
> 電腦視覺讓機器能夠看見、理解並推理視覺資訊。從邊緣檢測到擴散模型——機器感知的演進。

---

## CV 的核心任務 Core CV Tasks

| 任務 Task | 說明 Description | 輸入→輸出 |
|-----------|-----------------|----------|
| **Image Classification** | 判斷整張圖片的類別 | Image → Class label |
| **Object Detection** | 定位並分類圖中的物體 | Image → Bounding boxes + labels |
| **Semantic Segmentation** | 每個像素的分類 | Image → Pixel-wise class map |
| **Instance Segmentation** | 區分同類別的不同實例 | Image → Instance masks |
| **Image Generation** | 從文字/噪聲生成新圖片 | Text/Noise → Image |
| **Object Tracking** | 在影片中追蹤物體 | Video → Tracks |
| **Pose Estimation** | 估計人體/物體的姿態 | Image → Keypoints |

## CNN 骨幹網路演進 CNN Backbone Evolution

| 模型 | 年份 | 核心創新 | 參數 |
|------|------|---------|------|
| **LeNet-5** | 1998 | 第一個成功的 CNN（手寫數字） | 60K |
| **AlexNet** | 2012 | ImageNet 冠軍，GPU + ReLU + Dropout | 60M |
| **VGG** | 2014 | 深層堆疊 3×3 Conv，簡單但參數多 | 138M |
| **GoogLeNet (Inception)** | 2014 | 多尺度並行卷積 (Inception module) | 6.8M |
| **ResNet** | 2015 | 殘差連接 (skip connection)，可訓練超深網路 (152 層) | 25–60M |
| **DenseNet** | 2017 | 每一層連接到所有後續層 | 8–40M |
| **EfficientNet** | 2019 | Compound scaling，同時調深度/寬度/解析度 | 5–66M |
| **ConvNeXt** | 2022 | "Modernized CNN"，結合 Transformer 設計理念 | 29–198M |

## 目標檢測 Object Detection

### 兩階段 Two-Stage（精確但較慢）

| 模型 | 核心思想 |
|------|---------|
| **R-CNN** | Selective Search 提取區域 → CNN 分類 |
| **Fast R-CNN** | 共享卷積特徵 + RoI Pooling |
| **Faster R-CNN** | RPN (Region Proposal Network)，端到端訓練 |

### 單階段 One-Stage（快速）

| 模型 | 特點 |
|------|------|
| **YOLO** (v1→v10) | 將檢測轉為迴歸問題，網格預測 + 邊界框 |
| **SSD** | 多尺度特徵圖預測 |
| **RetinaNet** | Focal Loss 解決類別不平衡 |

### 檢測指標 Detection Metrics

| 指標 | 說明 |
|------|------|
| **IoU** (Intersection over Union) | 預測框與真實框的重疊度 |
| **mAP** (mean Average Precision) | 多類別、多 IoU 閾值下的平均精度 |

## Vision Transformer (ViT)

傳統 CV 使用 CNN，2020 年 ViT 證明純 Transformer 也能做到 SOTA。

```
Image → Patches (16×16) → Linear Projection → Transformer Encoder → MLP Head → Class
```

| vs CNN | CNN | ViT |
|--------|-----|-----|
| **歸納偏置 (Inductive Bias)** | 局部性、平移等變性（內建） | 需從資料中學習 |
| **資料需求** | 較少 | 大量（除非預訓練） |
| **全局視野** | 需要深層才能看到全局 | 第一層就是全局 self-attention |

### ViT 變體

| 變體 | 特點 |
|------|------|
| **Swin Transformer** | 分層 + 滑動窗口 attention |
| **DINO / DINOv2** | 自監督學習，優秀的視覺特徵 |
| **CLIP** | 文字-影像對比學習，zero-shot 分類 |

## 影像生成 Image Generation

### GAN (Generative Adversarial Network)

```
Generator ──→ Fake Image ──→ Discriminator ──→ Real/Fake
    │                              │
    └──── 對抗訓練 (Adversarial) ────┘
```

### Diffusion Models 擴散模型

```
Forward: Image → (add noise step by step) → Pure Noise
Reverse: Pure Noise → (denoise step by step) → Image
```

| 模型 | 特點 |
|------|------|
| **Stable Diffusion** | 在 latent space 中 diffusion，高效 |
| **DALL-E 3** | 文字理解力強，跟隨複雜 prompt |
| **Midjourney** | 美學品質最高的生成工具 |
| **Flux** | 開源，高品質，由 Stable Diffusion 原團隊開發 |

## CV 的現代趨勢 Modern CV Trends

| 趨勢 | 說明 |
|------|------|
| **Multi-modal** | 圖像 + 文字 + 音頻（GPT-4V, Gemini） |
| **Self-supervised Learning** | 無需標籤的預訓練（MAE, DINO, SimCLR） |
| **3D Vision** | NeRF, Gaussian Splatting |
| **Video Understanding** | 影片生成 (Sora)、動作識別 |
| **Edge CV** | 手機、IoT 上的輕量級視覺模型 |
