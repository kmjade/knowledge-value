# 多工具智能体

> [!summary] 概述
> Hermes Agent 内置 20+ 工具集，可以自然地在单个任务中组合使用多个工具。

---

## 示例：文件分析 + 网络搜索

```bash
hermes chat -q "读取 README.md，如果内容提到某个技术术语我
不理解的，帮我在网上搜索解释"
```

Hermes 会自动：
1. 使用 `read_file` 读取 README.md
2. 识别不熟悉的术语
3. 使用 `web_search` 搜索解释

---

## 示例：代码审查 + 修复

在交互式会话中：

```
用户: 审查 src/utils.py 的代码质量，修复发现的问题
```

Hermes 会：
1. `read_file` 读取文件
2. 分析代码质量
3. `patch` 修复问题
4. 如果需要，`terminal` 运行测试验证

---

## 工具组合模式

| 场景 | 工具组合 |
|------|---------|
| 代码审查 | read_file → patch → terminal(test) |
| 研究总结 | web_search → write_file |
| 部署 | terminal(build) → terminal(deploy) → web_fetch(health) |
| 数据分析 | read_file(csv) → execute_code → write_file(report) |
