# MCP 协议集成

> [!info] 
> Hermes Agent 原生支持 MCP (Model Context Protocol)。

```bash
hermes mcp add NAME --url ...     # 添加
hermes mcp list                   # 列出
hermes mcp serve                  # 作为 MCP 服务运行
```

参考：[MCP 文档](https://hermes-agent.nousresearch.com/docs/user-guide/features/mcp)
