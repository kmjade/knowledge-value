# 自定义工具开发

> [!info] 待编写
> 参考：[Contributor Quick Reference](https://hermes-agent.nousresearch.com/docs/developer-guide/)

在 `tools/` 目录下创建新文件，使用 `registry.register()` 注册：

```python
from tools.registry import registry

registry.register(
    name="my_tool",
    toolset="custom",
    schema={...},
    handler=my_handler,
)
```
