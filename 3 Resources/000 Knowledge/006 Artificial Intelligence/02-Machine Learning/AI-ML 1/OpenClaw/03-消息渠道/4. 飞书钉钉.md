# OpenClaw 飞书钉钉配置

> [!info] 概述
> 飞书和钉钉是国内企业常用的协作平台，OpenClaw 支持通过官方 API 集成。

---

## 飞书配置

### 创建飞书应用

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 进入「开发者后台」
3. 点击「创建企业自建应用」
4. 填写应用名称和描述

### 获取凭证

在应用详情页获取：
- **App ID**
- **App Secret**

### 配置权限

| 权限 | 用途 |
|------|------|
| `im:message` | 接收和发送消息 |
| `im:message:send_as_bot` | 以机器人身份发消息 |
| `contact:user.base:readonly` | 获取用户信息 |

### 配置事件订阅

1. 在「事件订阅」页面启用
2. 配置请求网址：
   ```
   https://your-gateway.com/webhook/feishu
   ```
3. 添加事件：`im.message.receive_v1`

### OpenClaw 配置

```json
{
  "channels": {
    "feishu": {
      "appId": "cli_xxxxxxxxxx",
      "appSecret": "xxxxxxxxxxxxxxxxxxxxxxxx",
      "enabled": true
    }
  }
}
```

### 命令行配置

```bash
openclaw channel add feishu \
  --app-id cli_xxxxxxxxxx \
  --app-secret xxxxxxxxxxxxxxxxxxxxxxxx
```

## 钉钉配置

### 创建钉钉应用

1. 访问 [钉钉开放平台](https://open.dingtalk.com/)
2. 进入「应用开发」
3. 创建「企业内部应用」
4. 选择「H5微应用」

### 获取凭证

- **Client ID**
- **Client Secret**

### 配置机器人

1. 在应用详情页找到「机器人与消息推送」
2. 启用机器人功能
3. 配置消息接收地址：
   ```
   https://your-gateway.com/webhook/dingtalk
   ```

### 权限配置

| 权限 | 用途 |
|------|------|
| `qyapi_message_corp_send` | 发送企业消息 |
| `contact_user.base_read` | 读取用户信息 |

### OpenClaw 配置

```json
{
  "channels": {
    "dingtalk": {
      "clientId": "dingxxxxxxxx",
      "clientSecret": "xxxxxxxxxxxxxxxx",
      "enabled": true
    }
  }
}
```

### 命令行配置

```bash
openclaw channel add dingtalk \
  --client-id dingxxxxxxxx \
  --client-secret xxxxxxxxxxxxxxxx
```

## 消息格式

### 飞书消息

```json
{
  "msg_type": "text",
  "content": {
    "text": "你好，我是 AI 助手"
  }
}
```

支持的消息类型：
- text（文本）
- post（富文本）
- image（图片）
- interactive（消息卡片）

### 钉钉消息

```json
{
  "msgtype": "text",
  "text": {
    "content": "你好，我是 AI 助手"
  }
}
```

支持的消息类型：
- text（文本）
- markdown（Markdown）
- link（链接）
- actionCard（卡片）

## 群聊配置

### 飞书群聊

```json
{
  "channels": {
    "feishu": {
      "groups": {
        "*": { "requireMention": true }
      }
    }
  }
}
```

### 钉钉群聊

将机器人添加到群聊后：

```json
{
  "channels": {
    "dingtalk": {
      "groups": {
        "*": { "requireMention": true }
      }
    }
  }
}
```

## 安全配置

### IP 白名单

在飞书/钉钉后台配置 Gateway 服务器 IP 白名单。

### 消息加密

OpenClaw 自动处理飞书/钉钉的消息加密/解密。

### 飞书测试

1. 在飞书中搜索你的机器人
2. 发送消息
3. 检查是否收到回复

### 钉钉测试

1. 在钉钉中找到你的机器人
2. 发送消息
3. 检查是否收到回复

---

## 常见问题

### 收不到消息

1. 检查事件订阅地址是否正确
2. 确认权限配置完整
3. 查看 Gateway 日志

### 发送失败

1. 检查 App Secret 是否正确
2. 确认有发送权限
3. 检查消息格式

### 群聊不响应

1. 确认机器人已加入群聊
2. 检查 `requireMention` 配置
3. 尝试 @ 机器人触发
