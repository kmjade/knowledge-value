# Skills Files

Claude Code 的配置文件系統，讓 AI 理解用戶的業務環境和習慣。

---

## 核心概念

- **Markdown 格式**：用 .md 文件定義技能
- **業務沉澱**：積累業務經驗和最佳實踐
- **個性化**：讓 AI 更貼合用戶需求

---

## 文件結構

```
~/.claude/skills/
  skill-name/
    SKILL.md
```

---

## 應用場景

| 場景 | 說明 |
|------|------|
| 競品分析 Skill | 結構化競品分析流程 |
| 自動化工作流 | 沉澱工作流搭建經驗 |
| 領域知識 | 注入專業領域知識 |

---

## 與兩層自動化的關係

在 [[two-layer-automation]] 中：
- Skills 文件是「業務理解」的載體
- 讓 AI 搭建的工作流更貼合實際需求
