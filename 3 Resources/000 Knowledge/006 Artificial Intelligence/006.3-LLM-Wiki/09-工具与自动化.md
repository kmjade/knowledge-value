# 09 — 工具與自動化

> LLM-Wiki 的工具鏈與自動化體系：從核心指令 `/wiki-compile` 到定時編譯、vault 健康檢查，實現知識庫的無人值守維護。

---

## 一、核心指令：/wiki-compile

### 指令語法

```bash
/wiki-compile <domain> [options]
```

### 選項

| 選項 | 說明 | 範例 |
|------|------|------|
| `<domain>` | 目標域標識 | `006`, `000`, `all` |
| `--incr` | 增量模式（僅處理 compiled: false） | `/wiki-compile 006 --incr` |
| `--full` | 全量模式（重建所有頁面） | `/wiki-compile 006 --full` |
| `--target <kb>` | 定向編譯特定子庫 | `/wiki-compile 006 --target 006.3` |
| `--dry-run` | 預覽模式，不實際寫入 | `/wiki-compile 006 --dry-run` |
| `--lint` | 編譯後自動執行 Lint | `/wiki-compile 006 --lint` |
| `--force` | 跳過確認，強制執行 | `/wiki-compile 006 --full --force` |

### 執行流程

```
/wiki-compile 006 --incr --lint
        │
        ├── 1. 讀取 CLAUDE.md (006 域 Schema)
        ├── 2. 掃描 raw/ 中 compiled: false 的文件
        ├── 3. 調用 LLM 提取概念/實體/關係
        ├── 4. 生成 wiki/ 頁面 (concepts/ entities/ sources/)
        ├── 5. 建立 wikilinks 網絡
        ├── 6. 更新 index.md + log.md
        ├── 7. 標記 compiled: true
        ├── 8. 執行 /lint 006
        └── 9. 輸出報告
```

---

## 二、輔助指令

### /wiki-query

```bash
/wiki-query "<question>" --kb 006.3
```

從指定知識庫中查詢並返回結構化答案。

### /wiki-lint

```bash
/wiki-lint <domain> [--fix]
```

| 選項 | 說明 |
|------|------|
| `--fix` | 自動修復可修復問題（如補全 frontmatter） |
| `--output <file>` | 將報告寫入文件 |

### /wiki-stats

```bash
/wiki-stats <domain>
```

輸出知識庫統計資訊：

```
Domain: 006 AI
─────────────────
Concepts:  11    Entities:  6
Sources:   1     Total:    18
Wikilinks: 87    Density:  4.8/page
Last compile: 2026-06-01
Health: 🟢 Healthy
```

### /wiki-graph

```bash
/wiki-graph <domain> [--depth 2]
```

生成知識圖譜數據（JSON 格式），可導入 Obsidian Graph 或其他可視化工具。

## 三、自動化腳本

### 定時編譯 Cron Job

```bash
# crontab — 每天凌晨 3:00 增量編譯
0 3 * * * cd /path/to/vault && /wiki-compile all --incr --lint >> /wiki-cron.log 2>&1
```

### Git 自動提交

```bash
#!/bin/bash
# auto-commit.sh — 編譯後自動 commit
/wiki-compile all --incr --lint
if [ $? -eq 0 ]; then
  git add wiki/
  git commit -m "wiki-compile: auto $(date +%Y-%m-%d)"
  git push
fi
```

### Vault 健康檢查

```bash
#!/bin/bash
# vault-health.sh — 全 vault 健康巡檢
echo "=== Vault Health Check: $(date) ==="
for domain in 000 004 005 006; do
  echo "--- Domain: $domain ---"
  /wiki-lint $domain --output "/tmp/lint-$domain.md"
  /wiki-stats $domain
done
echo "=== Done ==="
```

## 四、自動化策略

| 策略 | 觸發條件 | 操作 |
|------|---------|------|
| **即時編譯** | 新 raw 文件加入 | `/wiki-compile <domain> --incr` |
| **定時編譯** | Cron 排程（每日/每週） | `/wiki-compile all --incr --lint` |
| **事件觸發** | Git hook（pre-commit） | 檢查 frontmatter 完整性 |
| **手動觸發** | 重大結構變更 | `/wiki-compile <domain> --full` |
