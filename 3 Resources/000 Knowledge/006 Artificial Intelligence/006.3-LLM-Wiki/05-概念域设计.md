# 05 — 概念域設計

> 概念域（Concept Domain）是 LLM-Wiki 的知識組織核心——如何劃分域、設定概念粒度、選擇實體、建立溯源鏈，直接決定知識庫的可用性。

---

## 一、域劃分原則 Domain Partitioning

### 劃分標準

| 原則 | 說明 | 反例 |
|------|------|------|
| **主題內聚** | 同域概念圍繞同一主題 | 把「LLM」和「園藝」放在同一域 |
| **認知負載** | 每域 6–15 個概念為宜 | 單域 30+ 概念導致導航困難 |
| **邊界清晰** | 域的邊界應容易判斷 | 概念可同時歸入多個域 |
| **層級一致** | 同域概念抽象層級相近 | 「AI」與「Backpropagation」同域 |
| **DDC 對齊** | 域劃分應對齊 DDC 分類體系 | 域與 DDC 碼無關聯 |

### 域劃分決策樹

```
新概念歸屬判定：
  ┌─ 是否已有合適的域？ ──→ 是 ──→ 加入該域
  │
  └─ 否
      │
      ├─ 是否與現有域有強關聯？ ──→ 是 ──→ 作為子概念加入
      │
      └─ 否
          │
          └─ 是否可形成 ≥3 個概念的新域？ ──→ 是 ──→ 創建新域
              │
              └─ 否 ──→ 放入「其他」或等待更多概念
```

## 二、概念粒度標準 Concept Granularity

### 粒度層級

| 層級 | 說明 | 範例 | 每域數量 |
|------|------|------|----------|
| **L1: 領域級** | 整個學科/領域 | Artificial Intelligence | 0–1 |
| **L2: 子領域級** | 領域內分支 | NLP, Computer Vision | 3–8 |
| **L3: 核心概念級** | 子領域核心概念 | Transformer, RAG | 6–15 |
| **L4: 技術細節級** | 具體算法/技巧 | Multi-Head Attention | 通過 wikilinks 引用 |

### 粒度判斷標準

| 問題 | 答案為「是」→ 應為獨立概念 |
|------|---------------------------|
| 是否有多個來源討論它？ | ✅ |
| 是否可獨立成一段有意義的描述（>200 字）？ | ✅ |
| 是否與 ≥2 個其他概念有關聯？ | ✅ |
| 是否在領域內有公認的定義？ | ✅ |
| 是否可能被未來查詢單獨引用？ | ✅ |

## 三、實體選擇標準 Entity Selection

### 納入標準（滿足 ≥2 條）

| 標準 | 說明 |
|------|------|
| **代表性** | 在領域內具有標誌性地位 |
| **引用頻率** | 被 raw 文件多次提及 |
| **關聯密度** | 與 ≥3 個概念有直接關聯 |
| **持久性** | 非曇花一現的短期工具/人物 |
| **可操作性** | 對知識使用者有實際參考價值 |

### 排除標準

| 標準 | 說明 |
|------|------|
| 過於短暫 | 流行期 < 1 年的工具/框架 |
| 過於小眾 | 僅在 ≤1 篇 raw 中出現 |
| 無獨立價值 | 可完全歸入概念頁面描述 |
| 已廢棄 | 已被替代且無歷史參考價值 |

## 四、來源溯源鏈 Source Traceability

### 溯源層級

```
Source Page (wiki/sources/)
    │  ← 記錄 raw 文件路徑、原始 URL
    ▼
Entity Page (wiki/entities/)
    │  ← 標註來源頁面
    ▼
Concept Page (wiki/concepts/)
    │  ← 彙總多個來源
    ▼
Query Answer
    │  ← 可追溯至具體概念頁
    ▼
Output Document
       ← 最終引用
```

### 溯源標記格式

```markdown
## Sources
- [[source-page-1]] — 首次提到
- [[source-page-2]] — 補充定義
```

每條來源鏈接必須：
1. 指向具體的 source 頁面
2. 簡要說明該來源提供了什麼資訊
3. 在 source 頁面可反向追蹤到 raw 文件

```
│   └── Neural Networks (L3)
│
├── 域: ai-specialized
│   ├── NLP (L2)
│   ├── Computer Vision (L2)
│   └── Reinforcement Learning (L2)
│
├── 域: ai-frontier
│   ├── Generative AI (L2)
│   ├── LLMs (L2)
│   └── AI Agents (L3)
│
├── 域: ai-governance
│   ├── AI Ethics (L2)
│   └── AI Safety (L2)
│
└── 域: ai-meta
    └── LLM-Wiki (L2) ← 本庫
```
