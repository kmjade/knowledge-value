# 04 — 頁面模板體系

> LLM-Wiki 定義了三類核心頁面模板（Concept / Entity / Source），以及兩個輔助頁面（index / log），確保知識庫輸出的結構一致性。

---

## 一、模板總覽

| 模板 | 目錄 | 用途 | 維護者 |
|------|------|------|--------|
| **Concept** | wiki/concepts/ | 抽象概念、理論、方法論 | AI |
| **Entity** | wiki/entities/ | 具體人物、工具、平台、協議 | AI |
| **Source** | wiki/sources/ | 來源溯源頁 | AI |
| **Index** | wiki/index.md | 全局知識索引 | AI |
| **Log** | wiki/log.md | 編譯操作日誌 | AI |

---

## 二、Concept 模板（概念頁）

### Frontmatter

```yaml
---
title: <Concept Name in English>
aliases: [<Chinese Name>, <Alternative English>]
tags: [concept, <domain-tag>, <sub-tag>]
type: concept
domain: <domain-name>
created: YYYY-MM-DD
modified: YYYY-MM-DD
---
```

### 正文結構

```markdown
# <Concept Name> — <Chinese Name>

> 一句話定義（繁體中文 + English 術語）。

## 定義 Definition
- 形式化定義
- 核心屬性 (3–5 點)

## 核心特徵 Key Characteristics
| 特徵 | 說明 |
|------|------|
| ...  | ...  |

## 相關概念 Related Concepts
- [[related-concept-1]] — 關係說明
- [[related-concept-2]] — 關係說明

## 相關實體 Related Entities
- [[entity-1]] — 關聯說明

## 來源 Sources
- [[source-page-name]]
```

### 範例

參見 [[../wiki/concepts/LLM-Wiki|LLM-Wiki 概念頁]] 作為實際範例。

## 三、Entity 模板（實體頁）

### Frontmatter

```yaml
---
title: <Entity Name>
aliases: [<Chinese Name>, <Abbreviation>]
tags: [entity, <entity_type>, <domain-tag>]
type: entity
entity_type: <person|tool|platform|protocol|system|method>
domain: <domain-name>
created: YYYY-MM-DD
modified: YYYY-MM-DD
---
```

### 實體類型子模板

#### Person（人物）

| 欄位 | 說明 |
|------|------|
| **生平** | 生卒年份、國籍、主要領域 |
| **貢獻** | 核心理論/發現/著作 |
| **影響** | 對領域的影響與傳承 |
| **相關概念** | wikilinks |
| **相關人物** | wikilinks |

#### Tool（工具）

| 欄位 | 說明 |
|------|------|
| **概述** | 功能、開發者、發佈年份 |
| **核心功能** | 主要能力列表 |
| **使用場景** | 適用領域與限制 |
| **相關概念** | wikilinks |
| **替代工具** | 同類工具對比 |

#### Platform（平台）

| 欄位 | 說明 |
|------|------|
| **概述** | 平台定位、運營方 |
| **核心服務** | 提供的服務列表 |
| **生態** | API、插件、整合 |
| **相關實體** | wikilinks |

#### Protocol（協議/標準）

| 欄位 | 說明 |
|------|------|
| **概述** | 協議目的、制定組織 |
| **規範版本** | 當前版本、發佈日期 |
| **核心規範** | 關鍵技術要點 |
| **實現** | 主要實現清單 |
| **相關概念** | wikilinks |

## 四、Source 模板（來源頁）

### Frontmatter

```yaml
---
title: Source: <Source Title>
tags: [source, <source_type>]
type: source
source_type: <article|paper|book|conversation|webpage|documentation>
source_path: raw/<relative-path>
source_url: <original-url>
compiled_date: YYYY-MM-DD
created: YYYY-MM-DD
---
```

### 正文結構

```markdown
# Source: <Source Title>

## 摘要 Summary
- 2–3 句話概括

## 提取概念 Extracted Concepts
| 概念 | 說明 |
|------|------|
| ...  | ...  |

## 提取實體 Extracted Entities
| 實體 | 類型 | 說明 |
|------|------|------|
| ...  | ...  | ...  |

## 關鍵引用 Key Quotes
> 重要原文引用

## 原始連結 Original Link
- [URL](source_url)
```

## 五、Index 模板（索引頁）

```markdown
---
title: 知識庫索引
tags: [index, <domain-tag>]
created: YYYY-MM-DD
modified: YYYY-MM-DD
---

# 知識庫索引

> Wiki 內容目錄，按類別組織。

## 概念頁 Concepts
| 頁面 | 說明 |
|------|------|
| ...  | ...  |

## 實體頁 Entities
| 頁面 | 類型 | 說明 |
|------|------|------|
| ...  | ...  | ...  |

## 來源頁 Sources
| 頁面 | 來源 | 日期 |
|------|------|------|
| ...  | ...  | ...  |

## 統計 Statistics
| 指標 | 數量 |
|------|------|
| 概念頁 | N |
| 實體頁 | N |
| 來源頁 | N |
| **總計** | **N** |
```

## 六、Log 模板（日誌頁）

```markdown
## [YYYY-MM-DD] 操作類型 | 標題

**操作類型**: <ingest|query|lint|init>

**源文件**: N 篇
- `raw/path/file1.md`
- ...

**創建頁面** (N):
- `wiki/concepts/xxx.md` — 說明
- ...

**說明**: 詳細描述
```
