## 二、场景详细分析

### 场景 1：编写文档

```

Token：2,000

→ 只需语言规范，不需要代码相关 Skill
```

### 场景 2：开发 API

```
激活：7 个 Skill（~3,093 行）
未激活：mall-code-review
Token：12,000
节省：13,000 tokens (52%)
→ 复杂任务仍需较多 Skill，但无需审查
```

### 场景 3：代码审查

```
激活：4 个 Skill（~1,534 行）
Token：6,500
节省：18,500 tokens (74%)
→ 只需审查 + 规范 Skill，无需检索和开发流程
```

## 四、ROI 分析

### 投入

| 项目 | 时间 |
|------|:---:|
| 拆分 CLAUDE.md | 8 小时 |
| 编写 SKILL.md | 2 小时 |
| 测试验证 | 2 小时 |
| **总计** | **12 小时** |

### 回报（第一年）

| 项目 | 节省 |
|------|:---:|
| Token 成本 | $3,307 |
| 维护时间 | ~100 小时 |
| 响应速度 | 60% 提升 |

### ROI 计算

```
ROI = ($3,307 + 100h × $50/h) / (12h × $50/h)
    = $8,307 / $600
    = 13.8×
```
