# OpenCode 主题设置

> [!info] 概述
> OpenCode 支持多种主题，可以根据个人喜好自定义界面外观。

## 查看可用主题

在 TUI 中输入：

```
/themes
```

或使用快捷键 `ctrl+x → t`

---

## 内置主题

| 主题 | 说明 |
|------|------|
| default | 默认主题 |
| dark | 暗色主题 |
| light | 亮色主题 |
| dracula | Dracula 配色 |
| monokai | Monokai 配色 |
| gruvbox | Gruvbox 配色 |

---

## 切换主题

### 方法一：命令

```
/themes
```

然后选择想要的主题。

### 方法二：配置文件

```json
{
  "theme": "dark"
}
```

---

## 自定义主题

### 主题文件位置

```
~/.config/opencode/themes/
├── custom.json
└── my-theme.json
```

### 主题文件格式

```json
{
  "name": "my-theme",
  "colors": {
    "background": "#1a1a2e",
    "foreground": "#eaeaea",
    "primary": "#0f3460",
    "accent": "#e94560",
    "border": "#16213e",
    "text": "#eaeaea",
    "textMuted": "#888888"
  },
  "syntax": {
    "keyword": "#e94560",
    "string": "#4ecca3",
    "comment": "#888888",
    "function": "#00adb5",
    "number": "#f38181"
  }
}
```

---

## 颜色配置项

### 界面颜色

| 配置项 | 说明 |
|--------|------|
| `background` | 背景色 |
| `foreground` | 前景色 |
| `primary` | 主色 |
| `accent` | 强调色 |
| `border` | 边框色 |
| `text` | 文字颜色 |
| `textMuted` | 次要文字颜色 |

### 语法高亮

| 配置项 | 说明 |
|--------|------|
| `keyword` | 关键字 |
| `string` | 字符串 |
| `comment` | 注释 |
| `function` | 函数名 |
| `number` | 数字 |

---

## 主题推荐

### 暗色主题

- **Dracula** - 流行的暗色主题
- **Monokai** - 经典的代码编辑器主题
- **Gruvbox** - 复古风格暗色主题
- **Nord** - 北欧风格暗色主题

### 亮色主题

- **Solarized Light** - 护眼亮色主题
- **GitHub Light** - GitHub 风格
- **One Light** - Atom 风格

---

## 配置示例

### opencode.json

```json
{
  "theme": "dracula"
}
```

### 环境变量

```bash
export OPENCODE_THEME=dracula
```
