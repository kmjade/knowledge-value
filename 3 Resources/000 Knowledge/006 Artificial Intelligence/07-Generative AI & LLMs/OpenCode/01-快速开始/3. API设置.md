# OpenCode API 设置

> [!info] 概述
> OpenCode 本身不提供模型，需要配置 AI 服务商的 API Key。

## 快速开始：OpenCode Zen

如果你是第一次使用，建议优先选择 **OpenCode Zen**：
- 官方精选的一组模型
- 已经过测试和优化
- 开箱即用，适合新手
- **包含大量免费模型**

---

## 配置步骤

### 步骤 1：启动 OpenCode

```bash
opencode
```

> [!warning] 权限问题
> 如果出现 **EACCES: permission denied** 错误：
> ```bash
> sudo opencode
> ```

### 步骤 2：运行连接命令

在 TUI 中输入：

```
/connect
```

### 步骤 3：选择提供商

选择 **opencode** 使用 OpenCode Zen，或选择其他提供商：

| 提供商 | 说明 |
|--------|------|
| opencode | 官方精选模型（推荐新手） |
| anthropic | Claude 系列 |
| openai | GPT 系列 |
| google | Gemini 系列 |
| deepseek | DeepSeek |
| 其他... | 阿里百炼、GLM、MiniMax 等 |

### 步骤 4：获取 API Key

访问 https://opencode.ai/auth：
1. 登录账号
2. 添加账单信息（可选）
3. 创建 API Key

### 步骤 5：粘贴 API Key

回到终端，将 API Key 粘贴进去即可完成配置。

> [!note] 凭证存储
> API Key 会存储在 `~/.local/share/opencode/auth.json` 中。

---

## 模型选择

配置完成后，使用 `/model` 命令查看可用模型：

```
/model
```

### 免费模型

标记 **Free** 的模型可以免费使用：
- 适合日常开发
- 无需付费
- 可能有使用限制

### 付费模型

| 模型 | 特点 |
|------|------|
| Claude Sonnet | 代码能力强，推荐 |
| Claude Opus | 最强能力，价格较高 |
| GPT-4 | 综合能力强 |
| Gemini Pro | 多模态支持 |

## 其他提供商配置

### DeepSeek

```
/connect
→ 选择 deepseek
→ 输入 DeepSeek API Key
```

获取 API Key：https://platform.deepseek.com/

### Anthropic Claude

```
/connect
→ 选择 anthropic
→ 输入 Claude API Key
```

获取 API Key：https://console.anthropic.com/

### OpenAI

```
/connect
→ 选择 openai
→ 输入 OpenAI API Key
```

获取 API Key：https://platform.openai.com/

## 配置文件位置

| 配置 | 路径 |
|------|------|
| API 凭证 | `~/.local/share/opencode/auth.json` |
| 项目配置 | `./opencode.json` |
| 全局配置 | `~/.config/opencode/opencode.json` |
