# OpenCode 安装指南

> [!info] 概述
> OpenCode 提供多种安装方式，推荐优先使用官方脚本。

## 安装前准备

### 1. 操作系统要求

| 系统 | 支持情况 |
|------|----------|
| macOS | ✅ 完全支持 |
| Linux | ✅ 完全支持 |
| Windows | ⚠️ 推荐使用 WSL |

### 2. 终端工具

需要一个现代终端：

| 终端 | 平台 | 链接 |
|------|------|------|
| WezTerm | 跨平台 | https://wezterm.org/ |
| Alacritty | 跨平台 | https://alacritty.org/ |
| Ghostty | Linux/macOS | https://ghostty.org/ |
| Kitty | Linux/macOS | https://github.com/kovidgoyal/kitty |

### 3. API Key

OpenCode 本身不提供模型，需要配置 AI 服务商：

| 提供商 | 说明 |
|--------|------|
| OpenAI | GPT 系列 |
| Anthropic | Claude 系列 |
| Google | Gemini 系列 |
| DeepSeek | 国产模型 |
| 阿里百炼 | 国产模型 |
| GLM | 国产模型 |

## 安装方式

### 方式一：一键安装（推荐）

```bash
curl -fsSL https://opencode.ai/install | bash
```

**说明：**
- 自动下载最新版本
- 自动配置环境
- 适用于 macOS / Linux / WSL

### 方式二：Node.js 安装

```bash
# 使用 npm
npm install -g opencode-ai

# 使用 pnpm
pnpm install -g opencode-ai

# 使用 yarn
yarn global add opencode-ai
```

### 方式三：macOS Homebrew

```bash
# 推荐：官方 tap（更新更快）
brew install anomalyco/tap/opencode

# 或使用默认 tap（更新较慢）
brew install opencode
```

### 方式四：Windows 安装

#### 推荐：WSL
更稳定，完整支持所有功能。

#### Chocolatey
```powershell
choco install opencode
```

#### Scoop
```powershell
scoop install opencode
```

#### NPM
```powershell
npm install -g opencode-ai
```

### 方式五：Linux 安装

#### 通用方式
```bash
curl -fsSL https://opencode.ai/install | bash
```

#### Arch Linux
```bash
# 使用 pacman
sudo pacman -S opencode

# 或使用 paru
paru -S opencode-bin
```

### 方式六：Docker

```bash
docker run -it --rm ghcr.io/anomalyco/opencode
```

### 方式七：Mise

```bash
mise use -g github:anomalyco/opencode
```

### 方式八：手动下载

从 Releases 页面直接下载二进制文件：
https://github.com/anomalyco/opencode/releases

## 验证安装

```bash
opencode --version
```

如果输出版本号（如 `1.1.50`），说明安装成功。

---

## 启动 OpenCode

```bash
# 在当前目录启动
opencode

# 指定目录启动
opencode /path/to/project
```

> [!warning] 权限问题
> 如果出现 **EACCES: permission denied** 错误：
>
> ```bash
> # 修复目录权限
> sudo chown -R $(whoami) ~/.local
> chmod -R 755 ~/.local
>
> # 或使用 sudo 启动
> sudo opencode
> ```

## 安装问题排查

### 常见问题

| 问题 | 解决方案 |
|------|----------|
| 命令找不到 | 检查 PATH 环境变量 |
| 权限被拒绝 | 使用 sudo 或修复目录权限 |
| 网络超时 | 使用代理或切换网络 |
| 版本过旧 | 使用官方脚本重新安装 |

### 卸载

```bash
# npm 安装的
npm uninstall -g opencode-ai

# Homebrew 安装的
brew uninstall opencode
```
