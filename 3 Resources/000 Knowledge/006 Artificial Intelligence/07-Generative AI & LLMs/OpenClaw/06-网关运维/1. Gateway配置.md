# OpenClaw Gateway 配置

> [!info] 概述
> Gateway 是 OpenClaw 的核心组件，负责消息路由、代理调度和 API 服务。

## Gateway 基础

### 启动 Gateway

```bash
# 前台运行
openclaw gateway start

# 后台运行（守护进程）
openclaw gateway start --daemon

# 使用 systemd 服务
openclaw gateway install-service
```

### 停止 Gateway

```bash
# 停止服务
openclaw gateway stop

# 重启服务
openclaw gateway restart
```

## 配置文件

### 基础配置

```json
{
  "gateway": {
    "port": 18789,
    "host": "0.0.0.0",
    "controlUi": {
      "enabled": true
    }
  }
}
```

### 完整配置

```json
{
  "gateway": {
    "port": 18789,
    "host": "0.0.0.0",
    "controlUi": {
      "enabled": true,
      "root": null,
      "auth": {
        "enabled": false,
        "username": "admin",
        "password": "secret"
      }
    },
    "api": {
      "enabled": true,
      "prefix": "/api"
    },
    "logging": {
      "level": "info",
      "file": "~/.openclaw/logs/gateway.log"
    }
  }
}
```

## 配置项说明

### 网络配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `port` | 监听端口 | 18789 |
| `host` | 监听地址 | 0.0.0.0 |
| `ssl` | SSL 配置 | - |

### Control UI 配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `enabled` | 是否启用 | true |
| `root` | 自定义 UI 目录 | null |
| `auth.enabled` | 启用认证 | false |
| `auth.username` | 用户名 | admin |
| `auth.password` | 密码 | - |

### API 配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `enabled` | 是否启用 API | true |
| `prefix` | API 路径前缀 | /api |
| `rateLimit` | 速率限制 | - |

## SSL/TLS 配置

### 使用证书文件

```json
{
  "gateway": {
    "ssl": {
      "enabled": true,
      "cert": "/path/to/cert.pem",
      "key": "/path/to/key.pem"
    }
  }
}
```

### 使用 Let's Encrypt

```json
{
  "gateway": {
    "ssl": {
      "enabled": true,
      "letsEncrypt": {
        "enabled": true,
        "email": "admin@example.com",
        "domains": ["example.com"]
      }
    }
  }
}
```

## 命令行管理

### 查看状态

```bash
openclaw gateway status
```

输出示例：
```
Gateway is running
PID: 12345
Port: 18789
Control UI: http://127.0.0.1:18789
```

### 查看日志

```bash
# 实时日志
openclaw gateway logs

# 指定行数
openclaw gateway logs --lines 100

# 过滤日志
openclaw gateway logs --level error
```

### 配置检查

```bash
# 验证配置
openclaw gateway config check

# 查看配置
openclaw gateway config show
```

## 系统服务

### 安装为系统服务

```bash
# 安装服务
openclaw gateway install-service

# 启动服务
systemctl start openclaw

# 开机自启
systemctl enable openclaw

# 查看状态
systemctl status openclaw
```

### 服务配置

服务文件位于 `/etc/systemd/system/openclaw.service`

---

## 故障排除

### 端口被占用

```bash
# 查看端口占用
lsof -i :18789

# 或修改端口
openclaw gateway start --port 18790
```

### 权限问题

```bash
# 使用非特权端口（>1024）
# 或使用 root 运行
sudo openclaw gateway start
```

### 进程已存在

```bash
# 查找并终止进程
ps aux | grep openclaw
kill <PID>
```
