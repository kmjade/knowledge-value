# DDC 杜威十進分類法

## 定義
DDC 由 Melvil Dewey 於 1876 年發明，是全球使用最廣泛的圖書分類法。以十進位數字標記知識領域。

## 結構
- **10 大類** (000-900)
- **100 細分** (每大類 10 個子類)
- **小數擴展** 可無限細分

## 大類
| DDC | 領域 |
|:---:|:-----|
| 000 | 知識總論·電腦科學 |
| 100 | 哲學·心理學 |
| 200 | 宗教·神學 |
| 300 | 社會科學 |
| 400 | 語言學 |
| 500 | 自然科學 |
| 600 | 應用科學 |
| 700 | 藝術·休閒 |
| 800 | 文學 |
| 900 | 歷史·地理·傳記 |
