# 知識圖譜 Knowledge Graph

## 定義
以圖結構（節點=實體，邊=關係）組織和表示知識的資料庫。Google 於 2012 年正式提出此概念。

## 核心要素
- **實體** (Entity) — 真實世界中的物件或概念
- **關係** (Relation) — 實體之間的連結（如：生於、任職於）
- **屬性** (Property) — 實體的特徵描述

## 關聯技術
- RDF (Resource Description Framework) — 三元組標準
- SPARQL — 知識圖譜查詢語言
- Wikidata — 開放協作知識圖譜
