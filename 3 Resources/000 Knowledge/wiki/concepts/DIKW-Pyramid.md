# DIKW Pyramid

## 定义

DIKW 金字塔是描述数据 (Data)、信息 (Information)、知识 (Knowledge)、智慧 (Wisdom) 四个层级及其转化关系的信息科学基础模型。

> 来源: [[3 Resources/000 Knowledge/ram/Knowledge-Systems/01/01-概述.md|01]]

## 核心原理

### 四层级定义

| 层级 | 定义 | 例子 |
|------|------|------|
| **数据** | 原始符号，无上下文 | "37.5" |
| **信息** | 有上下文的数据 | "体温 37.5°C" |
| **知识** | 结构和理解的信息 | "正常 36-37°C，37.5 是低烧" |
| **智慧** | 应用知识做正确判断 | "先物理降温，持续升高就去医院" |

### 转化关系

```
数据 → (赋予上下文) → 信息 → (建立结构) → 知识 → (实践判断) → 智慧
```

每一层需要人类的认知加工才能跃迁到下一层。AI 可以辅助数据→信息和信息→知识的转化，但知识的应用（智慧）仍需要人类判断。

## 关键要点

1. 数据本身无意义，上下文赋予它价值
2. 知识是对信息的结构化理解
3. 智慧是知识的实践应用，包含价值判断
4. AI 在数据→知识阶段效率高，但智慧阶段需要人类
5. KOS 的核心目标是将信息有效组织为可检索的知识
