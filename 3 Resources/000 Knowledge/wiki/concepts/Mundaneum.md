# Mundaneum 曼德內姆

> **Google 之前的 Google** — 1895 年的「人类知识通用索引」
> Paul Otlet 和 Henri La Fontaine 毕生心血的物理结晶

---

## 定义

Mundaneum 是 [[Paul-Otlet]] 和 [[Henri-La-Fontaine]] 于 **1910 年**提出的概念，旨在建立**人类全部知识的通用索引系统**。其核心是一个包含 **1500 万张索引卡片**的物理知识库，辅以国际协作网络和 [[UDC]] 分类体系，被认为是**搜索引擎、维基百科和万维网**的物理先驱。

> "The Mundaneum is not a museum in the ordinary sense. It is a **center for the systematic organization of all human knowledge**." — Paul Otlet

---

## 核心组成部分

### 1. 通用书目目录 (RBU)

**Répertoire Bibliographique Universel** — Mundaneum 的核心数据库：

| 指标 | 数值 |
|:----|:----:|
| 卡片总数 | **1500 万+** |
| 卡片规格 | 7.5 × 12.5 cm (国际标准) |
| 分类体系 | [[UDC]] 国际十进分类法 |
| 覆盖语言 | 多语言 (以法语为主) |
| 高峰时期 | 1930 年代 |
| 协作网络 | 全球 300+ 机构 |

### 2. UDC 分类系统

Otlet 和 La Fontaine 专门为 Mundaneum 开发了 [[UDC]]，使其具备：

- **复合主题表达**：`:` 关联号连接多个概念
- **地理/时间维度**：`(1/9)` 地理复分 + `"..."` 时间复分
- **通用性**：跨语言、跨文化的统一分类语言

### 3. 物理基础设施

```
Mundaneum 建筑 (布鲁塞尔·蒙斯)

├── 大厅: 索引卡片目录柜 (1500 万张)
│   ├── 主题目录 (按 UDC 排列)
│   ├── 作者目录 (按字母排列)
│   └── 地理目录 (按地区排列)
│
├── 阅览室: 供研究者查阅
│
├── 档案库: 原始文献收藏
│
├── 博物馆: 知识史展览
│
└── 办公室: 国际合作管理
```

### 4. 国际协作网络

| 协作类型 | 描述 |
|:---------|:-----|
| 书目交换 | 与国际机构交换书目记录 |
| 联合编目 | 多国合作创建统一目录 |
| 标准制定 | UDC 的维护与推广 |
| 培训 | 文献工作方法的国际教学 |

## 时间线

| 年份 | 事件 |
|:----:|:-----|
| 1895 | Otlet 与 La Fontaine 创立 IIB（国际书目学会）|
| 1895 | 开始编译通用书目目录 (RBU) |
| 1905 | UDC 第一版出版 |
| 1910 | "Mundaneum" 名称正式提出 |
| 1913 | La Fontaine 获诺贝尔和平奖 |
| 1920 | 在布鲁塞尔帕莱·杜·蒙德建筑中建立 |
| 1924 | 获得比利时政府支持 |
| 1934 | Otlet 出版《Traité de Documentation》|
| 1941-1944 | 被纳粹占领，部分资料被毁 |
| 1944 | Otlet 逝世，Mundaneum 衰落 |
| 1998 | Mundaneum 在比利时蒙斯**重新开放为博物馆** |
| 2012 | Google 资助 Mundaneum 数字化项目 |
| 2013 | 被认定为**欧洲文化遗产** (European Heritage Label) |

## 与互联网的前驱关系

Mundaneum 在多个维度上预见了今天的互联网：

### 超文本 (1895)

> "Books and documents can be connected through a system of **associative links**, allowing the reader to navigate from one piece of information to another along paths of relevance."

— Otlet, 1895（比 Bush 的 Memex 早 50 年，比 Berners-Lee 的 WWW 早一个世纪）

### 搜索引擎

| Mundaneum 特性 | 现代等价 | 差距 |
|:---------------|:---------|:-----|
| 1500 万卡片索引 | Google 索引 (千亿级) | 规模差 10^5 |
| UDC 分类 + 混合检索 | 关键词 + 语义搜索 | 自动 vs 人工 |
| 国际书目协作 | Pagerank / 链接分析 | 人工 vs 算法 |
| 阅览室物理查询 | 客户端-服务器 | 物理 vs 数字 |

### 知识图谱

```
Mundaneum 概念           现代实现
─────────────            ────────
通用书目目录              Wikidata
UDC 分类                  SKOS · 本体
文献关联                  关联数据 (Linked Data)
全球知识索引               Google Knowledge Graph
```

### 未被实现的要素

- **实时更新**: Mundaneum 卡片更新延迟大 → 数字实时更新解决
- **全民访问**: 需亲临布鲁塞尔 → 互联网解决
- **自主发布**: 编辑集中在少数专家 → Wiki 模式解决
- **多媒体** (Otlet 预见到): 1931 年提出"Télé-Livre"（电视书），类似 iPad + 网络

## 衰落原因与教训

| 原因 | 说明 | 教训 |
|:-----|:-----|:-----|
| **技术限制** | 卡片系统扩展性有限 | 知识组织需要可扩展架构 |
| **政治环境** | 二战纳粹占领破坏 | 知识中心需分布式备份 |
| **资金不足** | 过度依赖政府资助 | 可持续模式至关重要 |
| **集中化** | 单一物理地点 | 分布式网络更健壮 |
| **缺乏自动化** | 纯人工编目 | AI 辅助知识编译 (如 LLM-Wiki) |

## 在 LLM-Wiki 系统中的映射

[[PARA+LLM-Wiki 融合系统]] 可以视为 Mundaneum 理念的现代实现：

| Mundaneum | LLM-Wiki |
|:----------|:----------|
| 1500 万卡片 | raw/ + wiki/ 知识库 |
| UDC 分类 | `#udc/` 标签 + DDC-UDC 映射 |
| 国际书目协作 | AI 自动编译 |
| 物理阅览室 | Obsidian Vault 本地查询 |
| 卡片索引 | wiki/index.md 知识索引 |
| 书目著录 | Frontmatter 标准化元数据 |
| Otlet 手工分类 | AI 自动 UDC 标引 (Phase 2.5) |
