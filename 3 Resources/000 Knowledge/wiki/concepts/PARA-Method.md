# PARA Method

## 定义

PARA 是由 Tiago Forte 提出的个人知识组织方法，将数字生活分为四个层级：Projects (项目)、Areas (领域)、Resources (资源)、Archives (归档)。

> 来源: [[3 Resources/000 Knowledge/ram/Knowledge-Systems/09-Personal-KM/09-Personal-KM.md|09-Personal-KM]]

## 核心原理

### 四层结构

| 层级 | 定义 | 问题 | 例子 |
|------|------|------|------|
| **P**rojects | 有截止日期的任务集 | "我在做什么？" | 装修商铺、学习 Python |
| **A**reas | 持续责任领域 | "我关注什么？" | 健康、财务、职业 |
| **R**esources | 知识资源库 | "我想参考什么？" | AI/ML 笔记、读书笔记 |
| **A**rchives | 已完成/过时内容 | "已完成什么？" | 完结项目、过期资料 |

### 信息流

```
Inbox → Projects (行动) → Archives (完成)
      → Areas (持续关注)
      → Resources (长期参考)
```

## 关键要点

1. PARA 的核心是**以行动为导向**组织信息
2. Projects 和 Areas 的区别: Projects 有终点，Areas 没有
3. 本 Vault 在 PARA 基础上叠加 LLM-Wiki 编译层
4. PARA 本质上是个人版的"分类法"
