# 维也纳学派

## 基本信息

- **活跃时期**: 1924–1936（纳粹上台后解散，成员流亡英美）
- **地点**: 维也纳大学
- **学派类型**: 科学哲学——**逻辑实证主义**（后称逻辑经验主义）
- **核心成员**: Moritz Schlick（创始人）、Rudolf Carnap、Otto Neurath、Hans Hahn
- **相关人物**: 维特根斯坦（不是成员但强烈影响）、《逻辑哲学论》在学派的讨论中扮演核心角色

## 核心主张

### 证实原则

一个陈述有认知意义，当且仅当它**可以被经验验证**（或它是逻辑/数学真理）。所有无法被验证的陈述——形上学、伦理学、美学——被降级为"无认知意义"（不是"假"，是"无意义"）。

### 统一科学

科学应该被**统一**——所有科学学科应该被还原为一个统一的语言（物理主义语言）。"统一科学百科全书"项目试图将所有科学建立在可观察的物理基础上。

### 拒斥形上学

形上学命题（如"绝对精神在历史中实现自身"——黑格尔）既不是真也不是假——它们根本**不是有效的认知陈述**。但这不等于它们没有情感或诗意价值——Neurath 明确区分了认知意义和情感意义。

## 历史命运

| 事件 | 影响 |
|------|------|
| 纳粹上台 | 学派在 1936 年 Schlick 被刺杀后解散，成员流亡英美 |
| Carnap 到了芝加哥 | 逻辑经验主义成为美国科学哲学的正统（1950s） |
| 奎因的批判（"两个教条"，1951）| 瓦解了分析/综合区分——证实原则的关键前提 |
| 库恩的历史主义转向（1962）| 展示了科学的历史和社会维度——与逻辑经验主义的非历史进路决裂 |
| 科学大战中的遗产 | 维也纳学派常被天真地视为"科学主义"的源头 |

## 在三层模型中的位置

维也纳学派是**科学层的认识论维度**的极端表达——一切知识都可以也应该通过科学方法来验证。三层模型继承了维也纳学派的严谨性，但拒绝了它的还原主义和形上学排除。
