# 卡尔·波普尔 (1902–1994)

## 基本信息

- **生卒**: 1902–1994
- **国籍**: 奥地利→英国（二战期间移居）
- **学派**: 批判理性主义
- **核心著作**: 《科学发现的逻辑》（1934/1959）、《开放社会及其敌人》（1945）、《猜想与反驳》（1963）

## 核心贡献

| 贡献 | 描述 |
|------|------|
| **可证伪性** | 科学划界标准——一个理论是科学的当它具有可被经验证伪的可能性。不是"可被证实"，而是"可被证伪" |
| **猜想与反驳** | 科学的运作不是归纳（从观察到理论），而是猜想（提出大胆理论）→反驳（试图推翻它） |
| **开放社会** | 将可证伪性延伸到政治哲学——开放社会允许对其根本原则的批判，封闭社会则反之 |
| **三个世界** | 世界1（物理）、世界2（心理）、世界3（客观知识——如理论和问题的世界） |

## 在三层模型中的位置

波普尔的核心关怀在**科学层**——科学是如何运作的，如何区分科学与非科学。但他也触及了知识层（世界3的客观知识）和学科层（开放社会作为学术自由的论证）。

## 与库恩的著名辩论

波普尔认为科学应该是**永续革命**的——科学家应时刻准备推翻自己的理论。库恩反驳：实际运作的科学是"常规科学"——科学家在范式内解谜而非试图推翻范式。这场辩论至今未息。
