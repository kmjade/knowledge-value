# Tim Berners-Lee

## 基本信息

- **类型**: 计算机科学家
- **国藉**: 英国
- **主要贡献**: 发明 World Wide Web, 提出 Semantic Web & Linked Data
- **现任**: W3C 名誉主席

## 描述

Tim Berners-Lee 是 Web 的发明者，也是语义网 (Semantic Web) 和关联数据 (Linked Data) 理念的提出者。他于 1989 年在 CERN 提出了 Web 的原始构想，2006 年提出关联数据四原则，为知识图谱和语义网奠定了理论基础。

### 主要贡献

| 年份 | 贡献 |
|------|------|
| 1989 | 提出 World Wide Web |
| 1994 | 创立 W3C |
| 2001 | 提出 Semantic Web 愿景 |
| 2006 | 提出 Linked Data 四原则 |
| 2010 | 五星开放数据模型 |

## 核心特征

- "关联数据四原则"是知识图谱 Web 化的理论基础
- 强调数据的开放和互联——"Raw Data Now!"
- 从文档的 Web 到数据的 Web 的完整愿景
