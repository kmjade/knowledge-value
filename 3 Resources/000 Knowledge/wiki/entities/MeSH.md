# Medical Subject Headings (MeSH)

## 基本信息

- **类型**: 受控词表 / 叙词表
- **创建**: 1960 年，美国国立医学图书馆 (NLM)
- **术语数**: 30,000+
- **使用**: PubMed/MEDLINE 索引基础
- **许可**: 公共领域 (美国政府)

## 描述

MeSH 是生物医学领域最权威的受控词表，用于索引 PubMed 数据库中的 3000 万+ 篇文献。每个 MeSH 词条包含 BT/NT/RT 关系层级，支持精准检索。

### 树状结构示例

```
Anatomy [A]
  Body Regions [A01]
    Head [A01.456]
      Face [A01.456.505]
        Eye [A01.456.505.420]
```

## 核心特征

- 每年更新，反映医学进展
- 层级树 + 关联词双结构
- 支持 explode 检索 (自动包含所有下位词)
- Pubmed 自动将同义词映射到 MeSH 规范词
