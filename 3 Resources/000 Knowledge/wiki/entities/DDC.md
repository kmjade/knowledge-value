# Dewey Decimal Classification (DDC)

## 基本信息

- **类型**: 图书分类法
- **创建**: 1876 年，Melvil Dewey
- **使用范围**: 135+ 国家，20 万+ 图书馆
- **许可**: OCLC 维护，需授权
- **状态**: 持续更新 (第 23 版)

## 描述

DDC 是最广泛使用的图书馆分类法，将人类知识分为 10 大类，每类下再细分 10 个子类，形成层级分明的十进制体系。

### 十大类

| 类号 | 类别 |
|------|------|
| 000 | 计算机科学、信息与总类 |
| 100 | 哲学与心理学 |
| 200 | 宗教 |
| 300 | 社会科学 |
| 400 | 语言 |
| 500 | 自然科学 |
| 600 | 技术/应用科学 |
| 700 | 艺术与娱乐 |
| 800 | 文学 |
| 900 | 历史与地理 |

### 细分示例

```
600 技术 (应用科学)
  610 医学与健康
    616 疾病
      616.9 传染病
```

## 核心特征

- 纯数字标记，语言无关
- 层级递进，数字越多越具体
- 每个主题有且仅有一个位置 (互斥性)
- 通过小数实现无限细分
