# 大卫·布鲁尔 (1942–)

## 基本信息

- **出生**: 1942
- **国籍**: 英国
- **背景**: 哲学→科学社会学
- **学派**: 爱丁堡学派——科学知识社会学（SSK）强纲领
- **核心著作**: 《知识与社会意象》（1976）

## 核心贡献

### 强纲领（Strong Programme）四原则

布鲁尔主张科学社会学应该用**同一类原因**解释科学知识的成功和失败：

| 原则 | 要求 |
|------|------|
| **因果性** | 寻找知识声称的社会（和其他）原因 |
| **无偏性** | 对"真"和"假"的声称使用相同的解释模式 |
| **对称性** | 同一种原因既可以解释真实也可以解释虚假的声称 |
| **反身性** | SSK 的模式必须适用于 SSK 本身 |

### 强纲领的激进性

对称性原则是最激进的：如果你用社会原因（如默顿的CUDOS规范）解释为什么物理学有效，你也必须用社会原因解释为什么炼金术失败——而不是简单地用"因为炼金术不真实"。对布鲁尔而言，"真实性"本身就是一个需要社会学解释的概念。

## 在三层模型中的位置

布鲁尔和 SSK 在三层模型的**元层次**中运作——他们分析的是"我们如何知道科学知识是可靠的"这个声称本身的社会建构过程。

## 关键争议

### 科学大战

1990 年代，科学家（以物理学家 Alan Sokal 为首）猛烈反击布鲁尔等人的 SSK——称之为"对科学的攻击"。Sokal 的著名恶作剧（向文化研究期刊投稿一篇关于"量子引力的解放性"的故意无意义论文并发表）成为"科学大战"的引爆点。
