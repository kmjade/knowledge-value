# 托马斯·库恩 (1922–1996)

## 基本信息

- **生卒**: 1922–1996
- **国籍**: 美国
- **背景**: 物理学→科学史→科学哲学
- **核心著作**: 《科学革命的结构》（1962）—— 20 世纪被引用最多的人文著作之一

## 核心贡献

| 贡献 | 描述 |
|------|------|
| **范式** | 科学共同体共享的整套预设、方法、范例。范式定义了什么是好问题和好答案 |
| **常规科学** | 科学的主体是"解谜"——在范式内解决范式定义的问题。这不是坏科学，而是科学唯一可行的运作方式 |
| **科学革命** | 当异常积累无法被范式消化→危机→范式竞争→范式转换。新范式与旧范式"不可通约" |
| **不可通约性** | 新旧范式之间没有中立的评判标准——概念、方法、甚至"事实"的含义都随范式而变 |

## 在三层模型中的位置

库恩的理论是三层模型最核心的理论来源——特别是**范式作为中介**连接科学层和学科层的双重角色。

## 与波普尔的著名辩论

库恩：波普尔的"永续革命"模型从未发生过——真正的科学 99% 的时间在做常规科学。波普尔描述的"试图推翻理论"其实是在范式内的解谜活动。科学教育是"教条式的"——不是为了培养批判者，而是为了让初学者接受范式。
