# Zettelkasten

## 基本信息

- **类型**: 知识管理方法
- **创建者**: Niklas Luhmann (尼克拉斯·卢曼)
- **创建**: 1950-60 年代
- **核心**: 原子笔记 + 双向链接
- **代表工具**: Obsidian, Roam Research, Logseq

## 描述

Zettelkasten (德文: "卡片盒") 是德国社会学家 Niklas Luhmann 发明的知识管理方法。核心理念是将知识分解为**原子化的卡片**，通过**双向链接**建立概念之间的语义网络。Luhmann 用这个方法在 30 年内发表了 70 本书和 400+ 篇论文。

### 核心原则

| 原则 | 说明 |
|------|------|
| **原子性** | 每张卡片只包含一个概念 |
| **链接性** | 卡片之间通过链接建立关联 |
| **自生长** | 链接网络随时间涌现出结构 |
| **独立上下文** | 每张卡片即使脱离原文也可理解 |

## 核心特征

- Obsidian 的 `[[wikilink]]` 直接源自 Zettelkasten 的链接理念
- 强调**涌现**而非预设——结构从链接中自行生长
- 本 Vault 的 wiki/ 概念页 = 数字化的 Zettelkasten
