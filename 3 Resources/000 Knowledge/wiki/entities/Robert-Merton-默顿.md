# 罗伯特·默顿 (1910–2003)

## 基本信息

- **生卒**: 1910–2003
- **国籍**: 美国
- **背景**: 社会学家——科学社会学的奠基人
- **核心著作**: 《科学规范结构》（1942）、《科学社会学》（1973）

## 核心贡献

### CUDOS 规范

默顿描述了使科学有效运作的**制度规范结构**：

| 规范 | 含义 |
|------|------|
| **公有性**（Communism） | 科学知识是公共财产——研究者放弃对其发现的专有权，换取承认 |
| **普遍性**（Universalism） | 科学声称的有效性不取决于声称者的种族、国籍、宗教或机构 |
| **无私利性**（Disinterestedness） | 研究者追求真理而非个人利益 |
| **有条理的怀疑**（Organized Skepticism） | 暂时不接受的悬置判断——直到经过批判性审查 |

### 默顿与科学的社会运行

默顿还贡献了：
- **马太效应**：已有声望的科学家从合作中获得了不成比例的信用（"富者愈富"）
- **多重发现**：许多科学发现在几乎同一时间被多人独立做出——表明发现不只是天才，而是科学的社会结构准备的产物
- **优先权争议**：科学史上频繁的争议——谁先发现了X——揭示了CUDOS 规范（尤其是公有性和无私利性）与制度现实之间的张力

## 在三层模型中的位置

默顿是**科学层的社会维度**的核心理论家——三层模型从默顿那里继承了对科学共同体自我治理的关注。
