# 信息科学 知识库总览

> 信息科学（Information Science）是研究信息的产生、组织、存储、检索、传播和利用的跨学科学科。它与图书馆学、档案学、文献学、知识管理等领域深度交叉，是信息时代的基础学科。

## 为什么要了解信息科学？

1. **信息爆炸时代的基础素养** — 会搜索 ≠ 会检索。信息科学教你系统性地找到、评估和使用信息
2. **知识管理的理论基础** — 你正在用的 Obsidian、Zettelkasten、PARA 都建立在知识组织的原理之上
3. **跨学科思维** — 信息科学连接技术（CS）、人文（图书馆）、管理（知识管理）
4. **学术必备** — 了解出版流程、同行评审、引文分析对做研究至关重要
5. **职业方向** — 数据管理、信息架构、知识管理、数字图书馆都是增长领域

## DIKW 金字塔

信息科学的核心模型——数据如何升华为智慧：

```
         /\
        /智慧\        ← 应用知识做出判断和决策
       /Wisdom\
      /────────\
     /   知识    \     ← 经过组织和理解的信息
    / Knowledge  \
   /──────────────\
  /      信息       \   ← 有上下文和意义的数据
 /   Information    \
/────────────────────\
        数据            ← 原始的、未处理的符号和事实
        Data
```

| 层级 | 定义 | 例子 |
|------|------|------|
| **数据** | 原始的、未加工的事实 | "37.5" |
| **信息** | 有上下文的数据 | "体温 37.5°C" |
| **知识** | 被理解和内化的信息 | "正常体温是 36-37°C，37.5 是低烧" |
| **智慧** | 应用知识做出判断 | "先物理降温，如果持续升高就去医院" |
