# 01-信息科学概述

> 信息科学是研究信息的属性、行为、流动以及处理信息以实现最优可访问性和可用性的学科。——这是一个关于"如何让正确的人在正确的时间找到正确的信息"的学科。

## 信息科学的跨学科本质

```
            ┌──────────────┐
            │   计算机科学   │
            │  (技术实现)    │
            └──────┬───────┘
                   │
    ┌──────────────┼──────────────┐
    │   图书馆学    │              │   认知科学    │
    │  (服务传统)    │              │  (用户行为)   │
    └──────┬───────┘              └──────┬───────┘
           │                             │
           └──────────┬──────────────────┘
                      ↓
              ┌──────────────┐
              │   信息科学     │
              │  (整合学科)    │
              └──────────────┘
                      ↑
    ┌─────────────────┼─────────────────┐
    │   档案学        │  管理学          │  传播学
    │  (记录保存)     │ (信息管理)       │ (信息传播)
    └─────────────────┘                 └─────────┘
```

## 学科名称演变

| 时期 | 名称 | 焦点 |
|------|------|------|
| 19 世纪末-20 世纪中 | 图书馆学 (Librarianship) | 图书的收藏、编目和借阅 |
| 1960s-1980s | 图书馆与信息科学 (LIS) | 信息检索、自动化进入图书馆 |
| 1990s-2000s | 信息科学 / 情报学 | 数字资源、信息系统 |
| 2000s-至今 | iSchools 运动 | 信息、技术与人——跨学科整合 |

> 中国学科目录中的"图书情报与档案管理"（1205）对应的正是这个领域。

## 信息生命周期

信息从创造到消亡（或长期保存）的全过程：

```
创建 → 采集 → 组织 → 存储 → 检索 → 传播 → 使用 → 保存/销毁
  ↑____________________________________________________↓
                      反馈与更新
```

每一阶段都是信息科学的研究对象：
- **创建**：学术写作、数据生产
- **组织**：分类、编目、知识组织学
- **存储**：数据库、数字仓储
- **检索**：搜索引擎、信息检索学
- **传播**：出版、开放获取
- **使用**：信息行为、信息素养
- **保存**：数字保存、档案学

## 信息科学的核心理念

| 理念 | 含义 |
|------|------|
| **相关性而非数量** | 好的信息系统不是返回最多的结果，而是返回最相关的结果 |
| **用户中心** | 信息系统的设计必须从用户的信息需求和信息行为出发 |
| **标准与互操作** | 不同系统之间能交换和理解数据——元数据标准是关键 |
| **长期可访问** | 信息不仅要今天能用，10 年、50 年后也要能用 |
| **信息伦理** | 隐私、知识产权、信息自由、数字鸿沟 |

## 关键人物

| 人物 | 贡献 |
|------|------|
| **Paul Otlet** (1868-1944) | 信息科学之父，创建通用十进制分类法 (UDC)，设想"世界脑" |
| **Vannevar Bush** (1890-1974) | 1945 年提出 Memex——超文本的先驱概念 |
| **Eugene Garfield** (1925-2017) | 创建 SCI 和影响因子，信息计量学奠基人 |
| **Tim Berners-Lee** | 发明 World Wide Web，推动语义网和关联数据 |
| **S. R. Ranganathan** (1892-1972) | "图书馆学五定律"，印度图书馆学之父 |

### 图书馆学五定律

> Ranganathan (1931) 提出的五条简洁而深刻的定律：

1. **书是为了用的** (Books are for use)
2. **每位读者有其书** (Every reader his/her book)
3. **每本书有其读者** (Every book its reader)
4. **节省读者的时间** (Save the time of the reader)
5. **图书馆是一个生长的有机体** (Library is a growing organism)

> 这五条至今仍是图书馆和信息系统设计的指导原则。用"信息"替换"书"，它们完全适用。
