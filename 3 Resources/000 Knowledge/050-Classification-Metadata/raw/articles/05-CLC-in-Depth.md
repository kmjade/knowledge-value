# CLC 中国图书馆分类法详解

> 1975年首次出版——22大类(A-Z)——字母+数字——中国大陆绝大多数图书馆标准

---

| 属性 | 值 |

| 全称 | 中国图书馆分类法（曾用名：中国图书馆图书分类法） |

## 三、独特特征

### 意识形态嵌入

**A 类**（马克思主义·列宁主义·毛泽东思想）独立成大类——全球主要分类系统中**独一无二**。这展示了学科分类体系的非中立性——分类反映社会和政治价值观。

### 中国知识传统

- **R2**（中国医学）独立于现代医学（R1）——传统/现代知识在分类体系中共存
- **K825.x** 按人物成就领域精细复分——传记在四体系中最系统

### 技术细分优势

```
T 工业技术
  └── TP 自动化·计算机
       └── TP317 应用软件
            ├── TP317.1  办公自动化
            ├── TP317.2  字处理
            ├── TP317.3  排版
            ├── TP317.4  桌面出版
            ├── TP317.42 图形设计
            └── TP317.46 图像处理
```

> CLC TP317 的细度远超 DDC 005.5（后者将所有办公应用归为一个总类）。

## 四、CLC vs 其他体系

| | CLC | DDC |
|------|------|------|
| 大类数 | 22 (A-Z) | 10 (000-900) |
| 编码 | 字母+数字 `TP317.4` | 纯数字 `006.686` |
| 维护方 | 国家图书馆 | OCLC |
| 意识形态 | A类独立设马克思主义 | 宗教在200（基督教主导） |
| 中医定位 | R2 独立子类 | 归入610医学 |
| 传记精度 | K825.x 极细 | 920 加职业号 |

---

## 五、在本Vault中的使用

本Vault以DDC为主体分类，CLC作为**辅助交叉对照**：

- 所有工具分类码页均含CLC对照
- 技术工具(TP317.x)的CLC细分比DDC更精确
- 中医学(R2)使用CLC作为首选分类
