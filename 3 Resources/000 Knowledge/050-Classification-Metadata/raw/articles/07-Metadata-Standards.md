# Metadata Standards-元数据标准

> 元数据 = 关于数据的数据——书有作者·标题·日期——这些就是元数据

## 核心标准

| Dublin Core: 15个元素——最简单——跨领域 |
|------|
| MODS/MARC: 图书馆专用——复杂但精密 |
| Schema.org: Google·Bing·Yahoo推动——网页结构化数据 |
