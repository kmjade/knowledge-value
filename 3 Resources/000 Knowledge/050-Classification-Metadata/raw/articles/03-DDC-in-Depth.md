# DDC 详解

> Melvil Dewey (1876) — 10大类 · 纯数字 · 小数制 · 全球最广泛使用的图书分类法

---

## 一、历史

|  年份  | 事件                       |
| :--: | ------------------------ |
| 1876 | 第一版——44页（仅12页分类表）        |
| 1951 | 第15版——正式命名为「杜威十进制分类法」    |
| 2003 | 第22版——更新004-006（计算机）     |
| 2011 | 第23版——最新印刷版，约3000页       |
|  现在  | **WebDewey** 为权威在线版，季度更新 |

## 四、优势与局限

| ✅ 优势 | ❌ 局限 |
|------|------|
| 十进制——人尽皆知 | 西方中心——200类基督教占9/10 |
| 层级明确——分类即排架 | 数字分配不均——600远大于200 |
| 全球200000+图书馆 | 单一排架——交叉学科被迫选一个位置 |
| WebDewey季度更新 | 更新滞后——新兴领域需等十几年 |

### 著名争议

| 争议 | 详情 |
|------|------|
| 宗教偏向 | 200类：220-280全是基督教，290是「其他宗教」 |
| LGBTQ分类 | 从「异常性心理」到「性取向」(306.76)——数十年修正 |
| 数字鸿沟 | AI伦理等新兴领域仍难以安置 |

### DDC vs UDC

| | DDC | UDC |
|------|------|------|
| 创始人 | Melvil Dewey (美) | Paul Otlet (比, 基于DDC扩展) |
| 分类策略 | 枚举+层级 | 分面+层级——连接符合并 |
| 灵活性 | 低——单一位置 | 高——多维度组合 |
| 使用范围 | 公共图书馆 | 专业/研究图书馆 |
| 符号 | 纯数字 `336.73` | 数字+符号 `336.73(73)"19"` |

## 五、本Vault中的应用

| 调整 | 说明 |
|------|------|
| DDC骨架 | 000·100·300·400·500·600·700·800·900 |
| 004独立 | 计算机科学从000抽出，单独构建 |
| 006合并 | AI并入`006 Artificial Intelligence` |
| 组合使用 | DDC作为宏观骨架 + 标签作为微观维度 |
