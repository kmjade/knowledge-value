# UDC 國際十進分類法 — In-Depth

> UDC（Universal Decimal Classification）是 Paul Otlet 與 Henri La Fontaine 於 1905 年基於 DDC 創建的文獻分類系統。其核心創新是**輔助符號體系**——通過 `+ / : = ( ) " "` 等符號組合主題，表達力遠超 DDC。

---

## 核心定位

| 維度 | 說明 |
|------|------|
| **創建者** | Paul Otlet & Henri La Fontaine（比利時） |
| **首版** | 1905（基於 DDC 第 5 版） |
| **維護方** | UDC Consortium（荷蘭海牙） |
| **類型** | 分面 + 層級分類法 |
| **適用範圍** | 專業 / 研究圖書館、文獻數據庫 |
| **使用國家** | 30+ 國（歐洲、拉美、非洲為主） |

---

## UDC vs DDC 核心差異

| 維度 | DDC | UDC |
|------|-----|-----|
| 分類策略 | **枚舉**——每個主題一個數字 | **分面**——通過符號組合多個維度 |
| 連接符 | 無（純層級） | 有——`+ / : = (…) "…"` |
| 靈活性 | 低 | **高**——可表達複雜關係 |
| 學習曲線 | 低——直觀 | 高——需學習符號系統 |
| 適用場景 | 公共 / 學校圖書館 | 專業 / 研究圖書館 |
| 核心思維 | 一本書 → 一個號碼 | 一本書 → 一個表達式 |

---

## 輔助符號體系（UDC 的精髓）

UDC 最偉大的創新：用符號讓類號可以**組合**。

| 符號 | 功能 | 範例 |
|------|------|------|
| **+** | 並列（兩主題並存） | `622+669` = 採礦與冶金 |
| **/** | 範圍（連續類號） | `621.3/.4` = 電氣工程到熱機 |
| **:** | 關係（主題交叉） | `37:004` = 教育與計算機 |
| **=** | 語言 | `61=111` = 醫學（英文） |
| **(0…)** | 形式 | `(035)` = 手冊、`(043)` = 論文 |
| **(…)** | 地點 | `(510)` = 中國、`(73)` = 美國 |
| **"…"** | 時間 | `"20"` = 21 世紀 |
| **(=…)** | 種族 / 民族 | `(=581)` = 中國人 |

### 實例對比

| 文獻主題 | DDC 23 | UDC |
|----------|:------:|:---:|
| 中國醫學史（英文） | 610.951 | `61(510)(091)=111` |
| 人工智能教育 | 006.3071 | `004.8:37` |
| 德國哲學詞典 | 103.43 | `1(038)=112.2` |
| 日本採礦與冶金 | 669.0952 | `622+669(520)` |
| 21 世紀中國高等教育 | 378.510905 | `378(510)"20"` |

## 十大類概覽

| 類 | 內容 | 備註 |
|:--:|------|------|
| **0** | 總論、計算機、信息 | 含 004 計算機科學 |
| **1** | 哲學、心理學 | — |
| **2** | 宗教、神學 | — |
| **3** | 社會科學、統計、政治、經濟 | — |
| **4** | ⚠️ **空**——原語言類已併入 8 | **UDC 獨有特徵** |
| **5** | 數學、自然科學 | — |
| **6** | 應用科學、醫學、科技 | 本 vault 重點對應領域 |
| **7** | 藝術、娛樂、體育 | — |
| **8** | 語言、語言學、文學 | 1980 年代吸收原第 4 類 |
| **9** | 地理、傳記、歷史 | — |

> 注意：UDC 沒有第 4 類——這是 UDC 不同於 DDC 的明顯特徵。

## 設計哲學

| DDC | UDC |
|-----|-----|
| 簡單——任何圖書館員都能用 | **精確**——能描述文獻的所有維度 |
| 為 **排架** 設計 | 為 **文獻檢索** 設計 |
| 公共圖書館首選 | 專業圖書館首選 |
