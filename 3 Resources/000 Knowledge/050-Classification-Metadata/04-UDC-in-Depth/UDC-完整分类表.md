# UDC 完整分類表

> UDC（Universal Decimal Classification）——Paul Otlet & Henri La Fontaine 1905 年創建——基於 DDC 但用輔助符號大幅擴展表達力。
> 以下為 UDC 主類（0–9）及主要細分（1–2 級）。UDC 不採用 DDC 的 3 位數字基礎碼，而是直接使用數字 + 小數點。

## UDC 標記法特點

### 數字結構

UDC 與 DDC 的數字結構有顯著差異：

| 特徵 | DDC | UDC |
|:----|:---|:----|
| 主類位數 | 3 位（000–999） | 1–2 位（0–9、00–99） |
| 小數點位置 | 第 3 位後固定 | 可出現在任意位置 |
| 類號長度 | 相對固定 | 可長可短——取決於精度 |
| 基礎類號 | 所有類號 3 位起 | 主類 1 位起，細分用． |

### 輔助符號

UDC 的精髓——這些符號讓類號可以**精確描述文獻的各個維度**：

| 符號 | 功能 | 使用範例 |
|:----:|------|----------|
| **+** | 並列（兩個獨立主題） | `622+669` = 採礦與冶金 |
| **/** | 範圍（連續類號區間） | `621.3/.4` = 從電氣工程到熱機 |
| **:** | 關係（主題間關聯） | `37:004` = 教育與計算機的關係 |
| **::** | 雙冒號（固定順序的關係） | `37::004` = 計算機輔助教育 |
| **=** | 語言 | `61=111` = 醫學文獻（英文） |
| **(0…)** | 形式 | `(035)` = 手冊、`(043)` = 學位論文 |
| **(…)** | 地點 | `(510)` = 中國、`(73)` = 美國 |
| **(=…)** | 種族·民族 | `(=581)` = 中國人 |
| **"…"** | 時間 | `"20"` = 21 世紀 |
| **.0…** | 觀點·視角 | `.002` = 技術觀點、`.003` = 經濟觀點 |
| **-0…** | 通用復分 | 材料、性能等通用屬性 |

### UDC 類號構建範例

```text
文獻：「21 世紀中國高等教育的經濟影響」

UDC 類號構建步驟：
1. 主類：378（高等教育）
2. 地點：(510)（中國）
3. 時間："20"（21 世紀）
4. 觀點：.003（經濟觀點）

結果：378(510)"20".003
```
