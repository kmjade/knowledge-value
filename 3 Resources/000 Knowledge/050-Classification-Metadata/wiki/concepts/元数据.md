# 元數據 Metadata

## 定義
關於數據的數據 — 描述資源的結構化信息，用於檢索、管理與互操作。

## 主要標準
| 標準 | 領域 | 說明 |
|:----|:----|:-----|
| **Dublin Core** | 通用 | 15 個核心元素（題名·作者·主題·日期等） |
| **MARC 21** | 圖書館 | 機讀編目格式，書目記錄標準 |
| **MODS** | 圖書館 | 基於 XML 的書目描述 |
| **DCAT** | 數據集 | 數據目錄詞彙表 |

## 核心元素 (Dublin Core)
Title·Creator·Subject·Description·Publisher·Contributor·Date·Type·Format·Identifier·Source·Language·Relation·Coverage·Rights
