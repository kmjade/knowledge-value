# CLC 中國圖書館分類法

## 定義
中國圖書館分類法 (CLC) 是中國大陸通用的圖書分類標準，由中國圖書館分類法編輯委員會編制。

## 結構
22 大類，以英文大寫字母標記：

| 大類 | 領域 |
|:---:|:-----|
| A | 馬克思主義·列寧主義 |
| B | 哲學·宗教 |
| C | 社會科學總論 |
| D | 政治·法律 |
| F | 經濟 |
| G | 文化·科學·教育 |
| I | 文學 |
| J | 藝術 |
| K | 歷史·地理 |
| N-X | 自然科學·應用科學 |
| Z | 綜合性圖書 |
