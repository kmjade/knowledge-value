# 分類原理 Classification Principles

## 核心要素
| 要素 | 說明 |
|:----|:-----|
| **類目** | 分類的單元——「哲學」「數學」 |
| **層級** | 大類→中類→小類，從一般到特殊 |
| **標記** | 數字或字母代表類目，便於排架與檢索 |
| **索引** | 幫助找到類目——分類的「地圖」 |

## 類型
- **列舉式**：把所有類目列出（DDC）
- **分面式**：用標準面組合（冒號分類法，UDC）
