# 阮岡納贊 S.R. Ranganathan
印度圖書館學家，冒號分類法 (Colon Classification) 創始人。

## 主要貢獻
- 冒號分類法 — 史上首個分面分類系統
- 圖書館學五定律
- 分面分析理論：Personality·Matter·Energy·Space·Time (PMEST)
