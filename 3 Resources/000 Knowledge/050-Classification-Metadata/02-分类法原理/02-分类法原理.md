# Principles of Classification

| 知识分类 vs 文献分类: 「世界」怎么分 vs 「书」怎么放——不是同一回事 |
|------|
| 文献保证(literary warrant): 有书——才有类目——没有书的类目是空的 |
| 用户保证(user warrant): 用户需要找什么——就建立什么类目 |

## 阮冈纳赞五定律

| 书是为了用的·每个读者有他的书·每本书有它的读者·节省读者的时间·图书馆是生长中的有机体 |
|------|
