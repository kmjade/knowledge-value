# What is Knowledge?？

> 柏拉图：「知识是得到辩护的真信念」——2500年后，一个三页的论文推翻了它。

## 传统定义 (JTB)

| 条件 | 含义 |
|------|------|
| 信念 (Belief) | S相信P——如果不信——不可能是知识 |
| 真 (Truth) | P是真的——如果是假的——只是「自以为知道」 |
| 辩护 (Justification) | S有理由相信P——不是蒙对的——否则是「幸运的猜测」 |

## Gettier 问题 (1963)

| Gettier用两页半论文推翻了JTB——构造了「真信念+辩护——但不是知识」的案例 |
|------|
| 例子: Smith有充分理由相信「得到工作的人口袋里有10枚硬币」——他自己恰好也是那个人——他的信念为真且有辩护——但这显然不是「知识」——只是巧合 |

## 后Gettier时代

| 方案 | 核心 |
|------|------|
| 无错误 (No false lemmas) | 辩护不能依赖任何错误的前提 |
| 因果理论 | 知识 = 真信念+事实和信念之间的适当因果链 |
| 可靠主义 | 知识由可靠的认知过程产生 |
| 德性认识论 | 知识来自认知德性(像伦理德性——但针对认知) |
