# Sources of Knowledge

> 我们怎么知道我们「知道」的那些东西？——理性还是经验？

## 理性主义 vs 经验主义

| | 理性主义 | 经验主义 |
|------|------|------|
| 知识来源 | 理性——先天 | 感官——后天 |
| 代表 | 笛卡尔·斯宾诺莎·莱布尼茨 | 洛克·休谟·贝克莱 |
| 数学是… | 纯粹理性的典范 | 从经验抽象而来 |
| 「白板」？ | 否——有先天观念 | 是——一切来自经验 |

## 康德的综合

| 先天综合判断——「经验提供内容——理性提供形式」——「没有内容的思维是空的——没有概念的直观是盲的」 |
|------|

## 其他来源

| 来源 | 说明 |
|------|------|
| 记忆 | 把过去的知识带到现在——但记忆不可靠(每一次回忆都是重构) |
| 证词 | 别人的话——我们大多数「知识」其实来自证词——不是亲自验证 |
| 直觉 | 「感觉是对的」——哲学家对直觉的依赖——但直觉可靠吗？ |
