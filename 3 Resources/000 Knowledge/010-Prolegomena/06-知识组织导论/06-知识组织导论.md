# Intro to Knowledge Organization

> 我们如何安排知识使其可检索？——这是知识组织的核心问题。

## 分类的哲学

| 亚里士多德的范畴: 实体·数量·性质·关系——西方分类思想的源头 |
|------|
| 「分类即政治」: 谁决定什么属于什么类别？——DSM(精神疾病诊断标准)是分类——也是权力 |

## 图书馆分类简史

| 亚历山大图书馆→中世纪修道院→培根→杜威(1876)→奥特勒(UDC·1905)→阮冈纳赞(冒号分类法·1933) |
|------|

## 知识组织的原则

| 原则 | 说明 |
|------|------|
| 互斥 | 每个项目只属于一个类别 |
| 穷尽 | 所有项目都有地方放 |
| 可扩展 | 新知识可以加入——不破坏结构 |
| 可检索 | 最终目标是找到——不是放在正确的位置 |
