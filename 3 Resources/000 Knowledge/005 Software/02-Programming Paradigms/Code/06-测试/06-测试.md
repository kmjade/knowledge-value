# 06-测试

## 测试金字塔

```
    /E2E\      少量——完整用户流程
   /------\
  / 集成测试 \   中等——模块交互
 /----------\
/  单元测试   \  大量——函数/方法
```

## 测试类型

| 类型 | 范围 | 速度 |
|------|------|------|
| 单元 | 单个函数 | ms |
| 集成 | 模块间 | s |
| E2E | 完整流程 | min |
| 性能 | 吞吐/延迟 | min |

## TDD 三步

```
RED —— 写失败测试
GREEN —— 最少代码让测试过
REFACTOR —— 重构优化
```

## 测试原则

| 原则 | 说明 |
|------|------|
| FIRST | Fast/Independent/Repeatable/Self-validating/Timely |
| 一断言 | 一个测试验证一件事 |
| 边界 | 测试空/零/最大/负值 |
