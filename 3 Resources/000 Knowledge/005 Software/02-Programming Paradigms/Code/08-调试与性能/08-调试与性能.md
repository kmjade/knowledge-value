# 08-调试与性能

## 调试方法

| 方法 | 适合 |
|------|------|
| print/log | 简单快速 |
| 断点调试 | 复杂逻辑——step in/out/over |
| 二分注释 | 定位引入 bug 的代码段 |
| rubber duck | 向"橡皮鸭"解释代码——往往自己发现问题 |

## 性能分析

| 工具 | 测什么 |
|------|--------|
| Profiler | CPU 热点——哪个函数最慢 |
| Memory Profiler | 内存泄漏——谁占内存 |
| Benchmark | 微基准——A vs B 谁更快 |
| APM | 生产环境——Datadog/New Relic |

## 常见性能问题

| 问题 | 解决 |
|------|------|
| N+1 查询 | 批量查询/eager loading |
| 无索引 | 加数据库索引 |
| 同步阻塞 | 异步/并发 |
| 大对象 | 分页/懒加载 |
