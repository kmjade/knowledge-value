# 03-数据结构与算法实现

> 理论在 [[3 Resources/000 Knowledge/004 Computer Science & technology/004.5-数据库与数据结构/02-数据结构与算法|CS KB]]。本章聚焦**代码实现**。

## 常用技巧

| 技巧 | 场景 | 模板 |
|------|------|------|
| 双指针 | 有序数组、链表 | `l=0; r=n-1; while l<r` |
| 滑动窗口 | 子串/子数组 | `window = set(); expand+shrink` |
| 哈希表 | 查找/去重 | `dict/hashmap` |
| 栈 | 括号/逆序 | `stack = []` |
| BFS/DFS | 图/树遍历 | `queue/recursion` |
| DP | 最优子结构 | `dp[i] = f(dp[i-1],...)` |

## 时间复杂度速查

| 操作 | 数组 | 链表 | 哈希 | BST |
|------|------|------|------|-----|
| 访问 | O(1) | O(n) | O(1)* | O(log n) |
| 搜索 | O(n) | O(n) | O(1)* | O(log n) |
| 插入 | O(n) | O(1) | O(1)* | O(log n) |
| 删除 | O(n) | O(1) | O(1)* | O(log n) |
