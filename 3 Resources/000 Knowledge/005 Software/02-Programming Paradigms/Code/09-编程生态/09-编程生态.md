# 09-编程生态

## 开源许可

| 许可 | 要求 | 适合 |
|------|------|------|
| **MIT** | 保留版权声明即可 | 最自由 |
| **Apache 2.0** | 专利授权 | 企业友好 |
| **GPL** | 衍生也须开源 | "传染性" |
| **AGPL** | 网络使用也须开源 | 最强 copyleft |

## 包管理

| 语言 | 包管理器 |
|------|----------|
| Python | pip / Poetry |
| JS/TS | npm / yarn / pnpm |
| Java | Maven / Gradle |
| Rust | Cargo |
| Go | go mod |
