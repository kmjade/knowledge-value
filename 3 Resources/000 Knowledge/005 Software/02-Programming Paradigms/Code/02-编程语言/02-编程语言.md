# 02-编程语言

## 类型系统

| | 静态 | 动态 |
|---|------|------|
| 检查时机 | 编译时 | 运行时 |
| 安全 | 高——早发现 | 低——测试覆盖 |
| 代表 | Java, Rust, TS | Python, JS, Ruby |

## 语言速览

| 语言 | 范式 | 擅长 | 典型应用 |
|------|------|------|----------|
| **Python** | 多范式 | 数据、AI、脚本 | ML、后端、自动化 |
| **JavaScript/TS** | 多范式 | Web | 前端全栈 |
| **Java** | OOP | 企业后端 | 金融、电商 |
| **Go** | 并发 | 网络服务 | 云原生、CLI |
| **Rust** | 系统 | 安全+性能 | 系统编程、WASM |
| **C/C++** | 系统 | 极致性能 | 游戏、嵌入式、OS |

## 选型指南

| 场景 | 推荐 |
|------|------|
| 快速原型 | Python |
| Web 全栈 | TypeScript |
| 企业后端 | Java/Go |
| 系统底层 | Rust |
| 数据/AI | Python |
