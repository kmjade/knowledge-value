# 必備插件

> [!summary] 功能概述
> IntelliJ IDEA 常用插件推薦，提升開發效率。

---

## 生產力插件

### Lombok
- **功能**：自動生成 getter/setter/constructor
- **安裝**：Settings → Plugins → 搜索 "Lombok"
- **注意**：需要項目添加 Lombok 依賴

### MyBatisX
- **功能**：MyBatis 增強插件
- **特點**：Mapper 接口與 XML 跳轉、代碼生成

### Rainbow Brackets
- **功能**：彩色括號匹配
- **特點**：快速識別代碼塊範圍

### Key Promoter X
- **功能**：快捷鍵提示
- **特點**：使用鼠標操作時提示對應快捷鍵

---

## 代碼質量

### SonarLint
- **功能**：代碼質量檢查
- **特點**：實時代碼分析、安全漏洞檢測

### Alibaba Java Coding Guidelines
- **功能**：阿里 Java 規範檢查
- **特點**：符合國內開發規範

### CheckStyle-IDEA
- **功能**：代碼風格檢查
- **特點**：可自定義檢查規則

---

## 主題美化

### Material Theme UI
- **功能**：Material Design 主題
- **特點**：多種配色方案

### One Dark Theme
- **功能**：Atom 風格主題
- **特點**：護眼深色主題

### Atom Material Icons
- **功能**：文件圖標美化
- **特點**：Material 風格文件圖標

---

## 工具類

### Translation
- **功能**：翻譯插件
- **特點**：選中文字快速翻譯

### GsonFormatPlus
- **功能**：JSON 轉 Java 類
- **特點**：快速生成數據類

### String Manipulation
- **功能**：字符串處理
- **特點**：大小寫轉換、排序等

### .gitignore
- **功能**：.gitignore 文件支持
- **特點**：語法高亮、模板生成

---

## 安裝方法

1. Settings → Plugins
2. 選擇 Marketplace 標籤
3. 搜索插件名稱
4. 點擊 Install
