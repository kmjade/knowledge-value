# 導航快捷鍵

> [!summary] 功能概述
> IntelliJ IDEA 導航相關快捷鍵速查表，幫助快速在項目中移動。

---

## 全局搜索

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| Search Everywhere | Double Shift | Double Shift |
| 查找操作 | Ctrl+Shift+A | Cmd+Shift+A |
| 查找類 | Ctrl+N | Cmd+O |
| 查找文件 | Ctrl+Shift+N | Cmd+Shift+O |
| 查找符號 | Ctrl+Alt+Shift+N | Cmd+Option+O |

---

## 文件導航

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 最近文件 | Ctrl+E | Cmd+E |
| 最近編輯文件 | Ctrl+Shift+E | Cmd+Shift+E |
| 切換文件標籤 | Alt+Left/Right | Ctrl+Left/Right |
| 關閉當前標籤 | Ctrl+F4 | Cmd+W |

---

## 代碼導航

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 跳轉到定義 | Ctrl+B | Cmd+B |
| 跳轉到實現 | Ctrl+Alt+B | Cmd+Option+B |
| 跳轉到超類 | Ctrl+U | Cmd+U |
| 跳轉到行 | Ctrl+G | Cmd+L |
| 文件結構 | Ctrl+F12 | Cmd+F12 |

---

## 位置導航

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 後退 | Ctrl+Alt+Left | Cmd+[ |
| 前進 | Ctrl+Alt+Right | Cmd+] |
| 最後編輯位置 | Ctrl+Shift+Backspace | Cmd+Shift+Delete |

---

## 工具窗口

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 項目窗口 | Alt+1 | Cmd+1 |
| 結構窗口 | Alt+7 | Cmd+7 |
| 終端 | Alt+F12 | Option+F12 |
| 版本控制 | Alt+9 | Cmd+9 |
| 隱藏所有窗口 | Ctrl+Shift+F12 | Cmd+Shift+F12 |

---

## 聲明導航

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 查看快速定義 | Ctrl+Shift+I | Option+Space |
| 查看類型定義 | Ctrl+Shift+B | Ctrl+Shift+B |
| 查看參數信息 | Ctrl+P | Cmd+P |
