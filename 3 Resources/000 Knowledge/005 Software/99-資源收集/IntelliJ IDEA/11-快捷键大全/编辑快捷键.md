# 編輯快捷鍵

> [!summary] 功能概述
> IntelliJ IDEA 編輯相關快捷鍵速查表，分為 Windows/Linux 和 macOS 兩個版本。

---

## 基本編輯

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 撤銷 | Ctrl+Z | Cmd+Z |
| 重做 | Ctrl+Shift+Z | Cmd+Shift+Z |
| 剪切 | Ctrl+X | Cmd+X |
| 複製 | Ctrl+C | Cmd+C |
| 粘貼 | Ctrl+V | Cmd+V |
| 全選 | Ctrl+A | Cmd+A |
| 刪除行 | Ctrl+Y | Cmd+Delete |
| 複製當前行 | Ctrl+D | Cmd+D |

---

## 代碼補全

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 基本補全 | Ctrl+Space | Ctrl+Space |
| 智能補全 | Ctrl+Shift+Space | Ctrl+Shift+Space |
| 語句補全 | Ctrl+Shift+Enter | Cmd+Shift+Enter |
| 參數信息 | Ctrl+P | Cmd+P |

---

## 代碼生成

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 代碼生成菜單 | Alt+Insert | Cmd+N |
| 覆蓋方法 | Ctrl+O | Ctrl+O |
| 實現方法 | Ctrl+I | Ctrl+I |
| 環繞代碼 | Ctrl+Alt+T | Cmd+Option+T |

---

## 多光標編輯

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 添加光標 | Alt+Click | Option+Click |
| 選擇下一個相同詞 | Alt+J | Ctrl+G |
| 選擇所有相同詞 | Alt+Ctrl+Shift+J | Ctrl+Cmd+G |
| 列選擇模式 | Alt+Shift+Insert | Cmd+Shift+8 |

---

## 代碼格式化

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 格式化代碼 | Ctrl+Alt+L | Cmd+Option+L |
| 優化導入 | Ctrl+Alt+O | Ctrl+Option+O |
| 自動縮進 | Ctrl+Alt+I | Ctrl+Option+I |

---

## 查找替換

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 查找 | Ctrl+F | Cmd+F |
| 替換 | Ctrl+R | Cmd+R |
| 全局查找 | Ctrl+Shift+F | Cmd+Shift+F |
| 全局替換 | Ctrl+Shift+R | Cmd+Shift+R |

---

## 注釋

| 功能 | Windows/Linux | macOS |
|------|---------------|-------|
| 行註釋 | Ctrl+/ | Cmd+/ |
| 塊註釋 | Ctrl+Shift+/ | Cmd+Shift+/ |
