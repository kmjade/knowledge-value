# 界面介紹

> [!summary] 功能概述
> IntelliJ IDEA 界面由多個工具窗口組成，了解界面佈局是高效使用 IDE 的第一步。

---

## 界面組成

### 主要區域

```
┌─────────────────────────────────────────────────────────────┐
│ 菜單欄 (Menu Bar)                                           │
├─────────────────────────────────────────────────────────────┤
│ 工具欄 (Toolbar)                                            │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│ 項目     │                                                  │
│ 工具窗口 │              編輯器區域                          │
│ (Project)│            (Editor Area)                         │
│          │                                                  │
│          │                                                  │
├──────────┴──────────────────────────────────────────────────┤
│ 工具窗口欄 (Tool Window Bar)                                │
│ [Project] [Structure] [Favorites] [TODO] [Problems] [...]  │
├─────────────────────────────────────────────────────────────┤
│ 狀態欄 (Status Bar)                                         │
└─────────────────────────────────────────────────────────────┘
```

---

## 主要區域說明

### 1. 菜單欄 (Menu Bar)
| 菜單 | 功能 |
|------|------|
| File | 文件操作、項目管理 |
| Edit | 編輯操作、查找替換 |
| View | 視圖控制、工具窗口 |
| Navigate | 導航、搜索 |
| Code | 代碼生成、重構 |
| Refactor | 重構操作 |
| Build | 構建項目 |
| Run | 運行、調試 |
| Tools | 工具集 |
| VCS | 版本控制 |
| Window | 窗口管理 |
| Help | 幫助文檔 |

### 2. 項目工具窗口 (Project)
- 顯示項目文件結構
- 快捷鍵：`Alt+1` (Win) / `Cmd+1` (Mac)

### 3. 編輯器區域 (Editor Area)
- 代碼編輯的主區域
- 支持多標籤頁
- 支持分屏顯示

### 4. 工具窗口欄 (Tool Window Bar)
| 窗口 | 快捷鍵 | 功能 |
|------|--------|------|
| Project | Alt+1 | 項目結構 |
| Structure | Alt+7 | 類結構 |
| Favorites | Alt+2 | 收藏夾 |
| TODO | Alt+6 | 待辦事項 |
| Problems | Alt+6 | 問題列表 |
| Terminal | Alt+F12 | 終端 |
| Database | - | 數據庫工具 |
| Version Control | Alt+9 | 版本控制 |

---

## 快速操作

### 全局搜索
- **Search Everywhere**: `Double Shift`
- **Find Action**: `Ctrl+Shift+A` / `Cmd+Shift+A`
- **Find Class**: `Ctrl+N` / `Cmd+O`
- **Find File**: `Ctrl+Shift+N` / `Cmd+Shift+O`

### 視圖控制
- **隱藏所有工具窗口**: `Ctrl+Shift+F12` / `Cmd+Shift+F12`
- **全屏模式**: `Ctrl+Alt+F` / `Ctrl+Cmd+F`

---

## 注意事項

> [!tip] 最佳實踐
> - 善用 `Double Shift` 快速搜索任何內容
> - 自定義工具窗口佈局以適應工作習慣
> - 使用 `Ctrl+Shift+A` 查找任何操作
