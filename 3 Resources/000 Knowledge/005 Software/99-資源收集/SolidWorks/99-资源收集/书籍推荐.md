# 書籍推薦

> [!summary] 概述
> 推薦的 SolidWorks 學習書籍，涵蓋入門到高級各個階段。

---

## 入門書籍

### 《SolidWorks 基礎教程》
- **適合對象**：零基礎初學者
- **內容**：界面、草圖、基本特徵
- **特點**：圖文並茂，步驟詳細

### 《SolidWorks 從入門到精通》
- **適合對象**：自學者
- **內容**：全面覆蓋軟件功能
- **特點**：配套視頻教程

---

## 進階書籍

### 《SolidWorks 高級教程》
- **適合對象**：有一定基礎的用戶
- **內容**：曲面、鈑金、仿真
- **特點**：案例驅動

### 《SolidWorks API 開發指南》
- **適合對象**：需要二次開發的用戶
- **內容**：宏、VBA、API 編程
- **特點**：代碼示例豐富

---

## 參考手冊

### 《SolidWorks 命令參考手冊》
- **用途**：快速查閱命令用法
- **特點**：按功能分類，便於檢索

---

## 選書建議

1. **初學者**：選擇圖多、步驟詳細的入門書
2. **進階者**：選擇案例豐富的實戰書
3. **開發者**：選擇 API 相關的技術書
