# 快捷鍵大全

> [!summary] 功能概述
> SolidWorks 快捷鍵是提高建模效率的重要工具，本文整理了常用的快捷鍵列表。

---

## 文件操作

| 快捷鍵 | 功能 |
|--------|------|
| Ctrl+N | 新建文件 |
| Ctrl+O | 打開文件 |
| Ctrl+S | 保存文件 |
| Ctrl+Shift+S | 另存為 |
| Ctrl+W | 關閉文件 |
| Ctrl+P | 打印 |
| Ctrl+Tab | 切換打開的文件 |

---

## 視圖操作

| 快捷鍵 | 功能 |
|--------|------|
| 空格 | 視圖方向對話框 |
| Ctrl+1 | 前視 |
| Ctrl+2 | 後視 |
| Ctrl+3 | 左視 |
| Ctrl+4 | 右視 |
| Ctrl+5 | 上視 |
| Ctrl+6 | 下視 |
| Ctrl+7 | 等軸測 |
| F | 整屏顯示全圖 |
| Z | 縮小 |
| Shift+Z | 放大 |

---

## 草圖操作

| 快捷鍵 | 功能 |
|--------|------|
| S | 快捷工具欄 |
| L | 直線 |
| C | 圓 |
| R | 矩形 |
| A | 圓弧 |
| D | 智能標註 |
| Esc | 退出當前命令 |
| Delete | 刪除選中對象 |

---

## 特徵操作

| 快捷鍵 | 功能 |
|--------|------|
| B | 拉伸 (Extrude) |
| R | 旋轉 (Revolve) |
| S | 掃描 (Sweep) |
| F | 圓角 (Fillet) |
| Ctrl+B | 重建模型 |

---

## 選擇操作

| 快捷鍵 | 功能 |
|--------|------|
| Ctrl+A | 全選 |
| Ctrl+點擊 | 多選 |
| Shift+點擊 | 連續選擇 |
| 框選 | 矩形區域選擇 |

---

## 編輯操作

| 快捷鍵 | 功能 |
|--------|------|
| Ctrl+Z | 撤銷 |
| Ctrl+Y | 重做 |
| Ctrl+C | 複製 |
| Ctrl+V | 粘貼 |
| Ctrl+X | 剪切 |

---

## 顯示操作

| 快捷鍵 | 功能 |
|--------|------|
| Ctrl+Shift+8 | 顯示/隱藏所有註解 |
| Ctrl+Shift+T | 顯示/隱藏所有參考平面 |

---

## 自定義快捷鍵

> [!tip] 自定義方法
> 1. 工具 → 自定義
> 2. 選擇「鍵盤」選項卡
> 3. 找到需要修改的命令
> 4. 輸入新的快捷鍵
