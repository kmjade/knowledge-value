# 视图操作

> [!summary] 功能概述
> AutoCAD 視圖縮放、平移與控制技術，是高效繪圖的核心技能。

---

## 功能介紹

### 功能定位
- **所屬模塊**：基礎入門
- **功能類型**：視圖控制
- **前置條件**：熟悉界面

### 應用場景
1. 放大查看細節區域
2. 縮小查看整體佈局
3. 平移到不同區域
4. 恢復上一視圖

---

## 縮放命令（ZOOM）

### 命令調用方式

| 方式 | 命令/操作 |
|------|----------|
| 命令行 | `ZOOM` 或 `Z` |
| 工具欄 | 視圖 → 縮放 |
| 鼠標 | 滾輪滾動 |

### 縮放選項

```
ZOOM
指定窗口角點，輸入比例因子 (nX 或 nXP)，或者
[全部(A)/中心(C)/動態(D)/範圍(E)/上一個(P)/比例(S)/窗口(W)/對象(O)] <實時>:
```

| 選項 | 說明 | 使用場景 |
|------|------|---------|
| 全部 (A) | 顯示全部圖形 | 查看整體佈局 |
| 範圍 (E) | 顯示所有對象 | 查看所有繪圖內容 |
| 窗口 (W) | 窗口選擇區域 | 放大特定區域 |
| 上一個 (P) | 恢復上一視圖 | 返回之前視圖 |
| 比例 (S) | 輸入縮放比例 | 精確縮放 |
| 中心 (C) | 指定中心點 | 以指定點為中心縮放 |

### 常用縮放操作

**實時縮放**：
- 滾動鼠標滾輪
- 按住滾輪並拖動

**窗口縮放**：
```
ZOOM → W
指定第一個角點
指定對角點
```

**恢復上一視圖**：
```
ZOOM → P
```

## 平移命令（PAN）

### 命令調用方式

| 方式 | 命令/操作 |
|------|----------|
| 命令行 | `PAN` 或 `P` |
| 工具欄 | 視圖 → 平移 |
| 鼠標 | 按住滾輪拖動 |
| 快捷菜單 | 右鍵 → 平移 |

### 平移操作

**鼠標操作**：
- 按住鼠標滾輪
- 拖動到目標位置
- 釋放滾輪

**命令操作**：
```
PAN
按住鼠標左鍵拖動
按 Esc 或 Enter 退出
```

## 視圖控制工具

### 導航欄

位於繪圖區域右側，包含：

| 工具 | 功能 |
|------|------|
| SteeringWheels | 導航控制盤 |
| ViewCube | 視圖立方體 |
- 縮放 | 縮放工具 |
- 平移 | 平移工具 |

### ViewCube

三維視圖導航工具（主要用於三維建模）：
- 點擊面、邊、角點切換視圖
- 拖動旋轉視圖
- 點擊方向標籤恢復正交視圖

### SteeringWheels

圓形導航菜單：
- 縮放
- 平移
- 中心
- 漫遊
- 向上/向下
- 環視

## 視口管理

### 創建視口

**命令**：`VPORTS`

**選項**：
| 選項 | 說明 |
|------|------|
| 兩個 | 垂直或水平分割為兩個視口 |
| 三個 | 分割為三個視口 |
| 四個 | 分割為四個視口 |
| 合併 | 合併相鄰視口 |

### 視口操作

- 在不同視口中可以設置不同的視圖
- 在一個視口中的修改會同步到所有視口
- 適合同時查看整體和細節

---

## 視圖管理

### 保存視圖

**命令**：`VIEW`

```
VIEW
新建
輸入視圖名稱: 零件細節
```

### 恢復視圖

```
VIEW
選擇已保存的視圖
置為當前
```

### 常用視圖快捷鍵

| 快捷鍵 | 視圖 |
|--------|------|
| `Z + Enter + A + Enter` | 全部顯示 |
| `Z + Enter + E + Enter` | 範圍顯示 |
| `Z + Enter + P + Enter` | 上一視圖 |
| `P + Enter` | 平移 |

---

## 實用技巧

### 鼠標操作匯總

| 操作 | 功能 |
|------|------|
| 滾輪向前 | 放大 |
| 滾輪向後 | 縮小 |
| 按住滾輪拖動 | 平移 |
| 雙擊滾輪 | 範圍縮放 |
| Shift + 滾輪 | 旋轉視圖（三維） |

### 視圖導航技巧

1. **使用窗口縮放**：快速定位到細節區域
2. **保存常用視圖**：為常用區域保存命名視圖
3. **利用上一視圖**：在放大和整體之間快速切換
4. **多視口協作**：大屏幕可使用多視口

---

## 注意事項

> [!warning] 重要提示
> - 縮放和平移不會改變圖形的實際尺寸
> - 使用滾輪縮放時，鼠標位置為縮放中心
> - 雙擊滾輪可快速顯示全部圖形
