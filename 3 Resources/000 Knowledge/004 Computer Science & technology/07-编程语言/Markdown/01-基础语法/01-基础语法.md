# 01 - Markdown 基础语法

Markdown 基础语法覆盖日常写作的 90% 场景，所有 Markdown 方言都支持。

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

> 井号后要有空格。通常只用 2-4 级。

**替代写法**（仅一二级）：

```markdown
一级标题
========

二级标题
--------
```

## 段落与换行

```markdown
这是第一段。

这是第二段。（空行分隔段落）

这是同一段，  
末尾两个空格强制换行。
```

## 强调

```markdown
*斜体* 或 _斜体_
**粗体** 或 __粗体__
***粗斜体*** 或 ___粗斜体___
~~删除线~~
```

实际效果：*斜体* **粗体** ***粗斜体*** ~~删除线~~

## 列表

### 无序列表

```markdown
- 项目一
- 项目二
  - 子项目（缩进两个空格）
  - 子项目二
* 也可以用星号
+ 或者加号
```

### 有序列表

```markdown
1. 第一步
2. 第二步
   1. 子步骤（缩进三个空格）
   2. 子步骤二
3. 第三步
```

### 任务列表（GFM）

```markdown
- [ ] 待办事项
- [x] 已完成
- [ ] 进行中
```

## 链接

```markdown
[显示文字](https://example.com)
[显示文字](https://example.com "鼠标悬停提示")
<https://example.com>  自动链接
[引用式链接][ref]

[ref]: https://example.com
```

## 图片

```markdown
![替代文字](图片URL)
![替代文字](图片URL "图片标题")
[![图片](图片URL)](链接URL)  点击图片跳转
```

## 引用

```markdown
> 这是一级引用
>> 这是二级嵌套引用
>>> 三级嵌套

> 引用可以包含其他元素
> - 列表
> - **强调**
> - [链接](https://example.com)
```

> 这是一级引用
>> 这是二级嵌套引用

## 水平线

```markdown
---
***
___
```

## 转义字符

```markdown

\# 不渲染为标题
\` 不渲染为代码
```
