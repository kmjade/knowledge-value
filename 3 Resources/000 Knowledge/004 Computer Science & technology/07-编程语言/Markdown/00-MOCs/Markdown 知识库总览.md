# Markdown 知识库总览

> Markdown 是一种轻量级标记语言，由 John Gruber 和 Aaron Swartz 于 2004 年创建。
> 它以纯文本格式编写，可以轻松转换为 HTML，是技术写作的事实标准。

## 为什么用 Markdown？

1. **纯文本** — 任何编辑器都能打开，永不失效
2. **易读易写** — 源码即排版，不需要学复杂语法
3. **版本控制友好** — Git diff 清晰可读
4. **生态丰富** — 静态网站生成器、笔记软件、文档工具都支持
5. **转格式灵活** — 可转为 HTML、PDF、Word、LaTeX 等
