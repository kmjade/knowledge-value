# CS 资源收集

## 📚 经典教材（按主题）

| 书名 | 主题 | 说明 |
|------|------|------|
| 《算法导论》(CLRS) | 算法 | 算法百科全书，参考用而非通读 |
| 《算法 (第 4 版)》Sedgewick | 算法 | 更友好的入门，Java 实现 |
| 《深入理解计算机系统》(CS:APP) | 系统 | 覆盖组成原理+OS+网络的最好的单本书 |
| 《计算机组成与设计》Patterson & Hennessy | 组成原理 | RISC-V 版，经典教材 |
| 《现代操作系统》Tanenbaum | 操作系统 | OS 经典，结合 Linux 源码 |
| 《计算机网络：自顶向下方法》Kurose & Ross | 网络 | 最好的网络入门教材 |
| 《数据库系统概念》Silberschatz | 数据库 | 数据库理论经典 |
| 《高性能 MySQL》 | 数据库实战 | MySQL 调优必读 |
| 《编译原理》(龙书) | 编程语言 | 经典但重——先读 Engineering a Compiler |
| 《设计模式》GoF | 软件工程 | 23 种经典设计模式 |
| 《重构》Martin Fowler | 软件工程 | 改善既有代码的设计 |
| 《DDIA》(设计数据密集型应用) | 分布式 | 分布式系统必读，2026 年仍是必读 |
