# 08 算法與複雜度

## Big-O 複雜度
| 複雜度 | 名稱 | 例子 |
|:--:|------|------|
| O(1) | 常數 | 陣列索引 |
| O(log n) | 對數 | 二分搜尋 |
| O(n) | 線性 | 遍歷陣列 |
| O(n log n) | 線性對數 | 快速排序 / 合併排序 |
| O(n²) | 平方 | 冒泡排序 |
| O(2ⁿ) | 指數 | 暴力破解 TSP |

## 核心演算法
| 演算法 | 類型 | 複雜度 |
|------|------|:--:|
| **Binary Search** | 搜尋 | O(log n) |
| **Quick Sort** | 排序 | O(n log n) |
| **Merge Sort** | 排序 | O(n log n) |
| **BFS** | 圖遍歷 | O(V+E) |
| **DFS** | 圖遍歷 | O(V+E) |
| **Dijkstra** | 最短路徑 | O(V²) |
