# 09 大數據與分佈式系統

## 大數據 4V
- **Volume** (規模) — TB–PB 級
- **Velocity** (速度) — 即時流處理
- **Variety** (多樣) — 結構化/半結構化/非結構化
- **Veracity** (真實性) — 數據品質

## 核心技術棧
| 層級 | 工具 | 用途 |
|------|------|------|
| 儲存 | HDFS、S3 | 分佈式文件系統 |
| 批次處理 | Hadoop MapReduce、Spark | 大規模離線計算 |
| 流處理 | Kafka、Flink | 即時數據管道 |
| 查詢 | Hive、Presto、Spark SQL | SQL on Big Data |
| NoSQL | HBase、Cassandra | 分佈式資料庫 |
