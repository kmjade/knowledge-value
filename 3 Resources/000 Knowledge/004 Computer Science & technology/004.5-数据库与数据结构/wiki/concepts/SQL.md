# SQL 結構化查詢語言

## 定義
SQL 是關聯式數據庫的標準查詢語言，用於定義 (DDL)、操作 (DML)、控制 (DCL) 和查詢 (DQL) 數據。

## 四大子語言
| 子語言 | 關鍵字 | 用途 |
|--------|--------|------|
| DDL | CREATE/ALTER/DROP | 定義結構 |
| DML | INSERT/UPDATE/DELETE | 操作數據 |
| DQL | SELECT | 查詢數據 |
| DCL | GRANT/REVOKE | 權限控制 |
