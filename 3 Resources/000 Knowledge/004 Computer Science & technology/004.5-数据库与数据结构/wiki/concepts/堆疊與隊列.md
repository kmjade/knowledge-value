# 堆疊與隊列 Stack & Queue

## Stack (LIFO 後進先出)
| 操作 | 複雜度 |
|------|:--:|
| push | O(1) |
| pop | O(1) |
| peek/top | O(1) |

應用: 函數調用堆疊、Undo/Redo、括號匹配、DFS

## Queue (FIFO 先進先出)
| 操作 | 複雜度 |
|------|:--:|
| enqueue | O(1) |
| dequeue | O(1) |
| peek/front | O(1) |

應用: BFS、任務排程、訊息隊列、印表機佇列
