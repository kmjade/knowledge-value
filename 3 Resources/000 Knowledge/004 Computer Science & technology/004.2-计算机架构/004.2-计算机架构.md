## 计算机架构的核心层次

```
┌──────────────────────────────────────┐
│          Application 应用层           │
│  程序看到的世界：变量、函数、对象        │
├──────────────────────────────────────┤
│     Instruction Set Architecture     │
│         指令集架构 (ISA)              │
│  x86 / ARM / RISC-V                  │
│  软件与硬件的分界线 ← 004.23          │
├──────────────────────────────────────┤
│       Microarchitecture 微架构        │
│  流水线、超标量、乱序执行、分支预测      │
│  ← 004.24 处理器架构                  │
├──────────────────────────────────────┤
│          Memory Hierarchy            │
│          存储层次                     │
│  L1→L2→L3 Cache → RAM → SSD/HDD     │
│  ← 004.25 存储器架构                  │
├──────────────────────────────────────┤
│         Digital Logic 数字逻辑        │
│  门电路、触发器、ALU、数据通路          │
│  ← 004.3 硬件                         │
└──────────────────────────────────────┘
```
