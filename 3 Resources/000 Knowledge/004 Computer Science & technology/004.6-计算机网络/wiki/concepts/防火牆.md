# 防火牆 Firewall

## 定義
根據安全規則監控和控制進出網絡流量的系統 — 網絡安全的第一道防線。

## 類型
| 類型 | 工作層 | 功能 |
|------|:--:|------|
| Packet Filter | L3–4 | IP+Port 過濾 (ACL) |
| Stateful | L3–4 | 追蹤連接狀態 |
| Application (WAF) | L7 | HTTP 內容檢測 |
| NGFW | L3–7 | IDS/IPS+DPI+應用感知 |

## 默認策略: 白名單 (預設拒絕，允許特定) > 黑名單 (預設允許，拒絕特定)
