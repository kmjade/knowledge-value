# 01 計算機硬件導論

## 硬件全景 (Hardware Panorama)

計算機硬件由六大子系統構成：

| 子系統 | 核心組件 | 功能 |
|--------|----------|------|
| **運算核心** | CPU (Central Processing Unit) | 執行指令、數據處理 |
| **平台基礎** | 主機板 (Motherboard) | 連接所有組件、供電、I/O |
| **記憶體** | RAM (Random Access Memory) | 執行時期暫存數據 |
| **存儲** | SSD / HDD (Storage) | 持久化數據保存 |
| **供電** | PSU (Power Supply Unit) | 交流轉直流、穩定供電 |
| **輸入輸出** | I/O Devices | 人機互動、數據交換 |

## Von Neumann vs Harvard 架構

| 特性 | Von Neumann | Harvard |
|------|-------------|---------|
| 指令/數據總線 | 共享同一總線 | 分離總線 |
| 瓶頸 | Von Neumann Bottleneck | 並行存取 |
| 應用 | 通用 PC | 嵌入式/DSP/微控制器 |
| 現代變體 | Modified Harvard (L1 指令/數據 Cache 分離) | — |

> 現代 PC 使用 Modified Harvard：CPU 內部 Cache 分離指令/數據，但外部記憶體總線共享。
