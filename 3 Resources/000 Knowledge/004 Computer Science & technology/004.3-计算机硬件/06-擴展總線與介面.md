# 06 擴展總線與介面 (Expansion Bus & Interfaces)

## PCI Express (PCIe)

### 世代頻寬

| 世代 | 每通道速率 (GT/s) | ×1 頻寬 | ×4 頻寬 | ×16 頻寬 | 編碼 |
|:----:|:-----------------:|:-------:|:-------:|:--------:|:----:|
| **PCIe 3.0** | 8 GT/s | ~1 GB/s | ~4 GB/s | ~16 GB/s | 128b/130b |
| **PCIe 4.0** | 16 GT/s | ~2 GB/s | ~8 GB/s | ~32 GB/s | 128b/130b |
| **PCIe 5.0** | 32 GT/s | ~4 GB/s | ~16 GB/s | ~64 GB/s | 128b/130b |
| **PCIe 6.0** | 64 GT/s | ~8 GB/s | ~32 GB/s | ~128 GB/s | PAM4+FEC |
| **PCIe 7.0** (2027) | 128 GT/s | ~16 GB/s | ~64 GB/s | ~256 GB/s | PAM4 |

### 通道配置 (Lane Configuration)

| 設備 | 典型 PCIe 通道需求 |
|------|:------------------:|
| GPU (獨立顯卡) | ×16 (Gen 4/5) |
| NVMe SSD | ×4 (Gen 3/4/5) |
| 網卡 (10GbE) | ×4 (Gen 3) |
| 音效卡/擷取卡 | ×1 |
| Thunderbolt 4 控制器 | ×4 (Gen 3) |

## SATA (Serial ATA)

| 版本 | 頻寬 | 應用 |
|:----:|:----:|------|
| SATA I | 1.5 Gb/s (150 MB/s) | 歷史 |
| SATA II | 3.0 Gb/s (300 MB/s) | 舊 HDD |
| SATA III | 6.0 Gb/s (600 MB/s) | SATA SSD / HDD |

## USB (Universal Serial Bus)

| 版本 | 最大速率 | 連接器 | 供電 (最大) | 年代 |
|:----:|:--------:|--------|:----------:|:----:|
| USB 2.0 | 480 Mbps | Type-A/B/Mini/Micro | 2.5W (5V/0.5A) | 2000 |
| USB 3.0/3.1 Gen 1 | 5 Gbps | Type-A/C | 4.5W (5V/0.9A) | 2008 |
| USB 3.1 Gen 2 | 10 Gbps | Type-A/C | 7.5W (5V/1.5A) | 2013 |
| USB 3.2 Gen 2×2 | 20 Gbps | Type-C only | 7.5W | 2017 |
| USB4 (Gen 2×2) | 20 Gbps | Type-C only | 100W (PD 3.0) | 2019 |
| USB4 (Gen 3×2) | 40 Gbps | Type-C only | 240W (PD 3.1) | 2022 |
| USB4 v2 | 80 Gbps | Type-C only | 240W+ | 2024 |

> USB4 = Thunderbolt 3 協議基礎。USB4 v2 = Thunderbolt 5 級別。

## Thunderbolt

| 版本 | 速率 | 介面 | PCIe 通道 | 特點 |
|:----:|:----:|------|:---------:|------|
| TB3 | 40 Gbps | USB-C | PCIe 3.0 ×4 | 雙 4K / 單 5K |
| TB4 | 40 Gbps | USB-C | PCIe 3.0 ×4 | 最低要求更嚴、Hub 支援 |
| TB5 | 80/120 Gbps | USB-C | PCIe 4.0 ×4 | 多 8K / 120Hz |

## M.2 介面

| Key | 缺口位置 | 協議 | 典型用途 |
|:---:|----------|------|----------|
| **M Key** | 右 | PCIe ×4 + SATA | NVMe SSD |
| **B Key** | 左 | PCIe ×2 + SATA | SATA SSD |
| **B+M Key** | 雙缺口 | PCIe ×2 + SATA | 相容 SATA/NVMe |
| **E Key** | 特殊 | PCIe ×1 + USB | WiFi/BT 網卡 |
| **A Key** | 特殊 | PCIe ×1 + USB | WiFi/BT (舊) |
