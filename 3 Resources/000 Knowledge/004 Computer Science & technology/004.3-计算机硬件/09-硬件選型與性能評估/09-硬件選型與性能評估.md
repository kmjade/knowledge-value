# 09 硬件選型與性能評估 (Hardware Selection & Benchmark)

## 用途匹配 (Use-Case Matching)

| 用途 | CPU 重點 | GPU 重點 | RAM | 存儲 | 預算範圍 |
|------|----------|----------|:---:|------|:--------:|
| **文書/上網** | i3/R3 | iGPU 即可 | 8–16GB | 512GB SSD | $300–500 |
| **輕度遊戲** (eSports) | i5/R5 | RTX 4060 / RX 7600 | 16GB | 1TB SSD | $600–900 |
| **3A 遊戲** (1440p) | i5/R5 (X3D) | RTX 4070S / RX 7800 XT | 32GB | 2TB NVMe | $1,200–1,800 |
| **4K 遊戲** | i7/R7 (X3D) | RTX 4090 / 5090 | 32GB | 2TB NVMe | $2,500+ |
| **內容創作** | i7-i9/R7-R9 | RTX 4070 Ti+ | 32–64GB | 2TB+ NVMe | $1,500–3,000 |
| **工作站** | Threadripper/Xeon W | RTX A 系列 / Pro | 64–128GB | RAID NVMe | $4,000+ |
| **伺服器** | EPYC / Xeon | 無 / ASPEED | 128–512GB ECC | RAID HDD/SSD | $3,000+ |
| **HTPC / NAS** | i3/N100 | iGPU | 8–16GB | HDD ×N | $300–800 |

## 瓶頸分析 (Bottleneck Analysis)

```
瓶頸 = 限制系統性能的最慢組件

檢查方法:
├── CPU 瓶頸: GPU 使用率 < 90% + CPU 使用率 > 90%
├── GPU 瓶頸: GPU 使用率 > 95% (遊戲中正常)
├── RAM 瓶頸: RAM 使用率 > 90% (交換至 SSD)
├── 存儲瓶頸: 磁碟使用率 100%、響應時間 > 100ms
└── 散熱瓶頸: 溫度 > 95°C → 降頻 (Thermal Throttling)
```

| 瓶頸類型 | 症狀 | 解決方案 |
|----------|------|----------|
| CPU 瓶頸 | GPU 閒置、FPS 不穩、0.1% Low 低 | 升級 CPU / 提高解析度 |
| GPU 瓶頸 | FPS 低但 CPU 空閒 | 升級 GPU / 降低畫質 |
| RAM 瓶頸 | 多工卡頓、程式崩潰 | 增加容量 / 雙通道 |
| VRAM 瓶頸 | 紋理載入慢、卡頓 | 降低紋理品質 / 升級 GPU |

## 性價比分析 (Price-Performance)

### C/P 值公式
```
C/P = 性能分數 / 價格
選購原則: 主流價位帶 C/P 最高，旗艦產品 C/P 遞減 (Diminishing Returns)
```

### GPU C/P 熱點 (2024–2025)
- **入門**: RTX 4060 / RX 7600 (~$300)
- **甜蜜點**: RTX 4070 Super / RX 7800 XT (~$500–600)
- **高階**: RTX 4070 Ti Super (~$800)
- **旗艦**: RTX 4090 (~$1,600, 但 C/P 低)

## 升級路徑 (Upgrade Path)

### 升級優先級

| 優先級 | 升級項目 | 條件 | 預期提升 |
|:------:|----------|------|:--------:|
| 1 | SSD (HDD→NVMe) | 仍用 HDD 開機 | 開機速度 10×↑ |
| 2 | GPU | 遊戲/渲染性能不足 | FPS 2-4×↑ |
| 3 | RAM | <16GB 或單通道 | 多工順暢 |
| 4 | CPU | CPU 瓶頸明顯 | 視世代差 |
| 5 | PSU | 瓦數不足/老舊 | 穩定性↑ |
| 6 | 機殼/散熱 | 溫度高 | 降頻問題解決 |

### 平台升級檢查

| 升級 | 需同步升級 |
|------|-----------|
| 新一代 CPU | 主機板 (Socket 不同) + 可能 RAM (DDR4→DDR5) |
| 新 GPU | 可能 PSU (瓦數不足) |
| DDR4→DDR5 | 主機板 (插槽不相容) |
| PCIe 4.0→5.0 SSD | 主機板 + CPU (需支援) |
