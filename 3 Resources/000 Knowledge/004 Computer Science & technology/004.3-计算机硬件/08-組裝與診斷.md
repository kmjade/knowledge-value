# 08 組裝與診斷 (Assembly & Diagnostics)

## DIY 組裝步驟 (Assembly Steps)

### 準備階段
| 步驟 | 動作 | 注意 |
|:----:|------|------|
| 1 | 準備工具 | 十字螺絲刀、散熱膏、束線帶、防靜電手環 |
| 2 | 工作環境 | 乾燥、無靜電、充足光線、大平面 |
| 3 | 檢查零件 | 對照清單、檢查針腳彎曲 (CPU Socket) |

### 組裝流程 (Build Order)

```
1. CPU → 主機板 (注意方向三角標記)
2. RAM → 主機板 (雙通道插槽: A2+B2 優先)
3. M.2 SSD → 主機板 (安裝前撕散熱片保護膜)
4. 散熱器 → CPU (散熱膏米粒大小 / X 塗法)
5. 主機板 → 機殼 (先裝 I/O 擋板!)
6. PSU → 機殼 (風扇朝下或朝前, 視機殼)
7. 接線:
   ├── 24-pin ATX (主機板主供電)
   ├── 8-pin EPS (CPU 供電)
   ├── PCIe 6+2-pin (GPU 供電)
   ├── SATA 供電 (HDD/SSD)
   ├── 前面板接線 (Power SW / Reset SW / HDD LED / Power LED ±)
   ├── USB 3.0 / USB-C 前面板
   └── 音頻前面板 (HD Audio)
8. GPU → PCIe 插槽 (最上方 ×16 槽)
9. 理線 (Cable Management)
10. 首次開機測試 (裸測可選)
```

### 首次開機檢查

| 檢查項 | 狀態 | 異常處理 |
|--------|:----:|----------|
| 風扇運轉 | 所有風扇應轉動 | 若否 → 檢查 24-pin/8-pin 連接 |
| POST 畫面 | 顯示 BIOS/UEFI Logo | 若無 → 檢查螢幕連接/GPU 供電 |
| Debug LED | 依主機板說明 | CPU→DRAM→VGA→BOOT 依次熄滅 |
| BIOS 識別 | CPU/RAM/SSD 正確顯示 | 若否 → 重新安裝該組件 |

## POST / Beep Codes (開機自我檢測)

### 常見 Beep Code (AMI BIOS)

| Beep | 含義 | 排查 |
|:----:|------|------|
| 1 短 | 正常啟動 ✅ | — |
| 2 短 | 記憶體錯誤 | 重插 RAM、清潔金手指、換插槽 |
| 3 短 | 基本記憶體錯誤 | 同上 |
| 4 短 | 系統計時器錯誤 | 主機板故障 |
| 5 短 | CPU 錯誤 | 重裝 CPU、檢查供電 |
| 6 短 | 鍵盤控制器錯誤 | USB 衝突 |
| 7 短 | CPU 異常中斷 | CPU/主機板 |
| 8 短 | 顯卡記憶體錯誤 | 重插 GPU、檢查供電 |
| 1 長 2 短 | 顯卡錯誤 | GPU 未正確安裝 |
| 連續長音 | 記憶體未安裝 | 檢查 RAM 安裝 |

### Debug LED 順序

```
開機 → [CPU 亮] → [DRAM 亮] → [VGA 亮] → [BOOT 亮] → 全部熄滅 → 進入 OS
          ↓卡住?          ↓卡住?        ↓卡住?       ↓卡住?
        CPU/供電問題    RAM 問題     GPU 問題     SSD/開機裝置
```

## 常見故障排除

### 無法開機 (No Boot)

| 症狀 | 可能原因 | 解決方案 |
|------|----------|----------|
| 完全無反應 | PSU 未通電 / 開關未開 | 檢查 PSU 開關、電源線、插座 |
| 風扇轉但無畫面 | RAM/GPU/CPU 問題 | 單根 RAM 測試、換插槽 |
| 風扇轉一下就停 | 短路保護 | 檢查主機板銅柱 (Standoff) |
| 無限重啟 | RAM 相容性 / XMP 不穩 | Clear CMOS、關閉 XMP |

### 藍屏 (BSOD)

| 錯誤碼 | 含義 | 可能原因 |
|--------|------|----------|
| MEMORY_MANAGEMENT | 記憶體錯誤 | RAM 故障 / XMP 不穩 |
| WHEA_UNCORRECTABLE_ERROR | 硬體錯誤 | CPU/RAM 不穩、電壓不足 |
| KERNEL_SECURITY_CHECK_FAILURE | 驅動程式 | 驅動衝突 |
| CRITICAL_PROCESS_DIED | 系統程序 | SSD 故障 |
| DPC_WATCHDOG_VIOLATION | SSD 驅動 | NVMe 驅動問題 |

### 過熱 (Overheating)

| 組件 | 正常溫度 | 危險溫度 | 排查 |
|------|:--------:|:--------:|------|
| CPU (待機) | 30–45°C | — | — |
| CPU (滿載) | 60–85°C | >95°C | 重塗散熱膏、檢查散熱器安裝 |
| GPU (滿載) | 65–85°C | >90°C | 清理灰塵、改善機殼風道 |
| SSD | 30–50°C | >70°C | 檢查散熱片、避免與 GPU 背板接觸 |

## 診斷工具 (Diagnostic Tools)

| 工具 | 用途 | 獲取 |
|------|------|------|
| **MemTest86** | RAM 穩定性測試 | USB 開機運行 |
| **CrystalDiskInfo** | SSD/HDD 健康狀態 (SMART) | Windows 免費 |
| **HWMonitor / HWiNFO** | 溫度/電壓/風扇監控 | 免費 |
| **OCCT** | CPU/GPU/PSU 壓力測試 | 免費 |
| **CPU-Z / GPU-Z** | 硬體規格識別 | 免費 |
| **BlueScreenView** | BSOD Dump 分析 | 免費 |
