# 計算機科學與技術 — DDC 004

> [!info] DDC 第 004 類 · 數據處理與計算機科學
> 原屬 DDC 000 總類——因內容龐大獨立建庫。
> 計算的本質、硬體與軟體的設計、數據的處理與管理。

---

## DDC 004 在分類體系中的位置

|   DDC   | 類目                       |        歸屬         |
| :-----: | ------------------------ | :---------------: |
| **000** | [[000 Knowledge\|總類·知識]] |       知識總論        |
| **004** | **計算機科學與技術**             |      ← 你在這裡       |
| **005** | 程序設計                     | 000 總類（高階語言·操作系統） |
| **006** | 人工智能                     |   000 總類（已獨立建庫）   |
| **600** | 技術                       | 技術大類（與 004 有大量交叉） |

> **為什麼 004 在 000 而不是 600？** —— DDC 1876 年設計時計算機還不存在。004 最初是「數據處理」的狹義類目，隨計算機科學膨脹而擴展，但歷史遺留歸在 000。600 技術類偏重於傳統應用工程（機械·電氣·化工）。

## 四大分類法對照：DDC · UDC · CLC · LCC

計算機科學在四大分類體系中的位置對照：

| 體系 | 類號 | 名稱 | 隸屬大類 |
|:----:|:----:|------|----------|
| **DDC** | 004 | Data processing & Computer science | 000 總類 |
| **UDC** | 004 | Computer science and technology | 0 科學與知識 |
| **CLC** | TP3 | 計算技術、計算機技術 | T 工業技術 |
| **LCC** | QA75.5–76.95 | Computer science | Q 科學（數學） |

### UDC 004 細分

UDC（國際十進分類法）的 004 比 DDC 更現代化，直接反映計算機科學的層次結構：

| UDC | 名稱 | 對應 DDC |
|:---:|------|:--------:|
| 004.2 | 計算機體系結構 Computer architecture | 004.22 |
| 004.3 | 計算機硬件 Hardware | 004.1 |
| 004.4 | 軟件 Software（含操作系統、程式語言） | 005 |
| 004.5 | 人機交互 Human-computer interaction | 004.019 |
| 004.6 | 數據 Data（含數據庫、大數據） | 004.5 |
| 004.7 | 計算機通信 Computer communication | 004.6 |
| 004.8 | 人工智能 Artificial intelligence | 006 |
| 004.9 | 應用導向技術（Web、VR、生物信息學等） | 004.9 |

> **UDC vs DDC 關鍵差異**：UDC 將 OS、程式語言歸入 004.4（軟件），DDC 則單列 005 為程序設計——這反映了 UDC 更偏向「計算機作為一個整體系統」的視角。

### CLC TP3 細分

CLC（中國圖書館分類法）將計算機歸入 **T 工業技術** 大類：

| CLC | 名稱 | 對應 DDC |
|:---:|------|:--------:|
| TP3-0 | 計算機理論與方法 | 004.01 |
| TP30 | 一般性問題（可靠性、安全） | 004 |
| TP31 | 計算機軟件（程式設計、數據庫） | 005 |
| TP32 | 一般計算器和計算機 | — |
| TP33 | 電子數字計算機 | 004.1 |
| TP34 | 電子模擬計算機 | — |
| TP35 | 混合電子計算機 | — |
| TP36 | 微型計算機（PC、單片機） | 004.16 |
| TP37 | 多媒體技術與計算機 | 006.7 |
| TP38 | 其他計算機（AI 專用、量子） | 004.1/006.3 |
| TP39 | 計算機應用 | 004.9 |
| TP391 | 信息處理（文字、圖像、模式識別） | 006.4 |

> **CLC 特點**：將「計算機」視為工業技術（T 類）而非總類（DDC 000），與中國對科技的分類邏輯一致——技術服務於生產力。

### LCC QA75.5–76.95 細分

LCC（美國國會圖書館分類法）將計算機科學歸入 **Q 科學** 下的 **QA 數學**：

| LCC | 名稱 | 對應 DDC |
|:---:|------|:--------:|
| QA75.5–76.765 | 一般著作、概論 | 004 |
| QA76.75–76.765 | 計算機軟件 Software | 005 |
| QA76.76 | 計算機程式設計 Programming | 005.1 |
| QA76.9 | 其他專題（安全、人機交互、數據庫） | 004.019/005.8 |
| TK7885–7895 | 計算機工程（硬件） | 621.39 |

> **LCC 特點**：將計算機科學視為數學的延伸（QA），硬件工程則歸入電氣工程（TK）。這反映了北美學術傳統——CS 源於數學系。
