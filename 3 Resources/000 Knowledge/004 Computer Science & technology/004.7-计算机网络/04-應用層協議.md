# 04 應用層協議

## HTTP (HyperText Transfer Protocol)
- **無狀態** — 每次請求獨立
- **Method**: GET (讀取), POST (提交), PUT (更新), DELETE (刪除)
- **狀態碼**: 2xx 成功 / 3xx 重定向 / 4xx 客戶端錯誤 / 5xx 伺服器錯誤
- **HTTPS** = HTTP + TLS (加密通道)

## DNS (Domain Name System)
域名 → IP 解析: 遞迴查詢 → Root DNS → TLD → Authoritative

## 其他協議
