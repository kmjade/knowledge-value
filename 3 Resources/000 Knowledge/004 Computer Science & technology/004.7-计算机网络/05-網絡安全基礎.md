# 05 網絡安全基礎

## CIA 三元組
- **機密性** (Confidentiality) — 加密、存取控制
- **完整性** (Integrity) — 數位簽章、HMAC
- **可用性** (Availability) — DDoS 防護、冗餘

## TLS/SSL 握手
Client Hello → Server Hello + 憑證 → 密鑰交換 → 對稱加密通訊

## 防火牆類型
| 類型 | 層級 | 功能 |
|------|:--:|------|
| Packet Filter | L3/4 | IP+Port 過濾 |
| Stateful | L3/4 | 追蹤連接狀態 |
| Application (WAF) | L7 | HTTP 內容檢測 |
| NGFW | L3–7 | IPS/IDS + DPI |

## 常見攻擊: DDoS、MITM、SQL Injection、XSS、Phishing
