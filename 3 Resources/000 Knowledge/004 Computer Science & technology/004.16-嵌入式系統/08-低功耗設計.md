# 08 低功耗設計

## 功耗來源
P_total = P_dynamic (切換) + P_static (漏電)
P_dynamic ∝ C × V² × f   (電容 × 電壓² × 頻率)

## 降功耗策略
| 策略 | 效果 |
|------|------|
| **降頻** | 線性降動態功耗 |
| **降壓** | 平方效果 (最有效!) |
| **休眠模式** | Sleep → Stop → Standby → Shutdown |
| **時鐘閘控** | 關閉閒置模塊時鐘 |
| **DMA** | CPU 休眠，DMA 傳輸數據 |

## STM32 低功耗模式
- **Sleep** — CPU 停，周邊運行，任意中斷喚醒
- **Stop** — 高速時鐘停，RTC 可運行，EXIT 喚醒
- **Standby** — 全關，僅 RTC/WKUP 喚醒，SRAM 丟失

## 物聯網場景
LoRa/NB-IoT/BLE — 目標：一顆鈕扣電池運行數年
