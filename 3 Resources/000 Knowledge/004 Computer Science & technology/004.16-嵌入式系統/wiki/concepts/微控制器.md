# 微控制器 Microcontroller (MCU)

## 定義
將 CPU、記憶體 (Flash/SRAM) 和 I/O 周邊整合在**單一晶片**上的微型計算機，專為嵌入式控制應用設計。

## 核心組成
| 模塊 | 功能 |
|------|------|
| **CPU 核心** | ARM Cortex-M / RISC-V / AVR |
| **Flash** | 程式存儲 (KB–MB) |
| **SRAM** | 運行時數據 (KB 級) |
| **GPIO** | 通用數位 I/O |
| **計時器** | PWM、輸入捕獲、RTC |
| **通訊介面** | UART/SPI/I2C/CAN |
| **ADC/DAC** | 類比↔數位轉換 |

## MCU vs MPU: MCU 內建 Flash+SRAM、mW 功耗、RTOS/裸機；MPU 需外部 DDR、W 級功耗、Linux
