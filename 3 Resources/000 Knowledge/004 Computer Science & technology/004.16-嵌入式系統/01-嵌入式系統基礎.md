# 01 嵌入式系統基礎

## 什麼是嵌入式系統
以**微控制器 (MCU)** 或**微處理器 (MPU)** 為核心，嵌入到設備中執行特定功能的計算機系統。與通用 PC 不同：資源受限、即時性要求、低功耗。

## 核心組成
| 組件 | 功能 |
|------|------|
| **MCU/MPU** | 中央處理 + 記憶體 |
| **記憶體** | Flash (程式)、SRAM (數據)、EEPROM |
| **I/O 介面** | GPIO、UART、SPI、I2C、ADC |
| **時鐘** | 系統時鐘、RTC |
| **電源管理** | 穩壓、休眠喚醒 |

## 設計流程
需求分析 → 硬體選型 → 原理圖 → PCB → 韌體開發 → 測試 → 量產
