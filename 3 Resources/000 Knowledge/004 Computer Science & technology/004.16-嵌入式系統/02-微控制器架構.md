# 02 微控制器架構

## MCU vs MPU
| | MCU | MPU |
|:--|:---|:---|
| 核心 | Cortex-M | Cortex-A |
| 記憶體 | 內建 Flash+SRAM | 外部 DDR |
| OS | Bare-metal / RTOS | Linux |
| 功耗 | mW 級 | W 級 |
| 應用 | 感測器、家電 | 手機、平板 |

## 核心架構
- **Harvard vs Von Neumann** — 程式與數據總線分離/合一
- **RISC vs CISC** — ARM (RISC) vs x86 (CISC)
- **管線 Pipeline** — 取指→解碼→執行→寫回

## 記憶體層次
Register → Cache → SRAM → Flash → 外部存儲
