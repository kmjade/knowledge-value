# 07 C51 / 8051 工具鏈 (C51 / 8051 Toolchain)

> C51 是 Keil 針對 8051/8052 微控制器的經典 C 編譯器，仍在許多 Legacy 產品和教學中使用。

## C51 編譯器特性 C51 Compiler Features

| 特性 | 說明 |
|:-----|:-----|
| **記憶體模型** | SMALL / COMPACT / LARGE（影響 data/pdata/xdata 選擇） |
| **code 關鍵字** | `code` 將常數放入 ROM (Flash) |
| **data 關鍵字** | `data` 直接尋址 RAM (128B) |
| **idata** | 間接尋址 RAM (256B) |
| **xdata** | 外部擴展 RAM (64KB) |
| **bit** | 位元變數 (bit-addressable) |
| **sfr** | 特殊功能暫存器定義 (`sfr P0 = 0x80;`) |
| **sbit** | 位元定址 SFR (`sbit LED = P0^0;`) |
| **_at_** | 絕對位址變數放置 |

## 8051 記憶體佈局 Memory Layout (Harvard Architecture)

```
8051 Memory Map
├── CODE  (64KB) — 程式碼 Flash
├── DATA  (128B) — 內部直接定址 RAM (data)
├── IDATA (256B) — 內部間接定址 RAM (idata)
├── XDATA (64KB) — 外部 RAM (xdata)
├── SFR   (128B) — 特殊功能暫存器 (P0/P1/TMOD/...)
└── BIT   (16B)  — 位元定址區 (20H-2FH)
```

## 記憶體模型 Memory Models

| 模型 | 默認變數存放 | 優點 | 缺點 |
|:-----|:------------|:-----|:-----|
| **SMALL** | data | 最快（直接尋址） | 僅 128B 可用 |
| **COMPACT** | pdata | 間接尋址 256B | 編譯器較慢 |
| **LARGE** | xdata | 完整 64KB | 最慢（MOVX 指令） |

## 中斷程式設計 Interrupt Programming

```c
#include <reg51.h>

// 外部中斷 0 (INT0)
void ISR_EXT0(void) interrupt 0 using 1 {
    // interrupt 0 = 中斷向量號
    // using 1     = 使用 Register Bank 1
    P1 = ~P1;   // Toggle P1
}

void main(void) {
    EA = 1;     // Enable All interrupts
    EX0 = 1;    // Enable INT0
    IT0 = 0;    // Level-triggered
    while(1);
}
```

### 8051 中斷向量表 Interrupt Vector Table

| Interrupt | Vector | 中斷源 | 旗標 |
|:----------|:------:|:-------|:-----|
| RESET | 0x0000 | 系統重置 | — |
| INT0 | 0x0003 | 外部中斷 0 | IE0 |
| Timer 0 | 0x000B | 定時器 0 | TF0 |
| INT1 | 0x0013 | 外部中斷 1 | IE1 |
| Timer 1 | 0x001B | 定時器 1 | TF1 |
| Serial | 0x0023 | UART (RI/TI) | RI/TI |
| Timer 2 | 0x002B | 定時器 2 (8052) | TF2 |

## 常用週邊代碼 On-Chip Peripheral Examples

```c
// UART 初始化 (9600 baud, 11.0592 MHz)
void UART_Init(void) {
    TMOD = 0x20;    // Timer 1, Mode 2 (8-bit auto-reload)
    TH1 = 0xFD;     // 9600 baud @ 11.0592MHz
    SCON = 0x50;    // Mode 1, REN=1
    TR1 = 1;        // Start Timer 1
}

// GPIO 位元操作
sbit LED = P1^0;
LED = 0;    // 低電平點亮 (8051 一般低電平驅動)
```

## 遺留提示 Legacy Note

> **C51 工具鏈仍在維護但已凍結特性**。新專案強烈建議使用 MDK-ARM + Cortex-M 系列。C51 僅適用於維護舊產品、教學 (大學 8051 課程)、或極低成本批量應用。
