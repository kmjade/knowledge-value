# 08 網絡編程

## Socket 核心概念
Socket = IP + Port = 網絡通訊端點。

## TCP Socket 流程
```python
# Server
s = socket.socket()
s.bind(('0.0.0.0', 8080))
s.listen()
conn, addr = s.accept()
data = conn.recv(1024)
conn.send(b'Hello')

# Client
c = socket.socket()
c.connect(('server', 8080))
c.send(b'GET')
response = c.recv(1024)
```

## 常見模式
- **阻塞 I/O** — 最簡單，每連接一個線程
- **非阻塞 + select/poll/epoll** — 單線程多連接
- **異步 (asyncio)** — Python async/await
