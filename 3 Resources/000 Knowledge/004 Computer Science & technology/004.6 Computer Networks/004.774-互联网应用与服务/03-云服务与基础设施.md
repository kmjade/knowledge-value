# 03 — 雲服務與基礎設施 Cloud Services & Infrastructure

> 從實體機房到 Serverless——計算資源的抽象層級不斷提升，責任界線也隨之移動。

---

## 服務模型 Service Models

| 層級 | 你管理 | 雲管理 | 範例 |
|------|--------|--------|------|
| **On-Premise** | 全部（硬體→應用） | 無 | 自有機房 |
| **IaaS** | OS + 運行時 + 應用 | 硬體 + 虛擬化 | AWS EC2, GCE |
| **PaaS** | 應用 + 數據 | OS + 運行時 | Heroku, Vercel |
| **SaaS** | 無 | 全部 | Gmail, Slack |
| **FaaS** | 函數代碼 | 全部 | AWS Lambda, Cloudflare Workers |

---

## 三大雲平台對比 Cloud Comparison

| 維度 | AWS | Azure | GCP |
|------|-----|-------|-----|
| **計算** | EC2, Lambda | VM, Functions | Compute Engine, Cloud Run |
| **存儲** | S3, EBS | Blob, Disk | Cloud Storage, Persistent Disk |
| **K8s** | EKS | AKS | GKE（最成熟） |
| **CDN** | CloudFront | Azure CDN | Cloud CDN |
| **AI/ML** | SageMaker | Azure ML | Vertex AI |
| **優勢** | 服務最多、市佔第一 | 企業集成（AD/O365） | BigQuery/K8s/ML |

---

## CDN 對比 CDN Comparison

| CDN | 特性 | 邊緣計算 | 適合 |
|-----|------|---------|------|
| **Cloudflare** | 全球網路 + DDoS 防護 | Workers | 中小型到企業 |
| **AWS CloudFront** | S3 原生集成 | Lambda@Edge | AWS 生態 |
| **Fastly** | 即時 purge，VCL 配置 | Compute@Edge | 高流量媒體 |
| **Vercel/Netlify** | Jamstack 原生 CDN | Edge Functions | 前端專案 |

---

## S3 / R2 物件存儲 Object Storage

| 服務 | 提供者 | 特點 | 價格 |
|------|--------|------|------|
| **S3** | AWS | 最成熟，生態最廣 | 按量 + 請求費 |
| **R2** | Cloudflare | 零流量費（egress free） | 僅存儲費 |
| **GCS** | GCP | 與 BigQuery 集成 | 按量 |
| **B2** | Backblaze | 最低存儲價 | ~$5/TB/月 |

---

## Serverless 架構要點 Serverless Essentials

- **冷啟動 (Cold Start)**：Lambda 首次調用延遲 100ms–2s，可預熱或使用 Provisioned Concurrency
- **無狀態**：狀態需外存（DynamoDB、Redis）
- **適用**：事件驅動、API Gateway、定時任務、ETL
