# 09 — 安全與合規 Security & Compliance

> 安全不是功能——是屬性。每一層（傳輸、應用、資料）都需要防禦縱深（Defense in Depth）。

---

## OWASP Top 10 (2021) 常見漏洞

| 排名 | 漏洞 | 說明 | 防禦 |
|------|------|------|------|
| **A01** | 權限控制失效 (BAC) | URL 直接訪問他人資源 | 伺服器端授權檢查 |
| **A02** | 加密失效 | 明文傳輸、弱算法 | TLS 1.3, bcrypt/argon2 |
| **A03** | Injection (SQL/XSS) | 未過濾輸入 | 參數化查詢、CSP |
| **A04** | 不安全設計 | 缺乏威脅建模 | Shift-left 安全 |
| **A05** | 安全配置錯誤 | 預設密碼、debug on | 硬化、IaC 審計 |
| **A06** | 易受攻擊的元件 | 過時依賴 | SCA (Dependabot, Snyk) |
| **A07** | 認證失效 | 弱密碼、無 MFA | MFA, OAuth2/OIDC |
| **A08** | 軟體與資料完整性 | 未驗證的 CI/CD | 簽章驗證、SBOM |
| **A09** | 日誌與監控失效 | 無法偵測入侵 | SIEM, 異常檢測 |
| **A10** | SSRF | 伺服器端請求偽造 | 白名單域名 |

## DDoS 防護 DDoS Protection

| 類型 | 攻擊層 | 防禦 |
|------|--------|------|
| **Volume-based** | L3/L4 (UDP flood, SYN flood) | Cloudflare Magic Transit, AWS Shield |
| **Protocol** | L3/L4 (SYN, Ping of Death) | Rate Limiting, SYN Cookie |
| **Application** | L7 (HTTP flood, Slowloris) | WAF, CAPTCHA, IP 信譽 |

## 安全標頭 Security Headers

| Header | 作用 | 推薦值 |
|--------|------|--------|
| **CSP** (Content-Security-Policy) | 限制資源來源，防 XSS | `default-src 'self'` |
| **HSTS** (Strict-Transport-Security) | 強制 HTTPS | `max-age=31536000; includeSubDomains` |
| **X-Frame-Options** | 防 Clickjacking | `DENY` 或 `SAMEORIGIN` |
| **X-Content-Type-Options** | 防 MIME sniffing | `nosniff` |
| **Referrer-Policy** | 控制 Referrer 外洩 | `strict-origin-when-cross-origin` |
| **Permissions-Policy** | 限制瀏覽器 API | `camera=(), geolocation=(self)` |

## CORS 安全配置 CORS Best Practices

- **絕不使用** `Access-Control-Allow-Origin: *` + `credentials: true`
- 動態回顯 `Origin` 需白名單驗證
- 僅開放必要的 Methods 和 Headers
- `Access-Control-Max-Age` 減少 Preflight 請求

---

## Let's Encrypt 免費 TLS 證書

- **ACME 協議**：自動化證書申請/續期
- **工具**：certbot, acme.sh, Caddy (內建)
- **限制**：每週每域名 50 張，DV 驗證僅（無 OV/EV）
- **生命週期**：90 天，建議 30 天自動續期
