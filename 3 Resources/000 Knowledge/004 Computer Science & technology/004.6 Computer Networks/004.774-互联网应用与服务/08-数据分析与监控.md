# 08 — 數據分析與監控 Analytics & Monitoring

> 你不能優化無法量測的東西。現代分析棧從前端埋點到後端追蹤，構建完整的可觀測性（Observability）體系。

---

## 網站分析 Web Analytics

| 工具 | 類型 | 隱私 | 價格 |
|------|------|------|------|
| **Google Analytics 4** | 事件驅動（免費） | 需 Cookie 同意 | 免費至大量 |
| **Plausible** | 輕量、隱私優先 | GDPR 友善，無 Cookie | $9/月起 |
| **Umami** | 自託管，開源 | 完全自主 | 免費（自託管） |
| **Matomo** | 開源，自託管 | 100% 數據所有權 | 免費（自託管） |
| **Mixpanel** | 產品分析 | 需同意 | 免費額度 + 付費 |

---

## 可觀測性三支柱 Three Pillars

| 支柱 | 解決問題 | 工具 |
|------|---------|------|
| **Logs** (日誌) | 發生什麼？ | Elasticsearch (ELK), Loki |
| **Metrics** (指標) | 系統多快/多忙？ | Prometheus + Grafana |
| **Traces** (追蹤) | 請求走了哪？ | Jaeger, OpenTelemetry |

---

## 日誌與儀表板 ELK / Grafana

| 元件 | ELK Stack | Grafana Stack |
|------|-----------|---------------|
| **收集** | Logstash / Beats | Grafana Agent / Alloy |
| **存儲** | Elasticsearch | Mimir (Metrics), Loki (Logs) |
| **可視化** | Kibana | Grafana |
| **追蹤** | APM (自帶) | Tempo |

| 工具 | 類型 | 強項 | 定價 |

## A/B 測試 A/B Testing

| 概念 | 說明 |
|------|------|
| **分流** | 用戶隨機分到 Control (A) vs Variant (B) |
| **指標** | Conversion Rate, CTR, Revenue per User |
| **統計** | p-value < 0.05，最小樣本量計算 |
| **工具** | Google Optimize (已停), VWO, LaunchDarkly, GrowthBook |
| **注意** | 窺探偏誤（Peeking）、多重測試校正 |
