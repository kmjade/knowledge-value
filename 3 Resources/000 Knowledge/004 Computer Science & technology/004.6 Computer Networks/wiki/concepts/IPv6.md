# IPv6 互聯網協議第六版

## 定義
IPv4 的繼任者 — 128-bit 地址空間 (340 undecillion 地址)，解決 IPv4 地址耗盡問題。

## IPv6 vs IPv4
| | IPv4 | IPv6 |
|:--|:-----|:-----|
| 地址 | 32-bit (4.3B) | 128-bit (3.4×10³⁸) |
| 格式 | 192.168.1.1 | 2001:db8::1 |
| NAT | 必須 | 不需要 |
| IPSec | 可選 | 內建 |
| 廣播 | 有 | 無 (改用 Multicast) |

## 過渡技術: Dual Stack (雙棧)、Tunneling (6to4/Teredo)、NAT64
