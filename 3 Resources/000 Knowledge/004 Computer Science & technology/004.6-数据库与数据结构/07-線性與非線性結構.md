# 07 線性與非線性結構

## 線性結構
| 結構 | 操作 | 複雜度 |
|------|------|:--:|
| **Array** | 隨機存取 O(1) | 插入 O(n) |
| **Linked List** | 插入頭/尾 O(1) | 搜尋 O(n) |
| **Stack** | push/pop (LIFO) | O(1) |
| **Queue** | enqueue/dequeue (FIFO) | O(1) |

## 非線性結構
| 結構 | 特點 | 應用 |
|------|------|------|
| **Tree** | 根節點 + 子樹 | 文件系統、DOM |
| **Binary Search Tree** | 左<根<右 | 快速查找 O(log n) |
| **Heap** | 最大/最小在根 | 優先級隊列 |
| **Hash Table** | key→hash→bucket | O(1) 查找 |
| **Graph** | 節點+邊 | 社交網絡、路由 |
