# CS 学习路径

> 对标大学计算机科学核心课程的自学路线。面向有编程经验的开发者，系统补齐 CS 基础。

## 阶段一：地基（第 1-3 周）

1. [[01-计算机科学概述]] — 建立全景认知
2. [[3 Resources/000 Knowledge/004 Computer Science & technology/004.5-数据库与数据结构/02-数据结构与算法|02-数据结构与算法]] — 开始刷题，每天 1-2 道
3. **实践**：用你熟悉的语言实现基本数据结构（链表、栈、哈希表）

## 阶段二：向下挖（第 4-7 周）

4. [[03-计算机组成原理]] — 理解"Hello World"如何在硬件上运行
5. [[04-操作系统]] — 进程、线程、虚拟内存、文件系统
6. **实践**：写一个简单的 shell、一个多线程程序

## 阶段三：向外看（第 8-11 周）

7. [[004.6 计算机网络]] — 从物理层到应用层，抓包分析 HTTP
8. [[06-数据库系统]] — SQL 精通、索引原理、事务隔离级别
9. **实践**：设计一个博客系统的数据库 schema + 常见查询优化

## 阶段四：系统与工程（第 12-16 周）

10. [[07-编程语言]] — 理解类型系统、编译 vs 解释
11. [[08-软件工程]] — 设计模式、测试策略、CI/CD
12. [[09-数据处理与分布式系统]] — MapReduce 思想、CAP 定理、Kafka 基础
13. **终极实践**：从头实现一个简单的 key-value 数据库

## 阶段五：深入子领域（可选，按兴趣）

| 方向 | 入口 | 適合 |
|------|------|------|
| 🤖 AI/ML | [[004.8-人工智能/004.8-人工智能\|004.8 AI]] | 想做 AI 應用或研究 |
| 🔌 嵌入式 | [[004.16-嵌入式系統/004.16-嵌入式系統\|004.16 嵌入式]] | 對硬體/物聯網感興趣 |
| 🌐 網絡深度 | [[004.6-计算机网络/004.6 Computer Networks\|004.6 網絡]] | 想深入協議/安全/編程 |
| 🗄️ DB/DSA 深度 | [[004.5-数据库与数据结构/004.5-数据库与数据结构\|004.5 DB+DSA]] | 面試準備/系統設計 |

> 💡 所有深度子域均含 **wiki/** 編譯產物——概念頁、實體頁、來源追溯。

## 前置知识

| 知识 | 最低要求 |
|------|----------|
| 编程 | 能用至少一门语言（Python/Java/Go/C++）写出能运行的程序 |
| 数学 | 高中代数 + 一点离散数学意识（集合并交、逻辑） |

## 选学路径

如果你有特定兴趣：

- **后端开发** → 02 算法 + 04 OS + 05 网络 + 06 数据库 + 09 分布式
- **AI/ML** → 02 算法 + 07 语言(Python 深入) + 09 数据处理
> 然后进入 [[AI 知识库总览|AI KB]]
- **系统/底层** → 03 组成原理 + 04 OS + 07 编译原理
- **面试突击** → 02 算法（重点刷题）+ 04 OS 基础 + 05 网络（HTTP/TCP）

> 这是一场马拉松——16 周是最短时间。CS 基础是"慢回报"投资：短时间看不到效果，但长期会让你对任何技术都能快速上手。
