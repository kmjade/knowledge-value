## 特点

- **大学核心对标** — 覆盖 CS 本科 9 门核心课程的知识框架
- **从硬件到算法** — 自底向上的全景视角（冯·诺依曼 → 分布式）
- **中文优先** — 核心术语保留英文对照
- **面试导向** — 算法章节附带复杂度速查和刷题平台
- **工程实用** — 软件工程章包含 Git、CI/CD、设计模式
- **双向链接** — 与 AI 知识库建立交叉引用
