# 05 計算機視覺

## 核心任務
| 任務 | 描述 | 算法 |
|------|------|------|
| **圖像分類** | 整張圖→類別 | ResNet, ViT |
| **物件偵測** | 框出物體 | YOLO, Faster R-CNN |
| **語義分割** | 像素級分類 | U-Net, DeepLab |
| **實例分割** | 區分同類個體 | Mask R-CNN |

## CNN 核心層
卷積 (特徵提取) → ReLU → Pooling (降維) → ... → FC (分類)

## 前沿
- **ViT** — Transformer 應用於視覺
- **SAM** — Meta 的 Segment Anything Model
- **NeRF** — 3D 場景重建
- **Stable Diffusion** — 文本→圖像生成
