# AI FAQ

### ML 還是 DL？
結構化數據、少量數據、可解釋性 → ML (XGBoost)。非結構化數據、大規模、高準確度 → DL。

### GPU 為什麼重要？
神經網絡訓練 = 大量矩陣乘法 → GPU 數千核心並行計算遠超 CPU。

### 什麼是 Transformer？
2017 年 Attention 機制取代 RNN — 並行處理、長距離依賴、可擴展 → 現代 LLM 的基石。

### 監督 vs 自監督學習？
監督: 人類標註 (昂貴)。自監督: 從數據本身創造標籤 (GPT 預測下一個詞) — LLM 訓練的核心。

### AGI 何時到來？
無共識。樂觀派: 5-10 年、悲觀派: 50+ 年、懷疑派: 不可能。
