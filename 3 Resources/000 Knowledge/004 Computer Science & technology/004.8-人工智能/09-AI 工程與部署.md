# 09 AI 工程與部署

## MLOps 管線
數據收集 → 數據驗證 → 訓練 → 評估 → 部署 → 監控 → 重訓練

## 推理部署
| 方式 | 工具 | 場景 |
|------|------|------|
| REST API | FastAPI/Flask | 同步請求 |
| 串流 | Kafka + 模型 | 即時推理 |
| 邊緣 | TensorFlow Lite / ONNX | 移動/IoT |
| 批次 | Spark + 模型 | 離線預測 |

## 關鍵工具: Docker (容器化)、Kubernetes (編排)、MLflow/W&B (實驗追蹤)、Triton/vLLM (高效推理)

## 模型優化: 量化 (FP32→INT8)、蒸餾 (大→小模型)、剪枝 (移除冗餘權重)
