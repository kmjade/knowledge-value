# 強化學習 Reinforcement Learning

## 定義
Agent 通過與環境互動，根據獎勵信號學習最優策略。不依賴標註數據，而是**試錯學習**。

## 核心循環: 觀察狀態 s → 選擇動作 a → 獲得獎勵 r → 轉移到新狀態 s' → 更新策略

## 算法
| 算法 | 原理 | 場景 |
|------|------|------|
| Q-Learning | 學習 Q(s,a) 表 | 離散小空間 |
| DQN | 神經網路近似 Q(s,a) | Atari 遊戲 |
| Policy Gradient | 直接優化策略 π(a|s) | 連續動作 |
| PPO | 穩定策略更新 | 機器人、遊戲 |
| AlphaZero | MCTS + 自對弈 | 圍棋、象棋 |
