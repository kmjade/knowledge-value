## 六、DDC vs UDC 归类分歧

```
                 ┌─ 005.133 (编程语言)
DDC              ├─ 006.74  (标记语言)
(按用途分)       ├─ 005.756 (数据库语言)
                 └─ 686.22  (排版语言 — LaTeX!)

                 ┌─ 004.432 (通用编程语言)
UDC              ├─ 004.438 (DSL)
(按本质分)       └─ 004.439 (标记/查询/其他)
```

| 语言 | DDC | UDC | 分歧原因 |
|------|:---|:---|------|
| Markdown | **`006.74`** 标记语言 | `004.439` 其他语言 | DDC: Web技术归006 |
| SQL | **`005.756`** 数据库 | `004.439` 查询语言 | DDC: 按应用领域归类 |
| LaTeX | **`686.22`** 印刷！ | `004.439` 排版语言 | DDC: 按最终用途(印刷)归类 |
| Verilog | **`621.392`** 电路 | `004.438` DSL | DDC: 归入电子工程而非CS |
