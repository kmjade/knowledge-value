## 🗺️ Java 學習路線圖 Learning Roadmap

```
Phase 1: 基礎語法 (2-4 週)
├── JVM/JDK/JRE 概念
├── 基本語法 (類型、流程控制、陣列、方法)
├── 物件導向 (類、繼承、多型、介面)
└── 常用類別 (String、集合、例外處理)

Phase 2: 進階特性 (4-6 週)
├── 泛型 (Generic)、註解 (Annotation)
├── Lambda、Stream API、Optional
├── IO/NIO、序列化
├── 併發基礎 (Thread、synchronized、Lock)
└── JVM 基礎 (記憶體模型、GC)

Phase 3: 工程化 (6-8 週)
├── Maven/Gradle 構建工具
├── Spring Boot 核心 (IoC、AOP、MVC)
├── 資料庫 (JPA、MyBatis、Flyway)
├── 測試 (JUnit 5、Mockito、Integration Test)
└── Docker / CI/CD

Phase 4: 架構與深入 (持續)
├── 設計模式在 Java 中的實踐
├── JVM 調優 (GC 日誌、Arthas、JMH)
├── 分散式 (Redis、Kafka、Spring Cloud)
├── 源碼閱讀 (Spring、Netty 核心模組)
└── 貢獻開源專案
```
