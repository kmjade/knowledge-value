## 9.1 創建型模式 Creational Patterns

### 單例模式 Singleton

```java
// 1. Enum 單例 (最安全，防反射/序列化破壞)
public enum ConfigManager {
    INSTANCE;

    private final Properties props = new Properties();

    ConfigManager() {
        // 加載配置...
    }

    public String get(String key) {
        return props.getProperty(key);
    }
}
// 使用: ConfigManager.INSTANCE.get("db.url")

// 2. 雙重檢查鎖 DCL (懶加載，需 volatile)
public class LazySingleton {
    private static volatile LazySingleton instance;

    private LazySingleton() {}

    public static LazySingleton getInstance() {
        if (instance == null) {
            synchronized (LazySingleton.class) {
                if (instance == null) {
                    instance = new LazySingleton();
                }
            }
        }
        return instance;
    }
}
```

### 工廠模式 Factory

```java
// 簡單工廠 (Static Factory)
public class NotificationFactory {
    public static Notification create(String type) {
        return switch (type) {
            case "EMAIL" -> new EmailNotification();
            case "SMS"   -> new SmsNotification();
            case "PUSH"  -> new PushNotification();
            default -> throw new IllegalArgumentException("Unknown: " + type);
        };
    }
}

// Factory Method — 延遲到子類決定
interface Dialog {
    Button createButton();
    default void render() {
        Button btn = createButton();
        btn.onClick(() -> System.out.println("Clicked!"));
    }
}
class WindowsDialog implements Dialog {
    public Button createButton() { return new WindowsButton(); }
}
```

### 建造者模式 Builder (+ Lombok)

```java
// 手寫 Builder
public class HttpClient {
    private final String url;
    private final int timeout;
    private final int retries;

    private HttpClient(Builder builder) {
        this.url = builder.url;
        this.timeout = builder.timeout;
        this.retries = builder.retries;
    }

    public static class Builder {
        private String url;
        private int timeout = 5000;
        private int retries = 3;

        public Builder url(String url)      { this.url = url; return this; }
        public Builder timeout(int timeout) { this.timeout = timeout; return this; }
        public Builder retries(int retries) { this.retries = retries; return this; }

        public HttpClient build() {
            if (url == null) throw new IllegalStateException("url required");
            return new HttpClient(this);
        }
    }
}

// Lombok @Builder 等價寫法
@Builder
public class HttpClientLombok {
    private String url;
    @Builder.Default private int timeout = 5000;
    @Builder.Default private int retries = 3;
}
// 使用: HttpClientLombok.builder().url("https://api.com").build()
```

### 原型模式 Prototype

```java
// Java Record — 不可變，複製即變體 (wither pattern)
record User(Long id, String name, String email) {
```

## 9.2 結構型模式 Structural Patterns

### 適配器模式 Adapter

```java
// 將舊版 Logger 適配到 SLF4J
class LegacyLogger {
    void logMessage(String msg) { System.out.println("[LEGACY] " + msg); }
}

class LegacyLoggerAdapter implements org.slf4j.Logger {
    private final LegacyLogger legacy = new LegacyLogger();

    public void info(String msg) { legacy.logMessage("INFO: " + msg); }
    public void error(String msg) { legacy.logMessage("ERROR: " + msg); }
    // ... 其他方法委託
}
```

### 裝飾器模式 Decorator

```java
// Java IO 就是裝飾器典範
// InputStream → BufferedInputStream → GZIPInputStream → DataInputStream

// 自定義裝飾器：快取資料庫查詢
class CachingUserRepository implements UserRepository {
    private final UserRepository delegate;
    private final Map<Long, User> cache = new ConcurrentHashMap<>();

    public Optional<User> findById(Long id) {
        return Optional.ofNullable(
            cache.computeIfAbsent(id, k -> delegate.findById(k).orElse(null))
        );
    }
}
```

### 代理模式 Proxy

```java
// JDK 動態代理 (AOP 基礎)
class LoggingProxy implements InvocationHandler {
    private final Object target;

    public static <T> T create(Class<T> iface, T target) {
        return (T) Proxy.newProxyInstance(
            iface.getClassLoader(),
            new Class[]{iface},
            new LoggingProxy(target)
        );
    }

    @Override
    public Object invoke(Object proxy, Method method, Object[] args) throws Throwable {
        System.out.println(">> " + method.getName());
        long start = System.nanoTime();
        Object result = method.invoke(target, args);
        System.out.println("<< " + method.getName() + " took " +
            TimeUnit.NANOSECONDS.toMillis(System.nanoTime() - start) + "ms");
        return result;
    }
}
```

## 9.3 行為型模式 Behavioral Patterns

### 觀察者模式 Observer

```java
// Java 9+ Flow API (Reactive Streams)
import java.util.concurrent.Flow.*;

class NewsPublisher implements Publisher<String> {
    private final List<Subscriber<? super String>> subscribers = new CopyOnWriteArrayList<>();

    public void subscribe(Subscriber<? super String> s) {
        subscribers.add(s);
        s.onSubscribe(new Subscription() {
            public void request(long n) {}
            public void cancel() { subscribers.remove(s); }
        });
    }

    public void publish(String news) {
        subscribers.forEach(s -> s.onNext(news));
    }
}
```

### 策略模式 Strategy (+ Lambda)

```java
// 傳統寫法
interface DiscountStrategy {
    double apply(double price);
}

// Lambda 策略 (現代化)
class PriceCalculator {
    private final Map<String, DiscountStrategy> strategies = Map.of(
        "VIP", price -> price * 0.8,
        "SALE", price -> price * 0.7,
        "NORMAL", price -> price
    );

    public double calculate(double price, String type) {
        return strategies.getOrDefault(type, p -> p).apply(price);
    }
}
```

### 責任鏈模式 Chain of Responsibility

```java
// 函數式鏈 (Java 8+)
@FunctionalInterface
interface Handler {
    Optional<String> handle(Request req);
}

Handler chain = req ->
    Optional.ofNullable(req.getHeader("Auth-Token"))
        .filter(t -> t.startsWith("Bearer "))
        .map(t -> "Authenticated");

// 組合多個 Handler
Handler authChain = req -> {
    if (req.hasHeader("Auth-Token")) return Optional.of("Token OK");
    if (req.hasHeader("Api-Key"))    return Optional.of("Key OK");
    return Optional.empty();
};
```

### 模板方法模式 Template Method

```java
abstract class DataProcessor {

    // Template Method (final 防止子類覆蓋)
    public final void process() {
        loadData();
        validateData();
        transformData();
        saveResults();
    }

    abstract void loadData();
    abstract void transformData();

    // Hook (可選覆蓋)
    void validateData() { /* 預設無操作 */ }
    void saveResults() { System.out.println("Done"); }
}
```

## 9.4 現代 Java 中的模式 Modern Patterns

### 依賴注入 Dependency Injection

```java
// Spring 構造器注入 (最推薦)
@Service
public class OrderService {
    private final PaymentService paymentService;
    private final InventoryService inventoryService;

    // 無需 @Autowired (Spring 4.3+ 單一構造子自動注入)
    public OrderService(PaymentService paymentService, InventoryService inventoryService) {
        this.paymentService = paymentService;
        this.inventoryService = inventoryService;
    }
}
```

### Sealed Class + Pattern Matching (代數資料型別)

```java
// ADT 風格 — 編譯期窮舉 (Java 21)
sealed interface Result<T> permits Success, Failure {}
record Success<T>(T value) implements Result<T> {}
record Failure<T>(String error) implements Result<T> {}

// 編譯器強制處理所有分支
String message = switch (result) {
    case Success(var value) -> "OK: " + value;
    case Failure(var error) -> "ERROR: " + error;
    // 不需 default — 編譯器保證窮舉
};
```

## 設計模式速查表 Quick Reference

| 模式 | 類型 | 核心思想 | Java 典型場景 |
|------|------|---------|-------------|
| **Singleton** | 創建 | 全域唯一 | Config, Connection Pool |
| **Factory** | 創建 | 延遲實例化 | DI Container, JDBC Driver |
| **Builder** | 創建 | 逐步構造 | HttpClient, Lombok @Builder |
| **Prototype** | 創建 | 複製 | Record wither, Object.clone |
| **Adapter** | 結構 | 轉換介面 | Legacy → SLF4J, API 相容 |
| **Decorator** | 結構 | 動態增強 | java.io 體系, 快取代理 |
| **Proxy** | 結構 | 控制訪問 | AOP, Lazy Loading |
| **Observer** | 行為 | 廣播通知 | Event Listener, Flow API |
| **Strategy** | 行為 | 互換演算法 | Comparator, Lambda 替換 |
| **Chain** | 行為 | 傳遞處理 | Filter Chain, Middleware |
| **Template** | 行為 | 骨架固定 | JdbcTemplate, RestTemplate |
| **Dependency Injection** | 現代 | 控制反轉 | Spring IoC Container |
