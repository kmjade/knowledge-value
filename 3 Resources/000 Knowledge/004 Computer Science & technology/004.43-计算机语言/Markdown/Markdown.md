### 为什么 DDC 归入 006 而非 005？

```
005 = 程序设计 (Programming)    → Python, Java, C++...
006 = 特殊计算机方法             → Web, 标记语言, 排版
      └── 006.74 = Markup languages (标记语言)
```

DDC 按**用途**分类：标记语言服务于文档/Web，而非"程序设计"。UDC 按**本质**分类：所有语言归 004.43，再用 `004.432`（通用PL）vs `004.439`（标记/其他）区分。
