# 02 - Markdown 扩展语法

这些语法来自 CommonMark、GFM 等扩展规范，大多数现代编辑器都支持。

## 表格

```markdown
| 左对齐 | 居中 | 右对齐 |
|:------|:----:|------:|
| 内容 | 内容 | 内容 |
| 长内容 | 短 | 中 |
```

| 左对齐 | 居中 | 右对齐 |
|:------|:----:|------:|
| 内容 | 内容 | 内容 |
| 长内容 | 短 | 中 |

> 冒号位置决定对齐：`|:---|` 左对齐，`|:---:|` 居中，`|---:|` 右对齐

## 围栏代码块

````markdown
```python
def fibonacci(n):
    """返回第 n 个斐波那契数"""
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```

```
无语法高亮的纯文本
```

```bash
echo "支持语法高亮"
```
````

**常用语言标识**：`python` `javascript` `bash` `json` `yaml` `sql` `html` `css` `java` `rust`

## 脚注

```markdown
这是一个有脚注的句子[^1]。

[^1]: 这是脚注内容。
```
