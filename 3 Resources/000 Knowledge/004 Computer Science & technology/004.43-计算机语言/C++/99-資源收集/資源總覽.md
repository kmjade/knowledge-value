## 🔧 工具生態 (Tooling Ecosystem)

### 編譯器 (Compilers)

| 編譯器 | 特點 | C++23 支援 |
|:---:|------|:---:|
| **GCC** | Linux 預設，生態最廣 | ✅ 13+ |
| **Clang** | 錯誤訊息友好，模組化設計 | ✅ 16+ |
| **MSVC** | Windows 首選，IDE 整合 | ✅ 2022+ |
| **Intel oneAPI** | HPC/SIMD 最佳化 | 部分 |

### 靜態分析 (Static Analysis)

| 工具 | 說明 |
|------|------|
| **clang-tidy** | LLVM 靜態分析，200+ 檢查項 |
| **cppcheck** | 輕量級 C/C++ 靜態分析 |
| **PVS-Studio** | 商業級靜態分析 (有免費授權) |
| **SonarQube** | 平台級程式碼品質管理 |

### 套件管理 (Package Managers)

| 工具 | 特點 |
|------|------|
| **vcpkg** | Microsoft 維護，CMake 原生整合 |
| **Conan** | Python-based，跨平台，去中心化 |
| **xmake** | Lua-based，輕量建置 + 套件管理 |

### 效能剖析 (Profilers)

| 工具 | 平台 | 說明 |
|------|:---:|------|
| **perf** | Linux | 核心級效能計數器 |
| **Intel VTune** | 跨平台 | 商業級效能分析 |
| **Valgrind/Cachegrind** | Linux | 快取/記憶體分析 |
| **Tracy** | 跨平台 | 即時 frame profiler (遊戲/圖形) |
