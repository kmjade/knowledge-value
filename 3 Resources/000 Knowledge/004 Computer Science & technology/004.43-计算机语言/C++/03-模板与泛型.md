# 03 — 模板與泛型 (Templates & Generics)

> 🎯 **學習目標**：掌握函數/類別模板、特化/偏特化、SFINAE、Concept (C++20)、variadic templates、fold expressions、CTAD。

---

## 3.1 函數模板 (Function Templates)

### 基本語法與推導

```cpp
// 基本函數模板
template<typename T>
T max(T a, T b) {
    return a > b ? a : b;
}

// 型別推導 (deduction)
auto r1 = max(3, 5);          // T=int
auto r2 = max(3.14, 2.71);    // T=double
// auto r3 = max(3, 3.14);    // ❌ T 模棱兩可 (int vs double)
auto r4 = max<double>(3, 3.14);  // ✅ 顯式指定 T=double

// 多模板參數
template<typename T, typename U>
auto add(T a, U b) -> decltype(a + b) {   // C++11 trailing return
    return a + b;
}
// C++14: 簡化為 auto + decltype 推導
template<typename T, typename U>
auto add_v2(T a, U b) {
    return a + b;
}
```

### 非型別模板參數 (Non-Type Template Parameter)

```cpp
// 編譯期常數作為模板參數
template<typename T, size_t N>
class Array {
    T data_[N];
public:
    constexpr size_t size() const { return N; }
    T& operator[](size_t i) { return data_[i]; }
    const T& operator[](size_t i) const { return data_[i]; }
    
    T* begin() { return data_; }
    T* end() { return data_ + N; }
};

Array<int, 5> arr;              // 5 個 int
static_assert(arr.size() == 5);

// C++20: 浮點數、類別型別也可作非型別模板參數
template<auto Value>             // C++17 auto 非型別參數
struct constant {
    static constexpr auto value = Value;
};
constant<42> c1;
constant<'A'> c2;

// C++20: 浮點非型別參數
template<double Val>
struct PiMultiplier {
    static constexpr double value = Val * 3.14159;
};
```

## 3.2 類別模板 (Class Templates)

```cpp
template<typename T>
class Stack {
    std::vector<T> data_;
public:
    void push(const T& val) { data_.push_back(val); }
    void push(T&& val) { data_.push_back(std::move(val)); }
    std::optional<T> pop() {
        if (data_.empty()) return std::nullopt;
        T val = std::move(data_.back());
        data_.pop_back();
        return val;
    }
    bool empty() const { return data_.empty(); }
    size_t size() const { return data_.size(); }
};
```

### CTAD (Class Template Argument Deduction) — C++17

```cpp
// C++17 前必須顯式指定型別
std::pair<int, double> p1{42, 3.14};
std::vector<int> v1{1, 2, 3, 4};

// C++17 CTAD: 編譯器從建構子推導
std::pair p2{42, 3.14};          // std::pair<int, double>
std::vector v2{1, 2, 3, 4};      // std::vector<int>
std::tuple t{1, "hello", 3.14};  // std::tuple<int, const char*, double>

// 自訂推導指南 (Deduction Guide)
template<typename T>
class Wrapper {
    T value_;
public:
    Wrapper(T v) : value_(std::move(v)) {}
};
// 若推導錯誤，可自訂：
// template<typename T> Wrapper(T) -> Wrapper<T>;

// 常見 CTAD 陷阱：
std::vector v3(5, 42);           // 5 個 42: vector<int>
std::vector v4{5, 42};           // 2 個元素 {5, 42}: vector<int>
// std::vector v5{1, 2L, 3};    // ❌ 型別不一致
```

## 3.3 特化與偏特化 (Specialization & Partial Specialization)

### 全特化 (Full Specialization)

```cpp
// Primary template (主要模板)
template<typename T>
struct TypeName {
    static const char* get() { return "unknown"; }
};

// Full specialization for int
template<>
struct TypeName<int> {
    static const char* get() { return "int"; }
};

// Full specialization for double
template<>
struct TypeName<double> {
    static const char* get() { return "double"; }
};

static_assert(std::string(TypeName<int>::get()) == "int");
```

### 偏特化 (Partial Specialization)

```cpp
// Primary template
template<typename T, typename U>
struct IsSame { static constexpr bool value = false; };

// Partial specialization: T==U
template<typename T>
struct IsSame<T, T> { static constexpr bool value = true; };

// 偏特化指針型別
template<typename T>
struct TypeName<T*> {
    static std::string get() { return std::string(TypeName<T>::get()) + "*"; }
};

// 偏特化 const 型別
template<typename T>
struct TypeName<const T> {
    static std::string get() { return "const " + std::string(TypeName<T>::get()); }
};

// 複雜偏特化：檢查特定容器
template<typename T>
struct is_vector : std::false_type {};

template<typename T, typename Alloc>
struct is_vector<std::vector<T, Alloc>> : std::true_type {};
```

### 標籤分派 (Tag Dispatch)

```cpp
// 利用型別標籤選擇不同實作
struct fast_t {};
struct accurate_t {};

template<typename T>
T compute(T x, fast_t) { return x * 2; }        // 快速但不精確

template<typename T>
T compute(T x, accurate_t) { /* 複雜計算 */ return x * 2.0000001; }

// 使用端
auto result = compute(42.0, fast_t{});  // compile-time dispatch
```

## 3.4 SFINAE (Substitution Failure Is Not An Error)

### 經典 SFINAE 範例

```cpp
// C++11 enable_if: 函數多載解析時排除不匹配的模板
template<typename T>
typename std::enable_if<std::is_integral<T>::value, T>::type
process(T val) {
    std::cout << "integral: " << val << "\n";
    return val * 2;
}

template<typename T>
typename std::enable_if<std::is_floating_point<T>::value, T>::type
process(T val) {
    std::cout << "floating: " << val << "\n";
    return val * 3.14;
}

// C++14: enable_if_t 簡化
template<typename T, typename = std::enable_if_t<std::is_arithmetic_v<T>>>
T add(T a, T b) { return a + b; }

// C++17: if constexpr — 取代 SFINAE 的更清晰方式
template<typename T>
T process_v2(T val) {
    if constexpr (std::is_integral_v<T>) {
        return val * 2;
    } else if constexpr (std::is_floating_point_v<T>) {
        return val * 3.14;
    }
}
```

### Detection Idiom (C++17/20)

```cpp
// 檢測型別是否有 .serialize() 方法
template<typename T, typename = void>
struct is_serializable : std::false_type {};

template<typename T>
struct is_serializable<T, std::void_t<decltype(std::declval<T>().serialize())>> 
    : std::true_type {};

// C++20 Concepts 更簡潔 (見 3.5)
```

## 3.5 Concepts (C++20)

> Concepts 是 C++20 最重要的特性之一：編譯期約束模板參數，提供清晰的錯誤訊息。

### 基本語法

```cpp
#include <concepts>

// 定義 Concept
template<typename T>
concept Numeric = std::integral<T> || std::floating_point<T>;

template<typename T>
concept Addable = requires(T a, T b) {
    { a + b } -> std::same_as<T>;   // a+b 合法且回傳 T
};

template<typename T>
concept Comparable = requires(T a, T b) {
    { a == b } -> std::convertible_to<bool>;
    { a < b }  -> std::convertible_to<bool>;
};

// 使用 Concept 的 4 種語法
// 1. requires clause
template<typename T>
    requires Numeric<T>
T square(T x) { return x * x; }

// 2. 簡潔語法 (constrained auto)
auto square2(Numeric auto x) { return x * x; }

// 3. template head 中
template<Numeric T>
T square3(T x) { return x * x; }

// 4. 結合 requires clause 與簡潔型別
template<typename T>
T process(T val) requires Addable<T> && Comparable<T> {
    return val + val;
}
```

### 標準庫 Concepts

| Concept | 定義 | 對應型別 |
|------|------|------|
| `std::integral<T>` | 整數型別 | int, long, unsigned, char, ... |
| `std::floating_point<T>` | 浮點型別 | float, double |
| `std::signed_integral<T>` | 有號整數 | int, long, ... |
| `std::unsigned_integral<T>` | 無號整數 | unsigned int, size_t, ... |
| `std::same_as<T, U>` | 完全相同型別 | — |
| `std::derived_from<T, Base>` | 繼承關係 | — |
| `std::convertible_to<T, U>` | 可隱式轉換 | — |
| `std::movable<T>` | 可移動 | std::string, std::vector |
| `std::copyable<T>` | 可複製 | int, std::string |
| `std::regular<T>` | 正則型別 | int, std::string |
| `std::invocable<F, Args...>` | 可呼叫 | lambda, function pointers |

### 複合 Requirement

```cpp
// 檢查容器介面
template<typename T>
concept Container = requires(T c) {
    typename T::value_type;              // 型別別名
    typename T::iterator;
    { c.begin() } -> std::same_as<typename T::iterator>;
    { c.end() } -> std::same_as<typename T::iterator>;
    { c.size() } -> std::convertible_to<size_t>;
    { c.empty() } -> std::convertible_to<bool>;
};

// 使用
template<Container C>
void printAll(const C& container) {
    for (const auto& elem : container)
        std::cout << elem << " ";
}
```

## 3.6 Variadic Templates (可變參數模板 — C++11)

### 基本語法

```cpp
// 遞迴終止條件
void print() { std::cout << "\n"; }

// 遞迴展開
template<typename T, typename... Args>
void print(T first, Args... rest) {
    std::cout << first;
    if constexpr (sizeof...(rest) > 0) std::cout << ", ";
    print(rest...);  // recursive call with remaining args
}

print(1, 2.0, "hello", 'c');
// Output: 1, 2, hello, c
```

### Fold Expressions (C++17) — 摺疊運算式

```cpp
// 四種摺疊形式
template<typename... Args>
auto sum(Args... args) {
    return (... + args);    // Unary right fold: (a + (b + (c + d)))
    // return (args + ...); // Unary left fold:  (((a + b) + c) + d)
}

template<typename... Args>
auto sum_init(Args... args) {
    return (args + ... + 0); // Binary fold (with init): ((((0 + a) + b) + c) + d)
}

// 完整摺疊運算符對照
// (... op args)   = (a op (b op (c op d)))
// (args op ...)   = (((a op b) op c) op d)
// (init op ... op args) = (((init op a) op b) op c)
// (args op ... op init) = (a op (b op (c op init)))

// 實戰：and/or 摺疊
template<typename... Args>
bool all_true(Args... args) { return (... && args); }

template<typename... Args>
bool any_true(Args... args) { return (... || args); }

// 逗號摺疊：批量執行
template<typename... Callables>
void callAll(Callables&&... funcs) {
    (funcs(), ...);  // 依次呼叫每個函數
}

// 輸出摺疊
template<typename... Args>
void println(Args&&... args) {
    ((std::cout << std::forward<Args>(args) << " "), ...);
    std::cout << "\n";
}
```

### 完美轉發 Variadic

```cpp
// make_unique 的 variadic 實作原理
template<typename T, typename... Args>
std::unique_ptr<T> make_unique(Args&&... args) {
    return std::unique_ptr<T>(new T(std::forward<Args>(args)...));
}

// tuple 建構
template<typename... Args>
auto make_tuple(Args&&... args) {
    return std::tuple<std::decay_t<Args>...>(std::forward<Args>(args)...);
}
```

## 3.7 模板元程式設計 (Template Metaprogramming — TMP)

```cpp
// 編譯期 factorial (經典 TMP)
template<unsigned N>
struct Factorial {
    static constexpr unsigned value = N * Factorial<N - 1>::value;
};
template<>
struct Factorial<0> {
    static constexpr unsigned value = 1;
};
static_assert(Factorial<5>::value == 120);

// C++14 後用 constexpr 函數更簡單：
constexpr unsigned factorial(unsigned n) {
    return n <= 1 ? 1 : n * factorial(n - 1);
}
static_assert(factorial(5) == 120);
```

### Type Traits 實作原理

```cpp
// 自製 type traits
template<typename T, T v>
struct integral_constant {
    static constexpr T value = v;
    using value_type = T;
    using type = integral_constant;
    constexpr operator T() const noexcept { return v; }
    constexpr T operator()() const noexcept { return v; }
};

using true_type = integral_constant<bool, true>;
using false_type = integral_constant<bool, false>;

// 自製 is_same
template<typename T, typename U>
struct is_same : false_type {};
template<typename T>
struct is_same<T, T> : true_type {};

// 自製 remove_reference
template<typename T> struct remove_reference      { using type = T; };
template<typename T> struct remove_reference<T&>  { using type = T; };
template<typename T> struct remove_reference<T&&> { using type = T; };
```

## 3.8 模板實例化與編譯 (Template Instantiation)

### 兩階段名稱查找 (Two-Phase Name Lookup)

```cpp
// Phase 1 (definition time): 非相依名稱 (non-dependent names)
// Phase 2 (instantiation time): 相依名稱 (dependent names)

template<typename T>
struct Container {
    void foo() {
        std::cout << "generic\n";  // Phase 1 — 非相依
    }
    
    void bar() {
        T::staticMethod();          // Phase 2 — 相依於 T
        // ⚠️ 需要 typename 宣告型別：
        // typename T::value_type val;
    }
};

// template disambiguator
template<typename T>
void callTemplateMethod() {
    T t;
    t.template foo<int>();  // ✅ template 關鍵字消除歧義
}
```

### extern template (C++11)

```cpp
// header.hpp
template<typename T>
void common_func(T val) { /* ... */ }

extern template void common_func<int>(int);     // 顯式抑制實例化
extern template void common_func<double>(double);

// impl.cpp
template void common_func<int>(int);             // 顯式實例化（僅此處生成）
template void common_func<double>(double);
```
