# 04 — 工作區與專案管理 Workspace & Project Management

> .code-workspace、Multi-root、tasks.json、launch.json、集成終端機——VS Code 的專案管理體系。

---

## .code-workspace 工作區文件

```json
{
  "folders": [
    { "path": "frontend", "name": "🟦 Frontend" },
    { "path": "../shared-lib", "name": "📦 Shared Library" },
    { "path": "backend", "name": "🟩 Backend" }
  ],
  "settings": {
    "editor.tabSize": 2,
    "files.exclude": { "**/node_modules": true }
  },
  "extensions": {
    "recommendations": ["dbaeumer.vscode-eslint", "esbenp.prettier-vscode"]
  },
  "launch": {
    "configurations": [],
    "compounds": []
  }
}
```

| 區塊 | 功能 |
|------|------|
| **folders** | 多根目錄定義，支援 name 別名 |
| **settings** | 工作區級設定，覆蓋 User Settings |
| **extensions** | 推薦擴展 (recommendations) / 不推薦 (unwanted) |
| **launch** | 調試配置 |

## Multi-Root Workspace 多根工作區

| 操作 | 方式 |
|------|------|
| **建立** | `File → Add Folder to Workspace...` |
| **保存** | `File → Save Workspace As...` (→ .code-workspace) |
| **搜尋範圍** | 搜尋時可選擇作用根目錄 |
| **設定作用域** | 每個 root 可有獨立 `.vscode/settings.json` |

```
my-project/
├── frontend/                ← Root 1
│   └── .vscode/settings.json
├── backend/                 ← Root 2
│   └── .vscode/settings.json
└── project.code-workspace   ← Workspace 定義
```

## tasks.json 任務自動化

```json
{
  "version": "2.0.0",
  "tasks": [
    {
      "label": "Build",
      "type": "shell",
      "command": "npm run build",
      "group": { "kind": "build", "isDefault": true },
      "problemMatcher": ["$tsc"]
    },
    {
      "label": "Test",
      "type": "shell",
      "command": "npm test",
      "group": { "kind": "test", "isDefault": true }
    }
  ]
}
```

| 屬性 | 說明 |
|------|------|
| **label** | 任務名稱 (Command Palette 顯示) |
| **type** | `shell` / `process` |
| **group** | `build` / `test` / `none` |
| **problemMatcher** | 解析編譯錯誤輸出 |
| **dependsOn** | 前置任務依賴 |
| **presentation** | 輸出面板行為 (reveal/echo) |
| **快捷鍵** | `Ctrl+Shift+B` (Build), `Ctrl+Shift+T` (Test 需手動綁定) |

## launch.json 調試啟動配置

```json
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Launch Node",
      "type": "node",
      "request": "launch",
      "program": "${workspaceFolder}/src/index.js",
      "env": { "NODE_ENV": "development" }
    }
  ]
}
```

| 屬性 | 說明 |
|------|------|
| **type** | 調試器類型 (node / python / cppdbg) |
| **request** | `launch` (啟動) / `attach` (附加到進程) |
| **program** | 入口文件 |
| **args** | 命令行參數 |
| **env** | 環境變數 |
| **preLaunchTask** | 啟動前執行的 task |

## 集成終端機 Integrated Terminal

| 操作 | Win/Linux | macOS |
|------|:--------:|:-----:|
| 開/關終端 | `` Ctrl+` `` | `` Cmd+` `` |
| 新增終端 | `Ctrl+Shift+` ` | `Cmd+Shift+` ` |
| 切換終端 | `Ctrl+PageUp/Down` | — |
| 分割面板 | `Ctrl+Shift+5` | `Cmd+Shift+5` |
| 清除終端 | `Ctrl+K` (需 focus) | `Cmd+K` |

| 設定項 | 預設 | 說明 |
|--------|:---:|------|
| `terminal.integrated.shell.linux` | `bash` | 預設 Shell |
| `terminal.integrated.fontFamily` | monospace | 終端字型 |
| `terminal.integrated.cwd` | `${workspaceFolder}` | 起始目錄 |
