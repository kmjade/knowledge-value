## 擴展管理技巧

| 操作 | 方法 |
|------|------|
| 批量匯出 | `code --list-extensions > extensions.txt` |
| 批量安裝 | `cat extensions.txt | xargs -L1 code --install-extension` |
| 禁用/啟用 | 擴展面板右鍵 → Disable/Enable |
| 禁用所有 (暫時) | `code --disable-extensions .` |
| 按 Profile 分組 | 為不同場景建立不同 Profile 的擴展集 |
