# 07 — 調試與診斷 Debugging & Diagnostics

> Debugger 配置、斷點/監看/呼叫堆疊、語言專屬調試 (Python/Node/C++)、效能分析。

---

## 調試面板 Debug Panel

| 區塊 | 說明 |
|------|------|
| **VARIABLES** | 目前作用域變數值，可雙擊修改 |
| **WATCH** | 自訂監看表達式 (ex: `arr.length > 0`) |
| **CALL STACK** | 函數呼叫鏈，點擊跳轉對應幀 |
| **BREAKPOINTS** | 所有斷點列表，可勾選啟用/禁用 |
| **LOADED SCRIPTS** | 已載入腳本 (Node.js) |

| 操作 | 快捷鍵 (F-key) | 說明 |
|------|:------:|------|
| 開始/繼續 | `F5` | Start / Continue |
| 逐過程 | `F10` | Step Over |
| 逐語句 | `F11` | Step Into |
| 跳出 | `Shift+F11` | Step Out |
| 停止 | `Shift+F5` | Stop |
| 切換斷點 | `F9` | Toggle Breakpoint |
| 條件斷點 | 右鍵斷點 → Edit Breakpoint | 僅當條件為真時中斷 |

---

## 斷點類型 Breakpoint Types

| 類型 | 說明 |
|------|------|
| **Line Breakpoint** | 最常用，點擊行號左側空白 |
| **Conditional Breakpoint** | 表達式為真時觸發 (ex: `i > 100`) |
| **Logpoint** | 不中斷，僅輸出日誌 (類似 console.log) |
| **Function Breakpoint** | 在函數入口中斷 (不限特定檔案) |
| **Data Breakpoint** | 變數值變更時中斷 (C++/C# 支援) |
| **Inline Breakpoint** | 在一行中的特定表達式中斷 (`Shift+F9`) |
| **Exception Breakpoint** | 未捕獲/已捕獲異常時中斷 |

## 語言專屬調試 Per-Language Debug

### Python

```json
{
  "type": "python",
  "request": "launch",
  "name": "Python: Current File",
  "program": "${file}",
  "console": "integratedTerminal",
  "justMyCode": false
}
```

| 擴展 | `ms-python.debugpy` |
|------|------|
| **條件斷點** | ✅ 支援 |
| **遠端調試** | `debugpy` 模組 attach |
| **Django/Flask** | 自動偵測框架，一鍵啟動 |

### Node.js

```json
{
  "type": "node",
  "request": "launch",
  "name": "Launch Program",
  "program": "${workspaceFolder}/app.js",
  "runtimeArgs": ["--inspect"]
}
```

| 功能 | 說明 |
|------|------|
| **Auto Attach** | 自動附加到 `node --inspect` 進程 |
| **Browser Debug** | Chrome/Edge 調試支援 (透過 `ms-js-debug`) |
| **Source Maps** | 自動解析 .map 文件，支援 TS 調試 |

### C/C++

```json
{
  "type": "cppdbg",
  "request": "launch",
  "name": "C++ Debug",
  "program": "${workspaceFolder}/build/main",
  "MIMode": "gdb",
  "miDebuggerPath": "/usr/bin/gdb"
}
```

| 擴展 | `ms-vscode.cpptools` |
|------|------|
| **調試器** | GDB (Linux), LLDB (macOS), Visual Studio Debugger (Windows) |
| **記憶體檢視** | 原始記憶體檢視器 |
| **反組譯** | 切換到反組譯視圖 |

## 效能分析 Profiling

| 功能 | 說明 |
|------|------|
| **JavaScript/Node** | 內建 CPU Profile / Heap Snapshot (`.cpuprofile`/`.heappprofile`) |
| **Python** | 整合 `cProfile` / `py-spy` |
| **C++** | 整合 `perf` (Linux) / Instruments (macOS) |
| **Flame Chart** | 可視化函數調用耗時分佈 |
