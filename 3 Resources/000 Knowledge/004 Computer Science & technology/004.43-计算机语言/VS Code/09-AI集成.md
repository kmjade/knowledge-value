# 09 — AI 集成 AI Integration

> GitHub Copilot、Cline、Continue——VS Code 已是 AI 輔助編程的主戰場。

---

## GitHub Copilot / Copilot Chat

| 功能 | 說明 | 快捷鍵 |
|------|------|:------:|
| **行內補全** | 灰字建議，Tab 接受 | 自動觸發 |
| **下一個建議** | 循環查看備選 | `Alt+]` / `Alt+[` |
| **觸發建議** | 手動觸發補全 | `Ctrl+Enter` (先打開建議) |
| **Chat 面板** | 對話式問答 | `Ctrl+Shift+I` |
| **Inline Chat** | 在編輯器內對話 | `Ctrl+I` |
| **Explain** | 解釋選取程式碼 | 選取 → Copilot 選單 |
| **Fix** | 修復選取程式碼 | 選取 → Copilot 選單 |
| **Generate Tests** | 產生測試 | `Ctrl+Shift+P` → Copilot |
| **Agent Mode** | 自主執行任務 (讀寫檔案、執行命令) | Chat 中切換 |

---

## Copilot Prompt 工程技巧

| 技巧 | 說明 | 範例 |
|------|------|------|
| **高層次引導** | 先描述目標再給細節 | "Create a function that validates an email address. Use regex." |
| **約束風格** | 指定風格/命名規則 | "Use functional style, no classes. camelCase naming." |
| **提供上下文** | 開啟相關檔案作為參考 | `#file:src/types.ts` 引用 |
| **迭代修正** | 用 Inline Chat 逐步優化 | 選取 → `Ctrl+I` → "refactor to use async/await" |
| **Workspace Agent** | 讓 Agent 跨文件作業 | `@workspace` 前綴引用整個專案 |

---

## Cline (自主 AI 編程代理)

| 特性 | 說明 |
|------|------|
| **定位** | VS Code 內的自主 AI 編程代理 (類似 Cursor Agent) |
| **能力** | 讀寫檔案、執行終端命令、瀏覽網頁、安裝依賴 |
| **模型** | Claude / GPT / 本地模型 (Ollama/LM Studio) |
| **Diff 檢視** | 每次修改顯示 Diff，用戶確認後應用 |
| **MCP 整合** | 支援 Model Context Protocol 擴展能力 |
| **安全性** | 執行命令前需用戶確認 (可設定 auto-approve) |
| **安裝** | Marketplace 搜尋 `Cline` (saoudrizwan.claude-dev) |

---

## Continue (開源 AI 助手)

| 特性 | 說明 |
|------|------|
| **定位** | 開源 AI 程式碼助手 |
| **模型選擇** | 支援任何 LLM (OpenAI/Anthropic/本地/Ollama) |
| **Chat** | `Cmd+L` / `Ctrl+L` 對話 |
| **Edit** | `Cmd+I` / `Ctrl+I` 編輯模式 |
| **Autocomplete** | Tab 補全 |
| **自訂規則** | `.continuerc.json` 定義專案規則 |
| **VS Code 擴展** | `Continue.continue` |

## AI 工具對比 Comparison

| 工具 | 類型 | 開源 | 價格 | 最強項 |
|------|------|:---:|------|--------|
| **GitHub Copilot** | 補全 + Chat + Agent | ❌ | $10/月 (個人) | 生態最完整 |
| **Cline** | 自主 Agent | ✅ | 自備 API Key | 自主任務執行 |
| **Continue** | 開源助手 | ✅ | 自備 API Key | 開源可自訂 |
| **Cursor** | AI-first IDE | ❌ | $20/月 | 最強 AI 編輯器 (非純 VS Code) |
| **Windsurf** | AI IDE | ❌ | $15/月 | Codeium 整合 |

## AI Code Review

| 工具/方法 | 說明 |
|-----------|------|
| **Copilot Code Review** | 選取程式碼 → `Ctrl+Shift+I` → "Review this code" |
| **Cline Code Review** | 請 Cline 審查整個檔案/PR |
| **GitHub Actions + AI** | CI 中整合 AI 審查 (如 CodeRabbit) |
| **Prompt 技巧** | "Find potential bugs, security issues, and suggest improvements" |
