# 02 — 安裝與配置 Installation & Configuration

> 跨平台安裝、settings.json 配置體系、Settings Sync、`code` CLI。

---

## 安裝方式 Installation

| 平台 | 方法 | 指令 |
|------|------|------|
| **Windows** | 官網下載 | winget: `winget install Microsoft.VisualStudioCode` |
| **Windows** | Chocolatey | `choco install vscode` |
| **macOS** | Homebrew | `brew install --cask visual-studio-code` |
| **Linux (Debian)** | .deb | `sudo dpkg -i code_*.deb` |
| **Linux (RHEL)** | .rpm | `sudo rpm -i code_*.rpm` |
| **Linux (Arch)** | pacman | `sudo pacman -S code` |
| **Linux (Snap)** | Snap | `sudo snap install code --classic` |

## settings.json 配置體系

```
配置層級 (Priority: 高 → 低)
  Workspace Settings   →  .vscode/settings.json  (專案級)
  User Settings        →  %APPDATA%/Code/User/settings.json
  Default Settings     →  內建預設 (唯讀)
```

| 配置類型 | 檔案位置 | 作用域 | 適用場景 |
|----------|---------|:------:|---------|
| **User Settings** | `~/.config/Code/User/settings.json` (Linux) | 全域 | 通用偏好 |
| **Workspace** | `<project>/.vscode/settings.json` | 單一專案 | 專案規則 |
| **Folder (multi-root)** | `.code-workspace` 內的 settings | 多根工作區 | 多專案共享 |
| **Profile Settings** | Profile 獨立儲存 | Profile 作用域 | 場景切換 |

## 常用 settings.json 配置

```json
{
  "editor.fontSize": 14,
  "editor.fontFamily": "'Cascadia Code', 'Fira Code', monospace",
  "editor.fontLigatures": true,
  "editor.tabSize": 2,
  "editor.insertSpaces": true,
  "editor.renderWhitespace": "boundary",
  "editor.minimap.enabled": false,
  "editor.bracketPairColorization.enabled": true,
  "files.autoSave": "onFocusChange",
  "terminal.integrated.fontSize": 13,
  "workbench.colorTheme": "One Dark Pro",
  "workbench.iconTheme": "material-icon-theme"
}
```

## Settings Sync 設定同步

| 功能 | 說明 |
|------|------|
| **同步內容** | settings.json, keybindings.json, 擴展列表, Profile, Snippets, UI State |
| **平台** | 透過 Microsoft / GitHub 帳號雲端同步 |
| **啟動** | `Ctrl+Shift+P` → `Settings Sync: Turn On` |
| **衝突** | 合併或覆蓋可選 |

## code CLI

| 指令 | 功能 |
|------|------|
| `code .` | 在 VS Code 開啟目前目錄 |
| `code file.txt` | 開啟指定檔案 |
| `code -r .` | 在目前視窗開啟 (reuse) |
| `code -n .` | 在新視窗開啟 (new window) |
| `code --diff file1 file2` | 比較兩個檔案差異 |
| `code -g file:10:5` | 開啟檔案並跳到第 10 行 |
| `code --install-extension <ext-id>` | 安裝擴展 |
| `code --list-extensions` | 列出已安裝擴展 |
| `code --disable-extensions` | 禁用所有擴展啟動 |

> **安裝 CLI**: macOS: `Cmd+Shift+P` → `Shell Command: Install 'code' command in PATH`
> Windows: 安裝時勾選 "Add to PATH"
