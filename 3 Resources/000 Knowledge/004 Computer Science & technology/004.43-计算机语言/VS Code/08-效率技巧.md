# 08 — 效率技巧 Productivity Tips

> Command Palette、完整快速鍵速查、Keybindings 自訂、Zen Mode、Profiles——讓 VS Code 飛起來。

---

## Command Palette 命令面板

| 操作 | 快捷鍵 |
|------|:------:|
| 開啟命令面板 | `Ctrl+Shift+P` / `F1` |
| 檔案快速開啟 | `Ctrl+P` |
| 切換到符號 | `Ctrl+Shift+O` (檔案內 `@`) / `Ctrl+T` (全域 `#`) |
| Recent Files | `Ctrl+P` → 空白鍵暫停 |
| 命令面板操作 | `>` 前綴 (預設)、`?` 說明 |

> **Ctrl+P 技巧**: 輸入 `file.ts:42` 跳到第 42 行，`@functionName` 跳到函數

## Keybindings 自訂

```json
// keybindings.json (Ctrl+Shift+P → Preferences: Open Keyboard Shortcuts (JSON))
[
  {
    "key": "ctrl+shift+t",
    "command": "workbench.action.tasks.test",
    "when": "editorTextFocus"
  },
  {
    "key": "ctrl+k ctrl+d",
    "command": "editor.action.formatDocument",
    "when": "editorTextFocus"
  }
]
```

| when 條件 | 說明 |
|-----------|------|
| `editorTextFocus` | 編輯器聚焦時 |
| `editorHasSelection` | 有選取文字時 |
| `terminalFocus` | 終端聚焦時 |
| `!editorHasMultipleSelections` | 非多游標時 |
| `resourceExtname == .ts` | 僅在 .ts 檔案 |

## Zen Mode 禪模式

| 操作 | 說明 |
|------|------|
| `Ctrl+K Z` | 進入 Zen Mode (全螢幕無干擾) |
| `Esc` × 2 | 退出 Zen Mode |
| **特性** | 隱藏側欄、狀態列、活動列、Tabs (可選)，僅保留編輯器 |

## Profiles 配置檔

| 操作 | 方式 |
|------|------|
| 建立 Profile | `Ctrl+Shift+P` → `Profiles: Create Profile` |
| 切換 Profile | 左下角齒輪 → Profile 選單 |
| 匯出 Profile | `Profiles: Export Profile` → GitHub Gist 或本地檔案 |

| Profile 範例 | 目的 | 擴展重點 |
|-------------|------|---------|
| **Python Dev** | Python 開發 | Python, Pylance, Jupyter, Ruff |
| **Web Dev** | 前端開發 | ESLint, Prettier, Tailwind, Live Server |
| **Writing** | 寫作/筆記 | Markdown All in One, Foam, Grammarly |
| **Minimal** | 極簡模式 | 僅核心功能，0 擴展 |

## 其他效率技巧

| 技巧 | 說明 |
|------|------|
| **Autosave** | `files.autoSave`: `onFocusChange` / `afterDelay` |
| **File Nesting** | `explorer.fileNesting` 將相關檔案折疊 |
| **Breadcrumbs** | `Ctrl+Shift+.` 聚焦 breadcrumb 導航 |
| **Color Picker** | 在 CSS/HEX 值上懸停顯示顏色選擇器 |
| **Reopen Closed Editor** | `Ctrl+Shift+T` 重開上一個關閉的頁面 |
| **Preview Mode** | 單擊檔案為預覽 (斜體 Tab)，雙擊固定 |
| **Tab 管理** | `Ctrl+Tab` 切換最近 (按順序) |
