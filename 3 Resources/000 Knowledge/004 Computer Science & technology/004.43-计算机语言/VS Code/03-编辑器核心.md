# 03 — 編輯器核心 Editor Core

> Multi-cursor、IntelliSense、Snippets、Emmet、RegEx 搜尋取代——VS Code 編輯器的核心戰鬥力。

---

## Monaco Editor 核心能力

| 能力 | 說明 |
|------|------|
| **語法高亮** | TextMate 語法 + Semantic Highlighting (LSP) |
| **括號配對著色** | `editor.bracketPairColorization.enabled` (預設啟用) |
| **程式碼折疊** | `Ctrl+Shift+[` / `Ctrl+Shift+]` |
| **縮排指示線** | `editor.guides.indentation` |
| **Minimap** | 右側程式碼縮圖 |
| **Sticky Scroll** | 捲動時固定區塊頂部上下文 |

## Multi-Cursor 多重游標

| 操作 | Win/Linux | macOS |
|------|:--------:|:-----:|
| 加入游標 (點擊) | `Alt+Click` | `Option+Click` |
| 向上加游標 | `Ctrl+Alt+Up` | `Cmd+Option+Up` |
| 向下加游標 | `Ctrl+Alt+Down` | `Cmd+Option+Down` |
| 選取所有相同 | `Ctrl+Shift+L` | `Cmd+Shift+L` |
| 下一個相同詞 | `Ctrl+D` | `Cmd+D` |
| 跳過選取 | `Ctrl+K Ctrl+D` | `Cmd+K Cmd+D` |
| 行尾加游標 | `Shift+Alt+I` | `Shift+Option+I` |

## Snippets 程式碼片段

| 操作 | 快捷鍵 |
|------|:------:|
| 插入 Snippet | `Ctrl+Shift+P` → `Insert Snippet` |
| 自訂 Snippet | `Ctrl+Shift+P` → `Snippets: Configure User Snippets` |

```json
// javascript.json (User Snippet 範例)
{
  "Console Log": {
    "prefix": "clg",
    "body": ["console.log('$1:', $1);$0"],
    "description": "Insert console.log"
  }
}
```

| 變數 | 說明 |
|------|------|
| `$1, $2, ...` | Tab 停駐點 |
| `$0` | 最後游標位置 |
| `${1:default}` | 帶預設值的佔位符 |
| `$TM_FILENAME` | 目前檔名 |
| `$CURRENT_YEAR` | 當前年份 |

## Emmet 快速編寫 HTML/CSS

| 語法 | 展開結果 |
|------|---------|
| `div.container>ul>li*5` | `<div class="container"><ul><li></li>×5</ul></div>` |
| `div#app+p.foo` | `<div id="app"></div><p class="foo"></p>` |
| `a{Click me}` | `<a href="">Click me</a>` |
| `m10` | `margin: 10px;` |
| `bdrs5` | `border-radius: 5px;` |
| **觸發** | `Tab` 或 `Ctrl+E` (Expand Abbreviation) |

## 搜尋與取代 Find & Replace

| 操作 | Win/Linux | macOS |
|------|:--------:|:-----:|
| 檔案內搜尋 | `Ctrl+F` | `Cmd+F` |
| 檔案內取代 | `Ctrl+H` | `Cmd+H` |
| 全域搜尋 | `Ctrl+Shift+F` | `Cmd+Shift+F` |
| 全域取代 | `Ctrl+Shift+H` | `Cmd+Shift+H` |

| Regex 選項 | 圖示 | 說明 |
|:----------:|:----:|------|
| `.*` | `.*` | 啟用正則表達式 |
| `Aa` | `Aa` | 區分大小寫 |
| `ab` | `ab` | 匹配整個單詞 |

> 常用 Regex: `\\b\\w+\\b` (單詞), `^\\s*$` (空行), `TODO|FIXME|HACK` (標記)
