# 物件導向編程 — Object-Oriented Programming

> 🎯 **學習目標**：掌握 Python 的 OOP 核心機制，包括 class/object、繼承/MRO、magic methods、property/descriptor、@dataclass、ABC 等。

---

## 📑 核心概念速覽

| 概念 (Concept) | 說明 (Description) | Python 關鍵詞 |
|:---|:---|:---|
| Class | 物件的藍圖（blueprint） | `class` |
| Object / Instance | 類別的實例 | `obj = MyClass()` |
| Encapsulation | 封裝：隱藏內部實作 | `_protected`, `__private` |
| Inheritance | 繼承：子類繼承父類屬性與方法 | `class Child(Parent)` |
| Polymorphism | 多型：同一介面，不同實作 | duck typing / method override |
| Abstraction | 抽象：定義介面、延遲實作 | `ABC`, `@abstractmethod` |
| Composition | 組合：以 has-a 替代 is-a | 將物件作為屬性 |
| MRO | 方法解析順序 | C3 linearization, `__mro__` |

## 🏗️ 類別與物件基礎

### 定義類別與創建實例

```python
class Person:
    """人類別（Person）"""

    # 類別屬性（class attribute）：所有實例共享
    species = "Homo sapiens"

    # 實例初始化（instance initializer）
    def __init__(self, name: str, age: int):
        # 實例屬性（instance attribute）：每個實例獨立
        self.name = name          # 公開屬性
        self._age = age           # 受保護屬性（約定）
        self.__secret = "hidden"  # 私有屬性（name mangling → _Person__secret）

    # 實例方法
    def greet(self) -> str:
        return f"Hi, I'm {self.name}, {self._age} years old."

    # 私有方法
    def __private_method(self):
        return self.__secret

# 創建實例
p1 = Person("Alice", 30)
p2 = Person("Bob", 25)

print(p1.greet())          # Hi, I'm Alice, 30 years old.
print(p1.species)          # Homo sapiens
print(Person.species)      # Homo sapiens

# Name mangling 存取私有屬性
print(p1._Person__secret)  # hidden (不推薦)
```

### __init__ vs __new__

| | `__new__` | `__init__` |
|:---|:---|:---|
| 調用順序 | 先於 `__init__` | 後於 `__new__` |
| 用途 | 創建並返回實例（singleton 等） | 初始化實例屬性 |
| 返回值 | 必須返回實例 | 返回 None |

```python
class Singleton:
    _instance = None

    def __new__(cls, *args, **kwargs):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance

    def __init__(self, value=None):
        if not hasattr(self, 'value'):
            self.value = value

a = Singleton(1)
b = Singleton(2)
print(a is b)        # True
print(a.value, b.value)  # 1 1
```

## 🧬 繼承與多型

### 單一繼承

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        raise NotImplementedError("Subclass must implement speak()")

class Dog(Animal):
    def speak(self):
        return f"{self.name} says Woof!"

class Cat(Animal):
    def speak(self):
        return f"{self.name} says Meow!"

# 多型：同一介面，不同行為
animals = [Dog("Buddy"), Cat("Kitty")]
for animal in animals:
    print(animal.speak())
# Buddy says Woof!
# Kitty says Meow!
```

### 多重繼承與 MRO (C3 Linearization)

| 規則 | 說明 |
|:---|:---|
| C3 Linearization | Python 的方法解析順序演算法 |
| DFS + 單調性 | 保證子類優先於父類、保持繼承順序 |
| `__mro__` | 查看類別的 MRO 順序 |
| `super()` | 按 MRO 順序調用下一個類別 |

```python
class A:
    def method(self):
        print("A.method")

class B(A):
    def method(self):
        print("B.method")
        super().method()

class C(A):
    def method(self):
        print("C.method")
        super().method()

class D(B, C):
    def method(self):
        print("D.method")
        super().method()

d = D()
d.method()
# D.method → B.method → C.method → A.method

print(D.__mro__)
# (<class 'D'>, <class 'B'>, <class 'C'>, <class 'A'>, <class 'object'>)
```

### Mixin 模式

```python
class SerializeMixin:
    """將物件序列化為 dict"""

    def to_dict(self) -> dict:
        return {
            k: v for k, v in self.__dict__.items()
            if not k.startswith('_')
        }

class LoggerMixin:
    """日誌混入"""

    def log(self, message: str):
        print(f"[{self.__class__.__name__}] {message}")

class User(SerializeMixin, LoggerMixin):
    def __init__(self, name, email):
        self.name = name
        self.email = email

u = User("Alice", "alice@example.com")
print(u.to_dict())  # {'name': 'Alice', 'email': 'alice@example.com'}
u.log("User created")
```

## ✨ Magic Methods (Dunder Methods)

| 分類 | Methods | 用途 |
|:---|:---|:---|
| 建構/銷毀 | `__init__`, `__new__`, `__del__` | 物件生命週期 |
| 字串表示 | `__str__`, `__repr__`, `__format__` | print/str/repr |
| 比較運算 | `__eq__`, `__ne__`, `__lt__`, `__le__`, `__gt__`, `__ge__` | `==`, `<`, `>` 等 |
| 算術運算 | `__add__`, `__sub__`, `__mul__`, `__truediv__` | `+`, `-`, `*`, `/` |
| 容器模擬 | `__len__`, `__getitem__`, `__setitem__`, `__contains__` | `len()`, `obj[key]`, `in` |
| 可呼叫 | `__call__` | 讓實例可被呼叫 |
| 上下文 | `__enter__`, `__exit__` | context manager |
| 屬性存取 | `__getattr__`, `__setattr__`, `__delattr__` | 屬性攔截 |

### 完整範例：實作一個 Vector 類別

```python
import math

class Vector:
    """二維向量，展示常見 magic methods"""

    def __init__(self, x: float, y: float):
        self.x = x
        self.y = y

    def __repr__(self) -> str:
        """開發者字串表示，eval(repr(obj)) 應重建物件"""
        return f"Vector({self.x!r}, {self.y!r})"

    def __str__(self) -> str:
        """使用者友好字串"""
        return f"({self.x}, {self.y})"

    def __abs__(self) -> float:
        return math.hypot(self.x, self.y)

    def __bool__(self) -> bool:
        return bool(abs(self))

    def __eq__(self, other: "Vector") -> bool:
        if not isinstance(other, Vector):
            return NotImplemented
        return self.x == other.x and self.y == other.y

    def __hash__(self) -> int:
        """若自定義 __eq__ 且需可 hash，須同時定義 __hash__"""
        return hash((self.x, self.y))

    def __add__(self, other: "Vector") -> "Vector":
        return Vector(self.x + other.x, self.y + other.y)

    def __mul__(self, scalar: float) -> "Vector":
        return Vector(self.x * scalar, self.y * scalar)

    def __call__(self):
        """讓實例可被呼叫"""
        print(f"Vector called: ({self.x}, {self.y})")

    def __getitem__(self, index: int) -> float:
        return (self.x, self.y)[index]

    def __len__(self) -> int:
        return 2

# 使用
v1 = Vector(3, 4)
v2 = Vector(1, 2)

print(repr(v1))        # Vector(3, 4)
print(str(v1))         # (3, 4)
print(abs(v1))         # 5.0
print(bool(v1))        # True
print(v1 == v2)        # False
print(v1 + v2)         # (4, 6)
print(v1 * 2)          # (6, 8)
print(v1[0])           # 3
print(len(v1))         # 2
v1()                   # Vector called: (3, 4)
```

## 🔐 Property 與 Descriptor

### @property

```python
class Temperature:
    def __init__(self, celsius: float):
        self._celsius = celsius

    @property
    def celsius(self) -> float:
        """讀取攝氏溫度"""
        return self._celsius

    @celsius.setter
    def celsius(self, value: float):
        """設置攝氏溫度（附帶驗證）"""
        if value < -273.15:
            raise ValueError("Temperature below absolute zero!")
        self._celsius = value

    @property
    def fahrenheit(self) -> float:
        """華氏溫度（唯讀計算屬性）"""
        return self._celsius * 9 / 5 + 32

t = Temperature(25)
print(t.celsius)       # 25
print(t.fahrenheit)    # 77.0
t.celsius = 30
print(t.fahrenheit)    # 86.0
# t.celsius = -300     # ValueError!
```

### Descriptor Protocol

```python
class ValidatedString:
    """Descriptor：驗證字串長度"""

    def __init__(self, min_len: int = 1, max_len: int = 100):
        self.min_len = min_len
        self.max_len = max_len
        self.data = {}  # 以實例 id 存儲值，避免類別級別共享

    def __set_name__(self, owner, name):
        self.name = name

    def __get__(self, obj, objtype=None):
        if obj is None:
            return self
        return self.data.get(id(obj), '')

    def __set__(self, obj, value):
        if not isinstance(value, str):
            raise TypeError(f"{self.name} must be a string")
        if not (self.min_len <= len(value) <= self.max_len):
            raise ValueError(
                f"{self.name} length must be between "
                f"{self.min_len} and {self.max_len}"
            )
        self.data[id(obj)] = value

class User:
    name = ValidatedString(min_len=2, max_len=50)
    bio = ValidatedString(min_len=0, max_len=500)

    def __init__(self, name: str, bio: str = ""):
        self.name = name
        self.bio = bio

u = User("Alice", "Python developer")
print(u.name)  # Alice
```

## 📦 @dataclass

```python
from dataclasses import dataclass, field, asdict, astuple
from typing import List

@dataclass(order=True, frozen=False)
class Product:
    """商品資料類別"""

    name: str
    price: float
    tags: List[str] = field(default_factory=list)  # 可變默認值必須用 field()
    _stock: int = field(default=0, repr=False)      # 排除 repr

    @property
    def stock(self) -> int:
        return self._stock

    def restock(self, qty: int):
        self._stock += qty

# 使用
p1 = Product("Laptop", 999.99, ["electronics"])
p2 = Product("Mouse", 29.99, ["electronics", "peripheral"])

print(p1)                    # Product(name='Laptop', price=999.99, tags=['electronics'])
print(p1 == p2)              # False
print(asdict(p1))            # {'name': 'Laptop', 'price': 999.99, 'tags': ['electronics']}

# 凍結 (frozen=True) 的 dataclass 不可變
@dataclass(frozen=True)
class Point:
    x: float
    y: float
```

## 🧪 抽象基底類別 (ABC)

```python
from abc import ABC, abstractmethod, abstractproperty

class Shape(ABC):
    """抽象形狀"""

    @abstractmethod
    def area(self) -> float:
        """計算面積"""
        ...

    @abstractmethod
    def perimeter(self) -> float:
        """計算周長"""
        ...

    def describe(self) -> str:
        """具體方法（concrete method）"""
        return f"Shape area={self.area():.2f}, perimeter={self.perimeter():.2f}"

class Circle(Shape):
    def __init__(self, radius: float):
        self.radius = radius

    def area(self) -> float:
        return 3.14159 * self.radius ** 2

    def perimeter(self) -> float:
        return 2 * 3.14159 * self.radius

class Rectangle(Shape):
    def __init__(self, width: float, height: float):
        self.width = width
        self.height = height

    def area(self) -> float:
        return self.width * self.height

    def perimeter(self) -> float:
        return 2 * (self.width + self.height)

# shape = Shape()  # TypeError: Can't instantiate abstract class
c = Circle(5)
print(c.describe())  # Shape area=78.54, perimeter=31.42
```

### ABC 註冊虛擬子類

```python
from abc import ABC, abstractmethod

class Drawable(ABC):
    @abstractmethod
    def draw(self):
        ...

# 無需繼承即註冊為虛擬子類
class LegacyRenderer:
    def draw(self):
        print("Legacy draw")

Drawable.register(LegacyRenderer)

print(issubclass(LegacyRenderer, Drawable))  # True
renderer = LegacyRenderer()
print(isinstance(renderer, Drawable))         # True
```

## 🔄 常用設計模式速覽

| 模式 | Python 實現 | 適用場景 |
|:---|:---|:---|
| Singleton | `__new__` 控制實例化 | 配置管理、日誌 |
| Factory | function / classmethod | 物件創建解耦 |
| Builder | method chaining | 複雜物件構建 |
| Observer | signal / callback | 事件驅動 |
| Strategy | function as argument | 演算法互換 |
| Decorator | `@decorator` 語法糖 | 橫切關注點 |

### Strategy Pattern 範例

```python
from typing import Callable

class Sorter:
    """排序器：策略模式"""

    def __init__(self, strategy: Callable = sorted):
        self.strategy = strategy

    def sort(self, data):
        return self.strategy(data)

# 不同策略
by_length = Sorter(strategy=lambda d: sorted(d, key=len))
by_reverse = Sorter(strategy=lambda d: sorted(d, reverse=True))

words = ["Python", "Go", "JavaScript", "C"]
print(by_length.sort(words))   # ['C', 'Go', 'Python', 'JavaScript']
print(by_reverse.sort(words))  # ['Python', 'JavaScript', 'Go', 'C']
```

## 🎯 重點回顧

1. **`self`** 是實例方法的隱式第一個參數，指向當前實例
2. **MRO** 決定多重繼承的方法查找順序，使用 C3 linearization
3. **magic methods** 是 Python 的運算符重載與協議實作核心
4. **`@property`** 實現 getter/setter，descriptor 提供更細粒度控制
5. **`@dataclass`** 簡化資料類別定義，自動生成 `__init__`/`__repr__`/`__eq__`
6. **ABC** 強制子類實作介面，`register()` 支援虛擬子類

---

> 💡 **最佳實踐**：優先組合 (composition) 而非繼承；使用 Mixin 橫向復用功能；@dataclass 替代手寫 boilerplate；ABC 定義清晰合約。
