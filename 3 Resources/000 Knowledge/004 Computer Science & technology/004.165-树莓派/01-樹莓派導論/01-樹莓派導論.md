# 01 樹莓派導論

## SBC 定義 (Single Board Computer)

單板計算機 (SBC) 是將處理器、記憶體、存儲接口、I/O 全部整合在一張電路板上的完整計算機。與傳統 PC 不同，SBC 強調低成本、低功耗、小體積與豐富的 GPIO 接腳，適合嵌入式應用、教育與原型開發。

> 截至 2024 年，樹莓派系列累計出貨超過 **6,000 萬台**，是全球最暢銷的單板計算機。

## 生態系統全景

```
軟體層          Raspberry Pi OS (Debian Bookworm) / Ubuntu
                └── 程式語言：Python (gpiozero)、C (libgpiod)
中介層          rpi-eeprom / bootloader / device tree
硬體層          BCM27xx SoC + RP1 I/O Controller (Pi 5)
週邊生態        HAT 擴展板、Camera Module、Sense HAT、AI Kit
社群            forums.raspberrypi.com、piwheels、MagPi
```

## MCU vs SBC 對比

| 特性 | 樹莓派 (SBC) | Arduino (MCU) | STM32 (MCU) |
|------|:-----------:|:------------:|:----------:|
| **核心** | ARM Cortex-A (應用處理器) | ATmega328 (8-bit) | ARM Cortex-M (32-bit) |
| **作業系統** | Linux (完整 OS) | 無 OS / Bare metal | FreeRTOS / Bare metal |
| **時脈** | 1.5–2.4 GHz | 16 MHz | 72–480 MHz |
| **RAM** | 512MB–8GB | 2KB SRAM | 20KB–1MB SRAM |
| **GPIO** | 26–40-pin, 3.3V | 14–20-pin, 5V | 多通道 3.3V/5V 容忍 |
| **功耗** | 2.5–12W | ~0.1W | ~0.1–0.5W |
| **適合** | 多工、網路、AI、多媒體 | 簡單控制、低功耗 | 即時控制、工業應用 |

## 與 004.16 嵌入式系統的分工

| 領域 | 004.16 嵌入式系統 | 004.165 樹莓派 |
|------|-------------------|----------------|
| 平台 | MCU (STM32/Arduino/ESP32) | Linux SBC (Raspberry Pi) |
| 重點 | RTOS、中斷、即時控制、低功耗 | Linux 應用、多媒體、網路服務 |
| 演算能力 | 受限，無 MMU | 完整 MMU，可跑 Docker/AI |
| 典型場景 | 傳感器讀取、馬達控制 | 上位機、邊緣運算、網關 |

> 在 UGV 小車專案中：**STM32 = 即時控制層（下位機），樹莓派 = ROS2 導航 + 視覺（上位機）**。兩者透過 UART/USB 通訊協作。

## 樹莓派 vs 其他 SBC

| SBC | SoC | RAM | 價格 | 亮點 |
|-----|-----|:---:|:----:|------|
| **Raspberry Pi 5** | BCM2712 | 4/8GB | $60-80 | 生態最強、文件最全 |
| Orange Pi 5 | RK3588S | 4-32GB | $60-130 | NPU 6 TOPS、更強性能 |
| Jetson Nano | Tegra X1 | 4GB | $149 | CUDA、AI 推理專用 |
| ROCK 5B | RK3588 | 4-16GB | $80-160 | PCIe 3.0 x4、M.2 NVMe |

> 選擇建議：**生態與社群優先 → Pi 5；AI 推理 → Jetson；性價比 → Orange Pi 5**。
