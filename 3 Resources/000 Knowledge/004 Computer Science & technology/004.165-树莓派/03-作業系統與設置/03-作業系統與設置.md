# 03 作業系統與設置 OS & Setup

## Raspberry Pi OS

官方作業系統，基於 Debian Bookworm（12），針對 ARM 架構深度優化。

| 版本 | 桌面環境 | 預裝軟體 | 映像大小 | 適用場景 |
|------|:-------:|----------|:-------:|----------|
| **Desktop (Full)** | PIXEL (Wayland) | 辦公/教育/編程全套 | ~3GB | 桌面替代、教學 |
| **Desktop** | PIXEL (Wayland) | 基本工具 | ~2GB | 一般開發 |
| **Lite** | 無 (純 CLI) | 最小系統 | ~0.5GB | 伺服器、嵌入式、headless |

> Pi 5 預設使用 **Wayland** (wayfire)，Pi 4 及以下仍可用 X11 兼容模式。

## 可選作業系統

| OS | 特點 | 適合 |
|----|------|------|
| **Ubuntu Server/Desktop** | 完整 Ubuntu 生態、Snap | 熟悉 Ubuntu 的開發者 |
| **DietPi** | 極輕量 (Debian-based) | 資源極度受限場景 |
| **LibreELEC** | Kodi 媒體中心專用 | HTPC 家庭影音 |
| **Home Assistant OS** | 智慧家居專用 | HA 中樞 |
| **Kali Linux** | 滲透測試工具集 | 安全測試 |
| **RetroPie** | 復古遊戲模擬器整合 | 遊戲機 |

## Raspberry Pi Imager 工具

官方燒錄工具 (Windows/macOS/Linux)，三步完成：

```
1. 選擇裝置 (Raspberry Pi 5 / 4 / Zero 2W ...)
2. 選擇 OS (Raspberry Pi OS / Ubuntu / ...)
3. 選擇儲存媒體 (microSD / USB)
```

**進階設定 (Ctrl+Shift+X)**：
- 設定 hostname (預設 `raspberrypi`)
- 啟用 SSH (密碼或 SSH key)
- 設定 WiFi SSID/密碼 (headless 必備)
- 設定使用者名稱與密碼
- 時區與鍵盤佈局

> 使用 Imager 進階設定可以在燒錄時一次性完成所有初始設定，無需開機後手動配置。

## 初始設置流程

### 方法一：有螢幕 + 鍵盤

```bash
# 開機後自動進入桌面/CLI
sudo raspi-config
```

### 方法二：Headless 無頭設置 (無螢幕)

```bash
# 1. 用 Imager 進階設定寫入 WiFi + SSH + user/password
# 2. 插入 microSD 開機，等待 2 分鐘
# 3. 尋找 Pi 的 IP：
ping raspberrypi.local           # macOS/Linux
# 或用路由器 DHCP 列表查找
# 4. SSH 連線：
ssh pi@raspberrypi.local
# 5. 首次登入後立即更新：
sudo apt update && sudo apt full-upgrade -y
```

## raspi-config 常用設定

| 選項 | 功能 |
|------|------|
| System Options → Wireless LAN | 設定 WiFi |
| System Options → Boot/Auto Login | 開機模式 (Desktop/CLI) |
| Interface Options → SSH | 啟用/關閉 SSH |
| Interface Options → I2C / SPI | 啟用通訊協議 |
| Interface Options → Camera | 啟用舊版 camera stack |
| Performance Options → Fan | 風扇 GPIO 與溫度閾值 |
| Localisation Options | 時區、鍵盤、Locale |

## rpi-eeprom 韌體管理

```bash
# 查看當前 bootloader 版本
sudo rpi-eeprom-update

# 更新至最新 stable 韌體
sudo rpi-eeprom-update -a

# 設定 boot order (Pi 5 支援 NVMe 開機)
sudo rpi-eeprom-config --edit
# BOOT_ORDER=0xf416  → NVMe 優先, 然後 SD, 然後 USB

# 查看 config
vcgencmd bootloader_config
```

## 建議的首次設定

```bash
# 1. 更新系統
sudo apt update && sudo apt full-upgrade -y

# 2. 安裝常用工具
sudo apt install -y git curl wget vim htop neofetch

# 3. 設定 firewall (可選)
sudo apt install -y ufw
sudo ufw allow ssh
sudo ufw enable

# 4. 啟用必要的 interfaces
sudo raspi-config nonint do_i2c 0
sudo raspi-config nonint do_spi 0

# 5. 重啟
sudo reboot
```
