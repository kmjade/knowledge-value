## 氣象站 Weather Station

### 架構

```
BME280 (I²C) → Python daemon → MQTT → InfluxDB → Grafana
```

### 步驟

```bash
# 1. 硬體接線
Pi 3.3V → BME280 VCC
Pi GND  → BME280 GND
Pi SDA  → BME280 SDA
Pi SCL  → BME280 SCL

# 2. 安裝依賴
sudo apt install -y python3-pip influxdb grafana
pip3 install adafruit-circuitpython-bme280 paho-mqtt

# 3. Python 採集腳本 (sensor_publisher.py)
```

```python
import time, json, board, adafruit_bme280
import paho.mqtt.client as mqtt

i2c = board.I2C()
bme = adafruit_bme280.Adafruit_BME280_I2C(i2c, address=0x77)
client = mqtt.Client()
client.connect("localhost", 1883)

while True:
    data = {
        "temperature": round(bme.temperature, 1),
        "humidity": round(bme.humidity, 1),
        "pressure": round(bme.pressure, 1)
    }
    client.publish("weather/station1", json.dumps(data))
    time.sleep(60)
```

```bash
# 4. Node-RED: MQTT in → InfluxDB out
# 5. Grafana: Data source → InfluxDB → Dashboard panels
```

## 智慧家居中樞 Smart Home Hub

### 架構

```
Pi 5 + Home Assistant OS
├── Zigbee Coordinator (USB dongle: Sonoff ZBDongle-P)
├── MQTT Broker (內建)
├── Node-RED (add-on)
└── 設備整合
    ├── Zigbee: 門窗感應器、智慧燈泡、插座
    ├── WiFi: ESPHome 自製傳感器
    └── 藍牙: BLE Tracker / 體溫計
```

### 推薦硬體

| 組件 | 型號 | 價格 |
|------|------|:----:|
| 主機 | Pi 5 (8GB) + NVMe 128GB | $130 |
| Zigbee dongle | Sonoff ZBDongle-P (CC2652P) | $18 |
| UPS HAT | PiJuice / Geekworm X708 | $30 |

> 使用 Home Assistant OS 直接燒錄，免去 Docker 配置複雜度。搭配 Tailscale 可在外網安全存取。

## 機器人大腦 Robot Brain

### 架構 (以 UGV 小車為例)

```
上位機 (樹莓派)
├── ROS2 Humble
├── Nav2 (路徑規劃)
├── SLAM (Cartographer / SLAM Toolbox)
├── 視覺：Camera Module 3 → YOLO 物件偵測
└── 通訊：UART/USB → STM32 (下位機)

下位機 (STM32)  — 參見 [[../004.16-嵌入式系統/004.16-嵌入式系統|004.16 嵌入式系統]]
├── 馬達控制 (PID)
├── 編碼器讀取 (里程計)
├── IMU (MPU6050)
└── 電池監控
```

> 完整 UGV 小車專案請參見 [[3 Resources/600 Applied Sciences/689 业余手工/08-電子DIY/09-進階專案/UGV小车/UGV小车|UGV 小車]]。

### 關鍵點

- Pi 5 透過 **UART** 與 STM32 通訊（115200–921600 bps）
- Pi 處理 **高層邏輯**：路徑規劃、視覺、地圖
- STM32 處理 **即時控制**：馬達 PWM、編碼器中斷
- Camera Module 3 提供視覺輸入，AI Kit (Hailo-8L) 可加速 YOLO 推理

## NAS 網絡附加存儲

```bash
# OpenMediaVault (OMV) 安裝
# 基於 Debian，專為 NAS 設計的 Web GUI

# 方法一：OMV 官方映像 (推薦)
# 從 omv-extras.org 下載 Raspberry Pi 映像 → 燒錄

# 方法二：在 Raspberry Pi OS 上安裝
curl -sSL https://github.com/OpenMediaVault-Plugin-Developers/installScript/raw/master/install | sudo bash

# 存取 http://<pi-ip>
# 預設帳號：admin / openmediavault
```

### 儲存配置建議

| 方案 | 硬碟 | RAID | 容量 | 適合 |
|------|------|:----:|:----:|------|
| 基本 | USB 3.0 外接 HDD | 無 | 1-4TB | 家庭備份 |
| 雙碟鏡像 | 2×USB HDD | RAID 1 | 鏡像 50% | 資料安全 |
| NVMe | M.2 HAT + NVMe SSD | 無 | 128GB-2TB | 高速存取 |

> Pi 5 + NVMe SSD + USB HDD = 低成本、低功耗的家庭 NAS。功耗僅 5-10W。

## Pi-hole 廣告攔截

```bash
# 一鍵安裝
curl -sSL https://install.pi-hole.net | bash

# 管理後台 http://<pi-ip>/admin
# 設定路由器 DNS 指向 Pi 的 IP 即可全屋去廣告
```

### Pi-hole 核心功能

| 功能 | 說明 |
|------|------|
| DNS 過濾 | 阻擋廣告/追蹤域名 (~150K 預設清單) |
| DHCP 伺服器 | 可取代路由器 DHCP |
| 統計面板 | 查詢次數、阻擋率、域名排行 |

> 建置簡單、資源消耗極低 (Pi Zero 2W 即可)，效果顯著。

## RetroPie 復古遊戲機

```bash
# 在 Raspberry Pi OS 上安裝
sudo apt install -y git dialog
git clone --depth=1 https://github.com/RetroPie/RetroPie-Setup.git
cd RetroPie-Setup
sudo ./retropie_setup.sh

# 或直接燒錄 RetroPie 映像
# 下載 https://retropie.org.uk/download/
```

### 支援平台

| 平台 | 模擬器 | 效能 (Pi 5) |
|------|--------|:-----------:|
| NES / SNES | lr-nestopia / lr-snes9x | ✅ 完美 |
| PS1 | lr-pcsx-rearmed | ✅ 完美 60fps |
| PSP | PPSSPP | ✅ 大部分流暢 |
| Dreamcast | lr-flycast | ✅ 完美 |
| N64 | lr-mupen64plus | ✅ 流暢 (Pi 5) |
| PS2 | AetherSX2 | ⚠️ 部分可玩 (Pi 5) |

> Pi 5 CPU 性能 (~2.4GHz Cortex-A76) 已可流暢模擬大部分 Dreamcast/PSP 遊戲。N64 在 Pi 5 上首次不再卡頓。
