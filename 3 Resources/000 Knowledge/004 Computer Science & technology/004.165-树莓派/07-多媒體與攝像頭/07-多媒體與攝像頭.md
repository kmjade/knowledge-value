## libcamera 工具鏈

Pi 5/4 使用 **libcamera** (取代舊版 raspistill/raspivid)。

### 靜態照片

```bash
# 基本拍攝
rpicam-still -o photo.jpg

# 指定解析度、旋轉
rpicam-still -o photo.jpg --width 1920 --height 1080 --rotation 180

# 延時攝影 (每 5 秒一張，共 60 張)
rpicam-still -t 300000 --timelapse 5000 -o timelapse_%04d.jpg

# 手動對焦 (Cam 3)
rpicam-still -o focused.jpg --autofocus-mode manual --lens-position 5.0
```

### 錄影

```bash
# 1080p@30fps, 30 秒
rpicam-vid -t 30000 -o video.h264 --width 1920 --height 1080

# 4K HEVC
rpicam-vid -t 10000 -o 4k.h265 --width 3840 --height 2160 --codec h265

# 含時間戳預覽
rpicam-vid -t 0 --inline -o - | cvlc stream:///dev/stdin --sout '#rtp{sdp=rtsp://:8554/}'
```

### libcamera 常用參數

| 參數 | 說明 |
|------|------|
| `-t` | 拍攝/錄影時長 (ms, 0=無限制) |
| `--width` / `--height` | 解析度 |
| `--rotation` | 旋轉 (0/90/180/270) |
| `--brightness` | 亮度 (-1.0 ~ 1.0) |
| `--contrast` | 對比 (0 ~ 32) |
| `--autofocus-mode` | manual / auto / continuous (Cam 3) |
| `-n` | 無預覽 |

## Raspberry Pi AI Kit (Hailo-8L)

| 參數 | 規格 |
|------|------|
| NPU | Hailo-8L |
| 算力 | 13 TOPS (INT8) |
| 功耗 | ~2.5W |
| 接口 | M.2 (PCIe 2.0 ×1) |
| 框架 | HailoRT, TAPPAS, ONNX |

```bash
# 安裝 Hailo 軟體
sudo apt install hailo-all
# 運行範例
hailortcli run /usr/share/hailo-models/yolov5m.hef
```

> AI Kit 透過 Pi 5 的 PCIe M.2 接口連接。支援 YOLOv5/v8、ResNet、MobileNet 等模型推理。適合邊緣視覺應用。

## 影片串流 (RTSP)

### 方案一：rpicam-vid + VLC

```bash
# 將 rpicam-vid 輸出管線至 VLC RTSP
rpicam-vid -t 0 --inline --width 1280 --height 720 --framerate 30 \
  -o - | cvlc stream:///dev/stdin \
  --sout '#rtp{sdp=rtsp://:8554/stream}' :demux=h264
```

### 方案二：ffmpeg

```bash
# libcamera → ffmpeg → RTMP (YouTube Live) 或 RTSP
rpicam-vid -t 0 --width 1920 --height 1080 --framerate 30 \
  --inline -o - | \
  ffmpeg -i - -c copy -f flv rtmp://a.rtmp.youtube.com/live2/YOUR_KEY
```

### 方案三：MJPG-Streamer (低延遲)

```bash
# 適合網頁預覽 (MJPEG)
sudo apt install -y cmake libjpeg8-dev
git clone https://github.com/jacksonliam/mjpg-streamer
cd mjpg-streamer/mjpg-streamer-experimental
make
./mjpg_streamer -i "input_libcamera.so" -o "output_http.so -w ./www"
# 訪問 http://<pi-ip>:8080/stream.html
```

## 雙顯示器配置

Pi 5/4 支援雙 HDMI 輸出：

```bash
# 鏡像模式
xrandr --output HDMI-1 --same-as HDMI-2

# 延伸桌面
xrandr --output HDMI-1 --auto --output HDMI-2 --auto --right-of HDMI-1

# 查看顯示器
kmsprint
```

## 音頻

| 功能 | 方式 |
|------|------|
| **HDMI 音頻** | 預設輸出，隨 HDMI 自動 |
| **3.5mm 耳機孔** | Pi 5 無 3.5mm 孔（需 USB 音效卡或 HDMI） |
| **USB 麥克風** | `arecord -l` 確認，即插即用 |
| **I2S DAC** | 需 overlay (`dtoverlay=hifiberry-dac`) |
| **藍牙音頻** | `bluetoothctl` 配對 A2DP 裝置 |
