# 05 傳感器與執行器 Sensors & Actuators

## 溫度與濕度傳感器

### DHT22 (AM2302) — 數位溫濕度

| 參數 | 範圍 | 精度 |
|------|------|:---:|
| 溫度 | -40°C ~ 80°C | ±0.5°C |
| 濕度 | 0% ~ 100% RH | ±2% RH |
| 通訊 | 單線 (One-wire 協定風格) | — |
| 取樣率 | 0.5 Hz (每 2 秒) | — |

```python
# 使用 adafruit-circuitpython-dht 庫
import adafruit_dht
import board
dht = adafruit_dht.DHT22(board.D4)  # GPIO 4
temp = dht.temperature  # °C
hum = dht.humidity      # %
```

> DHT22 需要 4.7kΩ–10kΩ pull-up 電阻於 DATA 線接 3.3V。

### BME280 — 精密溫濕壓

| 參數 | 範圍 | 精度 |
|------|------|:---:|
| 溫度 | -40°C ~ 85°C | ±0.5°C |
| 濕度 | 0% ~ 100% RH | ±3% |
| 氣壓 | 300 ~ 1100 hPa | ±1 hPa |
| 通訊 | I²C (0x76/0x77) 或 SPI | — |

```python
import board
import adafruit_bme280
i2c = board.I2C()
bme = adafruit_bme280.Adafruit_BME280_I2C(i2c, address=0x76)
print(f"{bme.temperature:.1f}°C, {bme.humidity:.1f}%, {bme.pressure:.1f}hPa")
```

| 傳感器 | 價格 | 精度 | 通訊 | 適用 |
|--------|:---:|:---:|------|------|
| DHT11 | $1 | ±2°C | One-wire | 入門 |
| DHT22 | $3 | ±0.5°C | One-wire | 一般 |
| BME280 | $5 | ±0.5°C/±1hPa | I²C/SPI | 氣象站 |
| DS18B20 | $2 | ±0.5°C | 1-Wire | 多點測溫 |

## 超音波測距 — HC-SR04

| 參數 | 規格 |
|------|------|
| 範圍 | 2cm ~ 400cm |
| 精度 | ±3mm |
| 工作電壓 | **5V** (需分壓 Echo 腳) |
| 腳位 | VCC / Trig / Echo / GND |

```python
from gpiozero import DistanceSensor
# echo=24, trigger=23
sensor = DistanceSensor(echo=24, trigger=23, max_distance=4)
print(f"{sensor.distance * 100:.1f} cm")
```

> ⚠️ Echo 輸出 5V，必須用分壓電阻 (1kΩ + 2kΩ) 降至 3.3V 才能接到 Pi GPIO！

## 慣性測量單元 — MPU6050

| 參數 | 規格 |
|------|------|
| 陀螺儀 | ±250/500/1000/2000 °/s |
| 加速度計 | ±2/4/8/16g |
| 通訊 | I²C (0x68) |
| 工作電壓 | 3.3V–5V |

```python
from mpu6050 import mpu6050
sensor = mpu6050(0x68)  # I²C address
accel = sensor.get_accel_data()  # {'x':..., 'y':..., 'z':...}
gyro = sensor.get_gyro_data()
```

> MPU6050 原生 3.3V I²C，可直連 Pi GPIO。適合姿態檢測、平衡機器人。

## 繼電器模組 Relay Module

| 類型 | 控制 | 負載 | 用途 |
|------|------|------|------|
| **5V 單路繼電器** | GPIO LOW 觸發 | AC 250V/10A | 家電控制 |
| **固態繼電器 SSR** | 3-32V DC | AC 24-380V | 靜音、頻繁切換 |

```python
from gpiozero import OutputDevice
relay = OutputDevice(17, active_high=False)  # LOW 觸發常見
relay.on()   # 繼電器吸合
relay.off()  # 釋放
```

> 繼電器模組通常自帶光耦隔離與反向二極體保護。驅動電流約 5-10mA，Pi GPIO 可直接驅動。

## 伺服馬達 Servo Motor

| 型號 | 扭矩 | 角度 | 電壓 | 用途 |
|------|:----:|:----:|:----:|------|
| **SG90** | 1.2 kg·cm | 0–180° | 4.8–6V | 小型機構 |
| **MG996R** | 10 kg·cm | 0–180° | 4.8–7.2V | 機器人關節 |

```python
from gpiozero import AngularServo
servo = AngularServo(18, min_pulse_width=0.5/1000, max_pulse_width=2.5/1000)
servo.angle = 0    # 0 度
servo.angle = 90   # 90 度
servo.angle = 180  # 180 度
```

> ⚠️ SG90 直接從 Pi 5V 腳取電可能造成壓降。建議外接 5V/2A 獨立電源，共地即可。

## LED 與 NeoPixel/WS2812

### 普通 LED

```python
from gpiozero import LED
led = LED(17)
led.on()
led.off()
# 需串聯 220Ω–470Ω 限流電阻
```

### WS2812B (NeoPixel) 可定址 RGB LED

| 參數 | 規格 |
|------|------|
| 電壓 | 5V (訊號 3.3V 可接受) |
| 協定 | 單線歸零碼 |
| 串接 | 理論無限 (每 LED ~60mA 全亮) |

```python
import board
import neopixel
pixels = neopixel.NeoPixel(board.D18, 8, brightness=0.5)  # 8 顆, GPIO 18
pixels[0] = (255, 0, 0)   # 紅
pixels[1] = (0, 255, 0)   # 綠
pixels.fill((0, 0, 255))  # 全藍
pixels.show()
```

> WS2812B 需要 5V 供電 (獨立電源)，DAT 腳接 GPIO 18 (PWM0)，3.3V 邏輯通常可接受。若不可靠，加 74AHCT125 做電平轉換。
