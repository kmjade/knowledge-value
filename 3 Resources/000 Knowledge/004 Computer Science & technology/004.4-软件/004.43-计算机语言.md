# 計算機語言 — UDC 004.43

> UDC `004.43` 是「計算機語言 Computer languages」的精確類號，隸屬 `004.4 軟件`。
> 與 DDC 不同的是，UDC 為**每一種語言提供獨立專號**，並透過輔助符號組合表達語言的應用場景。

---

## UDC 004.43 在分類體系中的位置

| 體系 | 類號 | 名稱 | 隸屬 |
|:---:|:----:|------|------|
| **UDC** | `004.43` | 計算機語言 | 004.4 軟件 → 004 計算機科學 → 0 總論 |
| **DDC** | `005.13` | 程式語言 | 005 程序設計 → 000 總類 |
| **CLC** | `TP312` | 程序語言、算法語言 | TP31 計算機軟件 → TP3 計算技術 |
| **LCC** | `QA76.7` | Programming languages | QA76 計算機科學 → QA 數學 |

> **UDC vs DDC 的關鍵差異**：UDC 將語言視為軟件的組成部分（004.43 ∈ 004.4），DDC 則將程序設計獨立為 005 大類。UDC 的結構更「物件導向」——語言是軟件的一種。

## 語言與應用場景的組合（UDC 輔助符號）

UDC 的核心優勢：通過輔助符號將語言與應用場景組合。

| 場景 | UDC 表達式 | 含義 |
|------|-----------|------|
| Python 用於 AI | `004.432.42:004.8` | Python + AI |
| JavaScript 用於 Web | `004.432.45:004.738.5` | JS + Web |
| C++ 用於遊戲引擎 | `004.432.2C++:004.92` | C++ + 計算機圖形學 |
| Rust 用於系統編程 | `004.432.6Rust:004.45` | Rust + OS |
| SQL 用於數據倉庫 | `004.439SQL:004.65` | SQL + 數據庫 |
| Python 用於數據科學 | `004.432.42:004.62` | Python + Data Science |
| Go 用於雲端後端 | `004.432.6Go:004.77` | Go + Cloud |
| Markdown 用於知識管理 | `004.438Markdown:001.1` | MD + KM |
| R 用於統計 | `004.432R:519.2` | R + 統計學 |

> `:` 是 UDC 的**關係符號**，表示兩個主題之間的關聯——不限於層級，可雙向導航。

## 語言分類法（按範式）

UDC 不強制按範式分類（那是 004.434–004.436 的職責），但以下對照有助於從範式視角導航：

| 範式 | UDC 範式碼 | 代表語言（UDC 專號） |
|------|:---:|------|
| **命令式** | — | C (`004.432.2C`), Go (`004.432.6Go`) |
| **物件導向** | 004.436 | Java (`004.432.4Java`), C++ (`004.432.2C++`) |
| **函數式** | 004.434 | Haskell (`004.434Haskell`), Clojure (`004.434Clojure`) |
| **邏輯式** | 004.435 | Prolog (`004.435Prolog`) |
| **多範式** | — | Python (`004.432.42`), Rust (`004.432.6Rust`) |
| **標記式** | 004.438 | HTML (`004.438HTML`), Markdown (`004.438Markdown`) |
| **腳本式** | 004.439 | Bash (`004.439Bash`), SQL (`004.439SQL`) |
