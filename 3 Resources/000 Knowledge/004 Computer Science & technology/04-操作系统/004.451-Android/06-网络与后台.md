# 06 網路與後台 Networking & Background

> Retrofit + OkHttp 為標準 HTTP 堆疊，Kotlin Coroutines + Flow 處理非同步，WorkManager/JobScheduler 管理後台任務，FCM 推送。
> Retrofit + OkHttp is the standard HTTP stack, Kotlin Coroutines + Flow handle async, WorkManager/JobScheduler manage background tasks, FCM for push.

## HTTP 網路堆疊 Stack

| 層級 Layer | 庫 Library | 職責 |
|------|------|------|
| **API 定義** | Retrofit | 宣告式 HTTP 介面 (Annotation-based) |
| **HTTP 引擎** | OkHttp | 連線池、攔截器、快取、HTTP/2 |
| **JSON 序列化** | Moshi / kotlinx.serialization / Gson | Request/Response body 序列化 |
| **圖片載入** | Coil / Glide | 非同步圖片載入與快取 |

```kotlin
// Retrofit 服務定義
interface ApiService {
    @GET("users/{id}")
    suspend fun getUser(@Path("id") userId: Int): Response<User>

    @POST("users")
    suspend fun createUser(@Body user: User): Response<User>
}

// OkHttp 攔截器
val okHttp = OkHttpClient.Builder()
    .addInterceptor { chain ->
        val request = chain.request().newBuilder()
            .addHeader("Authorization", "Bearer $token")
            .build()
        chain.proceed(request)
    }
    .connectTimeout(30, TimeUnit.SECONDS)
    .build()
```

## Kotlin Coroutines 層級

| 調度器 Dispatcher | 用途 | 執行緒池 |
|------|------|------|
| **Dispatchers.Main** | UI 更新 | 主執行緒 |
| **Dispatchers.IO** | 網路、檔案 I/O | 彈性執行緒池 (max 64) |
| **Dispatchers.Default** | CPU 密集工作 | CPU 核心數 |
| **Dispatchers.Unconfined** | 測試用，不限定 | 當前呼叫執行緒 |

```kotlin
// ViewModelScope — 典型模式
class UserViewModel(private val api: ApiService) : ViewModel() {
    private val _users = MutableStateFlow<List<User>>(emptyList())
    val users: StateFlow<List<User>> = _users.asStateFlow()

    fun loadUsers() {
        viewModelScope.launch(Dispatchers.IO) {
            val result = api.getUsers()
            _users.value = result
        }
    }
}
```

## 後台限制 Background Limits

| Android 版本 | 後台限制 |
|:---:|------|
| **6.0+** | Doze — 螢幕關閉後限制網路與 CPU |
| **8.0+** | 後台 Service 限制 — `startService()` 在後台拋出 IllegalStateException |
| **9+** | 後台 Sensor 限制 |
| **10+** | Scoped Storage — 限制檔案系統存取 |
| **11+** | 一次性權限、權限自動重置 |
| **13+** | 通知權限 (`POST_NOTIFICATIONS`) 需用戶授權 |
| **14+** | 更嚴格的 foreground service 類型聲明 |

## FCM (Firebase Cloud Messaging)

| 類型 | 說明 | 適合 |
|------|------|------|
| **Notification Message** | FCM SDK 自動顯示通知 | Marketing, 簡單 alert |
| **Data Message** | App 自行處理 payload | 自訂 UI, silent data sync |
| **Topic** | 訂閱/退訂主題群發 | 新聞分類、區域推送 |
| **Token** | 每個裝置唯一註冊 Token | 單一裝置定向推送 |

```kotlin
// FCM 繼承
class MyFirebaseMessagingService : FirebaseMessagingService() {
    override fun onNewToken(token: String) { /* 發送 token 到後端 */ }
    override fun onMessageReceived(message: RemoteMessage) { /* 處理 data payload */ }
}
```
