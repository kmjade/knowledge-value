# 004.451 Android

> Android 是基於 Linux 核心的開源行動作業系統，由 Google 主導開發，2008 年首發，現為全球市佔率最高的行動平台，涵蓋手機、平板、穿戴裝置、車載與 IoT。
> Android is an open-source mobile OS based on the Linux kernel, led by Google, first released 2008, now the world's dominant mobile platform spanning phones, tablets, wearables, automotive, and IoT.

## 架構總覽 Architecture Overview

| 層 Layer | 組件 Component | 說明 Description |
|:---|------|------|
| **Apps** | System + Third-party | Launcher, Dialer, Settings, 用戶應用 |
| **Java API Framework** | Activity Manager, Window Manager, Content Providers | 開發者面向的 Android API |
| **Android Runtime (ART)** | dex2oat, GC, JIT/AOT compilation | 執行 dex bytecode 的 runtime |
| **Native C/C++ Libraries** | libc, SQLite, OpenGL ES, WebKit, libv8 | HAL 層面上的原生庫 |
| **HAL** | Hardware Abstraction Layer | 標準化硬體介面 (Treble) |
| **Linux Kernel** | Binder, ashmem, wakelocks, SELinux | 修改過的 Linux 核心 |
