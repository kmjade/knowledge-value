# 04 介面與互動 UI & Interaction

> 從傳統 View 系統到 Jetpack Compose 宣告式 UI，Material Design 3 提供現代化設計語言，Navigation 管理多螢幕導航。
> From traditional View system to Jetpack Compose declarative UI, Material Design 3 provides a modern design language, with Navigation managing multi-screen routing.

## View 系統 vs Compose

| 概念 | View (Imperative) | Compose (Declarative) |
|------|------|------|
| **佈局容器** | `LinearLayout`, `ConstraintLayout`, `FrameLayout` | `Column`, `Row`, `Box`, `ConstraintLayout` |
| **列表** | `RecyclerView` + `Adapter` + `ViewHolder` | `LazyColumn` / `LazyRow` |
| **點擊** | `setOnClickListener {}` | `Modifier.clickable {}` |
| **動畫** | `ObjectAnimator`, `AnimatorSet` | `animate*AsState`, `AnimatedVisibility` |
| **主題** | `styles.xml` / `themes.xml` | `MaterialTheme` composable |

## Material Design 3 關鍵特性

| 特性 | API | 說明 |
|------|------|------|
| **Dynamic Color** | `dynamicColor` (MaterialTheme) | 從桌布提取色調 (Android 12+) |
| **Typography** | `Typography` + `TextStyle` | 設計 Token 定義文字層級 |
| **Shape** | `Shapes(small, medium, large)` | 圓角統一管理 |
| **Top App Bar** | `TopAppBar` / `CenterAlignedTopAppBar` | M3 風格頂欄 |
| **Navigation Bar** | `NavigationBar` + `NavigationBarItem` | 底部導航 |

```kotlin
// Material 3 Dynamic Color Theme
MaterialTheme(
    colorScheme = if (Build.VERSION.SDK_INT >= 31) {
        dynamicDarkColorScheme(LocalContext.current)
    } else {
        darkColorScheme()
    }
) { /* content */ }
```

## Navigation 導航

| 功能 | View 系統 | Compose |
|------|------|------|
| **框架** | Navigation Component (XML nav graph) | NavHost + composable destinations |
| **引數傳遞** | Safe Args plugin | Route string + NavType |
| **深層連結** | `<deepLink>` in nav graph | `deepLink` parameter |
| **返回棧** | `popUpTo` + `inclusive` | 同上，完全一致 |

```kotlin
NavHost(navController, startDestination = "home") {
    composable("home") { HomeScreen(navController) }
    composable("detail/{id}", arguments = listOf(
        navArgument("id") { type = NavType.IntType }
    )) { backStackEntry ->
        DetailScreen(backStackEntry.arguments?.getInt("id") ?: 0)
    }
}
```

## 螢幕適配 Screen Adaptation

| 策略 | 說明 |
|------|------|
| **WindowSizeClass** | `Compact` / `Medium` / `Expanded` — 官方適配方案 |
| **Configuration Qualifiers** | `layout-sw600dp`, `values-night`, `drawable-xhdpi` |
| **ConstraintLayout** | 相對於父層/兄弟 View 定位，避免嵌套 |
| **Foldable / Large Screen** | `Jetpack WindowManager` — 折疊螢幕與平板適配 |
| **Dark Theme** | `values-night/` + `isSystemInDarkTheme()` |
