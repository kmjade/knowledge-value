# 03 — 命令列對比 CLI Comparison

> 跨六大作業系統的等效命令速查表：Bash/zsh ↔ PowerShell ↔ adb ↔ hdc。
> Cross-OS equivalent commands quick reference: Bash/zsh ↔ PowerShell ↔ adb ↔ hdc for daily operations.

---

## 命令環境概覽 Shell Environment Overview

| OS | 預設 Shell | 替代 Shell | CLI 入口 | 腳本語言 |
|------|:---:|------|------|------|
| **Windows Server** | PowerShell 7 | cmd, WSL bash | Terminal / WinRM | PowerShell, Batch |
| **Linux** | Bash | zsh, fish, ash (Alpine) | SSH / tty | Bash, Python |
| **Unix (Solaris)** | ksh | bash, sh | SSH / console | ksh, Perl |
| **macOS** | zsh | bash, fish | Terminal.app / SSH | zsh, AppleScript |
| **Android** | sh (mksh) | — | adb shell / Termux | Shell script |
| **HarmonyOS** | — | — | hdc shell | — |

## 檔案系統操作 File System Operations

| 操作 | Linux/macOS/Unix (Bash/zsh) | Windows (PowerShell) | Android (adb shell) | HarmonyOS (hdc shell) |
|------|------|------|------|------|
| **列出目錄** | `ls -la` | `Get-ChildItem` / `ls` | `ls -la` | `ls -la` |
| **切換目錄** | `cd /path` | `cd C:\path` | `cd /path` | `cd /path` |
| **當前目錄** | `pwd` | `Get-Location` / `pwd` | `pwd` | `pwd` |
| **建立目錄** | `mkdir -p dir` | `New-Item -ItemType Directory dir` | `mkdir dir` | `mkdir dir` |
| **複製檔案** | `cp -r src dst` | `Copy-Item src dst -Recurse` | `cp src dst` | `cp src dst` |
| **移動檔案** | `mv src dst` | `Move-Item src dst` | `mv src dst` | `mv src dst` |
| **刪除檔案** | `rm -rf path` | `Remove-Item path -Recurse -Force` | `rm -rf path` | `rm -rf path` |
| **建立符號連結** | `ln -s target link` | `New-Item -ItemType SymbolicLink` | `ln -s target link` | `ln -s target link` |
| **檔案權限** | `chmod 755 file` | `icacls file /grant User:RX` | `chmod 755 file` | `chmod 755 file` |
| **變更擁有者** | `chown user:group file` | `takeown / icacls` | `chown user:group file` | `chown user:group file` |
| **搜尋檔案** | `find / -name "*.log"` | `Get-ChildItem -Recurse -Filter *.log` | `find / -name "*.log"` | `find / -name "*.log"` |
| **檔案內容搜尋** | `grep -r "pattern" /path` | `Select-String -Path "*.log" -Pattern "pattern"` | `grep -r "pattern" /path` | `grep -r "pattern" /path` |
| **磁碟用量** | `df -h ; du -sh *` | `Get-PSDrive ; Get-ChildItem \| Measure-Object` | `df -h ; du -sh *` | `df -h` |
| **檔案差異** | `diff file1 file2` | `Compare-Object (gc a) (gc b)` | `diff file1 file2` | `diff file1 file2` |

## 跨平台腳本等價物 Cross-Platform Script Equivalents

| 操作 | Bash | PowerShell | 說明 |
|------|------|------|------|
| **變數賦值** | `NAME="value"` | `$NAME = "value"` | 字串變數 |
| **條件判斷** | `if [ -f "$f" ]; then ... fi` | `if (Test-Path $f) { ... }` | 檔案存在檢查 |
| **迴圈** | `for i in $(seq 1 10); do ... done` | `for ($i=1; $i -le 10; $i++) { ... }` | 計數迴圈 |
| **管線** | `cat file \| grep pat \| wc -l` | `gc file \| Select-String pat \| Measure-Object` | 過濾統計 |
| **文字處理** | `awk '{print $1}' file` | `(gc file).Split()[0]` | 取第一欄 |
| **環境變數** | `export PATH=/new:$PATH` | `$env:Path = "/new;$env:Path"` | 設定 PATH |
| **命令替換** | `$(command)` | `$(command)` | 命令輸出代入變數 |

## ADB vs HDC 命令對比 (Android vs HarmonyOS)

| 操作 | adb (Android) | hdc (HarmonyOS) | 說明 |
|------|------|------|------|
| **列出設備** | `adb devices` | `hdc list targets` | 連接的裝置列表 |
| **Shell** | `adb shell` | `hdc shell` | 進入裝置 shell |
| **安裝應用** | `adb install app.apk` | `hdc install app.hap` | 安裝應用套件 |
| **檔案推送** | `adb push local /remote` | `hdc file send local /remote` | 推送檔案到裝置 |
| **檔案拉取** | `adb pull /remote local` | `hdc file recv /remote local` | 從裝置拉取檔案 |
| **截圖** | `adb exec-out screencap -p > file.png` | `hdc shell snapshot_display -f /tmp/s.png` | 截取螢幕 |
| **日誌** | `adb logcat` | `hdc hilog` | 查看系統日誌 |
| **重啟** | `adb reboot` | `hdc shell reboot` | 重啟裝置 |
| **卸載** | `adb uninstall pkg` | `hdc uninstall pkg` | 卸載應用 |
