# 04 — 開發環境對比 Development Environment Comparison

| **Linux** | — | VS Code, JetBrains (all), Eclipse | VS Code / Vim / Neovim | GCC + Make / CMake |
| **Unix** | — | — | Vim / Emacs | Oracle Developer Studio (Solaris) |

## 開發環境設置難度 Setup Difficulty

| OS | 設置複雜度 | 說明 |
|------|:---:|------|
| **Windows Server** | 🟡 中等 | Visual Studio 一鍵安裝，但 CLI 工具鏈需 winget/choco 補全 |
| **Linux** | 🟢 簡單 | apt install build-essential 即完成基礎設置 |
| **macOS** | 🟢 簡單 | xcode-select --install + Homebrew 即可 |
| **Android** | 🟡 中等 | Android Studio + SDK 下載大，AVD 需硬體加速 (HAXM/KVM) |
| **HarmonyOS** | 🟡 中等 | DevEco Studio + hdc + 開發者帳號，平台限制 Windows/Mac/Linux |
