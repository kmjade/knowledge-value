## SMB 跨平台互通詳解 SMB Cross-Platform Interoperability

| 場景 | Windows → Linux | Linux → Windows | macOS → Windows | macOS → Linux |
|------|:---:|:---:|:---:|:---:|
| **伺服器端** | Samba on Linux | Windows SMB share | SMB via macOS Sharing | Samba/nfsd on Linux |
| **用戶端掛載** | `net use Z: \\host\share` | `mount -t cifs` or `smbclient` | Finder → Connect to Server | Finder → Connect to Server |
| **驗證** | NTLM / Kerberos | AD join / local user | Kerberos (AD bind) | Kerberos / local |
| **SMB 版本** | SMB 3.1.1 (default) | Samba 4.x (SMB 3) | SMB 3.x (native) | SMB 3.x (native) |
| **加密** | SMB Encryption | SMB Encryption (Samba 4.11+) | SMB Signing | SMB Signing |
| **多通道** | ✅ (SMB Multichannel) | 🟡 (Samba 4.x experimental) | ❌ | ❌ |

## WSL (Windows Subsystem for Linux) 互操作詳解

| 功能 | WSL 2 | 說明 |
|------|:---:|------|
| **Linux 內核** | ✅ 完整 Linux kernel (Microsoft-built) | 非模擬，真實內核在輕量 VM 中 |
| **檔案系統互通** | ✅ `/mnt/c/` ↔ `\\wsl$\` | 雙向存取，9P 協議 |
| **網路互通** | ✅ `localhost` 共享 | Windows ↔ WSL 雙向 localhost |
| **GPU 加速** | ✅ CUDA / DirectML / OpenCL | WSLg 整合 |
| **GUI 應用** | ✅ WSLg (Wayland/Weston) | Linux GUI 原生顯示在 Windows |
| **systemd** | ✅ (WSL 0.67.6+) | 完整 systemd 支援 |
| **USB 直通** | 🟡 (usbipd-win) | 需要額外工具 |
| **Docker** | ✅ Docker Desktop WSL 2 backend | 容器直接在 WSL 中執行 |
| **效能** | ~92-95% of native Linux | 輕微 overhead |

### WSL 常見互操作場景

```bash
# 從 WSL 訪問 Windows 檔案
cd /mnt/c/Users/username/Documents

# 從 Windows 訪問 WSL 檔案（檔案總管）
explorer.exe .

# 從 WSL 呼叫 Windows 命令
powershell.exe Get-ComputerInfo
cmd.exe /c dir

# 從 Windows PowerShell 呼叫 WSL
wsl ls -la
wsl -d Ubuntu-22.04 python3 script.py
```

## 虛擬化巢狀與互操作 Virtualization Nesting

| 場景 | 支援 | 備註 |
|------|:---:|------|
| **Hyper-V 內執行 WSL 2** | ✅ | WSL 2 本質上使用 Hyper-V 輕量 VM |
| **Hyper-V 內執行 KVM** | ✅ (nested virtualization) | 需啟用 ExposeVirtualizationExtensions |
| **KVM 內執行 Hyper-V** | ❌ | 不支援 |
| **VMware 內執行 WSL 2** | ✅ | VMware 16+ with Hyper-V mode |
| **macOS (Apple Silicon) 內執行 Linux VM** | ✅ | UTM / Parallels (ARM64 guests only) |
| **macOS (Apple Silicon) 內執行 x86 Windows VM** | ❌ | 無 x86 模擬支援 |
| **Linux KVM 內執行 Windows VM** | ✅ | virtio drivers |
| **Docker Desktop 跨平台** | ✅ | Windows/Mac 使用 VM backend, Linux 原生 |
