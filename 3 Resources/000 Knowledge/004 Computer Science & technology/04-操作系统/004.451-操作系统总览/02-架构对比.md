# 02 — 作業系統架構對比 Architecture Comparison

> 六大作業系統的內核類型、行程排程、記憶體管理、檔案系統與安全模型的深度對比。
> Deep comparison of kernel types, process scheduling, memory management, filesystems, and security models across six operating systems.

---

## 內核類型對比 Kernel Type Comparison

| 維度 | Windows Server | Linux | Unix | macOS | Android | HarmonyOS |
|------|:---:|:---:|:---:|:---:|:---:|:---:|
| **內核類型** | Hybrid (NT) | Monolithic | Monolithic | Hybrid (XNU) | Monolithic (modified Linux) | Microkernel (自研) |
| **內核模式** | Kernel + Executive | Kernel space only | Kernel + modules | Mach (micro) + BSD (mono) | Linux kernel + Android patches | Deterministic latency microkernel |
| **模組化** | DLL + drivers | LKM (Loadable Kernel Modules) | Static/Varies | KEXT → System Extensions | LKM (limited) | 輕量級服務組件 |
| **驅動模型** | WDF (KMDF/UMDF) | Device Tree + drivers | Varies by vendor | IOKit (C++) | Linux driver model | HDF (Hardware Driver Foundation) |
| **IPC 基礎** | ALPC (Advanced LPC) | Unix domain sockets + D-Bus | SysV IPC + sockets | Mach ports + XPC | Binder | 分散式軟總線 (DSoftBus) |
| **中斷處理** | IRQL levels | Top/Bottom halves + tasklets | Interrupt levels | Mach IRQ model | Linux IRQ model | 確定性延遲中斷模型 |

## 行程排程器對比 Process Scheduler Comparison

| 維度 | Windows Server | Linux | Unix (Solaris) | macOS | Android | HarmonyOS |
|------|:---:|:---:|:---:|:---:|:---:|:---:|
| **排程類別** | Priority preemptive | CFS (→ EEVDF 6.6+) | Multi-level feedback queue (TS/IA/FSS) | Mach scheduler + BSD | CFS (modified) | 確定性延遲排程器 |
| **優先級範圍** | 0-31 (16+ real-time) | nice -20..+19 + RT(1-99) | 0-169 (60+ RT) | 0-127 (Mach) | nice + cgroups RT | 0-31 (確定性) |
| **時間片** | Variable (quantum) | Dynamic (vruntime) | Variable by class | Mach quantum (10ms default) | CFS dynamic | Fixed + adaptive |
| **NUMA 感知** | ✅ (since Win 7) | ✅ (sched domain) | ✅ (lgroup) | ✅ | ✅ | ✅ (分散式 NUMA) |
| **CPU 親和性** | Processor affinity mask | taskset/cpuset | pbind | thread_policy_set | taskset/cpuset | 裝置親和性 (device affinity) |
| **節能排程** | Core parking | EAS (Energy-Aware) | Power Aware Dispatcher | QoS 等級 | EAS (since 5.x) | 裝置級能效排程 |
| **即時支援** | ✅ (MMCSS/WinRT) | ✅ (PREEMPT_RT) | ✅ (RT class) | ✅ (real-time threads) | ✅ (RT patches) | ✅ (確定性 RT) |

## 記憶體管理對比 Memory Management Comparison

| 維度 | Windows Server | Linux | Unix | macOS | Android | HarmonyOS |
|------|:---:|:---:|:---:|:---:|:---:|:---:|
| **虛擬記憶體** | Demand paging + VAS | Demand paging + MMU | Demand paging | Mach VM + UBC | Demand paging (MMU) | 分散式虛擬記憶體 |
| **頁面大小** | 4KB / 2MB (large) | 4KB / 2MB / 1GB | 4KB-4MB (varies) | 4KB / 2MB (ARM) / 16KB (M1+) | 4KB / 2MB / 16KB | 4KB / 2MB / 16KB |
| **頁面替換** | Multi-page working set | LRU (active/inactive) + MGLRU | Two-handed clock | Mach LRU + UBC | LRU + MGLRU (Android 14+) | 自適應 LRU |
| **OOM 處理** | System OOM policies | OOM Killer → systemd-oomd | Varies per vendor | Jetsam (memorystatus) | LMK (Low Memory Killer) → LMKD | 確定性 OOM 管理 |
| **記憶體壓縮** | Memory Compression (Win10+) | zram/zswap | Varies | VM compressor (since Mavericks) | zram (default since 4.x) | 智慧壓縮引擎 |
| **Huge Pages** | Large Pages | THP (Transparent) | MPSS (Solaris) | Superpages | THP | 確定性大頁面 |
| **最大定址** | 256 TB (Win Server) | 256 TB (x86_64) | Varies | 128 TB (Apple Silicon) | 4GB (32-bit) / 256 TB (64-bit) | 取決於裝置類型 |

## 檔案系統對比 Filesystem Comparison

| 維度 | Windows Server | Linux | Unix | macOS | Android | HarmonyOS |
|------|:---:|:---:|:---:|:---:|:---:|:---:|
| **預設 FS** | NTFS | ext4 / XFS (RHEL) | UFS / ZFS (Solaris) | APFS | ext4 / f2fs | HMFS (自研) |
| **最大檔案** | 16 EB | 16 TB (ext4) / 8 EB (XFS) | 16 EB (ZFS) | 8 EB (APFS) | 4 GB (FAT32) / 16 TB (ext4) | 取決於裝置類型 |
| **日誌/快照** | Journal (NTFS) + VSS | Journal + LVM/Btrfs snapshots | ZFS snapshots | APFS snapshots + clones | ext4 journal | 版本式快照 |
| **寫時複製 CoW** | ❌ (ReFS: ✅) | ✅ (Btrfs, ZFS) | ✅ (ZFS) | ✅ (APFS) | ❌ (ext4) / ✅ (f2fs GC) | ✅ |
| **壓縮** | NTFS compression | ext4 (inline) / Btrfs/ZFS | ZFS compression | APFS compression | ❌ (FBE only) | 智慧分層壓縮 |
| **加密** | BitLocker (NTFS) | LUKS / fscrypt (ext4/f2fs) | ZFS encryption | FileVault (APFS) | FBE (File-Based Encryption) | 分級加密 |
| **跨平台支援** | ❌ (需第三方) | ✅ (讀取 NTFS via ntfs-3g) | Varies | ❌ (APFS 封閉) | ❌ (MTP/ext4) | 跨裝置透明存取 |

## 安全框架對比 Security Framework Comparison

| 維度 | Windows Server | Linux | Unix | macOS | Android | HarmonyOS |
|------|:---:|:---:|:---:|:---:|:---:|:---:|
| **MAC 框架** | Mandatory Integrity Control (MIC) | SELinux / AppArmor | Trusted Extensions (Solaris) | SIP (System Integrity Protection) | SELinux (Android) | 分級安全 (Graded Security) |
| **權限模型** | ACL + Privileges | DAC + capabilities | POSIX ACL | Sandbox entitlements + TCC | Runtime permissions + SELinux | 分級權限 (Graded Permissions) |
| **安全啟動** | Secure Boot + TPM 2.0 | UEFI Secure Boot | Varies | Secure Boot (T2/M-series) | Verified Boot (AVB) | Verified Boot |
| **應用簽名** | Authenticode / EV cert | GPG / package signing | Vendor-specific | Apple code signing | APK Signature v4 | 應用簽名 + 證書管理 |
| **沙箱** | AppContainer / LowBox | seccomp + namespaces | Zones / chroot | macOS sandbox (seatbelt) | Android app sandbox (UID) | App sandbox + 分散式隔離 |
| **入侵檢測** | Defender for Endpoint | AIDE / OSSEC / auditd | Varies | XProtect / MRT | Play Protect | 安全檢測引擎 |

## 啟動流程對比 Boot Process Comparison

| 階段 | Windows Server | Linux | macOS | Android | HarmonyOS |
|------|------|------|------|------|------|
| **韌體** | UEFI / BIOS | UEFI / BIOS / U-Boot | iBoot (Apple) | ABL (UEFI/U-Boot) | UEFI / U-Boot |
| **Boot Loader** | Windows Boot Manager | GRUB2 / systemd-boot | boot.efi | Little Kernel (LK) | 鴻蒙 Bootloader |
| **內核載入** | ntoskrnl.exe | vmlinuz + initramfs | XNU (Mach + BSD) | Linux kernel + initramfs | 鴻蒙微內核 |
| **初始化** | smss.exe → wininit | PID 1: systemd / init | launchd (PID 1) | init → Zygote | init 服務管理器 |
| **用戶空間** | csrss → winlogon → shell | systemd targets | launchd agents | Zygote → system_server | Ability Manager |
