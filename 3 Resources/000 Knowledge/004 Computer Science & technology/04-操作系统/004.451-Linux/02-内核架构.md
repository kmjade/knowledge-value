# 02 內核架構 Kernel Architecture

> Linux kernel 是 monolithic 架構，但支援動態載入模組。核心子系統：行程管理、記憶體管理、檔案系統 (VFS)、網路堆疊、裝置驅動。

## 架構總覽 Architecture Overview

```
Userspace (Ring 3)
  ├── Applications (bash, nginx, python)
  ├── GNU C Library (glibc) ── syscall interface
─── syscall boundary ──────────────────────────
Kernelspace (Ring 0)
  ├── Process Scheduler (CFS/EEVDF)
  ├── Memory Manager (virtual memory, page allocator)
  ├── VFS (Virtual File System)
  ├── Network Stack (TCP/IP, netfilter)
  └── Device Drivers (char, block, net)
```

## 行程調度器 Process Scheduler

| 調度器 | 版本 | 特點 |
|------|:---:|------|
| **O(n)** | 2.4 | 早期，簡單但 O(n) 複雜度 |
| **O(1)** | 2.6 | 常數時間調度，引入 runqueue |
| **CFS** | 2.6.23 | 完全公平調度器，紅黑樹，vruntime |
| **EEVDF** | 6.6 | Earliest Eligible Virtual Deadline First (取代 CFS)，更佳延遲保證 |

```bash
# 查看當前調度器與行程狀態
cat /proc/sched_debug | head -30
ps -eo pid,comm,rtprio,policy --sort=-rtprio | head
chrt -p <PID>              # 查看行程調度策略
```

## 虛擬記憶體 Virtual Memory

| 概念 | 說明 |
|------|------|
| **Page (頁)** | 固定大小記憶體單元，通常 4KB (x86_64) 或支援 huge page (2MB/1GB) |
| **Page Table** | 虛擬位址 → 實體位址映射，多級頁表 (PGD→PUD→PMD→PTE) |
| **TLB** | Translation Lookaside Buffer，頁表快取 |
| **Swap** | 將不活躍頁面交換到磁碟 |
| **OOM Killer** | 記憶體耗盡時選擇並殺掉程序 |

```bash
# 記憶體管理檢視
free -h                     # 記憶體使用概況
cat /proc/meminfo            # 詳細記憶體統計
vmstat 1 5                   # 虛擬記憶體統計
cat /proc/sys/vm/swappiness  # swap 傾向 (0-100)
sysctl vm.swappiness=10      # 降低 swap 傾向
```

## 核心模組 Kernel Modules

```bash
lsmod                        # 列出已載入模組
modinfo <module>             # 模組詳細資訊
modprobe <module>            # 載入模組 (含相依)
modprobe -r <module>         # 移除模組
ls /lib/modules/$(uname -r)/ # 可用模組目錄
```

## /proc 與 /sys 虛擬檔案系統

| 路徑 | 用途 |
|------|------|
| `/proc/cpuinfo` | CPU 詳細資訊 |
| `/proc/meminfo` | 記憶體統計 |
| `/proc/<PID>/` | 行程資訊 (fd, maps, status) |
| `/proc/sys/` | 核心參數 (sysctl 介面) |
| `/sys/class/` | 裝置類別 (net, block) |
| `/sys/kernel/` | 核心調試與參數 |
