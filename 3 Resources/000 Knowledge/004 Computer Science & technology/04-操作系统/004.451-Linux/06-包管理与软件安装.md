# 06 套件管理與軟體安裝 Package Management

> Linux 套件管理系統管理軟體的安裝、升級、相依性解析與移除。不同發行版使用不同的套件格式與工具鏈。

## 套件管理對比 Package Manager Comparison

| 家族 | 底層工具 | 高層工具 | 搜尋 | 安裝 | 移除 |
|------|------|------|------|------|------|
| **Debian** | dpkg | apt | `apt search` | `apt install` | `apt remove` |
| **Red Hat** | rpm | dnf | `dnf search` | `dnf install` | `dnf remove` |
| **Arch** | pacman | pacman/yay | `pacman -Ss` | `pacman -S` | `pacman -R` |
| **Alpine** | apk | apk | `apk search` | `apk add` | `apk del` |
| **openSUSE** | rpm | zypper | `zypper search` | `zypper install` | `zypper remove` |

## apt / dpkg (Debian/Ubuntu)

```bash
# 倉庫管理
apt update                          # 更新套件索引
apt upgrade                         # 升級全部套件
apt full-upgrade                    # 完整升級 (可移除衝突套件)

# 套件操作
apt install nginx                   # 安裝
apt remove nginx                    # 移除 (保留設定)
apt purge nginx                     # 完全移除 (含設定)
apt autoremove                      # 移除孤立相依

# 查詢
apt search keyword                  # 搜尋套件
apt show nginx                      # 套件詳細資訊
apt list --installed                # 列出已安裝

# dpkg 底層
dpkg -i package.deb                 # 手動安裝 .deb
dpkg -l | grep nginx               # 列出套件
dpkg -S /usr/bin/nginx             # 哪個套件擁有此檔案
```

## dnf / rpm (RHEL/Fedora)

```bash
dnf search nginx                    # 搜尋
dnf install nginx                   # 安裝
dnf remove nginx                    # 移除
dnf update                          # 更新全部
dnf history                         # 交易歷史
dnf history rollback <id>           # 復原到特定交易

rpm -ivh package.rpm                # 安裝
rpm -qa | grep nginx               # 查詢
rpm -ql nginx                       # 列出套件檔案
```

## pacman (Arch)

```bash
pacman -Syu                         # 同步並升級全部 (Arch 標配)
pacman -S nginx                     # 安裝
pacman -R nginx                     # 移除 (保留相依)
pacman -Rns nginx                   # 移除 + 相依 + 設定
pacman -Ss keyword                  # 搜尋
pacman -Q                           # 列出已安裝
pacman -Ql nginx                    # 列出套件檔案
```

## 通用套件格式 Universal Package Formats

| 格式 | 特點 | 隔離 | 自動更新 | 適用 |
|------|------|:---:|:---:|------|
| **Snap** | Canonical 維護，sandbox 隔離 | ✅ | ✅ | Ubuntu 預設 |
| **Flatpak** | Freedesktop 標準，桌面應用 | ✅ | ✅ | Fedora / 跨 distro |
| **AppImage** | 單一可執行檔，無需安裝 | ❌ | ❌ | 便攜/試用 |

```bash
# Snap
snap install vlc
snap list

# Flatpak
flatpak install flathub org.vlc.VLC
flatpak list

# AppImage
chmod +x app.AppImage
./app.AppImage
```

## 從原始碼編譯 Compile from Source

```bash
# 典型流程
git clone https://github.com/example/program.git
cd program
./configure --prefix=/usr/local     # 檢查環境 + 設定
make -j$(nproc)                     # 編譯
sudo make install                   # 安裝
make clean                          # 清理

# 安裝編譯工具鏈
apt install build-essential         # Debian
dnf groupinstall "Development Tools" # RHEL
```
