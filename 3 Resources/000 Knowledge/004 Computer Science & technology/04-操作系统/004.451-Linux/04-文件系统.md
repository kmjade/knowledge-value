# 04 檔案系統 File Systems

> Linux 透過 VFS (Virtual File System) 抽象層支援數十種檔案系統。核心概念：inode、掛載 (mount)、邏輯卷管理 (LVM)、權限模型。

## VFS 架構 VFS Architecture

```
User:  open() / read() / write() / close()
─── System Call ─────────────────────────
VFS:  統一抽象層 (inode, dentry, file, superblock)
─── Dispatch ────────────────────────────
Impl: ext4 │ XFS │ Btrfs │ ZFS │ NTFS │ tmpfs │ ...
─── Block I/O ───────────────────────────
Block: Block Layer → Device Driver → Disk
```

## 檔案系統對比 FS Comparison

| 檔案系統 | 最大卷 | 日誌 | CoW | 快照 | 場景 |
|------|:---:|:---:|:---:|:---:|------|
| **ext4** | 1 EiB | ✅ | ❌ | ❌ | 通用預設，穩定可靠 |
| **XFS** | 8 EiB | ✅ | ❌ | ❌ | 大檔案/高併發，RHEL 預設 |
| **Btrfs** | 16 EiB | ✅ | ✅ | ✅ | 快照/壓縮/RAID |
| **ZFS** | 256 ZiB | ✅ | ✅ | ✅ | 資料完整性、自癒 |
| **tmpfs** | RAM size | ❌ | ❌ | ❌ | 暫存 (`/tmp`, `/dev/shm`) |
| **F2FS** | 16 TiB | ✅ | ❌ | ❌ | Flash/NVMe/手機 |

## 掛載與 fstab Mount & fstab

```bash
# 掛載
mount /dev/sdb1 /mnt/data
mount -t ext4 -o noatime /dev/sdb1 /mnt/data

# 卸載
umount /mnt/data
umount -l /mnt/data        # lazy unmount

# fstab 條目格式
# <device>  <mount>  <type>  <options>  <dump>  <pass>
/dev/sdb1  /data    ext4    defaults,noatime  0  2
```

## LVM 邏輯卷管理 Logical Volume Manager

| 概念 | 說明 | 指令 |
|------|------|------|
| **PV** (Physical Volume) | 實體磁碟/分割區 | `pvcreate /dev/sdb` |
| **VG** (Volume Group) | PV 組成的儲存池 | `vgcreate vg0 /dev/sdb` |
| **LV** (Logical Volume) | 可動態調整的邏輯卷 | `lvcreate -L 10G -n data vg0` |

```bash
pvdisplay / vgdisplay / lvdisplay     # 查看狀態
lvextend -L +5G /dev/vg0/data         # 擴大 LV
resize2fs /dev/vg0/data               # 擴大檔案系統 (ext4)
```

## inode / 硬連結 / 符號連結

| 概念 | 說明 | 指令 |
|------|------|------|
| **inode** | 檔案元數據 (權限/大小/區塊指標)，不含檔名 | `stat file` / `ls -i` |
| **硬連結** | 多個檔名指向同一 inode (不可跨 FS) | `ln target link` |
| **符號連結** | 指向路徑的「捷徑」(可跨 FS) | `ln -s target link` |

```bash
ls -li                         # 顯示 inode 編號
df -i                          # 查看 inode 使用量
find / -inum <inode#>          # 找出指向特定 inode 的檔案
```

## 權限與 ACL Permissions & ACL

```bash
# 基本權限
chmod 755 script.sh             # rwxr-xr-x
chown user:group file.txt       # 更改擁有者/群組

# ACL (Access Control Lists)
getfacl file.txt                # 查看 ACL
setfacl -m u:bob:rw file.txt    # 授權 bob 讀寫
setfacl -x u:bob file.txt       # 移除 bob 的 ACL
```
