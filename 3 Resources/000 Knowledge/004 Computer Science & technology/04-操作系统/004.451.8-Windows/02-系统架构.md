# 02 — 系統架構 System Architecture

> Windows NT 核心架構：從 HAL 到使用者模式的完整堆疊，包括子系統、驅動模型與虛擬化層。
> Deep dive into NT kernel, HAL, user/kernel mode, subsystems (Win32, WinRT, WSL/WSLg), and driver model.

---

## 核心層次 Core Layers

| 層次 | 元件 | 檔案 | 說明 |
|:----:|:-----|:-----|:-----|
| User | Win32 Subsystem | csrss.exe, kernel32.dll, user32.dll | 傳統 Windows API |
| User | WinRT/UWP | — | Modern/Metro 應用執行時期 |
| User | WSL / LxSS | lxss.sys, wsl.exe | Linux ELF 二進位相容 |
| User | NTDLL | ntdll.dll | User-mode syscall stub |
| Kernel | Executive | ntoskrnl.exe | 物件管理、記憶體、I/O、安全 |
| Kernel | Kernel | ntoskrnl.exe | 排程、中斷、同步 |
| Kernel | HAL | hal.dll | 硬體抽象層 |
| Kernel | Drivers | *.sys | WDM / KMDF / UMDF 驅動 |

## 子系統詳解 Subsystems

### Win32 子系統

| 元件 | 用途 |
|:-----|:-----|
| csrss.exe | Client/Server Runtime — 主控臺、程序管理 |
| kernel32.dll | 基礎 API（檔案、記憶體、執行緒） |
| user32.dll | 視窗、訊息、UI 元件 |
| gdi32.dll | 圖形裝置介面 (legacy) |

### WinRT (Windows Runtime)

| 特性 | 說明 |
|:-----|:-----|
| 語言投影 | C++/CX, C#, JavaScript/Broker |
| 沙箱模型 | AppContainer — 限制檔案系統/網路 |
| 生命週期 | 暫停/恢復/終止管理 |
| 套件格式 | .msix / .appx |

### WSL 2 (Windows Subsystem for Linux)

| 版本 | 架構 | 特性 |
|:----:|:-----|:-----|
| WSL 1 | Syscall 轉譯層 (lxss.sys) | 無完整 Linux Kernel，IO 慢 |
| WSL 2 | Hyper-V VM (輕量) | 完整 Linux Kernel，Ext4 VHD，IO 快 |

| WSL 快速指令 | 用途 |
|:-------------|:-----|
| `wsl --install` | 安裝 WSL 2 + Ubuntu |
| `wsl --list --verbose` | 列出發行版 |
| `wsl --set-version <distro> 2` | 切換至 WSL 2 |
| `wsl --shutdown` | 關閉 WSL VM |
| `wsl ~ -d Ubuntu` | 啟動特定發行版 |

## 驅動模型 Driver Model

| 模型 | 全名 | 層級 | 說明 |
|:----:|:-----|:----:|:-----|
| WDM | Windows Driver Model | Kernel | 傳統核心模式驅動 |
| KMDF | Kernel-Mode Driver Framework | Kernel | WDM 的現代封裝，更安全 |
| UMDF | User-Mode Driver Framework | User | 用戶模式驅動（USB/列印） |
| WDDM | Windows Display Driver Model | Kernel | GPU 顯示驅動 |

### 驅動管理指令

| 指令 | 用途 |
|:-----|:-----|
| `driverquery` | 列出已安裝驅動 |
| `driverquery /v` | 詳細模式 |
| `pnputil /enum-drivers` | 枚舉驅動套件 |
| `sc query type= driver` | 查詢驅動服務 |

## 記憶體架構 Memory Architecture

| 區域 | User Mode | Kernel Mode |
|:-----|:---------|:------------|
| 32-bit | 2 GB (預設) / 3 GB (3GB switch) | 2 GB / 1 GB |
| 64-bit | 128 TB (實際 8 TB) | 128 TB |
| 分頁 | Paged pool / Non-paged pool | — |

## 關鍵系統行程 Key System Processes

| 行程 | 說明 |
|:-----|:-----|
| System (PID 4) | 核心模式執行緒宿主 |
| smss.exe | Session Manager — 啟動 csrss/winlogon |
| csrss.exe | Client/Server Runtime |
| wininit.exe | Windows 初始化 |
| services.exe | 服務控制管理員 (SCM) |
| lsass.exe | 本地安全驗證子系統 |
| svchost.exe | 服務宿主（多實例） |
| explorer.exe | Shell (桌面、工作列) |
