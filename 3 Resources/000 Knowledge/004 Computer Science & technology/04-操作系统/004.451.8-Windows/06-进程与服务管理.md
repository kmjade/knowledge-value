# 06 — 進程與服務管理 Process & Service Management

> Windows 進程監控、服務管理、效能監視器、排程任務與開機啟動項管理。
> Task Manager, ProcMon, perfmon, services.msc, schtasks, startup management.

---

## 工作管理員 Task Manager

### 快速鍵與啟動

| 操作 | 快捷鍵/指令 |
|:-----|:------------|
| 直接開啟 | `Ctrl + Shift + Esc` |
| 安全畫面 | `Ctrl + Alt + Del` → 選擇 |
| 命令列 | `taskmgr` |
| 精簡模式 | 雙擊圖表區域 |

### 分頁功能 Tabs Overview

| 分頁 | 內容 | 實用操作 |
|:----:|:-----|:---------|
| Processes | 執行中進程 + 資源使用 | 右鍵 → 移至詳情、結束工作樹 |
| Performance | CPU/Memory/Disk/Net/GPU 圖表 | 複製摘要文字 |
| App history | UWP 應用資源歷史 | 刪除使用量記錄 |
| Startup | 開機啟動項 | 啟用/停用、查看啟動影響 |
| Users | 各使用者資源使用 | 中斷連線 |
| Details | 進程詳細資訊 (PID) | 設定優先順序、親和性 |
| Services | 服務清單 | 開啟 services.msc |

## 命令行進程管理 CLI Process Management

### tasklist / taskkill

| 指令 | 用途 | 範例 |
|:-----|:-----|:-----|
| `tasklist` | 列出所有進程 | `tasklist /v` (詳細) |
| `tasklist /svc` | 顯示進程託管服務 | `tasklist /svc /fi "pid eq 1234"` |
| `tasklist /m` | 顯示載入的 DLL | `tasklist /m kernel32.dll` |
| `taskkill /pid` | 依 PID 終止 | `taskkill /pid 1234 /f` |
| `taskkill /im` | 依名稱終止 | `taskkill /im notepad.exe /f` |
| `taskkill /t` | 終止進程樹 | `taskkill /pid 1234 /t /f` |

### PowerShell 進程管理

```powershell
Get-Process                          # 列出所有進程
Get-Process -Name chrome             # 依名稱篩選
Get-Process | Sort-Object CPU -Descending | Select -First 10  # CPU Top 10
Stop-Process -Name notepad -Force    # 強制終止
Start-Process notepad.exe            # 啟動新進程
Wait-Process -Name notepad           # 等待進程結束
Get-Process -Id $pid | Select *      # 查看進程所有屬性
```

## 服務管理 Service Management

### services.msc & sc.exe

| 操作 | services.msc | sc.exe |
|:-----|:------------|:-------|
| 列出服務 | 圖形清單 | `sc query` |
| 查詢狀態 | 狀態欄 | `sc query "Spooler"` |
| 啟動 | 右鍵 → Start | `sc start "Spooler"` |
| 停止 | 右鍵 → Stop | `sc stop "Spooler"` |
| 設定啟動類型 | 內容 → 一般 | `sc config "Spooler" start= auto` |
| 建立服務 | — | `sc create "MySvc" binPath= "C:\app.exe"` |
| 刪除服務 | — | `sc delete "MySvc"` |

### 啟動類型 Startup Types

| 類型 | sc 值 | 說明 |
|:-----|:-----:|:-----|
| Automatic (自動) | auto | 系統啟動時啟動 |
| Automatic (Delayed) | delayed-auto | 所有 auto 之後啟動 (約 2 min) |
| Manual (手動) | demand | 按需啟動 |
| Disabled (已停用) | disabled | 無法啟動 |
| Automatic (Trigger Start) | — | 依觸發事件 (Win 10+) |

### PowerShell 服務管理

```powershell
Get-Service                                    # 列出服務
Get-Service -Name wuauserv                     # Windows Update 服務
Start-Service -Name wuauserv
Stop-Service -Name wuauserv -Force
Restart-Service -Name Spooler
Set-Service -Name wuauserv -StartupType Manual
```

### 核心服務 Core Services Reference

| 服務名稱 | 顯示名稱 | 說明 |
|:---------|:---------|:-----|
| wuauserv | Windows Update | 系統更新 |
| Spooler | Print Spooler | 列印多工緩衝 |
| W32Time | Windows Time | 時間同步 |
| WinRM | Windows Remote Management | PowerShell 遠端 |
| TermService | Remote Desktop Services | RDP 連線 |
| WlanSvc | WLAN AutoConfig | Wi-Fi 管理 |
| DHCP | DHCP Client | IP 自動取得 |
| BITS | Background Intelligent Transfer | 背景傳輸 (Windows Update) |

## 效能監視 Performance Monitoring

### perfmon / 效能監視器

| 指標 | 用途 | 計數器範例 |
|:-----|:-----|:-----------|
| CPU | Processor(_Total)\% Processor Time | CPU 使用率 |
| Memory | Memory\Available MBytes | 可用記憶體 |
| Disk | PhysicalDisk(_Total)\Avg. Disk sec/Read | 磁碟延遲 |
| Network | Network Interface\Bytes Total/sec | 網路流量 |
| Process | Process(*)\Handle Count | 控制代碼洩漏 |

### 資源監視器 resmon

```cmd
resmon    # Resource Monitor
```

| 分頁 | 監控內容 |
|:----:|:---------|
| Overview | CPU/Disk/Network/Memory 總覽 |
| CPU | 各進程 CPU + 控制代碼/模組 |
| Memory | 各進程記憶體使用 |
| Disk | 磁碟 I/O 各檔案 |
| Network | TCP 連線 + 監聽埠 |

## 排程任務 Task Scheduler

### schtasks 指令

| 指令 | 用途 |
|:-----|:-----|
| `schtasks /query` | 列出所有排程任務 |
| `schtasks /query /tn "\TaskName" /v /fo LIST` | 詳細資訊 |
| `schtasks /create /tn "MyTask" /tr "C:\script.bat" /sc DAILY /st 09:00` | 建立每日任務 |
| `schtasks /run /tn "MyTask"` | 立即執行 |
| `schtasks /delete /tn "MyTask" /f` | 刪除任務 |
| `schtasks /change /tn "MyTask" /disable` | 停用 |

### 常用觸發器與條件

| 觸發器 (Triggers) | 說明 |
|:------------------|:-----|
| On a schedule | 每日/每週/每月定時 |
| At logon | 使用者登入時 |
| At startup | 系統啟動時 |
| On idle | 閒置時 |
| On an event | 事件記錄觸發 |
| At task creation/modification | 任務建立時 |

## 開機啟動項 Startup

### 啟動位置

| 類型 | 位置 |
|:-----|:-----|
| 使用者 Startup 資料夾 | `%AppData%\Microsoft\Windows\Start Menu\Programs\Startup` |
| 全域 Startup 資料夾 | `%ProgramData%\Microsoft\Windows\Start Menu\Programs\Startup` |
| Registry: HKCU Run | `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` |
| Registry: HKLM Run | `HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run` |
| Registry: RunOnce | `HKCU\...\RunOnce`, `HKLM\...\RunOnce` |
| 排程任務 | Task Scheduler Library |
| 服務 | services.msc (auto start) |

### 管理工具

| 工具 | 用途 |
|:-----|:-----|
| Task Manager → Startup | 啟用/停用、影響評估 |
| Autoruns (Sysinternals) | 完整啟動項掃描 |
| `msconfig` → 服務 | 診斷模式開機 |
