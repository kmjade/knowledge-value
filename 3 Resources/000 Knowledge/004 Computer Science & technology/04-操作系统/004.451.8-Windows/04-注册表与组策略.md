# 04 — 註冊表與組策略 Registry & Group Policy

> Windows Registry 的五大根鍵結構、regedit/reg.exe 操作指令、群組原則 (gpedit.msc/secpol.msc) 與 ADMX 範本管理。
> HKLM/HKCU/HKCR/HKU/HKCC, Registry editing, Group Policy Editor, Security Policy, ADMX templates.

---

## Registry 五大根鍵 Five Root Keys

| 根鍵 | 縮寫 | 內容 | 實際位置 |
|:-----|:----:|:-----|:---------|
| HKEY_LOCAL_MACHINE | HKLM | 全機設定 | `%SystemRoot%\System32\Config\` |
| HKEY_CURRENT_USER | HKCU | 目前用戶設定 | `%UserProfile%\NTUSER.DAT` |
| HKEY_CLASSES_ROOT | HKCR | 檔案關聯 + COM | HKLM\Software\Classes + HKCU\Software\Classes |
| HKEY_USERS | HKU | 所有用戶設定 | SID 子鍵 |
| HKEY_CURRENT_CONFIG | HKCC | 目前硬體設定檔 | HKLM\SYSTEM\CurrentControlSet\Hardware Profiles |

---

## Registry 重要路徑 Key Registry Paths

| 路徑 | 用途 |
|:-----|:-----|
| `HKLM\SYSTEM\CurrentControlSet\Services` | 所有服務與驅動 |
| `HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion` | OS 版本資訊 |
| `HKLM\SYSTEM\CurrentControlSet\Control\Session Manager` | 開機設定 (boot execute) |
| `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` | 使用者啟動項 |
| `HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run` | 全域啟動項 |
| `HKLM\SOFTWARE\Policies` | 群組原則設定 |
| `HKCU\Control Panel\Desktop` | 桌面外觀設定 |
| `HKLM\SYSTEM\CurrentControlSet\Control\Terminal Server` | RDP 設定 |
| `HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced` | Explorer 設定 |

## Registry 操作指令 reg.exe

| 操作 | 指令 | 範例 |
|:-----|:-----|:-----|
| 查詢 | `reg query` | `reg query "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion" /v ProductName` |
| 新增 | `reg add` | `reg add "HKCU\Software\MyApp" /v Setting /t REG_DWORD /d 1 /f` |
| 刪除 | `reg delete` | `reg delete "HKCU\Software\MyApp" /v Setting /f` |
| 匯出 | `reg export` | `reg export "HKLM\SOFTWARE\MyApp" C:\backup.reg` |
| 匯入 | `reg import` | `reg import C:\backup.reg` |
| 比較 | `reg compare` | `reg compare "HKLM\Software\A" "HKLM\Software\B"` |

### 登錄值類型 Registry Value Types

| 類型 | reg 標記 | 說明 |
|:-----|:--------|:-----|
| REG_SZ | — | 字串 |
| REG_DWORD | /t REG_DWORD | 32-bit 整數 |
| REG_QWORD | /t REG_QWORD | 64-bit 整數 |
| REG_BINARY | /t REG_BINARY | 二進位 |
| REG_MULTI_SZ | /t REG_MULTI_SZ | 多行字串 |
| REG_EXPAND_SZ | /t REG_EXPAND_SZ | 含環境變數的字串 |

## 群組原則 Group Policy

### 管理工具

| 工具 | 用途 | 適用版本 |
|:-----|:-----|:---------|
| gpedit.msc | 本機群組原則編輯器 | Pro/Enterprise (Home 可手動安裝) |
| secpol.msc | 本機安全性原則 | Pro/Enterprise |
| rsop.msc | 原則結果集 (RSoP) | Pro/Enterprise |
| gpupdate /force | 強制更新群組原則 | 全部 |
| gpresult /h report.html | 匯出原則報告 | 全部 |
| gpmc.msc | 群組原則管理主控臺 | 僅 Server (AD) |

### 常用 GPUpdate 指令

```cmd
gpupdate                          # 更新變更的原則
gpupdate /force                   # 強制更新全部原則
gpupdate /target:computer         # 僅電腦設定
gpupdate /target:user             # 僅使用者設定
gpupdate /boot                    # 需要重開機的原則
gpresult /r                       # 顯示 RSoP 摘要
gpresult /h C:\gpo_report.html    # HTML 報告
```

### 常用群組原則路徑

| 原則 | GP 路徑 |
|:-----|:-------|
| 密碼複雜度 | Computer\Windows Settings\Security Settings\Account Policies\Password Policy |
| 帳戶鎖定 | Computer\...\Account Policies\Account Lockout Policy |
| UAC 設定 | Computer\...\Security Settings\Local Policies\Security Options |
| Windows Update | Computer\Admin Templates\Windows Components\Windows Update |
| BitLocker | Computer\Admin Templates\Windows Components\BitLocker Drive Encryption |
| 網路 QoS | Computer\Admin Templates\Network\QoS Packet Scheduler |
| 移除 OneDrive | Computer\Admin Templates\Windows Components\OneDrive |
| 關閉 Defender | Computer\Admin Templates\Windows Components\Microsoft Defender Antivirus |

## ADMX 範本 ADMX Templates

| 概念 | 說明 |
|:-----|:-----|
| ADMX | XML-based 原則定義檔案（取代 ADM） |
| ADML | 語言檔案（繁體中文：zh-TW） |
| Central Store | `\\domain\SYSVOL\domain\Policies\PolicyDefinitions\` |
| 本機路徑 | `%SystemRoot%\PolicyDefinitions\` |
| 第三方 ADMX | Chrome、Firefox、Office 365 提供自有 ADMX |

## 安全性原則 secpol.msc 常用設定

| 設定 | 建議值 |
|:-----|:------|
| 密碼長度下限 | 8-14 字元 |
| 密碼最長使用期限 | 42-90 天 |
| 帳戶鎖定閾值 | 5 次無效登入 |
| 鎖定持續時間 | 15-30 分鐘 |
| UAC: 管理員核准模式 | 已啟用 |
| 稽核登入事件 | 成功 + 失敗 |
| 關閉自動播放 | 所有磁碟機 |
| LAN Manager 驗證層級 | NTLMv2 回應，拒絕 LM & NTLM |
