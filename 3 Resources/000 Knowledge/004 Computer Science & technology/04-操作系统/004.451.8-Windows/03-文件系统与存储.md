## NTFS 核心特性 NTFS Core Features

### 權限模型 Permission Model

| 權限 | 檔案 | 目錄 |
|:-----|:----:|:----:|
| Read | 讀取內容 | 列出內容 |
| Write | 修改內容 | 建立檔案/子目錄 |
| Execute | 執行 | 遍歷目錄 |
| Delete | — | 刪除 |
| Full Control | 全部 | 全部 |
| Modify | Read+Write+Delete | Read+Write+Delete |

### NTFS 壓縮

```cmd
# 壓縮檔案/目錄
compact /c /s:C:\Data

# 查詢壓縮狀態
compact /q C:\Data

# 解壓縮
compact /u /s:C:\Data
```

### NTFS 符號連結 Symlinks

```cmd
# 建立符號連結 (管理員權限)
mklink link.txt target.txt          # 檔案符號連結
mklink /D link_dir target_dir       # 目錄符號連結
mklink /H hardlink.txt target.txt   # 硬連結
mklink /J junction_dir target_dir   # 目錄連接點 (Junction)
```

| 連結類型 | 跨磁碟區 | 目標刪除後 | 備註 |
|:---------|:--------:|:----------:|:-----|
| Symbolic Link | ✅ | 損壞 | 需管理員權限 |
| Hard Link | ❌ | 資料保留 | 僅檔案 |
| Junction | ✅ (本機) | 目標不存在 → 錯誤 | 僅目錄，不需管理員 |

## 磁碟管理 Disk Management

### diskmgmt.msc & diskpart

| 操作 | diskmgmt.msc (GUI) | diskpart (CLI) |
|:-----|:-------------------|:---------------|
| 列出磁碟 | 圖形化顯示 | `list disk` |
| 初始化 | 右鍵 → Initialize | `clean` / `convert gpt` |
| 建立磁碟區 | 右鍵 → New Simple Volume | `create partition primary` |
| 格式化 | 右鍵 → Format | `format fs=ntfs quick` |
| 變更磁碟機代號 | 右鍵 → Change Drive Letter | `assign letter=D` |
| 擴展磁碟區 | 右鍵 → Extend Volume | `extend` |

### 常用 diskpart 指令序列

```cmd
diskpart
list disk
select disk 1
clean                # ⚠️ 清除所有分割
convert gpt
create partition primary
format fs=ntfs quick label="Data"
assign letter=E
```

## Storage Spaces 儲存空間

| 復原類型 | 最少磁碟 | 容錯 | 容量效率 | 說明 |
|:---------|:--------:|:----:|:--------:|:-----|
| Simple | 1 | 無 | 100% | 類似 RAID 0 |
| Two-way mirror | 2 | 1 碟 | 50% | 類似 RAID 1 |
| Three-way mirror | 5 | 2 碟 | 33% | 高可用 |
| Parity | 3 | 1 碟 | ~67% | 類似 RAID 5 |
| Dual parity | 7 | 2 碟 | ~57% | 類似 RAID 6 |

```powershell
# 建立 Storage Space (PowerShell)
New-StoragePool -FriendlyName "Pool1" -PhysicalDisks $disks
New-VirtualDisk -StoragePoolFriendlyName "Pool1" -FriendlyName "VDisk1" `
    -Size 500GB -ResiliencySettingName Mirror
Initialize-Disk -Number $vd.Number -PartitionStyle GPT
New-Partition -DiskNumber $vd.Number -UseMaximumSize -DriveLetter D
Format-Volume -DriveLetter D -FileSystem NTFS
```

## BitLocker 加密

| 版本 | 適用對象 | 解鎖方式 | 管理模式 |
|:-----|:---------|:---------|:---------|
| 裝置加密 (Device Encryption) | Home | TPM only | 自動（需 MS 帳戶） |
| BitLocker | Pro/Enterprise | TPM, PIN, USB Key, Network Unlock | 完整控制 |
| BitLocker To Go | Pro/Enterprise | 密碼, Smart Card | 抽取式磁碟 |

### BitLocker 管理指令

```cmd
# 啟用 BitLocker (C: 磁碟，TPM 保護)
manage-bde -on C: -RecoveryPassword

# 查詢狀態
manage-bde -status C:

# 暫停保護（BIOS/韌體更新前）
manage-bde -protectors -disable C:

# 恢復保護
manage-bde -protectors -enable C:

# 備份修復金鑰至 AD
manage-bde -protectors -adbackup C:
```

| GPO 關鍵設定 | 路徑 |
|:-------------|:-----|
| 加密方法 | Computer\Admin Templates\Windows Components\BitLocker\ |
| 啟動驗證 | ...\Operating System Drives\Require additional authentication |
| 修復金鑰儲存 | ...\Choose how BitLocker-protected drives can be recovered |
