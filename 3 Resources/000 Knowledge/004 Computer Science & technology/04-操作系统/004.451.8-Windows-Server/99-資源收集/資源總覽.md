## 關鍵 PowerShell 模組 Essential PowerShell Modules

| 模組 Module | 用途 Purpose | 安裝命令 |
|:------|------|------|
| **ActiveDirectory** | AD 管理 | `Install-WindowsFeature RSAT-AD-PowerShell` |
| **DnsServer** | DNS 管理 | `Install-WindowsFeature RSAT-DNS-Server` |
| **DhcpServer** | DHCP 管理 | `Install-WindowsFeature RSAT-DHCP` |
| **Hyper-V** | Hyper-V 管理 | `Install-WindowsFeature Hyper-V-PowerShell` |
| **FailoverClusters** | 叢集管理 | `Install-WindowsFeature RSAT-Clustering-PowerShell` |
| **IISAdministration** | IIS 管理 (新) | `Install-WindowsFeature Web-Scripting-Tools` |
| **StorageReplica** | 儲存複本 | `Install-WindowsFeature Storage-Replica` |
| **NetworkController** | SDN 控制器 | `Install-WindowsFeature NetworkController` |
| **Microsoft.Graph** | Microsoft Graph API | `Install-Module Microsoft.Graph` |
| **Az** | Azure 管理 | `Install-Module Az` |
| **DSC (PSDesiredStateConfiguration)** | 期望狀態設定 | 內建 (Win 11+ / Server 2025 移除外置) |
