# 02 — 安裝與部署 Installation & Deployment

> 從手動安裝到大規模自動化部署：ADK/WDS/MDT 工具鏈、無人值守安裝、Windows Admin Center 管理。

---

## 部署工具鏈對比 Deployment Tools Comparison

| 工具 Tool | 規模 Scale | 場景 Scenario | 難度 Difficulty |
|:------|:---:|------|:---:|
| **手動安裝 Manual Install** | 1-5 台 | Lab / PoC | 🟢 低 |
| **autounattend.xml** | 1-50 台 | 批次無人值守 | 🟡 中 |
| **WDS (Windows Deployment Services)** | 50-500 台 | PXE 網路開機部署 | 🟡 中 |
| **MDT (Microsoft Deployment Toolkit)** | 50-1000 台 | Lite Touch (LTI) 部署 | 🟡 中 |
| **SCCM / MECM (Configuration Manager)** | 1000+ 台 | Zero Touch (ZTI) 企業部署 | 🔴 高 |
| **Azure Arc / Automanage** | 混合雲 | 雲端管理 + Hotpatch | 🟡 中 |
| **Windows Admin Center (WAC)** | 1-100 台 | Web-based 日常管理 | 🟢 低 |

## Windows ADK (Assessment & Deployment Kit) 元件 Components

| 元件 Component | 功能 Function | 命令/路徑 |
|:------|------|------|
| **Deployment Tools** | DISM, OSCDIMG, BCDBoot | `copype`, `MakeWinPEMedia` |
| **Windows PE** | WinPE 開機映像 | `WinPE_OCs` |
| **USMT (User State Migration Tool)** | 使用者設定遷移 | `scanstate`, `loadstate` |
| **VAMT (Volume Activation Mgmt Tool)** | KMS/MAK 啟用管理 | `VAMT.exe` |
| **Windows Performance Toolkit** | 效能分析 | `WPR`, `WPA` |
| **Windows Assessment Toolkit** | 系統評估 | `AXScfg.exe` |

## autounattend.xml 結構 Structure

```xml
<!-- 關鍵 Section 對照表 Key Sections -->
<!-- 1. windowsPE: 磁碟分區、Product Key -->
<!-- 2. offlineServicing: 驅動注入 -->
<!-- 3. generalize: Sysprep -->
<!-- 4. specialize: 電腦名稱、網路設定 -->
<!-- 5. oobeSystem: 管理員密碼、時區 -->
```

| 設定項 Setting | XML Path | 範例值 Example |
|:------|------|------|
| 磁碟分區 Disk Config | `windowsPE > DiskConfiguration` | GPT, EFI 100MB + MSR 16MB + OS |
| 產品金鑰 Product Key | `windowsPE > UserData > ProductKey` | 或 KMS Client Key |
| 語言/地區 Locale | `windowsPE > UILanguage` | zh-TW / en-US |
| 電腦名稱 Computer Name | `specialize > ComputerName` | SRV01-%RAND% |
| 管理員密碼 Admin Password | `oobeSystem > AdministratorPassword` | Base64 編碼 |
| 時區 Time Zone | `oobeSystem > TimeZone` | Taipei Standard Time |
| 自動登入次數 AutoLogon | `oobeSystem > AutoLogon` | 1 (只首次) |
| 加入網域 Domain Join | `specialize > Identification` | 需 Credentials |
| FirstLogonCommands | `oobeSystem > FirstLogonCommands` | PowerShell 腳本 |

### 生成 autounattend.xml

```powershell
# 使用 Windows System Image Manager (SIM)
# 1. 安裝 ADK → 啟動 Windows SIM
# 2. 載入 install.wim → 建立 Catalog
# 3. 配置各 Pass → 匯出 XML

# 或用 PowerShell 快速生成基本版
# 後用於 USB 或 WDS
```

## Windows Admin Center (WAC) 部署模式 Deployment Modes

| 模式 Mode | 架構 Architecture | 適用場景 Scenario |
|:------|------|------|
| **單機模式 Standalone** | 單一 WAC Gateway → 多 Server | 小型環境 (< 100 伺服器) |
| **HA 叢集模式** | WAC on Failover Cluster | 生產環境 |
| **Azure 模式** | WAC in Azure VM | 雲端管理 |
| **閘道模式 Gateway** | 經由單一 Gateway 代理 | DMZ / 跳板 |

### WAC 快速安裝 Quick Install

```powershell
# Download
Invoke-WebRequest -Uri "https://aka.ms/WACDownload" -OutFile WAC.msi

# 安裝 (port 443)
msiexec /i WAC.msi /qn /L*v log.txt SME_PORT=443 SSL_CERTIFICATE_OPTION=generate

# 或 PowerShell 一鍵
Install-WindowsAdminCenter -Port 443 -SslCertificateOption Generate
```

## sconfig — Server Core 管理工具 Server Core Configuration Tool

| 選項 # | 功能 Function | 說明 |
|:--:|------|------|
| 1 | Domain/Workgroup | 加入網域或工作群組 |

| 4 | Remote Management | 開啟 WinRM / RDP |
| 5 | Windows Update | 設定更新策略 |
| 6 | Download & Install | 下載並安裝更新 |
| 7 | Remote Desktop | 啟用 RDP |
| 8 | Network Settings | IP/DNS 設定 |
| 9 | Date and Time | 日期時間 |
| 10 | Telemetry | 遙測設定 |
| 11 | Activate Windows | 啟用授權 |
| 12 | Sign Out | 登出 |
| 13 | Restart | 重新啟動 |
| 14 | Shut Down | 關機 |
| 15 | Exit to CMD/PowerShell | 退出到命令列 |

## 部署流程最佳實踐 Deployment Best Practices

| 階段 Phase | 行動 Action | 工具 Tool |
|:---:|------|------|
| 1 | 準備 ISO → 擷取 install.wim | ADK + OSCDIMG |
| 2 | 建立 autounattend.xml | Windows SIM |
| 3 | 加入驅動/更新到映像 | DISM (`/Add-Driver`, `/Add-Package`) |
| 4 | PXE 開機設定 (可選) | WDS |
| 5 | 部署 Task Sequence (可選) | MDT / SCCM |
| 6 | 後部署設定 Post-deployment | PowerShell DSC |
| 7 | 驗證 + 基線 | WAC + Pester |
