# 07 — 儲存與檔案服務 Storage & File Services

> Windows Server 儲存堆疊：NTFS/ReFS 檔案系統、Storage Spaces 虛擬化儲存、SMB 通訊協定、DFS 命名空間與複寫、FSRM 與重複資料刪除。

| 特性 Feature | NTFS | ReFS | FAT32 | exFAT |

| **最大檔案 File Max** | 16 EB | 16 EB | 4 GB | 16 EB |
| **日誌 Journaling** | ✅ | ✅ (metadata only) | ❌ | ❌ |

| **自動修復 Auto-repair** | ❌ (需 chkdsk) | ✅ (online self-healing) | ❌ | ❌ |
| **去重 Dedup 支援** | ✅ | ✅ (Server 1709+) | ❌ | ❌ |
| **開機 Bootable** | ✅ | ❌ (Server 2025 Datacenter 支援) | ✅ | ❌ |
| **檔案層級加密 EFS** | ✅ | ❌ | ❌ | ❌ |
| **壓縮 Compression** | ✅ | ❌ | ❌ | ❌ |

## Storage Spaces 儲存空間

### 儲存集區與虛擬磁碟 Storage Pool & Virtual Disk

| 配置 Layout | 容錯 Fault Tolerance | 可用容量計算 | 建議場景 |
|:------|:---:|------|------|
| **Simple** | ❌ 無容錯 | N × Disk Size | 暫存 / 效能 |
| **Mirror (2-way)** | ✅ 1 磁碟 | (N / 2) × Disk | 一般生產 |
| **Mirror (3-way)** | ✅ 2 磁碟 | (N / 3) × Disk | 高可用 |
| **Parity (Single)** | ✅ 1 磁碟 | (N − 1) × Disk | 歸檔 / 備份 |
| **Parity (Dual)** | ✅ 2 磁碟 | (N − 2) × Disk | 極高可靠 |
| **Mirror-Accelerated Parity** | ✅ 混合 | 依比例 | 🟢 2025 新功能 |

### Storage Spaces 命令 Storage Spaces Commands

```powershell
# 建立儲存集區
$disks = Get-PhysicalDisk -CanPool $true
New-StoragePool -FriendlyName "Pool01" -StorageSubsystemFriendlyName "Windows Storage*" `
  -PhysicalDisks $disks

# 建立虛擬磁碟 (Mirror)
New-VirtualDisk -StoragePoolFriendlyName "Pool01" -FriendlyName "VDisk01" `
  -Size 500GB -ResiliencySettingName Mirror -ProvisioningType Fixed

# 初始化 + 格式化
Initialize-Disk -Number 2 -PartitionStyle GPT
New-Partition -DiskNumber 2 -UseMaximumSize -AssignDriveLetter F
Format-Volume -DriveLetter F -FileSystem ReFS -NewFileSystemLabel "HyperV-Data"
```

## SMB 通訊協定 SMB Protocol

| 版本 Version | Windows 版本 | 關鍵特性 Key Features |
|:---:|------|------|
| **SMB 1.0 / CIFS** | Win 2000 | Legacy，已棄用 Deprecated (安全風險) |
| **SMB 2.0** | Vista / Server 2008 | 減少指令、複合操作 |
| **SMB 2.1** | Win 7 / Server 2008 R2 | Large MTU support, 租約 Leases |
| **SMB 3.0** | Win 8 / Server 2012 | SMB Direct (RDMA), Multichannel, Transparent Failover |
| **SMB 3.1.1** | Win 10 / Server 2016+ | AES-256 加密, Pre-authentication Integrity, QUIC |

### SMB 安全設定

```powershell
# 停用 SMB 1.0 (重大安全保護)
Disable-WindowsOptionalFeature -Online -FeatureName SMB1Protocol
# 或
Set-SmbServerConfiguration -EnableSMB1Protocol $false

# 強制 SMB 加密
Set-SmbServerConfiguration -EncryptData $true -Force

# 強制 SMB 簽章
Set-SmbServerConfiguration -RequireSecuritySignature $true -Force

# 拒絕未加密存取
Set-SmbServerConfiguration -RejectUnencryptedAccess $true -Force

# SMB over QUIC (Server 2022+)
New-SmbServerCertificateMapping -Name "quic.contoso.com" `
  -Thumbprint "ABCD1234..." -StoreName My -Flags AllowQUIC
```

## DFS — 分散式檔案系統 Distributed File System

| 元件 Component | 功能 Function | 場景 Scenario |
|:------|------|------|
| **DFS Namespace (DFS-N)** | 統一名稱空間 `\\contoso.com\` | 多伺服器統一存取點 |
| **DFS Replication (DFS-R)** | 檔案複寫 (多站點同步) | 分支機構檔案同步、SYSVOL 複寫 |
| **DFSN Standalone** | 單一伺服器命名空間 | 小型部署 |
| **DFSN Domain-based** | AD-Integrated 命名空間 | 🟢 高可用建議 |

### DFS-N 設定 DFS Namespace Setup

```powershell
# 安裝 DFS 角色
Install-WindowsFeature FS-DFS-Namespace, FS-DFS-Replication -IncludeManagementTools

# 建立 Domain-based 命名空間
New-DfsnRoot -TargetPath "\\dc01\Shares" -Type DomainV2 -Path "\\contoso.com\Data"

# 加入資料夾目標 (多伺服器)
New-DfsnFolderTarget -Path "\\contoso.com\Data\HR" -TargetPath "\\srv01\HR"
New-DfsnFolderTarget -Path "\\contoso.com\Data\HR" -TargetPath "\\srv02\HR"

# 啟用 DFS-R 複寫
New-DfsReplicationGroup -GroupName "HR-RG" | Add-DfsrMember -ComputerName srv01,srv02
```

## FSRM — 檔案伺服器資源管理器 File Server Resource Manager

| 功能 Feature | 說明 Description | 範例 |
|:------|------|------|
| **配額 Quotas** | 限制目錄/磁碟區使用量 | 員工 Home folder 10GB 上限 |
| **檔案篩選 File Screening** | 封鎖特定副檔名 | 禁止 `.mp3`, `.mkv`, `.torrent` |
| **儲存報告 Storage Reports** | 定期產生使用率報告 | 每週 "Files by Owner" 報表 |
| **檔案管理工作 File Mgmt Tasks** | 自動過期/歸檔/刪除 | 30 天未存取檔案自動歸檔 |

```powershell
# 安裝 FSRM
Install-WindowsFeature FS-Resource-Manager

# 建立檔案群組 (禁止的副檔名)
New-FsrmFileGroup -Name "Blocked Media" `
  -IncludePattern @("*.mp3","*.avi","*.mkv","*.mp4","*.mov")

# 建立檔案篩選
New-FsrmFileScreen -Path "F:\Shares" -Template "Block Audio and Video Files"
```

## 重複資料刪除 Data Deduplication

| 選項 Option | 說明 Description | 適用場景 |
|:------|------|------|
| **General Purpose** | 預設，平衡效能與省量 | 一般檔案伺服器 |
| **Hyper-V** | 針對 VHDX 最佳化 | Hyper-V 儲存 |
| **Backup** | 針對備份軟體 (如 DPM) 優化 | 備份磁碟區 |

```powershell
# 安裝 Dedup
Install-WindowsFeature FS-Data-Deduplication

# 啟用 Dedup (磁碟區 D:)
Enable-DedupVolume -Volume D: -UsageType HyperV

# 手動觸發去重作業
Start-DedupJob -Volume D: -Type Optimization

# 查看節省空間
Get-DedupStatus -Volume D:
```
