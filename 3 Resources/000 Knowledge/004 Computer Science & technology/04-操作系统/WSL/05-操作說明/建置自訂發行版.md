# 建置自訂發行版

> [!info] 說明
> 從頭建立自訂的 WSL Linux 發行版。

## 建置流程

```mermaid
flowchart LR
    A[準備 rootfs] --> B[建立 TAR]
    B --> C[匯入 WSL]
    C --> D[設定發行版]
    D --> E[測試驗證]
```

## 方法一：使用現有 Linux 系統

### 建立最小 rootfs

```bash
# 在 Debian/Ubuntu 系統中
sudo debootstrap --variant=minbase jammy ./rootfs http://archive.ubuntu.com/ubuntu/

# 或使用 yum
sudo yum --installroot=/path/to/rootfs install @core
```

### 建立 TAR 封存

```bash
# 進入 rootfs 目錄
cd rootfs

# 建立 TAR 檔
sudo tar --numeric-owner -czvf ../custom-distro.tar.gz .
```

### 匯入 WSL

```powershell
wsl --import CustomDistro D:\WSL\CustomDistro custom-distro.tar.gz
```

## 方法二：使用 Docker

### 建立 Dockerfile

```dockerfile
# Dockerfile
FROM ubuntu:22.04

# 安裝必要套件
RUN apt-get update && apt-get install -y \
    sudo \
    curl \
    wget \
    git \
    && rm -rf /var/lib/apt/lists/*

# 建立使用者
RUN useradd -m -s /bin/bash developer && \
    echo "developer ALL=(ALL) NOPASSWD:ALL" >> /etc/sudoers

# 設定 WSL 預設設定
RUN mkdir -p /etc && \
    echo "[user]\ndefault = developer" > /etc/wsl.conf

# 設定工作目錄
WORKDIR /home/developer

# 設定環境變數
ENV LANG=C.UTF-8
ENV LC_ALL=C.UTF-8
```

### 建置並匯出

```bash
# 建置映像檔
docker build -t my-wsl-distro .

# 建立容器
docker create --name temp my-wsl-distro

# 匯出
docker export temp > my-wsl-distro.tar

# 清理
docker rm temp
```

### 匯入 WSL

```powershell
wsl --import MyDistro D:\WSL\MyDistro my-wsl-distro.tar
```

## 方法三：使用 chroot

### 建立 rootfs 結構

```bash
# 建立目錄結構
mkdir -p mydistro/{bin,dev,etc,home,lib,lib64,opt,proc,root,sbin,sys,tmp,usr,var}

# 複製必要檔案 (從現有系統)
# 這是一個簡化的範例
sudo debootstrap --variant=minbase bullseye ./mydistro http://deb.debian.org/debian/
```

### 設定基本系統

```bash
# 進入 chroot
sudo chroot mydistro /bin/bash

# 設定密碼
passwd

# 設定主機名稱
echo "mydistro" > /etc/hostname

# 設定時區
ln -sf /usr/share/zoneinfo/Asia/Taipei /etc/localtime

# 安裝額外套件
apt update
apt install -y sudo curl wget git vim

# 建立使用者
useradd -m -s /bin/bash user
passwd user
usermod -aG sudo user

# 設定 WSL 配置
cat > /etc/wsl.conf << 'EOF'
[user]
default = user

[interop]
enabled = true
appendWindowsPath = true
EOF

# 退出 chroot
exit
```

### 建立 TAR

```bash
cd mydistro
sudo tar --numeric-owner -czvf ../mydistro.tar.gz .
```

## 發行版最佳化

### 最小化大小

```bash
# 清理套件快取
apt clean
rm -rf /var/lib/apt/lists/*
rm -rf /var/cache/apt/archives/*

# 清理日誌
rm -rf /var/log/*

# 清理暫存檔
rm -rf /tmp/*
```

### 預先安裝工具

```bash
# 安裝開發工具
apt install -y build-essential git curl wget vim nano

# 安裝 Python
apt install -y python3 python3-pip

# 安裝 Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs
```

### 自訂設定腳本

```bash
# 建立 /root/setup.sh
cat > /root/setup.sh << 'EOF'
#!/bin/bash
# 首次啟動設定腳本

# 更新系統
apt update && apt upgrade -y

# 安裝額外套件
apt install -y $(cat /root/packages.list)

# 設定 Git
git config --global user.name "$GIT_NAME"
git config --global user.email "$GIT_EMAIL"

# 清理
rm /root/setup.sh
EOF

chmod +x /root/setup.sh
```

## 驗證發行版

### 啟動並測試

```powershell
# 啟動發行版
wsl -d MyDistro

# 在發行版內執行
# 檢查基本功能
uname -a
cat /etc/os-release
which sudo curl wget git
```

### 測試 WSL 整合

```bash
# 測試 Windows 程式執行
notepad.exe
explorer.exe .

# 測試網路
ping google.com

# 測試檔案系統
ls /mnt/c/
```

## 封裝與分發

### 建立可分發的封裝

```powershell
# 匯出發行版
wsl --export MyDistro mydistro-v1.0.tar.gz

# 建立安裝腳本
# install.ps1
$distroName = "MyDistro"
$installPath = "$env:LOCALAPPDATA\WSL\$distroName"
$tarFile = "mydistro-v1.0.tar.gz"

Write-Host "Installing $distroName..."
wsl --import $distroName $installPath $tarFile
wsl -d $distroName

Write-Host "Installation complete!"
```

### 建立解除安裝腳本

```powershell
# uninstall.ps1
$distroName = "MyDistro"

Write-Host "Uninstalling $distroName..."
wsl --terminate $distroName
wsl --unregister $distroName

Write-Host "Uninstallation complete!"
```
