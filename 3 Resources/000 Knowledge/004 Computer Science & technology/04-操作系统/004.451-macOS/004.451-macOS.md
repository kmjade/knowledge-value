# 004.451 macOS

> macOS 是 Apple 為 Mac 電腦開發的 Unix 作業系統，基於 Darwin (XNU 核心 + BSD)，從 2001 年的 Mac OS X Cheetah 到 2025 年的 macOS Sequoia 持續演化，現已遷移至 Apple Silicon (M1-M4) 架構。
> macOS is Apple's Unix-based operating system for Mac computers, built on Darwin (XNU kernel + BSD), evolving from Mac OS X Cheetah (2001) to macOS Sequoia (2025), now fully migrated to Apple Silicon (M1-M4).
