# 05 應用與套件管理 Apps & Package Management

> macOS 軟體生態：Homebrew (brew/cask/mas) 為終端套件管理、.app Bundle 為原生應用格式、Gatekeeper 控管安全、LaunchAgent/LaunchDaemon 管理背景服務。

## 安裝來源對比 Installation Source Comparison

| 來源 | 途徑 | 格式 | 範例 |
|------|------|------|------|
| **Homebrew** | CLI 套件管理 | Formula (原始碼/二進位) | `brew install git` |
| **Homebrew Cask** | GUI 應用安裝 | `.app` (GUI) | `brew install --cask firefox` |
| **Mac App Store** | Apple 官方商店 | `.app` (sandboxed) | `mas install 497799835` (Xcode) |
| **直接下載** | 官網 `.dmg` / `.pkg` | 手動安裝 | 拖曳到 /Applications |
| **mas-cli** | App Store CLI 前端 | — | `mas search`, `mas upgrade` |

## Homebrew 操作速查 Homebrew Cheat Sheet

```bash
# Formula (CLI 工具/函式庫)
brew search keyword               # 搜尋 formula
brew info git                     # 詳細資訊
brew install git                  # 安裝
brew upgrade git                  # 升級特定
brew uninstall git                # 移除
brew list                         # 列出已安裝 formula

# Cask (GUI 應用)
brew search --cask chrome         # 搜尋 GUI 應用
brew install --cask firefox       # 安裝 GUI 應用
brew list --cask                  # 列出已安裝 cask

# mas (Mac App Store CLI)
brew install mas                  # 安裝 mas
mas signin "apple@id.com"         # 登入 App Store
mas search Xcode                  # 搜尋
mas install 497799835             # 安裝 Xcode
mas upgrade                       # 升級所有 App Store 應用
```

## .app Bundle 結構 .app Bundle Structure

```
Example.app/
├── Contents/
│   ├── Info.plist          # 應用 metadata (Bundle ID, 版本, 最低 OS)
│   ├── MacOS/
│   │   └── Example         # 主執行檔
│   ├── Resources/          # 圖示/字型/nib/lproj
│   │   ├── AppIcon.icns
│   │   └── en.lproj/
│   ├── Frameworks/         # 內嵌 dylib/framework
│   └── _CodeSignature/     # 程式碼簽名
```

## Gatekeeper 與安全 Gatekeeper & Security

| 機制 | 功能 | 檢查 |
|------|------|:---:|
| **Gatekeeper** | 阻擋未簽名/未公證應用 | `spctl --assess /Applications/App.app` |
| **Notarization** | Apple 自動掃描惡意軟體並公證 | `spctl -a -v App.dmg` |
| **XProtect** | 內建反惡意軟體 (YARA 規則) | 自動更新 |
| **App Sandbox** | 限制 App 存取範圍 (App Store 強制) | entitlement 定義 |

```bash
# Gatekeeper 管理
spctl --status                     # 查看狀態
sudo spctl --master-disable        # 允許任何來源 (不建議)
sudo spctl --master-enable         # 恢復預設
xattr -d com.apple.quarantine App  # 移除檢疫標記
```

## LaunchAgent / LaunchDaemon 範例

```xml
<!-- ~/Library/LaunchAgents/com.example.script.plist -->
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "...">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.example.script</string>
    <key>ProgramArguments</key>
    <array>
        <string>/usr/local/bin/my-script.sh</string>
    </array>
    <key>StartInterval</key>
    <integer>3600</integer>
</dict>
</plist>
```
