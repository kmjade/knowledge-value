# 08 自動化與腳本 Automation & Scripting

> macOS 提供多層次自動化方案：Automator (圖形化工作流)、Shortcuts (跨裝置自動化)、AppleScript/JXA (系統級腳本)、launchd (排程背景任務)、第三方工具如 Hazel/Keyboard Maestro。

## 自動化方案對比 Automation Comparison

| 方案 | 類型 | 複雜度 | 適用場景 |
|------|------|:---:|------|
| **Automator** | GUI 拖曳工作流 | 🟢 低 | 檔案批次處理、PDF 合併 |
| **Shortcuts** | 現代工作流 (macOS 12+) | 🟢 低 | 跨裝置、快速自動化 |
| **AppleScript** | 系統級腳本語言 | 🟡 中 | 操控應用 (Mail, Finder) |
| **JXA** | JavaScript for Automation | 🟡 中 | JS 開發者友善 |
| **Shell Script** | bash/zsh 腳本 | 🟡 中 | 系統管理、檔案操作 |
| **launchd plist** | 定時/事件觸發 | 🔴 高 | 背景服務、排程任務 |
| **Hazel** | 資料夾自動整理 | 🟢 低 | 自動歸檔/標籤 |
| **Keyboard Maestro** | 巨集/觸發器 | 🟡 中 | 鍵盤/滑鼠自動化 |

## AppleScript 範例 AppleScript Examples

```bash
# 在終端執行 AppleScript
osascript -e 'display notification "Done!" with title "Build"'
osascript -e 'tell application "Finder" to get name of every disk'

# 從檔案執行
osascript myscript.scpt
```

```applescript
-- 操控 Finder：打開目錄
tell application "Finder"
    make new Finder window
    set target of front window to (POSIX file "/Users/me/Projects")
end tell

-- 操控 Chrome：開啟 URL
tell application "Google Chrome"
    activate
    open location "https://github.com"
end tell
```

## launchd 排程範例 Scheduled Jobs with launchd

```xml
<!-- ~/Library/LaunchAgents/com.user.daily-backup.plist -->
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "...">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.user.daily-backup</string>
    <key>ProgramArguments</key>
    <array>
        <string>/usr/local/bin/backup.sh</string>
    </array>
    <key>StartCalendarInterval</key>
    <dict>
        <key>Hour</key><integer>2</integer>
        <key>Minute</key><integer>0</integer>
    </dict>
    <key>StandardOutPath</key>
    <string>/tmp/backup.log</string>
    <key>StandardErrorPath</key>
    <string>/tmp/backup.err</string>
</dict>
</plist>
```

```bash
# 載入/啟動
launchctl load ~/Library/LaunchAgents/com.user.daily-backup.plist
launchctl start com.user.daily-backup

# 列出使用者排程
launchctl list | grep com.user
```

## 實用自動化腳本 Practical Automation Scripts

```bash
#!/bin/zsh
# cleanup.sh — 自動清理暫存與下載舊檔

# 清理超過 30 天的下載檔案
find ~/Downloads -type f -mtime +30 -exec rm -v {} \;

# 清空垃圾桶 (謹慎!)
# rm -rf ~/.Trash/*

# 清理 Xcode 衍生資料
rm -rf ~/Library/Developer/Xcode/DerivedData/*

echo "Cleanup complete at $(date)"
```

```bash
#!/bin/zsh
# watch_build.sh — 監看目錄變更後自動編譯
fswatch -o ~/Projects/myapp/src | while read; do
    echo "Change detected — rebuilding..."
    make -j8 && say "Build succeeded" || say "Build failed"
done
```
