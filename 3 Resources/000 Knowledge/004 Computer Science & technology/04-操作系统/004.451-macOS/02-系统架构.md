# 02 系統架構 System Architecture

> macOS 是分層架構：Darwin (XNU= Mach+BSD+I/O Kit) 為底層，往上依序為 Core Services、Application Services (WindowServer/Metal)、圖形/應用層。

## XNU 核心架構 XNU Kernel Architecture

| 組件 Component | 來源 | 職責 |
|------|------|------|
| **Mach** | CMU (微核心) | 行程/執行緒排程、IPC、VM、real-time 支援 |
| **BSD** | FreeBSD | 檔案系統、網路堆疊、POSIX API、使用者管理 |
| **I/O Kit** | Apple (C++ restricted subset) | 驅動程式框架，動態載入、電源管理 |
| **Sandbox** | Apple | 強制存取控制 (MAC)，App Sandbox / SIP 實作基礎 |

## 關鍵子系統 Key Subsystems

| 子系統 Subsystem | 功能 | 相關工具/檔案 |
|------|------|------|
| **launchd** (PID 1) | 系統與服務初始化，取代 init/cron/inetd | `launchctl`, plist 於 `/System/Library/LaunchDaemons/` |
| **WindowServer** | 合成式視窗管理、GPU 加速繪圖 | Metal, Quartz, Core Animation |
| **Metal** | 低開銷 GPU API (類似 Vulkan/DX12) | 3D 渲染、ML 加速 |
| **SIP** | System Integrity Protection — 唯讀系統卷 | `csrutil status/enable/disable` |
| **APFS** | Apple File System (macOS 10.13+) | 快照、克隆、加密、空間共享 |

## launchd 結構 launchd Hierarchy

```bash
# PID 1 層級
/System/Library/LaunchDaemons/     # 系統 daemon (root)
/Library/LaunchDaemons/             # 第三方 daemon
~/Library/LaunchAgents/             # 使用者 agent (登入時)
/Library/LaunchAgents/              # 全域 agent
/System/Library/LaunchAgents/       # 系統 agent

# 常用指令
launchctl list                       # 列出所有服務
launchctl load/unload plist          # 載入/卸載 plist
launchctl kickstart -k label         # 強制重啟服務
```

## SIP 與安全架構 Security Architecture

```bash
csrutil status                        # 查看 SIP 狀態
# System Integrity Protection status: enabled.
csrutil disable                       # 關閉 (需 Recovery 模式)
# 保護範圍: /System, /bin, /sbin, /usr (除 /usr/local)
# 限制: kext 載入、DTrace、NVRAM 修改
```
