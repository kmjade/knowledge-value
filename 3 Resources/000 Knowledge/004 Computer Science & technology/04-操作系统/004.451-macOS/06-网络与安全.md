# 06 網路與安全 Networking & Security

> macOS 基於 BSD 網路堆疊，預設防火牆為 pf (Packet Filter)。FileVault 提供全碟加密、Keychain 管理密碼、TCC (Transparency Consent Control) 控管隱私權限。

## 防火牆設定 Firewall Configuration

| 機制 | 層級 | 工具 |
|------|------|------|
| **pf** (Packet Filter) | 核心級 (BSD 繼承) | `/etc/pf.conf`, `pfctl` |
| **ALF** (Application Layer Firewall) | 應用層 (GUI) | 系統設定 → 網路 → 防火牆 |
| **socketfilterfw** | ALF CLI 管理 | `/usr/libexec/ApplicationFirewall/socketfilterfw` |

```bash
# pf 防火牆操作
sudo pfctl -s all                    # 查看所有規則/狀態
sudo pfctl -f /etc/pf.conf           # 載入規則
sudo pfctl -e / -d                   # 啟用/停用 pf

# ALF 防火牆 (應用層)
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --getglobalstate
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --setblockall on

# 查看監聽埠
sudo lsof -iTCP -sTCP:LISTEN -P -n   # TCP 監聽埠
netstat -an -p tcp | grep LISTEN      # BSD netstat
```

## SSH 與遠端存取 SSH & Remote Access

```bash
# 啟用遠端登入 (系統設定 → 共享 → 遠端登入)
sudo systemsetup -setremotelogin on   # 啟用 SSH

# SSH 設定檔
/etc/ssh/sshd_config                 # SSH daemon 設定
~/.ssh/config                         # 用戶 SSH 設定

# SSH key 管理
ssh-keygen -t ed25519 -C "mac@example.com"
ssh-copy-id user@host                # 複製公鑰到伺服器
ssh-add --apple-use-keychain ~/.ssh/id_ed25519  # 存入 Keychain
```

## FileVault 全碟加密 Full Disk Encryption

```bash
fdesetup status                       # 查看加密狀態
sudo fdesetup enable                  # 啟用 FileVault
sudo fdesetup disable                 # 停用 (解密)
sudo fdesetup list                    # 列出授權用戶

# 恢復密鑰
sudo fdesetup validaterecovery        # 驗證恢復密鑰
```

## Keychain 密碼管理 Keychain Management

```bash
# Keychain CLI
security find-generic-password -a "account" -s "service" -w
security add-generic-password -a "user" -s "WiFi" -w "pass"
security list-keychains
security unlock-keychain ~/Library/Keychains/login.keychain-db

# 查看憑證
security find-certificate -a -p     # PEM 格式輸出所有憑證
```

## TCC 隱私控制 Transparency Consent Control

| 權限類別 | 說明 | TCC 資料庫 |
|------|------|------|
| **Camera / Microphone** | 相機/麥克風存取 | `~/Library/Application Support/com.apple.TCC/TCC.db` |
| **Full Disk Access** | 完整磁碟存取 (終端/IDE 需要) | 系統設定 → 隱私權 |
| **Accessibility** | 輔助使用 (自動化工具需要) | `tccutil reset Accessibility` |
| **Screen Recording** | 螢幕錄製/分享 | `tccutil reset ScreenCapture` |

```bash
# TCC 管理
tccutil reset All                    # 重置所有權限 (謹慎!)
tccutil reset Camera com.apple.Terminal  # 重置特定應用權限
```

## 惡意軟體防護 Malware Protection

| 機制 | 說明 |
|------|------|
| **XProtect** | YARA 規則庫，自動更新，掃描已知惡意軟體 |
| **MRT** (Malware Removal Tool) | 自動移除已發現的惡意軟體 |
| **Gatekeeper** | 檢查應用簽名/公證狀態 |
| **Notarization** | Apple 伺服器端掃描開發者提交的應用 |
