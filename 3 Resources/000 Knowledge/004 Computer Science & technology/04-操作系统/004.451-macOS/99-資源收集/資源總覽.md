## macOS 終端工具的 Linux 對照 macOS CLI vs Linux

| Linux 工具 | macOS 對應 | 說明 |
|------|------|------|
| `apt` / `dnf` | `brew` | 套件管理 |
| `systemctl` | `launchctl` | 服務管理 |
| `iptables` | `pfctl` | 防火牆 |
| `ip` / `ifconfig` | `networksetup` / `ifconfig` | 網路設定 |
| `cron` | `launchd` plist | 排程任務 |
| `procfs` (`/proc`) | `sysctl` / `vmmap` / `heap` | 行程資訊 |
| `strace` | `dtruss` / `DTrace` | 系統呼叫追蹤 |
