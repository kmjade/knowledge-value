## 推薦學習路徑 Learning Path

| 階段 | 內容 | 預計時間 | 資源 |
|:----:|------|:--------:|------|
| **1. 入門** | 了解 HarmonyOS 生態、版本、基本概念 | 1~2 天 | 本 KB 01-02 章 |
| **2. 環境** | 安裝 DevEco Studio，建立第一個 HAP | 0.5 天 | 官方 Quick Start |
| **3. ArkTS** | 學習 ArkTS 語法、裝飾器、聲明式 UI | 3~5 天 | 本 KB 03-04 章 + 官方 Codelabs |
| **4. 組件** | ArkUI 常用組件與佈局實戰 | 5~7 天 | 官方 Component 文檔 |
| **5. 數據網絡** | RDB、KVStore、HTTP 請求 | 3~5 天 | 本 KB 06 章 |
| **6. 分散式** | 分散式數據、遷移、超級設備 | 5~7 天 | 本 KB 05 章 + 多設備實戰 |
| **7. 安全** | 權限、簽名、TEE | 2~3 天 | 本 KB 07 章 |
| **8. 發佈** | 簽名、打包、上架 AGC | 1~2 天 | 本 KB 09 章 |
| **9. 進階** | ArkCompiler、性能優化、NDK | 持續 | OpenHarmony 原始碼 |
