## 必備工具 Essential Tools

| 工具 | 類別 | 說明 |
|------|------|------|
| `htop` | 監控 | 互動式行程檢視器 |
| `ncdu` | 磁碟 | 互動式磁碟使用分析 |
| `tldr` | 文件 | 簡化的指令範例 (替代冗長 man) |
| `bat` | 檢視 | 語法高亮的 `cat` 替代 |
| `ripgrep` (`rg`) | 搜尋 | 超快速文字搜尋 |
| `fd` | 搜尋 | 更友善的 `find` 替代 |
| `jq` | JSON | 命令列 JSON 處理器 |
| `fzf` | 互動 | 模糊搜尋工具 |
| `tmux` | 終端 | 終端多工器 |
| `rsync` | 同步 | 檔案同步與備份 |
