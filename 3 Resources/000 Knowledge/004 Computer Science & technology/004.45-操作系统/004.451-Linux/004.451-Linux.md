# 004.451 Linux

> Linux 是基於 Unix 哲學的開源作業系統內核，由 Linus Torvalds 於 1991 年創建，現已成為伺服器、嵌入式、雲端和超級電腦的主導平台。
> Linux is an open-source Unix-like operating system kernel created by Linus Torvalds in 1991, now dominating servers, embedded, cloud, and supercomputing.

## 發行版對比 Distro Comparison

| 發行版 Distro | 套件管理 | 初始化系統 | 目標用戶 | 發布模型 |
|:---|:---|:---|:---|:---|
| **Ubuntu** | apt/dpkg | systemd | 桌面/伺服器 | LTS (2yr) + 半年 |
| **Debian** | apt/dpkg | systemd | 穩定性優先 | Stable/Testing/Unstable |
| **RHEL** | dnf/rpm | systemd | 企業伺服器 | 10yr lifecycle |
| **Arch** | pacman | systemd | 進階用戶 | Rolling release |
| **Alpine** | apk | OpenRC | 容器/嵌入式 | Stable branch |
| **Fedora** | dnf/rpm | systemd | 開發者/桌面 | 半年 release |
| **openSUSE** | zypper/rpm | systemd | 桌面/伺服器 | Leap (stable) + Tumbleweed (rolling) |
| **Gentoo** | portage | OpenRC/systemd | 極致定制 | Rolling (source-based) |
