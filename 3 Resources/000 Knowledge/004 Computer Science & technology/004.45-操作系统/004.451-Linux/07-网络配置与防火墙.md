# 07 網路配置與防火牆 Networking & Firewall

> Linux 網路堆疊提供從底層封包過濾到高層 DNS 解析的完整工具鏈。現代 distro 使用 NetworkManager 或 systemd-networkd 管理網路介面。

## 網路配置工具 Network Configuration

| 工具 | 適用 | 設定檔 |
|------|------|------|
| **NetworkManager** | 桌面/伺服器 | `/etc/NetworkManager/` |
| **systemd-networkd** | 伺服器/容器 | `/etc/systemd/network/` |
| **Netplan** | Ubuntu Server (18.04+) | `/etc/netplan/*.yaml` |
| **ifupdown** (`/etc/network/interfaces`) | Debian 傳統 | `/etc/network/interfaces` |

```bash
# 基本網路檢視
ip addr                         # 查看 IP 位址
ip route                        # 路由表
ip link set eth0 up/down        # 啟用/停用介面
nmcli device status             # NetworkManager 狀態
nmcli connection show           # 列出連線
```

## 防火牆：iptables / nftables / ufw

| 工具 | 世代 | 特點 |
|------|:---:|------|
| **iptables** | Legacy | 廣泛使用，規則豐富 |
| **nftables** | Modern | iptables 後繼，更簡潔高效 |
| **ufw** | 簡化層 | Ubuntu 的 iptables 前端 |
| **firewalld** | 動態 | RHEL 預設，zone-based |

### iptables 關鍵概念

| 鏈 (Chain) | 時機 | 說明 |
|------|------|------|
| **INPUT** | 進入本機 | 外部連入的封包 |
| **OUTPUT** | 離開本機 | 本機發出的封包 |
| **FORWARD** | 經過本機 | 路由轉發的封包 |
| **PREROUTING** | 路由前 | DNAT 修改目的地 |
| **POSTROUTING** | 路由後 | SNAT/Masquerade |

```bash
# iptables 常用規則
iptables -L -v -n                   # 列出規則 (數字格式)
iptables -A INPUT -p tcp --dport 80 -j ACCEPT    # 允許 HTTP
iptables -A INPUT -p tcp --dport 443 -j ACCEPT   # 允許 HTTPS
iptables -A INPUT -j DROP                          # 預設拒絕
iptables-save > /etc/iptables/rules.v4            # 儲存規則

# ufw 簡化操作
ufw allow 80/tcp                # 允許 HTTP
ufw allow from 192.168.1.0/24  # 允許子網路
ufw enable                      # 啟用防火牆
ufw status numbered             # 規則列表
```

### nftables 範例

```bash
nft add table inet filter
nft add chain inet filter input { type filter hook input priority 0\; }
nft add rule inet filter input tcp dport 80 accept
```

## SSH 安全連線

```bash
# 基本連線
ssh user@host
ssh -p 2222 user@host           # 指定埠號
ssh -i ~/.ssh/id_rsa user@host  # 使用金鑰

# 金鑰管理
ssh-keygen -t ed25519 -C "comment"     # 產生金鑰
ssh-copy-id user@host                  # 複製公鑰到遠端
ssh-agent bash && ssh-add              # 金鑰代理

# SSH 設定 (/etc/ssh/sshd_config)
# Port 22 → 改為非預設埠
# PermitRootLogin no
# PasswordAuthentication no → 僅金鑰登入
```

## DNS 解析：systemd-resolved

```bash
resolvectl status                # 查看 DNS 設定
resolvectl query example.com     # 手動查詢
systemd-resolve --flush-caches   # 清除快取
```

## 封包分析 Packet Analysis

```bash
# tcpdump — 命令列封包擷取
tcpdump -i eth0                  # 監聽介面
tcpdump -i eth0 port 80          # 僅 HTTP
tcpdump -i eth0 host 1.2.3.4    # 僅特定主機
tcpdump -w capture.pcap          # 寫入檔案

# ss — socket 統計 (取代 netstat)
ss -tlnp                         # 監聽中的 TCP port
ss -s                            # 統計摘要
ss -t state established          # 已建立的連線
```
