# 06 — HarmonyOS 數據與網絡

> HarmonyOS 提供多層次數據存儲方案（RDB / KVStore / Preferences）與完整網絡協議支援。
> 分散式數據能力使其與傳統移動 OS 在數據層有本質差異。

---

## 數據存儲方案總覽

| 存儲類型 | API | 適用場景 | 容量 | 分散式 |
|----------|-----|----------|:----:|:------:|
| **RDB (Relational DB)** | `@ohos.data.relationalStore` | 結構化數據、複雜查詢 | MB~GB | ✅ |
| **KVStore** | `@ohos.data.distributedKVStore` | 鍵值對、簡單配置 | KB~MB | ✅ |
| **Preferences** | `@ohos.data.preferences` | 輕量配置項 | <10KB | ❌ |
| **DataShare** | `@ohos.data.dataShare` | 跨應用數據共享 | — | ✅ |
| **文件存儲** | `@ohos.file.fs` | 文件讀寫 | 無限制 | 可選 |

---

## RDB (關聯式數據庫)

| 特性 | 說明 |
|------|------|
| **引擎** | SQLite 增強版 |
| **SQL 支援** | 標準 SQL92，含 JOIN、子查詢、事務 |
| **分散式** | 支援多設備間 RDB 同步 |
| **ORM** | 原生不提供 ORM，需手寫 SQL 或自行封裝 |
| **加密** | 支援數據庫級加密 |
| **備份** | 支援雲端備份與恢復 |
| **範例** | `store.executeSql('CREATE TABLE user(...)')` |

---

## 分散式 KVStore

| 特性 | 說明 |
|------|------|
| **數據模型** | Key-Value 鍵值對 |
| **同步** | 設備間自動同步 |
| **衝突策略** | LWW / 自定義 |
| **安全級別** | NO_LEVEL / S0~S4（詳見安全章節） |
| **Schema** | 支援定義 Value Schema 進行校驗 |

---

## 網絡通訊 Network

| 協議/技術 | API | 說明 |
|-----------|-----|------|
| **HTTP/HTTPS** | `@ohos.net.http` | 標準 HTTP 請求庫，支援 GET/POST/PUT/DELETE |
| **WebSocket** | `@ohos.net.webSocket` | 雙向即時通訊 |
| **Socket** | `@ohos.net.socket` | TCP/UDP Socket 通訊 |
| **Bluetooth** | `@ohos.bluetooth` | BLE 藍牙通訊（經典+低功耗） |
| **WiFi** | `@ohos.wifi` | WiFi 連接管理 |
| **NFC** | `@ohos.nfc` | NFC 讀寫及卡模擬 |
| **MQTT** | 三方庫 | IoT 輕量級消息協議 |
| **Fetch** | `@ohos.net.fetch` | 文件下載上傳 |

---

## DataShare 跨應用數據共享

| 概念 | 說明 |
|------|------|
| **Provider** | 數據提供方，實現 DataShareExtensionAbility |
| **Consumer** | 數據消費方，透過 `createDataShareHelper()` 訪問 |
| **URI** | `datashare:///{bundleName}/{moduleName}/{storeName}` |
| **CRUD** | 支援增刪改查操作 |
| **權限** | 需要 Provider 聲明 `dataShare` 權限並授權 |
| **場景** | 通訊錄共享、日曆同步、跨應用配置 |
