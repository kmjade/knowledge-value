# 07 — 網路與連接 Networking & Connectivity

> Windows 網路設定與診斷：netsh、ipconfig、Windows 防火牆、VPN/Proxy、RDP 遠端桌面與網路疑難排解。
> netsh, wf.msc firewall, VPN/proxy configuration, RDP, and network diagnostics tools.

## 網路基本指令 Basic Network Commands

### ipconfig — IP 設定

| 指令 | 用途 |
|:-----|:-----|
| `ipconfig` | 基本 IP 資訊 |
| `ipconfig /all` | 完整設定 (MAC, DHCP, DNS) |
| `ipconfig /release` | 釋放 DHCP 租約 |
| `ipconfig /renew` | 重新取得 DHCP 租約 |
| `ipconfig /flushdns` | 清除 DNS 快取 |
| `ipconfig /displaydns` | 顯示 DNS 快取 |
| `ipconfig /registerdns` | 重新註冊 DNS |

### 常用網路診斷指令

| 指令 | 用途 | 範例 |
|:-----|:-----|:-----|
| `ping` | 連通性測試 | `ping 8.8.8.8 -t` (連續) |
| `tracert` | 路由追蹤 | `tracert google.com` |
| `pathping` | 路徑 + 延遲統計 | `pathping google.com` |
| `nslookup` | DNS 查詢 | `nslookup google.com 8.8.8.8` |
| `netstat` | 連線狀態 | `netstat -ano` (含 PID) |
| `route print` | 路由表 | `route print -4` (僅 IPv4) |
| `arp -a` | ARP 快取表 | `arp -a` |
| `Get-NetIPAddress` | PowerShell IP | `Get-NetIPAddress -AddressFamily IPv4` |
| `Test-NetConnection` | PowerShell 連線測試 | `Test-NetConnection google.com -Port 443` |

## netsh — 網路 Shell

### netsh 核心指令

| 指令 | 用途 |
|:-----|:-----|
| `netsh interface ip show config` | 顯示 IP 設定 |
| `netsh interface ip set address "乙太網路" static 192.168.1.100 255.255.255.0 192.168.1.1` | 設定靜態 IP |
| `netsh interface ip set dns "乙太網路" static 8.8.8.8` | 設定 DNS |
| `netsh wlan show profiles` | 顯示 Wi-Fi 設定檔 |
| `netsh wlan show profile "SSID" key=clear` | 顯示 Wi-Fi 密碼 |
| `netsh advfirewall show currentprofile` | 防火牆狀態 |
| `netsh winsock reset` | 重設 Winsock |
| `netsh int ip reset` | 重設 TCP/IP |
| `netsh trace start scenario=NetConnection capture=yes` | 網路追蹤 |
| `netsh trace stop` | 停止追蹤 |

## Windows 防火牆 (wf.msc)

### 設定檔 Profiles

| 設定檔 | 說明 | 適用場景 |
|:-------|:-----|:---------|
| Domain | 網域網路 | 已加入 AD 網域 |
| Private | 私人網路 | 家庭/辦公室信任網路 |
| Public | 公用網路 | 公共 Wi-Fi (最嚴格) |

### 防火牆管理指令

```cmd
# GUI
wf.msc                             # 高階安全性防火牆

# 命令列 (netsh advfirewall / PowerShell)
netsh advfirewall set allprofiles state on
netsh advfirewall set allprofiles state off
netsh advfirewall firewall add rule name="MyApp" dir=in action=allow program="C:\app.exe"
netsh advfirewall firewall delete rule name="MyApp"
```

```powershell
# PowerShell NetSecurity
Get-NetFirewallProfile
Set-NetFirewallProfile -Profile Public -Enabled True
New-NetFirewallRule -DisplayName "Allow Port 8080" -Direction Inbound -LocalPort 8080 -Protocol TCP -Action Allow
Get-NetFirewallRule | Where DisplayName -like "*8080*"
Remove-NetFirewallRule -DisplayName "Allow Port 8080"
```

### 防火牆規則類型

| 類型 | 說明 |
|:-----|:-----|
| Inbound | 傳入流量控制 |
| Outbound | 傳出流量控制 (預設允許) |
| Connection Security | IPsec 規則 |

## VPN 設定 VPN Configuration

### 內建 VPN 用戶端

| 通訊協定 | 安全性 | 備註 |
|:---------|:------:|:-----|
| IKEv2 | 高 | 支援 MOBIKE (網路切換) |
| L2TP/IPsec | 中高 | 廣泛支援 |
| PPTP | 低 (已過時) | 不建議使用 |
| SSTP | 高 | SSL/TLS over 443，不易被封鎖 |
| WireGuard | 高 | 第三方支援 (Wg客户端) |

```powershell
# 建立 VPN 連線 (PowerShell)
Add-VpnConnection -Name "CompanyVPN" -ServerAddress "vpn.example.com" `
    -TunnelType IKEv2 -EncryptionLevel Required `
    -AuthenticationMethod Eap

# 連線/中斷
Connect-VpnConnection -Name "CompanyVPN"
Disconnect-VpnConnection -Name "CompanyVPN"
```

## 遠端桌面 RDP (Remote Desktop Protocol)

### RDP 設定

| 設定位置 | 路徑 |
|:---------|:-----|
| GUI 啟用 | Settings → System → Remote Desktop |
| 傳統 | `sysdm.cpl` → Remote 分頁 |
| Registry | `HKLM\SYSTEM\CurrentControlSet\Control\Terminal Server\fDenyTSConnections` (0=允許) |
| 防火牆規則 | Remote Desktop - User Mode (TCP 3389) |

### RDP 指令

```cmd
mstsc                  # 啟動遠端桌面連線
mstsc /v:192.168.1.50  # 直接連線
mstsc /admin            # 管理員模式 (Console session)
mstsc /multimon         # 使用所有監視器
mstsc /span             # 跨監視器展開
mstsc /f                # 全螢幕
```

### 內建 RDP 限制

| 版本 | RDP Host 支援 |
|:----:|:------------:|
| Home | ❌ (僅 RDP Client) |
| Pro | ✅ (1 使用者同時) |
| Enterprise | ✅ (1 使用者同時) |
| Server | ✅ (2 使用者 + RDS 授權可擴展) |

### RDP 檔案設定 (.rdp)

```
full address:s:192.168.1.50
username:s:DOMAIN\user
administrative session:i:1
multimon:i:1
audiocapturemode:i:0
redirectclipboard:i:1
redirectprinters:i:1
```

## Proxy 設定 Proxy Configuration

| 設定方式 | 路徑 |
|:---------|:-----|
| GUI | Settings → Network & Internet → Proxy |
| 傳統 | `inetcpl.cpl` → Connections → LAN settings |
| Registry | `HKCU\Software\Microsoft\Windows\CurrentVersion\Internet Settings` |
| netsh | `netsh winhttp set proxy proxy-server="proxy:8080"` |
| PAC 腳本 | 自動設定指令碼 URL |

## 網路診斷工具 Network Diagnostics

| 工具 | 指令 | 用途 |
|:-----|:-----|:-----|
| 網路疑難排解員 | `msdt.exe /id NetworkDiagnosticsNetworkAdapter` | 自動診斷 |
| 網路重設 | Settings → Network → Network Reset | 重裝所有網路介面卡 |
| Wireshark | 第三方 | 封包擷取分析 |
| pktmon | `pktmon start --capture` | 內建封包監視 (Win 10+) |
| TCPView | Sysinternals | TCP/UDP 連線圖形化 |

### 網路卡重設流程

```cmd
ipconfig /release
ipconfig /renew
ipconfig /flushdns
netsh winsock reset
netsh int ip reset
netsh winhttp reset proxy
:: 重新開機
```
