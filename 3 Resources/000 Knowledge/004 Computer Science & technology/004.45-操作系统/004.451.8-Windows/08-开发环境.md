## 套件管理員 Package Managers

### 比較 Comparison

| 特性 | winget | Chocolatey | Scoop |
|:-----|:------:|:----------:|:-----:|
| 內建 | ✅ Win 10+ | ❌ | ❌ |
| 來源 | Microsoft Community | Community | Community |
| 安裝目錄 | 系統標準 | `C:\ProgramData\chocolatey` | `~\scoop\` |
| 管理員權限 | 有時需要 | ✅ 需要 | ❌ 不需要 (User) |
| 可攜式應用 | ❌ | ❌ | ✅ |
| YAML manifest | ✅ | ❌ | ❌ (JSON) |
| GUI | ❌ | ❌ | ❌ |

### winget 常用指令

```powershell
winget search "vscode"                     # 搜尋套件
winget install Microsoft.VisualStudioCode  # 安裝
winget list                                 # 列出已安裝
winget upgrade                              # 顯示可更新
winget upgrade --all                        # 全部更新
winget uninstall Microsoft.VisualStudioCode # 解除安裝
winget export -o packages.json              # 匯出清單
winget import -i packages.json              # 從清單還原
winget show Microsoft.VisualStudioCode      # 顯示詳細資訊
```

### Chocolatey 常用指令

```powershell
choco install vscode -y
choco upgrade all -y
choco list --local-only
choco uninstall vscode -y
```

### Scoop 常用指令

```powershell
scoop install vscode
scoop update *
scoop list
scoop uninstall vscode
scoop bucket add extras      # 新增 bucket
```

## WSL 2 開發環境 WSL 2 Development

### 安裝與設定

```powershell
# 一鍵安裝 (Win 10 2004+ / Win 11)
wsl --install

# 指定發行版
wsl --install -d Ubuntu-24.04

# 列出可用發行版
wsl --list --online
```

### WSL 2 最佳設定 (.wslconfig)

```ini
# %UserProfile%\.wslconfig
[wsl2]
memory=8GB
processors=4
swap=4GB
localhostForwarding=true
nestedVirtualization=false

[experimental]
autoMemoryReclaim=gradual
sparseVhd=true
```

### WSL 與 Windows 互通

| 操作 | Windows → WSL | WSL → Windows |
|:-----|:-------------|:-------------|
| 檔案系統 | `\\wsl$\<distro>\` | `/mnt/c/` |
| 執行指令 | `wsl <command>` | `<windows_exe>.exe` |
| 環境變數 | WSLENV 共用 | WSLENV 共用 |
| 網路 | localhost 互通 | localhost 互通 |
| VS Code | Remote - WSL 擴展 | `code .` (在 WSL 內) |

### WSLg (GUI 支援) — Win 11

```bash
# WSLg 自動啟用，無需額外設定
# 直接在 WSL 內執行 GUI 應用
nautilus &          # Linux 檔案管理員
gedit &             # 文字編輯器
code &               # VS Code (WSL 安裝的)
```

## Docker Desktop

### 安裝與設定

```powershell
winget install Docker.DockerDesktop
```

| 功能 | 說明 |
|:-----|:-----|
| WSL 2 Backend | 使用 WSL 2 輕量 VM |
| Docker Compose | 內建 |
| Kubernetes | 一鍵啟用單節點 K8s |

### Docker WSL 整合

```powershell
# 在 Docker Desktop → Settings → Resources → WSL Integration
# 啟用目標發行版

# 之後在 WSL 內直接使用 docker
docker ps
docker compose up -d
```

## Git 設定 Git Configuration

```bash
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
git config --global core.autocrlf input    # WSL/Mac: input, Windows: true
git config --global init.defaultBranch main
git config --global pull.rebase true
```

### SSH 設定 (連接 GitHub)

```powershell
# PowerShell / WSL
ssh-keygen -t ed25519 -C "your@email.com"
# 將 ~/.ssh/id_ed25519.pub 內容新增至 GitHub Settings → SSH Keys

# 測試
ssh -T git@github.com
```
