# 04 檔案系統與儲存 File Systems & Storage

> macOS 10.13 起預設檔案系統為 APFS (Apple File System)：支援快照、空間共享、原生加密、clone 複製。Time Machine 提供自動備份，iCloud Drive 與系統深度整合。

## APFS 特性 APFS Features

| 特性 Feature | 說明 | 指令/工具 |
|------|------|------|
| **Snapshots** | 時間點唯讀快照，Time Machine 底層 | `tmutil localsnapshot` |
| **Space Sharing** | 多卷共享同一 container 空間 | Disk Utility / `diskutil` |
| **Clones** | 檔案複製僅複製 metadata，寫入時才分配 | 透明無感 |
| **Encryption** | 原生 AES-XTS 全碟加密 | FileVault / `fdesetup` |
| **Crash Protection** | 寫入時複製 (CoW) 保證資料一致性 | 內建 |
| **Sparse Files** | 延遲分配空間 | 自動 |

## 磁碟管理 Disk Management

```bash
diskutil list                       # 列出所有磁碟/卷宗
diskutil info /                     # 根卷宗詳細資訊
diskutil apfs list                  # APFS container/volume 清單
diskutil verifyVolume /             # 驗證卷宗
diskutil repairVolume /             # 修復卷宗

# 建立 APFS 快照
tmutil localsnapshot                # 本機快照
tmutil listlocalsnapshots /         # 列出本機快照

# FileVault 加密
fdesetup status                     # 查看加密狀態
sudo fdesetup enable                # 啟用 FileVault
```

## Time Machine 備份 Time Machine Backup

| 操作 | 終端指令 |
|------|------|
| 列出備份目標 | `tmutil destinationinfo` |
| 開始備份 | `tmutil startbackup` |
| 排除路徑 | `tmutil addexclusion ~/Downloads` |
| 比較快照差異 | `tmutil compare` |
| 從備份恢復 | `tmutil restore /path/to/file` |

## iCloud Drive 整合 iCloud Drive Integration

```bash
# iCloud Drive 路徑
~/Library/Mobile Documents/com~apple~CloudDocs/

# 透過終端操作
ls ~/Library/Mobile\ Documents/com~apple~CloudDocs/
# 檔案狀態標記: 雲端圖示 = 僅雲端; 下載圖示 = 已下載

# 強制同步
killall bird                       # 重啟 iCloud sync daemon
brctl log -w                       # 監看同步事件
```

## 檔案系統路徑結構 Key Paths

| 路徑 Path | 說明 |
|------|------|
| `/System/Volumes/Data/` | 使用者資料卷 (讀寫) — macOS Catalina+ |
| `/System/Volumes/Preboot/` | 開機前環境 |
| `/System/Volumes/VM/` | Swap 檔案所在 |
| `~/.Trash/` | 使用者垃圾桶 |
| `/Volumes/` | 掛載點 |
