# 07 開發環境配置 Development Environment

> macOS 是開發者首選平台之一：Unix 底層 + 商業軟體生態。Xcode Command Line Tools 提供編譯器/SDK，Homebrew 管理 runtime，Docker Desktop 支援容器開發，Rosetta 2 相容 x86 工具鏈。

## 開發起手式 Quick Start

```bash
# 1. 安裝 Xcode Command Line Tools
xcode-select --install

# 2. 安裝 Homebrew (見 05-应用与包管理)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 3. 安裝基礎開發工具
brew install git gh cmake pkg-config
brew install --cask visual-studio-code

# 4. 設定 git
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
```

## 語言執行環境 Language Runtimes

| 語言 | macOS 內建 | 建議安裝方式 | 版本管理 |
|------|:---:|------|------|
| **Python** | Python 3 (系統) | `brew install python` | pyenv / conda |
| **Node.js** | ❌ | `brew install node` | nvm / fnm |
| **Go** | ❌ | `brew install go` | gvm (可選) |
| **Ruby** | Ruby (系統，舊版) | `brew install ruby` | rbenv / rvm |
| **Rust** | ❌ | `curl --proto '=https' ... -sSf https://sh.rustup.rs | sh` | rustup |
| **Swift** | ✅ (Xcode 內建) | Xcode / Xcode CLT | xcrun |

```bash
# Python 虛擬環境
python3 -m venv venv && source venv/bin/activate

# Node.js 版本管理 (fnm — 更快的 nvm)
brew install fnm
fnm install --lts && fnm use lts-latest

# Go workspace
go env GOPATH                    # ~/go
export PATH="$PATH:$(go env GOPATH)/bin"
```

## Docker Desktop 設定 Docker Desktop Setup

| 項目 | Intel Mac | Apple Silicon |
|------|------|------|
| 原生架構 | x86_64 | arm64 (Apple Silicon 原生) |
| x86 模擬 | — | Rosetta 2 (`Use Rosetta for x86_amd64 emulation`) |
| 選項路徑 | Docker Desktop → Settings → General | 勾選 Rosetta |

```bash
brew install --cask docker       # 安裝 Docker Desktop

# 檢查架構
docker info --format '{{.OSType}}/{{.Architecture}}'
# Apple Silicon: linux/arm64   Intel: linux/x86_64

# 指定平台構建
docker build --platform linux/amd64 -t myapp:x86 .
docker build --platform linux/arm64 -t myapp:arm .
```

## Rosetta 2 使用 Rosetta 2 Usage

```bash
# 查看二進位架構
file /path/to/binary               # Mach-O 64-bit executable x86_64
lipo -info /path/to/binary         # Non-fat file: x86_64

# 以 Rosetta 2 執行 x86 程式
arch -x86_64 /bin/zsh              # 開啟 x86 shell
arch -x86_64 brew install foo      # 在 x86 模式下用 Homebrew

# 檢查是否在 Rosetta 環境下
sysctl -n sysctl.proc_translated   # 1 = Rosetta, 0 = native
```

## Xcode 工具鏈 Xcode Toolchain

```bash
# Xcode CLT 路徑
xcrun --show-sdk-path              # SDK 路徑: /Applications/Xcode.app/...
xcrun --sdk macosx --show-sdk-version  # SDK 版本
xcodebuild -version                # Xcode 版本
xcode-select -p                    # 目前使用的 Xcode 路徑

# Swift 編譯
swiftc hello.swift -o hello        # 編譯 Swift
swift package init --type executable  # 建立 Swift Package
```
