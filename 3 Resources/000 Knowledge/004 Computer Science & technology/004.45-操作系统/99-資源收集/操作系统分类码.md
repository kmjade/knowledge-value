# 操作系统分类码对照

> Windows / Linux / macOS 在四大图书馆分类体系中的精确分类号。

---

## 一、操作系统总类

| 体系 | 类号 | 名称 | 说明 |
|:----:|:----:|------|------|
| **DDC** | `005.43` | Systems programs / Operating systems | 005 程序设计的子类 |
| **UDC** | `004.45` | Operating systems | 004.4 软件的子类 |
| **CLC** | `TP316` | 操作系统 | T 工业技术 → TP 计算机 |
| **LCC** | `QA76.76.O63` | Operating systems | Q 科学 → QA 数学/CS |

## 二、Windows 各版本分类

### DDC

| 类号 | 版本/范围 |
|:----:|------|
| `005.446` | Operating systems for specific types of computers — **Windows 总类** |
| `005.432` | 特定系统。**注**：DDC 标准表中 Windows 归入 005.446 |
| `005.4469` | 其他特定系统（包括 Windows 10/11） |

> **DDC 23 官方**：Microsoft Windows → `005.446`  
> 也可用 `005.43` + 标准复分表 -T1-09（历史）细化版本演进

### UDC

| 类号 | 版本 |
|:----:|------|
| `004.451` | Specific operating systems（特定操作系统总类） |
| `004.451.8Windows` | Microsoft Windows（字母扩展） |

> UDC 使用字母扩展来区分具体产品：`004.451.8Windows 10`、`004.451.8Windows 11`

### CLC（中国图书馆分类法）

| 类号 | 名称 |
|:----:|------|
| `TP316` | 操作系统（总论） |
| `TP316.7` | **Windows 操作系统** |
| `TP316.86` | Windows Server / Windows NT |
| `TP316.7` | Windows 95/98/Me/2000/XP/Vista/7/8/10/11 均归此 |

> **CLC 中文分类特色**：将 Windows 桌面版和 Server 版分开

### LCC（美国国会图书馆分类法）

| 类号 | 名称 |
|:----:|------|
| `QA76.774.M434` | Microsoft Windows（总类） |
| `QA76.774.M435` | Windows 10 |
| `QA76.774.M436` | Windows 11 |
| `QA76.76.O63` | Operating systems（通用，可用于任何 OS） |

> **LCC Cutter 号**：`.M434` 是 Microsoft Windows 的 Cutter 号码

## 四、HarmonyOS 分类详解 🆕

| 体系 | 类号 | 说明 |
|:----:|:----:|------|
| **DDC** | `005.445` | 移动操作系统（与 Android 同类） |
| **UDC** | `004.451HarmonyOS` | UDC 字母扩展（或 `004.451OpenHarmony`） |
| **CLC** | **`TP316.4`** | **分布式操作系统** ← 主流分类 |
| **CLC (第二)** | `TN929.53` | 移动通信终端（双分类号标配） |
| **LCC** | `QA76.774.H37` | Cutter 号：`.H37` 从 HarmonyOS 派生 |

### CLC 为什么用 TP316.4？

鸿蒙核心设计理念是"分布式全场景操作系统"，因此中图法分类首选 **TP316.4（分布式操作系统）**：

```
TP316  操作系统
  ├── TP316.1  分时操作系统
  ├── TP316.2  实时操作系统 (RTOS)
  ├── TP316.3  批处理操作系统
  ├── TP316.4  分布式操作系统 ← 🆕 鸿蒙
  ├── TP316.5  网络操作系统
  ├── TP316.7  Windows
  └── TP316.8  Linux
```

> 实际编目中，大多数高校图书馆对鸿蒙图书采用**双分类号**：`TP316.4` + `TN929.53`（移动通信终端），反映其"分布式 OS + 移动应用开发"的双重属性。

### OpenHarmony 变体

| 变体 | CLC | 说明 |
|------|:---:|------|
| HarmonyOS (华为商用) | `TP316.4` | 全场景分布式 OS |
| OpenHarmony (开源) | `TP316.4` | 同分类号 |
| LiteOS (轻量 IoT) | `TP316.2` | 实时操作系统 |

## 六、快速查号规则

### DDC 构建规则

```
005 — 程序设计（Computer programming, programs, data）
  └── 005.4 — 系统程序（Systems programming and programs）
       └── 005.43 — 系统程序 / 操作系统（Systems programs; operating systems）
            └── 005.432 — 特定操作系统（Specific OS, e.g. Linux, Unix）
            └── 005.446 — 特定类型计算机的 OS（e.g. Windows, macOS）
```

### CLC 构建规则

```
TP — 自动化技术、计算机技术
  └── TP3 — 计算技术、计算机技术
       └── TP31 — 计算机软件
            └── TP316 — 操作系统
                 └── TP316.7 — Windows
                 └── TP316.81 — Unix
                 └── TP316.84 — macOS
                 └── TP316.85 — Linux
                 └── TP316.86 — Windows Server
                 └── TP316.89 — 移动 OS
```
