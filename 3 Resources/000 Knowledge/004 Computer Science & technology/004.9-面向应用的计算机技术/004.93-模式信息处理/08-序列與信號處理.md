# 08 序列與信號處理

## 傅里葉變換 (Fourier Transform)
| 變換 | 域 | 公式/特性 |
|------|-----|-----------|
| **FT** (Fourier Transform) | 連續 | F(ω) = ∫ f(t)·e⁻ʲʷᵗ dt |
| **DFT** (Discrete FT) | 離散 | N 點採樣 → N 頻率分量 |
| **FFT** (Fast FT) | 離散 | O(N log N) 加速 DFT |
| **STFT** (Short-Time FT) | 時頻 | 滑窗 → 頻譜圖 (Spectrogram) |

## 小波變換 (Wavelet Transform)
與傅里葉不同: 同時提供時間和頻率分辨率。多尺度分析 (DWT): 低通→近似、高通→細節。

## 時頻分析 (Time-Frequency Analysis)
| 方法 | 特性 |
|------|------|
| **STFT / Spectrogram** | 固定分辨率、計算簡單 |
| **CWT** (Continuous Wavelet) | 可變分辨率、低頻寬高頻窄 |
| **Wigner-Ville** | 高分辨率但有交叉項干擾 |
| **Hilbert-Huang** | 非線性/非平穩信號 |

## 數字濾波器 (Digital Filters)
| 類型 | 特性 | 設計方法 |
|------|------|---------|
| **FIR** (Finite Impulse Response) | 穩定、線性相位 | 窗函數法、頻率採樣 |
| **IIR** (Infinite Impulse Response) | 低階高效、非線性相位 | Butterworth、Chebyshev、Elliptic |

## 序列建模
| 模型 | 原理 | 應用 |
|------|------|------|
| **HMM** (Hidden Markov Model) | 隱狀態馬爾可夫鏈 + 發射 | 語音、手寫、生物序列 |
| **CRF** (Conditional Random Field) | 條件概率、全局歸一化 | 序列標註 (NER、分詞) |
| **DTW** (Dynamic Time Warping) | 時間軸非線性對齊 | 簽名驗證、手勢識別 |
| **RNN / LSTM / GRU** | 深度序列建模 | 端到端替代 HMM |
| **Transformer** | 自注意力並行 | 長序列建模 |

## 應用場景: 心電圖 (ECG) 分類 · 振動分析 · 雷達信號 · 地震數據
