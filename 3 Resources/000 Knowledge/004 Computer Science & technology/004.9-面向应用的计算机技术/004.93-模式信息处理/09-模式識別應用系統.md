# 09 模式識別應用系統

## 工業檢測 (Industrial Inspection)
| 場景 | 技術 | 要求 |
|------|------|------|
| 表面缺陷檢測 | 紋理分析、異常檢測 GAN | 高精度 (ppm 級) |
| 尺寸測量 | 邊緣檢測 + 標定 | 微米級精度 |
| 裝配驗證 | 模板匹配 + 深度學習 | 實時性 |
| 印刷質量 | OCR + 顏色比對 | 字符完整性 |

## 醫學影像 (Medical Imaging)
| 模態 | 任務 | 方法 |
|------|------|------|
| X-Ray / CT | 病灶檢測、分割 | U-Net、Faster R-CNN |
| MRI | 腦區分割、腫瘤分類 | 3D CNN、nnU-Net |
| 病理切片 | 細胞檢測、分級 | 滑窗 CNN、Multiple Instance Learning |
| 眼底圖 | 糖尿病視網膜病變 | 分類 CNN、Attention |
| 超聲 | 胎兒測量、器官分割 | 實時輕量模型 |

## 自動駕駛感知 (Autonomous Driving Perception)
| 任務 | 技術 |
|------|------|
| 目標檢測 2D/3D | YOLO、PointPillars |
| 語義分割 | DeepLab、Swin Transformer |
| 車道線檢測 | LaneNet、曲線擬合 |
| 交通標誌識別 | 細粒度分類 |
| 多傳感器融合 | Camera + LiDAR + Radar 前融合/後融合 |

## 視頻監控 (Video Surveillance)
- 行人重識別 (Re-ID): 跨攝像頭身份匹配
- 異常行為檢測: 軌跡異常、人群密度
- 車牌識別 (ALPR/ANPR): 定位 → 字符分割 → 識別

## 文檔數字化 (Document Digitization)
- 表單處理: 版面分析 + OCR + 字段提取
- 票據識別: 發票、支票結構化提取
- 歷史文獻: 手寫體識別、古籍 OCR

## 系統設計考量: 延遲 (Latency) · 吞吐量 (Throughput) · 邊緣部署 (Edge) · 模型壓縮 (Quantization/Pruning)
