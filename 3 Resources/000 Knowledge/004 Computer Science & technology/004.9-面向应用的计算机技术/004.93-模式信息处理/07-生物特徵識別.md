# 07 生物特徵識別

## 指紋識別 (Fingerprint Recognition)
| 階段 | 方法 |
|------|------|
| **增強** | Gabor 濾波、方向場估計 |
| **特徵提取** | 細節點 (Minutiae): 端點、分叉點 → 位置/方向/類型 |
| **匹配** | 細節點圖對齊 + 彈性匹配、深度嵌入比對 |
| **特點** | 最成熟、普遍應用於手機解鎖和出入境 |

## 人臉識別 (Face Recognition)
| 方法 | 原理 | 代表 |
|------|------|------|
| **Eigenface** | PCA 降維 → 子空間比對 | 早期方法 |
| **Fisherface** | LDA → 最大化類間距離 | 光照魯棒 |
| **LBPH** | LBP 直方圖比對 | 紋理特徵 |
| **DeepFace / FaceNet** | CNN + Triplet Loss | 深度度量學習 |
| **ArcFace** | Additive Angular Margin | SOTA 人臉識別 |

## 虹膜識別 (Iris Recognition)
Daugman 算法: 積分微分算子定位 → 極坐標展開 → Gabor 相位編碼 (IrisCode 2048-bit) → Hamming 距離匹配。

## 聲紋識別 (Voiceprint / Speaker Recognition)
- **文本相關**: 固定短語 → GMM-UBM / i-vector / x-vector
- **文本無關**: 任意語音 → d-vector (DNN embedding)

## 多模態融合 (Multimodal Fusion)
| 層級 | 融合方式 |
|------|---------|
| **特徵級** | 串聯不同模態特徵向量 |
| **分數級** | 各模態匹配分數加權求和 (最常用) |
| **決策級** | AND/OR、多數投票 |

## 評估指標
- **FAR** (False Accept Rate) — 假接受率
- **FRR** (False Reject Rate) — 假拒絕率
- **EER** (Equal Error Rate) — FAR = FRR 時的平衡點
- **ROC / DET 曲線** — 安全性 vs 便利性權衡
