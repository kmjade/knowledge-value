# 04 圖像處理與分析
---
## 空間域濾波 (Spatial Filtering)
| 濾波器 | 核 | 效果 |
|--------|-----|------|
| **均值濾波** | 均勻權重 | 平滑去噪 |
| **高斯濾波** | Gaussian kernel | 平滑、邊緣保留 |
| **中值濾波** | 非線性 | 椒鹽噪聲去除 |
| **雙邊濾波** | 空間+顏色權重 | 保邊平滑 |

## 頻率域濾波 (Frequency Filtering)
傅里葉變換 → 頻譜 → 低通 (平滑) / 高通 (邊緣) / 帶通濾波。FFT 加速卷積。

## 邊緣檢測 (Edge Detection)
| 算子 | 原理 | 特性 |
|------|------|------|
| **Sobel** | 一階導數 (Gx, Gy) | 簡單快速 |
| **Canny** | 多階段 (梯度→NMS→雙閾值→滯後) | 最優邊緣檢測器 |
| **Laplacian** | 二階導數、零交叉 | 對噪聲敏感 |
| **LoG** (Laplacian of Gaussian) | 先高斯平滑再 Laplacian | 斑點檢測 |

## 形態學 (Morphology)
- **膨脹 (Dilation)**: 擴大前景區域
- **腐蝕 (Erosion)**: 縮小前景區域
- **開運算 (Opening)**: 先腐蝕後膨脹 → 去噪
- **閉運算 (Closing)**: 先膨脹後腐蝕 → 填洞

## 圖像分割 (Segmentation)
| 方法 | 原理 |
|------|------|
| **閾值法 (Thresholding)** | 灰度直方圖 → Otsu、自適應閾值 |
| **區域生長 (Region Growing)** | 種子點 + 相似性準則 |
| **分水嶺 (Watershed)** | 梯度圖的拓撲分割 |
| **GrabCut** | Graph Cut + 高斯混合模型 |
| **U-Net / DeepLab** | 深度語義分割 |

## 卷積 (Convolution): 核心操作 — 濾波器滑動 → 加權求和。可分離卷積降低計算量。
