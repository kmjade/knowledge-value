# 01 模式識別導論

## 定義
模式識別 (Pattern Recognition) 是從數據中自動發現規律 (Pattern) 並用於分類或描述的科學。
核心問題: 給定輸入 x，預測類別 y 或描述結構。涵蓋統計模式識別、句法模式識別、神經網絡模式識別三大流派。

## 典型系統架構
| 階段 | 功能 | 技術 |
|------|------|------|
| **傳感 (Sensing)** | 採集物理信號 → 數字數據 | 攝像頭、麥克風、傳感器 |
| **預處理 (Preprocessing)** | 降噪、標準化、增強 | 濾波、歸一化、增強 |
| **特徵提取 (Feature Extraction)** | 原始數據 → 特徵向量 | SIFT、MFCC、深度特徵 |
| **分類/回歸 (Classification)** | 特徵 → 類別標籤 | SVM、神經網絡、KNN |
| **後處理 (Post-processing)** | 修正分類結果 | CRF、上下文修正 |

## 與相關領域的關係
| 領域 | 關係 |
|------|------|
| **機器學習 (ML)** | 模式識別 = ML 在感知數據上的應用 |
| **計算機視覺 (CV)** | CV = 模式識別 + 圖像理解 + 3D |
| **自然語言處理 (NLP)** | 文本分類、序列標註 → 模式識別 |
| **信號處理** | 預處理與特徵提取的基礎 |
| **深度學習** | 端到端模式識別，自動特徵學習 |

## 模式識別系統設計原則: 數據 → 特徵 → 模型 → 評估 → 迭代
