# 03 統計模式識別
---
## 貝葉斯決策理論 (Bayes Decision Theory)
- **先驗概率 P(ωᵢ)** — 類別出現頻率
- **類條件概率 P(x|ωᵢ)** — 似然
- **後驗概率 P(ωᵢ|x)** ∝ P(x|ωᵢ) · P(ωᵢ)
- **決策規則**: 選擇最大後驗 (MAP) 或最小風險

## 參數估計
| 方法 | 思想 | 特性 |
|------|------|------|
| **MLE** (最大似然估計) | 最大化 P(D|θ) | 無偏、漸進有效 |
| **MAP** (最大後驗估計) | 最大化 P(θ|D) ∝ P(D|θ)·P(θ) | 引入先驗、避免過擬合 |
| **貝葉斯估計** | 整個後驗分佈 P(θ|D) | 完全貝葉斯、計算昂貴 |

## 判別函數 (Discriminant Functions)
- **線性判別** — g(x) = wᵀx + w₀，兩類間的線性決策邊界
- **廣義線性判別** — 多項式、RBF 核映射
- **多類策略**: One-vs-All / One-vs-One

## 核心分類器
| 分類器 | 原理 | 特點 |
|--------|------|------|
| **SVM** (Support Vector Machine) | 最大間隔超平面 + 核技巧 | 高維有效、小樣本強 |
| **KNN** (K-Nearest Neighbors) | 多數投票 | 簡單、無訓練、計算昂貴 |
| **決策樹 (Decision Tree)** | 遞歸分割 (ID3/C4.5/CART) | 可解釋、易過擬合 |
| **隨機森林 (Random Forest)** | Bagging + 隨機特徵 | 抗過擬合、並行訓練 |
| **XGBoost/LightGBM** | Gradient Boosting | 表格數據 SOTA |
| **樸素貝葉斯 (Naive Bayes)** | 特徵條件獨立假設 | 快速、高維文本 |

## 評估指標: Accuracy / Precision / Recall / F1 / ROC-AUC / Confusion Matrix
