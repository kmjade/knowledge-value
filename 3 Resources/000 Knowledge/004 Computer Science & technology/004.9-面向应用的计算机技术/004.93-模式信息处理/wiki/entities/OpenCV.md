# OpenCV

Intel 發起 (1999) 的開源計算機視覺與機器學習軟件庫。C++ 核心 + Python/Java/JS 綁定。

## 核心模塊
| 模塊 | 功能 |
|------|------|
| **core** | 矩陣運算、數據結構 |
| **imgproc** | 圖像濾波、幾何變換、形態學 |
| **features2d** | SIFT、ORB、特徵匹配 |
| **objdetect** | Haar Cascade、HOG + SVM |
| **dnn** | DNN 推理 (Caffe/TF/ONNX) |
| **video** | 光流、背景減除 |

## 地位: 計算機視覺領域最廣泛使用的基礎庫，幾乎所有 CV 項目都直接或間接依賴。
