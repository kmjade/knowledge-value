# 01 InDesign 概述與工作區 Overview & Workspace

## InDesign 定位 vs Photoshop vs Illustrator

| | InDesign | Photoshop | Illustrator |
|---|----------|-----------|-------------|
| **類型** | 排版軟體 | 點陣圖編輯 | 向量圖形 |
| **單位** | 頁面/跨頁 | 像素/圖層 | 畫板/路徑 |
| **色彩模式** | CMYK + 專色 | RGB 優先 | CMYK/RGB |
| **文字** | 專業排版引擎 | 基本文字圖層 | 路徑文字 |
| **輸出** | PDF/X、Print | TIFF/PSD/JPG | SVG/EPS/AI |

## 工作區 UI 佈局 Workspace Layout

| 區域 | 內容 | 快捷鍵 |
|------|------|:------:|
| **應用程式列** | 選單、工作區切換 | — |
| **控制面板** | 上下文相關屬性 | — |
| **工具面板（左）** | 選擇/繪圖/文字/導航工具 | `Tab` 隱藏全部面板 |
| **面板塢（右）** | 頁面/圖層/連結/樣式等 | `Shift+Tab` 隱藏右側面板 |
| **文件視窗** | 頁面本體+ pasteboard | `W` 切換預覽模式 |
| **狀態列** | 縮放/頁碼/Preflight 指示 | `Ctrl+Alt+I` 文檔資訊 |

## 工作區自訂 Workspace Customization

- **儲存工作區**: `視窗 → 工作區 → 新增工作區` — 保存面板佈局+快捷鍵
- **預設工作區**: 基本功能 / 書籍 / 互動PDF / 印刷與校樣 / 排版規則
- **CC Libraries 面板**: 跨應用程式同步顏色/字元樣式/圖形資產

## 偏好設定 Preferences（`Ctrl+K`）

| 分類 | 關鍵設定 |
|------|----------|
| **一般** | 字體預覽大小、縮圖預覽 |
| **介面** | 主題（深色/淺色）、即時螢幕繪製 |
| **文字** | 智慧文字重排、連字、預設合成器 |
| **單位與增量** | 游標鍵增量 (0.25mm)、網格間隔 |
| **顯示效能** | 典型/高品質顯示、GPU 加速 |
| **拼字檢查** | 動態拼字檢查、自訂字典 |

## 快捷鍵速查 Essential Shortcuts

| 操作 | macOS | Windows |
|------|-------|---------|
| 選擇工具 | `V` | `V` / `Esc` |
| 直接選擇 | `A` | `A` |
| 文字工具 | `T` | `T` |
| 手形平移 | `H` / `Space+拖曳` | `H` / `Space+拖曳` |
| 縮放 | `Cmd+加減` | `Ctrl+加減` |
| 符合視窗 | `Cmd+0` | `Ctrl+0` |
| 隱藏面板 | `Tab` | `Tab` |
| 預覽模式切換 | `W` | `W` |
| 框架網格顯示/隱藏 | `Cmd+Option+I` | `Ctrl+Alt+I` |
| 快速套用 | `Cmd+Return` | `Ctrl+Enter` |

## CC Libraries 跨應用同步

- **支援資產**: 顏色、字元樣式、段落樣式、圖形、文字
- **同步範圍**: InDesign ↔ Illustrator ↔ Photoshop ↔ XD
- **協作**: 共用至團隊 CC Library，協作者自動取得更新
