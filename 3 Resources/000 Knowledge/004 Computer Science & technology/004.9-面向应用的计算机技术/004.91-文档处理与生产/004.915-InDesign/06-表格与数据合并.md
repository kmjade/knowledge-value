# 06 表格與數據合併 Tables & Data Merge

## 表格操作 Tables

| 操作 | 快捷鍵/方法 |
|------|------------|
| **建立表格** | `Ctrl+Alt+Shift+T` 或文字框內 `表格 → 插入表格` |
| **選取行/欄** | 游標移至表格上邊緣或左邊緣，變箭頭後點擊 |
| **增刪行/欄** | `Ctrl+9` 刪除行，`Ctrl+Backspace` 刪除欄 |
| **合併儲存格** | 選取多儲存格 → 右鍵 → 合併儲存格 |
| **分割儲存格** | 選取儲存格 → 右鍵 → 水平/垂直分割 |
| **表格樣式面板** | `視窗 → 樣式 → 表格樣式` |

## 表格與儲存格樣式 Table & Cell Styles

| 表格樣式屬性 | 儲存格樣式屬性 |
|-------------|---------------|
| 表格邊框/填色 | 文字格式 (字體/大小/顏色) |
| 列間隔線/欄間隔線 | 儲存格內邊距 |
| 首N列/末N列格式 | 段落樣式 |
| 首N欄/末N欄格式 | 對角線 |
| 交替填色模式 (斑馬紋) | 儲存格邊框/填色 |

> 交替填色: 表格樣式 → 填色 → 交替模式 → 每 N 列。快速設定斑馬條紋。

## Data Merge 數據合併

將 CSV/TXT 數據合併至 InDesign 版面，自動產生大量變體（名片/證書/標籤/地址）。

### 工作流 Data Merge Workflow

| 步驟 | 操作 |
|------|------|
| 1. 準備數據 | CSV/TXT (UTF-8)，首列為欄位名稱 |
| 2. 載入數據 | `視窗 → 程式 → 數據合併` → 選取數據來源 |
| 3. 插入欄位 | 拖曳欄位名到版面中 |
| 4. 預覽 | 勾選「預覽」檢視合併結果 |
| 5. 建立合併文檔 | 選取記錄範圍 → 建立合併文檔 |

### Data Merge 欄位語法

| 語法 | 說明 |
|------|------|
| `<<欄位名稱>>` | 插入數據欄位 |
| `<<欄位名稱>>` + 段落樣式 | 套用自動格式 |
| 文字框內混排欄位 | 可與靜態文字混合 |

> 圖片合併: 數據檔案中用 `@圖片` 欄位，值為圖片檔案路徑（相對於數據檔或絕對路徑）。

## XML 匯入 XML Import

| 步驟 | 操作 |
|------|------|
| 1. 載入 DTD/無 | `檢視 → 結構 → 顯示結構` 後 `檔案 → 匯入 XML` |
| 2. 對應標籤 | 結構面板 → 標籤對應至樣式 |
| 3. 置入內容 | 拖曳 XML 元素至版面或使用「合併 XML」 |

> 適合: 結構化數據（產品目錄、價格表）、自動化出版流程。

## 目錄自動產生 TOC Auto-Generation

| 步驟 | 操作 |
|------|------|
| 1. 定義段落樣式 | 標題使用 H1/H2/H3 樣式 |
| 2. 建立目錄樣式 | `版面 → 目錄樣式` → 新增 |
| 3. 產生目錄 | `版面 → 目錄` → 選取樣式 → 確定 |
| 4. 置入目錄 | 載入游標後點擊或拖曳至文字框 |

### 目錄樣式關鍵設定

| 設定 | 說明 |
|------|------|
| **包含段落樣式** | 哪些樣式納入目錄 (H1, H2…) |
| **項目樣式** | 目錄中各層級對應的字元/段落樣式 |
| **頁碼位置** | 在項目前 / 在項目後 |
| **定位點前導符** | 標題與頁碼間的填充符號 (………) |

## 實用技巧

- **表格轉文字 / 文字轉表格**: `表格 → 轉換`
- **表格內 Tab**: `Ctrl+Tab` (非直接按 Tab，Tab 會跳到下一格)
- **均分欄寬**: 選取多欄 → `表格 → 均分欄`
- **Data Merge 批次輸出 PDF**: 合併時選擇匯出 PDF（非產生 InDesign 文件），比大量頁面更經濟
