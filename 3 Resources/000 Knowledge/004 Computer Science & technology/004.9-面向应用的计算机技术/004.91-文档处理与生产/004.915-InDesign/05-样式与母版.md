# 05 樣式與母版 Styles & Masters

## 樣式體系總覽 Style Types

| 樣式類型 | 面板 | 快捷鍵 | 範圍 |
|----------|------|:------:|------|
| **字元樣式** | Character Styles | `Ctrl+Shift+F11` | 選取字元 |
| **段落樣式** | Paragraph Styles | `Ctrl+F11` | 整段文字 |
| **物件樣式** | Object Styles | `Ctrl+F7` | 框架/形狀/線條 |
| **表格樣式** | Table Styles | — | 表格整體 |
| **儲存格樣式** | Cell Styles | — | 表格內儲存格 |

## 段落樣式詳解 Paragraph Styles

| 屬性分類 | 內容 |
|----------|------|
| **一般** | 樣式名稱、基於 (Based On)、下一個樣式 |
| **基本字元格式** | 字體/大小/行距/字距/語言 |
| **進階字元格式** | 水平/垂直縮放、基線位移、旋轉 |
| **縮排與間距** | 對齊/縮排/前後段間距/首字放大 |
| **定位點** | Tab 定位點設定 |
| **段落線** | 段前/段後裝飾線 |
| **分行設定** | 禁則處理 (Kinsoku)、連字 (Hyphenation) |
| **GREP 樣式** | 正則匹配自動套用字元樣式 |
| **項目符號與編號** | 清單格式、編號層級 |

## 巢狀樣式 Nested Styles

在段落樣式中定義自動字元樣式轉換規則：

| 巢狀樣式語法 | 效果 |
|-------------|------|
| `[粗體] 至 1 個 :` | 到第一個冒號為止套用粗體 |
| `[斜體] 至 1 個 em 空格` | 到全形空格為止套用斜體 |
| `[小型大寫] 2 個字` | 前 2 字套用小型大寫 |
| `[無] 至 1 個定位點字元` | 清除定位點後樣式 |

> 典型用法: 字典條目（詞條 Bold + 定義 Regular）、目錄（編號 Bold + 標題 Regular）

## 物件樣式 Object Styles

| 可定義屬性 | 說明 |
|------------|------|
| **填色** | 底色/漸層 |
| **筆畫** | 邊框粗細/樣式/顏色 |
| **文字繞排** | 預設繞排方式 |
| **段落樣式** | 自動套用到框架內文字 |
| **效果** | 陰影/羽化/透明度 |
| **框架符合選項** | 內容符合方式 |
| **轉角效果** | 圓角/倒角 |

> 選取物件 → 物件樣式面板 → 新增樣式。`Alt+Click` 樣式名稱可套用並清除覆蓋。

## 主頁設計策略 Master Page Design

| 主頁類型 | 內容 | 用途 |
|----------|------|------|
| **A-主版** | 頁碼 + 頁首/頁尾 | 內文頁通用 |
| **B-章節首頁** | 章名 + 裝飾 + 無頁碼 | 章節開頭 |
| **C-空白** | 無任何元素 | 全頁圖/特殊頁 |
| **D-欄位佈局** | 預設文字框 + 欄 | 快速開始排版 |

### 主頁關鍵操作

| 操作 | 方法 |
|------|------|
| **套用主頁** | 拖曳主頁 → 目標頁面縮圖 |
| **覆蓋主頁項目** | `Ctrl+Shift` + 點擊主頁物件 |
| **分離所有物件** | 頁面面板選單 → 覆蓋所有主頁項目 |
| **基於主頁的主頁** | 新增主頁時選「基於主頁: [A-主版]」— 繼承並可疊加 |

## 替代版面 Alternate Layouts

| 用途 | 操作 |
|------|------|
| **不同尺寸** | 同一內容 A4 / Letter / A5 版本 |
| **不同方向** | 直式 / 橫式版本 |
| **液態版面規則** | 物件 → 液態版面 → 選規則（縮放/重新置中/輔助線） |

## CC Library 樣式同步

同步至 CC Library 的樣式可跨文件、跨應用使用：

- 段落樣式/字元樣式 → 右鍵 → 新增至 CC Library
- CC Library 中的樣式拖入文件即自動建立並連結
- 來源樣式更新 → 文件內連結樣式會提示更新
