# 04 圖像與圖形 Images & Graphics

## 置入圖像 Place Images

| 操作 | 快捷鍵/方法 | 說明 |
|------|:----------:|------|
| **置入** | `Ctrl+D` | 選取檔案後載入游標，點擊或拖曳置入 |
| **多圖置入** | `Ctrl+D` + 多選 | 游標顯示縮圖預覽，可用方向鍵切換 |
| **置入至框架** | 先選框架 → `Ctrl+D` | 圖像直接填入既有框架 |
| **拖放** | 從檔案總管/Bridge 拖入 | 快速置入，無選項對話框 |

## 連結 vs 內嵌 Linked vs Embedded

| | 連結 (Linked) | 內嵌 (Embedded) |
|---|:---:|:---:|
| **檔案大小** | 小 (僅預覽) | 大 (完整資料) |
| **更新** | 原始檔修改後自動提示 | 需手動重新內嵌 |
| **可攜性** | 需附原始檔 (Package) | 獨立自足 |
| **建議** | ✅ 標準工作流 | ❌ 僅小圖示/特殊情況 |

> 連結面板 (`Ctrl+Shift+D`): 管理所有置入資源、更新連結、重新連結、編輯原稿。

## 裁剪路徑與去背 Clipping Paths

| 方法 | 適用 | 操作 |
|------|------|------|
| **Photoshop 裁剪路徑** | 已定義路徑的 PSD/TIFF | 置入時自動偵測 |
| **Alpha 通道** | 帶遮罩的圖像 | 物件 → 裁剪路徑 → Alpha 通道 |
| **偵測邊緣** | 純色背景 | 物件 → 裁剪路徑 → 偵測邊緣（調整閾值/容許度） |
| **鋼筆工具繪製** | 精確去背 | 繪製路徑 → 選取圖+路徑 → 物件 → 裁剪路徑 |

## 文字繞排 Text Wrap

| 類型 | 圖示 | 效果 |
|------|:----:|------|
| **無文字繞排** | — | 文字直接覆蓋圖像 |
| **繞邊界框** | ▭ | 文字繞矩形邊界 |
| **繞物件形狀** | ○ | 文字沿物件輪廓繞排 |
| **上下繞排** | ═ | 文字只在物件上/下方 |
| **跳物件** | ↕ | 文字跳過物件所在行 |

> 面板: `視窗 → 文字繞排` (`Ctrl+Alt+W`)。可設定偏移量 (Offset) 控制繞排距離。

## 效果面板 Effects (`Ctrl+Shift+F10`)

| 效果 | 用途 | 關鍵參數 |
|------|------|----------|
| **陰影 Drop Shadow** | 立體浮出感 | 距離/角度/模糊/擴散 |
| **內陰影 Inner Shadow** | 凹陷效果 | 同上 |
| **羽化 Feather** | 邊緣柔化 | 羽化寬度 |
| **外光暈/內光暈** | 發光效果 | 不透明度/大小/展開 |
| **斜角與浮雕** | 3D 立體按鈕 | 樣式/深度/方向 |
| **漸層羽化** | 漸進式透明 | 線性/放射狀 |

## 鋼筆工具與路徑 Pen Tool & Paths

| 工具 | 快捷鍵 | 功能 |
|------|:------:|------|
| **鋼筆工具** | `P` | 建立錨點與貝茲曲線 |
| **新增錨點** | `+` | 在路徑上增加控制點 |
| **刪除錨點** | `-` | 刪除路徑上的控制點 |
| **轉換錨點** | `Shift+C` | 角點 ↔ 平滑點切換 |
| **直接選取** | `A` | 選取/調整個別錨點與方向線 |

## 實用技巧

- **適合內容**: 框架 → `物件 → 符合 → 內容符合框架` (等比填滿/等比符合)
- **圖像品質**: 檢視 → 顯示效能 → 高品質顯示 (`Ctrl+Alt+H` 切換)
- **圖層面板**: `F7` — 管理圖形/文字堆疊順序
- **PSD 圖層顯示**: 置入 PSD 可選擇顯示/隱藏特定圖層組合（Layer Comps）
- **Pencil tool**: `N` — 自由繪製路徑，非精確貝茲曲線時使用
