# InDesign FAQ

### InDesign vs Word vs Publisher 何時用哪個？

| 工具 | 適用場景 |
|------|----------|
| **InDesign** | 精確排版、多頁書籍/雜誌、印刷輸出、CMYK 色彩管理 |
| **Word** | 簡單文檔、協作編輯、非印刷需求 |
| **Publisher** | 簡易 DM/海報、非專業級商業印刷 |

### 為何我的連結圖片變成低解析度？

InDesign 預設顯示低解析度預覽以提升效能。`檢視 → 顯示效能 → 高品質顯示` (`Ctrl+Alt+H`) 即可。

### 「溢排文字」 (Overset Text) 如何解決？

文字超過文字框容量。文字框右下角紅色 `+` 圖示。解決方法: 點擊 `+` 後點擊下一頁/下一個文字框串文，或調整文字框大小。

### 何時用 CMYK？何時用 RGB？

- **CMYK** (青品黃黑): 所有印刷輸出。在 InDesign 中新建文件時選擇「列印」意圖。
- **RGB** (紅綠藍): 螢幕顯示、數位出版 (EPUB/Publish Online/互動PDF)。
- **轉換**: `編輯 → 透明度混合空間` 控制混合時使用的色彩空間。

### 如何把 InDesign 文件給沒有 InDesign 的人看？

1. **匯出 PDF** (`Ctrl+E`): 最通用方式
2. **封裝** (`Ctrl+Alt+Shift+P`): 收集所有素材，可在另一台有 InDesign 的電腦上繼續編輯
3. **IDML**: 匯出為 `InDesign 標記語言 (IDML)`，向下相容舊版

### 專色 (Spot Color) 和 CMYK 的區別？

| | CMYK | 專色 |
|---|------|------|
| **原理** | 四色網點疊印 | 預調油墨單色 |
| **色域** | 有限（無法印金色/螢光色） | 可印金屬色/螢光色/霓虹色 |
| **成本** | 標準 | 每多一色加價 |
| **色票** | — | Pantone / DIC / TOYO |

### InDesign 文件可以在舊版開啟嗎？

`檔案 → 另存新檔 → 格式: InDesign CS4 或更早 (IDML)`，或匯出為 IDML (InDesign Markup Language)。

### GREP 查找與普通文字查找有何不同？

| 功能 | 文字查找 | GREP 查找 |
|------|:------:|:--------:|
| 精確匹配 | ✅ | ✅ |
| 模式匹配 | ❌ | ✅ (正規表達式) |
| 自動格式化 | ❌ | ✅ (GREP 樣式即時套用) |
| 學習曲線 | 低 | 中 |

### 如何加速 InDesign？

- 關閉不需要的面板
- `檢視 → 顯示效能 → 典型顯示` (非高品質)
- 停用「即時螢幕繪製」(偏好設定 → 介面)
- 連結大型 PSD/AI 而非內嵌
- 分章節儲存為書冊 (.indb) 而非單一大型文件

### 中英文混排最佳實踐？

1. **複合字體 (Composite Font)**: 中英文各自指定字體
2. **字間距**: 使用 GREP 自動在中英文字元間插入 1/4 空格
3. **標點**: 使用全形標點 (，。：；！？)
4. **數字**: 半形數字，與中文間加微間距
