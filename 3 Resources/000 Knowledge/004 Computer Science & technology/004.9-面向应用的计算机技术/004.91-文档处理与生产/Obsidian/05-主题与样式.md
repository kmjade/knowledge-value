# 05 — 主題與樣式

## Theme Marketplace

> 設定 → 外觀 → 瀏覽 → 100+ 社群主題一鍵安裝

| 主題 | 風格 | 特色 | 適合 |
|------|------|------|------|
| **Minimal** | 極簡 | 高可定制、Style Settings 深度整合、卡片式佈局 | 🏆 全能 |
| **Blue Topaz** | 華麗 | Chinese-friendly、多彩、多功能按鈕 | 中國用戶 |
| **AnuPpuccin** | 柔和 | 糖果配色、圓角設計、Rainbow 資料夾 | 視覺派 |
| **Things** | 清爽 | macOS Things 風格、乾淨排版 | Mac 用戶 |
| **Catppuccin** | 溫暖 | Catppuccin 色調、四種風味 | 配色控 |
| **Prism** | 多彩 | 高對比、色彩豐富、Graph 渲染佳 | 重度用戶 |
| **Obsidian Nord** | 暗色 | 北歐冷淡風、護眼暗色調 | 暗色愛好者 |

## 主題安裝與切換

| 步驟 | 操作 |
|:----:|------|
| 1 | 設定 → 外觀 → 主題 → 管理 |
| 2 | 瀏覽社群主題 → 搜尋 → 安裝 |
| 3 | 點擊「使用」啟用 |
| 4 | 已安裝主題保留於 `.obsidian/themes/` |
| 5 | 使用 **Style Settings** 插件微調參數 |

## CSS Snippets 自定義片段

> 不需修改主題原始碼，透過 snippet 疊加樣式。

```
.obsidian/
└── snippets/
    ├── custom-font.css      ← 自定義字型
    ├── hide-ribbon.css      ← 隱藏側邊欄
    └── wider-editor.css     ← 加寬編輯區
```

### 常用 CSS Snippets

| Snippet | 效果 | 代碼量 |
|---------|------|:-----:|
| `wider-editor` | 編輯區最大寬度 100% | 3 行 |
| `hide-ribbon` | 隱藏左側功能區 | 2 行 |
| `rainbow-folders` | 資料夾彩色圖示 | 20 行 |
| `custom-checkbox` | 自定義任務勾選框樣式 | 15 行 |
| `code-wrap` | 代碼塊自動換行 | 2 行 |
| `no-titlebar` | 隱藏標題列（與 Hider 插件配合） | 3 行 |

### Snippet 範例：加寬編輯區

```css
/* wider-editor.css */
.markdown-source-view, .markdown-preview-view {
    max-width: 100%;
    margin: 0 auto;
}
```

> 設定 → 外觀 → CSS 片段 → 重新整理 → 啟用切換。

## Style Settings 插件

> 安裝主題後，透過 Style Settings 動態調整主題參數而無需寫 CSS。

可調整的參數範例（以 Minimal 為例）：

| 分類 | 可調參數 |
|------|----------|
| **配色** | 主色、強調色、背景色、文字色 |
| **排版** | 字型、字級、行高、段落間距 |
| **佈局** | 編輯區寬度、側邊欄位置、標籤樣式 |
| **卡片** | 卡片圓角、陰影、邊框 |
| **標題** | H1-H6 字級、顏色、字重 |
| **Graph** | 節點大小、連線粗細、顏色方案 |

## 自定義主題開發流程

```
1. 基於現有主題 Fork（推薦 Minimal）
2. 建立 .obsidian/themes/MyTheme/theme.css
3. 使用 Developer Tools (Ctrl+Shift+I) 調試 CSS
4. 發布到 GitHub → 提交 Obsidian Theme Gallery
```
