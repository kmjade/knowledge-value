# 06 - Obsidian 同步与备份

## 方案对比

| 方案 | 费用 | 跨平台 | 版本历史 | 难度 |
|------|------|--------|----------|------|
| **Obsidian Sync** | $5/月 | ✅ 全平台 | ✅ 1年 | ⭐ |
| **Git** | 免费 | 桌面 | ✅ 完整 | ⭐⭐⭐ |
| **iCloud** | 免费 | Apple | ❌ | ⭐ |
| **OneDrive/Google Drive** | 免费 | 部分 | ❌ | ⭐ |
| **Syncthing** | 免费 | ✅ 全平台 | ✅ 文件级 | ⭐⭐ |
| **Remotely Save** | 免费 | ✅ 全平台 | ❌ | ⭐ |

## Obsidian Sync（官方，推荐）

```
设置 → Obsidian Sync → 登录 → 选择同步仓库
```

**优点**：
- 端到端加密
- 增量同步，速度快
- 1 年版本历史
- 选择性同步文件夹

**缺点**：$5/月

## Git 同步（技术用户首选）

```bash
cd MyVault
git init
git add .
git commit -m "init vault"
git remote add origin <your-repo>
git push -u origin main
```

### 自动备份脚本

```bash
#!/bin/bash
# ~/sync-vault.sh
cd /path/to/vault
git add -A
git commit -m "auto sync $(date '+%Y-%m-%d %H:%M')"
git push
```

### .gitignore 推荐

```
.obsidian/workspace.json
.obsidian/workspace-mobile.json
.trash/
.DS_Store
```

> 💡 配合 Obsidian Git 插件可实现自动 commit + push

## iCloud / OneDrive（简易方案）

- **macOS/iOS**：iCloud 最方便
- **Windows/Android**：OneDrive / Google Drive
- ⚠️ 避免多个网盘同时同步 `.obsidian/` 造成冲突

## Syncthing（自托管同步）

- 完全免费，P2P 同步
- 适合隐私敏感用户
- 需在每台设备安装

## 备份 3-2-1 原则

```
3 份拷贝
2 种不同介质
1 份异地存储
```

**实操方案**：

```
主要编辑 → Obsidian Sync（实时同步）
本地备份 → Git（版本控制）
异地备份 → GitHub private repo（灾难恢复）
```

## 冲突处理

| 情境 | 处理 |
|------|------|
| 出现 `.sync-conflict` 文件 | 手动合并后用 Sync 插件清除 |
| Git merge conflict | 用 VS Code 解决冲突 |
| 笔记内容丢失 | Obsidian Sync 恢复历史版本 |
