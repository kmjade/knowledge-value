# 06 — 數據管理與同步

## 同步方案總覽

| 方案 | 成本 | 平台 | 衝突處理 | 版本歷史 | 速度 |
|------|:---:|------|:--------:|:-------:|:---:|
| **Obsidian Sync** | $5/月 | 全平台 | ✅ 自動合併 | ✅ 1 年 | 🟢 極快 |
| **iCloud** | 免費~$3 | Apple 生態 | ❌ 弱 | ❌ | 🟡 中 |
| **Git** | 免費 | 全平台 | ✅ git merge | ✅ 無限 | 🟢 快 |
| **OneDrive** | 免費~$7 | Win/Mac | ❌ 衝突文件 | ✅ 30 天 | 🟡 中 |
| **Google Drive** | 免費~$10 | 全平台 | ❌ 衝突文件 | ✅ 30 天 | 🔴 慢 |
| **Syncthing** | 免費 | 全平台 | ⚠️ 簡單 | ✅ 版本 | 🟢 快 |
| **Dropbox** | 免費~$10 | 全平台 | ⚠️ 簡單 | ✅ 30 天 | 🟢 快 |

## Obsidian Sync 詳解

| 特性 | 說明 |
|------|------|
| **E2E 加密** | AES-256 端到端加密，Obsidian 團隊也看不到你的數據 |
| **增量同步** | 只傳送變更部分，非同步整個 Vault |
| **選擇性同步** | 可按資料夾排除（如 `.obsidian/workspace.json`） |
| **版本歷史** | 保留 1 年歷史版本，可恢復任意時間點 |
| **衝突合併** | 自動偵測衝突並合併 Markdown 文本 |
| **多人共用** | 可建立 Shared Vault（Beta） |

> 設定 → Sync → 登入 Obsidian 帳號 → 選擇 Remote Vault → 設定同步規則。

## Git 同步方案

```bash
# Vault 根目錄初始化
git init
echo ".obsidian/workspace.json" >> .gitignore
echo ".obsidian/workspace-mobile.json" >> .gitignore
echo ".trash/" >> .gitignore

# 使用 Obsidian Git 插件自動提交
# 設定 → 社群插件 → Obsidian Git → 設定自動備份間隔
```

### Git vs Obsidian Sync

| 比較 | Git | Obsidian Sync |
|------|-----|---------------|
| **費用** | 免費 | $5/月 |
| **自動化** | Obsidian Git 插件 | 內建 |
| **移動端** | ⚠️ 需要 Working Copy (iOS) | ✅ 原生支援 |
| **衝突合併** | ✅ git merge | ✅ 自動合併 |
| **設定複雜度** | 🟠 中高 | 🟢 低 |
| **版本控制** | ✅ 完整 Git 歷史 | ✅ 1 年快照 |
| **協作** | ✅ PR / branch | ❌ 個人用 |
| **隱私** | 取決於 remote（GitHub 公開） | ✅ E2E 加密 |

## iCloud / OneDrive / Google Drive 對比

| 特性 | iCloud | OneDrive | Google Drive |
|------|:------:|:--------:|:------------:|
| **Apple 生態** | ✅ 原生 | ⚠️ 可 | ⚠️ 可 |
| **隱私** | 🔒 E2E 可選 | ⚠️ 無 E2E | ❌ 無 E2E |
| **衝突處理** | ❌ 易衝突 | ❌ 產生衝突文件 | ❌ 覆蓋 |
| **iOS 支援** | ✅ Obsidian 原生 | ⚠️ 需手動 | ⚠️ 需手動 |
| **建議** | ✅ Mac+iOS | ❌ 不建議 | ❌ 不建議 |

> ⚠️ **雲盤同步目錄的風險**：頻繁寫入 `.md` 檔案可能觸發雲盤的檔案鎖定/衝突機制，導致資料損失。建議只用於單向備份而非雙向同步。

## 附件管理 Attachment Management

| 設定 | 建議值 | 說明 |
|------|--------|------|
| **附件資料夾** | `Attachments/` | 集中管理，避免散落各處 |
| **鏈接格式** | `相對路徑` | 可攜性最佳，換電腦不移位 |
| **自動更新鏈接** | ✅ 開啟 | 移動檔案時自動更新 `[[鏈接]]` |
| **附件副檔名檢測** | 全部開啟 | 拖入圖片/PDF/音訊自動歸位 |

```
MyVault/
├── Attachments/          ← 所有圖片/附件集中於此
│   ├── screenshot-2026.png
│   └── diagram.excalidraw
├── 筆記.md               ← 引用 ![[Attachments/screenshot-2026.png]]
```

## 備份策略 Backup Strategy

| 策略 | 頻率 | 工具 | 說明 |
|------|------|------|------|
| **即時** | 每次變更 | Git + Obsidian Git 插件 | 版本控制 + 備份 |
| **每日** | 24h | 定時備份到外接硬碟 | `rsync` 或備份軟體 |
| **每週** | 7d | 打包壓縮上傳雲盤 | `tar czf vault.tar.gz MyVault/` |
| **異地** | 每月 | 雲端冷存儲 | S3 / Backblaze B2 |
| **3-2-1 原則** | — | 3 份拷貝、2 種媒介、1 份異地 | 黃金標準 |
