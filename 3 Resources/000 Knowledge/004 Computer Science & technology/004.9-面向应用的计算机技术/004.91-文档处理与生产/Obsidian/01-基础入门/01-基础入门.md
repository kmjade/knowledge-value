# 01 - Obsidian 基础入门

## 下载安装

| 平台 | 方式 |
|------|------|
| **Windows** | [官网下载](https://obsidian.md/download) .exe 安装包 |
| **macOS** | 官网下载 .dmg 或 `brew install --cask obsidian` |
| **Linux** | AppImage 或 `sudo snap install obsidian` |
| **iOS** | App Store 搜索 Obsidian |
| **Android** | Google Play 或 [官网 APK](https://obsidian.md/android) |

## 创建第一个仓库

1. 打开 Obsidian → 点击「创建新仓库」
2. 选择一个**空文件夹**作为仓库位置
3. 给仓库起个名字（如 `My Vault`）
4. 点击「创建」

> ⚠️ 仓库位置建议选在本地硬盘（非网盘同步目录），避免同步冲突。

## 界面概览

```
┌─────────────────────────────────────────┐
│ 左侧边栏  │     编辑区      │ 右侧边栏   │
│           │                 │           │
│ 📁 文件   │   Markdown      │ 🔗 反向链接│
│ 🔍 搜索   │   笔记内容      │ 🏷️ 标签    │
│ ⭐ 书签   │                 │ 📋 大纲    │
│           │                 │           │
├───────────┴─────────────────┴───────────┤
│              状态栏                      │
└─────────────────────────────────────────┘
```

**快捷键打开侧边栏**：
- `Ctrl+Shift+F` — 搜索
- 点击左侧「📁」— 文件列表
- 点击右侧「🔗」— 反向链接

## 命令面板

`Ctrl+P`（Mac: `Cmd+P`）打开命令面板，可以搜索执行所有操作：

```
Ctrl+P → 输入 "新建笔记" → Enter
Ctrl+P → 输入 "切换主题" → Enter
Ctrl+P → 输入 "图谱" → Enter
```

> 不记得快捷键？`Ctrl+P` 解决一切。

## 核心快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建笔记 |
| `Ctrl+O` | 快速打开笔记 |
| `Ctrl+P` | 命令面板 |
| `Ctrl+E` | 切换编辑/阅读模式 |
| `Ctrl+Shift+F` | 全局搜索 |
| `Ctrl+G` | 打开图谱视图 |
| `Ctrl+,` | 打开设置 |
