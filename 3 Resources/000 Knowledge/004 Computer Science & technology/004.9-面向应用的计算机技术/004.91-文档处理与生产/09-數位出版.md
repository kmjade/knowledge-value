# 09 數位出版

## 定義
數位出版 (Digital Publishing) 將文檔**以數位格式發佈**至電子閱讀器、Web 或印刷機。涵蓋電子書、靜態網站生成、CMS 出版與印刷生產流。

## 電子書格式詳解 eBook Formats Deep Dive
| 特徵 | EPUB 3 | MOBI/KF8 | PDF |
|------|------|------|------|
| 排版 | 回流式 (Reflowable) | 回流式 | 固定版面 |
| DRM | Adobe DRM / LCP | Amazon DRM | 可選 |
| 無障礙 | 原生支援 (WCAG) | 有限 | PDF/UA |
| 互動 | JS + HTML5 | KF8 有限 | 有限 |
| 市場 | 全球標準 (除 Kindle) | Amazon 獨佔 | 固定版面需求 |

### EPUB 內部結構 Internal Structure
```
book.epub (ZIP container)
├── mimetype                (application/epub+zip)
├── META-INF/
│   └── container.xml       (根檔案指向)
├── OEBPS/
│   ├── content.opf         (清單/spine/中繼資料)
│   ├── toc.ncx             (目錄 — EPUB 2)
│   ├── nav.xhtml           (目錄 — EPUB 3)
│   ├── chapter1.xhtml
│   └── styles.css
```

## 靜態網站生成器 Static Site Generators (SSG)
| 工具 | 語言 | 特點 |
|------|------|------|
| Hugo | Go | 極速建構 (ms 級)、豐富主題 |
| Jekyll | Ruby | GitHub Pages 原生支援 |
| Next.js (SSG mode) | JS/React | 混合 SSG + SSR |
| Astro | JS | 零 JS 輸出 (默認) |
| MkDocs / Docusaurus | Python/JS | 技術文檔專用 |

### SSG 出版流
```
Markdown 內容 + 模板 + 靜態資源
         │
         ▼
   SSG 建構 (Hugo/Jekyll)
         │
         ▼
   靜態 HTML/CSS/JS
         │
         ▼
   CDN 部署 (Netlify/Vercel/GitHub Pages)
```

## CMS 內容管理系統 Content Management Systems
| 平台 | 類型 | 適用場景 |
|------|------|------|
| WordPress | 傳統 CMS (PHP/MySQL) | 部落格、企業網站 (43% Web) |
| Ghost | 無頭 CMS (Node.js) | 會員制出版 |
| Strapi | 無頭 CMS (Node.js) | API-first、客製前端 |
| Contentful | SaaS 無頭 CMS | 企業內容基礎設施 |

## 印刷生產流程 Print Production Workflow
```
定稿文檔 (InDesign/LaTeX)
    │
    ▼
預檢 (Preflight) — 字型/圖像/出血檢查
    │
    ▼
PDF/X-4 輸出 — CMYK + ICC + 出血
    │
    ▼
拼版 (Imposition) — 印刷機版面排列
    │
    ▼
打樣 (Proof) — 軟打樣/硬打樣確認
    │
    ▼
製版 (Plate making) → 印刷 (Print) → 裝訂 (Binding)
```
