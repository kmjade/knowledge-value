# 05 文檔格式標準

## 定義
文檔格式標準 (Document Format Standards) 定義文檔的**結構化表示方式**，確保跨軟體、跨平台、跨時間的互通性。分為辦公文檔、技術文檔、電子書、歸檔等類別。

## ODF vs OOXML — 格式大戰 The Format War
| 維度 | ODF (OpenDocument) | OOXML (Office Open XML) |
|------|------|------|
| 標準 | ISO/IEC 26300 (2006) | ISO/IEC 29500 (2008) |
| 主要支持 | LibreOffice / Apache OpenOffice | Microsoft Office 2007+ |
| 檔案結構 | ZIP + XML (manifest.xml, content.xml) | ZIP + XML (更複雜的部件) |
| 擴展名 | .odt (文字) .ods (試算) .odp (簡報) | .docx / .xlsx / .pptx |
| 開放性 | 完全開放、社群驅動 | MS 主導、向後相容遺留格式 |
| 政治 | 歐盟/政府推薦 | 企業默認 |

## 技術文檔標準 Technical Documentation Standards
| 標準 | 類型 | 用途 |
|------|------|------|
| DocBook (OASIS) | XML/SGML | 技術書籍/手冊 — O'Reilly 使用 |
| DITA (Darwin Information Typing Architecture) | XML | 模組化技術文檔（IBM 開發） |
| S1000D | XML | 航空/國防技術出版物 |
| Markdown + Pandoc | 純文本 | 輕量技術文檔管道 |

## 電子書格式 eBook Formats
| 格式 | 基礎 | 特性 |
|------|------|------|
| EPUB 3.x | (X)HTML5 + CSS + ZIP | 回流式 (reflowable)、無障礙 (A11Y) |
| MOBI / KF8 | Palm 數據庫 / HTML5 | Amazon Kindle 專有 |
| PDF | PostScript | 固定版面 (fixed-layout) |

## 歸檔格式 Archival Formats
| 格式 | 標準 | 用途 |
|------|------|------|
| PDF/A-1/2/3/4 | ISO 19005 | 長期數位保存 (禁止外部依賴) |
| PDF/UA | ISO 14289 | 無障礙 PDF (螢幕閱讀器) |
| EPUB 3 (A11Y) | W3C WCAG | 無障礙電子書 |

## 格式選擇決策矩陣 Format Decision Matrix
| 場景 | 推薦格式 | 原因 |
|------|------|------|
| 企業辦公 | DOCX / ODT | 普遍相容 |
| 長期保存 | PDF/A / ODF | ISO 標準、開放 |
| 技術寫作 | DocBook / Markdown | 版本控制/自動化 |
| Web 發佈 | HTML / Markdown→SSG | 原生 Web |
| 電子書 | EPUB 3 | 回流式、無障礙 |
| 印刷 | PDF/X-4 | CMYK + 出血 |
