# Microsoft Word

## 定義
Microsoft Word 是全球使用最廣泛的 WYSIWYG 字處理軟體，1983 年首次發佈 (MS-DOS)，1989 年 Windows 版確立市場主導地位。屬於 Microsoft Office / Microsoft 365 套件核心組件。

## 演進歷程
| 版本 | 年代 | 關鍵 |
|------|:--:|------|
| Word for DOS | 1983 | 滑鼠支援的文字處理先驅 |
| Word for Windows 2.0 | 1991 | 開始取代 WordPerfect |
| Word 97 | 1997 | 引入 .doc 格式/Clippy |
| Word 2007 | 2007 | OOXML (.docx) + Ribbon UI |
| Word 2016+ | 2016 | 雲端協作、即時共同編輯 |
| Word Copilot | 2023 | AI 輔助寫作/摘要 |

## 核心格式: .docx (Office Open XML — ZIP + XML)。核心特性: 追蹤修訂、合併列印、VBA 巨集、Ribbon UI、Copilot AI 整合。

## 市場地位: 企業辦公標準 (90%+ 市佔率)、法律/政府/教育主導字處理器。競爭者: Google Docs (雲端)、LibreOffice Writer (開源)、Apple Pages (Apple 生態)。
