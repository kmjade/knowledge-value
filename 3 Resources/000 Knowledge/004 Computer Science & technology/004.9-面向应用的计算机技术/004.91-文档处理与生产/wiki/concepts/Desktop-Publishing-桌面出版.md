# 桌面出版 Desktop Publishing (DTP)

## 定義
桌面出版 (DTP) 是使用專業軟體將文字、圖像與圖形組合為精確印刷版面的技術。不同於字處理 (以內容流為主)，DTP 以**頁面為單位**精確控制每個元素的位置、色彩、出血。

## 核心內容
| 工具 | 類型 | 市場定位 |
|------|------|------|
| Adobe InDesign | 商業旗艦 | 專業出版標準 (雜誌/書籍/目錄) |
| QuarkXPress | 商業 | 出版業元老 (1987) |
| Scribus | 開源 | 免費跨平台 DTP |
| Affinity Publisher | 商業平價 | Adobe 替代品 (一次性購買) |
| LaTeX (排版視角) | 學術 | 結構化自動排版 |

## DTP 工作流
```
文字創作 (Word/Markdown) → 圖像準備 (PS/AI/GIMP) → 版面設計 (InDesign/Scribus)
    → 預檢 (Preflight: 字型/圖像/色彩) → PDF/X-4 輸出 → 印刷
```

## 核心概念
| 概念 | 說明 |
|------|------|
| CMYK | 印刷四原色 (Cyan, Magenta, Yellow, Key-black) |
| 出血 (Bleed) | 超出裁切線 3-5mm 的延伸區域 |
| ICC 色彩設定檔 | 跨設備色彩一致性 |
| 網點 (Halftone) | 模擬連續色調的印刷技術 |
| 主頁 (Master Page) | 統一套用頁碼/頁首/頁尾的模板 |
