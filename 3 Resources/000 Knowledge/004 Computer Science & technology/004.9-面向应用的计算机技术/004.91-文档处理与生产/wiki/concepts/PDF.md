# PDF Portable Document Format

## 定義
PDF (Portable Document Format) 是 Adobe 於 1993 年開發的跨平台頁面描述語言，2008 年成為 ISO 32000 國際標準。設計目標: 在任何裝置上忠實呈現文檔外觀。

## 核心內容
| 版本里程碑 | 年份 | 關鍵 |
|------|:--:|------|
| PDF 1.0 | 1993 | Adobe 推出，取代 PostScript 的發佈格式 |
| PDF 1.7 | 2006 | ISO 32000-1 基準 |
| PDF 2.0 | 2017 | ISO 32000-2; UTF-8、AES-256 |

| 專業子集 | 用途 |
|------|------|
| PDF/A | 長期歸檔 (禁止外部依賴/動態內容) |
| PDF/X | 印刷交換 (CMYK/ICC/出血) |
| PDF/UA | 無障礙 (Universal Accessibility) |
| PDF/E | 工程圖交換 |
| PDF/VT | 可變數據印刷 (帳單) |

## PDF 技術棧: PostScript 基礎 → 頁面描述語言 → ISO 32000 → 子集特化 → Web PDF (PDF.js/Mozilla)

## 核心特性: 字型嵌入 (跨平台字型一致性)、透明度 (1.4+)、數位簽章 (PKI/PAdES)、無障礙標籤 (PDF/UA)、表單 (AcroForms/XFA)
