# 07 — 工作流與模板

## Daily / Weekly Notes 週期筆記

> 核心插件 **Daily Notes** + 社群插件 **Periodic Notes** 實現。

| 類型 | 模板變數 | 範例路徑 | 頻率 |
|------|----------|----------|:---:|
| **Daily** | `{{date}}` | `Journal/YYYY/MM/YYYY-MM-DD.md` | 每日 |
| **Weekly** | `{{date:gggg-[W]ww}}` | `Journal/Weekly/gggg-[W]ww.md` | 每週 |
| **Monthly** | `{{date:YYYY-MM}}` | `Journal/Monthly/YYYY-MM.md` | 每月 |
| **Quarterly** | `{{date:YYYY-[Q]Q}}` | `Journal/Quarterly/YYYY-[Q]Q.md` | 每季 |
| **Yearly** | `{{date:YYYY}}` | `Journal/Yearly/YYYY.md` | 每年 |

### Daily Note 標準模板

```markdown
---
date: <% tp.date.now("YYYY-MM-DD") %>
tags: [daily]
---

# 📅 <% tp.date.now("YYYY-MM-DD (dddd)") %>

## 🎯 今日三件事
- [ ] 
- [ ] 
- [ ] 

## 📝 筆記
-

## 📊 今日時間紀錄
| 時段 | 任務 | 狀態 |
|------|------|:---:|
| 上午 | | |
| 下午 | | |

## 🔗 相關
- [[<% tp.date.now("YYYY-MM-DD", -1) %>|← 昨天]]
- [[<% tp.date.now("YYYY-MM-DD", 1) %>|明天 →]]
```

## PARA 工作流

> Tiago Forte 的 PARA 方法論在 Obsidian 中的落地。

| P.A.R.A | 資料夾 | 說明 | 範例 |
|:-------:|--------|------|------|
| **P**rojects | `1-Projects/` | 有截止日期的短期任務 | `撰寫 Q2 報告` |
| **A**reas | `2-Areas/` | 持續維護的責任領域 | `健康` `財務` |
| **R**esources | `3-Resources/` | 感興趣的主題倉庫 | `程式設計` `攝影` |
| **A**rchives | `4-Archives/` | 已完成/不活躍項目 | `2025 報稅` |

```
Vault/
├── 1-Projects/
├── 2-Areas/
├── 3-Resources/
├── 4-Archives/
├── Inbox/              ← 快速捕獲（待分類）
└── Journal/            ← 日記
```

| 操作 | 頻率 | 工具 |
|------|:---:|------|
| **捕獲** | 隨時 | QuickAdd → Inbox |
| **分類** | 每日 | 手動移動到 P/A/R/A |
| **回顧** | 每週 | Dataview 查詢 overdue 任務 |
| **歸檔** | 專案完成時 | 移到 Archives/ |

## Zettelkasten 卡片盒筆記法

> Niklas Luhmann 的方法論在 Obsidian 的實現。

| 概念 | Obsidian 實現 |
|------|--------------|
| **原子筆記 Atomic Note** | 一篇 `.md` 只講一個概念 |
| **連結 Link** | `[[概念A]]` `[[概念B]]` 建立關聯 |
| **索引 Index** | MOC 筆記，匯集相關筆記的起點 |
| **序列 Folge** | 使用 `[[筆記1]] → [[筆記2]] → [[筆記3]]` 鏈條 |
| **ID 命名** | `202605302315 卡片標題`（時間戳唯一 ID） |

### Zettelkasten vs PARA

| 維度 | Zettelkasten | PARA |
|------|:-----------:|:----:|
| **粒度** | 極細（一個概念一張卡） | 中等（專案/領域） |
| **組織** | 扁平 + 鏈接 | 層級資料夾 |
| **適合** | 研究、寫作、深度思考 | 生活、工作、專案管理 |
| **搜索** | 遍歷鏈接 | 資料夾定位 |
| **維護成本** | 🟠 中高 | 🟢 低 |

## Templater 模板引擎

> 社群插件 #2，超越核心模板插件的高級模板。

| 功能 | 語法 | 範例 |
|------|------|------|
| **日期** | `<% tp.date.now("YYYY-MM-DD") %>` | `2026-05-30` |
| **檔案創建** | `<% tp.file.create_new() %>` | 動態建立新筆記 |
| **游標位置** | `<% tp.file.cursor() %>` | 插入後游標停在這裡 |
| **系統剪貼簿** | `<% tp.system.clipboard() %>` | 貼上剪貼簿內容 |
| **用戶輸入** | `<% tp.system.prompt("標題?") %>` | 彈出對話框 |
| **JS 執行** | `<% ... %>` | 任意 JavaScript |

## QuickAdd 巨集與自動化

| 功能 | 說明 |
|------|------|
| **Template Choice** | 選擇模板快速建立筆記 |
| **Capture Choice** | 快速捕獲文字到指定筆記 |
| **Macro Choice** | 執行一系列命令的巨集 |
| **Multi Choice** | 組合以上多個操作 |

### QuickAdd 常見自動化範例

| 場景 | QuickAdd 設定 |
|------|--------------|
| 快速日記 | Capture → append to Daily Note |
| 新專案 | Template → 建立於 `1-Projects/` |
| 讀書筆記 | Template + prompt 輸入書名 |
| 會議記錄 | Macro → 建立筆記 + 插入日期 + 連結到專案 |

## Dataview 自動化查詢

### 常用查詢模板

```dataview
TABLE file.cday as "建立日期", status as "狀態"
FROM "1-Projects"
WHERE status != "done"
SORT file.cday DESC
```

```dataview
TASK
FROM "1-Projects"
WHERE !completed
GROUP BY file.link
SORT rows.file.ctime ASC
```

```dataview
LIST
FROM #idea AND -#archived
SORT file.mtime DESC
LIMIT 10
```
