# 07 - Obsidian 主题定制

## 推荐主题

| 主题 | 风格 | 适合 |
|------|------|------|
| **Minimal** | 极简 | 所有人（首推） |
| **Things** | macOS 风格 | Apple 用户 |
| **Blue Topaz** | 中式优雅 | 中文用户 |
| **AnuPpuccin** | 彩虹色系 | 活泼风格 |
| **Catppuccin** | 柔和配色 | 暗色模式爱好者 |
| **Dracula** | 经典暗色 | 程序员 |
| **Primary** | Material Design | 清爽现代 |

## 安装主题

```
设置 → 外观 → 主题 → 管理 → 搜索主题名 → 安装并使用
```

## 自定义 CSS

`.obsidian/snippets/` 下创建 `.css` 文件，设置中启用。

### 常用片段

```css
/* === 加宽编辑区 === */
.markdown-source-view.mod-cm6 .cm-content,
.markdown-reading-view .markdown-preview-view {
    max-width: 900px;
}

/* === 隐藏不用的 UI === */
.status-bar { display: none; }
.mod-left-split, .mod-right-split { min-width: 250px; }

/* === 美化链接 === */
.cm-hmd-internal-link, .internal-link {
    color: #4a9eff;
    text-decoration: none;
}

/* === 美化 callout === */
.callout {
    border-radius: 8px;
    border-left: 4px solid;
}

/* === 更舒适的行距 === */
.markdown-source-view .cm-line {
    line-height: 1.8;
}

/* === 自定义字体 === */
body {
    --font-text: "LXGW WenKai", "PingFang SC", sans-serif;
    --font-monospace: "JetBrains Mono", monospace;
}
```

## 中文字体推荐

| 字体 | 风格 | 免费 |
|------|------|------|
| **霞鹜文楷** | 楷体，优雅 | ✅ |
| **思源黑体** | 黑体，现代 | ✅ |
| **更纱黑体** | 等宽，编程 | ✅ |
| **霞鹜致宋** | 宋体，正式 | ✅ |

## 主题调参

安装 Style Settings 插件 → 打开设置 → Style Settings → 微调主题参数：

- 配色方案
- 字体大小
- 界面密度
- 图标样式
- 自定义 CSS

## 我的配色方案

Minimal 主题 + Style Settings：

```
Color scheme: Nord
Font size: 16px
Line width: 800px
Sidebar width: 280px
```
