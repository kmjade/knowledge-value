# Obsidian — 本地優先的知識操作系統

> Obsidian is a **local-first, plain Markdown** knowledge base that thinks like your brain — bidirectional links, graph view, and 2000+ plugins.
> 由 Shida Li & Erica Xu 創建，2020 年首發。個人使用完全免費。

## 核心理念 Philosophy

| 原則 | 說明 |
|------|------|
| **Local-first** | 數據 100% 在你手上，純 Markdown 檔案，無 vendor lock-in |
| **Plain Markdown** | 任何編輯器都能打開，無專有格式 |
| **Bidirectional Links** | `[[wikilink]]` 構建知識網絡，反向鏈接自動追蹤 |
| **Extensible** | 53+ 核心插件 + 2000+ 社群插件，高度可定制 |
| **Offline-first** | 無需網路，秒開秒寫，性能卓越 |

## 快速上手 Quick Start

```bash
# 1. 下載安裝
https://obsidian.md/download

# 2. 創建 Vault → 選擇空資料夾
# 3. 寫第一份筆記 → Ctrl+N
# 4. 建立連結 → [[筆記名]]
# 5. 打開命令面板 → Ctrl+P
# 6. 安裝必備插件 → 設定 → 社群插件 → 關閉安全模式
```
