# 02 — Obsidian 核心概念

## Wikilinks `[[]]` — 雙向鏈接

> Obsidian 的靈魂功能。每個 `[[筆記名]]` 自動建立雙向關係。

| 語法 | 效果 | 用例 |
|------|------|------|
| `[[筆記名]]` | 建立鏈接 | `[[PARA Method]]` |
| `[[筆記名\|顯示文字]]` | 自定義顯示 | `[[PARA Method\|PARA 方法論]]` |
| `[[筆記名#標題]]` | 鏈接到段落 | `[[PARA#Projects]]` |
| `[[筆記名#^block-id]]` | 鏈接到區塊 | `[[日記#^todays-todo]]` |
| `![[筆記名]]` | 嵌入全文 | `![[會議記錄]]` |
| `![[筆記名#標題]]` | 嵌入段落 | `![[設計稿#原型圖]]` |

> **雙向 = 我鏈你 + 你知道我鏈了你**。右側邊欄 → 反向鏈接面板 → 誰鏈了這篇筆記一覽無遺。

## Graph View 圖譜視圖

| 類型 | 快速鍵 | 說明 |
|------|--------|------|
| **全域圖譜** | `Ctrl+G` → 無過濾 | 整個 Vault 的知識網絡可視化 |
| **局部圖譜** | 右側邊欄 → 局部圖譜 | 僅顯示當前筆記的直接關聯 |
| **深度調整** | 圖譜設定 → 深度 | 控制顯示幾層連結（1-5） |
| **分組著色** | 圖譜設定 → Groups | 按 tag/path/name 給節點上色 |

```
       [筆記A] ──── [筆記B]
         │            │
         └── [筆記C] ─┘
              │
         [筆記D]
```

## Tags 與 Aliases

### Tags 標籤系統

```markdown
#tag                   單層標籤
#pkm/obsidian          嵌套標籤（子標籤）
```

| 特性 | Tags | Links |
|------|------|-------|
| **用途** | 分類/狀態標記 | 概念關聯 |
| **語法** | `#tag` | `[[筆記]]` |
| **反向查詢** | 標籤面板 | 反向鏈接面板 |
| **巢狀支援** | `#parent/child` | ❌ |
| **範例** | `#draft` `#pkm/tool` | `[[Obsidian]]` |

### Aliases 別名

```yaml
---
aliases: [PKM, 個人知識管理, Personal Knowledge Management]
---
```

同一篇筆記可以用多個名字引用。`[[PKM]]`、`[[個人知識管理]]` 都指向同一篇。

## YAML Frontmatter

```yaml
---
title: 筆記標題
aliases: [別名1, 別名2]
tags: [tag1, tag2]
created: 2026-05-30
updated: 2026-05-30
status: draft
---
```

| 常用欄位 | 說明 | Dataview 可查詢 |
|----------|------|:--------------:|
| `title` | 筆記標題 | ✅ |
| `aliases` | 別名列表 | ✅ |
| `tags` | 標籤 | ✅ |
| `created` | 建立日期 | ✅ |
| `updated` | 更新日期 | ✅ |
| `status` | 狀態（draft/review/done） | ✅ |
| `cssclass` | 自定義 CSS class | ❌ |

## Dataview 基礎查詢

> Dataview 是 Obsidian 的查詢引擎，將 Vault 變成可查詢的數據庫。

| 查詢類型 | 語法 | 範例 |
|----------|------|------|
| **LIST** | `LIST FROM #tag` | 列出所有 `#pkm` 筆記 |
| **TABLE** | `TABLE field1, field2 FROM "folder"` | 表格顯示 + 排序 |
| **TASK** | `TASK FROM "Projects"` | 列出所有未完成任務 |
| **CALENDAR** | `CALENDAR file.cday` | 依建立日期顯示日曆 |
| **WHERE** | `WHERE status = "draft"` | 過濾條件 |

```dataview
TABLE aliases, created, status
FROM #obsidian
SORT created DESC
WHERE status = "draft"
LIMIT 10
```

## Search 搜尋語法

| 運算子 | 語法 | 說明 |
|--------|------|------|
| 關鍵詞 | `obsidian` | 全文搜尋 |
| 精確 | `"知識管理"` | 精確短語 |
| **path:** | `path:Projects` | 限定目錄 |
| **tag:** | `tag:#obsidian` | 限定標籤 |
| **file:** | `file:學習` | 檔名匹配 |
| **line:** | `line:(TODO)` | 行內容搜尋 |
| **section:** | `section:(## 目標)` | 章節標題 |
| **task:** | `task:""` | 任務狀態 |
| **task-todo:** | `task-todo:""` | 未完成任務 |
| **AND / OR** | `obsidian AND pkm` | 邏輯組合 |
| **括號** | `(A OR B) AND C` | 分組 |
| **Regex** | `/正則表達式/` | 正則搜尋 |

## Command Palette 命令面板

`Ctrl+P`（Mac: `Cmd+P`）— 不記快捷鍵也不怕，一切皆可搜。

| 常用命令 | 關鍵字 |
|----------|--------|
| 新建筆記 | `new note` |
| 切換編輯/預覽 | `toggle edit` |
| 打開圖譜 | `graph` |
| 切換主題 | `theme` |
| 插入模板 | `template` |
| 全域搜尋 | `search` |
| 重整快取 | `reload` |

> 💡 命令面板支援模糊搜尋，輸入 `tn` 就能找到 `Toggle Note`。
