# 字處理器 Word Processor

## 定義
字處理器 (Word Processor) 是一種以 WYSIWYG (所見即所得) 為核心範式的文檔創作軟體，讓使用者在螢幕上直接看到最終輸出效果。

## 核心內容
| 演進 | 產品 | 年代 |
|------|------|:--:|
| 第一代 | WordStar / WordPerfect | 1979–1990 |
| 第二代 | MS Word for Windows | 1989 |
| 現代 | Google Docs / Office 365 | 2010s |
| AI 時代 | Copilot / Gemini in Docs | 2023+ |

| 核心功能 | 說明 |
|------|------|
| 樣式 (Style) | 段落/字元樣式 → 內容與格式分離 |
| 模板 (Template) | .dotx/.ott → 文檔起點 |
| 追蹤修訂 (Track Changes) | 協作審閱核心機制 |
| 合併列印 (Mail Merge) | 批量生成個人化文檔 |

## WYSIWYG 範式: 使用者直接操作格式化的文檔外觀，無需學習標記語法。學習門檻低但程式化控制弱。
