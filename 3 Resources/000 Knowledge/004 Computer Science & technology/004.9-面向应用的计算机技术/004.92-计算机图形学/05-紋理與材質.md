# 05 紋理與材質 — Textures & Materials

> 紋理是表面細節的載體，材質定義了光線如何與表面交互的完整規則。

---

## 5.1 UV 映射 UV Mapping

| 概念 | 說明 |
|------|------|
| **UV 座標** | 2D 紋理座標，$(u,v) \in [0,1]^2$，將 3D 表面展平到 2D 圖像 |
| **UV 展開 Unwrapping** | 將 3D 網格切割/展開為 2D 平面（最小化失真） |
| **接縫 Seam** | UV 邊界不連續處，需用 Texture Padding 避免滲色 |
| **UDIM** | 多 UV Tile 系統（1001, 1002...），支援超高解析度紋理 |

---

## 5.2 紋理濾波 Texture Filtering

| 濾波器 | 採樣方式 | 品質 |
|:---:|------|:---:|
| **Nearest** | 最近鄰 | 塊狀鋸齒 |
| **Bilinear** | 2×2 線性插值 | 平滑但遠處模糊 |
| **Trilinear** | Bilinear + Mipmap 層間插值 | 消除 Mipmap 跳變 |
| **Anisotropic** (AF) | 沿視線方向多採樣（可達 16×） | 減少斜視模糊 |

---

## 5.3 Mipmap 多級紋理

| 概念 | 說明 |
|------|------|
| **原理** | 預先產生 $1/2, 1/4, 1/8...$ 縮小版紋理 |
| **選擇 LOD** | 基於螢幕空間像素與紋理像素比（紋理梯度 $du/dx, dv/dy$） |
| **優點** | 減少 Cache Miss、消除摩爾紋 (Moiré)、提升效能 |
| **Mipmap Bias** | 手動偏移 LOD，控制銳利/模糊（過低 bias → 閃爍） |

---

## 5.4 紋理類型 Texture Types

| 類型 | 用途 |
|------|------|
| **Albedo / Base Color** | 基礎顏色（無光照資訊） |
| **Normal Map** | 擾動法線，增加表面細節（Tangent Space 儲存） |
| **Roughness Map** | 表面粗糙度（低值=鏡面，高值=漫反射） |
| **Metallic Map** | 金屬度（0=介電質 Dielectric，1=金屬） |
| **Ambient Occlusion (AO) Map** | 預計算局部遮蔽 |
| **Height/Displacement Map** | 頂點位移 → 真實幾何變形 |
| **Emission Map** | 自發光區域 |

---

## 5.5 PBR 材質管線 PBR Material Pipeline

| 工作流 | 紋理 | 適用標準 |
|------|------|----------|
| **Metallic/Roughness** | Albedo + Metallic + Roughness | Unreal Engine, Substance, glTF |
| **Specular/Glossiness** | Diffuse + Specular + Glossiness | 舊版 Unity, 傳統工作流 |

> **Disney Principled BRDF** 定義了藝術家友善的 PBR 參數集：Base Color, Metallic, Roughness, Specular, Anisotropic, Sheen, Clearcoat...

---

## 5.6 進階技術 Advanced Techniques

| 技術 | 說明 |
|------|------|
| **Virtual Texturing** | 只載入可見紋理 Tile 到 GPU 記憶體（ID Tech, UE5） |
| **Procedural Texturing** | 演算法生成紋理（噪聲、圖案），無限解析度 |
| **Texture Atlasing** | 多材質合併一張大紋理，減少 Draw Call |
| **Texture Compression** | BCn/ETC2/ASTC 格式，GPU 硬體解壓 |
