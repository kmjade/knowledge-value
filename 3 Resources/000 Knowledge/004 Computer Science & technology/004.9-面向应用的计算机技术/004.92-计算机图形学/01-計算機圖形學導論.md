# 01 電腦圖形學導論 — Introduction to Computer Graphics

> 電腦圖形學（Computer Graphics, CG）研究如何使用電腦生成、處理與顯示圖形影像，是視覺計算的核心學科。

---

## 1.1 定義 Definition

| 面向 | 描述 |
|------|------|
| **輸入** | 3D 場景描述（幾何、材質、光源、相機） |
| **處理** | 模擬光線傳播與物理交互 |
| **輸出** | 2D 像素陣列（影像 / 即時幀） |
| **核心挑戰** | 在有限計算資源下生成逼真影像（Photo-realism） |

---

## 1.2 歷史里程碑 Historical Milestones

| 年代 | 里程碑 | 意義 |
|:---:|------|------|
| 1963 | **Sketchpad** (Ivan Sutherland) | 第一個互動式圖形系統，CAD 始祖 |
| 1975 | **Phong Shading** (Bui Tuong Phong) | 奠定即時光照基礎 |
| 1980 | **Whitted Ray Tracing** (Turner Whitted) | 首次實現反射/折射光線追蹤 |
| 1986 | **Pixar 成立** / RenderMan | 離線渲染標準，REYES 演算法 |
| 1992 | **OpenGL 1.0** | 跨平台圖形 API 標準 |
| 1999 | **NVIDIA GeForce 256** — 首款 GPU | 硬體 T&L，開啟 GPU 時代 |
| 2006 | **CUDA** (NVIDIA) | GPGPU，GPU 通用計算 |
| 2018 | **NVIDIA RTX / Turing** | 硬體光線追蹤 (RT Core) |
| 2020 | **Unreal Engine 5 — Nanite + Lumen** | 虛擬化幾何 + 即時全域照明 |

---

## 1.3 應用領域 Applications

| 領域 | 典型應用 | 關鍵技術 |
|------|----------|----------|
| 🎬 **電影/VFX** | 特效、CGI 角色 | Path Tracing, RenderMan, Arnold |
| 🎮 **遊戲** | 即時 3D 渲染 | Rasterization, PBR, DLSS |
| ✏️ **CAD/CAM** | 工業設計、建築 | NURBS, CSG, B-Rep |
| 🥽 **VR/AR** | 沉浸式體驗 | 低延遲渲染、Foveated Rendering |
| 🔬 **科學視覺化** | 醫學影像、流體模擬 | Volume Rendering, Ray Marching |
| 🤖 **AI 圖形學** | NeRF, 3DGS, Stable Diffusion | 神經渲染、Diffusion Models |

---

## 1.4 學科關係 Disciplinary Relations

- **電腦視覺 (CV)**: 影像 → 3D（逆向）；CG: 3D → 影像（正向）
- **計算幾何**: 提供幾何資料結構與演算法基礎
- **人機互動 (HCI)**: 圖形介面與互動設計
- **AI/ML**: 神經渲染、降噪、超解析度（DLSS）
