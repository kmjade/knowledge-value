# 08 動畫與模擬 — Animation & Simulation

> 賦予 3D 場景生命：從角色動畫到物理模擬的技術全景。

---

## 8.1 骨骼動畫 Skeletal Animation

| 概念 | 說明 |
|------|------|
| **Skeleton / Rig** | 層級骨骼鏈（關節 Hierarchy），定義變形結構 |
| **Skinning** | 網格頂點跟隨骨骼變形 |
| **Linear Blend Skinning (LBS)** | 每個頂點受多根骨骼加權影響：$v' = \sum w_i \mathbf{M}_i v$ |
| **Dual Quaternion Skinning (DQS)** | 解決 LBS 的「糖紙效應」(Candy Wrapper) |
| **Animation Clip** | 一段骨骼動畫資料（每幀關節 Transform） |
| **Animation Blending** | 多動畫平滑過渡（Lerp/Slerp） |
| **IK (Inverse Kinematics)** | 給定末端位置 → 反算關節角度（CCD/FABRIK 演算法） |
| **FK (Forward Kinematics)** | 給定關節角度 → 計算末端位置 |

---

## 8.2 Blend Shape / Morph Target

| 概念 | 說明 |
|------|------|
| **原理** | 儲存頂點偏移量 (Delta)，線性混合多個形態 |
| **應用** | 臉部表情 (FACS)、肌肉變形、口型同步 (Lip Sync) |
| **限制** | 拓撲必須一致，記憶體消耗 ~ #頂點 × #BlendShapes |

---

## 8.3 物理模擬 Physics Simulation

| 類型 | 方法/演算法 | 應用 |
|------|----------|------|
| **剛體 Rigid Body** | Bullet, PhysX (約束求解器) | 碰撞、堆疊、破壞 |
| **軟體 Soft Body** | FEM (有限元), PBD (Position Based Dynamics) | 布料、果凍、橡膠 |
| **流體 Fluid** | SPH (粒子法), FLIP/APIC (混合), Grid-based | 水、煙、火 |
| **布料 Cloth** | 質點-彈簧 (Mass-Spring), PBD | 衣物、旗幟 |
| **毛髮 Hair/Fur** | 引導線 (Guide Hair) + 插值 + 碰撞 | 角色毛髮 (TressFX/HairWorks) |

---

## 8.4 粒子系統 Particle Systems

| 元件 | 說明 |
|------|------|
| **Emitter** | 產生粒子（速率、初始速度、生命週期） |
| **Dynamics** | 重力、風力、紊流 (Turbulence)、吸引力 |
| **Rendering** | Billboard (始終面向相機)、Mesh Particle、Ribbon Trail |
| **GPU Particles** | 使用 Compute Shader 模擬數百萬粒子 (VFX Graph / Niagara) |

## 8.5 動畫管線 Animation Pipeline

```
Rigging → Skinning → Keyframe/Mocap → Polish (Curves) → Export
```

| 工具 | 用途 |
|------|------|
| **Maya / Blender** | Rigging, Animation, Blend Shapes |
| **MotionBuilder** | Mocap 清理與重定向 |
| **Cascadeur** | AI 輔助物理動畫 |
| **Mixamo** | 自動 Rigging + 動畫庫 (Adobe) |

---

## 8.6 即時動畫技術 Real-Time Animation

| 技術 | 說明 |
|------|------|
| **Animation Blueprint** (UE) / **Animator** (Unity) | 狀態機驅動動畫邏輯 |
| **Procedural Animation** | 即時運算動作（IK 腳步、物理布娃娃） |
| **Motion Matching** | 從大型動畫資料庫即時匹配最優動作片段 |
| **ML Deformer** (UE5) | 神經網路替代傳統變形（肌肉/布料互動） |
