# 07 光線追蹤 — Ray Tracing

> 模擬光線從相機出發，與場景幾何/材質交互，生成物理精確影像的技術。

---

## 7.1 基礎光線追蹤 Basic Ray Tracing

### Whitted Ray Tracing (1980)

| 步驟 | 說明 |
|------|------|
| **Primary Rays** | 從相機射出，穿過每個像素 |
| **Ray-Triangle Intersection** | Möller–Trumbore 演算法（快速射線-三角形相交） |
| **Shadow Rays** | 從交點射向光源，判斷是否被遮擋 |
| **Reflection Rays** | 鏡面反射方向遞迴追蹤 |
| **Refraction Rays** | Snell 定律 ($n_1 \sin\theta_1 = n_2 \sin\theta_2$) 折射追蹤 |

### 渲染方程 The Rendering Equation (Kajiya, 1986)

$$L_o(p, \omega_o) = L_e(p, \omega_o) + \int_\Omega f_r(p, \omega_i, \omega_o) L_i(p, \omega_i) (\omega_i \cdot n) d\omega_i$$

> 輸出光線 = 自發光 + 所有入射光 × BRDF 的半球積分

## 7.2 Monte Carlo 路徑追蹤 Path Tracing

| 概念 | 說明 |
|------|------|
| **Monte Carlo Integration** | 隨機採樣估計積分，收斂速度 $O(1/\sqrt{N})$ |
| **Path Tracing** | 每像素追蹤多條完整光路徑 (Camera → ... → Light) |
| **Russian Roulette** | 隨機終止低貢獻路徑，保持無偏 |
| **Importance Sampling** | 按 BRDF/光源分佈偏好採樣，降低方差 |
| **Multiple Importance Sampling (MIS)** | 結合 BRDF 採樣 + 光源採樣權重 |

## 7.3 加速結構 Acceleration Structures

| 結構 | 說明 | 建構時間 | 遍歷速度 |
|------|------|:---:|:---:|
| **BVH** (Bounding Volume Hierarchy) | 層級包圍盒（AABB/OBB），業界標準 | 快 | 極快 |
| **KD-Tree** | 空間二分樹 | 慢 | 最快 |
| **Octree / Grid** | 均勻/自適應空間網格 | 最快 | 中等 |
| **TLAS/BLAS** (DXR/Vulkan RT) | 雙層 BVH：TLAS(場景級) + BLAS(物體級) | HW加速 | HW加速 |

## 7.4 降噪 Denoising

| 技術 | 說明 |
|------|------|
| **SVGF / ASVGF** | 時空方差導向濾波 (Spatiotemporal Variance-Guided Filtering) |
| **OIDN** (Intel Open Image Denoise) | AI 降噪，CPU/GPU |
| **OptiX Denoiser** | NVIDIA 光線追蹤降噪 |
| **DLSS Ray Reconstruction** | 神經網路超解析度 + 光追降噪（DLSS 3.5+） |

## 7.5 硬體光線追蹤 Hardware RT

| 元件 | 功能 |
|------|------|
| **RT Core** (NVIDIA Turing+) | 硬體 BVH 遍歷 + 射線-三角形相交 |
| **DXR** (DirectX Raytracing) | Microsoft 光追 API (RayGen/ClosestHit/Miss/AnyHit Shaders) |
| **Vulkan RT** (VK_KHR_ray_tracing) | Vulkan 光追擴展 |
| **Metal RT** | Apple GPU 光追 (M3+) |
| **Shader Execution Reordering (SER)** | Ada Lovelace: 動態重排線程，提升 divergent rays 效能 |
