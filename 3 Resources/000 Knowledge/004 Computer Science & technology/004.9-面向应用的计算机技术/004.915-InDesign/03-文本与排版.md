# 03 文本與排版 Text & Typography

## 文字工具總覽 Text Tools

| 工具 | 快捷鍵 | 用途 |
|------|:------:|------|
| **文字工具** | `T` | 建立文字框 or 點擊輸入 |
| **路徑文字工具** | `Shift+T` | 沿路徑排文 |
| **垂直文字工具** | — | 直排文字（中日韓常用） |
| **選取+連按文字框**: 從入口/出口串接文字框（串文 Threading） |

## 字元樣式 vs 段落樣式 Character vs Paragraph Styles

| | 字元樣式 | 段落樣式 |
|---|----------|----------|
| **範圍** | 選取的字元 | 整個段落（含 `Enter`） |
| **屬性** | 字體/大小/行距/字距/色彩/語言 | 上述 + 縮排/對齊/定位點/分行設定 |
| **快捷鍵** | `Ctrl+Shift+Num` | `Ctrl+Num` (需先定義) |
| **巢狀樣式** | ❌ | ✅ 可內嵌字元樣式 |

## OpenType 功能 OpenType Features

| 功能 | 中文名稱 | 效果 | 適用 |
|------|----------|------|------|
| **Liga** | 標準連字 | f+i → ﬁ (合字) | 拉丁 |
| **Smcp** | 小型大寫 | 小寫變小型大寫 | 拉丁 |
| **SwsH** | 上下文替代 | 依前後文變形 | 拉丁/阿拉伯 |
| **Palt** | 等比假名 | 日文假名字寬等比 | CJK |
| **Vrt2** | 直排替代 | 直排時自動替代字符 | CJK |
| **分數** | 自動分數 | 1/2 → ½ | 通用 |
| **上標/下標** | Superior/Inferior | 如 x²、H₂O | 通用 |

> 中文 OpenType: 常用 `Palt`（假名）、`Vrt2`（直排）、`halt`（半形）。透過「字元」面板 (Character Panel) 右側選單 → OpenType。

## 複合字體 Composite Fonts（中日韓+拉丁混排）

| 設定 | 說明 |
|------|------|
| **漢字** | 中文字體（如 Adobe 明體、思源宋體） |
| **假名** | 日文假名字體 |
| **標點** | 全形標點符號字體 |
| **符號** | 數學/貨幣/箭頭等 |
| **羅馬字** | 拉丁字母/數字字體（如 Minion Pro、Source Sans） |
| **數字** | 可獨立指定數字字體 |

> 路徑：`文字 → 複合字體 → 新增`。複合字體儲存在文件內，可匯出/匯入。

## 中文排版規則 Kinsoku（禁則處理）

| 類型 | 規則 | 說明 |
|------|------|------|
| **行首禁則** | 不置於行首 | ，。、：；！？」』）】 |
| **行尾禁則** | 不置於行尾 | 「『（【《 |
| **不可分割** | 不跨行斷開 | …—（em dash 前後） |
| **壓縮** | 行尾標點壓縮 | 句號/逗號半形寬度擠壓 |

> 設定: `段落面板 → 面板選單 → 禁則處理 (Kinsoku)` → 選擇「寬鬆」或「嚴格」。

## GREP 樣式 GREP Styles

| 用途 | GREP 表達式 | 套用樣式 |
|------|-------------|----------|
| 數字變粗體 | `\d+` | Bold 字元樣式 |
| 括號內變色 | `\(.+?\)` | Highlight 樣式 |
| 貨幣金額 | `[$¥]\d+[.,]\d{2}` | Price 樣式 |
| Email 底線 | `[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}` | Link 樣式 |
| 第一個句子變體 | `^.+?[.。!！?？]` | Intro 樣式 |

> 設定路徑: 段落樣式 → GREP 樣式 → 新增 GREP 樣式
