# 07 互動與數位出版 Interactive & Digital Publishing

## EPUB 輸出 EPUB Export

| 類型 | 說明 | 適用 |
|------|------|------|
| **流動式 EPUB (Reflowable)** | 文字自適應螢幕大小，可調整字體 | 小說、文字為主書籍 |
| **固定版面 EPUB (Fixed-Layout)** | 精確保留版面設計 | 圖文書、繪本、雜誌 |

### EPUB 匯出關鍵設定

| 設定 | 說明 |
|------|------|
| **版本** | EPUB 2.0.1 (相容性) / EPUB 3.0 (多媒體/直排) |
| **封面** | 指定影像 (JPEG/GIF) |
| **目錄樣式** | 使用文件 TOC 樣式自動產生 NCX 導覽 |
| **CSS 選項** | 產生 CSS / 保留本機覆蓋 |
| **中繼資料** | 標題/作者/出版社/ISBN/語言 |

### 物件匯出選項 (Object Export Options)

| 設定 | 效果 |
|------|------|
| **自訂點陣圖化** | 控制圖像壓縮與解析度 |
| **EPUB 文字** | 轉文字/轉影像/忽略 |
| **Alt 文字** | 無障礙替代文字 (EPUB3) |

## 互動式 PDF Interactive PDF

| 互動元素 | 說明 |
|----------|------|
| **按鈕 (Buttons)** | 導覽（下一頁/上一頁）、動作（開啟URL/播放影片） |
| **超連結 (Hyperlinks)** | URL、頁面跳轉、錨點 |
| **影片/音訊** | 內嵌或連結媒體 (`視窗 → 互動 → 媒體`) |
| **書籤 (Bookmarks)** | 自動由 TOC 產生或手動新增 |
| **表單欄位** | 文字欄位/核取方塊/下拉選單/簽名欄 |

### 互動面板 (`視窗 → 互動`)

| 面板 | 快捷鍵 |
|------|:------:|
| **按鈕與表單** | `Ctrl+Shift+F6` (舊版) |
| **動畫** | 設定移動/旋轉/縮放動畫 |
| **計時** | 設定延遲觸發的時間軸 |
| **物件狀態** | 多重狀態物件（點擊切換） |
| **超連結** | 管理所有連結 |

> 匯出互動式 PDF: `檔案 → 匯出 → Adobe PDF (互動式)` — 保留按鈕/連結/影片。

## Publish Online 線上發佈

| 特性 | 說明 |
|------|------|
| **平台** | Adobe 雲端託管，產生公開 URL |
| **互動支援** | 按鈕、動畫、影片、超連結 |
| **分析** | 內建瀏覽次數分析 |
| **限制** | 不支援 SWF、部分 PDF 表單功能 |
| **操作** | `檔案 → Publish Online` → 設定標題/描述/封面 |

## 固定版面 EPUB (Fixed-Layout EPUB)

| 關鍵設定 | 說明 |
|----------|------|
| **跨頁檢視** | 雙頁合成跨頁或獨立單頁 |
| **匯出範圍** | 所有頁面 / 頁面範圍 |
| **JPEG 品質** | 圖像壓縮品質 (高/中/低) |
| **對頁** | 模擬翻書效果（僅支援部分閱讀器） |

## 支援格式對照 Format Support Matrix

| 格式 | 互動 | 多媒體 | 精確版面 | 平台 |
|------|:---:|:-----:|:-------:|------|
| EPUB (Reflowable) | 有限 | 音訊/影片(EPUB3) | ❌ | 所有閱讀器 |
| EPUB (Fixed-Layout) | 有限 | 音訊/影片(EPUB3) | ✅ | Apple Books/Kobo |
| 互動式 PDF | ✅ | 影片(Flash 已淘汰) | ✅ | Acrobat Reader |
| Publish Online | ✅ | ✅ | ✅ | 瀏覽器 |
| DPS (舊) | ✅ | ✅ | ✅ | 已終止 |

## 實用技巧

- **Alt 文字**: 所有圖像設定 Alt 文字以符合 EPUB 無障礙標準
- **文章面板**: `視窗 → 文章` — 控制 EPUB 匯出時內容的閱讀順序
- **先檢查後匯出**: 使用 EPUB 互動式預檢 (`視窗 → 輸出 → Preflight`)
- **字體授權**: EPUB 內嵌字體需確認授權許可
