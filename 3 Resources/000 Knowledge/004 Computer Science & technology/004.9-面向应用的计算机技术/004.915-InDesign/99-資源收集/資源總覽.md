## 印刷廠溝通檢查清單

| ☐ | 項目 | 確認內容 |
|:--:|------|----------|
| ☐ | **出血** | 3mm 以上，裁切標記已啟用 |
| ☐ | **色彩** | CMYK + 專色，無殘留 RGB 物件 |
| ☐ | **解析度** | 圖像 ≥ 300ppi |
| ☐ | **字體** | 全部嵌入或轉外框 |
| ☐ | **TAC** | 總油墨覆蓋率 ≤ 300–320% |
| ☐ | **安全區** | 重要內容距離裁切邊 ≥ 5mm |
| ☐ | **Preflight** | 通過 PDF/X-4 設定檔檢查 |
| ☐ | **疊印預覽** | 檢查黑色疊印 / 白色鏤空正確 |
| ☐ | **頁數** | 騎馬釘裝訂: 頁數為 4 的倍數 |
