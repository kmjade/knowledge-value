# 09 腳本與自動化 Scripts & Automation

## 腳本技術棧 Scripting Stack

| 技術 | 語言 | 現狀 | 適用 |
|------|------|:----:|------|

### 常用 API 速查

| 操作 | ExtendScript |
|------|-------------|
| 取得當前文件 | `app.activeDocument` |
| 建立新文件 | `app.documents.add()` |
| 選取物件 | `app.selection[0]` |
| 建立文字框 | `doc.textFrames.add()` |
| 套用段落樣式 | `para.appliedParagraphStyle = doc.paragraphStyles.item("H1")` |
| 置入圖像 | `frame.place(File("/path/image.jpg"))` |
| 匯出 PDF | `doc.exportFile(ExportFormat.PDF_TYPE, File("/output.pdf"))` |
| 執行 GREP 查找 | `app.findGrepPreferences.findWhat = "\\d+"` |

### ExtendScript 範例：批次匯出 PDF

```javascript
// 將資料夾內所有 .indd 匯出為 PDF
var folder = Folder("/path/to/indd/");
var files = folder.getFiles("*.indd");
for (var i = 0; i < files.length; i++) {
    var doc = app.open(files[i]);
    var pdfPath = files[i].fullName.replace(/\.indd$/, ".pdf");
    doc.exportFile(ExportFormat.PDF_TYPE, File(pdfPath));
    doc.close(SaveOptions.NO);
}
```

## 批次處理 Batch Processing

### 方法一: GREP 查找/取代 (`Ctrl+F`)

| 範例 | 查找 | 取代 | 用途 |
|------|------|------|------|
| 移除多餘空格 | ` +` | ` ` | 清理排版 |
| 中英文間加空格 | `([\x{4e00}-\x{9fff}])([A-Za-z])` | `$1 $2` | CJK 排版 |
| 全形數字轉半形 | `[０-９]` | 逐項對應取代 | 數據標準化 |
| 移除空白行 | `\r\n\s*\r\n` | `\r\n` | 清理段落間距 |

### 方法二: GREP 樣式 (段落樣式內)

即時自動格式化（無需手動查找替換），見 [[03-文本与排版#GREP 樣式 GREP Styles|03 GREP 樣式]]。

### 方法三: Data Merge 批次

批次產生多份文件／PDF（見 [[06-表格与数据合并|06 Data Merge]]）。

### 方法四: 自訂腳本

ExtendScript 組合多步驟為一鍵操作：

| 場景 | 腳本化操作 |
|------|-----------|
| **定稿檢查** | Preflight + 字體清單 + 連結狀態 + 匯出 PDF |
| **格式轉換** | .indd → PDF → JPEG 縮圖 |
| **樣式清理** | 刪除未使用樣式、標準化命名 |
| **文件合併** | 多份 .indd 合併為書冊 (.indb) |

## UXP 插件開發

| 項目 | 說明 |
|------|------|
| **開發環境** | UXP Developer Tool + VS Code |
| **HTML/CSS/JS** | 面板 UI 使用網頁技術 |
| **API** | 與 ExtendScript 相似，但基於 Promise/async |
| **分發** | Adobe Creative Cloud Marketplace |
| **優勢** | 現代 ES6+、非同步、原生 UI 框架 |

### UXP vs ExtendScript 對照

| | ExtendScript | UXP |
|---|:---:|:---:|
| JS 版本 | ES3 (無 Promise) | ES6+ |
| UI | ScriptUI (舊) | HTML/CSS |
| 執行方式 | 同步 | 非同步 (Promise) |
| 存取範圍 | 完整 DOM | 逐步擴充中 |
| 建議 | 內部腳本/快速自動化 | 分發插件/複雜 UI |

## GREP 速查 GREP Quick Reference

| 符號 | 說明 | 範例 |
|:----:|------|------|
| `.` | 任意字元 | `c.t` → cat, cut |
| `\d` | 數字 | `\d{4}` → 4 位數字 |
| `\w` | 單字字元 | `[A-Za-z0-9_]` |
| `\s` | 空白字元 | 空格/Tab/換行 |
| `^` / `$` | 行首/行尾 | `^\d+` → 行首數字 |
| `\b` | 單字邊界 | `\bword\b` |
| `+` / `*` / `?` | 1+/0+/0或1 | `\d+\.?\d*` |
| `\|` | 或 | `cat\|dog` |
| `()` | 分組 | `(\d{3})-(\d{4})` |
| `[ ]` | 字元類 | `[aeiou]` 母音 |
| `(?i)` | 大小寫不區分 | `(?i)indesign` |
