# 02 文檔設置與版面 Document Setup & Layout

## 文檔創建參數 New Document

| 參數 | 說明 | 建議值 |
|------|------|--------|
| **意圖** | Print / Web / Mobile | Print 為印刷 |
| **頁數** | 起始頁數，後續可增減 | 依出版物 |
| **起始頁碼** | 首頁編號 | 1 或自訂 |
| **頁面大小** | 完成尺寸 (Trim Size) | A4: 210×297mm |
| **出血 (Bleed)** | 超出裁切線的延伸區 | 3mm（歐美）/ 3-5mm（亞洲） |
| **印刷邊界 (Slug)** | 出血外的備註區（裁切標記/色條） | 視印刷廠需求 |

## 頁面與跨頁 Pages & Spreads

- **頁面面板** (`F12`): 管理頁面順序、增刪、套用主頁
- **跨頁 (Spread)**: 多頁並排顯示（對頁），可勾選「允許文件頁面隨機排列」打散
- **移動頁面**: 拖曳縮圖 + `Alt` 複製
- **旋轉跨頁**: 選取跨頁 → 頁面面板選單 → 旋轉跨頁檢視

## 輔助線與網格 Guides & Grids

| 類型 | 用途 | 快捷鍵/操作 |
|------|------|------------|
| **邊界 (Margins)** | 文檔安全區 | 版面 → 邊界與欄 |
| **欄 (Columns)** | 文字欄分割 | 選取文字框 → 文字框選項 (`Ctrl+B`) |
| **尺標輔助線** | 自訂對齊線 | 從尺標拖出，`Ctrl+;` 顯示/隱藏 |
| **基準網格 (Baseline Grid)** | 文字基線對齊 | 偏好設定 → 網格 |
| **文檔網格** | 物件定位 | `Ctrl+'` 顯示/隱藏 |
| **智慧輔助線** | 自動對齊鄰近物件 | 檢視 → 網格與輔助線 → 智慧輔助線 |

## 主頁 Master Pages

| 概念 | 說明 |
|------|------|
| **主頁 (Master Page)** | 統一套用到多頁的版面模板（頁碼/頁首/頁尾/背景） |
| **A-主版** | 預設主頁，雙面文檔含左右頁 |
| **套用主頁** | 拖曳主頁縮圖到頁面，或右鍵 → 套用主頁至頁面 |
| **覆蓋主頁項目** | `Ctrl+Shift+Click` 選取主頁物件後釋放，即可在單頁修改 |
| **多重主頁** | 不同章節/版型用不同主頁 (A-Master, B-Chapter, C-Cover…) |

## 頁碼與章節 Numbering & Sections

| 操作 | 方法 |
|------|------|
| **自動頁碼** | 在主頁繪製文字框 → `文字 → 插入特殊字元 → 標記 → 目前頁碼` |
| **章節起始** | 右鍵頁面 → 編號與章節選項 → 起始頁碼 + 章節前綴 |
| **頁碼樣式** | 阿拉伯數字 / 羅馬數字 (i,ii…) / 英文字母 |
| **章節標記** | 段落樣式定義 → 項目符號與編號 → 清單類型: 編號 → 層級1 |

## 實用技巧

- **`Alt+拖曳輔助線`**: 水平↔垂直輔助線切換
- **雙擊尺標交會處**: 重設零點
- **`Ctrl+Alt+G`**: 選取頁面上所有輔助線（方便刪除）
- **頁面工具 (`Shift+P`)**: 選取個別頁面調整大小（多尺寸文檔）
- **液態版面 (Liquid Layout)**: 頁面工具選取後，面板中選擇縮放規則（用於 Alternate Layout）
