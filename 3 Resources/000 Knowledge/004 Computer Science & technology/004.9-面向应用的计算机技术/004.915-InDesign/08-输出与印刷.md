# 08 輸出與印刷 Output & Print Production

## PDF 輸出預設 PDF Presets

| 預設 | 用途 | 相容性 | 關鍵特性 |
|------|------|:------:|----------|
| **高品質列印** | 桌上型印表機校樣 | Acrobat 5+ | 300ppi、不移除透明度 |
| **PDF/X-1a:2001** | 傳統印刷（最保守） | Acrobat 4+ | CMYK+專色、全部平面化、嵌入字體 |
| **PDF/X-3:2002** | 印刷（支援 RGB） | Acrobat 4+ | 允許 RGB/Lab + ICC、平面化 |
| **PDF/X-4:2010** | 現代印刷（推薦） | Acrobat 7+ | 保留透明度、圖層、ICC |
| **最小檔案大小** | Email/網頁預覽 | Acrobat 6+ | 72ppi、高壓縮 |

### PDF/X-4 關鍵參數 (推薦印刷用)

| 參數 | 建議值 |
|------|--------|
| **色彩轉換** | 無色彩轉換 (保留 CMYK+R+G+B 數值) |
| **輸出方式** | 轉換為目的地 (保留數值) |
| **目的地** | Document CMYK (或印刷廠指定 ICC) |
| **影像壓縮** | 雙立方縮減取樣至 300ppi、JPEG 高品質 |
| **標記與出血** | 裁切標記 + 3mm 出血 |

## Preflight 預檢

即時偵測文檔輸出問題 (`視窗 → 輸出 → Preflight`)

| 預設設定檔 | 檢查範圍 |
|------------|----------|
| **基本** | 遺失字體、遺失連結、溢排文字 |
| **數位出版** | EPUB 相容性、Alt 文字 |
| **PDF/X-1a** | 色彩空間、透明度平面化、字體嵌入 |
| **PDF/X-4** | 透明度保留、色彩設定、影像解析度 |

### 自訂 Preflight 設定檔

`Preflight 面板選單 → 定義設定檔 → 新增`。可檢查：

- 影像: 最低/最高解析度、色彩空間、壓縮方式
- 文字: 最小字級、字體類型、溢排文字
- 色彩: 專色數量、RGB 存在、色彩設定檔
- 頁面: 出血存在、空白頁、頁面大小一致

## Package 封裝

收集所有關聯資源以交付印刷廠 (`檔案 → 封裝`)

| 收集內容 | 說明 |
|----------|------|
| **InDesign 文件 (.indd)** | 原始排版檔 |
| **字體** | 使用的字體（確認授權） |
| **連結圖像** | 所有置入的圖像原始檔 |
| **ICC 設定檔** | 色彩設定檔 |
| **PDF 校樣** | 低解析度預覽用 PDF |
| **說明報告 (.txt)** | 字體清單、連結清單、列印設定摘要 |

## 疊印與陷印 Overprint & Trapping

| 概念 | 說明 |
|------|------|
| **疊印 (Overprint)** | 前景色不鏤空背景，油墨疊印（對黑色常用 100% K 疊印） |
| **陷印 (Trapping)** | 相鄰色塊間微小重疊，防止印刷套印偏差露白 |
| **內建陷印** | `視窗 → 輸出 → 陷印預設` — InDesign 可自動計算陷印 |

### 疊印預覽

`檢視 → 疊印預覽` (`Ctrl+Alt+Shift+Y`) — 模擬油墨疊印效果，檢查黑色疊印/白色鏤空。

## 色彩管理 Color Management

| 概念 | 說明 |
|------|------|
| **CMYK** | 青 Cyan / 洋紅 Magenta / 黃 Yellow / 黑 blacK — 印刷四色 |
| **專色 (Spot Color)** | 預混油墨，非 CMYK 合成 — Pantone/DIC/TOYO 色票 |
| **RGB** | 螢幕顯示色域 > 印刷色域，需注意溢色 (Out of Gamut) |
| **ICC 設定檔** | 設備色彩特性描述，確保跨設備一致 (Japan Color / Fogra / GRACoL) |

### 色彩設定 (`編輯 → 色彩設定`)

| 設定 | 建議 |
|------|------|
| **工作空間 RGB** | sRGB (網頁) / Adobe RGB (印刷) |
| **工作空間 CMYK** | Japan Color 2001 Coated (日本) / Fogra 39 (歐洲) / GRACoL (北美) |
| **色彩管理政策** | 保留內嵌設定檔 (Preserve Embedded Profiles) |

### 分色預覽 Separations Preview

`視窗 → 輸出 → 分色預覽` — 檢查各色版、專色、總油墨覆蓋率 (TAC ≤ 300-320%)
