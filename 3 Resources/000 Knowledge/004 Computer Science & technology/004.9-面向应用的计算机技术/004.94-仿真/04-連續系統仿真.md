# 04 連續系統仿真 Continuous System Simulation

## 核心: 微分方程的數值求解
連續系統由**常微分方程 (ODE)** 或**偏微分方程 (PDE)** 描述，仿真即為對其進行數值積分。

## 常微分方程數值方法 ODE Numerical Methods

| 方法 | 階數 | 特點 | 誤差 |
|------|:--:|------|:--:|
| **Euler 顯式法** | 1st | 最簡單，大步長不穩定 | O(h) |
| **Euler 隱式法** | 1st | 無條件穩定但需求解方程 | O(h) |
| **Heun 法 (改進 Euler)** | 2nd | 預測-校正，精度提升 | O(h²) |
| **Runge-Kutta 4 (RK4)** | 4th | 精度/計算量最佳平衡 | O(h⁴) |
| **Dormand-Prince (DOPRI)** | 5(4) | 自適應步長，ODE45 使用 | O(h⁵) |
| **Adams-Bashforth-Moulton** | 可變 | 多步法，ODE113 使用 | — |

## 剛性問題 Stiff ODE
系統含快慢兩種時間尺度時 → 顯式法要求極小步長 → 需隱式法 (BDF/ODE15s)。

## 區塊圖建模 Block Diagram Modeling (Simulink 方法)

| 區塊類型 | 功能 | Simulink 範例 |
|----------|------|---------------|
| 積分器 Integrator | 1/s，積累輸入 | Integrator block |
| 增益 Gain | 乘以常數 | Gain block |
| 求和 Sum | 加/減信號 | Sum block |
| 傳遞函數 Transfer Fcn | s-domain 動態 | Transfer Fcn block |
| 非線性 Nonlinearity | 飽和、死區、遲滯 | Saturation block |
| 信號源 Source | Step、Sine、Pulse | Step block |
| 示波器 Scope | 信號可視化 | Scope block |

## 控制系統仿真案例
PID 控制器調參 (Kp, Ki, Kd) → 時域指標 (上升時間、超調量、穩態誤差) → 頻域分析 (Bode、Nyquist)。典型模型: 質量-彈簧-阻尼系統 (mẍ + bẋ + kx = F)。

## PDE 仿真方法概述
有限差分法 (FDM)、有限元素法 (FEM)、有限體積法 (FVM) → 離散化後轉化為大型 ODE 系統。
