# 06 基於代理的建模 Agent-Based Modeling (ABM)

## ABM 定義
自底向上 (Bottom-Up) 的建模方法：定義個體**代理 (Agent)** 的行為規則，通過代理間的**互動 (Interaction)** 及代理與**環境 (Environment)** 的互動，觀察系統層面的**湧現行為 (Emergent Behavior)**。

## ABM 三要素

| 要素 | 描述 | 設計要點 |
|------|------|----------|
| **代理 Agent** | 自治個體，有內部狀態和行為規則 | 異質性 (Heterogeneity)、學習/適應 |
| **環境 Environment** | 代理生存和互動的空間 | 離散網格或連續空間、資源分佈 |
| **互動 Interaction** | 代理間/代理-環境的訊息交換 | 局部鄰域、網路拓撲 (小世界/無尺度) |

## 湧現 Emergence
微觀規則 → 宏觀模式：個體簡單行為的疊加產生意想不到的全局行為。
範例: 鳥群 (Boids Rules: Separation/Alignment/Cohesion) → 集體遷移模式。

## ABM 適用場景 (當系統具備以下特徵)

| 特徵 | 說明 |
|------|------|
| 異質個體 | 個體差異不可忽略 |
| 局部互動 | 個體僅與鄰近個體交流 |
| 自適應行為 | 個體能學習和調整策略 |
| 空間維度重要 | 空間位置影響行為 |
| 網絡效應 | 社交網絡影響傳播 |

## 工具與框架

| 工具 | 語言 | 特點 |

| **NetLogo** | Logo | 教育用，簡單快速原型 |

| **Mesa** | Python | 輕量、可與數據科學整合 |
| **Repast** | Java/Python | 學術級，大規模仿真 |
| **MASON** | Java | 高性能，Swarm 後繼 |

## 經典應用案例

| 領域 | 模型 | 核心機制 |
|------|------|----------|
| **流行病 (Epidemic)** | SIR、SEIR | 感染-恢復動態、接觸網路 |
| **市場 (Market)** | 零智能代理、Santa Fe 人工股市 | 訂單簿、策略演化 |
| **人群疏散 (Crowd)** | 社會力模型 (Social Force) | 目標驅動+排斥力 |
| **生態 (Ecology)** | 捕食者-獵物 (Lotka-Volterra ABM) | 出生/死亡/捕食規則 |
| **輿論 (Opinion)** | Deffuant、Hegselmann-Krause | 有限信任的態度更新 |
