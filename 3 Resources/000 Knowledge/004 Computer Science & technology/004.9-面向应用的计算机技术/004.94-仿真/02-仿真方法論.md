# 02 仿真方法論 Simulation Methodology

## 建模範式 Modeling Paradigms

| 範式 | 描述 | 代表框架/工具 |
|------|------|---------------|
| **DEVS** | 離散事件系統規範，分層模組化建模 | ADEVS、PythonPDEVS |
| **SysML** | 系統建模語言，基於 UML 擴展 | Cameo、MagicDraw |
| **Bond Graph** | 能量流和多物理域統一建模 | 20-sim |
| **Modelica** | 基於方程的非因果建模語言 | Dymola、OpenModelica |
| **UML 狀態圖** | 離散狀態轉換建模 | AnyLogic、Simulink Stateflow |

## 仿真專案工作流 Project Workflow
問題定義 (Problem Formulation) → 概念模型 (Conceptual Model) → 數學模型 (Mathematical Model) → 電腦實現 (Implementation) → 驗證 Verification (建對了模型?) → 確認 Validation (建了對的模型?) → 實驗設計 DOE → 運行 → 統計分析 → 結果溝通

## 隨機數生成 Random Number Generation
- **偽隨機數 PRNG**: Linear Congruential Generator (LCG)、Mersenne Twister (MT19937)
- **隨機變量生成**: 逆變換法 (Inverse Transform)、接受-拒絕法 (Acceptance-Rejection)
- **常見分佈**: Uniform、Normal (Box-Muller)、Exponential、Poisson、Gamma
- **隨機數種子 (Seed)**: 確保實驗可重現性 (Reproducibility)

## 仿真優化 Simulation Optimization

| 方法 | 描述 | 適用場景 |
|------|------|----------|
| 響應曲面法 RSM | 統計回歸建立元模型 | 連續決策變量 |
| 啟發式/元啟發式 | GA、PSO、Simulated Annealing | 組合優化 |
| Ranking & Selection | 統計比較多個方案 | 有限候選方案 |
| 梯度估計 | IPA、LR/SF 方法 | 連續優化 |
| Bayesian Optimization | 高斯過程代理模型 | 昂貴的仿真評估 |
