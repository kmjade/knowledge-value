# 01-图书情报学概述

> 图书馆学+情报学——知识的管理和服务的科学

## LIS的三大支柱

| 支柱 | 核心问题 |
|------|------|
| 图书馆学 | 如何收集·保存·提供信息？ |
| 情报学 | 如何检索·分析·利用信息？ |
| 信息科学 | 信息本身的性质和行为？ |

## 关键转变

| 从藏书(保存) → 服务(提供) → 赋能(教用户自己找) |
