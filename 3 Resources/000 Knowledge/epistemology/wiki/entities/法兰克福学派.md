# 法兰克福学派

## 基本信息

- **成立**: 1923 年（法兰克福大学社会研究所）
- **活跃时期**: 1930s–至今
- **学派类型**: 批判理论——马克思主义、精神分析和社会科学的综合
- **第一代核心成员**: Max Horkheimer、Theodor Adorno、Herbert Marcuse、Walter Benjamin、Erich Fromm
- **第二代**: Jürgen Habermas
- **第三代**: Axel Honneth

## 对科学的批判

法兰克福学派对三层模型的每个层都提出了激进的批判：

| 层 | 法兰克福学派的批判 |
|----|-------------------|
| **科学层** | "工具理性"将一切还原为效率和算计——科学方法成为统治的技术，而非解放的工具 |
| **知识层** | 实证主义将"知识"窄化为可量化的命题——排斥了批判性反思和解放性知识 |
| **学科层** | 学术学科的结构复制了社会权力的结构——学科的"中立性"是意识形态 |

### 核心概念

| 概念 | 提出者 | 含义 |
|------|--------|------|
| **工具理性批判** | Horkheimer/Adorno | 理性被还原为手段—目的计算——失去了反思目的本身的能力。科学是工具理性的典范 |
| **文化产业** | Adorno/Horkheimer | 知识（包括科学知识）可以被商品化和工业化——"科学传播"成为"科学消费品"的生产 |
| **解放性认知兴趣** | Habermas | 不同于科学的"技术性认知兴趣"（控制自然），人类也有"解放性认知兴趣"——摆脱不必要的社会约束。批判理论服务于这个兴趣 |
| **知识与人类兴趣** | Habermas | 三种认知兴趣对应三种科学：经验—分析科学（技术兴趣）、历史—解释学科学（实践兴趣）、批判性科学（解放兴趣） |

## 在三层模型中的位置

法兰克福学派处于三层模型的**边界和元层次**——它不仅研究科学，而且批判科学本身的社会功能和局限性。它提醒三层模型不要忽视**权力**——学科不仅组织知识，也有能力压制某些知识。

## 相关概念

- [[科学哲学]] — 法兰克福学派提供的是对科学的**社会政治批判**，不是传统科学哲学
- [[科学社会学]] — 法兰克福学派的研究对象有大量重叠，但方法是批判的而非描述的
