# UDC：国际十进分类法

## 基本信息

- **全称**: Universal Decimal Classification
- **创始人**: [[奥特勒|Paul Otlet]] 与 Henri La Fontaine
- **首次出版**: 1905 年（基于 DDC 第五版）
- **当前维护方**: UDC Consortium（荷兰海牙）
- **使用范围**: 全球 130+ 个国家，尤其在欧洲、拉丁美洲的专业/研究图书馆
- **类型**: 分面+层级分类法

## 与 DDC 的核心区别

UDC 最初基于 DDC，但奥特勒加入了关键的创新——**分面分类**：

| | DDC | UDC |
|----|-----|-----|
| 分类策略 | 枚举（每个主题一个数字） | 分面（通过符号组合多个维度） |
| 连接符 | 无（纯层级） | 有——+ / : = (...) "..." |
| 灵活性 | 低 | 高——可以表达复杂关系 |
| 复杂度 | 低——直观 | 高——需要学习符号系统 |
| 应用 | 公共和学校图书馆 | 专业和研究图书馆 |

### UDC 的辅助符号表

| 符号 | 功能 | 示例 |
|------|------|------|
| **+** | 并列 | 54+66 = 化学和物理 |
| **/** | 范围 | 53/54 = 物理学和化学 |
| **:** | 关系 | 17:7 = 伦理与艺术的关系 |
| **(=...)** | 种族/民族 | (=214.58) = 罗姆人 |
| **"..."** | 时间 | "19" = 20世纪 |
| **(...)** | 地点 | (510) = 中国 |
