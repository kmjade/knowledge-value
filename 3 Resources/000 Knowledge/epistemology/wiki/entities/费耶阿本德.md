# 保罗·费耶阿本德 (1924–1994)

## 基本信息

- **生卒**: 1924–1994
- **国籍**: 奥地利→美国→瑞士
- **背景**: 戏剧→物理学→科学哲学（波普尔的学生，但与老师彻底分道）
- **核心著作**: 《反对方法》（1975）——科学哲学的"坏孩子"

## 核心贡献

### "怎么都行"（Anything Goes）

费耶阿本德的极端论点：不存在普遍有效的科学方法论规则。历史上的科学进步往往是通过**打破方法规则**而取得的——伽利略使用了宣传和修辞（而非"方法"）来推进日心说。

### 四项核心主张

| 主张 | 含义 |
|------|------|
| **方法论无政府主义** | "怎么都行"不是方法——是反对方法论的权威主义。规则可以也应该被打破 |
| **反划界** | 科学与非科学之间没有有意义的区分。西方医学和传统医学——为什么前者是"科学"？ |
| **自由社会中的科学** | 科学不应该在国家中享有特权地位。教会与国家分离 → 科学与国家也应分离 |
| **多元主义** | 新理论应该通过**与尽可能多的替代理论比较**来发展，而不是仅与"事实"比较 |

## 在三层模型中的位置

费耶阿本德位于三层模型的**边缘**——他挑战了所有三层的基本假设：没有可靠的生产方法（科学层），没有客观的知识（知识层），学科分类是权力的伪装（学科层）。

## 关键争议

费耶阿本德是最被误解的科学哲学家之一。"怎么都行"不是提倡混乱，而是**拒绝方法论规则的暴政**。他的核心洞察：方法论规则（如波普尔的可证伪性或标准统计方法）在保护正常科学的同时，也系统性地排斥了创新。
