# 01-分类法概述

> 分类是认知的基本操作——把世界分成「可管理的块」

## 分类法的要素

| 要素 | 说明 |
|------|------|
| 类目 | 分类的单元——「哲学」「数学」「物理」 |
| 层级 | 大类→中类→小类——从一般到特殊 |
| 标记 | 数字或字母——代表类目——便于排架和检索 |
| 索引 | 帮助找到类目——不是分类——是「地图」 |

## 分类法的类型

| 列举式(DDC): 把所有类目都列出来 / 分面式(冒号分类法): 用标准面组合 |
|------|
