# 05-计算机视觉 (Computer Vision)

> 计算机视觉是 AI 的一个子领域，研究如何让计算机从图像和视频中获取高层次的语义理解——模拟（甚至超越）人类视觉系统。

## CV 的核心任务

| 任务 | 含义 | 输入 → 输出 |
|------|------|------------|
| **图像分类** | 整个图像属于哪一类 | 图片 → "猫" |
| **目标检测** | 图像中有什么物体，在哪里 | 图片 → 多个 (类别 + 边界框) |
| **语义分割** | 每个像素属于哪一类 | 图片 → 逐像素类别标签 |
| **实例分割** | 区分同类的不同个体 | 图片 → "猫A" vs "猫B" 分别标出 |
| **姿态估计** | 人的关节点在哪 | 图片 → 骨骼关键点坐标 |
| **图像生成** | 从文本/噪声生成新图像 | "一只戴墨镜的猫" → 图像 |
| **图像超分辨率** | 低分辨率 → 高分辨率 | 模糊图 → 清晰图 |

## CNN 骨干网络 (Backbone)

### 经典架构演进

| 架构 | 年份 | 关键创新 | Top-5 错误率 (ImageNet) |
|------|------|----------|--------------------------|
| AlexNet | 2012 | 首次用 GPU 训练深度 CNN | 15.3% |
| VGG-16/19 | 2014 | 小而统一的 3×3 卷积核，深度更重要 | 7.3% |
| GoogLeNet (Inception) | 2014 | Inception 模块（多尺度并行卷积） | 6.7% |
| ResNet | 2015 | 残差连接，训练 152 层 | 3.6% |
| DenseNet | 2017 | 密集连接，每层连接所有前层 | 优于 ResNet |
| EfficientNet | 2019 | 用 NAS 搜索最优深度/宽度/分辨率比例 | State-of-the-art (当时) |

### ResNet 残差块

```
输入 x ──→ Conv → BN → ReLU → Conv → BN ──→ ⊕ → ReLU → 输出
  └──────────────────────────────────────────┘
            恒等映射 (跳跃连接)
```

> 核心思想：学习残差 F(x) = H(x) - x 比学习 H(x) 本身更容易。这让 1000+ 层的网络可训练。

## 目标检测

### 两阶段检测器（更准确，更慢）

| 算法 | 核心思想 |
|------|----------|
| **R-CNN 系列** | 区域提议 → CNN 分类。Fast R-CNN → Faster R-CNN（RPN 端到端） |

### 单阶段检测器（更快）

| 算法 | 核心思想 |
|------|----------|
| **YOLO** (You Only Look Once) | 一次性回归边界框和类别。速度极快，适合实时应用 |
| **SSD** | 多尺度特征图预测。平衡速度和精度 |

### YOLO 版本演进

| 版本 | 年份 | 特点 |
|------|------|------|
| YOLOv1 | 2016 | 开创性端到端检测 |
| YOLOv3 | 2018 | 多尺度预测，特征金字塔 |
| YOLOv5 | 2020 | PyTorch 实现，工程优化好 |
| YOLOv8 | 2023 | Ultralytics 维护，易于使用 |
| YOLOv10+ | 2024- | 更强的性能和效率 |

## 语义分割

| 算法 | 核心思想 |
|------|----------|
| **FCN** | 全卷积网络，将全连接层替换为卷积 |
| **U-Net** | 编码器-解码器 + 跳跃连接，医学图像分割标准 |
| **DeepLab** | 空洞卷积（Atrous Convolution）扩大感受野 |
| **SegFormer** | 基于 Transformer 的分割 |

## 视觉 Transformer (ViT)

2020 年起，Transformer 架构进入 CV 领域：

| 方法 | 核心思想 |
|------|----------|
| **ViT** | 将图像切成 patch，当作 token 序列输入 Transformer |
| **Swin Transformer** | 层级结构 + 窗口注意力，更适合 CV 的层次化特征 |
| **DETR** | Transformer 端到端目标检测 |
| **CLIP** | 图像-文本对比学习（OpenAI） |
| **SAM** (Segment Anything) | 提示式通用分割（Meta） |

> 2024-2026 年趋势：ViT 和 CNN 混合架构逐渐成为主流，纯 CNN 的优势在缩小。

## 图像生成

### 主要方法

| 方法 | 代表模型 | 原理 |
|------|----------|------|
| **GAN** (生成对抗网络) | StyleGAN | 生成器 vs 判别器对抗训练 |
| **VAE** (变分自编码器) | Stable Diffusion 的前身 | 编码到隐空间再解码 |
| **Diffusion Models** | Stable Diffusion, DALL-E 3, Midjourney | 学习逆向去噪过程 |
| **Autoregressive** | DALL-E 1, Parti | 逐像素/逐 token 生成 |

> 2026 年，**Diffusion 模型是图像生成的主流**，Stable Diffusion (开源) 和 Midjourney/DALL-E (商业) 是最广泛使用的工具。

### Stable Diffusion 的工作原理

```
文本提示 → CLIP 文本编码器 → 文本嵌入
                                        ↓
随机噪声 → U-Net (去噪) ← 文本条件 ←——→ 逐步去噪 → 输出图像
```
