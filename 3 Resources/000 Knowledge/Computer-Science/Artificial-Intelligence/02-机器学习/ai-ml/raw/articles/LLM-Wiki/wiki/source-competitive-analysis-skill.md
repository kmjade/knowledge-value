# 競品分析 Skill 資料摘要

> [!summary] 摘要
> 使用 Claude Code 配合競品分析 Skill 進行自動化競品分析，將 5 個競品的分析時間從 2 天壓縮到 2 小時。

---

## 核心要點

### 工作流程

```
定義競品列表
    ↓
Agent Browser 抓取信息
    ↓
Skills 分析處理
    ↓
生成結構化報告
```

### 效率提升

| 指標 | 傳統方式 | 自動化方式 |
|------|----------|------------|
| 5 個競品 | 2 天 | 2 小時 |
| 信息收集 | 手動搜索 | 自動抓取 |
| 分析報告 | 手動撰寫 | 自動生成 |

---

## 關鍵洞見

### Skill 的價值

- 沉澱業務經驗
- 讓分析更結構化
- 可重複使用

### 自動化的邊界

- AI 負責：信息收集、結構化分析
- 人類負責：戰略判斷、決策
