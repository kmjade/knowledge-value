## 方式一：使用镜像站（推荐）

### 推荐镜像站

| 镜像站 | 地址 | 特点 |
|--------|------|------|
| Gemini 镜像站 | [xsimplechat.com](https://xsimplechat.com) | 稳定快速，推荐首选 |
| Gemini 中文版 | [ai.huoyachat.com](https://ai.huoyachat.com) | 中文界面优化 |
| Gemini 官网镜像 | [gptokk.com](https://www.gptokk.com) | 完整官方功能 |

### 使用步骤

1. 访问上述任一镜像站
2. 使用 Google 账号登录
3. 开始使用 Gemini

---

## 方式二：使用 API

### 获取 API Key

1. 访问 [AI Studio](https://aistudio.google.com/apikey)（需科学上网）
2. 创建 API Key
3. 在国内直接调用 API

### Python 示例

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")
model = genai.GenerativeModel('gemini-2.5-pro')

response = model.generate_content("你好")
print(response.text)
```

> [!tip] 提示
> API 调用在国内可以直接访问，无需代理。

## 方式三：CLI 工具

### 配置代理

```bash
# 设置代理
export HTTP_PROXY=http://127.0.0.1:7890
export HTTPS_PROXY=http://127.0.0.1:7890

# 运行 Gemini CLI
gemini
```

### 或使用 API Key

```bash
# 设置 API Key
export GEMINI_API_KEY=your-api-key

# 运行（无需代理）
gemini
```

## 各方式对比

### 镜像站

**优点**
- 无需配置
- 中文界面友好
- 功能完整

**缺点**
- 可能有访问限制
- 部分功能可能延迟

### API

**优点**
- 可编程控制
- 国内直连
- 稳定可靠

**缺点**
- 需要开发能力
- 有配额限制

### 代理

**优点**
- 官方完整体验
- 无功能限制

**缺点**
- 需要付费代理
- 可能不稳定

## 常见问题

### 镜像站打不开？

1. 尝试其他镜像站
2. 清除浏览器缓存
3. 检查网络连接

### API 调用失败？

1. 检查 API Key 是否正确
2. 确认配额是否用尽
3. 查看错误信息

### 如何选择？

| 需求 | 推荐 |
|------|------|
| 日常聊天 | 镜像站 |
| 开发应用 | API |
| 完整功能 | 代理 + 官网 |
