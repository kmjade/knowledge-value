# OpenClaw Pi 代理

> [!info] 概述
> Pi 是 OpenClaw 内置的 AI 代理框架，让你可以快速部署和使用 AI 助手。

## 什么是 Pi

Pi (Personal Intelligence) 是 OpenClaw 的默认代理：
- 开箱即用，无需配置
- 支持多种模型提供商
- 可扩展工具和能力

---

## Pi 配置

### 基础配置

```json
{
  "agents": {
    "pi": {
      "provider": "anthropic",
      "model": "claude-sonnet-4-6",
      "enabled": true
    }
  }
}
```

| 配置项 | 说明 |
|--------|------|
| `provider` | 模型提供商 |
| `model` | 模型名称 |
| `enabled` | 是否启用 |

### 完整配置

```json
{
  "agents": {
    "pi": {
      "provider": "anthropic",
      "model": "claude-sonnet-4-6",
      "systemPrompt": "你是一个有帮助的AI助手。",
      "tools": ["web_search", "code_interpreter"],
      "maxTokens": 4096,
      "temperature": 0.7
    }
  }
}
```

## 模型配置

### 支持的提供商

| 提供商 | 配置值 | 模型示例 |
|--------|--------|----------|
| Anthropic | `anthropic` | claude-sonnet-4-6 |
| OpenAI | `openai` | gpt-4o |
| DeepSeek | `deepseek` | deepseek-chat |
| Google | `google` | gemini-pro |

### 模型选择

```bash
# CLI 切换模型
openclaw agent set pi --model claude-opus-4-7
```

## 工具配置

### 启用工具

```json
{
  "agents": {
    "pi": {
      "tools": [
        "web_search",
        "code_interpreter",
        "file_manager"
      ]
    }
  }
}
```

### 工具权限

```json
{
  "agents": {
    "pi": {
      "tools": {
        "web_search": { "enabled": true },
        "code_interpreter": { "enabled": true },
        "file_manager": {
          "enabled": true,
          "allowedPaths": ["/home/user/documents"]
        }
      }
    }
  }
}
```

## 会话管理

### 会话配置

```json
{
  "agents": {
    "pi": {
      "session": {
        "maxHistory": 50,
        "contextWindow": 128000,
        "summarizeThreshold": 100000
      }
    }
  }
}
```

| 配置项 | 说明 |
|--------|------|
| `maxHistory` | 最大历史消息数 |
| `contextWindow` | 上下文窗口大小 |
| `summarizeThreshold` | 自动摘要阈值 |

### 清除会话

```bash
# 通过消息渠道
/reset

# CLI 命令
openclaw session clear --agent pi
```

## 多代理路由

### 配置多个代理

```json
{
  "agents": {
    "pi": {
      "provider": "anthropic",
      "model": "claude-sonnet-4-6"
    },
    "coder": {
      "provider": "anthropic",
      "model": "claude-sonnet-4-6",
      "systemPrompt": "你是一个专业的编程助手。"
    },
    "writer": {
      "provider": "openai",
      "model": "gpt-4o",
      "systemPrompt": "你是一个专业的写作助手。"
    }
  }
}
```

### 切换代理

```bash
# CLI 切换
openclaw agent use coder

# 消息中切换
/agent coder
```

## API 调用

### 通过 API 使用 Pi

```bash
curl -X POST http://localhost:18789/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "agent": "pi",
    "message": "你好，请介绍一下你自己"
  }'
```

### 流式响应

```bash
curl -X POST http://localhost:18789/api/chat/stream \
  -H "Content-Type: application/json" \
  -d '{
    "agent": "pi",
    "message": "写一首关于春天的诗"
  }'
```

## 高级配置

### 响应缓存

```json
{
  "agents": {
    "pi": {
      "cache": {
        "enabled": true,
        "ttl": 3600
      }
    }
  }
}
```

### 重试策略

```json
{
  "agents": {
    "pi": {
      "retry": {
        "maxAttempts": 3,
        "backoff": "exponential"
      }
    }
  }
}
```

## 命令行操作

```bash
# 查看代理列表
openclaw agent list

# 查看代理详情
openclaw agent show pi

# 创建新代理
openclaw agent create my-agent

# 删除代理
openclaw agent delete my-agent

# 测试代理
openclaw agent test pi --message "Hello"
```
