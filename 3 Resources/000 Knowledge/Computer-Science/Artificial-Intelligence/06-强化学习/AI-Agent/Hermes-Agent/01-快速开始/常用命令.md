# Hermes Agent CLI 命令速查表

> [!summary]
> Hermes Agent 的 CLI 是一个完整的终端用户界面（TUI），支持多行编辑、斜杠命令自动补全、对话历史、中断重定向和流式工具输出。

---

## 启动命令

```bash
hermes                          # 交互式会话（默认）
hermes chat -q "Hello"          # 单次查询
hermes -m model/name            # 指定模型
hermes --provider deepseek       # 指定提供商
hermes --toolsets "web,terminal" # 指定工具集
hermes -s skill-name             # 预加载技能
hermes -s skill1,skill2          # 多个技能
hermes --continue                # 恢复最近会话
hermes --resume <id>             # 恢复指定会话
hermes -w                        # worktree 模式（并行智能体）
hermes --yolo                    # 跳过危险命令审批
hermes chat --verbose            # 详细调试输出
```

---

## 快捷键

| 键 | 动作 |
|----|------|
| `Enter` | 发送消息 |
| `Alt+Enter` / `Ctrl+J` | 插入换行（多行输入） |
| `Tab` | 接受自动建议 / 补全斜杠命令 |
| `Ctrl+C` | 中断 Agent（2秒内双击强制退出） |
| `Ctrl+D` | 退出 |
| `Ctrl+Z` | 挂起到后台（Unix），`fg` 恢复 |
| `Ctrl+B` | 语音录制开关（需启用 voice 模式） |
| `Ctrl+V` / `Alt+V` | 粘贴文本/图片 |

### 中断 Agent

- Agent 工作时输入新消息 + Enter → 中断当前操作，立即处理新指令
- `Ctrl+C` → 中断当前操作
- 待处理的终端命令会立即终止（SIGTERM → 1秒后 SIGKILL）

## 状态栏

```
⚕ claude-sonnet-4 │ 12.4K/200K │ [██████░░░░] 6% │ $0.06 │ 15m
```

| 元素 | 说明 |
|------|------|
| 模型名称 | 当前模型（超过26字符截断） |
| Token 数量 | 已用 / 最大上下文窗口 |
| 上下文条 | 颜色编码填充指示器 |
| 成本 | 预估会话成本 |
| 时长 | 已过会话时间 |

### 上下文颜色编码

| 颜色 | 阈值 | 含义 |
|------|------|------|
| 🟢 绿色 | < 50% | 空间充足 |
| 🟡 黄色 | 50-80% | 逐渐填满 |
| 🟠 橙色 | 80-95% | 接近上限 |
| 🔴 红色 | ≥ 95% | 接近溢出——建议 `/compress` |

## 斜杠命令

输入 `/` 弹出自动补全下拉菜单。技能自动注册为斜杠命令。

### 会话控制

| 命令 | 说明 |
|------|------|
| `/new` | 新会话 |
| `/title <name>` | 命名当前会话 |
| `/model [name]` | 显示或切换模型 |
| `/compress` | 手动压缩上下文 |
| `/undo` | 撤销上一次交换 |
| `/retry` | 重发上一次消息 |
| `/clear` | 清屏 + 新会话 |
| `/rollback [N]` | 恢复文件系统快照 |
| `/resume [name]` | 恢复命名会话 |

### 后台任务

| 命令 | 说明 |
|------|------|
| `/background <prompt>` | 在后台独立会话中运行提示 |
| `/agents` | 查看活跃 Agent 和运行中任务 |

后台任务在守护线程中运行完全独立的 Agent 会话，不影响主会话。完成后结果以面板形式显示。

### 配置与工具

| 命令 | 说明 |
|------|------|
| `/tools` | 管理工具开关 |
| `/skill <name>` | 加载技能 |
| `/skills browse` | 浏览技能中心 |
| `/reload` | 重载 .env 环境变量 |
| `/config` | 查看当前配置 |

### 显示与风格

| 命令 | 说明 |
|------|------|
| `/skin [name]` | 切换 CLI 主题 |
| `/personality <name>` | 切换个性（15 种内置 + 自定义） |
| `/verbose` | 循环：off → new → all → verbose |
| `/reasoning <level>` | 推理程度：none/minimal/low/medium/high/xhigh |
| `/voice on/off/tts` | 语音模式 |
| `/yolo` | 切换审批绕过 |
| `/indicator [style]` | 切换 TUI 动画风格 |
| `/statusbar` | 切换状态栏 |

### 个性列表

`helpful` `concise` `technical` `creative` `teacher` `kawaii` `catgirl` `pirate` `shakespeare` `surfer` `noir` `uwu` `philosopher` `hype`

## 快捷命令 (Quick Commands)

在 config.yaml 中定义无 LLM 调用的快捷命令：

```yaml
# ~/.hermes/config.yaml
quick_commands:
  status:
    type: exec
    command: systemctl status hermes-agent
  gpu:
    type: exec
    command: nvidia-smi --query-gpu=utilization.gpu,memory.used --format=csv,noheader
```

然后在聊天中输入 `/status` 或 `/gpu` 即可直接执行。

## 多行输入

两种方式：

1. `Alt+Enter` 或 `Ctrl+J` — 插入换行
2. 反斜杠续行 — 以 `\` 结尾表示继续：

```
Write a function that:\
1. Takes a list of numbers\
2. Returns the sum
```

## 忙碌输入模式

控制 Agent 工作时按 Enter 的行为：

```yaml
# ~/.hermes/config.yaml
display:
  busy_input_mode: "interrupt"  # 默认：中断并处理新消息
  # busy_input_mode: "queue"    # 排队：等当前任务完成后再处理
```

## 常用配置

```bash
# 工具预览截断
hermes config set display.tool_preview_length 80

# 完成时响铃
hermes config set display.bell_on_complete true

# 上下文压缩
hermes config set compression.enabled true
hermes config set compression.threshold 0.50

# 模型切换
hermes model
```

## 会话管理

```bash
hermes sessions list         # 列出会话
hermes sessions browse       # 交互式浏览
hermes sessions rename ID T  # 重命名
hermes sessions delete ID    # 删除
hermes sessions export OUT   # 导出 JSONL
hermes sessions stats        # 统计
```

## 安装与更新

```bash
hermes update          # 更新到最新版
hermes doctor          # 健康检查
hermes doctor --fix    # 自动修复
hermes uninstall       # 卸载
```
