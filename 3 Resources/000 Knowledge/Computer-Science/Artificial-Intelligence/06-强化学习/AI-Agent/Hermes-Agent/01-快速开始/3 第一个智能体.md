# 第一个智能体

> [!summary] 概述
> 启动 Hermes Agent 并完成第一个任务——Hermes Agent 是一个 AI 智能体框架，你通过自然语言对话让它帮你完成工作。

---

## 启动 Hermes Agent

### 方法一：交互式会话

```bash
hermes
```

启动后你会进入一个交互式聊天界面。输入任何问题或任务，Hermes 会自动使用合适的工具。

### 方法二：单次查询

```bash
hermes chat -q "帮我查看当前目录下有哪些文件"
```

### 方法三：带上技能预设

```bash
# 加载 Python 调试技能
hermes -s python-debugpy

# 加载多个技能
hermes -s python-debugpy,code-review
```

---

## 第一个对话任务

### 任务 1：文件浏览

启动 `hermes` 后，尝试：

```
列出当前目录下的所有文件，按修改时间排序
```

Hermes 会自动使用 `search_files` 工具来完成。

### 任务 2：代码分析

```
阅读 src/main.py，分析它的主要功能和潜在问题
```

Hermes 会使用 `read_file` 工具读取文件，然后给出分析。

### 任务 3：Shell 命令

```
检查磁盘使用情况，告诉我哪些目录占用空间最大
```

Hermes 会使用 `terminal` 工具执行 `du` 命令。

### 任务 4：创建文件

```
创建一个 Python 脚本 hello.py，打印 "Hello, Hermes!"
```

Hermes 会使用 `write_file` 工具创建文件。

## 会话中的常用命令

在 `hermes` 交互式会话中，你可以使用斜杠命令：

| 命令 | 说明 |
|------|------|
| `/help` | 查看所有命令 |
| `/model model-name` | 切换模型 |
| `/skill skill-name` | 加载技能 |
| `/tools` | 管理工具开关 |
| `/new` | 开始新会话 |
| `/history` | 查看对话历史 |
| `/save` | 保存对话到文件 |
| `/quit` | 退出 |

## 会话管理

```bash
# 列出最近会话
hermes sessions list

# 恢复上一次会话
hermes --continue

# 恢复特定会话
hermes --resume <session-id>

# 搜索历史会话
# 在会话中直接问：我们之前关于XX的讨论结果是什么？
```
