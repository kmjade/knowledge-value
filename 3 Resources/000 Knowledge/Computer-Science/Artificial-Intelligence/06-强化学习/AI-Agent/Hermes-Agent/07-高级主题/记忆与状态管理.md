# 记忆与状态管理

> [!info]
> Hermes Agent 的 Memory 系统支持跨会话持久记忆。

```bash
hermes memory setup    # 配置记忆后端
hermes memory status   # 查看状态
```

可插拔后端：内置 SQLite、Honcho、Mem0 等。
