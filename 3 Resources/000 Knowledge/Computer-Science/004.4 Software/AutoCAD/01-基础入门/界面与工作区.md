# 界面与工作区

> [!summary] 功能概述
> AutoCAD 2026 的用戶界面組成與工作空間配置，是學習 AutoCAD 的第一步。

---

## 功能介紹

### 功能定位
- **所屬模塊**：基礎入門
- **功能類型**：環境配置
- **前置條件**：已安裝 AutoCAD 2026

### 應用場景
1. 初次使用 AutoCAD 時的界面熟悉
2. 切換不同工作空間以適應不同任務
3. 自定義界面佈局以提高工作效率

---

## 界面組成

### 1. 應用程序菜單（Application Menu）

位於左上角的紅色 "A" 圖標，包含：
- 新建 (New)
- 打開 (Open)
- 保存 (Save)
- 另存為 (Save As)
- 導出 (Export)
- 打印 (Print)

### 2. 快速訪問工具欄（Quick Access Toolbar）

位於應用程序菜單右側，默認包含：
- 新建、打開、保存
- 撤銷、重做
- 打印

> [!tip] 自定義快速訪問工具欄
> 右鍵點擊工具欄 → 自定義快速訪問工具欄 → 添加常用命令

### 3. 功能區（Ribbon）

主要的命令訪問區域，由多個選項卡組成：

| 選項卡 | 主要功能 |
|--------|---------|
| 默認 (Home) | 繪圖、修改、圖層、標註 |
| 插入 (Insert) | 塊、外部參照、點雲 |
| 註釋 (Annotate) | 文字、標註、表格 |
| 參數化 (Parametric) | 幾何約束、標註約束 |
| 視圖 (View) | 視圖控制、視口 |
| 管理 (Manage) | 應用程序、自定義 |
| 輸出 (Output) | 打印、導出 |

### 4. 繪圖區域（Drawing Area）

- 中央的黑色區域為繪圖空間
- 左下角顯示坐標系圖標 (UCS)
- 包含視圖控制按鈕（右上角）

### 5. 命令行（Command Line）

位於底部，用於：
- 輸入命令
- 查看提示信息
- 選擇命令選項

> [!tip] 快捷鍵
> `Ctrl + 9` 可顯示/隱藏命令行

### 6. 狀態欄（Status Bar）

位於最底部，包含：
- 坐標顯示
- 繪圖輔助工具（柵格、捕捉、正交等）
- 視圖控制工具

## 工作空間

### 預設工作空間

AutoCAD 2026 提供多種預設工作空間：

| 工作空間 | 適用場景 |
|---------|---------|
| 草圖與註釋 | 二維繪圖與標註 |
| 三維基礎 | 三維建模入門 |
| 三維建模 | 三維建模高級 |
| AutoCAD 經典 | 傳統界面風格 |

### 切換工作空間

**方法一**：快速訪問工具欄
1. 點擊快速訪問工具欄右側的下拉箭頭
2. 選擇「工作空間」
3. 選擇所需的工作空間

**方法二**：狀態欄
1. 點擊狀態欄右側的齒輪圖標
2. 選擇所需的工作空間

**方法三**：命令行
```
WSCURRENT
輸入工作空間名稱: 草圖與註釋
```

### 自定義工作空間

1. 配置界面佈局
2. 點擊快速訪問工具欄下拉箭頭
3. 選擇「將當前工作空間另存為」
4. 輸入工作空間名稱

## 實用技巧

### 界面個性化

| 設置項 | 路徑 |
|--------|------|
| 背景顏色 | 選項 → 顯示 → 顏色 |
| 字體大小 | 選項 → 顯示 → 字體 |
| 光標大小 | 選項 → 顯示 → 十字光標大小 |
| 功能區位置 | 拖動功能區標籤 |

### 常用設置

> [!tip] 機械製圖推薦設置
> - 背景顏色：白色或淺灰色（便於打印預覽）
> - 十字光標大小：100（滿屏十字線）
> - 顯示線寬：開啟

---

## 注意事項

> [!warning] 重要提示
> - 不要過度自定義界面，保持與標準一致便於協作
> - 定期保存工作空間設置
> - 學會使用鍵盤快捷鍵，減少鼠標操作
