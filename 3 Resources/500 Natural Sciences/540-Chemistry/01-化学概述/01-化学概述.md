# 01-化学概述

> 化学 = 「中心科学」——连接物理和生物——研究物质及其变化。

## 化学的历史

| 时期 | 核心 |
|------|------|
| 炼金术 | 寻找贤者之石——把铅变金——积累了实验技术 |
| 拉瓦锡 (1780s) | 质量守恒——推翻燃素说——化学成为定量科学 |
| 道尔顿 (1808) | 原子论——每种元素有独特原子量 |
| 门捷列夫 (1869) | 元素周期表——预言了未发现元素的性质 |
| 20世纪 | 量子化学——理解化学键的本质 |

## 化学的分支

| 分支 | 研究对象 |
|------|------|
| 有机化学 | 含碳化合物——生命分子·药物·塑料 |
| 无机化学 | 金属·矿物·催化剂 |
| 物理化学 | 热力学·量子·动力学 |
| 分析化学 | 检测和测量 |
| 生物化学 | 蛋白质·DNA·代谢 |
