# 09-化学与生活

> 化学无处不在——从你早上刷牙到你晚上吃药。

## 药物

| 概念 | 说明 |
|------|------|
| 阿司匹林 | 最成功的药物——1899上市——每年消耗4万吨 |
| 青霉素 | 弗莱明1928偶然发现——第一个抗生素——挽救了上亿人 |
| 靶向药 | 不杀所有快速分裂的细胞——精准攻击癌细胞特征 |
| 药物研发 | 平均10年+$10亿——从1万个候选到1个上市 |

## 材料

| 材料 | 化学 |
|------|------|
| 塑料 | 聚乙烯·PET·尼龙——高分子化学——白色污染危机 |
| 半导体 | 硅——超纯(99.9999999%) ——掺杂改变电性 |
| 复合材料 | 碳纤维+树脂——比钢强比铝轻——飞机·F1 |

## 日常化学

| 肥皂: 一端亲水一端亲油——包裹油污——水冲走 |
|------|
| 防晒霜: 有机(吸收UV) + 无机(ZnO/TiO₂·反射UV) |
| 发酵: 酵母把糖变酒精和CO₂——面包·啤酒·白酒——最古老的化学工业 |
