# 08-分析化学

> 「这是什么？有多少？」——化学检测的科学。

## 主要技术

| 技术 | 原理 | 用途 |
|------|------|------|
| **色谱** (HPLC·GC) | 混合物在固定相和流动相中分配 | 分离和定量混合物 |
| **质谱** (MS) | 离子按质荷比分离 | 鉴定分子·蛋白质组学 |
| **光谱** (UV-Vis·IR) | 物质吸收/发射特定波长 | 结构鉴定 |
| **NMR** | 原子核在磁场中共振 | 有机结构测定——最强大的工具 |
| **X射线衍射** | 晶体衍射X射线 | 三维结构——DNA双螺旋就是这样发现的 |

## 色谱

| 混合物→通过色谱柱→不同组分以不同速度移动→分离→检测 |
|------|
| 气相色谱(GC): 挥发·热稳定的——香料·农药 |
| 液相色谱(HPLC): 不挥发——药物·蛋白质——应用最广 |

## 分析化学的应用

| 法医: 毒物检测·DNA·指纹化学 |
|------|
| 环境: 水中污染物·空气PM2.5·土壤重金属 |
| 食品: 农药残留·添加剂·营养成分 |
| 兴奋剂检测: 奥运会的分析化学 |
