# 06-无机化学

> 碳以外的化学——金属·矿物·配位化合物。

## 主要领域

| 领域 | 内容 |
|------|------|
| 主族元素化学 | s区和p区元素——Na·Al·Si·P·S·Cl |
| 过渡金属化学 | d区元素——颜色丰富——多价态——催化 |
| 配位化学 | 金属离子被配体包围——[Fe(CN)₆]⁴⁻ |
| 固态化学 | 晶体结构——半导体·超导·磁性材料 |

## 过渡金属的特征

| 特征 | 原因 |
|------|------|
| 颜色丰富 | d-d电子跃迁——吸收可见光——CuSO₄蓝色 |
| 多种氧化态 | Fe²⁺/Fe³⁺·Mn²⁺~Mn⁷⁺ |
| 催化 | 提供反应表面或中间体——Haber法用Fe |
| 磁性 | 未成对电子——顺磁性·铁磁性 |

## 配位化合物

| 中心金属+配体——配位数常见4或6 |
|------|
| 叶绿素(Mg²⁺)·血红素(Fe²⁺)·维生素B12(Co³⁺) ——都是天然配位化合物 |
| 顺铂 [PtCl₂(NH₃)₂] ——抗癌药物——DNA交联 |
