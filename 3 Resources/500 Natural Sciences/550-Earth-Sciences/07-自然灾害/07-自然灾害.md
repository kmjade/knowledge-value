# 07-自然灾害

> 地球的力量——美丽且致命。

## 地震

| 每年~50万次——有感的~10万次——破坏性的~100次 |
|------|
| 最大记录: 1960智利9.5级 |
| 死亡最多: 1556陕西地震——83万人(估计) |
| 建筑规范是最好的防震——「地震不杀人——建筑杀人」 |

## 海啸

| 海底地震/滑坡→水体位移→长波长(数百km) ——海上是小波浪——到岸边变成水墙 |
|------|
| 2004印度洋海啸: 9.1级——23万人死亡——14国受影响 |

## 火山

| 喷发类型 | 特征 |
|------|------|
| 夏威夷式 | 温和——熔岩流——可预测 |
| 普林尼式 | 爆炸性——火山灰柱到平流层——维苏威(79AD掩埋庞贝) |
| 超级火山 | 文明威胁级——黄石·多巴——全球冬天 |
