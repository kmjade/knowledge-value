# 08-气候变化

> 地球正在变暖——科学共识是明确的——争论在于怎么办。

## 证据

| 全球平均温度: 相比工业化前上升~1.2°C——过去十年是最热的十年 |
|------|
| CO₂: 从280 ppm(1750) → 420 ppm(2023) ——300万年来最高 |
| 冰芯: 南极冰芯记录80万年的CO₂——现在远超自然范围 |

## 影响

| 影响 | 现状 |
|------|------|
| 极端天气 | 热浪·洪水·干旱频率和强度增加 |
| 冰川融化 | 北极海冰每十年减少13%——格陵兰每天融化~3000亿吨 |
| 生态系统 | 珊瑚白化·物种迁徙·物候改变 |
| 粮食安全 | 某些地区减产——威胁全球粮食供应 |

## 应对

| 减缓 (Mitigation) | 减少排放——可再生能源·碳定价·碳汇 |
|------|------|
| 适应 (Adaptation) | 面对不可避免的变化——海堤·抗旱作物·预警系统 |
| 巴黎协定 (2015) | 目标<2°C——争取<1.5°C——各国自愿承诺——不够 |

## 争议与挑战

| 不是「有没有」的争议——是「谁的排放·谁付钱·怎么减」 |
|------|
| 全球北方排放了历史累计的大部分——全球南方承受最大影响——气候正义 |
