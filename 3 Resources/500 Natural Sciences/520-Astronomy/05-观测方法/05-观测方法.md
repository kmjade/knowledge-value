# 05-观测方法

> 望远镜是天文学家的时间机器——看向远方就是看向过去。

## 电磁波谱的窗口

| 波段 | 观测对象 | 代表设备 |
|------|------|------|
| 射电 | 冷气体·脉冲星·宇宙微波背景 | FAST(中国天眼)·ALMA |
| 红外 | 尘埃·恒星形成区·系外行星 | JWST |
| 可见光 | 恒星·星系——最传统 | Hubble·地面望远镜 |
| 紫外/X射线 | 高温气体·黑洞吸积盘·超新星遗迹 | Chandra·XMM-Newton |
| γ射线 | 最剧烈的事件——GRB | Fermi |

## 多信使天文学

| 2017: GW170817——首次同时探测到引力波+电磁波——双中子星合并 |
|------|
| 引力波 (LIGO/Virgo): 时空的涟漪——黑洞合并·中子星合并 |
| 中微子 (IceCube): 穿透一切——从超新星和活动星系核来 |
| 宇宙线: 高能粒子——来源仍不完全清楚 |

## 大气的影响

| 为什么把望远镜放太空？——大气吸收·湍流(星星「眨眼」)·光污染 |
|------|
| 自适应光学: 地面望远镜通过变形镜面实时补偿大气湍流——接近太空的清晰度 |
