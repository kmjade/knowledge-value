# 01-天文学概述

> 天文学是最古老的科学——人类仰望星空至少4万年了。

## 从神话到科学

| 时期 | 核心 |
|------|------|
| 史前 | 巨石阵·金字塔——天文观测用于历法和宗教 |
| 古希腊 | 托勒密地心说——本轮均轮——预测行星位置 |
| 哥白尼 (1543) | 日心说——地球不是中心——「哥白尼革命」 |
| 开普勒 (1609) | 行星运动三定律——椭圆轨道——不是完美的圆 |
| 伽利略 (1609) | 望远镜指向天空——木星的卫星·金星的相位——颠覆 |
| 牛顿 (1687) | 万有引力——同一规律支配苹果和月球 |

## 天文学的尺度

| 单位 | 定义 | 例子 |
|------|------|------|
| AU (天文单位) | 日地距离 ~1.5亿km | 太阳-木星: 5.2 AU |
| 光年 | 光走一年的距离 ~9.46万亿km | 最近的恒星: 4.2光年 |
| 秒差距 (pc) | 3.26光年 | 银河系直径: ~30 kpc |

## 天文学的分支

| 分支 | 研究对象 |
|------|------|
| 行星科学 | 太阳系天体 |
| 恒星天文 | 恒星的结构和演化 |
| 星系天文 | 星系的结构和动力学 |
| 宇宙学 | 宇宙整体的起源和演化 |
