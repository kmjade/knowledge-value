# 04-星系与宇宙

> 数千亿颗恒星的集合——银河系只是可观测宇宙中2万亿星系中的一个。

## 银河系

| 特征 | 数据 |
|------|------|
| 类型 | 棒旋星系 |
| 直径 | ~10万光年 |
| 恒星数 | 1000-4000亿 |
| 中心 | 超大质量黑洞 (人马座A*, ~400万M☉) |
| 我们的位置 | 猎户臂——距中心~2.6万光年 |

## 星系分类 (哈勃序列)

| 类型 | 特征 |
|------|------|
| 椭圆 | 年老·红色·无气体——恒星随机运动 |
| 旋涡 | 盘+旋臂——气体丰富——恒星形成活跃 |
| 不规则 | 无规则形状——往往是被引力扰乱的结果 |

## 暗物质

| 星系旋转太快——可见物质不足以提供引力——应该飞散——但没有 |
|------|
| 暗物质 ~27%宇宙——只通过引力作用——不知道是什么粒子 |
| 候选粒子: WIMPs·轴子——寻找了几十年——还没找到 |

## 大尺度结构

| 星系不是均匀分布的——形成纤维状「宇宙网」——星系团·超星系团——之间是巨大的空洞(voids) |
