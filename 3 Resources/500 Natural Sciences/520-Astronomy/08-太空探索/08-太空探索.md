# 08-太空探索

> 从斯普特尼克到SpaceX——人类走出摇篮的征程。

## 里程碑

| 年份 | 事件 |
|:---:|------|
| 1957 | 斯普特尼克1号——第一颗人造卫星——太空时代开始 |
| 1961 | 加加林——第一个进入太空的人 |
| 1969 | 阿波罗11号——人类首次登月——「一小步」 |
| 1977 | 旅行者号发射——至今仍在飞行——已进入星际空间 |
| 1990 | 哈勃太空望远镜——革命性的图像 |
| 1998 | 国际空间站(ISS)开始建设——至今最大的太空合作 |
| 2021 | JWST发射——哈勃的继任者——红外波段 |

## 当代太空竞赛

| 玩家 | 特征 |
|------|------|
| NASA | 阿尔忒弥斯计划——重返月球——最终目标火星 |
| SpaceX | 可回收火箭——星舰——火星殖民愿景 |
| 中国 | 嫦娥探月·天宫空间站·火星车祝融——系统推进 |
| Blue Origin | 新谢泼德(亚轨道)·新格伦——缓慢但稳定 |

## 火星任务

| 为什么是火星？——最近的可能曾经有生命的行星——有大气(薄)·有水冰 |
|------|
| 挑战: 辐射·微重力(骨骼流失)·心理(隔离·距离——通信延迟4-24分钟) |
| NASA目标2030s载人登火——SpaceX更激进——这代人可能看到 |

## 商业航天

| SpaceX降低发射成本~10倍——可回收火箭革命 |
|------|
| 卫星互联网 (Starlink) ——数万颗卫星——天文观测者的噩梦(光污染) |
| 太空旅游: 亚轨道($25万+)起步——目前是超级富豪的玩具 |
