# 02-太阳系

> 我们的宇宙家园——一颗恒星、八颗行星、数百颗卫星、无数小行星和彗星。

## 太阳系家族

| 天体 | 特征 |
|------|------|
| **太阳** | 99.86%的质量——G型主序星——通过核聚变发光发热 |
| **水星** | 最小·最近——无大气——温差600°C |
| **金星** | 「地球的邪恶双胞胎」——失控温室效应——表面460°C |
| **地球** | 唯一已知有生命的行星——液态水·板块构造·月球 |
| **火星** | 红色星球——曾经有液态水——人类下一个目标 |
| **木星** | 最大——大红斑(持续300年的风暴)——79+卫星 |
| **土星** | 壮观的光环——密度小于水(理论上能浮) |
| **天王星** | 「躺着转」——轴倾斜98° |
| **海王星** | 最远——太阳系最高的风速(2100km/h) |

## 小行星带与柯伊伯带

| 小行星带 (火星-木星之间) | 大部分是岩石——谷神星(最大·矮行星) |
|------|------|
| 柯伊伯带 (海王星外) | 冰质天体——冥王星曾被认为第九大行星——2006年降级为矮行星 |

## 太阳系的形成

| ~46亿年前: 分子云坍缩→原太阳+原行星盘→行星通过吸积形成 |
|------|
| 内太阳系: 高温→岩石行星(水·金·地·火) |
| 外太阳系: 低温→气体巨行星(木·土)+冰巨行星(天·海) |
