# 09-动物保护

> 人类世的大灭绝——我们如何保护剩下的？

## 受威胁程度

| IUCN红色名录: |
|------|
| 灭绝(EX)·野外灭绝(EW)·极危(CR)·濒危(EN)·易危(VU)·近危(NT)·无危(LC) |
| 评估了~15万种——其中~28%受威胁 |

## 标志性保护案例

| 物种 | 状况 | 恢复原因 |
|------|:---:|------|
| 大熊猫 | 易危(从濒危降级) | 栖息地保护·圈养繁殖·「旗舰物种」 |
| 美洲野牛 | 近危(从~300到~50万) | 保护区和私人牧场 |
| 加州秃鹰 | 极危(从27到~500) | 圈养繁殖·全部个体都被追踪 |
| 北部白犀牛 | 仅剩2头(雌性) | 功能性灭绝——正在尝试IVF |

## 保护的挑战

| 挑战 | 说明 |
|------|------|
| 人兽冲突 | 老虎吃牛·大象毁庄稼——当地人为什么要保护？——需要补偿机制 |
| 非法贸易 | 象牙·犀牛角·虎骨——年交易额~$200亿——有组织犯罪 |
| 筹资 | 保护资金严重不足——发展中国家保护区「纸上公园」 |
| 「魅力物种」偏差 | 熊猫得到最多关注——两栖类正在无声地灭绝 |
