# 04-动物生理

> 动物身体如何运作——从循环到神经。

## 主要系统

| 系统 | 功能 |
|------|------|
| 循环 | 运输氧气·营养·废物——心脏+血管 |
| 呼吸 | 气体交换——鳃(水生)·肺(陆生)·气管(昆虫) |
| 消化 | 分解食物——吸收营养 |
| 神经 | 感知·处理·响应——神经元·脑 |
| 内分泌 | 激素——慢但持久——调节生长·代谢·繁殖 |

## 循环系统对比

| 类型 | 特征 | 代表 |
|------|------|------|
| 开放 | 血液直接浸泡器官——血淋巴 | 昆虫·软体动物 |
| 闭合 | 血液在血管中——高效 | 环节动物·脊椎动物 |
| 二腔心 | 鱼——一条回路 | 鱼 |
| 三腔心 | 两心房一心室——部分混合 | 两栖·大多数爬行 |
| 四腔心 | 完全分离——高效供养 | 鸟·哺乳·鳄鱼 |

## 体温调节

| 变温(ectotherm) | 靠环境调节体温——省能量——但活动受限——「冷血」 |
|------|------|
| 恒温(endotherm) | 内部产热——高代谢——可全天活动——代价: 需要吃很多 |
