# 01-动物学概述

> 动物王国——从海绵到人类——~150万已知物种。

## 动物的特征

| 特征 | 说明 |
|------|------|
| 多细胞 | 有分化的组织和器官 |
| 异养 | 不能光合作用——必须吃 |
| 运动 | 至少生命周期的某个阶段可以主动运动 |
| 神经和肌肉 | 快速响应环境(除了海绵) |
| 有性繁殖 | 大多数——但有例外(孤雌生殖) |

## 动物的系统树

| 海绵→刺胞动物(水母·珊瑚)→扁形动物→软体动物→环节动物(蚯蚓)→节肢动物(昆虫·蜘蛛·虾)→棘皮动物(海星)→脊索动物(鱼·蛙·蜥·鸟·哺乳) |
|------|
