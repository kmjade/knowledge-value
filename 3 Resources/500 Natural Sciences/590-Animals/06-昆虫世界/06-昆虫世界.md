# 06-昆虫世界

> 地球上最成功的动物——占已知动物物种的>75%。

## 昆虫为什么成功？

| 外骨骼 | 轻·坚固——保护——但必须蜕皮才能生长 |
|------|------|
| 飞行 | 唯一的无脊椎飞行者——逃脱·扩散·找食物 |
| 变态 | 幼虫和成虫不竞争——利用不同资源 |
| 体型小 | 更多生态位——更短的世代——快速适应 |
| 高繁殖力 | 一只蚜虫一季可产生数十亿后代(理论上) |

## 主要目

| 目 | 物种 | 特征 |
|------|:---:|------|
| 鞘翅目(甲虫) | 40万 | 「所有动物中上帝最爱甲虫」——Haldane |
| 鳞翅目(蝶·蛾) | 18万 | 翅膀有鳞片——毛虫→蛹→成虫 |
| 膜翅目(蜂·蚁) | 15万 | 社会性——有些可育·大多数不育 |
| 双翅目(蝇·蚊) | 15万 | 一对翅膀——「真苍蝇」 |

## 社会性昆虫

| 蚂蚁·蜜蜂·白蚁: 生殖分工——女王(繁殖)·工蚁/蜂(不育·劳动)——超个体(superorganism) |
|------|
| 蚂蚁战争: 不同蚁群打仗——规模可比人类——策略复杂(包围·伏击) |
