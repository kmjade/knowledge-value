# 03-动物行为

> 动物为什么做它们做的事？——四个为什么(Tinbergen)。

## Tinbergen的四问

| 问题 | 解释类型 |
|------|------|
| 机制 | 刺激-反应——什么触发了行为？ |
| 发育 | 在个体一生中如何发展？——先天 vs 学习 |
| 功能 | 如何帮助生存和繁殖？——适应价值 |
| 演化 | 在演化史上如何起源？——系统发生 |

## 先天 vs 学习

| 先天: 基因编码——不需要经验——蜘蛛织网·鸟迁徙·婴儿吸吮 |
|------|
| 学习: 经验改变行为——印记(Lorenz的鹅)·条件反射·观察学习 |
| 大多数行为是两者结合——基因提供「倾向」——经验塑造「实现」 |

## 社会行为

| 行为 | 例子 |
|------|------|
| 合作 | 狼群围猎·蜜蜂舞蹈·蚂蚁军团 |
| 利他 | 工蜂不育——为女王牺牲——亲缘选择(帮亲属繁殖=帮自己的基因) |
| 冲突 | 争夺资源·领地·配偶 |
