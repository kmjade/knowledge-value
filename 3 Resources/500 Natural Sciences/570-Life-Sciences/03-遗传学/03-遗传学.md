# 03-遗传学

> 基因——遗传信息的基本单位——从豌豆到人类基因组。

## 孟德尔遗传

| 孟德尔(1866)——豌豆实验——7个特征——发现了遗传的数学规律 |
|------|
| 等位基因: 每个基因有两个拷贝——显性(A)和隐性(a) |
| 分离律: 配子只得到一个拷贝 |
| 自由组合: 不同基因独立分配(除非在同一染色体上——连锁) |

## DNA 结构

| Watson & Crick (1953): 双螺旋——两条链——A=T, G≡C |
|------|
| 糖-磷酸骨架 + 碱基对——像螺旋梯 |
| 人类基因组: 30亿碱基对——~2万个基因——但只占DNA的~1.5% |

## 中心法则

| DNA → RNA → 蛋白质 |
|------|
| 转录: DNA→mRNA(在核内) / 翻译: mRNA→蛋白质(在核糖体) |
| 逆转录: 有些病毒RNA→DNA(逆转录酶)——HIV |

## 表观遗传

| 不改变DNA序列——但改变基因表达——可以遗传 |
|------|
| DNA甲基化·组蛋白修饰——环境(饮食·压力)可以影响 |
| 为什么同卵双胞胎不完全相同？——表观遗传 |
