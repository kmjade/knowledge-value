# 02-细胞生物学

> 细胞是生命的基本单位——所有生命都由细胞构成(或曾经是细胞)。

## 细胞学说

| (1) 所有生命由细胞构成 (2) 细胞是生命的基本单位 (3) 所有细胞来自已有细胞 |

## 原核 vs 真核

| | 原核 | 真核 |
|------|------|------|
| 大小 | ~1 μm | ~10-100 μm |
| 核 | 无 | 有(膜包围) |
| 细胞器 | 无膜包围的 | 线粒体·ER·高尔基等 |
| 例子 | 细菌·古菌 | 动物·植物·真菌 |

## 细胞器

| 细胞器 | 功能 |
|------|------|
| 线粒体 | 能量工厂——ATP——「细胞的发电站」 |
| 内质网 (ER) | 蛋白质合成和折叠——粗面(核糖体) +滑面(脂质) |
| 高尔基体 | 分拣·包装·运输——「细胞的邮局」 |
| 溶酶体 | 消化——分解废物——「细胞的胃」 |
| 叶绿体 | 光合作用——只在植物和藻类 |

## 内共生理论

| 线粒体和叶绿体曾经是自由生活的细菌——被吞噬——留下来——成为细胞的一部分 |
|------|
| 证据: 它们有自己的DNA·双层膜·独立分裂——像细菌 |
