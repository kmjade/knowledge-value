# 05-进化论

> 「生物学中没有任何东西是有意义的——除非在进化之光下。」(Dobzhansky)

## 达尔文的自然选择

| 三个条件: (1)变异 (2)遗传 (3)差异繁殖 |
|------|
| 结果: 有利于生存和繁殖的变异在种群中越来越常见——「适应」 |
| 不是「为了生存」——是「碰巧活下来了」——自然选择没有目的 |

## 进化的证据

| 证据 | 例子 |
|------|------|
| 化石记录 | 从鱼到两栖——提塔利克鱼(Tiktaalik) |
| 比较解剖学 | 同源结构——人手·鲸鳍·蝙蝠翼——相同的骨骼蓝图 |
| DNA | 人类和黑猩猩~98.8%相同——设计者不会这么「偷懒」 |
| 生物地理学 | 岛屿物种像最近大陆的——不是像其他岛屿的 |
| 直接观察 | 抗生素耐药·COVID变异——在实时发生 |

## 现代综合

| 达尔文+孟德尔遗传学+群体遗传学——进化 = 等位基因频率随时间的变化 |
|------|
| 漂变: 随机变化——在小种群中尤其重要——不是「选择」——是「运气」 |
| 基因流: 种群间迁移——引入新等位基因 |
