# 01-生命科学概述

> 生物学 = 生命的科学——从分子到生态系统。

## 生命的特征

| 特征 | 说明 |
|------|------|
| 细胞组成 | 所有生命由细胞构成——除了病毒(争议) |
| 新陈代谢 | 从环境获取能量和物质——维持自身 |
| 稳态 | 维持内部环境稳定——体温·pH·血糖 |
| 生长与发育 | 从小到大——基因调控的过程 |
| 繁殖 | 产生后代——DNA传递 |
| 演化 | 种群随时间变化——自然选择 |

## 生物学的层级

| 分子→细胞→组织→器官→系统→个体→种群→群落→生态系统→生物圈 |
|------|
| 还原论: 从分子理解生命 / 系统论: 涌现——整体>部分之和 |

## 生物学的分支

| 分支 | 层级 |
|------|------|
| 分子生物学 | 分子 |
| 细胞生物学 | 细胞 |
| 遗传学 | 基因 |
| 生理学 | 器官·系统 |
| 生态学 | 种群·群落·生态系统 |
| 进化生物学 | 所有层级——统一理论 |
