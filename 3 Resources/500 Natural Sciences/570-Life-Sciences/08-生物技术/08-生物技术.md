# 08-生物技术

> 利用生命系统为人类服务——从酿酒到基因编辑。

## 里程碑

| 年份 | 事件 |
|:---:|------|
| ~8000 BCE | 驯化植物和动物——最早的生物技术——但不知道原理 |
| ~4000 BCE | 用酵母发酵——面包·啤酒 |
| 1973 | 重组DNA——Boyer & Cohen——「基因工程」诞生 |
| 1978 | 基因工程胰岛素(大肠杆菌生产)——糖尿病患者的福音 |
| 1996 | 多莉羊——第一个用体细胞克隆的哺乳动物 |
| 2012 | CRISPR-Cas9——基因编辑革命 |
| 2020 | mRNA疫苗(COVID)——几十年的研究在一年内拯救了数百万生命 |

## 生物技术的应用

| 领域 | 应用 |
|------|------|
| 医药 | 胰岛素·单克隆抗体·CAR-T(改造T细胞攻击癌症) |
| 农业 | GMO(抗虫·耐除草剂)——黄金大米(维生素A) |
| 工业 | 酶(洗涤剂)·生物燃料·生物塑料 |
| 法医 | DNA指纹——1986年首次用于定罪 |
