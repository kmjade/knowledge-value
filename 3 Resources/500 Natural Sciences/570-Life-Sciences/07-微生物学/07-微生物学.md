# 07-微生物学

> 看不见的世界——微生物统治地球——无论是在数量上还是在影响上。

## 微生物的多样性

| 类群 | 特征 |
|------|------|
| 细菌 | 原核——无处不在——有些致病·大部分无害或有益 |
| 古菌 | 原核——极端环境——不是细菌 |
| 病毒 | 不是细胞——寄生——DNA或RNA+蛋白外壳 |
| 真菌 | 真核——酵母·霉菌——分解者 |
| 原生生物 | 真核——单细胞——变形虫·草履虫 |

## 微生物组

| 人体内有~40万亿细菌(和细胞数相当)——大多数在肠道 |
|------|
| 功能: 消化·免疫·合成维生素——甚至影响情绪(肠-脑轴) |
| 失衡→炎症性肠病·肥胖·抑郁——粪便移植治疗C.diff感染 |

## 抗生素耐药

| 抗生素滥用→耐药菌→「超级细菌」——每年~70万人死于耐药感染 |
|------|
| 2050预测: 每年1000万人——超过癌症——后抗生素时代的恐惧 |
