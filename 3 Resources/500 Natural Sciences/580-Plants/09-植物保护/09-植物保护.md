# 09-植物保护

> 40%的植物物种面临灭绝风险——植物灭绝 = 生态系统崩溃。

## 威胁

| 威胁 | 影响 |
|------|------|
| 栖息地丧失 | 最大威胁——森林砍伐·农业扩张·城市化 |
| 气候变化 | 改变分布——高山植物无处可退 |
| 入侵物种 | 竞争·杂交——本地物种被取代 |
| 过度采集 | 药用植物·观赏植物·木材 |

## 种子库

| 斯瓦尔巴全球种子库(挪威)——北极永冻土里——保存了~120万份种子样本 |
|------|
| 千年种子银行(英国): 目标是保存25%的世界植物物种的种子 |

## 保护策略

| 就地保护: 保护区·国家公园——保护栖息地 |
|------|
| 迁地保护: 植物园·种子库——保护物种 |
| 恢复生态学: 重新引入——修复退化的生态系统 |
