# 05-植物生理

> 植物如何运作——激素·信号·对环境做出反应。

## 植物激素

| 激素 | 功能 |
|------|------|
| 生长素 (Auxin) | 向光性·顶端优势·根的生长 |
| 赤霉素 (Gibberellin) | 茎的伸长·种子萌发 |
| 细胞分裂素 (Cytokinin) | 细胞分裂·延缓衰老 |
| 乙烯 | 果实成熟——「一个烂苹果坏了一筐」 |
| 脱落酸 (ABA) | 休眠·抗旱——关闭气孔 |

## 向性

| 向光性 | 向光弯曲——生长素分布不均(背光侧多)——生长快→弯向光 |
|------|------|
| 向地性 | 根向下(正向)——茎向上(负向)——平衡石(淀粉粒)感知重力 |

## 昼夜节律

| 植物有「生物钟」——气孔开闭·叶片运动·开花时间 |
|------|
| 光周期: 短日植物(菊花·大豆)vs 长日植物(小麦·菠菜)——用光周期判断季节 |
