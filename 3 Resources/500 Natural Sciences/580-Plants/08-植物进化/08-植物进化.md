# 08-植物进化

> 从绿藻到开花植物——5亿年的绿色革命。

## 登陆的挑战与解决方案

| 挑战 | 解决方案 |
|------|------|
| 干燥 | 角质层(蜡质)·气孔(可控开闭) |
| 支撑 | 木质素——坚硬的维管组织 |
| 营养吸收 | 根——从土壤吸水吸矿 |
| 繁殖(无水) | 花粉——不需要水来传递雄配子 |

## 花与传粉的协同进化

| 花和传粉者(昆虫·鸟·蝙蝠)共同进化——互利的军备竞赛 |
|------|
| 蜂媒花: 蓝·黄·紫外图案(蜜蜂能看到)——甜味·平台 |
| 鸟媒花: 红(鸟能看到)·管状·无味(鸟嗅觉差) |
| 蛾媒花: 白色·夜间开放·强烈的香气 |
| 欺骗: 某些兰花模仿雌蜂——雄蜂试图交配——传粉——「性欺骗」 |

## 果实与种子传播

| 风(蒲公英)·水(椰子)·动物(吃果实·排泄种子·粘在毛上)——「弹射」(豆荚爆裂) |
