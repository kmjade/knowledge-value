# 03-电磁学

> 电与磁的统一——19世纪物理学最伟大的成就——麦克斯韦方程组。

## 麦克斯韦方程组

| 方程 | 含义 |
|------|------|
| 高斯定律(电) | 电荷产生电场——电场线从正到负 |
| 高斯定律(磁) | 没有磁单极子——磁场线总是闭合 |
| 法拉第定律 | 变化的磁场产生电场——发电机的原理 |
| 安培-麦克斯韦 | 电流+变化的电场产生磁场 |

## 电磁波

| 麦克斯韦预言: 变化的电场和磁场互相产生——以光速传播——光就是电磁波！ |
|------|
| 赫兹1887年实验证实——无线电·微波·X射线——都是电磁波——只是频率不同 |

## 应用

| 应用 | 原理 |
|------|------|
| 发电机 | 法拉第定律——机械能→电能 |
| 电动机 | 洛伦兹力——电流在磁场中受力 |
| 变压器 | 电磁感应——改变电压——交流电远距离传输 |
| 无线通信 | 振荡电路产生电磁波——天线接收 |
