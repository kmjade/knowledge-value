# 06-相对论

> 时空不是绝对的——爱因斯坦颠覆了我们对现实最根本的直觉。

## 狭义相对论 (1905)

| 两个假设: (1)物理定律在所有惯性系中相同 (2)光速不变 |
|------|
| 后果: 时间膨胀——运动时钟变慢——GPS必须校正(每天38微秒) |
| 长度收缩——运动物体在运动方向上变短 |
| 同时性的相对性——「同时」取决于观察者 |
| E=mc²——质量是能量的浓缩——1克物质=21千吨TNT |

## 广义相对论 (1915)

| 等效原理: 引力=加速度——你分不清是在地球上还是在加速的火箭里 |
|------|
| 引力 = 时空弯曲——物质告诉时空如何弯曲——时空告诉物质如何运动 |
| 水星近日点进动——牛顿解释不了——爱因斯坦自然地解释了 |

## 广义相对论的验证

| 验证 | 内容 |
|------|------|
| 1919日食 | 星光被太阳弯曲——证实爱因斯坦——他成了全球名人 |
| 引力红移 | 光爬出引力场——频率降低——Pound-Rebka实验(1959) |
| 引力波 | 2015 LIGO首次探测——黑洞合并——时空的涟漪 |
| 黑洞照片 | 2019 EHT——M87*——看到了「事件视界的影子」 |
