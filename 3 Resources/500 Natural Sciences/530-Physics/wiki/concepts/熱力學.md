# 熱力學 Thermodynamics

## 定義
熱力學研究熱、功和能量轉換的規律。

## 四大定律

| 定律 | 內容 |
|------|------|
| **第零** | 熱平衡的傳遞性 |
| **第一** | ΔU = Q − W (能量守恆) |
| **第二** | ΔS ≥ 0 (熵增原理) |
| **第三** | T→0 時 S→常數 |

## 熵 Entropy
S = k_B·ln(W)。熵度量系統的無序度——宇宙的熵永恆增加。
