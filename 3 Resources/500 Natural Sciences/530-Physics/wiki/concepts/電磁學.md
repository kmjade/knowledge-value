# 電磁學 Electromagnetism

## 定義
電磁學研究電荷、電流產生的電場和磁場，由 Maxwell 方程組統一描述。

## Maxwell 方程組

| 方程 | 物理意義 |
|------|---------|
| ∇·E = ρ/ε₀ | 電場源於電荷 (Gauss) |
| ∇·B = 0 | 無磁單極 (Gauss磁) |
| ∇×E = −∂B/∂t | 變化的磁場產生電場 (Faraday) |
| ∇×B = μ₀J + μ₀ε₀∂E/∂t | 電流+變化的電場產生磁場 (Ampère-Maxwell) |

## 電磁波
Maxwell 預測：c = 1/√(μ₀ε₀) ≈ 3×10⁸ m/s = 光速。光是電磁波。
