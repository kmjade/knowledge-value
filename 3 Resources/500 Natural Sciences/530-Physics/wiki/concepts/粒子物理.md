# 粒子物理 Particle Physics

## 定義
粒子物理研究物質的最基本組成和它們之間的相互作用。

## 標準模型

| 類別 | 粒子 |
|------|------|
| **夸克 Quarks** | u, d, c, s, t, b |
| **輕子 Leptons** | e, μ, τ, ν_e, ν_μ, ν_τ |
| **規範玻色子** | γ(電磁), W⁺/W⁻/Z(弱), g(強) |
| **Higgs 玻色子** | H (2012 發現) |

## 四種基本力

| 力 | 媒介粒子 | 相對強度 |
|----|---------|:--:|
| 強力 | gluon | 1 |
| 電磁力 | photon | 10⁻² |
| 弱力 | W/Z | 10⁻⁷ |
| 重力 | graviton(?) | 10⁻³⁹ |
