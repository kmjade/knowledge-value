# 量子力學 Quantum Mechanics

## 定義
量子力學描述微觀粒子（原子、電子、光子）的行為——與經典物理截然不同。

## 核心原理

| 原理 | 說明 |
|------|------|
| **波粒二象性** | 粒子同時具有波和粒子的性質 |
| **不確定性原理** | Δx·Δp ≥ ℏ/2 (Heisenberg) |
| **疊加原理** | |ψ⟩ = α|0⟩ + β|1⟩ |
| **測量塌縮** | 觀測使量子態塌縮到一個本徵態 |
| **量子糾纏** | EPR 對——超距關聯 |

## Schrödinger 方程

```
iℏ·∂ψ/∂t = Ĥψ
```

## 詮釋之爭

| 詮釋 | 核心觀點 |
|------|---------|
| **哥本哈根** | 測量導致波函數塌縮 |
| **多世界** | 每次測量宇宙分裂 |
| **隱變量** | 量子隨機性下有確定性 |
