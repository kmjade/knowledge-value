# 相對論 Relativity

## 狹義相對論 Special Relativity (1905)

| 原理 | 說明 |
|------|------|
| **光速不變** | c 在所有慣性系中相同 |
| **時間膨脹** | Δt' = γ·Δt, γ = 1/√(1−v²/c²) |
| **長度收縮** | L' = L/γ |
| **質能等價** | **E = mc²** |

## 廣義相對論 General Relativity (1915)

```
G_μν = 8πG·T_μν

時空彎曲 = 物質-能量分佈
```

- 重力 = 時空的幾何彎曲
- 預測：黑洞、重力波 (2015 LIGO 證實)、光線彎曲
