# 凝聚態物理 Condensed Matter Physics

## 定義
凝聚態物理研究大量粒子組成的系統的宏觀和微觀性質——固體、液體、超導體等。

## 核心概念

| 概念 | 說明 |
|------|------|
| **能帶理論** | 固體中電子的能量分佈 |
| **超導 Superconductor** | 零電阻 + Meissner 效應 |
| **拓撲絕緣體** | 表面導電、內部絕緣 |
| **量子霍爾效應** | 二維電子氣的精確電導量子化 |
