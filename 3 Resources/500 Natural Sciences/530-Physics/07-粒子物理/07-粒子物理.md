# 07-粒子物理

> 物质的最基本成分——标准模型——人类最精确的理论之一。

## 标准模型

| 类别 | 粒子 | 说明 |
|------|------|------|
| 夸克 (6种) | 上·下·粲·奇·顶·底 | 组成质子和中子(uud, udd) |
| 轻子 (6种) | 电子·μ子·τ子+3种中微子 | 电子是唯一稳定的带电轻子 |
| 玻色子 (5种) | 光子·胶子·W±·Z·希格斯 | 传递力的粒子 |

## 希格斯机制

| 2012年LHC发现希格斯粒子——「上帝粒子」——最后的拼图 |
|------|
| 希格斯场: 遍布全宇宙——粒子通过与它相互作用获得质量 |
| 没有希格斯——电子无质量→原子不存在→我们没有 |

## 超越标准模型

| 问题 | 说明 |
|------|------|
| 暗物质 | 标准模型没有候选粒子——需要新物理 |
| 中微子质量 | 标准模型预测零质量——但中微子振荡证明有质量 |
| 物质-反物质不对称 | 大爆炸应产生等量——但宇宙几乎全是物质——为什么？ |
| 等级问题 | 为什么引力比其他力弱那么多？ |

|------|
| 最大的科学实验——每年产生~90PB数据——WWW在此发明 |
