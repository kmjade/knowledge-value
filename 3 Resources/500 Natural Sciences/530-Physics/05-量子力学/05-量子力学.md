# 05-量子力学

> 微观世界的规则——反直觉——但这是实验证实的事实。

## 核心概念

| 概念 | 说明 |
|------|------|
| **波粒二象性** | 光和物质既是波又是粒子——取决于你怎么看——双缝实验 |
| **不确定性原理** | Δx·Δp ≥ ℏ/2——位置和动量不能同时精确知道——不是测量技术的限制——是自然的本性 |
| **叠加态** | 在被测量前——粒子同时处于多个状态——薛定谔的猫 |
| **测量问题** | 测量「坍缩」波函数——为什么？——哥本哈根诠释:就这样——多世界:没有坍缩——世界分叉了 |
