# 04-热力学

> 能量、热量、熵——从蒸汽机到宇宙热寂。

## 四大定律

| 定律 | 内容 |
|:---:|------|
| 第零 | 热平衡的传递性——温度计的基础 |
| 第一 | 能量守恒——ΔU=Q−W——第一类永动机不可能 |
| 第二 | 熵不减——热量不会自发从冷到热——第二类永动机不可能 |
| 第三 | 绝对零度不可达到 |

## 熵

| 熵 = 无序度——微观状态数的度量——时间之箭 |
|------|
| 为什么鸡蛋不会自发「反打破」？——熵增——不是不可能——是概率极低 |
| 生命是「局部的熵减」——以环境的熵增为代价 |

## 热力学过程

| 过程 | 特征 |
|------|------|
| 等温 | 温度恒定——与热源接触 |
| 绝热 | 无热交换——压缩→升温 |
| 卡诺循环 | 最理想的热机——效率只取决于高低温热源的温度 |

## 统计力学

| 玻尔兹曼: 热力学是统计的——温度=分子平均动能——熵 S=k log W |
|------|
| 架起了微观(分子)和宏观(热力学)的桥梁 |
