# 08-凝聚态物理

> 最多物理学家的领域——研究「多体」的涌现行为——「More is Different」(P.W. Anderson)。

## 能带理论

| 晶体中电子的能级形成「能带」——解释了为什么有些材料导电、有些不 |
|------|
| 导体: 价带和导带重叠——电子自由移动 |
| 绝缘体: 带隙大——电子跃不过去 |
| 半导体: 带隙小——可调——晶体管的基础 |

## 超导

| 某些材料在低温下电阻为零——电流可以永远流动 |
|------|
| BCS理论(1957): 电子配对(库珀对)——通过晶格振动(声子)相互吸引 |
| 高温超导(>77K液氮): 1986发现——铜氧化物——机制仍不完全清楚——凝聚态最大的未解之谜 |

## 拓扑材料

| 2016诺贝尔奖: 拓扑相变——用拓扑学(数学)分类物质相 |
|------|
| 拓扑绝缘体: 表面导电——内部绝缘——鲁棒——不受杂质影响 |
| 潜在应用: 量子计算——拓扑量子比特——对抗退相干 |

## 自旋电子学

| 利用电子的自旋(不只是电荷)存储和处理信息——MRAM·自旋晶体管 |
