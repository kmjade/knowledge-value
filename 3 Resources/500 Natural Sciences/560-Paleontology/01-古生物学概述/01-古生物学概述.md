# 01-古生物学概述

> 古生物学 = 通过化石阅读地球的生命史。

## 核心概念

| 概念 | 说明 |
|------|------|
| 化石 | 过去生命的遗骸或痕迹——不仅仅是骨头——足迹·粪便·羽毛 |
| 化石记录 | 不完整——99%以上的物种没有留下化石——我们看到的只是冰山一角 |
| 深层时间 | 地质时间尺度——「百万年」是最小单位 |

## 化石化的条件

| 快速埋藏 | 防止腐烂和食腐——洪水·火山灰·沙暴 |
|------|------|
| 硬体部分 | 骨头·壳·牙齿——软组织极少保存(特殊条件下有) |
| 缺氧环境 | 减缓分解——深海·沼泽 |

## 古生物学方法

| 方法 | 用途 |
|------|------|
| 比较解剖学 | 与现存生物比较——推断功能 |
| 同位素分析 | 碳-14定年(<5万年)·铀-铅定年(百万-数十亿年) |
| CT扫描 | 不破坏化石看内部结构 |
| 系统发生学 | 用特征矩阵重建演化树 |
