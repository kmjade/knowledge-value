# 05-哺乳动物崛起

> 从恐龙的阴影到统治地球——哺乳动物的6600万年。

## 中生代的哺乳动物

| 和恐龙共存了1亿多年——但都是小型的·夜行的·「老鼠水平」 |
|------|
| 为什么小？——白天是恐龙的世界——哺乳动物在夜晚活动 |
| 优势: 恒温·毛发·乳汁·复杂牙齿——只是还没机会「爆发」 |

## K-Pg灭绝后

| 6600万年前: 小行星撞击——恐龙(非鸟)灭绝——哺乳动物迎来机会 |
|------|
| 100万年内: 哺乳动物体型增大10倍——生态位「空缺」被迅速填充 |
| 几千万年内: 蝙蝠飞上天·鲸回到海洋·马从狐狸大小演化成今天的马 |

## 关键类群

| 类群 | 特征 |
|------|------|
| 灵长类 | 树栖→立体视觉→大脑增大→人类 |
| 鲸类 | 从陆生(巴基斯坦古鲸)到完全水生——演化最快的转变之一 |
| 有蹄类 | 草原扩张→奔跑适应——马·鹿·牛 |
| 食肉目 | 猫科·犬科·熊——哺乳动物的顶级捕食者 |
