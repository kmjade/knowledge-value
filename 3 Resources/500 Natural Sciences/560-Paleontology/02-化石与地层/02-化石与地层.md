# 02-化石与地层

> 地层是地球的日历——化石是日历上的标记。

## 地层学原理

| 原理 | 内容 |
|------|------|
| 叠覆原理 | 下面的地层更老——除非构造作用翻转 |
| 化石层序律 | 相同的化石出现在相同时代的地层——用于全球对比 |
| 切割关系 | 切割其他地层的岩体更年轻 |
