# 08-古生态学

> 重建古代的生态系统——不只是单个生物——是它们之间的关系。

## 方法

| 方法 | 用途 |
|------|------|
| 功能形态学 | 从形态推断生态——牙齿→食物·四肢→运动 |
| 同位素 | C¹³→食物类型·O¹⁸→温度 |
| 微痕 | 牙齿上的微划痕→最后一餐吃了什么 |
| 共现分析 | 哪些物种经常一起出现？ |
