## 💡 學習建議

# 指南

**零基礎/初中水平**：
```
Khan Academy → 中国大学MOOC → B站
```

**大学预习/复习**：
```
中国大学MOOC → MIT OCW → Coursera
```

**专业深造**：
```
MIT OCW → edX → 学术论文
```

### 學習节奏

| 课程类型 | 建議每周投入 |
|----------|--------------|
| 短课程（<10小时） | 3-5小时 |
| 中等课程（10-30小时） | 5-8小时 |
| 长课程（>30小时） | 8-12小时 |

# 方法

1. **先看大纲**：了解课程结构和要求
2. **按序學習**：不要跳跃，循序渐进
# 整理
4. **完成作业**：练习是掌握的關鍵
5. **参与讨论**：在論壇提问和交流
6. **定期复习**：巩固已学內容
