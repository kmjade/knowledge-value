# 統計學 Statistics

## 定義
統計學是收集、分析、解釋和呈現數據的科學。

## 核心概念

| 概念 | 說明 |
|------|------|
| **描述統計** | 均值、中位數、標準差 |
| **推斷統計** | 從樣本推斷總體 |
| **假設檢驗** | p值、顯著性水平 |
| **置信區間** | 參數的可能範圍 |
| **回歸分析** | 變量關係建模 |
