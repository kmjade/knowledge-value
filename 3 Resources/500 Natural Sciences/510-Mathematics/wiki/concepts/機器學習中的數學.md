# 機器學習中的數學 Mathematics for ML

## 必備數學領域

| 領域 | ML 應用 |
|------|--------|
| **線性代數** | 數據矩陣、PCA、SVD、神經網絡權重 |
| **微積分** | 梯度下降、反向傳播 |
| **概率論** | 貝葉斯推斷、生成模型 |
| **統計學** | 假設檢驗、回歸、評估指標 |
| **優化理論** | SGD、Adam、凸優化 |

## 梯度下降

```
θ := θ − α·∇J(θ)
α = 學習率, ∇J = 損失函數的梯度
```
