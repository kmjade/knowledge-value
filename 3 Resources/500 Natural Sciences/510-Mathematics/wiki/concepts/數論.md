# 數論 Number Theory

## 定義
數論研究整數的性質和關係——「數學的女王」(Gauss)。

## 核心概念

| 概念 | 說明 |
|------|------|
| **素數 Prime** | 只能被 1 和自身整除 |
| **算術基本定理** | 每個整數可唯一分解為素數乘積 |
| **同餘 Congruence** | a ≡ b (mod n) |
| **歐拉函數 φ(n)** | ≤n 且與 n 互質的數的個數 |
| **費馬小定理** | a^(p-1) ≡ 1 (mod p) |

## 與黎曼猜想的關聯

素數定理：π(x) ~ x/ln(x)。黎曼猜想若為真，將給出素數分佈的最精確誤差界。
