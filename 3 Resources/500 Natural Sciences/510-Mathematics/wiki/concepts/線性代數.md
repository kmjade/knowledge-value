# 線性代數 Linear Algebra

## 定義
線性代數研究向量空間和線性變換，是現代數學和計算科學的基石。

## 核心概念

| 概念 | 說明 |
|------|------|
| **向量 Vector** | 有大小和方向的量 |
| **矩陣 Matrix** | 數字的矩形陣列 |
| **行列式 Determinant** | 方陣的縮放因子 |
| **特徵值 Eigenvalue** | Av = λv |
| **線性變換** | 保持向量加法和標量乘法的映射 |

## 應用

- ML：數據表示、PCA、SVD
- 計算機圖形：3D變換
- 量子力學：算符、態向量
