# 複變函數 Complex Analysis

## 定義
複變函數研究定義在複數平面上的函數。複數 z = x + iy，其中 i² = −1。

## 核心概念

| 概念 | 說明 |
|------|------|
| **解析函數** | 可複微分的函數（比實可微強得多） |
| **Cauchy-Riemann 方程** | 解析性的充要條件 |
| **留數定理** | ∮f(z)dz = 2πi·ΣRes |
| **保角映射** | 保持角度不變的變換 |

## 與黎曼猜想的關聯

黎曼 ζ 函數是複變函數：
ζ(s) = Σ(1/n^s)，s 為複數。
黎曼猜想：ζ(s) 的所有非平凡零點都在臨界線 Re(s) = 1/2 上。
