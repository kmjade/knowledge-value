# 概率論 Probability Theory

## 定義
概率論研究隨機現象的數學規律。

## 核心概念

| 概念 | 公式/說明 |
|------|---------|
| **概率公理** | P(Ω)=1, P(∅)=0, 0≤P≤1 |
| **條件概率** | P(A\|B) = P(A∩B)/P(B) |
| **貝葉斯定理** | P(A\|B) = P(B\|A)P(A)/P(B) |
| **期望值** | E[X] = Σ x·P(x) |
| **方差** | Var(X) = E[(X−E[X])²] |
| **大數定律** | 樣本均值 → 期望值 |

## 概率分佈

| 分佈 | 適用場景 |
|------|---------|
| 正態 Normal | 自然測量誤差 |
| 二項 Binomial | n次獨立試驗成功次數 |
| 泊松 Poisson | 稀有事件計數 |
