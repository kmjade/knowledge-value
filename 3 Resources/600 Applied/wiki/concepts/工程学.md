# 工程學 Engineering
應用科學原理設計與建造結構、機器與系統。

## 主要分支
| 分支 | 應用領域 |
|:----|:--------|
| **土木工程** | 建築·橋樑·道路 |
| **機械工程** | 機械·動力·製造 |
| **電機工程** | 電力·電子·通訊 |
| **化學工程** | 化工·材料·製藥 |

## 工程方法
- 需求分析 → 設計 → 原型 → 測試 → 迭代
