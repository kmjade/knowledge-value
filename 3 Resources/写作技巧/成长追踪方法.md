> [!note] 成长追踪 / Growth Tracking
> 量化个人成长轨迹的系统化方法

---

## 💡 核心想法

成长追踪是通过数据和指标记录个人进步的方式：
- 📊 可视化进步轨迹
- 🎯 量化目标和成就
- 🔍 识别模式和趋势
- 🌟 激励持续改进

## 📏 追踪维度

### 个人成长多维度

| 维度 | 指标示例 | 追踪频率 |
|------|---------|---------|
| **健康** | 体重、运动时间、睡眠时长 | 每日 |
| **学习** | 阅读页数、学习时间、技能练习 | 每日 |
| **工作** | 完成任务、项目进度、工作时长 | 每周 |
| **财务** | 收入、支出、储蓄率 | 每月 |
| **关系** | 社交活动、家庭时间 | 每周 |
| **精神** | 冥想时间、感恩日记 | 每日 |

### 追踪模板

```markdown
---
date: 2026-01-19
growth_score: 7/10
---

## 今日追踪

### 健康
- [ ] 运动 30 分钟
- [ ] 睡眠 7+ 小时
- [ ] 饮水 2000ml

### 学习
- 阅读 20 页 ✅
- 学习新概念 ✅

### 工作
- 完成 3 个任务 ✅

### 精神
- 冥想 10 分钟 ✅
- 写感恩日记 ✅

## 成长洞察
今天在时间管理上有进步，提前完成了重要任务。
```

## 🎯 SMART 追踪

### 目标设定与追踪

```markdown
## 2026年成长目标

### 健康
**目标**: 体重从75kg降至70kg
**起点**: 75kg (1月1日)
**终点**: 70kg (12月31日)
**进度**: 75kg → 74.5kg → 74kg → ...

**追踪**:
- 每周一早上测量体重
- 记录运动日志
- 每月回顾饮食

### 学习
**目标**: 完成12门在线课程
**起点**: 0门 (1月1日)
**终点**: 12门 (12月31日)
**进度**: 3/12 (25%)

**追踪**:
- 每周学习10小时
- 每月完成1门课程
- 记录学习笔记到 Zettels
```

## 🔧 追踪工具

### Obsidian PARA 追踪系统

| PARA 分类 | 追踪内容 | 更新频率 |
|----------|----------|---------|
| **1 Projects** | 项目进度、里程碑完成 | 每周 |
| **2 Areas** | 领域健康度、长期趋势 | 每月 |
| **3 Resources** | 追踪模板、分析方法 | 按需 |
| **5 Zettels** | 每日/每周追踪数据 | 每日/每周 |

### Dataview 查询示例

```dataview
TABLE WITHOUT ID
  dateformat(file.mtime, "yyyy-MM-dd") as "日期",
  growth_score as "成长评分",
  exercise as "运动",
  reading as "阅读"
FROM "5 Zettels"
WHERE contains(tags, "growth-tracking")
SORT file.mtime DESC
LIMIT 30
```

## 📈 成长分析

### 每周回顾

```markdown
## 第3周回顾 (1月15-21日)

### 成功之处
- ✅ 运动习惯坚持 7/7 天
- ✅ 阅读完成 1 本书
- ✅ 工作效率提升 20%

### 需要改进
- ❌ 睡眠时间不足（平均 6.5 小时）
- ❌ 社交时间偏少

### 下周计划
- 调整睡眠作息，争取 7+ 小时
- 安排 2 次社交活动
- 保持运动习惯
```

### 月度总结

```markdown
## 1月总结

### 关键数据
- 运动：26/31 天 (84%)
- 阅读：4 本书 (目标 4 本 ✅)
- 工作：完成 12 个项目
- 睡眠：平均 6.8 小时 (目标 7 小时 ⚠️)

### 成长亮点
- 建立了晨间运动习惯
- 完成了 PARA 系统整理
- 学会了使用触发器自动化

### 改进方向
- 提升睡眠质量
- 增加社交活动
- 优化工作时间分配

### 2月目标
- 睡眠达到 7 小时
- 阅读 5 本
- 开始新的学习项目
```

## 🔄 成长循环

```
设定目标 → 追踪数据 → 分析洞察 → 调整策略 → 设定新目标

1. 设定可衡量的成长目标
2. 持续追踪相关数据
3. 定期反思和分析趋势
4. 根据洞察调整策略
5. 设定下一个成长目标
```

## 📊 成长指标库

### 健康指标
- 体重、BMI
- 运动时间、步数
- 睡眠时长、质量
- 饮食记录

### 学习指标
- 阅读页数/书籍
- 学习时间
- 技能练习次数
- 完成的课程

### 工作指标
- 完成任务数
- 项目进度
- 工作时长
- 深度工作块数

### 财务指标
- 收入、支出
- 储蓄率
- 投资回报
- 债务减少

---

> [!tip] 实践建议
> 1. **从少量指标开始** - 不要追踪太多东西
> 2. **保持一致性** - 每天同一时间记录
> 3. **定期回顾** - 每周分析数据和趋势
> 4. **可视化进步** - 使用图表激励自己

> [!question] 思考
> 我最想在哪些方面看到成长？如何设定可追踪的指标？
> - 健康：运动、睡眠、饮食
> - 学习：阅读、技能、课程
> - 工作：项目、效率、收入
> - 关系：社交、家庭、友谊
