# 05-救赎论

> 人犯了罪——怎么得救？谁做的工？上帝？人？合作？这是救赎论——系统神学中最具争议的部分——直接关系到「谁可以得救」和「为什么」。

## 救赎的秩序 (Ordo Salutis)

| 步骤 | 含义 |
|------|------|
| 拣选 | 上帝在永恒中拣选得救的人 |
| 呼召 | 上帝通过福音呼召人 |
| 重生 | 圣灵使人灵性活过来 |
| 归信 | 人悔改 + 相信 |
| 称义 | 上帝宣告罪人为义 |
| 成圣 | 圣灵逐渐改变人的品格 |
| 得荣 | 最终——复活——完全像基督 |

---

## 称义

> 「人称义是因着信——不在乎遵行律法。」(罗 3:28)

| 概念 | 内容 |
|------|------|
| **称义 = 宣告为义** | 不是「使人变成义」(那是成圣) ——是「宣告」为义——法律用语 |
| **归算** | 基督的义「归算」给信徒——如同债务转移到另一个账户 |

### 天主教 vs 新教

| 维度 | 天主教 (天特会议) | 新教 (宗教改革) |
|------|-----------------|----------------|
| 称义 | 信心 + 行为 + 圣礼 | 唯独信心 (sola fide) |
| 恩典 | 恩典使称义可能——但需要人的配合 | 唯独恩典 (sola gratia) ——全主动 |

> 1999 年《因信称义联合声明》——信义宗和天主教会宣布：双方在称义教义上取得「基本共识」——500 年后的突破。

## 预定论

> 谁得救——是你选择的——还是上帝选择的？

### 加尔文主义 (改革宗)

| TULIP | 含义 |
|-------|------|
| **T**otal depravity | 全然败坏——人不能自救 |
| **U**nconditional election | 无条件拣选——上帝选择不是基于人的任何功德 |
| **L**imited atonement | 限定赎罪——基督为选民而死 |
| **I**rresistible grace | 不可抗拒的恩典——上帝呼召的人一定来 |
| **P**erseverance of saints | 圣徒的坚忍——真信徒不会失落 |

### 阿米念主义

| 要点 | 加尔文主义反对 |
|------|--------------|
| 人有自由意志回应或拒绝恩典 | 人的意志被罪捆绑——不能回应 |
| 拣选基于上帝预知人的信心 | 拣选是无条件的 |
| 恩典可被抗拒 | 恩典不可抗拒 |
| 信徒可能失落 | 真信徒不会失落 |

> 这个争论从 17 世纪持续至今——多特会议 (1618-19) 谴责了阿米念主义——但在循道宗、五旬宗及许多福音派中阿米念主义影响深远。
