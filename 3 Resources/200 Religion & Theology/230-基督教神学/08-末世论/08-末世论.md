# 08-末世论

> 「末世论」不只是一张未来的时间表——它是关于**终局**的教义：历史往哪里去？死亡之后是什么？宇宙的终点是什么？基督教的答案：不是毁灭——是**更新**。

## 末世论的核心事件

| 事件 | 内容 |
|------|------|
| **基督再来** | 耶稣将「驾云降临」——不是隐秘的——是公开的 |
| **死人复活** | 所有死去的人将复活——身体和灵魂重新结合——但身体被更新 |
| **最后审判** | 所有人的生命被审判 |
| **新天新地** | 不是「上天堂」——是天堂**降临**到地上——上帝与人同住 |

## 千禧年争议 (启 20)

| 立场 | 基督再来 vs 千禧年 | 代表 |
|------|------------------|------|
| **前千禧年** | 基督再来 → 千禧年 | 早期教父·部分福音派 |
| **后千禧年** | 千禧年 → 基督再来 (教会在地上建立「准天国」) | 18-19世纪乐观派 |
| **无千禧年** | 千禧年 = 象征——现在是「千禧年」——基督再来 = 终局 | 奥古斯丁·加尔文 |

---

## 被提 (Rapture)

> 「被提」——信徒在基督再来时被「提」到空中——达秘 (Darby, 19世纪) 推广的观念。

| 立场 | 内容 |
|------|------|
| **灾前被提** | 信徒在七年大灾难**前**被提 |
| **灾中被提** | 在中间的 3.5 年被提 |
| **灾后被提** | 大灾难**后**被提——和基督再来是同一事件 |

> 被提是相对较新的教义——大多数基督徒从未听说过——直到 1830 年代。东正教完全拒绝这个观念。
