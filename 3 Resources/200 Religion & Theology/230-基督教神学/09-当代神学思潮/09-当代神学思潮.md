# 09-当代神学思潮

> 神学不是一成不变的——每个时代都在问新问题。20 世纪是神学最动荡的世纪：两次世界大战摧毁了「进步」的乐观——神学家被迫追问：上帝在哪里？被压迫者、女性和其他宗教的信徒开始要求神学回应**他们**的问题。

## 巴特与新正统

> 「上帝就是上帝。」——卡尔·巴特 (Karl Barth, 1886-1968)

| 概念 | 内容 |
|------|------|
| **核心** | 自由派神学把神学变成了人类学——上帝变成了人类情感的投影。巴特恢复「上帝的道」作为神学的唯一基础 |
| **《罗马书释义》 (1919)** | 「炸弹落在神学家的操场上」——扭转了 19 世纪的自由派神学 |

---

## 解放神学

| 代表 | 古斯塔沃·古铁雷斯 (Gustavo Gutiérrez, 1971) |
|------|------|

| 原则 | 内容 |
|------|------|
| **「优先选择穷人」** | 上帝站在穷人、被压迫者的一边——这不是「政治上左倾」——是圣经本身：出埃及 = 被压迫者的解放 |
| **实践优先** | 神学不是坐在书斋里写——是在**参与**穷人的解放实践中反思 |
| **罪的结构性** | 罪不只是个人的——社会结构（经济、政治）也可能是「罪的制度」 |

---

## 女性主义神学

| 问题 | 圣经和传统基督教是否系统性地贬低了女性？ |
|------|------|

| 进路 | 内容 |
|------|------|
| **修正主义** | 圣经本身不贬低女性——是**翻译和诠释**的问题 |
| **重建主义** | 圣经是父权制的产物——需要从女性经验出发重建神学 |
| **上帝的语言** | 「父」——这个隐喻是否必须？——「母」的隐喻也被圣经使用 (赛 66:13) |

---

## 进程神学

| 基础 | 怀特海 (Whitehead) 的进程哲学 |
|------|------|

| 核心 | 现实不是由「实体」构成——是由「事件/进程」构成。上帝也在进程中——他不是「不变的」——他**经历**世界的变化并与之一同受苦 |
|------|------|
| 与传统冲突 | 上帝不是全能的——是「有说服力的」——不是强制的。他不能单方面阻止恶——只能通过说服促使人向善 |
