# 04-圣灵论

> 圣灵是三位一体中最「神秘」的一位。父是规划者、子是执行者——圣灵是**应用者**。圣灵把基督成就的救赎**应用**在信徒的生命中。

## 圣灵的位格

> 圣灵不是一个「能力」或「力量」——是一个**位格**。

| 证据 | 内容 |
|------|------|
| 有意志 | 「圣灵……随己意分给各人」(林前 12:11) |
| 有情感 | 「不要叫上帝的圣灵担忧」(弗 4:30) |
| 可被亵渎 | 「亵渎圣灵的——永不得赦免」(可 3:29) ——一个「力量」不能被亵渎 |

### 和子句争议 (Filioque)

> 圣灵「从父和子」发出 (西方教会) ——还是「从父」发出 (东方教会)？

| 这导致了 1054 年东西教会大分裂——至今未解决 |

## 圣灵的工作

| 工作 | 内容 |
|------|------|
| **重生** | 使人「从灵而生」——成为新造的人 (约 3:5) |
| **内住** | 圣灵住在信徒里面——「你们的身体是圣灵的殿」(林前 6:19) |
| **成圣** | 圣灵逐渐改变信徒的品格——结出圣灵的果子 |
| **恩赐** | 圣灵赐予不同的属灵恩赐——为了建立教会 |

### 圣灵的果子 (加 5:22-23)

| 果子 | 九个品格：仁爱·喜乐·和平·忍耐·恩慈·良善·信实·温柔·节制 |
|------|------|

---

## 属灵恩赐

| 恩赐 | 内容 |
|------|------|
| 教导、治理、服事、劝勉、施舍 | 「日常恩赐」——但同样是圣灵给的 |
| 说方言、预言、医病 | 「超常恩赐」——灵恩派强调——停止论者认为这些恩赐已随使徒时代结束 |

### 停止论 vs 持续论

| 立场 | 核心 |
|------|------|
| **停止论** | 神迹性恩赐（方言·预言·医病）随使徒时代结束——不再存在 |
| **持续论** | 所有恩赐持续到基督再来 |
