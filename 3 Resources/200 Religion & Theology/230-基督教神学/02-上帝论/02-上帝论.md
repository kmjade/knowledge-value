# 02-上帝论

> 系统神学的第一个主题——也是最深的：上帝是谁？基督教说：上帝是三位一体的——一个本质，三个位格。这听起来像「1=3」——但两千年来的神学家说：不是矛盾，是奥秘。

## 三位一体

### 圣经基础

| 经文 | 含义 |
|------|------|
| 太 28:19 | 「奉父、子、圣灵的名施洗」——不是「名们」——是一个名——却有三个 |
| 约 1:1 | 「道就是上帝」——耶稣 (道) 被等同为上帝 |
| 约 14-16 | 圣灵是「另一位保惠师」——与耶稣不同——但同样是上帝 |

### 核心术语

| 术语 | 含义 |
|------|------|
| **ousia** (本质) | 上帝是**一个**本质——一个「什么」 |
| **hypostasis** (位格) | 上帝有**三个**位格——三个「谁」 |
| **homoousios** (同质) | 子与父**本质相同**——不是相似的——是同一本质 |

### 错误的三位一体观

| 异端 | 错误 |
|------|------|
| **形态论** (Modalism) | 父、子、圣灵是同一个上帝的三种「面具」——不是三个位格 |
| **亚流主义** (Arianism) | 子是受造物——有开始——「曾有一段时间他不存在」 |
| **三神论** (Tritheism) | 三个上帝——不是一本质 |

---

## 上帝的属性

### 可传递 vs 不可传递

| 类型 | 属性 | 含义 |
|------|------|------|
| **不可传递** (只属于上帝) | 自存 | 上帝不需要任何东西存在 |
| | 不变 | 上帝的本质不改变 |
| | 永恒 | 上帝在时间之外 (或无始终) |
| | 全在 | 上帝存在于所有空间 |
| **可传递** (可与人类似) | 爱、公义、智慧、圣洁 | 人有这些——但上帝以无限的完美拥有它们 |

---

## 创造论

| 立场 | 内容 |
|------|------|
| **从无创造** (ex nihilo) | 上帝不是用已有材料造世界——是从「无」中造出——意味着物质世界不是永恒的 |
| **护理** (Providence) | 上帝没有造了就走——他持续地**维系**和**掌管**世界 |

### 创造与科学

- 年轻地球创世论：六天 = 144 小时
- 日-代论：每一天 = 一个时代
- 框架假说：创世记 1 是文学框架——不是科学教科书
- 有神进化论：进化是上帝创造的**方式**
