# 02-基督教与文化

> 尼布尔 (H. Richard Niebuhr) 在《基督与文化》(1951) 中提出五个模型——至今仍是讨论基督教与文化关系的基本框架。

## 五种模型

### 1. 基督反对文化

| 核心 | 教会是与周围文化**完全对立**的反文化共同体 |
|------|------|
| 经文 | 「不要爱世界和世界上的事」(约一 2:15) |
| 代表 | 德尔图良·修道主义·门诺派·托尔斯泰 |
| 问题 | 完全退出——世界留给「该撒」——放弃了文化使命 |

### 2. 基督属于文化

| 核心 | 基督是文化成就的**顶峰**——文化是通向上帝的道路 |
|------|------|
| 代表 | 19世纪自由派·施莱尔马赫 |
| 问题 | 文化吸收了基督——基督教变成了「中产阶级的道德」 |

### 3. 基督在文化之上

| 核心 | 恩典不摧毁自然——而是**完善**自然 |
|------|------|
| 代表 | 阿奎那·天主教 | 
| 问题 | 对文化的批判不够——现存秩序被「施洗」而非被挑战 |

### 4. 基督与文化悖论

| 核心 | 基督徒活在两个国度——永远处于**张力**中 |
|------|------|
| 经文 | 「该撒的归该撒——上帝的归上帝」 |
| 代表 | 路德——两国论 |
| 问题 | 可能太安于现状——对不公义的文化沉默 |

### 5. 基督转化文化

| 核心 | 文化是堕落的——但也是**可救赎的**——基督来更新一切 |
|------|------|
| 经文 | 「看哪——我将一切都更新了」(启 21:5) |
| 代表 | 奥古斯丁·加尔文·凯波尔 (Abraham Kuyper) |
| 名言 | 「在人类存在的整个领域中——没有一寸是基督不宣称归属他的」——凯波尔 |
