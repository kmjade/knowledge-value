# 05-公共伦理

> 「行公义，好怜悯，存谦卑的心，与你的上帝同行。」(弥 6:8) 基督教伦理不只是个人的——它有**公共的**面向。

## 社会公义

| 主题 | 圣经基础 |
|------|---------|
| **穷人** | 旧约律法中保护穷人的条款——安息年·禧年——债务赦免 |
| **寄居者** | 「你们要怜爱寄居的——因为你们在埃及地也作过寄居的」(申 10:19) |
| **工人** | 「工人得工价是应当的」(路 10:7) ——雅各书 5:4 指控克扣工资的富人 |

---

## 生命伦理

| 议题 | 基督教视角 |
|------|-----------|
| **堕胎** | 生命始于何时？——不同宗派立场不同 |
| **安乐死** | 「不可杀人」——但「不必要地延长痛苦」是否也是错误的？ |
| **基因编辑** | 「扮演上帝」——还是「行使治理的使命」？ |
| **临终关怀** | 不是「杀死」——是「在死亡来临时——不加速也不阻挡——陪伴」 |

## 环境伦理

| 基础 | 「治理这地」(创 1:28) ——不是**掠夺**——是**管家** |
|------|------|
| 当代 | 「创造关怀」(Creation Care) ——不只是「末世的安慰」——是对上帝创造的尊重 |
