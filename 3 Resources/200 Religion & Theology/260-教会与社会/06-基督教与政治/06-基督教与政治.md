# 06-基督教与政治

> 基督徒应该参与政治吗？如果是——以什么方式？「基督教国家」？「基督徒作为公民」？还是「教会作为先知的声音」？

## 三种进路

| 进路 | 核心 | 代表 |
|------|------|------|
| **政治神学** (Political Theology) | 政治概念是神学概念的世俗化——揭露政治的「宗教性」 | 施米特 (Schmitt) |
| **公共神学** (Public Theology) | 神学可以也应该参与公共辩论——用公共语言 | Stackhouse |
| **激进正统** (Radical Orthodoxy) | 世俗是现代性的幻觉——教会应恢复其作为**真正公共空间**的地位 | Milbank |

---

## 基督徒的政治参与

| 模式 | 内容 |
|------|------|
| **退出** | 政治是「属世的」——基督徒不应参与 |
| **参与** | 基督徒应参与——但不以「基督教政党」的方式——而是**作为盐和光**分散在各领域 |
| **基督教政党** | 欧洲的基督教民主党模式 |
| **先知模式** | 教会作为一个**整体**对政治权力说话——不卷入政党政治——但发声 |

## 当代议题

| 议题 | 张力 |
|------|------|
| 民族主义 | 上帝的国家超越所有地上的国家——但爱祖国是自然的 |
| 移民 | 「寄居者」的圣经传统 vs 国家主权的现代观念 |
| 战争 | 正义战争理论 vs 和平主义——基督教历史上两者都有传统 |
