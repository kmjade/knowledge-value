# 03-宣教学

> 「你们要去——使万民作我的门徒。」(太 28:19) 宣教不只是地理上的「去」——更是文化上的「翻译」——把福音从一种文化传递到另一种文化。

## 宣教的圣经基础

| 主题 | 内容 |
|------|------|
| **亚伯拉罕之约** | 「地上的万族都要因你得福」(创 12:3) ——宣教的应许从一开始就在 |
| **大使命** | 太 28:19-20 ——「去·施洗·教导」 |
| **使徒行传** | 福音从耶路撒冷→犹太全地→撒玛利亚→地极 |

---

## 处境化 (Contextualization)

| 概念 | 内容 |
|------|------|
| **定义** | 将福音的「不变信息」用「可变的文化形式」表达出来 |
| **不是** | 改变福音以适应文化——那是妥协 |
| **是** | 像翻译——意思不变——语言变了 |

### 处境化的光谱

| 过犹不及 | 问题 |
|---------|------|
| **过度处境化** | 印度教的「基督教瑜伽」——福音被文化淹没 |
| **不足处境化** | 非洲教会唱 18 世纪德国圣诗——「这是什么？」 |

---

## 宣教的历史范式

| 时代 | 范式 |
|------|------|
| 早期教会 | 见证 + 殉道——「血的种子」 |
| 中世纪 | 修道院——凯尔特修士「朝圣者基督」 |
| 殖民时代 | 宣教 + 殖民——复杂遗产——好与坏交织 |
| 20世纪 | 从「西方到其余」→「从各处到各处」——全球基督教兴起 |
| 当代 | 整全使命——传福音 + 社会行动——不可分割 |
