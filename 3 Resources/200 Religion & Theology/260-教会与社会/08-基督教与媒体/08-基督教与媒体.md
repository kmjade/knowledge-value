# 08-基督教与媒体

> 每一场媒体革命——都是**神学**事件。印刷术催生了宗教改革。广播给了葛培理。互联网在重塑基督教的未来——我们还不知道它会把教会带向哪里。

## 媒体的神学

| 原则 | 内容 |
|------|------|
| **道成肉身** | 上帝用人的语言（文字）沟通——沟通的「媒介」本身是有神学意义的 |
| **邻舍关系** | 屏幕上的「邻舍」是真实的邻舍吗？——约翰说「我盼望当面谈」(约二 12) |
| **偶像工厂** | 媒体不只是「传递信息」——它**塑造**欲望——塑造我们爱什么 |

---

## 每一次媒体革命中的教会

| 革命 | 教会的反应 |
|------|-----------|
| 文字 | 圣经→正典——信仰被固化 |
| 印刷术 | 路德——没有印刷机就没有宗教改革 |
| 广播/电视 | 电视布道家——但也使讲道变成了「表演」 |
| 互联网 | 博客→播客→直播崇拜——教会正在学习 |

---

## 数字时代的独特挑战

| 挑战 | 回应 |
|------|------|
| 注意力 | 通知→碎片化——「静默」成为激进的反文化操练 |
| 真实性 | 在线可以伪装——「真实的团契」需要身体同在 |
| AI | ChatGPT 写讲章？——效率 vs 呼召 |
| 算法 | 算法塑造我们的欲望——谁能塑造算法？ |
