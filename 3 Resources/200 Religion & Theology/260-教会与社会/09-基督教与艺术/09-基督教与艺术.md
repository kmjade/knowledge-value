# 09-基督教与艺术

> 「起初——上帝创造。」(创 1:1) 圣经的第一个行动是创造——而人类是按上帝的形象造的——所以我们是**创造的创造者**。艺术不是奢侈——是人性的表达。

## 圣经中的艺术

| 例子 | 内容 |
|------|------|
| **会幕与圣殿** | 比撒列和亚何利亚伯——被圣灵充满——用**艺术**为上帝建造 |
| **诗篇** | 诗歌——不是神学论文——是用最美的语言表达最深的信仰 |
| **大卫** | 音乐家——弹琴驱走扫罗的恶魔——艺术有灵性的力量 |

---

## 历史上的张力

| 时期 | 态度 |
|------|------|
| 早期教会 | 图像争议——不可拜偶像——但也发展出基督教艺术 |
| 拜占庭 | 圣像争辩 (Iconoclasm) ——图像被毁——后恢复——神学上辩护为「道成肉身的见证」 |
| 宗教改革 | 加尔文——只允许音乐——不允许视觉艺术——「清教徒的空白」 |
| 现代 | 「基督教艺术」= 次等？——还是艺术本身可以为上帝的创造作见证？ |

---

## 艺术的呼召

| 原则 | 内容 |
|------|------|
| **创造不是复刻** | 上帝是创造者——我们不是「复制圣经场景」——是**创造新的美** |
| **美不需要「用途」** | 马利亚打破香膏——「枉费」——但耶稣说：「她做的是美事」 |
| **艺术作为先知** | 音乐和视觉可以**说出**讲道不能说出的——触及理性绕不过的部分 |
