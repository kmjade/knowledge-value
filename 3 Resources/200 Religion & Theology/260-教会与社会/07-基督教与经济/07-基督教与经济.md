# 07-基督教与经济

> 「一个人不能事奉两个主……你们不能又事奉上帝——又事奉玛门。」(太 6:24) 耶稣谈论金钱比谈论天堂和地狱加起来还多——经济不是神学的「附录」——是核心议题。

## 旧约的经济伦理

| 制度 | 内容 |
|------|------|
| **安息年** | 每七年——债务赦免——土地休耕——奴隶释放 |
| **禧年** | 每五十年——土地归还原主——彻底的经济「重置」 |
| **拾穗法** | 收割时不收尽田角——留给穷人 |
| **禁止高利贷** | 向穷人收取利息被禁止 |

## 当代立场

| 立场 | 核心 | 代表 |
|------|------|------|
| **资本主义肯定** | 自由市场创造财富——帮助穷人脱贫——「财富创造是爱的行动」 |  Brian Griffiths |
| **资本主义批判** | 资本主义崇拜「增长」——制造不公——需要「先知性」的批判 | 解放神学 |
| **社会经济** | 合作社·公平贸易——不是国有化——也不是纯粹市场 | 基督教社会主义传统 |
