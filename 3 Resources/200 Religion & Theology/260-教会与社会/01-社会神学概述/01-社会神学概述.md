# 01-社会神学概述

> 教会是「山上的城」(太 5:14) ——不是隐藏的——是**可见的**。社会神学追问这个可见的共同体如何存在于世界中。

## 教会的双重身份

| 身份 | 含义 |
|------|------|
| **天国的子民** | 教会不属于这个世界——她的终极忠诚是上帝 |
| **世界中的盐和光** | 但教会被**差遣**到世界中——不是逃离——是进入 |

---

## 公共神学 (Public Theology)

| 原则 | 内容 |
|------|------|
| 神学是对所有人说的——不只是对信徒 | 用公共语言——而不是「教会黑话」 |
| 三个公众 | 教会·学术界·公民社会——神学要同时对话三者 |

---

## 核心张力

| 张力 | 问题 |
|------|------|
| 分别为圣 vs 入世 | 太圣洁就隔离——太入世就同化 |
| 先知 vs 祭司 | 指责社会的罪 vs 为社会的和平代求 |
| 普遍恩典 vs 特殊恩典 | 非基督徒的文化成就——是「恩典」吗？ |
