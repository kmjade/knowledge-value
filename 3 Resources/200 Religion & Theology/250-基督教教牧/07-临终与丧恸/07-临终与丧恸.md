# 07-临终与丧恸

> 「死啊——你得胜的权势在哪里？」(林前 15:55) 基督徒面对死亡不是没有悲伤——是有**盼望**的悲伤。牧者作为「死亡的见证人」——陪伴人走完最后一程。

## 临终陪伴

| 原则 | 内容 |
|------|------|
| **在那里** | 不是说什么——是**在**。握着病人的手——读诗篇 23 |
| **倾听** | 「你有什么想说的？」——未了的心愿、恐惧、感恩 |
| **祷告** | 不是「一定会好起来」——是「无论发生什么——上帝与你同在」 |
| **家属** | 疲惫的照顾者也需要牧养——「你还好吗？」 |

---

## 哀悼的过程

> 库伯勒-罗斯 (Kübler-Ross) 五阶段——不是线性的——是螺旋的：

| 阶段 | 内容 |
|------|------|
| 否认 | 「这不是真的」 |
| 愤怒 | 「为什么是我/他？」——对上帝、医生产生愤怒 |
| 讨价还价 | 「如果……我愿意……」 |
| 抑郁 | 深沉的悲伤——这**不是**缺乏信心——是人类的正常反应 |
| 接受 | 不是「没事了」——是**学会了带着失去活下去** |

> 约伯撕裂衣服——然后他敬拜。两者都是信心的表达。

---

## 葬礼

| 要素 | 内容 |
|------|------|
| **福音** | 葬礼的核心是**复活**——不是悼念死者的美德 |
| **诚实** | 承认「这个人不完美」——不用把所有人都说成圣人 |
| **盼望** | 「不是永别——是再见」 |
| **文化敏感** | 尊重家庭的文化传统——但避免与基督教信仰冲突的仪式 |
