# 06-婚姻与家庭

> 婚姻是上帝设立的——但教会往往只在婚礼那天才想起这个教义。家庭事工不只是「婚前辅导」——是从摇篮到坟墓的全生命周期关怀。

## 婚前辅导

| 主题 | 内容 |
|------|------|
| **期望** | 「你认为婚姻是什么？」——发现隐藏的期望 |
| **沟通** | 冲突模式——你们吵架后会怎么和好？ |
| **财务** | 钱——离婚的三大原因之一 |
| **原生家庭** | 「你从你的父母那里学到了什么关于婚姻的？」 |

---

## 婚姻中的牧养

| 阶段 | 挑战 | 牧养回应 |
|------|------|---------|
| 新婚 | 磨合——两个世界碰撞 | 门训——不是只说「不要离婚」 |
| 养育 | 孩子改变一切——关系被挤压 | 鼓励定期约会——保护婚姻 |
| 中年 | 空巢·中年危机 | 重新发现彼此 |
| 晚年 | 健康·退休·照顾 | 陪伴——不是「你们已经没问题了」 |

---

## 困难议题

| 议题 | 牧养原则 |
|------|---------|
| **离婚** | 既认真对待婚姻的永久性——又怜悯破碎的人——不是二选一 |
| **家暴** | 安全第一——不是「为婚姻忍忍」——施暴者需要后果 |
| **不婚/单身** | 教会不应把单身当作「二等状态」——保罗是单身 |
