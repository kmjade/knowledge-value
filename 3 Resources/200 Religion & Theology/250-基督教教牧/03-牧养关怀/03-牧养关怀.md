# 03-牧养关怀

> 「与哀哭的人同哭。」(罗 12:15) 牧养关怀的核心不是「解决问题」——是**同在**。在你学到任何辅导技巧之前——你先要学会**坐**在有痛苦的人旁边。

## 牧养辅导 vs 专业辅导

| 维度 | 牧养辅导 | 专业心理咨询 |
|------|---------|------------|
| 基础 | 圣经·神学 | 心理学理论 |
| 目标 | 灵性成长 + 解决问题 | 心理健康 |
| 关系 | 牧者→会友——长期 | 治疗师→来访——限时 |
| 界限 | 较模糊——探访·吃饭 | 较清晰——50分钟 |
| **转介** | 严重精神疾病——**必须转介**——牧者不是精神科医生 |

---

## 聆听的艺术

| 原则 | 内容 |
|------|------|
| **先听后说** | 「快快的听——慢慢的说」(雅 1:19) ——大多数人倾听是为了「准备回答」——不是理解 |
| **不急于「解决」** | 约伯的朋友前七天是对的——他们**坐着**——不说话——一开口就错了 |
| **反映式聆听** | 「你似乎在说……」——确认你听懂了——而不是「我也有类似经历……」 |

---

## 探访

| 类型 | 内容 |
|------|------|
| **医院探访** | 短——读一段诗篇——祷告——不逗留 |
| **家庭探访** | 在自然环境中认识会友——他们的书架比周日早晨的问候告诉你的更多 |
| **危机探访** | 突发——事故·丧亲——**你在那里**——比你说什么都重要 |

---

## 辅导的局限

| 情况 | 行动 |
|------|------|
| 自杀风险 | **立即转介**——联系危机干预 |
| 精神疾病 | 转介精神科——持续牧养支持 |
| 虐待 | 保护受害者——报告——不是「维护教会名声」 |
