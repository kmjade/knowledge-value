# 09-牧者自我关怀

> 牧师是「灵魂的医生」——他们自己也需要医治。统计数据说每月有 1500 位牧师离开服事——不是离开「那个教会」——是离开「牧职」——burnout。你不能从空杯子里倒水。

## Burnout 的信号

| 信号 | 内容 |
|------|------|
| **情感枯竭** | 不再关心会友的故事——秘密盼着他们不要来 |
| **灵性枯干** | 圣经变成了「讲章材料」——祷告变成了「聚会程序」 |
| **愤世嫉俗** | 「这些人永远不会改变」——当初的呼召变成了日常工作 |
| **身体信号** | 失眠·久病·倦怠——身体比灵魂先说话 |

---

## 安息的操练

| 操练 | 内容 |
|------|------|
| **周间的安息日** | 牧师不能在周日安息——那周一是你的「安息日」 |
| **每日的安静** | 至少30分钟——不是为了准备讲章——是与上帝独处 |
| **退修** | 每年至少一次——离开——安静——不是「在职培训」——是**与上帝** |

---

## 支持系统

| 支持 | 内容 |
|------|------|
| **属灵同伴** | 不是会友——是可以倾诉软弱的人 |
| **督导** | 有经验的牧者——定期见面 |
| **专业辅导** | 牧师也需要看咨询师——不是「信心不够」——是智慧的 |
| **保护家庭** | 牧者配偶不是「无偿同工」——保护婚姻和家庭时间 |
