# 08-冲突与纪律

> 教会冲突是不可避免的——因为教会里有**人**。问题不是「会不会有冲突」——是「怎么处理」。逃避冲突的牧者——和不追求和睦的牧者——都在伤害教会。

## 冲突的来源

| 来源 | 内容 |
|------|------|
| **神学差异** | 不是所有分歧都是「真理 vs 异端」——有些是可以容忍的 |
| **权力与控制** | 「我们一直是这么做的」——传统 vs 改变 |
| **个性冲突** | 不是教义问题——是两个人合不来 |
| **未解决的伤害** | 五年前的某句话——没人处理——变成了毒根 |

---

## 马太福音 18 章原则

| 步骤 | 内容 |
|------|------|
| 1. 私下谈 | 「趁着只有他和你在一处的时候」(太 18:15) ——不是先到处说 |
| 2. 带一两个人 | 不是「证人」——是帮助**分辨和调解** |
| 3. 告诉教会 | 不是公审——是最后的属灵权柄介入 |
| 4. 若仍不悔改 | 看为「外邦人」——即不在教会成员之列 |

---

## 教会纪律

| 目的 | 不是惩罚——是**挽回**——不是「开除」——是「你走得太远了——我们想要你回来」 |
|------|------|

| 情况 | 行动 |
|------|------|
| 持续性不悔改的罪 | 实施纪律——但始终带着挽回的心 |
| 教义上的严重错误 | 「可容忍」与「不可容忍」——需要智慧分辨 |

### 错误的方式

| 错误 | 说明 |
|------|------|
| 不处理 | 「时间会治愈」——不会——会溃烂 |
| 公开羞辱 | 不是纪律——是报复 |
