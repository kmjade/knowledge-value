# 02-讲道学

> 「你当竭力在上帝面前得蒙喜悦——作无愧的工人——按着正意分解真理的道。」(提后 2:15) 讲道不只是「分享」——是上帝通过人的声音说话。

## 讲道的类型

| 类型 | 内容 | 优点 | 风险 |
|------|------|------|------|
| **释经讲道** | 一段经文的逐节讲解 | 忠于圣经 | 可能枯燥——变成课堂 |
| **主题讲道** | 一个主题——引用多段经文 | 全面 | 可能脱离上下文——proof-texting |
| **叙事讲道** | 用故事的方式讲 | 吸引人 | 可能弱化教义 |
| **先知性讲道** | 针对社会议题——以上帝的名义说话 | 有力 | 可能变成政治演说 |

---

## 讲章的准备

| 步骤 | 内容 |
|------|------|
| 1. **选择经文** | 根据教会年历？逐卷系列？还是当下需要？ |
| 2. **研经** | 上下文·原文·历史背景——不是为了找「亮点」——是理解作者的意思 |
| 3. **找出核心信息** | 一句话——这段经文在说什么？——如果不能用一句话说清——说明还没想清楚 |
| 4. **搭建结构** | 引言→主体（2-4点）→结论——不是三段式论文——是引导会众与经文相遇 |
| 5. **应用** | 不只是「知道了什么」——是「现在要做什么」？ |
| 6. **祷告** | 不是「准备完了」——是「上帝——求你说话」 |

---

## 讲道的危险

| 危险 | 说明 |
|------|------|
| **以自己为中心** | 讲自己的故事比讲圣经多 |
| **讨好听众** | 「耳朵发痒」——只讲舒服的——不讲扎心的 |
| **律法主义** | 每次的结论都是「你要更努力」——而不是「基督为你成全了」 |

---

## 当代挑战

| 挑战 | 回应 |
|------|------|
| 注意力变短 | 讲道不必更短——但需要更清晰的结构 |
| 屏幕文化 | 用视觉辅助——但不是代替 |
| 后现代 | 不只讲「这是真理」——也讲「为什么这与你的生活有关」 |
