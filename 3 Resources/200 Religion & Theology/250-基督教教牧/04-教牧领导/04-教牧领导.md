# 04-教牧领导

> 领导不是「让别人做你想做的事」——是「帮助别人做上帝要他们做的事」。教会领导——不是 CEO 模式——是仆人模式。

## 领导的圣经模式

| 模式 | 内容 |
|------|------|
| **仆人领导** | 耶稣洗门徒的脚——「你们中间谁愿为大——就必作你们的用人」(太 20:26) |
| **牧人领导** | 走在前面——不是驱赶——是**带领** |
| **管家领导** | 教会不是「我的」——是上帝的——我只是管家 |

---

## 异象与策略

| 要素 | 内容 |
|------|------|
| **异象** | 上帝要这个教会成为什么？——不是你想出来的——是**分辨**出来的 |
| **使命** | 这个教会具体做什么来实现异象？ |
| **策略** | 怎么做？——步骤·资源·时间线 |

---

## 建立团队

| 原则 | 内容 |
|------|------|
| **恩赐导向** | 不是「有空的人做事」——是「有恩赐的人服事」 |
| **授权** | 摩西听从叶忒罗——分权——「你独自一人办理不了」(出 18) |
| **发展领袖** | 保罗→提摩太→忠心的人——领导力不是「储存」——是「复制」 |

---

## 教会行政

| 领域 | 内容 |
|------|------|
| **财务** | 透明——预算——不是只凭「信心」——是凭「忠心」 |
| **人事** | 聘牧·同工评估·冲突处理 |
| **建筑** | 教堂不只是「功能性」的——它**说话**——美观·庄严·欢迎 |
| **法律** | 注册·税务·儿童保护政策——教会不是游离于法律之外 |
