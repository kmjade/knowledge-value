# 05-基督教教育

> 「教养孩童——使他走当行的道。」(箴 22:6) 基督教教育不只是一周一个小时的主日学——是帮助人从「不知道」到「知道」——再到「活出来」。

## 基督教教育的对象

| 年龄 | 内容 | 方法 |
|------|------|------|
| 儿童 | 圣经故事·基本教义·品格 | 讲故事·手工·唱歌 |
| 青少年 | 信仰与理性·护教学·身份 | 对话·小组·营会 |
| 成人 | 门训·圣经研究·神学 | 课程·小组·一对一 |
| 长者 | 回顾人生·智慧传承 | 见证·团契 |

---

## 教理问答 (Catechesis)

| 传统 | 问答 |
|------|------|
| 《海德堡要理问答》(1563) | 「你生与死的唯一安慰是什么？」——「我不属于自己——属于我信实的救主耶稣基督。」 |
| 《威斯敏斯特小要理问答》(1647) | 「人的首要目的是什么？」——「荣耀上帝——并以他为乐——直到永远。」 |

> 教理问答不是「洗脑」——是**给信徒语言**——当他们不知道该怎么表达信仰的时候——已经有人为教会写了最好的表述。

---

## 教育的模式

| 模式 | 内容 |
|------|------|
| **课堂式** | 教师讲→学生听——传统模式 |
| **对话式** | 苏格拉底式——提问引导 |
| **体验式** | 服事中学习——「做中学」 |
| **门训式** | 生命影响生命——不只是「上课」——是一起生活 |

---

## 当代挑战

| 挑战 | 回应 |
|------|------|
| 圣经文盲 | 很多会友从未完整读过一卷书——从基础开始 |
| 文化多元 | 不是「一套教材教所有人」 |
| 数字时代 | 在线课程——但不能取代面对面的团契 |
