# 08-五旬宗与灵恩派

> 1906年洛杉矶阿苏撒街——五旬宗的「大爆炸」。今天五旬宗/灵恩派约 6.5 亿——基督教内增长最快的运动。

## 五旬宗的核心

| 特征 | 内容 |
|------|------|
| **圣灵的洗** | 不同于归信——是「第二次恩典」——通常以说方言为证据 |
| **属灵恩赐** | 方言·预言·医病·赶鬼——不是「使徒时代结束了」——是「仍然在今天」 |
| **敬拜** | 活泼·自由·身体参与——举手·跳舞·呼喊 |

## 灵恩运动 (1960s-)

| 与五旬宗的区别 | 不离开自己的宗派——**在天主教·圣公会·路德宗中**经历灵恩 |
|------|------|
| 第三次浪潮 | 温约翰 (John Wimber) ——「神迹奇事」——葡萄园运动 |

## 争议

| 争议 | 内容 |
|------|------|
| 成功神学 (Prosperity Gospel) | 「信耶稣就发财得医治」——被广泛批评 |
| 情感主义 | 追求「体验」超过追求真理？ |

## 全球影响

| 地区 | 五旬宗增长最快——非洲·拉丁美洲——亲自体验上帝的力量——不受西方神学限制 |
|------|------|
