# 03-东正教

> 「正教」(Orthodox) = 正确的荣耀 + 正确的教义。约 2.2 亿信徒。

## 与天主教的分歧

| 议题 | 天主教 | 东正教 |
|------|--------|--------|
| 教皇 | 普世最高权威 | 「同等中的首席」——无管辖权 |
| 和子句 | 圣灵从父「和子」发出 | 从父发出 |
| 无原罪受孕 | 信理 | 不承认 |
| 圣餐 | 无酵饼 | 有酵饼 |
| 神职人员婚姻 | 独身(拉丁礼) | 司铎可结婚(祝圣前)——主教独身 |

## 礼仪生活

| 核心 | Divine Liturgy——不是讲道——是**天堂在地上的临现** |
|------|------|
| 圣像 | 不是偶像——是「道成肉身的见证」——通过物质触摸神圣 |
| 大斋 | 严格的禁食——一年约180天 |

## 东正教大家庭

| 教会 | 特征 |
|------|------|
| 希腊正教会 | 君士坦丁堡普世牧首——「同等中的首席」 |
| 俄罗斯正教会 | 最大——约1亿信徒 |
| 东方正教会 (Oriental) | 451年迦克墩后分裂——科普特·亚美尼亚 |
