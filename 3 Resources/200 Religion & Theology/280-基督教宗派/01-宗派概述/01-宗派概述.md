# 01-宗派概述

> 耶稣祷告「使他们都合而为一」(约 17:21) ——但今天有数千个宗派。这是基督教的耻辱——还是多样性的见证？

## 为什么有宗派？

| 原因 | 内容 |
|------|------|
| 历史 | 东西分裂(1054)·宗教改革(1517)——重大事件产生新宗派 |
| 神学 | 洗礼·圣餐·预定·恩赐——不同意就分开 |
| 文化 | 同一神学在不同文化中演变成不同宗派 |
| 政治 | 「教随国定」——国王选宗派——人民跟着 |
