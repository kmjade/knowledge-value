# 06-圣公会

> 亨利八世因为离婚问题与罗马决裂——但圣公会自认不是「新教宗派」——而是天主教和改革宗之间的「中道」(Via Media)。约 8500 万。

## 核心

| 特征 | 内容 |
|------|------|
| **中道** | 不是罗马——也不是日内瓦——是两者之间 |
| **公祷书** (Book of Common Prayer) | 圣公会的核心——不是信仰告白——是共同**祷告** |
| **圣经·传统·理性** | 三个权威来源——「三脚凳」(胡克) |

## 兰贝斯四纲 (1888)

| 四纲 | 合一的基础：圣经·信经·圣礼·历史主教制 |
|------|------|
