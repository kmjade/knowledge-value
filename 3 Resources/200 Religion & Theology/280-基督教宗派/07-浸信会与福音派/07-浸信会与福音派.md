# 07-浸信会与福音派

> 浸信会 = 信徒洗礼 + 会众制 + 宗教自由。全球约 1 亿。福音派是一个跨宗派运动——不是宗派。

## 浸信会的核心

| 特征 | 内容 |
|------|------|
| **信徒洗礼** | 只有认信后才能受洗——浸入水中——不是婴儿洗 |
| **会众制** | 每个地方教会自治——没有主教和总会管辖 |
| **宗教自由** | 在 17 世纪为所有人都能自由信仰而奋斗 |

## 福音派

| 特征 | 内容 |
|------|------|
| 跨宗派 | 浸信会·长老会·五旬宗——不同宗派共享福音派信仰 |
| 核心 | 归信主义·圣经权威·十字架中心·行动主义 (Bebbington四角) |
| 全球 | 全球福音派正在快速增长——尤其在非洲·拉美·亚洲 |

## 当代福音派

| 议题 | 张力 |
|------|------|
| 政治 | 美国福音派政治化——特朗普时代的争议 |
| 文化战争 | 堕胎·LGBTQ+——福音派是「道德多数」吗？ |
