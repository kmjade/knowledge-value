# 02-天主教

> 全球 13 亿信徒——最大的基督教宗派。天主教宣称自己是「从使徒传下来的唯一、至圣、至公的教会」。

## 特征

| 特征 | 内容 |
|------|------|
| **教皇** | 彼得继承者——「磐石」——无谬误 (1870年定义) |
| **圣经 + 圣传** | 启示的两个来源——不是「唯独圣经」 |
| **七圣礼** | 洗·坚振·圣餐·告解·婚配·圣秩·傅油 |
| **马利亚** | 圣母——无原罪受孕 (1854) ——蒙召升天 (1950) |
| **变质说** | 圣餐中饼和酒的本质变为基督的身体和血 |

## 关键转折

| 事件 | 内容 |
|------|------|
| 天特会议 (1545-63) | 反宗教改革——澄清教义 |
| 梵一 (1869-70) | 教皇无谬误 |
| 梵二 (1962-65) | 现代化——本地语言礼仪——与其他宗教对话 |
