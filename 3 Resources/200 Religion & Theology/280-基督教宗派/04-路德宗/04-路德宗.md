# 04-路德宗

> 全球约 7500 万——最早的抗议宗——马丁·路德的遗产。

## 核心

| 信念 | 内容 |
|------|------|
| 唯独恩典·唯独信心·唯独圣经 | 三个「唯独」 |
| 律法与福音 | 律法显明罪——福音赐予赦免——讲道必须两者都有 |
| 圣餐 | 「同质说」——基督的身体「在·与·下」饼和酒临在 |

## 路德宗的特征

| 特征 | 内容 |
|------|------|
| 信义宗协和书 (1580) | 路德宗信条的总集 |
| 礼仪 | 保留较多传统——在「天主教去掉的」和「改革宗去掉的」之间 |
| 两国论 | 上帝的左手(国家——维持秩序)和右手(教会——赐予恩典) |
| 分布 | 德国·北欧——国家教会传统 |

## 主要分支

| 分支 | 特征 |
|------|------|
| 路德宗-密苏里会 (LCMS) | 保守——美国 |
| 美国福音路德教会 (ELCA) | 自由——女性按立·LGBTQ+ |
