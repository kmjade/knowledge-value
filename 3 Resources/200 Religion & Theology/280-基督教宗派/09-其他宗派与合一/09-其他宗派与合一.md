# 09-其他宗派与合一

## 卫理宗 (Methodist)

| 起源 | 约翰·卫斯理——18世纪英国复兴——「方法」——每日灵修操练 |
|------|------|
| 神学 | 阿米念主义——自由意志——成圣的强调——「完全的爱」 |
| 特征 | 小组系统——社会关怀——医院·学校 |
| 约 8000 万 | 联合卫理公会 (UMC) ——美国最大之一 |

## 门诺会 (Mennonite)

| 起源 | 重洗派——16世纪——因坚持「信徒洗礼」被天主教和新教都迫害 |
|------|------|
| 特征 | 和平主义——简朴——与世分离 |
| 阿米什人 | 门诺会的一支——拒绝现代技术 |

## 贵格会 (Quaker)

| 起源 | 乔治·福克斯——17世纪英国 |
|------|------|
| 特征 | 「内在之光」——静默等候——**没有圣礼**——**没有牧师**——人人都是祭司 |
| 贡献 | 废奴运动·监狱改革·和平主义 |

## 复临安息日会

| 特征 | 守安息日 (周六) ——健康信息——全球教育网络 |
|------|------|

## 合一运动的现状

| 现状 | 组织合并——但文化差异和神学差异仍然巨大 |
|------|------|
| 希望 | 不是「一个宗派」——是「在差异中承认彼此是基督徒」 |
