# 05-改革宗

> 加尔文 + 慈运理 + 诺克斯……改革宗传统以「上帝的主权」为中心。约 8000 万长老会 + 改革宗教会成员。

## 核心

| 信念 | 内容 |
|------|------|
| **上帝的主权** | 一切都在上帝的掌管之下——包括救恩 |
| **圣约神学** | 上帝与人立约——旧约的应许在新约中实现 |
| **TULIP** | 加尔文主义五要点 (见救赎论) |
| **圣餐** | 属灵临在——不是变质——也不是纯纪念 |

## 改革宗的特征

| 特征 | 内容 |
|------|------|
| 教会治理 | 长老制——地方堂会→区会→总会 |
| 信条 | 威斯敏斯特信条 (1647) · 海德堡要理问答 (1563) |
| 文化使命 | 凯波尔 (Abraham Kuyper) ——「没有一寸不是基督的」——改革宗对社会文化的积极转化 |

## 主要分支

| 分支 | 特征 |
|------|------|
| 长老会 (PCUSA) | 主流——自由 |
| 改革宗 (CRC, URC) | 较保守 |
| PCA | 保守长老会——美国增长 |
