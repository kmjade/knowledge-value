# 07-门徒训练

> 「来跟从我。」(太 4:19) ——耶稣不是叫人来听课——是叫人来**跟着**他。门徒训练不是「学一套教义」——是学一种生活方式。

## 门徒的核心

| 要素 | 内容 |
|------|------|
| **跟随** | 有一位在前面——你在后面——不只是相信——是跟随 |
| **效法** | 「学生不能高过先生——但学成了不过和先生一样」(路 6:40) |
| **代价** | 「若有人要跟从我——就当舍己——背起他的十字架」(太 16:24) |

---

## 门徒训练的圣经模式

### 耶稣的模式

| 阶段 | 内容 |
|------|------|
| 「来——看」 | 耶稣邀请门徒与他同住——观察他的生活 |
| 「来——跟从我」 | 放下渔网——全职跟随 |
| 「来——与我同在」 | 十二门徒——更深入的教学 |
| 「去——为我而活」 | 复活后——差遣 |

### 保罗的模式

> 「你们该效法我——像我效法基督一样。」(林前 11:1)

| 模式 | 保罗→提摩太→「忠心能教导别人的人」(提后 2:2) |
|------|------|
| 四代传递——保罗→提摩太→忠心的人→别人 |

---

## 属灵指导

| 要素 | 内容 |
|------|------|
| **定义** | 一个更有经验的基督徒帮助另一个人留意上帝在她/他生命中的作为 |
| **与辅导的区别** | 辅导解决问题——属灵指导帮助**分辨上帝的工作** |
| **历史** | 沙漠教父——「阿爸——给我一句话」——一句可以反复咀嚼的指引 |

---

## 当代门训实践

| 形式 | 内容 |
|------|------|
| **小组** | 现代教会最基本的门训单元 |
| **一对一** | 定期的门徒关系——读经、祷告、问责 |
| **生命对生命** | 不只是「教」——是一起生活——一起吃饭——一起服事 |
