# 09-当代灵性实践

> 21世纪的基督徒怎么灵修？在智能手机的时代如何「静默」？在996的节奏中如何「安息」？当代灵性不是发明新东西——是在新处境中重新发现旧智慧。

## 退修 (Retreat)

| 类型 | 内容 |
|------|------|
| **静默退修** | 3-8天的安静——不说话——只祷告 |
| **依纳爵退修** | 30天神操——或在日常生活中分9个月完成 |
| **日常退修** | 不是「去某个地方」——是在日常中划出「旷野时间」 |

---

## 数字时代的灵修

| 挑战 | 回应 |
|------|------|
| 信息过载 | 「你们要休息，要知道我是上帝」——每天关机1小时 |
| 碎片化注意力 | 读经 App 好用——但纸本圣经让眼睛慢下来 |
| 社交媒体焦虑 | 「比较」是灵修的天敌——操练「隐藏」 |
| AI | 让 AI 写祷告？——工具可以辅助——但关系不能外包 |

---

## 新形态教会与实践

| 形态 | 内容 |
|------|------|
| **家庭教会** | 地下教会传统——在受逼迫地区延续至今 |
| **新兴教会** (Emerging Church) | 后现代语境中重新想象教会——对话取代独白 |
| **弥散教会** | 疫情后——在线聚会——虚拟圣餐的争议 |

---

## 日常生活中的灵修

| 实践 | 内容 |
|------|------|
| **呼吸祷告** | 在通勤、排队的间隙——在心里说一句短祷 |
| **意识省察** | 依纳爵的「每日省察」——今天上帝在哪里出现了？我今天在哪里拒绝了上帝？ |
| **安息日** | 每周24小时——不工作——不是「什么都不做」——是「做与六天不同的事」 |
| **身体灵修** | 走路祷告·瑜伽式伸展（基督徒改编）——身体参与敬拜 |
