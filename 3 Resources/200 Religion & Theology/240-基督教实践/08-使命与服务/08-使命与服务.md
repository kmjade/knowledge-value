# 08-使命与服务

> 「你们往普天下去——传福音给万民听。」(可 16:15) ——大使命。但耶稣也说：「我饿了——你们给我吃。」(太 25:35) 传福音和服务——不是二选一。

## 大使命 vs 大诫命

| | 大使命 | 大诫命 |
|------|--------|--------|
| 内容 | 使万民作门徒 | 爱上帝 + 爱邻舍 |
| 焦点 | 传讲 | 行动 |
| 张力 | 是否只有传福音才算「宣教」？ | 是否所有善行都是「宣教」？ |

> 当代共识：**言行合一**——「用话语——也在必要时用话语」的说法虽然巧妙——但圣方济可能没说过——而且耶稣两者都做了。

## 宣教的历史形态

| 时期 | 模式 | 特点 |
|------|------|------|
| 早期教会 | 见证 | 日常生活中分享信仰——殉道者的血是种子 |
| 中世纪 | 修道院 | 爱尔兰修士——「朝圣者基督」——旅行布道 |
| 殖民时代 | 殖民宣教 | 宣教士随殖民者进入——复杂遗产 |
| 19-20世纪 | 信心差会 | 戴德生——中国内地会——「不募款——只祷告」 |
| 当代 | 整全使命 | 传福音 + 社会行动——不可分割 |

## 基督教慈善的历史

| 创新 | 内容 |
|------|------|
| **医院** | 罗马帝国没有公共医疗——基督徒创建了第一所医院 |
| **孤儿院** | 早期教会收养弃婴 |
| **教育** | 主日学校运动 (18世纪英国) ——让贫穷儿童学会阅读 |
| **废奴** | 威伯福斯 (Wilberforce) ——受福音驱动的政治行动 |

## 当代公义议题

| 议题 | 基督徒回应 |
|------|-----------|
| 贫穷 | 不只是施舍——是**赋能**——Microfinance、公平贸易 |
| 人口贩卖 | 国际正义使命 (IJM) ——律师+社工+教会 |
| 环境 | 「创造关怀」——「治理这地」(创 1:28) 意味着**保护**——不是掠夺 |
