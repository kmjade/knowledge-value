# 03-崇拜与礼仪

> 「礼仪」(Liturgy) ——希腊语 leitourgia = 「公众的工作」。崇拜不是观众运动——是上帝子民一起**献上**的。

## 崇拜的要素

| 要素 | 内容 |
|------|------|
| **圣道** | 读经 + 讲道——上帝通过他的话说话 |
| **圣餐** | 记念基督的死——领受恩典 |
| **祷告** | 群体性的赞美·认罪·祈求 |
| **音乐** | 圣诗·诗歌——「歌唱是双倍的祷告」(奥古斯丁) |

---

## 教会年历

| 节期 | 时间 | 焦点 |
|------|------|------|
| **将临期** | 圣诞前四周 | 等候——基督的降临 |
| **圣诞期** | 12/25-1/6 | 庆祝——道成肉身 |
| **显现期** | 1/6-大斋前 | 基督向世界显现 |
| **大斋期** (四旬期) | 40天 | 悔改·禁食·默想基督的苦难 |
| **复活期** | 复活节-五旬节 | 庆祝——基督复活 |
| **五旬节** | 复活后50天 | 圣灵降临——教会的诞生 |
| **常年期** | 其余时间 | 日常的基督徒生活 |

> 教会年历不只是日历——是**时间的门训**——每年重走一遍基督的生平。

---

## 崇拜的两种风格

| 维度 | 礼仪崇拜 (高教会) | 自由崇拜 (低教会) |
|------|-----------------|-----------------|
| 结构 | 固定礼仪书 | 较自由 |
| 音乐 | 管风琴·圣诗班 | 乐队·当代诗歌 |
| 氛围 | 庄严·神秘 | 亲密·活泼 |
| 宗派 | 天主教·东正教·圣公会·路德宗 | 浸信会·五旬宗·福音派 |

---

## 圣乐

| 传统 | 代表 |
|------|------|
| 格里高利圣咏 | 单声·无伴奏——最古老的西方音乐传统 |
| 众赞歌 | 路德——「坚固保障」——会众自己唱 |
| 黑人灵歌 | 被奴役者的信仰表达——蓝调·爵士的源头 |
| 当代敬拜 | Hillsong, Bethel ——全球化敬拜运动 |
