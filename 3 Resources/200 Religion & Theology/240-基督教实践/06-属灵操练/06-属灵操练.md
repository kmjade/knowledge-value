# 06-属灵操练

> 保罗说：「我是攻克己身——叫身服我。」(林前 9:27) 属灵操练不是赚取恩典的手段——是把你自己放在恩典的**河道**上。

## 内在操练

### 禁食

| 维度 | 内容 |
|------|------|
| **含义** | 自愿戒除食物——为了专注祷告 |
| **不是** | 减肥——示威——「感动」上帝 |
| **是** | 身体对灵魂说：「我比你想象中更需要上帝。」 |
| **模式** | 部分禁食 (但以理)、完全禁食、间歇禁食 |

### 静默与独处

| 概念 | 内容 |
|------|------|
| **静默** | 停止说话——也停止内心的噪音——「你们要休息——要知道我是上帝」(诗 46:10) |
| **独处** | 离开人群——不是逃避——是为了更好地**回去** |

---

## 外在操练

### 简朴

| 维度 | 内容 |
|------|------|
| **内在简朴** | 单一的心——「眼睛若了亮——全身就光明」 |
| **外在简朴** | 有意的少消费——不是因为穷——是因为「够了」 |

### 服事

| 原则 | 内容 |
|------|------|
| **隐藏的服事** | 耶稣说：「不要叫左手知道右手所做的」 |
| **小事的服事** | 服事不是「做大事」——是做别人不愿意做的小事 |

---

## 群体操练

| 操练 | 内容 |
|------|------|
| **认罪** | 向另一个人承认你的罪——打破罪的隐秘力量 |
| **敬拜** | 不是为了「感觉好」——是把上帝应得的荣耀归给他 |
| **庆祝** | 基督徒不只是严肃的——信仰里有深刻的喜乐 |

---

## 操练的危险

| 危险 | 说明 |
|------|------|
| **把操练当作目的** | 操练是工具——不是目的——目的是上帝 |
| **骄傲** | 「我禁食了一周」——属灵的虚荣 |
| **与恩典脱节** | 操练不是「表现」——是「把自己放在可以被恩典找到的位置」 |
