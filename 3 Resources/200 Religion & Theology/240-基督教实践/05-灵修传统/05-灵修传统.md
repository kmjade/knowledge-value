# 05-灵修传统

> 两千年来——基督徒在追问同一个问题：怎样「不住地祷告」(帖前 5:17)？不同的时代给出不同的回答——形成了丰富的灵修传统。

## 沙漠教父 (3-5世纪)

| 人物 | 教导 |
|------|------|
| **安东尼** (251-356) | 第一位隐修士——退入埃及沙漠——「你的斗室会教导你一切」 |
| **沙漠教父箴言** | 一个修士问：「我该做什么？」长者说：「去——坐在你的斗室里——你的斗室会教导你一切。」 |

| 核心操练 | 静默·独处·「心的守护」——不断回到上帝面前 |
|---------|------|

---

## 本笃会 (6世纪)

> 圣本笃 (480-547) ——《圣本笃会规》——「祈祷与工作」(Ora et Labora)

| 三个誓言 | 贫穷·贞洁·顺服 |
|----------|------|
| **稳定** | 不四处游荡——委身于一个地方、一个群体——这是反文化的 |
| **礼仪日课** | 一天七次祷告——把一天分别为圣 |

---

## 依纳爵灵修 (16世纪)

> 依纳爵·罗耀拉 (1491-1556) ——耶稣会创始人——《神操》

| 核心 | 内容 |
|------|------|
| **神操** | 30天的默想退修——四「周」——跟随基督的生平 |
| **在一切事上看见上帝** | 上帝不只是「在教堂里」——在日常工作和决定中寻找他 |
| **分辨** | 如何知道一个冲动是来自上帝——还是来自自己？——「神慰」和「神枯」 |

---

## 奥秘派传统

| 人物 | 核心 |
|------|------|
| 艾克哈特大师 (Meister Eckhart, 13-14世纪) | 上帝在灵魂的最深处——「上帝的根基和灵魂的根基是同一个根基」 |
| 十架约翰 (16世纪) | 「灵魂的暗夜」——在一切安慰被剥夺时——信心反而被炼净 |
| 大德兰 (16世纪) | 《七宝楼台》——灵魂像蚕变成蝴蝶——逐渐进入与上帝的合一 |

---

## 现代灵修运动

| 人物 | 贡献 |
|------|------|
| 傅士德 (Richard Foster) | 《属灵操练礼赞》——1978年——重新引介古典属灵操练给当代基督徒 |
| 卢云 (Henri Nouwen) | 「负伤的治疗者」——用自身的破碎来服事破碎的人 |
| 魏乐德 (Dallas Willard) | 《心灵的重塑》——门徒训练不只是行为改变——是心灵的改变 |
