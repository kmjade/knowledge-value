# 04-圣礼实践

> 基督设立了洗礼和圣餐——不是抽象的神学概念——是**触摸得到的恩典**。水、饼、酒——最普通的物质承载最深的奥秘。

## 洗礼

### 实践模式

| 宗派 | 方式 | 年龄 | 神学重点 |
|------|------|------|---------|
| 东正教 | 浸水三次——奉三一之名 | 婴儿 | 进入教会——重生 |
| 天主教 | 注水 | 婴儿 | 除去原罪——进入恩典 |
| 浸信会 | 全身入水 | 认信后 | 公开见证——「与基督同死同复活」 |
| 长老会 | 洒水/注水 | 婴儿 | 约的记号——如旧约割礼 |

### 洗礼的意义

| 维度 | 内容 |
|------|------|
| **与基督联合** | 「所以我们借着洗礼归入死——和他一同埋葬」(罗 6:4) |
| **洁净** | 罪的赦免——「起来——求告他的名受洗——洗去你的罪」(徒 22:16) |
| **进入教会** | 「都从一位圣灵受洗——成了一个身体」(林前 12:13) |

---

## 圣餐/主的晚餐

### 礼仪结构

| 部分 | 内容 |
|------|------|
| **预备** | 认罪·赦罪 |
| **圣道礼仪** | 读经·讲道·信经 |
| **圣餐礼仪** | 祝圣——「这是我的身体」——分饼分杯 |
| **差遣** | 祝福——「去吧——在平安中服事主」 |

### 频率

| 传统 | 频率 |
|------|------|
| 天主教 | 每次弥撒 (每日) |
| 东正教 | 每次圣礼 (主日+节日) |
| 改革宗 | 每月或每季 |
| 早期教会 | 每次聚会：「他们天天同心合意……擘饼」(徒 2:46) |

## 其他圣礼？

| 宗派 | 圣礼数量 | 除洗礼和圣餐外 |
|------|:---:|------|
| 天主教/东正教 | 7 | 坚振·告解·婚配·圣秩·傅油 |
| 新教 | 2 | — |
| 贵格会 | 0 | 全部生活都是圣礼——不需要特殊仪式 |

---

## 圣礼的核心张力

| 张力 | 问题 |
|------|------|
| **客观 vs 主观** | 圣礼是上帝的作为——还是人的回应？ |
| **形式 vs 自由** | 礼仪固定——还是每次不同？ |
| **神秘 vs 理解** | 需要理解才能领受——还是奥秘超越理解？ |
