# 02-祷告

> 「主啊，请教我们祷告。」——门徒的请求。耶稣给了他们主祷文。两千年后——基督徒仍在用这最简单的祷告词——也仍在探索祷告的深度。

## 祷告的类型

| 类型 | 内容 | 例子 |
|------|------|------|
| **赞美** | 为上帝的本性赞美他 | 诗篇 8 |
| **感恩** | 为上帝的作为感谢 | 诗篇 103 |
| **认罪** | 承认自己的罪 | 诗篇 51 |
| **祈求** | 为自己和他人求 | 主祷文「我们日用的饮食」 |
| **代求** | 为他人祷告 | 摩西为以色列代求 |
| **叹息** | 「说不出来的叹息」(罗 8:26) ——最深层的祷告 |

---

## 主祷文 (太 6:9-13)

| 部分 | 内容 | 焦点 |
|------|------|------|
| 我们在天上的父 | 关系——上帝是父亲 | 向上 |
| 愿人都尊你的名为圣 | 上帝的荣耀 | 向上 |
| 愿你的国降临 | 上帝的治理 | 向上 |
| 我们日用的饮食 | 每日需求 | 向自己 |
| 免我们的债 | 赦免 | 向自己 |
| 不叫我们遇见试探 | 保护 | 向自己 |

---

## 默观祷告 (Contemplation)

> 不是「说很多话」——是「安静在上帝面前」。

| 传统 | 内容 |
|------|------|
| **归心祷告** (Centering Prayer) | 选一个「圣词」——当思绪飘走时——温柔地回到圣词 |
| **耶稣祷文** | 「主耶稣基督，上帝的儿子，怜悯我罪人。」——呼吸的节奏 |
| **属灵阅读** (Lectio Divina) | 读→默想→祷告→默观——四步进入上帝的同在 |

---

## 祷告的挣扎

| 问题 | 回应 |
|------|------|
| 「我祷告时总是分心」 | 分心是正常的——察觉分心→温柔地回来——这就是操练 |
| 「上帝不回答」 | 沉默不等于不在——「你在哪里？」——有时上帝通过沉默塑造信心 |
| 「我不会祷告」 | 诗篇是祷告的学校——用诗篇祷告——让别人写的祷词替你说话 |
