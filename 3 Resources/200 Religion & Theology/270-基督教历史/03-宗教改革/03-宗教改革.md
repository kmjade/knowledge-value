# 03-宗教改革 (1500-1650)

> 1517年10月31日——马丁·路德在维滕堡教堂门上贴出《九十五条论纲》——欧洲被撕裂——西方基督教永远改变了。

## 宗教改革的先声

| 人物 | 贡献 |
|------|------|
| 威克里夫 (14世纪) | 英文圣经翻译——「改教的晨星」 |
| 胡斯 (15世纪) | 波希米亚改教家——被烧死——「你们今天烧死一只鹅——一百年后会有一只天鹅」 |

## 路德

| 事件 | 内容 |
|------|------|
| 《九十五条》(1517) | 抗议赎罪券——不是计划革命——是学术讨论 |
| 沃尔姆斯会议 (1521) | 「这是我的立场——我不能不这样做」——被逐出教会 |
| **唯独圣经·唯独恩典·唯独信心** | 宗教改革的三个唯独 |

## 加尔文

| 著作 | 《基督教要义》(1536) ——系统神学的奠基 |
|------|------|
| 日内瓦 | 神权共和国——教会的改革模式 |
| 预定论 | 救赎完全出于上帝的主权 |

## 激进改革

| 团体 | 特征 |
|------|------|
| 重洗派 | 信徒洗礼——政教分离——门诺派的祖先 |
| 胡特尔派 | 财产共有 |

## 天主教的反宗教改革

| 事件 | 内容 |
|------|------|
| 天特会议 (1545-1563) | 回应新教——澄清天主教教义——改革教会弊端 |
| 耶稣会 (1540) | 依纳爵创立——天主教改革的主力 |

## 宗教战争的后果

| 合约 | 内容 |
|------|------|
| 奥格斯堡和约 (1555) | 「教随国定」——诸侯决定领地宗教 |
| 三十年战争 (1618-1648) | 欧洲最血腥的内战之一——威斯特伐利亚和约→宗教宽容 |
