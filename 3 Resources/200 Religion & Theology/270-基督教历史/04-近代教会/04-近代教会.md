# 04-近代教会 (1650-1900)

> 启蒙运动挑战了基督教的知识权威，但正是在这个「怀疑的世纪」，教会经历了最大的复兴和全球扩张。

## 启蒙运动的挑战

| 挑战 | 内容 |
|------|------|
| **理性主义** | 康德的「纯粹理性界限」——启示被理性审判 |
| **科学** | 达尔文 (1859) ——《物种起源》——创世记的字面解读被动摇 |
| **历史批判** | 圣经像其他古代文本一样被「批判」 |

## 大觉醒运动

| 浪潮 | 人物 |
|------|------|
| **第一次** (1730s-40s) | 爱德华兹·怀特菲尔德·卫斯理 |
| **第二次** (1790s-1840s) | 芬尼——帐篷聚会——美国边疆的信仰大爆发 |

## 宣教大世纪 (19世纪)

| 人物 | 贡献 |
|------|------|
| 威廉·克理 (1792) | 「现代宣教之父」——去印度 |
| 戴德生 (1865) | 中国内地会——不募款——只祷告 |
| 李文斯顿 | 非洲——探险家+宣教士+废奴主义者 |

## 自由派 vs 基要派

| 立场 | 核心 |
|------|------|
| **自由派** | 接受历史批判——强调「登山宝训」的伦理——淡化神迹 |
| **基要派** | 《基要真理》(1910-15) ——五要点——圣经无误·童贞生子·代赎·复活·再来 |
