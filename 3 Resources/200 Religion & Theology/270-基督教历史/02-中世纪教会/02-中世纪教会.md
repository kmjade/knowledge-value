# 02-中世纪教会 (500-1500)

> 「黑暗时代」——但教会在这一千年里建造了欧洲文明：修道院保存了古典文本，大学从中诞生，经院哲学为现代科学铺了路。但十字军和异端裁判所也是这一时代的阴暗面。

## 修道主义

| 人物 | 贡献 |
|------|------|
| 本笃 (480-547) | 《圣本笃会规》——祈祷与工作——西方修道的奠基 |
| 爱尔兰修士 | 在欧洲大陆建立修道院——「文明的使者」 |

## 教皇制的兴起

| 人物 | 事件 |
|------|------|
| 利奥一世 (5世纪) | 确立教皇权威——「彼得继承者」 |
| 格里高利七世 (11世纪) | 与亨利四世的叙任权斗争——「卡诺莎之辱」 |
| 英诺森三世 (13世纪) | 教皇权力的巅峰——第四次拉特兰会议 |

## 东西分裂 (1054)

| 原因 | 和子句争议 + 文化差异 + 政治对立 |
|------|------|
| **结果** | 基督教永久分裂为西方 (天主教) 和东方 (东正教) ——至今未合一 |

## 十字军东征 (1095-1291)

| 动机 | 收复圣地·宗教热情·政治野心——混合 |
|------|------|
| 遗产 | 暴力历史——基督教-伊斯兰关系的创伤至今未愈 |

## 经院哲学

| 人物 | 贡献 |
|------|------|
| 安瑟伦 | 「信仰寻求理解」——本体论论证 |
| 阿奎那 (1225-1274) | 《神学大全》——亚里士多德 + 基督教——经院哲学的巅峰 |
