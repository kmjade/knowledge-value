# 09-当代趋势

> 21世纪的教会面对一个「后基督教」的西方、爆炸性增长的南半球、和数字化带来的全新挑战和机遇。

## 后基督教世界

| 现象 | 内容 |
|------|------|
| **欧洲** | 教堂变成博物馆、酒店、清真寺——「基督教是祖父的信仰」 |
| **美国** | 「无宗教者」(Nones) 快速增长——但福音派仍是重要的政治力量 |

## 新兴教会 (Emerging Church)

| 特征 | 内容 |
|------|------|
| 后现代 | 对话取代独白——讲故事取代系统讲道 |
| 怀疑 | 欢迎怀疑者——不是「把怀疑压下去」 |
| 实践 | 重视社会行动——不是只关注「死后上天堂」 |

## 巨型教会 (Megachurch)

| 特征 | 每周出席 2000+ 人——专业制作——如 Hillsong, Willow Creek |
|------|------|
| 批评 | 「宗教消费主义」——缺少深度团契 |

## 性别与性

| 议题 | 张力 |
|------|------|
| 女性按立 | 许多宗派已经按立——保守宗派反对 |
| 同性恋 | 教会在这个议题上严重分裂——可能是21世纪最大的分裂原因 |

## 数字教会

| 现象 | 内容 |
|------|------|
| 在线聚会 | COVID 加速——「虚拟圣餐」——神学争议 |
| AI | ChatGPT 写讲章？——效率 vs 呼召 |
