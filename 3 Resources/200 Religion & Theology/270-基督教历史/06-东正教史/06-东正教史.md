# 06-东正教史

> 1054 年大分裂之后——东方教会走了自己的路。君士坦丁堡陷落 (1453) 后——「第三罗马」莫斯科成了东正教的中心。

## 拜占庭帝国 (330-1453)

| 特征 | 内容 |
|------|------|
| **政教合一** | 皇帝是教会的保护者——但不是教皇式的领袖——「交响乐」(symphonia) |
| **七次大公会议** (325-787) | 所有东方教会承认的——之后西方还有更多——东方不承认 |
| **圣像争辩** (726-843) | 圣像破坏 vs 圣像敬礼——最终恢复——「大斋期第一主日」是「正教得胜主日」 |

## 君士坦丁堡陷落 (1453)

| 后果 | 希腊正教会在奥斯曼帝国下存活——但失去了政治权力 |
|------|------|

## 俄罗斯正教会

| 事件 | 内容 |
|------|------|
| 基辅罗斯受洗 (988) | 弗拉基米尔大公——选择了东正教——「希腊人的美让我们无法抗拒」 |
| 莫斯科——第三罗马 | 「两个罗马已陷——第三罗马屹立——第四罗马不会有了」 |
| 苏维埃时期 | 迫害——教堂被毁——神职人员被处决——地下教会 |

## 东正教的特征

| 特征 | 内容 |
|------|------|
| 圣礼 | 礼仪 (Divine Liturgy) 是核心——不是讲道 |
| 圣像 | 窗户朝向天国——不是「偶像」 |
| 神化 (Theosis) | 救赎不只是「免罪」——是**成为像上帝**——「上帝成为人——为要让人成为神」(亚他那修) |
| 教会的分裂 | 君士坦丁堡和莫斯科——2018年因乌克兰问题正式分裂 |
