# 07-全球基督教

> 20世纪的「巨大反转」：1900年，80%的基督徒在欧洲和北美。今天，超过60%在非洲、亚洲和拉丁美洲。基督教的「重心」已经南移。

## 南半球的崛起

| 地区 | 特征 |
|------|------|
| **非洲** | 1900年：900万信徒 → 今天：6亿+ ——五旬宗爆炸性增长 |
| **拉丁美洲** | 从天主教单一 → 福音派和五旬宗快速增长 |
| **亚洲** | 韩国——世界上最大的教会 (汝矣岛纯福音教会) ——中国地下教会 |

## 后殖民基督教

| 问题 | 宣教是「西方的宗教殖民」吗？ |
|------|------|
| 回应 | 今天非洲和亚洲的基督教已经不是「西方」的——它被本土文化**处境化** |

## 全球的神学新声音

| 神学 | 地区 |
|------|------|
| 解放神学 | 拉丁美洲 |
| 黑人神学 | 非洲和非洲裔 |
| 故事神学 (Dalit) | 印度——最低种姓的声音 |
| Minjung 神学 | 韩国——被压迫民众 |
