# 08-中国教会史

> 基督教四次传入中国——前三次都消亡了。第四次至今仍在。

## 四次传入

### 第一次：景教 (635-845)

| 事件 | 内容 |
|------|------|
| 阿罗本 (635) | 从波斯来——唐太宗接见——「大秦景教流行中国碑」 |
| 消亡 | 唐武宗灭佛 (845) ——景教被波及——在中国消亡 |

### 第二次：也里可温 (13-14世纪)

| 元代 | 蒙古人带来的基督教——随元朝覆亡而消失 |

### 第三次：明末清初 (1580-1724)

| 人物 | 贡献 |
|------|------|
| 利玛窦 (1583) | 耶稣会士——「合儒补儒」——穿儒服——用儒家术语传播基督教 |
| 礼仪之争 | 祭祖和祭孔是礼仪还是偶像崇拜？——教皇禁止——康熙大怒——禁教 |

### 第四次：近代至今 (1807-)

| 人物 | 贡献 |
|------|------|
| 马礼逊 (1807) | 第一位来华新教宣教士——翻译圣经——编纂字典 |
| 戴德生 (1865) | 中国内地会——跨宗派——「信心差会」 |
| 义和团 (1900) | 数万基督徒和宣教士被杀 |

## 20世纪中国教会

| 时期 | 内容 |
|------|------|
| 1920s-40s | 本色化运动——脱离西方宗派——建立中国基督徒自己领导的教会 |
| 1949后 | 三自爱国运动——与外国差会断绝关系 |
| 文革 (1966-76) | 所有宗教活动被禁——地下教会坚持 |
| 1980s后 | 教会恢复——惊人增长——估计 4000万-1亿+ 基督徒 |

## 当代：家庭教会 vs 三自教会

| 家庭教会 | 不向政府登记——独立的聚会点 |
|------|------|
| 三自教会 | 在政府登记的教会——「自治·自养·自传」 |
| 张力 | 两个群体之间复杂的张力——不是简单的「真 vs 假」 |
