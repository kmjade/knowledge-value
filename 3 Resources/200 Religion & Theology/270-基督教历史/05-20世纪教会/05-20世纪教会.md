# 05-20世纪教会

> 两次世界大战、纳粹德国的教会挣扎、普世合一运动、五旬宗全球爆炸、梵二大公会议——20世纪是教会史上最动荡、也是最有活力的一百年。

## 世界大战与教会

| 事件 | 内容 |
|------|------|
| 纳粹德国 | 德国教会分裂——「德意志基督徒」支持希特勒——「认信教会」(巴特·潘霍华) 抵抗 |
| 潘霍华 | 《追随基督》——参与刺杀希特勒密谋——1945年被处决 |
| 二战后 | 「上帝之死」神学——奥斯维辛后还能说上帝是全能全善的吗？ |

## 普世合一运动

| 事件 | 内容 |
|------|------|
| 世界基督教协进会 (WCC, 1948) | 新教+东正教的普世合一平台 |
| 限制 | 天主教未加入——但以观察员参与——福音派批评 WCC 太自由 |

## 梵二大公会议 (1962-1965)

| 改革 | 内容 |
|------|------|
| 礼仪 | 用本地语言——不是拉丁文 |
| 圣经 | 鼓励平信徒读经 |
| 其他宗教 | 承认其他宗教也含有「真理的光辉」 |

## 五旬宗与灵恩运动

| 事件 | 内容 |
|------|------|
| 阿苏撒街 (1906) | 洛杉矶——五旬宗全球运动的起点 |
| 灵恩运动 (1960s-) | 五旬宗的经验进入主流宗派——天主教灵恩复兴 |
| 今天 | 五旬宗/灵恩派约6.5亿人——基督教第二大群体 |

## 福音派的兴起

| 人物 | 贡献 |
|------|------|
| 葛培理 (1918-2018) | 全球布道大会——超过2亿人听过他讲道 |
| 斯托得·巴刻 | 福音派学术复兴 |
