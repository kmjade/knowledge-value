# 01-早期教会 (30-500)

> 从五旬节到奥古斯丁——五百年间，基督教从耶路撒冷的一个小教派变成了罗马帝国的国教。

## 使徒时代 (30-100)

| 事件 | 内容 |
|------|------|
| 五旬节 (30) | 圣灵降临——教会诞生——三千人受洗 |
| 保罗的宣教旅程 | 三次旅行布道——小亚细亚→希腊→罗马 |
| 耶路撒冷会议 (49) | 外邦人不需要受割礼——决定性的转折 |
| 尼禄迫害 (64) | 彼得和保罗殉道 |
| 耶路撒冷陷落 (70) | 圣殿被毁——犹太教和基督教彻底分道 |

## 迫害时代 (100-313)

| 迫害 | 最严重的是戴克里先 (303-311) ——「大迫害」 |
|------|------|
| 殉道者 | 伊格纳修·坡旅甲·游斯丁·佩尔佩图阿 |
| 回应 | 殉道的血是「种子」(德尔图良) ——迫害越强——教会越增长 |

## 君士坦丁转折 (313)

| 米兰敕令 (313) | 基督教合法化——不是国教——但不再被迫害 |
|------|------|
| 尼西亚会议 (325) | 第一次大公会议——亚流争议——《尼西亚信经》 |
| 狄奥多西 (380) | 基督教成为罗马帝国唯一合法宗教——历史的反转 |

## 关键教父

| 教父 | 贡献 |
|------|------|
| 奥古斯丁 (354-430) | 《忏悔录》《上帝之城》——西方神学之父 |
| 亚他那修 | 对抗亚流——捍卫基督的神性 |
| 耶柔米 | 拉丁文武加大译本——西方教会的圣经 |
