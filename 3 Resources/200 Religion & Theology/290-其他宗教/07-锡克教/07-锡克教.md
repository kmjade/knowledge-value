# 07-锡克教

> 约 3000 万信徒——主要在北印度旁遮普。Guru Nanak 在 16 世纪创立——融合了伊斯兰教的一神论和印度教的轮回观——但超越了二者。

## 核心信念

| 信念 | 一个上帝——无形——真理——创造者——「Ik Onkar」(一个创造者) |
|------|------|
| 轮回 | 接受——目标是打破轮回——与上帝合一 |
| **反对** | 种姓制度·偶像崇拜·祭司阶层·苦行主义——人人平等 |
| **服务** (Seva) | 无私服务——在Gurdwara扫地·在Langar做饭——这是信仰的表达 |

## 十古鲁谱系

| 古鲁 | 在位 | 贡献 |
|------|:---:|------|
| 1. 那纳克 (Nanak) | 1469-1539 | 创立者 |
| 2. 安格德 (Angad) | 1539-1552 | 创古鲁穆克文字(Gurmukhi) |
| 3. 阿玛尔·达斯 | 1552-1574 | 制度化Langar |
| 4. 拉姆·达斯 | 1574-1581 | 建阿姆利则圣城 |
| 5. 阿尔琼 (Arjan) | 1581-1606 | 编《阿迪·格兰特》——建金庙——第一位殉道古鲁 |
| 6. 哈尔·戈宾德 | 1606-1644 | 武装化——「Miri-Piri」(世俗-灵性双剑) |
| 10. 戈宾德·辛格 | 1675-1708 | 创Khalsa(1699)——古鲁传承终止于经典 |
| **古鲁·格兰特·沙希卜** | 1708- | 「活的古鲁」——不仅是经书——是第十一位永恒的古鲁 |

## 《古鲁·格兰特·沙希卜》

| 特征 | 内容 |
|------|------|
| 长度 | 1430页——用古鲁穆克文书写 |
| 内容 | 诗歌·赞美诗——不仅来自锡克古鲁——还包括印度教和伊斯兰教圣徒的作品 |
| 地位 | 一切决定由它裁决——每天随机翻开一页求指引(Hukamnama) |

## 五 K (Khalsa)

| 符号 | 含义 |
|------|------|
| Kesh | 不剪发——接受上帝创造的样子 |
| Kangha | 木梳——整洁 |
| Kara | 铁手镯——与上帝的联结 |
| Kachera | 短裤——自制 |
| Kirpan | 短剑——保护弱者 |

## 金庙 (Harmandir Sahib)

| 特征 | 内容 |
|------|------|
| 位置 | 阿姆利则——池中央——四个入口(向所有方向/所有种姓开放) |
| Langar | 每天免费供应数万份餐食——任何人——不分宗教·种姓·性别 |
| 1984年事件 | 蓝星行动——印度军队攻入金庙——锡克教徒深刻的集体创伤 |
