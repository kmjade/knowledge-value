# 07-锡克教

> 约 3000 万信徒——主要在北印度旁遮普。Guru Nanak 在 16 世纪创立——融合了伊斯兰教的一神论和印度教的轮回观——但超越了二者。

## 核心

| 信念 | 一个上帝——无形——真理——创造者 |
|------|------|
| 轮回 | 接受——目标是打破轮回——与上帝合一 |
| **反对** | 种姓制度·偶像崇拜·祭司阶层——人人平等 |

## 五 K (Khalsa)

| 符号 | 含义 |
|------|------|
| Kesh | 不剪发——接受上帝创造的样子 |
| Kangha | 木梳——整洁 |
| Kara | 铁手镯——与上帝的联结 |
| Kachera | 短裤——自制 |
| Kirpan | 短剑——保护弱者 |

## 实践

| 实践 | 内容 |
|------|------|
| 金庙 (阿姆利则) | 最神圣的圣地 |
| Langar | 免费食堂——任何人——不分宗教·种姓——都可以来吃 |
| 经典 | 《古鲁·格兰特·沙希卜》——不仅是经书——是「活的古鲁」 |
