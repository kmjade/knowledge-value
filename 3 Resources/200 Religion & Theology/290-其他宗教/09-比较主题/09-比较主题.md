# 09-比较主题

## 终极实在

| 宗教 | 终极实在 |
|------|---------|
| 犹太教 | 独一上帝——超越——创造主 |
| 伊斯兰教 | 安拉——绝对独一——无形象 |
| 印度教 | 梵——非人格终极——但也有人格化的神 |
| 佛教 | 涅槃——不是「神」——是熄灭贪嗔痴的状态 |

## 来世

| 宗教 | 来世观 |
|------|--------|
| 犹太教 | 模糊——有些流派有复活观 |
| 伊斯兰教 | 天堂(Jannah)·火狱(Jahannam) ——具体描述 |
| 印度教 | 轮回——直到解脱 |
| 佛教 | 轮回——但不是「灵魂」在转——是业的延续 |

## 伦理基础

| 宗教 | 黄金法则 |
|------|---------|
| 犹太教 | 「爱人如己」(利 19:18) |
| 伊斯兰教 | 「你希望别人怎样待你——你就怎样待人」(圣训) |
| 印度教 | 「不要对别人做那些如果别人对你做你会感到痛苦的事」 |
| 佛教 | 「以己度他情」(法句经) |

## 经典的地位

| 宗教 | 经典观 |
|------|--------|
| 伊斯兰教 | 逐字启示——不可翻译 |
| 犹太教 | 成文+口传——需要拉比诠释 |
| 印度教 | 多经典——吠陀最权威——但有哲学诠释空间 |
| 佛教 | 多种经典——不同宗派用不同版本 |
