# 06-道教(宗教维度)

> 道家哲学在 DDC 180 已深入覆盖(14文件)。本章从**宗教**维度补充：科仪·符箓·神仙崇拜·民间实践。

## 宗教维度 vs 哲学维度

| 哲学 (DDC 180) | 宗教 (DDC 290) |
|:---|:---|
| 道·德·无为·自然 | 科仪·斋醮·符箓·丹药 |
| 老子的「道可道非常道」 | 太上老君信仰 |

## 历史脉络

| 时期 | 发展 |
|------|------|
| **东汉** (2世纪) | 张道陵创天师道(五斗米道)——四川鹤鸣山——道教制度化起点 |
| **魏晋南北朝** | 上清派·灵宝派兴起——葛洪《抱朴子》——外丹术高峰 |
| **唐代** | 道教为国教——老子被尊为「太上玄元皇帝」——内丹兴起 |
| **宋金元** | 王重阳创全真道(1167)——三教合一——修道院制度 |
| **明清** | 全真·正一两大派——民间道教融合——扶乩·关帝崇拜 |
| **现代** | 1949后大陆萎缩——台湾·香港保留完整——近年大陆渐复 |

## 两大流派

| 流派 | 特征 | 分布 |
|------|------|------|
| **正一道** | 天师世袭——符箓·斋醮——可以结婚——住家 | 南方·台湾 |
| **全真道** | 出家修道——三教合一——内丹为主——道观修行 | 北方·全真十方丛林 |

## 神仙体系

| 层级 | 神 | 职司 |
|:---:|------|------|
| 最高 | 三清 | 元始天尊·灵宝天尊·道德天尊 |
| 天界 | 玉皇大帝 | 天界主宰 |
| 星宿 | 紫微·北斗·南斗 | 掌管命运 |
| 八仙 | 吕洞宾·钟离权·铁拐李等 | 道教中最受欢迎的民间神仙 |
| 民间 | 关帝(关羽)·妈祖·城隍·土地 | 被道教吸纳的民间信仰 |

## 实践

| 实践 | 内容 |
|------|------|
| **科仪/斋醮** | 道士主持的集体仪式——祈禳·超度——敲击法器·步罡踏斗 |
| **符箓** | 道士画的符——驱邪·治病·护身——「符」是写给鬼神的公文 |
| **风水** | 阴阳五行——择地·择时——影响建筑和墓葬 |
| **外丹** | 炼制丹药——追求长生不老——唐代多位皇帝死于丹药中毒 |
| **内丹** | 人体内部的炼丹——精气神转化——与哲学维度交叉 |

## 道教与民间信仰的边界

| 道教 | 民间信仰 |
|------|----------|
| 有经典·有师承·有教团 | 无系统·地方性·口头传承 |
| 道士·道观·度牒 | 神婆·乩童·庙祝 |
| 实际上两者高度混融——「中国的道教」很大程度上就是「制度化的民间信仰」 |
