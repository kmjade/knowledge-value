# 03-伊斯兰教

> 「伊斯兰」= 顺服 (上帝)。穆斯林 = 顺服者。全球 ~19 亿——增长最快的宗教之一。

## 基本数据

| 先知 | 穆罕默德 (~570-632) ——最后的先知——「封印」 |
|------|------|
| 经典 | 古兰经——阿拉伯语——逐字启示——不可翻译(只能「解释」) |
| 降示 | 610 CE 在希拉山洞——哲卜利勒天使传述——历时23年 |

## 古兰经的结构

| 特征 | 内容 |
|------|------|
| 苏拉 (章) | 114章——麦加章(早期·短·末日警告) + 麦地那章(后期·长·律法) |
| 编排 | 大致从长到短——不是时间顺序 |
| 语言 | 古典阿拉伯语——押韵散文(saj') ——「不可模仿性」(i'jaz) |

## 五功

| 功 | 内容 |
|----|------|
| **念** (Shahada) | 「万物非主——唯有真主——穆罕默德是主的使者」 |
| **礼** (Salat) | 每日五次礼拜——朝向麦加 |
| **斋** (Sawm) | 斋月 (Ramadan) ——日出到日落禁食 |
| **课** (Zakat) | 施舍——2.5% 财产 |
| **朝** (Hajj) | 一生至少一次去麦加朝圣 |

## 沙里亚法的来源

| 来源 | 说明 |
|------|------|
| **古兰经** | 第一来源——约500节法律经文 |
| **圣训** (Hadith) | 穆罕默德的言行录——六大圣训集 |
| **公议** (Ijma) | 学者的共识 |
| **类比** (Qiyas) | 从已知推断未知 |

## 逊尼 vs 什叶

| 分支 | 比例 | 分歧 | 主要分布 |
|------|:---:|------|------|
| 逊尼 | 85% | 穆罕默德没有指定继承人——由社群选举 | 中东·北非·南亚·东南亚 |
| 什叶 | 15% | 穆罕默德指定了堂弟阿里——血统继承 | 伊朗(90%+)·伊拉克·黎巴嫩 |

## 苏菲主义

| 概念 | 内容 |
|------|------|
| 核心 | 伊斯兰的神秘维度——不是「说」信仰——是「尝」信仰 |
| 道团 (Tariqa) | 导师-弟子传承链——冥想技巧——赞念(dhikr) |
| 著名苏菲 | 鲁米——13世纪波斯诗人——《玛斯纳维》 |
| 争议 | 圣徒崇拜·音乐·舞蹈——部分正统派视为异端 |

## 伊斯兰与现代性

| 议题 | 张力 |
|------|------|
| **世俗化** | 土耳其模式(凯末尔主义) vs 伊朗模式(教法学家治国) |
| **女性** | 头巾(hijab)是压迫还是身份？——经文诠释的多样性 |
| **吉哈德** | 字面义「奋斗」——大吉哈德(内心) vs 小吉哈德(自卫) |

## 伊斯兰教与基督教

| 共同 | 同一个上帝 (Allah) ——亚伯拉罕·摩西·耶稣都是先知 |
|------|------|
| 分歧 | 耶稣不是上帝的儿子——没有被钉十字架——古兰经改正了圣经的「篡改」 |
