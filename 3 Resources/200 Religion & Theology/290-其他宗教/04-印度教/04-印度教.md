# 04-印度教

> 印度教不是「一个宗教」——是一个包含无数神祇、哲学体系、修行法门的宗教家族。没有单一创始人、没有单一经典、没有单一信条。

## 核心概念

| 概念 | 内容 |
|------|------|
| **梵** (Brahman) | 终极实在——宇宙的本源——「你是那个」(Tat tvam asi) |
| **阿特曼** (Atman) | 个体的真实自我——最终与梵合一 |
| **业** (Karma) | 行动的因果法则——善有善报 |
| **轮回** (Samsara) | 生-死-再生的循环 |
| **法** (Dharma) | 宇宙秩序·个体责任——「做你该做的」 |
| **解脱** (Moksha) | 从轮回中解脱——与梵合一 |

## 三大神 (Trimurti)

| 神 | 角色 | 坐骑 | 配偶 |
|----|------|------|------|
| 梵天 (Brahma) | 创造者 | 天鹅 | 萨拉斯瓦蒂(智慧) |
| 毗湿奴 (Vishnu) | 维护者——十大化身(罗摩·黑天·佛陀) | 金翅鸟 | 拉克什米(财富) |
| 湿婆 (Shiva) | 毁灭和转化者——舞蹈之神(Nataraja) | 白牛 | 帕尔瓦蒂 |

## 瑜伽四道 (Marga)

| 道 | 方式 | 适合 |
|----|------|------|
| **业瑜伽** (Karma) | 无私行动——不执着结果 | 行动者 |
| **智瑜伽** (Jnana) | 哲学思辨——追问「我是谁」 | 思辨者 |
| **信瑜伽** (Bhakti) | 虔诚奉献——爱神——唱颂 | 情感者 |
| **王瑜伽** (Raja) | 冥想·调息·八支瑜伽——帕坦伽利体系 | 修行者 |

## 经典

| 经典 | 内容 | 时期 |
|------|------|------|
| 吠陀 (Veda) | 最古老的——赞美诗·祭祀 | ~1500-500 BCE |
| 奥义书 (Upanishads) | 哲学——梵我合一 | ~800-200 BCE |
| 薄伽梵歌 (Bhagavad Gita) | 最流行的——阿周那的困惑·黑天的教导 | ~200 BCE |
| 往世书 (Puranas) | 神话·宇宙论——18部主要 | ~300-1000 CE |

## 吠檀多三派

| 学派 | 核心 | 创始人 |
|------|------|------|
| **不二论** (Advaita) | 梵与阿特曼完全同一——世界是幻(maya) | 商羯罗 |
| **殊胜不二论** | 梵与个体不一不异——但有个体性 | 罗摩奴阇 |
| **二元论** (Dvaita) | 梵与个体完全不同——神是主·我是仆 | 摩陀婆 |

## 种姓制度

| 四种姓 | 婆罗门(祭司)·刹帝利(武士)·吠舍(商人)·首陀罗(劳工) |
|------|------|
| 达利特 | 「不可接触者」——在四种姓之外 |
| 现代 | 法律上废除——社会现实中仍存在——保留政策(配额)引发争议 |

## 当代印度教

| 议题 | 内容 |
|------|------|
| **印度教民族主义** | Hindutva——印度教国家认同——莫迪与BJP |
| **全球化** | 瑜伽国际化——ISKCON(奎师那知觉)——灵性导师(guru)现象 |
| **与伊斯兰教关系** | 印巴分治的创伤——阿约提亚清真寺-神庙争议 |
