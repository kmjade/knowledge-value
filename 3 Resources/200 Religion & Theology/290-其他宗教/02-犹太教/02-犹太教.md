# 02-犹太教

> 最小的亚伯拉罕宗教——但它的孩子（基督教和伊斯兰教）覆盖了世界一半以上的人口。

## 基本数据

| 信徒 | ~1500万——以色列·美国 |
|------|------|
| 经典 | 塔纳赫 (Tanakh) + 塔木德 (Talmud) ——「口传妥拉」 |
| 核心 | 「以色列啊，你要听！耶和华我们上帝是独一的主」(申 6:4) |

## 经典的层级

| 层级 | 内容 | 地位 |
|------|------|:---:|
| **成文妥拉** | 摩西五经 (Torah) | 最高权威 |
| **先知书** (Nevi'im) | 约书亚→列王纪→以赛亚等 | 次权威 |
| **圣卷** (Ketuvim) | 诗篇·箴言·约伯记等 | 第三层 |
| **口传妥拉 → 塔木德** | 米示拿(Mishnah) + 革马拉(Gemara) — 拉比讨论汇编 | 实践权威 |

## 分支

| 分支 | 特征 | 全球占比 |
|------|------|:---:|
| 正统派 | 严格遵守律法——安息日·饮食——不混合 | ~20% |
| 现代正统派 | 守律法+参与现代社会——「妥拉与科学」 | 增长中 |
| 保守派 | 保持传统——但适应现代 | ~30% |
| 改革派 | 强调伦理——放松礼仪——男女平等 | ~35% |
| 重建派 | 犹太教是不断进化的文明——圣经是人写的 | 小众 |

## 实践

| 实践 | 内容 |
|------|------|
| 安息日 | 周五日落→周六日落——不工作——家庭聚餐 |
| 逾越节 | 纪念出埃及——Seder 晚餐——无酵饼(matzo) |
| 赎罪日 (Yom Kippur) | 最庄严——25小时禁食——忏悔 |
| 住棚节 | 搭建临时棚——纪念旷野漂泊 |
| 光明节 | 纪念马加比起义——点烛台(hanukkiah)八天 |
| 饮食律法 | Kashrut——不混奶肉——只吃分蹄反刍动物 |

## 卡巴拉 (Kabbalah)

| 概念 | 内容 |
|------|------|
| 生命树 (Sefirot) | 10个属性——上帝通过它们创造和维持世界 |
| 光辉之书 (Zohar) | 13世纪——卡巴拉最重要文本——传统归摩西·德·莱昂 |
| 当代影响 | 麦当娜带红卡巴拉手绳——学术界争议「是古代传统还是中世纪发明」 |

## 反犹主义与锡安主义

| 时期 | 事件 |
|------|------|
| 中世纪 | 十字军屠杀·西班牙驱逐(1492)·血祭诽谤 |
| 现代 | 德雷福斯事件(1894)→赫茨尔发起锡安主义 |
| 大屠杀 | 纳粹屠杀600万犹太人——改变了犹太教的地理和神学 |
| 以色列建国 | 1948——犹太教从「流散宗教」变成「有国土的宗教」 |

## 犹太教与基督教

| 共同 | 同一本旧约——同一个上帝——十诫 |
|------|------|
| 分歧 | 耶稣是不是弥赛亚？——这是根本分歧 |
| 对话 | 二战后的「犹太-基督教对话」——教宗若望保禄二世称犹太人为「长兄」 |
