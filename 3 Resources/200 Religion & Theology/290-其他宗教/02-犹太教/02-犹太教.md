# 02-犹太教

> 最小的亚伯拉罕宗教——但它的孩子（基督教和伊斯兰教）覆盖了世界一半以上的人口。

## 基本数据

| 信徒 | ~1500万——以色列·美国 |
|------|------|
| 经典 | 塔纳赫 (Tanakh) + 塔木德 (Talmud) ——「口传妥拉」 |
| 核心 | 「以色列啊，你要听！耶和华我们上帝是独一的主」(申 6:4) |

## 分支

| 分支 | 特征 |
|------|------|
| 正统派 | 严格遵守律法——安息日·饮食——不混合 |
| 保守派 | 保持传统——但适应现代 |
| 改革派 | 强调伦理——放松礼仪——男女平等 |

## 实践

| 实践 | 内容 |
|------|------|
| 安息日 | 周五日落→周六日落——不工作——家庭聚餐 |
| 逾越节 | 纪念出埃及——Seder 晚餐 |
| 赎罪日 (Yom Kippur) | 最庄严——禁食——忏悔 |

## 犹太教与基督教

| 共同 | 同一本旧约——同一个上帝——十诫 |
|------|------|
| 分歧 | 耶稣是不是弥赛亚？——这是根本分歧 |
