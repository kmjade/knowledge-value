# 05-佛教(宗教维度)

> 佛学在 DDC 180 已从**哲学**维度深入覆盖(14文件)。本章从**宗教**维度补充：仪轨·宗派组织·民间佛教实践。

## 宗教维度 vs 哲学维度

| 哲学 (DDC 180) | 宗教 (DDC 290) |
|:---|:---|
| 四谛·空性·心性分析 | 寺庙·仪轨·朝圣·节日 |
| 禅宗的「不立文字」 | 信徒的烧香·磕头·求签 |

## 实践

| 实践 | 内容 |
|------|------|
| 三皈依 | 皈依佛·皈依法·皈依僧 |
| 供佛 | 花·香·灯·果——不是拜偶像——是表达敬意 |
| 朝圣 | 蓝毗尼(出生)·菩提伽耶(觉悟)·鹿野苑(初转)·拘尸那罗(涅槃) |
| 卫塞节 | 纪念佛陀诞生·觉悟·涅槃——南传最重要节日 |

## 佛教宗派的宗教形态

| 宗派 | 特征 |
|------|------|
| 上座部 | 僧侣托钵——在家众供养——「积累功德」 |
| 大乘 | 观音崇拜——净土念佛——「阿弥陀佛」 |
| 藏传 | 转经筒·风马旗·喇嘛·唐卡——最丰富的仪式传统 |
| 民间佛教 | 烧纸钱·拜观音求子·放生——与「经典佛教」差异很大 |
