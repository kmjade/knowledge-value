# 05-佛教(宗教维度)

> 佛学在 DDC 180 已从**哲学**维度深入覆盖(14文件)。本章从**宗教**维度补充：仪轨·宗派组织·民间佛教实践。

## 宗教维度 vs 哲学维度

| 哲学 (DDC 180) | 宗教 (DDC 290) |
|:---|:---|
| 四谛·空性·心性分析 | 寺庙·仪轨·朝圣·节日 |
| 禅宗的「不立文字」 | 信徒的烧香·磕头·求签 |

## 戒律与僧团 (Vinaya & Sangha)

| 概念 | 内容 |
|------|------|
| **比丘** (Bhikkhu) | 受具足戒的僧侣——227条戒律(上座部) |
| **比丘尼** | 女性僧侣——戒条更多——历史上传承中断——当代复兴争议 |
| **三衣** | 僧袍——不蓄财——靠乞食和供养生活 |
| **雨安居** (Vassa) | 雨季三月不出门——密集修行——结束时供僧衣(Kathina) |

## 三传佛教的宗教形态

| 传承 | 地区 | 宗教特征 |
|------|------|------|
| **上座部** | 斯里兰卡·泰·缅·柬·老 | 僧侣托钵——在家众供养——「积累功德」——短期出家 |
| **汉传** | 中·日·韩·越 | 素食·农禅·早晚课诵——最「制度化」的佛教 |
| **藏传** | 西藏·蒙古·不丹 | 转经筒·风马旗·喇嘛·唐卡——最丰富的仪式传统——活佛转世 |

## 佛教节日

| 节日 | 日期 | 内容 |
|------|------|------|
| **卫塞节** (Vesak) | 五月满月 | 纪念佛陀诞生·觉悟·涅槃——南传最重要 |
| **佛诞日** | 四月初八(汉传) | 浴佛——「天上天下，唯我独尊」 |
| **盂兰盆节** | 七月十五 | 超度祖先——源自目连救母 |
| **腊八节** | 腊月初八 | 佛陀成道日——喝腊八粥 |

## 民间佛教

| 实践 | 说明 |
|------|------|
| 烧香拜佛 | 求平安·求财运·求子——与「经典佛教」的涅槃追求差异大 |
| 放生 | 慈悲实践——但生态争议（放生=变相杀生） |
| 观音崇拜 | 悲悯女神——中国最受欢迎的佛教形象——「家家观世音」 |
| 寺庙经济 | 门票·功德箱·头香拍卖——商业化批评 |
