# 其他宗教 知识库总览 (DDC 290)

## 世界宗教地图

| 宗教 | 信徒约数 | 起源 | 核心 | 经典 | 类型 |
|------|:---:|------|------|------|:---:|
| 犹太教 | 1500万 | 中东 ~2000 BCE | 妥拉·盟约 | 塔纳赫+塔木德 | 一神论 |
| 伊斯兰教 | 19亿 | 阿拉伯 610 CE | 五功·古兰经 | 古兰经+圣训 | 一神论 |
| 印度教 | 12亿 | 印度 ~1500 BCE | 梵·业·法 | 吠陀+奥义书 | 多神/一元 |
| 佛教 | 5亿 | 印度 ~500 BCE | 四谛·八正道 | 三藏(依宗派异) | 非神论 |
| 锡克教 | 3000万 | 旁遮普 1500 CE | 一神·五K | 古鲁·格兰特·沙希卜 | 一神论 |
| 道教(宗教) | 数千万 | 中国 2世纪 CE | 道·修炼 | 道藏 | 多神 |

| 摩门教 | 1700万 | 美国 1830 | 现代先知 | 摩门经+圣经 | 一神(多元) |

## 比较宗教关键术语

| 术语 | 含义 |
|------|------|
| **一神论** (Monotheism) | 只有一位上帝——犹太教·基督教·伊斯兰教·锡克教 |
| **多神论** (Polytheism) | 多位神祇——印度教(表面多神·本质一元)·道教宗教 |
| **非神论** (Nontheism) | 终极实在不是人格神——佛教·耆那教 |
| **万有在神** (Panentheism) | 万有在神中——神既超越又内在——印度教梵·卡巴拉 |
| **泛灵论** (Animism) | 万物有灵——日本神道教·部落宗教 |
| **综摄** (Syncretism) | 混合不同宗教元素——高台教·民间宗教 |
