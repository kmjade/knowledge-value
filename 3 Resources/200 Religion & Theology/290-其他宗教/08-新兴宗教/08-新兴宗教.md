# 08-新兴宗教

> 「新兴宗教」——在过去200年内兴起的宗教运动。有些已成为全球性宗教，有些仍在小规模发展。

## 后期圣徒 (摩门教)

| 创立 | 约瑟·斯密 (1830) ——美国纽约——《摩门经》 |
|------|------|
| 经典 | 《摩门经》+《教义和圣约》+《无价珍珠》+ 圣经——四部标准经典 |
| 核心教义 | 现代先知——圣殿——家谱——「人类可以成为神」(exaltation) |
| 一夫多妻 | 1890年正式放弃——但原教旨主义分支仍在实践 |
| 约 1700 万 | 总部——盐湖城——全球增长最快的本土美国宗教 |

## 耶和华见证人

| 特征 | 不输血(基于使徒行传15:29)·不服兵役·逐户传道 |
|------|------|
| 教义 | 拒绝三位一体——耶稣是上帝长子但不是上帝——1914年基督隐形降临 |
| 守望台 | 最广泛发行的宗教杂志之一 |
| 约 850 万 | 全球——最知名的「敲门人」——240种语言 |

## 巴哈伊信仰

| 创立 | 巴哈欧拉 (1863) ——波斯 |
|------|------|
| 核心 | 上帝唯一·宗教同源·人类一体——渐进启示(manifestations of God) |
| 先知名单 | 亚伯拉罕·摩西·佛陀·耶稣·穆罕默德·巴布·巴哈欧拉——都是上帝的显现 |
| 全球治理 | 世界正义院(海法)——每五年选举——无神职人员 |
| 约 700 万 | 最成功的「新兴世界宗教」——分布最广(仅次于基督教) |

## 其他新兴宗教

| 团体 | 特征 | 信徒约数 |
|------|------|:---:|
| **山达基** | L. 罗恩·贺伯特(1954)——Dianetics——好莱坞明星——争议最大 | 数万 |
| **高台教** (Cao Dai) | 越南(1926)——融合佛·道·儒·基督——独特的圣殿——维克多·雨果是圣人 | 400万+ |
| **拉斯塔法里** | 牙买加——海尔·塞拉西一世是上帝化身——雷鬼音乐——dreadlocks | 100万 |
| **法轮功** | 1992年中国——气功+佛法——1999年被取缔——海外活跃 | 未确证 |

## "邪教" vs "新兴宗教"

| 问题 | 谁定义？ |
|------|----------|
| 学术术语 | NRM (New Religious Movement) ——中性描述 |
| 公众话语 | 「cult」= 脑控·封闭·危险——往往是「我的是宗教，你的是邪教」 |
| 张力 | 基督教早期被罗马视为「邪教」——时间改变合法性 |

## 新兴宗教的共同模式

| 模式 | 说明 |
|------|------|
| **魅力型领袖** | 创始人往往有非凡的个人魅力和启示声称 |
| **新经典** | 在已有圣典之外增加新的启示文本 |
| **千禧年/末世论** | 强调即将到来的世界变革 |
| **制度化挑战** | 第一代创始人死后如何传承——这是所有新兴宗教的关键考验 |
