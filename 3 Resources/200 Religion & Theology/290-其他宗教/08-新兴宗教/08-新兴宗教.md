# 08-新兴宗教

> 「新兴宗教」——在过去200年内兴起的宗教运动。有些已成为全球性宗教，有些仍在小规模发展。

## 后期圣徒 (摩门教)

| 创立 | 约瑟·斯密 (1830) ——美国——《摩门经》 |
|------|------|
| 特征 | 现代先知——圣殿——家谱——「人类可以成为神」 |
| 约 1700 万 | 总部——盐湖城 |

## 耶和华见证人

| 特征 | 不输血·不服兵役·逐户传道——拒绝三位一体 |
|------|------|
| 约 850 万 | 全球——最知名的「敲门人」 |

## 巴哈伊信仰

| 创立 | 巴哈欧拉 (1863) ——波斯 |
|------|------|
| 核心 | 所有宗教来自同一个上帝——人类合一——世界和平 |
| 约 700 万 | 最成功的「新兴世界宗教」 |

## 其他

| 团体 | 特征 |
|------|------|
| 山达基 | 争议——好莱坞明星——秘密教义 |
| 高台教 | 越南——融合佛·道·儒·基督——独特的圣殿 |
