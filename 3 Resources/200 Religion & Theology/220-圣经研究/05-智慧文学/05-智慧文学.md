# 05-智慧文学

> 智慧文学不问「上帝做了什么」(那是历史书)——也不问「上帝要什么」(那是律法书和先知书)——它问：「人该怎么活？」

## 诗篇 (Psalms)

### 诗篇的类型

| 类型 | 例子 | 特征 |
|------|------|------|
| **赞美诗** | 诗 8, 19, 104 | 赞美上帝的创造和作为 |
| **哀歌** | 诗 22, 42, 137 | 「我的上帝——你为什么离弃我？」——最多的一类 |
| **感恩诗** | 诗 30, 116 | 被拯救后的感谢 |
| **君王诗** | 诗 2, 72, 110 | 关于大卫王朝 |
| **智慧诗** | 诗 1, 119 | 律法 vs 恶人的道路 |

> 诗篇是**祷告词**——不是神学论文。它教人如何把一切情感——喜乐、愤怒、绝望、疑问——带到上帝面前。

## 约伯记 (Job)

> 义人为什么受苦？约伯的三个朋友说：「你一定犯了罪。」约伯说：「我没有——我要上帝回答我。」上帝最终回答了——但答案不是解释——是**旋风中的显现**。

| 部分 | 内容 |
|------|------|
| 序幕 (1-2) | 天上的对话——上帝允许撒但试炼约伯 |
| 对话 (3-31) | 三个朋友的循环——「你受苦因为你有罪」——约伯抗议 |
| 以利户 (32-37) | 年轻人插话——不完全对——但引入后面旋风 |
| 上帝的回答 (38-41) | 「我立大地根基的时候——你在哪里？」——上帝不解释——他显现 |
| 尾声 (42) | 约伯：「我从前风闻有你——现在亲眼看见你。」 |

> 约伯记的答案：不是「为什么受苦」——是「在苦难中，上帝与你同在。」

## 箴言 (Proverbs)

| 特征 | 内容 |
|------|------|
| **形式** | 短小格言——「敬畏耶和华是知识的开端」 |
| **原则 vs 应许** | 「教养孩童使他走当行的道……」——这是一般原则——不是绝对保证——约伯就是反例 |

---

## 传道书 (Ecclesiastes)

> 「虚空的虚空——凡事都是虚空。」——最「非正统」的圣经书卷。

| 主题 | 内容 |
|------|------|
| **人生的虚空** | 智慧、享乐、工作——最终都指向死亡——「日光之下并无新事」 |
| **结论** | 在虚空的人生中——享受当下、敬畏上帝——这是**人所能做的最好的事** |
