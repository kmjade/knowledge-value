# 08-保罗书信

> 保罗——原名扫罗——从迫害教会的人变成教会历史上最有影响力的传教士。他的 13 封书信奠定了基督教神学的基础。

## 保罗生平

| 阶段 | 内容 |
|------|------|
| 出身 | 大数人——罗马公民——法利赛人——迦玛列门下 |
| 归信 | 大马士革路上——「扫罗！扫罗！你为什么逼迫我？」(使 9) |
| 三次宣教旅程 | 小亚细亚→希腊——建立多个教会 |
| 殉道 | ~公元 64-67——罗马——被斩首 |

---

## 核心神学

### 因信称义

> 「人称义是因着信——不在乎遵行律法。」(罗 3:28)

| 概念 | 内容 |
|------|------|
| **问题** | 所有人——犹太人和外邦人——都犯了罪——都在上帝的审判之下 |
| **解决** | 耶稣在十字架上的死——代替罪人承受审判 |
| **领受** | 不是靠做善事——是靠「信」——信耶稣的代赎 |

### 恩典

> 「你们得救是本乎恩——也因着信——这不是出于自己——乃是上帝所赐的。」(弗 2:8)

| 概念 | 内容 |
|------|------|
| 恩典 = 不配得的恩惠 | 上帝不是因为人好才救人——是因为他**自己好** |

---

## 主要书信

| 书信 | 主题 |
|------|------|
| **罗马书** | 最系统的神学论述——因信称义·以色列的命运 |
| **哥林多前后** | 教会问题——分裂·道德·属灵恩赐·**爱的篇章**(林前 13) |
| **加拉太书** | 最激烈的——保罗捍卫「福音的自由」——反对要求外邦人受割礼 |
| **以弗所书** | 教会是基督的身体——「天上地下一切所有的——都在基督里同归于一」 |
| **腓立比书** | 喜乐——「我靠着那加给我力量的——凡事都能做」 |
| **腓利门书** | 最个人的——为一个逃跑的奴隶阿尼西母求情 |

---

## 保罗与耶稣

| 批评 | 保罗「发明」了基督教——耶稣宣扬天国的伦理——保罗把它变成了「信耶稣得救」的宗教 |
|------|------|
| 辩护 | 保罗的信息根植于早期教会的传统——「我当日所领受又传给你们的」 |
