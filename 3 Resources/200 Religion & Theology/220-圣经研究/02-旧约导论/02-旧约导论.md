# 02-旧约导论

> 犹太人称它为 Tanakh (תנ״ך): Torah (律法) + Nevi'im (先知) + Ketuvim (圣卷)。基督徒称它为旧约——与耶稣的「新约」相对。但不论怎么称呼——它是人类最古老的持续阅读的文本之一。

## 旧约的三部分

### Torah (律法书/五经)

| 卷 | 希伯来名 | 主题 |
|----|---------|------|
| 创世记 | Bereshit (起初) | 创造·堕落·洪水·先祖 |
| 出埃及记 | Shemot (名字) | 摩西·出埃及·西奈之约·十诫 |
| 利未记 | Vayikra (他呼叫) | 祭祀·洁净·圣洁法典 |
| 民数记 | Bamidbar (在旷野) | 旷野漂流 40 年·人口普查 |
| 申命记 | Devarim (话语) | 摩西的临终讲道——重申律法 |

### Nevi'im (先知书)

| 类别 | 卷 |
|------|-----|
| 前先知 | 约书亚记·士师记·撒母耳记·列王纪 |
| 后先知 | 以赛亚·耶利米·以西结——十二小先知 |

### Ketuvim (圣卷)

| 类别 | 卷 |
|------|-----|
| 诗歌 | 诗篇·箴言·约伯记 |
| 五卷 (Megillot) | 雅歌·路得记·耶利米哀歌·传道书·以斯帖记 |
| 其他 | 但以理书·以斯拉-尼希米记·历代志 |

---

## 古代近东背景

| 文化 | 与圣经的关系 |
|------|------------|
| **美索不达米亚** | 《吉尔伽美什史诗》——洪水叙事与创世记 6-9 惊人相似 |
| **埃及** | 出埃及记的历史背景——摩西与法老 |
| **迦南** | 巴力崇拜——旧约先知持续批判的对象 |
| **赫梯条约** | 申命记的结构类似赫梯宗主条约——暗示圣经使用了当时的政治形式 |

---

## 关键主题

| 主题 | 内容 |
|------|------|
| **约** (Covenant) | 上帝与人立约——诺亚之约·亚伯拉罕之约·西奈之约·大卫之约 |
| **出埃及** | 从奴役到自由——旧约的核心叙事 |
| **律法** (Torah) | 不是「法律」——是「教导」——如何与上帝和邻舍相处 |
| **圣殿** | 从会幕到所罗门圣殿——上帝住在子民中间 |
