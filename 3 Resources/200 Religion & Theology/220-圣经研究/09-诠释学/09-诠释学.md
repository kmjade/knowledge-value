# 09-诠释学

> 「你怎么读圣经？」这个问题决定了你从圣经中看到什么。字面解读和寓意解读可能从同一段经文得出完全相反的结论。释经学就是关于「怎么读」的学问。

## 历史上的四种诠释进路

### 1. 字面/历史释经

| 特征 | 关注文本在原始语境中的含义——「作者想说什么？」 |
|------|------|

| 方法 | 内容 |
|------|------|
| 文法分析 | 原文语法和词汇 |
| 历史背景 | 当时的文化、政治、宗教环境 |
| 文学类型 | 诗歌不能当法律条文读——启示文学不能当历史叙事读 |

### 2. 寓意释经 (Allegory)

| 代表 | 俄利根 (Origen, ~185-254) |
|------|------|
| 例子 | 雅歌——不是情诗——是**基督爱教会**的寓意。「两只乳房」= 旧约和新约 |

| 问题 | 寓意可以无限延伸——任何人都可以读出任何意思 |

### 3. 历史批判法

| 方法 | 问题 |
|------|------|
| 来源批判 | 这卷书用了什么来源？ |
| 形式批判 | 这段话在口传时期是什么形式（诗歌·格言·神迹故事）？ |
| 编修批判 | 作者/编者如何选择和组织材料？有什么神学目的？ |

### 4. 读者回应批判

| 核心 | 意义不是「在文本里」——是读者和文本「互动」中产生的 |
|------|------|
| 解放神学 | 被压迫者从圣经中读出「解放」——特权者读不到 |

---

## 释经的当代问题

| 问题 | 内容 |
|------|------|
| **女性主义释经** | 圣经是父权制的文本——还是被父权制**误读**的文本？ |
| **后殖民释经** | 圣经被殖民者用作殖民工具——同时被被殖民者用作解放工具 |
