# 啟示文學 Apocalyptic Literature

## 定義
啟示文學是猶太-基督教獨特的文學體裁，通過象徵性異象揭示隱藏的屬天真相和末世的盼望。

## 聖經中的啟示文學

| 書卷 | 時代 | 核心信息 |
|------|:--:|------|
| **但以理書** | 6th C BCE | 人子異象、七十個七 |
| **啟示錄** | 1st C CE | 終末審判、新天新地 |

## 啟示錄結構

| 段落 | 章節 | 內容 |
|------|:--:|------|
| 七教會書信 | 1–3 | 對亞細亞七教會的信息 |
| 七印 | 4–7 | 羔羊揭開封印 |
| 七號 | 8–11 | 審判的號角 |
| 七碗 | 15–16 | 最後的災難 |
| 新天新地 | 21–22 | 終末盼望 |

## 解釋方法

| 學派 | 觀點 |
|------|------|
| **過去派 Preterist** | 已在 1 世紀應驗 |
| **歷史派 Historicist** | 預言教會歷史 |
| **未來派 Futurist** | 尚未應驗 |
| **理想派 Idealist** | 象徵善惡爭戰 |
