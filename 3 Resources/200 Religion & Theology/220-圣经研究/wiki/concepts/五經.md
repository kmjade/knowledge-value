# 五經 Pentateuch (Torah)

## 五卷書

| 書卷 | 主題 | 關鍵內容 |
|------|------|---------|
| **創世記** | 起源 | 創造、墮落、洪水、族長 |
| **出埃及記** | 救贖 | 摩西、十災、十誡、會幕 |
| **利未記** | 聖潔 | 獻祭、潔淨律法 |
| **民數記** | 曠野 | 漂流 40 年 |
| **申命記** | 重述 | 律法重述、臨別贈言 |

## 底本學說 Documentary Hypothesis (Wellhausen)

| 底本 | 年代 | 特徵 |
|------|:--:|------|
| **J** (Yahwist) | 10th C BCE | 用 Yahweh、生動敘事 |
| **E** (Elohist) | 9th C BCE | 用 Elohim、較抽象 |
| **D** (Deuteronomist) | 7th C BCE | 申命記風格 |
| **P** (Priestly) | 6th C BCE | 禮儀、家譜、結構化 |
