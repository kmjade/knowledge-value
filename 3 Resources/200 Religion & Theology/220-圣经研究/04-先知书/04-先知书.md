# 04-先知书

> 先知不是算命先生——他们是「上帝的检察官」——代表上帝起诉以色列的背约。先知书的模式：控诉→审判→盼望。

## 先知的角色

| 角色 | 内容 |
|------|------|
| **上帝的代言人** | 「耶和华如此说……」——先知不发表个人意见 |
| **约的检察官** | 依据西奈之约——控诉以色列违约 |
| **社会公义的呼声** | 为穷人、孤儿、寡妇发声——反对宗教虚伪 |

---

## 三大先知

### 以赛亚 (Isaiah)

| 部分 | 章 | 主题 |
|------|-----|------|
| 第一以赛亚 | 1-39 | 审判——亚述威胁——「圣哉！圣哉！圣哉！」 |
| 第二以赛亚 | 40-55 | 安慰——被掳巴比伦——「预备主的道路」——**受苦的仆人** |
| 第三以赛亚 | 56-66 | 回归后——新天新地 |

> 以赛亚书被称为「旧约的福音书」——第 53 章的**受苦仆人**——基督徒认为是耶稣的预言。

### 耶利米 (Jeremiah)

| 主题 | 内容 |
|------|------|
| **流泪的先知** | 耶利米为同胞流泪——但必须宣布审判 |
| **新约的应许** | 「我要将我的律法放在他们里面，写在他们心上」(31:33) ——旧约中最明确的「新约」预言 |
| **圣殿讲道** | 不要依靠圣殿——「圣殿！圣殿！圣殿！」——礼仪不能代替公义 |

### 以西结 (Ezekiel)

| 特征 | 内容 |
|------|------|
| 被掳中的先知 | 在巴比伦——看见上帝荣耀离开圣殿——又在异象中看见圣殿复兴 |
| 枯骨复生 | 第 37 章——平原上的枯骨——复活 |
| 象征性行动 | 侧卧 390 天——剃头分三份——用行为做预言 |

## 十二小先知

| 先知 | 时代 | 核心信息 |
|------|------|---------|
| 何西阿 | 北国灭亡前 | 不忠的妻子——上帝对以色列不离不弃的爱 |
| 阿摩司 | 同 | 社会公义——「唯愿公平如大水滚滚」 |
| 弥迦 | 同 | 「行公义·好怜悯·存谦卑的心与你的上帝同行」 |
| 约拿 | 亚述时期 | 去尼尼微——「普世的」上帝 |
| 哈巴谷 | 巴比伦威胁 | 「义人必因信得生」 |

---

## 预言的双重性质

| 层次 | 内容 |
|------|------|
| **近期应验** | 对当时听众有直接意义——如「亚述必败」 |
| **远期指向** | 终极的复兴——基督徒认为指向耶稣和末世 |
