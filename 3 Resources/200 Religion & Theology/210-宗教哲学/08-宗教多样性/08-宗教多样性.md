# 08-宗教多样性

> 世界上有成千上万种宗教——它们给出相互矛盾的关于终极实在的宣称。如果基督教是对的——穆斯林错了？如果佛教是对的——一神论都错了？还是——所有宗教都是通向同一座山的不同路径？

## 三种立场

### 排他论 (Exclusivism)

| 核心 | 只有一个宗教是真的——其他宗教的信徒不能得救 |
|------|------|
| 代表 | 传统基督教、传统伊斯兰教 |
| 问题 | 一个印度农民从未听过福音——他因为生在错误的地方而下地狱——这公平吗？ |

### 包容论 (Inclusivism)

| 核心 | 只有在自己的宗教中才能得救——但其他宗教的信徒如果「隐含地」回应了真理——也可能被拯救 |
|------|------|
| 代表 | 卡尔·拉纳 (Karl Rahner) ——「无名基督徒」 |
| 问题 | 这改变了其他宗教——把他们变成了「匿名基督徒」——没有真正尊重差异性 |

### 多元论 (Pluralism)

| 核心 | 所有宗教都在回应同一个终极实在——但由于文化、历史、语言的差异——形成不同的「表征」 |
|------|------|
| 代表 | 约翰·希克 (John Hick) |
| 比喻 | 盲人摸象——每个人摸到不同部分——都描述了真实的某个方面 |

#### 希克的多元论

> 终极实在本身 (the Real an sich) 是不可言说的——各个宗教的「上帝」「梵」「空」「道」是这个实在在人类经验中的不同显现。

| 问题 | 如果终极实在是不可言说的——希克怎么知道它「是」所有宗教共同指向的？ |
|------|------|

## 宗教冲突的真正根源？

| 理论 | 内容 |
|------|------|
| **内容冲突** | 基督教说「耶稣是上帝」——伊斯兰教说「耶稣不是上帝」——这不可能都是真的 |
| **救赎途径** | 靠信仰 (基督教) vs 靠行为 (部分佛教) vs 靠顺服 (伊斯兰教) ——不可能都是唯一途径 |
