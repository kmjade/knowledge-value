# 神聖屬性 Divine Attributes

## 傳統屬性 (Omni- 屬性)

| 屬性 | 含義 |
|------|------|
| **全能 Omnipotence** | 能做出任何邏輯上可能的行動 |
| **全知 Omniscience** | 知道一切過去、現在、未來的事 |
| **全善 Omnibenevolence** | 完全的道德善良 |
| **永恆 Eternal** | 存在於時間之外 (無時間性) |
| **不變 Immutable** | 本質不變 |

## 哲學問題

### 全能悖論
「上帝能否創造一塊祂舉不起的石頭？」→ 重新定義全能 = 能做邏輯上可能的事。

### 全知與自由意志
如果上帝預先知道我會做 X，我還能自由選擇不做 X 嗎？
