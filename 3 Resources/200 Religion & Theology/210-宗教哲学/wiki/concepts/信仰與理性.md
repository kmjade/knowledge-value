# 信仰與理性 Faith & Reason

## 四種立場

| 立場 | 觀點 | 代表人物 |
|------|------|---------|
| **理性主義** | 信仰必須通過理性檢驗 | Locke |
| **信仰主義** | 信仰高於理性、獨立於理性 | Kierkegaard, Tertullian |
| **和諧論** | 信仰與理性相輔相成 | **Aquinas**, Plantinga |
| **衝突論** | 信仰與理性本質衝突 | Dawkins (新無神論) |

## 帕斯卡的賭注 Pascal's Wager

```
        上帝存在    上帝不存在
相信      無限收益    有限損失
不相信    無限損失    有限收益

∴ 相信上帝是理性的選擇
```

## 改革宗認識論 Reformed Epistemology (Plantinga)

對上帝的信仰可以是**恰當基本的 (properly basic)**——不需要證據支持，就像對他人心靈或外部世界的信仰一樣。
