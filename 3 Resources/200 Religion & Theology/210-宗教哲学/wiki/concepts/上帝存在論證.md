# 上帝存在論證 Arguments for God's Existence

## 本體論證 Ontological Argument (Anselm, 11th C)

1. 上帝定義為「無法設想比之更偉大的存在」
2. 存在於現實中 > 僅存在於心智中
3. 若上帝僅存在於心智中，則可設想一個更偉大的存在（現實中存在的上帝）
4. ∴ 上帝必須存在於現實中

> Kant 批評：「存在不是謂詞」

## 宇宙論證 Cosmological Argument (Aquinas)

**五路論證 (Five Ways)**：
1. 運動 → 第一推動者
2. 因果 → 第一因
3. 偶然性 → 必然存在
4. 完美等級 → 最完美者
5. 目的論 → 設計者

## 目的論證 Teleological Argument (Paley)

手錶類比：在荒野中發現手錶→推斷有製錶師。宇宙的精細調校同樣指向設計者。

## 道德論證 Moral Argument (Kant)

客觀道德法則的存在需要一個神聖立法者作為終極基礎。
