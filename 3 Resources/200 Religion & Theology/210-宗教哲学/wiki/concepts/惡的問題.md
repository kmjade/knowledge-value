# 惡的問題 Problem of Evil

## 邏輯問題 (Epicurus, 3rd C BCE)

```
1. 上帝是全能的 → 可以消除惡
2. 上帝是全善的 → 願意消除惡
3. 惡存在
∴ 全能全善的上帝不存在？
```

## 神義論回應 Theodicies

| 神義論 | 核心觀點 | 提出者 |
|--------|---------|--------|
| **自由意志辯護** | 惡是自由意志的必然代價 | Plantinga |
| **靈魂塑造** | 苦難塑造美德和品格 | Irenaeus, Hick |
| **更大善** | 惡是實現更大善的手段 | Aquinas |
| **最佳可能世界** | 這是一切可能世界中最好的 | Leibniz |

## 證據問題 Evidential Problem

邏輯問題可通過自由意志辯護回應。但證據問題更難：**為什麼有如此多的惡？** 大屠殺、兒童癌症——這些「過度的」惡如何與全善的上帝調和？
