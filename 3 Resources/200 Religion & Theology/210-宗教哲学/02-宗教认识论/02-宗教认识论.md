# 02-宗教认识论

> 如果有上帝——我们怎么知道？通过自然理性？通过神圣的经文？通过个人的宗教体验？还是不需要「知道」——信仰就够了？

## 宗教信念的基础

| 来源 | 说明 | 问题 |
|------|------|------|
| **理性** | 用论证证明上帝存在 (03章) | 如果论证有缺陷——信仰就站不住脚？ |
| **经典** | 上帝通过经典启示自身 | 怎么知道这本经典是真的启示？循环论证？ |
| **体验** | 个人直接体验到上帝 | 幻觉？不同宗教体验到不同的神——谁是真的？ |
| **传统** | 信仰社群的传承 | 传统本身需要辩护 |
| **信仰** | 不需要证据——信仰本身就是基础 | 凭什么信这个神而不是那个神？ |

## 宗教经验

### 威廉·詹姆斯《宗教经验之种种》(1902)

| 特征 | 内容 |
|------|------|
| **不可言说** | 无法用语言充分描述 |
| **理智性** | 虽然是情感体验——但包含深刻的洞察 |
| **短暂性** | 通常持续几分钟到几小时 |
| **被动性** | 体验者感觉被「抓住」——不是主动产生 |

### 宗教经验作为证据？

| 正方 | 反方 |
|------|------|
| 如果有「表面原则」——如果一个人似乎体验到 X，我们应该信任这个体验除非有反驳理由 | 幻觉、癫痫、药物——都可以产生类似的「体验」 |
| Alston: 宗教体验和感官知觉在认识论上是对称的 | 感官知觉有交叉验证——宗教体验没有 |

## 神迹 (Miracles)

### 休谟的论证

> 「神迹是对自然法则的违反……没有任何证据足以证明一个神迹——除非那个证据的反面比它试图建立的事实更像个神迹。」

| 休谟的要点 | 内容 |
|-----------|------|
| 自然法则的压倒性证据 | 我们有无数的证据支持自然法则——要推翻它需要同样压倒性的证据 |
| 证人的不可靠 | 神迹的目击者通常来自无知和野蛮的民族 |
| 对立神迹 | 不同宗教都声称神迹——它们互相抵消 |

| 类型 | 内容 | 例子 |
