# OpenClaw Skills 技能

> [!info] 概述
> Skills 是可复用的能力模块，让你可以为 AI 添加特定领域的专业能力。

## 什么是 Skills

Skills 是预定义的能力包，包含：
- 系统提示词
- 工具配置
- 示例对话
- 知识库

---

## 内置 Skills

### 通用技能

| Skill | 说明 |
|-------|------|
| `general` | 通用对话（默认） |
| `translator` | 翻译助手 |
| `summarizer` | 内容摘要 |
| `coder` | 编程助手 |

### 专业技能

| Skill | 说明 |
|-------|------|
| `data_analyst` | 数据分析 |
| `researcher` | 研究助手 |
| `writer` | 写作助手 |
| `teacher` | 教学助手 |

---

## 使用 Skills

### 命令行

```bash
# 列出可用技能
openclaw skill list

# 启用技能
openclaw skill enable coder --agent pi

# 禁用技能
openclaw skill disable coder --agent pi
```

### 消息切换

```
/skill coder
```

## 创建自定义 Skill

### Skill 目录结构

```
~/.openclaw/skills/my-skill/
├── skill.json      # Skill 配置
├── system.md       # 系统提示词
├── examples.md     # 示例对话
└── tools.json      # 工具配置
```

### skill.json

```json
{
  "name": "my-skill",
  "version": "1.0.0",
  "description": "我的自定义技能",
  "author": "Your Name",
  "tags": ["custom", "productivity"],
  "requires": ["web_search"],
  "systemPrompt": "system.md",
  "examples": "examples.md"
}
```

### system.md

```markdown
你是一个专业的编程助手，擅长以下领域：

- Python 开发
- JavaScript/TypeScript 开发
- 系统架构设计
- 代码审查

在回答问题时：
1. 提供清晰的代码示例
2. 解释关键概念
3. 给出最佳实践建议
```

### examples.md

```markdown
## 示例 1：代码生成

用户: 写一个 Python 函数计算斐波那契数列
助手: 当然，这是一个使用动态规划的实现...

## 示例 2：代码审查

用户: 请审查这段代码...
助手: 我来分析这段代码...
```

### tools.json

```json
{
  "tools": [
    "web_search",
    "code_interpreter"
  ],
  "config": {
    "code_interpreter": {
      "languages": ["python", "javascript"]
    }
  }
}
```

## 管理 Skills

### 安装 Skill

```bash
# 从本地安装
openclaw skill install /path/to/skill

# 从 URL 安装
openclaw skill install https://example.com/skill.tar.gz
```

### 更新 Skill

```bash
openclaw skill update my-skill
```

### 删除 Skill

```bash
openclaw skill delete my-skill
```

## Skill 配置

### 全局配置

```json
{
  "skills": {
    "defaultSkill": "general",
    "autoEnable": true,
    "searchPaths": [
      "~/.openclaw/skills",
      "/usr/share/openclaw/skills"
    ]
  }
}
```

### Agent 配置

```json
{
  "agents": {
    "pi": {
      "skills": ["general", "coder"],
      "defaultSkill": "general"
    }
  }
}
```

---

## 组合 Skills

### 多技能组合

```json
{
  "agents": {
    "pi": {
      "skills": ["coder", "translator", "writer"]
    }
  }
}
```

AI 会根据消息内容自动选择合适的技能。

### 技能优先级

```json
{
  "agents": {
    "pi": {
      "skillPriority": ["coder", "translator", "general"]
    }
  }
}
```

## Skill 最佳实践

### 1. 单一职责

每个 Skill 应专注于一个领域：
- ✅ `python-expert` - Python 专家
- ✅ `sql-helper` - SQL 助手
- ❌ `all-in-one` - 全能助手

### 2. 清晰的提示词

系统提示词应包含：
- 角色定义
- 能力范围
- 回复风格
- 限制条件

### 3. 实用的示例

提供多样化的示例对话：
- 常见问题
- 边缘情况
- 错误处理
