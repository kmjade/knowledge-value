# OpenClaw 快速开始

> [!info] 概述
> 安装 OpenClaw，运行新手引导，并与你的 AI 助手聊天，全程大约 5 分钟。

## 你需要准备

- **Node.js** — 推荐 Node 24（也支持 Node 22.16+）
- **API Key** — 来自模型提供商（Anthropic、OpenAI、DeepSeek 等）

> [!tip] Windows 用户
> 支持原生 Windows 和 WSL2。WSL2 更稳定，推荐用于完整体验。

---

## 快速设置

### 步骤 1：安装 OpenClaw

**macOS / Linux**
```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

**Windows (PowerShell)**
```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

**其他安装方式**
```bash
# npm
npm install -g openclaw@latest

# Docker
docker run -it openclaw/openclaw
```

### 步骤 2：运行新手引导

```bash
openclaw onboard --install-daemon
```

向导会引导你：
1. 选择模型提供商
2. 设置 API Key
3. 配置 Gateway

大约需要 2 分钟。

### 步骤 3：验证 Gateway 运行

```bash
openclaw gateway status
```

你应该会看到 Gateway 正在监听端口 18789。

### 步骤 4：打开仪表板

```bash
openclaw dashboard
```

这会在浏览器中打开 Control UI（默认 http://127.0.0.1:18789/）

### 步骤 5：发送第一条消息

在 Control UI 聊天中输入消息，你应该会收到 AI 回复。

## 连接消息渠道

想用手机聊天？最快设置的渠道：

### Telegram（推荐）

1. 在 Telegram 中创建 Bot（@BotFather）
2. 获取 Bot Token
3. 配置：
```bash
openclaw channel add telegram --token YOUR_BOT_TOKEN
```

### WhatsApp

1. 运行配对命令：
```bash
openclaw channel add whatsapp
```
2. 扫描二维码完成配对

---

## 常用命令

| 命令 | 说明 |
|------|------|
| `openclaw onboard` | 运行新手引导 |
| `openclaw dashboard` | 打开 Control UI |
| `openclaw gateway status` | 查看网关状态 |
| `openclaw gateway start` | 启动网关 |
| `openclaw gateway stop` | 停止网关 |
| `openclaw gateway restart` | 重启网关 |
