# OpenClaw 新手引导

> [!info] 概述
> 使用 `openclaw onboard` 命令完成引导式设置，大约需要 2 分钟。

## CLI 新手引导

### 启动引导

```bash
openclaw onboard --install-daemon
```

`--install-daemon` 参数会安装系统服务，让 Gateway 在后台持续运行。

### 引导步骤

1. **欢迎信息**
   - 显示 OpenClaw 版本和基本信息

2. **选择模型提供商**
   - Anthropic (Claude)
   - OpenAI (GPT)
   - Google (Gemini)
   - DeepSeek
   - 其他...

3. **输入 API Key**
   - 粘贴你的 API Key
   - 配置会被保存到 `~/.openclaw/openclaw.json`

4. **配置 Gateway**
   - 选择端口（默认 18789）
   - 配置 Control UI

5. **完成设置**
   - Gateway 自动启动
   - 显示访问地址

---

## macOS 应用引导

如果你使用 macOS 应用：

1. 打开 OpenClaw 应用
2. 首次启动会显示引导界面
3. 按照提示完成配置

## 配对流程

### 手机渠道配对

**Telegram**
```bash
openclaw channel add telegram
# 输入 Bot Token
```

**WhatsApp**
```bash
openclaw channel add whatsapp
# 扫描二维码
```

**飞书**
```bash
openclaw channel add feishu
# 配置 App ID 和 Secret
```

### 安全配对

为确保安全，OpenClaw 会：
- 强制执行私信配对
- 支持允许列表配置
- 可设置群组访问规则

## 验证安装

### 检查 Gateway 状态

```bash
openclaw gateway status
```

输出示例：
```
Gateway is running on port 18789
Control UI: http://127.0.0.1:18789
```

### 测试对话

1. 打开 Control UI：
```bash
openclaw dashboard
```

2. 发送测试消息：
```
你好，请介绍一下你自己
```

3. 检查是否收到正常回复

---

## 常见问题

### Gateway 无法启动

1. 检查端口是否被占用
2. 确认 Node.js 版本正确
3. 检查配置文件语法

### API Key 无效

1. 确认 Key 是否正确
2. 检查账户余额
3. 确认 API 访问权限

### 无法连接渠道

1. 检查网络连接
2. 确认 Token/凭证正确
3. 查看 Gateway 日志
