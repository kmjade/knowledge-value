# OpenClaw 本地模型

> [!info] 概述
> OpenClaw 支持本地部署的模型，让你可以在自己的硬件上运行 AI。

## 支持的本地方案

| 方案 | 特点 | 适用场景 |
|------|------|----------|
| Ollama | 简单易用 | 个人开发 |
| vLLM | 高性能 | 生产部署 |
| LM Studio | GUI 界面 | 桌面使用 |
| LocalAI | API 兼容 | 自托管 |

---

## Ollama 集成

### 安装 Ollama

**macOS / Linux**
```bash
curl -fsSL https://ollama.ai/install.sh | sh
```

**Windows**
下载安装包：https://ollama.ai/download

### 下载模型

```bash
# 下载 Llama 3
ollama pull llama3

# 下载 Mistral
ollama pull mistral

# 下载 Qwen
ollama pull qwen2
```

### 启动 Ollama

```bash
ollama serve
```

默认监听 `http://localhost:11434`

---

## 配置 OpenClaw 使用 Ollama

### 配置文件

```json
{
  "providers": {
    "ollama": {
      "baseURL": "http://localhost:11434/v1",
      "apiKey": "ollama"
    }
  },
  "agents": {
    "pi": {
      "provider": "ollama",
      "model": "llama3"
    }
  }
}
```

### 命令行配置

```bash
openclaw provider add ollama \
  --base-url http://localhost:11434/v1 \
  --api-key ollama

openclaw agent set pi --provider ollama --model llama3
```

## 模型参数配置

### 基础参数

```json
{
  "agents": {
    "pi": {
      "provider": "ollama",
      "model": "llama3",
      "parameters": {
        "temperature": 0.7,
        "top_p": 0.9,
        "top_k": 40,
        "num_predict": 2048
      }
    }
  }
}
```

| 参数 | 说明 | 范围 |
|------|------|------|
| `temperature` | 随机性 | 0-2 |
| `top_p` | 核采样 | 0-1 |
| `top_k` | Top-K 采样 | 1-100 |
| `num_predict` | 最大生成长度 | - |

## vLLM 集成

### 安装 vLLM

```bash
pip install vllm
```

### 启动服务

```bash
vllm serve meta-llama/Llama-3-8b-chat-hf \
  --host 0.0.0.0 \
  --port 8000
```

### OpenClaw 配置

```json
{
  "providers": {
    "vllm": {
      "baseURL": "http://localhost:8000/v1",
      "apiKey": "none"
    }
  }
}
```

## LM Studio 集成

### 设置 LM Studio

1. 下载安装 LM Studio
2. 在「Local Server」中启动服务
3. 默认端口：1234

### OpenClaw 配置

```json
{
  "providers": {
    "lmstudio": {
      "baseURL": "http://localhost:1234/v1",
      "apiKey": "none"
    }
  }
}
```

## 性能优化

### GPU 配置

```json
{
  "providers": {
    "ollama": {
      "baseURL": "http://localhost:11434/v1",
      "gpu": {
        "enabled": true,
        "layers": 35
      }
    }
  }
}
```

### 量化配置

Ollama 支持多种量化格式：

```bash
# 4-bit 量化（默认）
ollama pull llama3:8b-q4_0

# 8-bit 量化
ollama pull llama3:8b-q8_0
```

## 混合部署

### 本地 + 云端混合

```json
{
  "agents": {
    "pi": {
      "provider": "anthropic",
      "model": "claude-sonnet-4-6",
      "fallback": {
        "provider": "ollama",
        "model": "llama3"
      }
    }
  }
}
```

### 负载分配

```json
{
  "router": {
    "rules": [
      {
        "condition": "tokens < 1000",
        "agent": "local-agent"
      },
      {
        "condition": "tokens >= 1000",
        "agent": "cloud-agent"
      }
    ]
  }
}
```

## 故障排除

### Ollama 连接失败

1. 确认 Ollama 正在运行
2. 检查端口是否正确
3. 确认模型已下载

### 内存不足

1. 使用量化模型
2. 减少 GPU 层数
3. 切换到更小的模型

### 响应慢

1. 启用 GPU 加速
2. 减少生成长度
3. 使用更快的模型
