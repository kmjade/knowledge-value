# OpenClaw 安装方式

> [!info] 概述
> OpenClaw 支持多种安装方式，推荐使用官方安装脚本。

## 安装脚本（推荐）

### macOS / Linux

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

### Windows (PowerShell)

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

---

## npm 安装

```bash
# 使用 npm
npm install -g openclaw@latest

# 使用 pnpm
pnpm install -g openclaw@latest

# 使用 yarn
yarn global add openclaw
```

---

## Docker 安装

```bash
# 拉取镜像
docker pull openclaw/openclaw

# 运行容器
docker run -it openclaw/openclaw

# 持久化配置
docker run -it -v ~/.openclaw:/root/.openclaw openclaw/openclaw
```

---

## 从源代码安装

```bash
# 克隆仓库
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# 安装依赖
npm install

# 构建
npm run build

# 链接全局命令
npm link
```

---

## 验证安装

```bash
# 检查版本
openclaw --version

# 检查帮助
openclaw --help
```

---

## 平台特定说明

### Windows

- 支持原生 Windows 和 WSL2
- **推荐 WSL2**：更稳定，完整体验
- PowerShell 脚本需要管理员权限

### macOS

- 支持 Intel 和 Apple Silicon
- 可使用 Homebrew 安装依赖

### Linux

- 支持 Ubuntu、Debian、Fedora 等
- 需要安装构建工具

---

## 更新 OpenClaw

```bash
# npm 安装的
npm update -g openclaw

# 安装脚本
curl -fsSL https://openclaw.ai/install.sh | bash
```

---

## 卸载

```bash
# npm 安装的
npm uninstall -g openclaw

# 清理配置
rm -rf ~/.openclaw
```
