# OpenClaw 配置文件

> [!info] 概述
> OpenClaw 配置位于 `~/.openclaw/openclaw.json`。

## 配置文件位置

| 配置类型 | 路径 |
|----------|------|
| 主配置 | `~/.openclaw/openclaw.json` |
| 状态目录 | `~/.openclaw/` |

---

## 基础配置示例

```json
{
  "gateway": {
    "port": 18789,
    "controlUi": {
      "enabled": true
    }
  },
  "providers": {
    "anthropic": {
      "apiKey": "your-api-key"
    }
  }
}
```

---

## 主要配置项

### Gateway 配置

```json
{
  "gateway": {
    "port": 18789,
    "host": "0.0.0.0",
    "controlUi": {
      "enabled": true,
      "root": null
    }
  }
}
```

| 配置项 | 说明 |
|--------|------|
| `port` | Gateway 监听端口 |
| `host` | 监听地址 |
| `controlUi.enabled` | 是否启用 Control UI |
| `controlUi.root` | 自定义 UI 目录 |

---

---

### 渠道配置

```json
{
  "channels": {
    "telegram": {
      "token": "your-bot-token",
      "enabled": true
    },
    "whatsapp": {
      "enabled": true
    }
  }
}
```

---

### 安全配置

```json
{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+15555550123"],
      "groups": {
        "*": { "requireMention": true }
      }
    }
  },
  "messages": {
    "groupChat": {
      "mentionPatterns": ["@openclaw"]
    }
  }
}
```

| 配置项 | 说明 |
|--------|------|
| `allowFrom` | 允许的发件人列表 |
| `requireMention` | 群组中是否需要提及 |
| `mentionPatterns` | 提及模式 |

---

### 环境变量

| 变量 | 说明 |
|------|------|
| `OPENCLAW_HOME` | 主目录 |
| `OPENCLAW_STATE_DIR` | 状态目录 |
| `OPENCLAW_CONFIG_PATH` | 配置文件路径 |

---

## 配置最佳实践

1. **安全配置**
   - 使用 `allowFrom` 限制访问
   - 群组设置 `requireMention`
   - 不要提交 API Key 到版本控制

2. **性能优化**
   - 只启用需要的渠道
   - 合理配置会话隔离

3. **维护**
   - 定期备份配置文件
   - 记录自定义配置
