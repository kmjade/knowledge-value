# OpenClaw 微信集成

> [!warning] 注意
> 微信集成存在限制，请了解风险后使用。

## 集成方式

### 方式对比

| 方式 | 难度 | 稳定性 | 功能 |
|------|------|--------|------|
| 企业微信 | ⭐⭐ | ✅ 稳定 | 完整 |
| 微信公众号 | ⭐⭐ | ✅ 稳定 | 受限 |
| Web 协议 | ⭐⭐⭐ | ⚠️ 风险 | 完整 |

---

## 企业微信（推荐）

### 创建企业微信应用

1. 访问 [企业微信管理后台](https://work.weixin.qq.com/)
2. 进入「应用管理」
3. 创建应用

### 获取凭证

- **企业 ID (CorpID)**
- **应用 AgentID**
- **应用 Secret**

### 配置回调

1. 设置 API 接收
2. 配置 URL：
   ```
   https://your-gateway.com/webhook/wework
   ```
3. 设置 Token 和 EncodingAESKey

### OpenClaw 配置

```json
{
  "channels": {
    "wework": {
      "corpId": "wwxxxxxxxx",
      "agentId": 1000001,
      "secret": "xxxxxxxxxxxxxxxx",
      "token": "your-token",
      "encodingAESKey": "your-aes-key",
      "enabled": true
    }
  }
}
```

### 命令行配置

```bash
openclaw channel add wework \
  --corp-id wwxxxxxxxx \
  --agent-id 1000001 \
  --secret xxxxxxxxxxxxxxxx
```

## 微信公众号

### 适用场景

- 公众号消息回复
- 订阅号/服务号

### 配置步骤

1. 登录微信公众平台
2. 进入「基本配置」
3. 配置服务器地址
4. 设置 Token

### 限制

- 需要认证服务号才能使用大部分 API
- 消息回复有 5 秒超时限制
- 每月主动消息有限额

---

## Web 协议（不推荐）

> [!danger] 高风险
> 此方式违反微信服务条款，可能导致账号封禁。

### 风险

- 账号可能被封禁
- 协议不稳定，随时可能失效
- 需要频繁处理验证码

### 替代方案

建议使用企业微信，它提供了完整的官方 API 支持。

## 安全配置

### 企业微信

```json
{
  "channels": {
    "wework": {
      "allowFrom": ["user1", "user2"],
      "groups": {
        "*": { "requireMention": true }
      }
    }
  }
}
```

### 访问控制

- 配置可信 IP
- 设置用户白名单
- 限制群组访问

## 常见问题

### 企业微信不回复

1. 检查回调 URL 配置
2. 确认 Secret 正确
3. 查看 Gateway 日志

### 公众号超时

公众号消息回复有 5 秒限制：
- 优化 AI 响应速度
- 使用异步回复机制
- 配置「客服消息」

### 权限问题

确认应用有必要的权限：
- 发送消息权限
- 获取用户信息权限
