# OpenClaw 渠道概览

> [!info] 概述
> OpenClaw 支持多种消息渠道，让你的 AI 助手可以在各个平台上与你对话。

## 添加渠道

### 命令行方式

```bash
# 列出可用渠道
openclaw channel list

# 添加渠道
openclaw channel add <渠道名>

# 查看渠道状态
openclaw channel status <渠道名>

# 删除渠道
openclaw channel remove <渠道名>
```

### 配置文件方式

```json
{
  "channels": {
    "telegram": {
      "token": "your-bot-token",
      "enabled": true
    },
    "whatsapp": {
      "enabled": true
    }
  }
}
```

## 渠道权限

### 私信权限

```json
{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+15555550123", "+15555550124"]
    }
  }
}
```

### 群组权限

```json
{
  "channels": {
    "whatsapp": {
      "groups": {
        "*": { "requireMention": true },
        "group-123": { "requireMention": false }
      }
    }
  }
}
```

| 配置项 | 说明 |
|--------|------|
| `allowFrom` | 允许的发送者白名单 |
| `requireMention` | 群组中是否需要 @ 提及 |
| `groups` | 按群组 ID 配置权限 |

## 多渠道管理

### 会话隔离

OpenClaw 自动隔离不同渠道的会话：
- 每个渠道有独立的会话历史
- 支持跨渠道用户识别
- 可配置共享上下文

### 统一回复

```json
{
  "gateway": {
    "unifiedResponse": true
  }
}
```

启用后，AI 回复会自动适配各渠道的消息格式。
