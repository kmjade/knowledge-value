# 機器學習

## 📚 內容摘要

### 監督式學習
- **分類**：預測類別標籤
  - 決策樹、隨機森林、SVM
  - 應用：垃圾郵件檢測、圖像識別
- **迴歸**：預測連續值
  - 線性迴歸、嶺迴歸
  - 應用：房價預測、趨勢分析

### 非監督式學習
- **聚類**：發現數據模式
  - K-Means、層次聚類、DBSCAN
  - 應用：用戶分群、異常檢測
- **降維**：減少特徵數量
  - PCA、t-SNE、UMAP
  - 應用：視覺化、特徵提取

### 深度學習
- **CNN**：卷積神經網路
  - 應用：圖像處理、目標檢測
- **RNN/LSTM**：循環神經網路
  - 應用：序列預測、時間序列
- **Transformer**：注意力機制
  - 應用：NLP、語言模型
