# 快速开始

## 选择使用方式

| 方式 | 适合人群 | 特点 |
|------|----------|------|
| 网页版 | 普通用户 | 最简单，无需安装 |
| CLI 工具 | 开发者 | 终端操作，功能强大 |
| API | 开发者 | 集成到应用 |

---

## 方式一：网页版（推荐新手）

### 步骤 1：访问 Gemini

**国内用户**：使用镜像站
- [Gemini 镜像站](https://xsimplechat.com)
- [Gemini 中文版](https://ai.huoyachat.com)
- [Gemini 官网镜像](https://www.gptokk.com)

**海外用户**：直接访问
- [Gemini 官网](https://gemini.google.com)

### 步骤 2：登录
使用 Google 账号登录即可开始使用。

### 步骤 3：开始对话
直接在输入框输入问题，Gemini 会即时响应。

---

## 方式二：CLI 工具（推荐开发者）

### 安装

**快速运行（无需安装）**
```bash
npx https://github.com/google-gemini/gemini-cli
```

**全局安装（npm）**
```bash
npm install -g @google/gemini-cli
```

**全局安装（Homebrew - macOS/Linux）**
```bash
brew install gemini-cli
```

### 认证

启动后选择认证方式：
1. **Login with Google**（推荐）- 免费额度：60次/分钟，1000次/天
2. **API Key** - 需要[获取 API Key](https://aistudio.google.com/apikey)

### 使用

```bash
# 启动交互模式
gemini

# 非交互模式
gemini "解释什么是机器学习"

# 指定文件
gemini "分析这个代码" --file main.py
```

---

## 方式三：API 调用

### 获取 API Key

1. 访问 [AI Studio](https://aistudio.google.com/apikey)
2. 点击「Create API Key」
3. 复制生成的 Key

### Python 示例

```python
import google.generativeai as genai

# 配置 API Key
genai.configure(api_key="YOUR_API_KEY")

# 创建模型实例
model = genai.GenerativeModel('gemini-2.5-pro')

# 生成内容
response = model.generate_content("你好，介绍一下你自己")
print(response.text)
```

### 安装 SDK

```bash
pip install google-generativeai
```
