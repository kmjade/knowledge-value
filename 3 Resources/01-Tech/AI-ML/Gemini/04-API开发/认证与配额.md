# 认证与配额

## 认证方式

### 方式一：API Key（推荐新手）

#### 获取 API Key

1. 访问 [AI Studio](https://aistudio.google.com/apikey)
2. 点击「Create API Key」
3. 选择或创建 Google Cloud 项目
4. 复制生成的 Key

#### 使用 API Key

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")
```

```bash
# 环境变量
export GEMINI_API_KEY=your-api-key
```

### 方式二：OAuth 2.0（推荐生产环境）

#### 服务账号

```python
from google.oauth2 import service_account
import google.generativeai as genai

credentials = service_account.Credentials.from_service_account_file(
    'path/to/service-account.json',
    scopes=['https://www.googleapis.com/auth/cloud-platform']
)

genai.configure(credentials=credentials)
```

#### 用户认证

```bash
# 安装 gcloud CLI
gcloud auth application-default login
```

```python
# 自动使用应用默认凭证
import google.generativeai as genai
genai.configure()
```

### 方式三：Vertex AI（企业环境）

```python
from vertexai.preview.generative_models import GenerativeModel

# 需要设置 GOOGLE_CLOUD_PROJECT 环境变量
model = GenerativeModel("gemini-2.5-pro")
```

## 上下文缓存

### 启用缓存

```python
import google.generativeai as genai

model = genai.GenerativeModel('gemini-2.5-pro')

# 创建缓存
cache = genai.caching.CachedContent.create(
    model='gemini-2.5-pro',
    display_name='my-cache',
    contents=[
        {"role": "user", "parts": [{"text": "长文档内容..."}]}
    ],
    ttl=3600  # 1小时过期
)

# 使用缓存
model = genai.GenerativeModel.from_cached_content(cache)
response = model.generate_content("关于这个文档的问题")
```

### 缓存价格优势

使用缓存可节省 **75%** 的输入成本。

## 最佳实践

### 成本优化

1. **使用缓存** - 对于重复的上下文
2. **选择合适的模型** - 简单任务用 Flash
3. **控制输出长度** - 设置 max_output_tokens
4. **批量处理** - 合并多个请求

### 错误处理

```python
from google.api_core.exceptions import ResourceExhausted
import time

def safe_generate(model, prompt, max_retries=3):
    for attempt in range(max_retries):
        try:
            return model.generate_content(prompt)
        except ResourceExhausted:
            if attempt < max_retries - 1:
                wait_time = (attempt + 1) * 2
                print(f"配额耗尽，等待 {wait_time} 秒后重试...")
                time.sleep(wait_time)
            else:
                raise
```
