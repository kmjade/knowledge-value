# 高级功能

## Google Search Grounding

使用 Google 搜索获取实时信息：

```bash
# 启用搜索增强
gemini "2026年最新的 AI 发展动态" --search
```

### 适用场景
- 实时信息查询
- 新闻事件
- 最新技术动态
- 事实核查

---

## Shell 命令执行

Gemini CLI 可以执行 Shell 命令：

```bash
# 让 Gemini 执行命令
gemini "列出当前目录的所有 Python 文件"

# 自动执行模式
gemini "查找并删除所有 .tmp 文件" --execute
```

### 安全控制
- 默认会询问确认
- 使用 `--execute` 自动执行
- 危险操作需要额外确认

## 文件操作

### 读取文件
```bash
gemini "分析这个配置文件" --file config.json
```

### 编辑文件
```bash
# 让 Gemini 修改文件
gemini "在这个函数中添加类型注解" --file main.py --edit
```

### 创建文件
```bash
gemini "创建一个 FastAPI 项目模板" --create
```

---

## 多文件分析

```bash
# 分析多个文件
gemini "比较这两个实现" --file v1.py --file v2.py

# 分析整个项目
gemini "总结项目架构" --dir ./src --recursive
```

## 对话检查点

保存和恢复复杂对话：

```bash
# 保存检查点
> /checkpoint save feature-design

# 恢复检查点
> /checkpoint restore feature-design

# 列出检查点
> /checkpoint list
```

## 自定义上下文

### GEMINI.md 文件

在项目根目录创建 `GEMINI.md`：

```markdown
# 项目背景

这是一个电商后台管理系统，使用以下技术栈：
- 前端：React + TypeScript
- 后端：Node.js + Express
- 数据库：PostgreSQL

## 代码规范

- 使用函数式组件
- 遵循 ESLint 配置
- 测试覆盖率 > 80%

## 常用命令

- npm run dev: 开发服务器
- npm test: 运行测试
- npm run build: 构建生产版本
```

### 作用
- 提供项目背景信息
- 定义代码规范
- 加速上下文理解

## 工作流自动化

### 脚本集成

```bash
#!/bin/bash
# 自动代码审查脚本

echo "正在审查代码..."
gemini "审查最近的代码变更，检查是否有安全问题和性能问题" --dir ./src --diff

echo "生成报告..."
gemini "生成代码审查报告" --format markdown > review-report.md
```

### CI/CD 集成

```yaml
# GitHub Actions 示例
name: AI Code Review
on: [pull_request]
jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: AI Review
        uses: google-github-actions/run-gemini-cli@v1
        with:
          prompt: "审查这个 PR 的代码变更"
```
