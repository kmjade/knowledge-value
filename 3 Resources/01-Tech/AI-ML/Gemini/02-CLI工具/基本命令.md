# 基本命令

## 启动方式

| 命令 | 说明 |
|------|------|
| `gemini` | 启动交互模式 |
| `gemini "提示词"` | 单次问答模式 |
| `gemini --help` | 显示帮助信息 |
| `gemini --version` | 显示版本号 |

---

## 交互模式

### 启动
```bash
gemini
```

### 基本操作

| 操作 | 说明 |
|------|------|
| 输入文本 | 直接输入问题，按 Enter 发送 |
| `Ctrl+C` | 退出当前会话 |
| `Ctrl+D` | 退出程序 |
| `↑` / `↓` | 浏览历史输入 |

### 会话管理

| 命令 | 说明 |
|------|------|
| `/clear` | 清除当前对话 |
| `/exit` | 退出程序 |
| `/help` | 显示帮助 |

---

## 文件操作

### 指定文件
```bash
# 分析单个文件
gemini "分析这个代码" --file main.py

# 分析多个文件
gemini "比较这两个文件" --file a.py --file b.py
```

### 目录分析
```bash
# 分析整个目录
gemini "总结这个项目" --dir ./src
```

---

## 模型选择

```bash
# 指定模型
gemini --model gemini-2.5-pro

# 可用模型
# gemini-2.5-pro（默认）
# gemini-2.5-flash
# gemini-3-pro
```

---

## 输出控制

| 参数 | 说明 |
|------|------|
| `--json` | JSON 格式输出 |
| `--markdown` | Markdown 格式输出 |
| `--quiet` | 静默模式，仅输出结果 |

---

## 会话保存

### 保存会话
```bash
# 保存当前会话
gemini --save-session my-session
```

### 恢复会话
```bash
# 恢复之前的会话
gemini --restore-session my-session
```

---

## 非交互模式

适用于脚本和自动化：

```bash
# 单次执行
gemini "解释这段代码" --file script.py

# 管道输入
cat README.md | gemini "总结这个文档"

# 输出到文件
gemini "生成代码文档" --file main.py > docs.md
```

---

## 常用命令速查

```bash
# 快速问答
gemini "什么是 REST API?"

# 代码分析
gemini "分析这个函数的时间复杂度" --file algorithm.py

# 代码生成
gemini "用 Python 实现一个简单的 HTTP 服务器"

# 文档生成
gemini "为这个函数生成文档注释" --file utils.py

# 翻译
gemini "将以下内容翻译成英文: 你好世界"

# 总结
cat article.txt | gemini "总结这篇文章的核心观点"
```
