# 安装与配置

## 系统要求

| 要求 | 说明 |
|------|------|
| Node.js | 版本 20 或更高 |
| 操作系统 | macOS, Linux, Windows |
| 网络 | 需要访问 Google 服务 |

---

## 安装方式

### 方式一：npx（推荐尝鲜）

无需安装，直接运行：
```bash
npx https://github.com/google-gemini/gemini-cli
```

### 方式二：npm 全局安装

```bash
npm install -g @google/gemini-cli
```

### 方式三：Homebrew（macOS/Linux）

```bash
brew install gemini-cli
```

---

## 版本选择

| 版本 | 安装命令 | 说明 |
|------|----------|------|
| Stable | `npm install -g @google/gemini-cli@latest` | 稳定版，每周二发布 |
| Preview | `npm install -g @google/gemini-cli@preview` | 预览版，每周二发布 |
| Nightly | `npm install -g @google/gemini-cli@nightly` | 每日构建版 |

## 认证配置

### 方式一：Google 账号登录（推荐）

1. 启动 Gemini CLI：
   ```bash
   gemini
   ```

2. 选择「Login with Google」

3. 按提示在浏览器中完成认证

**免费配额**：
- 60 请求/分钟
- 1000 请求/天
- Gemini 2.5 Pro 模型

### 方式二：API Key 认证

1. 获取 API Key：
   - 访问 [AI Studio](https://aistudio.google.com/apikey)
   - 点击「Create API Key」
   - 复制生成的 Key

2. 配置环境变量：
   ```bash
   export GEMINI_API_KEY="your-api-key"
   ```

3. 启动：
   ```bash
   gemini
   ```

**免费配额**：
- 100 请求/天
- Gemini 2.5 Pro 模型

## 组织账户配置


## 常见问题

### 安装失败

**问题**：npm 安装超时
**解决**：使用国内镜像
```bash
npm config set registry https://registry.npmmirror.com
npm install -g @google/gemini-cli
```

### 认证失败

**问题**：浏览器认证无法完成
**解决**：
1. 检查网络连接
2. 尝试使用 API Key 认证
3. 清除缓存后重试

### Windows 路径问题

**问题**：Windows 下路径识别错误
**解决**：使用正斜杠或转义反斜杠
