# OpenCode oh-my-openagent

> [!info] 概述
> oh-my-openagent 是 OpenCode 的高级代理框架，用于构建复杂的 AI 代理工作流。

## 什么是 oh-my-openagent

oh-my-openagent 是一个代理配置集合，提供：
- 预配置的专业代理
- 可复用的代理模板
- 最佳实践工作流

---

## 安装

```bash
# 克隆仓库
git clone https://github.com/opencode/oh-my-openagent

# 复制到配置目录
cp -r oh-my-openagent/agents ~/.config/opencode/agents/
```

## 包含的代理

### 开发类代理

| 代理 | 说明 |
|------|------|
| `frontend-dev` | 前端开发专家 |

### 文档类代理

| 代理 | 说明 |
|------|------|
| `tech-writer` | 技术文档写作 |
| `api-docs` | API 文档生成 |

---

## 使用代理

### 列出可用代理

```
/agents
```

### 切换代理

按 `Tab` 键切换，或：

```
@frontend-dev 帮我优化这个组件
```

---

## 自定义代理

### 创建代理配置

```json
{
  "my-custom-agent": {
    "mode": "primary",
    "model": "anthropic/claude-sonnet-4-20250514",
    "prompt": "你是一个专业的 [角色]，专注于 [任务]...",
    "tools": {
      "write": true,
      "edit": true,
      "bash": true
    }
  }
}
```

### 代理模板

```
.opencode/agents/
└── my-agent/
    └── AGENT.md
```

## 工作流示例

### 前端开发工作流

```
@frontend-dev 创建一个登录组件
@test-generator 为登录组件生成测试
@code-analyst 审查代码质量
```

### 后端开发工作流

```
@backend-dev 设计 API 接口
@security 检查安全性
@tech-writer 生成 API 文档
```

## 最佳实践

1. **选择合适的代理**：根据任务选择专业代理
2. **组合使用**：多个代理协作完成复杂任务
3. **自定义代理**：根据团队需求定制代理
4. **版本控制**：将代理配置提交到 Git
