# OpenCode 生态系统

> [!info] 概述
> OpenCode 拥有丰富的生态系统，包括 MCP 服务器、Skills、工具等。

## 核心组件

### OpenCode CLI
命令行工具，核心入口。

### OpenCode TUI
终端用户界面，交互式使用。

### OpenCode Web
Web 界面，图形化操作。

### OpenCode VSCode Extension
VSCode 集成插件。
