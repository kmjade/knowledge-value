# OpenCode 模型选择指南

> [!info] 概述
> OpenCode 支持多种 AI 模型，选择合适的模型可以平衡成本和效果。

## 模型分类

### 按能力

| 级别 | 模型 | 适用场景 |
|------|------|----------|
| 顶级 | Claude Opus, GPT-4 | 复杂任务、架构设计 |
| 高级 | Claude Sonnet, GPT-4o | 日常开发、代码审查 |
| 中级 | Claude Haiku, GPT-3.5 | 简单任务、快速响应 |
| 入门 | DeepSeek, 开源模型 | 学习、测试 |

### 按价格

| 类型 | 模型 | 价格水平 |
|------|------|----------|
| 免费 | OpenCode Zen Free, 部分开源模型 | 免费 |
| 低价 | DeepSeek, 阿里百炼 | ¥1-5/百万 token |
| 中价 | Claude Haiku, GPT-3.5 | $0.25-1/百万 token |
| 高价 | Claude Sonnet, GPT-4 | $3-15/百万 token |
| 顶级 | Claude Opus | $75-150/百万 token |

---

## 推荐配置

### 个人开发者

```json
{
  "model": "deepseek/deepseek-coder"
}
```

**理由**：性价比最高，编程能力强。

### 团队开发

```json
{
  "model": "anthropic/claude-sonnet-4-20250514"
}
```

**理由**：平衡性能和成本，适合大多数场景。

### 复杂项目

```json
{
  "model": "anthropic/claude-opus-4-20250514"
}
```

**理由**：最强能力，适合架构设计和复杂推理。

## 切换模型

### TUI 中切换

```
/model
```

### 配置默认模型

```json
{
  "model": "anthropic/claude-sonnet-4-20250514"
}
```

### 按任务切换

```
/model claude-opus  # 复杂任务
/model deepseek     # 日常开发
```

---

## 免费模型

### OpenCode Zen Free

```
/model
→ 选择标记 Free 的模型
```

### 开源模型

通过 Ollama 运行本地模型：

```bash
ollama run codellama
```

---

## 成本优化建议

1. **简单任务用便宜模型**：问答、简单代码
2. **复杂任务用好模型**：架构设计、重构
3. **使用免费模型测试**：验证流程后再用付费模型
4. **设置用量限制**：避免意外消耗
