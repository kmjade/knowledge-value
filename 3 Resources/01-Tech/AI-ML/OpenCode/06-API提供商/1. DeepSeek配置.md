# OpenCode DeepSeek 配置

> [!info] 概述
> DeepSeek 是国内领先的 AI 模型提供商，性价比高，适合国内用户使用。

## 为什么选择 DeepSeek

| 优势 | 说明 |
|------|------|
| 国产模型 | 国内访问稳定 |
| 性价比高 | 价格比海外模型便宜 |
| 中文优化 | 对中文理解更好 |
| 编程能力强 | 代码生成质量高 |

---

## 获取 API Key

1. 访问：https://platform.deepseek.com/
2. 注册/登录账号
3. 进入 API Keys 页面
4. 创建新的 API Key

---

## 配置步骤

### 方式一：TUI 配置

```
/connect
→ 选择 deepseek
→ 输入 DeepSeek API Key
```

### 方式二：配置文件

```json
{
  "providers": {
    "deepseek": {
      "api_key": "${DEEPSEEK_API_KEY}",
      "base_url": "https://api.deepseek.com"
    }
  }
}
```

### 方式三：环境变量

```bash
export DEEPSEEK_API_KEY=sk-xxxx
```

## 可用模型

| 模型 | 说明 | 特点 |
|------|------|------|
| `deepseek-chat` | 通用对话模型 | 平衡性价比 |
| `deepseek-coder` | 编程专用模型 | 代码能力强 |
| `deepseek-reasoner` | 推理模型 | 复杂推理任务 |

---

## 使用 DeepSeek

### 指定模型

```
/model deepseek-coder
```

### 配置默认模型

```json
{
  "model": "deepseek/deepseek-coder"
}
```

## 使用建议

### 适用场景
- 国内用户
- 成本敏感项目
- 中文编程任务
- 日常开发工作

### 不适用场景
- 需要最强推理能力
- 复杂系统设计
- 英文为主的项目
