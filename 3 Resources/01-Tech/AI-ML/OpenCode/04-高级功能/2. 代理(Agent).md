# OpenCode 代理(Agent)

> [!info] 概述
> 代理是 OpenCode 中专门化的 AI 助手，每个代理可以配置独立的系统提示词、模型、工具访问权限。

## 代理类型

| 类型 | 说明 | 切换方式 |
|------|------|----------|
| **主代理（primary）** | 直接交互的主要助手 | `Tab` 键循环切换 |
| **子代理（subagent）** | 由主代理调用执行专门任务 | 主代理自动调用或 `@代理名` |

---

## 内置代理

### 主代理

#### Build（默认）
- 启用了所有工具
- 适合需要完整文件操作和系统命令访问权限的日常开发工作

#### Plan
- 专为代码分析和规划设计的受限代理
- `edit` 和 `bash` 操作会触发审批提示
- 适合只分析代码不修改的场景

### 子代理

#### General
- 通用子代理，拥有完整的工具访问权限
- 可以修改文件
- 适用于研究复杂问题和执行多步骤任务

#### Explore
- 只读子代理，无法修改任何文件
- 适合快速查找文件、搜索代码关键字
- 速度快且不会产生副作用

### 系统代理（隐藏）

| 代理名称 | 职责 | 触发时机 |
|----------|------|----------|
| Compaction | 压缩上下文 | 上下文长度达到阈值时 |
| Title | 生成会话标题 | 新会话开始后 |
| Summary | 创建会话摘要 | 需要会话摘要时 |

---

## 代理使用方式

### 切换主代理

按 `Tab` 键在主代理之间循环切换，按 `Shift+Tab` 反向切换。

### 调用子代理

#### 自动调用
主代理根据任务性质判断需要某个子代理时，会自动调用。

#### 手动调用
```
@general help me search for this function
@explore 找到所有使用这个 API 的地方
```

---

## 会话导航

子代理运行时会创建独立的子会话：

| 快捷键 | 操作 |
|--------|------|
| `<Leader>→` | 向前循环：父会话 → 子会话 |
| `<Leader>←` | 向后循环 |
| `<Leader>↑` | 直接返回父会话 |

## 代理配置

### 方式一：opencode.json

```json
{
  "agent": {
    "build": {
      "mode": "primary",
      "model": "anthropic/claude-sonnet-4-20250514",
      "prompt": "{file:./prompts/build.txt}",
      "tools": {
        "write": true,
        "edit": true,
        "bash": true
      }
    },
    "code-reviewer": {
      "description": "Reviews code for best practices",
      "mode": "subagent",
      "model": "anthropic/claude-sonnet-4-20250514",
      "prompt": "You are a code reviewer...",
      "tools": {
        "write": false,
        "edit": false
      }
    }
  }
}
```

### 方式二：AGENT.md 文件

```
.opencode/agents/
└── my-agent/
    └── AGENT.md
```

---

## 配置字段说明

| 字段 | 说明 |
|------|------|
| `mode` | `primary` 或 `subagent` |
| `model` | 使用的模型 |
| `prompt` | 系统提示词 |
| `tools` | 工具权限配置 |
| `description` | 代理描述 |

---

## CLI 命令

```bash
# 创建代理
opencode agent create

# 列出代理
opencode agent list
```

## 自定义代理示例

### 代码审查代理

```json
{
  "code-reviewer": {
    "description": "Reviews code for security and performance",
    "mode": "subagent",
    "model": "anthropic/claude-sonnet-4-20250514",
    "prompt": "You are a code reviewer. Focus on security, performance, and maintainability.",
    "tools": {
      "write": false,
      "edit": false,
      "read": true,
      "grep": true
    }
  }
}
```

### 测试生成代理

```json

    "prompt": "You are a test engineer. Generate comprehensive unit tests.",
    "tools": {
      "write": true,
      "edit": true,
      "read": true
    }
  }
}
```
