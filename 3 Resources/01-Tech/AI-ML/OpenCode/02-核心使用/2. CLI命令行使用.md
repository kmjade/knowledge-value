# OpenCode CLI 命令行使用

> [!info] 概述
> OpenCode CLI 不只是一个命令工具，而是一个**可编程的 AI 开发接口**。

## 架构层次

```
┌─────────────────────────────────────┐
│ 交互层: tui / web / attach          │
├─────────────────────────────────────┤
│ 执行层: run / agent / mcp / github  │
├─────────────────────────────────────┤
│ 服务层: serve / acp                 │
└─────────────────────────────────────┘
```

本质是 **AI Agent + 本地开发环境 + 可编程 CLI + Server API** 的统一入口。

---

## 基础命令

| 命令 | 说明 |
|------|------|
| `opencode` | 默认启动 TUI |
| `opencode run "msg"` | 非交互执行任务 |
| `opencode --help` | 查看帮助 |
| `opencode --version` | 查看版本 |

## TUI 启动参数

| 标志 | 简写 | 说明 |
|------|------|------|
| `--continue` | `-c` | 继续上一次会话 |
| `--session` | `-s` | 指定会话 ID |
| `--fork` | — | 分叉会话 |
| `--prompt` | — | 初始提示词 |
| `--model` | `-m` | 指定模型 |
| `--agent` | — | 指定代理 |
| `--port` | — | 监听端口 |
| `--hostname` | — | 主机地址 |

## 核心 CLI 命令

### agent - 代理管理

```bash
# 创建自定义代理
opencode agent create

# 列出所有代理
opencode agent list
```

### auth - 认证管理

```bash
# 登录
opencode auth login

# 列出认证
opencode auth list

# 登出
opencode auth logout
```

### attach - 连接远程后端

```bash
opencode attach
```

### github - GitHub 自动化

```bash
# 安装 GitHub App
opencode github install

# 运行 GitHub 任务
opencode github run
```

### mcp - MCP 服务管理

```bash
# 添加 MCP 服务器
opencode mcp add

# 列出 MCP 服务器
opencode mcp list

# MCP 认证
opencode mcp auth

# MCP 调试
opencode mcp debug
```

### models - 查看模型列表

```bash
opencode models
```

### run - 非交互执行

```bash
# 执行单个任务
opencode run "分析项目结构"

# 指定模型
opencode run -m claude-sonnet "优化代码"

# 指定代理
opencode run --agent plan "审查代码"
```

### serve - 启动 API 服务

```bash
opencode serve
```

### web - 启动 Web 界面

```bash
opencode web
```

## 使用场景

### 自动化脚本

```bash
# 自动化代码审查
opencode run "审查代码并生成报告" > review.md

# CI/CD 集成
opencode run "运行测试并修复失败用例"
```

### 批量操作

```bash
# 批量重构
for dir in packages/*; do
  opencode run "重构 $dir"
done
```

### 远程执行

```bash
# 连接远程服务器
opencode attach --host remote-server

# 执行任务
opencode run "部署到生产环境"
```

---

## 高级用法

### 组合命令

```bash
# 分析后执行
opencode run "分析项目" && opencode run "生成文档"
```

### 输出重定向

```bash
# 导出分析报告
opencode run "分析代码质量" > report.md
```

### 环境变量

```bash
# 设置模型
export OPENCODE_MODEL=claude-sonnet

# 执行
opencode run "优化代码"
```
