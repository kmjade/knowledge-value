# OpenCode 知识库导航

> [!info] 概述
> OpenCode 是一款开源 AI 编程协作工具，定位为基于命令行（CLI）的 AI 编程 Agent，可深度参与项目全流程开发。

## 核心定位

OpenCode 并非简单的代码生成器，而是能读取项目、理解上下文、自主执行开发任务的 AI 编程助手：
- **运行在终端中的 AI 程序员助手**
- 脱离网页聊天框的局限，直接在终端中运行
- 深度理解整个项目，全程参与真实的编码、修改与重构过程
