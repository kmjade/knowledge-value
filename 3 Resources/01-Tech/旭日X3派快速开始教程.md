# 旭日X3派快速开始教程

> [!info] 教程概述
> 本教程详细介绍旭日X3派的准备工作、系统安装、远程登录和快速体验，帮助用户快速上手开发。

## 🔧 1. 准备工作

### 1.1 硬件准备

#### 基础配件

| 配件 | 说明 |
|------|------|
| **旭日X3派开发板** | 核心开发板 |
| **电源适配器** | 12V/2A或更高规格 |
| **Micro SD卡** | 建议Class 10，容量≥32GB |
| **Type-C数据线** | 用于供电和调试 |
| **HDMI线** | 用于视频输出（可选） |
| **显示器** | HDMI接口（可选） |
| **网线** | 有线网络连接（可选） |
| **USB转TTL模块** | 串口调试（可选） |

#### 摄像头配件

| 摄像头类型 | 说明 |
|-----------|------|
| **USB摄像头** | 标准USB接口摄像头 |
| **MIPI CSI摄像头** | 旭日X3派专用MIPI摄像头 |

#### 显示屏配件

| 显示屏类型 | 说明 |
|-----------|------|
| **HDMI显示屏** | 标准HDMI接口显示屏 |

### 1.2 软件准备

#### 开发机环境

- **操作系统**：Ubuntu 18.04/20.04 或 Windows 10/11
- **Python**：Python 3.7+
- **SD卡烧录工具**：
  - Windows：BalenaEtcher、Raspberry Pi Imager
  - Linux：dd命令、BalenaEtcher
- **串口工具**（可选）：
  - Windows：PuTTY、SecureCRT
  - Linux：minicom、picocom

#### 远程登录工具

- **VNC客户端**：RealVNC、TightVNC
- **SSH客户端**：
  - Windows：PuTTY、Git Bash
  - Linux/Mac：终端ssh命令

## 🔥 2. 安装系统

### 2.1 镜像准备

#### 下载系统镜像

1. 访问旭日X3派开发者文档
2. 下载最新的系统镜像文件（.img格式）
3. 验证镜像文件的完整性（MD5/SHA256）

#### 镜像说明

系统镜像通常包含：
- Ubuntu Linux操作系统
- 预装的开发工具和库
- BPU驱动和推理库
- 示例代码和文档

### 2.2 烧录系统

#### Windows烧录步骤

1. **插入SD卡**：
   - 将Micro SD卡插入SD卡读卡器
   - 将读卡器插入电脑USB接口

2. **启动烧录工具**：
   - 下载并安装BalenaEtcher
   - 启动BalenaEtcher

3. **选择镜像**：
   - 点击"Flash from file"
   - 选择下载的系统镜像文件（.img）

4. **选择目标**：
   - 点击"Select target"
   - 选择Micro SD卡设备

5. **开始烧录**：
   - 点击"Flash!"开始烧录
   - 等待烧录完成（可能需要几分钟）

6. **完成提示**：
   - 看到烧录成功提示后，关闭BalenaEtcher
   - 安全移除SD卡

#### Linux烧录步骤

```bash
# 1. 查看SD卡设备
lsblk

# 2. 卸载SD卡所有分区（假设SD卡是/dev/sdX）
sudo umount /dev/sdX*

# 3. 烧录镜像到SD卡
sudo dd if=x3pi_image.img of=/dev/sdX bs=4M status=progress conv=fsync

# 4. 同步并完成
sudo sync

# 5. 移除SD卡
sudo eject /dev/sdX
```

### 2.3 启动系统

#### 首次启动步骤

1. **插入SD卡**：
   - 将烧录好的SD卡插入旭日X3派

2. **连接电源**：
   - 将电源适配器连接到旭日X3派
   - 接通电源

3. **观察启动过程**：
   - 旭日X3派开始启动
   - LED指示灯会闪烁
   - 如果连接了HDMI显示器，可以看到启动画面

4. **等待系统启动完成**：
   - 通常需要30-60秒
   - 系统启动后会显示登录界面

## 🔌 3. 远程登录

### 3.1 串口登录

#### 硬件连接

1. **连接USB转TTL模块**：
   - GND → GND
   - TX → RX
   - RX → TX

2. **连接到电脑**：
   - 将USB转TTL模块插入电脑USB接口

#### 串口工具配置

**参数设置**：
- 波特率：115200
- 数据位：8
- 停止位：1
- 校验位：None
- 流控制：None

**Linux使用minicom**：
```bash
# 安装minicom
sudo apt install minicom

# 连接串口（假设串口设备是/dev/ttyUSB0）
sudo minicom -D /dev/ttyUSB0

# 退出：Ctrl+A，然后按Q
```

**Windows使用PuTTY**：
1. 打开PuTTY
2. 设置串口参数
3. 点击"Open"连接

### 3.2 网络状态确认

#### 查看IP地址

```bash
# 查看所有网络接口
ip addr

# 或使用ifconfig
ifconfig
```

**网络接口说明**：
- `eth0` - 有线网络接口
- `wlan0` - 无线网络接口

#### 默认账户信息

| 用户名 | 密码 | 权限 |
|--------|------|------|
| `sunrise` | `sunrise` | 普通用户 |
| `root` | `root` | 超级用户 |

> [!warning] 安全提示
> - 首次登录后建议修改默认密码
> - 生产环境不要使用默认密码

### 3.3 VNC登录

#### 配置VNC服务

**启动VNC服务器**：
```bash
# 切换到sunrise用户
su sunrise

# 启动VNC服务器
vncserver :1

# 设置VNC密码（首次运行时）
```

**停止VNC服务器**：
```bash
vncserver -kill :1
```

#### 连接VNC客户端

1. **打开VNC客户端**（RealVNC Viewer、TightVNC Viewer）
2. **输入服务器地址**：`<IP地址>:1`
3. **输入密码**：设置的VNC密码
4. **连接**：点击连接

**使用SSH隧道**（推荐）：
```bash
# 在本地电脑执行
ssh -L 5901:localhost:5901 sunrise@<旭日X3派IP>

# 然后VNC客户端连接到 localhost:5901
```

### 3.4 SSH登录

#### SSH连接命令

```bash
# 基本SSH连接
ssh sunrise@<旭日X3派IP>

# 指定端口连接（默认端口22）
ssh -p 22 sunrise@<旭日X3派IP>

# 使用密钥认证
ssh -i ~/.ssh/id_rsa sunrise@<旭日X3派IP>
```

#### 配置SSH密钥（可选）

**生成SSH密钥对**：
```bash
# 在本地电脑生成密钥
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
```

**复制公钥到开发板**：
```bash
# 方法1：使用ssh-copy-id
ssh-copy-id sunrise@<旭日X3派IP>

# 方法2：手动复制
ssh sunrise@<旭日X3派IP>
mkdir -p ~/.ssh
echo "<你的公钥内容>" >> ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
chmod 700 ~/.ssh
```

## ⚡ 4. 快速体验

### 4.1 LED指示灯控制

#### 示例代码

```python
import hobotLed
import time

# 初始化LED
led = hobotLed.HobotLed()

# 设置LED颜色和亮度
# 参数: (r, g, b), 取值范围0-255
led.set_led((255, 0, 0))   # 红色
time.sleep(1)
led.set_led((0, 255, 0))   # 绿色
time.sleep(1)
led.set_led((0, 0, 255))   # 蓝色
time.sleep(1)
led.set_led((255, 255, 0)) # 黄色

# 关闭LED
led.set_led((0, 0, 0))
```

#### 运行示例

```bash
# 创建LED控制脚本
cd ~/hobot_led_demo/
python led_demo.py
```

### 4.2 GPIO读写操作

#### 示例代码

```python
import hobot_gpio as gpio

# 初始化GPIO
gpio.init()

# 设置GPIO模式
gpio.set_mode(gpio.BOARD)  # 使用物理引脚编号
# 或
gpio.set_mode(gpio.BCM)    # 使用BCM编号

# 配置GPIO为输出
gpio.setup(12, gpio.OUT)

# 输出高电平
gpio.output(12, gpio.HIGH)

# 输出低电平
gpio.output(12, gpio.LOW)

# 配置GPIO为输入
gpio.setup(11, gpio.IN)

# 读取GPIO状态
value = gpio.input(11)
print(f"GPIO 11 状态: {value}")

# 清理GPIO
gpio.cleanup()
```

#### GPIO引脚说明

- **物理引脚编号**：对应开发板物理上的引脚号（1-40）
- **BCM编号**：对应芯片的GPIO编号

### 4.3 图像分类算法示例

#### Mobilenet v1示例

```python
import hobot_dnn
import hobot_vio
import cv2

# 初始化摄像头
cam = hobot_vio.Camera()
cam.open_cam(hobot_vio.CAMERA_USB_ID)

# 加载Mobilenet v1模型
model = hobot_dnn.Model("mobilenet_v1")

# 读取类别标签
with open("synset_words.txt", "r") as f:
    labels = f.readlines()

# 获取图像
image = cam.get_img()
image_bgr = cv2.cvtColor(image, cv2.COLOR_RGB2BGR)

# 模型推理
outputs = model.forward(image)

# 解析结果
class_id = outputs[0].argmax()
confidence = outputs[0][class_id]
label = labels[class_id].strip()

print(f"类别: {label}")
print(f"置信度: {confidence:.2f}")

# 关闭摄像头
cam.close_cam()
```

#### 运行方法

```bash
cd ~/samples/mobilenet_v1/
python mobilenet_demo.py
```

### 4.4 USB摄像头使用

#### 基本使用

```python
import hobot_vio
import cv2

# 初始化摄像头
cam = hobot_vio.Camera()

# 打开USB摄像头
cam.open_cam(hobot_vio.CAMERA_USB_ID)

# 获取一帧图像
image = cam.get_img()

# 显示图像
cv2.imshow("USB Camera", image)
cv2.waitKey(0)
cv2.destroyAllWindows()

# 关闭摄像头
cam.close_cam()
```

#### 连续获取图像

```python
import hobot_vio
import cv2

# 初始化摄像头
cam = hobot_vio.Camera()
cam.open_cam(hobot_vio.CAMERA_USB_ID)

while True:
    # 获取图像
    image = cam.get_img()
    
    # 显示图像
    cv2.imshow("USB Camera", image)
    
    # 按'q'退出
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

# 关闭摄像头
cam.close_cam()
cv2.destroyAllWindows()
```

## 📦 5. 硬件接口说明

### 5.1 接口总览

旭日X3派提供丰富的硬件接口：

| 接口类型 | 功能 | 用途 |
|---------|------|------|
| **40PIN接口** | GPIO、I2C、SPI、UART | 传感器、执行器、通信 |
| **电源接口** | 12V DC电源 | 系统供电 |
| **调试串口** | UART | 串口调试 |
| **有线网口** | 1000Mbps Ethernet | 有线网络 |
| **HDMI接口** | HDMI视频输出 | 显示输出 |
| **USB接口** | USB 3.0 | USB设备 |
| **USB摄像头** | USB摄像头 | 图像采集 |
| **MIPI CSI** | MIPI CSI摄像头 | 图像采集 |
| **Micro SD** | TF卡槽 | 存储 |
| **Wi-Fi** | 无线网络 | 无线网络 |
| **蓝牙** | 蓝牙 | 短距离通信 |
| **40PIN** | GPIO扩展 | 外设扩展 |
| **音频接口** | 麦克风/耳机 | 音频输入输出 |

### 5.2 电源接口

- **输入电压**：12V DC
- **推荐电流**：≥2A
- **接口类型**：DC插头
- **供电方式**：12V/2A电源适配器

### 5.3 调试串口

- **接口类型**：UART
- **波特率**：115200
- **数据位**：8
- **停止位**：1
- **校验位**：None

### 5.4 有线网口

- **接口类型**：RJ45
- **速度**：1000Mbps
- **支持协议**：TCP/IP

### 5.5 HDMI接口

- **接口类型**：HDMI
- **分辨率**：最高支持1080p
- **用途**：视频输出

### 5.6 USB接口

- **接口类型**：USB 3.0
- **数量**：2个（典型）
- **用途**：外设连接、调试

### 5.7 USB摄像头

- **接口类型**：USB
- **支持格式**：USB Video Class (UVC)
- **分辨率**：取决于摄像头

### 5.8 MIPI摄像头

- **接口类型**：MIPI CSI
- **支持分辨率**：最高1080p
- **优势**：低延迟、高性能

### 5.9 Micro SD接口

- **卡类型**：Micro SD/SDHC/SDXC
- **容量建议**：≥32GB
- **速度等级**：Class 10或更高

### 5.10 无线连接

#### Wi-Fi
- **标准**：IEEE 802.11 b/g/n/ac
- **频段**：2.4GHz/5GHz
- **模式**：Station/AP模式

#### 蓝牙
- **版本**：BLE 4.2
- **用途**：短距离通信

### 5.11 40PIN

#### 管脚定义

40PIN接口包含：
- **GPIO** - 通用输入输出
- **I2C** - I2C总线
- **SPI** - SPI总线
- **UART** - 串口通信
- **PWM** - 脉冲宽度调制
- **5V/3.3V/GND** - 电源和地

#### 管脚复用关系

部分GPIO引脚支持多功能复用：
- GPIO ↔ PWM
- GPIO ↔ UART
- GPIO ↔ I2C
- GPIO ↔ SPI

### 5.12 音频接口

#### 驱动加载

```bash
# 加载音频驱动
sudo modprobe snd-bcm2835
```

# 播放音频
aplay audio.wav

# 录音

```bash

```
