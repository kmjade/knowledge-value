## 💻 快速开始

### 🎯 10分钟快速入门
```python
# 你的第一个Python程序
print("Hello, Python!")

# 基础数据类型
name = "Python"        # 字符串
version = 3.9          # 浮点数  
is_awesome = True     # 布尔值

# 简单函数
def greet(language):
    return f"Hello, {language}!"

print(greet("Python"))  # 输出: Hello, Python!
```
