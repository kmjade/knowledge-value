# 工具推薦

LLM Wiki 生態相關的推薦工具。

---

## 必備工具

| 工具 | 用途 | 必要性 | 鏈接 |
|------|------|--------|------|
| **Obsidian** | 瀏覽 Wiki、圖譜視圖、實時預覽 | 強烈推薦 | [obsidian.md](https://obsidian.md) |
| **Obsidian Web Clipper** | 瀏覽器一鍵剪藏網頁為 Markdown | 強烈推薦 | [瀏覽器擴展](https://obsidian.md/clipper) |
| **Git** | 版本控制、歷史記錄 | 推薦 | [git-scm.com](https://git-scm.com) |

---

## 可選工具

| 工具 | 用途 | 說明 |
|------|------|------|
| **Dataview** | 基於 frontmatter 生成動態表格 | Obsidian 插件 |
| **Marp** | 從 Wiki 內容生成演示文稿 | Obsidian 插件 |
| **qmd** | Wiki 頁面搜索引擎 | 大規模時需要 |

---

## Obsidian 推薦設置

### 附件目錄設置

```
Settings → Files and links → Attachment folder path → raw/assets/
```

### 下載附件快捷鍵

```
Settings → Hotkeys → 搜索 "Download" → 綁定 Ctrl+Shift+D
```

### 圖譜視圖

- 查看 Wiki 結構
- 識別樞紐頁
- 發現孤立頁
