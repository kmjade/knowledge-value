# Agent Browser

讓 AI Agent 能夠瀏覽和操作網頁的概念和能力。

---

## 核心功能

- **網頁抓取**：自動獲取網頁內容
- **實時信息**：獲取最新網絡信息
- **交互操作**：模擬用戶操作網頁

---

## 應用場景

| 場景 | 說明 |
|------|------|
| 競品分析 | 實時抓取競品網頁信息 |
| 數據收集 | 自動收集網絡數據 |
| 內容監控 | 監控網頁內容變化 |

---

## 與 Claude Code 的協作

配合 [[claude-code]] 進行：
- 自動化信息收集
- 實時數據分析
- 結構化報告生成
