# Claude Code + n8n 工作流資料摘要

> [!summary] 摘要
> Claude Code 與 n8n 結合實現兩層自動化，讓 AI 幫你搭建自動化工作流，實現指數級效率提升。

---

## 核心要點

### 兩層自動化模式

| 層級 | 角色 | 說明 |
|------|------|------|
| 第一層 | n8n 執行 | 7x24 小時運行，處理固定流程 |
| 第二層 | Claude Code 搭建 | AI 幫你創建和配置工作流 |

### 效率對比

- 傳統方式：一天最多 3-5 個工作流
- 自動化方式：一天幾十個工作流

---

## 關鍵洞見

### 為什麼有效？

> 維護知識庫的繁重工作不是閱讀和思考，而是記賬——更新交叉引用、保持摘要最新、標註矛盾。

### 關鍵前提

> 「如果你不理解自己要自動化的流程，你只是在規模化混亂。」
