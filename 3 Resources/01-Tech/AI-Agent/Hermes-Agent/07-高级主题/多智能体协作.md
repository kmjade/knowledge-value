# 多智能体协作

> [!info] 
> Hermes Agent 支持多种多智能体模式：

- **delegate_task**：子智能体委派（最多 3 并发）
- **tmux 多开**：完全独立的 Hermes 实例
- **Cron + Profile**：定时任务使用不同配置
- **Kanban**：多智能体工作队列
