# 协议规范

> [!summary] 概述
> Hermes Agent 使用标准的 **OpenAI 兼容 function calling** 协议进行工具调用。

> [!warning] 不是 XML 标签协议
> Hermes Agent 不使用 `<tool_call]>` XML 标签。
> 那是 Hermes 3 **模型**的旧格式。Hermes Agent **框架**使用标准 JSON function calling。

---

## OpenAI Function Calling 协议

Hermes Agent 与 LLM 之间的工具调用采用 OpenAI 兼容格式：

### 请求中的工具定义

```json
{
  "messages": [...],
  "tools": [
    {
      "type": "function",
      "function": {
        "name": "read_file",
        "description": "Read a text file with line numbers and pagination",
        "parameters": {
          "type": "object",
          "properties": {
            "path": {
              "type": "string",
              "description": "Path to the file"
            },
            "offset": {
              "type": "integer",
              "default": 1
            },
            "limit": {
              "type": "integer",
              "default": 500
            }
          },
          "required": ["path"]
        }
      }
    }
  ]
}
```

### LLM 返回的工具调用

```json
{
  "role": "assistant",
  "content": null,
  "tool_calls": [
    {
      "id": "call_abc123",
      "type": "function",
      "function": {
        "name": "read_file",
        "arguments": "{\"path\": \"/home/user/test.py\", \"limit\": 50}"
      }
    }
  ]
}
```

### 工具结果返回

```json
{
  "role": "tool",
  "tool_call_id": "call_abc123",
  "content": "1|import os\n2|\n3|def main():\n..."
}
```

## 消息序列示例

```
System: 你是 Hermes Agent...（工具定义 + 环境信息）
   ↓
User:   帮我看看 /etc/hosts 文件内容
   ↓
LLM:    tool_call: read_file(path="/etc/hosts", limit=100)
   ↓
Tool:   "1|127.0.0.1 localhost\n2|..."
   ↓
LLM:    /etc/hosts 文件包含以下内容：127.0.0.1 localhost...
```

---

## JSON Schema 参数类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `string` | 字符串 | `"/tmp/test.txt"` |
| `integer` | 整数 | `100` |
| `number` | 浮点数 | `0.7` |
| `boolean` | 布尔 | `true` |
| `array` | 数组 | `["a", "b"]` |
| `object` | 嵌套对象 | `{"key": "val"}` |

### 常用约束

| 属性 | 说明 |
|------|------|
| `type` | 数据类型 |
| `description` | 参数说明 |
| `enum` | 允许值列表 |
| `default` | 默认值 |
| `required` | 标记必需参数（在父级 `required` 数组中） |

---

## 多工具并行调用

LLM 可以在一次响应中调用多个工具：

```json
{
  "tool_calls": [
    {
      "id": "call_1",
      "function": {"name": "read_file", "arguments": "{\"path\": \"a.py\"}"}
    },
    {
      "id": "call_2",
      "function": {"name": "read_file", "arguments": "{\"path\": \"b.py\"}"}
    }
  ]
}
```

---

## 与 Hermes 3 模型协议的区别

| | Hermes Agent 框架 | Hermes 3 模型 |
|--|------------------|--------------|
| **格式** | OpenAI function calling (JSON) | `<tool_call]>` XML 标签 |
| **系统角色** | 自动由框架构建 | 需要手动在 system prompt 中注入 |
| **适用场景** | 框架使用者无需关心 | 直接使用模型时需要 |
