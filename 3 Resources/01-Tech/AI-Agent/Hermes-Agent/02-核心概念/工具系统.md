# 工具系统

> [!summary] 概述
> Hermes Agent 的工具系统——通过 `toolsets` 管理工具，底层使用 OpenAI 兼容的 function calling 协议。

> [!warning]
> Hermes Agent 的工具是框架内置的，不需要用户用 Python 定义和注册。
> 用户只需通过 `hermes tools` 开关工具集，框架自动处理工具的定义、发现和调用。

---

## 工具集管理

### 交互式管理

```bash
hermes tools          # curses UI 交互式开关
```

### 命令行管理

```bash
hermes tools list          # 列出所有工具及状态
hermes tools enable NAME   # 启用工具集
hermes tools disable NAME  # 禁用工具集
```

工具变更需要 `/reset`（新会话）生效。

## 工具按平台配置

```bash
hermes tools    # 在 UI 中可按平台（CLI、Telegram、Discord 等）分别开关
```

这样可以让 Telegram 上的 Hermes 不能执行终端命令，而 CLI 上的可以。

## 功能工具详解

### terminal — Shell 命令

| 功能 | 说明 |
|------|------|
| 前台命令 | `terminal(command, timeout=N)` |
| 后台进程 | `terminal(background=True, notify_on_complete=True)` |
| PTY 模式 | `terminal(pty=True)` — 交互式 CLI 工具 |

### file — 文件操作

| 工具 | 用途 |
|------|------|
| `read_file` | 按行读取（替代 cat/head/tail） |
| `write_file` | 覆盖写入（替代 echo heredoc） |
| `search_files` | 内容搜索 + 文件查找（替代 grep/find/ls） |
| `patch` | 精确替换编辑（替代 sed/awk） |

### web — 网络

| 工具 | 用途 |
|------|------|
| `web_search` | 搜索引擎查询 |
| `web_fetch` | 提取网页内容 |

## 工具的底层机制

### OpenAI Function Calling

Hermes Agent 使用标准的 OpenAI 兼容 function calling 协议，不是 XML 标签：

```json
// 发送给 LLM 的工具定义
{
  "type": "function",
  "function": {
    "name": "read_file",
    "description": "Read a text file with line numbers",
    "parameters": {
      "type": "object",
      "properties": {
        "path": {"type": "string", "description": "File path"},
        "limit": {"type": "integer", "default": 500}
      },
      "required": ["path"]
    }
  }
}
```

LLM 返回：
```json
{
  "tool_calls": [{
    "id": "call_xxx",
    "function": {
      "name": "read_file",
      "arguments": "{\"path\": \"/tmp/test.txt\"}"
    }
  }]
}
```

### 工具注册（开发者视角）

每个工具在 `tools/` 目录下有一个 `.py` 文件，通过 `registry.register()` 注册：

```python
# tools/registry.py 中的注册模式
registry.register(
    name="read_file",
    toolset="file",
    schema={...},
    handler=lambda args, **kw: read_file(...),
    check_fn=check_requirements,
)
```

工具注册（开发者视角）参考 [官方开发者文档](https://hermes-agent.nousresearch.com/docs/developer-guide/)。
