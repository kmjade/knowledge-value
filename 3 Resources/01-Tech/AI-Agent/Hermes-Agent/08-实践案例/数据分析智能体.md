# 数据分析智能体

> [!summary] 概述
> 使用 Hermes Agent 进行数据分析——读取数据、执行分析、生成报告。

---

## 交互式分析

```bash
hermes
```

```
用户: 读取 sales.csv，分析月度销售趋势，
      找出增长最快的产品类别，生成图表
```

Hermes 会：
1. `read_file` 读取 CSV
2. `execute_code` 运行 Python 分析（pandas、matplotlib）
3. `write_file` 保存图表和报告

---

## 配合 Jupyter

```bash
# 加载 Jupyter 技能
hermes -s jupyter-live-kernel "分析 data/experiment_results.csv，
运行统计检验，生成结论"
```

---

## 定时数据报告

```bash
# 每周一生成销售周报
hermes cron create "0 9 * * 1" "读取 ~/data/sales_this_week.csv，
分析销售数据，生成周报保存到 ~/reports/"
```
