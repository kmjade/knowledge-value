# 天气查询智能体

> [!summary] 概述
> 使用 Hermes Agent（框架）完成天气查询任务——通过自然语言对话即可，无需编写 Python 代码。

> [!tip] 使用 Hermes Agent 不需要写代码
> Hermes Agent 是一个 CLI 框架，你只需要用自然语言描述任务。

---

## 方式一：交互式会话（推荐）

```bash
# 启动 Hermes
hermes

# 在会话中说：
# "帮我查一下北京今天的天气"
```

Hermes 会自动使用 `web_search` 工具查询天气并回复。

---

## 方式二：单次任务

```bash
hermes chat -q "查询北京、上海、广州今天的天气，做成一个对比表格"
```

---

## 方式三：定时天气报告

```bash
# 每天早 8 点发送天气报告
hermes cron create "0 8 * * *" "查询北京今天的天气并告诉我"
```

---

## 扩展：创建天气技能

如果经常查询天气，可以创建一个 Skill：

```bash
# 在 Hermes 会话中完成一次天气查询后，让它创建技能
# 对 Hermes 说: "把我刚才查询天气的方式保存为一个技能"
```
