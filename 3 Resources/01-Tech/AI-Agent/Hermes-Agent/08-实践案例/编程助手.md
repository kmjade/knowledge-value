# 编程助手

> [!summary] 概述
> Hermes Agent 作为日常编程助手的使用模式。

---

## 代码生成

```bash
hermes chat -q "在 ~/myproject 中创建一个 FastAPI 用户认证服务，
包含 JWT token、登录/注册端点、数据库模型"
```

---

## 代码审查

```bash
hermes chat -q "审查 ~/myproject/src/ 下所有 Python 文件，
指出安全问题、性能问题和代码风格问题"
```

---

## Debugging

在交互式会话中直接贴入错误信息：

```
用户: 运行 pytest 时遇到这个错误：
E   TypeError: expected str, bytes or os.PathLike object, not NoneType
帮我找出原因并修复
```

Hermes 会读取相关文件、分析错误、修复代码、运行测试确认。

---

## 使用专门的编程技能

```bash
# 加载代码审查技能
hermes -s github-code-review

# 加载 TDD 工作流
hermes -s test-driven-development
```
