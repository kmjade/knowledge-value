# 简介

> [!summary] 概述
> Hermes Agent 是 Nous Research 开发的开源 AI 智能体**框架**，可在终端、消息平台和 IDE 中自主完成任务。

---

## 什么是 Hermes Agent

Hermes Agent 是一个开源的 AI 智能体框架，定位类似于 Claude Code（Anthropic）、Codex CLI（OpenAI）。它运行在你的终端中，通过工具调用与系统交互——执行命令、读写文件、搜索网络等。

> [!warning] 与 Hermes 3 模型的区别
> - **Hermes Agent** = AI 智能体**框架**（本主题）
> - **Hermes 3** = Nous Research 的 **LLM 模型**（8B/70B/405B 参数）
> - Hermes Agent 可以使用 Hermes 3 模型，也可以使用任何其他提供商（OpenRouter、Anthropic、DeepSeek 等）

### 核心定义

| 概念 | 说明 |
|------|------|
| **智能体框架 (Agent Framework)** | 运行 LLM + 工具调用的完整系统 |
| **工具调用 (Tool Calling)** | 智能体调用外部工具的能力（终端、文件、网络等） |
| **Skills 技能** | 可复用的程序性知识，智能体从经验中学习并持久化 |
| **Memory 记忆** | 跨会话的持久事实存储 |

---

## 核心特性

### 1. 技能系统 (Skills)

Hermes Agent 独有的能力——从经验中学习，将工作流程保存为技能文档：

```bash
# 创建技能
hermes skills install <url>

# 会话中加载技能
/skill python-debugpy
```

技能会在未来会话中自动加载，使智能体越来越擅长你的具体任务。

### 2. 持久记忆 (Memory)

记住你是谁、偏好、环境细节：

```python
# 智能体自动保存记忆
# 下次会话时这些信息会被注入
```

### 3. 多平台网关 (Gateway)

同一智能体运行在多个平台：

```bash
hermes gateway setup   # 配置 Telegram、Discord 等
hermes gateway run     # 启动网关
```

### 4. 提供商无关

```bash
hermes model           # 交互式切换模型/提供商
hermes chat -m anthropic/claude-sonnet-4
hermes chat --provider deepseek
```

### 5. 完整的工具系统

| 工具集 | 功能 |
|--------|------|
| `terminal` | Shell 命令和进程管理 |
| `file` | 文件读写搜索 |
| `web` | 网络搜索和内容提取 |
| `browser` | 浏览器自动化 |
| `delegation` | 子智能体任务委派 |
| `skills` | 技能浏览和管理 |
| 20+ 更多 | ... |

## 应用场景

### 1. 软件开发

```bash
hermes -s claude-code
# 或直接在会话中说：
"在 ~/myapp 中实现用户认证系统"
```

### 2. 系统管理

```bash
hermes chat -q "检查所有 Docker 容器状态，重启异常服务"
```

### 3. 数据分析

```bash
# 会话中：
"读取 data.csv，分析销售趋势，生成可视化报告"
```

### 4. 定时任务

```bash
hermes cron create "every 6h" "检查服务器健康状态并报告"
```

### 5. 内容创作

```bash
hermes -s excalidraw "画一个微服务架构图"
```

## 快速开始

```bash
# 安装
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

# 启动
hermes

# 配置
hermes setup
```
