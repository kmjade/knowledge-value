# 工具调用格式

> [!summary] 概述
> Hermes Agent 使用标准的 **OpenAI 兼容 function calling** 格式，不是 XML 标签。

> [!warning]
> Hermes Agent 框架使用 JSON function calling 协议。
> `<tool_call]>` XML 标签是 Hermes 3 **模型**的旧格式，仅模型直调时使用。

---

## Function Calling 格式

### LLM 请求中的工具定义

```json
{
  "type": "function",
  "function": {
    "name": "read_file",
    "description": "Read a text file with line numbers",
    "parameters": {
      "type": "object",
      "properties": {
        "path": {"type": "string", "description": "File path"},
        "offset": {"type": "integer", "default": 1},
        "limit": {"type": "integer", "default": 500}
      },
      "required": ["path"]
    }
  }
}
```

### LLM 返回的工具调用

```json
{
  "role": "assistant",
  "content": null,
  "tool_calls": [
    {
      "id": "call_abc123",
      "type": "function",
      "function": {
        "name": "read_file",
        "arguments": "{\"path\": \"/tmp/test.py\", \"limit\": 50}"
      }
    }
  ]
}
```

### 工具执行结果

```json
{
  "role": "tool",
  "tool_call_id": "call_abc123",
  "content": "1|import os\n2|\n3|def main():\n..."
}
```

## 单参数调用

### 天气查询

```json
{
  "tool_calls": [{
    "id": "call_w1",
    "function": {
      "name": "web_search",
      "arguments": "{\"query\": \"北京今天天气\"}"
    }
  }]
}
```

### 文件读取

```json
{
  "tool_calls": [{
    "id": "call_f1",
    "function": {
      "name": "read_file",
      "arguments": "{\"path\": \"/etc/hosts\", \"limit\": 20}"
    }
  }]
}
```

---

## 多参数调用

```json
{
  "tool_calls": [{
    "id": "call_e1",
    "function": {
      "name": "terminal",
      "arguments": "{\"command\": \"docker ps\", \"timeout\": 30, \"workdir\": \"/home/user\"}"
    }
  }]
}
```

---

## 并行调用（多工具同时）

```json
{
  "tool_calls": [
    {
      "id": "call_1",
      "function": {"name": "read_file", "arguments": "{\"path\": \"src/main.py\"}"}
    },
    {
      "id": "call_2",
      "function": {"name": "read_file", "arguments": "{\"path\": \"src/utils.py\"}"}
    },
    {
      "id": "call_3",
      "function": {"name": "read_file", "arguments": "{\"path\": \"src/config.py\"}"}
    }
  ]
}
```

---

## 参数类型

| 类型 | JSON 表示 | 示例 |
|------|----------|------|
| `string` | `"..."` | `"北京"` |
| `integer` | 数字 | `100` |
| `number` | 浮点 | `3.14` |
| `boolean` | `true/false` | `true` |
| `array` | `[...]` | `["a", "b"]` |
| `object` | `{...}` | `{"key": "val"}` |

---

## 对比：框架 vs 模型格式

| | Hermes Agent 框架 | Hermes 3 模型 |
|--|------------------|--------------|
| 格式 | JSON function calling | `<tool_call]>` XML |
| 多工具 | 并行 tool_calls 数组 | 逐个 XML 标签 |
| 使用方 | 框架自动处理 | 模型使用者手动解析 |
