# 計算機網絡學習路徑

> [!info] 學習路徑概述
> 本路徑涵蓋從網絡基礎到進階實戰的完整學習計劃，適合零基礎或有一定基礎的學習者。

---

## 🎯 學習目標

完成本學習路徑後，你將能夠：
- ✅ 掌握各層核心協議的工作原理
- ✅ 具備基本的網絡安全意識和防護能力
- ✅ 獨立進行網絡故障排查

---

## 📅 學習階段

### 階段一：基礎入門（1-2周）

> 🎓 目標：建立網絡基礎概念，理解基本工作原理

#### 第1周：網絡基礎理論
- **Day 1-2**: [[OSI七層模型]] 和 [[TCP/IP協議棧]]
  - 理解分層架構
  - 對比兩個模型
  - 畫出數據封裝/解封裝流程圖

- **Day 3-4**: [[IP地址與子網劃分]]
  - IPv4 地址分類
  - 子網掩碼計算
  - CIDR 表示法
  - 實踐：手動計算子網劃分

- **Day 5**: [[網絡拓撲結構]]
  - 了解常見的網絡拓撲
  - 理解拓撲設計的優缺點

- **Day 6-7**: 復習與實踐
  - 使用 [[網絡診斷工具]] 中的 ping 命令測試網絡連通性

#### 第2周：應用層協議
- **Day 1-2**: [[HTTP/HTTPS]]
  - 理解請求/響應模型
  - 常見狀態碼
  - HTTPS 加密原理

- **Day 3-4**: [[DNS]] 和 [[DHCP]]
  - DNS 解析過程
  - DHCP 工作原理

- **Day 5**: [[應用層協議]] 其他協議概覽
  - FTP, SMTP, SSH 等

- **Day 6-7**: 實踐專案
  - 搭建簡單的 Web 服務器測試 HTTP
  - 配置 DNS 服務器

#### 🎯 階段一驗收
- [ ] 能說出 OSI 七層模型的每一層功能
- [ ] 能手算簡單的子網劃分
- [ ] 能看懂 HTTP 報文結構
- [ ] 能使用 ping, nslookup 等基礎工具

---

### 階段二：協議深入（2-3周）

> 🎓 目標：深入理解核心協議的工作機制

#### 第3-4周：傳輸層與網絡層
- **Day 1-3**: [[TCP協議深入]]
  - 三次握手/四次揮手
  - 序列號和確認號
  - 滑動視窗與流量控制
  - 慢啟動與擁塞控制

- **Day 4-5**: [[UDP協議]]
  - UDP 的特點
  - 適用場景
  - TCP vs UDP 對比

- **Day 6-7**: [[IP協議與路由]]
  - IP 數據包格式
  - 路由選擇算法
  - 靜態路由 vs 動態路由

#### 第5周：數據鏈路層
- **Day 1-2**: [[Ethernet]] 與 [[MAC地址]]
  - 以太網幀格式
  - MAC 地址與 ARP 協議

- **Day 3-4**: [[VLAN]] 與 [[交換機]]
  - VLAN 劃分
  - 交換機工作原理
  - STP 生成樹協議

- **Day 5-7**: 實踐專案
  - 使用 Wireshark 抓包分析 TCP 連接
  - 配置 VLAN 並測試通信

#### 🎯 階段二驗收
- [ ] 能詳細描述 TCP 三次握手過程
- [ ] 能解釋 TCP 擁塞控制機制
- [ ] 能看懂 Wireshark 抓到的 TCP 數據包
- [ ] 理解 ARP 協議的作用

---

### 階段三：網絡架構（2-3周）

> 🎓 目標：了解不同網絡架構的設計與應用

#### 第6-7周：局域網與廣域網
- **Day 1-3**: [[局域網(LAN)]]
  - 以太網標準 (10/100/1000/10000 Mbps)
  - 交換機配置基礎
  - 冗餘設計 (VRRP, HSRP)

- **Day 4-5**: [[廣域網(WAN)]]
  - 專線技術
  - MPLS 概念
  - [[VPN虛擬專用網絡]] 基礎

- **Day 6-7**: 實踐專案
  - 設計一個小型企業 LAN 拓撲

#### 第8周：無線與雲端網絡
- **Day 1-2**: [[無線網絡(Wi-Fi)]]
  - 802.11 標準演進
  - WEP/WPA/WPA2/WPA3
  - 無線網絡設計要點

- **Day 3-5**: [[雲端計算網絡]]
  - 虛擬私有雲端 (VPC)
  - 容器網絡 (Docker/K8s)
  - [[SDN軟體定義網絡]] 概念

#### 🎯 階段三驗收
- [ ] 能設計基本的 LAN 拓撲
- [ ] 了解不同無線標準的區別
- [ ] 理解雲端計算網絡的虛擬化特性

---

### 階段四：安全與運維（2-3周）

> 🎓 目標：掌握網絡安全與運維的基本技能

#### 第9-10周：網絡安全基礎
- **Day 1-3**: [[網絡安全基礎]]
  - CIA 三元組
  - 常見威脅類型
  - 安全策略制定

- **Day 4-5**: [[加密與認證]]
  - 對稱加密 vs 非對稱加密
  - 數字簽名
  - PKI 體系
  - TLS/SSL 協議

- **Day 6-7**: [[防火牆技術]]
  - 包過濾防火牆
  - 狀態檢測防火牆
  - 防火牆規則設計

#### 第11-12周：網絡監控與故障排查
- **Day 1-2**: [[網絡監控]]
  - SNMP 協議
  - 常見監控指標
  - 監控工具選擇

- **Day 3-4**: [[故障排查]]
  - 分層排查法
  - 常見故障案例
  - 故障排查工具綜合使用

- **Day 5-7**: 實踐專案
  - 配置基礎監控系統
  - 排查模擬的網絡故障

#### 🎯 階段四驗收
- [ ] 能識別基本的網絡安全威脅
- [ ] 理解 HTTPS 加密原理
- [ ] 具備網絡故障排查的基本思路

---

### 階段五：進階實戰（持續）

> 🎓 目標：綜合應用，深入特定領域

#### 專項深入學習
根據興趣選擇方向：
- 🌐 **雲端計算網絡**: 深入學習 AWS/阿里雲端網絡服務
- 🔒 **網絡安全**: 深入學習滲透測試、安全加固
- 🤖 **SDN/NFV**: 學習軟體定義網絡技術
- 📊 **網絡自動化**: 學習 Ansible/Python 網絡編程
- 🌍 **大規模網絡**: 學習 BGP、MPLS 等運營商技術

#### 專案實踐建議
1. **搭建家庭實驗室**
   - 搭建虛擬網絡環境
   - 搭建監控系統
   - 設計網絡拓撲
   - 實施 VLAN

2. **參與開源項目**
   - 貢獻網絡開源工具
   - 學習最佳實踐

3. **效能優化專案**
   - 分析網絡性能瓶頸
   - 實施負載均衡

## 🛠️ 實踐工具清單

學習過程中需要掌握的工具：

### 基礎診斷
- ✅ `ping` - 連通性測試
- ✅ `traceroute` - 路由追蹤
- ✅ `nslookup`/`dig` - DNS查詢
- ✅ `netstat`/`ss` - 端口狀態

### 協議分析
- ✅ Wireshark - 圖形界面抓包
- ✅ tcpdump - 命令行抓包

### 網絡模擬
- ✅ Cisco Packet Tracer
- ✅ GNS3

### 監控工具
- ✅ Nagios
- ✅ Zabbix
- ✅ Prometheus + Grafana

詳見：[[06-實踐工具]]
