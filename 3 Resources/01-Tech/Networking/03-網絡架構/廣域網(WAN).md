## 🌍 WAN基礎

### 定義

WAN 是連接遠距離地理區域的電信網路，通常由電信運營商提供。

### WAN特點

| 特點 | 說明 |
|------|------|
| 地理範圍 | 跨城市、國家、大陸 |
| 傳輸速率 | 低至高，取決於技術 |
| 延遲 | 通常較高 |
| 成本 | 高（需租用線路） |
| 可靠性 | 取決於運營商 |

### WAN應用場景

- 企業多個辦公室之間連接
- 分支機構與總部連接
- 雲端訪問
- 網際網接入

## 🔧 WAN技術

### 專線技術 (Dedicated Line)

| 技術 | 速度 | 特點 |
|------|------|------|
| T1 | 1.544 Mbps | 北美標準 |
| E1 | 2.048 Mbps | 歐洲標準 |
| DS3 | 44.736 Mbps | 企業級 |
| OC-3 | 155.52 Mbps | 運營商級 |
| 光纖專線 | 100Mbps-10Gbps+ | 高速、低延遲 |

### MPLS (Multiprotocol Label Switching)

**定義**：MPLS 是一種在IP網路中標籤數據包的技術。

**優勢**：
- 流量工程：控制數據包路徑
- VPN支持：易於構建企業虛擬專用網
- QoS支持：優先級處理流量
- 速度快：交換效率高

**MPLS標籤**：
```
┌─────────────────────────────────┐
│ 標籤 (20 bits)          │
├─────────────────────────────────┤
│ 實驗值 (3) │ TTL (1) │ EXP (3) │ 標籤值 (13) │
└─────────────────────────────────┘
```

### VPN技術

**定義**：VPN (Virtual Private Network) 在公網上構建安全的專用網絡。

**VPN類型**：
1. **Site-to-Site VPN**：連接多個網站
2. **Remote Access VPN**：移動用戶接入企業網絡
3. **SSL VPN**：通過瀏覽器接入
4. **IPSec VPN**：網絡層加密

### 負載均衡

**定義**：將流量分佈到多個鏈路，提高可用性和性能。

**貢載均衡類型**：
1. **基於鏈路**：按輪詢、哈希等分配
2. **基於應用程式**：根據應用程式類型分配
3. **基於策略**：根據自定義規則分配

## 🔄 WAN vs LAN

| 特性 | LAN | WAN |
|------|-----|-----|
| 地理範圍 | 房間、建築物、校園 | 城市、國家、大陸 |
| 所有權 | 企業擁有 | 運營商提供 |
| 速率 | 高（100Mbps-100Gbps） | 變化（Kbps-10Gbps） |
| 成本 | 低（一次性） | 高（月租費） |
| 管理 | 企業IT部門 | 運營商+企業IT |
| 協議 | Ethernet | ATM, Frame Relay, MPLS, SDH |

## 🏗️ WAN設計

### WAN拓撲

#### 星型拓撲

```
        [總部]
            │
     ┌────┴────┐
     │         │
[分支1] [分支2] [分支3]
```

**優勢**：
- 結構簡單
- 故障隔離
- 易於管理

**劣勢**：
- 中心點單點故障
- 依賴中心鏈路容量

#### 網狀拓撲

```
        [分支1] ────────┐
         │           │
        [分支2] ──── [中心點] ──── [分支4]
         │           │           │
        [分支3] ────────┘           │
                                    │
                                [運營商]
```

**優勢**：
- 冗餘路徑
- 高可用性

**劣勢**：
- 結構複雜
- 成本高

### 冗餘設計

#### 冗餘鏈路

- **主備鏈路**：主鏈路故障時切換到備鏈路
- **負載分担**：正常時兩條鏈路同時使用

#### 冗餘設備

- **雙路由器**：兩台路由器作為網關
- **VRRP/HSRP**：虛擬路由器冗餘協議
- **雙ISP**：連接兩個不同運營商

### 路由選擇

**靜態路由**：
- 手動配置路由
- 適用於小型、穩定網路

**動態路由**：
- 協議自動學習路由
- 適用於中大型網路
- 協議：OSPF, BGP, RIP
