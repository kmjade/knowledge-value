# 交换机 (Switch)

## 📚 简介

交换机是计算机网络中用于连接多个设备并转发数据包的网络设备。它工作在OSI模型的数据链路层（第2层），根据MAC地址进行数据转发。

## 📝 核心概念

### 基本功能
- **MAC地址学习**: 学习连接设备的MAC地址
- **数据转发**: 根据MAC地址表转发数据帧
- **广播风暴控制**: 防止广播风暴
- **VLAN支持**: 虚拟局域网划分

### 交换机类型

| 类型 | 描述 | 应用场景 |
|------|------|----------|
| 二层交换机 | 基于MAC地址转发 | 办公室、小型网络 |
| 三层交换机 | 支持IP路由 | 企业核心网络 |
| 管理型交换机 | 支持SNMP、CLI管理 | 企业网络 |
| 非管理型交换机 | 即插即用 | 家庭、小型办公室 |
