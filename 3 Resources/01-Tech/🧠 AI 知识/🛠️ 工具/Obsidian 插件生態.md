# Obsidian 插件生態

## 定義

Obsidian 擁有豐富的社區插件市場（1000+ 插件），可根據需求自定義功能，支持 API 和 Webhook 集成。

## 插件類型

### 核心插件
社區中最流行和功能強大的插件，包括：

- [[Obsidian Dataview 插件]] - 強大的查詢和數據可視化
- [[Obsidian Templater 插件]] - 模板引擎，支持 JavaScript 腳本
- [[Obsidian Tasks 插件]] - 任務管理插件
- [[Obsidian Local REST API]] - 提供 HTTP API 訪問筆記
- [[Obsidian Flashcards 插件]] - 閃念學習插件

### 官方插件
- **Daily Notes**: 每日日記自動創建
- **Backlinks**: 反向連結面板
- **Outliner**: 大綱視圖
- **Graph View**: 知識網絡視圖
- **File Recovery**: 版本歷史

## 插件管理

### 安裝插件
1. 打開設置 → 社區插件
2. 瀏覽或搜索插件
3. 一鍵安裝並啟用

### 插件性能
- ✅ 只安裝必需的插件
- ✅ 定期檢查插件更新
- ❌ 避免安裝過多插件影響性能
- 詳見：[[Obsidian 性能優化]]

## 最佳實踐

### 插件選擇
- 明確需求後再安裝
- 優先選擇活躍維護的插件
- 查看插件評分和用戶反饋

### 插件配置
- 仔細閱讀插件文檔
- 根據個人需求調整設置
- 備份重要插件配置
