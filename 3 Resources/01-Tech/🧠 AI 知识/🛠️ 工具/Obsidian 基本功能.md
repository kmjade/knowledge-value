# Obsidian 基本功能

## 筆記管理

### 快速寫作
- 支持實時預覽和 Markdown 語法
- 自動保存，從不遺失
- 所見即所得的編輯體驗

### 標籤系統
- 使用 `#標籤名` 創建分類
- 每個筆記可添加多個標籤
- 支持標籤嵌套：`#tag1/subtag2`
- 詳見：[[Obsidian 搜索優化]]

### 文件夾組織
- 創建邏輯化的文件夾結構
- 支持文件夾嵌套和並行組織
- 建議使用 PARA 方法論

### 屬性面板
- 每個筆記可添加自定義屬性
- 支持創建日期、修改日期、狀態等屬性
- 使用 YAML frontmatter 格式

## 知識管理

### 反向連結
- 查看哪些筆記連結到當前筆記
- 在狀態欄顯示反向連結數量
- 快速了解知識關聯
- 詳見：[[Obsidian 雙向連結系統]]

### 大綱視圖
- 將相關筆記組織成大綱
- 可折疊展開，查看子主題
- 視覺化知識結構

### 圖表視圖
- 將筆記轉換為表格視圖
- 按自定義列顯示筆記屬性
- 支持排序、篩選、分組
