# 設定最佳實務做法

> [!info] 目標
> 建立一個高效、穩定的 WSL 開發環境。

## 安裝後設定清單

```mermaid
flowchart LR
    A[安裝 WSL] --> B[更新系統]
    B --> C[安裝開發工具]
    C --> D[設定 Git]
    D --> E[設定 VS Code]
    E --> F[優化設定]
```

## 1. 系統更新

### 更新 Linux 發行版

```bash
# Ubuntu/Debian
sudo apt update && sudo apt upgrade -y

# 安裝建議套件
sudo apt install build-essential git curl wget unzip -y
```

### 更新 WSL

```bash
# 在 Windows PowerShell 執行
wsl --update
```

## 2. 檔案系統最佳實務

### 專案位置

> [!tip] 重要
> 將專案放在 Linux 檔案系統內以獲得最佳效能。

```bash
# ✅ 推薦 - Linux 檔案系統
cd ~
mkdir projects
cd projects

# ❌ 避免 - 從 Linux 存取 Windows 檔案
cd /mnt/c/Users/username/projects
```

### 效能比較

| 位置 | 效能 | 說明 |
|------|------|------|
| `~/projects` | ⭐⭐⭐⭐⭐ | WSL 2 Linux 檔案系統 |
| `/mnt/c/` | ⭐⭐ | 跨檔案系統存取 |

### 從 Windows 存取 Linux 檔案

```bash
# 在檔案總管中開啟
explorer.exe .

# 取得 Linux 路徑的 Windows 路徑
wslpath -w /home/user/projects
# 輸出: \\wsl$\Ubuntu\home\user\projects
```

## 3. 終端機設定

### Windows Terminal 設定

推薦設定 (`settings.json`)：

```json
{
    "profiles": {
        "defaults": {
            "font": {
                "face": "Cascadia Code",
                "size": 12
            },
            "colorScheme": "One Half Dark",
            "cursorShape": "bar",
            "startingDirectory": "~"
        }
    }
}
```

### 安裝更好的 Shell

```bash
# 安裝 Zsh
sudo apt install zsh -y

# 安裝 Oh My Zsh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# 或安裝 Fish
sudo apt install fish -y
chsh -s /usr/bin/fish
```

## 4. Git 設定

### 基本設定

```bash
# 設定使用者資訊
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

# 設定預設分支名稱
git config --global init.defaultBranch main

# 設定憑證管理
git config --global credential.helper /mnt/c/Program\ Files/Git/mingw64/bin/git-credential-manager.exe
```

### SSH 金鑰設定

```bash
# 產生 SSH 金鑰
ssh-keygen -t ed25519 -C "your@email.com"

# 啟動 SSH Agent
eval "$(ssh-agent -s)"

# 加入金鑰
ssh-add ~/.ssh/id_ed25519

# 複製公鑰
cat ~/.ssh/id_ed25519.pub | clip.exe
```

## 5. VS Code 整合

### 安裝遠端開發擴充功能

## 6. WSL 設定優化

### .wslconfig 設定

建立 `%UserProfile%\.wslconfig`：

```ini
[wsl2]
# 記憶體限制
memory=8GB

# 處理器數量
processors=4

# 置換空間
swap=2GB

# 本地化主機
localhostForwarding=true

# 系統呼叫相容性
syscall=true
```

### /etc/wsl.conf 設定

```ini
[boot]
# 啟用 systemd
systemd=true

[automount]
# 掛載選項
options = "metadata,umask=22,fmask=11"

[network]
# 主機名稱
hostname = wsl

[interop]
# 啟用 Windows 程序整合
enabled = true
appendWindowsPath = true
```

## 7. 開發環境設定

### Python 開發

```bash
# 安裝 pyenv
curl https://pyenv.run | bash

# 安裝 Python
pyenv install 3.12.0
pyenv global 3.12.0

# 安裝 Poetry (套件管理)
curl -sSL https://install.python-poetry.org | python3 -
```

### Node.js 開發

```bash
# 安裝 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安裝 Node.js
nvm install --lts
nvm use --lts

# 安裝 pnpm
npm install -g pnpm
```

### Go 開發

```bash
# 安裝 Go
wget https://go.dev/dl/go1.21.0.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.21.0.linux-amd64.tar.gz

# 設定 PATH
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
```

## 8. 容器開發

### Docker 設定

```bash
# 安裝 Docker CLI
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

# 將使用者加入 docker 群組
sudo usermod -aG docker $USER
```

### 檢查 WSL 資源使用

```bash

free -h

# 磁碟使用
df -h

# Windows 工作
# 在 PowerShell 中
wsl --status
```
