# Obsidian 核心特性

## 定義

# 管理

## 核心特性

### 1. 本地 Markdown 筆記
- 所有筆記都以純文本 Markdown 格式存儲
- 您擁有完全的數據所有權
- 可以隨時離線訪問，無需網絡連接

### 2. 雙向連結
- 使用 `[[筆記名稱]]` 語法創建筆記間的連結
- 類似大腦的神經網絡，知識相互關聯
- 可快速跳轉到相關筆記
- 詳見：[[Obsidian 雙向連結系統]]

### 3. 可擴展性
- 豐富的社區外掛市場（1000+ 外掛）
- 可根據需求自定義功能
- 支持 API 和 Webhook 整合
- 詳見：[[Obsidian 外掛生態]]

### 4. 跨平台支持
- Windows、macOS、Linux、iOS、Android
- 多設備同步（Obsidian Sync、Git、第三方）

### 5. 多視圖模式
- 文檔視圖、圖表視圖、看板視圖
- 可自定義主題、外掛、CSS
