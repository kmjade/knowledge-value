## 📌 原始摘录

### 核心內容

#### AGENTS.md：给AI的工作手册

还记得我们刚才創建的 `AGENTS.md` 檔案吗？

這個檔案非常非常重要，决定你的智能体是否跟你合得来。

你需要赋予它：

> - 你叫什么名字
> - 你是什么性格
> - 檔案该放哪里
> - 有哪些规矩必须遵守

# 更新

这是你给你的智能体赋予的最强脑力。这里建議設計的时候加上"最强大脑"几个字，别问为什么，问就是大佬们都在用。效果就是好。

#### 完整模板

如果你想让 AI 更懂你，用這個完整版：

```
# AI 管家工作規範
```

#### 个性化定制

AGENTS.md这东西就是定义你跟谁对话，無限定制，你可以随便改。

想让AI说话带点幽默感？加一句：

```
## 性格
```

想让AI模仿某个风格？直接写：

```
## 交互风格
```

想让AI專注某个领域？指定：

```
## 专业领域
```

AI 会讀取這個檔案，然后按你说的来。

以下是我个人左脑模板，情绪價值拉满！(其实有的內容是不用写的，但是装了不少的skill外掛后，会对当前的skill目錄進行权限封定，导致自然讀取失败，因此锚定到记忆中，指令遵循更好。)
