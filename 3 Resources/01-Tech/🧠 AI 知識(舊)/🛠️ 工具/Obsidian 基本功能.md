# Obsidian 基本功能

# 管理

### 快速寫作
- 支持實時預覽和 Markdown 語法
- 自動儲存，從不遺失
- 所見即所得的編輯體驗

### 標籤系統
- 使用 `#標籤名` 創建分類
- 每個筆記可新增多個標籤
- 支持標籤嵌套：`#tag1/subtag2`
- 詳見：[[Obsidian 搜尋優化]]

### 檔案夾組織
- 創建邏輯化的檔案夾結構
- 支持檔案夾嵌套和並行組織
# 方法

### 屬性面板
- 每個筆記可新增自定義屬性
# 修改
- 使用 YAML frontmatter 格式

### 反向連結
# 查看

### 大綱視圖
- 將相關筆記組織成大綱
# 查看

- 按自定義列顯示筆記屬性
