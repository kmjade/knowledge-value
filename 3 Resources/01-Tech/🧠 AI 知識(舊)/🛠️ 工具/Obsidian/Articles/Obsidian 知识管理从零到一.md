# 指南

### 步骤 1：安裝 Obsidian
1. 前往官方站点下載并安裝。
2. 創建本地 Vault（資料夾），建議命名为 `SecondBrain`。

### 步骤 2：搭建基礎结构（PARA）
```
SecondBrain/
├─ 0 Inbox/
├─ 1 Projects/
├─ 2 Areas/
├─ 3 Resources/
└─ 4 Archives/
```
- 在每个資料夾下創建 README 简要說明。

### 步骤 3：遷移核心筆記
- 选取 **3‑5** 篇最重要的筆記，放入对应資料夾。
- 为每篇筆記新增 Frontmatter（參考 `_templates/standard_note.md`）并打上標籤。

### 步骤 4：建立双链与 MOC
# 知識
- 在每篇筆記里加入相關双链，逐步構建網路。

### 步骤 5：安裝核心外掛
- 在外掛市场搜尋并启用：Templater、Dataview、Calendar、Excalidraw、Tag Wrangler。
# 配置

### 步骤 6：日常使用 & 回顧
- **每日**：在 `0 Inbox` 捕获灵感、任務。
# 更新
- **每月**：合并重复筆記，歸檔完成專案。

| 問題 | 解決方案 |
|------|----------|
# 管理
| 是否需要編程能力？ | 不需要。双链、標籤、MOC 都是原生功能。高级功能如 Dataview 查詢、Templater 腳本可逐步學習。 |
| 如何遷移已有筆記？ | 采用分批遷移：先遷移價值最高的筆記，保持原始格式，随后在 Obsidian 中新增双链和標籤。 |
