# Obsidian 外掛生態

## 定義

Obsidian 擁有豐富的社區外掛市場（1000+ 外掛），可根據需求自定義功能，支持 API 和 Webhook 整合。

## 外掛類型

### 核心外掛
社區中最流行和功能強大的外掛，包括：

- [[Obsidian Dataview 外掛]] - 強大的查詢和數據可視化
- [[Obsidian Templater 外掛]] - 模板引擎，支持 JavaScript 腳本
# 管理
- [[Obsidian Local REST API]] - 提供 HTTP API 訪問筆記
- [[Obsidian Flashcards 外掛]] - 閃念學習外掛

### 官方外掛
- **Daily Notes**: 每日日記自動創建
- **Backlinks**: 反向連結面板
- **Outliner**: 大綱視圖
- **Graph View**: 知識網絡視圖

# 管理

### 安裝外掛
1. 打開設置 → 社區外掛
2. 瀏覽或搜尋外掛
3. 一鍵安裝並啟用
