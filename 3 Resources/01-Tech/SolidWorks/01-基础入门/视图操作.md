# 視圖操作

> [!summary] 功能概述
> 視圖操作是 SolidWorks 中最常用的基本操作，包括旋轉、縮放、平移視圖，以及視圖方向和顯示模式的切換。

---

## 鼠標操作

### 三鍵鼠標操作

| 操作 | 功能 | 說明 |
|------|------|------|
| 中鍵按住拖動 | 旋轉視圖 | 繞模型中心旋轉 |
| 中鍵 + Shift + 拖動 | 平移視圖 | 移動視圖位置 |
| 中鍵 + Ctrl + 拖動 | 縮放視圖 | 放大/縮小視圖 |
| 中鍵滾輪 | 縮放 | 快速縮放視圖 |
| 中鍵點擊 | 縮放全部 | 顯示所有內容 |

### 鼠標手勢

> [!tip] 鼠標手勢
> 按住右鍵拖動可觸發鼠標手勢，快速訪問常用命令。

---

## 視圖方向

### 標準視圖

| 視圖 | 快捷鍵 | 說明 |
|------|--------|------|
| 前視 | Ctrl+1 | 正面視圖 |
| 後視 | Ctrl+2 | 背面視圖 |
| 左視 | Ctrl+3 | 左側視圖 |
| 右視 | Ctrl+4 | 右側視圖 |
| 上視 | Ctrl+5 | 頂部視圖 |
| 下視 | Ctrl+6 | 底部視圖 |
| 等軸測 | Ctrl+7 | 三維等軸測視圖 |

### 視圖工具欄

- **視圖定向**：打開視圖方向對話框
- **上一視圖**：返回上一個視圖
- **正視於**：將所選平面正對屏幕

---

## 顯示模式

### 顯示樣式

| 模式 | 說明 |
|------|------|
| 帶邊線著色 | 顯示面顏色和邊線 |
| 著色 | 僅顯示面顏色 |
| 消除隱藏線 | 隱藏不可見邊線 |
| 隱藏線可見 | 顯示所有邊線 |
| 線架圖 | 僅顯示邊線 |

### 顯示選項

- **陰影**：顯示模型陰影
- **透視圖**：透視顯示效果
- **剖面視圖**：剖切顯示模型內部

---

## 視圖控制

### 縮放操作

| 操作 | 說明 |
|------|------|
| 局部放大 | 框選區域放大 |
| 放大/縮小 | 動態縮放 |
| 放大所選範圍 | 放大選中對象 |
| 整屏顯示全圖 | 顯示全部內容 |

### 旋轉操作

- **旋轉視圖**：自由旋轉
- **旋轉 about 屏幕中心**：以屏幕中心旋轉

---

## 注意事項

> [!tip] 最佳實踐
> - 熟練掌握鼠標操作可大幅提高效率
> - 常用視圖方向可設置快捷鍵
> - 使用空格鍵快速打開視圖方向面板
