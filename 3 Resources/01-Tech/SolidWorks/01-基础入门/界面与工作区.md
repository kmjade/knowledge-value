# 界面與工作區

> [!summary] 功能概述
> SolidWorks 用戶界面是進行三維建模的工作環境，包含菜單欄、工具欄、特徵管理器、圖形區域等核心組件。

---

## 功能介紹

### 功能定位
- **所屬模塊**：基礎入門
- **功能類型**：界面認知
- **前置條件**：無

### 應用場景
1. 初次接觸 SolidWorks 時熟悉工作環境
2. 自定義工作界面以提高工作效率
3. 切換不同的工作模式（零件、裝配、工程圖）

---

## 界面組成

### 主要區域

```
┌─────────────────────────────────────────────────────────────┐
│ 菜單欄 (Menu Bar)                                           │
├─────────────────────────────────────────────────────────────┤
│ 標準工具欄 (Standard Toolbar)                                │
├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│ 特徵     │                                                  │
│ 管理器   │              圖形區域                             │
│ (Feature │            (Graphics Area)                       │
│ Manager) │                                                  │
│          │                                                  │
│ ├ 設計樹 │                                                  │
│ ├ 屬性   │                                                  │
│ └ 配置   │                                                  │
│          │                                                  │
├──────────┴──────────────────────────────────────────────────┤
│ 狀態欄 (Status Bar)                                         │
└─────────────────────────────────────────────────────────────┘
```

### 1. 菜單欄 (Menu Bar)
- **文件 (File)**：新建、打開、保存、打印
- **編輯 (Edit)**：撤銷、重做、剪切、複製、粘貼
- **視圖 (View)**：顯示設置、工具欄配置
- **插入 (Insert)**：添加特徵、參考幾何體
- **工具 (Tools)**：草圖工具、測量、檢查
- **窗口 (Window)**：窗口排列、切換
- **幫助 (Help)**：文檔、教程

### 2. 命令管理器 (CommandManager)
- 根據當前工作模式動態顯示相關工具
- **零件模式**：特徵、草圖、評估、DimXpert
- **裝配模式**：裝配、佈局、草圖、評估
- **工程圖模式**：視圖佈局、註解、草圖、評估

### 3. 特徵管理器 (FeatureManager)
- **設計樹**：顯示模型的特徵歷史
- **屬性管理器**：顯示當前操作的屬性設置
- **配置管理器**：管理零件的不同配置

### 4. 圖形區域 (Graphics Area)
- 三維模型顯示區域
- 支持鼠標操作（旋轉、縮放、平移）

---

## 工作區類型

### 零件 (Part)
- 創建單個零件的三維模型
- 包含草圖、特徵等建模工具

### 裝配體 (Assembly)
- 組裝多個零件
- 包含配合、干涉檢查等工具

### 工程圖 (Drawing)
- 從三維模型生成二維圖紙
- 包含視圖、標註、注解工具

---

## 注意事項

> [!warning] 重要提示
> - 界面佈局可以自定義，建議初學者先使用默認佈局
> - 不同工作模式下的工具欄會有所不同

> [!tip] 最佳實踐
> - 使用快捷鍵 S 打開快捷工具欄
> - 右鍵點擊工具欄可以自定義顯示的工具
