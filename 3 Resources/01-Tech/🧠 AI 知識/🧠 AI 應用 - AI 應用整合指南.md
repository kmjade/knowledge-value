## 知識體系

### AI 領域地圖

```
AI 知識體系
├── 1. 基礎知識 Base Knowledge
│   ├── 編程語言（Python、JavaScript、Java）
│   ├── 數學（線性代數、概率論）
│   ├── 統計機構（統計、機器學習）
│   └── 系統設計（架構模式、微服務）
│
├── 2. 深度學習 Deep Learning
│   ├── 深度學習 Deep Learning
│   ├── 生成模型 Generative Models
│   ├── 傳強學習 Reinforcement Learning
│   └── 轉移學習 Transfer Learning
│
├── 3. AI 應用 AI Applications
│   ├── LLM 應用 Large Language Models
│   ├── RAG（檢索增強生成）
│   ├── Agent 開發 Multi-Agent Systems
│   └── 工具集成 Tool Integration
│
└── 4. AI 產品開發
│       ├── Prompt Engineering
│       ├── Fine-tuning
│       ├── 部署與優化
│       └── 安全與合規
```

## 學習路徑

### 路徑一：基礎 → 應用（適合初學者）

**目標人群**：AI 初學者、剛開始學習的開發者
**時間安排**：3-6個月
**重點**：快速上手，實踐項目

**學習步驟**：
1. **月1-2月**：Python 基礎 + LangChain
   - [ ] 熟悉 Python 基礎語法
   - [ ] 學習 LangChain 快速入門
   - [ ] 實現第一個 AI 應用

2. **月3-4月**：Python 深度 + AI 概念
   - [ ] 學習 Python 進階
   - [ ] 理解 LLM 工作原理
   - [ ] 實現 RAG 應用

3. **月5-6月**：實踐項目 + 深度
   - [ ] 完成一個完整的 AI 應用項目
   - [ ] 學習向量數據庫
   - [ ] 部署到雲端
