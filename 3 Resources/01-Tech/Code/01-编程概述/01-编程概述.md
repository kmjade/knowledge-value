# 01-编程概述

> 编程是将人类意图转化为计算机可执行指令的过程。DDC 005 涵盖编程的全景。

## 编程范式

| 范式 | 核心思想 | 代表 |
|------|----------|------|
| **命令式** | 告诉计算机"怎么做"——一步步指令 | C, Python |
| **面向对象** | 数据+行为封装为对象 | Java, C++ |
| **函数式** | 计算=数学函数求值, 无副作用 | Haskell, Scala |
| **声明式** | 描述"要什么"而非"怎么做" | SQL, HTML |
| **响应式** | 数据流+变化传播 | RxJS, SwiftUI |

## 编译 vs 解释

| | 编译型 | 解释型 | JIT |
|---|--------|--------|-----|
| 过程 | 源码→机器码→执行 | 逐行执行 | 热点编译 |
| 速度 | 快 | 慢 | 接近编译 |
| 代表 | C, Rust, Go | Python, Ruby | Java, JS(V8) |
