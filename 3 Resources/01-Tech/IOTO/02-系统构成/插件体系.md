# 插件体系

> [!summary] 概述
> IOTO 包含 5 個配套插件：3 個基礎插件 + 2 個增強插件，共同支撐知識生產工作流。

---

## 插件概覽

```
IOTO 插件體系
├── 基礎插件 (Essential)
│   ├── IOTO Update        # 框架部署中樞
│   ├── Framework Settings # 配置中心
│   └── IOTO Dashboard     # 生產大屏
│
└── 增強插件 (Advanced)
    ├── Template Generator # 模板編輯器
    └── Sync Scripts Generator # 同步腳本
```

---

## 基礎插件

### IOTO Update

**定位**：框架部署中樞

> [!info] 版本信息
> - **最新版本**: v2.0.7
> - **更新日期**: 2026-04-24
> - **GitHub**: [shawndotty/ioto-update](https://github.com/shawndotty/ioto-update)

| 功能 | 說明 |
|------|------|
| 更新 IOTO Framework | 保持框架最新版本 |
| 更新同步模板 | 同步個人模板配置 |
| 一鍵下載 | 模板、腳本、CSS 片段 |
| 熱更新 | 保持生產環境最新 |
| 樣式注入 | 自動應用 IOTO 樣式 |
| 插件管理 | 確保依賴插件安裝 |

**核心價值**：確保生產環境始終最新，無需手動維護。

---

### Framework Settings

**定位**：IOTO 配置中心

| 功能 | 說明 |
|------|------|
| 目錄初始化 | 一鍵創建 IOTO 文件夾結構 |
| 語言切換 | 中英文界面切換 |
| 熱鍵自動配置 | 快捷鍵自動映射 |
| 項目創建 | 快速創建新項目 |
| 打開任務列表 | 快速訪問當前任務 |

**核心價值**：集中管理所有配置，降低使用門檻。

---

### IOTO Dashboard

**定位**：IOTO 生產大屏

> [!info] 版本信息
> - **最新版本**: v2.0.3
> - **更新日期**: 2026-03-23
> - **GitHub**: [shawndotty/ioto-dashboard](https://github.com/shawndotty/ioto-dashboard)

| 功能 | 說明 |
|------|------|
| 工作流導航 | Input/Output/Task/Outcome 階段切換 |
| 雙視圖模式 | 筆記視圖 + 任務視圖 |
| 動態過濾 | 多維過濾項目、日期、狀態 |
| Subject 過濾 | v2.0.3 新增，按主題過濾 |
| 多語言支持 | 英文/簡中/繁中 |

**核心價值**：知識資產分布和狀態一目了然。

---

## 增強插件

### Template Generator

**定位**：GUI 模板編輯器

> [!info] 版本信息
> - **最新版本**: v2.0.5
> - **更新日期**: 2026-04-17
> - **GitHub**: [shawndotty/ioto-template-generator](https://github.com/shawndotty/ioto-template-generator)

| 功能 | 說明 |
|------|------|
| 可視化編輯 | 無需編寫代碼 |
| 自定義模板 | 創建專屬模板 |
| 模板管理 | 統一管理所有模板 |
| 快捷鍵配置 | 自動關聯快捷鍵 |
| Plan 模式 | v2.0.5 新增支持 |

**核心價值**：個性化定制，適應獨特需求。

---

### Sync Scripts Generator

**定位**：多平台同步配置引擎

> [!info] 版本信息
> - **最新版本**: v2.0.7
> - **更新日期**: 2026-04-24
> - **GitHub**: [shawndotty/sync-script-generator](https://github.com/shawndotty/sync-script-generator)

| 功能 | 說明 |
|------|------|
| 同步腳本生成 | 為 IOTO 和 OB Sync With MDB 生成腳本 |
| Notion 同步 | 支持 Notion 數據庫同步 |
| 多工作空間支持 | v2.0.7 新增，Notion 多工作空間 |
| 高階同步配置 | 細粒度控制 |

**核心價值**：打通 Obsidian 與雲端數據庫的聯動通道。

## 安裝順序

### 推薦安裝流程

```
1. 安裝 IOTO Update
   └→ 自動安裝依賴插件

2. 運行 Framework Settings
   └→ 初始化目錄結構
   └→ 配置快捷鍵

3. 安裝 IOTO Dashboard
   └→ 可選，按需安裝

4. 安裝增強插件（進階）
   └→ Template Generator
   └→ Sync Scripts Generator
```

---

## 插件協作關係

```
IOTO Update ──下載──→ Templates/Scripts
       │
       ↓
Framework Settings ──初始化──→ 文件夾結構
       │                      │
       ↓                      ↓
IOTO Dashboard ←──讀取──→ 筆記/任務數據
       │
       ↓
Template Generator ──定制──→ 自定義模板
       │
       ↓
Sync Scripts Generator ──同步──→ 雲端
```
