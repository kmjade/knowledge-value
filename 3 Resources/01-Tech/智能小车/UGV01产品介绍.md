# UGV01 产品介绍

> [!info] 产品概述
> UGV01是一款具备超强的越野通过能力和抗震性能、开源全部代码可二次开发的移动机器人履带底盘。

## 📦 产品特性

### 核心特性

| 特性 | 描述 |
|------|------|
| **多独立悬挂系统** | 每侧配有四个独立悬挂，大幅度减少复杂地形冲击 |
| **优秀爬坡能力** | 重负载爬坡时，悬挂收缩，后轮会提供额外支撑 |
| **编码器电机** | 电机带有编码器，可以获得速度反馈来进行速度闭环控制 |
| **OLED屏幕** | 搭载0.96英寸OLED屏幕，实时显示设备信息 |
| **边充边放** | 带有充电接口和自动下载电路，支持边充电边使用 |
| **电压电流监控** | UPS供电模块板载INA219采集芯片，方便实时监控 |
| **大容量电池** | 3S 18650电池，7800mAh大容量，输出电流更大，电机动力更强 |
| **多电压输出** | UPS供电模块提供5V和3.3V输出，用于扩展其它设备 |
| **电池保护** | 板载锂电池保护电路，具有防过充、防过放、防过流和短路保护功能 |

### 软件特性

- **Arduino IDE开发** - 例程代码使用Arduino IDE开发，无需手动配置编译环境
- **Web端控制** - ESP32开机自动建立WIFI热点，可使用手机或电脑连接并登录控制页面
- **无需APP** - 只需安装基于Chromium的浏览器即可，不需要下载app
- **独立使用** - 下位机ESP32可用于驱动直流电机、总线舵机等，即使不安装上位机也可以单独使用
- **上位机扩展** - 可扩展树莓派、Jetson Nano、地平线旭日X3等多种上位机
- **完全开源** - 全部代码开源并提供丰富的开发文档和教程
- **灵活通信** - JSON指令可通过串口、USB、HTTP、ESP-NOW等方式下达
- **配置灵活** - 绝大部分配置设置可由JSON指令实现，例如不需要连接USB线即可配置WIFI连接

## 🔧 驱动板详解

### General Driver for Robots

基于ESP32的多功能机器人驱动板，集成多种功能模块。

#### 核心模块

| 模块 | 功能 | 特性 |
|------|------|------|
| **WiFi/蓝牙** | 无线通信 | 支持AP和STA模式 |
| **OLED屏幕** | 信息显示 | 0.96英寸，4行显示 |
| **IMU模块** | 姿态感知 | 九轴传感器 |
| **SD卡槽** | 数据存储 | 支持TF卡 |
| **舵机接口** | 执行器控制 | 总线舵机、PWM舵机 |
| **GPIO** | 通用IO | IO4、IO5等 |
| **电机驱动** | 运动控制 | 带编码器闭环控制 |

#### 通信接口

| 接口 | 功能 | 连接方式 |
|------|------|----------|
| **UART** | 串口通信 | 连接上位机 |
| **USB** | USB通信 | 固件更新、通信 |
| **WiFi** | 无线网络 | Web控制、HTTP指令 |
| **蓝牙** | 短距离通信 | 设备互联 |
| **ESP-NOW** | 低延迟通信 | 多机协同 |

### 上位机扩展

#### 支持的上位机

- **树莓派** - 树莓派4B、树莓派5等
- **Jetson系列** - Jetson Nano、Jetson Orin Nano等
- **旭日系列** - 地平线旭日X3等

#### 连接方式

1. **40PIN UART接口** - 直接连接树莓派、Jetson Nano等
2. **USB接口** - 通过USB线连接（需要拆卸小车）

#### 通信方式

- 串口通信（UART）
- JSON指令交互
- 实时数据反馈

## 💡 使用建议

> [!tip] 首次使用
> 1. 检查固件版本
> 2. 安装电池（注意正负极）
> 3. 更新固件到最新版本
> 4. 测试基本功能

> [!tip] 扩展开发
> 1. 选择合适的上位机
> 2. 配置通信接口
> 3. 安装上位机程序
> 4. 开发自定义功能

> [!warning] 安全注意
> - 电池接反禁止充电
> - 推荐使用高放电倍率电池
> - 重载爬坡注意稳定性
> - Web端断开会自动停止运动
