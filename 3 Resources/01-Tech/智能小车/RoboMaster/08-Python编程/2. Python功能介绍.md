# 2. RoboMaster Python 功能介绍

> [!info] 本章概述
> 介绍 RoboMaster EP Python 编程的高级功能，包括多机通信系统和自定义 UI 系统。

## 2.1 多机通信系统

多机通信系统是一种基于局域网实现多台机器人之间相互通信的功能，让用户可以进行多台机器人协同工作，实现多台机器人编队表演等功能。

### 工作原理

```mermaid
flowchart LR
    R1[EP 机器人 1] <-->|局域网| R2[EP 机器人 2]
    R2 <-->|局域网| R3[EP 机器人 3]
    R1 <-->|局域网| R3
```

### 使用条件

| 条件 | 说明 |
|------|------|
| **网络环境** | 所有机器人加入同一局域网 |
| **编程方式** | 使用官方 Python API |
| **App 版本** | 支持多机通信功能 |

### 功能特性

- 基于局域网的实时通信
- 支持多台机器人协同工作
- 实现编队表演等功能

### 使用步骤

1. 将所有目标机器人加入到同一个局域网内
2. 使用官方提供的 Python API 进行编程
3. 定义机器人之间的通信机制
4. 实现具体的多机协同功能

## 2.2 多机通信 API

### 初始化通信

```python
import multi_comm

# 初始化多机通信
multi_comm.init()
```

### 发送消息

```python
# 发送消息给指定机器人
multi_comm.send(target_id, message)

# 广播消息给所有机器人
multi_comm.broadcast(message)
```

### 接收消息

```python
# 接收消息
def on_message_received(sender_id, message):
    print(f"收到来自 {sender_id} 的消息: {message}")

# 注册回调
multi_comm.on_recv(on_message_received)
```

## 2.3 多机协同示例

### 编队控制示例

```python
import robot_ctrl
import chassis_ctrl
import multi_comm

# 机器人 ID
ROBOT_ID = 1

def start():
    """初始化"""
    multi_comm.init()
    print(f"机器人 {ROBOT_ID} 就绪")

def receive_message(sender_id, message):
    """接收消息回调"""
    if message == "MOVE_FORWARD":
        chassis_ctrl.move(1, 0, 0, 0.7)
    elif message == "TURN_LEFT":
        chassis_ctrl.move(0, 0, 90, 45)

def main():
    """主程序"""
    # 注册消息接收回调
    multi_comm.on_recv(receive_message)
    
    # 主控机器人发送编队命令
    if ROBOT_ID == 1:
        # 所有机器人前进
        multi_comm.broadcast("MOVE_FORWARD")
        chassis_ctrl.move(1, 0, 0, 0.7)

def stop():
    """清理"""
    multi_comm.deinit()
```

### 协同任务示例

```python
import robot_ctrl
import chassis_ctrl
import gun_ctrl
import multi_comm
import time

ROBOT_ID = 1

def collaborative_task():
    """协同任务示例"""
    if ROBOT_ID == 1:
        # 主控机器人：发送指令
        multi_comm.send(2, "SCOUT")
        multi_comm.send(3, "GUARD")
        
        # 执行自己的任务
        chassis_ctrl.move(2, 0, 0, 0.5)
        
    elif ROBOT_ID == 2:
        # 侦查机器人
        chassis_ctrl.move(3, 0, 0, 0.7)
        gun_ctrl.fire_once()
        
    elif ROBOT_ID == 3:
        # 守卫机器人
        chassis_ctrl.move(0, 0, 90, 30)
```

## 2.4 自定义 UI 系统

RoboMaster EP 支持编程自定义 UI 系统，通过 Python 编写虚拟控件，自由设计交互界面，拓展无限应用可能。

### UI 控件类型

| 控件 | 说明 |
|------|------|
| **Button** | 按钮 |
| **Text** | 文本显示 |
| **Input** | 输入框 |
| **Dropdown** | 下拉菜单 |
| **Toggle** | 开关 |
| **Stage** | 画布 |

### 创建 UI 示例

```python
from ui import *

# 创建按钮
btn_forward = Button()
btn_forward.set_text("前进")
btn_forward.set_position(100, 100)
btn_forward.set_size(80, 40)

def on_btn_click():
    chassis_ctrl.move(1, 0, 0, 0.7)

btn_forward.on_click(on_btn_click)

# 创建文本
txt_status = Text()
txt_status.set_text("状态: 就绪")
txt_status.set_position(10, 10)

# 创建输入框
input_speed = Input()
input_speed.set_position(100, 200)
input_speed.set_size(150, 30)
```

### UI 交互示例

```python
from ui import *
import chassis_ctrl

class ControlPanel:
    """控制面板"""
    
    def __init__(self):
        # 创建控件
        self.btn_up = Button()
        self.btn_up.set_text("↑")
        self.btn_up.set_position(100, 50)
        self.btn_up.on_click(self.move_forward)
        
        self.btn_down = Button()
        self.btn_down.set_text("↓")
        self.btn_down.set_position(100, 150)
        self.btn_down.on_click(self.move_backward)
        
        self.btn_left = Button()
        self.btn_left.set_text("←")
        self.btn_left.set_position(50, 100)
        self.btn_left.on_click(self.turn_left)
        
        self.btn_right = Button()
        self.btn_right.set_text("→")
        self.btn_right.set_position(150, 100)
        self.btn_right.on_click(self.turn_right)
        
        self.txt_info = Text()
        self.txt_info.set_text("控制面板")
        self.txt_info.set_position(10, 10)
    
    def move_forward(self):
        chassis_ctrl.move(0.5, 0, 0, 0.5)
        self.txt_info.set_text("前进")
    
    def move_backward(self):
        chassis_ctrl.move(-0.5, 0, 0, 0.5)
        self.txt_info.set_text("后退")
    
    def turn_left(self):
        chassis_ctrl.move(0, 0, -45, 30)
        self.txt_info.set_text("左转")
    
    def turn_right(self):
        chassis_ctrl.move(0, 0, 45, 30)
        self.txt_info.set_text("右转")

# 启动面板
panel = ControlPanel()
```

## 常见问题

### 1. 多机通信延迟

**解决方案：**
- 确保网络环境良好
- 减少消息发送频率
- 使用简洁的数据格式

### 2. UI 控件不显示

**解决方案：**
- 检查控件位置是否在屏幕范围内
- 确认控件已正确创建
- 验证回调函数是否正确绑定

### 3. 消息丢失

**解决方案：**
- 增加消息确认机制
- 检查网络连接状态
- 使用消息队列缓存
