# 3. RoboMaster Python API

> [!info] 本章概述
> RoboMaster EP App 内置 Python API 完整参考，包括机器人控制、底盘、云台、发射器等模块。

---

## API 模块概览

App 内置 Python 使用以下控制模块前缀：

| 模块前缀 | 说明 |
|----------|------|
| `robot_ctrl` | 机器人主体控制 |
| `chassis_ctrl` | 底盘控制 |
| `gimbal_ctrl` | 云台控制 |
| `gun_ctrl` | 发射器控制 |
| `led_ctrl` | LED 灯效控制 |
| `armor_ctrl` | 装甲板控制 |
| `camera_ctrl` | 相机控制 |
| `vision_ctrl` | 视觉识别 |
| `sensor_ctrl` | 传感器接口 |

---

## 3.1 机器人控制 (robot_ctrl)

### 机器人信息

#### `robot_ctrl.get_battery_percentage()`

获取机器人电池电量。

| 参数 | 说明 |
|------|------|
| **返回** | 电池电量百分比 (0, 100] |

**示例：**
```python
battery = robot_ctrl.get_battery_percentage()
print(f"电量: {battery}%")  # 输出: 电量: 85%
```

---

#### `robot_ctrl.get_version()`

获取机器人固件版本。

| 参数 | 说明 |
|------|------|
| **返回** | 固件版本字符串 |

**示例：**
```python
version = robot_ctrl.get_version()
print(f"版本: {version}")
```

---

#### `robot_ctrl.get_sn()`

获取机器人 SN 序列号。

| 参数 | 说明 |
|------|------|
| **返回** | SN 序列号字符串 |

---

### 运动模式

#### `robot_ctrl.set_mode(mode)`

设置机器人运动模式。

| 参数 | 说明 |
|------|------|
| `mode` | 模式常量 |

**模式常量：**

| 常量 | 说明 |
|------|------|
| `robot_ctrl.MODE_CHASSIS_LEAD` | 底盘跟随模式 |
| `robot_ctrl.MODE_GIMBAL_LEAD` | 云台跟随模式 |
| `robot_ctrl.MODE_FREE` | 自由模式 |

**示例：**
```python
# 设置自由模式
robot_ctrl.set_mode(robot_ctrl.MODE_FREE)
```

---

## 3.2 底盘控制 (chassis_ctrl)

### 移动控制

#### `chassis_ctrl.move(x, y, z, speed)`

控制底盘相对移动。

| 参数 | 类型 | 说明 |
|------|------|------|
| `x` | float | 前进距离（米） |
| `y` | float | 横移距离（米） |
| `z` | float | 旋转角度（度） |
| `speed` | float | 移动速度（米/秒） |

**示例：**
```python
# 前进 1 米，速度 0.7 m/s
chassis_ctrl.move(1, 0, 0, 0.7)

# 左转 90 度
chassis_ctrl.move(0, 0, 90, 45)

# 左平移 0.5 米
chassis_ctrl.move(0, 0.5, 0, 0.5)
```

---

#### `chassis_ctrl.set_speed(x, y, z)`

设置底盘速度。

| 参数 | 类型 | 说明 |
|------|------|------|
| `x` | float | 前进速度（米/秒） |
| `y` | float | 横移速度（米/秒） |
| `z` | float | 旋转角速度（度/秒） |

**示例：**
```python
# 以 0.5 m/s 前进
chassis_ctrl.set_speed(0.5, 0, 0)
```

---

#### `chassis_ctrl.stop()`

停止底盘运动。

**示例：**
```python
chassis_ctrl.stop()
```

---

### 状态查询

#### `chassis_ctrl.get_speed()`

获取底盘当前速度。

| 参数 | 说明 |
|------|------|
| **返回** | (vx, vy, vz) 速度元组 |

---

#### `chassis_ctrl.get_attitude()`

获取底盘姿态角。

| 参数 | 说明 |
|------|------|
| **返回** | (pitch, roll, yaw) 姿态元组 |

---

#### `chassis_ctrl.get_position()`

获取底盘位置。

| 参数 | 说明 |
|------|------|
| **返回** | (x, y, z) 位置元组 |

---

## 3.3 云台控制 (gimbal_ctrl)

### 角度控制

#### `gimbal_ctrl.set_angle(pitch, yaw)`

设置云台绝对角度。

| 参数 | 类型 | 说明 |
|------|------|------|
| `pitch` | float | 俯仰角（度） |
| `yaw` | float | 偏航角（度） |

**示例：**
```python
# 云台指向正前方
gimbal_ctrl.set_angle(0, 0)

# 云台向左转 30 度
gimbal_ctrl.set_angle(0, 30)
```

---

#### `gimbal_ctrl.rotate(pitch, yaw)`

控制云台相对旋转。

| 参数 | 类型 | 说明 |
|------|------|------|
| `pitch` | float | 俯仰角偏移（度） |
| `yaw` | float | 偏航角偏移（度） |

**示例：**
```python
# 相对左转 15 度
gimbal_ctrl.rotate(0, 15)
```

---

#### `gimbal_ctrl.resume()`

云台回中。

**示例：**
```python
gimbal_ctrl.resume()
```

---

### 速度控制

#### `gimbal_ctrl.set_speed(pitch_speed, yaw_speed)`

设置云台旋转速度。

| 参数 | 类型 | 说明 |
|------|------|------|
| `pitch_speed` | float | 俯仰角速度（度/秒） |
| `yaw_speed` | float | 偏航角速度（度/秒） |

---

### 状态查询

#### `gimbal_ctrl.get_angle()`

获取云台当前角度。

| 参数 | 说明 |
|------|------|
| **返回** | (pitch, yaw) 角度元组 |

---

| `brightness` | int | 亮度值 (0-255) |


## 3.5 LED 控制 (led_ctrl)

### 灯效设置

#### `led_ctrl.set_led(led_id, r, g, b, effect)`

设置 LED 灯效。

| 参数 | 类型 | 说明 |
|------|------|------|
| `led_id` | int | LED ID |
| `r` | int | 红色分量 (0-255) |
| `g` | int | 绿色分量 (0-255) |
| `b` | int | 蓝色分量 (0-255) |
| `effect` | int | 灯效常量 |

**LED ID 常量：**

| 常量 | 说明 |
|------|------|
| `led_ctrl.LED_ALL` | 所有 LED |
| `led_ctrl.LED_TOP` | 顶部 LED |
| `led_ctrl.LED_BOTTOM` | 底部 LED |

**灯效常量：**

| 常量 | 说明 |
|------|------|
| `led_ctrl.EFFECT_ON` | 常亮 |
| `led_ctrl.EFFECT_OFF` | 熄灭 |
| `led_ctrl.EFFECT_FLASH` | 闪烁 |
| `led_ctrl.EFFECT_BREATH` | 呼吸 |

**示例：**
```python
# 设置所有 LED 为红色常亮
led_ctrl.set_led(led_ctrl.LED_ALL, 255, 0, 0, led_ctrl.EFFECT_ON)

# 设置顶部 LED 为蓝色呼吸
led_ctrl.set_led(led_ctrl.LED_TOP, 0, 0, 255, led_ctrl.EFFECT_BREATH)
```

## 3.10 多机通信 (multi_comm)

### 初始化

#### `multi_comm.init()`

初始化多机通信。

---

#### `multi_comm.deinit()`

释放多机通信资源。

---

### 消息收发

#### `multi_comm.send(target_id, message)`

发送消息给指定机器人。

| 参数 | 类型 | 说明 |
|------|------|------|
| `target_id` | int | 目标机器人 ID |
| `message` | str | 消息内容 |

---

#### `multi_comm.broadcast(message)`

广播消息给所有机器人。

| 参数 | 类型 | 说明 |
|------|------|------|
| `message` | str | 消息内容 |

---

注册消息接收回调。

| 参数 | 类型 | 说明 |
|------|------|------|
| `callback` | function | 回调函数 |

**回调函数格式：**
```python
def callback(sender_id, message):
    # 处理接收到的消息
    pass
```

## 完整示例

### 综合控制程序

```python
import robot_ctrl
import chassis_ctrl
import gimbal_ctrl
import gun_ctrl
import led_ctrl
import time

def start():
    """程序初始化"""
    # 检查电量
    battery = robot_ctrl.get_battery_percentage()
    print(f"电量: {battery}%")
    
    # 设置自由模式
    robot_ctrl.set_mode(robot_ctrl.MODE_FREE)
    
    # 设置 LED
    led_ctrl.set_led(led_ctrl.LED_ALL, 0, 255, 0, led_ctrl.EFFECT_ON)

def main():
    """主程序"""
    # 云台回中
    gimbal_ctrl.resume()
    time.sleep(1)
    
    # 前进 1 米
    chassis_ctrl.move(1, 0, 0, 0.7)
    time.sleep(2)
    
    # 左转 90 度
    chassis_ctrl.move(0, 0, 90, 45)
    time.sleep(2)
    
    # 发射
    gun_ctrl.fire_once()
    
    # 云台扫描
    for angle in [-30, 0, 30, 0]:
        gimbal_ctrl.set_angle(0, angle)
        time.sleep(0.5)

def stop():
    """程序清理"""
    chassis_ctrl.stop()
    led_ctrl.set_led(led_ctrl.LED_ALL, 0, 0, 0, led_ctrl.EFFECT_OFF)
```
