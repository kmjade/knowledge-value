## 技术架构

### 开发环境

| 组件 | 说明 |
|------|------|
| **MCU** | STM32 系列 |
| **IDE** | Keil MDK-ARM |
| **配置工具** | STM32CubeMX |
| **调试器** | ST-Link / J-Link |

### 目录结构

```
DevelopmentBoard-Examples/
├── Imu/                    # IMU 示例
├── PWM/                    # PWM 示例
├── RemoteControl/          # 遥控器示例
├── RM_OLED/                # OLED 示例
├── SDCard/                 # SD 卡示例
├── USB/                    # USB 示例
├── UWB/                    # UWB 示例
└── README.md               # 说明文档
```

### 通用结构

每个示例目录包含：

```
Example_Name/
├── Drivers/                # 硬件驱动
├── Inc/                    # 头文件
├── Src/                    # 源代码
├── MDK-ARM/                # Keil 工程文件
├── *.ioc                   # STM32CubeMX 配置
├── .mxproject              # CubeMX 工程配置
└── keilkill.bat            # 编译清理脚本
```

## 快速开始

### 环境准备

1. **安装 Keil MDK-ARM**
   - 下载地址：https://www.keil.com/
   - 版本要求：MDK 5.x

2. **安装 STM32CubeMX**
   - 下载地址：https://www.st.com/zh/development-tools/stm32cubemx.html
   - 用于工程配置和代码生成

3. **安装 ST-Link 驱动**
   - 用于程序下载和调试

### 使用方法

```bash
# 克隆仓库
git clone https://github.com/RoboMaster/DevelopmentBoard-Examples.git

# 进入示例目录
cd DevelopmentBoard-Examples/Imu

# 使用 Keil 打开工程
# MDK-ARM/Project.uvprojx
```

### 编译下载

1. 使用 Keil MDK-ARM 打开 `MDK-ARM/Project.uvprojx`
2. 编译工程 (Build)
3. 连接 ST-Link 调试器
4. 下载程序 (Download)
