# 3. RoboMaster EP 红外深度传感器

> [!info] 本章概述
> 介绍 RoboMaster EP 红外深度传感器的工作原理、产品特性、通讯协议和 Python API。

---

## 3.1 介绍

红外深度传感器的设计是基于飞行时间 **TOF (Time of Flight)** 原理：

- 传感器发出经调制的近红外光
- 遇物体后反射
- 传感器通过计算光线发射和反射时间差或相位差，来计算距离物体的距离

---

## 3.2 产品特性

### 探测原理

红外深度传感器的探测面积：

![TOF 探测原理](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/tof.png)

传感器发出的是一个角度为 **20°** 的圆锥光。

### 光斑与距离关系

光斑直径 D 与距离 Dist 的关系：

$$D = 2 \times Dist \times \tan(10°)$$

| 距离 (m) | 光斑直径 (cm) |
|----------|---------------|
| 0.5 | 17.6 |
| 1.0 | 35.3 |
| 2.0 | 70.5 |
| 4.0 | 141.0 |

## 3.5 控制命令

### 开启测量

```
ir_distance_sensor_measure_on
```

**描述：** 打开红外深度传感器数据输出，输出频率为 20 Hz

### 关闭测量

```
ir_distance_sensor_measure_off
```

**描述：** 关闭红外深度传感器数据输出

## 3.6 Python API

### 基本使用

```python
from robomaster import robot

ep_robot = robot.Robot()
ep_robot.initialize(conn_type="ap")

# 获取传感器对象
ep_sensor = ep_robot.sensor

# 获取距离值（单位：mm）
distance = ep_sensor.get_distance(id=1)
print(f"距离: {distance} mm")

ep_robot.close()
```

### 持续测量示例

```python
import time
from robomaster import robot

def distance_monitor():
    """持续监测距离"""
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="ap")
    
    ep_sensor = ep_robot.sensor
    
    print("开始测量距离...")
    print("按 Ctrl+C 停止")
    
    try:
        while True:
            distance = ep_sensor.get_distance(id=1)
            print(f"距离: {distance} mm")
            time.sleep(0.05)  # 20Hz 输出频率
    except KeyboardInterrupt:
        print("\n停止测量")
    
    ep_robot.close()

if __name__ == "__main__":
    distance_monitor()
```

### 避障示例

```python
import time
from robomaster import robot

def obstacle_avoidance():
    """使用红外传感器避障"""
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="ap")
    
    ep_chassis = ep_robot.chassis
    ep_sensor = ep_robot.sensor
    
    SAFE_DISTANCE = 300  # 安全距离 300mm
    
    try:
        while True:
            distance = ep_sensor.get_distance(id=1)
            print(f"距离: {distance} mm")
            
            if distance < SAFE_DISTANCE:
                # 检测到障碍物，停止并后退
                print("检测到障碍物！后退...")
                ep_chassis.drive_speed(x=-0.2, y=0, z=0, timeout=1)
                time.sleep(1)
                # 转向
                ep_chassis.drive_speed(x=0, y=0, z=30, timeout=1)
                time.sleep(1)
            else:
                # 安全，前进
                ep_chassis.drive_speed(x=0.3, y=0, z=0, timeout=1)
            
            time.sleep(0.1)
    
    except KeyboardInterrupt:
        print("\n停止")
        ep_chassis.drive_speed(x=0, y=0, z=0, timeout=1)
    
    ep_robot.close()

if __name__ == "__main__":
    obstacle_avoidance()
```

## 应用场景

### 1. 避障导航

- 实时检测前方障碍物距离
- 配合底盘实现自动避障
- 适用于室内导航

### 2. 物体检测

- 检测物体是否在指定范围内
- 配合机械臂实现抓取定位
- 精确距离测量

### 3. 墙壁跟随

- 保持与墙壁固定距离
- 实现沿墙导航
- 适用于迷宫探索

## 常见问题

### 1. 距离读数不稳定

**解决方案：**
- 确认目标物体表面平整
- 避免强光直射干扰
- 检查传感器安装是否稳固

### 2. 无法获取距离数据

**解决方案：**
- 确认 UART 连接正确
- 检查波特率是否为 115200
- 确认传感器已开启测量模式

### 3. 测量距离偏差大

**解决方案：**
- 考虑光斑直径影响
- 确认目标物体在测量范围内
- 检查目标物体颜色和反光率
