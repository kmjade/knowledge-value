## 快速导航

### 机械臂与机械爪

**功能特性：**
- 机械臂支持 FPV 精准遥控
- 机械爪支持夹力控制
- 水平移动范围：0-0.22 米
- 垂直移动范围：0-0.15 米
- 机械爪开合距离：约 10 厘米

**核心 API：**

| 功能 | API |
|------|-----|
| 相对位置运动 | `move(x, y)` |
| 绝对位置运动 | `moveto(x, y)` |
| 回中控制 | `recenter()` |
| 打开机械爪 | `gripper.open(power)` |
| 关闭机械爪 | `gripper.close(power)` |

📖 详细文档：[[1. 机械臂与机械爪]]

---

### 舵机

**功能特性：**
- 支持角度模式和速度模式
- 角度范围：0° ~ 360°
- 速度范围：0 ~ 49 rpm
- 支持 485 和 PWM 控制

**控制模式：**

| 模式 | 油门范围 | 输出 |
|------|----------|------|
| 角度模式 | 2.5% ~ 12.5% | 0° ~ 360° |
| 速度模式（顺时针） | 2.5% ~ 7.5% | 49rpm ~ 0rpm |
| 速度模式（逆时针） | 7.5% ~ 12.5% | 0rpm ~ -49rpm |

**核心 API：**

| 功能 | API |
|------|-----|
| 设置角度 | `set_angle(id, angle)` |
| 获取角度 | `get_angle(id)` |

📖 详细文档：[[2. 舵机]]

---

### 红外深度传感器

**功能特性：**
- 基于 TOF (Time of Flight) 原理
- 测量范围：0.1m ~ 8m
- 探测角度：20° 圆锥光
- 输出频率：20 Hz

**技术规格：**

| 参数 | 值 |
|------|-----|
| 通讯接口 | UART |
| 波特率 | 115200 |
| 数据位 | 8 |
| 停止位 | 1 |
| 奇偶校验 | None |

**核心 API：**

| 功能 | API |
|------|-----|
| 获取距离 | `sensor.get_distance(id)` |

📖 详细文档：[[3. 红外深度传感器]]

---

### 传感器转接模块

**功能特性：**
- 支持 ADC 模拟采集（12-bit, 0-3.3V）
- 支持 GPIO 数字输入输出
- 每个模块两个传感器接口
- 可接入温度、压力、测距等传感器

**引脚定义：**

| 引脚 | 功能 |
|------|------|
| VCC | 电源正极，输出 3.3V |
| GND | 电源地 |
| I/O | 电平输入，0 ~ 3.3V |
| AD | 模拟电压输入，0 ~ 3.3V |

**核心 API：**

| 功能 | API |
|------|-----|
| 获取 ADC 值 | `sensor_adapter.get_adc_value(id, port, pin)` |
| 获取 IO 值 | `sensor_adapter.get_io_value(id, port, pin)` |

📖 详细文档：[[4. 传感器转接模块]]

---

### UART 接口

**功能特性：**
- 第三方平台与 EP 连接的通信接口
- 支持明文 SDK 控制
- 可连接单片机实现分布式控制

**串口配置：**

| 参数 | 值 |
|------|-----|
| 波特率 | 115200 |
| 数据位 | 8 |
| 停止位 | 1 |
| 奇偶校验 | None |

**核心 API：**

| 功能 | API |
|------|-----|
| 发送字符串 | `uart.write_string(data)` |
| 读取字符串 | `uart.read_string()` |
| 发送字节 | `uart.write_bytes(data)` |
| 读取字节 | `uart.read_bytes(size)` |

📖 详细文档：[[5. UART接口]]
