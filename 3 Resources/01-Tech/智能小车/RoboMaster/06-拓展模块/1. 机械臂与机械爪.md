# 1. RoboMaster EP 机械臂与机械爪

> [!info] 本章概述
> 介绍 RoboMaster EP 机械臂与机械爪的功能特性、使用说明和 PWM 控制接口。

---

## 1.1 介绍

机械臂支持 FPV 精准遥控，机械爪配合机械臂使用，支持夹力控制，用户可在 App 中通过第一人称视角操控机械臂和机械爪完成任务。

![机械臂与机械爪](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/arm&gripper.png)

---

## 1.2 使用说明

用户可以控制机械臂的移动范围、机械爪的开合距离。

### 移动范围

| 组件 | 移动范围 |
|------|----------|
| **机械臂水平移动** | 0 - 0.22 米 |
| **机械臂垂直移动** | 0 - 0.15 米 |
| **机械爪开合距离** | 约 10 厘米 |

---

> [!warning] 安全警告
> 1. 当机械臂或机械爪处于工作状态时，请尽量避免对其施加外力
> 2. 请勿碰撞或损伤机械臂或机械爪，避免导致性能下降或舵机运行异常
> 3. 避免因用身体部位接触机械臂或机械爪旋转或尖锐部分而导致受伤
> 4. 及时清理水滴、水晶弹残渣等异物，避免腐蚀结构表面

---

## 1.3 机械爪 PWM 接口说明

机械爪支持力矩控制模式。

![PWM 接口引脚](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/arm_pwm.png)

### 引脚定义

| 序号 | 引脚 |
|------|------|
| 1 | 485A/PWM |
| 2 | 485B |
| 3 | VCC-12V |
| 4 | GND |

### PWM 信号参数

- **频率：** 50Hz
- **占空比范围：** 2.5% ~ 12.5%

### 占空比与夹力对应关系

| 占空比 | 夹力 |
|--------|------|
| 2.5% ~ 7.5% | 闭合力度 [最大, 0] |
| 7.5% ~ 12.5% | 开合力度 [0, 最大] |

## 1.4 Python API

### 机械臂控制

```python
from robomaster import robot

ep_robot = robot.Robot()
ep_robot.initialize(conn_type="ap")

# 获取机械臂对象
ep_arm = ep_robot.robotic_arm

# 绝对位置移动
ep_arm.moveto(x=100, y=50).wait_for_completed()

# 相对位置移动
ep_arm.move(x=20, y=10).wait_for_completed()

# 回中
ep_arm.recenter().wait_for_completed()

# 获取当前位置
pos = ep_arm.get_position()
print(f"位置: x={pos[0]}, y={pos[1]}")

ep_robot.close()
```

### 机械爪控制

```python
from robomaster import robot

ep_robot = robot.Robot()
ep_robot.initialize(conn_type="ap")

# 获取机械爪对象
ep_gripper = ep_robot.gripper

# 打开机械爪（功率 1-100）
ep_gripper.open(power=50)

# 关闭机械爪
ep_gripper.close(power=50)

# 暂停动作
ep_gripper.pause()

ep_robot.close()
```

## 完整示例：抓取物体

```python
import time
from robomaster import robot

def grip_object():
    """机械臂抓取物体示例"""
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="ap")
    
    ep_arm = ep_robot.robotic_arm
    ep_gripper = ep_robot.gripper
    
    # 1. 打开机械爪
    print("打开机械爪...")
    ep_gripper.open(power=50)
    time.sleep(1)
    
    # 2. 移动到目标位置
    print("移动机械臂...")
    ep_arm.moveto(x=150, y=0).wait_for_completed()
    
    # 3. 关闭机械爪抓取
    print("抓取物体...")
    ep_gripper.close(power=80)
    time.sleep(1)
    
    # 4. 抬起机械臂
    print("抬起物体...")
    ep_arm.moveto(x=150, y=80).wait_for_completed()
    
    # 5. 移动到放置位置
    print("移动到放置位置...")
    ep_arm.moveto(x=50, y=80).wait_for_completed()
    
    # 6. 放下物体
    print("放下物体...")
    ep_arm.moveto(x=50, y=0).wait_for_completed()
    ep_gripper.open(power=50)
    time.sleep(1)
    
    # 7. 回中
    print("回中...")
    ep_arm.recenter().wait_for_completed()
    
    ep_robot.close()
    print("抓取完成！")

if __name__ == "__main__":
    grip_object()
```

## 常见问题

### 1. 机械臂移动不到位

**解决方案：**
- 检查移动范围是否超出限制（水平 0-0.22m，垂直 0-0.15m）
- 确认机械臂没有被物理阻挡
- 检查 `wait_for_completed()` 是否已调用

### 2. 机械爪夹力不足

**解决方案：**
- 增加 `power` 参数值（1-100）
- 检查物体是否在机械爪开合范围内
- 确认机械爪没有异物阻碍

### 3. PWM 控制无响应

**解决方案：**
- 确认 PWM 信号频率为 50Hz
- 检查占空比是否在 2.5% ~ 12.5% 范围内
- 确认引脚连接正确
