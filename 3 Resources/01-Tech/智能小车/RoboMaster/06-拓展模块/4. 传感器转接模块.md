# 4. RoboMaster EP 传感器转接模块

> [!info] 本章概述
> 介绍 RoboMaster EP 传感器转接模块的功能、引脚说明和 Python API 使用方法。

---

## 4.1 介绍

传感器转接模块是为了方便用户将温度、压力、测距等传感器接入 RoboMaster EP 而设计的：

- 可在 Scratch 编程环境中获取传感器数据信息
- 每个模块均有 **两个传感器接口**
- 两个接口功能相同

![传感器转接模块](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/pinboard.png)

---

## 4.2 引脚说明

### 端口定义

每个传感器转接模块有两个端口（Port1 和 Port2），引脚功能相同：

| 端口 | 引脚 | 功能 |
|------|------|------|
| **Port1** | VCC | 电源正极，输出电压 3.3V |
| | GND | 电源地 |
| | I/O | 电平输入，输入范围 0 ~ 3.3V |
| | AD | 模拟电压输入，输入范围 0 ~ 3.3V |
| **Port2** | 同 Port1 | 同 Port1 |

---

## 4.3 功能特性

### 数字 I/O 功能

- **电平输入：** 0 ~ 3.3V
- **可读取高低电平状态**
- 适用于数字传感器（如限位开关、红外对管等）

### ADC 模拟采集

- **输入范围：** 0 ~ 3.3V
- **精度：** 12-bit ADC
- **适用：** 模拟传感器（如电位器、光敏电阻、温度传感器等）

---

## 4.4 Python API

### 基本使用

```python
from robomaster import robot

ep_robot = robot.Robot()
ep_robot.initialize(conn_type="ap")

# 获取传感器转接模块对象
ep_sensor_adapter = ep_robot.sensor_adapter

# 获取 ADC 值（通道 0-3）
adc_value = ep_sensor_adapter.get_adc_value(id=1, port=1, pin=1)
print(f"ADC 值: {adc_value}")

# 获取 IO 值
io_value = ep_sensor_adapter.get_io_value(id=1, port=1, pin=1)
print(f"IO 值: {io_value}")

ep_robot.close()
```

### ADC 读取示例

```python
import time
from robomaster import robot

def adc_monitor():
    """持续监测 ADC 值"""
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="ap")
    
    ep_sensor_adapter = ep_robot.sensor_adapter
    
    print("开始监测 ADC 值...")
    print("按 Ctrl+C 停止")
    
    try:
        while True:
            # 读取 Port1 的 ADC 值
            adc = ep_sensor_adapter.get_adc_value(id=1, port=1, pin=1)
            # 转换为电压（3.3V 参考）
            voltage = adc * 3.3 / 4095  # 12-bit ADC
            print(f"ADC: {adc}, 电压: {voltage:.2f}V")
            time.sleep(0.1)
    except KeyboardInterrupt:
        print("\n停止监测")
    
    ep_robot.close()

if __name__ == "__main__":
    adc_monitor()
```

### IO 读取示例

```python
import time
from robomaster import robot

def io_monitor():
    """监测 IO 电平变化"""
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="ap")
    
    ep_sensor_adapter = ep_robot.sensor_adapter
    
    print("监测 IO 电平...")
    
    try:
        while True:
            # 读取 Port1 的 IO 值
            io_state = ep_sensor_adapter.get_io_value(id=1, port=1, pin=1)
            
            if io_state == 1:
                print("IO 状态: 高电平 (3.3V)")
            else:
                print("IO 状态: 低电平 (0V)")
            
            time.sleep(0.1)
    except KeyboardInterrupt:
        print("\n停止监测")
    
    ep_robot.close()

if __name__ == "__main__":
    io_monitor()
```

## 应用示例

### 光敏传感器

```python
import time
from robomaster import robot

def light_sensor():
    """光敏传感器示例"""
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="ap")
    
    ep_sensor_adapter = ep_robot.sensor_adapter
    ep_led = ep_robot.led
    
    from robomaster import led
    
    print("光敏传感器控制 LED...")
    
    try:
        while True:
            # 读取光敏传感器 ADC 值
            adc = ep_sensor_adapter.get_adc_value(id=1, port=1, pin=1)
            
            # 根据光照强度设置 LED 亮度
            brightness = int(adc * 255 / 4095)
            
            ep_led.set_led(
                comp=led.COMP_ALL,
                r=brightness,
                g=brightness,
                b=brightness,
                effect=led.EFFECT_ON
            )
            
            print(f"光照强度: {adc}, LED 亮度: {brightness}")
            time.sleep(0.1)
    
    except KeyboardInterrupt:
        print("\n停止")
    
    ep_robot.close()

if __name__ == "__main__":
    light_sensor()
```

### 电位器控制

```python
import time
from robomaster import robot

def potentiometer_control():
    """电位器控制舵机角度"""
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="ap")
    
    ep_sensor_adapter = ep_robot.sensor_adapter
    ep_servo = ep_robot.servo
    
    print("用电位器控制舵机...")
    
    try:
        while True:
            # 读取电位器 ADC 值
            adc = ep_sensor_adapter.get_adc_value(id=1, port=1, pin=1)
            
            # 映射到舵机角度 (0-360°)
            angle = int(adc * 360 / 4095)
            
            # 设置舵机角度
            ep_servo.set_angle(id=1, angle=angle)
            
            print(f"ADC: {adc}, 角度: {angle}°")
            time.sleep(0.05)
    
    except KeyboardInterrupt:
        print("\n停止")
    
    ep_robot.close()

if __name__ == "__main__":
    potentiometer_control()
```

## 常见问题

### 1. ADC 读数异常

**解决方案：**
- 确认传感器电压在 0 ~ 3.3V 范围内
- 检查 VCC 和 GND 连接是否正确
- 验证 AD 引脚连接是否正常

### 2. IO 读取不稳定

**解决方案：**
- 确认输入电平稳定
- 添加适当的上拉/下拉电阻
- 检查是否有信号干扰

### 3. 无法识别模块

**解决方案：**
- 确认模块 ID 正确
- 检查模块是否正确连接
- 重启机器人后重试
