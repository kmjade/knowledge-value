# 2. RoboMaster EP 舵机

> [!info] 本章概述
> 介绍 RoboMaster EP 舵机的控制模式、引脚说明和 PWM 控制方法。

---

## 2.1 介绍

舵机的油门控制方式除了支持 485 控制，还可以进行 PWM 控制，控制模式包含：

- **速度模式**
- **角度模式**

> [!note] 控制模式说明
> 舵机的控制模式需要通过官方的编程接口（Scratch / Python）进行切换，并且会记录在舵机内部，不会随着舵机掉电而重置。使用 PWM 控制前请确认舵机当前的控制模式。

---

## 2.2 引脚说明

舵机上的 485 管脚和 PWM 管脚复用。

![舵机引脚](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/arm_pwm.png)

### 引脚定义

| 序号 | 引脚 |
|------|------|
| 1 | 485A/PWM |
| 2 | 485B |
| 3 | VCC-12V |
| 4 | GND |

---

## 2.3 控制说明

在 PWM 控制方式下，舵机对应的输入输出：

### PWM 参数

| 参数 | 值 |
|------|-----|
| **脉冲周期** | 50Hz |
| **油门范围** | 2.5% ~ 12.5% |

### 控制模式对应关系

| 控制模式 | 脉冲周期 | 油门范围 | 舵机输出 |
|----------|----------|----------|----------|
| **角度模式** | 50Hz | 2.5% ~ 12.5% | 0° ~ 360° |
| **速度模式** | 50Hz | 2.5% ~ 7.5% | 49rpm ~ 0rpm，顺时针 |
| **速度模式** | 50Hz | 7.5% ~ 12.5% | 0rpm ~ -49rpm，逆时针 |

## 2.4 Python API

### 舵机控制示例

```python
from robomaster import robot

ep_robot = robot.Robot()
ep_robot.initialize(conn_type="ap")

# 获取舵机对象
ep_servo = ep_robot.servo

# 设置舵机角度（ID: 1-3, 角度: 0-360）
ep_servo.set_angle(id=1, angle=90)

# 获取舵机角度
angle = ep_servo.get_angle(id=1)
print(f"舵机 1 当前角度: {angle}°")

ep_robot.close()
```

### 多舵机协调控制

```python
import time
from robomaster import robot

def servo_demo():
    """舵机控制示例"""
    ep_robot = robot.Robot()
    ep_robot.initialize(conn_type="ap")
    
    ep_servo = ep_robot.servo
    
    # 设置多个舵机角度
    print("设置舵机角度...")
    ep_servo.set_angle(id=1, angle=45)
    time.sleep(0.5)
    ep_servo.set_angle(id=2, angle=90)
    time.sleep(0.5)
    ep_servo.set_angle(id=3, angle=180)
    time.sleep(1)
    
    # 读取各舵机角度
    for i in range(1, 4):
        angle = ep_servo.get_angle(id=i)
        print(f"舵机 {i}: {angle}°")
    
    # 回到初始位置
    print("回到初始位置...")
    ep_servo.set_angle(id=1, angle=0)
    ep_servo.set_angle(id=2, angle=0)
    ep_servo.set_angle(id=3, angle=0)
    
    ep_robot.close()
    print("完成！")

if __name__ == "__main__":
    servo_demo()
```

## PWM 控制示例

### 角度模式

```python
import time

# 使用 PWM 控制舵机角度
# 50Hz PWM，占空比 2.5% ~ 12.5% 对应 0° ~ 360°

def angle_to_duty(angle):
    """角度转占空比"""
    # 2.5% = 0°, 12.5% = 360°
    duty = 2.5 + (angle / 360) * 10
    return duty

# 示例：设置舵机到 180°
target_angle = 180
duty_cycle = angle_to_duty(target_angle)
print(f"目标角度: {target_angle}°, 占空比: {duty_cycle}%")
```

### 速度模式

```python
import time

# PWM 控制舵机速度
# 50Hz PWM
# 2.5% ~ 7.5%: 顺时针，49rpm ~ 0rpm
# 7.5% ~ 12.5%: 逆时针，0rpm ~ -49rpm

def speed_to_duty(speed, direction='cw'):
    """
    速度转占空比
    speed: 0-49 rpm
    direction: 'cw' 顺时针, 'ccw' 逆时针
    """
    if direction == 'cw':
        # 顺时针：2.5% ~ 7.5%
        duty = 7.5 - (speed / 49) * 5
    else:
        # 逆时针：7.5% ~ 12.5%
        duty = 7.5 + (speed / 49) * 5
    return duty

# 示例：以 25rpm 顺时针旋转
target_speed = 25
duty_cycle = speed_to_duty(target_speed, 'cw')
print(f"目标速度: {target_speed}rpm, 占空比: {duty_cycle}%")
```

## 常见问题

### 1. 舵机不响应

**解决方案：**
- 确认控制模式（角度/速度）是否正确
- 检查 PWM 频率是否为 50Hz
- 确认占空比在 2.5% ~ 12.5% 范围内

### 2. 角度控制不准确

**解决方案：**
- 确认舵机处于角度模式
- 检查机械负载是否过大
- 验证 PWM 信号稳定性

### 3. 速度控制方向错误

**解决方案：**
- 检查占空比范围
  - 2.5% ~ 7.5%: 顺时针
  - 7.5% ~ 12.5%: 逆时针
- 确认舵机处于速度模式
