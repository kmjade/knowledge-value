# 1. RoboMaster SDK 安装

> [!info] 本章概述
> RoboMaster Python SDK 安装指南，支持 Windows、Linux 和 macOS 平台。

---

## 环境要求

### Python 版本

| 平台 | Python 版本 | 架构 |
|------|-------------|------|
| **Windows** | 3.6.6 - 3.8.9 | 64 位 |
| **Ubuntu** | 3.7+ | 64 位 |
| **macOS** | 3.7+ | 64 位 |

> [!warning] 重要提示
> - 必须使用 **64 位** Python 解释器
> - Python SDK 适配 **3.6.6 以上至 3.8.9** 版本
> - 否则会影响 Python SDK 的使用

> [!tip] Python 环境安装
> 如果尚未安装 Python 环境，请参考 [[2. 编程环境安装|编程环境安装]]

---

## 1.1 安装 SDK 到 Windows 平台

### 1.1.1 开发环境准备

在使用 Python SDK 前，需要确保编程平台端安装对应的 Python 环境：

| 步骤 | 说明 |
|------|------|
| 1 | 下载 Python 3.7-3.8 (64位) |
| 2 | 安装时勾选 `Add Python to PATH` |
| 3 | 验证安装：`python --version` |

---

### 1.1.2 安装 VC 库环境（可选）

> [!note] 何时需要
> 如果使用 SDK 时出现 `libmedia` 相关错误，需要安装 VC 库。

**下载地址：**
- [GitHub RoboMaster SDK repository](https://github.com/dji-sdk/robomaster-sdk)
- [Gitee RoboMaster SDK repository](https://gitee.com/xitinglin/RoboMaster-SDK)

![](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/vc_exe.png)

> [!warning] 错误示例
> 如果出现以下问题，请执行此安装：
> 
> ![](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/libmedia_err.png)

---

### 1.1.3 安装 VC Build Tools

> [!note] 何时需要
> 如果安装 SDK 时出现 `netifaces` 编译错误，需要安装 VC Build Tools。

**下载地址：**
- [GitHub RoboMaster SDK repository](https://github.com/dji-sdk/robomaster-sdk)
- [Gitee RoboMaster SDK repository](https://gitee.com/xitinglin/RoboMaster-SDK)

![](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/vs_build_tool.png)

> [!warning] 错误示例
> 不安装 VC build tools，安装 SDK 会报以下错误：
> 
> ![](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/neti_err.jpg)

---

### 1.1.4 安装 RoboMaster Python SDK

**步骤：**

1. 点开开始菜单，在搜索框中输入 `cmd`
2. 在搜索结果中，对着命令提示符程序，单击鼠标右键
3. 菜单中点击选择 **以管理员身份运行**
4. 输入以下指令：

```bash
# 安装 SDK
pip install robomaster
```

**网络较差时使用国内镜像：**

```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple robomaster
```

> [!tip] 安装失败处理
> 如果出现下列情况，请参考 [[2. 编程环境安装|Python 编程环境安装]]：
> 
> ![](https://robomaster-dev.readthedocs.io/zh-cn/latest/_images/pip_install_error.jpg)

---

### 1.1.5 升级 RoboMaster SDK

```bash
pip install --upgrade robomaster
```

## 1.2 安装 SDK 到 Linux 平台

### Ubuntu 环境

**安装 Python 环境：**

```bash
# 更新软件包列表
sudo apt update

# 安装 Python 和 pip
sudo apt install python3.7 python3-pip
```

**安装 RoboMaster SDK：**

```bash
pip3 install robomaster
```

**升级 RoboMaster SDK：**

```bash
pip3 install --upgrade robomaster
```

> [!tip] 树莓派安装
> 树莓派下的 Python SDK 安装教程可参考 [sdk install on Raspberry Pi.7z](https://github.com/dji-sdk/robomaster-sdk)

---

## 1.3 安装 SDK 到 macOS X 平台

**安装 Python 环境：**

```bash
# 使用 Homebrew 安装 Python
brew install python@3.8
```

**安装 RoboMaster SDK：**

```bash
pip3 install robomaster
```

**升级 RoboMaster SDK：**

```bash
pip3 install --upgrade robomaster
```

## 验证安装

### 检查 SDK 版本

```python
# 测试 SDK 是否安装成功
import robomaster
print(robomaster.__version__)
```

### 查看安装信息

```bash
# 查看 SDK 版本
pip show robomaster

# 查看 SDK 安装位置
pip show robomaster | grep Location

# 列出已安装的包
pip list | grep robomaster
```

## 常见问题

### 1. pip 不是内部或外部命令

**解决方案：**
- 确保 Python 已添加到 PATH
- 使用 `python -m pip install robomaster`
- 重新安装 Python 并勾选 `Add Python to PATH`

### 2. 安装速度慢

**解决方案：使用国内镜像**

```bash
# 清华镜像
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple robomaster

# 阿里镜像
pip install -i https://mirrors.aliyun.com/pypi/simple robomaster
```

### 3. 权限错误

**解决方案：**

```bash
# Windows - 使用用户安装
pip install robomaster --user

# Linux/macOS - 使用 sudo
sudo pip install robomaster
```

### 4. 网络连接错误

**解决方案：**

```bash
# 使用代理
pip install --proxy http://proxy.example.com:8080 robomaster

# 或使用国内镜像
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple robomaster
```

### 5. 版本冲突

**解决方案：**

```bash
# 创建虚拟环境
python -m venv robomaster-env

# 激活虚拟环境
# Windows
robomaster-env\Scripts\activate
# Linux/macOS
source robomaster-env/bin/activate

# 安装 SDK
pip install robomaster
```
