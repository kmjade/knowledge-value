## 02_chassis - 底盘控制

### 移动控制

```python
# 相对移动
ep_chassis.move(x=1, y=0, z=0, xy_speed=0.7).wait_for_completed()

# 速度控制
ep_chassis.drive_speed(x=0.5, y=0, z=0, timeout=5)

# 麦轮速度
ep_chassis.drive_wheels(w1=100, w2=100, w3=100, w4=100)
```

### 示例列表

| 示例 | 功能 |
|------|------|
| `move()` | 相对位置移动 |
| `moveto()` | 绝对位置移动 |
| `drive_speed()` | 速度控制 |
| `drive_wheels()` | 麦轮速度控制 |
| `get_attitude()` | 获取姿态角 |
| `get_position()` | 获取位置 |
| `get_speed()` | 获取速度 |

## 05_vision - 视觉识别

### 识别类型

| 类型 | 说明 |
|------|------|
| `DETECT_FACE` | 人脸识别 |
| `DETECT_MARKER` | 标签识别 |
| `DETECT_LINE` | 线识别 |

### 示例代码

```python
from robomaster import vision

# 启用人脸识别
ep_vision.enable_detection(vision.DETECT_FACE)

# 获取识别结果
faces = ep_vision.get_detect_info(vision.DETECT_FACE)

# 禁用识别
ep_vision.disable_detection(vision.DETECT_FACE)
```

## 07_led - LED 控制

### 灯效控制

```python
from robomaster import led

# 设置红色常亮
ep_led.set_led(comp=led.COMP_ALL, r=255, g=0, b=0, effect=led.EFFECT_ON)

# 设置蓝色呼吸
ep_led.set_led(comp=led.COMP_ALL, r=0, g=0, b=255, effect=led.EFFECT_BREATH)

# 关闭所有灯
ep_led.set_led(comp=led.COMP_ALL, r=0, g=0, b=0, effect=led.EFFECT_OFF)
```

### 灯效常量

| 常量 | 说明 |
|------|------|
| `EFFECT_ON` | 常亮 |
| `EFFECT_OFF` | 熄灭 |
| `EFFECT_FLASH` | 闪烁 |
| `EFFECT_BREATH` | 呼吸 |
| `EFFECT_SCROLLING` | 流水 |

## 12_drone - 无人机控制

### 飞行控制

```python
# 起飞
tl_flight.takeoff().wait_for_completed()

# 前进 100cm
tl_flight.forward(distance=100).wait_for_completed()

# 旋转 90°
tl_flight.rotate(angle=90).wait_for_completed()

# 降落
tl_flight.land().wait_for_completed()
```

### 挑战卡

```python
# 开启挑战卡检测
tl_flight.mission_pad_on()

# 飞到挑战卡位置
tl_flight.go(x=0, y=0, z=100, speed=50, mid="m1").wait_for_completed()

# 关闭挑战卡检测
tl_flight.mission_pad_off()
```

## 运行示例

### 下载并运行

```bash
# 克隆仓库
git clone https://github.com/dji-sdk/RoboMaster-SDK.git
cd RoboMaster-SDK

# 安装依赖
pip install -r requirements.txt

# 运行示例
python examples/01_robot/01_get_version.py
```

### 常用命令

```bash
# 安装 SDK
pip install robomaster

# 更新 SDK
pip install --upgrade robomaster

# 查看版本
pip show robomaster
```
