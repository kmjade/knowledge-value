# UGV01与UGV02对比

> [!info] 对比概述
> 本文详细对比Waveshare UGV01履带机器人和UGV02六轮机器人的特性、规格和适用场景，帮助用户选择最适合自己需求的产品。

## 📊 核心特性对比

### 主要差异

| 特性 | UGV01 | UGV02 |
|------|-------|-------|
| **底盘类型** | 履带底盘 | 六轮底盘 |
| **驱动方式** | 四轮驱动 | 六轮四驱 |
| **轮胎/履带** | 履带 | 柔软橡胶轮胎 |
| **悬挂系统** | 多独立悬挂系统（每侧4个） | 无悬挂系统 |
| **减震方式** | 悬架减震 | 轮胎减震 |
| **地形适应性** | 极强（复杂地形） | 强（越野） |
| **平地稳定性** | 中等 | 优秀 |
| **爬坡能力** | 优秀（悬挂收缩提供额外支撑） | 优秀（六轮四驱） |
| **扩展导轨** | 1020欧标型材 × 2 | 1020欧标型材 × 2（203mm间距） |
| **驱动板** | General Driver for Robots（ESP32） | General Driver for Robots（ESP32） |
| **电机** | 带编码器电机 | JGB37-520带编码器电机 |
| **编码器** | 支持 | 支持 |
| **速度控制** | 闭环速度控制 | 闭环速度控制 |

### 共同特性

| 特性 | 说明 |
|------|------|
| **驱动板** | 基于ESP32的多功能机器人驱动板 |
| **OLED屏幕** | 0.96英寸OLED屏幕 |
| **IMU模块** | 九轴传感器 |
| **SD卡槽** | 支持TF卡 |
| **WiFi/蓝牙** | 支持AP和STA模式 |
| **舵机接口** | TTL总线舵机、PWM舵机 |
| **GPIO扩展** | IO4、IO5等通用IO |
| **UPS供电** | 3S 18650锂电池UPS供电模块 |
| **INA219** | 电压电流采集芯片 |
| **边充边放** | 支持边充电边放电 |
| **JSON指令** | 支持JSON指令通信 |
| **多种通信** | 串口、USB、HTTP、ESP-NOW |
| **上位机扩展** | 支持树莓派、Jetson Nano等 |
| **开源** | 全部代码开源 |
| **Web端控制** | 内置Web控制界面 |

## 🎯 适用场景对比

### UGV01 - 履带机器人

**适合场景**：

| 场景 | 说明 |
|------|------|
| **复杂地形** | 碎石、沙地、泥地等复杂地形 |
| **越野探险** | 户外越野、野外探险 |
| **复杂环境** | 楼梯、台阶等障碍环境 |
| **科研教育** | 机器人教学、科研项目 |
| **竞赛应用** | 机器人竞赛、挑战赛 |

**优势**：
- ✅ 极强的地形通过能力
- ✅ 优秀的越野性能
- ✅ 多独立悬挂系统
- ✅ 重负载爬坡能力强

**劣势**：
- ❌ 平地速度相对较慢
- ❌ 噪音较大
- ❌ 能耗相对较高
- ❌ 结构较复杂

---

### UGV02 - 六轮机器人

**适合场景**：

| 场景 | 说明 |
|------|------|
| **平地移动** | 室内外平坦地面 |
| **越野越野** | 草地、土路等越野环境 |
| **巡逻巡检** | 工厂巡检、安防巡逻 |
| **科研教育** | 机器人教学、科研项目 |
| **竞赛应用** | 机器人竞赛、挑战赛 |

**优势**：
- ✅ 优秀的平地稳定性
- ✅ 较高的移动速度
- ✅ 能耗相对较低
- ✅ 结构相对简单
- ✅ 噪音较小

**劣势**：
- ❌ 地形通过能力不如履带
- ❌ 复杂地形适应性一般
- ❌ 无悬挂系统

## 💡 选择建议

### 选择UGV01的情况

**推荐选择**：

1. **需要复杂地形通过能力**
   - 碎石、沙地、泥地
   - 楼梯、台阶等障碍
   - 极端越野环境

2. **重负载应用**
   - 需要承载较重设备
   - 需要强力爬坡能力

3. **科研竞赛需求**
   - 机器人竞赛
   - 复杂环境挑战赛

**不推荐选择**：
- 主要在平坦地面使用
- 对噪音敏感的应用
- 对续航有高要求

---

### 选择UGV02的情况

**推荐选择**：

1. **平地或轻度越野**
   - 室内外平坦地面
   - 草地、土路等轻度越野
   - 一般户外环境

2. **需要较高速度**
   - 巡逻巡检
   - 快速移动应用

3. **教学和科研**
   - 机器人教学
   - 一般科研项目

**不推荐选择**：
- 需要复杂地形通过能力
- 需要爬楼梯或台阶
- 极端越野环境
