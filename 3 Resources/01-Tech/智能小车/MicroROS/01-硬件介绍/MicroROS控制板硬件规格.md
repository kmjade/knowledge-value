# MicroROS控制板硬件规格

> [!summary] 概述
> 亞博智能 MicroROS 機器人控制板完整硬件規格，基於 ESP32-S3 雙核處理器，專為 ROS2 機器人開發設計。

---

## 主控芯片

### ESP32-S3 規格

| 項目 | 規格 |
|------|------|
| 處理器 | ESP32-S3 雙核 Xtensa LX7 |
| CPU 頻率 | 240MHz |
| 內存 | 512KB SRAM |
| 閃存 | 8MB |
| 無線通訊 | 2.4GHz WiFi 4/5, Bluetooth 5.0 |
| 工作電壓 | 3.3V |
| 工作溫度 | -40°C ~ +85°C |

### 核心特性

- 雙核高性能處理器
- 支持 WiFi 4/5 無線連接
- 支持藍牙 5.0
- 豐富的 GPIO 接口
- 多路 PWM 輸出
- 多路 ADC 輸入

---

## 板載資源

### 電機接口

| 接口 | 規格 | 說明 |
|------|------|------|
| 電機接口 | 4路 | 310 編碼器電機 |
| 驅動方式 | PWM + 編碼器 | 支持速度閉環控制 |
| 編碼器類型 | 增量式編碼器 | AB 相輸出 |

### 舵機接口

| 接口 | 規格 | 說明 |
|------|------|------|
| PWM 舵機 | 2路 (S1/S2) | 標準 PWM 舵機接口 |
| 脈衝範圍 | 500-2500μs | 對應 0-180° |
| 電源 | 5V | 獨立供電 |

### 傳感器

| 傳感器 | 型號 | 功能 |
|--------|------|------|
| IMU | 六軸姿態傳感器 | 加速度計 + 陀螺儀 |
| 激光雷達接口 | MS200 | 支持 MS200 激光雷達 |

### 通訊接口

| 接口 | 規格 | 用途 |
|------|------|------|
| Type-C | USB 串口 | 固件燒錄、串口通訊 |
| 雷達接口 | 串口 | MS200 雷達連接 |
| 天線接口 | 外置天線 | WiFi/藍牙信號增強 |

### 電源系統

| 接口 | 規格 | 說明 |
|------|------|------|
| 電池接口 | 7.4V T型 | 連接 2S 鋰電池 |
| 充電接口 | DC 8.4V | 充電器接口 |
| PD 供電 | Type-C | 為樹莓派5供電 |
| 電源開關 | - | 總電源開關 |

## 工作模式

### 正常運行模式

- 加載 ROS2 節點
- 接收指令控制外設
- 發布傳感器數據
- MCU 指示燈：每三秒快閃兩下

### 配置模式

**進入方式**：上電時按住 BOOT 按鍵

- 啟動 AP 熱點模式
- WiFi SSID：`microROS_xxxxxx`
- 密碼：`12345678`
- MCU 指示燈：每秒閃一次

### 測試模式

**進入方式**：上電時按住 KEY1 按鍵

- 測試所有電機
- 測試舵機
- 測試 IMU
- 測試蜂鳴器和 LED
- MCU 指示燈：每三秒快閃三次

### 低電壓模式

**觸發條件**：電池電壓 < 7.0V

- 蜂鳴器報警
- MCU 指示燈快閃
- 限制電機功率
