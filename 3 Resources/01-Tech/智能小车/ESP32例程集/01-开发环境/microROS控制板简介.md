# microROS控制板简介

> microROS控制板是一款轻量级的ROS2驱动控制器，基于ESP32S3双核开发板，专为机器人应用设计。

---

## 📋 核心特性

### 主要功能

| 类别 | 功能 | 描述 |
|------|------|------|
| **控制核心** | ESP32-S3双核 | 高性能双核处理器，支持WiFi和蓝牙5.0 |
| **电机控制** | 四路编码器电机 | 支持310编码器电机，PWM控制 |
| **舵机控制** | 两路PWM舵机 | 标准PWM舵机接口 |
| **传感器** | 六轴IMU | 内置姿态传感器，提供姿态信息 |
| **激光雷达** | MS200雷达接口 | 支持激光雷达导航 |
| **电源管理** | PD电源 | 为树莓派5供电，支持7.4V电池 |
| **通讯接口** | 多种通讯方式 | WiFi、蓝牙、串口通讯 |

---

### 状态指示灯说明

#### MCU指示灯 (GPIO45)

![[Pasted image 20260219005727.png]]

| 状态 | 闪烁频率 | 描述 |
|------|---------|------|
| **正常模式** | 每三秒快闪两下 | 系统正常运行 |
| **低电压模式** | 每100毫秒闪一次 (快闪) | 电池电量不足 (<7.0V) |
| **配置模式** | 每秒闪一次 (慢闪) | 进入AP配置模式 |
| **测试模式** | 每三秒快闪三次 | 系统测试模式 |

## 🔌 接口使用说明

![[Pasted image 20260219005834.png]]

### 电池供电系统

#### 电池连接
- **电池规格**: 7.4V 锂电池 (2S)
- **接口类型**: T型接口
- **电池容量**: 推荐 2000mAh-5000mAh
- **低电压保护**: <7.0V 时蜂鸣器报警，MCU指示灯快闪

#### 充电连接
- **充电器规格**: DC 8.4V 电源适配器
- **接口类型**: DC圆孔接口
- **充电电流**: 推荐 2A-3A
- **充电时间**: 约2-3小时 (满电池)

## ⚙️ 工作模式说明

### 1. 正常运行模式

**进入条件**: 系统正常启动，无特殊按键操作

**功能**:
- 加载ROS2节点
- 接收指令并控制外设
- 发布IMU数据
- 连接microROS代理

**MCU指示灯**: 每三秒快闪两下

### 2. 配置模式

**进入条件**: 上电时按住BOOT按键不放

**功能**:
- 启动AP模式 (热点)
- 连接WiFi配置
- 修改ROS2参数
- 查看系统信息

**MCU指示灯**: 每秒闪一次 (慢闪)

**默认SSID**: `microROS_xxxxxx` (xxxxxx为MAC地址后6位)

**默认密码**: `12345678`

### 3. 测试模式

**进入条件**: 上电时按住自定義按键 (KEY1) 不放

**功能**:
- 测试所有电机
- 测试舵机
- 测试IMU
- 测试蜂鸣器
- 测试LED指示灯

**MCU指示灯**: 每三秒快闪三次

### 4. 低电压模式

**进入条件**: 电池电压 < 7.0V

**功能**:
- 蜂鸣器报警
- MCU指示灯快闪
- 限制电机功率
- 建议充电

**MCU指示灯**: 每100毫秒闪一次 (快闪)
