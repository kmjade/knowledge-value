# 基本设置

> [!summary] 功能概述
> AutoCAD 2026 的單位設置、繪圖界限與選項配置，是開始繪圖前的必要準備工作。

---

## 功能介紹

### 功能定位
- **所屬模塊**：基礎入門
- **功能類型**：環境配置
- **前置條件**：熟悉界面

### 應用場景
1. 新建文件時設置繪圖單位
2. 設置繪圖區域範圍
3. 配置軟件選項以提高效率
4. 自定義系統變量

---

## 單位設置（UNITS）

### 命令調用方式

| 方式 | 命令/操作 |
|------|----------|
| 命令行 | `UNITS` 或 `UN` |
| 菜單 | 格式 → 單位 |

### 圖形單位對話框

#### 長度類型

| 類型 | 格式 | 示例 |
|------|------|------|
| 建築 | 英尺-分數英寸 | 1'-3 1/2" |
| 小數 | 十進制 | 15.5000 |
| 工程 | 英尺-小數英寸 | 1'-3.50" |
| 分數 | 分數英寸 | 15 1/2 |
| 科學 | 指數 | 1.55E+01 |

> [!tip] 機械製圖建議
> 使用「小數」類型，精度設為 0.00 或 0.000

#### 角度類型

| 類型 | 格式 | 示例 |
|------|------|------|
| 十進制度數 | 度 | 45.0000 |
| 度/分/秒 | DMS | 45d0'0" |
| 百分度 | 梯度 | 50.0000g |
| 弧度 | 弧度 | 0.7854r |
| 勘測單位 | N/S/E/W | N 45d0'0" E |

> [!tip] 機械製圖建議
> 使用「十進制度數」，精度設為 0.0

#### 插入比例

設置插入塊時的縮放單位：
- 機械製圖：毫米 (mm)
- 建築製圖：毫米或米

## 繪圖界限（LIMITS）

### 命令調用方式

| 方式 | 命令/操作 |
|------|----------|
| 命令行 | `LIMITS` |
| 菜單 | 格式 → 圖形界限 |

### 設置步驟

```
LIMITS
指定左下角點或 [開(ON)/關(OFF)] <0.0000,0.0000>: 0,0
指定右上角點 <420.0000,297.0000>: 594,420
```

### 常用圖紙尺寸

| 圖幅 | 尺寸 (mm) | 繪圖界限設置 |
|------|----------|--------------|
| A0 | 841 × 1189 | 0,0 到 1189,841 |
| A1 | 594 × 841 | 0,0 到 841,594 |
| A2 | 420 × 594 | 0,0 到 594,420 |
| A3 | 297 × 420 | 0,0 到 420,297 |
| A4 | 210 × 297 | 0,0 到 297,210 |

### 界限開關

- **開 (ON)**：阻止在界限外繪圖
- **關 (OFF)**：允許在界限外繪圖

> [!tip] 建議
> 設置為「關」，便於靈活繪圖；打印時使用佈局控制範圍

## 選項配置（OPTIONS）

### 命令調用方式

| 方式 | 命令/操作 |
|------|----------|
| 命令行 | `OPTIONS` 或 `OP` |
| 菜單 | 工具 → 選項 |
| 快捷菜單 | 右鍵 → 選項 |

### 重要選項卡

#### 顯示選項卡

| 設置項 | 推薦值 | 說明 |
|--------|--------|------|
| 窗口元素 - 配色方案 | 深色/淺色 | 界面主題 |
| 十字光標大小 | 100 | 滿屏十字線 |
| 顯示精度 - 圓弧和圓的平滑度 | 1000 | 提高顯示質量 |

#### 打開和保存選項卡

| 設置項 | 推薦值 | 說明 |
|--------|--------|------|
| 另存為 | AutoCAD 2018 圖形 | 兼容性 |
| 自動保存 | 開啟，10 分鐘 | 防止數據丟失 |
| 每次保存均創建備份副本 | 開啟 | 備份文件 |

#### 打印和發布選項卡

| 設置項 | 推薦值 | 說明 |
|--------|--------|------|
| 默認打印樣式表 | monochrome.ctb | 黑白打印 |
| 打印到文件 | 開啟 | 便於 PDF 輸出 |

#### 用戶系統配置選項卡

| 設置項 | 推薦值 | 說明 |
|--------|--------|------|
| 快捷菜單 | 開啟 | 右鍵快捷菜單 |
| 繪圖區域中使用快捷菜單 | 開啟 | |

#### 草圖設置選項卡

| 設置項 | 推薦值 | 說明 |
|--------|--------|------|
| 自動捕捉設置 | 全部開啟 | 對象捕捉輔助 |
| 自動捕捉標記大小 | 適中 | |

## 系統變量

### 常用系統變量

| 變量名 | 默認值 | 說明 |
|--------|--------|------|
| `LTSCALE` | 1.0 | 線型比例 |
| `DIMSCALE` | 1.0 | 標註比例 |
| `TEXTSIZE` | 2.5 | 文字高度 |
| `MIRRTEXT` | 0 | 鏡像文字是否反轉 |
| `OSMODE` | 4133 | 對象捕捉模式 |

### 修改系統變量

```
LTSCALE
輸入新線型比例因子 <1.0000>: 10
```

## 機械製圖推薦設置

### 新建文件設置清單

- [ ] 單位：小數，精度 0.00
- [ ] 角度：十進制度數，精度 0.0
- [ ] 插入比例：毫米
- [ ] 繪圖界限：根據圖幅設置
- [ ] 十字光標：100%
- [ ] 自動保存：開啟，10 分鐘

### 創建模板文件

將上述設置保存為模板：
1. 完成所有設置
2. `SAVEAS` → 選擇 `.dwt` 格式
3. 命名為「機械製圖模板.dwt」

## 實用技巧

### 配置文件的導出與導入

**導出配置**：
```
OPTIONS → 配置 → 輸出
```

**導入配置**：
```
OPTIONS → 配置 → 輸入
```

### 重置選項

恢復默認設置：
```
OPTIONS → 配置 → 重置
```

## 注意事項

> [!warning] 重要提示
> - 不同項目可能需要不同的設置，建議創建多個模板
> - 修改系統變量會影響當前圖形
> - 重要的設置變更前先備份文件
