# 文件管理

> [!summary] 功能概述
> AutoCAD 文件的創建、打開、保存與模板使用，是繪圖工作的基礎操作。

---

## 功能介紹

### 功能定位
- **所屬模塊**：基礎入門
- **功能類型**：文件操作
- **前置條件**：熟悉界面

### 應用場景
1. 創建新的工程圖文件
2. 打開現有圖紙進行編輯
3. 保存繪圖成果
4. 使用模板提高效率

---

## 文件類型

### AutoCAD 主要文件格式

| 格式 | 擴展名 | 說明 |
|------|--------|------|
| DWG | `.dwg` | 標準繪圖文件 |
| DXF | `.dxf` | 數據交換格式 |
| DWT | `.dwt` | 模板文件 |
| DWS | `.dws` | 標準文件 |
| BAK | `.bak` | 備份文件 |
| SV$ | `.sv$` | 自動保存文件 |

---

## 新建文件

### 方法一：應用程序菜單
1. 點擊左上角紅色 "A" 圖標
2. 選擇「新建」→「圖形」
3. 選擇模板文件

### 方法二：快速訪問工具欄
- 點擊「新建」圖標

### 方法三：命令行
```
NEW (或 QNEW)
```

### 方法四：快捷鍵
- `Ctrl + N`

### 使用模板創建文件

新建文件時可選擇模板：
- `acad.dwt` - 英制模板
- `acadiso.dwt` - 公制模板
- 自定義模板（機械製圖模板）

> [!tip] 機械製圖建議
> 創建包含標準圖層、標註樣式、圖框的模板文件

---

## 打開文件

### 方法一：應用程序菜單
1. 點擊「打開」→「圖形」
2. 選擇文件
3. 點擊「打開」

### 方法二：快速訪問工具欄
- 點擊「打開」圖標

### 方法三：命令行
```
OPEN
```

### 方法四：快捷鍵
- `Ctrl + O`

### 打開選項

| 選項 | 說明 |
|------|------|
| 打開 | 正常打開 |
| 以只讀方式打開 | 無法保存修改 |
| 局部打開 | 只加載指定圖層 |

---

## 保存文件

### 保存（Save）

**方法**：
- 快速訪問工具欄 → 保存圖標
- 命令：`QSAVE`
- 快捷鍵：`Ctrl + S`

### 另存為（Save As）

**方法**：
- 應用程序菜單 → 另存為 → 圖形
- 命令：`SAVEAS`
- 快捷鍵：`Ctrl + Shift + S`

### 自動保存

**設置路徑**：選項 → 打開和保存 → 文件安全措施

| 設置項 | 推薦值 |
|--------|--------|
| 自動保存 | 開啟 |
| 保存間隔分鐘數 | 10-15 |
| 每次保存均創建備份副本 | 開啟 |

---

## 模板文件

### 模板的作用

模板文件包含預設置的：
- 圖層設置
- 標註樣式
- 文字樣式
- 圖框和標題欄
- 塊定義

### 創建模板

1. 新建文件並完成設置
2. 保存為 `.dwt` 格式
3. 存放於模板目錄

### 模板目錄位置

```
C:\Users\[用戶名]\AppData\Local\Autodesk\AutoCAD 2026\Rxx.x\chs\Template\
```

### 機械製圖模板建議內容

- 圖層：輪廓線、中心線、尺寸線、剖面線、文字
- 標註樣式：GB 標準標註樣式
- 文字樣式：仿宋體、宋體
- 圖框：A0-A4 標準圖框

---

## 文件修復

### 恢復損壞的文件

**方法一**：使用 RECOVER 命令
```
RECOVER
選擇要恢復的文件
```

**方法二**：使用審核命令
```
AUDIT
是否更正檢測到的錯誤？[是(Y)/否(N)] <N>: Y
```

### 查找自動保存文件

自動保存文件位於：
```
C:\Users\[用戶名]\AppData\Local\Temp\
```
查找 `.sv$` 文件，重命名為 `.dwg`

---

## 實用技巧

### 文件命名規範

建議命名格式：
```
[項目編號]_[零件名稱]_[版本號].dwg
例：P001_齒輪軸_V01.dwg
```

### 版本兼容性

| AutoCAD 版本 | 可打開的 DWG 版本 |
|--------------|------------------|
| 2026 | 2018-2026 |
| 2024 | 2018-2024 |

> [!warning] 注意
> 使用「另存為」選擇較低版本格式，以便與舊版本兼容
