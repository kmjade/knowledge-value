# 快捷键大全

> [!summary] 功能概述
> AutoCAD 2026 常用快捷鍵與命令別名速查表，提高繪圖效率的必備參考。

---

## 功能介紹

### 功能定位
- **所屬模塊**：基礎入門
- **功能類型**：參考文檔
- **前置條件**：無

### 使用說明

AutoCAD 快捷鍵分為兩類：
1. **功能鍵/組合鍵**：如 `Ctrl + S`
2. **命令別名**：如 `L` 對應 `LINE` 命令

## 自定義快捷鍵

### 修改命令別名

1. 輸入 `ALIASEDIT` 或 `PGP`
2. 編輯 `acad.pgp` 文件
3. 添加或修改別名

### PGP 文件格式

```
L,          *LINE
C,          *CIRCLE
```

### 重新加載 PGP

```
REINIT
選擇 PGP 文件
```
