# 🏷️ 耳穴疗法知识库标签系统

## 📋 标签体系概述

本知识库采用层次化标签系统，通过多维度分类实现知识的精准组织和快速检索。

### 🎯 标签设计原则
- **层次化**：采用父子标签结构
- **语义化**：标签名称具有明确语义
- **标准化**：统一标签命名规范
- **可扩展**：预留扩展空间
- **易检索**：便于搜索和筛选

## 🏷️ 标签使用规范

### 📝 标签命名规则
- **小写字母**：统一使用小写字母
- **连字符分隔**：使用连字符(-)分隔单词
- **语义明确**：标签名称具有明确含义
- **长度适中**：标签长度控制在合理范围
- **避免重复**：避免语义重复的标签

### 🔗 标签组合使用
- **主标签+子标签**：`#tcm/acupuncture`
- **多标签组合**：`#clinical #safe #effective`
- **层次化标签**：`#technique/acupuncture/manual`
- **功能性标签**：`#priority/high #status/active`

### 📊 标签统计管理
- **标签使用频率**：定期统计标签使用情况
- **标签清理**：定期清理无用和重复标签
- **标签标准化**：统一标签命名和使用规范
- **标签优化**：根据使用情况优化标签体系

## 🔍 标签检索技巧

### 🎯 精确检索
- **单一标签**：搜索特定标签内容
- **组合标签**：使用多个标签组合检索
- **排除标签**：使用减号排除特定标签
- **模糊匹配**：使用通配符进行模糊搜索

### 📱 高级检索
- **标签层次**：检索父标签和子标签
- **标签逻辑**：使用AND、OR、NOT逻辑
- **时间过滤**：结合时间标签过滤
- **类型过滤**：根据内容类型标签过滤
