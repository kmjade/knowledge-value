# 知識庫

### 🎯 標籤設計原則
- **层次化**：采用父子標籤结构
- **语义化**：標籤名称具有明确语义
- **標準化**：統一標籤命名規範
- **可擴展**：预留擴展空間
- **易檢索**：便于搜尋和篩選

## 🏷️ 標籤使用規範

### 📝 標籤命名规则
- **小写字母**：統一使用小写字母
- **连字符分隔**：使用连字符(-)分隔单词
- **语义明确**：標籤名称具有明确含義
- **长度适中**：標籤长度控制在合理范围
- **避免重复**：避免语义重复的標籤

### 🔗 標籤组合使用
- **主標籤+子標籤**：`#tcm/acupuncture`
- **多標籤组合**：`#clinical #safe #effective`
- **层次化標籤**：`#technique/acupuncture/manual`
- **功能性標籤**：`#priority/high #status/active`
