## 🛠️ 使用技巧

### 🔗 双向連結
- 使用 `[[筆記名称]]` 創建内部連結
- 使用 `[[筆記名称|顯示文本]]` 自定义連結文本
- 使用 `![[圖片名称]]` 嵌入圖片

### 🏷️ 標籤功能
- 使用 `#標籤名` 創建標籤
- 支持嵌套標籤 `#父標籤/子標籤`
- 標籤可以自動聚合相關內容

### 📝 模板使用
- 新建筆記时可使用预设模板
- 模板位于 `08-模板` 資料夾
- 支持日常筆記、研究筆記等多种模板
