# Obsidian

Obsidian 是一个基于 Markdown 的个人知识管理工具，在 LLM Wiki 方法论中扮演 IDE 角色。

## 基本信息
- 类型: 工具
- 状态: active

## 在 LLM Wiki 中的定位
- 作为知识库的宿主平台
- 负责 frontmatter 元数据管理
- 提供 wikilinks 双向链接机制
- 管理 Obsidian Git 版本控制
