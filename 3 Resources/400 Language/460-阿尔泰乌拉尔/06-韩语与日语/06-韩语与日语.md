# 06-韩语与日语

> 东亚最大的分类难题——韩语和日语属于哪个语系？

## 分类之争

| 阿尔泰假说: 韩语和日语有阿尔泰底层——语法相似——但同源词太少 |
|------|
| 独立语系: 许多学者将韩语和日语各自视为独立语系 |
| 南岛关联: 日语可能和南岛语系有远古联系——底层词汇 |

## 韩语

| ~8200万——韩文(训民正音·1443创制)——世界上最科学的文字之一——「聪明的早晨」 |
|------|
| 敬语系统: 极其复杂——根据说话对象选择不同的动词形式和词汇 |
| 汉字使用: 韩国减少——朝鲜完全废除——但同音词太多导致歧义 |

## 日语

| ~1.3亿——混合文字: 汉字+平假名+片假名——可能是最复杂的书写系统 |
|------|
| 敬语: 同样复杂——尊敬语·谦让语·丁宁语——社会等级的语言镜子 |
