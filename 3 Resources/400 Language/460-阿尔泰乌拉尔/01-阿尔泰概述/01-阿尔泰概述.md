# 01-阿尔泰语系概述

> 从土耳其到日本——欧亚大陆北部的语系——是否存在「阿尔泰语系」仍在争论。

## 阿尔泰假说

| 传统观点: 突厥·蒙古·通古斯三大语族有共同祖先——「阿尔泰语系」 |
|------|
| 扩展: 韩语和日语可能也属阿尔泰——争议更大 |
| 当前: 多数语言学家倾向于「语言联盟」——不是遗传——是长期接触的趋同 |

## 特征

| 特征 | 说明 |
|------|------|
| 黏着语 | 多个后缀——一个接一个——边界清晰 |
| 元音和谐 | 词内元音在前后/圆唇上保持一致 |
| SOV语序 | 动词在最后 |
| 无语法性别 | 「他」和「她」是同一个词——除非从上下文推断 |
