# 04-通古斯语族

> 西伯利亚和满洲的语言——使用者不多但分布极广。

## 主要语言

| 满语: 清朝皇室语言——今天母语者不到20人——但满文文献浩如烟海 |
|------|
| 鄂温克语(Evenki): 西伯利亚——~1.5万 |
| 赫哲语: 黑龙江——极度濒危 |

## 满语的衰落

| 18世纪: 清朝的官方语言之一——乾隆用满语写诗 |
|------|
| 19世纪: 汉族移民涌入满洲→汉语成为通用语——满语开始衰落 |
| 今天: 被联合国教科文组织列为「极度濒危」 |
