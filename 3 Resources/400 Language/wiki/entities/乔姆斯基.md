# 喬姆斯基 Noam Chomsky
美國語言學家，生成語法創始人，認知科學先驅。

## 主要貢獻
- 普遍語法 (Universal Grammar)：語言能力是天生的
- 生成語法：有限規則生成無限句子
- 深層結構 → 表層結構的轉換規則
- 最簡方案
