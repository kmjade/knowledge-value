# 索緒爾 Ferdinand de Saussure
現代語言學之父，結構主義的奠基人。

## 主要貢獻
- 語言 (langue) vs 言語 (parole) 之分
- 共時 vs 歷時研究
- 符號學：能指 (signifier) · 所指 (signified)
- 語言作為符號系統——任意性原則

## Sources
- 索緒爾《普通語言學教程》(1916)
