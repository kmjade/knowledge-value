# 語言學 Linguistics
研究人類語言的科學，涵蓋語音、語法、語義及語言演化的系統性探索。

## 核心分野
| 分野 | 研究對象 | 核心問題 |
|:----|:--------|:--------|
| **語音學** | 語音的物理屬性 | 語音如何產生/感知？ |
| **音韻學** | 語音的功能模式 | 哪些語音區別意義？ |
| **形態學** | 詞的內部結構 | 詞如何構成？ |
| **句法學** | 句子的結構 | 詞如何組合為句子？ |
| **語義學** | 意義 | 句子意義如何決定？ |
| **語用學** | 上下文中的意義 | 如何理解言外之意？ |

## 主要理論
- **結構主義語言學**：語言是符號系統（索緒爾）
- **生成語法**：普遍語法·深層結構（喬姆斯基）
- **功能語言學**：語言功能決定形式（Halliday）

## Sources

- 喬姆斯基《句法結構》(1957)
