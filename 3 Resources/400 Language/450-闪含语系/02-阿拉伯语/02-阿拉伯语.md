# 02-阿拉伯语

> 闪语族最大语言——~4亿使用者——《古兰经》的语言。

## 双言现象 (Diglossia)

| 标准阿拉伯语 (Fusha) | 书面·媒体·正式场合——所有阿拉伯人都学——但没有人日常说 |
|------|------|
| 方言 (Ammiyya) | 日常口语——埃及·黎凡特·海湾·马格里布——互相可懂度低 |

## 语言特征

| 特征 | 说明 |
|------|------|
| 书写 | 阿语字母——右到左——28个辅音字母——短元音不标 |
| 词根系统 | k-t-b(写) → 数十个相关词——系统性的派生 |
| 语法性别 | 阴阳两性——动词·形容词·代词都分 |
| 数 | 单数·双数·复数——双数是闪语的独特特征 |

## 阿拉伯语的全球影响

| 科学: 中世纪阿拉伯语是科学语言——algebra·algorithm·alcohol——都来自阿拉伯语 |
|------|
| 宗教: 《古兰经》降示的语言——18亿穆斯林的礼拜语言 |
