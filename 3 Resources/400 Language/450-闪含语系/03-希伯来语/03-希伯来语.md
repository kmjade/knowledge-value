# 03-希伯来语

> 唯一成功复兴的「死语言」——从祈祷书到日常对话的奇迹。

## 复兴史

| 公元前: 古希伯来语——圣经的语言 |
|------|
| 公元2世纪: 作为口语灭绝——但作为礼拜和文学语言存活了1800年 |
| 19世纪末: Eliezer Ben-Yehuda ——「如果可以有犹太国家——为什么不能有犹太语言？」 |
| 1948: 以色列建国——希伯来语成为官方语言——今天的母语者~900万 |

## 语言特征

| 闪语特征: 三辅音词根·右到左书写·22个辅音字母 |
|------|
| 现代希伯来语吸收了欧洲语言的句法——不是「纯」闪语了 |
| 圣经希伯来语和现代希伯来语的差异≈莎士比亚英语和现代英语 |
