# 07-闪含语法特征

> 三辅音词根——闪含语系最独特的语法创新。

## 词根-模式形态

| 词根: 三个辅音——携带核心意义——模式: 元音和附加辅音——携带语法信息 |
|------|
| 阿拉伯语 k-t-b(写): kitāb(书) · kātib(作家) · maktab(办公室) · kutub(多本书) |
| 类似乐高——词根+模式=无穷的组合 |

## 其他特征

| 特征 | 说明 |
|------|------|
| 语法性别 | 阴阳两性——动词·形容词·代词都标记 |
| 双数 | 除了单数和复数——闪语有专门的双数形式 |
| 状态 | 绝对态和构造态——「国王的书」= kitāb al-malik vs kitāb-u l-maliki |
