# 02-汉语族

> 汉语不是「一种语言」——是一族互相不可懂的语言变体——由共同的文字系统连结。

## 主要变体

| 变体 | 母语者 | 特征 |
|------|:---:|------|
| 普通话 | 9亿+ | 4声调·北方为基础——标准语 |
| 粤语 | 8500万 | 6-9声调·保留入声——香港·广东 |
| 吴语 | 8000万 | 浊音·复杂变调——上海·浙江 |
| 闽南语 | 4700万 | 最古老——台湾·福建·东南亚 |
| 客家话 | 4800万 | 迁徙历史——分布最广 |
| 湘语 | 3800万 | 新老派分化——湖南 |

## 「方言」还是「语言」？

| 语言学标准: 互相可懂性——粤语和普通话互相不可懂→不同语言 |
|------|
| 政治/文化标准: 共享文字——共同文化认同——统称「中文」 |

## 汉语统一的力量

| 共同的书写系统是汉语族统一的凝聚力——无论怎么读——「山」就是「山」 |
|------|
