# 08-汉藏比较研究

> 重建原始汉藏语——和印欧比较相比——汉藏比较还处于早期。

## 挑战

| 挑战 | 说明 |
|------|------|
| 缺少古老记录 | 印欧语有2000+年文本——汉藏语(除汉语)最早文献<1500年 |
| 形态稀少 | 汉语几乎没有屈折——可比的材料少 |
| 大量借词 | 数千年接触——同源还是借来？——难以区分 |
| 内部多样性 | 藏缅语族极多样——有些分支几乎没有研究 |

## 原始汉藏语的重建尝试

| 基本语序: 可能SOV(藏缅) → SVO(汉语·壮侗) |
|------|
| 原始汉藏语可能无声调——声调是后起的——不同的分支独立发展 |
| 「我」「你」同源: 汉语 ŋa·藏语 ŋa·缅甸语 ŋa ——最可靠的同源词之一 |
