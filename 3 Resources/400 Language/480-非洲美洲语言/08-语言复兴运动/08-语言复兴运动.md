# 08-语言复兴运动

> 能否挽救濒死语言？——希伯来语证明可能——但需要「国家意志」。

## 复兴案例

| 毛利语(新西兰): 1970s~5%毛利儿童会说→2020s~20%——语言巢(kōhanga reo) |
|------|
| 夏威夷语: 从~50母语者(1980s) → ~2000+——沉浸式学校(pūnana leo) |
| 威尔士语: 从50万人(1901)触底→现在稳定~50万——双语路牌·威尔士语媒体 |

## 复兴的要素

| 要素 | 说明 |
|------|------|
| 代际传递 | 家长必须教孩子——这是唯一的长期解决方案 |
| 官方地位 | 法律承认——学校教育——政府服务 |
| 经济价值 | 如果语言能帮你找到工作——复兴就容易 |
| 社区意愿 | 外部强迫无效——必须社区自己驱动 |
