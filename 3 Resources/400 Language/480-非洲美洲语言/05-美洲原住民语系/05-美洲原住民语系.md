# 05-美洲原住民语言

> 从阿拉斯加到火地岛——人类最后定居的大陆——语言多样性曾经极丰富。

## 殖民的冲击

| 1492前: 可能~2000种语言——今天: ~600种——且大多数极度濒危 |
|------|
| 寄宿学校: 美国和加拿大强迫原住民孩子上英语/法语学校——「杀死印第安人——拯救人」 |
| 代际断裂: 祖父母说——父母听得懂——孩子一句不会 |

## 幸存的语言

| 纳瓦霍语(Navajo): 美国最大——~17万——二战密码员使用的语言 |
|------|
| 克丘亚语: 南美最大——~800万——印加帝国的语言——秘鲁·玻利维亚官方语言 |
| 瓜拉尼语: 巴拉圭——~500万——唯一被广泛使用的原住民语言——多数巴拉圭人双语 |

## 多式综合特征

| 一个「词」可以是一个完整句: 「我在为他把那个漂亮的黑色大东西拿来」——一个动词搞定 |
