# 07-古代汉语

> 阅读先秦文本——和现代汉语是「两种语言」——但用的是同一套文字。

## 分期

| 时期 | 时代 | 特征 |
|------|------|------|
| 上古汉语 | 先秦·汉(~前1000-200 CE) | 《诗经》《论语》——单音节为主——词类灵活 |
| 中古汉语 | 南北朝·唐·宋(~200-1200) | 唐诗宋词——声调系统形成——双音节化开始 |
| 近代汉语 | 元·明·清(~1200-1900) | 白话——接近现代——《红楼梦》 |

## 古汉语的特征

| 特征 | 例子 |
|------|------|
| 词类活用 | 「春风又绿江南岸」——「绿」用作动词——「使…变绿」 |
| 宾语前置 | 「何陋之有？」=「有何陋？」 |
| 使动/意动 | 「孔子登东山而小鲁」——「以鲁为小」 |
| 省略 | 「一鼓作气——再(鼓)而衰——三(鼓)而竭」 |

## 入声的消失

| 中古汉语有入声(-p/-t/-k结尾)——普通话消失了——但粤语·闽南语·客家话保留了 |
|------|
| 「白」古读 bak ——普通话 bái ——粤语 baak6 |
| 所以唐诗用普通话读可能不押韵——用粤语读就押韵了 |
