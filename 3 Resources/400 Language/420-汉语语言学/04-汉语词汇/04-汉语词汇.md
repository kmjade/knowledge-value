# 04-汉语词汇

> 汉语的词——单音节到多音节——从古代到现代的双音节化。

## 单音节 vs 双音节

| 古汉语以单音节词为主——现代汉语以双音节词为主——「双音节化」是汉语词汇史最大的趋势 |
|------|
| 古: 学·习·朋·友 → 今: 学习·朋友——但很多单字仍独立存在 |

## 构词法

| 方法 | 说明 | 例子 |
|------|------|------|
| 并列 | 两个同义/近义词素 | 朋友·道路·学习 |
| 偏正 | 修饰+中心 | 汽车·手机·蓝天 |
| 主谓 | 主语+谓语 | 地震·心疼·年轻 |
| 动宾 | 动词+宾语 | 吃饭·睡觉·上课 |
| 补充 | 动词+结果/方向 | 看见·打破·跑来 |

## 外来词

| 方式 | 例子 |
|------|------|
| 音译 | 咖啡·沙发·巧克力·幽默 |
| 意译 | 电话·计算机·民主·科学 |
| 音+意 | 啤酒(beer——啤+酒类) |
| 和制汉语 | 日本用汉字翻译西方概念——中国反向借入: 社会·经济·哲学·自由 |

## 成语与熟语

| 成语: 四字固定——大多来自典故——「画蛇添足」「亡羊补牢」 |
|------|
| 歇后语: 前句+后句——「孔夫子搬家——净是书(输)」 |
