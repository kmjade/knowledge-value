# 漢字系統 Chinese Writing System

## 六書 Six Principles

| 類型 | 說明 | 例子 |
|------|------|------|
| **象形** | 描繪物體形狀 | 日、月、山 |
| **指事** | 抽象符號 | 上、下、一 |
| **會意** | 組合表意 | 休 (人+木) |
| **形聲** | 形旁+聲旁 (90%+) | 晴 (日+青) |
| 轉注、假借 | 用字方法 | — |

## 字體演變
甲骨文 → 金文 → 篆書 → 隸書 → 楷書 → 簡體字
