# 現代漢語演變 Modern Chinese Evolution

## 白話文運動 (1919 五四)
- 文言 → 白話
- 胡適、魯迅推動

## 簡化字 (1956/1964)
- 2,235 個簡化字
- 減少筆畫、提高識字率

## 普通話推廣
- 以北京語音為標準音
- 以北方話為基礎方言
- 以典範白話文為語法規範

## 當代趨勢
- 網絡用語: 給力、躺平、內卷
- 中英混合: "這個 project 的 deadline"
- 拼音縮寫: yyds、xswl
