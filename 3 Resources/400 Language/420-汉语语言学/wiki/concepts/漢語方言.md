# 漢語方言 Chinese Dialects

## 七大方言區
| 方言 | 代表地 | 使用者佔比 |
|------|------|:--:|
| **官話 Mandarin** | 北京 | ~70% |
| **吳語 Wu** | 上海、蘇州 | ~8% |
| **粵語 Yue** | 廣州、香港 | ~5% |
| **閩語 Min** | 福建、台灣 | ~5% |
| 客家話 Hakka | 梅州 | ~4% |
| 湘語 Xiang | 長沙 | ~3% |
| 贛語 Gan | 南昌 | ~2% |

## 差異維度
- 語音（最大差異）
- 詞彙（部分不同）
- 語法（較小差異）
