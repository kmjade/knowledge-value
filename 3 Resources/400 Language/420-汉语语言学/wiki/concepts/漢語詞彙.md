# 漢語詞彙 Chinese Lexicon

## 構詞方式
| 方式 | 例子 |
|------|------|
| **複合** | 火車 (fire+vehicle=train) |
| **派生** | 老虎 (prefix+)、桌子 (-suffix) |
| **重疊** | 媽媽、看看、紅紅的 |

## 詞彙特點
- 雙音節化趨勢（古單音→今雙音）
- 豐富的量詞系統（一隻鳥、一本書、一條魚）
- 大量成語（四字格）
