# 09-汉语教学

> 如何教汉语给非母语者？——声调和汉字是最大的挑战。

## 汉语作为第二语言的难点

| 难点 | 说明 |
|------|------|
| 声调 | 大多数语言没有声调——学习者「听不出来」「发不出来」 |
| 汉字 | 需要记忆数千个——拼音只能辅助——不能替代 |
| 语法 | 表面上简单(无屈折)——但虚词(了·着·过·把)极难掌握 |
| 成语 | 文化背景密集——「画蛇添足」的字面意思不等于实际意思 |

## HSK (汉语水平考试)

| 级别 | 词汇量 | 能力 |
|:---:|:---:|------|
| HSK 1 | 150 | 简单问候——自我介绍 |
| HSK 3 | 600 | 日常交流——旅行 |
| HSK 5 | 2500 | 读新闻——流畅交流 |
| HSK 6 | 5000+ | 接近母语理解力 |
