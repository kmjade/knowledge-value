# 05-汉语语法

> 汉语是孤立语的典型——靠语序和虚词——没有屈折变化。

## 核心特征

| 特征 | 说明 |
|------|------|
| SVO 语序 | 主语-动词-宾语——「我爱你」——基本顺序 |
| 无形态变化 | 动词不因人称/时态变化——「我吃/你吃/昨天吃」都用「吃」 |
| 虚词重要 | 「了」「着」「过」「的」「地」「得」——语法全靠它们 |
| 量词 | 数词和名词之间必须用量词——「一个人」不是「一人」 |

## 时体系统

| 标记 | 功能 | 例子 |
|------|------|------|
| 了 | 完成/变化 | 我吃了饭·天亮了 |
| 着 | 持续 | 门开着 |
| 过 | 经历 | 我去过北京 |
| 在 | 进行 | 我在吃饭 |

## 话题结构

| 汉语是「话题突显」语言——不同于英语的「主语突显」 |
|------|
| 「那本书我已经看完了」——主题=那本书——「鱼我爱吃」=「(至于)鱼——我爱吃」 |

## 「把」字句

| 把+宾语+动词——「我把饭吃完了」——处置义——宾语被「处置」了 |
|------|
| 独特的中文结构——没有欧洲语言的直接对应 |
