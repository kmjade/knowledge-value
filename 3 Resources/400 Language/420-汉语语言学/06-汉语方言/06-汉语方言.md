# 06-汉语方言

> 「方言」是政治称谓——语言学上——很多汉语方言应被视为不同语言。

## 七大(或十大)方言区

| 方言 | 使用人口 | 区域 | 特征 |
|------|:---:|------|------|
| 官话(普通话基础) | 10亿+ | 北方·西南 | 4个声调·无入声 |
| 吴语 | 8000万 | 上海·浙江·苏南 | 浊音保留·复杂连读变调 |
| 粤语 | 8500万 | 广东·香港·海外 | 6-9声调·保留入声(-p/-t/-k) |
| 闽南语 | 4700万 | 福建·台湾·东南亚 | 最古老——保留上古汉语特征 |
| 客家话 | 4800万 | 广东·福建·台湾 | 「宁卖祖宗田——不忘祖宗言」 |
| 湘语 | 3800万 | 湖南 | 浊音·新老派分化 |
| 赣语 | 4800万 | 江西 | — |

## 方言差异

| 差异维度 | 极端例子 |
|------|------|
| 语音 | 温州话对其他方言区的人来说像外语 |
| 词汇 | 「肥皂」: 胰子(北方)·番碱(客家)·雪文(sabun——闽南·借自马来) |
| 语法 | 粤语「我食咗饭先」=普通话「我先吃了饭」——副词位置不同 |

## 方言保护

| 城市化+普通话推广→方言衰退——很多「方言」正在消失 |
|------|
| 粤语和闽南语相对强壮(有媒体和海外社区) ——但儿童使用率在下降 |
