# 01-汉语概述

> 汉语 = 世界上使用人数最多的母语——~14亿人——有文字记载3000年以上。

## 汉语的基本事实

| 特征 | 说明 |
|------|------|
| 语系 | 汉藏语系——汉语族 |
| 使用人数 | ~14亿母语者——主要在中国、台湾、新加坡 |
| 文字 | 汉字(语素文字)——不是字母——每个字有独立意义 |
| 声调 | 4个声调(普通话)——声调区别意义——mā·má·mǎ·mà |
| 语法 | 孤立语——语序(SVO)——虚词重要——无屈折变化 |

## 汉语的独特性

| 特征 | 说明 |
|------|------|
| 无时态屈折 | 「了」「过」「在」表达时间——不是动词变形 |
| 量词 | 一本书·一张纸·一条鱼——英语没有的量词系统 |
| 主题突显 | 「那本书我已经看完了」——主题在句首——不同于英语主语优先 |

## 普通话 vs 方言

| 普通话(官话): 标准语——基于北京音——10亿+使用者 |
|------|
| 粤语(广东话): 9个声调——保留古汉语入声——~8500万 |
| 吴语(上海话): 浊音保留——~8000万 |
| 闽南语: 最古老的汉语分支之一——~4700万 |
