# 08-现代汉语演变

> 语言是活的——汉语在21世纪如何变化？

## 普通话的推广

| 1956: 普通话成为标准——基于北京语音——北方话为基础方言——典范现代白话文为语法规范 |
|------|
| 成功: 全国普及率从~50%(1980s)到>80%(2020s) |
| 代价: 方言快速衰退——「双语」(普通话+方言)正在变成「单语」(普通话) |

## 网络语言的冲击

| 现象 | 例子 |
|------|------|
| 字母词 | GG·YYDS·PUA·栓Q |
| 旧词新义 | 雷(震惊)·晒(炫耀)·潜水(在线但不发言) |
| 谐音 | 蓝瘦香菇(难受想哭)·杯具(悲剧) |
| 缩略 | 人艰不拆(人生艰难不拆穿) |

## 语言接触

| 英语→汉语: 借词(咖啡·沙发) · 句式(「被…」——被动式的扩展) |
|------|
| 日语→汉语: 大量学术词汇——社会·经济·哲学·共和——「和制汉语」 |
