# 05-计算语言学

> 语言学+计算机科学——让机器处理人类语言。

## 核心任务

| 任务 | 说明 |
|------|------|
| 分词 | 「我爱北京天安门」→我/爱/北京/天安门 |
| 词性标注 | 名词·动词·形容词——「我/r 爱/v 北京/ns」 |
| 句法分析 | 主谓宾——依存树——短语结构树 |
| 命名实体识别 | 识别人名·地名·组织——「马云在杭州创建了阿里巴巴」 |
| 情感分析 | 正面/负面/中性——「这菜味道还行」=？ |

## 从规则到统计到神经

| 1990s: 手写规则——if「了」then完成体 |
|------|
| 2000s: 统计方法——大量语料+概率——机器翻译革命 |
| 2010s-: 深度学习——BERT·GPT——端到端——不需要语言学知识——效果好得惊人 |
