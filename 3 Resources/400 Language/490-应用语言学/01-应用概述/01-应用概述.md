# 01-应用语言学概述

> 语言学走出象牙塔——解决真实世界的问题。

## 应用语言学的领域

| 领域 | 核心问题 |
|------|------|
| 语言教学 | 怎么最有效地教/学一门语言？ |
| 翻译 | 怎么在语言之间传递意义？ |
| 语言政策 | 国家应该用什么语言？ |
| 计算语言学 | 怎么让计算机理解和生成语言？ |
| 临床语言学 | 语言障碍怎么诊断和治疗？ |
| 法律语言学 | 这个嫌疑人的供词是他写的吗？ |
