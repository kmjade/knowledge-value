# 02-第一语言习得

> 孩子如何「长」出语言——不是「学」——是天性。

## 习得的时间线

| 年龄 | 里程碑 |
|:---:|------|
| 0-6月 | 咕咕——所有婴儿发的声音都差不多——包括失聪婴儿 |
| 6-12月 | 咿呀学语——开始像母语的声音——失聪婴儿的咿呀声减少 |
| 12-18月 | 第一个词——单字句——「奶」=「我要喝奶」 |
| 18-24月 | 词汇爆发——双词句——「爸爸车」 |
| 2-3岁 | 句子——语法爆炸——过度泛化(「我跑过的」而非「我跑了」) |
| 5岁 | 基本掌握母语语法 |

## 关键期假说

| 有一个「窗口」——如果在这个窗口内不接触语言——永远学不会 |
|------|
| Genie案例(1970): 被隔离到13岁——从未正常习得语法——支持关键期存在 |
