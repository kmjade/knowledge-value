# 05-古典汉语(文言文)

> 东亚的拉丁语——2000年的官方书面语言——中国·日本·韩国·越南的学术共同语。

## 古典汉语的特征

| 极简: 单字词为主——省略主语·介词·连词——需要读者「填补」 |
|------|
| 词类灵活: 「春风又绿江南岸」——「绿」用作动词 |
| 没有标点: 古代文本必须「断句」——这是学习古典汉语最大的挑战 |

## 汉文文化圈

| 日本·韩国·越南: 使用汉字——用汉文写作——用本地语读(训读) |
|------|
| 直到19世纪末——东亚的学术语言是文言——不是任何一种口语 |
| 日本「脱亚入欧」后——文言作为区域通用语的地位崩塌 |
