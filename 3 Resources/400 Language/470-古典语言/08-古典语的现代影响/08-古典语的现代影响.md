# 08-古典语的现代影响

> 古典语言塑造了现代词汇、思维模式和身份认同。

## 词汇的活化石

| 新概念用古典词根命名——tele+vision(希腊+拉丁) ——「远+看」= 电视 |
|------|
| 科学: 所有物种的拉丁学名——国际通用——跨越语言障碍 |
| 医学: 几乎所有解剖学术语来自拉丁/希腊 |

## 政治与身份

| 希伯来语复兴: 犹太民族国家的语言基础 |
|------|
| 印度梵语政治: 印地语净化运动——去除波斯/英语借词——用梵语词根创造新词 |
| 土耳其语言改革: 去除阿拉伯/波斯词——用「纯土耳其语」词根——但词根有限 |
