# 02-拉丁语

> 罗马帝国的语言——千年后仍然「活」在教会·法律·科学中。

## 拉丁语的历史

| 古典拉丁(~100 BCE-200 CE): 西塞罗·维吉尔——「黄金拉丁」 |
|------|
| 通俗拉丁: 士兵和商人说的话——不是古典拉丁——是它演化出了罗曼语族 |
| 中世纪拉丁: 教会·大学·科学——全欧洲的知识分子通用语——牛顿的《原理》用拉丁语 |
| 教会拉丁: 梵蒂冈官方语言至今——但弥撒现在通常用本地语言 |

## 拉丁语的特征

| 高度屈折: 名词5个格(主·属·与·宾·夺)·动词复杂变位·自由语序 |
|------|
| 无冠词——不像罗曼后代——「puella」=「女孩」或「一个女孩」或「那个女孩」 |

## 为什么学拉丁？

| 理解西方文化的根——法律·医学·植物学的术语——训练逻辑思维 |
|------|
