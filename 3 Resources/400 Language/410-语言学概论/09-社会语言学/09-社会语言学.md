# 09-社会语言学

> 语言和社会如何互相塑造——「你怎么说话」透露了「你是谁」。

## 方言 vs 语言

| 「一种语言是有陆军和海军的方言」(Weinreich) |
|------|
| 瑞典语和挪威语相互可懂——但是不同的「语言」 |
| 广东话和普通话互相不可懂——但是同一种「语言」的「方言」 |
| 政治和身份——不是语言学——决定了什么是「语言」 |

## 语言变体

| 变体 | 说明 |
|------|------|
| 地域方言 | 地理变化——美国vs英国英语 |
| 社会方言 | 阶级·教育·族群——「received pronunciation」vs Cockney |
| 语域 | 正式程度——对老板vs对朋友 |
| 语码转换 | 同一对话中切换语言/方言——双语者常见 |

## 语言与身份

| 口音 = 身份的声学标志——人们会调整口音来融入或区隔群体 |
|------|
| 语言歧视: 某些口音被看作「不聪明」——不是语言学的判断——是社会偏见 |

## 语言接触

| 洋泾浜: 简化沟通——无母语者 / 克里奥尔: 洋泾浜变成母语——完整语法 |
|------|
| 新加坡英语(Singlish) — 克里奥尔特性——但政府推行「讲正确英语运动」 |
