# 08-历史语言学

> 语言怎么随时间变化——重建逝去语言——追溯语言家族。

## 比较法

| 比较不同语言中的同源词(cognates) ——重建原始语言(proto-language) |
|------|
| 英语 father·德语 Vater·拉丁语 pater·梵语 pitṛ——都来自原始印欧语 *ph₂tḗr |
| 通过比较——可以推断原始印欧语中p变成了日耳曼语的f (格林定律) |

## 语言变化类型

| 变化 | 例子 |
|------|------|
| 语音变化 | 古英语 stān → 现代英语 stone (ā→ō) |
| 语法化 | 实词→功能词: 「去吃饭」的「去」从实义动词变成表示意图 |
| 语义变化 | nice: 愚蠢(古)→精细→好(今) |
| 借词 | 英语70%词汇是借来的(主要来自法语和拉丁语) |

## 格林定律

| 原始印欧语 | → 日耳曼语 (英语) |
|------|------|
| p | f (pater→father) |
| t | th (tres→three) |
| k | h (cord→heart) |
| d | t (decem→ten) |
| g | k (genus→kin) |
