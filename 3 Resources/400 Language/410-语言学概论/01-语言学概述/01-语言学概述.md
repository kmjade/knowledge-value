# 01-语言学概述

> 语言学 = 语言的科学研究——不是「学语言」——是研究「语言本身」——人类最独特的能力。

## 语言学的核心问题

| 问题 | 分支 |
|------|------|
| 语言的声音怎么发出？ | 语音学 |
| 声音怎么组织成有意义的系统？ | 音系学 |
| 词怎么构成？ | 形态学 |
| 词怎么组成句子？ | 句法学 |
| 意义怎么产生？ | 语义学 |
| 语境怎么影响意义？ | 语用学 |
| 语言怎么随时间变化？ | 历史语言学 |
| 语言和社会怎么互动？ | 社会语言学 |

## 索绪尔的关键区分

| 区分 | 含义 |
|------|------|
| 语言 (langue) vs 言语 (parole) | 抽象系统 vs 实际使用 |
| 共时 vs 历时 | 某一时刻的语言 vs 语言随时间的变化 |
| 能指 vs 所指 | 声音形式 vs 概念——「cat」和🐱的关系是任意的 |

## 乔姆斯基革命

| 普遍语法: 所有人类语言共享一套深层结构——孩子不是「学」语言——是「长」语言 |
|------|
| 关键证据: 孩子听到的输入有限且不完美——却能产出无限多从未听过的正确句子——「刺激贫乏」 |
