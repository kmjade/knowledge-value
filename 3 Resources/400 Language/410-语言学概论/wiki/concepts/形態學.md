# 形態學 Morphology

## 定義
形態學研究詞的內部結構和構詞規則。

## 語素 Morpheme
最小意義單位：
- **自由語素**: book, run (可獨立使用)
- **黏著語素**: -s (複數), un- (否定), -ing (進行)

## 構詞方式
| 方式 | 例子 |
|------|------|
| 派生 Derivation | happy → unhappy |
| 屈折 Inflection | walk → walked |
| 複合 Compounding | black + board = blackboard |
