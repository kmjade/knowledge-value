# 社會語言學 Sociolinguistics

## 定義
社會語言學研究語言與社會的關係——社會因素如何影響語言使用。

## 核心概念
| 概念 | 說明 |
|------|------|
| **方言 Dialect** | 地區或社會群體的語言變體 |
| **語碼轉換 Code-switching** | 在同一對話中切換語言 |
| **語言與身份** | 口音、用詞反映社會階層 |
| **語言接觸** | pidgin、creole 的產生 |
