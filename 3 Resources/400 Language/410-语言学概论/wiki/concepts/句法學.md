# 句法學 Syntax

## 定義
句法學研究句子結構——詞語如何組合成合乎語法的句子。

## Chomsky 的貢獻
- **生成語法 Generative Grammar**：有限的規則生成無限的句子
- **句法樹 Parse Tree**：句子的層級結構
- **普遍語法 Universal Grammar**：人類天生具有語言能力

## 核心概念
| 概念 | 說明 |
|------|------|
| 詞組結構 Phrase Structure | NP, VP, PP |
| X-bar 理論 | 所有詞組有統一的內部結構 |
| 移位 Movement | wh-移位、NP-移位 |
