# 語義學 Semantics

## 定義
語義學研究語言中的**意義**——詞語、短語、句子的含義。

## 核心概念
| 概念 | 說明 |
|------|------|
| **詞義關係** | 同義、反義、上下位 (dog ⊂ animal) |
| **語義特徵** | [+人類], [-生命], [+雄性] |
| **命題邏輯** | 句子的真值條件 |
| **預設 Presupposition** | "法國的國王是禿頭" 預設法國有國王 |
