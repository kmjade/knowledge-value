# 歷史語言學 Historical Linguistics

## 定義
歷史語言學研究語言如何隨時間**演變**、語言之間的**譜系關係**。

## 核心概念
| 概念 | 說明 |
|------|------|
| **語音演變** | Grimm's Law: PIE *p → Germanic *f |
| **比較法** | 通過比較同源詞重建原始語言 |
| **語言譜系樹** | 印歐語系、漢藏語系、阿爾泰語系 |
| **語法化** | 實詞 → 虛詞 (如 "把" 從動詞變為介詞) |
