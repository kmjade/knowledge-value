# 音系學 Phonology

## 定義
音系學研究語音在特定語言中如何組織成**有意義的系統**——哪些語音區別是有意義的（音位），哪些只是變體。

## 核心概念
| 概念 | 說明 |
|------|------|
| **音位 Phoneme** | 區別意義的最小語音單位 /p/ vs /b/ |
| **同位音 Allophone** | 同一音位的不同變體 [p] vs [pʰ] |
| **最小對立對** | pat vs bat → /p/ vs /b/ 在英語中是不同音位 |
