# 02-语音学

> 语言的声音——从声带到声波到耳朵——语音的物理存在。

## 发音语音学

| 发音部位 | 例子 |
|------|------|
| 双唇 | /p/ /b/ /m/ |
| 唇齿 | /f/ /v/ |
| 齿间 | /θ/ /ð/ (think, this) |
| 齿龈 | /t/ /d/ /s/ /z/ /n/ |
| 硬腭 | /ʃ/ (sh) /j/ (yes) |
| 软腭 | /k/ /g/ /ŋ/ (ng) |
| 声门 | /h/ |

## 发音方式

| 方式 | 特征 | 例子 |
|------|------|------|
| 塞音 | 完全阻塞→突然释放 | /p/ /t/ /k/ |
| 擦音 | 窄缝→摩擦 | /f/ /s/ /ʃ/ |
| 塞擦音 | 塞+擦 | /tʃ/ (ch) |
| 鼻音 | 气流从鼻腔 | /m/ /n/ /ŋ/ |
| 近音 | 接近但不摩擦 | /w/ /j/ /l/ |

## 国际音标 (IPA)

| 一个符号=一个音——跨语言通用——不需要知道拼写就知道发音 |
|------|
| 英语「fish」可拼成 ghoti (gh=tough, o=women, ti=nation) ——IPA才不会这么混乱 |
