# 07-语用学

> 语境中的意义——说的话 ≠ 传达的意思。

## Grice 的合作原则

| 准则 | 内容 |
|------|------|
| 量的准则 | 给足够的信息——不要太多——不要不够 |
| 质的准则 | 说真话——不要说没证据的 |
| 关系的准则 | 说相关的话 |
| 方式的准则 | 清楚、简洁、有序——避免歧义 |

## 会话含义

| 故意违反准则→产生「含义」 |
|------|
| 「你觉得我的新发型怎么样？」——「你今天穿的衣服真漂亮。」 |
| 故意不回答→含义 = 「我不喜欢你的发型——但我不想直接说」 |

## 言语行为 (Austin & Searle)

| 说话 = 做事 |
|------|
| 言内行为 (locutionary): 说出这句话 |
| 言外行为 (illocutionary): 说话的目的——承诺·请求·警告 |
| 言后行为 (perlocutionary): 对听者产生的效果——被说服·被吓到 |
| 「我宣布你们为夫妻」——这句话本身完成了结婚这个行为 |

## 礼貌原则

| Brown & Levinson: 面子——积极面子(被喜欢) · 消极面子(不被强加) |
|------|
| 不同文化有不同的礼貌策略——日语有完整的敬语系统——汉语用「您」 |
