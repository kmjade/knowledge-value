## 📋 学习检查清单

### 入门阶段
- [ ] 理解 OSI 七层模型和 TCP/IP 四层模型
- [ ] 掌握 IP 地址和子网划分
- [ ] 了解常用的应用层协议 (HTTP, DNS, DHCP)
- [ ] 会使用 ping, traceroute, netstat 等基础工具

### 进阶阶段
- [ ] 深入理解 TCP 的可靠传输机制
- [ ] 掌握路由协议原理 (OSPF, BGP)
- [ ] 熟悉交换机、路由器的基本配置
- [ ] 会使用 Wireshark 进行抓包分析

### 高级阶段
- [ ] 掌握网络安全基础知识
- [ ] 理解 VPN 和防火墙技术
- [ ] 熟悉 SDN 和云计算网络
- [ ] 具备网络故障排查和优化能力
