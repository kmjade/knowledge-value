# AI 常见问题

> 关于人工智能的十大常见误解——以及事实。

## 十大常见误解

| ❌ 误解 | ✅ 事实 |
|---------|---------|
| AI 会像人类一样思考 | 当前的 AI（包括 LLM）是模式匹配和统计推断，没有意识、意图或真正的理解。它们处理的是 token 概率分布，不是思维 |
| AI 会取代所有工作 | AI 会替代**任务**而非**工作**。历史表明技术改变工作的组成而非总量。但过渡期可能很痛苦，需要社会政策配套 |
| 数据越多模型越好 | 数据质量 > 数据数量。噪声数据反而降低性能。Data-centric AI 是当前的重要方向 |
| ChatGPT 知道一切 | LLM 会生成看似合理但实际错误的内容（幻觉），且对自己的知识边界没有意识。它们不是知识库，是语言模型 |
| 深度学习 = AI 的全部 | DL 只是 ML 的一个子集，ML 只是 AI 的一个子集。符号 AI、进化算法、知识图谱等方法仍然有价值 |
| AI 模型越大越好 | 模型规模收益递减。小模型的微调和优化（如 LoRA、量化）在很多任务上可达到大模型的效果 |
| 训练一个 AI 模型需要超级计算机 | 许多任务用单个 GPU 甚至 Google Colab 免费版就能完成。LoRA 微调、量化技术大幅降低了门槛 |
| AI 是黑箱，无法解释 | 可解释性 AI (XAI) 是活跃研究领域。SHAP、LIME、注意力可视化等工具让模型决策越来越可解释 |
| 有 AGI 时间表 = AI 在 X 年内超越人类 | 不同研究者对 AGI 的预测差异极大——从 5 年到 100 年不等。这些是观点，不是科学共识 |
| 开源模型不如闭源 | 2025-2026 年，开源模型（如 Llama 4、Qwen、DeepSeek）在许多基准上接近甚至匹配闭源模型 |

## AI 能力 vs AI 炒作

如何区分真正的 AI 能力和营销话术？

| 迹象 | 含义 |
|------|------|
| 演示很惊艳，但只有 demo 没有产品 | 可能只在特定条件下工作 |
| "AI-powered" 标签 | 可能只是 if-else 逻辑 |
| 不提评估指标和基准 | 没有可验证的性能数据 |
| 声称解决一切问题 | AI 是解决特定任务的工具，不是万能药 |
| 回避失败案例 | 真正理解 AI 的人会坦诚讨论局限 |

## 入门者的常见困惑

### "我需要学多少数学？"

| 目标 | 数学要求 |
|------|----------|
| 使用 AI API / 工具 | 基本不需要 |
| Fine-tune 模型 | 微积分基础、线性代数入门 |
| 理解和修改模型架构 | 线性代数、概率论、微积分 |
| 做 AI 研究 | 扎实的数学基础（以上全部 + 优化理论） |

### "Python 还是其他语言？"

**Python**。没有争议。整个 AI 生态（PyTorch、Hugging Face、LangChain）都以 Python 为中心。

### "我需要 GPU 吗？"

- 学习阶段：Google Colab 免费 GPU 足够
- 小规模训练/Fine-tune：单张消费级 GPU（RTX 3060+）即可
- 大规模训练：需要集群（公司级投入）
