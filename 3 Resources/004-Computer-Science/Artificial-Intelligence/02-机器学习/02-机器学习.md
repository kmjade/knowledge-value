# 02-机器学习

> 机器学习是 AI 的子领域，研究如何让计算机**从数据中学习**——即通过经验自动改进性能，而不需要显式编程。

## 机器学习的三种范式

| 范式 | 数据要求 | 学习方式 | 典型任务 |
|------|----------|----------|----------|
| **监督学习** | 带标签的数据 (x → y) | 学习输入到输出的映射 | 分类、回归 |
| **无监督学习** | 无标签的数据 | 发现数据中的隐藏结构 | 聚类、降维、异常检测 |
| **强化学习** | 环境反馈（奖励/惩罚） | 试错学习最优策略 | 游戏、机器人控制 |

## 核心概念

### 训练 vs 推理

| 阶段 | 输入 | 做什么 | 产出 |
|------|------|--------|------|
| **训练** | 训练数据 + 标签 | 调整模型参数，最小化损失 | 训练好的模型 |
| **推理** | 新数据（未见过的） | 用训练好的模型做预测 | 预测结果 |

### 过拟合 vs 欠拟合

| 问题 | 表现 | 原因 | 解决 |
|------|------|------|------|
| **欠拟合** | 训练集和测试集都表现差 | 模型太简单 | 增加模型复杂度、更多特征 |
| **过拟合** | 训练集性能远好于测试集 | 模型记住了噪声，学得太复杂 | 正则化、更多数据、Dropout、早停 |

> 机器学习的核心挑战不是让模型在训练集上表现好，而是**泛化**到未见过的数据。

### 偏差-方差权衡

| 概念 | 含义 | 高的情况 |
|------|------|----------|
| **偏差 (Bias)** | 模型的系统性错误 | 欠拟合——模型假设太强 |
| **方差 (Variance)** | 对训练数据微小变化的敏感度 | 过拟合——模型太灵活 |

```
总误差 = 偏差² + 方差 + 不可约误差
```

## 监督学习算法速览

### 分类算法

| 算法 | 核心思想 | 优点 | 缺点 |
|------|----------|------|------|
| **逻辑回归** | 线性模型 + sigmoid 输出概率 | 简单、可解释、快速 | 只能处理线性可分 |
| **决策树** | 树形 if-else 规则 | 高度可解释 | 容易过拟合 |
| **随机森林** | 多棵决策树投票 | 鲁棒、不易过拟合 | 模型大、训练慢 |
| **SVM** | 找到最大化间隔的超平面 | 高维数据表现好 | 大数据集慢 |
| **KNN** | 看最近的 K 个邻居投什么 | 无需训练 | 推理慢、高维退化 |
| **XGBoost/LightGBM** | 梯度提升树 | 表格数据竞赛王者 | 超参数多 |

### 回归算法

| 算法 | 输出 | 应用 |
|------|------|------|
| **线性回归** | 连续值 | 房价预测 |
| **岭回归/Lasso** | 连续值（+正则化） | 防止过拟合 |
| **多项式回归** | 非线性关系 | 曲线拟合 |

## 无监督学习

| 方法 | 目标 | 算法 | 应用 |
|------|------|------|------|
| **聚类** | 将相似数据分组 | K-Means, DBSCAN, 层次聚类 | 客户分群、图像分割 |
| **降维** | 减少特征数量，保留信息 | PCA, t-SNE, UMAP | 可视化、特征压缩 |
| **密度估计** | 估计数据分布 | GMM (高斯混合模型) | 异常检测 |

## 机器学习项目流程

```
问题定义 → 数据收集 → 数据清洗 → 特征工程 → 模型选择 → 训练 → 评估 → 部署 → 监控
```

### 数据集划分

```
完整数据集
├── 训练集 (60-80%)  ← 用于训练模型参数
├── 验证集 (10-20%)  ← 用于调超参数和早停
└── 测试集 (10-20%)  ← 只用一次！最终评估
```

### 评估指标

| 任务 | 常用指标 |
|------|----------|
| 分类 | 准确率、精确率、召回率、F1、AUC-ROC |
| 回归 | MAE、MSE、RMSE、R² |
| 聚类 | 轮廓系数、Davies-Bouldin 指数 |

## 特征工程

> "数据和特征决定了机器学习的上限，模型和算法只是在逼近这个上限。"

| 技术 | 说明 |
|------|------|
| 归一化/标准化 | 将不同范围的特征缩放到统一区间 |
| 缺失值处理 | 填充（均值/中位数/模型预测）或删除 |
| 类别编码 | One-hot、Label Encoding、Target Encoding |
| 特征组合 | 创建交互特征（如 BMI = 体重/身高²） |
| 特征选择 | 移除无关或冗余特征 |
