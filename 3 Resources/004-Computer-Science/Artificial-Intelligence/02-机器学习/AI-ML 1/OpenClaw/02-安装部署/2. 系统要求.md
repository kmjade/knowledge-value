# OpenClaw 系统要求

## Node.js 要求

| 版本 | 支持情况 |
|------|----------|
| Node 24 | ✅ 推荐 |
| Node 22.16+ | ✅ 支持 |
| Node 22 以下 | ❌ 不支持 |

### 检查 Node 版本

```bash
node --version
```

### 安装 Node.js

**官方安装**
- https://nodejs.org/

**使用 nvm（推荐）**
```bash
# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 安装 Node 24
nvm install 24
nvm use 24
```

## 操作系统要求

### Windows

| 版本 | 支持情况 |
|------|----------|
| Windows 10 | ✅ |
| Windows 11 | ✅ |
| WSL2 | ✅ 推荐 |

> [!tip] Windows 用户
> WSL2 提供更稳定的体验，推荐用于完整功能。

### macOS

| 版本 | 支持情况 |
|------|----------|
| macOS 12+ | ✅ |
| macOS 14+ | ✅ 推荐 |

支持 Intel 和 Apple Silicon（M1/M2/M3）。

### Linux

| 发行版 | 支持情况 |
|--------|----------|
| Ubuntu 20.04+ | ✅ |
| Debian 11+ | ✅ |
| Fedora | ✅ |
| Arch Linux | ✅ |

## 硬件要求

### 最低要求

| 组件 | 要求 |
|------|------|
| CPU | 2 核 |
| 内存 | 4 GB |
| 存储 | 1 GB |

### 推荐配置

| 组件 | 要求 |
|------|------|
| CPU | 4 核+ |
| 内存 | 8 GB+ |
| 存储 | 5 GB+ |

## 网络要求

- 需要访问模型提供商 API
- 如果使用国内模型，可能需要配置代理
- 消息渠道可能需要稳定的网络连接

|------|----------|
| Control UI | 18789 |
| Gateway | 18789 |

确保这些端口未被占用。
