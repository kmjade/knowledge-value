# 多模态能力

## 概述

Gemini 原生支持多种模态，可以理解和生成文本、图像、音频和视频内容。

---

## 支持的模态

| 模态 | 输入 | 输出 |
|------|------|------|
| 文本 | ✅ | ✅ |
| 图像 | ✅ | ❌ (使用 Imagen) |
| 音频 | ✅ | ❌ (使用 TTS) |
| 视频 | ✅ | ❌ (使用 Veo) |

---

## 文本处理

### 基本文本生成

```python
import google.generativeai as genai

model = genai.GenerativeModel('gemini-2.5-pro')
response = model.generate_content("写一篇关于 AI 的短文")
print(response.text)
```

### 长文本处理

```python
# 处理超长文本
with open("long_article.txt", "r") as f:
    long_text = f.read()

response = model.generate_content(f"总结以下内容：\n{long_text}")
```

### 多语言支持

| 功能 | 支持 |
|------|------|
| 语言数量 | 24+ |
| 翻译 | ✅ |
| 多语言对话 | ✅ |

## 图像处理

### 图像理解

```python
import PIL.Image

image = PIL.Image.open("photo.jpg")
response = model.generate_content(["描述这张图片", image])
```

### 支持的图像格式

| 格式 | 支持 |
|------|------|
| JPEG | ✅ |
| PNG | ✅ |
| WebP | ✅ |
| GIF | ✅ (首帧) |
| BMP | ✅ |

### 图像分析任务

| 任务 | 示例 |
|------|------|
| 描述 | "描述这张图片的内容" |
| 识别 | "图片中有哪些物体" |
| OCR | "提取图片中的文字" |
| 分析 | "分析这张图表的趋势" |
| 比较 | "比较这两张图片的差异" |

### 代码截图分析

```python
code_image = PIL.Image.open("code_screenshot.png")
response = model.generate_content([
    "识别这段代码并找出潜在问题",
    code_image
])
```

## 视频处理

### 视频理解

```python
# 上传视频
video_file = genai.upload_file(path="video.mp4")

# 等待处理
import time
while video_file.state.name == "PROCESSING":
    time.sleep(10)
    video_file = genai.get_file(video_file.name)

# 分析视频
response = model.generate_content([
    "总结这个视频的内容，列出关键事件的时间点",
    video_file
])
```

### 支持的视频格式

| 格式 | 支持 |
|------|------|
| MP4 | ✅ |
| MOV | ✅ |
| AVI | ✅ |
| WebM | ✅ |

### 视频分析任务

| 分析 | "分析这个教学视频的讲解质量" |

## 多模态组合

### 图文混合输入

```python
image1 = PIL.Image.open("before.png")
image2 = PIL.Image.open("after.png")

response = model.generate_content([
    "比较这两张图片，描述变化",
    image1,
    image2
])
```

### 视频音频混合

```python
video = genai.upload_file("video.mp4")
audio = genai.upload_file("narration.mp3")

response = model.generate_content([
    "分析视频内容和旁白的匹配程度",
    video,
    audio
])
```

## 最佳实践

### 图像处理

1. 使用适当分辨率（推荐 512x512 以上）
2. 提供清晰的指令
3. 对于 OCR，确保文字清晰可读

### 视频处理

1. 控制视频时长（推荐 < 1小时）
2. 等待处理完成再发送请求
3. 使用具体的时间点引用

### 文件上传

```python
# 检查文件状态
file = genai.get_file(file.name)
print(f"状态: {file.state.name}")

# 删除不再需要的文件
genai.delete_file(file.name)
```
