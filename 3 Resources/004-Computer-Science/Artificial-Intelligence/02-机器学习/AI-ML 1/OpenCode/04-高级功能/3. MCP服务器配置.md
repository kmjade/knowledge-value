# OpenCode MCP 服务器配置

> [!info] 概述
> MCP（Model Context Protocol）是一种开放协议，允许为 OpenCode 接入外部工具和数据源。

## MCP 类型

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| **本地（local）** | 通过命令行启动的 MCP 进程 | 本地工具和脚本 |
| **远程（remote）** | 通过 HTTPS 连接的远端服务 | 云端服务和第三方平台 |

> [!warning] 上下文消耗提醒
> 每个 MCP 服务器都会占用对话的上下文空间。建议只启用当前任务实际需要的 MCP 服务器。

---

## 基础配置结构

```json
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "my-mcp-server": {
      "type": "local",
      "enabled": true
    },
    "another-mcp-server": {
      "type": "remote",
      "url": "https://api.example.com/mcp"
    }
  }
}
```

---

## 本地 MCP 配置

```json
{
  "mcp": {
    "filesystem": {
      "type": "local",
      "command": "mcp-filesystem",
      "args": ["/path/to/allowed/dir"],
      "enabled": true
    }
  }
}
```

### 配置字段

| 字段 | 说明 |
|------|------|
| `type` | 必须为 `local` |
| `command` | MCP 服务器启动命令 |
| `args` | 命令行参数 |
| `env` | 环境变量 |
| `enabled` | 是否启用 |

---

## 远程 MCP 配置

```json
{
  "mcp": {
    "github": {
      "type": "remote",
      "url": "https://api.github.com/mcp",
      "headers": {
        "Authorization": "Bearer ${GITHUB_TOKEN}"
      },
      "enabled": true
    }
  }
}
```

### 配置字段

| 字段 | 说明 |
|------|------|
| `type` | 必须为 `remote` |
| `url` | MCP 服务器 URL |
| `headers` | HTTP 请求头 |
| `enabled` | 是否启用 |

---

## CLI 命令

```bash
# 添加 MCP 服务器
opencode mcp add

# 列出 MCP 服务器
opencode mcp list

# MCP 认证
opencode mcp auth

# MCP 调试
opencode mcp debug
```

---

## 常用 MCP 服务器

### 文件系统

```json
{
  "mcp": {
    "filesystem": {
      "type": "local",
      "command": "mcp-server-filesystem",
      "args": ["--root", "/path/to/project"]
    }
  }
}
```

### GitHub

```json
{
  "mcp": {
    "github": {
      "type": "local",
      "command": "mcp-server-github",
      "env": {
        "GITHUB_TOKEN": "${GITHUB_TOKEN}"
      }
    }
  }
}
```

### PostgreSQL

```json
{
  "mcp": {
    "postgres": {
      "type": "local",
      "command": "mcp-server-postgres",
      "args": ["postgresql://user:pass@localhost/db"]
    }
  }
}
```

## 权限控制

MCP 工具受权限配置控制：

```json
{
  "permission": {
    "filesystem_*": "ask",
    "github_*": "allow"
  }
}
```

---

## 调试 MCP

```bash
# 查看日志
opencode mcp debug

# 测试连接
opencode mcp debug --test
```

---

## 最佳实践

1. **按需启用**：只启用当前任务需要的 MCP
2. **权限控制**：对敏感 MCP 设置 `ask` 权限
3. **定期清理**：删除不再使用的 MCP 配置
4. **日志监控**：定期检查 MCP 日志
