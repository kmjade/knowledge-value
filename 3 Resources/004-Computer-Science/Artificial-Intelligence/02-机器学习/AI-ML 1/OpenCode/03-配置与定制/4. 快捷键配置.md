# OpenCode 快捷键配置

> [!info] 概述
> OpenCode 提供了丰富的快捷键，可以显著提高操作效率。

## 快捷键前缀

OpenCode 使用 `ctrl+x` 作为快捷键前缀：

- 先按 `ctrl+x`
- 然后按对应字母

例如：`ctrl+x → c` 表示先按 `ctrl+x`，松开后按 `c`

---

## 核心快捷键

| 快捷键 | 命令 | 功能 |
|--------|------|------|
| `ctrl+x → c` | `/compact` | 压缩上下文 |
| `ctrl+x → d` | `/details` | 切换详情显示 |
| `ctrl+x → e` | `/editor` | 打开外部编辑器 |
| `ctrl+x → h` | `/help` | 打开帮助面板 |
| `ctrl+x → i` | `/init` | 创建 AGENTS.md |
| `ctrl+x → l` | `/sessions` | 会话列表 |
| `ctrl+x → m` | `/models` | 列出模型 |
| `ctrl+x → n` | `/new` | 新建会话 |
| `ctrl+x → q` | `/exit` | 退出 TUI |
| `ctrl+x → r` | `/redo` | 重做操作 |
| `ctrl+x → s` | `/share` | 分享会话 |
| `ctrl+x → t` | `/themes` | 切换主题 |
| `ctrl+x → u` | `/undo` | 撤销操作 |
| `ctrl+x → x` | `/export` | 导出对话 |

---

## 代理切换

| 快捷键 | 功能 |
|--------|------|
| `Tab` | 向前切换主代理 |
| `Shift+Tab` | 向后切换主代理 |
| `<Leader>a` | 打开代理列表 |

---

## 会话导航

| 快捷键 | 功能 |
|--------|------|
| `<Leader>→` | 向前循环子会话 |
| `<Leader>←` | 向后循环子会话 |
| `<Leader>↑` | 返回父会话 |

---

## 自定义快捷键

### 配置文件

```json
{
  "keybindings": {
    "custom": "ctrl+k",
    "save": "ctrl+s",
    "format": "ctrl+shift+f"
  }
}
```

### 环境变量

```bash
export OPENCODE_KEYBINDINGS_CUSTOM=ctrl+k
```

---

## Leader 键

默认 Leader 键是 `ctrl+x`，可以自定义：

```json
{
  "keybindings": {
    "leader": "ctrl+space"
  }
}
```
