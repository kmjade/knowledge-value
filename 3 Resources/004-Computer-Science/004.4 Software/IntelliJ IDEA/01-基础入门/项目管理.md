# 項目管理

> [!summary] 功能概述
> IntelliJ IDEA 項目管理涵蓋項目的創建、導入、配置等操作，是開發工作的起點。

---

## 創建項目

### 新建項目

1. **方式一：歡迎界面**
   - 點擊「New Project」

2. **方式二：菜單欄**
   - File → New → Project

3. **配置項目**
   - 選擇項目類型（Java、Maven、Gradle 等）
   - 設置項目名稱和路徑
   - 選擇 SDK

### 項目類型

| 類型 | 說明 |
|------|------|
| Empty Project | 空項目 |
| Java | 純 Java 項目 |
| Maven | Maven 構建項目 |
| Gradle | Gradle 構建項目 |
| Spring Boot | Spring Boot 項目 |

---

## 導入項目

### 從現有代碼導入

1. File → New → Project from Existing Sources
2. 選擇項目目錄
3. 選擇構建工具（Maven/Gradle/IDEA）

### 導入方式

| 方式 | 適用場景 |
|------|----------|
| Maven | pom.xml 項目 |
| Gradle | build.gradle 項目 |
| IDEA | .idea 目錄存在的項目 |
| 從源碼 | 無構建工具的項目 |

---

## 項目結構

### .idea 目錄
- 存儲項目配置
- 包含編碼設置、庫依賴等

### 標準結構

```
project-name/
├── src/
│   ├── main/
│   │   ├── java/        # Java 源碼
│   │   └── resources/   # 資源文件
│   └── test/
│       ├── java/        # 測試代碼
│       └── resources/   # 測試資源
├── target/              # 構建輸出 (Maven)
├── pom.xml              # Maven 配置
└── .idea/               # IDEA 配置
```

---

## SDK 配置

### 添加 JDK

1. File → Project Structure → SDKs
2. 點擊「+」添加 JDK
3. 選擇 JDK 安裝路徑

### 設置項目 JDK

1. File → Project Structure → Project
2. 選擇 Project SDK

---

## 注意事項

> [!tip] 最佳實踐
> - 使用版本控制管理代碼，而非 .idea 目錄
> - 統一團隊的 JDK 版本
> - 合理設置源碼目錄標記
