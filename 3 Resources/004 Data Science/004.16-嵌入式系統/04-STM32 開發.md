# 04 STM32 開發

## STM32 產品線 (STMicroelectronics)
| 系列 | 核心 | 特點 |
|:-----|:-----|------|
| **F0** | Cortex-M0 | 低成本入門 |
| **F1** | Cortex-M3 | 經典主流 |
| **F4** | Cortex-M4 | DSP + FPU |
| **F7** | Cortex-M7 | 高效能 |
| **H7** | Cortex-M7+M4 | 雙核旗艦 |
| **L0/L4** | Cortex-M0+/M4 | 超低功耗 |
| **G0/G4** | Cortex-M0+/M4 | 新一代主流 |

## 開發環境
- **STM32CubeIDE** — 官方免費 IDE (Eclipse-based)
- **STM32CubeMX** — 圖形化配置工具 (時鐘樹、腳位)
- **HAL / LL 庫** — 硬體抽象層 vs 低層驅動

## 關鍵實戰
1. GPIO 點亮 LED → 按鍵中斷
2. UART 序列通訊 → printf 重定向
3. ADC 採集 + DMA 傳輸
4. PWM 控制馬達/舵機
5. I2C/SPI 連接感測器
