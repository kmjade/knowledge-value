# 08-消费者行为

> 人们为什么买？——不只是「需要」——心理·社会·文化。

## 消费者决策模型

| 问题识别→信息搜索→评估→购买→购后行为 |
|------|
| 大部分购买是低介入——不需要走完整流程——「看到了——买了」 |

## 影响消费的因素

| 因素 | 例子 |
|------|------|
| **心理** | 动机(Maslow)·感知·态度·学习 |
| **个人** | 年龄·职业·生活方式·个性 |
| **社会** | 家庭·参考群体·社会阶层·文化 |
| **情境** | 时间压力·环境(音乐→买更贵的酒)·心情 |

## 行为经济学的消费启示

| 默认选项: 默认选择的威力——「自动续费」 |
|------|
| 锚定: 第一个价格锚定——「原价999——现在299」 |
| 稀缺: 「仅剩3件！」——不是假的——但确实刺激购买 |
| 社会证明: 「1000人已购买」——别人买我也买 |

## 隐私与个性化

| 「我的手机在偷听我吗？」——技术上说没有——但行为定向广告让人感觉被监视 |
|------|
| 个性化 vs 隐私——「方便」的代价——GDPR·CCPA——消费者在觉醒 |
