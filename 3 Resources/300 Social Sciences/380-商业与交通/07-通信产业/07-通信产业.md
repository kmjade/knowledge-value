# 07-通信产业

> 信息的传递——从电报到5G——通信是现代社会的基础设施。

## 通信的演变

| 时代 | 技术 | 影响 |
|------|------|------|
| 1G | 模拟语音——大哥大 | 移动通信诞生 |
| 2G | 数字——短信 | 文本通信 |
| 3G | 移动互联网 | 智能手机元年 |
| 4G | 宽带移动——视频 | 短视频·直播·共享经济 |
| 5G | 低延迟——大连接 | 自动驾驶·工业4.0·IoT |

## 网络效应

| 一个网络的价值与用户数量的平方成正比 (Metcalfe定律) |
|------|
| 电话: 一个人有电话 = 没用——两个人 = 有用——越多越有用 |
| 结果: 先发优势巨大——赢家通吃——微信·WhatsApp |

## 平台经济

| 平台 = 连接双方(或多方)的中介——价值在于网络效应 |
|------|
| 交易平台: 淘宝·Uber·Airbnb |
| 社交平台: 微信·抖音·Instagram |
| 关键: 冷启动问题——「先有鸡还是先有蛋」 |
