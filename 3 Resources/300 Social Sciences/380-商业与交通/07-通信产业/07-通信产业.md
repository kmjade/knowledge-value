# 07-通信产业

## 通信的演变

| 时代 | 技术 |
|------|------|
| 1G | 模拟——大哥大 |
| 2G | 数字——短信——诺基亚 |
| 3G | 移动互联网——iPhone 时刻 |
| 4G | 视频·直播·移动支付——颠覆一切 |
| 5G | 低延迟——物联网·自动驾驶——但现在还没「杀手应用」 |
