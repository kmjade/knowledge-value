# 06-电子商务

> 在线买卖——从亚马逊到直播带货。

## 电商模式

| 模式 | 说明 | 代表 |
|------|------|------|
| B2C | 企业→消费者 | 京东·亚马逊 |
| C2C | 消费者之间 | 闲鱼·eBay |
| B2B | 企业间 | 阿里巴巴国际站 |
| D2C | 品牌直接卖给消费者——绕过中间商 | Warby Parker |
| 社交电商 | 在社交媒体中购物——直播带货 | 抖音·拼多多 |

## 电商的关键要素

| 要素 | 内容 |
|------|------|
| 流量 | 用户怎么找到你？——搜索·推荐·社交 |
| 转化 | 浏览→购买——优惠·信任·方便 |
| 履约 | 下单→收货——速度·准确·包装 |
| 售后 | 退货·客服——决定了会不会再来 |

## 直播带货

| 中国现象: 2023年直播电商GMV ~$700B——主播=新型渠道 |
|------|
| 为什么有效？——实时互动·信任关系·限时优惠·娱乐 |
| 争议: 假货·刷单·头部主播垄断——「最低价」挤压品牌利润 |
