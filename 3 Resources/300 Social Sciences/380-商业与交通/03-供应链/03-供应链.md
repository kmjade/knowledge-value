# 03-供应链

> 从原材料到消费者手中——全球化的血管。

## 供应链的环节

| 计划→采购→制造→交付→退货 |
|------|
| 每一个环节都可能断裂——COVID暴露了全球供应链的脆弱 |

## Just-in-Time vs Just-in-Case

| JIT (精益) | 最低库存——零件需要时才到——丰田模式——成本最低 |
|------|------|
| JIC (稳健) | 缓冲库存——不怕中断——成本更高 |
| 趋势：从JIT向「战略库存」回摆——供应链安全 > 效率 |

## 供应链风险

| 风险 | 例子 |
|------|------|
| 地缘政治 | 芯片——台湾→全球依赖——风险集中 |
| 自然灾害 | 泰国洪水(2011)——全球硬盘短缺 |
| 单一供应商 | 一颗芯片断供——整车停产 |
| 物流中断 | 苏伊士运河堵塞(2021)——全球贸易每天$100亿卡住 |

## 可持续供应链

| 消费者和监管要求: 供应链透明——碳足迹——劳工权益 |
|------|
| 从「最低成本」到「负责任采购」——冲突矿产·血汗工厂·碳关税 |
| 区块链溯源: 钻石·咖啡——「从矿山到手掌」——但成本高 |
