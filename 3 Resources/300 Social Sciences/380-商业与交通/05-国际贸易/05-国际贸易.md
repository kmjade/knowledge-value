# 05-国际贸易

> 跨越国界的买卖——商业的全球化。

## 为什么贸易？

| 比较优势 (Ricardo): 即使一国在所有产品上都劣于另一国——贸易仍有好处——聚焦相对擅长 |
|------|
| 竞争优势 (Porter): 不是「先天」——是后天构建的——产业集群·创新·制度 |

## 贸易壁垒

| 壁垒 | 形式 |
|------|------|
| 关税 | 对进口征税——最透明 |
| 非关税 | 配额·许可证·标准·检验——往往更隐蔽 |
| 技术壁垒 | 安全标准·环保要求——合理但可能被滥用为保护主义 |

## 贸易协定

| 层次 | 例子 |
|------|------|
| 双边 | 美国-日本·中国-澳大利亚 |
| 区域 | EU·USMCA·RCEP·CPTPP |
| 多边 | WTO——多哈回合停滞——多边主义危机 |

## 贸易战

| 2018-中美贸易战: 关税→出口下降→供应链转移→消费者付更多钱 |
|------|
| 谁赢了？——经济学家共识: 都输了——但政治逻辑不同 |
| 近岸外包 (nearshoring): 把生产搬到「附近」——墨西哥替代中国——「友岸外包」 |
