# 01-商业概论

> 商业 = 商品和服务的**交换**——买卖——连接生产者和消费者。

## 市场结构

| 结构 | 特征 | 例子 |
|------|------|------|
| 完全竞争 | 无数卖家——同质产品——价格接受者 | 农产品 |
| 垄断竞争 | 差异化——品牌——广告战 | 服装·餐厅 |
| 寡头 | 少数企业——相互依赖——价格战或共谋 | 航空·电信 |
| 垄断 | 唯一卖家——定价能力 | 公用事业 |

## 价值链

| 供应商→制造→分销→零售→消费者 | 每个环节加价值——也分利润 |
|------|------|
| 波特价值链: 主要活动(入向物流·运营·出向物流·营销·服务)+支持活动(人力·技术·采购) |

## 商业模式

| 模式 | 核心 | 例子 |
|------|------|------|
| B2C | 企业对消费者 | 零售·电商 |
| B2B | 企业对企业 | 原材料·软件SaaS |
| 平台 | 连接双方——网络效应 | Uber·Airbnb·淘宝 |
| 订阅 | 定期付费——经常性收入 | Netflix·SaaS |
| Freemium | 免费基础——付费高级 | Spotify·Dropbox |

## 商业的演变

| 时代 | 特征 |
|------|------|
| 传统 | 实体店·面对面·本地 |
| 连锁 | 标准化·规模——沃尔玛 |
| 电商 | 在线·全球·24/7——亚马逊 |
| 新零售 | 线上+线下·数据驱动·即时 |
