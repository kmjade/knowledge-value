# 04-交通运输

> 移动人和货物——商业的物理基础。

## 运输方式

| 方式 | 优势 | 劣势 | 适合 |
|------|------|------|------|
| 海运 | 成本最低——大宗 | 最慢——港口依赖 | 石油·矿石·消费品 |
| 铁路 | 中等成本——可靠 | 线路固定——转运 | 煤炭·谷物·集装箱 |
| 公路 | 门到门——灵活 | 拥堵·成本高·污染 | 最后一公里·短途 |
| 空运 | 最快 | 最贵——体积重量限制 | 高价值·时效性强 |
| 管道 | 持续·安全·低成本 | 固定——初期投资大 | 石油·天然气 |

## 最后一公里

| 物流中最贵最难的部分——从配送中心到你家门口 |
|------|
| 电商爆发→快递包裹暴增——外卖·即时配送——「30分钟达」 |
| 挑战: 成本·拥堵·环境·劳动(骑手权益) |

## 智能物流

| 技术 | 应用 |
|------|------|
| IoT传感器 | 实时跟踪——温度·震动——冷链 |
| 自动驾驶 | 长途货运·最后一公里机器人·无人机 |
| AI路线优化 | 动态规划——省油·省时 |
| 预测分析 | 提前知道需求——货物提前就位 |
