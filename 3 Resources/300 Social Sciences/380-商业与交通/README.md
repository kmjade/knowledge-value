## 🔀 DDC 380 ↔ DDC 600 边界

DDC 380 处理 **社会经济维度**（商业模式、产业政策、市场结构），DDC 600 处理 **技术/工程维度**（如何建造、如何实现）：

```
社会维度 (DDC 380)                    技术维度 (DDC 600)
──────────────────                    ──────────────────
通信产业·运营商·频谱政策     ←→       通信工程·5G·光纤 (DDC 621)
物流·供应链·贸易路线         ←→       交通工程·自动驾驶 (DDC 629)
电子商务·平台经济            ←→       Web技术·支付系统 (DDC 004-006)
商业理论·贸易政策            ←→       管理科学·运筹学 (DDC 650)
```
