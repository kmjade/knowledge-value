# 05-回归分析

> 回归 = 用一个或多个变量(X)预测另一个变量(Y)。它是社会科学和机器学习中使用最广泛的统计工具。

## 线性回归

| 模型 | Y = β₀ + β₁X + ε |
|------|------|
| β₁ | X每增加1单位——Y的平均变化 |
| R² | 模型解释的Y的变异比例——0到1 |
| 假设 | 线性·独立·正态·等方差 |

## 多元回归

| Y = β₀ + β₁X₁ + β₂X₂ + ... |
|------|
| 「控制」 | 多元回归的核心——「在其他变量不变的情况下——X₁的影响是……」 |

## 逻辑回归

| 用途 | Y是二元的(0/1)——「是否购买」「是否患病」 |
|------|------|
| 输出 | 概率——通过 logit 变换——不是直接预测Y |

## 回归的陷阱

| 陷阱 | 说明 |
|------|------|
| 相关 ≠ 因果 | 「冰淇淋销量和溺水人数相关」——不是因为冰淇淋导致溺水——是夏天 |
| 多重共线性 | X之间高度相关——β系数不稳定 |
| 过度拟合 | 太多变量——拟合了噪声 |
