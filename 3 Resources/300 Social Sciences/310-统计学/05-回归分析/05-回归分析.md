# 05-回归分析

> 量化变量之间的关系——统计建模的主力工具。

## 线性回归

| Y = β₀ + β₁X₁ + β₂X₂ + ... + ε |
|------|
| β₀: 截距——X=0时Y的预测值 |
| βᵢ: 斜率——Xᵢ每增加1单位——Y平均变化βᵢ单位 |

## R² 与解释力

| R² = 模型解释的变异 / 总变异 |
|------|
| R² = 0.8: 模型解释了80%的变异——不错 |
| ⚠️ R²高≠模型好——过拟合时R²也很高 |

## 多元回归的陷阱

| 陷阱 | 说明 |
|------|------|
| **多重共线性** | X之间高度相关→系数估计不稳定——VIF检测 |
| **遗漏变量** | 遗漏了影响Y又与X相关的变量→系数有偏 |
| **反向因果** | X→Y 还是 Y→X？——需要工具变量/实验 |

## 控制变量

| 「控制了」 = 保持其他变量不变——看X的「净」效应 |
|------|
| 例子: 「控制了教育水平后——城市居民收入仍高于农村居民」 |

## logistic 回归

| 当Y是0/1 (买/不买·存活/死亡) ——不能直接用线性回归 |
|------|
| 输出: 概率(0到1之间)——用logit链接函数 |
