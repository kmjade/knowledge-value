# 09-软件与应用

## R

| 特点 | 统计学家写的——为统计而生 |
|------|------|
| 包 | tidyverse(数据处理)·ggplot2(可视化)·lme4(混合模型)·brms(贝叶斯) |

## Python

| 特点 | 通用编程语言——统计 + ML |
|------|------|
| 包 | pandas(数据)·scikit-learn(ML)·statsmodels(统计)·PyMC(贝叶斯) |

## 应用领域

| 领域 | 典型问题 |
|------|---------|
| 医学 | 新药比安慰剂好吗？(RCT + t检验) |
| 商业 | 哪类用户更可能流失？(逻辑回归) |
| 社会科学 | 教育对收入的影响？(回归 + 因果推断) |
| AI/ML | 图像识别·推荐系统——统计学是ML的基础 |
