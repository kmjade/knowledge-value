# 02-描述统计

> 把一堆数字变成一幅画——描述统计是数据探索的第一步。

## 集中趋势

| 指标 | 公式/定义 | 何时用 |
|------|------|------|
| **均值** (Mean) | Σx/n | 对称分布——无极端值 |
| **中位数** (Median) | 中间值 | 偏态分布——有极端值(如收入) |
| **众数** (Mode) | 最频值 | 分类数据——多峰分布 |

## 离散度

| 指标 | 含义 | 说明 |
|------|------|------|
| **方差** (σ²) | 与均值偏差的平方均值 | 单位是原单位的平方——不直观 |
| **标准差** (σ) | 方差的平方根 | 恢复原单位——最常用 |
| **IQR** | Q3 − Q1 | 稳健——不受极端值影响 |
| **极差** (Range) | Max − Min | 最粗糙——仅两个点 |

## 分布形状

| 特征 | 说明 |
|------|------|
| **偏度** (Skewness) | 正偏(右尾长)——收入分布 / 负偏(左尾长)——考试成绩 |
| **峰度** (Kurtosis) | 高峰(厚尾)——金融收益 / 低峰(薄尾)——均匀分布 |

## 可视化

| 图表 | 用途 |
|------|------|
| 直方图 | 看分布——一维 |
| 箱线图 | 中位数+IQR+异常值——多组对比 |
| 散点图 | 两变量关系——寻找模式和异常 |
| 热力图 | 相关性矩阵——多维概览 |

## 相关 vs 因果

> ⚠️ 相关 ≠ 因果。冰淇淋销量↗溺水死亡↗——不是因为冰淇淋导致溺水——是「夏天」这个混杂变量。
