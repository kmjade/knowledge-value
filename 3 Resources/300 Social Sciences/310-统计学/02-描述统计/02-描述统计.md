# 02-描述统计

> 用几个数字概括一堆数据——这就是描述统计的魔法。

## 中心趋势

| 指标 | 公式/含义 | 何时用 |
|------|----------|--------|
| 均值 (Mean) | Σx/n | 对称分布——无极端值 |
| 中位数 (Median) | 中间值 | 有极端值——收入数据 |
| 众数 (Mode) | 出现最多的 | 分类数据 |

## 离散程度

| 指标 | 含义 |
|------|------|
| 方差 (σ²) | 每个值与均值的差距的平方的平均——单位是「平方」 |
| 标准差 (σ/s) | 方差的平方根——**最常用**——与原始数据同单位 |
| IQR | Q3 - Q1——中间 50% 的范围——对极端值稳健 |

## 分布形状

| 特征 | 含义 |
|------|------|
| 偏度 (Skewness) | 左偏(尾巴在左)·右偏(尾巴在右——如收入) |
| 峰度 (Kurtosis) | 尖峰 vs 扁峰——厚尾意味着更多「极端事件」 |

## 可视化

| 图 | 用途 |
|----|------|
| 直方图 | 看分布形状 |
| 箱线图 | 五数概括——中位数·四分位·极端值 |
| 散点图 | 两变量的关系 |
