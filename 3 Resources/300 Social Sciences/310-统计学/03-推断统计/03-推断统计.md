# 03-推断统计

> 从样本推断总体——统计学的"魔法"。

## 抽样分布

| 概念 | 说明 |
|------|------|
| **中心极限定理** | 无论总体分布如何——样本均值的分布趋近正态——n≥30 |
| **标准误** (SE) | σ/√n ——样本均值的标准差——n越大SE越小 |

## 置信区间

| 95% CI: x̄ ± 1.96 × SE |
|------|
| 解释: 如果我们重复抽样无数次——95%的区间会包含真实的μ |
| ⚠️ 不是「μ有95%的概率在区间内」——μ是固定的——区间是随机的 |

## 假设检验

| 步骤 | 内容 |
|------|------|
| 1 | 设 H₀ (零假设) ——「没有效应」 |
| 2 | 设 H₁ (备择假设) ——「有效应」 |
| 3 | 计算检验统计量——看数据与H₀的「距离」 |
| 4 | 计算p值——在H₀为真的假设下看数据这个极端或更极端的概率 |
| 5 | p < α (常用0.05) → 拒绝H₀ |

## p值争议

| p < 0.05 | ≠ | 效应是真的 |
|------|------|------|
| p > 0.05 | ≠ | 没有效应 |
| p值与样本量有关——大样本→微小的效应也显著 |
| ASA (2016): 「p值不能衡量效应的概率、大小或重要性」 |

## 两类错误

| | H₀ 为真 | H₀ 为假 |
|------|:---:|:---:|
| 拒绝 H₀ | 第一类(α) | ✅ |
| 不拒绝 H₀ | ✅ | 第二类(β) |
