# 01-统计学概述

> 统计学 = 从数据中学习的科学。描述统计告诉你「数据长什么样」，推断统计告诉你「从样本能对总体说什么」。

## 描述 vs 推断

| | 描述统计 | 推断统计 |
|------|---------|---------|
| 目的 | 总结数据 | 从样本推断总体 |
| 工具 | 均值·标准差·图表 | 假设检验·置信区间 |
| 例子 | 「平均身高 170cm」 | 「总体平均身高在 168-172 之间(95%置信)」 |

## 总体与样本

| 概念 | 含义 |
|------|------|
| 总体 (Population) | 你关心的**全部**——所有选民·所有产品 |
| 样本 (Sample) | 你实际测量的**部分**——随机抽取的 1000 人 |
| 参数 vs 统计量 | 总体参数(μ, σ)是固定的未知——样本统计量(x̄, s)是随机的已知 |

## 数据的类型

| 类型 | 例子 | 运算 |
|------|------|------|
| 定类 (Nominal) | 性别·颜色 | 只能数 |
| 定序 (Ordinal) | 满意度排名 | 可以排序 |
| 定距 (Interval) | 温度(°C) | 可以加减 |
| 定比 (Ratio) | 身高·收入 | 可以乘除——有绝对零点 |
