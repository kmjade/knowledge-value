# 04-推断统计

> 从样本推断总体——这是统计学的核心。你测量了 1000 个人的身高，你能说「全国人民平均身高是 X」吗？

## 假设检验

| 步骤 | 内容 |
|------|------|
| 1. 提出假设 | H₀ (零假设·无效应) vs H₁ (备择假设) |
| 2. 选检验 | t检验·卡方·ANOVA——取决于数据和问题 |
| 3. 计算p值 | 在H₀为真的前提下——观察到这样的数据(或更极端)的概率 |
| 4. 决策 | p < α(通常0.05) → 拒绝H₀ |

## p值的误解

| 误解 | 事实 |
|------|------|
| 「p值是H₀为真的概率」 | ❌ p值假设H₀为真——不是H₀的概率 |
| 「p>0.05说明没有效应」 | ❌ 只能说「证据不足以拒绝H₀」——不等于「没有」 |
| 「p<0.05说明效应很大」 | ❌ p值与**样本量**有关——大样本下微小效应也显著 |

## 置信区间

| 95% CI | 「如果我们重复抽样无数次——95%的置信区间会包含真值」 |
|------|------|
| 不是 | 「真值有95%的概率落在这个区间内」——这是贝叶斯的解读 |

## I类与II类错误

| | H₀为真 | H₀为假 |
|------|:---:|:---:|
| 拒绝H₀ | I类错误(α) | ✅ |
| 不拒绝H₀ | ✅ | II类错误(β) |
