# 08-统计软件

> 工具塑造思维——选择合适的统计软件。

## R vs Python

| | R | Python |
|------|:---:|:---:|
| 出生 | 为统计而生 | 通用编程语言 |
| 优势 | 统计建模·ggplot2·CRAN | 机器学习·生产部署·pandas |
| 学习曲线 | 陡(奇怪的语法) | 平(通用语法) |
| ggplot2 vs matplotlib | ggplot2 优雅 | matplotlib 灵活但啰嗦 |

## 其他工具

| 工具 | 适用场景 |
|------|------|
| **SPSS** | 社会科学——点菜单——不用写代码 |
| **Stata** | 经济学——计量经济学——do文件 |
| **SAS** | 医药·金融——企业级——贵 |
| **Excel** | 快速探索——但容易出错——版本控制噩梦 |

## Jupyter Notebook

| 优点: 代码+图表+文字——可复现的研究——分享方便 |
|------|
| 缺点: 隐藏状态——cell执行顺序不直观——不适合生产 |
