# 03-概率论基础

> 概率是统计学的语言——它量化了不确定性。

## 基本规则

| 规则 | 公式 |
|------|------|
| 加法 (互斥) | P(A∪B) = P(A) + P(B) |
| 乘法 (独立) | P(A∩B) = P(A)P(B) |
| 条件概率 | P(A|B) = P(A∩B)/P(B) |
| **贝叶斯定理** | P(A|B) = P(B|A)P(A)/P(B) |

## 常见分布

| 分布 | 适用 | 参数 |
|------|------|------|
| **正态** | 自然·测量误差 | μ, σ |
| **二项** | n次独立试验——成功次数 | n, p |
| **泊松** | 单位时间/空间的计数 | λ |
| **指数** | 等待时间 | λ |

## 核心定理

| 定理 | 内容 |
|------|------|
| **大数定律** | 样本越大——样本均值越接近总体均值 |
| **中心极限定理** | 无论原分布是什么——样本均值的分布→正态——n≥30 |
