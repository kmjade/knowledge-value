## 🔀 统计学在 DDC 中的位置

统计学的独特之处在于它是 **少数跨 DDC 大类的学科**：

```
DDC 310 (社会科学)
     ↑ 应用
DDC 510 (数学)           ← 数学基础（概率论·测度论）
     ↑
DDC 004 (计算机)         ← 计算实现（R·Python·Stan）
     ↓ 应用
DDC 150 (心理学)         ← 实验设计与假设检验
DDC 330 (经济学)         ← 计量经济学
DDC 370 (教育学)         ← 教育测量与评估
```
