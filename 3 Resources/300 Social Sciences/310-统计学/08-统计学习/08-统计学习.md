# 08-统计学习

> 统计学习 = 统计学 + 计算。当你有 1000 个变量——传统回归不够了——需要新的方法。

## 偏差-方差权衡

| | 偏差高 | 方差高 |
|------|--------|--------|
| 问题 | 欠拟合——太简单 | 过拟合——太复杂 |
| 解决 | 增加模型复杂度 | 正则化·更多数据 |

## 交叉验证

| k-fold CV | 把数据分成k份——每次用k-1份训练——1份测试——轮换k次——取平均 |
|------|------|
| **为什么？** | 避免「在训练集上表现好——在测试集上一塌糊涂」 |

## 正则化

| 方法 | 作用 |
|------|------|
| Ridge (L2) | 把系数**缩小**——但不归零 |
| Lasso (L1) | 把不重要的系数**归零**——自动选变量 |

## 常用算法

| 算法 | 用途 |
|------|------|
| 决策树/随机森林 | 分类——可解释 |
| SVM | 高维分类 |
| K-means | 聚类——无监督 |
