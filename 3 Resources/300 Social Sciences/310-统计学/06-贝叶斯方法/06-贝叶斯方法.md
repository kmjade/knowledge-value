# 06-贝叶斯方法

> 先验信念 + 数据 = 更新后的信念。贝叶斯不是公式——是思维方式。

## 贝叶斯定理回顾

| P(H|D) = P(D|H)×P(H) / P(D) |
|------|
| P(H): 先验——在看到数据前对H的信念 |
| P(D|H): 似然——如果H为真——看到这个数据的概率 |
| P(H|D): 后验——看到数据后对H的更新信念 |

## 频率 vs 贝叶斯

| | 频率学派 | 贝叶斯学派 |
|------|------|------|
| 概率 | 长期频率 | 信念的程度 |
| 参数 | 固定但未知 | 随机——有分布 |
| 先验 | 不存在/不承认 | 必须指定 |
| 优点 | 「客观」 | 自然整合先验知识 |
| 缺点 | p值难以解释 | 先验的主观性 |

## 先验的选择

| 类型 | 说明 |
|------|------|
| **无信息先验** | 对所有可能值一视同仁——让数据说话 |
| **信息先验** | 已有知识——来自以往研究/专家 |
| **共轭先验** | 先验和后验同分布——数学上方便 |

## MCMC (Markov Chain Monte Carlo)

| 现代贝叶斯的主力——当后验没有解析形式时——用模拟来逼近 |
|------|
| 核心思想: 在参数空间中随机游走——在高概率区域停留更久 |
| 主流工具: Stan·PyMC·JAGS |
