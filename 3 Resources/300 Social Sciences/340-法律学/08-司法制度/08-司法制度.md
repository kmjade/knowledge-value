# 08-司法制度

> 法院、法官、律师、陪审团——正义的「生产线」。

## 法院体系

| 层级 | 大陆法系 | 英美法系 |
|------|------|------|
| 初审 | 地方法院 | 地方法院(District Court) |
| 上诉 | 高等法院 | 上诉法院(Court of Appeals) |
| 终审 | 最高法院 | 最高法院(Supreme Court) |

## 法官的角色

| | 大陆法系 | 英美法系 |
|------|------|------|
| 角色 | 「公务员法官」——适用立法——职业官僚 | 「法官造法」——有更大裁量权——从律师中选拔 |
| 庭审 | 纠问式——法官主导调查 | 对抗式——双方律师对抗——法官中立裁判 |

## 陪审团 vs 参审制

| | 陪审团 (英美) | 参审制 (大陆) |
|------|------|------|
| 组成 | 12位普通公民 | 职业法官+非职业法官(如德) |
| 决定 | 陪审团定事实·法官定法律 | 一起决定 |
| 优点 | 公民参与——民主的「学校」 | 专业+民众 |
| 缺点 | 成本高·媒体影响·不给出理由 | 法官可能主导 |

## 替代性纠纷解决 (ADR)

| 类型 | 特征 |
|------|------|
| **调解** | 第三方帮助协商——无约束力 |
| **仲裁** | 第三方裁决——有约束力——比法院快 |
| **在线** (ODR) | 电商纠纷——eBay每年解决数千万案件——全自动 |
