# 07-国际法

> 国际法是「法」吗？没有世界议会·没有世界警察——但它确实在起作用。

## 国际法的来源

| 来源 | 内容 |
|------|------|
| 条约 | 国家间的「合同」——维也纳条约法公约 |
| 习惯 | 长期实践 + 法律确信——各国认为「这是法律」 |
| 一般原则 | 「诚实信用」「禁止反言」——各国承认的共通原则 |

## 核心领域

| 领域 | 内容 |
|------|------|
| 海洋法 | 领海12海里·专属经济区200海里 |
| 国际人道法 | 日内瓦公约——战争中保护平民和战俘 |
| 国际人权法 | 世界人权宣言——公民权利·政治权利·经济社会权利 |
| 国际刑法 | 种族灭绝·战争罪·反人类罪——ICC(国际刑事法院) |

## 国际法的执行力

| 没有中央执法机构 | 靠国家自愿——报复·制裁·「点名羞辱」 |
|------|------|
| 「几乎所有的国家几乎在所有时候遵守几乎所有的国际法」 | ——但违反时有发生——且不一定有后果 |
