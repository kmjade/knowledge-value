# 03-民法

> 民法 = 私人之间的法律。调整平等主体之间的人身关系和财产关系——你不需要和政府打交道，你和其他人打交道。

---

## 民法的范围

| 领域 | 内容 |
|------|------|
| **合同法** | 契约的成立·履行·违约·救济 |
| **侵权法** | 过失·故意·严格责任——「你伤害了我——赔」 |
| **物权法** | 所有权·占有·抵押——「这是我的」 |
| **家庭法** | 婚姻·离婚·子女抚养·继承 |

---

## 合同四要素

| 要素 | 说明 |
|------|------|
| **要约** (Offer) | 我愿意以 X 价格卖——明确的意愿表示 |
| **承诺** (Acceptance) | 我接受——mirrors the offer |
| **对价** (Consideration) | 双方都付出/得到有价值的东西（大陆法系用「原因」替代） |
| **意图** (Intention) | 双方有意建立法律关系 |

---

## 民法基本原则

| 原则 | 含义 |
|------|------|
| **意思自治** | 私人可以自由决定是否缔约·与谁缔约·内容为何——民法的灵魂 |
| **诚实信用** | 行使权利、履行义务应诚实守信——「帝王条款」 |
| **公序良俗** | 不违反公共秩序和善良风俗——意思自治的边界 |
| **过错责任** | 无过错则无责任——除非法律另有规定（严格责任） |

---

## 侵权 vs 犯罪

| | 侵权 (Tort) | 犯罪 (Crime) |
|------|------|------|
| 谁起诉 | 受害者（私人） | 国家（公诉） |
| 结果 | 赔偿（金钱） | 惩罚（监禁·罚款） |
| 证明标准 | 优势证据（>50%） | 排除合理怀疑（>95%） |
| 目的 | 填补损害 → 恢复原状 | 惩罚·威慑·改造 |

---

## 两大法系中的民法

| | 大陆法系 (Civil Law) | 英美法系 (Common Law) |
|------|------|------|
| 法律渊源 | 成文法典——如《法国民法典》《德国民法典》 | 判例法为主——法官造法 |
| 思维方式 | 演绎——从法条到案件 | 归纳——从判例中找规则 |
| 物权 | 物权法定原则——种类和内容由法律规定 | 衡平法上的财产权——更灵活 |
| 合同 | 「原因」理论 | 「对价」理论 |

> 物权法定原则：物权的种类和内容由法律规定，不能自由创设。所有权 = 占有·使用·收益·处分——最完整的物权。
