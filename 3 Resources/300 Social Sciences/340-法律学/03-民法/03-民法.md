# 03-民法

> 民法 = 私人之间的法律——你不需要和政府打交道——你和其他人打交道。

## 民法的范围

| 领域 | 内容 |
|------|------|
| **合同法** | 契约的成立·履行·违约·救济 |
| **侵权法** | 过失·故意·严格责任——「你伤害了我——赔」 |
| **物权法** | 所有权·占有·抵押——「这是我的」 |
| **家庭法** | 婚姻·离婚·子女抚养·继承 |

## 合同四要素

| 要素 | 说明 |
|------|------|
| **要约** (Offer) | 我愿意以X价格卖——明确的意愿表示 |
| **承诺** (Acceptance) | 我接受—— mirrors the offer |
| **对价** (Consideration) | 双方都付出/得到有价值的东西 |
| **意图** (Intention) | 双方有意建立法律关系 |

## 侵权 vs 犯罪

| | 侵权 (Tort) | 犯罪 (Crime) |
|------|------|------|
| 谁起诉 | 受害者(私人) | 国家(公诉) |
| 结果 | 赔偿(金钱) | 惩罚(监禁·罚款) |
| 标准 | 优势证据(>50%) | 排除合理怀疑(>95%) |

## 物权

| 物权法定原则——物权的种类和内容由法律规定——不能自由创设 |
|------|
| 所有权: 占有·使用·收益·处分——最完整的物权 |
