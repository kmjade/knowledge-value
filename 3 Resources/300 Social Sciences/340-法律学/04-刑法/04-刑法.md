# 04-刑法

> 国家定义犯罪并施加惩罚——刑法是最"重"的法律——涉及自由甚至生命。

## 犯罪构成

| 要素 | 拉丁术语 | 说明 |
|------|------|------|
| **行为** | Actus Reus | 有犯罪行为——自愿的身体行为 |
| **意图** | Mens Rea | 有犯罪意图——故意·明知·轻率·疏忽 |
| **无抗辩** | — | 无正当防卫·无紧急避险·无精神病 |

## 刑罚的目的

| 目的 | 说明 |
|------|------|
| **报应** (Retribution) | 罪有应得——「以眼还眼」——道德直觉 |
| **威慑** (Deterrence) | 让别人不敢犯罪(一般)·让罪犯不敢再犯(特殊) |
| **隔离** (Incapacitation) | 把危险的人和社会隔离——监狱 |
| **改造** (Rehabilitation) | 教育·培训——让罪犯回来做守法公民 |

## 刑事程序

| 阶段 | 内容 |
|------|------|
| 侦查 | 警察收集证据——搜查令·逮捕 |
| 起诉 | 检察官决定是否起诉——自由裁量 |
| 审判 | 排除合理怀疑——无罪推定——被告有权保持沉默 |
| 量刑 | 法官根据法律和案件情况决定刑罚 |

## 程序正义

| 米兰达警告: 「你有权保持沉默——你说的任何话都可能成为呈堂证供——你有权请律师…」 |
|------|
| 非法证据排除: 毒树之果——非法获取的证据不能用 |
