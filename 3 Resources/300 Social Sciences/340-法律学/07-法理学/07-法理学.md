# 07-法理学

> 「法律是什么？」——这不是一个简单的问题。

## 自然法

| 核心: 法律的有效性取决于其道德内容——「恶法非法」 |
|------|
| 古典: 阿奎那——自然法是上帝理性在人类世界的参与 |
| 现代: Fuller——法律的「内在道德」——八项原则(公开·不溯及·清晰…) |
| 批评: 谁的「自然」？谁的「道德」？——太模糊 |

## 法律实证主义

| 核心: 法律的有效性与道德无关——「恶法亦法」——法律是社会事实 |
|------|
| Austin: 法律 = 主权者的命令——以制裁为后盾 |
| Hart: 初级规则(义务)+次级规则(承认·改变·裁判)——「承认规则」是法律体系的终极标准 |
| 批评: 纳粹德国的「法律」——难道我们只能说「这很糟糕——但它是法律」？ |

## 德沃金的批评

| 法律不仅包括规则——还包括**原则**——原则有「重量」(weight) ——需要权衡 |
|------|
| 「疑难案件」: 法官不是在「创造」法律——是在「发现」法律——通过建构性解释 |

## 法律现实主义

| 核心: 法律不是纸上的规则——是法官实际做什么 |
|------|
| 霍姆斯: 「法律的生命不在于逻辑——而在于经验」 |
| 「坏人的视角」: 如果你是一个坏人——你关心的不是道德——是法官会怎么判你 |
