# 01-法律学概述

> "Where there is no law, there is no freedom." — John Locke

## 法的本质

| 理论 | 核心 |
|------|------|
| **自然法** | 法律之上有更高的「自然正义」——不正义的法律不是法律 |
| **实证法** | 法律 = 主权者制定的规则——有效≠道德 |
| **法律现实主义** | 法律不是纸上的——是法官实际怎么判的 |

## 世界主要法系

| 法系 | 特征 | 代表国家 |
|------|------|------|
| **大陆法系** (Civil Law) | 成文法典——法官适用法律——不创造先例 | 法·德·日·中 |
| **英美法系** (Common Law) | 判例法——法官通过判决「造法」——遵循先例(stare decisis) | 英·美·加·澳 |
| **伊斯兰法系** | 沙里亚——古兰经+圣训 | 沙特·伊朗·部分中东 |
| **混合法系** | 大陆+英美+习惯法混合 | 南非·以色列·菲律宾 |

## 法律的分类

| 分类 | 内容 |
|------|------|
| **公法 vs 私法** | 公法: 国家与个人(宪法·刑法·行政法) / 私法: 个人之间(民法·商法) |
| **实体法 vs 程序法** | 实体: 规定权利义务 / 程序: 规定如何实现权利义务 |

## 法律渊源

| 渊源 | 说明 |
|------|------|
| 宪法 | 最高法——一切法律不能与之抵触 |
| 法律 | 立法机关制定的规范性文件 |
| 行政法规 | 行政机关的授权立法 |
| 判例 | 英美法系的核心——大陆法系的参考 |
| 习惯 | 长期形成的社会规范——被法律承认 |
