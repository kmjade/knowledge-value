# 01-法律学概述

## 法的特征

| 特征 | 内容 |
|------|------|
| 规范性 | 规定「应当」如何——不是描述「实际」如何 |
| 强制性 | 有国家暴力为后盾 |
| 普遍性 | 适用于所有人——不是针对个人 |

## 两大法系

| | 大陆法系 | 英美法系 |
|------|---------|---------|
| 来源 | 成文法典 | 判例 (stare decisis) |
| 法官角色 | 适用法典 | 「法官造法」 |
| 地区 | 欧洲大陆·中国·日本 | 英国·美国·澳大利亚 |

## 法的渊源

| 渊源 | 内容 |
|------|------|
| 宪法 | 最高法律——所有法律必须合宪 |
| 法律 | 立法机关制定 |
| 行政法规 | 行政机关根据授权制定 |
| 判例 | 英美法系中具有约束力 |
| 习惯 | 长期实践中形成——最古老的法源 |
