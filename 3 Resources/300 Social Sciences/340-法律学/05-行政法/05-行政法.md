# 05-行政法

> 约束政府权力的法律——当政府做错了——公民如何救济。

## 行政法的核心问题

| 如何确保庞大的行政机器依法运作——而不变成「依法行政」的遮羞布——用来证明任何行为的合法性？ |

## 依法行政原则

| 原则 | 内容 |
|------|------|
| **法律优位** | 行政行为不得违反法律——「消极」——不违背就行 |
| **法律保留** | 重要事项必须由法律(而非行政法规)规定——「积极」 |

## 行政行为的司法审查

| 审查维度 | 内容 |
|------|------|
| **合法性审查** | 行政机关是否超越了法律授予的权力？ |
| **合理性审查** | 决定是否「如此不合理以至于任何合理的机构都不会做出」？ |
| **程序审查** | 是否遵循了正当程序？——听证·告知·说明理由 |

## 比例原则

| 比例原则是行政法的「帝王条款」 |
|---|
| 1. 适当性: 手段是否有助于目的实现？ |
| 2. 必要性: 是否有更少侵害的手段？ |
| 3. 狭义比例: 手段的侵害与目的的利益是否成比例？ |
