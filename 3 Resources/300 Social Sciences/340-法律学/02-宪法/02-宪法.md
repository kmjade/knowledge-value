# 02-宪法

> 宪法 = 国家的根本大法——约束权力的权力。

## 宪法的功能

| 功能 | 内容 |
|------|------|
| **组织权力** | 建立政府机构——分配职权——设定程序 |
| **限制权力** | 基本权利不可侵犯——通过制衡防止暴政 |
| **象征** | 国家认同的标志——「我们人民」 |

## 宪法的类型

| 维度 | 类型 | 例子 |
|------|------|------|
| 形式 | **成文** (codified) | 美国·德国·中 |
| | **不成文** (uncodified) | 英国——宪法惯例+法律+判例 |
| 修改 | **刚性**——修宪困难 | 美国(2/3国会+3/4州) |
| | **柔性**——修宪简单 | 英国(普通立法即可) |

## 基本权利

| 权利类型 | 内容 |
|------|------|
| **自由权** | 言论·信仰·集会·迁徙——国家不得干涉 |
| **平等权** | 法律面前人人平等——禁止歧视 |
| **社会权** | 教育·医疗·社会保障——国家有义务提供 |
| **程序权** | 公正审判·无罪推定·反酷刑 |

## 权力分立

| 分支 | 职能 | 制衡 |
|------|------|------|
| **立法** (国会) | 立法律·批预算 | 弹劾·调查·否决 |
| **行政** (总统/内阁) | 执法·外交 | 否决权(立法的制约) |
| **司法** (法院) | 解释法律·违宪审查 | 宣告立法/行政违宪 |

## 违宪审查

| 模式 | 说明 | 例子 |
|------|------|------|
| **美国模式** | 普通法院在具体案件中审查 | 马伯里诉麦迪逊(1803) |
| **欧陆模式** | 专门的宪法法院——抽象审查 | 德国·法国·中 |
| **英国模式** | 弱审查——法院宣告「不相容」——议会决定改不改 | 英国《人权法》 |
