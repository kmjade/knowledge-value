# 04-民法

> 「民法是市民社会的宪法。」——它调整平等主体之间的财产关系和人身关系。

## 民法典的结构 (德国民法典式)

| 编 | 内容 |
|----|------|
| 总则 | 人·物·法律行为·代理·时效 |
| 物权 | 所有权·抵押权·占有——对物的直接支配 |
| 债权 | 合同·侵权·不当得利·无因管理 |
| 亲属 | 婚姻·亲子·收养·扶养 |
| 继承 | 法定继承·遗嘱·遗赠 |

## 合同

| 要素 | 要约 + 承诺 = 合意 |
|------|------|
| 违约 | 不履行→损害赔偿——或实际履行 |

## 侵权

| 原则 | 过错责任——「无过错则无责任」——但现代有无过错责任(产品·危险活动) |
|------|------|
| 赔偿 | 恢复原状——或金钱赔偿 |

## 物权 vs 债权

| 物权 | 对世权——对抗所有人——排他 |
|------|------|
| 债权 | 对人权——只对抗特定债务人 |
