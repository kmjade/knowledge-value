# 06-国际法

> 国际法是「法」吗？没有世界警察——但国家大体遵守它。

## 国际法的特殊性

| 国内法 | 国际法 |
|------|------|
| 立法机关·警察·法院 | 无世界立法机关·无世界警察·无强制管辖权 |
| 强制执行力 | 靠国家同意——靠互惠——靠声誉 |

## 国际法的渊源

| 渊源 (ICJ规约第38条) | 说明 |
|------|------|
| **条约** | 国家之间的书面协议——最重要的渊源 |
| **习惯国际法** | 国家实践+法律确信——普遍·一致 |
| **一般法律原则** | 各国法律体系共有的原则——如诚实信用 |
| **判例和学说** | 辅助渊源 |

## 国际法的核心领域

| 领域 | 内容 |
|------|------|
| **国际人权法** | 公民权利·政治权利·经济社会文化权利 |
| **国际人道法** | 战争法——日内瓦公约——保护平民和战俘 |
| **国际刑法** | 灭绝种族·战争罪·危害人类罪——ICC |
| **海洋法** | 领海(12海里)·专属经济区(200海里)·公海 |

## 国际法的执行困境

| 国际法院(ICJ): 只能审理国家同意的案子——判决靠自觉 |
|------|
| 安理会: 大国否决权——五常之一反对就暂停 |
| 现实: 权力仍然是最重要的——法律为权力提供了正当性 |
