# 06-行政法

> 行政法控制政府——确保它依法行事，保护公民免受行政权力的侵害。

## 核心原则

| 原则 | 内容 |
|------|------|
| 法律优先 | 行政行为不能违反法律 |
| 法律保留 | 某些事项必须有法律明确授权——不能行政机关自己说了算 |
| 比例原则 | 手段和目的之间要成比例——不能用大炮打蚊子 |

## 行政行为

| 类型 | 内容 |
|------|------|
| 行政许可 | 批准或拒绝——「你可以建这栋楼」 |
| 行政处罚 | 罚款·吊销执照 |
| 行政强制 | 强制执行——拆除违建 |

## 司法审查

| 法院审查行政行为是否合法——不是审查「好不好」——是审查「合不合法」 |
|------|
| 越权无效 | 行政机关做的事情超出了法律授权→无效 |
