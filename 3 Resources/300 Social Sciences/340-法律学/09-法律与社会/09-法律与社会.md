# 09-法律与社会

> 法律不是孤立的——它在社会中运作——被社会塑造——也塑造社会。

## 法律与社会变迁

| 方向 | 例子 |
|------|------|
| 社会变迁→法律变化 | 工业化→劳动法·环保法 |
| 法律→社会变迁 | Brown v. Board → 废除种族隔离 |
| 法律落后于社会 | AI创作——版权法还在追 |

## 法律与社会运动

| 法律既是压迫的工具——也是解放的工具 |
|------|
| 民权运动: 利用法院挑战种族隔离——Brown v. Board (1954) |
| LGBTQ+权利: 从刑事化到婚姻平等——Obergefell v. Hodges (2015) |
| 策略: 「法律动员」——把社会议题转化为法律语言——进入法院 |

## 科技与法律的碰撞

| 领域 | 挑战 |
|------|------|
| **AI与责任** | 自动驾驶撞了人——谁负责？程序员？车主？AI？ |
| **隐私** | GDPR——被遗忘权——数据是「新石油」 |
| **区块链与智能合约** | Code is Law? ——代码能替代法律吗？ |
| **算法歧视** | AI招聘工具歧视女性——谁来监管算法？ |

## 法律与经济发展

| 「法治」是经济增长的关键条件 (Douglass North) |
|------|
| 产权保护→投资激励→增长 |
| 但「纸上的法治」≠「实践中的法治」——关键在于执行 |
