# 05-刑法

> 国家 vs 个人——刑法是最严厉的法律——它剥夺自由甚至生命。

## 犯罪构成

| 要素 | 内容 |
|------|------|
| 构成要件该当性 | 行为符合法律规定的犯罪描述 |
| 违法性 | 没有正当防卫·紧急避险等违法阻却事由 |
| 有责性 | 有责任能力——不是精神病人——有过错 |

## 罪刑法定

| 「法无明文规定不为罪——法无明文规定不处罚」 |
|------|
| 禁止事后法·禁止类推·禁止不确定的刑罚 |

## 刑罚的目的

| 理论 | 核心 |
|------|------|
| 报应 | 你做了坏事——所以你应该受苦——「应得的」 |
| 威慑 | 让潜在犯罪者不敢——一般预防+特殊预防 |
| 改造 | 让罪犯变成守法公民 |
| 隔离 | 把危险的人和社会隔开 |

## 正当程序
