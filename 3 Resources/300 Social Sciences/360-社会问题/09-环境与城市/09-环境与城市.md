# 09-环境与城市

> 到2050年——68%的人口将居住在城市——城市是环境问题的原因——也是解决方案。

## 环境正义

| 环境污染不成比例地影响穷人和少数族裔——「环境种族主义」 |
|------|
| 垃圾填埋场·化工厂·高速公路——总是在「弱势」社区 |
| 这不是偶然——是土地便宜+政治权力弱的结果 |

## 城市化的挑战

| 挑战 | 说明 |
|------|------|
| **住房** | 可负担住房危机——「士绅化」——驱逐原住民 |
| **交通** | 拥堵·污染——私家车 vs 公共交通——「15分钟城市」 |
| **非正规居住** | 全球~10亿人住在贫民窟——水电·卫生·安全 |
| **隔离** | 富人和穷人住在不同世界——空间上的不平等 |

## 气候正义

| 碳排放最少的人(全球南方)承担气候变化的最高代价 |
|------|
| 富裕国家造成的气候问题——贫穷国家买单——代际不公平——当代人消耗·后代人承受 |
| 「只是过渡」(Just Transition): 绿色转型不能牺牲工人和边缘社区 |

## 可持续城市

| 紧凑·混合用途·公共交通导向——哥本哈根·新加坡 |
|------|
| 不是「回到乡村」——是高密度城市+绿色空间+可再生能源 |
| 挑战: 钱·政治意愿·既得利益(房地产·汽车产业) |
