# 04-种族与歧视

> 种族是社会建构的——但其影响是真实的和毁灭性的。

## 种族的社会建构

| 生物学上——人类基因差异的绝大部分(>85%)——在「种族」内部而非之间 |
|------|
| 但「种族」作为社会类别——塑造了机会·待遇·身份——真实的社会后果 |
| 「如果你被当作某一种族对待——你就『是』那个种族」 |

## 歧视的类型

| 类型 | 说明 |
|------|------|
| **个人歧视** | 个体的偏见行为——公开或微妙 |
| **制度歧视** | 组织规则和程序——表面上中立——实际偏向某些群体 |
| **结构歧视** | 历史累积的不平等——不需要任何个人「歧视」——系统自动复制 |

## 特权 (Privilege)

| 特权 = 不是因为「你做了什么」——而是因为「你是什么」——而获得的系统性优势 |
|------|
| 如: 白人不必向孩子解释「警察可能因为你的肤色而对你不同」 |
| 特权不等于「你的人生很容易」——而是「你的种族没有让它更难」 |

## 平权行动 (Affirmative Action)

| 目的: 纠正历史和结构的不平等——在教育和就业中给少数群体优先 |
|------|
| 支持: 没有积极干预——平等只是纸上的 |
| 反对: 「逆向歧视」——违反功绩原则——污名化受益者 |
| 美国最高法院2023: 禁止大学招生中的种族考量 |
