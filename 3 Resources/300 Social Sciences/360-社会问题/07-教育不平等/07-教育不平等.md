# 07-教育不平等

> 教育本应是伟大的平等器 (equalizer) ——但往往它复制了不平等。

## 不平等的来源

| 来源 | 影响 |
|------|------|
| **家庭背景** | 父母的收入·教育·「文化资本」——是学业成就最强的预测器 |
| **学校质量** | 富裕地区的学校——更好的教师·设施·资源——「学区房」 |
| **同伴效应** | 你和谁一起上学很重要——影响动力·期望·行为 |

## 教育再生产 (Bourdieu)

| 学校不是中立的——它偏好中产阶级的「文化资本」——并把它伪装成「天赋」 |
|------|
| 上流阶级的品味·语言·行为——在学校被奖励——被当作「聪明」 |
| 工人阶级的孩子——不是「笨」——是游戏规则不是为他们设计的 |

## PISA 与全球比较

| OECD的PISA: 15岁学生数学·阅读·科学——每三年 |
|------|
| 芬兰模式: 无标准化考试·教师高度专业·平等第一 |
| 上海/新加坡: 高分——但高压·补习·睡眠不足 |
| 核心发现: 教育公平和卓越可以并存 |

## 高等教育

| 大学扩招→「文凭通胀」——学士不如以前值钱——但不上大学更难 |
|------|
| 学生贷款: 美国$1.7万亿——拖累一代人的经济前景 |
| 第一代大学生: 进了大学——但缺少「隐性课程」——退学率高 |
