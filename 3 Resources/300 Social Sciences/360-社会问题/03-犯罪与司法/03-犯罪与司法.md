# 03-犯罪与司法

> 犯罪是社会秩序的裂缝——司法既是对裂缝的修补——也可能扩大它。

## 犯罪的解释

| 理论 | 核心 |
|------|------|
| **紧张理论** (Merton) | 社会目标(财富)人人都想要——但合法手段不均衡——张力→犯罪 |
| **社会控制** (Hirschi) | 为什么不犯罪？——与社会的联结(依恋·投入·参与·信念) |
| **标签理论** | 被贴上「罪犯」标签的人——更难回归——变成了标签预测的人 |
| **破窗理论** | 小混乱(破窗)不修→信号:「没人管」→更严重的犯罪 |
| **日常活动** | 犯罪 = 有动机的犯罪者+合适的靶子+缺乏有效守护——机会 |

## 大规模监禁 (Mass Incarceration)

| 美国: 全球监禁率最高——占世界人口5%——占世界囚犯25% |
|------|
| 原因: 毒品战争·三振出局法·量刑指南·私人监狱利润 |
| 后果: 少数族裔不成比例——家庭破碎——重返社会难——「重犯循环」 |

## 恢复性司法

| 替代报应性司法: 不是「你犯了罪——惩罚你」 |
|------|
| 受害者-犯罪者-社区三方对话——修复伤害——道歉——赔偿——和解 |
| 证据: 再犯率低于传统司法——受害者满意度更高 |
