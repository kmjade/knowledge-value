# 03-犯罪与司法

## 犯罪学理论

| 理论 | 核心 |
|------|------|
| 理性选择 | 犯罪 = 成本收益分析——被抓概率×惩罚 < 收益 → 犯罪 |
| 社会学习 | 犯罪是学来的——周围有人犯罪——你也学 |
| 紧张理论 | 社会设定的目标(财富)——合法手段不够——走非法途径 |
| 标签理论 | 「罪犯」标签 → 自我实现 → 真成罪犯 |

## 监禁

| 美国 | 全球最高监禁率——占世界人口5%——占世界囚犯25% |
|------|------|
| **大规模监禁**不只关人——摧毁家庭·社区——种族不成比例 |

## 恢复性司法

| vs 报应性司法 | 不是「你犯了罪→惩罚你」——是「你造成伤害→修复伤害」 |
|------|------|
| 过程 | 受害者·罪犯·社区——对话——赔偿——恢复 |
