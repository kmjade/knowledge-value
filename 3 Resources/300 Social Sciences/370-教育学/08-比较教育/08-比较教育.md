# 08-比较教育

> 别的国家怎么做教育？——我们可以学什么——但不能简单复制。

## 教育系统对比

| 国家 | 特征 | 优势 | 挑战 |
|------|------|------|------|
| **芬兰** | 晚入学(7岁)·无标准化考试·教师硕士·平等第一 | 高公平·高成就 | 同质性高(难复制) |
| **新加坡** | 早期分流·双语·高强度·高投入 | PISA顶尖 | 压力大·补习军备 |
| **日本** | 集体主义·午餐值日·塾产业 | 高均等·公民性 | 创造力·个体性 |
| **美国** | 分权·地方控制·巨大不平等 | 顶尖大学·创新 | 系统性不平等 |

## 政策借鉴 (Policy Borrowing)

| 为什么「借鉴」常常失败？ |
|------|
| 去语境化: 把芬兰模式搬到美国——但芬兰有强大的教师工会、高社会信任、小规模 |
| 符号性借鉴: 政治家借鉴别国的「语言」来推进自己已有的议程——而非真正学习 |
| 「PISA 旅游」: 各国教育部长飞往芬兰——回来做报告——然后一切照旧 |

## 全球教育趋势

| 趋势 | 内容 |
|------|------|
| **标准化** | PISA·共同核心——全球教育趋于相似 |
| **私人化** | 私立学校·补习·EdTech——教育越来越市场化 |
| **STEM焦虑** | 各国担忧STEM人才不足——但数据不明确 |
