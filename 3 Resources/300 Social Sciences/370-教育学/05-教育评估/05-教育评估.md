# 05-教育评估

> 「不是所有重要的都可以测量——不是所有可测量的都重要。」

## 形成性 vs 总结性

| | 形成性 | 总结性 |
|------|------|------|
| 目的 | 改善学习——反馈 | 判断——打分·排名 |
| 时机 | 学习过程中 | 学习结束后 |
| 例子 | 课堂提问·反馈·小测 | 期末考试·高考 |
| 比喻 | 厨师尝汤 | 客人品汤 |

## 评估的质量

| 标准 | 含义 |
|------|------|
| **效度** | 测到了想测的东西吗？——高考数学测的是数学能力还是应试技巧？ |
| **信度** | 换一天考——分数差不多吗？——作文评分——不同老师给同一篇不同分 |
| **公平** | 对所有背景的学生都公平吗？——文化偏差——SAT词汇偏向白人中产 |

## 标准化测试的争议

| 支持 | 客观·可比·问责——「我们怎么知道学校在教？」 |
|------|------|
| 反对 | 窄化课程(只教考的)·应试教学·高利害→作弊·压力·「为考而教」 |
| Campbell定律 | 「一个社会指标被用于决策——它就不再是好的指标」——人会操控 |

## 替代评估

| 档案袋 | 收集学生一段时间的作品——展示成长——比一次考试更有意义 |
|------|------|
| 表现评估 | 做一件真实的事——写一篇论文·做一个项目——而非选择题 |
| 自我/同伴评估 | 培养元认知——但学生可能不诚实或不准确 |
