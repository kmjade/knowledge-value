# 05-教育评估

## 评估的类型

| 类型 | 内容 |
|------|------|
| 形成性 | 在学习过程中——为改进——「你哪里还没懂？」 |
| 总结性 | 学期末——考试——为判断——「你学会了多少？」 |
| 诊断性 | 学习前——找问题——「你缺什么前提知识？」 |

## 标准化考试

| 优点 | 客观·可比·大规模——「公平」 |
|------|------|
| 问题 | 窄化课程——「教考试」——应试——创造性被牺牲——文化偏见(语言·背景) |

## 信度 vs 效度

| 信度 | 一致性——再测一次——分数差不多？ |
|------|------|
| 效度 | 你测的真是你想测的吗？——数学考试要读很长的英语——测的是阅读——不是数学 |
