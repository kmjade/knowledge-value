# 03-课程与教学

> 教什么(课程)和怎么教(教学)——两翼。

## 课程类型

| 类型 | 说明 |
|------|------|
| **显性课程** | 写在课表上的——数学·语文·科学 |
| **隐性课程** | 不教而教——排队·服从·竞争·性别角色 |
| **空无课程** (Null) | 学校不教的——为什么？——女性历史·工人运动·批判性媒体素养 |

## 课程设计

| Tyler 原理 (1949) | 四个问题: 目标→经验→组织→评价 |
|------|------|
| 螺旋课程 (Bruner) | 同一主题在不同年级重复——越来越深——「任何学科都可以以智力上诚实的方式教给任何年龄的孩子」 |

## 教学方法

| 方法 | 特征 | 适合 |
|------|------|:---:|
| **直接教学** | 教师讲——结构化——明确——高效 | 基础知识·技能 |
| **探究式** | 学生提出问题——自己探索 | 深层理解·科学 |
| **合作学习** | 小组合作——互相教 | 社交技能·复杂任务 |
| **翻转课堂** | 课前看视频——课堂讨论练习 | 需要讨论的科目 |

## 布鲁姆分类学

| 层次(低→高) | 记忆→理解→应用→分析→评价→创造 |
|------|------|
| 大多数考试只考前两层——这就是问题 |
