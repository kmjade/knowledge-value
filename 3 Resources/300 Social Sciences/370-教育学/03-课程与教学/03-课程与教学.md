# 03-课程与教学

## 课程设计

| 问题 | 「什么知识最有价值？」(斯宾塞)——课程是选择——不是「全部」——是「选什么」 |
|------|------|

| 取向 | 内容 |
|------|------|
| 学术理性 | 学科结构——数学·科学·文学——系统的知识体系 |
| 社会效率 | 培养「有用的」技能——STEM·职业 |
| 学生中心 | 兴趣和需求导向——不是「学什么」——是「学生想学什么」 |
| 社会改造 | 批判种族·阶级·性别——「课程即政治」(Freire·Apple) |

## 教学法

| 方法 | 特征 |
|------|------|
| 直接教学 | 教师讲——清晰·高效——适合新手/程序性知识 |
| 探究式 | 学生自己提问·发现——深度学习——但需要高支持 |
| 合作学习 | 小组——互相教——社交和学术双赢 |
| 差异化 | 同一教室——不同学生——不同任务——「因材施教」 |
