# 02-学习理论

## 行为主义

| 核心 | 学习 = 刺激-反应的联结——奖励强化——惩罚消退 |
|------|------|
| 代表 | 巴甫洛夫(条件反射)·斯金纳(操作条件) |
| 应用 | 程序化教学——小步子·即时反馈 |

## 认知主义

| 核心 | 学习 = 信息加工——大脑不是「黑箱」——关注记忆·理解·问题解决 |
|------|------|
| 代表 | 皮亚杰——认知发展阶段 |
| 关键 | 先备知识——「最重要的单一因素——是学习者已经知道什么」(Ausubel) |

## 建构主义

| 核心 | 知识不是「传递」的——是学习者主动**建构**的 |
|------|------|
| 代表 | 维果茨基——「最近发展区」——脚手架 |
| 教学 | 不是讲——是「创造环境」——让学生自己发现 |

## 社会文化理论

| 学习是**社会**过程 | 在互动中发生——在文化工具(语言·符号)中完成 |
|------|------|
| 情境学习 | 真实情境中学——不是抽象课堂——「学徒制」 |
