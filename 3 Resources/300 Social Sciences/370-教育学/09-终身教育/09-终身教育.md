# 09-终身教育

> 学校教给你的——5年后可能就过时了。终身学习不是「选择」——是「必需」。

## 为什么需要终身教育？

| 因素 | 影响 |
|------|------|
| **技术变革** | AI自动化——现有技能贬值——需要持续更新 |
| **寿命延长** | 活到90岁——工作到70岁——「一次教育用一辈子」不现实 |
| **职业变化** | 一生换5-7次职业——「转行」需要再教育 |
| **知识爆炸** | 人类知识每13个月翻一番(有些估计)——跟不上 |

## 终身教育的形式

| 形式 | 例子 | 优势 | 局限 |
|------|------|------|------|
| 正规 | 回大学读学位 | 系统化·认证 | 贵·时间固定 |
| 非正规 | 在线课程·职业认证 | 灵活·实用 | 质量参差 |
| 非正式 | 自学·YouTube·阅读 | 免费·自主 | 无认证·难坚持 |
| 工作场所 | 企业培训·导师制 | 相关·即时应用 | 局限于工作需求 |

## 成人学习 (Andragogy — Knowles)

| 成人学习不同于儿童: |
|------|
| 自我导向: 成人想要控制学习——不是「被教」 |
| 经验基础: 成人有丰富经验——学习必须联系这些经验 |
| 问题中心: 成人想解决实际问题——不是抽象知识 |
| 内在动机: 成人学习是因为「我需要」——不是「老师说」 |
