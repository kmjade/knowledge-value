# 06-教育技术

## EdTech 的演变

| 时代 | 工具 |
|------|------|
| 广播/电视 | 「空中学校」——单向传播 |
| 计算机 | CAI——程序化教学——「电子翻页」 |
| 互联网 | MOOC——2012「MOOC元年」——完成率低——但仍有价值 |
| AI | 自适应学习——Khan Academy·Duolingo——个性化路径 |

## 技术有效吗？

| 「没有显著的媒体效应」(Clark) | 关键是**教学设计**——不是媒体本身——好的教学法在纸上也有效 |
|------|------|
| 但 | 技术能做纸做不到的事——模拟·即时反馈·大数据 |

## 数字鸿沟

| 硬件 | 有设备吗？(一级鸿沟) |
|------|------|
| 使用 | 是用来学习——还是用来娱乐？(二级) |
| 创造 | 是消费内容——还是创造？ |
