# 06-教育技术

> 从黑板到AI——技术承诺变革教育——但历史说「不要期望太多」。

## 技术的承诺与现实

| 每次新技术: 「这将变革教育！」 |
|------|
| 广播(1920s)·电视(1950s)·计算机(1980s)·互联网(1990s)·MOOC(2010s)·AI(2020s) |
| 结果: 大多数只是「旧酒装新瓶」——课堂上依然以教师讲授为主 |

## EdTech 的关键教训

| 教训 | 说明 |
|------|------|
| 技术是工具——不是答案 | 烂教学+好技术=烂教学——甚至更差 |
| 教师的角色不会消失 | 技术辅助教师——不替代教师——至少目前 |
| 数字鸿沟 | 技术反而扩大了不平等——有设备vs没设备 |
| MOOC的教训 | 注册100万——完成<5%——最受益的是「已经受过良好教育的人」 |

## AI 教育的当前前沿

| 应用 | 内容 |
|------|------|
| **个性化辅导** | AI 1对1——Khan Academy的Khanmigo——24/7可用的助教 |
| **自动评分** | 作文·编程——节省教师时间——但AI评分可被「套路」 |
| **作弊?** | ChatGPT可以写论文——教育界恐慌——「我们是禁用它还是教学生用它？」 |
