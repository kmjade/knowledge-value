# 07-教育政策

> 教育政策决定了「谁学·学什么·谁付钱」——它是价值观的战场。

## 教育政策的核心议题

| 议题 | 张力 |
|------|------|
| **公平 vs 卓越** | 「每个孩子都成功」vs「培养顶尖人才」 |
| **公共 vs 私人** | 政府提供 vs 市场选择——学券·特许学校 |
| **中央 vs 地方** | 国家标准统一 vs 地方因地制宜 |
| **学术 vs 职业** | 上大学 vs 学技能——社会偏见:「蓝领=失败」 |

## 择校与学券

| 支持: 竞争提升质量——穷人也能逃离差学校——家长有权选择 |
|------|------|
| 反对: 侵蚀公立教育——隔离增加(按种族·阶层)——信息不对称:「好学校」是营销 |
| 证据: 学券的效果混合——智利大规模学券实验——结果分化 |

## 教师政策

| 教师质量是影响学生成就的最重要**校内**因素 |
|------|
| 但如何衡量教师质量？——学生考试成绩增值？——可能扭曲教学行为 |
| 芬兰: 教师来自前10%毕业生——硕士学位——高度自主——高度信任 |

## 教育经费

| 钱从哪里来？——地方房产税(美)→富裕区学校有钱——贫穷区没钱——「合法的不平等」 |
|------|
| 「钱重要吗？」——有争论——但「钱花在哪里」可能比「花多少」更重要 |
