## 🔀 学科交叉关系

社会科学天然是交叉学科：哲学为它提供基础，数学为它提供方法，心理学为它提供个体行为模型。

```
                    ┌──────────────────────────────┐
                    │     DDC 100 哲学·心理学        │
                    │  ┌──────────┬──────────────┐  │
                    │  │ 伦理学    │ 心理学       │  │
                    │  │ DDC 170  │ DDC 150      │  │
                    │  └────┬─────┴──────┬───────┘  │
                    └───────┼────────────┼──────────┘
                            │ 规范性基础  │ 个体行为
                            ↓            ↓
                    ┌──────────────────────────────┐
                    │     DDC 300 社会科学          │
                    │  ┌──────┬──────┬──────┬─────┐ │
                    │  │政治学│经济学│法律学│教育学│ │
                    │  └──────┴──────┴──────┴─────┘ │
                    └──────────────┬───────────────┘
                                   │ 数据与方法
                                   ↓
                    ┌──────────────────────────────┐
                    │  DDC 500 自然科学              │
                    │  ┌─────────────────────────┐  │
                    │  │ 统计学 (DDC 310)         │  │
                    │  │ ← 数学基础 (DDC 510)    │  │
                    │  └─────────────────────────┘  │
                    └──────────────────────────────┘
```
