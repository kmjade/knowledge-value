# 03-人生礼仪

> 从出生到死亡——人生的重要过渡——需要仪式来标记。

## 通过礼仪 (Van Gennep)

| 三个阶段: |
|------|
| **分离** (Separation): 离开旧身份——「告别」 |
| **阈限** (Liminal): 在「之间」——不是旧的——也还不是新的——不确定——危险的 |
| **融合** (Incorporation): 回归社会——以新身份——「欢迎回来」 |

## 人生阶段的仪式

| 阶段 | 仪式 | 核心 |
|------|------|------|
| 出生 | 满月·洗礼·命名 | 欢迎新成员——赋予身份 |
| 成年 | 成人礼·割礼·Bar Mitzvah | 「你不再是孩子了」 |
| 婚姻 | 婚礼 | 两个家族·新的家庭单位 |
| 死亡 | 葬礼 | 告别——处理后事——安慰活着的人 |

## 婚礼的多样性

| 文化 | 特征 |
|------|------|
| 中国 | 红色——聘礼·嫁妆——敬茶——「闹洞房」 |
| 印度 | 多天——曼海蒂(手绘)——绕火七圈 |
| 犹太 | Chuppah(婚棚)——踩碎玻璃——「即使在最快乐的时候——我们记得失落」 |
| 现代 | 个性化——「这是我们的婚礼」——传统元素被选择和重新编排 |

## 死亡仪式

| 功能: 处理死者遗体·安慰生者·确认社区——死者走好·活着继续 |
|------|
| 多样性: 土葬·火葬·天葬(西藏)——天葬喂鹰是「布施」的最后一次 |
| 变化: 越来越世俗化·简洁化——「绿色葬礼」——回归自然 |
