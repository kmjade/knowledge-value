# 02-节日与庆典

## 中国岁时节日

| 节日 | 日期 | 核心 |
|------|------|------|
| 春节 | 正月初一 | 团圆·除旧——年兽·压岁钱·春晚 |
| 元宵 | 正月十五 | 灯·汤圆——「中国情人节」 |
| 清明 | 4月4-6 | 扫墓·踏青——生与死的交汇 |
| 端午 | 五月初五 | 屈原·龙舟·粽子·艾草 |
| 中秋 | 八月十五 | 月圆人圆——月饼·嫦娥 |
| 重阳 | 九月初九 | 登高·敬老——「九九」长寿 |

## 节日的功能

| 功能 | 内容 |
|------|------|
| 时间标记 | 给一年以节奏——「过日子」 |
| 社会聚合 | 回家——团聚——「春运」是人类最大的年度迁徙 |
| 文化传承 | 孩子通过节日学会「我们是谁」 |
| 消费 | 现代节日 = 「消费节」——双十一 = 人造节日 |
