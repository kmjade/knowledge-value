# 02-节日与庆典

> 节日是时间的节点——群体聚在一起——在「非常」时间中确认「正常」。

## 节日的类型

| 类型 | 例子 |
|------|------|
| **岁时节日** | 春节·冬至·中秋节——农事周期 |
| **宗教节日** | 圣诞节·开斋节·排灯节·卫塞节 |
| **生命礼仪** | 婚礼·成人礼——个人生命阶段 |
| **国家节日** | 国庆·独立日——「想象的共同体」的仪式化 |

## 节日的功能

| 打破日常——节日是「阈限」(liminal) 时间——在平常规则暂停的空间中 |
|------|
| 社会凝聚: 一起做事——加强「我们感」 |
| 压力释放: 狂欢节——倒置权力——穷人当「国王」——Bakhtin的「狂欢化」 |
| 消费: 现代节日越来越商业化——「双十一」是人造的节日——纯消费 |

## 中国传统节日

| 节日 | 时间 | 核心 |
|------|------|------|
| 春节 | 农历正月初一 | 团圆·红包·年兽·年夜饭 |
| 元宵 | 正月十五 | 灯·汤圆——春节的句号 |
| 清明 | 4月5日前后 | 扫墓·踏青——纪念祖先 |
| 端午 | 农历五月初五 | 粽子·龙舟·屈原 |
| 中秋 | 农历八月十五 | 月饼·赏月·团圆 |
| 重阳 | 农历九月初九 | 登高·敬老·菊花 |

## 节日的变迁

| 传统→现代: 春节从「爆竹驱年兽」到「看春晚·发微信红包」 |
|------|
| 商业化: 每个节日 = 购物节——「情人节巧克力是糖果公司发明的」 |
| 全球化: 圣诞节在中国——春节在海外——节日超越起源文化 |
