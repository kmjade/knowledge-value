# 09-当代民俗

> 民俗不是「过去」的——它在互联网、城市和全球化中不断生成。

## 数字民俗

| 形式 | 例子 |
|------|------|
| Internet Meme | 快速变异·匿名创作·社群传播——本质上是民间故事的数字形式 |
| 城市传说 | 「一个朋友的朋友…」——消失的搭车客·肾被盗传说——恐惧的叙事 |
| Creepypasta | 网络恐怖故事——Slender Man——集体创作的怪物 |
| 挑战 | Ice Bucket Challenge——仪式·传播·病毒——当代民俗 |

## 数字民俗的特征

| 变异极快 | 传统民间故事变异需要代际——meme几小时内就有几十个变体 |
|------|------|
| 作者匿名 | 谁做了第一个「Distracted Boyfriend」meme？——不知道——不重要 |
| 全球+地方 | 全球模板(表情包格式)——地方填充(本国政治人物) |

## 民俗旅游

| 游客想要「真实的」民俗体验——但「真实性」一旦被表演——就不是原来的了 |
|------|
| 「迪士尼化」的民俗: 干净·安全·可预测——失去了民俗的「脏」和「野」 |
| 谁受益？——外来者赚钱·本地人失去文化控制——「被观看」的不平等 |

## 非遗保护

| UNESCO《保护非物质文化遗产公约》(2003): 将民间传统列为「人类遗产」 |
|------|
| 成就: 提高意识——资金——记录濒危传统 |
| 问题: 冻结(freezing)——传统是活的——「保护」变成「凝固」——谁有话语权？ |
