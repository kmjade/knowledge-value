# 05-服饰与身体

> 身体不是「自然的」——它是文化的画布。

## 服饰的功能

| 功能 | 说明 |
|------|------|
| 保护 | 防寒·防晒——最基本 |
| 装饰 | 美——地位——「好看」 |
| 身份 | 制服·传统服饰——「我是谁」——性别·阶层·职业 |
| 仪式 | 婚礼礼服·丧服·节日盛装 |

## 传统服饰

| 文化 | 服饰 | 场合 |
|------|------|------|
| 中国 | 旗袍·汉服(复兴)·唐装 | 婚礼·节日·文化展示 |
| 日本 | 和服·浴衣 | 成人式·茶道·夏祭 |
| 印度 | Sari·Salwar Kameez | 日常·婚礼 |
| 苏格兰 | Kilt(格子裙) | 正式场合——民族象征 |

## 身体改造

| 实践 | 文化 | 含义 |
|------|------|------|
| 纹身 | 波利尼西亚·日本·当代全球 | 从部落标记到个人表达 |
| 缠足 | 中国(10-20世纪) | 美的标准——也是控制女性的工具 |
| 割礼 | 犹太·穆斯林·部分非洲 | 宗教契约·成人仪式——女性割礼争议 |
| 整形 | 韩国·巴西·全球 | 「改造身体」——美是消费的 |

## 服饰的政治

| 头巾 (Hijab): 压迫还是身份？——取决于谁在说——穆斯林女性自己的声音多样 |
|------|
| 禁止 vs 强制 → 女性选择权——「不要替我决定——让我自己决定」 |
| 校服: 平等(消除贫富差异)vs 压制个性——永远争论的话题 |
