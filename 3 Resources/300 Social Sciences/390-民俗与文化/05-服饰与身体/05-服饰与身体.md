# 05-服饰与身体

## 传统服饰

| 汉服 | 交领右衽·宽袍大袖——近年「汉服复兴」——年轻一代 |
|------|------|
| 旗袍/唐装 | 近代「发明」的传统——但成了中国的文化符号 |
| 和服/韩服 | 日本·韩国——与汉服同源——各自演变 |

## 服饰的功能

| 保暖 | 最基本 |
|------|------|
| 身份 | 官服的颜色·补子——「看衣服——知道你是谁」 |
| 表达 | 「我属于这个群体」——亚文化·街头·Lolita |

## 身体修饰

| 纹身·缠足·割礼·整容 | 文化在身体上「书写」——有些已废除——有些仍在争议 |
|------|------|
