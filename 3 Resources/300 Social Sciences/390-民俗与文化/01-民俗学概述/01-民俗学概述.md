# 01-民俗学概述

> 民俗 = 一个群体的**非正式**的文化——不是官方·不是精英·不是「高雅」——是**日常生活**中的传统。

## 什么是民俗？

| 领域 | 口传(故事·谚语)·习俗(节日·仪式)·物质(服饰·建筑·食物)·表演(音乐·舞蹈) |
|------|------|

## 民俗的特征

| 特征 | 说明 |
|------|------|
| 口头性 | 通过口耳相传——不是书本——不是官方 |
| 匿名性 | 不知道谁「发明」的——是「集体创作」 |
| 变异性 | 同一故事——不同版本——不是「错误」——是活的传统 |
| 传承性 | 代代相传——但每一代都改造——传统是动词不是名词 |

## 民俗学的历史

| 时期 | 核心 |
|------|------|
| 19世纪 | 浪漫民族主义——民俗 = 民族精神的源泉——格林兄弟 |
| 20世纪早期 | 「残余论」——民俗 = 古代的「残余」——即将消失 |
| 20世纪后期 | 民俗 = 底层·边缘·被忽视的声音——不只是「残余」——是「抵抗」 |
| 当代 | 民俗无处不在——城市传说·互联网迷因——民俗是活的 |

## 民俗的功能

| 功能 | 说明 |
|------|------|
| 教育 | 传递价值观和社会规范——「狼来了」教诚实 |
| 娱乐 | 故事·音乐·节日——社区的「乐趣」 |
| 身份 | 「我们的」故事——强化群体认同——「我们是谁」 |
| 社会控制 | 「不能做X——会有报应」——用传统规范行为 |
