# 06-民间信仰

> 不是「官方宗教」——是「老百姓怎么信」——往往比教义更鲜活。

## 民间信仰的特征

| 实用主义: 「灵就拜——不灵就换」——没有严格的教义——重在「灵验」 |
|------|
| 综摄 (Syncretism): 佛+道+民间+地方神——混合——不觉得矛盾 |
| 「平时不烧香——临时抱佛脚」——典型的功利性信仰 |

## 中国民间信仰

| 要素 | 内容 |
|------|------|
| 祖先崇拜 | 扫墓·祭祖·牌位——「祖先的保佑」 |
| 地方神 | 妈祖(海神)·关帝(关公)·城隍(城市保护神)·土地公 |
| 风水 | 阴阳五行——择地·择时——「气场」 |
| 算命占卜 | 八字·面相·求签——「想知道未来」 |

## 民间信仰 vs 官方宗教

| 民间信仰 | 官方宗教 |
|------|------|
| 口头·灵活·地方性 | 经典·制度化·普世 |
| 没有「神学家」 | 有教义·有神职人员 |
| 「做了」比「信了」重要 | 「信」是核心 |

## 全球化中的民间信仰

| 新纪元运动 (New Age): 水晶·占星·冥想——去语境化的民间信仰元素 |
|------|
| 文化挪用: 西方人用美洲原住民的仪式——原住民说「那是我们的——你们不懂」 |
| 网络算命: AI看相·星座App——传统的数字化——仍然是「想知道未来」 |
