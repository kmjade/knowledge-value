# 07-口头传统

> 在文字之前——人类通过口耳相传来保存和传递知识。

## 口头传统的类型

| 类型 | 例子 |
|------|------|
| 神话 | 创世神话·英雄神话——解释「为什么世界是这样」 |
| 传说 | 地方故事——「这座山为什么叫这个名字」——半历史 |
| 民间故事 | 灰姑娘·「从前…」——各种文化都有类似版本 |
| 谚语 | 「三个和尚没水喝」——浓缩的智慧 |
| 笑话 | 幽默——社会批评——「说了不该说的——用笑话」 |

## 口头传统的特征

| 公式化 | 重复·套话——帮助记忆——「很久很久以前…」 |
|------|------|
| 变异性 | 同一故事不同版本——不是「错误」——是传统的活力 |
| 表演性 | 讲故事 = 表演——语调·表情·互动——不是「读」——是「演」 |
| 社群性 | 听众参与——「然后呢？」——共同创造 |

## 口头 vs 书面

| 文字的出现 = 革命——但失去了一些东西 |
|------|
| Ong: 口头文化 = 聚合的·情境的·贴近生活的 / 书写文化 = 分析的·抽象的·疏离的 |
| 「荷马问题」: 《伊利亚特》和《奥德赛》是口传史诗被记录下来——还是文字创作的？——可能两者都是 |

## 数字口头传统

| 互联网 = 新型口头文化？——meme = 现代民间故事——快速变异·匿名·群体创造 |
|------|
| 「网络用语」(slay·bruh·栓Q)——口头性的数字回归 |
| 社交媒体 = 全球的「篝火」——故事还在被讲——形式变了 |
