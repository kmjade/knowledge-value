# Expert Advisor (EA 智能交易系统)

## 定义

Expert Advisor (EA) 是 MetaTrader 平台的自动交易系统，用 MQL 语言编写，可以自动分析市场行情、生成交易信号并执行买卖操作，实现 24/7 无人值守交易。

## 核心原理

### EA 工作流程

```
┌──────────────┐
│  接收 Tick    │  ← 每次价格变动触发
└──────┬───────┘
       ▼
┌──────────────┐
│  分析市场     │  ← 技术指标计算、条件判断
└──────┬───────┘
       ▼
┌──────────────┐
│  生成信号     │  ← 买入/卖出/持有决策
└──────┬───────┘
       ▼
┌──────────────┐
│  执行交易     │  ← 自动下单 + 止损止盈 + 仓位管理
└──────┬───────┘
       ▼
┌──────────────┐
│  记录日志     │  ← 交易记录、错误处理
└──────────────┘
```

### EA 的三大功能

| 功能 | 说明 |
|------|------|
| **自动分析** | 实时计算技术指标、识别图表形态、多时间框架分析 |
| **自动交易** | 根据预设策略执行开仓、平仓、加仓、减仓 |
| **自动风控** | 止损、止盈、跟踪止损、最大回撤限制 |

## 关键要点

1. **策略 vs EA** — 策略是逻辑，EA 是代码实现
2. **回测是必须的** — 实盘前必须用历史数据验证
3. **过拟合陷阱** — 回测完美 ≠ 实盘盈利
4. **风险管理** — 任何 EA 都需要资金管理和最大回撤控制
5. **MQL5 比 MQL4 快 20 倍** — 对高频策略差异显著

### 开发流程

```
1. 策略设计  → 定义交易规则（入场、出场、风控）
2. 代码编写  → MQL4/MQL5 实现策略逻辑
3. 回测验证  → 策略测试器历史数据模拟
4. 优化参数  → 遗传算法寻找最优参数组合
5. 模拟盘    → 真实市场环境验证
6. 实盘运行  → VPS 24/7 运行
```
