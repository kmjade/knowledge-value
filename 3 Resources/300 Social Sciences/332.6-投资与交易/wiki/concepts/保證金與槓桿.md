# 保證金與槓桿 Margin & Leverage

## 定義
槓桿允許交易者用少量保證金控制大額倉位。例如 100:1 槓桿 = 1% 保證金率。

## 計算

```
所需保證金 = 合約價值 / 槓桿倍數

例：1 標準手 EURUSD ($100,000)，槓桿 100:1
保證金 = $100,000 / 100 = $1,000
```

## 槓桿對比

| 槓桿 | 保證金率 | $1,000 可控制 | 風險 |
|:---:|:--:|------|:--:|
| 10:1 | 10% | $10,000 | 🟢 安全 |
| 100:1 | 1% | $100,000 | 🟡 適中 |
| 500:1 | 0.2% | $500,000 | 🔴 高風險 |

## 爆倉計算

```
保證金比例 = 淨值 / 已用保證金 × 100%
< 100% → 無法開新倉
< 50% → 強制平倉 (Stop Out)
```

> ⚠️ 新手建議槓桿 ≤ 100:1。
