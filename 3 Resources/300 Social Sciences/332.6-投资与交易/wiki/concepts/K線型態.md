# K線型態 Candlestick Patterns

## 定義
K線（陰陽燭）是技術分析最基本的價格可視化工具，每根K線顯示一段時間內的開盤價、收盤價、最高價和最低價。

## K線結構

```
陽線（收盤 > 開盤）：買方主導
陰線（收盤 < 開盤）：賣方主導

上影線 → 最高價
實體   → 開盤↔收盤
下影線 → 最低價
```

## 關鍵反轉型態

| 型態 | 特徵 | 信號 |
|------|------|:--:|
| **錘子線 Hammer** | 長下影 + 小實體，出現在下跌趨勢 | 底部反轉 |
| **吊頸線 Hanging Man** | 長下影 + 小實體，出現在上升趨勢 | 頂部反轉 |
| **吞沒型態 Engulfing** | 第二根K線完全吞沒第一根 | 強反轉 |
| **十字星 Doji** | 開盤≈收盤 | 猶豫/可能反轉 |
| **射擊之星 Shooting Star** | 長上影 + 小實體 | 頂部反轉 |
