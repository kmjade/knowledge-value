# 點差與交易成本 Spread & Trading Costs

## 點差 Spread

```
買價 (Ask) − 賣價 (Bid) = 點差 (Spread)

EURUSD: 1.08500 / 1.08501 → 點差 = 0.1 pips
```

## 每點價值 Pip Value

| 手數 | 單位 | EURUSD 每點價值 |
|:--:|------|:--:|
| 1 標準手 | 100,000 | ~$10 |
| 0.1 迷你手 | 10,000 | ~$1 |
| 0.01 微型手 | 1,000 | ~$0.10 |

## 總交易成本

```
總成本 = 點差 + 佣金 + 隔夜利息 (Swap)

ECN 帳戶：
  點差：0.0–0.3 pips (EURUSD)
  佣金：$3–7/手/邊
  總成本 ≈ 0.6–1.0 pips

標準帳戶：
  點差：1.0–2.0 pips (EURUSD)
  佣金：無
  總成本 ≈ 1.0–2.0 pips
```
