# 09 交易心理 Trading Psychology

> 市場上 90% 的交易者虧損——不是因為技術不好，而是因為管不住自己。

## 交易心理的三大敵人 Three Enemies

### 😱 恐懼 Fear

| 表現 | 後果 |
|------|------|
| 害怕虧損不敢進場 | 錯過機會 |
| 盈利後過早平倉 | 賺小賠大 |
| 持倉中反覆看盤 | 情緒消耗、過早出場 |

### 😤 貪婪 Greed

| 表現 | 後果 |
|------|------|
| 賺了還想賺更多 | 不設止盈，利潤回吐 |
| 浮盈加倉 | 加大風險敞口 |
| 看到別人賺錢跟風 | FOMO (Fear of Missing Out) |

### 🤬 報復心態 Revenge Trading

| 表現 | 後果 |
|------|------|
| 虧損後立即反手 | 雙向被掃 |
| 加大倉位想翻本 | 越虧越多 |
| 不做分析貿然進場 | 連續虧損 |

## 常見認知偏誤 Cognitive Biases

| 偏誤 | 說明 | 例子 |
|------|------|------|
| **確認偏誤** | 只看到支持自己觀點的資訊 | 做多後只看利多新聞 |
| **損失厭惡** | 虧損的痛苦是盈利快樂的 2 倍 | 不願止損，越套越深 |
| **近因效應** | 過度重視最近發生的事件 | 連續盈利後過度自信 |
| **錨定效應** | 被某個數字鎖定思維 | 「我買在 1.10，一定要等它回來」 |
| **過度自信** | 高估自己的判斷能力 | 不設止損、重倉 |
| **沉沒成本** | 已投入的不願放棄 | 虧損加倉以攤平成本 |

## 建立交易紀律 Building Discipline

### 1. 交易前檢查清單 Pre-Trade Checklist

```
□ 趨勢方向確認（D1/W1）
□ 支撐/阻力位標記
□ 有明確的入場信號？
□ 止損設在哪？（R:R ≥ 1:2？）
□ 風險 ≤ 2%？
□ 今天有沒有重要數據發布？
□ 當前情緒狀態：冷靜？焦慮？興奮？
   → 如果情緒不穩定：今天不交易。
```

### 2. 交易規則 (貼在螢幕旁)

```
1. 永遠設止損
2. 每筆風險 ≤ 2%
3. 不追漲殺跌
4. 不逆大趨勢
5. 不在數據發布前 5 分鐘進場
6. 連續虧損 3 筆 → 停止當日交易
7. 每筆交易必須寫進日誌
```

### 3. 交易日誌 (心理部分)

除了技術面記錄外，還應記錄：

```
進場時情緒：□ 冷靜  □ 焦慮  □ 興奮  □ 報復心態
持倉時感受：___________________
出場時情緒：___________________

這筆交易是否按照計劃執行？ □ 是  □ 否
如果沒有，為什麼？___________
```

## 克服常見心理障礙

### 「我必須把虧的錢賺回來」

> ❌ 這是報復心態。市場不在乎你虧了多少——只在乎你接下來的決定。
> ✅ 暫停交易。虧損後至少休息 2 小時，最好等到明天。

### 「這筆一定會盈利」

> ❌ 過度自信。沒有人能預測市場。
> ✅ 交易是概率遊戲。接受每筆交易都可能虧損，做好風控。

### 「再等一等就會回來了」

> ❌ 損失厭惡 + 沉沒成本。市場不會因為你套住了就回來。
> ✅ 嚴格執行止損。虧小錢比虧大錢好一萬倍。

## 職業交易員的日常 Routine

```
07:00  檢查經濟日曆
07:30  復盤昨日交易
08:00  掃描圖表，標記關鍵位
09:00  制定當日交易計劃（不超過 3 個機會）
─── 交易時段 ───
       嚴格按計劃執行
       每筆交易記錄
       連續虧損 3 筆 → 停止
─── 收盤後 ───
17:00  填寫交易日誌
17:30  分析當日交易（成功/失敗原因）
18:00  關閉所有交易軟件，離開市場
```

> 💡 **交易日不超過 6 小時**。疲勞交易 = 虧損交易。

## 最後的話

> 交易中最難的不是找到好的入場點，而是在虧損時按計劃止損，在盈利時讓利潤奔跑，在沒有信號時什麼都不做。

> **這是一場與自己的戰爭。市場永遠是對的。**
