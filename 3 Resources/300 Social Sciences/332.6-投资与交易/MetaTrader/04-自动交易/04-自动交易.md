# 04 - MetaTrader 自动交易（EA）

## EA 是什么？

> EA（Expert Advisor）= 自动交易机器人

根据预设规则，自动完成：分析 → 开仓 → 调整止损 → 平仓。

## EA 的安装

```
1. 下载 .ex4 或 .ex5 文件
2. 放入文件夹：
   MT4: 文件 → 打开数据文件夹 → MQL4 → Experts
   MT5: 文件 → 打开数据文件夹 → MQL5 → Experts
3. 重启 MT4/MT5 或在导航器中右键 → 刷新
4. 拖拽 EA 到图表
5. 启用「允许自动交易」
```

## EA 使用前检查

```
□ EA 是否有实盘记录？（Myfxbook 验证）
□ EA 最大回撤是多少？你能接受吗？
□ EA 是否包含马丁策略？（→ 高风险）
□ EA 是否包含网格策略？（→ 高风险）
□ 先在模拟账户跑 2-4 周
```

## 风险警告

```
⚠️ 市场上 95% 以上的 EA 不盈利

常见骗局：
❌ 回测曲线完美 → 用了作弊手段
❌ 保证月收益 XX% → 庞氏骗局
❌ 免费 EA + 指定经纪商 → 赚你返佣
❌ 马丁格尔 EA → 终将爆仓

✅ 好的 EA：有可验证的实盘记录 + 合理的回撤 + 透明的策略逻辑
```

## MT4/MT5 信号服务

```
终端 → 信号 → 订阅信号提供者

本质：跟单交易
⚠️ 同样需要仔细审查信号提供者的历史记录
```

## VPS（虚拟专用服务器）

```
EA 需要 24 小时运行 → 需要 VPS

推荐：MT4/MT5 内置 VPS 服务（与经纪商绑定）
或：AWS、阿里云、腾讯云 Windows 主机

配置：2 核 + 4GB 内存 + 低延迟网络
```
