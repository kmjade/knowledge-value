## 订单类型

### 市价单（Market Order）

```
即时按当前价格成交。

买入（做多）：按 Ask 价买入
卖出（做空）：按 Bid 价卖出
```

### 挂单（Pending Order）

| 类型 | 说明 | 场景 |
|------|------|------|
| **Buy Limit** | 在现价下方买入 | 抄底 |
| **Sell Limit** | 在现价上方卖出 | 摸顶 |
| **Buy Stop** | 在现价上方买入 | 突破追涨 |
| **Sell Stop** | 在现价下方卖出 | 突破追空 |

```
价格 ↑
  Sell Limit  ← 现价上方挂单卖出
  ──────── 当前价格 ────────
  Buy Stop    ← 现价上方追涨
  ──────────────────────
  Sell Stop   ← 现价下方追空
  ──────── 当前价格 ────────
  Buy Limit   ← 现价下方挂单买入
价格 ↓
```

## 止损与止盈

```
止损（Stop Loss）：亏损到某价位自动平仓
止盈（Take Profit）：盈利到某价位自动平仓

下单时必须设置！
```
