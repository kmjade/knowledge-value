# 05 - MetaTrader 策略回测

## 回测入口

```
MT4: 工具栏 → 策略测试（Ctrl+R）
MT5: 显示 → 策略测试（Ctrl+R）
```

## 回测参数设置

```
□ 选择 EA / 指标
□ 选择交易品种（EURUSD 等）
□ 选择时间周期（H1 等）
□ 选择时间范围（最近 1 年）
□ 选择模型（MT5 支持：1分钟OHLC / 每个点基于实时点）
□ 设置初始资金（建议 10,000）
□ 设置杠杆（1:100 等）

→ 点击「开始」
```

## 回测结果解读

### 关键指标

| 指标 | 含义 | 好标准 |
|------|------|--------|
| **净利润** | 总盈利 - 总亏损 | > 0 |
| **最大回撤** | 最大连续亏损幅度 | < 30% |
| **胜率** | 盈利交易 / 总交易 | 不重要 |
| **盈亏比** | 平均盈利 / 平均亏损 | > 1.5 |
| **夏普比率** | 风险调整收益 | > 1 |
| **交易次数** | 样本量 | > 100 才有统计意义 |

### 常见陷阱

```
❌ 回测曲线 45 度上涨无回撤 → 99% 作弊
❌ 胜率 90%+ → 大概率未来回撤会爆
❌ 只测了 3 个月 → 数据太少
❌ 回测完美实盘亏 → 过拟合

✅ 合理期待：
  - 年化收益 20-50%
  - 最大回撤 < 30%
  - 交易次数 > 200
```

## 优化（Optimization）

```
策略测试 → 优化 → 选择优化参数

例如：优化 MA 周期
  参数：MA Period = 10 → 200，步长 10
  测试 20 种组合 → 找出最优

⚠️ 过度优化的风险：
  参数完美匹配历史=未来一定失效
  留 30% 数据做 out-of-sample 验证
```

## 前向测试

```
回测（模拟）≠ 实盘（真实）

1. 回测通过 → 
2. 模拟账户跑 2-4 周 →
3. 小资金实盘跑 1 月 →
4. 正常资金实盘
```
