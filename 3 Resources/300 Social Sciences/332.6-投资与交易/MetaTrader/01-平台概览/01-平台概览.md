# 01 - MetaTrader 平台概览

## 下载安装

| 平台 | MT4 | MT5 |
|------|-----|-----|
| Windows | mt4setup.exe | mt5setup.exe |
| macOS | 通过 Wine/CrossOver 或 WebTerminal | 同左 |
| iOS | App Store → MetaTrader 4/5 | 同左 |
| Android | Google Play → MetaTrader 4/5 | 同左 |
| Web | 经纪商提供的 WebTerminal | 同左 |

> 桌面版功能最全。移动端适合监控和快速下单。

## 账户类型

### 模拟账户 vs 实盘账户

| | 模拟账户 | 实盘账户 |
|---|----------|----------|
| 资金 | 虚拟 | 真实 |
| 行情 | 可能延迟 | 实时 |
| 滑点 | 无/小 | 真实存在 |
| 心态 | 无压力 | 有压力 |
| 用途 | 学习、测试 | 真实交易 |

> 强烈建议：至少 1-3 个月模拟交易后再实盘。

### 账户类型（经纪商）

| 类型 | 点差 | 佣金 | 适合 |
|------|------|------|------|
| Standard | 较高（含佣金） | 无 | 新手 |
| ECN | 极低 | 有（$/手） | 短线交易 |
| Cent | 美分账户 | 低 | 超小资金 |
| Islamic | 常规 | 无隔夜利息 | 穆斯林用户 |

## MT4 vs MT5 详解

### 选 MT4 如果：

```
✅ 只做外汇交易
✅ 依赖社区 EA（MT4 资源最丰富）
✅ 习惯简单界面
✅ 需要大量现成指标
```

### 选 MT5 如果：

```
✅ 交易股票、期货等多资产
✅ 需要市场深度（DOM）
✅ 需要更快的回测速度
✅ 需要内置经济日历
✅ 需要更多时间周期
```

## 安装后的第一件事

```
1. 登录模拟账户
2. 打开市场报价（Ctrl+M）
3. 右键 → 显示全部品种
4. 打开 EURUSD 图表
5. 熟悉界面布局
```
