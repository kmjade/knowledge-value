# 02-官僚制

## 韦伯的理想型

| 特征 | 内容 |
|------|------|
| 层级制 | 明确的上下级——命令链 |
| 分工 | 专业化——每个人有固定职责 |
| 规则导向 | 按章办事——不是按人 |
| 非人格化 | 不因人而异——法律面前平等 |
| 功绩制 | 按能力选拔——不是世袭·不是选举 |

## 官僚制的优缺点

| 优点 | 效率·可预测·公平——法治的保障 |
|------|------|
| 问题 | 僵化·「官僚主义」·目标置换(规则变成目的)·创新困难 |

## 后官僚制

| 趋势 | 从层级→网络·从规则→结果·从封闭→透明·从垄断→竞争 |
|------|------|
