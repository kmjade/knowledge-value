# 02-官僚制

> 韦伯: 官僚制是「最理性的组织形式」——但也是「铁笼」。

## 韦伯的理想型官僚制

| 特征 | 内容 |
|------|------|
| **层级制** | 明确的上下级——命令链——谁向谁报告 |
| **分工** | 专业化——每个人有明确定义的职责 |
| **规则导向** | 按规则办事——不是按人情——「非人格化」 |
| **功绩制** | 按能力录用和晋升——不是按出身或关系 |
| **档案管理** | 书面记录——「写在纸上」——可追溯 |

## 官僚制的弊端

| 问题 | 说明 |
|------|------|
| **目标置换** | 遵守规则变成了目的——忘了「为什么」——「官僚主义」 |
| **僵化** | 难以适应变化——规则一旦建立很难改 |
| **寡头铁律** | Michels: 任何组织最终都会被少数精英控制 |
| **反功能** | Merton: 规则催生最低合规——「我只做规定内的事」 |

## 街头官僚 (Lipsky, 1980)

| 警察·教师·社工——一线工作者——他们不是「执行政策」——他们**就是**政策 |
|------|
| 自由裁量权: 必须的(每个案例都不同)——但裁量就是制定政策 |
| Coping mechanisms: 工作量太大→分类·简化——贫困的公民得不到同等服务 |

## 后官僚制时代

| NPM批评: 官僚制太慢·太笨——引入市场机制→管制机构·外包·绩效指标 |
|------|
| 但「去官僚化」也有代价: 碎片化——协调困难——责任模糊——「谁该负责？」 |
