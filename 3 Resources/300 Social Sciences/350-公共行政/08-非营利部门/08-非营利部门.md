# 08-非营利部门

> 政府失灵和市场失灵之间的「第三部门」。

## 为什么存在非营利？

| 理论 | 说明 |
|------|------|
| **政府失灵** | 政府提供「中位选民」想要的服务——少数群体需求被忽视——NPO填补 |
| **市场失灵** | 信息不对称(捐赠者无法判断服务质量)——非营利的「不分配约束」=信任信号 |
| **志愿失灵** | NPO也有问题: 资源不足·业余·家长式——需要政府合作 |

## 非营利 vs 政府 vs 企业

| | 非营利 | 政府 | 企业 |
|------|:---:|:---:|:---:|
| 资金来源 | 捐赠·资助·服务费 | 税收 | 销售收入 |
| 目标 | 使命 | 公共价值 | 利润 |
| 治理 | 理事会(志愿者) | 选举·任命 | 股东·董事会 |
| 约束 | 不分配利润 | 法律·预算 | 市场竞争 |

## 社会企业

| 用商业手段解决社会问题——「双重底线」(社会+财务) ——是否是「新瓶旧酒」？ |
|------|
| 例子: Grameen Bank(小额贷款)·TOMS(买一双捐一双) |
| 争议: 「使命漂移」——为了生存越来越像企业——社会使命淡了 |
