# 09-当代挑战

> 21世纪的公共行政——旧模式能否应对新挑战？

## 危机管理

| 疫情暴露: 政府应对危机的能力参差不齐 |
|------|
| 成功因素: 快速决策·科学沟通·社会信任·基层执行 |
| 「黑天鹅」vs「灰犀牛」: 罕见但高冲击 vs 显而易见但被忽视 |

## 信任危机

| 全球趋势: 公众对政府的信任下降——OECD国家平均信任度~45% |
|---|
| 原因: 丑闻·无能感·社交媒体放大不满·「精英背叛」叙事 |
| 后果: 低信任→低合规→高执行成本→「越不信任越证明不信任」 |

## 可持续发展

| 行政如何回应气候危机？ |
|------|
| 绿色采购·碳预算·气候适应规划——行政不是旁观者——是执行者 |
| 代际正义: 今天的政策影响未来几代人——但选举周期只有4-5年 |

## 平台治理

| Uber·Airbnb: 平台绕过传统监管——政府如何回应？ |
|------|
| 禁止(柏林禁Uber)·吸收(立法规制)·合作(数据共享) |
| 核心: 谁有权力——政府还是平台？——「代码就是法律」的挑战 |
