# 03-公共政策制定

## 政策循环

| 阶段 | 内容 |
|------|------|
| 议程设置 | 为什么有些问题成为「议题」——有些被忽略？ |
| 政策形成 | 提出解决方案——分析利弊 |
| 决策 | 立法·行政命令——正式采纳 |
| 执行 | 官僚机构将政策变为现实——**最难的环节** |
| 评估 | 政策有效吗？——反馈→新一轮循环 |

## 执行为什么难？

| 原因 | 资源不足·目标模糊·多层级协调·一线官僚的自由裁量·利益相关者的抵抗 |
|------|------|

## 渐进主义 (Lindblom)

| 「现实的决策不是全面理性分析——是『摸着石头过河』——小幅调整——边做边改」 |
|------|
