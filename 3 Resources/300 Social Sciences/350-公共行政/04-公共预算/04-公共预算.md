# 04-公共预算

## 预算的功能

| 功能 | 不只是「记账」——是资源配置·经济调控·政治博弈 |
|------|------|

## 预算过程

| 阶段 | 内容 |
|------|------|
| 编制 | 各部门提交——财政部汇总 |
| 审批 | 立法机关审议——美国国会可以修改——英国议会基本橡皮图章 |
| 执行 | 花钱——可以调整吗？——取决于制度 |
| 审计 | 事后检查——有没有违规？——有没有浪费？ |

## 税收原则

| 原则 | 效率(不扭曲经济)·公平(横向·纵向)·简单(普通人能懂) |
|------|------|
| 公债 | 政府借钱——当代人花——后代人还？——代际公平 |
