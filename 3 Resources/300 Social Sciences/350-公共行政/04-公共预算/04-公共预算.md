# 04-公共预算

> 预算 = 政府最重要的政策文件——数字就是优先级。

## 预算的功能

| 功能 | 说明 |
|------|------|
| **分配** | 钱花在哪里——军事？教育？医疗？ |
| **稳定** | 财政政策——对抗经济周期 |
| **再分配** | 从富人到穷人——从一代到另一代 |
| **控制** | 行政机关花多少钱——立法机关批准——权力制衡 |

## 预算过程

| 阶段 | 内容 |
|------|------|
| 编制 | 行政部门提出预算草案——各部门「要钱」 |
| 审批 | 立法机关审查——修改——批准——「钱袋子的权力」 |
| 执行 | 拨款——支出——采购——项目 |
| 审计 | 事后审查——钱按意图花了吗？——问责 |

## 预算改革

| 改革 | 核心 |
|------|------|
| **绩效预算** | 不是「给你多少钱」——是「你产生了什么结果」 |
| **零基预算** (ZBB) | 每年从零开始——每分钱都重新论证——理论上好——实践太难 |
| **参与式预算** | 公民直接决定部分预算——巴西Porto Alegre首创 |

## 预算政治

| 预算不是「技术」——是「政治」——「谁得到什么——多少——谁付钱」 |
|------|
| 渐进主义 (Lindblom): 今年预算 ≈ 去年预算 + 增量调整——而非理性全面分析 |
| 「猪肉桶」(pork barrel): 议员为选区争取项目——买选票——浪费 |
