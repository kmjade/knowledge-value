# 05-人力资源管理

> 政府最大的资产是人——公务员的质量决定了公共服务的质量。

## 公务员制度的核心

| 原则 | 内容 |
|------|------|
| **功绩制** | 按能力录用——不是按政治关系——不是「分赃制」(spoils system) |
| **中立** | 公务员服务于国家——不服务于某个政党——换政府不换公务员 |
| **终身制** | 职业保障——不受政治清洗——可以「说真话」 |
| **考试录用** | 通过竞争性考试——确保基本能力——但可能过于死板 |

## 激励难题

| 公共部门的激励不同于私人部门: |
|------|
| 没有利润指标——「绩效」难以衡量 |
| 难以解雇——「铁饭碗」——可能滋生懈怠 |
| 公共服务动机 (PSM): 有人选择公共部门不是因为钱——是因为「想为社会做贡献」 |

## 代表性官僚制

| 官僚机构应该「看起来像」它服务的社会——在种族·性别·阶层上的代表 |
|------|
| 被动代表: 人员构成反映社会构成 |
| 主动代表: 代表性能转化为对少数群体利益的实际推动 |
