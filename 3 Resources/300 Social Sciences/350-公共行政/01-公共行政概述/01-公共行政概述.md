# 01-公共行政概述

> 威尔逊 (Woodrow Wilson, 1887): 「行政应该在政治领域之外。」政治决定方向——行政负责执行。

## 政治 vs 行政二分

| | 政治 | 行政 |
|------|------|------|
| 谁 | 民选官员 | 职业公务员 |
| 做什么 | 制定政策——「应该做什么」 | 执行政策——「怎么做」 |
| 逻辑 | 选举·民意·妥协 | 效率·专业·规则 |
| **现实** | 二分是理想——实际上行政绝非「纯技术」——它不可避免地涉及价值判断 |

## 范式变迁

| 时期 | 范式 |
|------|------|
| 1880s-1940s | 政治-行政二分·科学管理 |
| 1940s-1970s | 行为主义·官僚政治 |
| 1980s-2000s | 新公共管理 (NPM) ——市场·竞争·绩效 |
| 2000s- | 新公共治理·网络·合作·公民参与 |
