# 06-电子政务

> 从排队到上网——信息技术如何重塑政府服务。

## 电子政务的阶段

| 阶段 | 内容 | 例子 |
|------|------|------|
| 1. 信息发布 | 政府网站——信息在线——单向 | 政策文件·通知 |
| 2. 交互 | 表格下载·邮件联系——双向 | 在线投诉 |
| 3. 交易 | 在线完成完整服务 | 报税·办证·缴费 |
| 4. 转型 | 重新设计流程——不是把纸搬到网上——是重新思考 | 爱沙尼亚「数字国家」 |

## 数字政府的前沿

| 领域 | 内容 |
|------|------|
| **AI政务** | ChatGPT式问答——智能审批——预测性执法 |
| **区块链** | 土地登记——学历认证——不可篡改 |
| **开放数据** | 政府数据公开——民间创新——但不等于「透明」 |

## 数字鸿沟与挑战

| 挑战 | 说明 |
|------|------|
| **数字鸿沟** | 老年人·低收入·农村——在线服务=排除他们 |
| **隐私** | 政府拥有海量数据——监控风险——中国社会信用体系 |
| **安全** | 黑客攻击政府系统——勒索软件——数据泄露 |

## 爱沙尼亚案例

| 99%政府服务在线——电子居民——X-Road数据交换层——「只问一次」原则 |
|------|
| 成功因素: 小国·高信任·后发优势(1991后从零建)——最难复制的是信任 |
