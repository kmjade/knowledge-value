# 06-电子政务

## 发展阶段

| 阶段 | 内容 |
|------|------|
| 信息发布 | 网站——政府信息公开 |
| 单向互动 | 下载表格 |
| 双向互动 | 在线提交·在线支付 |
| 转型 | 流程再造——不只是「把纸变成电子」——是重新设计 |

## 开放数据

| 原则 | 政府数据——向社会免费开放——机器可读——任何人可用 |
|------|------|
| 价值 | 创业(天气·交通API)·监督(预算透明)·研究 |

## 数字政府 vs 电子政务

| 电子政务 = 技术工具化了现有流程 | 数字政府 = 重新设计政府本身——以数据为核心 |
|------|------|

## AI 在公共行政

| 应用 | 智能客服·自动审批·风险预测(税务欺诈) ——但算法透明和公平是问题 |
|------|------|
