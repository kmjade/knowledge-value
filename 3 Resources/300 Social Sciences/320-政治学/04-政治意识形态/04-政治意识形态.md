# 04-政治意识形态

> 意识形态 = 回答"世界应该是什么样"的一套信念体系。

## 左右光谱

| 左 ← | → 右 |
|------|------|
| 平等 | 自由 |
| 集体 | 个体 |
| 国家干预 | 市场自发 |
| 激进变革 | 保守传统 |

## 主要意识形态

| 意识形态 | 核心信念 | 经济 | 社会 |
|------|------|:---:|:---:|
| **自由主义** | 个人自由——法治——有限政府 | 市场主导+安全网 | 宽容·进步 |
| **保守主义** | 传统·秩序·渐进——不信任激进变革 | 市场自由 | 传统价值 |
| **社会主义** | 平等——生产资料公有或共管 | 国家/集体主导 | 团结优先 |
| **社会民主主义** | 资本主义+强福利国家 | 混合经济 | 进步 |
| **自由意志主义** | 个人自由最大化——最小政府 | 彻底自由市场 | 彻底个人自由 |

## 意识形态的"终结"？

| 福山 (1992) | 历史终结于自由民主——没有更好的替代方案 |
|------|------|
| 21世纪现实 | 民粹主义·民族主义·威权回潮——历史没有终结 |
