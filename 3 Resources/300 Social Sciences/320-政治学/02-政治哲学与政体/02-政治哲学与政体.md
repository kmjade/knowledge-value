## Part A: 政治哲学——公正社会的蓝图

### 核心思想家谱系

| 人物 | 核心 | 著作 |
|------|------|------|
| **柏拉图** | 哲人王·正义 = 各司其职 | 《理想国》 |
| **亚里士多德** | 人是政治的动物·混合政体最优 | 《政治学》 |
| **马基雅维利** | 目的正当→手段正当·现实政治 | 《君主论》 |
| **霍布斯** | 自然状态 = 战争·利维坦（绝对主权） | 《利维坦》 |
| **洛克** | 自然权利·有限政府·革命权 | 《政府论》 |
| **卢梭** | 社会契约·公意·人民主权 | 《社会契约论》 |
| **马克思** | 阶级斗争·国家是阶级压迫的工具 | 《共产党宣言》 |
| **罗尔斯** | 无知之幕·正义二原则 | 《正义论》(1971) |

### 核心张力

| 张力 | 问题 |
|------|------|
| **自由 vs 平等** | 更多自由 → 不平等；更多平等 → 限制自由 |
| **个人 vs 集体** | 权利在个人？还是在社群？ |
| **理想 vs 现实** | 应该怎样 vs 实际怎样 |

## Part B: 政体类型——权力的组织方式

### 亚里士多德的经典分类

| 统治者 | 好的形式 | 坏的形式 |
|------|------|------|
| 一人 | 君主制 | 僭主制 |
| 少数 | 贵族制 | 寡头制 |
| 多数 | 政体 (Polity) | 民主（暴民） |

### 现代政体光谱

| 类型 | 特征 | 例子 |
|------|------|------|
| **自由民主** | 自由选举 + 公民权利 + 法治 + 权力制衡 | 德·英·日 |
| **选举威权** | 有选举但不自由，反对派被压制 | 俄罗斯·土耳其 |
| **一党制** | 一个政党控制，无竞争 | 中国·越南 |
| **绝对君主** | 君主有实权 | 沙特·阿联酋 |
| **军事政权** | 军队直接或间接统治 | 缅甸 (2021后) |
| **失败国家** | 政府不能提供基本秩序 | 索马里 |

### 总统制 vs 议会制

| | 总统制 | 议会制 |
|------|:---:|:---:|
| 行政产生 | 总统独立选举，固定任期 | 总理由议会产生，可被不信任案推翻 |
| 立法-行政 | 分离，制衡 | 融合，"高效的" |
| 稳定性 | 僵局风险，但任期固定 | 灵活，但政府可能频繁更换 |
| 例子 | 美国·巴西·韩国 | 英国·德国·日本 |

### 混合政体：半总统制

> 总统 + 总理——法国模式。"共治" (cohabitation)：总统和总理来自不同政党时必须合作。
