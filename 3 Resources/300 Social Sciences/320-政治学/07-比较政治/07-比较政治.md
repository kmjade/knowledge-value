# 07-比较政治

> 为什么有些国家民主——有些专制？为什么北欧福利好——美国没有全民医保？比较政治通过跨国比较寻找答案。

## 核心问题

| 问题 | 典型发现 |
|------|---------|
| 民主化的条件？ | 经济发展·中产阶级·公民社会·外部压力 |
| 为什么有些国家失败？ | 「资源诅咒」——石油富国反而专制·腐败 |
| 福利国家的起源？ | 左翼政党·工会力量·种族同质性 |

## 比较方法

| 方法 | 内容 |
|------|------|
| 大N统计 | 跨国数据——回归分析——找普遍模式 |
| 小N案例 | 深度比较几个国家——找出因果机制 |
| 过程追踪 | 追踪一个案例的历史——「怎么发生的」 |

## 民主化浪潮 (Huntington)

| 波 | 时间 | 例子 |
|----|------|------|
| 第一波 | 1828-1926 | 英国·美国·法国 |
| 第二波 | 1943-1962 | 战后·去殖民化 |
| 第三波 | 1974- | 葡萄牙·拉美·东欧——仍在继续？还是回潮？ |
