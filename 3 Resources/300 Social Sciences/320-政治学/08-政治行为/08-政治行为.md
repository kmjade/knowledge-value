# 08-政治行为

> 人们如何参与政治——投票、抗议、捐款——以及为什么不参与。

## 投票行为

| 模型 | 核心 |
|------|------|
| **社会学模型** (Columbia) | 你投给「和你一样的人」——阶级·宗教·种族 |
| **心理模型** (Michigan) | 政党认同——是「共和党人」还是「民主党人」——从小形成 |
| **理性选择** (Downs) | 投票 = 成本>收益? ——如果大概率是「浪费的一票」——为什么不投？ |

## 投票悖论

| 单张选票改变选举结果的概率≈0——但还有人投票——为什么？ |
|------|
| 表达性投票: 投票不是工具(改变结果)——是表达(我是谁) |
| 公民责任感: 「如果我都不投——凭什么抱怨？」 |

## 社会运动

| 特征 | 内容 |
|------|------|
| 资源动员理论 | 有钱·有人·有组织——才能运动 |
| 政治机会结构 | 政权脆弱时——运动成功率高 |
| 框架 (Framing) | 如何叙事——「这不公平」vs「这是我的权利」 |

## 公共舆论

| 舆论不稳定 (Converse): 大多数人没有「意识形态」——回答随机的 |
|------|
| Zaller: 舆论是精英话语的「回声」——人们接收最近收到的信息 |
