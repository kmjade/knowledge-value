# 04-选举与政党

## 选举制度

| 制度 | 规则 | 效果 |
|------|------|------|
| **多数制** (FPTP) | 一个选区选一人——票多者赢 | 两党制——赢者通吃——英国·美国 |
| **比例代表制** (PR) | 按得票比例分配席位 | 多党制——小党有机会——以色列·荷兰 |
| **混合制** | FPTP + PR 混合 | 德国·新西兰 |

## 杜瓦杰定律

| 多数制 → 两党制 | 比例代表制 → 多党制 |
|------|------|

## 政党的功能

| 功能 | 内容 |
|------|------|
| 聚合利益 | 把社会各群体的诉求整合为政策方案 |
| 选拔领袖 | 培养和提名候选人 |
| 动员选民 | 让支持者出来投票 |
| 组织政府 | 赢得选举→组建内阁 |

## 政党体系

| 体系 | 特征 | 例子 |
|------|------|------|
| 两党制 | 两个大党轮流 | 美国 |
| 温和多党制 | 3-5个党 | 德国 |
| 碎片多党制 | 很多小党——需要联合政府 | 意大利·以色列 |
| 一党制 | 只有一个党(合法) | — |
