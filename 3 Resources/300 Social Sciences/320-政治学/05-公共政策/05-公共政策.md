# 05-公共政策

> 政府做什么——以及不做什么——都是政策。

## 政策循环

```
议程设置 → 政策制定 → 决策 → 执行 → 评估
    ↑                                    │
    └──────────── 反馈 ──────────────────┘
```

## 谁影响政策？

| 行动者 | 如何影响 |
|------|---------|
| 民选官员 | 立法——设定方向 |
| 官僚 | 执行——有「街头官僚」的自由裁量 |
| 利益集团 | 游说·竞选捐款 |
| 媒体 | 设定议程——「CNN效应」 |
| 公众 | 选举·民意·抗议 |

## 政策分析框架

| 框架 | 核心 |
|------|------|
| 理性模型 | 全面分析→选最优方案——太理想 |
| 渐进主义 | 小步调整——「摸着石头过河」——现实 |
| 垃圾桶模型 | 问题·方案·政治——随机匹配——混乱 |
