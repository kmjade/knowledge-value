# 01-政治学概述

> 政治 = 谁得到什么、何时、如何。— Harold Lasswell (1936)

## 核心概念

| 概念 | 定义 |
|------|------|
| **权力** (Power) | A 让 B 做 B 本来不会做的事——Dahl 的定义 |
| **权威** (Authority) | 被承认的、合法的权力——不需要强制 |
| **合法性** (Legitimacy) | 被统治者对统治的接受——韦伯的三种合法性 |
| **国家** (State) | 在一定领土上垄断暴力的组织——韦伯 |

## 韦伯的三种合法性

| 类型 | 基础 | 例子 |
|------|------|------|
| **传统型** | 「历来如此」 | 君主制——国王 |
| **魅力型** (Charismatic) | 领袖的个人魅力 | 革命领袖——拿破仑 |
| **法理型** (Legal-rational) | 制度和法律 | 现代民主国家——选举 |

## 政治学的分支

| 分支 | 问题 |
|------|------|
| 政治理论 | 正义是什么？自由应该怎么定义？ |
| 比较政治 | 为什么有些国家民主化——有些不？ |
| 国际关系 | 为什么国家之间打仗？ |
| 公共政策 | 政府应该做什么？怎么做？ |

## 三种权力面孔

| 面孔 | 说明 |
|------|------|
| 第一面孔 | 公开决策——A 在投票中赢了 B |
| 第二面孔 | 议程设置——不让某些议题进入讨论(Bachrach & Baratz) |
| 第三面孔 | 意识形态——让 B 甚至不觉得自己被支配了(Lukes) |
