# 01-政治学概述

> 拉斯韦尔 (Harold Lasswell): 「政治是谁得到什么、何时、如何。」政治学是研究权力和资源分配的学科。

## 核心概念

| 概念 | 含义 |
|------|------|
| **权力** | A 让 B 做 B 本来不会做的事 |
| **权威** | 合法的权力——被统治者认可的权力 |
| **国家** | 在特定领土上垄断合法暴力的组织(韦伯) |
| **主权** | 最高权力——对外独立·对内最高 |

## 学科分支

| 分支 | 问题 |
|------|------|
| 政治哲学 | 什么是正义？最好的政体是什么？ |
| 比较政治 | 为什么有些国家民主有些专制？ |
| 国际关系 | 国家为什么打仗？ |
| 公共政策 | 政府应该做什么？怎么做？ |
