# 05-国际关系

> 为什么国家之间打仗？为什么有时能合作？国际关系理论试图回答。

## 三大理论

| 理论 | 核心 | 分析层次 |
|------|------|:---:|
| **现实主义** | 国际体系是无政府的——国家追求权力和安全——自助 | 体系 |
| **自由主义** | 制度·贸易·民主促进和平——合作可能 | 制度/国家 |
| **建构主义** | 利益和身份是社会建构的——规范和话语塑造行为 | 观念 |

## 现实主义核心

| 概念 | 内容 |
|------|------|
| **无政府状态** | 没有世界政府——国家最终只能靠自己 |
| **安全困境** | 我加强军备为了安全→你感到威胁→你也加强→我却不安全了 |
| **均势** | 大国之间的实力平衡——防止霸权 |
| **霸权稳定论** | 有一个霸主维护秩序——体系稳定 |

## 民主和平论

| 自由民主国家之间不打仗 (几乎没有例外) |
|------|
| 解释: 制度约束(选民的战争成本)·规范(和平解决争端)·贸易依赖 |
| 批评: 民主国家和非民主国家之间照样打 |

## 国际制度

| 制度 | 功能 |
|------|------|
| **联合国** | 集体安全——安理会——维和 |
| **WTO** | 贸易规则——争端解决 |
| **IMF/世界银行** | 金融稳定/发展援助——但条件性引发争议 |
