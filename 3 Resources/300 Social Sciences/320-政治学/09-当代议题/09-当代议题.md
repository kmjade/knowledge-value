# 09-当代议题

> 21世纪的政治挑战——旧理论能解释新世界吗？

## 民粹主义

| 特征 | 内容 |
|------|------|
| 核心叙事 | 「纯洁的人民」vs「腐败的精英」 |
| 左右都有 | 左翼民粹(经济)·右翼民粹(文化/民族) |
| 原因 | 全球化失利者·文化焦虑·社交媒体放大 |

## 政治极化

| 美国: 民主党和共和党的意识形态距离——内战以来最大 |
|------|
| 原因: 分类(sorting)·媒体「回音室」·初选制度(极端候选人) |
| 后果: 立法僵局·社会撕裂·民主倒退 |

## 技术政治

| 议题 | 内容 |
|------|------|
| **监视资本主义** | 平台收集数据→预测/操纵行为——Zuboff |
| **算法治理** | 谁写代码——谁统治——信用评分·预测警务·内容审核 |
| **假信息** | 民主需要共同的事实基础——这个基础正在瓦解 |

## 民主的未来

| 乐观: 民主是人类政治演化的终点 (福山) |
|------|
| 悲观: 民主正在全球退潮——威权主义找到了新模式 |
| 现实: 民主需要持续维护——不是「达到了」就一劳永逸 |
