# 06-国际关系

> 国际体系是无政府的——没有「世界政府」。国家之上没有更高的权威——这是国际关系的起点。

## 三大理论传统

| 理论 | 核心 | 代表 |
|------|------|------|
| **现实主义** | 国家追求权力和安全——国际关系 = 权力斗争 | Morgenthau·Mearsheimer |
| **自由主义** | 贸易·国际制度·民主可以促进和平 | Keohane·「民主和平论」 |
| **建构主义** | 利益和身份不是固定的——是**社会建构**的 | Wendt——「无政府是国家造成的」 |

## 核心概念

| 概念 | 内容 |
|------|------|
| 主权 | 国家之上无权威——不干涉内政 |
| 安全困境 | A加强国防→B感到不安全→B也加强→A更不安全——军备竞赛 |
| 软实力 | 不是强迫——是**吸引**——文化·价值观(Nye) |
| 国际制度 | UN·WTO·IMF——自由主义的核心工具 |
