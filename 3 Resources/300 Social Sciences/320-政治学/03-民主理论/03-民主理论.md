# 03-民主理论

> 民主不是「多数人投票」那么简单——它是人类迄今为止最复杂的政治发明。

## 民主的经典定义

| 理论家 | 定义 |
|------|------|
| **林肯** | «of the people, by the people, for the people» |
| **熊彼特** | 「通过竞争性选举获取权力的制度安排」——精英竞争 |
| **达尔** (Dahl) | 多元政体——参与+竞争+自由——七个制度条件 |

## 达尔的多元政体条件

| # | 条件 |
|:--:|------|
| 1 | 选举产生的官员 |
| 2 | 自由、公正、定期的选举 |
| 3 | 包容性(普选权) |
| 4 | 参选权 |
| 5 | 言论自由 |
| 6 | 替代性信息(媒体自由) |
| 7 | 结社自由 |

## 民主的模型

| 模型 | 核心 | 批评 |
|------|------|------|
| **代议民主** | 选代表——他们替你做决定 | 精英主义——代理人问题 |
| **参与民主** | 公民直接参与——不只是投票 | 太费时间——大多数人不愿意 |
| **协商民主** (Deliberative) | 理性讨论——寻求共识——Habermas | 理想主义——现实中权力不平等 |
| **激进民主** | 民主永远是未完成的——不断挑战现状 | 「永远革命」——不稳定 |

## 民主化的条件

| 条件 | 说明 |
|------|------|
| 经济发展 | Lipset (1959): 越富裕越可能民主——相关性≠因果 |
| 公民文化 | Almond & Verba: 参与感+信任+对系统的忠诚 |
| 国际环境 | 民主国家群体对威权国家的压力 |
| 精英分裂 | 威权精英内部分裂是民主化的窗口 |
