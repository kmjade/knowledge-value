# 02-政体类型

> 政治体制是权力的组织方式——从绝对君主到直接民主。

## 亚里士多德的分类

| 统治者 | 好的形式 | 坏的形式 |
|------|------|------|
| 一人 | 君主制 | 僭主制 |
| 少数 | 贵族制 | 寡头制 |
| 多数 | 政体(Polity) | 民主(暴民) |

## 现代政体光谱

| 类型 | 特征 | 例子 |
|------|------|------|
| **自由民主** | 自由选举+公民权利+法治+权力制衡 | 德·英·日 |
| **选举威权** | 有选举——但不自由——反对派被压制 | 俄罗斯·土耳其 |
| **一党制** | 一个政党控制——无竞争 | 中国·越南 |
| **绝对君主** | 君主有实权 | 沙特·阿联酋 |
| **军事政权** | 军队直接或间接统治 | 缅甸(2021后) |
| **失败国家** | 政府不能提供基本秩序 | 索马里 |

## 总统制 vs 议会制

| | 总统制 | 议会制 |
|------|:---:|:---:|
| 行政 | 总统独立选举——固定任期 | 总理由议会产生——可被不信任案推翻 |
| 立法-行政 | 分离——制衡 | 融合——「高效的」 |
| 稳定 | 僵局风险——但任期固定 | 灵活——但政府可能频繁更换 |
| 例子 | 美国·巴西·韩国 | 英国·德国·日本 |

## 混合政体：半总统制

| 总统+总理——法国模式——「共治」(cohabitation)：总统和总理来自不同政党时必须合作 |
