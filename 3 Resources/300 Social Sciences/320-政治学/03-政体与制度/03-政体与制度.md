# 03-政体与制度

## 政体光谱

| 政体 | 特征 |
|------|------|
| 自由民主 | 自由选举·法治·言论自由·权力制衡 |
| 选举威权 | 有选举——但不自由——「竞争性威权主义」 |
| 威权 | 无自由选举——权力集中——通过压制维持 |
| 极权 | 控制一切——思想·经济·私人生活——意识形态驱动 |

## 政府形式

| 形式 | 特征 | 例子 |
|------|------|------|
| 总统制 | 总统 = 国家元首 + 行政首长——直接选举 | 美国 |
| 议会制 | 首相从议会中产生——可被不信任投票罢免 | 英国 |
| 半总统制 | 总统 + 总理——双头行政 | 法国 |
| 联邦 vs 单一制 | 地方有宪法自治 vs 中央集权 | 美国 vs 法国 |

## 制度

| 制度 | 功能 |
|------|------|
| 宪法 | 最高法律——限制权力 |
| 司法独立 | 法院不受政治干预 |
| 文官制度 | 专业·非党派的行政机构 |
