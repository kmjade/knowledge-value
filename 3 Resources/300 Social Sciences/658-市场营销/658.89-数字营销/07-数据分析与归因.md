# 07 數據分析與歸因

> GA4/GTM 實施、UTM 參數體系、歸因模型 (First/Last/Linear/Data-Driven)、CRO 轉化率優化、A/B 測試統計。

## GA4 vs Universal Analytics

| 維度 | Universal Analytics (GA3) | GA4 |
|------|---------------------------|-----|
| 數據模型 | Session-based | Event-based |
| 指標 | Bounce Rate, Session Duration | Engagement Rate, Avg Engagement Time |
| 跨平台 | Web only | Web + App 統一數據流 |
| 歸因 | Last Non-Direct Click (默認) | Data-Driven Attribution (默認) |
| 報表 | 預設固定報表 | 自定義探索報表 (Explorations) |
| 隱私 | Cookie 依賴 | 信號增強 (Google Signals), 無 Cookie 建模 |
| 免費限額 | 10M Hits/月 | 無 Hit 限制 (事件參數有限制) |

## UTM 參數體系

| 參數 | 必填 | 說明 | 示例 |
|------|:----:|------|------|
| utm_source | ✅ | 流量來源 | google, baidu, wechat, douyin |
| utm_medium | ✅ | 媒介類型 | cpc, organic, email, social |

## 歸因模型對比

| 模型 | 邏輯 | 適用場景 | 優點 | 缺點 |
|------|------|----------|------|------|
| First Click | 100% 歸於首次接觸 | 品牌認知階段 | 識別獲客渠道 | 忽略後續助攻 |
| Last Click | 100% 歸於末次接觸 | 直接轉化場景 | 簡單直觀 | 忽略前期影響 |
| Linear | 均分給所有觸點 | 週期長、觸點多 | 公平分配 | 不分影響力大小 |
| Time Decay | 越近轉化權重越大 | 短決策週期 | 體現近期助攻 | 忽略初始觸發 |
| Position-Based | 40-20-40 (首-中-末) | 電商、Lead Gen | 兼顧首末 | 忽視中間觸點 |
| Data-Driven | 算法分配 (Shapley 值) | 數據充足時 | 最科學 | 需大量轉化數據 |

## CRO 轉化率優化框架

| 階段 | 動作 | 工具/方法 |
|------|------|-----------|
| 1. 研究 | 數據分析 + 用戶調研 (Hotjar/錄屏/問卷) | GA4, Hotjar, 用戶訪談 |
| 2. 假設 | 基於發現形成優化假設 | Hypothesis statement 模板 |
| 3. 優先級 | ICE/PIE 框架排序 | RICE Score (Reach×Impact×Confidence/Effort) |
| 4. 設計 | 低保真 → 高保真原型 | Figma, Sketch |
| 5. 測試 | A/B 測試 or 多變量測試 | Optimizely, VWO, Google Optimize |
| 6. 學習 | 歸檔測試結果, 提煉洞察 | Testing Log, Wiki |

## A/B 測試統計基礎

| 概念 | 說明 | 行業標準 |
|------|------|:--------:|
| 樣本量 | 達到統計顯著所需最小流量 | 計算器: Evan Miller, Optimizely |
| 顯著性水平 α | Type I Error (假陽性) | 一般 95% (α=0.05) |
| 統計功效 1-β | Type II Error (假陰性) | 一般 80% |
| MDE | 最小可檢測效應 (Minimum Detectable Effect) | 根據基線轉化率設定 |
| 測試週期 | 包含完整商業週期 (通常 ≥2 週) | 避開節假日異常 |
| SRM | Sample Ratio Mismatch 檢測 | p > 0.01 無顯著偏差 |

## 核心分析指標儀表板

| 維度 | 指標 | 計算方式 |
|------|------|----------|
| 流量 | Sessions, Users, Pageviews | GA4 標準 |
| 獲客 | 各渠道 Session %, 新用戶 % | GA4 Acquisition Report |
| 參與度 | Engagement Rate, Avg Session Duration | GA4 Engagement Report |
| 轉化 | Conversion Rate, Goal Completions | GA4 Conversions |
| 收入 | Revenue, ARPU, AOV | Ecommerce Tracking |
| 留存 | DAU/MAU, Day 1/7/30 retention | GA4 Retention |
