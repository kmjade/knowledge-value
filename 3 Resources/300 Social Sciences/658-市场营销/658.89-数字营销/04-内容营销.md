# 04 內容營銷

> 內容策略 → 創作 → 分發 → 量化。博客/白皮書/案例/短視頻矩陣。SEO 寫作 × 內容 ROI × 內容日曆。

## 內容矩陣 (Content Matrix)

| 內容類型 | 目標受眾 | 目標 | 中國平台 | 全球平台 |
|----------|----------|------|----------|----------|
| 博客/公眾號文章 | 中層漏斗 | SEO 獲客, 專業信任 | 微信公眾號, 知乎 | Blog, Medium |
| 短視頻/直播 | 頂層漏斗 | 曝光, 種草, 轉化 | 抖音, 快手, 視頻號 | YouTube Shorts, TikTok, Reels |
| 白皮書/電子書 | 底層漏斗 | Lead Gen, B2B 轉化 | 36Kr, 虎嗅 | HubSpot, Gated Content |
| 案例研究 | 底層漏斗 | 信任背書, 轉化加速 | 小紅書筆記 | Website, LinkedIn |
| 圖文/海報 | 頂層漏斗 | 品牌曝光, 社交傳播 | 小紅書, 朋友圈 | Instagram, Pinterest |
| 播客/音頻 | 中層漏斗 | 深度影響, 專業 PGC | 喜馬拉雅, 小宇宙 | Spotify, Apple Podcasts |
| Webinar/直播課 | 底層漏斗 | 教育轉化, B2B | 騰訊會議, 小鵝通 | Zoom, Demio |

## 內容策略框架

| 階段 | 動作 | 工具/方法 |
|------|------|-----------|
| 1. 審計 | 盤點現有內容, 分析 Gap | Screaming Frog, 內容清單 |
| 2. 受眾 | Buyer Persona, 搜索意圖分析 | Google Analytics, 問卷調研 |
| 3. 關鍵詞 | 關鍵詞研究 → 主題聚類 (Topic Cluster) | Ahrefs, SEMrush |
| 4. 日曆 | 季度/月度內容日曆, 分配渠道 | Notion, Google Calendar |
| 5. 創作 | 寫作 (SEO 規範), 設計, 視頻製作 | Surfer SEO, Canva, CapCut |
| 6. 分發 | 自有 → 付費 → 賺取渠道組合 | Buffer, Hootsuite, 新榜 |
| 7. 量化 | 流量/轉化/SEO 排名/社交互動 | GA4, Ahrefs Dashboards |

## SEO 寫作規範

| 元素 | 最佳實踐 |
|------|----------|
| Title Tag | 50-60 字元, 關鍵詞前置, 吸引點擊 |
| Meta Description | 150-160 字元, 包含 CTA, 摘要價值 |
| H1 | 一個, 包含主關鍵詞 |
| H2-H3 結構 | 層級清晰, 長尾關鍵詞覆蓋 |
| 首段 | 前 100 字內出現主關鍵詞 |
| 內容長度 | 信息型 1500-2500 字; 商業型 1000-2000 字 |
| 多媒體 | 圖片+Alt 標籤, 影片嵌入, 圖表 |
| 內部鏈接 | 3-5 條相關文章鏈接, 錨文本多樣化 |

## 內容分發模型 (PESO)

| 渠道類型 | 說明 | 示例 |
|----------|------|------|
| Paid 付費 | 付費推廣 | 信息流廣告, SEM, KOL 投放 |
| Earned 賺取 | 自然傳播 | PR 報導, UGC, 口碑分享 |
| Shared 社交 | 社交媒體 | 微博轉發, 微信朋友圈, LinkedIn |
| Owned 自有 | 自有渠道 | 官網 Blog, 公眾號, 郵件列表 |

## 內容 ROI 量化

| 指標 | 公式 / 衡量方式 |
|------|-----------------|
| 有機流量 | GA4 自然流量 (Sessions) |
| 關鍵詞排名 | Ahrefs/SEMrush 排名追蹤 |
| 轉化率 | Landing Page 轉化率 |
| 內容獲客成本 | 內容總成本 ÷ 內容帶來的新 Leads |
| 社交互動 | 點贊+評論+轉發+收藏 |
| 品牌搜索量 | Google Trends / 百度指數 |
| 內容資產複利 | 疊加每篇內容的持續月流量 |
