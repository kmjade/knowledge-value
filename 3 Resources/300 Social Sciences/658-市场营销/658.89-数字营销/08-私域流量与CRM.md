# 08 私域流量與 CRM

> 公域 vs 私域流量模式。微信生態 (公眾號/企業微信/視頻號/小程序) 閉環。會員體系設計、RFM 模型、CDP 客戶數據平台。

## 公域 vs 私域

| 維度 | 公域流量 | 私域流量 |
|------|----------|----------|
| 定義 | 平台擁有的流量池, 需付費或算法獲取 | 品牌自主可控的用戶池 |
| 所有權 | 平台 (淘寶/抖音/百度) | 品牌自己 |
| 觸達成本 | 每次觸達需付費 (CPC/CPM) | 邊際成本接近零 |
| 觸達方式 | 廣告競價 / 算法推薦 | 公眾號推送/社群/1v1 |
| 關係深度 | 弱關係, 一次性 | 強關係, 可持續互動 |
| 複購驅動 | 搜索/推薦再次觸達 | 社群/消息直接喚起 |
| 典型陣地 | 淘寶搜索, 抖音推薦頁 | 微信社群, 公眾號, 企業微信 |
| LTV 表現 | 較低 (流失不可控) | 較高 (可持續經營) |

## 微信生態矩陣

| 組件 | 定位 | 核心功能 | 私域角色 |
|------|------|----------|----------|
| 公眾號 | 內容中樞 | 圖文推送, 模板消息, 菜單 | 內容種草 + 服務通知 |
| 企業微信 | 客戶連接 | 1v1/社群, 歡迎語, 快捷回覆 | 銷售賦能 + SCRM |
| 小程序 | 交易載體 | 商城, 會員中心, 活動頁 | 交易閉環 |
| 視頻號 | 短視頻/直播 | 短視頻, 直播帶貨 | 公域引流入口 |
| 微信支付 | 支付+營銷 | 支付有禮, 商家券 | 復購觸發 |
| 搜一搜 | 搜索入口 | 品牌專區, 服務直達 | 主動搜索攔截 |

## 會員體系設計

| 層級 | 門檻 | 權益設計 | 目標 |
|------|------|----------|------|
| 普通會員 | 註冊/授權 | 積分, 簽到, 基礎折扣 | 留存 |
| 銀卡 | 累積消費 ¥500 | + 生日禮, 優先客服 | 複購 |
| 金卡 | 累積消費 ¥2000 | + 免運費, 新品優先, 專屬活動 | 忠誠 |
| 鑽石卡 | 累積消費 ¥5000+ | + 私人顧問, 線下活動, 共創 | 品牌大使 |
| 付費會員 | 年費 ¥99-365 | 會員價, 專屬品, 線上課 | 鎖定 LTV |

## RFM 模型

| 維度 | 定義 | 分數 (1-5) | 計算方式 |
|------|------|:----------:|----------|
| R - Recency | 最近一次消費時間 | 5=最近 | 天數越少分數越高 |
| F - Frequency | 消費頻率 | 5=最頻繁 | 次數越多分數越高 |
| M - Monetary | 消費金額 | 5=最高 | 金額越大分數越高 |

### RFM 客戶分群

| 分群 | R | F | M | 策略 |
|------|:--:|:--:|:--:|------|
| 重要價值客戶 | 5 | 5 | 5 | VIP 專屬服務, 新品優先體驗 |
| 重要發展客戶 | 5 | 3-4 | 3-4 | Cross-sell, 會員升級引導 |
| 重要保持客戶 | 3-4 | 5 | 4-5 | 流失預警, 專屬優惠召回 |
| 重要挽留客戶 | 1-2 | 5 | 4-5 | 大力度優惠, 1v1 專人跟進 |
| 一般價值客戶 | 4-5 | 2-3 | 2-3 | Upsell 引導, 內容培育 |
| 流失客戶 | 1-2 | 2-3 | 1-2 | 廉價引流品召回, 或放棄 |

## CDP 客戶數據平台

| 功能模塊 | 說明 | 代表工具 |
|----------|------|----------|
| 數據接入 | SDK/API 接入多渠道數據 | Segment, mParticle, 神策 |
| ID Mapping | 跨設備/跨渠道 One ID | 手機號/UnionID/DeviceID 關聯 |
| 標籤體系 | 屬性標籤 + 行為標籤 + 預測標籤 | 靜態/動態/算法標籤 |
| 人群圈選 | 任意條件組合創建人群包 | 可導出至廣告平台/Marketing Automation |
| 分析洞察 | 漏斗, 留存, 路徑分析 | 群畫像對比 (Cohort) |
| 營銷觸達 | 對接短信/EDM/Push/公眾號 | 自動化旅程+即時觸發 |

## 私域運營指標

| 指標 | 公式 / 定義 |
|------|-------------|
| 私域用戶規模 | 公眾號粉絲 + 企業微信好友 + 社群成員 (去重) |
| 好友添加率 | 新增好友數 ÷ 曝光/進店人數 |
| 社群留存率 | 週/月社群成員留存率 |
| 私域 GMV | 私域渠道總成交額 |
| 私域佔比 | 私域 GMV ÷ 總 GMV |
| 訊息互動率 | (閱讀+點讚+評論) ÷ 粉絲數 |
| 客戶流失率 | 刪好友/退群/取關 ÷ 總私域用戶 |
