# 07-金融与货币

> 钱的价格是利率——利率影响一切。从央行货币政策到资本市场，从有效市场假说到行为金融。

---

## Part A: 货币与银行

### 货币的功能

| 功能 | 说明 |
|------|------|
| **交易媒介** | 不用以物易物 |
| **计价单位** | 统一的尺度 |
| **价值储藏** | 可以存着——以后用（通胀侵蚀这个功能） |

### 利率的决定

| 名义利率 = 实际利率 + 预期通胀 |
|------|
| 央行设定基准利率（联邦基金利率 / LPR）——影响整个利率体系 |
| 负利率：日本·欧元区实验——存钱要付钱——效果争议 |

### 金融危机三幕

| 危机 | 原因 | 教训 |
|------|------|------|
| **1929 大萧条** | 股市泡沫 + 银行挤兑 + 金本位 | 央行必须做最后贷款人 |
| **2008 次贷** | 房地产泡沫 + 衍生品 + 影子银行 | 系统性风险——太大不能倒 |
| **1997 亚洲** | 热钱涌入 + 固定汇率 + 短期外债 | 资本账户开放需谨慎 |

---

## Part B: 资本市场

### 有效市场假说 (EMH — Fama)

| 股价反映了**所有**公开信息——你不能「打败市场」——除非有内幕信息 |
|------|
| 含义：猴子扔飞镖选股——和基金经理的表现差不多 |

### 风险与回报

| 原则 | 高风险 → 高期望回报——但「期望」≠「保证」 |
|------|------|
| 分散化 | 「不要把鸡蛋放在一个篮子里」——降低非系统性风险 |

### CAPM (资本资产定价模型)

> 资产的期望回报 = 无风险利率 + β × 市场风险溢价
> β：资产对市场的敏感度——β>1 = 激进型，β<1 = 防御型

### 行为金融

| 挑战 EMH | 投资者不是理性的——泡沫·恐慌·过度反应 |
|------|------|
| 例子 | 2008 金融危机——有效市场？ |

|------|------|
| CBDC | 央行数字货币——数字人民币——可编程货币——隐私争议 |
