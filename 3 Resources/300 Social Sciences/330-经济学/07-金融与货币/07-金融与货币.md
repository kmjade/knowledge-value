# 07-金融与货币

> 钱的价格是利率——利率影响一切——从你的房贷到国家的经济增长。

## 货币的功能

| 功能 | 说明 |
|------|------|
| **交易媒介** | 不用以物易物 |
| **计价单位** | 统一的尺度——「这个值3个面包」 |
| **价值储藏** | 可以存着——以后用(通胀侵蚀这个功能) |

## 利率的决定

| 名义利率 = 实际利率 + 预期通胀 |
|------|
| 央行设定基准利率(联邦基金利率/LPR) ——影响整个利率体系 |
| 负利率: 日本·欧元区实验——存钱要付钱——效果争议 |

## 金融危机

| 危机 | 原因 | 教训 |
|------|------|------|
| 1929大萧条 | 股市泡沫+银行挤兑+金本位 | 央行必须做最后贷款人 |
| 2008次贷 | 房地产泡沫+衍生品+影子银行 | 系统性风险——太大不能倒 |
| 1997亚洲 | 热钱涌入+固定汇率+短期外债 | 资本账户开放需谨慎 |

## 数字货币

| 比特币 (2009) | 去中心化——固定供应(2100万)——不是货币——是数字黄金 |
|------|------|
| CBDC | 央行数字货币——数字人民币——可编程货币——隐私争议 |
