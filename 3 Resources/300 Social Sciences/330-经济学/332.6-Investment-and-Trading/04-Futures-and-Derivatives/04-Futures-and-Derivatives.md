# 04 期貨與衍生品 Futures & Derivatives

> 衍生品：買賣的不是資產本身，而是關於資產的合約。

## 衍生品家譜 Derivative Family

```
衍生品 Derivatives
├── 遠期合約 Forward — 場外、客製化
├── 期貨 Futures — 場內、標準化
│   ├── 商品期貨 (原油、黃金、農產品)
│   ├── 金融期貨 (股指、國債、外匯)
│   └── CFD (差價合約) — 場外零售
├── 選擇權 Options
│   ├── 買權 Call — 買入的權利
│   └── 賣權 Put — 賣出的權利
└── 互換 Swap — 利率互換、貨幣互換
```

### 期貨 vs CFD

| | 期貨 Futures | CFD 差價合約 |
|---|-------------|-------------|
| 監管 | 交易所監管 | 經紀商自行管理 |
| 到期日 | 有（需轉倉） | 無 |
| 對手方 | 中央清算 | 經紀商 |
| 最小合約 | 較大 | 可微調 (0.01手) |
| 適合 | 專業/機構 | 零售交易者 |

## 選擇權 Options

### 基本概念

| 術語 | 含義 |
|------|------|
| **Call (買權)** | 以履約價**買入**標的的權利 |
| **Put (賣權)** | 以履約價**賣出**標的的權利 |
| **履約價 Strike** | 約定的交易價格 |
| **到期日 Expiry** | 合約失效日期 |
| **權利金 Premium** | 買方付給賣方的費用 |
| **價內 ITM** | 履約價優於市價（有內在價值） |
| **價外 OTM** | 履約價劣於市價（僅時間價值） |
| **價平 ATM** | 履約價 ≈ 市價 |

### 基本策略

| 策略 | 構成 | 觀點 | 風險 |
|------|------|------|------|
| Long Call | 買 Call | 看漲 | 限於權利金 |
| Long Put | 買 Put | 看跌 | 限於權利金 |
| Covered Call | 持股 + 賣 Call | 中性偏漲 | 機會成本 |
| Protective Put | 持股 + 買 Put | 對沖下跌 | 權利金 |
| Straddle | 買 Call + 買 Put | 大波動（方向不明） | 雙倍權利金 |

## CFD 交易要點

CFD (差價合約) 是零售交易者最常用的衍生品工具。

| 特性 | 說明 |
|------|------|
| 標的 | 股票、指數、外匯、商品、加密貨幣 |
| 槓桿 | 通常 5:1–30:1（取決於監管） |
| 做空 | 與做多一樣容易 |
| 成本 | 點差 + 隔夜利息 |
| 股息調整 | 持有多頭倉位時收到（扣除稅後） |
