## 資金管理 Money Management

### 核心規則

| 規則 | 數值 | 說明 |
|------|------|------|
| **每筆風險上限** | ≤ 2% | 單筆交易最大虧損佔總資金的比例 |
| **每日風險上限** | ≤ 5% | 當日累計虧損達標即停止交易 |
| **每週風險上限** | ≤ 10% | 當週虧損超標 → 下週減倉 |

### 倉位計算公式

```
手數 = (帳戶餘額 × 風險%) / (止損點數 × 每點價值)

例：
$10,000 帳戶 × 2% = $200 風險
止損 50 點 × EURUSD ($10/標準手) = $500/標準手
手數 = $200 / $500 = 0.4 手
```

### 風險報酬比 Risk-Reward Ratio (R:R)

```
R:R = 潛在盈利 / 潛在虧損

R:R 1:1 → 需 >50% 勝率才能盈利
R:R 1:2 → 只需 >34% 勝率即可盈利
R:R 1:3 → 只需 >25% 勝率即可盈利
```

> 💡 永遠追求 **R:R ≥ 1:2**。

### R:R 與勝率矩陣

| 勝率 \ R:R | 1:1 | 1:2 | 1:3 |
|-----------|-----|-----|-----|
| 30% | ❌ 虧 | ❌ 虧 | ❌ 虧 |
| 40% | ❌ 虧 | ✅ 盈 | ✅ 盈 |
| 50% | ➖ 平 | ✅ 盈 | ✅ 盈 |
| 60% | ✅ 盈 | ✅ 盈 | ✅ 盈 |

## 常見風控錯誤 Common Mistakes

| 錯誤 | 後果 |
|------|------|
| **不設止損** | 一筆虧損清空帳戶 |
| **移動止損（擴大）** | 違反原始計劃，情緒化決策 |
| **Martingale 加倍** | 連續虧損時爆倉加速 |
| **過度交易** | 點差/手續費吃掉利潤 |
| **報復性交易** | 虧損後急於翻本 → 更大虧損 |
| **贏小虧大** | R:R < 1:1 的系統必死 |

## 止損策略 Stop Loss Strategies

### 固定點數止損

```cpp
// 最簡單：固定 N 點止損
int StopLoss = 50;  // 50 點
```

### 技術止損 (結構止損)

```
止損放在：
- 最近的前期低點之下 (做多)
- 最近的前期高點之上 (做空)
- 支撐/阻力位之外 + 5–10 點緩衝
```

### ATR 止損 (波動率止損)

```
止損 = 1.5 × ATR(14)

以當前波動率為基準，避免被噪音掃出場。
```

### 移動止損 Trailing Stop

```cpp
// MQL4 移動止損範例
void TrailingStop() {
   double trail_points = 30;  // 30 點
   for(int i = 0; i < OrdersTotal(); i++) {
      if(OrderSelect(i, SELECT_BY_POS) && OrderMagicNumber() == MAGIC) {
         if(OrderType() == OP_BUY) {
            if(Bid - OrderOpenPrice() > trail_points * Point) {
               double new_sl = Bid - trail_points * Point;
               if(new_sl > OrderStopLoss())
                  OrderModify(OrderTicket(), OrderOpenPrice(), new_sl, 
                             OrderTakeProfit(), 0);
            }
         }
      }
   }
}
```

## 回撤控制 Drawdown Control

| 回撤 | 狀態 | 行動 |
|------|------|------|
| < 10% | 🟢 正常 | 維持策略 |
| 10–15% | 🟡 警告 | 減倉 50% |
| 15–20% | 🟠 危險 | 停止交易、復盤 |
| > 20% | 🔴 失控 | **立即停止**、重新評估策略 |

## 凱利公式 Kelly Criterion

```
f* = (bp − q) / b

f* = 最優倉位比例
b  = 盈虧比 (淨盈利/淨虧損)
p  = 勝率
q  = 1 − p

例：勝率 45%、盈虧比 2:1
f* = (2×0.45 − 0.55) / 2 = 0.175
最佳倉位 = 17.5%（通常使用 1/2 或 1/4 凱利）
實際倉位 = 4–8%
```

> ⚠️ 凱利公式假設你知道真實勝率和盈虧比，這在現實中很難確定。使用半凱利 (½ Kelly) 更安全。
