## 投資流派 Investment Styles

### 價值投資 Value Investing

- **核心**：買被低估的優質公司，長期持有
- **代表人物**：巴菲特、葛拉漢
- **關鍵指標**：PE < 行業平均、PB < 2、ROE > 15%、高股息率

### 成長投資 Growth Investing

- **核心**：買高增長公司，忍受高估值
- **典型**：科技股、生物醫藥
- **指標**：營收增長 >20%、廣闊的 TAM

### 指數投資 Index Investing

- **核心**：買 ETF，獲取市場平均回報
- **優勢**：低費用、分散風險、不用選股
- **推薦**：SPY (標普500)、QQQ (納斯達克100)、滬深300ETF

### 技術交易 Technical Trading

- **核心**：基於價格走勢和指標，不關心基本面
- **工具**：K 線型態、均線、MACD、RSI
- **時間框架**：日內 → 波段

## 關鍵財務指標 Key Financial Metrics

| 指標 | 公式 | 含義 |
|------|------|------|
| **PE (本益比)** | 股價 / EPS | 越低越便宜（但需看行業） |
| **PB (市淨率)** | 股價 / 每股淨資產 | <1 = 破淨（潛在低估） |
| **ROE** | 淨利潤 / 股東權益 | >15% 優秀 |
| **股息率** | 每股股息 / 股價 | 穩定現金流的來源 |
| **負債率** | 總負債 / 總資產 | <50% 健康 |
| **毛利率** | 毛利 / 營收 | 行業越高越好 |
