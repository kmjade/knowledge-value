# 01 投資基礎 Investment Basics

> 交易之前，先理解遊戲規則。

## 金融市場類型 Market Types

| 市場 | 交易標的 | 特點 |
|------|---------|------|
| **股票市場** | 公司股份 | T+1 (A股)、T+0 (港股/美股) |
| **外匯市場** | 貨幣對 | 24h、全球最大市場 ($7.5T/日) |
| **期貨市場** | 商品/指數合約 | 到期日、保證金交易 |
| **債券市場** | 政府/公司債 | 固定收益 |
| **加密貨幣** | BTC/ETH 等 | 24/7、高波動 |

## 交易基礎概念

### 訂單類型 Order Types

| 類型 | 說明 | 用途 |
|------|------|------|
| **市價單** Market Order | 以當前市價立即成交 | 快速進場/出場 |
| **限價單** Limit Order | 指定價格成交 | 精確入場 |
| **止損單** Stop Loss | 價格觸及後自動平倉 | 限制虧損 |
| **止盈單** Take Profit | 價格觸及後自動平倉 | 鎖定利潤 |
| **掛單** Pending Order | 條件觸發後生效 | 突破/回調策略 |

### 交易成本 Trading Costs

| 成本 | 說明 |
|------|------|
| **點差 Spread** | 買價與賣價的差額（外匯主要成本） |
| **佣金 Commission** | 每手固定費用（ECN 帳戶） |
| **隔夜利息 Swap** | 持倉過夜的利息（多空利率不同） |
| **滑點 Slippage** | 預期與實際成交價的差異 |

### 保證金與槓桿 Margin & Leverage

```
槓桿 100:1 → 1% 保證金 → $1,000 可控制 $100,000 倉位
槓桿 500:1 → 0.2% 保證金 → $200 可控制 $100,000 倉位
```

| 槓桿 | 優點 | 風險 |
|------|------|------|
| 低 (10–30x) | 安全 | 資金效率低 |
| 中 (50–100x) | 平衡 | 適度風險 |
| 高 (200–500x) | 小資金大倉位 | **爆倉風險極高** |

> ⚠️ 高槓桿是一把雙刃劍。建議新手使用不超過 100:1。

### 倉位計算 Position Sizing

```
手數 = (帳戶餘額 × 風險比例%) / (止損點數 × 每點價值)

例：$10,000 帳戶，風險 2% ($200)，止損 50 點，EURUSD (每點 ≈ $10/標準手)
手數 = $200 / (50 × $10) = 0.4 標準手
```

## 新手常見錯誤 Rookie Mistakes

1. ❌ **不設止損** — 一筆虧損吃掉十筆盈利
2. ❌ **過度交易** — 每天 10+ 筆，手續費吃掉利潤
3. ❌ **逆勢加倉** — Martingale 策略終將爆倉
4. ❌ **不設風險上限** — 單筆風險 >5% 等於自殺
5. ❌ **沒有交易日誌** — 不知道自己為什麼虧錢
