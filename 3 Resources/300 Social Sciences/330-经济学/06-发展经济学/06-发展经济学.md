# 06-发展经济学

> 为什么有些国家富裕、有些国家贫穷？这是经济学的终极问题。

## 增长的引擎

| 因素 | 机制 |
|------|------|
| **资本积累** | 投资→更多机器·基础设施→生产率↑ |
| **人力资本** | 教育·健康→更聪明的劳动力 |
| **技术** | 创新→全要素生产率(TFP) ——「索洛剩余」 |
| **制度** | 产权保护·法治·合同执行→激励投资 |

## 制度的决定性作用

| Acemoglu & Robinson (2012): 《国家为什么会失败》 |
|------|
| 包容性制度→繁荣 / 汲取性制度→贫困 |
| 例子: 朝鲜vs韩国——相同文化·地理·历史起点——制度分歧导致天壤之别 |

## 贫困陷阱

| 贫困本身导致持续贫困——没有足够的储蓄投资——营养不良降低生产力 |
|------|
| 破解: 外部援助·小额贷款·条件现金转移(如巴西Bolsa Família) |

## 随机对照试验革命 (RCT)

| 2019诺贝尔奖: Banerjee·Duflo·Kremer |
|------|
| 用实验方法测试发展干预——「证据为基础的发展」 |
| 例子: 驱虫→出勤率↑→终身收入↑ (极低成本——高回报) |
