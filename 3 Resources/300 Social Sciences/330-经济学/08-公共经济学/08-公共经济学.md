# 08-公共经济学

> 政府应该做什么？税收和支出如何影响经济和社会？

## 市场失灵的政府回应

| 市场失灵 | 政府回应 |
|------|------|
| 外部性(污染) | 碳税·排放交易 |
| 公共品(国防) | 政府直接提供 |
| 信息不对称 | 强制披露·许可制度 |
| 垄断 | 反垄断·价格管制 |

## 税收原则

| 原则 | 说明 |
|------|------|
| **效率** | 税收扭曲越少越好——「不伤害经济」 |
| **公平** | 横向公平(相同的人付相同的税)·纵向公平(富人多付) |
| **简单** | 容易理解和遵守——复杂的税制导致逃税和合规成本 |

## 税种

| 税种 | 特点 |
|------|------|
| **所得税** | 累进(多得多付)——直接税——最「可见」 |
| **增值税** (VAT) | 消费环节征收——间接税——效率高但对穷人影响大 |
| **企业税** | 全球化时代——跨国公司避税——最低税率竞赛 |
| **财产税** | 最不扭曲——但政治阻力最大 |

## 福利国家

| 类型 (Esping-Andersen) | 特征 | 例子 |
|------|------|------|
| **自由主义** | 最低安全网——市场主导 | 美·英·澳 |
| **保守/统合** | 社会保险——维持地位差异 | 德·法 |
| **社会民主** | 普遍主义——去商品化 | 北欧 |
