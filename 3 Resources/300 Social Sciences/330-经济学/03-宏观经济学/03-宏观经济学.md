# 03-宏观经济学

> 总体经济的运行：增长、波动、就业、通胀——以及政府如何影响它们。

## 三大指标

| 指标 | 含义 | 关键 |
|------|------|------|
| **GDP** | 一国在一年内生产的最终商品和服务的总价值 | 衡量经济规模 |
| **失业率** | 劳动力中正在找工作但找不到的人 | 自然失业率≠零 |
| **通胀** | 物价总水平持续上升 | 温和通胀(2%)是健康的 |

## 总需求-总供给 (AD-AS)

| | AD (总需求) | AS (总供给) |
|------|------|------|
| 组成 | C+I+G+(X−M) | 劳动·资本·技术 |
| 扩张 | 货币/财政刺激 | 长期——技术进步 |

## 财政政策

| 工具 | 内容 |
|------|------|
| **政府支出** (G) | 基建·军费·转移支付——乘数效应 |
| **税收** (T) | 个人所得税·企业税·增值税 |
| **自动稳定器** | 失业救济(衰退时自动增加)——累进税 |
| **赤字与债务** | 短期可以赤字——长期债务需可控 |

## 货币政策

| 工具 | 说明 |
|------|------|
| **利率** | 央行降息→借贷便宜→消费投资↑→通胀↑ |
| **量化宽松** (QE) | 央行购买金融资产→增加流动性——2008后常用 |
| **通胀目标** | 大多数央行目标~2%——太低(通缩比通胀更可怕) |

## 经济周期

| 阶段 | 特征 |
|------|------|
| 扩张 | GDP↑·就业↑·通胀缓慢上升 |
| 顶峰 | 资源紧张·通胀压力 |
| 衰退 | GDP↓·失业↑ |
| 谷底 | 触底——开始复苏 |
