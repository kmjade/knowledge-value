# 05-发展经济学

> 为什么有些国家富有——有些极度贫困？这是经济学最重要——也是最难的问题。

## 增长引擎

| 引擎 | 内容 |
|------|------|
| 物质资本 | 机器·基础设施——「硬件」 |
| 人力资本 | 教育·健康——「软件」 |
| 技术 | 全要素生产率——同样投入——更多产出 |
| 制度 | 产权·法治·低腐败——增长的「地基」 |

## 贫困陷阱

| 太穷→无法储蓄→无法投资→持续穷 | 「贫穷的恶性循环」 |
|------|------|

## 援助有效吗？

| 正方 (Sachs) | 援助填补「贫困陷阱」——启动增长 |
|------|------|
| 反方 (Easterly) | 援助破坏本地制度——「白人的负担」——自下而上的市场更有效 |
