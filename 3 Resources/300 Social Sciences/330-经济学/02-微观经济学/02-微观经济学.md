# 02-微观经济学

## 供给与需求

| 定律 | 内容 |
|------|------|
| 需求定律 | 价格↑ → 需求量↓ |
| 供给定律 | 价格↑ → 供给量↑ |
| **均衡** | 供给 = 需求——市场出清 |

## 弹性

| 类型 | 公式 | 含义 |
|------|------|------|
| 需求价格弹性 | %ΔQ / %ΔP | >1 = 弹性(奢侈品)·<1 = 刚性(必需品) |
| 收入弹性 | %ΔQ / %Δ收入 | 奢侈品>1·必需品<1·劣等品<0 |

## 市场结构

| 结构 | 厂商数 | 产品 | 进入壁垒 | 例子 |
|------|:---:|------|:---:|------|
| 完全竞争 | 很多 | 同质 | 无 | 农产品 |
| 垄断竞争 | 很多 | 差异化 | 低 | 餐厅 |
| 寡头 | 少数 | 均可 | 高 | 电信·航空 |
| 垄断 | 一个 | 唯一 | 极高 | 公用事业 |
