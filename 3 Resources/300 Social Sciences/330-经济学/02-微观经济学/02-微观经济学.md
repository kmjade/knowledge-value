# 02-微观经济学

> 个体决策的加总如何形成市场结果——「看不见的手」的科学。

## 供需模型

| | 需求 (Demand) | 供给 (Supply) |
|------|------|------|
| 定律 | 价格↑ → 需求↓ | 价格↑ → 供给↑ |
| 移动因素 | 收入·偏好·替代品 | 技术·成本·预期 |
| 弹性 | 消费者对价格变化的敏感度 | 生产者对价格变化的敏感度 |

## 市场结构

| 结构 | 卖家数 | 产品 | 进入壁垒 | 例子 |
|------|:---:|------|:---:|------|
| 完全竞争 | 无数 | 同质 | 无 | 农产品 |
| 垄断竞争 | 很多 | 差异化 | 低 | 餐厅·服装 |
| 寡头 | 少数 | 同质/异质 | 高 | 电信·航空 |
| 垄断 | 1 | — | 极高 | 公用事业·专利药 |

## 市场失灵

| 类型 | 说明 | 对策 |
|------|------|------|
| **外部性** | 私人成本 ≠ 社会成本——污染 | 征税·配额·产权(科斯) |
| **公共品** | 非竞争+非排他——国防·灯塔 | 政府提供 |
| **信息不对称** | 柠檬市场——Akerlof 的二手车 | 信号·担保·监管 |
| **垄断** | 无竞争——高价——低效 | 反垄断法·价格管制 |

## 博弈论基础

| 概念 | 说明 |
|------|------|
| **囚徒困境** | 个体理性→集体非理性——双方都坦白 |
| **纳什均衡** | 给定对方的策略——我没有动机改变 |
| **重复博弈** | 如果明天还要见面——合作可能 |
| **可信承诺** | 口头威胁没用——需要实际行动 |
