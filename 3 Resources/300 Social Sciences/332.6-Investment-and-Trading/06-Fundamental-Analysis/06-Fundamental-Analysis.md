# 06 基本面分析 Fundamental Analysis

> 技術分析告訴你「何時」交易，基本面分析告訴你「為何」交易。

## 基本面分析的層次

```
全球宏觀
  └── 央行政策 (利率、QE/QT)
       └── 國家經濟 (GDP、就業、通膨)
            └── 行業趨勢
                 └── 公司基本面 (財報、估值)
```

## 宏觀經濟關鍵指標 Macro Indicators

### GDP (國內生產總值)

| 指標 | 說明 |
|------|------|
| 發布 | 每季（初值→修正值→終值） |
| 含義 | 經濟總產出，最全面的經濟衡量 |
| 市場反應 | 高於預期 → 本幣走強 |

### 通膨 Inflation

| 指標 | 重點 |
|------|------|
| **CPI** (消費者物價指數) | 核心 CPI 剔除食品能源，是央行最關注的 |
| **PCE** (個人消費支出) | Fed 首選通膨指標，目標 2% |
| PPI (生產者物價指數) | 領先指標，傳導到 CPI |

### 就業 Employment

| 指標 | 頻率 | 影響 |
|------|------|:---:|
| **非農就業 NFP** | 每月 | ⭐⭐⭐⭐⭐ |
| 失業率 | 每月 | ⭐⭐⭐⭐ |
| 平均時薪 | 每月 | ⭐⭐⭐（通膨壓力） |
| **初請失業金** | 每週 | ⭐⭐⭐ |

### 央行政策 Central Bank Policy

| 央行 | 貨幣 | 政策工具 |
|------|------|---------|
| **Fed** (美聯儲) | USD | 聯邦基金利率、QE/QT、點陣圖 |
| **ECB** (歐央行) | EUR | 再融資利率、APP/PEPP |
| **BOJ** (日銀) | JPY | 政策利率、YCC、QQE |
| **BOE** (英銀) | GBP | 銀行利率 |
| **PBOC** (人民銀行) | CNY | LPR、MLF、存款準備金率 |

### 央行政策對貨幣的影響

```
加息 → 本幣走強（利差擴大，資金流入）
降息 → 本幣走弱（利差縮小，資金流出）
鷹派言論 → 暗示加息 → 本幣走強
鴿派言論 → 暗示降息 → 本幣走弱
```

## PMI (採購經理人指數)

| PMI | 含義 |
|-----|------|
| > 50 | 經濟擴張 |
| < 50 | 經濟收縮 |
| 趨勢 | 連續上升/下降比絕對值更重要 |

- **ISM 製造業 PMI** (美國) — 每月第一個工作日
- **Markit PMI** — 初值 (Flash) 比終值更受關注

## 地緣政治與市場情緒

| 事件類型 | 典型反應 |
|---------|---------|
| 戰爭/衝突 | 避險貨幣走強 (USD/JPY/CHF)、股市跌 |
| 貿易戰/制裁 | 相關貨幣貶值 |
| 政治不穩定 | 資本外流 |
| 自然災害 | 短期衝擊，重建利好 |

## 新聞交易要點 News Trading Tips

### 數據發布時的典型波動

```
預期 vs 實際：
實際 >> 預期 → 市場暴漲（Surprise 效應）
實際 ≈ 預期 → 波動有限（已 Price-in）
實際 << 預期 → 市場暴跌
```

### 注意事項

1. ⚠️ **不要在數據發布瞬間下單** — 點差拉大、滑點嚴重
2. ⚠️ **等第一根 K 線收盤** — 給市場 5–15 分鐘消化
3. ⚠️ **不只數據重要，修正值也重要** — NFP 經常大幅修正上月數據
4. ⚠️ **央行講話比數據更重要** — Powell 一句話可以翻轉市場
