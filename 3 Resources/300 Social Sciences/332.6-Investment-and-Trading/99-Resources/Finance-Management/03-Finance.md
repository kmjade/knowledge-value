## 💵 預算管理

### 50/30/20 法則

```
稅後收入分配：
┌─────────────────────────────────┐
│  50% 必需支出                    │
│  (住房、食物、交通、水電)         │
├─────────────────────────────────┤
│  30% 儲蓄投資                    │
│  (緊急基金、退休、投資)           │
├─────────────────────────────────┤
│  20% 可選支出                    │
│  (娛樂、購物、旅行)               │
└─────────────────────────────────┘
```

## 💡 財務原則

### 核心原則

1. **儲蓄第一** - 收入 - 儲蓄 = 支出
2. **避免高息債務** - 信用卡利息會吞噬財富
3. **長期投資** - 讓複利為你工作
4. **分散風險** - 不把所有資金投入單一資產
5. **定期回顧** - 每月檢查財務健康狀況

### 財務自由公式

```
財務自由指數 = (被動收入 × 12) / 年支出

目標：被動收入 ≥ 年支出
```
