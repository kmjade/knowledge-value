# 投資與交易 — 常見問題 FAQ

## 新手入門

**Q: 需要多少資金才能開始交易？**

A: 外匯保證金交易最低可從 $100 開始（微型手 0.01）。但建議至少 $1,000–$5,000，才有足夠的風險緩衝。A 股需要 ¥1,000+。**永遠只用你輸得起的錢交易。**

**Q: Demo (模擬帳戶) 要練多久？**

A: 至少 2–4 週，直到你能連續 20 筆交易嚴格執行策略（不一定是盈利的）。如果模擬都做不到紀律，實盤只會更糟。

**Q: 哪個貨幣對最適合新手？**

A: **EURUSD** — 點差最低、流動性最好、技術分析最有效。避免 GBPJPY（波動過大）和異國貨幣對（流動性差）。

## 技術分析

**Q: 最好的技術指標是什麼？**

A: 沒有「最好」的指標，只有最適合你的。入門推薦組合：**MA (50/200) + RSI (14) + 支撐阻力位**。少即是多——超過 3 個指標只會讓你猶豫不決。

**Q: 哪個時間框架最好？**

A: 取決於你的風格：
- Scalping → M1–M5
- 日內交易 → M15–H1
- 波段交易 → H4–D1
- 頭寸交易 → D1–W1

**新手建議從 H1 或 H4 開始**——不會太快也不會太慢。

## 風險管理

**Q: 止損設多遠？**

A: 沒有固定數字。用 ATR 或結構止損：
- ATR 法：止損 = 1.5–2 × ATR(14)
- 結構法：放在最近的高低點之外 + 5–10 點緩衝

**Q: 應該同時持倉多少個？**

A: 新手 ≤ 2 個，經驗者 ≤ 5 個。更重要的是**總風險敞口**——所有倉位合計風險不應超過帳戶的 5%。

## EA / 量化交易

**Q: 買 EA 能賺錢嗎？**

A: 大多數市售 EA 是 Martingale 或 Grid 策略——短期盈利，長期爆倉。**沒有「聖杯」EA**。如果你不懂 EA 的原理，就不要用它。

**Q: MQL4 還是 MQL5？**

A: 純外匯 → MQL4（經紀商支援更廣）。未來可能做多市場或需要更強回測 → MQL5。

## 心理與紀律

**Q: 連續虧損怎麼辦？**

A: 
1. **立即停止交易**（當天不再做）
2. 檢查交易日誌：是按計劃虧損（OK）還是情緒化交易（問題）？
3. 如果是按計劃虧損 → 正常波動，繼續執行
4. 連續虧損 5+ 筆 → 暫停一週，復盤策略

**Q: 如何克服報復性交易？**

A: 制定硬性規則：「連續虧損 3 筆 → 今日停止」。把規則寫下來貼在螢幕上。必要時設定 MT4/MT5 的每日最大虧損限制。

## 實務

**Q: 全職交易可行嗎？**

A: 可行但極難。你需要：
- 至少 6–12 個月穩定盈利記錄
- 足夠的生活費儲備（至少 1 年）
- 穩定的交易系統（Sharpe > 1.0）
- 強大的心理素質

> 💡 建議先兼職交易，直到交易收入穩定超過正職收入後再考慮全職。

**Q: 如何選擇經紀商？**

A: 優先看監管（FCA/ASIC > CySEC > 離岸）。其次看點差、出金速度。**永遠先用最小金額測試出金流程**。不要把所有資金放在一個經紀商。
