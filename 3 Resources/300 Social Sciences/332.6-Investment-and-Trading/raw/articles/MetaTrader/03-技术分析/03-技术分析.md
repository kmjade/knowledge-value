# 03 - MetaTrader 技术分析

## 图表类型

| 类型 | 快捷键 | 适合 |
|------|--------|------|
| **K 线图** | Alt+1 | 最常用 |
| **柱状图** | Alt+2 | 类似 K 线 |
| **折线图** | Alt+3 | 趋势判断 |

## 核心内置指标

### 趋势指标

| 指标 | 用法 |
|------|------|
| **MA（移动平均线）** | 判断趋势方向 |
| **Bollinger Bands** | 波动区间、突破信号 |
| **Parabolic SAR** | 趋势反转信号 |
| **Ichimoku** | 综合趋势 + 支撑阻力 |

### 震荡指标

| 指标 | 用法 |
|------|------|
| **RSI** | 超买 > 70，超卖 < 30 |
| **MACD** | 金叉/死叉信号 |
| **Stochastic** | 超买超卖区域 |
| **CCI** | 极端行情识别 |

### 成交量指标

| 指标 | 用法 |
|------|------|
| **Volume** | 成交量柱状图 |
| **OBV** | 量价关系 |

## 绘图工具

```
插入菜单 / 左侧工具栏：

├── 水平线          ← 支撑/阻力位
├── 趋势线          ← 连接高低点
├── 通道            ← 平行通道
├── 斐波那契回调     ← 黄金分割回撤位
├── 斐波那契扩展     ← 目标位预测
├── 文字标签        ← 标注
└── 矩形/椭圆       ← 标记区间
```

## 模板（Template）

```
常用指标组合 → 保存为模板 → 一键应用

右键图表 → 模板 → 保存模板
右键图表 → 模板 → 加载模板
```

### 推荐模板

```
短线交易模板：
  EMA 20 + EMA 50 + RSI(14) + Stochastic(5,3,3)

中线交易模板：
  SMA 50 + SMA 200 + MACD(12,26,9) + RSI(14)

趋势交易模板：
  EMA 20 + EMA 50 + EMA 200 + Ichimoku + ADX(14)
```

## 多时间周期分析

```
自上而下：

月线 → 大趋势方向
  ↓
周线 → 中期趋势
  ↓
日线 → 入场区域
  ↓
4 小时 → 精确入场
  ↓
1 小时 → 确认信号
```
