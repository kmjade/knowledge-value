# Alexander Elder — 亚历山大·埃尔德

> 专业交易员、精神病学家。《交易为生》作者，三位一体交易体系的创立者。

## 基本信息

| 维度 | 内容 |
|------|------|
| 出生 | 俄罗斯 (苏联) |
| 职业 | 精神病学家 → 专业交易员 |
| 代表作 | 《交易为生》(Trading for a Living, 1993) |
| 核心贡献 | 交易心理 + 技术分析 + 风险管理 三位一体 |
| 网站 | elder.com |

## 核心思想: 三位一体

```
  交易成功
  ┌───┼───┐
  心  方  资
  理  法  金
```

1. **心理 (Psychology)** — 交易 80% 是心理
2. **方法 (Method)** — 技术分析系统（三重滤网）
3. **资金 (Money)** — 风险管理规则（2% + 6%）

## 三重滤网系统 (Triple Screen)

| 滤网 | 时间框架 | 功能 |
|------|----------|------|
| 第一层 | 长期 (周线) | 识别趋势方向 |
| 第二层 | 中期 (日线) | 在趋势中寻找回调 |
| 第三层 | 短期 (小时线) | 精确入场时机 |

## 关键规则

- **2% 规则**: 单笔风险 ≤ 2%
- **6% 规则**: 月度风险 ≤ 6%
- **不要亏损加仓**
- **记录交易日志**
