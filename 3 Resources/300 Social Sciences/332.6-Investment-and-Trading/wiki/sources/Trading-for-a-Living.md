# 《交易为生》Trading for a Living

> 交易心理 + 技术分析 + 风险管理三位一体的经典之作。

## 核心框架

```
第一部分: 交易心理 (Psychology)
  - 个人心理 vs 群体心理
  - AA 原则（承认亏损上瘾）
  
第二部分: 技术分析 (Method)
  - 三重滤网系统 (Triple Screen)
  - Elder-ray 指标
  - 强力指数 (Force Index)
  
第三部分: 风险管理 (Money)
  - 2% 规则
  - 6% 规则
  - 出场策略
```
