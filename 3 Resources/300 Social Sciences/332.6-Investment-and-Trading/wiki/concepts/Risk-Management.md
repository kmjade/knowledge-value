# Risk Management (风险管理)

## 定义

风险管理是交易中最核心的生存法则：**先想亏损，再想盈利**。通过仓位计算、止损策略和资金管理，控制每笔交易的风险敞口，确保不会因单次亏损而爆仓。

## 核心原理

### 固定比例法

```
每笔交易风险 = 总资金的 1-2%

例如：账户 $10,000
  每笔最大亏损 = $10,000 × 2% = $200

止损 20 点 → 手数 = 200 / (20 × 10) = 1 手
止损 50 点 → 手数 = 200 / (50 × 10) = 0.4 手
```

### 仓位计算公式

```
手数 = (本金 × 风险%) / (止损点数 × 每点价值)
```

### 风险三层结构

```
第一层：仓位控制
  ├── 单笔风险 ≤ 总资金 1-2%
  └── 总持仓风险 ≤ 总资金 5-6%

第二层：止损纪律
  ├── 每笔必须有止损
  ├── 止损基于技术位（支撑/阻力/ATR）
  └── 移动止损保护利润

第三层：心理纪律
  ├── 连续亏损 → 减仓或暂停
  ├── 报复性交易 → 禁止
  └── 不追单
```

## 关键要点

1. **2% 规则** — 单笔亏损不超过总资金 2%
2. **止损必要** — 永远不要不带止损交易
3. **盈亏比 > 1.5** — 平均盈利至少是平均亏损的 1.5 倍
4. **最大回撤 < 30%** — 超过 30% 回撤需要战略调整
5. **胜率不重要** — 30% 胜率 + 3:1 盈亏比 = 盈利

### 止损策略

| 策略 | 方法 |
|------|------|
| 技术位止损 | 支撑/阻力位外侧 |
| ATR 止损 | ATR × 2 距离 |
| 移动止损 | 价格有利方向移动 X 点后跟随 |
| 时间止损 | N 根 K 线未达预期则退出 |
