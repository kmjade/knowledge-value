# Backtesting (策略回测)

## 定义

策略回测是用历史市场数据模拟交易策略执行的过程，评估策略在过去的表现。它是实盘交易前的**必经步骤**，但回测表现 ≠ 未来收益。

## 核心原理

### 回测流程

```
1. 选择策略    → 定义入场/出场/风控规则
2. 设置参数    → 品种、周期、时间范围、初始资金
3. 运行回测    → 历史数据模拟每笔交易
4. 分析报告    → 净利润、最大回撤、胜率、盈亏比
5. 优化参数    → 调优但避免过拟合
6. 验证稳健性  → 不同品种、不同时段交叉验证
```

### MetaTrader 回测入口

```
MT4: 工具栏 → 策略测试（Ctrl+R）
MT5: 显示 → 策略测试（Ctrl+R）
```

## 关键要点

### 核心指标解读

| 指标 | 含义 | 好标准 |
|------|------|--------|
| **净利润** | 总盈利 - 总亏损 | > 0 |
| **最大回撤** | 最大连续亏损幅度 | < 30% |
| **胜率** | 盈利交易 / 总交易 | 不重要（< 50% 正常） |
| **盈亏比** | 平均盈利 / 平均亏损 | > 1.5 |
| **夏普比率** | 风险调整收益 | > 1 |
| **交易次数** | 样本量 | > 100 才有统计意义 |
| **利润因子** | 总盈利 / 总亏损 | > 1.5 |

### 常见陷阱

```
❌ 回测曲线 45° 上涨无回撤 → 99% 作弊（未来函数/重绘）
❌ 胜率 90%+ → 大概率马丁/网格，终将爆仓
❌ 只测了 3 个月 → 样本不足，无统计意义
❌ 回测完美实盘亏 → 过拟合 (Curve Fitting)
❌ 不计算滑点/手续费 → 过度乐观
❌ MT4 单线程 → 速度慢但结果可能更真实
```

### 避免过拟合

```
每增加一个优化参数 → 自由度 +1 → 过拟合风险 ↑

 简单策略（2-3 参数）→ 样本外测试 → 更可能实盘盈利
 复杂策略（10+ 参数）→ 回测完美 → 实盘大概率亏
```
