# 凱利公式 Kelly Criterion

## 定義
凱利公式計算在已知勝率和盈虧比的前提下，使長期資本增長率最大化的最優倉位比例。

## 公式

```
f* = (bp − q) / b

f* = 最優倉位比例
b  = 盈虧比 (淨盈利 / 淨虧損)
p  = 勝率
q  = 1 − p (虧損率)
```

## 計算示例

```
勝率 p = 45%, 盈虧比 b = 2:1
f* = (2 × 0.45 − 0.55) / 2 = 0.175

理論最優倉位 = 17.5%

⚠️ 實際使用半凱利：
實際倉位 = ½ × 17.5% = 8.75%
```

## 凱利 vs 固定比例

| 策略 | 勝率45%, 盈虧比2:1 |
|------|:--:|
| 全倉 (100%) | 終將爆倉 |
| 凱利 (17.5%) | 長期最大增長 |
| 半凱利 (8.75%) | 穩健增長（推薦） |
| 固定 2% | 保守但安全 |

## 注意事項

> ⚠️ 凱利公式假設你知道真實的勝率和盈虧比——在實際交易中這些是估計值。使用半凱利或更保守的比例。
