# Economic Calendar (经济日历)

## 定义

经济日历是列出即将发布的经济数据、央行决议和重要事件的日程表，是基本面交易者最重要的信息工具。高影响力事件（如利率决议、非农就业、CPI）通常引发市场剧烈波动。

## 核心原理

### 事件影响力分级

```
🔴 高影响 → 央行决议、非农、CPI、GDP、PMI
🟡 中影响 → 零售销售、消费者信心、库存
⚪ 低影响 → 常规讲话、次要数据

波动率规则: 高影响事件 ±30 分钟内避免开仓
```

### 日历使用策略

```
事件前 (T-2h):
  → 了解预期值 vs 前值
  → 提前设好止损（可能扩大的波动）

事件发布 (T=0):
  → 关注实际值 vs 预期值的偏差
  → 偏差越大，波动越大

事件后 (T+30min):
  → 等市场消化价格
  → 顺趋势方向入场
```

## 关键要点

1. **实际 vs 预期** — 偏差是市场反应的驱动因素
2. **央行决议 > 经济数据** — 利率决议影响最大
3. **提前减仓** — 高影响事件前降杠杆
4. **避开新闻交易** — 新手不建议赌数据
