## 🏓 ping

### 功能

測試網路连通性和延遲。

### 基本用法

```bash
ping 目標主机
```

### 常用参数

| 参数 | 說明 |
|------|------|
| -c | 發送包的数量 |
| -i | 發送间隔（秒） |
| -s | 包大小 |
| -t | 持續ping（Windows） |
| -4/-6 | 强制使用IPv4/IPv6 |

### 使用示例

```bash
# 基本測試
ping www.baidu.com

# 發送4个包
ping -c 4 www.baidu.com

# 指定包大小
ping -s 1024 www.baidu.com

# Windows持續ping
ping -t www.baidu.com
```

### 輸出解析

```
64 bytes from 110.242.68.66: icmp_seq=1 ttl=54 time=12.3 ms
│     │    │                 │      │      │    │
│     │    │                 │      │      │    └─ 響應時間（延遲）
│     │    │                 │      │      └───── TTL（生存時間）
│     │    │                 │      └────────── 序列号
│     │    │                 └───────────────── 收到的字节数
│     │    └────────────────────────────────── 源IP地址
│     └──────────────────────────────────────── ICMP回覆
└──────────────────────────────────────────── 數據包大小
```

### 故障判斷

| 现象 | 可能原因 |
|------|----------|
| `Request timeout` | 目標不可达或防火墙阻止 |
| `Destination host unreachable` | 網路不可达 |
| `TTL expired` | 路由循环 |
| 延遲很高 | 網路拥塞或路徑过长 |

## 🗺️ traceroute / tracert

### 功能

# 顯示

### 基本用法

```bash
# Linux/Mac
traceroute 目標主机

# Windows
tracert 目標主机
```

### 使用示例

```bash
traceroute www.google.com
tracert www.google.com
```

### 輸出示例

```
traceroute to www.google.com (142.250.185.36), 30 hops max, 60 byte packets
 1  192.168.1.1 (192.168.1.1)  1.234 ms  1.112 ms  1.089 ms
 2  10.0.0.1 (10.0.0.1)  12.345 ms  12.234 ms  12.123 ms
 3  202.96.128.86 (202.96.128.86)  15.678 ms  15.567 ms  15.456 ms
...
```

### 工作原理

1. 發送TTL=1的包，第一跳路由器返回超时
2. 發送TTL=2的包，第二跳路由器返回超时
3. 逐步增加TTL，直到到达目的地

## 🔍 nslookup / dig

### 功能

DNS查詢工具，用于網網域名稱稱解析。

### nslookup（Windows通用）

```bash
# 基本查詢
nslookup www.baidu.com

# 查詢特定类型記錄
nslookup -type=MX baidu.com  # 電子郵件交换記錄
nslookup -type=NS baidu.com  # 名称伺服器記錄

# 指定DNS伺服器
nslookup www.baidu.com 8.8.8.8
```

### dig（Linux/Mac，功能更强）

```bash
# 基本查詢
dig www.baidu.com

# 简洁輸出
dig +short www.baidu.com

# 查詢特定类型
dig MX baidu.com
dig NS baidu.com
dig TXT baidu.com

# 追踪DNS解析過程
dig +trace www.baidu.com

# 反向查詢（IP → 網網域名稱稱）
dig -x 8.8.8.8
```

### 輸出解析


## 📊 netstat / ss

### 功能

# 顯示

### netstat（传统工具）

```bash
# 顯示
netstat -t

# 顯示
netstat -u

# 顯示
netstat -l

# 顯示
netstat -n

# 顯示
netstat -p

# 组合使用
netstat -tunlp  # 常用组合
```

### ss（现代工具，更快速）

```bash
# 顯示
ss -t

# 顯示
ss -u

# 顯示
ss -l

# 顯示
ss -p

# 顯示
ss -n

# 组合使用
ss -tunlp  # 常用组合
```

### 常用示例

```bash
# 尋找占用80端口的进程
netstat -tunlp | grep :80
ss -tunlp | grep :80

# 查看
netstat -tn
ss -tn

# 統計連接数
netstat -an | awk '/^tcp/ {print $6}' | sort | uniq -c
```

## 🌐 ifconfig / ip

### 功能

# 查看

### ifconfig（传统工具，Linux）

```bash
# 查看
ifconfig

# 查看
ifconfig eth0

# 配置
sudo ifconfig eth0 192.168.1.100 netmask 255.255.255.0

# 启用/禁用接口
sudo ifconfig eth0 up
sudo ifconfig eth0 down
```

### ip（现代工具，Linux）

```bash
# 查看
ip addr show

# 查看
ip addr show eth0

# 配置
sudo ip addr add 192.168.1.100/24 dev eth0

# 刪除IP地址
sudo ip addr del 192.168.1.100/24 dev eth0

# 查看
ip route show

# 新增路由
sudo ip route add default via 192.168.1.1
```

### ipconfig（Windows）

```bash
# 查看
ipconfig

# 查看
ipconfig /all

# 刷新DNS缓存
ipconfig /flushdns

# 释放IP地址
ipconfig /release

# 重新获取IP地址
ipconfig /renew

# 顯示
route print
```

## 🔗 arp

### 功能

# 管理

```bash
# 查看
arp -a

# 查看
arp -a 192.168.1.1

# 新增静态ARP条目
sudo arp -s 192.168.1.1 aa:bb:cc:dd:ee:ff

# 清空ARP缓存
sudo arp -d -a
```

## 🛣️ route

### 功能

# 查看

### Linux

```bash
# 查看
route -n

# 新增默认网关
sudo route add default gw 192.168.1.1

# 新增特定路由
sudo route add -net 192.168.2.0/24 gw 192.168.1.2

# 刪除路由
sudo route del default
```

### Windows

```bash
# 查看
route print

# 新增路由
route add 192.168.2.0 mask 255.255.255.0 192.168.1.2

# 刪除路由
route delete 192.168.2.0

# 新增永久路由
route -p add 192.168.2.0 mask 255.255.255.0 192.168.1.2
```

## 🎯 故障排查流程

```
1. 檢查本地連接
   ifconfig/ipconfig
   檢查IP是否正确分配

2. 測試网关连通性
   ping 网关IP
   ping 192.168.1.1

3. 測試DNS解析
   nslookup www.baidu.com

4. 測試外网连通性
   ping 8.8.8.8
   ping www.baidu.com

5. 追踪路由
   traceroute www.baidu.com

6. 檢查端口和服务
   netstat/ss

7. 檢查防火墙规则
   iptables -L (Linux)
   netsh advfirewall (Windows)
```
