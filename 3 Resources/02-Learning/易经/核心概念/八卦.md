# 八卦 (Eight Trigrams)

> "八卦成列，象在其中矣" - 《系辞传》

## 📖 概念定义

八卦是由三个阴阳爻组成的八种基本符号，是易经的基础构成单元。每卦代表自然界的基本现象和人类生活的基本方面。

---

## 🔢 八卦总表

### ☰ 乾卦 (天)
- **符号**：☰ (☊☊☊)
- **自然**：天
- **属性**：健、刚、动
- **方位**：西北
- **季节**：秋冬之交
- **家庭**：父
- **身体**：头

### ☷ 坤卦 (地)
- **符号**：☷ (☋☋☋)
- **自然**：地
- **属性**：顺、柔、静
- **方位**：西南
- **季节**：夏秋之交
- **家庭**：母
- **身体**：腹

### ☳ 震卦 (雷)
- **符号**：☳ (☋☋☊)
- **自然**：雷
- **属性**：动、起、奋
- **方位**：东
- **季节**：春
- **家庭**：长男
- **身体**：足

## 🔗 与六十四卦的关系

### 📐 构成原理
- **上卦+下卦**：两个八卦重叠形成六十四卦
- **示例**：乾(☰) + 坤(☷) = 泰卦(䷊)
- **数学原理**：8 × 8 = 64卦

### 🎯 常见组合

## 📊 记忆方法

### 🎨 口诀记忆
**八卦取象歌**：
- 乾三连，坤六断
- 震仰盂，艮覆碗
- 离中虚，坎中满
- 兑上缺，巽下断

### 🖼️ 形象记忆
- **乾**：三条连续线，像天一样完整
- **坤**：三条断线，像大地分段
- **震**：下阳上阴，像雷从地起
- **艮**：上阳下阴，像山止于上

### 🏠 家庭记忆
将八卦与家庭成员对应，便于记忆：
```
父(乾) 母(坤)
长男(震) 长女(巽)
中男(坎) 中女(离)
少男(艮) 少女(兑)
```
