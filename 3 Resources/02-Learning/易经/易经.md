# 知識庫

> "易有太极，是生两仪。两仪生四象，四象生八卦。" - 《系辞传》

## 🎯 概览

易经，又称周易，是中国古代占卜和哲学经典，包含六十四卦及其爻辞，为中华文化提供了深刻的生活智慧和實踐指导。

---

## 📜 歷史背景与發展

### 🏛️ 起源传说

**三圣作易**的传说：
- **伏羲氏**：仰观天文，俯察地理，创作八卦（先天八卦）
- **周文王**：在羑里演六十四卦，系卦辞爻辞（后天八卦）
- **孔子**：作《十翼》（即《易传》），系統化周易哲学

### 📚 文獻形成

#### 甲骨文与金文时期
- 占卜甲骨上的数字卦象
- 商周青铜器上的卦象符号
- 最早的阴阳爻雏形

#### 周代成书
- **《周易》**：西周至春秋时期成书
- **《易传》**：战国至汉代成书
- 合称《易经》，成为儒家五经之首

#### 历代發展
- **汉代**：孟喜、京房开创象数易学
- **魏晋**：王弼扫除象数，开创义理易学
- **宋代**：程颐、朱熹融合象数义理

# 方法

#### 1. 蓍草占法（大衍筮法）
**步骤**：
1. 取50根蓍草，用去1根（太极）
2. 将49根随意分成两堆（两仪）
3. 从右手堆取1根（人）
4. 左右堆分别四四数之（四象）
5. 余数置一旁，重复三次得一爻
6. 如此六次得一卦

**特點**：复杂但最传统，象征意义豐富

#### 2. 铜钱占法（金钱卦）
**步骤**：
1. 准备三枚铜钱
2. 双手握铜钱，心中默念問題
3. 6次投掷，記錄每次的結果
4. 根据正反面组合得出卦象

**解读**：
- 3字（正面）= 老阳（变爻，记为O）
- 3背（反面）= 老阴（变爻，记为X）
- 1字2背 = 少阳（不变，记为—）
- 2字1背 = 少阴（不变，记为- -）

#### 3. 時間起卦法
# 方法
- 年月日时数字相加得出上卦
- 年月日时数字相加除以8余数为下卦
- 总和除以6余数为动爻

**特點**：简单快捷，适合日常使用

# 方法

#### 1. 数字法
- 随机抽取两个数字（1-8）
- 分别对应上下卦
- 可加第三个数字定动爻

#### 2. 物象法
- 观察周围自然现象
- 根据现象特征对应卦象

#### 3. 心灵感应法

### 🔍 解卦要點

1. **卦辞**：整體形势和基本含義

## ❓ 常见問題 FAQ

### 基礎問題

**Q: 學習易经需要什么基礎？**
# 方法

**Q: 易经是迷信吗？**
A: 易经不是迷信，而是一部包含深刻哲学思想和生活智慧的典籍。占卜只是易经的一个應用程式方面，更重要的是理解其背后的哲学思想和人生智慧。

**Q: 學習易经需要多长時間？**
# 方法

**Q: 需要學習哪些经典文獻？**
A: 建議先读现代解读版（如曾仕强、南怀瑾等），再读经典注解（王弼、朱熹等），最后研读原著《周易》和《易传》。

# 方法

**Q: 如何记忆六十四卦？**
A: 采用分组记忆法，按八卦分组每组8卦；制作记忆卡片，使用间隔重复軟體；學習卦象关系（错卦、综卦、互卦）；结合生活实际理解。

**Q: 占卜結果不准怎么办？**
A: 占卜不是预测未来，而是提供決策參考。重点不在准不准，而在于理解卦象背后的智慧，学会从多角度思考問題。

**Q: 如何驗證學習效果？**
# 分享

**Q: 有必要學習繁体字版吗？**
A: 如果目標是研究经典，建議學習繁体字版。如果只是應用程式和了解，简体字现代解读版就足夠了。

### 深度問題

**Q: 易经的哲学体系是什么？**
A: 易经的核心哲学包括：天人合一的宇宙观、阴阳对立統一的辩证思维、中正之道的價值取向、变易永恒的發展观念。

**Q: 如何理解卦象的真实含義？**
A: 卦象是象征符号，需要透过现象看本质。不要拘泥于字面意思，要理解其背后的哲学思想和人生智慧。多读多思考，结合实际体会。

**Q: 易经与中医有什么关系？**
A: 易经是中医的理论基礎，阴阳五行、整體观念、辨证施治等中医核心思想都来源于易经。學習易经有助于深入理解中医。
