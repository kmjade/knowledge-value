## 學習方法

### 高效學習框架

#### 1. PARA 系統

```
Projects  - 需要行動的短期任務
Areas    - 長期責任和領域
Resources - 感興趣的主題和參考資料
Archive   - 已完成或不再需要的項目
```

**核心原則**:
- 按行動價值分類
- 定期檢視和歸檔
- 保持系統簡單
- 養成日常習慣

---

#### 2. 間隔重複（閃念學習）

**複習間隔**:
```
第1次：立即複習
第2次：1天後
第3次：3天後
第4次：1週後
第5次：2週後
第6次：1個月後
第7次：3個月後
```

**記憶技巧**:
1. 聯想記憶法：問題 + 圖像 = 記憶
2. 記憶宮殿法：將知識點放在熟悉空間
3. 故事記憶法：將知識編織成故事

---

#### 3. 費曼學習法

**步驟**:
1. **選擇概念** - 選擇要學習的主題
2. **教學講解** - 用簡單語言解釋給他人
3. **識別空白** - 發現理解不足的地方
4. **重新學習** - 針對空白再次學習
5. **簡化總結** - 用更簡單的方式重新組織

---

#### 4. PDCA 循環

```
Plan（計劃）→ Do（執行）→ Check（檢查）→ Act（改進）
    ↑                                           ↓
    └──────────────────────────────────────────────┘
```

## 學習路徑

### 路徑一：計算機網絡基礎

**目標人群**: 軟件開發者、DevOps 工程師、IT 從業者
**時間安排**: 2-3個月
**重點**: 理解網絡原理，掌握實踐技能

**學習步驟**:

1. **第1-2週**: 基礎知識
   - [ ] 學習 OSI 七層模型
   - [ ] 理解 TCP/IP 協議棧
   - [ ] 掌握網絡拓撲結構
   - [ ] 學習 IP 地址與子網劃分

2. **第3-4週**: 核心協議
   - [ ] 深入 HTTP 協議
   - [ ] 理解 TCP/UDP 協議機制
   - [ ] 學習 DNS 工作原理
   - [ ] 了解應用層協議

3. **第5-6週**: 網絡架構
   - [ ] 學習局域網設計
   - [ ] 了解廣域網架構
   - [ ] 探索無線網絡技術
   - [ ] 學習雲端網絡基礎

4. **第7-8週**: 安全與管理
   - [ ] 掌握網絡安全基礎
   - [ ] 學習網絡監控工具
   - [ ] 掌握故障排查方法
   - [ ] 實踐網絡診斷工具

**推薦資源**:
- [[3 Resources/02-Learning/计算机网络/README.md]]
