# 五行 (Five Elements)

> "五行者，金木水火土也，生成万物。" - 《尚书》

## 📖 概念定义

五行是中国古代哲学的重要概念，指金、木、水、火、土五种基本物质及其运动变化规律。五行学说解释了宇宙万物之间的相互关系和变化规律，与阴阳学说共同构成了中国传统哲学的基础。

## 🔢 五行总览

### 🪵 木 (Wood)
- **特性**：生长、升发、条达、舒畅
- **季节**：春季
- **方位**：东方
- **颜色**：青色、绿色
- **气候**：风
- **脏腑**：肝、胆
- **情绪**：怒
- **八卦对应**：震(☳)、巽(☴)

### 🔥 火 (Fire)
- **特性**：温热、上炎、光明、变化
- **季节**：夏季
- **方位**：南方
- **颜色**：红色
- **气候**：暑
- **脏腑**：心、小肠
- **情绪**：喜
- **八卦对应**：离(☲)

### 🪨 土 (Earth)
- **特性**：承载、生化、包容、稳定
- **季节**：长夏(季夏)
- **方位**：中央
- **颜色**：黄色
- **气候**：湿
- **脏腑**：脾、胃
- **情绪**：思
- **八卦对应**：坤(☷)、艮(☱)

### 💎 金 (Metal)
- **特性**：收敛、肃杀、清洁、坚硬
- **季节**：秋季
- **方位**：西方
- **颜色**：白色
- **气候**：燥
- **脏腑**：肺、大肠
- **情绪**：悲
- **八卦对应**：兑(☑)、乾(☰)

### 💧 水 (Water)
- **特性**：滋润、下行、寒凉、闭藏
- **季节**：冬季
- **方位**：北方
- **颜色**：黑色、蓝色
- **气候**：寒
- **脏腑**：肾、膀胱
- **情绪**：恐
- **八卦对应**：坎(☵)

## 🔄 五行关系

### 🔄 相生关系 (生成关系)
```
木生火 → 火生土 → 土生金 → 金生水 → 水生木
```

**解释**：
- **木生火**：木可以燃烧产生火
- **火生土**：燃烧后形成灰烬(土)
- **土生金**：土中孕育金属矿物
- **金生水**：金属表面凝结水珠
- **水生木**：水滋润树木生长

### ⚔️ 相克关系 (克制关系)
```
木克土 → 土克水 → 水克火 → 火克金 → 金克木
```

**解释**：
- **木克土**：树木根系穿透土壤
- **土克水**：土能阻挡和控制水流
- **水克火**：水能灭火
- **火克金**：火能熔化金属
- **金克木**：金属工具能砍伐树木
