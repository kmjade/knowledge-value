# 代数方程

> "代数不仅是关于符号的艺术，更是关于发现未知数的智慧。" - 佚名

## 📖 定义

**方程**：含有未知数的等式，表示两个表达式相等的数学陈述。

**解**：使等式成立的未知数的值。

**通式**：ax + b = c，其中 a、b、c 是已知数，x 是未知数

---

## 📐 方程类型

### 1. 一元一次方程
形式为 ax + b = 0 的方程

**求解步骤：**
```
1. 移项：ax = -b
2. 系数化为1：x = -b/a
```

**示例：**
```
解方程：2x + 6 = 14

2x + 6 = 14
2x = 14 - 6
2x = 8
x = 4
```

### 2. 一元二次方程
形式为 ax² + bx + c = 0 的方程

**求根公式：**
```
x = [-b ± √(b² - 4ac)] / 2a

其中：Δ = b² - 4ac 称为判别式
```

**判别式的意义：**
| Δ > 0 | Δ = 0 | Δ < 0 |
|-------|-------|-------|
| 两个不相等实根 | 两个相等实根 | 无实根（两个共轭复根） |

**示例：**
```
解方程：x² - 5x + 6 = 0

a = 1, b = -5, c = 6
Δ = (-5)² - 4×1×6 = 25 - 24 = 1 > 0

x = [5 ± √1] / 2
x₁ = (5 + 1)/2 = 3
x₂ = (5 - 1)/2 = 2
```

**因式分解法：**
```
x² - 5x + 6 = 0
(x - 2)(x - 3) = 0
∴ x = 2 或 x = 3
```

### 3. 一元高次方程
次数大于2的方程：axⁿ + bxⁿ⁻¹ + ... = 0

**常见类型：**
- **三次方程**：ax³ + bx² + cx + d = 0
- **四次方程**：ax⁴ + bx³ + cx² + dx + e = 0

### 4. 分式方程
分母含有未知数的方程

**求解步骤：**
1. 找出公分母
2. 去分母，化为整式方程
3. 求解整式方程
4. **检验**：排除使分母为零的解

**示例：**
```
解方程：1/(x-1) + 2/(x+1) = 3

公分母：(x-1)(x+1)

(x+1) + 2(x-1) = 3(x-1)(x+1)
x + 1 + 2x - 2 = 3(x² - 1)
3x - 1 = 3x² - 3
3x² - 3x - 2 = 0

x = [3 ± √(9 + 24)] / 6 = [3 ± √33] / 6

检验：x ≠ 1, x ≠ -1 ✓
```

## 🔑 核心公式

### 乘法公式
```
平方差公式：a² - b² = (a + b)(a - b)

完全平方公式：
(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²

完全立方公式：
(a + b)³ = a³ + 3a²b + 3ab² + b³
(a - b)³ = a³ - 3a²b + 3ab² - b³
```

### 韦达定理（二次方程）
对于方程 ax² + bx + c = 0 的两根 x₁, x₂：

```
x₁ + x₂ = -b/a    （根的和）
x₁ × x₂ = c/a     （根的积）
```

## 📊 方程分类汇总

| 类型 | 一般形式 | 求解方法 |
|------|----------|----------|
| 一元一次 | ax + b = 0 | 移项、化简 |
| 一元二次 | ax² + bx + c = 0 | 求根公式、因式分解 |
| 分式方程 | 含分母的方程 | 去分母、检验 |
| 根式方程 | 含根式的方程 | 两边平方、检验 |
| 绝对值方程 | \|x\| = a | 分类讨论 |
| 指数方程 | aˣ = b | 取对数 |
| 对数方程 | logₐx = b | 化为指数形式 |

## 🧮 常见因式分解

### 提公因式法
```
ax + ay = a(x + y)
3x² + 6x = 3x(x + 2)
```

### 分组分解法
```
ax + ay + bx + by = a(x + y) + b(x + y) = (a + b)(x + y)
x³ - x + x² - 1 = x(x² - 1) + (x² - 1) = (x + 1)(x² - 1)
```

### 特殊公式分解
```
x² - 9 = (x + 3)(x - 3)              (平方差)
x² + 4x + 4 = (x + 2)²               (完全平方)
x³ - 8 = (x - 2)(x² + 2x + 4)        (立方差)
```

## 🔍 应用场景

### 几何问题
```
矩形的周长是20cm，长比宽多2cm，求长和宽。

设宽为 x cm，则长为 (x + 2) cm

2(x + x + 2) = 20
2(2x + 2) = 20
4x + 4 = 20
4x = 16
x = 4

答：宽4cm，长6cm
```

### 行程问题
```
甲乙两地相距300km，甲车以60km/h的速度出发，
2小时后乙车以80km/h的速度从对面出发，
求两车何时相遇。

设乙车出发后 x 小时相遇

60(x + 2) + 80x = 300
60x + 120 + 80x = 300
140x = 180
x = 9/7 ≈ 1.29 小时

答：乙车出发后约1.29小时相遇
```

### 利润问题
```
商品进价200元，标价300元，打x折销售，
利润率为50%，求x。

售价 = 进价 × (1 + 利润率)
300 × (x/10) = 200 × 1.5
30x = 300
x = 10

答：不打折
```

## 💡 解题技巧

### 1. 移项要变号
```
左边 → 右边：+ → -，× → ÷
2x + 3 = 7 → 2x = 7 - 3 = 4
3x = 12 → x = 12 ÷ 3 = 4
```

### 2. 去分母要两边同乘
```
x/2 + x/3 = 10
6 × (x/2 + x/3) = 6 × 10
3x + 2x = 60
5x = 60
x = 12
```

### 3. 二次方程优先试因式分解
```
x² - 7x + 12 = 0
找两个数，积为12，和为7
3 和 4 满足
(x - 3)(x - 4) = 0
x = 3 或 x = 4
```

### 4. 分式方程、根式方程必须检验
检验是否有增根（使原方程无意义的解）
