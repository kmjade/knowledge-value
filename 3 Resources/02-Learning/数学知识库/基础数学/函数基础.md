# 函数基礎

> "函数是数学的血脉，贯穿于整个数学体系。" - 佚名

## 📖 定义

### 函数的定义
函数是一种特殊的对应关系，对于定义域内的每一个輸入值 x，都有唯一確定的輸出值 y 与之对应。

记作：**y = f(x)**

其中：
- f：函数名
- x：自变量（輸入）
- y：因变量（輸出）

### 函数的三要素
| 要素 | 含義 | 示例 |
|------|------|------|
| 定义域 | 自变量 x 的取值范围 | y = √x，定义域 x ≥ 0 |
| 对应法则 | x 与 y 的对应规则 | y = 2x + 1 中"2x+1" |
| 值域 | 函数值的集合 | y = 2x + 1 (x∈ℝ)，值域 ℝ |

---

# 方法

### 1. 解析式法
用数学公式表示函数关系。

```
例1：y = 2x + 1（一次函数）
例2：y = x²（二次函数）
例3：y = 2ˣ（指数函数）
例4：y = log₂x（对数函数）
```

### 2. 表格法
用表格列出 x 与 y 的对应值。

| x | 0 | 1 | 2 | 3 | 4 |
|---|---|---|---|---|---|
| y = 2x + 1 | 1 | 3 | 5 | 7 | 9 |

### 3. 图像法
用坐标系中的曲线表示函数关系。

### 4. 箭头图法
用箭头表示元素的对应关系（离散函数）

```
A = {1, 2, 3}, B = {2, 4, 6}
f: A → B, f(x) = 2x

1 → 2
2 → 4
3 → 6
```

## 🔑 基本初等函数

### 1. 一次函数（线性函数）
**形式**：y = kx + b（k ≠ 0）

| 参数 | 含義 | 几何意义 |
|------|------|----------|
| k | 斜率 | 直线的倾斜程度 |
| b | 截距 | 直线与y轴交点的纵坐标 |

**特殊情形：**
- 正比例函数：y = kx（b = 0）
- 常函数：y = b（k = 0）

**示例图像特征：**
```
k > 0：上升直线
k < 0：下降直线
k = 0：水平直线
```

### 2. 二次函数
**形式**：y = ax² + bx + c（a ≠ 0）

**顶点式**：y = a(x - h)² + k
- 顶点坐标：(h, k)
- 对称轴：x = h

**交点式**：y = a(x - x₁)(x - x₂)
- x₁, x₂：与x轴的交点横坐标

**a 的作用：**
```
a > 0：开口向上，有最小值
a < 0：开口向下，有最大值
|a| 越大：开口越小
```

### 3. 反比例函数
**形式**：y = k/x（k ≠ 0）

**图像特征：**
```
k > 0：第一、三象限的双曲线
k < 0：第二、四象限的双曲线
```

### 4. 指数函数
**形式**：y = aˣ（a > 0, a ≠ 1）

**性质：**
| a > 1 | 0 < a < 1 |
|-------|-----------|
| 单调递增 | 单调递减 |
| 过点(0,1) | 过点(0,1) |
| y轴是下渐近线 | y轴是上渐近线 |

### 5. 对数函数
**形式**：y = logₐx（a > 0, a ≠ 1）

**性质：**
| a > 1 | 0 < a < 1 |
|-------|-----------|
| 单调递增 | 单调递减 |
| 过点(1,0) | 过点(1,0) |
| x轴是右渐近线 | x轴是左渐近线 |

**对数恒等式：**
```
a^logₐx = x
logₐ(aˣ) = x
logₐ1 = 0
logₐa = 1
```

## 📊 函数性质

### 单调性
**定义**：在区间上，x₁ < x₂ 时 f(x₁) < f(x₂) 称为增函数，f(x₁) > f(x₂) 称为减函数。

# 方法
```
增函数：f'(x) > 0
减函数：f'(x) < 0
```

### 奇偶性
| 奇函数 | 偶函数 |
|--------|--------|
| f(-x) = -f(x) | f(-x) = f(x) |
| 关于原点对称 | 关于y轴对称 |
| 例：y = x³, y = sin x | 例：y = x², y = cos x |

### 周期性
**定义**：存在非零常数 T，使得 f(x + T) = f(x)，则称 f 是周期函数。

```
例：y = sin x，周期 T = 2π
    y = tan x，周期 T = π
```

### 有界性
**定义**：存在 M > 0，使得 |f(x)| ≤ M，则称 f 有界。

```
例：y = sin x，有界（|sin x| ≤ 1）
    y = x，無界
```

## 🔍 函数運算

### 函数的四则運算
```
(f ± g)(x) = f(x) ± g(x)
(f · g)(x) = f(x) · g(x)
(f / g)(x) = f(x) / g(x)，g(x) ≠ 0
```

### 复合函数
设 y = f(u)，u = g(x)，则 y = f[g(x)] 称为复合函数。

**示例：**
```
y = sin(2x + 1)
外层：y = sin u
内层：u = 2x + 1
```

### 反函数
若 y = f(x) 是一一对应函数，则其反函数记作 y = f⁻¹(x)

**求法：**
1. 从 y = f(x) 解出 x
2. 交换 x 和 y
3. 写出反函数解析式

**示例：**
```
求 y = 2x + 1 的反函数

x = (y - 1)/2
互换：y = (x - 1)/2

反函数：y = (x - 1)/2
```

## 📐 常见函数總結

| 函数 | 解析式 | 定义域 | 值域 | 单调性 |
|------|--------|--------|------|--------|
| 常函数 | y = c | ℝ | {c} | 不变 |
| 一次函数 | y = kx + b | ℝ | ℝ | k > 0 增，k < 0 减 |
| 二次函数 | y = ax² + bx + c | ℝ | 取决于 a | 不完全单调 |
| 反比例 | y = k/x | x ≠ 0 | y ≠ 0 | 分区间单调 |
| 指数函数 | y = aˣ | ℝ | (0, +∞) | 取决于 a |
| 对数函数 | y = logₐx | (0, +∞) | ℝ | 取决于 a |
| 正弦函数 | y = sin x | ℝ | [-1, 1] | 周期变化 |

## 📚 练习题

### 基礎题
1. 求函数 y = √(x - 2) 的定义域
2. 判斷 y = x³ + x 的奇偶性
3. 画出 y = 2x + 1 的图像
4. 已知 f(x) = x² + 1，求 f(2) 和 f(a+1)

### 进阶题
1. 已知 f(x) 是偶函数，f(1) = 3，求 f(-1)
2. 求函数 y = 2ˣ 的反函数
3. 判斷 y = x² - 2x 在区间 (1, +∞) 上的单调性
4. 已知 f(x + 1) = x² - 1，求 f(x)
