# 平面几何

> "几何是视觉化的数学，是理解空間的基礎。" - 欧几里得

## 📖 基本概念

### 点、线、面
| 元素 | 定义 | 表示 |
|------|------|------|
| 点 | 空間中的基本位置，無大小 | A, B, C... |
| 线 | 由無数点组成，無限延伸 | l, m, n... |
| 面 | 由無数线组成，二维平面 | α, β, γ... |

### 线的位置关系
| 关系 | 定义 | 表示 |
|------|------|------|
| 平行 | 永不相交的直线 | l ∥ m |
| 垂直 | 相交成90°的直线 | l ⊥ m |
| 斜交 | 相交但不成90° | l 与 m 相交 |
| 重合 | 完全相同的直线 | l 与 m 重合 |

---

## 📐 角的概念

### 角的定义
由两条射线共用一个端点形成的圖形。

### 角的分類

| 角的类型 | 范围 | 示意图 |
|----------|------|--------|
| 零角 | 0° | |
| 锐角 | 0° < α < 90° | |
| 直角 | α = 90° | ┐ |
| 钝角 | 90° < α < 180° | |
| 平角 | α = 180° | — |
| 周角 | α = 360° | ○ |

### 特殊角关系

**互补角**：
```
两个角的和为90°
若 ∠A + ∠B = 90°，则 ∠A 与 ∠B 互补
```

**互余角**：
```
两个角的和为180°
若 ∠A + ∠B = 180°，则 ∠A 与 ∠B 互余
```

**对顶角**：
```
两条直线相交形成的相对的两个角相等
```

**同位角、内错角、同旁内角**：
```
被两条平行线所截时：
- 同位角相等
- 内错角相等
- 同旁内角互补
```

## 🔑 三角形

### 三角形的分類

**按边分類：**
| 类型 | 特征 |
|------|------|
| 等边三角形 | 三边相等 |
| 等腰三角形 | 两边相等 |
| 不等边三角形 | 三边不等 |

**按角分類：**
| 类型 | 特征 |
|------|------|
| 锐角三角形 | 三个角都是锐角 |
| 直角三角形 | 有一个直角 |
| 钝角三角形 | 有一个钝角 |

### 三角形的基本性质

**边的关系**：
```
三角形两边之和大于第三边
a + b > c, b + c > a, c + a > b

三角形两边之差小于第三边
|a - b| < c, |b - c| < a, |c - a| < b
```

**角的关系**：
```
三角形内角和为180°
∠A + ∠B + ∠C = 180°
```

### 特殊三角形

**等边三角形**：
- 三边相等，三个角都是60°
- 高、中线、角平分线三线合一

**等腰三角形**：
- 两腰相等，底角相等
- 顶角平分线垂直平分底边

**直角三角形**：
- 勾股定理：a² + b² = c²
- 30°角所对直角边等于斜边的一半
- 斜边上的中线等于斜边的一半

### 三角形的重要线段

| 线段 | 定义 | 性质 |
|------|------|------|
| 高 | 顶点到对边的垂线段 | 三条高交于垂心 |
| 中线 | 顶点到对边中点的连线 | 三条中线交于重心 |
| 角平分线 | 角内到两边距离相等的线 | 三条角平分线交于内心 |

## 📊 四边形

### 四边形分類

```
梯形
 ├─ 直角梯形
 ├─ 等腰梯形
 └─ 一般梯形

平行四边形
 ├─ 矩形
 │   ├─ 正方形
 │   └─ 长方形
 ├─ 菱形
 │   └─ 正方形
 └─ 一般平行四边形
```

### 平行四边形性质

```
对边平行且相等
AB ∥ CD, AB = CD
BC ∥ AD, BC = AD

对角线互相平分
AC 与 BD 交点为 O，则 OA = OC, OB = OD

对角相等
∠A = ∠C, ∠B = ∠D
```

### 矩形
**性质**：
- 具有平行四边形所有性质
- 四个角都是直角
- 对角线相等

### 菱形
**性质**：
- 四边相等
- 对角线互相垂直
- 对角线平分一组对角

### 正方形
**性质**：
- 具有矩形所有性质
- 具有菱形所有性质
- 四边相等，四个角都是直角
- 对角线相等且互相垂直平分

## 🎯 圆

### 圆的基本概念
| 概念 | 定义 |
|------|------|
| 圆心 | 圆的中心，到圆上各点距离相等 |
| 半径 (r) | 圆心到圆上任意一点的距离 |
| 直径 (d) | 通過圆心的弦，d = 2r |
| 弦 | 連接圆上两点的线段 |
| 弧 | 圆上两点间的部分 |
| 圆心角 | 顶点在圆心的角 |
| 圆周角 | 顶点在圆上的角 |

### 圆的重要性质

**垂径定理**：
```
垂直于弦的直径平分这条弦及其所对的两条弧
```

**圆心角与圆周角**：
```
同弧所对的圆周角等于圆心角的一半
∠AOB = 2 × ∠ACB
```

**切线**：
```
切线垂直于过切点的半径
```

### 圆的度量

```
周长：C = 2πr = πd
面积：S = πr² = π(d/2)²

弧长：l = (nπr)/180 （n为中心角度数）
扇形面积：S = (nπr²)/360 = ½lr
```

## 🔍 重要定理

### 勾股定理
在直角三角形中，两直角边的平方和等于斜边的平方。

```
a² + b² = c²
```

**常见勾股数**：
| a | b | c |
|---|---|---|
| 3 | 4 | 5 |
| 5 | 12 | 13 |
| 6 | 8 | 10 |
| 7 | 24 | 25 |
| 8 | 15 | 17 |
| 9 | 40 | 41 |

### 三角形相似
两个三角形若对应角相等、对应边成比例，则称它们相似。

**判定条件**（满足其一即可）：
```
1. 两角对应相等 (AA)
2. 两边成比例且夹角相等 (SAS)
3. 三边成比例 (SSS)
4. 斜边和直角边成比例（仅限直角三角形）
```

**相似三角形性质**：
```
相似比：k = a₁/a₂ = b₁/b₂ = c₁/c₂
面积比：S₁/S₂ = k²
```

### 全等三角形
两个三角形若能完全重合，则称它们全等。

**判定条件**：
| 条件 | 說明 |
|------|------|
| SSS | 三边对应相等 |
| SAS | 两边及夹角对应相等 |
| ASA | 两角及夹边对应相等 |
| AAS | 两角及对边对应相等 |
| HL | 斜边和直角边对应相等（直角三角形） |

## 📐 常用公式

### 三角形面积

| 公式 | 适用条件 |
|------|----------|
| S = ½ah | 底a，高h |
| S = ½ab·sinC | 两边及夹角 |
| 海伦公式 | 三边已知 |

**海伦公式**：
```
设半周长 p = (a + b + c)/2
则面积 S = √[p(p-a)(p-b)(p-c)]
```

### 特殊圖形面积

```
正方形：S = a²
矩形：S = ab
平行四边形：S = ah
菱形：S = ½ × 对角线1 × 对角线2
梯形：S = ½(a + b)h
圆：S = πr²
圆环：S = π(R² - r²)
```

## 📚 练习题

### 基礎题
1. 求等边三角形边长为4时的高和面积
2. 证明三角形内角和为180°
3. 已知直角三角形两直角边为3和4，求斜边
4. 圆的半径为5，求面积和周长

### 进阶题
1. 证明三角形中位线定理
2. 已知菱形对角线长分别为6和8，求边长和面积
3. 已知圆半径为r，求内接正六边形面积
4. 证明：直角三角形斜边上的中线等于斜边的一半
