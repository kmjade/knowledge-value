# 线性代数

> "线性代数是数据科学和机器学习的基础。" - 佚名

## 📖 基本概念

### 标量、向量、矩阵、张量

| 概念 | 表示 | 维度 | 示例 |
|------|------|------|------|
| 标量 | 小写字母 | 0维 | a = 5 |
| 向量 | 小写加粗 | 1维 | x = [1, 2, 3]ᵀ |
| 矩阵 | 大写加粗 | 2维 | A = [[1,2],[3,4]] |
| 张量 | 大写加粗 | ≥3维 | T[i,j,k] |

---

## 🔢 向量

### 向量定义

n维列向量：
```
x = [x₁, x₂, ..., xₙ]ᵀ
```

### 向量运算

**加法**：
```
x + y = [x₁ + y₁, x₂ + y₂, ..., xₙ + yₙ]ᵀ
```

**标量乘法**：
```
α·x = [α·x₁, α·x₂, ..., α·xₙ]ᵀ
```

**点积（内积）**：
```
x · y = xᵀy = Σᵢ xᵢyᵢ
```

**外积**：
```
x ⊗ y = xyᵀ
结果为 n×m 矩阵
```

### 向量范数

**L₁范数**（曼哈顿距离）：
```
||x||₁ = Σᵢ |xᵢ|
```

**L₂范数**（欧几里得距离）：
```
||x||₂ = √(Σᵢ xᵢ²)
```

**L∞范数**：
```
||x||∞ = max |xᵢ|
```

**p-范数**：
```
||x||ₚ = (Σᵢ |xᵢ|ᵖ)^(1/ᵖ)
```

### 向量性质

**正交**：x·y = 0
**正交归一**：x·x = 1
**平行**：y = kx（k为标量）

## 📊 矩阵

### 矩阵定义

m×n 矩阵：
```
A = [aᵢⱼ] m×n = ┌a₁₁ a₁₂ ... a₁ₙ┐
                │a₂₁ a₂₂ ... a₂ₙ│
                └aₘ₁ aₘ₂ ... aₘₙ┘
```

### 矩阵运算

**加法**（同维矩阵）：
```
C = A + B，cᵢⱼ = aᵢⱼ + bᵢⱼ
```

**标量乘法**：
```
B = αA，bᵢⱼ = α·aᵢⱼ
```

**矩阵乘法**：
```
C = AB（A是m×n，B是n×p）
cᵢⱼ = Σₖ aᵢₖ·bₖⱼ
```

**示例**：
```
A = [1 2]    B = [5 6]
    [3 4]        [7 8]

AB = [1×5+2×7  1×6+2×8] = [19 22]
     [3×5+4×7  3×6+4×8] = [43 50]
```

### 特殊矩阵

| 矩阵类型 | 定义 | 性质 |
|----------|------|------|
| 单位矩阵 | 对角线为1，其余为0 | AI = IA = A |
| 对角矩阵 | 仅对角线有非零元素 | D = diag(d₁,d₂,...) |
| 对称矩阵 | Aᵀ = A | aᵢⱼ = aⱼᵢ |
| 反对称矩阵 | Aᵀ = -A | aᵢᵢ = 0 |
| 正交矩阵 | AᵀA = AAᵀ = I | A⁻¹ = Aᵀ |
| 幂等矩阵 | A² = A | 投影矩阵 |
| 正定矩阵 | xᵀAx > 0（x≠0） | 所有特征值>0 |

## 🔑 矩阵性质

### 转置
```
Aᵀ 的 (i,j) 元素 = A 的 (j,i) 元素
```

**性质**：
```
(Aᵀ)ᵀ = A
(A+B)ᵀ = Aᵀ + Bᵀ
(AB)ᵀ = BᵀAᵀ
```

### 逆矩阵
```
AA⁻¹ = A⁻¹A = I
```

**存在条件**：方阵且 det(A) ≠ 0

**性质**：
```
(A⁻¹)⁻¹ = A
(AB)⁻¹ = B⁻¹A⁻¹
(Aᵀ)⁻¹ = (A⁻¹)ᵀ
```

### 迹（Trace）
主对角线元素之和：
```
tr(A) = Σᵢ aᵢᵢ
```

**性质**：
```
tr(A) = tr(Aᵀ)
tr(A+B) = tr(A) + tr(B)
tr(AB) = tr(BA)
```

### 行列式（Determinant）
```
2×2: det(A) = a₁₁a₂₂ - a₁₂a₂₁

3×3: 展开第一行
det(A) = a₁₁det(M₁₁) - a₁₂det(M₁₂) + a₁₃det(M₁₃)
```

**性质**：
```
det(Aᵀ) = det(A)
det(AB) = det(A)det(B)
det(A⁻¹) = 1/det(A)
```

## 📐 矩阵分解

### 特征值与特征向量

**定义**：对于方阵 A，若存在非零向量 v 和标量 λ 使得：
```
Av = λv
```
则 λ 称为 A 的特征值，v 称为对应的特征向量。

**求解**：
```
det(A - λI) = 0
```

**示例**：
```
A = [2 1]
    [1 2]

det(A - λI) = det([2-λ 1]) = (2-λ)² - 1 = 0
                    [1 2-λ]

λ² - 4λ + 3 = 0
λ₁ = 1, λ₂ = 3
```

### 奇异值分解（SVD）

任意 m×n 矩阵 A 可以分解为：
```
A = UΣVᵀ
```

- U：m×m 正交矩阵
- Σ：m×n 对角矩阵（对角线为奇异值）
- V：n×n 正交矩阵

**应用**：
- 数据压缩
- 推荐
- 图像处理
- 主成分分析（PCA）

### 特征值分解

对称矩阵 A 可以分解为：
```
A = QΛQᵀ
```

- Q：正交矩阵（特征向量）
- Λ：对角矩阵（特征值）

## 🔍 线性方程组

### 矩阵形式
```
Ax = b
```

### 解的性质

| det(A) | 解的情况 |
|--------|----------|
| det(A) ≠ 0 | 唯一解 x = A⁻¹b |
| det(A) = 0 | 无解或无穷多解 |

### 高斯消元法

**步骤**：
1. 行初等变换化为行阶梯形
2. 回代求解

**示例**：
```
2x + y - z = 8  →  2x + y - z = 8
-3x - y + 2z = -11 →    -5y + 7z = 2
-2x + y + 2z = -3 →    8z = 8

回代：z = 1, y = 1, x = 4
```

## 💡 重要定理

| 定理 | 内容 |
|------|------|
| 基底扩充定理 | 线性无关向量可扩充为基底 |
| 秩-零化度定理 | rank(A) + nullity(A) = n |
| 谱定理 | 对称矩阵可对角化 |
| 逆矩阵定理 | 多个等价条件 |
| 西尔维斯特不等式 | rank(AB) ≤ min(rank(A), rank(B)) |

## 📚 常见操作汇总

| 操作 | 公式 | 条件 |
|------|------|------|
| 矩阵加法 | C = A + B | 同维度 |
| 矩阵乘法 | C = AB | A的列数=B的行数 |
| 转置 | Aᵀ | 任意矩阵 |
| 逆矩阵 | A⁻¹ | 方阵，det(A)≠0 |
| 伪逆 | A⁺ = (AᵀA)⁻¹Aᵀ | 列满秩 |
| 特征值 | det(A-λI)=0 | 方阵 |
| SVD | A=UΣVᵀ | 任意矩阵 |

## 📚 练习题

### 基础题
1. 计算 x = [1,2,3]ᵀ 和 y = [4,5,6]ᵀ 的点积
2. 求 A = [1,2;3,4] 的行列式和逆矩阵
3. 求 A = [2,1;1,2] 的特征值和特征向量
4. 高斯消元解方程组：x+y+z=6, 2x-y+z=3, x+2y-z=2

1. 证明对称矩阵的特征向量两两正交

3. 判断矩阵 A = [2,1;1,2] 是否正定
4. 求 rank(A) 和 nullity(A)，其中 A = [1,2,3;4,5,6]
