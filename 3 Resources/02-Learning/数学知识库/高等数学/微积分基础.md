# 微积分基礎

> "微积分是研究变化的数学艺术。" - 牛顿与莱布尼茨

## 📖 概述

微积分是研究连续变化的数学分支，主要包含两个核心概念：
- **微分**：研究变化率（切线斜率）
- **积分**：研究累积量（面积、体积）

---

## 📐 极限

### 极限的定义
当 x 無限接近 a 时，函数 f(x) 無限接近 L，记作：
```
lim(x→a) f(x) = L
```

### 极限的性质

| 性质 | 公式 | 說明 |
|------|------|------|
| 唯一性 | 极限值唯一 | 只能有一个极限 |
| 局部保号性 | f(x) > 0，则极限≥0 | 符号关系保持 |
| 四则運算 | lim[f±g] = lim f ± lim g | 運算可分项 |
| 复合函数 | lim(f[g(x)]) = f[lim g(x)] | 可代入 |

### 重要极限

**极限 1：**
```
lim(x→0) (sin x) / x = 1
```

**极限 2：**
```
lim(x→∞) (1 + 1/x)ˣ = e
lim(x→0) (1 + x)^(1/x) = e
```

其中 e ≈ 2.71828 是自然对数的底。

# 方法

1. **直接代入法**：连续函数可直接代入
   ```
   lim(x→2) (x + 3) = 2 + 3 = 5
   ```

2. **因式分解法**：處理 0/0 型
   ```
   lim(x→1) (x² - 1)/(x - 1)
   = lim(x→1) (x + 1)(x - 1)/(x - 1)
   = lim(x→1) (x + 1)
   = 2
   ```

3. **有理化法**：處理含根号
   ```
   lim(x→0) (√(1+x) - 1)/x
   = lim(x→0) [(√(1+x)-1)(√(1+x)+1)]/[x(√(1+x)+1)]
   = lim(x→0) x/[x(√(1+x)+1)]
   = 1/2
   ```

4. **洛必达法则**：處理 0/0 或 ∞/∞ 型
   ```
   lim(x→0) (sin x)/x
   = lim(x→0) (cos x)/1
   = 1
   ```

## 🔑 导数

### 导数的定义
函数在某点的变化率，即切线的斜率。

```
f'(x) = lim(Δx→0) [f(x + Δx) - f(x)] / Δx
```

**几何意义**：曲线在某点切线的斜率
**物理意义**：瞬时速度、加速度

### 基本导数公式

| 函数 f(x) | 导数 f'(x) |
|-----------|-----------|
| C（常数） | 0 |
| xⁿ | n·xⁿ⁻¹ |
| aˣ | aˣ · ln a |
| eˣ | eˣ |
| ln x | 1/x |
| logₐx | 1/(x · ln a) |
| sin x | cos x |
| cos x | -sin x |
| tan x | sec²x |
| cot x | -csc²x |

### 求导法则

**四则運算：**
```
(f ± g)' = f' ± g'
(f · g)' = f'·g + f·g'      （乘积法则）
(f/g)' = (f'·g - f·g')/g²   （商法则）
```

**复合函数（链式法则）：**
```
若 y = f(u)，u = g(x)
则 y' = f'(u) · g'(x)

例：y = sin(2x)
y' = cos(2x) · 2 = 2cos(2x)
```

**反函数：**
```
若 y = f(x)，x = f⁻¹(y)
则 (f⁻¹)'(y) = 1/f'(x)
```

### 高阶导数
导数的导数。

```
f''(x) = [f'(x)]'      （二阶导数）
f'''(x) = [f''(x)]'     （三阶导数）
f⁽ⁿ⁾(x)                 （n阶导数）
```

### 导数的應用程式

**1. 求切线方程：**
```
y = f(x)，在 x₀ 处的切线：
y - f(x₀) = f'(x₀)(x - x₀)
```

**2. 判斷单调性：**
```
f'(x) > 0：函数递增
f'(x) < 0：函数递减
```

**3. 求极值：**
```
f'(x₀) = 0 且 f''(x₀) > 0 → 极小值
f'(x₀) = 0 且 f''(x₀) < 0 → 极大值
```

## 📊 积分

### 不定积分
导数的逆運算。

```
∫ f(x) dx = F(x) + C
其中 F'(x) = f(x)，C 为常数
```

### 基本积分公式

| 被积函数 f(x) | 不定积分 |
|---------------|----------|
| xⁿ | xⁿ⁺¹/(n+1) + C (n ≠ -1) |
| 1/x | ln|x| + C |
| aˣ | aˣ/ln a + C |
| eˣ | eˣ + C |
| sin x | -cos x + C |
| cos x | sin x + C |
| sec²x | tan x + C |
| 1/(1 + x²) | arctan x + C |

### 积分法则

**线性性质：**
```
∫ [f(x) ± g(x)] dx = ∫ f(x) dx ± ∫ g(x) dx
∫ k·f(x) dx = k ∫ f(x) dx
```

**换元积分法：**
```
∫ f[g(x)]·g'(x) dx = ∫ f(u) du  （设 u = g(x)）

例：∫ 2x·cos(x²) dx
设 u = x²，du = 2x dx
= ∫ cos u du
= sin u + C
= sin(x²) + C
```

**分部积分法：**
```
∫ u·dv = u·v - ∫ v·du

例：∫ x·eˣ dx
设 u = x, dv = eˣ dx
du = dx, v = eˣ
= x·eˣ - ∫ eˣ dx
= x·eˣ - eˣ + C
= eˣ(x - 1) + C
```

### 定积分
計算区间 [a, b] 上曲线与x轴围成的面积。

```
∫[a,b] f(x) dx = F(b) - F(a)
```

**几何意义**：曲線下的面积
**物理意义**：位移、功、品質

**微积分基本定理：**
```
若 F'(x) = f(x)
则 ∫[a,b] f(x) dx = F(b) - F(a)
```

## 🔍 微积分應用程式

### 1. 求面积
```
計算 y = x²，x 从 0 到 1 的面积

∫[0,1] x² dx = [x³/3]₀¹ = 1/3 - 0 = 1/3
```

### 2. 求体积（旋转体）
```
绕x轴旋转的体积：
V = π ∫[a,b] f(x)² dx
```

### 3. 求弧长
```
曲线弧长：
L = ∫[a,b] √(1 + [f'(x)]²) dx
```

### 4. 求极值和最值
```
求 y = x³ - 3x + 1 的极值

y' = 3x² - 3 = 0
x = ±1

y'' = 6x
x = 1: y'' = 6 > 0 → 极小值
x = -1: y'' = -6 < 0 → 极大值
```

## 📚 常见导数与积分对照

| 函数 | 导数 | 原函数（积分） |
|------|------|---------------|
| xⁿ | n·xⁿ⁻¹ | xⁿ⁺¹/(n+1) + C |
| eˣ | eˣ | eˣ + C |
| ln x | 1/x | x(ln x - 1) + C |
| sin x | cos x | -cos x + C |
| cos x | -sin x | sin x + C |
| arctan x | 1/(1+x²) | x·arctan x - ½ln(1+x²) + C |
