## 🎯 學習建議

### 學習顺序建議

**初学者路徑：**
```
数学之美 → 什么是数学 → 高中数学精讲
     ↓
微积分入门 → 线性代数入门 → 概率統計入门
```

**大学路徑：**
```
# 教程
     ↓
# 分析
```

# 方法

1. **教材 + 視訊**：配合Coursera、MIT OpenCourseWare等公开课
2. **理论 + 练习**：每学一节都要做习题
3. **基礎 + 拓展**：先掌握基礎，再看相關读物
4. **輸入 + 輸出**：尝试用自己的话解釋概念
