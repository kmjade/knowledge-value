## 原子任务 1: 黎曼ζ函数基础

### 任务目标
理解黎曼ζ函数的定义、收敛性质和基本公式

### 预估时间
2-3 小时

### 完成标准
- [ ] 能写出ζ函数的原始定义
- [ ] 理解收敛区域
- [ ] 掌握欧拉乘积公式
- [ ] 能计算ζ(2)等简单值

### 核心内容

#### 1.1 ζ函数定义

**原始定义（1859年，Bernhard Riemann）**：

```
ζ(s) = Σ (n=1 to ∞) 1/n^s
```

其中 s = σ + it 是复数，σ > 1 时收敛

#### 1.2 收敛区域

```
区域 I: Re(s) > 1     绝对收敛
区域 II: Re(s) > 0      条件收敛（需要解析延拓）
区域 III: Re(s) ≤ 0    发散（需要解析延拓）
```

#### 1.3 欧拉乘积公式（1737年，Euler）

```
ζ(s) = Π (p is prime) 1/(1 - p^(-s))
```

**重要意义**：
- 连接ζ函数与素数
- 这是数论与复分析结合的桥梁
- 证明了ζ函数的所有非平凡零点都在 critical strip (0 < Re(s) < 1) 内

#### 1.4 特殊值

| s | ζ(s) | 说明 |
|----|-------|------|
| 2 | π²/6 | Basel 问题（1735年解决） |
| -1 | -1/2 | 简单的负整数 |
| 0 | -1/2 | 解析延拓后的值 |

### 练习题

1. **练习 1.1**：证明当 Re(s) > 1 时，级数 Σ(1/n^s) 收敛

   **提示**：使用积分判别法

2. **练习 1.2**：从欧拉乘积公式推导素数无限多的证明

   **提示**：假设只有有限个素数，推出矛盾

3. **练习 1.3**：计算ζ(4)的精确值（使用已知结果）

   **答案**：π⁴/90

## 原子任务 2: 解析延拓

### 任务目标
理解ζ函数如何从收敛区域延拓到整个复平面

### 预估时间
1-2 小时

### 完成标准
- [ ] 理解解析延拓的概念
- [ ] 掌握函数方程
- [ ] 理解对称性 ζ(s) = ζ(1-s)
- [ ] 了解 trivial zeros

### 核心内容

#### 2.1 解析延拓概念

**定义**：将一个只在某些区域收敛的函数扩展到更大的定义域，保持解析性

**方法**：
- 黎曼使用围道积分
- 利用函数方程和对称性

#### 2.2 函数方程（Functional Equation）

**黎曼公式（1859年）**：

```
ζ(s) = 2^s × π^(s-1) × sin(πs/2) × Γ(1-s) × ζ(1-s)
```

其中 Γ(s) 是伽马函数（Gamma function）

**重要性质**：
```
ζ(1-s) 的值 ⟷ ζ(s) 的值
```

这给出了ζ函数的对称性

#### 2.3 对称性

**Critical strip**：0 < Re(s) < 1

**对称轴**：Re(s) = 1/2（critical line）

```
        ↑ Im(s)
        |
   ...  |  ...
--------+-------  Re(s) = 1/2
        |
   ...  |  ...
        ↓
```

#### 2.4 Trivial Zeros（平凡零点）

**位置**：s = -2, -4, -6, -8, ... （负偶整数）

**来源**：sin(πs/2) 项的零点

**重要性**：这些零点完全已知，不是猜想的关注点

### 练习题

1. **练习 2.1**：验证 ζ(-2) = 0

   **提示**：使用函数方程

2. **练习 2.2**：证明 ζ(1-s) = ζ(s) 当 Re(s) = 1/2

   **提示**：分析函数方程的绝对值

### 验证检查

- [ ] 理解解析延拓的必要性
- [ ] 掌握函数方程的结构
- [ ] 理解对称性和critical line
- [ ] 识别所有trivial zeros

## 原子任务 3: 猜想陈述与含义

### 任务目标
准确理解和陈述黎曼猜想

### 预估时间
2-3 小时

### 完成标准
- [ ] 能准确陈述黎曼猜想
- [ ] 理解non-trivial zeros的定义
- [ ] 理解critical line的意义
- [ ] 理解等价表述

### 核心内容

#### 3.1 黎曼猜想的正式陈述

**黎曼猜想（1859年）**：

```
黎曼ζ函数的所有非平凡零点
都具有实部 1/2
```

**数学表述**：

```
If ζ(ρ) = 0 and ρ is non-trivial,
then Re(ρ) = 1/2
```

#### 3.2 非平凡零点的定义

**排除**：
- Trivial zeros: s = -2, -4, -6, ...
- s = 1（ζ函数的simple pole）

**Non-trivial zeros**：满足 0 < Re(ρ) < 1 的其他零点

#### 3.3 重要等价表述

**表述 1：素数分布误差**

```
π(x) = Li(x) + O(√x · log x)
```

其中：
- π(x) = 小于等于 x 的素数个数
- Li(x) = log integral（对数积分）

**黎曼猜想 ⟷ 素数分布误差较小**

**表述 2：零点实部界限**

```
如果黎曼猜想错误，则存在某个零点 ρ
使得 Re(ρ) > 1/2
```

定义：
```
Θ = max{Re(ρ) : ζ(ρ) = 0, ρ is non-trivial}
```

黎曼猜想 ⟷ Θ = 1/2

#### 3.4 零点的对称性

**如果 ρ 是非平凡零点**：

```
那么 1 - ρ, ρ̅ (共轭), 1 - ρ̅ 都是非平凡零点
```

这意味着零点关于critical line和实轴对称

### 练习题

1. **练习 3.1**：证明 ζ(s) 在 critical strip 内有无穷多个零点

   **提示**：利用解析延拓和性质定理

2. **练习 3.2**：假设存在一个零点 ρ 且 Re(ρ) ≠ 1/2，推导π(x)的误差项

   **提示**：使用显式公式（Explicit Formula）

### 验证检查

- [ ] 能准确陈述黎曼猜想

- [ ] 理解critical line的概念

## 原子任务 4: 素数分布

### 任务目标
理解黎曼猜想与素数分布的联系

### 预估时间
1-2 小时

### 完成标准
- [ ] 理解素数计数函数π(x)
- [ ] 理解素数定理
- [ ] 理解黎曼猜想对误差项的影响
- [ ] 掌握显式公式的基本形式

### 核心内容

#### 4.1 素数计数函数

**定义**：

```
π(x) = #{p ≤ x : p is prime}
```

**例子**：
- π(10) = 4（素数：2, 3, 5, 7）
- π(20) = 8（素数：2, 3, 5, 7, 11, 13, 17, 19）

#### 4.2 素数定理（Prime Number Theorem, 1896）

**定理陈述**：

```
lim (x → ∞) π(x) / (x / log x) = 1
```

**等价形式**：

```
π(x) ~ x / log x
```

**意义**：描述了大范围内素数的平均分布

#### 4.3 黎曼猜想与误差项

**素数定理的精确形式**：

```
π(x) = Li(x) + error term
```

**黎曼猜想的影响**：

```
如果黎曼猜想正确：
  error term = O(√x · log x)

如果黎曼猜想错误（存在零点 ρ 且 Re(ρ) > 1/2）：
  error term = O(x^Re(ρ) / log x)
```

**对比**：
- 猜想正确：误差 ≈ √x · log x
- 猜想错误：误差可能远大于 √x · log x

#### 4.4 显式公式（Explicit Formula, Riemann, 1859）

**基本形式**：

```
π(x) = Li(x) - Σ_ρ Li(x^ρ) + 小量修正
```

其中：
- 求和遍历所有非平凡零点 ρ
- 每个零点对素数分布有"震荡"影响

### 练习题

1. **练习 4.1**：计算π(100)和π(1000)

   **答案**：π(100) = 25, π(1000) = 168

2. **练习 4.2**：比较 Li(x)和x/log x在不同x值的差异

   **提示**：计算几个点，如x=100, 1000, 10000

## 原子任务 5: 研究历史与进展

### 任务目标
了解黎曼猜想的研究历史和重要进展

### 预估时间
2-3 小时

### 完成标准
- [ ] 了解1859年黎曼的原始工作
- [ ] 掌握关键历史里程碑
- [ ] 了解现代重要结果
- [ ] 理解当前研究状况

### 核心内容

#### 5.1 原始论文（1859）

**黎曼论文**：《Über die Anzahl der Primzahlen unter einer gegebenen Größe》（论不超过给定大小的素数个数）

**关键贡献**：
1. 定义了ζ函数
2. 建立了解析延拓
3. 发现了函数方程
4. 提出了关于零点的猜想
5. 给出了显式公式

#### 5.2 关键历史里程碑

| 年份 | 数学家 | 贡献 |
|------|----------|--------|
| 1859 | Riemann | 提出猜想，定义ζ函数 |
| 1896 | Hadamard, de la Vallée Poussin | 证明素数定理（无需猜想） |
| 1900 | Hilbert | 将猜想列入8个问题中的第8个 |
| 1903 | von Mangoldt | 计算了前15个零点 |
| 1914 | Hardy | 证明了无穷多个零点在critical line上 |
| 1921 | Hardy & Littlewood | 证明了零点密度定理 |
| 1942 | Selberg | 引入迹公式等重要工具 |
| 1948 | Turing | 发展计算方法，检查了大量零点 |
| 1972 | Montgomery | 发现零点对之间的相关性 |
| 1974 | Levinson | 证明了至少1/3零点在critical line上 |
| 1989 | Conrey | 改进到至少2/5零点 |
| 2004 | Gourdon & Demarte | 提出了新的等价表述 |

#### 5.3 现代重要结果

##### 5.3.1 零点比例

**关键定理**：
```
如果定义 N(T) = #{ρ : 0 < Im(ρ) < T, Re(ρ) = 1/2}
        Z(T) = #{ρ : 0 < Im(ρ) < T}

那么：
  lim (T → ∞) N(T) / Z(T) = ?
```

**已知结果**：
- Levinson (1974): > 1/3
- Conrey (1989): > 2/5
- 当前最佳比例: > 2/5

**意义**：越来越多零点被证明在critical line上

##### 5.3.2 计算验证

**截至2025年**：
- 已计算并验证前约 10^13 个零点
- 所有已验证的零点都在critical line上
- 无反例发现

##### 5.3.3 等价表述

**Weil 猜想（曲线）**：推广到代数曲线的ζ函数
**Berry-Keating 猜想**：关于算术函数的随机矩阵理论
**高斯环猜想**：在有限域上的推广

#### 5.4 当前研究状况

**研究方向**：
1. **解析数论**：研究ζ函数的性质
2. **计算数论**：大规模计算零点
3. **随机矩阵理论**：研究零点的统计性质
4. **量子混沌**：寻找物理学联系

**未解决问题**：
- 是否所有非平凡零点都在critical line上？
- 零点的分布规律是什么？
- ζ函数在critical strip上还有什么未知性质？

## 原子任务 6: 重要意义与应用

### 任务目标

### 核心内容

#### 6.1 数学意义

**基础数论核心问题**：
- 素数分布的精确描述
- 解析数论与代数数论的桥梁
- ζ函数理论的基石

**推动的数学发展**：
- 复分析方法
- 解析数论
- 代数几何
- 表示论

#### 6.2 应用领域

##### 6.2.1 密码学

**RSA加密的安全性**：
- 基于大整数分解的困难性
- 更精确的素数分布 → 更好的因数分解算法
- 黎曼猜想 → 影响素数检测算法的效率

##### 6.2.2 计算复杂性

**素性测试算法**：
- AKS素性测试（2002年，多项式时间）
- 改进算法依赖于素数分布的精确性
- 黎曼猜想 → 可能影响素性测试的复杂度

##### 6.2.3 物理学

**量子混沌与随机矩阵**：
- ζ函数的零点与随机矩阵的特征值有惊人相似性
- 物理学（量子混沌、核物理）提供新视角
- 黎曼猜想可能在物理学中有对应

#### 6.3 千禧年大奖

**克雷数学研究所（2000年）**：
- 奖金100万美元
- 7个千禧年问题之一
- 目前仅Poincaré猜想（2003年）被解决

**获奖条件**：
- 证明或反证黎曼猜想
- 证明必须被数学界接受至少2年

#### 6.4 与其他数学问题的联系

**关联问题**：
1. **Goldbach猜想**：每个大于2的偶数都是两个素数之和
2. **孪生素数猜想**：存在无穷多对相差2的素数
3. **ABC猜想**：涉及素数加法性质
4. **Langlands纲领**：更广泛的数学统一框架

**黎曼猜想的解决**可能推动这些问题的进展

3. **第三遍（综合巩固）**：
   - 建立概念间的联系
   - 尝试解决综合性问题
   - 总结关键定理和公式

### 笔记方法

- **公式卡片**：制作关键公式的速查卡片
- **概念图**：绘制概念之间的思维导图
- **问题清单**：记录未解决的问题
- **复习计划**：定期回顾已学内容

### 持续学习

- **每周学习时间**：至少5-10小时
- **复习频率**：每两周复习一次
- **扩展阅读**：根据兴趣选择深入方向
- **实际计算**：尝试计算一些ζ函数值
