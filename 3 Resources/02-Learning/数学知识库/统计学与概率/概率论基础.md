# 概率论基础

> "概率是可能性的度量，是理性决策的基础。" - 佚名

## 📖 概率定义

### 概率
在相同条件下，大量重复试验中事件 A 发生的频率的稳定值，称为事件 A 发生的概率，记作 P(A)。

### 概率的基本性质

| 性质 | 描述 | 公式 |
|------|------|------|
| 非负性 | 概率非负 | 0 ≤ P(A) ≤ 1 |
| 规范性 | 必然事件概率为1 | P(Ω) = 1 |
| 可加性 | 互斥事件概率相加 | P(A∪B) = P(A) + P(B) |

**特殊事件：**
```
不可能事件：P(Ø) = 0
必然事件：P(Ω) = 1
```

## 🔑 概率计算

### 1. 古典概型
等可能结果的有限样本空间。

```
P(A) = A中有利的结果数 / 所有可能的结果数
```

**示例：**
```
掷一枚骰子，点数大于4的概率：

可能结果：1, 2, 3, 4, 5, 6（共6个）
大于4的结果：5, 6（共2个）

P(点数>4) = 2/6 = 1/3
```

### 2. 条件概率
在事件 B 发生的条件下，事件 A 发生的概率。

```
P(A|B) = P(AB) / P(B)
```

**示例：**
```
袋中有3个红球、2个白球，不放回地取两次，
求第一次取红球后第二次也取红球的概率。

P(第二次红|第一次红) = P(两次红)/P(第一次红)
= (3×2/5×4) / (3/5)
= (6/20) / (3/5)
= (3/10) / (3/5)
= 1/2

或直接理解：第一次取走1个红球后，
袋中剩2红2白，概率 = 2/4 = 1/2
```

### 3. 乘法公式
```
P(AB) = P(A) · P(B|A) = P(B) · P(A|B)
```

**示例：**
```
甲、乙两人独立射击，甲命中率0.8，乙命中率0.6，
求两人都命中的概率。

P(甲中且乙中) = P(甲中) × P(乙中)
= 0.8 × 0.6 = 0.48
```

### 4. 全概率公式
```
设 B₁, B₂, ..., Bₙ 是完备事件组（互斥且并集为Ω），
则对任意事件 A：

P(A) = Σ P(Bᵢ) · P(A|Bᵢ)
```

**示例：**
```
两个袋子，袋1有3红7白，袋2有6红4白。
随机选一袋，从中取一球为红，求选袋1的概率。

P(红) = P(袋1)×P(红|袋1) + P(袋2)×P(红|袋2)
= 0.5 × 3/10 + 0.5 × 6/10
= 0.15 + 0.3 = 0.45
```

### 5. 贝叶斯公式
```
P(Bᵢ|A) = P(Bᵢ) · P(A|Bᵢ) / Σ P(Bⱼ) · P(A|Bⱼ)
```

**示例：**
```
疾病患病率1%，检测准确率99%。
某人检测结果为阳性，求真正患病的概率。

P(患病|阳性) = P(患病)×P(阳性|患病) / [P(患病)×P(阳性|患病) + P(健康)×P(阳性|健康)]
= 0.01 × 0.99 / (0.01 × 0.99 + 0.99 × 0.01)
= 0.0099 / 0.0198 = 0.5
```

## 📐 独立性

### 事件独立
两个事件 A, B 相互独立，当且仅当：
```
P(AB) = P(A) · P(B)
```

**等价条件：**
```
P(A|B) = P(A)
P(B|A) = P(B)
```

**示例：**
```
掷两枚骰子，求都出现6的概率。

P(第一枚6) = 1/6
P(第二枚6) = 1/6
P(都6) = 1/6 × 1/6 = 1/36
```

### 多个事件独立
```
P(AB...N) = P(A) · P(B) · ... · P(N)
```

## 📊 常见概率分布

### 1. 伯努利分布（0-1分布）
单次试验只有两种结果（成功/失败）。

```
P(X = 1) = p
P(X = 0) = 1 - p = q

期望：E(X) = p
方差：D(X) = pq
```

### 2. 二项分布
n 次独立伯努利试验中成功的次数。

```
P(X = k) = C(n,k) · pᵏ · qⁿ⁻ᵏ

其中：C(n,k) = n! / [k!(n-k)!]

期望：E(X) = np
方差：D(X) = npq
```

**示例：**
```
掷10次硬币，求恰好6次正面的概率。

P(X = 6) = C(10,6) × (0.5)⁶ × (0.5)⁴
= 210 × 1/64 × 1/16
= 210/1024 ≈ 0.205
```

### 3. 泊松分布
单位时间/空间内事件发生的次数。

```
P(X = k) = (λᵏ · e⁻ˡ) / k!

期望：E(X) = λ
方差：D(X) = λ
```

**应用场景：**
- 电话呼叫次数
- 网站访问量
- 放射性衰变

### 4. 正态分布（高斯分布）
最重要的连续概率分布。

```
概率密度函数：
f(x) = (1/σ√2π) · e^[-(x-μ)²/(2σ²)]

期望：E(X) = μ
方差：D(X) = σ²
```

**标准正态分布：**
```
μ = 0, σ = 1
记作：X ~ N(0,1)
```

**重要性质：**
| 区间 | 概率 |
|------|------|
| (μ - σ, μ + σ) | 68.27% |
| (μ - 2σ, μ + 2σ) | 95.45% |
| (μ - 3σ, μ + 3σ) | 99.73% |

## 🔍 随机变量

### 定义
将随机试验的结果映射到数值的函数。

**类型：**
- **离散型随机变量**：取有限或可列无限个值
- **连续型随机变量**：取某一区间的任意值

### 期望（均值）
```
离散型：E(X) = Σ xᵢ · P(X = xᵢ)
连续型：E(X) = ∫ x · f(x) dx
```

### 方差
```
D(X) = E[X - E(X)]² = E(X²) - [E(X)]²
标准差：σ = √D(X)
```

### 性质
```
E(aX + b) = aE(X) + b
D(aX + b) = a²D(X)
E(X + Y) = E(X) + E(Y)
D(X + Y) = D(X) + D(Y) （独立时）
```

## 📚 常用公式汇总

### 排列组合
```
排列：A(n,k) = n! / (n-k)!
组合：C(n,k) = n! / [k!(n-k)!]

示例：从5人中选3人排列
A(5,3) = 5!/2! = 60

示例：从5人中选3人组合
C(5,3) = 5!/(3!2!) = 10
```

### 概率公式
| 公式 | 用途 |
|------|------|
| P(A∪B) = P(A) + P(B) - P(AB) | 求并集概率 |
| P(A|B) = P(AB)/P(B) | 条件概率 |
| P(AB) = P(A)·P(B\|A) | 乘法公式 |
| P(A|B) = P(B\|A)·P(A)/P(B) | 贝叶斯公式 |
