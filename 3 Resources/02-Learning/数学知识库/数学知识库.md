# 数学知识库 (Mathematics Knowledge Base)

> "数学是科学的皇后。" - 卡尔·高斯

## 🎯 概览

数学是研究数量、结构、变化和空间等概念的一门学科。它不仅是自然科学的基础，也是逻辑思维和抽象能力训练的重要工具。本知识库系统整理了从基础数学到高等数学的各个分支。

## 🔍 快速参考

### 📐 重要公式

| 类别 | 公式 | 应用场景 |
|------|------|----------|
| 二次方程 | x = (-b ± √(b²-4ac))/2a | 求解二次方程 |
| 勾股定理 | a² + b² = c² | 直角三角形 |
| 欧拉公式 | e^(iπ) + 1 = 0 | 复数、三角函数 |
| 微积分基本定理 | ∫[a,b] f(x)dx = F(b) - F(a) | 定积分 |
| 正态分布 | f(x) = (1/σ√2π)e^(-(x-μ)²/(2σ²)) | 统计分布 |

### 🔑 核心概念
- **函数**：输入与输出的对应关系
- **极限**：变量趋近某值时的行为
- **导数**：变化率、切线斜率
- **积分**：面积、累积量
- **概率**：事件发生的可能性
- **统计**：数据分析和推断
