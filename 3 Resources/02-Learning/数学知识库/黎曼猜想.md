# 黎曼猜想（Riemann Hypothesis）

> [!quote] 数学中最重要的未解决问题之一

## 基本陈述

### 黎曼猜想（1859年）

**表述**：
> 黎曼ζ函数的所有非平凡零点都具有实部 1/2

**数学形式**：
```
若 ζ(ρ) = 0 且 ρ 是非平凡零点
则 Re(ρ) = 1/2
```

---

## 核心概念

### 1. 黎曼ζ函数

**定义**（Bernhard Riemann, 1859）：
```
ζ(s) = Σ (n=1 to ∞) 1/n^s
其中 s = σ + it 是复数，σ > 1 时收敛
```

**解析延拓**：
- 通过围道积分和欧拉乘积公式
- 将ζ函数延拓到整个复平面（s = 1 除外）

**欧拉乘积公式**（Euler, 1737）：
```
ζ(s) = Π (p 是素数) 1/(1 - p^(-s))
```

### 2. 收敛区域

```
区域 I:   Re(s) > 1      → 绝对收敛
区域 II:  0 < Re(s) ≤ 1    → 条件收敛（需要解析延拓）
区域 III: Re(s) ≤ 0      → 发散（需要解析延拓）
```

### 3. 平凡零点

**位置**：s = -2, -4, -6, -8, ... （负偶整数）

**来源**：sin(πs/2) 项的零点

**非平凡零点**：满足 0 < Re(s) < 1 的其他零点

### 4. Critical Strip（临界带）

**定义**：
```
Critical Strip: {s ∈ C : 0 < Re(s) < 1}
```

**Critical Line（临界线）**：Re(s) = 1/2

黎曼猜想声称：所有非平凡零点都位于临界线上

## 与素数的联系

### 素数计数函数

**定义**：
```
π(x) = #{p ≤ x : p 是素数}
```

**例子**：
- π(10) = 4（素数：2, 3, 5, 7）
- π(20) = 8（素数：2, 3, 5, 7, 11, 13, 17, 19）

### 素数定理（Hadamard, 1896）

```
lim (x → ∞) π(x) / (x / log x) = 1
```

**等价形式**：
```
π(x) ~ x / log x
```

### 误差项

**精确形式**：
```
π(x) = Li(x) + 误差项
```

其中 Li(x) = ∫₂ˣ dt/log t 是对数积分函数

**黎曼猜想的影响**：

如果黎曼猜想正确：
```
误差项 = O(√x · log x)
```

如果黎曼猜想错误（存在零点 ρ 且 Re(ρ) > 1/2）：
```
误差项 = O(x^Re(ρ) / log x)  （可能远大于 √x · log x）
```

## 等价表述

### 1. 素数分布形式

**形式 A**：
```
π(x) = Li(x) + O(√x · log x)
```

**形式 B**：
```
π(x) = Li(x) + O(x^(1/2+ε))  对任意 ε > 0
```

### 2. 零点实部形式

**定义**：
```
Θ = max{Re(ρ) : ζ(ρ) = 0, ρ 是非平凡零点}
```

黎曼猜想 ⇔ Θ = 1/2

### 3. 密度形式

**零点密度**（设 Z(T) 为 critical strip 中高度 < T 的零点数）：
```
lim (T → ∞) N(T) / Z(T) = ?
```

已知结果：
- Levinson (1974): > 1/3
- Conrey (1989): > 2/5
- 当前最佳: > 0.515...

## 研究历史

### 关键里程碑

| 年份 | 数学家 | 贡献 |
|------|----------|--------|
| 1859 | Riemann | 提出猜想，定义ζ函数 |
| 1896 | Hadamard | 证明素数定理（无需猜想） |
| 1903 | von Mangoldt | 计算前15个零点 |
| 1914 | Hardy | 证明无穷多个零点在临界线上 |
| 1921 | Hardy & Littlewood | 证明零点密度定理 |
| 1942 | Selberg | 引入迹公式等重要工具 |
| 1948 | Turing | 发展计算方法，检查了大量零点 |
| 1972 | Montgomery | 发现零点对之间的相关性 |
| 1974 | Levinson | 证明至少1/3的零点在临界线上 |
| 1989 | Conrey | 改进到至少2/5的零点在临界线上 |
| 2004 | Gourdon & Demartel | 提出新的等价表述 |

### 现代进展

截至2025年：
- 已计算并验证前约 10¹³ 个零点
- 所有已验证零点都在临界线上
- 无反例发现

## 应用意义

### 1. 密码学

**RSA 加密的安全性**：
- 基于大整数分解的困难性
- 更精确的素数分布 → 更好的因数分解算法

### 2. 计算复杂性

**素性测试算法**：
- AKS 素性测试（2002年，多项式时间）
- 改进算法依赖于素数分布的精确性

### 3. 物理学

**量子混沌与随机矩阵**：
- ζ函数的零点与随机矩阵的特征值有惊人相似性
- 物理学（量子混沌、核物理）提供新视角

## 相关概念

### 相关数论问题

**Goldbach 猜想**：
> 每个大于2的偶数都是两个素数之和

**孪生素数猜想**：
> 存在无穷多对相差2的素数

**ABC 猜想**：
> 涉及素数加法性质

**Langlands 纲领**：
> 更广泛的数学统一框架

黎曼猜想的解决可能推动这些问题的进展。
