# Notion

## 基本信息

| 属性 | 值 |
|------|------|
| **类型** | 全能工作空间 |
| **平台** | Web, Windows, macOS, iOS, Android |
| **存储** | 云端存储 |
| **许可** | Freemium（个人免费） |
| **官网** | https://notion.so |

## 描述

Notion 是一款集笔记、数据库、Wiki、任务管理于一体的全能工作空间。它以其数据库功能和灵活的 Block 结构著称，适合团队协作和项目管理。

## 核心特征

- **Block 编辑器** — 一切皆 Block，拖拽重组
- **数据库** — 表格、看板、日历、画廊等多种视图
- **模板系统** — 可复用的页面模板
- **AI 功能** — 内置 Notion AI
- **协作权限** — 精细的分享和权限控制
- **集成自动化** — API + Zapier/Make 连接

## 与 Obsidian 对比

| 维度 | Notion | Obsidian |
|------|--------|----------|
| **数据存储** | 云端 | 本地 |
| **格式** | 私有格式 | 标准 Markdown |
| **离线** | 弱 | 强 |
| **数据库** | ⭐ 核心功能 | 需插件 |
| **链接** | 双向链接 | ⭐ 双向链接 + 图谱 |
| **协作** | ⭐ 原生团队协作 | 弱 |
| **定制** | 中等 | ⭐ 2000+ 插件 |
| **AI** | 内置 Notion AI | 通过 Claude Code |
