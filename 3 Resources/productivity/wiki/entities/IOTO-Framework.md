# IOTO Framework

## 基本信息

| 属性 | 值 |
|------|------|
| **类型** | Obsidian 工作流框架 |
| **开发者** | 开源社区 |
| **平台** | Obsidian 插件生态 |
| **理念** | 输入-组织-跟踪-输出 |

## 描述

IOTO (Input → Organize → Track → Output) 是基于 Obsidian 的开源工作流框架，提供标准化的插件体系、脚本工具和文件夹架构，帮助用户在 Obsidian 中高效实现 PKM 全流程。

## 核心特征

- **四步工作流** — Input → Organize → Track → Output
- **插件体系** — 精选配置好的 Obsidian 插件组合
- **脚本工具** — Template Generator, Sync Scripts, Dashboard
- **标准文件夹架构** — 预定义的目录结构
- **快捷键体系** — 高效操作的键盘快捷方式

## 模块组件

| 模块 | 功能 |
|------|------|
| Dashboard | 统一操作面板 |
| Template Generator | 自动生成笔记模板 |
| Sync Scripts Generator | 同步脚本生成 |
| Update System | 框架版本更新 |

## 在本系统中的关系

IOTO 是 `_meta/IOTO/` 目录下的模板系统，为本 Vault 的 PARA 结构提供：

- 项目模板（创建新项目时使用）
- 笔记模板（创建新笔记时使用）
- 工作流参考（快捷键、操作流程）
