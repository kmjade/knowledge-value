## 摘要

一篇实操教程，介绍如何使用 Claude Code + LLM Wiki + Obsidian 三者结合，一站式打通 AI 知识库。强调用 Skills 实现 Wiki 自动化，不再需要多个软件切换。

> LLM wiki 最大的作用就是为整个知识库加上记忆，这样就算你的 Agent 由于上下文过长，导致失去记忆，也能通过 LLM wiki 迅速知道你的整个知识库。

> 我直接采用 Claude Code，用 skills 的方式来实现 wiki，这样这三个东西就都在一个软件中，不再需要切来切去。
