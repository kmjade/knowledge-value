# Second Brain (第二大脑)

## 定义

第二大脑是 Tiago Forte 提出的信息管理理念，通过 PARA 方法和 CODE 流程构建一个"外部大脑"，将信息从生物学大脑卸载到数字系统，实现知识的结构化存储和创造性输出。

## 核心原理

### CODE 流程

```
C → Capture   捕获有价值的信息
O → Organize  按行动性组织（PARA）
D → Distill   提炼核心见解（渐进式摘要）
E → Express   表达与创造（输出驱动）
```

### 与生物学大脑的对比

| 生物学大脑 | 第二大脑 |
|-----------|----------|
| 记忆不可靠 | 永久存储 |
| 容量有限 | 无限扩展 |
| 会遗忘 | 有版本历史 |
| 线性思考 | 网状链接 |
| 受情绪影响 | 客观记录 |

## 关键要点

1. **外化而非记忆** — 大脑是用来创造，不是用来存储
2. **行动性组织** — PARA 是核心组织方式
3. **输出驱动** — 只保存能帮助未来创造的信息
4. **渐进式摘要** — 从粗到细逐层提炼

### 渐进式摘要 (Progressive Summarization)

```
Layer 1: 原文摘录
Layer 2: 加粗关键句
Layer 3: 高亮核心词
Layer 4: 用自己的话总结
Layer 5: 重新组合创造
```
