# OKR (Objectives and Key Results)

## 定义

OKR 是 Intel 发明、Google 推广的目标管理框架。O (Objective) 定义想去哪里，KR (Key Results) 定义如何知道已到达。核心公式：**O = 定性方向，KR = 定量结果**。

## 核心原理

### OKR 公式

```
Objective (目标)   +   3-5 Key Results (关键结果)
"我想达成什么"           "如何衡量已达成"
定性、激励性              定量、可测量
```

### Google 十大 OKR 原则

1. 目标鼓舞人心
2. 目标有时限（通常是季度）
3. 3-5 个目标，每个 3-5 个 KR
4. KR 必须可衡量
5. KR 要有挑战性（信心指数 50-70%）
6. OKR 公开透明
7. 不与薪酬挂钩
8. 60-70% 由下而上
9. 不与任务清单混淆
10. 定期评分和复盘

## 关键要点

| 要素 | 目标 | 关键结果 |
|------|------|----------|
| **性质** | 定性的 | 定量的 |
| **回答** | 想去哪里？ | 到了没有？ |
| **数量** | 3-5 个 | 每个 O 配 3-5 个 |
| **挑战度** | 60-70% 信心 | 有挑战但不绝望 |

### 常见错误

| 错误 | 修正 |
|------|------|
| KR 是任务列表 | KR 必须是结果，不是行为 |
| 目标太保守 | 目标应该有适当挑战 |
| 数量太多 | 聚焦，3-5 个 O 足够 |
| 缺乏跟踪 | 每周更新进度 |

- 评分 (0.0-1.0) 或百分比
- 月度检查 + 季度复盘
