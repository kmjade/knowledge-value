# Zettelkasten (卡片盒笔记法)

## 定义

Zettelkasten 是德国社会学家 Niklas Luhmann 发明的知识管理方法，通过原子化笔记和密集链接构建知识网络。Luhmann 用此法在 30 年内发表了 70 本书和 400 多篇论文。

## 核心原理

### 五大原则

1. **原子化 (Atomicity)** — 每张卡片只记录一个概念
2. **链接优先 (Connectivity)** — 价值来自链接，而非卡片本身
3. **用自己话说 (Own Words)** — 内化知识的唯一途径
4. **保持关联 (Relevance)** — 只记与当前思考相关的内容
5. **信任系统 (Trust)** — 可放心的"忘掉"已记录的内容

### 三类笔记

```
💡 Fleeting Notes    → 临时想法，1-2 天内处理
📝 Literature Notes  → 阅读摘要，用自己的话写
📌 Permanent Notes   → 永久知识卡片，原子化+链接
```

## 关键要点

1. **每张卡片一个概念** — 原子化是链接的前提
2. **编号系统** — Luhmann 的分支编号 (1 → 1.1 → 1.1a)
3. **"意外相遇"** — 遍历链接时应产生新的洞见
4. **不用文件夹** — 链接替代分类层级

### Luhmann 编号

```
1 社会学理论
  1.1 社会系统
    1.1a 沟通概念
    1.1b 观察概念
  1.2 功能分化
```
