# Zettelkasten 知识库总览

> Zettelkasten（卡片盒笔记法）是德国社会学家 Niklas Luhmann 发明的笔记方法。
> 他用 9 万张手写卡片，产出了 70 本专著和 400+ 篇论文。
> 核心秘密不是"写得多"，而是"链接得巧"。

## 核心原则

| 原则 | 说明 |
|------|------|
| **原子性** | 一张卡片 = 一个想法 |
| **链接性** | 卡片之间互相关联 |
| **以自己的话写** | 不是摘抄，是理解后的重新表达 |
| **信任涌现** | 不预设结构，让结构从链接中自然生长 |

## 三类笔记

```
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│  闪念笔记     │  │  文献笔记     │  │  永久笔记     │
│  Fleeting     │  │  Literature   │  │  Permanent    │
│  临时捕获     │→ │  阅读摘录     │→ │  深度思考     │
│  1-2 天内处理 │  │  加入自己想法 │  │  永久保留     │
└──────────────┘  └──────────────┘  └──────────────┘
```

## Zettelkasten 的力量

> Luhmann 说："不是我写书，是我的卡片盒在写书。"

当一个主题下积累了足够多互相链接的卡片，写作就是——**整理已有的卡片，串成一条线**。
