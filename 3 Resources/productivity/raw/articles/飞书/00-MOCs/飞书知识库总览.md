# 飞书知识库总览

> 飞书（Feishu/Lark）是字节跳动出品的先进企业协作平台，集即时通讯、文档、多维表格、日历、会议、审批、自动化于一体。
> 2016 年内部孵化，2019 年对外发布。

## 知识库结构

- [[飞书学习路径]] — 从零到精通的推荐学习
- [[01-平台概览]] — 注册、版本、定价、国际版
- [[02-即时通讯]] — 消息、群组、频道、表情
- [[03-文档协作]] — 飞书文档、表格、思维笔记
- [[04-多维表格]] — Bitable 数据库（类 Airtable）

- [[07-机器人开发]] — 飞书 Bot、Webhook、消息卡片
- [[08-管理与安全]] — 管理员后台、权限、安全
- [[飞书资源收集]] — 官方、社区、开发者资源

## 为什么用飞书？

| 特性 | 说明 |
|------|------|
| **All-in-One** | 聊天 + 文档 + 表格 + 日程 + 会议 |
| **文档协作** | 多人实时编辑，评论/批注 |
| **多维表格** | 类 Airtable 的在线数据库 |
| **审批自动化** | 灵活的自定义审批流 |
| **Bot 开放平台** | 丰富的 API 和机器人框架 |
| **国际版 Lark** | 海外可用，数据合规 |

## 版本对比

| 版本 | 价格 | 适合 |
|------|------|------|
| **免费版** | ¥0 | 小团队（<50人） |
| **商业版** | ¥25/人/月 | 中小企业 |
| **企业版** | ¥50/人/月 | 大型企业 |
| **旗舰版** | 议价 | 超大型企业 |
