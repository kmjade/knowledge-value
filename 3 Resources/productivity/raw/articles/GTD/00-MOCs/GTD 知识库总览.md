# GTD 知识库总览

> GTD（Getting Things Done）是 David Allen 提出的时间管理方法论。
> 核心思想：**把大脑中的一切事项清空到外部系统，让大脑专注思考而非记忆。**

## GTD 五步流程

```
  收集 ──→ 理清 ──→ 整理 ──→ 回顾 ──→ 执行
  Capture  Clarify  Organize  Reflect   Engage
    ↑                                      │
    └──────────────────────────────────────┘
                 ↻ 持续循环
```

## 为什么学 GTD？

| 收益 | 说明 |
|------|------|
| **减压** | 大脑不再囤积待办，焦虑下降 |
| **清晰** | 清楚知道下一步做什么 |
| **掌控** | 对工作和生活更有掌控感 |
| **高效** | 减少决策疲劳，聚焦重要事项 |
| **信任系统** | 不再依赖记忆力，信任外部系统 |
