# 07 - GTD 工具实践

## 原则

> 最好的工具是你真正会用的工具。

## 方案 A：Obsidian GTD

### 插件组合

```
Tasks       → 任务管理
Calendar    → 日历视图
Dataview    → 汇总查询
QuickAdd    → 快速捕获
Templater   → 模板
```

### 目录结构

```
GTD/
├── 0-收集篮.md          ← QuickAdd 一键捕获
├── 1-下一步行动.md       ← @电脑/@手机/...
├── 2-项目清单.md         ← 每个项目的下一步
├── 3-等待清单.md
├── 4-将来也许.md
├── 5-每周回顾.md
└── 参考资料/
```

### 模板：下一步行动

```markdown
---
tags: [gtd/next-action]
context: "{{context}}"    # @电脑 @手机 @办公室
energy: "{{energy}}"      # 高/中/低
created: {{today}}
---

# {{title}}

- [ ] {{action}} 📅 {{due}}

---
项目：[[project]]
```

### Dataview 查询下一步行动

```dataview
TASK
FROM "GTD"
WHERE !completed AND contains(tags, "next-action") AND context = "@电脑"
GROUP BY context
SORT created ASC
```

## 方案 B：Notion GTD

### 数据库结构

```
Tasks 数据库
├── 属性：任务名、状态、情境、项目、截止日、优先级
├── 关联 → Projects 数据库
├── 看板视图：按状态
└── 分组视图：按情境

Projects 数据库
├── 属性：项目名、状态、截止日
├── 汇总 → 任务数、完成率
└── 看板视图
```

### 收集模板

```
页面模板 "快速捕获"
  └── 标题：{{输入}}
  └── 状态：待理清
  └── 自动放入 Inbox 数据库
```

## 方案 C：飞书 GTD

### 多维表格

```
任务表
├── 字段：任务名、状态、情境、项目、截止日、负责人
├── 表单视图 → 快速收集
├── 看板视图 → 按状态
├── 画廊视图 → 按情境
└── 自动化 → 到期提醒
```

### 飞书自动化

```
触发：任务截止日 = 今天
动作：发送飞书消息提醒
```

## 方案 D：纸笔 GTD

```
工具：
├── A5 活页本
├── 标签贴纸（@电脑 @手机 @办公室）
└── 每月/每周模板打印

最适合：减少屏幕时间、喜欢手写、简单优先
```

## 选择建议

```
想要灵活 + 本地 → Obsidian
想要协作 + 数据库 → Notion
团队使用 + 国内 → 飞书
想要简单 + 免费 → 纸笔 / Microsoft To Do
```
