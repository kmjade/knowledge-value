# OKR 知识库总览

> OKR（Objectives and Key Results）是目标管理框架，由 Intel 的 Andy Grove 发明，John Doerr 引入 Google 并发扬光大。
> 核心理念：**设定雄心勃勃的目标，用可量化的关键结果追踪进展。**

## OKR 公式

```
O（目标）+ KR（关键结果）= OKR

O = 我要去哪里？（定性的、鼓舞人心的）
KR = 我怎么知道我到了？（定量的、可衡量的）
```

## OKR 的超级力量

| 力量 | 说明 |
|------|------|
| **聚焦** | 一次只做 3-5 件最重要的事 |
| **对齐** | 上下左右目标一致 |
| **追踪** | 每周检查进展，不靠记忆 |
| **挑战** | 设定超出舒适区的目标 |

## OKR 经典例子

```
O：打造行业最好的移动端体验

KR1：应用商店评分从 3.8 提升到 4.5
KR2：页面加载时间从 3 秒降到 1.5 秒
KR3：NPS 从 30 提升到 60
KR4：崩溃率低于 0.1%
```

## OKR vs KPI

| | OKR | KPI |
|---|-----|-----|
| 性质 | 目标 + 路径 | 健康指标 |
| 频率 | 季度 | 持续 |
| 挑战性 | 高（60-70% 完成即好） | 期望 100% |
| 用途 | 推动变革 | 监控运行 |
| 关联 | — | 奖金/绩效 |
