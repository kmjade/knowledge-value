# 05 - Obsidian 高级技巧

## Templater 模板引擎

安装 Templater 插件后，创建模板：

```markdown
---
title: <% tp.file.title %>
tags: []
created: <% tp.date.now("YYYY-MM-DD") %>
---

# <% tp.file.title %>

## 内容

<% tp.file.cursor() %>

## 相关笔记

- [[相关笔记]]
```

**常用 Templater 命令**：

| 命令 | 效果 |
|------|------|
| `tp.date.now("YYYY-MM-DD")` | 当前日期 |
| `tp.file.title` | 文件标题 |
| `tp.file.cursor()` | 光标位置 |
| `tp.file.creation_date()` | 文件创建日期 |
| `tp.file.move()` | 移动文件 |
| `tp.system.prompt()` | 弹出输入框 |

## Dataview 查询引擎

### 列出所有含某标签的笔记

````markdown
```dataview
LIST
FROM #ai
SORT file.ctime DESC
```
````

### 表格视图

````markdown
```dataview
TABLE title, created, tags
FROM "3 Resources"
WHERE contains(tags, "markdown")
SORT created DESC
LIMIT 10
```
````

### 任务汇总

````markdown
```dataview
TASK
FROM "1 Projects"
WHERE !completed
GROUP BY file.link
```
````

### Dataview 常用函数

| 函数 | 说明 |
|------|------|
| `file.link` | 笔记链接 |
| `file.ctime` | 创建时间 |
| `file.mtime` | 修改时间 |
| `file.tags` | 所有标签 |
| `file.folder` | 所在文件夹 |
| `contains(list, item)` | 列表包含检查 |

## Canvas 白板

创建 `.canvas` 文件 → 可视化布局。

```
Canvas 支持：
├── 笔记卡片（嵌入 md 笔记）
├── 图片卡片
├── 网页嵌入
├── 连线/箭头
└── 分组框
```

**用例**：
- 头脑风暴
- 项目规划图
- 知识地图
- PPT 替代

## CSS 片段

`.obsidian/snippets/` 下放 `.css` 文件 → 设置中启用。

**常用片段示例**：

```css
/* 加宽编辑区 */
.markdown-source-view { max-width: 900px; }

/* 隐藏属性面板 */
.metadata-container { display: none; }

/* 更改链接颜色 */
.cm-hmd-internal-link { color: #4a9eff; }
```

## QuickAdd 快速捕获

配置 QuickAdd → 一键创建笔记、添加任务、调用模板。

```
QuickAdd 可以：
├── 捕获灵感（弹出输入框 → 写入指定笔记）
├── 模板调用（一键套用模板）
├── 宏命令（组合多个操作）
└── 日记模板（每日自动创建）
```

## 多仓库管理

```
~/
├── Vault-Personal/       ← 个人知识库
├── Vault-Work/           ← 工作知识库
└── Vault-Study/          ← 学习笔记
```

> 一个 Obsidian 窗口可以打开多个仓库 → `Ctrl+Shift+N`
