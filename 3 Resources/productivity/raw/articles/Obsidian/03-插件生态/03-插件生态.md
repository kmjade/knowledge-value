# 03 - Obsidian 插件生态

## 开启社区插件

设置 → 社区插件 → 关闭「安全模式」

> ⚠️ 安全模式阻止运行第三方代码，关闭后即可安装插件。

## 必装插件（按场景）

### 核心增强

| 插件 | 功能 | 优先级 |
|------|------|--------|
| **Templater** | 高级模板引擎 | ⭐⭐⭐⭐⭐ |
| **QuickAdd** | 快速捕获/宏命令 | ⭐⭐⭐⭐⭐ |

## 插件安装方式

1. 设置 → 社区插件 → 浏览 → 搜索插件名 → 安装 → 启用
2. 或者通过 BRAT 插件安装 Beta 版本

## 插件安全

| 注意点 | 说明 |
|--------|------|
| **开源审查** | 优先选择 GitHub Star 多的插件 |
| **权限最小** | 只装真正需要的插件 |
| **定期更新** | 过时插件可能有安全风险 |
| **备份先行** | 大规模装插件前先备份仓库 |
