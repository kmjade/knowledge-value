## 为什么用 Notion？

| 特性 | 说明 |
|------|------|
| **All-in-One** | 笔记 + 数据库 + 项目管理 + Wiki |
| **Block 架构** | 一切皆 Block，自由拖拽组合 |
| **数据库** | 类 Airtable 的强大数据库 |
| **协作** | 实时多人编辑，评论系统 |
| **模板** | 海量社区模板，开箱即用 |
| **AI** | 内置 AI 写作/翻译/总结 |
| **跨平台** | Web / Win / Mac / iOS / Android |
