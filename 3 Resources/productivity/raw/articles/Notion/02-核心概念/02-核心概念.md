# 02 - Notion 核心概念

## Block（块）

**一切皆 Block**。文字、图片、列表、数据库……都是 Block。

### 常用 Block 类型

| 类型 | 命令 | 说明 |
|------|------|------|
| 文本 | 直接输入 | 普通段落 |
| 标题 | `/h1` `/h2` `/h3` | 三级标题 |
| 列表 | `/bullet` `/numbered` | 无序/有序 |
| 待办 | `/todo` | 任务清单 |
| 折叠 | `/toggle` | 可展开折叠块 |
| 引用 | `/quote` | 引用块 |
| 分割线 | `/div` | 水平分割线 |
| 代码 | `/code` | 代码块（支持语法高亮） |
| 图片 | `/image` | 上传或链接图片 |
| 视频 | `/video` | 嵌入视频 |
| 书签 | `/bookmark` | URL 书签预览 |
| 文件 | `/file` | 上传附件 |
| 表格 | `/table` | 简单表格 |
| 数据库 | `/database` | 数据库（下一节详解） |
| 数学 | `/equation` | KaTeX 数学公式 |
| 面包屑 | `/bread` | 路径导航 |
| 子页面 | `/page` | 内嵌子页面 |

### Callout 标注

```text
/callout    蓝色信息框
/red         红色背景
/orange      橙色背景
/yellow      黄色背景
/green       绿色背景
```

### 列布局

```
拖拽 Block → 拖到另一个 Block 的右侧 → 自动并排
```

最多支持 5 列布局。

## 页面（Page）

```
页面 = Block 的容器
```

- 每个页面有独立图标（emoji / 自定义上传）
- 页面可以无限嵌套（页面套页面）
- 页面有面包屑导航

## 数据库（Database）

Notion 最强大的功能，类似 Airtable。

### 数据库创建方式

```
/inline     → 内嵌数据库（在页面内）
/full page  → 独立全页数据库
```

### 六种视图

| 视图 | 命令 | 适用场景 |
|------|------|----------|
| **表格** | Table | 数据一览 |
| **看板** | Board | 项目管理 |
| **时间线** | Timeline | 甘特图 |
| **日历** | Calendar | 日程安排 |
| **列表** | List | 简化清单 |
| **画廊** | Gallery | 图片/卡片展示 |

### 属性类型

| 类型 | 说明 |
|------|------|
| 文本 | 单行文本 |
| 数字 | 数值 |
| 选择 | 单选下拉 |
| 多选 | 多选标签 |
| 日期 | 日期 + 提醒 |
| 人员 | 分配成员 |
| 文件 | 上传附件 |
| 复选框 | 是/否 |
| URL | 链接 |
| 邮箱 | 邮箱地址 |
| 电话 | 电话号码 |
| 公式 | 计算字段 |
| 关联 | 关联其他数据库 |
| 汇总 | 从关联数据库汇总 |
| 创建时间 | 自动时间戳 |
| 最后编辑 | 自动更新时间 |

## 工作区（Workspace）

```
工作区
├── 私有页面
├── 共享页面
└── 团队空间（Teamspace）
```

> 免费版可创建 1 个私有工作区 + 无限共享页面。
