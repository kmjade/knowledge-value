> UDC (Universal Decimal Classification) is a system for classifying books and documents, invented by Melvil Dewey.

## 🎯 什麼是 UDC？What is UDC？

### 起源 History

- **發明人**: Melvil Dewey（梅爾維爾·杜威）
- **年份**: 1876 年
- **原名**: Decimal Classification（十進分類法）
- **廣泛使用**: 全球超過 150 個國家的圖書館

### 特點 Features

1. **十進制結構** Decimal System
   - 使用 0-9 十個數字
   - 每個數字代表一個大類
   - 可以無限細分

2. **層級結構** Hierarchical Structure
   ```
   6 應用科學
   62 工業技術
   621 機械工程
   621.3 電氣工程
   621.31 電機工程
   ```

3. **組配原則** Synthesis
   - 可以使用輔助符號組合
   - 支持多面向檢索

## 🔧 使用指南 Usage Guide

### 分類原則 Classification Principles

1. **主題優先** - 選擇主要主題
2. **最細原則** - 使用最詳細的分類
3. **組配原則** - 使用輔助符號

### 輔助符號 Auxiliary Symbols

| 符號 | 用法 | 示例 |
|------|------|------|
| `:` | 關係組配 | 62:621 工業技術與機械工程 |
| `/` | 範圍 | 62/69 應用科學全部類目 |
| `=` | 語言 | =20 表示英語 |
| `(...)` | 地區 | (510) 中國 |
