## 核心内容摘要

### 基础理论
- **定义**：耳穴疗法是指在耳廓特定反应点进行刺激，通过经络传导防治疾病的治疗方法
- **理论基础**：基于"耳为宗脉之所聚"理论和全息反射原理
- **历史发展**：从《黄帝内经》记载到现代标准化发展
- **现代认识**：结合神经解剖学、生理学等现代医学理论

### 耳穴定位
- **标准耳穴**：93个国家标准耳穴
- **分布规律**：按倒置胎儿分布，与身体部位相对应
- **定位方法**：解剖定位、比例定位、反应点定位
- **主治功能**：按系统、功效、脏腑分类主治

### 刺激方法
- **主要方法**：毫针针刺、耳穴按压、耳穴按摩、耳穴电针、耳穴放血等
- **方法选择**：根据病情、患者体质、治疗目的选择合适方法
- **操作规范**：严格的无菌操作、准确的穴位定位、适宜的刺激强度
- **治疗参数**：刺激时间、频次、疗程的科学设置

### 临床应用
- **主要适应症**：
  - 神经系统疾病：失眠、头痛、癫痫
  - 精神心理疾病：抑郁、焦虑、神经衰弱
  - 心血管系统疾病：高血压、冠心病
  - 呼吸系统疾病：哮喘、支气管炎
  - 消化系统疾病：胃炎、溃疡、便秘
  - 妇科疾病：痛经、月经不调、更年期综合征
  - 骨伤科疾病：颈椎病、腰椎间盘突出
  - 五官科疾病：近视、鼻炎、耳鸣
- **疗效特点**：安全性高、适应症广、疗效确切、患者依从性好

### 安全性
- **总体安全性**：高安全性治疗技术，不良反应率<3%
- **禁忌症**：耳部感染、肿瘤、严重出血性疾病等
- **注意事项**：特殊人群（孕妇、儿童、老年人）需谨慎使用
- **应急处理**：制定完善的不良反应处理预案
