### Obsidian Markdown 語法

| 語法 | 說明 |
|------|------|
| `[[Note]]` | 內向連結 |
| `![[Note]]` | 嵌入連結，不顯示 |
| `# Heading` | 標題 |
| `## Heading` | 二級標題 |
| `**text**` | 粗體 |
| `` `code` `` | 行內代碼 |
| `> [!info]` | 呼叫塊 |
