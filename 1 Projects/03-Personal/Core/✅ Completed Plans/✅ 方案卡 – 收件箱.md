> [!question] 方案概述
> 本方案旨在優化收件箱的捕獲與搬移流程，透過 QuickAdd 宏自動將完成的筆記搬至「✅ Completed Plans」資料夾，提升資訊整理效率。

> [!tip] 使用說明
> 1. 勾選 ✅ 清單時，直接在 Obsidian 中點擊方框即可自動打勾。  
> 2. 完成所有任務後，將 `status` 改為 `completed`，或自行新增 `completed: true` 欄位。  
> 3. 若已安裝 QuickAdd，可使用下方提供的「搬移已完成方案卡」宏自動搬移。

### 使用小技巧
1. **快速勾選**：在 Obsidian 中點擊 `[ ]` 方框即可自動變成 `[x]`，同時 `status` 欄位不會自動變更，若想同步，完成所有 ✅ 清單後手動把 `status: draft` 改為 `status: completed`。  
2. **自動搬移宏範例**（直接 copy‑paste 到 QuickAdd 設定）：

```json
{
  "name": "搬移已完成方案卡",
  "type": "macro",
  "command": "quickAdd",
  "args": {
    "macroName": "Move Completed Plan",
    "file": "{{file.path}}",
    "targetFolder": "0 Personals/✅ Completed Plans"
  }
}

let pages = dv.pages('"0 Personals/📥 00_InBox"')
               .where(p => p.tags && p.tags.includes('✅'));
dv.table(['狀態','未完成任務'],
         pages.map(p => [p.status, p.file.tasks.filter(t=>!t.completed).length]));
         
```

3. **Dataview 追蹤**（已修正語法）：

```text
TABLE status, length(filter(file.tasks, t => !t.completed)) AS "未完成任務"
FROM "0 Personals/📥 00_InBox"
WHERE contains(file.tags, "✅")
SORT file.ctime DESC
```

> 若想使用 **DataviewJS**，可改寫為：

```javascript
let pages = dv.pages('"0 Personals/📥 00_InBox"')
               .where(p => p.tags && p.tags.includes('✅'));
dv.table(['狀態','未完成任務'],
         pages.map(p => [p.status, p.file.tasks.filter(t=>!t.completed).length]));
```
