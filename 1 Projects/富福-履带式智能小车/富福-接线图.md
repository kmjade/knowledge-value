# 富福 v1 接線指南

## Arduino Mega 引腳分配

| 功能 | 引腳 | 說明 |
|------|:---:|------|
| **左电机 PWM** | D5 | 速度控制 (0–255) |
| **左电机 DIR** | D4 | 方向 |
| **右电机 PWM** | D6 | 速度控制 |
| **右电机 DIR** | D7 | 方向 |
| **左编码器 A** | D2 | 中断 INT0 |
| **左编码器 B** | D3 | 中断 INT1 |
| **右编码器 A** | D18 | 中断 INT5 |
| **右编码器 B** | D19 | 中断 INT4 |
| **超声波 Trig** | D22/24/26 | 前/左/右 |
| **超声波 Echo** | D23/25/27 | 前/左/右 |
| **UART TX** | TX1 (D18) | → Pi 5 RX (GPIO15) |
| **UART RX** | RX1 (D19) | ← Pi 5 TX (GPIO14) |
| **电压检测** | A0 | 电池分压读取 |

## Raspberry Pi 5 GPIO

| 功能 | GPIO | 说明 |
|------|:---:|------|
| **UART TX** | GPIO14 (Pin 8) | → Arduino RX |
| **UART RX** | GPIO15 (Pin 10) | ← Arduino TX |
| **I2C SDA** | GPIO2 (Pin 3) | → IMU (MPU6050) |
| **I2C SCL** | GPIO3 (Pin 5) | → IMU (MPU6050) |
| **I2C SDA (第二路)** | GPIO0 (Pin 27) | → INA219 |
| **I2C SCL (第二路)** | GPIO1 (Pin 28) | → INA219 |

## USB 设备分配

| 设备 | 接口 | 说明 |
|------|:---:|------|
| **LiDAR** | USB 3.0 Port 1 | `/dev/ttyUSB0` |
| **Camera** | USB 3.0 Port 2 | `/dev/video0` |
| **Arduino Mega** | USB (编程用) | 调试/烧录时连接 |

## 电源拓扑

```
12V 锂电池 (3S 5000mAh)
    │
    ├──→ TB6612 电机驱动 (12V 直供)
    │       ├──→ 左电机
    │       └──→ 右电机
    │
    ├──→ mini560 降压 (12V→5V)
    │       ├──→ Raspberry Pi 5 (5V 3A via GPIO Pin 2/4)
    │       ├──→ Arduino Mega (5V via Vin)
    │       ├──→ LiDAR (USB 5V)
    │       └──→ YDLIDAR X4 (5V)
    │
    └──→ 电压分压电路 (12V→3.3V for A0) → Arduino ADC
```

## 接线检查清单

- [ ] 电机驱动电源端子拧紧（大电流）
- [ ] 所有 GND 共地 (电池 GND = 降压 GND = Pi GND = Arduino GND)
- [ ] 编码器 AB 相接线正确（A→中断引脚，B→普通IO）
- [ ] UART 交叉连接 (Pi TX → Arduino RX, Pi RX → Arduino TX)
- [ ] UART 电平匹配 (Pi 3.3V, Arduino 5V — 使用电平转换或直接连接，Arduino 5V 可容忍 Pi 3.3V 输出)
- [ ] 急停开关串联在电池正极
- [ ] 保险丝接在电池正极最前端
