## 二、硬件方案

### 2.1 底盘 — 履带式

> **推荐方案 A** — UGV01 履带底盘，独立悬挂越野能力强，已有成熟驱动方案。

### 2.2 电机与驱动

| 组件 | 型号 | 参数 | 数量 |
|------|------|------|:---:|
| **直流减速电机** | JGB37-520 / MG513 | 12V, 300–500RPM, 编码器 | 2 |
| **电机驱动** | L298N / TB6612 / BTS7960 | 7–12V, 双路 H 桥 | 1 |
| **编码器** | 霍尔 AB 相 | 11/13 PPR (电机内置) | 2 |

### 2.3 主控制器

| 层级 | 方案 | 说明 |
|------|------|------|
| **上位机** | Raspberry Pi 5 (8GB) | ROS2 Humble, Nav2, 视觉 |
| **下位机** | Arduino Mega / STM32F103 | PID 闭环, 编码器读取, IMU |

> **上位机备选**：RDK X3（地平线，5 TOPS BPU AI 加速）— 如需端侧 AI 推理

**上下位机分工**：
- **上位机 (Pi 5)**：ROS2 节点、Nav2 导航、SLAM、摄像头处理、Web 控制面板
- **下位机 (Arduino/STM32)**：电机 PID 速度闭环、编码器里程计、IMU 读取、超声波避障

### 2.4 视觉云台 — Pan-Tilt Camera

> 🆕 选用 **Waveshare 2-Axis Pan-Tilt Camera Module** — 与 Pi 5 深度整合的二自由度摄像云台。

| 参数 | 值 |
|------|-----|
| **自由度** | 2-DOF：PAN ±180° (360°) / TILT -45°~90° (135°) |
| **舵机** | ST3215 总线舵机 ×2，30kg.cm 堵转扭矩 |
| **角度传感器** | 12位 360° 磁编码器（闭环控制） |
| **下位机** | ESP32-D0WD-V3 |
| **上位机接口** | 40PIN GPIO 直插 Pi 5（UART 通信） |
| **板载 IMU** | QMI8658 (6轴) + AK09918C (3轴罗盘) |
| **板载驱动** | TB6612FNG 电机驱动芯片 |
| **供电** | 12V（与底盘统一电源） |
| **重量** | 441g |
| **摄像头** | Pi Camera 3 / USB Camera |
| **开源** | ESP32 固件 + Pi 主程序 + STEP 模型 + 原理图 |

**功能亮点**：
- Web 浏览器控制界面（实时画面 + 云台操控）
- AI 检测：人脸检测、运动检测、颜色识别、手势识别
- 云台自稳：IMU 自动水平校正
- JSON 指令集控制
- LED 补光灯、拍照/录像/Photo Gallery
- 板载 LiDAR 接口

**与富福的整合方式**：
- 40PIN 直插 Pi 5 GPIO，无需额外接线
- 云台 ESP32 通过 GPIO UART 与 Pi 5 通信
- 底盘 Arduino 通过 USB/UART 与 Pi 5 通信
- 云台和底盘各有一个下位机，Pi 5 统一调度

```
Pi 5 (ROS2)
  ├── /dev/ttyAMA0 (GPIO UART) → Pan-Tilt ESP32 (云台控制)
  ├── /dev/ttyUSB0            → Arduino Mega (底盘控制)
  ├── /dev/ttyUSB1            → YDLIDAR X4
  └── USB Camera / Pi Cam     → 摄像头
```

## 三、软件架构

### 3.1 系统栈

```
┌─────────────────────────────────┐
│        应用层 Application        │
│  Web控制 / 自主导航 / 视觉巡线   │
├─────────────────────────────────┤
│        导航层 Navigation         │
│  Nav2 (Planner/Controller/AMCL) │
├─────────────────────────────────┤
│         ROS2 中间件              │
│  Node/Topic/Service/TF2         │
├─────────────────────────────────┤
│        驱动层 Driver             │
│  电机驱动 / LiDAR / IMU / Cam   │
├─────────────────────────────────┤
│        硬件层 Hardware           │
│  Pi 5 + Arduino + 传感器        │
└─────────────────────────────────┘
```

### 3.2 ROS2 节点设计

| 节点 | 功能 | Topic/Service |
|------|------|---------------|
| `/fufu_base` | 底盘控制 (Arduino 串口) | `/cmd_vel` → serial |
| `/fufu_odom` | 里程计发布 | `/odom` (编码器+IMU融合) |
| `/fufu_lidar` | LiDAR 驱动 | `/scan` |
| `/fufu_pantilt` | 🆕 云台控制 (ESP32 串口) | `/pantilt/cmd` → JSON |
| `/fufu_camera` | 摄像头图像 | `/image_raw` |
| `/fufu_ultrasonic` | 超声波距离 | `/ultrasonic` |
| `/fufu_teleop` | 键盘/手柄遥控 | → `/cmd_vel` |
| `/fufu_nav2` | Nav2 自主导航 | Nav2 标准接口 |

### 3.3 通信协议

**底盘通信** — UART 115200bps (Pi 5 ↔ Arduino), JSON：

```json
// 上位机 → 下位机（控制指令）
{"cmd": "vel", "left": 0.5, "right": 0.5}
{"cmd": "stop"}
{"cmd": "pid", "kp": 1.2, "ki": 0.3, "kd": 0.05}

// 下位机 → 上位机（状态回传）
{"enc_left": 1234, "enc_right": 1230}
{"imu": {"ax": 0.1, "ay": 0.0, "az": 9.8, "gx": 0.01, "gy": 0.0, "gz": 0.0}}
{"battery": 11.8}
{"ultrasonic": [45, 120, 60]}
```

**云台通信** — GPIO UART (Pi 5 ↔ Pan-Tilt ESP32), JSON：

```json
// 云台基础控制 — 设定绝对角度
{"T":133,"X":90,"Y":30,"SPD":50,"ACC":10}
// T=133: CMD_GIMBAL_CTRL_SIMPLE
// X: Pan角度 (-180~180), Y: Tilt角度 (-30~90)
// SPD: 速度(0=最快), ACC: 加速度(0=最快)

// 云台连续控制 — 平滑转动
{"T":134,"X":90,"Y":0,"SX":30,"SY":20}
// T=134: CMD_GIMBAL_CTRL_MOVE
// SX/SY: X/Y轴速度

// 云台停止
{"T":135}

// 云台自稳 — IMU水平校正
{"T":136,"s":1,"y":0}
// s=1: 开启自稳, y: 目标与地面夹角

// UI摇杆控制
{"T":137,"X":1,"Y":0,"SPD":80}
// X: -1(左)/0(停)/1(右), Y: -1(下)/0(停)/1(上)

// LED灯控制
{"T":132,"IO5":255}
// 0=灭, 255=最亮
```

## 四、组装步骤

### Phase 1：底盘与动力 (Day 1–2)

1. 组装履带底盘框架
2. 安装电机 + 编码器到框架
3. 安装驱动轮 + 履带
4. 连接电机驱动板 → 电机
5. 下位机 (Arduino) 烧录电机测试代码
6. **验证**：遥控前进/后退/转向正常

### Phase 2：电源与主控 (Day 2–3)

1. 安装电池 + 降压模块
2. 固定 Pi 5 + Arduino 到扩展板
3. 连接上下位机 UART
4. 配置 Pi 5：烧录 Raspberry Pi OS + ROS2 Humble
5. **验证**：上下位机通信正常

### Phase 3：传感器与云台 (Day 3–5)

1. 安装 Pan-Tilt 云台模块到车体前部
2. 40PIN 直插 Pi 5 GPIO
3. 安装 Pi Camera 到云台
4. 安装 LiDAR 在顶部
5. 安装超声波×3 (前/左/右)
6. 烧录云台 ESP32 固件
7. 配置 Pi 5：禁用蓝牙（释放 GPIO UART）
6. **验证**：所有传感器数据可读取

### Phase 4：ROS2 集成 (Day 4–6)

1. 创建 ROS2 workspace
2. 编写底盘控制节点 (`/fufu_base`)
3. 实现里程计 (`/fufu_odom`)
4. 集成 LiDAR/IMU/Camera 驱动
5. 实现键盘/手柄遥控
6. **验证**：`ros2 topic list` 所有话题正常

### Phase 5：自主导航 (Day 6–8)

1. 配置 Nav2 参数 (针对履带底盘调参)
2. SLAM 建图 (手动遥控 + cartographer/slam-toolbox)
3. 保存地图
4. AMCL 定位 + 路径规划
5. **验证**：给定目标点，小车自主导航到达

## 五、关键参数

### 5.1 运动学模型

```
履带差速模型 (Differential Drive)：

        v = (v_left + v_right) / 2
        ω = (v_right - v_left) / track_width

其中 track_width = 两履带中心距 (~200mm for UGV01)
```

### 5.2 PID 参数 (初始值)

| 参数 | 值 | 说明 |
|------|:---:|------|
| Kp | 1.2 | 比例增益 |
| Ki | 0.3 | 积分增益 |
| Kd | 0.05 | 微分增益 |
| 编码器 PPR | 11 | 电机每圈脉冲数 |
| 减速比 | 1:30 | JGB37-520 |
| 轮径 | 60mm | 驱动轮直径 |

### 5.3 预期性能

| 指标 | 目标值 |
|------|:---:|
| 最高速度 | 0.5 m/s |
| 续航 | 2–4 小时 |
| 负载 | 3 kg |
| 爬坡 | 15° |
| 控制延迟 | <50ms |
| 建图精度 | ±5cm |

## 六、控制方式

### 6.1 遥控模式

| 方式 | 实现 | 适用场景 |
|------|------|----------|
| **键盘** | `ros2 run teleop_twist_keyboard` | 开发调试 |
| **手柄** | `ros2 run joy joy_node` + Xbox/PS 手柄 | 实地操控 |
| **Web 面板** | Flask + rosbridge WebSocket | 手机浏览器遥控 |
| **手机 App** | ROS-Mobile (Android) | 移动端操控 |

### 6.2 自主导航模式

1. **SLAM 建图**：手动遥控小车遍历环境
2. **保存地图**：`ros2 run nav2_map_server map_saver_cli`
3. **自主导航**：设定目标点 → Nav2 规划路径 → 自主行驶
