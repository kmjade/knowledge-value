### 使用小技巧
1. **快速勾選**：在 Obsidian 中點擊 `[ ]` 方框即可自動變成 `[x]`，完成所有 ✅ 清單後手動把 `status: draft` 改為 `status: completed`，或在 QuickAdd 中加入「搬移至」宏自動搬到 `✅ Completed Plans`。  
2. **自動搬移**：在 QuickAdd 設定一個「完成」宏，動作 → *Move file to* → `0 Personals/✅ Completed Plans/`，保持收件箱整潔。  
3. **Dataview 追蹤**：在任意筆記加入以下查詢，即可彙總所有方案卡的進度：

```dataview
TABLE status,
      length(filter(file.tasks, (t) => !t.completed)) AS "未完成任務"
FROM "0 Personals/📥 00_InBox"
WHERE contains(file.tags, "✅")
SORT file.ctime DESC

```
