# 智谱 AI 配置实战

> **前置需求**: [[1 Projects/Claudian/Claudian 系统搭建指南.md]] 已完成系统安装
> **目标**: 完成智谱 AI API Key 获取和 CC Switch 配置

## 概述

本教程教你完成智谱 AI 平台的注册、认证、订阅，并配置 CC Switch，让 Claudian 能够调用国内智谱的 AI 服务。

**为什么选择智谱？**
- ✅ 国内访问无需梯子
- ✅ 响应速度快
- ✅ 性价比高（¥40/月，调用不限）
- ✅ 兼容 Claude Code 接口

---

## STEP 4: 注册智谱平台 & 获取 API Key

### 4.1 注册账号

**开放平台地址**
https://open.bigmodel.cn

**推荐方式**: 微信扫码登录（最方便）

**注册步骤**
1. 打开平台网站
2. 点击微信扫码登录
3. 扫码后绑定手机号
4. 完善信息（可跳过无关项）

> [!info] 提示
> 某些页面可直接点「跳过」，无需填写全部信息

### 4.2 实名认证

**必要性**
- 智谱平台要求实名认证后才能使用付费服务
- 流程简单，与微信实名认证类似

**认证方式**
- 刷脸认证（推荐）
- 身份证认证

> [!warning] 注意
> 实名认证是必须步骤，否则无法订阅套餐

### 4.3 订阅套餐

**推荐套餐: GLM Coding Lite**

| 项目 | 费用 |
|-----|------|
| 首月 | ¥20 |
| 次月起 | ¥40/月 |
| 调用次数 | 不限 |

**订阅步骤**
1. 进入套餐选择页面
2. 选择「GLM Coding Lite」套餐
3. 扫码付款
4. 等待订阅激活

**查看订阅状态**
- 付款成功后在平台可查看订阅详情
- 确认状态为「已激活」

### 4.4 复制 API Key

**创建 API Key**
1. 点击「添加新的 API Key」
2. 系统生成密钥字符串
3. 点击复制按钮

> [!danger] 重要安全提示
> ⚠️ **妥善保存 API Key**
> - 这是访问 AI 服务的唯一凭证
> - 不要泄露给他人
> - 建议保存到密码管理器
> - 后续配置会用到

---

## STEP 5: 在 CC Switch 里配置智谱

### 5.1 添加智谱

**操作步骤**
1. 打开 CC Switch 应用
2. 点击右上角的 **+** 号
3. 选择「Claude Code 供应商」
4. 点击 **Zhipu GLM**

### 5.2 填入 API Key

**配置项**
| 字段 | 值 |
|-----|---|
| API Key | 刚才复制的密钥字符串 |
| 主模型 | `glm-4.7` |

**配置步骤**
1. 将 API Key 粘贴到对应字段
2. 在主模型字段填入 `glm-4.7`
3. 点击「保存」

🎉 **配置完成！** 此时 CC Switch 已准备好为 Claudian 提供 AI 服务。

## STEP 6: MCP 服务器（已预配置）

### MCP 是什么？

**简单理解**
- 智谱 AI 模型虽然聪明，但它原生能力有限
- 看不了图片、上不了网、读不了网页
- MCP 服务器就是给 AI 装上「眼睛」和「网线」

### 已配置的 4 个 MCP 服务器

#### 1. zai-mcp-server
- **功能**: 视觉理解
- **能力**: 看懂截图、图片，甚至分析视频

#### 2. web-search-prime
- **功能**: 联网搜索
- **能力**: 获取实时信息、搜索最新资料

#### 3. web-reader
- **功能**: 网页读取
- **能力**: 抓取网页内容、提取结构化数据

#### 4. zread
- **功能**: 开源仓库
- **能力**: 访问 GitHub 等开源项目的文档和代码

### 如何使用 MCP

**调用方式**
在 Claudian 对话框里输入 `@` 即可呼出 MCP 列表，选择你需要的能力。

**状态**
✅ 打开仓库就能用，无需额外配置

### 6.1 如何自行添加 MCP

**方法一: 手动添加**
1. 点击左下角小齿轮（设置）
2. 打开「Claudian」→「MCP 服务器」
3. 添加新的 MCP 配置

**方法二: 对话添加**
直接让 Claudian 帮你添加：
```
帮我添加一个新的 MCP 服务器...
```

> [!info] 详细说明
> 更多 MCP 服务器介绍见 [[2 Areas/Claudian/Claudian MCP 服务器.md]]

## 验证配置

### 测试 AI 连接

1. 打开 Obsidian
2. 打开 Claudian 对话框
3. 输入测试命令：
```
/obsidian 你好，请介绍一下你自己
```

如果收到正常回复，说明配置成功！

### 常见问题排查

| 问题 | 可能原因 | 解决方案 |
|-----|---------|---------|
| AI 无响应 | API Key 错误 | 检查 CC Switch 中 API Key 是否正确 |
| 响应很慢 | 网络问题 | 检查网络连接 |
| 提示未订阅 | 订阅未激活 | 登录智谱平台查看订阅状态 |
| MCP 不可用 | MCP 配置问题 | 检查 `.claude/mcp_servers.json` |
