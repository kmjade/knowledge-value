## 🛠️ 开发环境

### 硬件要求
- MicroROS-Board控制板
- 树莓派5 或 Jetson（作为ROS主控）
- USB转串口模块
- WiFi模块（板载）
- 电源供应（电池或适配器）

### 软件环境

```yaml
ESP32开发:
  IDE: VS Code + ESP-IDF插件
  工具链: ESP-IDF v5.x
  调试: flash-tool + 串口监视器

ROS2开发:
  系统: Ubuntu 22.04 LTS (虚拟机或树莓派)
  ROS版本: ROS2 Humble Hawksbill
  开发工具: VS Code + ROS2插件
  可视化: Rviz2 + Rqt

MicroROS:
  代理: micro-ros-agent
  组件: ESP32-micro-ros-component
```
