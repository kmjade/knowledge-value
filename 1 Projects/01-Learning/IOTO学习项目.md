## 学习内容

### 核心理念

- **PKM 2.0** - 从知识管理到知识生产
- **成果导向** - 以终为始，让产出目标决定输入
- **项目驱动** - 项目是认知防火墙，聚焦精力和时间
- **无感联结** - 自动归类，减少分类决策成本

### 系统构成

```
IOTO = 1套框架 + 5个插件

核心框架：项目驱动的极简文件夹结构
├── 0-辅助/IOTO    # 脚本、模板
├── 1-输入         # 原始资料、碎片笔记
├── 2-输出         # 闪念笔记、卡片笔记
├── 3-任务         # 项目任务清单
└── 4-成果         # 项目成果沉淀

基础插件：
- IOTO Update      # 框架部署中枢
- Framework Settings # 配置中心
- IOTO Dashboard   # 生产大屏

增强插件：
- Template Generator # 模板编辑器
- Sync Scripts Generator # 同步脚本
```

### 核心工作流

```
项目创建流程（低频）：
意图 → 命名 → 自动基建 → 动态扩展

笔记创建流程（高频）：
Alt+1 输入 → Alt+2 输出 → Alt+4 成果
```
