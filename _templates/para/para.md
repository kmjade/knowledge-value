> [!info] PARA 模板选择指南
> 使用这些模板快速创建新的 PARA 笔记。

## 模板说明

| 模板 | 用途 | 适用场景 |
|------|------|----------|
| **Project 模板** | 创建有明确目标的项目 | 需要完成的具体任务 |
| **Area 模板** | 创建长期责任领域 | 需要持续维护的责任 |
| **Resource 模板** | 创建学习资源 | 感兴趣的主题、文章、书籍 |
| **Archive 模板** | 归档内容 | 完成的项目或不再活跃的内容 |
| **Zettelkasten 模板** | 创建原子化笔记 | 构建知识网络、核心概念 |

## 如何使用

1. 在对应 PARA 文件夹中创建新笔记
2. 使用 Templater 插件应用模板
3. 或手动复制模板内容到新文件
4. 填写必要的元数据（创建日期、标签等）
5. 根据需要添加具体内容

## PARA 方法论快速参考

- **Projects** - 有明确开始和结束日期
- **Areas** - 没有截止日期的长期责任
- **Resources** - 持续感兴趣的话题
- **Archives** - 完成或不再活跃的项目
- **Zettels** - 原子化笔记，构建知识网络

> [!tip] 定期回顾
> - Projects: 每周回顾
> - Areas: 每月回顾
> - Resources: 按需回顾
> - Archives: 每季度清理
> - Zettels: 按需整理和链接
