# PARA 方法原子化指南

> 本指南將 PARA 方法拆分為原子化、可執行的最小單元，便於理解、學習和實施。

## 📚 指南概覽

### 原子化原則

**最小可執行單元**：
- ✅ **獨立性**：可單獨完成，不依賴其他任務
- ✅ **明確性**：有清晰的輸入和輸出
- ✅ **可測試性**：有明確的完成標準
- ✅ **不可分割性**：無法再拆分為更小單元

### 原子化策略
1. **按功能分層**：概念、流程、工具、模板分離
2. **按執行順序**：從收集到執行的完整鏈條
3. **按目標對象**：Projects、Areas、Resources、Archives 分別處理
4. **按使用場景**：日常、週度、月度分別定義
