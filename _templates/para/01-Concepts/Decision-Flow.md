# 任務目標
決定新資訊的歸屬類別，使用 PARA 三問測試進行快速分類

## PARA 決策流程（三問測試）

收到新資訊 → 立即提問並執行：

### 1️⃣ 需要行動嗎？

**問題**：這個資訊需要我採取具體行動嗎？

**判斷標準**：
- ✅ 有明確目標和期限 → **Projects**
- ❌ 否 → 繼續下一個問題

**Projects 特徵**：
- 有明確的完成目標
- 需要連續執行
- 有截止日期
- 產出具體成果

---

### 2️⃣ 影響生活領域嗎？

**問題**：這個資訊屬於我需要持續維護的生活領域嗎？

**判斷標準**：
- ✅ 需要持續維護 → **Areas**
- ❌ 否 → 繼續下一個問題

**Areas 特徵**：
- 沒有「完成」狀態
- 需要定期投入
- 是生活的重要領域
- 有最低標準需要達成

---

### 3️⃣ 有興趣學習嗎？

**問題**：這個資訊是我有興趣收集和學習的內容嗎？

**判斷標準**：
- ✅ 純粹收集資料 → **Resources**
- ❌ 否 → **刪除**

**Resources 特徵**：
- 沒有完成壓力
- 純粹收集資訊
- 可自由探索
- 未來可能使用

---

## 決策流程圖

```
收到新資訊
    ↓
1️⃣ 需要行動嗎？
    ├─ 是（有明確目標和期限）→ 🎯 Projects
    │
    └─ 否 → 繼續
        ↓
2️⃣ 影響生活領域嗎？
    ├─ 是（需要持續維護）→ 🔴 Areas
    │
    └─ 否 → 繼續
        ↓
3️⃣ 有興趣學習嗎？
    ├─ 是（純粹收集資料）→ 📚 Resources
    │
    └─ 否 → 🗑️ 刪除
```

---

## 決策原則

### 核心原則：即時處理
- **接收 1 分鐘內**：決定初步歸類
- **24 小時內**：完成基本整理
- **週內**：放入合適位置

### 避免「稍後處理」心理陷阱
- ❌ 「等一下再處理」
- ❌ 「這個可能有用的」
- ❌ 「先收集後再想」
- ❌ 「以後會用到」

## 快速決策檢查清單

### 判斷 Projects
- [ ] 有明確的完成目標？
- [ ] 需要連續執行？
- [ ] 有時間限制？
- [ ] 產出具體成果？

### 判斷 Areas
- [ ] 沒有「完成」狀態？
- [ ] 需要持續投入？
- [ ] 是生活的重要領域？
- [ ] 有最低標準需要達成？

### 判斷 Resources
- [ ] 沒有完成壓力？
- [ ] 純粹收集資訊？
- [ ] 可自由探索？
- [ ] 未來可能使用？

### 判斷是否刪除
- [ ] 已過時？
- [ ] 無任何價值？
- [ ] 重複內容？
- [ ] 不再相關？
