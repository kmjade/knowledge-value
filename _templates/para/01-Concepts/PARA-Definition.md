# 任務目標
理解 PARA 方法的定義、特徵和識別標準

## PARA 定義

| 字母 | 意義 | 描述 |
|------|------|------|
| **P** | Projects | 有明確目標和期限的項目 |
| **A** | Areas | 持續維護的生活領域 |
| **R** | Resources | 有興趣的主題資源 |
| **A** | Archives | 已完成或過時的內容 |

## 四個類別的核心特徵

### Projects（項目）
- ✅ **明確完成條件**：知道什麼時候算完成
- 🎯 **具體目標**：有可衡量的成果
- ⏰ **時間期限**：有截止日期
- 📋 **行動清單**：需要執行的具體任務
- 🔄 **暫時性**：完成後就結束

### Areas（領域）
- 🔄 **永續性**：需要持續投入維持
- 📊 **標準維持**：有最低標準需要達到
- 📅 **定期檢視**：需要定期監控和調整
- 🎯 **持續進步**：永遠可以更好

### Resources（資源）
- 🔍 **探索性**：沒有時效壓力，可以自由探索
- 📚 **累積性**：持續收集和整理
- 🎯 **參考性**：需要時可快速取用
- 💡 **啟發性**：提供靈感和學習機會

### Archives（歸檔）
- 📦 **非活躍**：從其他類別移出的不活躍內容
- 🗄️ **歷史記錄**：保留過去的工作和學習記錄
- 📋 **參考價值**：有時候可能需要查閱
- 🧹 **保持乾淨**：讓主系統保持精簡

## 識別標準

### 如何判斷內容屬於 Projects？
問自己：**"這需要連續行動才能完成嗎？"**
- 有明確的完成目標 → Projects
- 需要連續執行 → Projects
- 有時間限制 → Projects

### 如何判斷內容屬於 Areas？
問自己：**"這需要持續維護嗎？"**
- 沒有「完成」狀態 → Areas
- 需要定期投入 → Areas
- 是生活的重要領域 → Areas

### 如何判斷內容屬於 Resources？
問自己：**"這是純粹收集的資訊嗎？"**
- 沒有完成壓力 → Resources
- 可自由探索 → Resources
- 未來可能使用 → Resources

### 如何判斷內容屬於 Archives？
問自己：**"這已完成或不再活躍了嗎？"**
- 項目已完成 → Archives
- 領域已不再相關 → Archives
- 資源已過時 → Archives

## 範例對比

### Projects 範例
| 是 Projects | 不是 Projects | 區別說明 |
|-----------|--------------|----------|
| 完成季度銷售報告（截止：本週五） | 銷售報告範本收集 | 有明確截止日期 |
| 規劃年度家庭旅行（目標：7月出發） | 旅遊攻略資料 | 有具體目標和計劃 |
| 通過專業認證考試（考試日：3月15日） | 認證考試相關資料 | 有明確考試目標 |
| 整理居住環境（目標：月底完成） | 收納整理技巧 | 有完成條件 |

### Areas 範例
| 是 Areas | 不是 Areas | 區別說明 |
|---------|-----------|----------|
| 運動習慣維持（每週3次） | 準備馬拉松比賽 | 沒有完成狀態 |
| 財務規劃管理（每月審視） | 存錢買車目標 | 持續維護的責任 |
| 職業技能提升（持續學習） | 學習Python程式設計 | 長期的職業發展 |
| 家庭關係維護（定期聯繫） | 規劃家庭聚會 | 持續的關係投入 |

### Resources 範例
| 是 Resources | 不是 Resources | 區別說明 |
|-----------|--------------|----------|
| 攝影技巧筆記收藏 | 學習攝影課程 | 純粹收集資訊 |
| Python 程式設計資料庫 | 開發 Python 專案 | 參考資料 |
| 旅行攻略和心得 | 規劃具體旅行 | 靈感和學習資源 |
| 創意設計靈感收集 | 創設計專案 | 啟發性內容 |

### Archives 範例
| 是 Archives | 不是 Archives | 區別說明 |
|-----------|--------------|----------|
| 2023年已完成的專案 | 進行中的專案 | 已完成歸檔 |
| 過期的市場分析報告 | 最新的市場趨勢 | 過時內容 |
| 不再感興趣的愛好收藏 | 當前興趣愛好 | 非活躍內容 |

## 練習題目

### 練習 1
**情境**：準備參加下個月的英語認證考試

**判斷**：屬於哪個類別？
A. Projects
B. Areas
C. Resources
D. Archives

**答案**：A. Projects

**分析**：
- 需要連續行動：準備、複習、參加考試
- 有明確目標：通過認證考試
- 有截止日期：下個月
- 完成後就結束：考試完成後此任務結束

### 練習 2
**情境**：收集 Python 程式設計相關的文章和教學資源

**判斷**：屬於哪個類別？
A. Projects
B. Areas
C. Resources
D. Archives

**答案**：C. Resources

**分析**：
- 沒有完成壓力：沒有明確的學習期限
- 純粹收集資訊：只是收集參考資料
- 可自由探索：可以隨時查看和學習
- 未來可能使用：將來需要參考時使用

### 練習 3
**情境**：維持每週運動3次的健康習慣

**判斷**：屬於哪個類別？
A. Projects
B. Areas
C. Resources
D. Archives

**答案**：B. Areas

**分析**：
- 沒有「完成」狀態：運動習慣沒有完成的一天
- 需要持續投入：每週都需要運動
- 定期監控和調整：需要定期檢查運動情況
- 是生活的重要領域：健康是核心領域

### 練習 4
**情境**：2023年已完成的年度工作報告

**判斷**：屬於哪個類別？
A. Projects
B. Areas
C. Resources
D. Archives

**答案**：D. Archives

**分析**：
- 已完成：2023年的報告已完成
- 非活躍：不再需要繼續處理
- 歷史記錄：保留作為參考
- 保持系統乾淨：從主系統移除
