# OpenCode 快速调用 / Quick OpenCode Call

> [!tip] 快速调用模式
> 使用预设参数快速调用 OpenCode，无需选择模型和任务类型。

## 🚀 快速执行

<%*
// 使用快速执行函数
// 默认使用 qwen2.5:7b 模型，问答任务
const result = await tp.user.quickExecute(tp, 'qa', 'qwen2.5:7b');
-%>

<% result %>

---

## ⚙️ 快速选项

如需更改默认参数，可以在模板中修改：

| 快捷模式 | 模型 | 说明 |
|----------|------|------|
| 默认问答 | qwen2.5:7b | 日常问答、解释 |
| 编程助手 | qwen2.5-coder:7b | 代码生成、调试 |
| 快速响应 | qwen2.5:3b | 轻量级任务 |
| 高质量 | qwen2.5:14b | 复杂任务、深度分析 |

---

## 📝 自定义预设

如需使用其他预设，修改模板中的参数：

```
<% await tp.user.quickExecute(tp, 'task_type', 'model_name') %>
```

可用任务类型：
- `qa` - 问答
- `generation` - 代码生成
- `analysis` - 分析
- `summarization` - 总结
