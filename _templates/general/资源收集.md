## 🔗 PARA 分类建议

处理后根据资源性质移动到：

| 资源类型 | 目标位置 | 示例 |
|---------|----------|------|
| 技术文档/教程 | `3 Resources/01-Tech/` | Obsidian 插件教程 |
| 学习资源/课程 | `3 Resources/02-Learning/` | Python 课程 |
| 生产力工具/方法 | `3 Resources/03-Productivity/` | 时间管理方法 |
| 兴趣爱好内容 | `3 Resources/04-Interests/` | 摄影/音乐资源 |
| 参考资料/文档 | `3 Resources/05-Reference/` | API 文档 |

### 二级分类建议

根据资源内容进一步分类：

- `01-Tech/`：
  - `01-编程/`
  - `02-AI/`
  - `03-工具/`
  - `04-网络安全/`

- `02-Learning/`：
  - `01-在线课程/`
  - `02-书籍/`
  - `03-论文/`
  - `04-教程/`

- `03-Productivity/`：
  - `01-方法/`
  - `02-工具/`
  - `03-系统/`
