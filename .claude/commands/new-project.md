# New Project

Create a new project note in the PARA structure.

## Usage
```
/new-project Project Name
```

## What is a Project?
- Has a clear outcome/goal
- Has a deadline or end date
- Can be completed or cancelled
- Requires active work

## Examples
- "Website Redesign Q1"
- "Q2 Sales Report"
- "Plan Family Vacation"

## Creates
- `1 Projects/Project Name.md`
- Pre-filled with goals, tasks, and progress tracking
