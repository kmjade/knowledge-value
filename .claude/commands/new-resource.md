# New Resource

Create a new resource note in the PARA structure.

## Usage
```
/new-resource Resource Topic
```

## What is a Resource?
- Topic of interest
- Reference material
- No immediate action required
- For future exploration

## Examples
- Machine Learning
- Productivity Tools
- Recipe Collection
- Travel Ideas

## Creates
- `PARA/Resources/Resource Topic.md`
- Pre-filled with content, links, and ideas sections
