調用 skill-creator 技能來創建新的 Claude Code 技能（Skill）。

## 使用方式

```
/create-skill 技能名稱或描述
```

或直接運行此命令，進入互動式技能創建流程。

## Command vs Skill 的區別

| 類型 | 位置 | 調用方式 | 適用場景 |
|------|------|----------|----------|
| Command | `.claude/commands/` | `/命令名` | 簡單的單步或多步任務 |
| Skill | `.claude/skills/` | 自動觸發或手動調用 | 複雜工作流、需要多檔案配置 |

## 執行步驟

1. 調用 `skill-creator` 技能
2. 根據用戶需求，skill-creator 會引導完成：
   - 技能名稱和描述
   - 觸發條件設置
   - 執行邏輯定義
   - 相關配置檔案創建

## 何時創建 Skill 而非 Command

- 需要自動觸發（基於檔案類型、關鍵詞等）
- 涉及複雜的多步驟工作流
- 需要與外部工具/API 整合
- 需要維護狀態或配置

## 範例

用戶：`/create-skill 自動格式化 Markdown 表格`

執行：調用 skill-creator，引導創建一個能自動檢測並格式化 Markdown 表格的技能。
