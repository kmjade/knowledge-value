帮助创建新的 Claude Code 自定义指令（Command）。

## 使用方式

```
/create-command 指令名称 指令描述
```

或直接运行此命令，交互式创建。

## 什么是 Command

Command 是放在 `.claude/commands/` 目录下的 `.md` 文件，用户可以通过 `/指令名` 直接调用。

## 执行步骤

1. 询问用户以下信息（如未提供）：
   - 指令名称（将作为文件名，如 `my-command.md` → `/my-command`）
   - 指令用途描述
   - 执行步骤
   - 使用示例

2. 按照标准格式创建指令文件：
   ```markdown
   简短描述指令的用途。

   ## 使用方式

   ```
   /指令名 [参数]
   ```

   ## 执行步骤

   1. 步骤一
   2. 步骤二
   ...

   ## 示例

   用户：`/指令名 参数`
   结果：...
   ```

3. 将文件保存到 `.claude/commands/` 目录

## 指令设计原则

- **单一职责**：每个指令只做一件事
- **清晰命名**：名称应直观反映功能
- **提供示例**：帮助用户理解如何使用
- **步骤明确**：执行逻辑要清晰可预测

## 示例

用户：`/create-command daily-summary 生成今日工作总结`

创建：`.claude/commands/daily-summary.md`
