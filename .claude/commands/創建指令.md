幫助創建新的 Claude Code 自定義指令（Command）。

## 使用方式

```
/create-command 指令名稱 指令描述
```

或直接運行此命令，互動式創建。

## 什麼是 Command

Command 是放在 `.claude/commands/` 目錄下的 `.md` 檔案，用戶可以通過 `/指令名` 直接調用。

## 執行步驟

1. 詢問用戶以下資訊（如未提供）：
   - 指令名稱（將作為檔案名，如 `my-command.md` → `/my-command`）
   - 指令用途描述
   - 執行步驟
   - 使用範例

2. 按照標準格式創建指令檔案：
   ```markdown
   簡短描述指令的用途。

   ## 使用方式

   ```
   /指令名 [參數]
   ```

   ## 執行步驟

   1. 步驟一
   2. 步驟二
   ...

   ## 範例

   用戶：`/指令名 參數`
   結果：...
   ```

# 儲存

## 指令設計原則

- **單一職責**：每個指令只做一件事
- **清晰命名**：名稱應直觀反映功能
- **提供範例**：幫助用戶理解如何使用
- **步驟明確**：執行邏輯要清晰可預測

## 範例

用戶：`/create-command daily-summary 生成今日工作總結`

創建：`.claude/commands/daily-summary.md`
