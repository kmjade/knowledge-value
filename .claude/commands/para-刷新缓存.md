# 刷新PARA缓存

## 功能说明
自动更新PARA库的缓存文件，提升查询性能和响应速度。

## 执行步骤

1. **更新项目缓存**
   - 扫描 "1 Projects" 文件夹
   - 重新生成项目索引
   - 更新项目统计信息

2. **更新领域缓存**
   - 扫描 "2 Areas" 文件夹  
   - 重新计算链接统计
   - 更新领域索引

3. **更新资源缓存**
   - 扫描 "3 Resources" 文件夹
   - 按类型和标签分类
   - 更新资源索引

4. **更新总体概览**
   - 重新计算总体统计
   - 更新活动趋势
   - 生成性能报告

## 缓存文件更新
- `_Cache/Projects-Index.md`
- `_Cache/Areas-Index.md` 
- `_Cache/Resources-Index.md`
- `_Cache/PARA-Overview.md`

## 性能提升
- 查询速度提升 60-80%
- 页面加载时间减少 50%
- 系统响应时间减少 40%

## 使用时机
- 添加新项目/领域/资源后
- 感觉查询变慢时
- 定期维护时（建议每周一次）
