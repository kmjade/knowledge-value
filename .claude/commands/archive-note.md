# Archive Note

Move a note to the Archive folder.

## Usage
```
/archive-note [reason]
```

## When to Archive
- Project is completed or cancelled
- Area is no longer relevant
- Resource is outdated
- Information no longer needed

## Process
1. Copy note content to Archive template
2. Add summary and status
3. Move original to `PARA/Archive/`
4. Create link from original location

## What Gets Archived
- ✅ Completed projects
- ⏸️ On-hold projects (temporarily)
- ❌ Cancelled projects
- Outdated resources
- Obsolete areas
