# First-CN-ZH (中文优先)

Review and ensure all content follows Chinese-first principles:

1. **Code**: Comments, commit messages, and documentation in Chinese
2. **Notes**: Note titles, tags, and content in Chinese
3. **Punctuation**: Use Chinese punctuation (，。：；？！"")
4. **Configuration**: Frontmatter and settings can use Chinese

Check the current file/selection and apply Chinese-first conventions where appropriate.
