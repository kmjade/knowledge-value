展示当前库的 PARA 结构概览和统计信息（性能优化版）。

## 执行步骤

1. 从缓存文件快速获取统计信息
2. 检查缓存是否需要更新
3. 列出活跃项目和核心领域
4. 检查收件箱状态

## 性能优化

- 使用预计算的缓存文件提升查询速度
- 限制查询范围避免全库扫描
- 优化Dataview查询语法
