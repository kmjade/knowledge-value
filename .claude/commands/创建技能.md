调用 skill-creator 技能来创建新的 Claude Code 技能（Skill）。

## 使用方式

```
/create-skill 技能名称或描述
```

或直接运行此命令，进入交互式技能创建流程。

## Command vs Skill 的区别

| 类型 | 位置 | 调用方式 | 适用场景 |
|------|------|----------|----------|
| Command | `.claude/commands/` | `/命令名` | 简单的单步或多步任务 |
| Skill | `.claude/skills/` | 自动触发或手动调用 | 复杂工作流、需要多文件配置 |

## 执行步骤

1. 调用 `skill-creator` 技能
2. 根据用户需求，skill-creator 会引导完成：
   - 技能名称和描述
   - 触发条件设置
   - 执行逻辑定义
   - 相关配置文件创建

## 何时创建 Skill 而非 Command

- 需要自动触发（基于文件类型、关键词等）
- 涉及复杂的多步骤工作流
- 需要与外部工具/API 集成
- 需要维护状态或配置

## 示例

用户：`/create-skill 自动格式化 Markdown 表格`

执行：调用 skill-creator，引导创建一个能自动检测并格式化 Markdown 表格的技能。
