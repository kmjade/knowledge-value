# New Area

Create a new area note in the PARA structure.

## Usage
```
/new-area Area Name
```

## What is an Area?
- Ongoing responsibility
- No defined end date
- Requires maintenance over time
- Standards to maintain

## Examples
- Health
- Finance
- Relationships
- Career
- Home

## Creates
- `PARA/Areas/Area Name.md`
- Pre-filled with standards, checklist, and notes
