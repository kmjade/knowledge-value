## 执行步骤

1. 列出 `0 Personals/📥 00_InBox/` 下所有文件和子文件夹内容
2. 分析每个文件的内容和用途
3. 根据以下规则建议归属：
   - 属于某个项目 → `1 Projects/项目名/`
   - 是某领域的知识 → `2 Areas/领域名/`
   - 是通用素材/模板 → `3 Resources/资源名/`
   - 无用内容 → 建议删除
4. 列出整理建议清单，等待用户确认
5. 用户确认后执行移动
