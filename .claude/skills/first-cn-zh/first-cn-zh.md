# First-CN-ZH Skill

This skill enforces Chinese-first (中文优先) practices for both code development and Obsidian note writing.

## Usage

Call this skill when you want to:
- Create or edit content following Chinese-first principles
- Review existing content for Chinese language consistency
- Convert English content to Chinese where appropriate

## Principles

### For Code
- Write comments, documentation, and explanations in Chinese
- Use Chinese for commit messages and PR descriptions
- README and documentation files in Chinese
- Variable names can be in English (for readability) but explanations in Chinese

### For Obsidian Notes
- Create notes with Chinese titles and content
# 標籤
- Frontmatter properties can use Chinese values
- Dataview queries and inline fields in Chinese

### General
- Use Chinese punctuation (，。：；？！"")
- Maintain consistent terminology
- Prefer full-width characters for Chinese text

## Trigger

This is a manual skill. Use the `/first-cn-zh` command to activate it when needed.
