# First-CN-TW Skill

This skill enforces Traditional Chinese-first (繁體中文優先) practices for both code development and Obsidian note writing.

## Usage

Call this skill when you want to:
- Create or edit content following Traditional Chinese-first principles
- Review existing content for Traditional Chinese language consistency
- Convert Simplified Chinese to Traditional Chinese where appropriate

## Principles

### For Code
- Write comments, documentation, and explanations in Traditional Chinese
- Use Traditional Chinese for commit messages and PR descriptions
- README and documentation files in Traditional Chinese
- Variable names can be in English but explanations in Traditional Chinese

### For Obsidian Notes
- Create notes with Traditional Chinese titles and content
- Use Traditional Chinese tags (e.g., #標籤/標籤名)
- Frontmatter properties can use Traditional Chinese values
- Dataview queries and inline fields in Traditional Chinese

### General
- Use Traditional Chinese punctuation (，。：；？！"", but keep as needed)
- Maintain consistent terminology with Traditional Chinese standards
