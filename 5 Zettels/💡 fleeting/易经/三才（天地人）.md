# 三才（天地人）

## 定义

三才，又称"天地人三才"，是中国古代哲学中关于宇宙构成的基本概念，认为宇宙由天、地、人三个基本要素构成，它们相互依存、相互作用，形成一个统一的整体。

## 核心思想

### 三才的内涵

- **天**：指自然界、宇宙、天道、规律
- **地**：指大地、自然万物、物质世界
- **人**：指人类、人类社会、人的主体性

### 三才关系

```
    天（天道规律）
      ↑
      │ (感通)
      │
      │
地（万物载体）←→ 人（万物之灵）
      (承载)
```

**特点**：
1. **天人合一**：天与人相感相通，人应顺应天道
2. **天人相应**：人体与自然相应，天人相应于理
3. **天、地、人并重**：三者缺一不可，构成完整的世界

## 起源与发展

### 起源

# 系統
  - 《说卦传》："立天之道，曰阴与阳；立地之道，曰柔与刚；立人之道，曰仁与义。"

- **道家**：强调人与自然的和谐
  - 道法自然，天人合一

- **儒家**：注重人的主体性和道德修养
  - 仁义礼智，内圣外王

### 历史发展

**先秦时期**
- 《周易》奠定三才理论基础
- 老子、孔子从不同角度阐释三才

**汉代**
- 董仲舒提出"天人感应"
# 系統

**宋明理学**
- 二程、朱熹进一步发展三才理论
- 理气心性之说与三才结合

## 主要流派

### 1. 儒家三才观

**核心观点**：
- 天为道德之源
- 地为万物之母
- 人为万物之灵

**特点**：
- 强调人的道德修养
- 主张"内圣外王"
- 追求天人合一

**代表人物**：孔子、孟子、朱熹

### 2. 道家三才观

**核心观点**：
- 天道自然
- 地法自然
- 人法自然

**特点**：
- 强调顺应自然
- 追求无为而治
- 返璞归真

**代表人物**：老子、庄子

### 3. 医家三才观

**核心观点**：
- 天为阳气
- 地为阴气
- 人为阴阳调和

**应用**：
- 中医理论
- 养生保健
- 疾病防治

**特点**：
- 整体观念
- 阴阳平衡
- 辨证施治

## 三才的具体应用

### 1. 中医学

**理论应用**：
- **天**：气候、季节、昼夜变化
- **地**：地理环境、水土、食物
- **人**：体质、情志、生活习惯

**养生原则**：
- 顺应天时：春生夏长秋收冬藏
- 适应地利：因地制宜
- 调和人事：修身养性

### 2. 建筑学

**风水理论**：
- 天文因素：方位、星象
- 地理因素：地形、地貌
- 人文因素：居住者的需求

### 3. 农业生产

**三才农业观**：
- 天时：季节、气候
- 地利：土壤、水源
# 管理

**特点**：因地制宜，因时制宜

### 4. 军事战略

**三才用兵**：
- 天：天时、气象
- 地：地形、地势
- 人：人心、士气

## 现代意义

### 1. 生态哲学

- 人与自然和谐共生
- 可持续发展理念
- 生态文明建设

# 管理

- 天时（机遇）
- 地利（环境）
- 人和（团队）

### 3. 个人修养

- 顺应自然规律
- 注重身心和谐
- 追求全面发展

## 经典语录

1. **《周易·系辞下》**："有天道焉，有人道焉，有地道焉。"

2. **《说卦传》**："立天之道，曰阴与阳；立地之道，曰柔与刚；立人之道，曰仁与义。"

3. **老子《道德经》**："人法地，地法天，天法道，道法自然。"

4. **董仲舒《春秋繁露》**："天、地、人，万物之本也。天生之，地养之，人成之。"
