# 《周易》

## 定义

《周易》，又称《易经》，是中国古代最重要的经典著作之一，位列"五经"之首。它是一部探讨宇宙、人生、社会变化规律的哲学著作，也是占卜和预测的经典文本。

## 基本信息概览

| 项目 | 内容 |
|------|------|
| **成书年代** | 西周至春秋时期 |
| **作者** | 相传周文王、周公、孔子等 |
| **地位** | 五经之首 |
| **性质** | 哲学著作、占卜经典 |
| **字数** | 约1.5万字（不含传） |

## 核心概念

### 1. 易

"易"有三种含义：
- **不易**：宇宙永恒不变的规律
- **变易**：万物不断变化的本质
- **简易**：大道至简的智慧

### 2. 阴阳

**阴阳是《周易》的核心范畴**：
- 阳：积极、主动、光明、刚强
- 阴：消极、被动、阴暗、柔顺

**阴阳关系**：
- 阴阳对立统一
- 阴阳互根互用
- 阴阳消长转化

### 3. 八卦

**基本符号**：由阴爻（--）和阳爻（—）组成

**八卦构成**：

```
☰ 乾（天）    ☴ 巽（风）    ☶ 艮（山）    ☷ 坤（地）
☱ 兑（泽）    ☲ 离（火）    ☵ 坎（水）    ☳ 震（雷）
```

**八卦含义**：

| 卦名 | 符号 | 自然 | 象征 |
|------|------|------|------|
| 乾 | ☰ | 天 | 刚健、领导、父 |
| 坤 | ☷ | 地 | 柔顺、包容、母 |
| 震 | ☳ | 雷 | 动荡、觉醒、长男 |
| 巽 | ☴ | 风 | 渗透、温顺、长女 |
| 坎 | ☵ | 水 | 险陷、陷溺、中男 |
| 离 | ☲ | 火 | 附丽、明亮、中女 |
| 艮 | ☶ | 山 | 静止、稳重、少男 |
| 兑 | ☱ | 泽 | 喜悦、表达、少女 |

### 4. 六十四卦

**形成**：八卦两两相叠，形成六十四卦

**结构**：
- 上卦（外卦）
- 下卦（内卦）
- 六爻（从下到上：初、二、三、四、五、上）

**示例**：

```
  上卦：☲（离/火）
      ————
      ———
      ———
      ———
      ———
      ————
  下卦：☰（乾/天）

  火天大有卦
```

## 《周易》的结构

### 经（本经）

**内容**：
- 卦画（64卦）
- 卦辞（解释卦义）
- 爻辞（解释爻义）

**特点**：
- 语言古奥，含义深奥
- 占卜的核心文本
- 哲学思想的载体

### 传（十翼）

**相传孔子所作，用以解释《周易》**：

1. **《彖传》** - 解释卦辞
2. **《象传》** - 解释卦象和爻象
3. **《系辞传》** - 总论《周易》哲学
4. **《文言传》** - 专门解释乾坤两卦
5. **《说卦传》** - 解释八卦
6. **《序卦传》** - 解释六十四卦排列顺序
7. **《杂卦传》** - 杂论六十四卦关系

## 核心思想

### 1. 变易之道

**核心观点**：宇宙万物都在不断变化

**重要概念**：
- 穷则变，变则通，通则久
- 物极必反
- 否极泰来

### 2. 阴阳平衡

**核心观点**：阴阳协调是理想状态

**应用**：
- 阴阳失调导致失衡
- 阴阳调和达到平衡
- 中庸之道

### 3. 三才之道

**参见**：[[三才（天地人）]]

- 天道：阴阳
- 地道：柔刚
- 人道：仁义

### 4. 君子之道

**修养原则**：
- 自强不息（乾卦）
- 厚德载物（坤卦）
- 进退有度
- 守正不阿

## 实际应用

### 1. 哲学思辨

**思想价值**：
- 变化观：认识变化的规律
- 整体观：万物相互联系
- 辩证观：阴阳对立统一

**影响领域**：
- 中国哲学
- 军事战略
- 政治治理
- 文化艺术

### 2. 占卜预测

**占卜方法**：
- 摇卦
- 筮法
- 梅花易数

**占卜原则**：
- 诚心正意
- 结合实际
- 不可沉迷

### 3. 养生保健

**中医理论**：
- 阴阳平衡是健康基础
- 五行相生相克
- 气血调和

**养生方法**：
- 顺应四时
- 调和阴阳
- 情志舒畅

### 4. 管理决策

**管理智慧**：
- 识别时机
- 权衡利弊
- 应对变化

## 经典卦例

### 乾卦（天行健）

**卦辞**："元亨利贞"

**核心思想**：
- 自强不息
- 积极进取
- 刚健有为

**应用**：领导、创业、发展

### 坤卦（地势坤）

**卦辞**："元亨，利牝马之贞"

**核心思想**：
- 厚德载物
- 柔顺包容
- 承载万物

**应用**：合作、包容、稳定

## 学习方法

### 初学者

1. **了解基本概念**：阴阳、八卦、六十四卦
2. **研读原文**：先读卦辞、爻辞
3. **参考注解**：阅读《周易正义》等注本
4. **结合实际**：联系生活理解

### 进阶者

1. **深入研究**：学习十翼
2. **哲学思辨**：理解易理
3. **比较研究**：与其他经典对照
4. **实践应用**：应用于决策分析

## 历史影响

### 对中国文化的影响

- **哲学思想**：奠定中国哲学基础
- **医学理论**：中医阴阳五行理论
- **建筑风水**：建筑布局与方位
- **艺术创作**：书画、音乐、文学
- **处世智慧**：为人处世的指导

### 对世界的影响

- **韩国**：易学深入文化
- **日本**：易学应用于企业
- **西方**：作为哲学思想研究
- **现代**：与现代科学结合（如计算机二进制）
