# 光貓賬戶權限體系

## 概述

電信光貓使用雙賬戶系統來控制訪問權限，這是光貓設計中的重要安全機制。

## 兩個賬戶類型

# 管理

# 修改

# 修改
# 配置
# 配置

### 2. useradmin（普通用戶賬戶）

- **權限範圍**：
# 配置
# 查看
  - 有限的設置權限
- **密碼獲取**：電信提供，在設備標籤上

## 賬戶限制

- 用戶名和密碼長度：最多 16 個字符
- 不允許包含空格
- 權限分離是安全設計的一部分
