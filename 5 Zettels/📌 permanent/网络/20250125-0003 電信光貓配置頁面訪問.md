# 電信光貓配置頁面訪問

## 概述

訪問電信光貓的配置頁面需要特定步驟和正確的地址，這是獲取超級密碼的關鍵步驟。

## 訪問步驟

### 1. 確定光貓 IP 地址

默認地址：`192.168.1.1`

檢查方法：
- Windows: 使用 `ipconfig` 命令查看默認網關
- macOS/Linux: 使用 `ifconfig` 或 `ip route` 查看
- 可能被修改過，需要實際檢查

### 2. 登錄 useradmin 賬戶

使用 useradmin 賬戶登錄光貓管理界面（不要關閉登錄窗口）

### 3. 訪問配置頁面

直接訪問以下 CGI 接口：
```
http://[光貓IP]/cgi-bin/baseinfoSet.cgi
```

示例：
```
http://192.168.1.1/cgi-bin/baseinfoSet.cgi
```

### 4. 查看配置信息

在配置頁面中可以看到以下關鍵信息：

```json
{
  "baseinfoSet_TELECOMACCOUNT": "telecomadmin",
  "baseinfoSet_TELECOMPASSWORD": "[加密的密碼數據]"
}
```

## CGI 接口說明

- **路徑**：`/cgi-bin/baseinfoSet.cgi`
- **用途**：獲取光貓基本配置信息
- **權限要求**：需要先登錄 useradmin 賬戶
- **數據格式**：JSON 格式

## 注意事項

> [!important] 重要
> - 必須先登錄 useradmin 賬戶
> - 不要關閉登錄窗口
> - 直接訪問 CGI 接口，不通過管理界面導航
