# 光貓密碼解密技術

## 概述

電信光貓的 telecomadmin 密碼使用 ASCII 編碼 + 凱撒密碼（位移 4）進行加密，需要解密後才能使用。

## 解密流程

### 1. 獲取加密密碼

訪問光貓配置頁面：
```
http://192.168.1.1/cgi-bin/baseinfoSet.cgi
```

找到以下字段：
```json
"baseinfoSet_TELECOMPASSWORD":"120&105&112&105&103&115&113&101&104&113&109&114&57&56&49&54&56&56&53&54&"
```

### 2. ASCII 轉換

將數字轉換為 ASCII 字符：

```python
# 示例代碼
a = [120, 105, 112, 105, 103, 115, 113, 101, 104, 113, 109, 114, 57, 56, 49, 54, 56, 56, 53, 54]
for i in a:
    print(chr(i))
```

輸出結果：`xipigsqehqmr98168856`

### 3. 凱撒密碼解密

使用**位移為 4 的凱撒密碼**進行解密：

| 加密字符 | 解密後 | 說明 |
|---------|--------|------|
| x | t | 位移 -4 |
| i | e | 位移 -4 |
| p | l | 位移 -4 |
| i | e | 位移 -4 |
| g | c | 位移 -4 |
| s | o | 位移 -4 |
| q | m | 位移 -4 |
| e | a | 位移 -4 |
| h | d | 位移 -4 |
| q | m | 位移 -4 |
| m | i | 位移 -4 |
| r | n | 位移 -4 |

數字部分保持不變。

### 4. 組合最終密碼

將解密後的前綴 + "telecomadmin" + 數字部分組合：

```
telecomadmin + 98168856 = telecomadmin98168856
```

## 加密原理

這種加密方式屬於**簡單的對稱加密**：
- 第一層：ASCII 數值編碼
- 第二層：凱撒密碼（位移 4）
- 安全性：較低，容易被破解
