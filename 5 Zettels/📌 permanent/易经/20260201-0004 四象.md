## 🎯 四象的构成

### 形成原理

```
太极
  └─→ 两仪（阴阳）
       ├─→ 阳爻 (⚊)
       │    ├─→ 阳阳 = 太阳 (⚌)
       │    └─→ 阳阴 = 少阴 (⚍)
       │
       └─→ 阴爻 (⚋)
            ├─→ 阴阳 = 少阳 (⚎)
            └─→ 阴阴 = 太阴 (⚏)
```

### 符号说明

- **太阳 (⚌)**：两个阳爻重叠，至阳之象
- **少阴 (⚍)**：上阳下阴，阳多阴少
- **少阳 (⚎)**：上阴下阳，阴多阳少
- **太阴 (⚏)**：两个阴爻重叠，至阴之象

## 🌍 四象的意义

### 1. 宇宙演化的阶段

四象体现了宇宙生成的四个层次：

| 层次 | 概念 | 状态 | 含义 |
|------|------|------|------|
| 0 | 无极 | 虚无 | 宇宙之前的状态 |
| 1 | 太极 | 混沌 | 阴阳未分的本源 |
| 2 | 两仪 | 分化 | 阴阳开始分化 |
| 3 | 四象 | 展开 | 阴阳进一步组合 |
| 4 | 八卦 | 形成 | 万物开始显现 |

### 2. 阴阳的动态平衡

- **太阳**：阳极
- **少阴**：阳转阴的开始
- **少阳**：阴转阳的开始
- **太阴**：阴极

### 3. 时间和空间的维度

- **时间**：四季（春夏秋冬）
- **空间**：四方位（东南西北）
- **状态**：四种基本能量状态

## 🔗 四象与八卦的关系

### 四象生八卦

每个四象加上一个阴爻或阳爻，形成两个八卦：

```
太阳 (⚌)
  ├─→ 阳 = 乾卦 (☰)
  └─→ 阴 = 兑卦 (☱)

少阴 (⚍)
  ├─→ 阳 = 离卦 (☲)
  └─→ 阴 = 震卦 (☳)

少阳 (⚎)
  ├─→ 阳 = 巽卦 (☴)
  └─→ 阴 = 坎卦 (☵)

太阴 (⚏)
  ├─→ 阳 = 艮卦 (☶)
  └─→ 阴 = 坤卦 (☷)
```

### 八卦的阴阳属性

| 四象 | 衍生的八卦 | 共同特征 |
|------|-----------|---------|
| 太阳 | 乾、兑 | 金行属性，刚健 |
| 少阴 | 离、震 | 火、木行，生发 |
| 少阳 | 巽、坎 | 木、水行，柔顺 |
| 太阴 | 艮、坤 | 土行属性，包容 |

## 🎨 四象图

### 先天四象图

```
    太阳 (南)
      ☺
少阴   少阳
☻      ☼
    太阴 (北)
      ☽
```

**含义**：
- 太阳在南：天阳之极
- 太阴在北：地阴之极
- 少阴在东：东方生发
- 少阳在西：西方收敛

### 四象循环

```
春 → 夏 → 秋 → 冬 → 春
少阴 → 太阳 → 少阳 → 太阴 → 少阴
```
