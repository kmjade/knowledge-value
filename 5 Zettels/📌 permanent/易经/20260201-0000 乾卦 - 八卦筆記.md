# 乾卦

> [!info] 八卦之一 - 乾為天，代表陽與創造
> **別名**：乾天、天卦、龍卦

---

## 🔢 核心概念

### 基本屬性
- **[[3 Resources/02-Learning/易经/核心概念/阴阳]]** - 陽爻（一畫，象天）
- **[[3 Resources/02-Learning/易经/核心概念/八卦]]** - 由三陽爻組成的八種基本符號

### 組構原理
- **先天八卦** - 伏羲八卦方位（乾南坤艮兌巽離）
- **文王八卦** - 文王八卦方位（乾南坎艮兌離）
- **後天八卦** - 上下卦組合（上卦+下卦）

## 🔗 離爻組成

### 初爻（底畫）
```
☰ (陽/天) - 炁、健、元、剛
```
**象徵**：陽之始，萬物資始，創造之元
**應用**：
- 事業開始 - 頴陽之志，確立目標
- 新年規劃 - 規劃年度規劃和目標

---

### 二爻（中畫）
```
☷ (地/柔) - 藏、順、靜、安
```
**象徵**：柔中帶剛，以柔順物，保持中道
**應用**：
- 隨時適應 - 逢凶化吉，以退為進，守柔得中
- 謉承上級 - 協助陽爻，保持連貫

---

### 三爻（上畫）
```
☳ (天/健) - 剛、剛、動、悅
```
**象徵**：君子終日乾乾，剛健不息
**應用**：
- 持續進取 - 終不懈怠，保持前進
- 正確決策 - 遇循正道，果斷行為

---

## 📚 卦辭速

### 納辭
```
乾：元亨利貞（大吉大利貞）
```
**解讀**：元，亨也，利貞也，首吉
**卦理**：乾卦為六十四卦之首，大通大利貞，是為上上卦之象

## 🧠 應理筆記

### 讀憶體驗
> [!question] 你認為乾卦最能代表什麼樣的人格特徵？
