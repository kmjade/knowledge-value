## 资源概述

Obsidian 是一个基于 Markdown 的知识管理工具，具有以下特点：

### 核心特性

- **双向链接**：在笔记之间建立连接
- **图谱视图**：可视化知识网络
- **插件系统**：丰富的社区插件
- **本地存储**：数据完全在本地

### 推荐插件

| 插件名称 | 用途 |
|----------|------|
| Dataview | 数据查询和展示 |
| Templater | 模板引擎 |
| QuickAdd | 快速添加笔记 |
| Graph Analysis | 图谱分析 |
