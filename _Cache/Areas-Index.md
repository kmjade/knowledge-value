## 关联项目

```dataview
TABLE without ID
  file.link AS "领域",
  file.outlinks.length AS "链接数",
  filter(file.outlinks, (l) => startswith(l.path, "1 Projects")) AS "关联项目"
FROM "2 Areas"
WHERE file.name != this.file.name
SORT file.outlinks.length DESC
LIMIT 10
```
