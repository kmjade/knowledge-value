# 📑 Index

> 字母索引 - 按字母顺序组织的快速查找工具

## 📝 什么是 Index？

Index 是一种按照字母顺序组织的笔记列表，提供：

- 快速的内容查找
- 主题的字母分类
- 知识点的快速定位

## 🎯 使用场景

- 记得笔记名称但不确定位置
- 想要浏览所有以特定字母开头的笔记
- 快速定位特定主题

## 📚 如何使用

1. 选择目标首字母
2. 浏览该字母下的笔记列表
3. 点击链接跳转到目标笔记

## 💡 提示

Index 通常与 MOC 配合使用：
- MOC 用于按主题探索
- Index 用于按名称查找
