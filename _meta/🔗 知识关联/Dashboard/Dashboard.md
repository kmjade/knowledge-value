# 📊 Dashboard

> 知识库仪表板 - 提供整体概览和快速访问

## 📝 什么是 Dashboard？

Dashboard 是知识库的控制中心，提供：

- 整体结构概览
- 重要内容的快速访问
- 最近更新的笔记
- 知识库统计数据

## 🎯 核心功能

- **导航中心**：快速访问各个 PARA 区域
- **最近更新**：查看最近修改的内容
- **统计信息**：了解知识库规模和增长
- **快捷方式**：常用笔记的快速入口

## 📚 视图选项

1. **概览视图**：展示所有主要分类
2. **时间视图**：按时间顺序查看内容
3. **统计视图**：查看知识库的量化数据

## 💡 使用建议

- 将 Dashboard 设置为首页
- 定期查看以了解知识库状态
- 根据需要自定义显示内容
