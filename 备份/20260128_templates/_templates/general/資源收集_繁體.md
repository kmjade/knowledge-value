## 🔗 PARA 分類建議

處理後根據資源性質移動到：

| 資源類型 | 目標位置 | 範例 |
|---------|----------|------|
| 技術文件/教學 | `3 Resources/01-Tech/` | Obsidian 外掛教學 |
| 學習資源/課程 | `3 Resources/02-Learning/` | Python 課程 |
| 生產力工具/方法 | `3 Resources/03-Productivity/` | 時間管理方法 |
| 興趣愛好內容 | `3 Resources/04-Interests/` | 攝影/音樂資源 |
| 參考資料/文件 | `3 Resources/05-Reference/` | API 文件 |

### 二級分類建議

根據資源內容進一步分類：

- `01-Tech/`：
  - `01-程式設計/`
  - `02-AI/`
  - `03-工具/`
  - `04-網路安全/`

- `02-Learning/`：
  - `01-線上課程/`
  - `02-書籍/`
  - `03-論文/`
  - `04-教學/`

- `03-Productivity/`：
  - `01-方法/`
  - `02-工具/`
  - `03-系統/`
  - `04-工作流/`
